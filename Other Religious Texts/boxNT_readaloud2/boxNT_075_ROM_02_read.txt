Oroomusa.
2.
Ũ wé nùpue lée nùpue á wee cítí bìa ká, á ũ ɲúhṹ máa lé, fo wee síiní ho ũten. Lé bìo á bìa fo wee cítí, bán wárá ɓàn síi lé hĩ̀a ũnɛ́n mún wee wé. 
À wa ɓɛ̀n zũ le bìa wee wé hɔ̃́n wárá so síi á le Dónbeenì á à cítí làa bìo térénna. 
À ũnɛ́n na wee cítí ɓa nùpua à ũ ɲɔǹ wee wé ɓa wárá ɓàn síi, fo so wee leéka le ũ ɲúhṹ á à lé le Dónbeenì cítíi yi le? 
Tàá bìo le Dónbeenì sĩi se, á wee ɲì mí yi, á zũ mu bè-tente wéró, á fo so wee lá làa bè-kã́amáa le? Zũń le lé bṹn tentemu so wee ɲàa fo à ũ yèrèmá ũ yilera lè ũ wárá. 
Àwa, á bìo fo yí wi ũ ɲí dɛ̀ɛ, á yí wi ũ yèrèmá ũ yilera lè ũ wárá, á le lònbe-sũmúi lé dìo fo ɓɛ̀n ǹ dé ũten yi le Dónbeenì sĩ-cĩ̀ílè wizonle zoǹ. Bṹn zoǹ ká le è fĩ̀ le cítíi làa bìo térénna. 
Le è sàání ɓa nùpua yi á à héha lè ɓa bè-wénia. 
Bìa wee wé mu bè-tentewà làa sòobɛ́ɛ, á wee cà ho cùkú lè le yèni le Dónbeenì cɔ̃́n, lè le mukãnì binbirì, bán le è na le mukãnì na máa vé yi. 
Ká bìa kã́a le Dónbeenì yi, na pã́ ho tũ̀iá poni à ɓa tà mu bè-kora bìo, bán le è zéenínáa le mí yi wee cĩ̀ ɓa yi làa sòobɛ́ɛ. 
Le lònbee lè le yi vɛ́ɛ á à yí bìa ɓúenɓúen na wee wé mu bè-kora. Mu ù ɓúaká làa bìa lé ɓa *zúifùwa, á à dãnáa bìa yínɔń ɓa zúifùwa. 
Ɛ̀ɛ ká bìa wee wé mu bè-tentewà, bán le Dónbeenì á à dé ho cùkú yi, le è wé á ɓa à yí le kɔ̃̀nbii á à kɛń ho hɛ́ɛrà yi. Le nín-yání ì wé mu làa bìa lé ɓa zúifùwa, á bìa yínɔń ɓa zúifùwa màhã́ à sĩ̀. 
Mu bon, le Dónbeenì yí máa hueeka nùpua mín yi. 
Lé bṹn nɔn á bìa ɓúenɓúen na wó mu bè-kora à ɓa yí zũ a *Mɔyiize làndá, bán húmú bìo á làndá níi mía yi. Ká bìa zũ ho làndá mu à ɓa wee wé mu bè-kora, bán á à cítí lè ho làndá. 
Mu bon, bìa térénna le Dónbeenì yahó yínɔń bìa wee ɲí a Mɔyiize làndá mí dòn ká ɓa máa bè ho yi. Bìa wee wé bìo ho làndá le ɓa wé, bán lé bìa le Dónbeenì wee wé à ɓa térén le yahó. 
Hen ká bìa yínɔń ɓa zúifùwa na ó o Mɔyiize làndá mía cɔ̃́n á wee wé bìo ho làndá bò, se ho làndá mu wi ɓa yiwa. 
Bṹn wee zéení le bìo ho làndá le ɓa wé wé á wi ɓa sĩa yi, á ɓa yiwa lé mu sɛɛ́ràsa. Lé bìo á pã̀ahṹ ɓúi à ɓarɛ́n míten yilera pĩ́ ɓa bìo, ká pã̀ahṹ ɓúi à hã tà ɓa ɲii. 
Le Dónbeenì cítíi fĩ̀ nɔ̀nzoǹ, ká mu bìo á à zũń. Bṹn zoǹ ká le è dĩ̀n o Yeesu *Krista wán á à cítíráa ɓa nùpua ɓa bìo ɓúenɓúen na sà yi dã́ní yi, làa bìo le Dónbeenì bín-tente bíaráa mu, dìo á ĩ wee bue. 
Ɛ̀ɛ ká ũnɛ́n na lé o zúifù, á zũ a *Mɔyiize làndá, á le ũ sĩi wan le Dónbeenì bìo yi, 
fo zũ bìo sĩ le Dónbeenì yi. Lé ho làndá nɔn á fo zũnáa bìo se. 
Fo láa mu yi le fo lé ɓa muiiwà ya-dí, á lé bìa wi le tíbírí yi khoomu. 
Fo le fo lé bìa yí zũ mu bìo nì-kàránlo, á lé ɓa háyúwá nì-kàránlo. Lé bìo fo zũ le mu bìo zũńló lè ho tũ̀iá poni á wi ho làndá yi. 
Àwa, ũnɛ́n na màhã́ le fo lé bìa ká nì-kàránlo ɓɛ̀n wó kaka yí máa kàránnáa ũten? Fo wee hení mu na ɓa nùpua yi le ɓa yí ɲuaa bìo, à ũnɛ́n màhã́ wee ɲuaa le we? 
Fo wee kàrán ɓa nùpua le ho hã́-fénló lè ho bá-fénló ɓa khí yi, à ũnɛ́n màhã́ wee wé mu le we? Fo ɲina hã wɔ̃̀nna à ũ màhã́ wee zoo ɲuaa mu bìo hã wɔ̃̀nna zĩní yi le we? 
Fo wee yòoní ũten ho làndá bìo yi, à le Dónbeenì yèni dén ɓɛ̀n wee yáa bìo fo yí máa bè ho làndá mu yi bìo yi. 
Mu bon, bìo kà lé bìo túara le Dónbeenì bíonì vũahṹ yi: «Lé minɛ́n bìo yi á bìa yí zũ le Dónbeenì wee yáaráa le yèni.» 
Mu bon, ká fo wee wé bìo ho làndá bò, se ũ *kúiiró bìo ɲúhṹ wi. Ká fo *kúio à ũ màhã́ yí máa bè bìo ho làndá bò henía yi, se fo ka lòn nùpue na yí kúio. 
Ká yìa yí kúio ò o ɓɛ̀n wee bè bìo ho làndá bò henía yi, á le Dónbeenì so máa càtí ɓànso lòn nùpue na kúio le? 
Á yìa yí kúio ò o wee bè bìo ho làndá bò henía yi lé wón khíi cítí ũnɛ́n na yí máa bè ho yi, ká ũnɛ́n ho làndá mu ɲɔǹ wã́a túara nɔn yi, á fo mún kúio. 
Mu bon, mu yínɔń o nùpue sãnía wee wé ɓànso là a zúifù, á mu mún yínɔń ho kúiiró wee wé o là a zúifù. 
Ká a zúifù binbirì lé yìa lé orɛ́n zoó mí sĩi yi. Á ho kúiiró binbirì, hón lé le sĩi na yèrèmáa. Bṹn lé le Dónbeenì Hácírí wee wé mu, ká yínɔń ho kúiiró na ó o Mɔyiize làndá bò. O zúifù binbirì yí máa yí le yèni nùpua cɔ̃́n. Le Dónbeenì lé dìo ó o wee yí le yèni cɔ̃́n.
