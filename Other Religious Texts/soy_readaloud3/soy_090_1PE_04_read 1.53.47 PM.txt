4.
Kirisi yɛ uipiŋɛ kɛ́mɛɛ ntóósi konlɛ. Lɛ̃ nnyɑ, ɑni ukɛmúŋɛ́ kɛ̃ kɛcɔ m múílɛ̃ nɔn nɛ mɛ̃́ tɛ nɔkɛ́ íwɛ li. Nkó yɛɛ uipiŋɛ kɛ́mɛɛ ntóósi n kṍ yɛ̀ɛ̀ felɛ uu ɑkópɛ piwɑi yɑ́. 
Lɛ̃ nnyɑ, kɑí nɛ nɛni n kpísi, ɑ́i pisɛ rɛ nɔkɛ́ Uléécɑɑ mɛlɑ n tikilɛ̃ nɔn kɔ nɛ ituŋɛ tɔrɔɔ nnyí kɑi nní kɛtẽ kɛcɑ́ɑ́ nɔ́ n tisɛ nɔ́inɛ́soi le. Áni kɑpɛ lɛ̃ kɛ nɔ́inɛ́piŋɛ ii n kɑ́ipinkɛɛ n tikilɛ̃. 
Asei kɛcɑ́ɑ́, nɔ pɛɛ mɛkɛɛ pɛpɛɛ ɑ́pi Uléécɑɑ n wuru mɛlɑ mɛyɑ̃́ wɑlɛ, nɔn mɛ́kpɛrinkpɛwɑi nɛ pikɑ́ipinkɛɛ nɛ pítɑnirɛ́ nɛ ilukɛ picɑɑinkɛɛ nɛ pítɑnirɛ́ ikɛŋɛnɛ wɑi nɔn piléé nyɔ́ɔnsɛ. 
Li nkpéni lipite pɛ̃ nnɔ́ɔ yilɛ rɛ ɑ́ni yɛ kɔ mɛwɑi kópɛ mɛ̃ piwɑi rikpɑ́ pimɛcɔ. Lɛ̃ nnyɑ kɑpi nɔ́ lɑ́misi. 
Pi yɛ́ kɑm ɑpi pimɛwɑi Uléécɑɑ kɛ́ɛ́nkɛɛ. Uyɛɛ umɛcirɛ cɑ́pinɛlɛnlɛ ukɛ́ nɛ pinyɑ́nnyɑ̃́ nɛ pikpɔkpɔ nnɛ́í túhɑɑnɛ. 
Lɛ̃ nnyɑ kɛ Kirisi uu Nsímɛ́ Kɛcirɛ riyóó uú nɛ pikpɔkpɔ ricuruu pɛ́nɛ. Pi pimɛwɑi kɛcɑ́ɑ́ nɛ pi túhɑɑnɛlɛ yɑrɛ ḿpɑ́ kɛsoipipi kɛ́ye mɛcɔ. Amɑ́ Nfɑ́ɑsɔnɛ nnyɑ, pi yɛ́ nkpéni fe ɑpi Uléécɑɑ kɛ́mɛɛ nfɑ́ɑ m mɑ́. 
Ḿpɑ́ yo kɛtɔ yɛ nnyɑ́ɑ̃ wɑlɛ. Lɛ̃ nnyɑ, ɑni mɛsɔhɔ m mɑ́ nɔn ḿpɑ́ yo wɑi nɛ kɛmúŋɛ́, nɔkɛ́ nɛ piyómɛyɑ́hɑɑ n fe nnyɑ. 
Lɛ̃ nnyɑ, lɛ̃ kɑni yɛ́ n wɑ lɛɛ rɛ ɑni mɛsɛ́rɛ ńsɔnɛ n lɑkɑɑnɛ. Ńlɑ yɛ̀ɛ̀ ɑkópɛ mɛyɑ̃ tinnɛlɛ. 
Ani kusɑ́nɛ n wɑpɑɑnɛ. Áni kɑpɛ yɛ kɛfɑ-i n ŋmɔpu nɔn nɛ pisɑ́nɛ yɔ́su. 
Uléécɑɑ yɛ ḿpɑ́ nɔ́ úye linyinɛ hɛlɛ nɛ limpehẽ. Lɛ̃ nnyɑ, ḿpɑ́ úye ukɛ́ ulihɛɛhɛɛ kpísi uú nɛ picɔ n lɛ́ni. Nɔn lɛ̃ n wɑi, nɔ pɛɛ lɛ̃ pisoi lɛ pɛɛ Uléécɑɑ lihɛɛhɛɛ ńsɔnɛ m múílɛ̃. 
Lin úye n hɛ rɛ ukɛ́ yɛ nsímɛ́ riyóó, liute ukɛ́ yɛ nɛ Uléécɑɑ nnɔɔ símisi. Lin úye n hɛ rɛ ukɛ́ yɛ picɔ lɛ̃́, liute ukɛ́ ńnɑŋɛ mmɛ̃ kɛ Uléécɑɑ uu u n hɛ nɛ picɔ n lɛ́ni Uléécɑɑ ukɛ́ nɛ ḿpɑ́ yo kɛ́mɛɛ Yeesu Kirisi rinyiri nnyɑ ríyu yɛ̃. Yeesu Kirisi uyɛɛ ḿpɑ́ píyei-pɔ mɛyɔ́ɔpi nɛ ńnɑŋɛ te! Amí. 
Pinɛ́sɑ́nɛ lɑlɑ kɛcirɛ, nɔ́kɛnɛ́wuu ɑ́kɛ kɑpɛ tɔ nɛ lɛ̃ kɑi nní nɛ ḿpɑpɛ nɔ́ m peikɛɛ yɑrɛ nnɑ mɛɛ nɔ́kɛnɛ́cɑ́ɑ́ kɔɔnu. Ái kɑpɛ nɔ́ n we yɑrɛ lisɑ́ŋɑ linyinɛ yɛ ni. 
Amɑ́ likɛ́ mɛyɑ̃́ nɔ́ n lɑ́ɑ́rú rɛ Kirisi iwɛ icɔ kɑni le nnyɑ. Limɛcɔ kɑi sɔ́nti likɛ́ nɔ́ n lɑ́ɑ́rú, kumúŋɛ́ kpɛ-i kuú hɑ umɛyɔɔpi ń lesɛnɛ ukɛ́ pisoi nyísɛ. 
Pisoi pin Kirisi nnyɑ nɔ́ n lɑ́misi, likɛ́ nɔ́ n lɑ́ɑ́rú rɛ Uléécɑɑ Nfɑɑsɔnɛ, Nfɑ́ɑyukɔɔ́ yɛ nɔ́kɛnɛ́mɛɛ́ welɛ. 
Nɔ́ úkɑ úu kɑpɛ íwɛ n le rɛ u usoi kpu nɛ́ɛ u linyinɛ yɑ́ɑi nɛ́ɛ u mɛwɑi kópɛ mɛnyinɛ wɑ nɛ́ɛ u picɔ nsímɛ́ nnyinɛ kɛ́mɛɛ tɑ́hɑnkɛɛ nnyɑ. 
Amɑ́ in tɛ unyinɛ yɛ íwɛ le rɛ u Kirisi ukɔ́ nnyɑ, ɑ́i kɑpɛ isɛi u n we. Amɑ́ ukɛ́ mɛníŋɛ rinyíri tɛ̃ nnyɑ Uléécɑɑ ríyu wɑisɛ. 
Ituŋɛ yɛ tulɛ rɛ Uléécɑɑ ukɛ́ nɛ pisoi túhɑɑnɛ. Upikɔ́ kuú nɛ korɑɑnɛnɛ. In tɛ tɔ̃́ kɑi nɛ korɑɑnɛnɛ, íye kɑi yɛ́ hɑ pɛɛ kɛ́tɔ-pɔ pɛpɛɛ ɑ́pi ń lɑ pikɛ́ Uléécɑɑ Nsímɛ́ Kɛcirɛ kutu rińcɔ wɑ? 
Li Uléécɑɑ Nsímɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: In tɛ ɑsei ute riyu pilɑlɛ yɛ pɑpisi, íye kɑi yɛ́ hɑ pɛɛ kɛ́tɔ-pɔ pɛpɛɛ ɑ́pi Uléécɑɑ m pɑkɑrɛlɛ̃ nɛ pikópɛkɔɔ́ wɑ? 
Lɛ̃ nnyɑ, pɛpɛɛ Uléécɑɑ mɛlɑ kɛ́mɛɛ íwɛ n le pikɛ́ nɛ lisɔnɛ piwɑi n nɔŋ́lɛ̃, ɑpi Uléécɑɑ, ɑsei ute yɛɛ pi n wɑ ɑnipɛ-i pimɛcirɛ wɑi.
