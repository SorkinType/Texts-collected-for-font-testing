Epireeyɛ.
11.
Nfɑtɛnɛ yɛ piŋmurɛi lɛ rɛ lɛ̃ kɛ usoi uu n tɑ́lɛ̃ yɛ piwɑi mɑsilɛ, tɛ lɛ̃ kɛ usoi úu n nyɑ́ni yɛ kɔ welɛ. 
Pikpurɛ nfɑtɛnɛ nnyɑ kɑpí pɛɛ ipɑkɑrɛ yɛ̃. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɑri nyu rɛ rinɔ́ɔ kɛ Uléécɑɑ uú nɛ ḿpɑ́ yo wɑ. Ai pɛɛ wɑi rɛ lɛ̃ kɑri nní n nyɑ́ni ɑ́i lɛlɛɛ kúyɛnɛ ń we kɛ́mɛɛ léeri. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Apɛɛli uu Uléécɑɑ inyɔ́ɔnsɛ wɑ ii Kɑyɛɛ ikɔ́ ńsɔnɛ tɔ́su. Mmɛ̃ nnyɑ kɑpi u sée rɛ ɑsei ute. Uléécɑɑ ricuruu yɛɛ uinyɔ́ɔnsɛ iyɛ̃ pɑkɑrɛ rɛ i nyɑmlɛ. Inyɔ́ɔnsɛ iyɛ̃ nnyɑ, ḿpɑ́ kuu nní ń kpu, ukɛcɑ́ɑ́ nsímɛ́ yɛ nɛ welɛ tɔn n kómɛi. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Uléécɑɑ uu Enɔki kpíipɔ úu pɛɛ nkpɔ yɛ̃. Kɛ Uléécɑɑ uu u n kpíipɔ nnyɑ kúu nkpɔ yɛ̃. Uléécɑɑ ukɛ́ kɛlenɛ u n kpíípɔ, Nléécɑɑsimɛ́ yɛ pɛɛ́ mɑɑ rɛ uisoi yɛ nɛ Uléécɑɑ sɑ́lɛ. 
Nfɑtɛnɛ nnsɑ́ n we, úkɑ isoi íi yɛ́ fe ikɛ́ nɛ Uléécɑɑ n sɑ́. Li we rɛ nkó yɛɛ Uléécɑɑ n yɑ́ɑ́si ukɛ́ nɛ kɛfɑ tɛnɛ rɛ Uléécɑɑ yɛ welɛ. U yɛ pɛpɛɛ u n wɛ́ɛ́si ihɛ́ɛ́ pɑlɛ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Nowee uu lɛ̃ kɛ Uléécɑɑ uu u n símisi rɛ li yɛ́ wɑ ŋmurɛi úu kɔ pɛɛ kɑhɑnɛ li n yɛ̃. Kuu Uléécɑɑ riwurɛ m mɑ́ nnyɑ, uu kutɑ́kɑi kɑrii uú nɛ ukɛyɔ pikɔ́ ɑyu lɔlu. Nfɑtɛnɛ mmɛ̃ nnyɑ kuú nɛ kɛtẽ nnɛ́í pisoi ɑkópɛ pɑ, uu pɛɛ kɔ mmɛ̃ nnyɑ pɑnsɛ ɑsei ute. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Apirɑhɑm uu Uléécɑɑ iséi ŋmurɛi uu kɛtẽ kɛ̃ kɑpi n yekeilɛ̃ tɛ ukɛ́ mɑsí n te kɛ́mɛɛ sĩ́. U pɛɛ tɔ́sulɛ úu kɔ nyu yei kuu sĩ́. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kuú kɑ yɑrɛ usɑ́nɛ uu kɛtẽ kɛ̃ kɑpi n yekeilɛ̃ tɛ ukɛ́ mɑsí n te kɛ́mɛɛ́ tonɛ un ɑcɑ́ŋíí kɛ́mɛɛ sói. Isɑɑki nɛ Yɑkupu kɑpi umɛcɔ kɛtẽ kɛ̃ n yekeilɛ̃ yɛ kɔ pɛɛ kei sóilɛ. 
Apirɑhɑm yɛ pɛɛ kuyu kpɛɛ ɑkúi pɑpɛ-pɑpɛ kɛcɑ́ɑ́ ḿ mɔ́mɑɑlɛ̃ mɛ́nlɛ. Uléécɑɑ ricuruu yɛɛ kuyu kpɛ̃ umúŋɛ́, uyɛɛ kɔ kuumɔmɛ. 
Nfɑtɛnɛ mɛɛ kɔ tíyɛsɛ kɛ Sɑrɑ ricuruu uu kɛwɑ̃́ mɑri, ɑŋmɛ̃ nyɛ-i kuu yɛ́ pɛɛ m mɑ́ri ɑn kɔ pitɔ́mpɔ mɑsí. U pɛɛ nyulɛ rɛ uyɛɛ rinɔ́ɔ n yekei yɛ ɑsei ute lɛ. 
Lɛ̃ nnyɑ kɛ sipírɛ̃ ɑsi usoi usɛ yɛɛ kɛkpɔ nɛ nnyɑ́ɑ̃ piwɑi ḿ mɑsí kɛ́mɛɛ léemɛ, sin kulúi we yɑrɛ kɛléécɑɑ ɑwɑrɛpi nɛ mínimɑɑ ritimɛ nnyɛnɛ mɛɛ ńn kúkɛɛ̃ ń we. 
Pinnɛ́í yɛ pɛɛ nfɑtɛnɛ múílɛnlɛ ɑpí weri ɑpí nɛ kpíni ɑ́pi lɛ̃ kɑpi pi n yekeilɛ̃ yɔsí, ɑmɑ́ kɛtɑɑ-pɔ kɑpi mɛnípɛɛ li kpɑsi, ɑi pi lɑrisi ɑpi hɑ́ɑ rɛ pi pisɑ́nɛ nɛ picéetɔŋɛ́ lɛ pɛɛ kɛtẽ kɛcɑ́ɑ́ ń cɛmɛ́lɛ̃. 
Pɛpɛɛ nní lɛ̃ n símisi yɛ mɛyíkíyiki nyísɛlɛnlɛ rɛ kɛtẽ kɑpi wɛ́ɛ́si. 
In pɛɛ kei kɑpi n léeri kɛ pikumííri ɑkú wepɔ, pi yɛ́ ncée yɛ̃ ɑpi kei pɛlɛ. 
Asei kɛcɑ́ɑ́, kɛtẽ kɛɛ nkɛ́ ń fe kɑpí nɛ nɑ́ɑ́pɔ. Lɛlɛɛ kɛyómɛcɑɑ-pɔ kɛkɔ́. Lɛ̃ nnyɑ kɛ isɛi íi Uléécɑɑ we rɛ pikɛ́ u sée rɛ Piuleecɑɑ, kuu nní kuyu n yekeilɛ̃ ukɛ́ pi pɑ nnyɑ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Apirɑhɑm uu Isɑɑki kpísi rɛ ukɛ́ nɛ nyɔ́ɔnsɛ pɛɛ́ uyɛ-i kɑi u m peikɛɛ. Ukɛpipi cirɛníŋɛ kuú pɛɛ lɛ̃ pɑlɛ̃ pikɛ́ nɛ nyɔ́ɔnsɛ, Uléécɑɑ un kɔ pɛɛ́ nɛ ɑnɔ́ɔ u yekei 
uu u mɑɑ rɛ: Isɑɑki kɛ́mɛɛ́ kɑɑ siyɑ́sɛ yɛnunɛ ɑpi yɛ ripɔ́nyíri si séi. 
U pɛɛ tɑ́lɛ̃ tɛ Uléécɑɑ yɛ ńnɑŋɛ mɑ́lɛ hɑ́i un yɛ́ fe uu usoi pikpɔkpɔ kɛ́mɛɛ yukusɛ. Lɛ̃ nnyɑ kɛ Uléécɑɑ uu ukɛpipi u pɛsɛ ɑi wɑi yɑrɛ nkpɔ kɛ́mɛɛ kuu kɛ yukusɛ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Isɑɑki uu Yɑkupu nɛ Esɑu rinɔ́ɔ sɔnɛ ritikisɛ un tɑ́lɛ̃ tɛ li kóso-pɔ lɛ̃ wɑinɛ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ ituŋɛ ii tu rɛ Yɑkupu ukɛ́ kpu, uu Yosɛfi piŋmɑ́nɛ kɛtɛ́ ḿpɑ́ úye rinɔ́ɔ sɔnɛ tíkisɛ, uu ukusɛlɛ kɛcɑ́ɑ́ nyíki uú nɛ Uléécɑɑ ríyu wɑisɛ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Yosɛfi uú pɛɛ kɛkpɔ nyɑhɑi, uu símisi rɛ Isirɑyɛɛli pikɔ́ yɛ́ kɑm ɑpi Esipiti kɛteni-i lelu, uu kɔ símisi lɛ̃ kɑpi yɛ́ mɑsí uɑkɔhɔ n wɑ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Moisi usɑɑ nɛ uuni ɑpi u mɑri ɑpi u pésu iwɑrɛ itɑɑni. Kɑpi n yɛ̃́ tɛ kɛwɑ̃́ yɛ pinyɑ́nɛi nyɑḿ nnyɑ, ɑ́pi isé kɛ uyɔ́ɔpi uu ḿ pɑ iwɑmɛ wɑ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Moisi uu pɑpisi, uu yúlu rɛ ɑ́pi kɑpɛ yɛ u sée rɛ Esipiti Uyɔɔpi ukpére kɛpipi. 
Uu wɛ́ɛ rɛ pikɛ́ u nɛ Uléécɑɑ pikɔ́ kɛsẽ́ íwɛ n wɑi nɛ kuu yɛ́ ituŋɛ keri inyinɛ-i ɑkópɛ kɛ́mɛɛ ńnɑrɛ n kómɛi. 
U pɛɛ yɛ̃́ tɛ ɑsei kɛcɑ́ɑ́, un Kirisi mɛcɔ ɑlɛ̃́ kɛ́mɛɛ́ n we, li mɛmɑ́ píimɑ lɛ ɑi tɔ́su Esipiti kɛtẽ kɛcɑ́ɑ́ lisɔnɛ nnɛ́í. Liriyíkí rɛ ihɛ́ɛ́ yɛɛ u m mɛ́mpɔ kuú pɛɛ nyɑ́nɛipɔ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kuu Esipiti kɛteni-i yisi úu kɔ uyɔ́ɔpi kuwɔi wuri. Liriyíkí rɛ u pɛɛ tɑ́silɛ yɑrɛ u pɛɛ uyɛɛ úu kúyɛnɛ ń we mɛyíkíyiki nyɑ́nilɛ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kuu nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ li, uu ɑnɔnɔɔ mɛ́nyɛ pɔ́riinkɛɛ rɛ uyɛɛ n kóni úu kɑpɛ kɑ ukɛ́ Isirɑyɛɛli pikɔ́ sipipi foí ricɑ. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Isirɑyɛɛli pikɔ́ ɑpi Míni-Wɛɛ́-Nkoi ritɛ́ŋ́ yɑrɛ kɛtẽ kóimɑ. Amɑ́ ɑi Esipiti pikɔ́ pɛɛ pitɛ́ŋ́ ḿ peenu pɔ̃́ muuni. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ Isirɑyɛɛli pikɔ́ ɑpi Yeriko imɛlɛ siyɑ́ɑ sisɛɛi pikɔ́ɔ́nú mɑsí, ii lólɛsi. 
Nfɑtɛnɛ mɛɛ tíyɛsɛ kɛ unɔ́si wɑ́sɑ́nkɑí Rɑhɑpi uu ńsɔnɛ pɛpɛɛ pimúŋɑɑ n hɑ́pɔ yɔsí. Lɛ̃ nnyɑ kɑ́pi u nɛ pɛpɛɛ ɑ́pi Uléécɑɑ m pɑkɑrɛlɛ̃ ripɛ́nɛ pikɛ́ kṍ. 
Ń-ye kɑm yɛ́ kɔ pisímɛ́ rikpɑ́? Nɛn tɛ kɛ́ Seteyɔɔ nɛ Pɑrɑɑki nɛ Sɑmsɔɔ nɛ Yɛfutee nɛ Tɑfiti nɛ Sɑmuyɛɛli nɛ ɑntepuyɛ nkɔ́ símisi kɛ́ tɑ́ŋɛnɛ, ituŋɛ yɛ́ nɛ́ ripɑ́rí. 
Nfɑtɛnɛ yɛ tíyɛsɛlɛ picɔ ɑpi ɑtɔpi wɑi ɑpi siyɔ́ɔpitẽ feriyɛntɛ ɑpi kɔ lɛlɛɛ ɑsei n tíkilɛ̃ wɑi, ɑpi Uléécɑɑ rinɔɔ kuu n yekei yɛnu, ɑpi kɔ ɑ́kinɑɑ́ inɔɔ mɔ́ŋnisɛ, 
ɑpi nnɑ kutoi kopu, ɑpi ŋmɛ́hɛrɛɛ ɑ́pi pi yɛ̃ pikɛ́ nɛ ɑtɛ́hɛ pi pɑrii, ɑpi kufɔ́ŋ finɛ ɑpi limɛmɑ́ɑ́ ɑtɔpi kɛ́mɛɛ rikɔhɔ yɛnu ɑpi kɔ ɑtɔpilee sɑnɛ feriyɛ ɑpi lɑ́kɑsɛ. 
Pinɔ́si pinyinɛ pikɔ́ pɛɛ n kpí, ɑpi pikpɔkpɔ kɛ́mɛɛ yisi ɑpi nfɑ́ɑ kpɑ́. Api pisoi picɔ pɔ̃́ íwɛ píimɑ lukɛisɛ, ɑpi yúlu rɛ ɑ́pi ikɛmɛɛ́ lelu. Pikpɔkpɔ kɛ́mɛɛ mɛyisɛ kɑpi lɛ̃ lɑ pikɛ́ hɑ mɛyíkíyiki yisi nnyɑ. 
Api picɔ pɔ̃́ kɔ nnyei nɑm, ɑpi isɛ́í pi súúkɛɛ, hɑ́i ricuruu ɑpi pi pɑhɑɑnkɛɛ ɑpi kukpɑniilee pi tɑni. 
Api ɑpɑrɛ pi tɑpisi ɑpi kóni, ɑpi pi peikɛɛ, ɑpi pi sɛkɛrɛntɛ ɑpi kériinɛ, ɑpi ɑtɛ́hɛ nɛ pi kóni. Pi pɛɛ tóórinkɛɛlɛ pin ɑsɑ́ŋkonɛ nɛ ɑpoikonɛ tɑ́ntilɛ̃, ɑ́pi líkɑ mɑ́, pisoi pin pi wéékusɛlɛ̃ pin íwɛ pi wɑi. 
Kɛtẽ ɑ́kɛ nɛ sɑ́ rɛ pɛ̃ pikɛ́ kɛkɛmɛɛ́ n we. Pi pɛɛ ɑcɛsi kóimɑ-i nɛ ɑyɔ́pɛ kɛcɑ́ɑ́ nɛ ɑpɑrɛpoo-i nɛ ɑtenhorɛ-i tóórinkɛɛlɛ. 
Pinnɛ́í mmɛ̃ nfɑtɛnɛ yɛ tíyɛsɛlɛ ɑpi pikɛcɑ́ɑ́ nsímɛ́ sɔnɛ símisi, ɑmɑ́ ɑ́pi lɛ̃ kɛ Uléécɑɑ uú pɛɛ n yekei yɛ̃. 
Ái líkɑ nnyɑ, Uléécɑɑ yɛ pɛɛ lisɔnɛ lɛɛ lɛ̃ n fe nɛ rɔ́ lɑlɛ, úu pɛɛ́ lɑ pɛ̃ ŋmɑnɛ pikɛ́ li yɛ̃, tɔ̃́ ɑ́ri kɑpɛ kɛ́mɛɛ n we.
