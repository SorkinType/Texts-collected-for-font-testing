Kɑlɑɑti.
6.
Pimɑ́rɛcɔ, nɔ́ mpí kɛ Nfɑ́ɑsɔnɛ nn nní ń séni, nɔn yɛ unyinɛ n yɛ̃ un ɑkópɛ n wɑi, ɑni yɛ m purú ɑní nɛ ncée sɔnɛ-i u pɛsɛri. Ani yɛ nɔ́mɛnɛ́círɛ tĩ tɛ mɛwɑi mɛ̃ mɛcɔ ɑ́mɛ kɑpɛ nɔ́ lɛɛmɛ. 
Ani ncɔni n lɛ́nɑɑnɛ. Lɛ̃ kɑni yɛ́ nɛ Kirisi isé rintíki yɛ lɛ̃. 
Úye un m músu rɛ úu kusɑ we, kúu kɔ pɛɛ líkɑ tu, umɛcirɛ kuu kírɑɑsɛ. 
Ḿpɑ́ úye ukɛ́ yɛ uricuruu mɛwee ripɑí. Un n yɛ̃ tɛ li rikɔ́kɔri we ukɛ́ umɛcirɛ nɛ múŋɛisɛnɛ, ɑ́i kɑpɛ nɛ picɔ. 
Li we rɛ ḿpɑ́ úye yɛɛ yɛ́ céri uu símisi lɛ̃ kuu n wɑ. 
Li pisɛ rɛ usoi nkó kɑpi Uléécɑɑ Nsímɛ́ n céési ukɛ́ ulikɔ́ nɛ uucélɑɑ kɛcɑ́ɑ́ pípɑí n lɛ́ni. 
Áni kɑpɛ nɔ́mɛnɛ́círɛ kirɑɑsɛ, ɑ́pi yɛ Uléécɑɑ sɛ́nnyi. Lɛ̃ kɛ usoi uu n lukɛsi kuu yɛ kpɑsi. 
Úye un umɛlɑ likɔ́ n lukɛsi, uu umɛlɑ likɔ́ yɛ́nu. Amɑ́ úye un Nfɑ́ɑsɔnɛ likɔ́ n lukɛsi, uu Nfɑ́ɑsɔnɛ mɛlɑ yɛnu, lɛlɛɛ nní nfɑ́ɑ tɛnɛcirɛ́. 
Mɛwɑi sɔnɛ piwɑi ɑ́pi kɑpɛ yɛ rɔ́ pi. Li we rɛ tɔn lɛ̃ piwɑi nɛ n ŋmɔ́ɔ́púlɛ̃, ituŋɛ yɛ sɔ́ntilɛ, ḿpɑ́ úye ukɛ́ lɛ̃ kuu n lukɛsi kpɑsi. 
Lɛ̃ nnyɑ kɑi nkpéni nní lɛ̃ kuwɑi ń we, tɔkɛ́ pisoi nnɛ́í ńlɑ n nyísɛlɛ̃, ɑí nɛ tɔ́su tɔ́ nɛ pɛ̃ tɔ́ɔ Kirisi nɛ kɛfɑ n tɛnɛlɛ̃. 
Ani ripɑí lɛ̃ kɑm nkpéni tinɛ́cúruu nɛ kunɛ́nípɛ nɔ́ n wɔ́i mɛkónɛ mɛkónɛ. 
Mpí nnɛ́í pɛɛ n wɛ́ɛ́si rɛ pisoi pikɛ́ pi pɑkɑrɛ pɛɛ nɔ́ cɔ́ŋlɛ̃ tɛ ɑni kérii. Ápi lɑ pikɛ́ Kirisi nkpɔ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ nnyɑ íwɛ inyinɛ li. 
Pikéké pɛ̃ ricuruu ɑ́pi isé tíkilɛ̃, ɑmɑ́ pi pɛɛ́ lɑ rɛ nɔ́ɔkɛ́ kérii pikɛ́ nɛ nɔ́pinɛ́ké pɛ̃ nnyɑ rikɔ́kɔri n wɑi. 
In nɛ́ɛ, kɛ Upíimɑ Yeesu Kirisi uu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ n kpu ŋmɑnɛ yɛɛ nɛ̃́ tikɔ́kɔri wɑisɛ. Líkɑ licɔ ɑ́i likɛcɑ́ɑ́ kpɑ́lɛ̃. Kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ nkpɔ mɛɛ tíyɛsɛ kɛ kɛtẽ kɛcɑ́ɑ́ likɔ́ ɑ́i kɔ nkpéni kɛnɛ́mɛɛ kulɑ́ɑ kúkɑ mɑ́, ɑ́m kɔ nkpéni pikɑ́ipinkɛɛ li kpɑ́lɛ̃. 
Ái nké nɛ́ɛ nkécirɛ́ yɛɛ linyinɛ yoriyɛ, ɑmɑ́ Uléécɑɑ ukɛ́ rɔ́ pɑnsɛsɛ pisoi fɑlɛ yɛɛ yɛ yoriyɛ. 
Mpí nnɛ́í pɛɛ piisoi kɛ́mɛɛ nsímɛ́ mmú n tíkilɛ̃ kɑḿ nɛ welu rɛ Uléécɑɑ ukɛ́ nkíŋniŋɛ pi hɛ uu kɔ íwɛ pi téni. Nkíŋniŋɛ nɛ píwɛtu pikɛ́ kɔ Uléécɑɑ kuyu Isirɑyɛɛli kɛcɑ́ɑ́ n we. 
Kɑi nɛni nɛ n kpísi, úkɑ úu kɑpɛ picɔ́ŋ nɛ́ rikpɑ́. Iwɔ́i yɛ inɛ́piŋɛ kɛcɑ́ɑ́ nyísɛlɛnlɛ rɛ nɛ Yeesu kɛcɑ́ɑ́ íwɛ lilɛ. 
Pimɑ́rɛcɔ, Urɔ́píimɑ Yeesu Kirisi ukɛ́ ípɛɛlɛ́ɛ nɔ́ nyísɛ. Amí.
