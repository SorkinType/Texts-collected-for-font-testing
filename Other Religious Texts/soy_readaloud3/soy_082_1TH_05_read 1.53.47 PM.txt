1 Tɛsɑloniki.
5.
Pimɑ́rɛcɔ, ɑ́i nkpɑ́ni pisɛ rɛ pikɛ́ kɛyɑ́ɑ kɛ̃ nɛ kumúŋɛ́ kpɛ-i kɛ mɛwɑi mɛ̃ nnɛ́í ɑmɛ n wɑinɛ kɛcɑ́ɑ́ rítɛlɛ́ nɔ́ wɔ́ipɔ. 
Nɔ̃́ ticuruu yɛ nyulɛ ńsɔnɛ rɛ kɛyɑ́ɑ kɛ̃ kɛ Upíimɑ uu n sɔ́nti yɛ rɔ́ kɑ́ikɑinɛ yɑrɛ kɛ uyɑɑ uu yɛ kɛsinɛ rinkɑ́ikɑi. 
Kumúŋɛ́ kpɛ-i kɛ pisoi ɑpi yɛ ḿ mɑɑ rɛ: Nkíŋniŋɛ yɛ tonɛlɛ, ncɔ́ŋ ńkɑ ńn kɔ we, kumúŋɛ́ kpɛ-i kɛ ncɑɑi nn yɛ mɛkɑ́ikɑi pi rikipi, yɑrɛ kɛ mɛmɑ́rɛnlõ ɑmɛ yɛ ufɑni rinkɑ́ikɑi mɛcɔ, piukɑ úu yɛ́ ŋmɛ́hɛrɛɛ. 
Amɑ́ nɔ̃́ pinɛ́mɑ́rɛcɔ, ɑ́ni kuŋmɑhɑ-i we, kɛyɑ́ɑ kɛ̃ ɑ́kɛ yɛ́ kɔ nɔ́ rikɑikɑi yɑrɛ kɛ uyɑɑ uu yɛ rinkɑ́ikɑi. 
Nɔ nɔ́nnɛ́nɛ́í mɛtɛ́í nɛ kɛtúŋɛ́ pikɔ́ lɛ. Áni kɛsinɛ nɛ kuŋmɑhɑ pikɔ́. 
Kɑi nní lɛ̃, ɑ́i pisɛ rɛ tɔkɛ́ picɔ mɛcɔ n lɔni. Amɑ́ li pisɛ rɛ tɔkɛ́ n nyɑ́ni ɑri ḿpɑ́ yo n wɑi nɛ kɛmúŋɛ́. 
Kɛsinɛ kɛ pɛpɛɛ yɛ n lɔpilɛ ɑpi yɛ lɔpilɛ, kɛsinɛ kɛ pítɑnirɛ́ pɔ́ɔ yɛ põ. 
Amɑ́ tɔ̃́ nɛ kɛtúŋɛ́ pikɔ́ lɛ. Li kɔ pisɛ rɛ tɔkɛ́ ḿpɑ́ yo n wɑi nɛ kɛmúŋɛ́. Li pisɛ rɛ tɔkɛ́ nfɑtɛnɛ nɛ ńlɑ kpísi ɑri wɑisɛ yɑrɛ kɛwenitukɑnkɑ kɑpi yɛ nɛ ihímɛ rińloró, ɑri kɔ pirɔ́yulɑlɛ kɑrí nɛ n tɑ́lɛ̃ kpísi ɑri kípi yɑrɛ ritɔpikɔŋ. 
Uléécɑɑ úu rɔ́ wɑ rɛ ukɛ́ ukuwɔi kɛrɔ́cɑ́ɑ́ n wɑilɛ̃, ɑmɑ́ u lɑlɛ tɔkɛ́ nɛ Urɔ́píimɑ Yeesu Kirisi kɛfɑ tɛnɛ ukɛ́ nɛ tirɔ́yu lɔ. 
Tɔ̃́ nnyɑ kɛ Kirisi uu kpu rɛ, tɔn n lɔni nɛ́ɛ tɔn n nyɑ́ni, tɔ́ nɛ uyɛ̃ tɔkɛ́ kɛsẽ́ nfɑ́ɑ m mɑ́. 
Kɑi nní lɛ̃, ɑni itisi hɛkɑɑnɛ ɑní n ŋmɔ́ɔ́púsɛnɛlɛ̃ yɑrɛ lɛ̃ kɑni nní n wɑpɑɑnɛri mɛcɔ. 
Pimɑ́rɛcɔ, ɑni pɛpɛɛ nɔ́kɛnɛ́mɛɛ n kɛisi, pɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ rɛ pikɛ́ nɔ́pinɛ́kpéenwɑi pin nɔ́ céési m pɑkɑrɛlɛ̃. 
Ani ńlɑ pi nyísɛ ńsɔnɛ ɑni pi m pɑkɑrɛlɛ̃, pikɛi pɛ̃ kɑpi n wɑi nnyɑ. Ani n tũ likɛ́ n lɑ́ɑ́rú. 
Pimɑ́rɛcɔ, tɔ nɛ nɔ́ téni rɛ ɑni pikɑ́nkɔ́ cɛ́si, ɑni piwɑmɛi itisi kpɑ́, ɑni pinɑ́rɛi lɛ́ni, ɑní nɛ pinnɛ́í ísuúlu n le. 
Ani nɔ́mɛnɛ́círɛ tĩ, úkɑ úu kɑpɛ yɛ mɛwɑi kópɛ piwɑi ucɔ cɛɛi. Amɑ́ ɑni ḿpɑ́ píyei n wɛ́ɛ́si rɛ nɔkɛ́ lisɔnɛ n wɑpɑɑnɛ ɑni kɔ pisoi nnɛ́í lisɔnɛ wɑi. 
Nɔ́ɑnɛ́pɔ́ɔ ɑkɛ́ mɛsɛ́rɛ n lɑ́ɑ́rú. 
Ani ḿpɑ́ píyei kɛyómɛ n yɑ́ɑ́si. 
Ani yɛ ḿpɑ́ mɛwɑi mɛ́ye-i Uléécɑɑ yɑ́ɑ́si rɛ nɛ upikɛi. Lɛ̃ kɛ Uléécɑɑ uú nɛ nɔ́ ń lɑ yɛ lɛ̃ nɔ́ nɛ Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ. 
Áni kɑpɛ Nfɑ́ɑsɔnɛ pikɛi tɑ́pisi. 
Áni kɑpɛ yɛ nsímɛ́ kɛ Nfɑ́ɑsɔnɛ nn nɔ́ ń símisi hɑ́ikusɛ. 
Amɑ́ ɑni yɛ ḿpɑ́ yo ńsɔnɛ ripɑí ɑni lɛɛ ń nyɑḿ múlú. 
Ani nɛ nɔ́mɛnɛ́círɛ mɛwɑi kópɛ nnɛ́í mpuri tɔ. 
Uléécɑɑ yɛɛ yɛ nkíŋniŋɛ n hɛ ukɛ́ tíyɛsɛ ɑni pɔ́lɔ́ upikɔ́ ukɛ́ nɔ́inɛ́soi nnɛ́í rimúlú ńsɔnɛ, kɑi n kpísi nfɑ́ɑ kuu nɔ́ ń hɛ nɛ kumííri nɛ ipiŋɛ, ɑ́ni kɑpɛ nɛ ɑkópɛ m mɑ́ kɛyɑ́ɑ kɛ̃ kɛ Urɔ́píimɑ Yeesu Kirisi uu m pɛɛri nnyɑ. 
Uyɛɛ nɔ́ ń sée yɛ li wɑinɛ. Un íye n yekei, lɛ̃ kuu yɛ ritiki. 
Pimɑ́rɛcɔ, ɑni yɛ kɔ kɛrɔ́cɑ́ɑ́ Uléécɑɑ yɑ́ɑ́si. 
Ani pimɑ́rɛcɔ tɔrɔɔ nnɛ́í mɛmɑ́rɛcɔ ripirɑ ɑni yɑ́hɑɑnɛ ńsɔnɛ. 
Nɛ nɛ nɔ́ téni rɛ ɑni Urɔ́píimɑ rinyiri ripɑí ɑni pimɑ́rɛcɔ tɔrɔɔ nnɛ́í rítɛlɛ́ ntí kɛ́ɛ́ni. 
Urɔ́píimɑ Yeesu Kirisi ukɛ́ ípɛɛlɛ́ɛ nɔ́ nyísɛ.
