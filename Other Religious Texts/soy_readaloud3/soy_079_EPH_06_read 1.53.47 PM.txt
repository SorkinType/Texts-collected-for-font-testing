Efɛɛsi.
6.
Nɔ̃́ siwɑ̃́, ɑni nɔ́pinɛ́te m pɑkɑrɛlɛ̃ yɑrɛ kɛ Upíimɑ uu ń lɑ. Lɛlɛɛ nɛ n sɑ́ rɛ nɔkɛ́ wɑ yɛ lɛ̃. 
A yɛ upɔ́sɑ́ɑ nɛ upɔ́ni ríyu wɑisɛ. Tinɔ́ɔ foí yɛ tɛ̃ kɛ Uléécɑɑ uu n hɛ uu rinɔ́ɔ yekei. 
Lɛlɛɛ rɛ li yɛ́ nɛ pɔ́ sɔnɛsi ɑɑ kɔ kɛtẽ kɛcɑ́ɑ́ nfɑ́ɑ tɑɑ-tɑɑ yɛnu. 
Nɔ̃́ siwɑ̃́ pisɑɑ, ɑ́ni kɑpɛ nɔ́sinɛ́wɑ̃́ kɛ́yu m pɑ́ɑ́púsɛ, ɑmɑ́ ɑni si n céési sikɛ́ kɛcírɛ wɑ yɑrɛ kɛ Upíimɑ uu ń lɑ. 
Nɔ̃́ pilɑ́si, ɑni yɛ nɔ́pinɛ́yɔsɑɑ kɛtẽ nté kɛcɑ́ɑ́ pɑkɑrɛ ɑni ɑsei nɛ ríyu pi n wɑisɛlɛ̃ yɑrɛ kɑni Kirisi m pɑkɑrɛlɛ̃. 
Ái pin nɔ́kɛnɛ́kúrí n we pin nɔ́ n nyɑ́ni ŋmɑnɛ kɑni yɛ́ɛ́ lɛ̃ wɑ yɑrɛ nɔ lɑlɛ likɛ́ sisoipipi n lɑ́ɑ́rú. Amɑ́ ɑni pi m pɑkɑrɛlɛ̃ tɛ nɔ Kirisi pilɑsi nnyɑ, ɑní n wɛ́ɛ́si rɛ nɔkɛ́ Uléécɑɑ mɛlɑ nɛ kɛfɑ kɛsɛ n wɑi. 
Ani nɛ mpɔ́ɔnɑrɛ pi n kɛisi yɑrɛ Upíimɑ pilɑsi. Ái kɑpɛ n we yɑrɛ kɛsoipipi kɑni kɛisi. 
Ani n nyu rɛ ḿpɑ́ pɔ́ úye pɔn ulɑ́si nɛ́ɛ pɔn mɛpɔ́círɛ n te, lisɔnɛ lɛ̃ kɑɑ n wɑ kulɑɑ kɑɑ sɔ́nti pɔkɛ́ Upíimɑ kɛ́mɛɛ yɛ̃́. 
Nɔ̃́ píyɔsɑɑ, nɔ́ɔkɛ́ kɔ limɛcɔ nɔ́pinɛ́kɛikɔ́ n wɑi. Ani nɔ́mɛnɛ́círɛ tĩ, ɑ́ni kɑpɛ pi n sɛnkɛɛ nɛ rikpɑ́kɑ́rɑ́. Ani n nyu rɛ nɔ́ nɛ pɛ̃ usɑɑ yɛ kɛyómɛcɑɑ-pɔ welɛ. Úu yɛ usoi kɛyu-i ripɑí uú nɛ n túhɑɑnɛ. 
Ntɔ́rɔɔ yɛ nkpéni mmú rɛ ɑni nɔ́ nɛ Upíimɑ kupɛ́nɛcɔ kɛ́mɛɛ unnɑŋɛ píimɑ kpísi ɑní nɛ m pɑ́ɑ́pú. 
Ani Uléécɑɑ itɔpilũ nnɛ́í kuu nɔ́ m pɑ kpísi ɑní nɛ kuníri ɑpinɛ kɑku nɔ́ n wɑi lɔ́ɔ́ ɑ́ku yɛ́ pɛɛ nɔ́ feriyɛ. 
Ái sisoipipi kɑrí nɛ tɔpu. Amɑ́ lɛlɛɛ iyɔ́ɔpi n tṹ nɛ lɛlɛɛ n-yɔ́ɔpinɑŋɛ m mɑ́ nɛ lɛlɛɛ kɛtẽ nté kuŋmɑhɑ kɛ́mɛɛ ńnɑŋɛ m mɑ́ lin pɛkɛsi kɑrí nɛ tɔpu. Tɔ kɔ nɛ ɑníri kpɑ́kɑ́rɑ́ nyɛɛ kɛyómɛcɑɑ-pɔ ń we tɔpulɛ. 
Lɛ̃ nnyɑ, ɑni nkpéni Uléécɑɑ itɔpilũ nnɛ́í kuu nɔ́ ḿ pɑ kpísi, ɑní nɛ siyɑ́ɑ kópɛ kɛ́mɛɛ tɑ́pisi, ɑ́i yɛ́ pɛɛ nɔ́ feriyɛ. Nɔn n tɑ́pɑɑlɛ̃ nɛ kɛ́tɔ-pɔ, líkɑ ɑ́i yɛ́ pɛɛ fe likɛ́ nɔ́inɛ́piŋɛ riyɔɔsɛ. 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ nɔ́inɛ́piŋɛ riyɔɔsɛ. Ani nsímɛ́sei pisímɛ́ kpísi ɑni wɑisɛ kútɑmpɑ́lɑ́, ɑni kɛhɑlɛ-i ku n télɛ̃. Ani kɔ ɑsei pitiki kpísi ɑni wɑisɛ kɛwenitukɑnkɑ. 
Ani Nsímɛ́ Kɛcirɛ mɛɛ yɛ nɛ nkíŋniŋɛ ń kɑ piyóó n lɑ. Nkɛ́ nɔ́ n we yɑrɛ ɑnɛ́ɛ́ri kɑni tɑnɑɑlɛ̃. 
Ani yɛ ḿpɑ́ yei nɔ́nnɛ́fɑtɛnɛ kpísi yɑrɛ kuwenikpɑ́ncúkú kɑni yɛ́ nɛ ukpɑ́ɑ́ree Setɑni ihimɛ mɛlõ ń kpu. 
Nɔ́pinɛ́yulɑlɛ pikɛ́ ritɔpikɔŋ, Uléécɑɑ Nsímɛ́ nkɛ́ ritɔpitɛ́hɛ kɛ Uléécɑɑ Nfɑɑsɔnɛ nn nɔ́ ḿ pɑ. 
Ani yɛ kɔ lɛ̃ kɛ Uléécɑɑ Nfɑɑsɔnɛ nn nɔ́ n nyísɛ ritiki ɑ́ni nɛ mɛsɛ́rɛ iyɑ́hɑɑ nnɛ́í mpuri n yɑ́ɑ́si nɔn kɔ u welu. Áni kɑpɛ lɔpilɛ, ɑni nɔ́mɛnɛ́círɛ kpísi ɑní pɑ, ɑni mɛsɛ́rɛ nɛ kɛfɑ kɛsɛ upikɔ́ nnɛ́í kɛcɑ́ɑ́ n yɑ́ɑ́si. 
Ani yɛ kɔ kɛnɛ́cɑ́ɑ́ u yɑ́ɑ́si rɛ ukɛ́ tíyɛsɛ kɛ́ yɛ nɛn nnɔ́ɔ n wúkulɛ, kɛ́ nɛ ikɑri n céési pisoi pikɛ́ lɛlɛɛ Nsímɛ́ Kɛcirɛ kɛ́mɛɛ m pékɑɑlɛ̃ ceri. 
Mmɛ̃ kɛ Uléécɑɑ uú nɛ nɛ́ tummɛ rɛ kɛ́ nté ukɛikɔ́ yɛɛ nɛ urinyiri n tṹ. Mmɛ̃ nnyɑ kɑm ɑkpɑnii kɛ́mɛɛ we. Nɛ lɑlɛ kɛ́ yɛ n símisi nɛ ikɑri yɑrɛ kɑi m pisɛ. 
Unɛ́sɑ́nɛ kɛcirɛ Tisiiki yɛ Upíimɑ ukɛikɔ́ lɛ nɛ ɑsei. U yɛ́ hɑpɔ uu lɛlɛɛ nɛ́ n wɑi nnɛ́í nɔ́ kɛ́ɛ́nkɛɛ nɔ́ɔ kɔ ceru mmɛ̃-i kɑm ń we nɛ lɛ̃ kɑm n wɑi. 
Lɛ̃ nnyɑ kɑm nɔ́kɛnɛ́mɛɛ u tummɛ rɛ ukɛ́ nɔ́ símisi mmɛ-i kɑri ń we ukɛ́ kɔ nɛ nɔ́ɑnɛ́kíŋ niŋukusɛ. 
Sɑ́ɑ Uléécɑɑ nɛ Upíimɑ Yeesu Kirisi pikɛ́ pirɔ́mɑ́rɛcɔ Yeesu pikɔ́ nnɛ́í nkíŋniŋɛ nɛ ńlɑ nɛ nfɑtɛnɛ hɛ. 
Uléécɑɑ ukɛ́ pɛ̃ nnɛ́í pɛɛ Urɔ́píimɑ Yeesu ń lɑ ɑ́i nɛ kɛ́tɔ ípɛɛlɛ́ɛ nyísɛ.
