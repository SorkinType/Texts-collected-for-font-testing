Kɑlɑɑti.
5.
Kirisi yɛ ilɑ́si kɛ́mɛɛ rɔ́ lésɛlɛ rɛ tɔkɛ́ mɛyíkíyiki mɛrɔ́círɛ n te. Lɛ̃ nnyɑ, ɑni n we nɔn pɑ́ɑ́pú, ɑ́ni kɑpɛ kɔ nɔ́mɛnɛ́círɛ ilɑ́si kɛ́mɛɛ pitɑnɛ rikpɑ́. 
Ani kutu ricɔ, nɛ̃́ Pɔɔli kɛ́ nɔ́ símisi ɑni kóm tɛ nɔn n tíyɛ pin n kérii, Kirisi úu pɛɛ líkɑ nɔ́ yóriyɛ. 
Nɛ́ kɔ pimɑ́ɑ nɔ́ rikpɑ́ rɛ úye un n tíyɛ pin u rinké, isé nnɛ́í kɑi pɛɛ pisɛ rɛ ukɛ́ ritiki. 
Nɔ́ɔ nní ń lɑ rɛ nɔkɛ́ isé ritiki nɔkɛ́ nɛ Uléécɑɑ inipɛɛ-i wɑ ɑsei pite, nɔ́ nɛ Kirisi yɛ túúnnɛlɛ ɑni Uléécɑɑ ipɛɛlɛ́ɛ pɑɑ. 
Tɔ́ɔ nní nɛ kɛfɑ u n tɛnɛlɛ̃, tɔ̃́ nɛ mɛ̃́ tɛ u yɛ́ rɔ́ wɑisɛ ɑsei pite. Nfɑ́ɑsɔnɛ mɛɛ tíyɛsɛlɛ̃ kɑri mɛ̃́ tɛ Uléécɑɑ yɛ́ ntɔ́fɑtɛnɛ nnyɑ rɔ́ wɑisɛ ɑsei pite. 
Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ, nké nɛ́ɛ nkécirɛ́ ńn yɛ líkɑ yoriyɛ. Lɛlɛɛ kulɑ́ɑ m mɑ́ yɛɛ usoi ukɛ́ nfɑtɛnɛ m mɑ́ mɛɛ ńlɑ n nyísɛlɛ̃. 
Nɔ korɑɑnɛlɛ ńsɔnɛ hɑ́i. Wóo nkpéni nsẽ́ nɔ́ púnɛisɛ un nɛ nɔ́ yúlu rɛ ɑ́ni kɑpɛ ɑsei ritiki? 
Nsímɛ́ mmɛ̃ kɑpí nɛ nní ɑtu nɔ́ n tɑkɑi ńn Uléécɑɑ yɛɛ nɔ́ n séi kɛ́mɛɛ léeri. 
Pi yɛ mɑɑ rɛ: Nkpɔ́nɔ́tɔ pitɛ fíí pɛɛ yɛ kukpɛ́ɛcɑri nnɛ́í nyɛ́pukusɛ. 
Amɑ́ ḿpɑ́ nɛ lɛ̃, nɛ nɛ Upíimɑ tɑ́lɛ̃ tɛ nɔ sɔ́nti nɔkɛ́ ncée sɔnɛ kɛ́mɛɛ pɛɛmɛ. Amɑ́ uyɛɛ nní nɔ́ n cɔ́ŋlɛ̃, tɛ úye kɛcirɛ loo, Uléécɑɑ yɛ́ kutu u ripɑsɑ. 
Pimɑ́rɛcɔ, nɛn pɛɛ́ n céési rɛ li pisɛ rɛ pisoi pikɛ́ kérii, yo nnyɑ kɑpi yɛ́ pɛɛ nkpɑ́ni nɛ́ n wéékusɛlɛ̃? Nɛn lɛ̃ n céési, Kirisi nkpɔ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ ńn yɛ́ pɛɛ úkɑ kɛ́mɛɛ ɑwólɛ nsímɛ́. 
Pɛpɛɛ nní nɔ́ n cɔ́ŋlɛ̃ tɛ ɑni kérii ɑ́pi kɑpɛ nɛ piké ŋmɑnɛ ripɑ́. Pikɛ́ mɛsɛ pimɛcirɛ cɑmɛsi! 
Pimɑ́rɛcɔ, Uléécɑɑ yɛ ilɑ́si kɛ́mɛɛ nɔ́ lésɛlɛ rɛ ɑni nɔ́mɛnɛ́círɛ n te. Ḿpɑ́ nɛ lɛ̃, ɑ́ni kɑpɛ nɛ riwɑ́ɑ́nɑ́ rɛ nɔ nɔ́mɛnɛ́círɛ te nnyɑ ɑní nɛ lɛ̃ kɑni n kɑ́ipinkɛɛ n wɑi. Amɑ́ ɑni n lɑkɑɑnɛ nɔn kɔ pikɛi wɑpɑɑnɛ. 
Nsímɛ́ nsɛ kɛ́mɛɛ kɛ isé nnɛ́í ií we. Lɛlɛɛ rɛ: A upɔ́cɔ n lɑ yɑrɛ ripɔ́cúruu. 
Amɑ́ nɔn nɔ́mɛnɛ́cɔpɛcirɛ n kɛ́hɛnnɛ nɔn n tɑkɑinkɛɛ, ɑni nɔ́mɛnɛ́círɛ tĩ. Insɑ́ lɛ̃, nɔ́ nɔ́mɛnɛ́círɛ kónɑɑnɛ. 
Lɛ̃ nnyɑ, nɛ lɑ kɛ́ nɔ́ símisi rɛ ɑni tíyɛ Nfɑ́ɑsɔnɛ nkɛ́ nɔ́ n séni, ɑ́ni yɛ́ pɛɛ́ uyɛ-i nɔ́linɛ́kɑ́ipi-kɑ́ipi n wɑi. 
Ái lɛ̃ kɑni ń lɑ kɛ Nfɑ́ɑsɔnɛ nń lɑ. Ái kɔ Nfɑ́ɑsɔnɛ lilɑlɑ kɛ nɔ́ɔ kɔ lɑ. Nfɑ́ɑsɔnɛ lilɑlɑ nɛ lɛ̃ kɛ kɛsoipipi ɑkɛ ń lɑ ɑ́i yɛ risɑ́nɛ. Lɛ̃ kɑí we nnyɑ kɑ́ni yɛ fe nɔkɛ́ lisɔnɛ kɑni ń lɑ n wɑi. 
Amɑ́ in Nfɑ́ɑsɔnɛ mɛɛ nɔ́ séni, isé íi pɛɛ kɔ ńnɑŋɛ nɔ́kɛnɛ́cɑ́ɑ́ mɑ́. 
Lɛ̃ kɛ kɛsoipipi likɑ́ipi-kɑ́ipi ɑi yɛ n léemɛ ɑí n nyɑ́ni kpɑ̃́ii yɛ nní: Iwɑ́sɑ́ piwɑpɑɑnkɛɛ nɛ mɛ́kpɛrinkpɛwɑi nɛ mɛsɛicirɛ́wɑi, 
nɛ piléényɔ́ɔnsɛ nɛ mɛnírihélɑɑwɑi nɛ ilɑ́ɑrɔ piwɑpɑɑnɛ nɛ picɛ́ŋɑɑnkɛɛ nɛ kufɑtoi nɛ kuwɔ́i nɛ íkɛŋɛnɛ nɛ pikóólɛntɛnɛ nɛ pitele siwúípi piwɑpɑɑnkɛɛ 
nɛ ucɔ likɔ́ pikɑ́ipinkɛɛ nɛ pítɑ nirɛ́ nɛ pilukɛ cɑɑinkɛɛ nɛ ɑnyɑ̃́ nɛ lilinyinɛcɔ. Nɛ lɑ kɛ́ nɔ́ péhelɛ yɑrɛ kɑm pisímɛ́ nɔ́ m mɑsí rɛ: Pɛ̃ nnɛ́í pɛɛ limɛwɑi mɛ̃ n wɑpisi ukɑ úu Uléécɑɑ iyɔɔpi kɛ́mɛɛ mɛlonɛ mɑ́. 
Amɑ́ lɛ̃ kɛ Nfɑ́ɑsɔnɛ pɔ́ɔ yɛ usoi n hɛ yɛ nní: Ńlɑ nɛ mpɔ́ɔnɑrɛ nɛ nkíŋniŋɛ nɛ ḿpurɛ nɛ kusoi piwɑi nɛ nfɑnɑrɛ nɛ ɑsei pitiki 
nɛ ńniŋɛ nɛ mɛcírɛ pimúlú. Isé íi mɛwee mɛ̃ kɛsu. 
Kirisi Yeesu pikɔ́ yɛ pimɛwee nɛ pililɑlɑ nɛ pilikɑ́ipi-kɑ́ipi pikpíkɛ́ mɑsilɛ ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii. 
Nfɑ́ɑsɔnɛ mɛɛ Uléécɑɑ nfɑɑ rɔ́ hɛɛlɛ̃. Lɛ̃ nnyɑ, tɔkɛ́ tíyɛ nkɛ́ rɔ́ n séni. 
Ári kɑpɛ pɑnsɛ píyuwɛ́lɑɑ tɔkɛ́ mɛrɔ́cɔpɛcirɛ ɑnɔ́ɔ n wɛ́lɑɑnɛ tɔn kɔ mɛrɔ́cɔpɛcirɛ kufɑtoi wɑpɑɑnɛ.
