Efɛɛsi.
5.
Nɔ siwɑ̃́ sinyinɛ lɛ kɛ Uléécɑɑ uu ń lɑ. Lɛ̃ nnyɑ, ɑni umɛcɔ n wɑi. 
Ńlɑ nkɛ́ nɔ́inɛ́soi nnɛ́í kɛ́mɛɛ n we. Ani yɛ̃ lɛ̃ kɛ Kirisi uu rɔ́ ń lɑ uu unfɑɑ kpísi uu tɔ̃́ nnyɑ hɛ yɑrɛ inyɔ́ɔnsɛsɛɛ, Uléécɑɑ uu i yɔ́su ńsɔnɛ. 
Lɛ̃ kɑ́i nɛ n sɑ́ rɛ likɛ́ Uléécɑɑ sipipi kɛcɔpɛ n we yɛ nní: Mɛ́woo mɛ́woo pinɔsiwɛ́lɑɑ nɛ mɛ́kpɛrinkpɛwɑi nɛ́ɛ mɛmɑ́ nnɑ́ɑ́pí nkɔ́ ńkɑ ńn kɑpɛ nkpɑ́ni nɔ́kɛnɛ́cɔpɛ n we. 
Ái kɔ nɛ sɑ́ rɛ nɔkɛ́ ɑsímɛ́ kópɛ nɛ́ɛ inírisimɛ́ nɛ́ɛ ɑkpɛ́ri kópɛ m mɑ́. Ái nyɑḿ. Amɑ́ Uléécɑɑ ipɑkɑrɛ ŋmɑnɛ ikɛ́ n we. 
Ani kõ ńsɔnɛ rɛ mɛ́woo mɛ́woo mɛwɑi ute úkɑ nɛ úkpɛrinkpɛkɔɔ́ úkɑ nɛ mɛmɑ́ nnɑ́ɑ́pí ukɔ́ úkɑ úu lɛ̃ kɛ Uléécɑɑ nɛ Kirisi ɑpi piiyɔɔpi kɛ́mɛɛ nɛ pipisoi m mɛ̃́ kɛ́mɛɛ líkɑ yɛnunɛ. Mɛmɑ́ nnɑ́ɑ́pí pɔ̃́ nɛ kɔ piléényɔ́ɔnsɛ picɔ lɛ. 
Áni kɑpɛ tíyɛ unyinɛ ukɛ́ ńwoosimɛ́ nɛ nɔ́ kírɑɑsɛ. Lɛ̃ nnɛ́í nnyɑ kɛ Uléécɑɑ kuwɔi ɑku pisoi pɛɛ ɑ́pi u m pɑkɑrɛlɛ̃ kɛcɑ́ɑ́ we. 
Lɛ̃ nnyɑ, nɔ́ nɛ pɛ̃ ɑ́ni kɑpɛ ńkɑ kɛ́mɛɛ m pɛ́nɛlɛ̃. 
Nɔ pɛɛ mɛkɛɛ mɛsei kuŋmɑhɑ-i welɛ. Nɔ nkpéni nɔ́ nɛ Upíimɑ kupɛ́nɛcɔ nnyɑ mɛtɛ́í-i welɛ. Lɛ̃ nnyɑ, ɑni n sɔ́nɛ likɛ́ n nyísɛlɛ̃ tɛ nɔ mɛtɛ́í sipipi lɛ. 
Likumúŋɛ́ rɛ lɛlɛɛ yɛ mɛtɛ́í kɛ́mɛɛ n léemɛ lɛɛ nfɑnɑrɛ nɛ nsímɛ́sei pitiki nɛ ɑsei. 
Lɛ̃ nnyɑ, ɑni n wɛ́ɛ́si nɔkɛ́ lɛlɛɛ nɛ Upíimɑ n sɑ́ n nyu. 
Nɔ́ nɛ kuŋmɑhɑ kɛ́mɛɛ mɛwɑi ɑ́ni kɑpɛ ńkɑ kɛ́mɛɛ m pɛ́nɛlɛ̃, ɑmɑ́ ɑni yɛ mɛ yɑ́mnɛ. 
Mɛsei sei lɛ̃ kɛ pisoi pɛ̃ ɑpi mɛyɑɑ n wɑi pisímɛ́ ricuruu yɛ isɛi welɛ. 
Amɑ́ pi yɛ pin pimɛwɑi mɛ̃ n yɑ́mnɛ, mɛ yɛ mɛtɛ́í kɛ́mɛɛ léémɛlɛ ɑmɛ kúcere wɑi. 
Lɛ̃ nnyɑ kɑpí mɑɑ rɛ: Pɔ́ nkó pɔ́ɔ nní ń lɔni, ɑ yisi. A pikpɔkpɔ kɛ́mɛɛ yisi Kirisi ukɛ́ pɛɛ mɛtɛ́í pɔ́ nyísɛ. 
Lɛ̃ nnyɑ, ɑni nɔ́nnɛ́sẽ́ kɛcɑ́ɑ́ musí ńsɔnɛ. Áni kɑpɛ n sɔ́nɛ yɑrɛ piníri, ɑmɑ́ ɑni n sɔ́nɛ nɛ mɛwɛ́ɛ́sɛsɔhɔ. 
Áni kɑpɛ tíyɛ ituŋɛ ikɛ́ mɛ́woo n tɔ́su, li we rɛ siyɑ́ɑ nsí-i kɑri nní ń we ɑ́si nyɑḿ. 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ mɛníri n tũ. Ani riŋmɔ́ɔ́pú ɑni Upíimɑ mɛlɑ ceru. 
Aɑni kɑpɛ yɛ pítɑ m pomɑɑlɛ̃. Lɛ̃ nɛ mɛ́woo mɛ́woo mɛwɑi lɛ. Amɑ́ ɑni tíyɛsɛ Uléécɑɑ Nfɑɑsɔnɛ nkɛ́ nɔ́ n yipɑɑlɛ̃. 
Ani ipɑkɑrɛ siyomɛ nɛ ɑkɛ́hɛ nɛ siyómɛ ncɔpuri kɛ Nfɑ́ɑsɔnɛ nn nɔ́ n hɛ nɛ nɔ́mɛnɛ́cɔpɛcirɛ n ŋmɔ́ɔ́púsɛnɛlɛ̃. Ani kɛfɑ kɛsɛ nɛ ipɑkɑrɛ siyomɛ Upíimɑ yõ. 
Ani yɛ ḿpɑ́ píyei nɛ ḿpɑ́ yo kɛ́mɛɛ Sɑ́ɑ Uléécɑɑ pɔɔnɛsɛ nɛ Urɔ́píimɑ Yeesu Kirisi rinyiri. 
Kirisi ipɑkɑrɛ kɑni m mɑ́ nnyɑ, ḿpɑ́ úye ukɛ́ ucɔ n wɑisɛlɛ̃. 
Nɔ̃́ pinɔ́si, ḿpɑ́ úye ukɛ́ uulɑ n wɑisɛlɛ̃ yɑrɛ kuu Upíimɑ n wɑisɛlɛ̃. 
Li we rɛ utisi yɛɛ unɔ́si uwɛ́ɛ́sɛ yɑrɛ kɛ Kirisi uu Uléécɑɑ icɑ́pinɛ nnɛ́í uwɛ́ɛ́sɛ mɛcɔ. Iyɛɛ uipiŋɛ, uyɛɛ iriyu. 
Lɛ̃ nnyɑ, ḿpɑ́ unɔ́si úye ukɛ́ uulɑ n wɑisɛlɛ̃ yɑrɛ kɛ Kirisi icɑ́pinɛ ii Kirisi n wɑisɛlɛ̃. 
Nɔ̃́ pinɔ́sikɔ́, ḿpɑ́ úye ukɛ́ uunɔsi n lɑ yɑrɛ kɛ Kirisi uu Uléécɑɑ icɑ́pinɛ ń lɑ uú hɑ nɛ unfɑɑ kpísi uu iyɛ̃ nnyɑ hɛ yɑrɛ inyɔ́ɔnsɛsɛɛ. 
U lɛ̃ wɑlɛ rɛ un míni nɛ unsímɛ́ nɛ pipɑ́ɑ́lɑ́ pi m mɑsí, pikɛ́ pɛɛ pɑnsɛ Uléécɑɑ pikɔ́. 
U lɛ̃ lɑlɛ Uléécɑɑ icɑ́pinɛ ikɛ́ ukɛyu-i ríyu píimɑ m mɑ́, ikɛ́ n kpɑ̃́iilɛ̃, ncɑɑi ńn kɑpɛ ríkɑ i n tɛŋ́lɛ̃ nɛ́ɛ íi kɑpɛ ncɑɑi féé ńkɑ m mɑ́. 
Limɛcɔ kɛ pinɔ́sikɔ́ pɔ́ɔ yɛ́ kɔ pipinɔsi n lɑ yɑrɛ kɑpi pimɛcirɛ ń lɑ. Úye un uunɔsi n lɑ, uricuruu kuu lɛ̃ lɑ. 
Mɛsei yɛ mɛ rɛ úkɑ úu yɛ píkɑi uipiŋɛ cirɛ niŋni. Ḿpɑ́ úye yɛɛ uipiŋɛ n nyɔ́ɔnsɛlɛ uu kɔ ikɛcɑ́ɑ́ m pɑílɛ̃ yɑrɛ kɛ Kirisi uu Uléécɑɑ icɑ́pinɛ n wɑi mɛcɔ. 
Kɑri uipiŋɛ ilesɛ nnyɑ kuu lɛ̃ wɑ. 
Li wɔlɑɑlɛ̃ tɛ: Lɛ̃ nnyɑ kɛ utisi uu usɑ́ɑ nɛ úni yisinɛ, u nɛ unɔ́si ɑpí hɑ pɛ́nɛ ɑpi pɑnsɛ ipiŋɛ isɛ. 
Nsímɛ́ mɛɛ kɛlõ nkɛ́-i m pékɑɑlɛ̃ yɛ mpíimɑ lɛ. Nɛ mɑɑ rɛ Kirisi nɛ uicɑ́pinɛ nkɔ́ yɛ mu. 
Amɑ́ nɔ̃́ kɑm kɔ nɛ lɛ̃ tee. Li pisɛ rɛ ḿpɑ́ unɔ́sikɔ́ úye ukɛ́ uunɔsi n lɑ yɑrɛ kuu umɛcirɛ ń lɑ, unɔ́si pɔ́ɔkɛ́ kɔ uulɑ m pɑkɑrɛlɛ̃.
