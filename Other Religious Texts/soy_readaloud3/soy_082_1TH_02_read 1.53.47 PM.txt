1 Tɛsɑloniki.
2.
Pimɑ́rɛcɔ, nɔ̃́ ticuruu yɛ ceri rɛ mɛrɔ́tuipɔ nɔ́kɛnɛ́mɛɛ ɑ́i mɛ́woo wɑ. 
Nɔ kɔ nyu rɛ pi rɔ́ lɑ́mɑɑnkɛɛlɛ ɑpi íwɛ rɔ́ wɑi Filipu kuyu-pɔ. Amɑ́ Uléécɑɑ yɛ ikɑri rɔ́ hɛlɛ ɑri Nsímɛ́ Kɛcirɛ mɛɛ ukɛmɛɛ n léeri nɔ́ símisi ḿpɑ́ nɛ ncɔ́ŋ ncɔ́ŋ kɑri n yɛ̃́. 
Ái líkɑ, mpúnnɛ ńn icélɑɑ iyɛ̃ kɑri nní pisoi n céesi kɛ́mɛɛ́ we, nfɑsimɛ́ kópɛ ńkɑ ńn kɔ ikɛmɛɛ́ we, ɑ́ri kɔ wɛ́ɛ́si tɔkɛ́ unyinɛ kirɑɑsɛ. 
Amɑ́ ḿpɑ́ píyei kɑri yɛ símisi yɑrɛ lɛ̃ kɛ Uléécɑɑ uu ń lɑ. Uléécɑɑ yɛɛ rɔ́ peenu uu yɛ́nu rɛ tɔ́ fe nnyɑ kuu Unsímɛ́ Kɛcirɛ rɔ́ rimúísɛ rɛ tɔkɛ́ nɛ n tikɔ́ɔ́nú. Ái sisoipipi nkiŋniŋɛ kɑri tíkilɛ̃, ɑmɑ́ Uléécɑɑ yɛɛ yɛ kɛsoipipi kɛfɑ nsímɛ́ ń céri kɑrí lɑ likɛ́ n lɑ́ɑ́rú. 
Nɔ̃́ ticuruu yɛ nyulɛ ńsɔnɛ rɛ ɑ́ri píkɑi nɔ́ kírɑɑsɛ, irɔ́fɑɑci kɛ́mɛɛ nfɑsimɛ́ ńkɑ ńn kɔ pékɑɑlɛ̃ tɛ siwóó kɑri wɛ́ɛ́si, Uléécɑɑ yɛɛ urɔ́seérɑkɔɔ́. 
Ári úkɑ ipɑkɑrɛ wɛ́ɛ́si, ɑ́ri kɔ nɔ̃́ nɛ pinyinɛ ikɔ́ wɛ́ɛ́si. 
Tɔ̃́ nɛ́ kɔ pɛɛ fe ɑri ntɔ́nɑŋɛ nɔ́ nyísɛ rɛ tɔ Kirisi pitumɛ nnyɑ. Amɑ́ tɔ pɛɛ́ nɛ nɔ́ lɑ́ɑ́rúlɛ yɑrɛ umɑ́rɛni yɛɛ nɛ ukɛwɑ̃ kutu n cɔ́lɛ̃. 
Tɔ mɛyɑ́nsei nɔ́ lɑlɛ, ɑri Nsímɛ́ Kɛcirɛ nɔ́ símisi. Ticuruu ḿpɑ́ ntɔ́fɑ́ɑ kɑí pɛɛ pisɛ tɔkɛ́ nɔ́ pɑ, tɔ́ nɔ́ m pɑnɛ. Nɔ tɔ́ ripɔ́lɔ́lɛ likɛi kɛcirɛ. 
Pimɑ́rɛcɔ, nɛ nyu rɛ nɔ léisɛlɛ̃ lɛ̃ kɑri n kɛisi mɛyíkíyiki ɑi rɔ́ pelu. Kɛtúŋɛ́ nɛ kɛsinɛ kɑri kɛisi ńsɔnɛ ɑ́ri kɑpɛ nɛ nɔ́ unyinɛ ncɔni rimpɔ́lɔ́ kumúŋɛ́ kpɛ-i kɑri Nsímɛ́ Kɛcirɛ mɛɛ Uléécɑɑ-i n léeri nɔ́ n símisi. 
Nɔ́ɔ nní Uléécɑɑ nɛ kɛfɑ n tɛnɛlɛ̃, nɔ́ nɛ Uléécɑɑ ricuruu nɔ́ɔ pirɔ́seérɑkɔɔ́ rɛ mɛrɔ́finɛ yɛ nɔ́kɛnɛ́cɔpɛ kpɑ̃́iilɛnlɛ mɛn kɛcirɛ we, ɑ́mɛ kɔ ncɑɑi ńkɑ mɑ́. 
Nɔ nyu rɛ tɔ ḿpɑ́ nɔ́ úye rimúílɛ yɑrɛ kɛ úyɔsɑɑ uu yɛ usiwɑ̃ timmúlú. 
Tɔ nɔ́ riŋmɔ́ɔ́púsɛlɛ ɑri ńnɑŋɛ nɔ́ hɛ ɑri kɔ mɛyɑ́nsei nɔ́ pisɛ rɛ ɑni sɔ́nɛ yɑrɛ lɛ̃ kɛ Uléécɑɑ uu ń lɑ. Uléécɑɑ uyɛɛ nɔ́ sée rɛ ɑni uiyɔɔpi kɛ́mɛɛ lompɔ ɑni kɔ kɛsẽ́ tíyu m mɑ́. 
Tɔ kɔ Uléécɑɑ licɔ kɛcɑ́ɑ́ pipɑkɑrɛ kpɑriisɛlɛ. Kumúŋɛ́ kpɛ-i kɑri Nléécɑɑsimɛ́ nɔ́ n símisi, ɑ́ni kutu n ticɔ ɑ́ni n yɔsí yɑrɛ sisoipipi nsímɛ́, ɑmɑ́ nɔ kutu n ticɔlɛ, ɑni n yɔ́su yɑrɛ Uléécɑɑ nsímɛ́. Li kɔ pɛɛ mɛsei Uléécɑɑ nsímɛ́ lɛ. N yɛ nɔ́ pɛpɛɛ nɛ kɛfɑ n tɛnɛ kɛ́mɛɛ kɛisilɛ. 
Pimɑ́rɛcɔ, lɛlɛɛ Uléécɑɑ icɑ́pinɛ yɛɛ nɛ Kirisi Yeesu kɛfɑ n tɛnɛlɛ̃ Yutee kɛteni-i n wɑ licɔ lɛɛ nɔ́kɛnɛ́mɛɛ wɑ. Íwɛ iyɛ̃ kɛ Pisuifi ɑpi Uléécɑɑ icɑ́pinɛ iyɛ̃ n wɑ icɔ kɛ nɔ́pinɛ́yukɔ́ ɑpi nɔ́ lukɛisɛ. 
Pɛpɛɛ Upíimɑ Yeesu nɛ ɑntepuyɛ kṍ ɑpi kɔ tɔ̃́ íwɛ wɑi. Ápi Uléécɑɑ ripɔ́ɔ we. Pi kɔ sisoipipi nnɛ́í pilɑɑrɔ lɛ. 
Pi nɑ́ɑ́silɛ pikɛ́ rɔ́ púnɛisɛ ɑ́ri kɑpɛ fe tɔkɛ́ mpuri sɑnɛ pikɔ́ nsímɛ́ mmɛ mɛɛ yɛ́ piriyu n lɔ símisi. Piɑkópɛ kɑpi n wɑpɑɑri kɛcɑ́ɑ́ kɑpi nní ɑcɔ kpɑ́lɛ̃. Amɑ́ Uléécɑɑ kuwɔi yɛ pitínɛ pi mɑsilɛ. 
Pimɑ́rɛcɔ, kɑrí nɛ kɛtɑɑ nɔ́ n wɑi nkɑ́ripi ɑ́i rɛ nɛ ntɔ́fɑsimɛ́, ɑmɑ́ nɛ irɔ́piŋɛ, tɔ mɛyɑ́nsei kɑ́ipilɛ rɛ tɔkɛ́ piyɛ́nɛ nɔ́ rikpɑ́. 
Tɔ kɑ́ipilɛ rɛ tɔkɛ́ nɔ́kɛnɛ́mɛɛ pɛɛpɔ. Nɛ̃́ Pɔɔli yɛ nɑ́ɑ́silɛ ɑ́i mɛsɛ, ɑmɑ́ Setɑni úu ncée hɛ. 
Nɔ̃́ ŋmɑɑ, ɑ́i úkɑ ucɔ, nɔ́ɔ lɛ̃ kɑrí nɛ n tɑ́lɛ̃, nɔ́ɔ ntɔ́pɔ́ɔnɑrɛ, nɔ́ɔ kɔ ɑrɔ́tɔpi kɑri n feriyɛ linyísɛ-nyísɛ kɑri yɛ́ nɛ rikɔ́kɔri n wɑ kumúŋɛ́ kpɛ-i kɛ Urɔ́píimɑ Yeesu uú hɑ m pɛɛri. 
Mɛsei yɛ mɛ, nɔ̃́ kɑrí nɛ rikɔ́kɔri wɑi, nɔ́ɔ kɔ ntɔ́pɔ́ɔnɑrɛ.
