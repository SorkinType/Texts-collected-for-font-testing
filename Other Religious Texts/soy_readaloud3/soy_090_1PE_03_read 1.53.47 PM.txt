Limɛcɔ kɑí we nɛ nɔ̃́ pinɔ́si kɛ́mɛɛ, ɑni nɔ́mɛnɛ́círɛ ricɛpisɛ, ḿpɑ́ nɔ́ úye ukɛ́ uulɑ m pɑkɑrɛlɛ̃. Lɛlɛɛ yɛ́ tíyɛsɛ pɛ̃ pilɑ pɛɛ ɑ́pi nɔ́kɛnɛ́cɔpɛ Uléécɑɑ nsímɛ́ nɛ kɛfɑ n tɛnɛlɛ̃ ɑpi nɔ́inɛ́soi sɔnɛ yɛnu. Ḿpɑ́ nɔnsɑ́ nsímɛ́ mmɛ̃ n símisi, pi yɛ́ pɛɛ́ nɛ Uléécɑɑ kɛfɑ tɛnɛ 
rɛ pi nɔ́inɛ́soi sɔnɛ kɑni n le nyɑ́ni nnyɑ. 
Áni kɑpɛ nɛ ipiŋɛ kɛcɑ́ɑ́ mɛlɑpɛ lɛkɑɑ-lɛkɑɑ kutu n cɔlɛ̃, yɑrɛ nní pinyúpipɑhɑɑ nɛ wurɑ ikólɛ́ nɛ ilũ sɔnɛ pitɑ́nti. 
Amɑ́ lɛlɛɛ Uléécɑɑ kɛ́mɛɛ ríyu m mɑ́ lɛɛ nní mɛlɑpɛ lɛkɑɑ-lɛkɑɑ mɛɛ usoi kɛfɑ-i m pékɑɑlɛ̃. Ámɛ yɛ cɑɑi, ɑ́mɛ yɛ kɔ fɔni. Lɛlɛɛ nɔ́nnɛ́fɑnɑrɛ nɛ nɔ́inɛ́soi niŋɛ-niŋɛ kɑni n le. 
Limɛcɔ kɛ pɛɛ mɛkɛɛ pinɔ́si pɛɛ Uléécɑɑ nɛ kɛfɑ n tɛnɛlɛ̃ pin pipilɑ pɑkɑrɛlɛ̃ pin kɔ nɛ Uléécɑɑ tɑ́lɛ̃ ɑpi yɛ pɛɛ lɑpisi. 
Piusɛ yɛɛ nní Sɑrɑ yɛɛ pɛɛ ulɑ Apirɑhɑm m pɑkɑrɛlɛ̃ hɑ́i uu yɛ nɛ u séi rɛ uusɑɑ. Nɔn lisɔnɛ n wɑi, nɔnsɑ́ líkɑ n wuru, nɔ pɑnsɛlɛ ukɛpirɛ pikɔ́. 
Limɛcɔ nɔ̃́ pitisi, ɑni nɛ nɔ́pinɛ́nɔ́si n we nɔn ńsɔnɛ pi nɑrɛsi rɛ ɑ́pi pɑ́ɑ́pú nnyɑ. Nfɑ́ɑ mmɛ̃ kɛ Uléécɑɑ uu nɔ́ n hɛ fɑɑlɑɑ, nɔ́ nɛ pɛ̃ kuu kɛsẽ́ n hɛ. Lɛ̃ nnyɑ, ɑni ríyu pi n wɑisɛlɛ̃. Lin lɛ̃ n we, líkɑ ɑ́i yɛ́ pɛɛ nɔ́pinɛ́yómɛyɑ́hɑɑ kɛ́mɛɛ nɔ́ púnɛisɛ. 
Ntɔ́rɔɔ kɑm nɔ́ n símisinɛ yɛ mmú rɛ ɑni kɛmúŋɛ́ kɛsɛ m mɑ́ nɔn kɛfɑ pɛ́nɛlɛ̃. Umɑ́rɛcɔ nlɑ nkɛ́ nɔ́kɛnɛ́mɛɛ́ n we, ɑni yɛ kɔ ucɔ íwɛ téni ɑni kɔ nɔ́mɛnɛ́círɛ n cɛpisɛlɛ̃. 
Áni kɑpɛ yɛ mɛwɑi kópɛ n cɛikɛɛnɛ, ɑ́ni kɑpɛ yɛ kɔ ɑlɛ̃́ n cɛikɛɛnɛ, ɑmɑ́ ɑni yɛ rinɔ́ɔ sɔnɛ usoi ritikisɛ. Lɛ̃ nnyɑ kuu nɔ́ sée rɛ ɑni yɛ lɛ̃ wɑ, rinɔ́ɔ sɔnɛ rikɛ́ kɔ nɛ nɔ̃́ n tikilɛ̃ yɑrɛ kuu ḿ mɑ. 
Li Uléécɑɑ Nsímɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Usoi un mɛyíkíyiki n lɑ ukɛ́ isoi sɔnɛ li likɛ́ kɔ siyɑ́ɑ nnɛ́í u n lɑ́ɑ́rú, li pisɛ rɛ ukɛ́ urilempi nɛ nsímɛ́ kópɛ tĩ. Uu kɔ unnɔɔ nɛ nnɔ́ɔmɛsimɛ́ kɑ́ɑ́sɛ. 
Ukɛ́ umɛcirɛ nɛ mɛwɑi kópɛ tɔ. Uu lisɔnɛ n wɑi. Ukɛ́ nkíŋniŋɛ piwɛ́lɑɑ nɛ kɛ́yu m mɑhɑnlɛ̃. 
Upíimɑ yɛ pɛpɛɛ ɑsei n tíkilɛ̃ inípɛɛ tɑpɑɑlɛnlɛ. Un kɔ pipiyomɛyɑ́hɑɑ kutu cɔlɛ̃. Amɑ́ u yɛ pɛpɛɛ mɛwɑi kópɛ n wɑi kɛpirɛ ritɔlɛ. 
In tɛ lisɔnɛ piwɑi kɑní nɛ kɛ́yu mɑhɑnlɛ̃, wóo yɛ́ pɛɛ íwɛ nɔ́ n wɑi? 
Ticuruu in tɛ lisɔnɛ kɑni wɑi nnyɑ kɑni íwɛ le, likɛ́ nɔ́ n lɑ́ɑ́rú. Áni kɑpɛ pisoi n wuru, líkɑ ɑ́i kɑpɛ yɛ kɔ nfɑsimɛ́ nɔ́ tɑ̃. 
Amɑ́ ɑni nɔ́sinɛ́fɑ-i n nyu rɛ Kirisi yɛɛ Upíimɑ, ɑni kɔ ríyu u n wɑisɛ. Ani yɛ nɔ́mɛnɛ́círɛ n cɑ́pinɛlɛ̃ nɔn nɛ mɛ̃́ tɛ pin nɔ́ m pisɛ rɛ yo nnyɑ kɑni nní mɛyɑ̃́ nɛ u tɑ́lɛ̃ 
nɔkɛ́ rinɔ́ɔ ńsɔnɛ yɔsí nɛ ipɑkɑrɛ. Ani kɛfɑ n kpɑ̃́iilɛ̃, pɛpɛɛ nɛ nɔ́ n nɑ́ɑ́si rɛ pikɛ́ nɔ́rinɛ́yu kpu pin nɔ́ lɑ́misi rɛ nɔ Kirisi tikilɛ̃ nnyɑ, pikɛ́ isɛi li. 
In Uléécɑɑ mɛlɑ, li kutɔsi we nɔkɛ́ lisɔnɛ n wɑi nɔn nɛ íwɛ le nɛ kɑni yɛ́ nɔ́mɛnɛ́wɑi kópɛ kɛcɑ́ɑ́ nnyɑ íwɛ n le. 
Asei kɛcɑ́ɑ́, nkpɔ cirɛníŋɛ kóló kɛ Kirisi uu pisoi ɑkópɛ kɛcɑ́ɑ́ kpu. Uricuruu úu ɑkópɛ ɑ́kɑ mɑ́, ɑmɑ́ uu pikópɛkɔɔ́ kɛcɑ́ɑ́ kpi rɛ ukɛ́ nɛ Uléécɑɑ kɛ́mɛɛ pi hɑ nnyɑ. Pi u kpulɛ uu kpi yɑrɛ kɛ kɛsoipipi ɑkɛ yɛ n kpu mɛcɔ, ɑmɑ́ Uléécɑɑ Nfɑɑsɔnɛ nn u yukusɛ, 
hɑ́i uú nɛ pɛpɛɛ nkpɔ kuhorɛ-i n kpɑniilɛ̃ Nléécɑɑsimɛ́ yóó. 
Pɛ̃ ɑ́pi pɛɛ Uléécɑɑ pɑkɑrɛ kumúŋɛ́ kpɛ-i kuú pɛɛ́ nɛ pi n kɑhɑri úu lɑ ukɛ́ ukuwɔi yukusɛ. Pɛɛ́ uyɛ-i kɛ Nowee uú pɛɛ ukutɑkɑi kɑrii. Pisoi nkɑ́ripi, pipɑhɑ ŋmɑnɛ pɛɛ kukɛmɛɛ lompɔ, ɑku míni kɛcɑ́ɑ́ n tɔsilɛ̃ Uléécɑɑ uú nɛ piɑyu lɔlu. 
Limɛni mɛ̃ yɛ míni kɑni yɛ nɛni nní ń wólɛ ɑi nɔ́ɑnɛ́yu lɔlu kumííri lɛ. Ápi míni nɔ́ wólɛ rɛ likɛ́ nɔ́inɛ́piŋɛ kɛcɑ́ɑ́ mɛ́kpɛrinkpɛ hɛɛrɛ. Pi míni nɔ́ wólɛ rɛ likɛ́ nyísɛ rɛ nɔ nɛ Uléécɑɑ ténilɛ rɛ ukɛ́ nɛ Yeesu Kirisi mɛyisɛ pikpɔkpɔ kɛ́mɛɛ ritiki ukɛ́ nɛ nɔ́kɛnɛ́múŋɛ́ sɔnukusɛ. 
Uyɛɛ kɛyómɛcɑɑ ritɑɑ́ uú hɑ Uléécɑɑ kulukɛ-lukɛ-i kɛ́yukɔɔ́tonɛ-i tonɛ. Uɑnipɛ-i kɛ Uléécɑɑ uu piléécɑɑtumɛ nɛ lɛ̃ nnɛ́í lɛɛ kei ríyu nɛ ńnɑŋɛ m mɑ́ wɑ un likɛcɑ́ɑ́ tṹ.
