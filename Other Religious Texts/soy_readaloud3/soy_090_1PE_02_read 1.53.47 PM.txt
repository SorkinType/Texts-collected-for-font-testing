1 Piyɛɛ.
2.
Lɛ̃ nnyɑ, ɑni rikpɑ́kɑ́rɑ́ nnɛ́í mpuri nɛ nnɔ́ɔmɛ nɛ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi nɛ kufɑtoi nɛ iwɔ́hɔ́ riyɑ́. 
Ani Uléécɑɑ nsímɛ́ n lɑ yɑrɛ kɛ siwɑ̃́ fɛnfɛ̃í ɑsi yɛ nní mɛpɛ sɔnɛ ń lɑ, likɛ́ tíyɛsɛ nɔkɛ́ ńsɔnɛ pɑpisi, likɛ́ kɔ nɔ́ɑnɛ́yu lɔ nnyɑ. 
Nɔ Upíimɑ isoi sɔnɛ pilélé mɑsilɛ. 
Ani nɛ u rikɔ, uyɛɛ ripɑrɛ kɛcirɛ tɛɛ yɛ nfɑ́ɑ n hɛ kɛ pisoi ɑpi n kɛ́si, Uléécɑɑ pɔ́ɔ ri wɛ́ɛ rɛ tɛtɛɛ uinipɛɛ-i rinɛ́íkɔɔ́. 
Nɔ̃́ ticuruu, nɔ́ɔkɛ́ kɔ n we yɑrɛ ɑpɑrɛ nyɛɛ nfɑ́ɑ m mɑ́. Nyɛ̃ kɛ Uléécɑɑ uu kpísi, uu tíyɛsɛ Unfɑɑsɔnɛ nń nɛ ukɛyɔ u mɔm. Kei kɑní yɛ́ piléékɔɔ́ icɑ́pinɛ n wɑi, piléékɔɔ́ pɛɛ pinfɑɑ n kpísi pin nɛ inyɔ́ɔnsɛ kuu ń lɑ u wɑi, uyɛ̃ un kɔ Yeesu Kirisi nnyɑ i pɑkɑrɛntɛ. Uléécɑɑ Nfɑɑsɔnɛ kɛ́mɛɛ kɛ liinyɔ́ɔnsɛ iyɛ̃ ii léeri. 
Li Uléécɑɑ Nsímɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Nɛ ripɑrɛ kɛ Uléécɑɑ uu n wɛ́ɛ tin nnɛ́í mɑ́ kpísilɛ kɛ́ Siyɔɔ-pɔ kɑrii rikɛ́ pɑnsɛ kɛnui ripɑrɛ. Úye un nɛ ri n tɑ́lɛ̃, úu isɛi mɛlukɛ mɑ́. 
Tipɑrɛ tɛɛ nɔ́ mpí nɔ́ɔ nní nɛ kɛfɑ n tɛnɛlɛ̃ kɛ́mɛɛ kulɑ́ɑ píimɑ mɑ́lɛ. Amɑ́ ri pɛpɛɛ ɑ́pi Uléécɑɑ nɛ kɛfɑ n tɛnɛlɛ̃ kɛ́mɛɛ́ welɛ yɑrɛ lɛ̃ kɑpi nní Uléécɑɑ Nsímɛ́ ritɛlɛ́-i n wɔ́i rɛ: Tipɑrɛ kɛ pimɔ́mɛ ɑpi n fómpɔ tɛɛ nkpéni pɑnsɛ tikɛcirɛ tɛɛ kɛnui-i ń we. Li kɔ wɔlɑɑlɛ̃ tɛ: Tipɑrɛ rinyinɛ yɛ ri kɛ pisoi ɑpi yɛ rinkpɔ́u, kupɑrɛcɑŋɑɑ kunyinɛ yɛ ku kpɛɛ yɛ pisoi n lóisɛ. 
Kɛ pisoi pɛ̃ ɑpi Uléécɑɑ nsímɛ́ pipɑkɑrɛ ń yɛ nnyɑ kɑpi ti rikpɔu. Lɛ̃ kɑi kɔ hɑ́i mɛkɛɛ-mɛ we rɛ likɛ́ pi wɑ. 
Amɑ́ nɔ̃́ nɛ mpuri mmɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ lɛ. Nɔ piléékɔɔ́ icɑ́pinɛ lɛ pɛɛ uyɔ́ɔpi inyɔ́ɔnsɛ n wɑ. Nɔ Uléécɑɑ pikɔ́ lɛ. U nɔ́ɑnɛ́yu lɔlɛ uu mɛtɛ́í kɛ́mɛɛ nɔ́ yekei rɛ ɑni umɛwɑi píimɑ pisoi riyóó. 
Áni pɛɛ mɛkɛɛ Uléécɑɑ mpuri pikɔ́, ɑmɑ́ nɔ nkpéni umpuri pikɔ́ lɛ. Uléécɑɑ úu pɛɛ mɛkɛɛ íwɛ nɔ́ tẽ́, ɑmɑ́ u nkpéni íwɛ nɔ́ ténlɛ. 
Pinɛ́sɑ́nɛ lɑlɑ kɛcirɛ, nɔ kɛtẽ nté pisɑ́nɛ lɛ pɛɛ pincee n tɔ́su. Nɛ lɑ kɛ́ nɔ́ riŋmɔ́ɔ́púsɛ rɛ ɑni nɔ́mɛnɛ́círɛ tĩ, ɑ́ni kɑpɛ likɑ́ipi-kɑ́ipi lɛɛ nɛ nɔ́ n tɔpu n tikilɛ̃. 
Nɔ́inɛ́soi kɑni n le nɔn nɛ pɛpɛɛ ɑ́pi Uléécɑɑ n wuru kɛ́mɛɛ́ we ikɛ́ n nyɑḿ. Lin lɛ̃ n wɑ, kei kɑpi yɛ́ pɛɛ nɔ́ ń wɔ́hɔsi rɛ nɔ mɛwɑi kópɛ wɑi, ɑpi kei nɔ́mɛnɛ́wɑi sɔnɛ yɛnu ɑpi Uléécɑɑ ríyu wɑisɛ kɛyɑ́ɑ kɛ̃ kuu n sɔ́nti. 
Ani Upíimɑ nnyɑ sisoipipi iyɔɔpi nnɛ́í yɛɛ kɛtẽ kɛcɑ́ɑ́ ń we m pɑkɑrɛlɛ̃. Ufoí yɛɛ uyɔ́ɔpi yɛɛ kɛtẽ nnɛ́í kɛcɑ́ɑ́ ńnɑŋɛ n te. 
Ani kɔ pɛpɛɛ umɛtene ń we m pɑkɑrɛlɛ̃. Pɛ̃ kuu túmɛi rɛ pikɛ́ yɛ piwɑikópɛkɔɔ́ kutu ripɑsɑ, pikɛ́ kɔ pɛpɛɛ lisɔnɛ n wɑi ríyu wɑisɛ. 
Lɛ̃ kɛ Uléécɑɑ uu ń lɑ lɛɛ rɛ ɑni lisɔnɛ n wɑi, piníri pɛɛ nɛ nɔ́ n kɛ́ŋɛnɛ pikɛ́ nɛ rinsɛ́ɛ́ nnyɑ. 
Ani n we yɑrɛ pisoi pɛɛ ɑ́pi úkɑ kɛcɑ́ɑ́ n tṹ, ɑmɑ́ ɑ́ni kɑpɛ pɛɛ rɛ kɑ́ni nní úkɑ kɛcɑ́ɑ́ ń tṹ nnyɑ, nɔ́ rikpɑ́kɑ́rɑ́ mɛwɑi m mɑ́ nɔn mɛkɛcɑ́ɑ́ yɑ́riilɛ̃. Ani n we yɑrɛ Uléécɑɑ pikɛikɔ́. 
Ani yɛ ḿpɑ́ úye m pɑkɑrɛlɛ̃. Ani nɔ́pinɛ́mɑ́rɛcɔ n lɑ, ɑni Uléécɑɑ n wuru, ɑni kɔ uyɔ́ɔpi ríyu n wɑisɛlɛ̃. 
Nɔ̃́ pikɛikɔ́, li pisɛ rɛ nɔkɛ́ nɔ́mɛnɛ́círɛ ricɛpisɛ ɑni nɔ́pinɛ́kɛisɑɑ n wuru. Ái pɛ̃ isoi yɛɛ ń nyɑḿ pin kɛfɑ lɑ́ɑ́rú ŋmɑnɛ, ɑmɑ́ nɛ pɛ̃ mɛkɔ́ mɛɛ m pɑ́ɑ́pú kɑni yɛ́ m pɑkɑrɛlɛ̃. 
Li nyɑmlɛ rɛ ḿpɑ́ pin usoi íwɛ n lukɛisɛ kɑ́i pɛɛ́ nɛ n sɑ́ rɛ ukɛ́ n le, li ríyu mɑ́lɛ rɛ ukɛ́ Uléécɑɑ kuú nɛ ɑsei n tíkilɛ̃ nnyɑ n kɑhɑri. 
Asei kɛcɑ́ɑ́, tíyu ríye kɑi mɑ́ rɛ usoi ukɛ́ uncɑɑi nnyɑ íwɛ n yɛnɑɑlɛ̃. Amɑ́ in tɛ usoi yɛ lisɔnɛ wɑi un kɔ pɛɛ́ nɛ íwɛ kɑhɑri, li Uléécɑɑ inipɛɛ-i lisɔnɛ lɛ. 
Lɛ̃ kɛ Uléécɑɑ uú nɛ nɔ́ n sée rɛ ɑni yɛ wɑ yɛ lɛ̃. Kirisi pɔ̃́ nɛ kɔ nɔ́kɛnɛ́cɑ́ɑ́ íwɛ lilɛ uú nɛ nɔ́ nyísɛ rɛ ɑni uncee n tikilɛ̃. 
Uyɛ̃ úu ɑkópɛ ɑ́kɑ wɑilɛ̃, nnɔ́ɔmɛ ńkɑ ńn kɛtumɛ unnɔɔ-i léemɛ. 
Pi u lɑ́mɑɑnkɛɛlɛ, ɑmɑ́ úu úkɑ ɑlɛ̃́ cɛɛi, ɑpi íwɛ u lukɛisɛ, ɑmɑ́ úu úkɑ sisikɛɛ, ɑmɑ́ umɛcirɛ kuu kpísi uu Uléécɑɑ yɛɛ ɑsei nɛ n túhɑɑnɛ ɑnipɛ-i wɑi. 
Arɔ́kópɛ kuu kpísi uu ukɛcɑ́ɑ́ wɑi ɑpí nɛ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii rɛ ɑkópɛ ɑkɛ́ kɛrɔ́cɑ́ɑ́ le, tɔkɛ́ pɛɛ lɛlɛɛ ń nyɑḿ Uléécɑɑ inipɛɛ-i n wɑi nnyɑ. Uɑkpɛ̃ nyɛɛ tíyɛsɛ kɛ tɔ́ɔ nɛ peí. 
Nɔ pɛɛ mɛkɛɛ-mɛ nyɛ́rɛilɛ yɑrɛ sipoipipi sɛɛ m pɔ. Amɑ́ nɔ nkpéni usɛɛsẽ́ nɛ uyɛɛ nɔ́kɛnɛ́cɑ́ɑ́ ḿ pɑílɛ̃ kɛ́mɛɛ pɛlɛlɛ.
