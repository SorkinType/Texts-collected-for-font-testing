Epireeyɛ.
10.
Asei kɛcɑ́ɑ́, lisɔnɛ lɛɛ kɛpirɛ n wemɛ kumííri kɛ isé ii lésɛ ii nyísɛ, ɑ́i liricuruu kii nyísɛlɛ̃. Lɛ̃ nnyɑ, ḿpɑ́ likɛ́ íye wɑ, íi yɛ́ fe ikɛ́ ḿpɑ́ píyei inyɔ́ɔnsɛ isɛ iyɛ̃ kɑpi ḿpɑ́ kuŋmɛ̃ kúye mɛsɛ́rɛ n wɑi nɛ tíyɛsɛ pɛpɛɛ lɛ̃ Uléécɑɑ n yɑ́ɑ́si pikɛ́ n nyɑḿ nɛ kɛ́tɔ-pɔ. 
In pɛɛ lɛ̃, ɑ́i yɛ́ pɛɛ nní n we pin inyɔ́ɔnsɛ iyɛ̃ piwɑi tíyɛɛ? Li yɛ́ pɛɛ nní n welɛ pɛpɛɛ inyɔ́ɔnsɛ iyɛ̃ n wɑi pin mɛcírɛníŋɛ pisɔnɑɑ mɑsí, ɑkópɛ ɑ́ɑ yɛ́ pɛɛ pikɛmúŋɛ́-i pi pɛkɛsi. 
Amɑ́ inyɔ́ɔnsɛ iyɛɛ mɛníŋɛ pisoi ɑkópɛ ḿpɑ́ kuŋmɛ̃ kúye pi léisɛntɛ. 
Li we rɛ ɑnɑ́ɑlɑ mɛnyɛ nɛ ɑpoilɑ mɛnyɛ ɑ́mɛ yɛ́ fe mɛkɛ́ ɑkópɛ hɛɛrɛ. 
Lɛ̃ nnyɑ kɛ Kirisi uu kɛtẽ kɛcɑ́ɑ́ kɑ uu Uléécɑɑ mɑɑ rɛ: Áɑ inyɔ́ɔnsɛ nɛ́ɛ lihɛɛhɛɛ lɑ. Amɑ́ ipiŋɛ kɑɑ nɛ́ mɔ̃́. 
Ái isɛɛ kɑpi n tóroisɛ múkú-múkú nɛ́ɛ inyɔ́ɔnsɛ kɑpi yɛ n wɑ ɑpí nɛ usoi ɑkópɛ u sɑ́rɛi kɑɑ́ yɔsí. 
Kei kɑm pɛɛ́ mɑɑ rɛ: Nɛ́ɛ nkó. Nɛ sɔ́nɛpɔlɛ, Uléécɑɑ, kɛ́ mɛpɔ́lɑ wɑ yɑrɛ lɛ̃ kɑpi Isé ritɛlɛ́ kɛ́mɛɛ kɛnɛ́cɑ́ɑ́ n wɔ́i mɛcɔ. 
Kirisi yɛ nté mɑlɛ kɛlenɛ rɛ: Áɑ inyɔ́ɔnsɛ nɛ́ɛ lihɛɛhɛɛ lɑ, ɑ́ɑ kɔ ŋmurɛi pɔkɛ́ inyɔ́ɔnsɛ kɑpi yɛ n wɑ ɑpí nɛ usoi ɑkópɛ sɑ́rɛi yɔsí, in kɔ nɛ isé kɛcɑ́ɑ́ kɑpi lɛ̃ wɑ. 
Uu kɔ kpɑ́ rɛ: Nɛ sɔ́nɛpɔlɛ kɛ́ mɛpɔ́lɑ wɑ. Inyɔ́ɔnsɛ kpurɛ kuu lɛ̃ kpu uu iféé kpísi uu iripɔhɔ wɑi. 
Uléécɑɑ mɛlɑ mɛ̃ kɛ Yeesu Kirisi uu n wɑ uu umɛcirɛ kpísi uú nɛ mɛcírɛníŋɛ nyɔ́ɔnsɛ nnyɑ kɑri ɑkópɛ kɛ́mɛɛ́ le. 
Ḿpɑ́ Uléécɑɑ inyɔ́ɔnsɛ uwɑi úye yɛ̀ɛ̀ ḿpɑ́ kɛweesi kɛ́ye ukɛkɛi kɛ́mɛɛ́ n welɛ un upikɛi wɑi, un mɛsɛ́rɛ inyɔ́ɔnsɛ isɛ iyɛ̃ wɑi, liinyɔ́ɔnsɛ iyɛ̃ íi yɛ kɔ píkɑi fe ikɛ́ ɑkópɛ lesɛ. 
Amɑ́ kɛ Kirisi pɔ́ɔ mɛsɛ ŋmɑnɛ inyɔ́ɔnsɛ kɑpi yɛ n wɑ ɑpí nɛ usoi ɑkópɛ sɑ́rɛi n wɑ, uu pɛɛ limɛmɑ́ɑ́ Uléécɑɑ kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i ḿpɑ́ píyei n tũ. 
Kei kuu nkpéni nɛni mɛ̃́ Uléécɑɑ ukɛ́ upilɑɑrɔ ukɛnyɑlɛ-i riwulɑsɛ. 
Inyɔ́ɔnsɛ isɛ kuu mɛcírɛníŋɛ wɑ uú nɛ Uléécɑɑ pikɔ́ pɑnsɛsɛ pisoi pɛɛ ḿpɑ́ píyei-pɔ ɑkópɛ kɛ́mɛɛ n lélɑɑlɛ̃. 
Lɛ̃ kɛ Nfɑ́ɑsɔnɛ pɔ́ɔ kɔ tíyɛsɛ ɑri céru. Li we rɛ Nfɑ́ɑsɔnɛ yɛ pɛɛ́ mɑɑ rɛ: 
Upíimɑ yɛ mɑɑ rɛ: Áni yɛ̃ɛ, nkómɛinɛ kɑḿ nɛ pi n sée yɛ mmú: Siyɑ́ɑ sɛ̃ mɛtɔ́mpɔ-mɑɑ́, nɛ́ inɛ́sé piɑkiŋ kɛ́mɛɛ wɑ. Am kɔ pimɛsɔhɔ kɛ́mɛɛ i wɔlu ɑḿ mɛɛlúsɛ. 
Uu kɔ kpɑ́ rɛ: Ám yɛ́ kɔ píkɑi piɑkópɛ nɛ pincɑɑi kɛcɑ́ɑ́ léisɛ. 
Asei kɛcɑ́ɑ́, in tɛ pi kɛlõ pisoi ɑkópɛ sɑ́rɛi, inyɔ́ɔnsɛ kɑpi yɛ n wɑ ɑpí nɛ usoi ɑkópɛ sɑ́rɛi íi kɔ kei piwée kpɑ́lɛ̃. 
Lɛ̃ nnyɑ pimɑ́rɛcɔ, tɔ tɑ́lɛnlɛ rɛ Yeesu mɛnyɛ yɛ ncée rɔ́ hɛlɛ tɔkɛ́ yɛ nɛ tirɔ́pɔ́ɔ kɛlõ sɔnɛ riyíkí-i lompɔ. 
Uipiŋɛ yɛɛ kusɑ́ŋɑ́ɑ́ kɑpi kɛnyɔ́ɔnsɛ-i rinkénɛ. Iyɛ̃ kuú nɛ ritiki uú nɛ ncée fɑlɛ mɛɛ yɛ nfɑ́ɑ n hɛ rɔ́ yɛ́u. 
Lɛ̃ nnyɑ, tɔ Uléécɑɑ usinɑ mɑ́lɛ yɛɛ Uléécɑɑ kɛyɔ iwɛ́ɛ́sɛ n tṹ. 
Lɛ̃ nnyɑ, tɔkɛ́ kɛfɑ kɛsɛ nɛ Uléécɑɑ rikɔ ɑri kɔ nfɑtɛnɛ mɛyɑ̃́ m mɑ́ nkɛ́ iníŋí m mɑ́. Simúŋɛ́ kópɛ sɛɛ sirɔ́fɑ kɛ́mɛɛ ń we sikɛ́ n hɛɛrɛlɛ̃ ɑsí lelu, tɔkɛ́ kɔ irɔ́piŋɛ nɛ míni sɔnɛ n wolɛlɛ̃. 
Tɔkɛ́ lɛ̃ kɑrí nɛ n tɑ́lɛ̃ ɑnípɛ kɛtɛ́ rimúlú, irɔ́piŋɛ íi kɑpɛ riyɔɔ. Li we rɛ uyɛɛ rinɔ́ɔ n yekei yɛ ɑsei ute lɛ. 
Ani tíyɛ tɔkɛ́ n lɛ́nɑɑnɛ tɔkɛ́ nɛ ńlɑ nɛ mɛwɑi sɔnɛ piwɑi kɛ́mɛɛ́ n ŋmɔ́ɔ́púsɛnɛlɛ̃ nnyɑ. 
Ári kɑpɛ yɛ tɔ́ nɛ pɛ̃ tɔ́ɔ yɛ n cɑ́pinɛ riyɑ́ tɔkɛ́ iyɑɑ n we yɑrɛ kɑi picɔ kɛ́mɛɛ inyɛ́kii m pɑnsɛ mɛcɔ. Amɑ́ tɔkɛ́ yɛ mɛníŋɛ ikɑri n kpɑriisɛnɛ, kɑni nní mɛyíkíyiki n yɛ̃́ tɛ kɛyɑ́ɑ tɔrɔɔ yɛ nyɑhɑimɛlɛ nnyɑ. 
Li we rɛ kɑri nní nsímɛ́sei picére ḿ mɑsí nnyɑ, tɔn kɔ nɛ n we tɔn nɛ mɛrɔ́lɑ kɛ́mɛɛ ɑkópɛ n wɑi, inyɔ́ɔnsɛ íkɑ íi pɛɛ́ we kɑri yɛ́ n wɑ pikɛ́ nɛ ɑkópɛ rɔ́ n sɑ́rɛi. 
Amɑ́ ńtɛrii kɛ usoi uu yɛ́ pɛɛ́ nɛ n tũ un nɛ Uléécɑɑ kɛtɑhɑi kuú nɛ pisoi n túhɑɑnɛnɛ nɛ nnɑ mɛɛ pipɑkɑrɛcirɛ́ ń torɛnɛ kutoi mɛ̃́. 
Usoi un Moisi isé rinlɔ́ɔ́, pisoi pitɛ́ nɛ́ɛ pitɑɑni pin m mɑɑ rɛ mɛsei yɛ mɛ, pi yɛ u kpulɛ, ɑ́i pisɛ rɛ pikɛ́ íwɛ u tẽ́. 
Nɔ yɛ̃ tɛ uyɛɛ Uléécɑɑ Kɛpipi rińtɑkɑ́ri uu fómpɔ yɛ nɛ ɑkpɑnii nyɛ̃ ɑtɔ́ŋɛ́-tɔŋɛ́ sɑ́lɛ rɛ u nkómɛinɛ mɛnyɛ mɛɛ u m pɑnsɛsɛ Uléécɑɑ ukɔ́ wɑisɛ likpɑ́rɑ́kɔɔ́, uu kɔ Uléécɑɑ Nfɑɑsɔnɛ mɛɛ yɛ Uléécɑɑ nfɑnɑrɛ usoi n nyísɛ lɑ́mɑɑnkɛɛ nnyɑ. 
Li we rɛ tɔ nyulɛ uyɛɛ ḿ mɑɑ rɛ: Nɛ̃́ kɑí nɛ sɑ́ rɛ kɛ́ yɛ pisoi mɛwɑi pi cɛɛi, nɛ́ɛ yɛ́ɛ́ pisoi ihɛ́ɛ́ pi pɑ. U kɔ mɑɑ rɛ: Nɛ̃́ Upíimɑ, nɛ́ɛ nɛ nnɛ́puri pikɔ́ túhɑɑnɛnɛ. 
Ái kukɑhɑ we rɛ usoi ukɛ́ Uléécɑɑ yɛɛ nfɑ́ɑ n te ɑnipɛ-i lóipɔ. 
Amɑ́ ɑni siyɑ́ɑ foí sɛ-i kɛ mɛtɛ́í ɑmɛ nɔ́ rinkpɑ̃́ii léisɛ. Kɑi pikpɑ̃́ii nɔ́ ḿ mɑsí, nɔ pɛɛ likumúŋɛ́ kpɛ-i íwɛ píimɑ kɛ́mɛɛ nɛ pinɑ́ɑ mɛyɑ̃ tiŋmɔ́ɔ́púlɛ: 
Api yɛ pinyinɛ-i ɑlɛ̃́ nɛ risoiwuí kɛ́mɛɛ isɛi nɔ́ n wɑpɑɑsɛ pin kɔ nɔ́ wéékusɛlɛ̃. Ani kɔ pinyinɛ-i pɛpɛɛ íwɛ iyɛ̃ icɔ kɛ́mɛɛ ń we kɛcɑ́ɑ́ kpɑ́ ɑni pi lɛ́ni. 
Asei kɛcɑ́ɑ́, nɔ yɛ pɛɛ pikpɑniikɔ́ íwɛ ténlɛ, ɑni yɛ kɔ nɛ mpɔ́ɔnɑrɛ ŋmurɛi rɛ pisoi pikɛ́ nɔ́linɛ́kɔ́ nɔ́ yɔsí, nɔn nyu rɛ nɔ mɛmɑ́ píimɑ mɑ́lɛ mɛɛ lɛ̃ n fe mɛn kɔ ḿpɑ́ píyei we. 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ lɛ̃ kɑní nɛ n tɑ́lɛ̃ fóm. Ihɛ́ɛ́ píimɑ inyinɛ yɛ likɛmɛɛ́ welɛ. 
Asei kɛcɑ́ɑ́, li nɔ́ pisɛlɛ rɛ nɔkɛ́ nŋmɔ́ɔ́pú m mɑ́, nɔn Uléécɑɑ mɛlɑ piwɑi m mɑsí, nɔkɛ́ pɛɛ́ nɛ lɛ̃ kɑni m mɛ̃́ n yɛ̃́. 
Nléécɑɑsimɛ́ ɑtɛlɛ́ yɛ mɑɑ rɛ: Asei kɛcɑ́ɑ́, li tisɛ nkɑ́ripi, nkɑ́ripi fíí ŋmɑɑ, uyɛɛ n sɔ́nti ukɛ́ nɛ n tuinɛ. Úu yɛ́ nɑ́ŋɑipɔ. 
Unɛ́kɔ́ yɛɛ inɛ́nípɛɛ-i ɑsei n tíkilɛ̃ yɛ́ unfɑtɛnɛ nnyɑ nfɑ́ɑ wɑ. Amɑ́ un kɛpirɛ-mɛ̃ tintɔpi, ɑ́m yɛ́ pɛɛ uipuri n lɑ. 
Amɑ́ in tɔ́ɔ, ɑ́i tɔ́ɔ pɛpɛɛ yɛ rintɔ́pi ɑpí nɛ pɔlu. Amɑ́ tɔ̃́ nɛ pɛpɛɛ yɛ nɛ kɛfɑ n tɛnɛ ɑi piriyu lɔlu lɛ.
