2 Timotee.
2.
Pɔ̃́ kɛnɛ́pipi, ɑ ípɛɛlɛ́ɛ kɛ Uléécɑɑ uu tɔ́ nɛ Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ rɔ́ n nyísɛ nɛ ritiki ɑɑ́ nɛ itisi mɛpɔ́círɛ n hɛɛlɛ̃. 
Mmɛ̃ nnɛ́í kɑɑ nnɛ́nɔ́ɔ-i n kṍ píseérɑkɔɔ́ kulúi kɛyu-i, ɑ pisoi kɑɑ́ nɛ ripɔ́ɔ n lɑ́ɑ́rú pɛɛ yɛ́ n fe ɑpi picɔ céési ritɔ́sɛ. 
Pɔ́ɔkɛ́ kɔ ŋmurɛi, tɔ́ nɛ mpɔ̃́ tɔkɛ́ íwɛ li yɑrɛ Kirisi Yeesu usɔ́ɔ́cɑ kɛcirɛ. 
Usɔ́ɔ́cɑ un upikɛi-i n we, un n lɑ umɛwɑi mɛkɛ́ nɛ uuwɛ́ɛ́sɛ n sɑ́, úu yɛ pɛpɛɛ ɑ́pi isɔ́ɔ́cɑ n wɑi likɔ́ nɛ kutu n cɔlɛ̃. 
Ukɑ́hɑmɛwurɛ ukɛ́ nɛ ń kpɑsi, li pisɛ rɛ ukɛ́ itóó isé rimúlú. 
Ulɑmɛ yɛɛ n lɔ̃ ɑi u toru yɛɛ yɛ́ ufoíé uiwɛ pilukɛ li. 
A lɛ̃ kɑm nní pɔ́ n símisi kõ, Upíimɑ yɛ́ ḿpɑ́ yo-i mɛsɔhɔ pɔ́ hɛ. 
A Yeesu Kirisi yɛɛ nkpɔ-i n yisi uu nfɑ́ɑ yɛnu kɛcɑ́ɑ́ léisɛ. U Tɑfiti kɛyɑsɛ lɛ yɑrɛ lɛ̃ kɛ Nsímɛ́ Kɛcirɛ kɑm nɔ́ n símisi nn n tee. 
Nsímɛ́ Kɛcirɛ mmɛ̃ nnyɑ kɑm nní íwɛ le, ticuruu ɑpi ɑkpɑnii nɛ́ wɑi yɑrɛ uwɑikópɛkɔɔ́. Amɑ́ ɑ́pi Uléécɑɑ nsímɛ́ pɔ̃́ ɑkpɑnii wɑ. 
Lɛ̃ nnyɑ kɑm íwɛ iyɛ̃ nnɛ́í ŋmurɛi pɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ nnyɑ, Kirisi Yeesu ukɛ́ nɛ pɛ̃ ticuruu riyu n lɔ ɑpi kɔ umɛyɔɔpi tɛnɛcirɛ́ yɛ́nu. 
Nsímɛ́ mmɛ̃ kɑm nní nɔ́ n símisi yɛ nsímɛ́sei lɛ rɛ tɔ́ nɛ uyɛ̃ tɔn kɛsẽ́ n kpu, nɛ uyɛ̃ kɑri kɔ kɛsẽ́ nfɑ́ɑ yɛnunɛ. 
Tɔn ikɑri n wɑ, tɔ́ nɛ uyɛ̃ yɛ́ kɛsẽ́ iyɔ́ɔpi li. Tɔn u n ŋmɛriyɛ, uyɛ̃ ticuruu yɛ́ rɔ́ ŋmɛriyɛ. 
Tɔnsɑ́ ɑsei n tikilɛ̃, uyɛ̃ nɛ ḿpɑ́ píyei ɑsei tikilɛnlɛ, likumúŋɛ́ rɛ úu yɛ́ fe ukɛ́ umɛcirɛ kɛpirɛ ritɔ. 
Nɛ lɑ rɛ ɑ ńsɔnɛ pi léisɛ rɛ pikɛ́ Uléécɑɑ ripɑí ɑ́pi kɑpɛ inɔ́ɔsimɛ́ kɛcɑ́ɑ́ n cɛ́ŋɑɑnkɛɛ. Pitele pɛ̃ mpuri ɑ́pi yɛ líkɑ yoriyɛ, ɑmɑ́ pɛpɛɛ nní kutu n cɔ́lɛ̃ kɑi yɛ cɑɑi. 
A riŋmɔ́ɔ́pú ɑɑ wɑi Uléécɑɑ ukɛ́ pɔ́ pɑkɑrɛ yɑrɛ ukɛikɔ́ uyɛ̃ kɛ upikɛi ɑ́pi yɛ́ isɛi n wɑ, uyɛɛ ɑsei n símisi lin kɛcírɛ we. 
A íkɛŋɛnɛ kpɑ́rɑ́ iyɛɛ íi líkɑ n yóriyɛ íi kɔ Uléécɑɑ kɛ́mɛɛ léeri kɑɑsɛ. Pɛpɛɛ íkɛŋɛnɛ iyɛ̃ mpuri-i pimɛcirɛ n tɑnɑɑlɛ̃ yɛɛ nɛ Uléécɑɑ kɛtɑɑ n wɑilɛ nnɑ́nnɑ́ri. 
Piicélɑɑ yɛɛ n we yɑrɛ kúcɑpɑɑ́ kpɛɛ ipiŋɛ n le. Pisoi pɛ̃ mpuri pɛɛ nní Imenee nɛ Filɛti. 
Pi ɑsei pitíki riyɑ́lɛ ɑpi pisoi n tee rɛ Uléécɑɑ yɛ nkpɔ-i piyukusɛ rɔ́ mɑsilɛ. Lɛlɛɛ tíyɛsɛ kɛ pisoi mɛyɑ̃ ɑpi nfɑtɛnɛ fóm. 
Ḿpɑ́ nɛ lɛ̃, tikúi kɛcirɛ kɛ Uléécɑɑ uu n yekei yɛ nɛ kɑriilɛnlɛ. Li rikɛcɑ́ɑ́ wɔ́lɑɑlɛ̃ tɛ: Upíimɑ yɛ upikɔ́ nyulɛ. Li kɔ wɔlɑɑlɛ̃ tɛ: Usoi nkó yɛɛ Upíimɑ rinyiri n sélei, ukɛ́ mɛwɑi kópɛ kɛpirɛ ritɔ. 
Ái wurɑ ɑsɔ́ɔnlũ nɛ nwóóweni ɑsɔ́ɔnlũ ŋmɑnɛ nyɛɛ kɛ́yɔ mɑɑ kɛ́mɛɛ́ we. Sihee nɛ ɑpɛ́rɛ́cɑripi pɔ̃́ yɛ kɔ welɛ. Kɛnyɑ̃́yɑɑ kɑpi yɛ nɛ ɑcɑ́ripi kɛcirɛ pikɛi wɑ. Ḿpɑ́ kɛyɑ́ɑ kɛ́ye kɑpi yɛ nɛ ɑcɑ́ripi fɑkɛ-fɑkɛ pɔ̃́ pikɛi wɑ. 
Limɛcɔ kɑi kɔ pisoi kɛ́mɛɛ́ we: Úye un mɛ́kpɛrinkpɛ mɛwɑi n tíyɛ, u yɛ́ n we yɑrɛ kusɔ́ɔnlũ kpɛ̃ kɑpí nɛ ɑnyɑ̃́ kɛ́mɛɛ n kɛisi. Pi yɛ́ ilúkɛ́ u yekei uu pɑnsɛ ukɛikɔ́, uu yɛ uukɛisɑɑ yoriyɛ. Api yɛ pɛɛ́ nɛ ḿpɑ́ pikɛi sɔnɛ píye u wɑi. 
A iŋmɑ́nɛ likɑ́ipi-kɑ́ipi nɛ kɛtɑɑ wɑ, nɔ́ nɛ pɛ̃ nɔ́ɔ Upíimɑ n sélei nɛ kɛfɑ kɛsɛ, ɑni mɛwɑi sɔnɛ ncee nɛ nfɑtɛnɛ nɛ ńlɑ nɛ nkíŋniŋɛ wɛ́ɛ́si. 
Amɑ́ ɑ inírisimɛ́ yɛɛ íi kulɑ́ɑ kúkɑ m mɑ́ pipɑ́sɑinɛ kɛsi. Pɔ̃́ ticuruu yɛ nyu rɛ pitele kii yɛ nɛ tɛnɛ. 
Ái kɔ pɛɛ pisɛ rɛ Upíimɑ ukɛikɔ́ ukɛ́ yɛ pitele wɛ́ɛ́si. Li pisɛ rɛ ukɛ́ ḿpɑ́ úye n lɑ, un yɛ́ fe uu yɛ pisoi céési un kɔ ísuúlu mɑ́. 
Li kɔ pisɛ rɛ ukɛ́ yɛ pɛpɛɛ yɛ unsímɛ́ n kɛ́si céési ńniŋɛ kɛ́mɛɛ. In n lɑ, Uléécɑɑ yɛ́ tíyɛsɛ ɑpi pimɛfinɛ consɛ ɑpi ɑsei ceru 
ɑpi nfɑsimɛ́ sɔnɛ tiki, ɑpi kuníri kpɛɛ pi n tĩ́ pin kumɛlɑ wɑi ripinɛ kɛ́mɛɛ́ lelu.
