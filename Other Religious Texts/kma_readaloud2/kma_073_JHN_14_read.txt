Jɔɔn.
14.
Yisa die dɩ balɩ a yɩ ʋ kʋaŋandɩɩsɩrɩba dɩ, “Nɩ da vaa nɩ sʋgɩtɩ chʋʋsɩ. Nɩ yi Ŋmɩŋ yada ta bɩ yi n gbaŋ yada. 
Juone pam bie n Chʋa Ŋmɩŋ tigiŋ me ta maŋ gara dɩ n wʋnsɩ a yɩ nɩ. Dɩ tɩŋ ka yi wusie n tɩŋ kaaŋ balɩ nɩ. 
Maŋ keŋ ga a wʋnsɩ a kpatɩ kʋaŋ chaaŋ, n nan viiri keŋ a nagɩ nɩ a ga jigidieke n benne wo, amʋ nɩ nan dɩ bie jigidieke n benne wo. 
Nɩ sɩba sie dieke dɩ ganana jigidieke n ganana wa.” 
Tomasi die dɩ pɩasɩ wa dɩ, “N Yɔmʋtieŋ, tɩ ka sɩba jigidieke fʋ ganana wa, die wɩa lalɩa tɩ baa tɩ yi a ye sieku a ga mi?” 
Yisa dɩ yiŋŋi a balɩ wa dɩ, “Manɩŋ n yine sieku aŋaŋ wusieke aŋaŋ miivoli. Vuoŋ wori a baaŋ bɩagɩ keŋ n Chʋa Ŋmɩŋ jigiŋ ntaala ʋ daagɩ n ma. 
Dɩɩ yi nɩ tɩŋ sɩba mɩŋ, nɩ tɩŋ nan dɩ sɩba n Chʋa wa gbaŋ. Ama a nyɩŋ jinne yaa gamma nɩ sɩba wa ta ye wo mɩŋ.” 
Filipi die dɩ balɩ wa dɩ, “N Yɔmʋtieŋ, wudieke tɩ yaala wʋnna, dagɩ tɩ fʋ Chʋa wa.” 
Yisa dɩ yiŋŋi balɩ wa dɩ, “Filipi, n bie nɩ jigiŋ a yʋasɩ mɩŋ, ama fʋ ka sɩba mɩŋ? Vuodieke nɩŋ mana dɩ yene mɩŋ ye n Chʋa wa mɩŋ. Bɩa wɩa fʋ baarɩ dɩ, ‘Dagɩ tɩ fʋ Chʋa wa’? 
Filipi, ka yi yada dɩ n bie wo n Chʋa wa ma, ta n Chʋa dɩ bie n ma.” Yisa die dɩ balɩ a yɩ ʋ kʋaŋandɩɩsɩrɩba dɩ, “Wudiekemba n bala nɩ wa ka nyɩŋ n jigiŋ, ama a nyɩŋ wa n Chʋa Ŋmɩŋ jigiŋ. Wʋnɩŋ ʋ benne n ma maŋ tʋma tʋʋmaha gie. 
Die wɩa, nɩ yi mɩŋ yada dɩ n bie wo n Chʋa Ŋmɩŋ ma, ta n Chʋa Ŋmɩŋ dɩaŋ dɩ bie n ma. Dɩ daa die, ama nɩ yi mɩŋ yada mamachi tʋŋ diekemba n tʋnnana wa wɩa. 
Wusie maŋ bala nɩ, vuodieke mana dɩ yine mɩŋ yada nan tʋŋ tʋŋ diekemba n tʋnnana wa, ta ʋ tʋʋma gbaŋ nan tɩaŋ n tʋʋmaha, dama n gara n Chʋa Ŋmɩŋ jigiŋ mɩŋ. 
Ta n nan yi wudieke mana nɩ jʋʋsɩna a yaala n saaŋ ma, amʋ wʋnɩŋ Ŋmɩŋ nan ye bɩrɩŋ n wɩa. 
Dɩɩ yi nɩ jʋʋsɩ mɩŋ jadieke mana n saaŋ ma, n nan yɩ nɩ. 
“Dɩɩ yi nɩ cho mɩŋ, nɩ nan dɩ dɩ n mɩraha, 
ta n nan jʋʋsɩ n Chʋa Ŋmɩŋ aŋ ʋ yɩ nɩ Halɩkasɩka aŋ ʋ suŋŋime nɩ ta bemme nɩ jigiŋ saŋŋa mana. 
Halɩkasɩka nan dagɩ nɩ wudieke dɩ yine wusie. Tɩŋgbaŋka gie vuosi kaaŋ bɩagɩ a tuo Halɩkasɩka dama ba ka yese wo yaa a sɩba wa. Ama nɩnɩŋ nɩŋ, nɩ sɩba Halɩkasɩka vuodieke dɩ benne nɩ jigiŋ ta nan dɩ bie nɩ jigiŋ yaa gamma haahuu wo. 
“Maŋ keŋ ga n kaaŋ vaa nɩ sɩba bikpiisi. N nan viiri a keŋ nɩ jigiŋ. 
Dɩ kaaŋ yʋasɩ tɩŋgbaŋka gie vuosi kaaŋ bɩ ye mɩŋ bɩbra, ama nɩnɩŋ nɩ nan ye mɩŋ. Dama n nan dɩ beri n mɩsɩ ma; die wɩa nɩ gbaŋ nɩ nan dɩ bie nɩ mɩsɩ ma. 
Maŋ keŋ yiŋŋi a keŋ n mɩsɩ ma, nɩ nan dɩ sɩba dɩ n bie wo n Chʋa wa ma, ta nɩ dɩaŋ nɩ bie n ma sɩba n benne nɩ ma dene wo. 
“Vuodiekemba dɩ tuosinene n mɩraha ta dɩ ha banɩŋ ba yine vuodiekemba dɩ chone mɩŋ; ta n Chʋa nan dɩ cho be, ta n gbaŋ n nan dɩ cho be ta yuori a dagɩ ba vuodieke n yine.” 
Judasi (vuoke gie ka yi Judasi Asikaroti wa) dɩ pɩasɩ wa dɩ, “N Yɔmʋtieŋ, bɩa yine fʋ bala dɩ nan yuori fʋ gbaŋ a dagɩ tɩ, ama kaaŋ yuori fʋ gbaŋ a dagɩ tɩŋgbaŋka gie vuosi?” 
Yisa die dɩ yiŋŋi a balɩ wa dɩ, “Vuodieke nɩŋ mana dɩ chone mɩŋ nan dɩ dɩɩ n dagɩkʋ. Ta n Chʋa Ŋmɩŋ nan dɩ cho wo, ta manɩŋ aŋaŋ n Chʋa nan keŋ ʋ jigiŋ a bemme aŋaŋ wa. 
Ama vuodieke nɩŋ mana dɩ kana ka cho mɩŋ ka dɩɩ n dagɩkʋ. Wudiekemba nɩ wʋnna wa ka nyɩŋ wa n gbaŋ gbaŋ n jigiŋ, ama a nyɩŋ wa n Chʋa Ŋmɩŋ vuodieke dɩ tʋnna mɩŋ wa jigiŋ. 
“N bala wɩaha gie mana a yɩa nɩ n yene bie nɩ jigiŋ naa. 
Ama Susuŋŋiru, Ŋmɩŋ Halɩkasɩka vuodieke n Chʋa Ŋmɩŋ dɩ bala ʋ tʋŋ n saaŋ ma wa, nan dagɩ nɩ jaaŋ mana ta vaa nɩ tɩɩnsɩ wudiekemba mana n bala nɩ wa. 
“Sʋgɩdʋagɩŋ maŋ yɩa nɩ daa sʋgɩdʋagɩ dieke dɩ yine sɩba tɩŋgbaŋka gie sʋgɩdʋagɩŋ. Die wɩa, nɩ daa vaa jaaŋ mugisi nɩ yaa ŋmaamɩŋ yalla nɩ. 
Nɩ wɔŋ a wʋŋ n bala nɩ dɩ ‘N gara mɩŋ ama n nan viiri keŋ nɩ jigiŋ.’ Dɩɩ yi nɩ seŋ cho mɩŋ, nɩ nan dɩ yaa sʋgɩfɩalɩŋ dɩ n yiŋŋi gara n Chʋa Ŋmɩŋ jigiŋ, dama ʋ tɩaŋ mɩŋ mɩŋ. 
N bala nɩ wɩaha gie lele ta a wa yi, amʋ a keŋ yi, nɩ nan yi yada. 
N kaaŋ balɩ aŋaŋ nɩ a yʋasɩ, dama tɩŋgbaŋka gie naaŋ Sitaani kieŋ mɩŋ, ama ʋ wo n hagɩrɩŋ, 
ama n yaala tɩŋgbaŋka gie vuosi sɩmma dɩ n cho n Chʋa Ŋmɩŋ aŋaŋ wusie; die wɩa maŋ yie jaaŋ mana sɩba ʋ dagɩna mɩŋ dene wo. “Nɩ vaa tɩ gamma.”
