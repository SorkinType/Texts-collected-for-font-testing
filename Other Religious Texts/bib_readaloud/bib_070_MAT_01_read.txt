Yeezuu Krista lemim nyɩnta kʊ  MATIYEE  a gʊrsɩra ba.
1.
Yeezuu yaabɔɔ wɔɔ ba n nɩ: Yeezuu ɩ yɩrɛ ɩ ʊ, cir *Davɩd Nyɩ m, Abraham do gʊaa m sɔ. 
Abraham n Izakkɩ yɩ, Izakkɩ n Zakɔɔb yɩ, Zakɔɔb n Ziida kan a dannɔ kɩ yɩ, 
Ziida n Farɛɛs kan Zaaraa kɩ yɩ, ŋ da m Tamaar m, Farɛɛs n Ɛsrɔm yɩ, Ɛsrɔm n Aram yɩ, 
Aram n Aminadaab yɩ, Aminadaab n Naasɔɔn yɩ, Naasɔɔn n Salmɔɔn yɩ, 
Salmɔɔn n Boozɩ yɩ, a da m Rahab m, Boozɩ n Zobɛɛd yɩ, a da m Rutɩ m, Zobɛɛd n Yesee yɩ, 
Yesee n cir Davɩd yɩ, Davɩd n Salomɔɔn yɩ, a da nyɩnta Iirii lʊ, 
Salomɔɔn n Roboam yɩ, Roboam n Abɩya yɩ, Abɩya n Azaa yɩ, 
Azaa n Zozafat yɩ, Zozafat n Zooram yɩ, Zooram n Ozɩyas yɩ, 
Ozɩyas n Zowatan yɩ, Zowatan n Akaazɩ yɩ, Akaazɩ n Ezekɩyas yɩ, 
Ezekɩyas n Manase yɩ, Manase n Amɔɔn yɩ, Amɔɔn n Zozɩyas yɩ, 
Zozɩyas n Zekonɩyas kan a dannɔ kɩ yɩ. A n a ga kʊ ʊ kan *Babɩlɔnnɩ lɔmbaa tarɛ dɔmɩm bɩ kɩ. 
Babɩlɔnnɩ lɔmbaa bɩ jɛ ʊ bɩ, Zekonɩyas Salatɩyɛl yɩ, Salatɩyɛl n Zorobabɛl yɩ, 
Zorobabɛl n Abiyud yɩ, Abiyud n Eliyakim yɩ, Eliyakim n Azɔɔr yɩ, 
Azɔɔr n Sadɔk yɩ, Sadɔk n Akim yɩ, Akim n Eliyud yɩ, 
Eliyud n Eleyazaar yɩ, Eleyazaar n Matan yɩ, Matan n Zakɔɔb yɩ, 
Zakɔɔb n Maarii zim Zozɛf yɩ, Maarii bɩ do ɩ Yeezuu yɩ n n'a birm Krista. 
Yaabɔɔ wɔɔ haay lɛ a zɛ kʊ ra bueeresi-si. A n a sa Abraham ma an ta m Davɩd ra bueeresi, an a sa Davɩd ma an ta m Babɩlɔnnɩ lɔmbaa ma bueeresi, an a sa Babɩlɔnnɩ lɔmbaa ma an ta m Krista ra bueeresi. 
Yeezuu yɩrɛ bɩ zaa ba n nɩ: A da Maarii bɩ, nyɩnta a ʊ Zozɛf jaan, bɩ, an lɛɛ ka kʊ Maarii n a ba Zozɛf lʊ bɩ, *Sɛnt-Ɛspri a ka Maarii n jɩsɩ sa. 
A jaan Zozɛf bɩ kʊ gʊaa mɩŋŋa m bɩ, a n a ŋʊa k'a a da nyannɩ ʊ y. A n a hɔɔn da k'a wɔ saa kʊ gɔsɩ b'a dɔ y. 
K'a n'a hɔɔn darɛ ncɩnaaʊ bɩ, Zuuba Malɛɛka bʊr zi nɩnnarɛ nɔ ʊ, an a hɩ m, a ʊ: «Zozɛf, Davɩd do gʊaa, ɩ bɩ nyi ba k'ɩ Maarii sa ɩ ba ɩ lʊ y, bala, Sɛnt-Ɛspri paŋŋa y'a ka, an nɩ jɩsɩ bɩ m, 
a nyɩ yɩm yar, k'ɩ tɔ ka Yeezuu, bala, a m n'a do bɩ bʊmbɔrɛ a mimbʊnyaarɔ nɔ ʊ.» 
Naa do woo… naa, ba ʊ kʊ meerbaa kʊ Woso a ba kan a lɛsinnɩsorazaa lɛ kɩ bɩ n kʊ yɩ. 
A ba n nɩ: nyɩmbʊɛɛrɛ k'a n yar dɔ bɩ jɩsɩ sim, an nyɩ yɩ yar, n y'a tɔ karɛ Emanʊwɛl, a jɩ lɛ taa k'a hɩ, a ʊ: Woso ɩ kan wɔɔ kɩ. 
Zozɛf k'a bɩ gu hinceem ma bɩ, a hɔ kʊ Zuuba malɛɛka b'a hɩ m bɩ ba: A Maarii gaa, an a ba a lʊ, 
bɩ, a n a dɔ lʊ y, an ta an aa dɔmɩm k'a nyɩyar bɩ yɩ ʊ bɩ ku, an a tɔ ka Yeezuu.
