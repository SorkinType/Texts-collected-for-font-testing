Fil.
3.
Mɛ‐daa ana yɛ mi‐supu ana, ɩmʋ nɩɩ ɩ san mʋ ɩ gyɛ yɛ, fanɛ ansi i gyii lɩɩ fan maa waa kʋkʋlʋn yɛ Ɔnyɩrɩpɛ Yesu sʋ mʋ! Nɩɩ ɩ man gyɛ asɩn tɔɔrɔɔsa sa mɛ fɛɛ n maa bɩla n kyʋrɔɔ atɔ kʋlʋn maŋa n sa fanɛ, amaa ɩ gyɛ nkyɛtɔ sa fanɛ nɩn. 
Fan kɩɩ yɛ ɔdan lɩɩ asa mʋ nɩɩ be du fɛɛ igyoonɔ nɩɩ ba gyɛ alibi awaapʋ, nɩɩ ba yɩlɛ kayɩɩ‐wʋlɛ kɩtɩn sʋ. 
Lɩɩ fɛɛ anɛ mʋ nɩɩ an bɔla Ɩbwaarɛ Kayaayu mʋ sʋ an sun mʋ, nɩɩ an kaapʋ anɛ n‐yɩɩ lɩɩ anɛ kɩwaa kʋkʋlʋn yɛ Krisito Yesu sʋ mʋ, anɛ nɩɩ ba tɩŋɛ iduu kesintin tɔ mʋ. Nɩɩ a man taa anɛ n‐yɩɩ tʋʋ kayɩɩ‐wʋlɛ kɩtɩn sʋ. 
Naafɔɔ ngbaa mɛ n sʋ ɔkpa n taa mɛ n‐yɩɩ tʋʋ kanɩn awaasa sʋ. Nɩɩ ɔkʋ kan ɛɛ fɛ nfɛɛrɛ fɛɛ ɔ sʋ ɔkpa ɔ taa mʋ n‐yɩɩ tʋʋ kayɩɩ‐wʋlɛ kɩtɩn sʋ mʋ, naafɔɔ mɛ ɩ sʋ ɔkpa n kyɔ mʋ. 
Ba tɩn mɛ oduu kakɛ buruwasapu, nɩɩ n gyɛ Ɩsɩrayɩ obii nɩn, nɩɩ n lɩɩ Bengyamin kebunɔ, nɩɩ n gyɛ Hibriwu obii gbaa‐gbaa, nɩɩ taa lɩɩ nbara mʋ kibuu sʋ mʋ, n kyɩna n gyɛ Farasii nɩn, 
nɩɩ lɩɩ mɛ n‐yɩɩ kɩkpan Gyiwu kabwaarɛ‐sun tɔ mʋ, n nyanla Krisito kʋbʋ‐kʋbʋ mʋ, nɩɩ lɩɩ ɔkʋ kɩnyɛ ɩkpa kigyiigi lɩɩ Gyiwu nbara mʋ kibuu sʋ mʋ, n man sʋ kilibi ɩmʋ tɔ. 
Amaa kʋtɔ kʋmaa mʋ nɩɩ n daa kɩɩ fɛɛ ɩ gyɛ tɔnɔ sa mɛ mʋ, nbɩɩnbɩɩ bɩrɛ lɩɩ Yesu Krisito sʋ mʋ, ɩ man bɩla ɩ gyɛ sɛhn mɛ asɛ. 
Man gyɛ ɩmʋ nkʋn, kʋtɔ kʋmaa nɩmʋ gyɛ mɛ asɛ fɛɛ atɔ fɔyɩsa, lɩɩ fɛɛ ɩmʋ nɩɩ ɩ gyɛ atɔ kparɛ sa mɛ mʋ, ɩ gyɛ n ba bɩɩ Mɛ‐Nyɩrɩpɛ Yesu Krisito. Lɩɩ Krisito sʋ mʋ n pan kʋtɔ kʋmaa, nɩɩ n kɩɩ ɩmʋ fɛɛ kɩgyaakpa sʋ atɔ, kɛ n nyɛ‐nyɛ Krisito. 
Saamʋ, n waa kʋkʋlʋn yɛ mʋ kakpaa, yɛgɛ n man bɩla n sʋ n gbaa‐gbaa mɛ ɩkpa kikyiigi lɩɩ nbara mʋ kibuu sʋ, amaa ɩkpa kikyiigi mʋ nɩɩ ɩ bɔla kɩkɔɔlɛ Krisito gyii sʋ. Nɩɩ ɩkpa kikyiigi nɩmʋ, mʋ nɩɩ ɩ lɩɩ Ɩbwaarɛ asɛ mʋ, gyan kɩkɔɔlɛ‐gyii sʋ nɩn. 
Nɩɩ kʋtɔ mʋ nɩɩ n biti ɩ gyɛ fɛɛ, n baa n nyi Krisito yɛ mʋ kelewu kyiŋi ɔlʋn mʋ, kɛ n gyii mʋ awʋrʋfɔ mʋ ɩkʋ, kɛ n ba waa fɛɛ n wu bʋga yɛ mʋ, 
saamʋ ɩ waa nɛnɛ Ɩbwaarɛ ɩ nyɛ kyiŋi ngbaa mɛ lɩɩ alewupu tɔ, ba nkpa tɔ. 
N mɛɛ kaapʋ nɩn fɛɛ n tɛɛ nyɛ ɩmʋ abɛɛ n tɛɛ wʋla waa kyɛɛkyɛɛ naa lʋwɛ. Amaa n kpan mɛ n‐yɩɩ kɛ n kɩtaa kʋtɔ mʋ nɩɩ Krisito Yesu tɩɩ mɛ waa mʋ lɛɛ mʋ tɔ. 
Mɛ‐daa ana yɛ mi‐supu ana, n mɛɛ fɛ nfɛɛrɛ fɛɛ mɛ kɩbaa tɛɛ kɩtaa ɩmʋ. Amaa kʋtɔ kʋlʋn mʋ nɩɩ n waa mʋ ɩ gyɛ fɛɛ, kʋtɔ kʋmaa mʋ nɩɩ kɩ wʋla kyʋn mɛ kamaa mʋ, n tan kɩmʋ sʋ, kɛ n waa aniya fʋʋ kʋtɔ mʋ nɩɩ kɩ dɛ mi ansi tɔ mʋ. 
Ɩmʋ sʋ n kpan mɛ n‐yɩɩ nɩn n sɩlɛ ntintin naa lʋwɛ, kɛ n nyɛ‐nyɛ nkpa kakɛɛ mʋ nɩɩ Ɩbwaarɛ bɔla Krisito Yesu sʋ tɩɩ asa mʋ. 
Nɩɩ anɛ mʋ nɩɩ an dan kɩkɔɔlɛ‐gyii mʋ tɔ mʋ kpini, ɩ dagaa a maa sʋ nfɛɛrɛ kʋlʋn nɩmʋ. Amaa nɩɩ fanɛ akʋ bɛɛ kan bɛɛ fɛ nfɛɛrɛ mʋ nɩɩ ɩ kwɩɩ mʋ, Ɩbwaarɛ ii biti ɔ lɛɛ ɩmʋ gbaa kasɛ sa fanɛ. 
Nɩɩ kʋtɔ mʋ nɩɩ ki tiri ɩ gyɛ fɛɛ, fan yɛgɛ a kɩtaa kɩkaapʋ mʋ nɩɩ an wʋla nyɛ mʋ nkʋn. 
Mɛ‐daa ana yɛ mi‐supu ana, fan kasɛ mɛ awaasa tɔ, kɛ fan pɛɛsɛ bamʋ nɩɩ ba tɛ kakyɩna mʋ nɩɩ an kaapʋ tɔ yɩla sa fanɛ mʋ. 
Lɩɩ fɛɛ kanan mʋ nɩɩ n wʋla tɔwɛ fanɛ, nɩɩ nbɩɩnbɩɩ n bɩla n tɔwɛ ɩmʋ yɛ ansikyu gbaa, yɛ asa gaalaagaa bamʋ kakyɩna tɔ du nɩn fɛɛ ba maa kisi Krisito kelewu oyu laasa mʋ sʋ asɩn. 
Asa anɩmʋ bɩrɛ, bamʋ kala ana bee biti ba muuri, nɩɩ bamʋ ɩtɔ nɩɩ bee sun, nɩɩ bamʋ ipeeli‐sɩn tɔ nɩɩ bɛɛ kʋsaa bamʋ n‐yɩɩ. Nɩɩ bamʋ nfɛɛrɛ ɩ gyan dulinyaa atɔ wʋlɛ nkʋn sʋ nɩn. 
Amaa anɛɛnɛ bɩrɛ, anɛ ayɛ bʋ sʋsʋ nɩn. Nɩɩ ɩ gyʋgyʋʋ anɛ an gyoo anɛ Ɔmɔlɩgɛpʋ, Ɔnyɩrɩpɛ Yesu Krisito mʋ nɩɩ ee biti ɔ lɩɩ tʋtɔ ba mʋ. 
Nɩɩ ɔ kan ba mʋ, ee biti ɔ kyɛɛgɛ anɛ n‐yɩɩ‐wʋlɛ mʋ nɩɩ ii wu falɛ, kɛ ɔ waa ɩmʋ fɛɛ mʋ gbaa‐gbaa kayɩɩ‐wʋlɛ mʋ nɩɩ kɩ sʋ bʋnyaa mʋ. Ɔlʋn mʋ nɩɩ ɔ sʋ nɩɩ ɛɛ taalɛ ɛɛ yɛgɛ kʋtɔ kʋmaa kɩ baa kɩ bʋ mʋ kasɛ mʋ, nɩɩ ee biti ɔ taa waa kanɩn.
