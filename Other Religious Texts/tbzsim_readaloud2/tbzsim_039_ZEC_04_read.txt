Sɑkɑrii.
4.
Kɛ̀ Kuyie ntɔ̃nnì dìì n nɑ̀ɑ́nnɛ̀, kɛ̀ dìi wɛ̃tɛní kɛ́ nh entɛ bɛ̀ ɔ̃ɔ̃ entɛ mɛ̀ɛ̀ botí onìtì wèè yɛ̃̀ nkɛ duɔ́. 
Kɛ̀ dì m bekɛ n yɑ̀ mù, kɛ̀ n dɔ̀: N yɑ̀ tɛsɔɔfitíkɛ̀kɛ̀tɛ̀ ntɛ, kɛ̀ mɛkùɔ̀ ncũɔ̃́ tɛ cuokɛ̀ tɛ yómmɛ̀ kɛ̀ sifitibúkúsí bo sìyiekɛ̀ nɛ̀ dɛ kó iwɛ̃ĩ. 
Kɛ̀ odìfíè kó dɛtie ncómmú tɛ tɑkɛ́, muù ntɛ bɑkù yoú, mutɛ̃mù ku cɑ̃̀nku. 
Kɛ̀ m bekɛ Kuyie ntɔ̃nnì dìì n nɑ̀ɑ́nnɛ̀ kɛ dɔ̀: N Yiɛ̀ diɛ m̀bɛnkú bɑ? 
Kɛ̀ dì dɔ̀: A í bɑntɛ́ dɛ̀ bɛnkúmɑ̀ɑ̀? Kɛ̀ n dɔ̀: N yí bɑntɛ́. 
Kɛ̀ Kuyie ntɔ̃nnì n nɑ́kɛ́ kɛ dɔ̀: Nɑ́kɛ́ ti nnɑ́ɑǹtì Sodobɑbɛɛdi kɛ tú: Dɛ̀ í tú nɛ̀ o wɛ̃rímú mmu yoo nɛ̀ o kpetì, dɛ̀ tu nɛ̀ n kó Muyɑɑ́ mmu. Mí ndi Yiɛ̀ nKuyie Mpɑkɛdɑɑ míì bɛ́i. 
Ditɑ̃rì diɛrì fɔ̃́ nhɑ í tùɔ̀kɛ dɛ̀mɑrɛ̀ Sodobɑbɛɛdi ììkɛ̀, ò bo dɑ pɔntɛ kɛ́dɛ́ítɛ́ kɛ́dɛitɛ ditɑ̃́rì ò yóó cɔ́ɔ́ ndì n cĩ́ɛ̃̀tɛ̀ yómmɛ̀, kɛ̀ bɛnìtìbɛ̀ ndi sɑ̃ntí kɛ tú: Dì wenni! Dì wenni! 
Kɛ̀ Kuyie nyíɛ́ kɛ́ n nɑ́kɛ́ kɛ dɔ̀: 
Sodobɑbɛɛdi weè pũ̀ṹtɛ́ n cĩ́ɛ̃̀tɛ̀ kɛ̀ weè yóó tɛ̀ dèè. Kɛ̀ dɛ̀ɛ yié nkɑ̀ɑ bɑntɛ́ kɛ dò mmí nKuyie mmíì ò tɔ̃nko. 
Wèè do senku Sodobɑbɛɛdi tɔ̃mmú ò mù ketírí dìì mɔ̀nnì kɛ̀ mù yí dɛu, dɛ̀ bo ò nɑrikɛ ò bo yɑ̀ dìì mɔ̀nnì ò mɑɑ́mɛ̀ n cĩ́ɛ̃̀tɛ̀. 
Kɛ̀ n wɛ̃tɛ kɛ́bɛ́innɛ̀ Kuyie ntɔ̃nnì kɛ dɔ̀: Odìfíè kó dɛtie ndɛndɛ dɛ̀ɛ̀ còḿmú tɛfitíkɛ̀kɛ̀tɛ̀ tɑkɛ́, muù ntɛ bɑkù yoú, mutɛ̃mù kucɑ̃̀nku, dɛ̀ bɛnkú bɑ? 
Kɛ̀ n yíɛ́ kɛ́bekɛ́ kɛ dɔ̀: Kɛ̀ odìfíè kó ibɑkɛ ìdɛ́ì ìì mɛ mmɔkɛ isɔɔkũɔ̃ti, kɛ̀ odìfíè kó mɛkùɔ̀ nyì tũ̀ nkɛ do mmɛsɔɔ, ndɛ̀ bɛnkú bɑ? 
Kɛ̀ dì dɔ̀: A í yɛ̃́ dɛ̀ bɛnkúmɑ̀ɑ̀? Kɛ̀ n dɔ̀: Áɑ̀, n yiɛ̀, n yí yɛ̃́. 
Kɛ̀ dì dɔ̀: Bɛnìtìbɛ̀ bɛ̀dɛ́bɛ̀ mbɛ n tɑ̃ɑ̃tɛ́ bɛ̀ kɛ̀ bɛ̀ cóu mmɛkùɔ̀ nkɛ̀ bɛ̀ còḿmú Kuyie Mpɑkɛdɑɑ ììkɛ̀.
