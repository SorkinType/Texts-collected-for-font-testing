Tyípɩ́nɩ̀.
11.
Tɛ́ Yesu tɛ̃ntõ̀ névye ànɛ̂ Yesu yõ sɛ̃́pú tɛ́lɔ́ nónó-á sõ̀ mɛ Nsyude gbɛ̃̀nyah sɔkɔ, pè kè noh nɛ, nósyɔ́ tɔ-á mɛ Liyel wɛy fɛ̃ syi, nónó náh pɩ Nsyifunɔlɛ. 
Képah tɛ̃̀nɛ, Pyɛrɩ-á pɔ ǹ syɩ kyɩ Yerusalɛmɩ sɔkɔ gbĩ́ nɛ̂nɛ, Yesu yõ sɛ̃́pú Nsyifunɔ kyɩ tyah lékàhlɛ ǹnɛ n kɔ̃, 
tɛ́ nɛ, à se sɔ̃́npɩ́ kyɩ de, à yõke tɔlɛ n yõ névye tĩ̀nnɛ, nónó náh pɩ Nsyifunɔlɛ nɩ? 
Képah sɔkɔ, tir mɛ̀-á pɩ sɔ̃́, Pyɛrɩ ké yah yo wah pɔ dal mɔ pé tyɩ́ wárwár nɛ, 
páh sõ̀ mɛ Nsyope kwil mɛ nírílɛ n pɩ, tɛ́ Liyel wɛ̃́kɩ́nɔ́ yɔ́lɛ pé wɛ̃kɩ. Páh kwâh yɔ́ wɛ kè mɛ wil yĩ̂nyõ fwɔ̀pĩ̀n-ǹgbɛ̃ yɔ́ sɔ̃́, ké vyɛ̃ynɔ ńyã́hlɛ tɛ̃́nɔ́lɛ, tɛ́ mɛ n tiki n pi à pɔ yĩni tɛ̃ pé tnɔ̂. 
Páh mɛ ké yáhnɔ́ sɔkɔ díkídíkí, pé yípɛ̂y wɛ ké sɔkɔ, ànɛ̂ gbɛ̃́nkyɔlɛ, ànɛ̂ yî nónó-á n til, à tahlɩ kèpye tyɩ́. 
Képah tɛ̃̀ sɔkɔ, pé yĩ́npɔ̃́ yɔ́ noh kè mɛ n yo pé tyɩ́ nɛ: «Pyɛrɩ, yuku n ko sé sɔkɔ ǹ kɔ.» 
Ǹtɛ pé mɛ yo yĩ́npɔ̃́ mɛ̀ tyɩ́ nɛ pé Tɛ̃̀ ǹgbɛ̃, pé náh képah pɩ n pi, ńkɛ̃́ páh pɩ sɔ̃́ pilɛ, dêl kwâh yɔ́ ńtɛ̃ náh de yah pé vyãh ànɛ̂ kwâh yɔ́ ńtɛ̃nɛ, nɛ̂-á névilɛ n silki Liyel yah. 
Képah tɛ̃̀nɛ, yĩ́npɔ̃́ mɛ̀ tɔ̃ wil yĩ̂nyõ, mɛ n yo nɛ ńkɛ̃́: «Liyel-á kwâh nɛ̂ wɛ̃kɩ kwáhpépɛ̌ynɛ, áwɔ tyɩ́ náh ké wɛ̃́kɩ́nɔ́lɛ nɛ, kǎh mɛ névilɛ n silki Liyel yah.» 
Képah tɛ̃̀-á pɩ sɛ̃́ kɛ tɔ̃́nɔ́, tɛ́ Liyel mɛ tɔ̃ kénɛ kwâh tɔkɔ dɔkɔ yĩ̂nyõ. 
Tɛ́ légbĩ́ mɛ̀ ó sɔkɔ, díbí nɛ́pĩ̂ tɔ̃́mɩ́ mó mɛ pɔ yĩn tɛ̃ péwɔ mɛ́-ńsah gbô yah sɔkɔ. Ǹnɩ̂-á tõ pélɛ Sesarɩ kwil, pè pɔ péwɔ tyɩ́. 
Tɛ́ Liyel Mirki mɛ yo péwɔ tyɩ́ nɛ, pé sɔkɔ pélɛ, pé káh tɔ̀tɔ̀ pɩ. Pé yṹnpyé nɛ́pĩ̂ kénúkò nónó-á yɔ̀, pépi mɛ pé sõ̀ tɛ̃ à yuku Sesarɩ kwil. Pé pól kyɩ de Kɔrnɛy gbô sɔkɔ. 
Képah sɔkɔ, Kɔrnɛy tɔ mó yo pé tyɩ́ nɛ, páh sõ̀ mɛ pé gbô sɔkɔ, tɛ́ yĩ̂nyõ tõ̀npɩ yɔ́ wil mɔ pé yah, tɛ́ yo pé tyɩ́ nɛ ńkɛ̃́, pé nɛy tɛkɩ n mɔ Nsyope kwil. Tɛ̃̀ kyɩ Simo tɛ̃ n pɔ, nɛ̂-á pè n ye Pyɛrɩlɛ. 
Wǎh kɩ pɔ wɛlɔ yísyɔ́ yo pé tyɩ́, nónó-á kɩ pé pwah mɔ ànɛ̂ pé gbô tãn póllɛ. 
Kɔrnɛy-á képah yo tɛ sɛ̃́ péwɔ Pyɛrɩ tyɩ́, tɛ́ pé fyɔ̀ tyah à tyilɛ n yo pé tyɩ́, Liyel Mirki tiki Kɔrnɛypi tɔ yõ, sɔ̃́-á lésõ kè tiki pépi yõ ké tyah gbĩ́nɛ. 
Kǎh pɩ sɛ̃́, péwɔ dyɔ to kõ Tɛ̃̀ ǹgbɛ̃ Yesu wɛy mɛ̀ tyɩ́lɛ, wǎh nɛ̂ yo nɛ ńkɛ̃́: «Nsyɔ̃ névye wel wilki ni sɔkɔ, ǹtɛ yépi tɛ̃̀ wɔlɛ, Liyel kɩ yé tɔ̃ sah ǹ Mirkilɛ kɩ yé pɩ yályál.» 
Ǹtɛ pépi-á sɛ̃ Tɛ̃̀ ǹgbɛ̃ Yesu Krista yõ, tɛ́ Liyel mɛ ǹ Mirki nɛ̂nɛ pépi kɔ̃, wǎh fyɔ̀ kélɛ no nɔ́pi tɔ kɔ̃ tɛ pépi tyɩ́ kɔ̃lɛ, nɛ̂ se péwɔlɛ kɩ fɛ̃ Liyel tir yah kõ nɩ? 
Képah sɔkɔ, névye-á wɛlɔ nɔ́pi noh sɛ̃́, pé pól vɩ, tɛ́ Liyel yĩnnɛ n gbilki mɛ n yo nɛ, Liyel-á ńtyɩ̃́nɔ ké wɛ̃̀kɩ̀lɛ névye tɔ kɔ̃ nónó náh pɩ Nsyifunɔlɛ, pè kɩ fɛ̃ pé nnɔ vi mɔ, kɩ tyɛ́-ńkɛ̃̂ min wɛ dɛ́! 
Yetyɛ̃nɩ ko mɔ gbĩ́nɛ, pè sõ̀ mɛ Yesu yõ sɛ̃́púlɛ n fõh. Képah ye, pè to karkɩ. Pé túkù sɔkɔ Fenisi gbɛ̃̀nyah, túkù sɔkɔ Syipri gbɛ̃̀nyah, túkù mɛ sɔkɔ Antyɔsyɩ kwil. Tɛ́ pè náh sõ̀ mɛ Yesu tyɩ́lɛ n yo ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́, nɛ́ Nsyifunɔ ó tyɩ́ náh. 
Ǹtɛ túkù sõ̀ mɛ pé sɔkɔ, nónó-á n wil Syipri gbɛ̃̀nyah sɔkɔ ànɛ̂ Sirɛnɩ kwil. Pépi-á pɔ Antyɔsyɩ sɔkɔ, pè sõ̀ mɛ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo no tɛ́lɔ́ tɔ tyɩ́ Tɛ̃̀ ǹgbɛ̃ Yesu tyɩ́ sɔkɔ. 
Tɛ́ Tɛ̃̀ ǹgbɛ̃ sõ̀ mɛ tɛ̃́nwɛnɔlɛ pélɛ n kɔ̃. Kǎh pɩ sɛ̃́, névye náhnáh vi mɔ Tɛ̃̀ ǹgbɛ̃ Yesu wɛ̃̀kɩ̀ yõ, à pɩ ǹ yõ sɛ̃́púlɛ. 
Képah tɛ̃̀nɛ, Yerusalɛmɩ Yesu yõ sɛ̃́pú tĩ̀nkì-á képah noh sɛ̃́, pè Barnabasɩ tɛkɩ mɔ Antyɔsyɩ sɔkɔ. 
Barnabasɩlɛ, nɛ́pépɛ̌y ye lésõ ǹnɛ. Liyel Mirki sõ̀ mɛ à tɔkɔ, ànɛ̂ Liyel yõ sɛ̃́nɔ́ mɛ ǹ sɔkɔ kègbɩlɛ. Képah ye, wǎh kyɩ yĩni tɛ̃, tɛ́ Liyel pèpɔrɩ pɩ́nɔ́ wɛ nɛ́npɔ́ Yesu yõ sɛ̃́pú tyɩ́ sɛ̃́, ǹ nɩ vɩ. À mɛ sṍré mɔ pé pól sɔkɔ nɛ, pè gbah tɛ̃ pé wrɔ́ tel tɛ̃ Tɛ̃̀ ǹgbɛ̃nɛ pé nnɔ póllɛ. Képah tɛ̃̀nɛ, tɔ́wû ǹgbɛ̃ fɛ̃ syi sɛ̃ Tɛ̃̀ ǹgbɛ̃ Yesu yõ. 
Képah náh, Barnabasɩ sɔkɔ Tarsɩ kwil, Sole yáhnkɔ̃-ńsah. 
Wǎh kyɩ à wɛ, à à tɛ̃ pɔ Antyɔsyɩ sɔkɔ. Pé nímí tɛ̃ à yi vi pól pɩ nɛ́npɔ́ Yesu yõ sɛ̃́pú tĩ̀nkìlɛ, à névye náhnáh tɔ wɛ kwɔ Yesu wɛ̃̀kɩ̀lɛ. Nɛ́npɔ́ ǹgbò sɔkɔ ye pè Yesu tõ̀ sã́hpú ye Kretyɛ̃bɩlɛ. 
Tɛ́ kénɛ sèpĩ̀n sɔkɔ, Liyel tɛ̃ntõ̀ syínyopu yísyɔ́ wil Yerusalɛmɩ kwil, à pɔ Antyɔsyɩ sɔkɔ. 
Pé sɔkɔ núkú yɔ́ sõ̀ mɛ pè n ye Akabusilɛ. Tɛ́ Liyel Mirki gbõ̀ yõ, ǹmɔ yuku kè yo nɛ ńkɛ̃́, dǒ kwéy, kwɛ́npékè-á kɩ n de n pi kèkõyṍ pól sɔkɔ. Képah pɩ́nɔ́ pɔ Klode wɛ, Wrome kwíltãm yõ̀tɛ̃̀nɛ. 
Képah sɔkɔ, Yesu tõ̀ sã́hpú kè yah tɔkɔ, à yɩ̃nɛ pé nɛ́núkù núkú pól gbã̀n yî wɛ́nɔ́nɔlɛ nɛ, páh kɩ pé gbã̀n yî yísyɔ́ tuke mɔ, kɩ tɛkɩ mɔ Yesu yõ sɛ̃́pú kɔ̃ Nsyude gbɛ̃̀nyah sɔkɔ. 
Pǎh kè yah tɔkɔ sɔ̃́, pè kè pɩ sɛ̃́, tɛ́ mɛ yî nɔ́pilɛ Barnabasɩ ànɛ̂ Sole gbõ̀ dahbɩ, pépi mɛ kyɩ sélɛ Yesu yõ sɛ̃́pú yah tãm kɔ̃.
