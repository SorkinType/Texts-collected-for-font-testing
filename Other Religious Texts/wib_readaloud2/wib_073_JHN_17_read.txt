Nsyɔ̃.
17.
Yesu-á képah yo tɛ sɛ̃́, à ǹ yah yɔ̃ yĩ̂nyõ tɛ́ nɛ: «Bɔ̌, ké gbĩ́ ye nɔ núkúnúkú. Gbílkínɔ́ sah ńmɔ yõ, ńmɔ tɔ mó gbílkínɔ́ sah á yõ. 
Áwɔ ye tɛ̃́nwɛnɔlɛ ń kɔ̃ no pól yõ, nɛ mé tyɛ́-ńkɛ̃̂ minnɛ pé pól kɔ̃, nónó-á á dahbɩ ń gbõ̀. 
Áwɔ nɛ̂-á Liyel núkúnínɛ kègbɩ yõ, pàh áwɔ pɩ̃, tɛ́ ńmɔ Yesu Krista pɩ̃, ńmɔ nɛ̂-á á tɛkɩ mɔ, képah ye tyɛ́-ńkɛ̃̂ minnɛ. 
Wáh tõ̀ nɛ̂ dahbɩ ń gbõ̀ pɩ pɩ tɛ̃̀nɛ, mé ye kè pɩ tɛ. Ńmɔ ye gbílkínɔ́ sah á yõ sétáh yõ sɔkɔ. 
Núkúnúkú tɛ̃̀nɛ Bɔ̌, gbílkínɔ́ nɛ̂-á sõ̀ mɛ ńmɔ yõ á gbɔ̀pɔ kèkõyṍ mɔ́-ńkɛ̃̂ gbĩ́ dùkùlɛ, képahlɛ ńmɔ kɔ̃ á gbɔ̀pɔ sɔkɔ ńtɔ̃. 
Wáh nónó wilki kèkõyṍ névye sɔkɔ à ńmɔ kɔ̃, mé ye á dùkùlɛ pè wɛ̃kɩ. Áwɔ tyɩ́ ye sõ̀ pélɛ, tɛ́ á mɛ pélɛ ńmɔ gbõ̀ dahbɩ, pè mɛ á wɛy fɛ̃ syi. 
Wáh wɛlɔ nónó yo ń tyɩ́, mé ye sè yo pé tyɩ́, pè mɛ sè fɛ̃ syi. Pè pɩ̃ yĩ́ĩ́ nɛ, máh wil áwɔ tyɩ́, tɛ́ sɛ̃ ké yõ nɛ áwɔ-á tɛkɩ mɔ ńnɛ. Képah ye, núkúnúkú tɛ̃̀nɛ, pè pɩ̃ nɛ wáh yî nónó póllɛ ńmɔ kɔ̃, sǎh wil áwɔ tyɩ́. 
Pǎh pɩ áwɔ tyɩ́lɛ sɛ̃́, képah ye mé álɛ n ni pépi tyɩ́. Mé náh mɛ álɛ n ni kèkõyṍ névye tɛ̃̀ wɔ tyɩ́ dɛ́! Wáh nónónɛ ń gbõ̀ dahbɩ, pépi tyɩ́ ye mé n ni álɛ. 
Nónó pól-á áwɔ tyɩ́lɛ, ńmɔ tɔ tyɩ́ ye pélɛ. Nónó pól-á ńmɔ tyɩ́lɛ, áwɔ tɔ tyɩ́ ye pélɛ. Tɛ́ pépi gbõ̀ yõ, gbílkínɔ́ mɛ n sah ńmɔ yõ. 
Ǹtɛ dǒ kwéy, ńmɔ kɩ n pɔ n pi áwɔ tyɩ́. Mé náh kɩ tɔ̃ n mɛ n pi kèkõyṍ sɔkɔ. Ǹtɛ pépi wɔ kɩ mɛ kèkõyṍ. Bɔ̌, áwɔ nɛ̂-á pèkè kɛ̃́nɛ, pè tɛ̃ n sah á tǎhkɩ̀ gbõ̀ yõ, képah nɛ̂-á á ńmɔ kɔ̃. Képah sɔkɔ, pè kɩ pɩ núkúlɛ sɔ̃́-á áwɔ ànɛ̂ ńmɔlɛ núkúlɛ. 
Máh sõ̀ mɛ pélɛ gbĩ́ nɛ̂nɛ, mé pè tɛ̃ sah á tǎhkɩ̀ gbõ̀ yõ, képah nɛ̂-á á ń kɔ̃. Mé pé yõ syɩkɩ pallɛ, ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́ náh yɩkɩ pé sɔkɔ, à wil nɛ́núkù mɛ̀ tyɩ́ sɔkɔ, nɛ̂ tyɩ́-á sõ̀ yɩ̃nɛ kè yɩkɩ, á wɛy sɛ́bɛ́y sɔkɔ tyi wã́rkɩ́nsàhnɔ́ kɔ̃ sè wɛ n wil. 
Ǹtɛ núkúnúkú, ńmɔ-á n pi áwɔ tyɩ́, képah ye mé tyi nɔ́pilɛ n yo tɛ́ ń sah kèkõyṍ sɔkɔ, nónó-á kɩ pè kɔ̃ pè kɩ mɛ nɩ̀vɩ̀nɩ̀ yèsyɩ̀kɩ̀ sɔkɔ ńmɔ tyɩ́ kɔ̃lɛ. 
Mé á wɛy yo pé tyɩ́. Pè náh tɔ̃ pɩ kèkõyṍ tyɩ́lɛ, sɔ̃́-á ńmɔ náh pɩ kèkõyṍ tyɩ́lɛ. Képah ye, kèkõyṍ névye pé dyɔlɛ n yɔ. 
Mé náh mɛ álɛ n ni nɛ, á pè tɛ̃ n yal kèkõyṍ dɛ́! Ǹtɛ nɛ n yah pé yõ, pèkè pɩ́-ò káh tɛ̃ wɛ pé tyɩ́. 
Sɔ̃́-á kèkõyṍ tyɩ́ náh ńmɔlɛ, sɛ̃́ ye kèkõyṍ tyɩ́ náh pépi tɔlɛ. 
Pè wilki n sah á tyɩ́ yõ á gbɩ gbõ̀ yõ. Á wɛynɛ, gbɩ ye kélɛ. 
Wáh ńmɔ tõ sɔ̃́ kèkõyṍ sɔkɔ, sɛ̃́ ye mé pé tɔlɛ n tõ kèkõyṍ sɔkɔ. 
Ńmɔ ń gblɔ̌ynɛ álɛ n kɔ̃ ń tɔ̂ɔ́ pépi yĩnnɛ, nɛ pé tɔ nɛ n mɛ áwɔ tyɩ́ yõ gbɩ tɛ̃̀ yõ. 
Ǹtɛ mé náh mɛ álɛ n ni pépi ó yĩnnɛ dɛ́! Nónó tɔ-á kɩ pɔ n sɛ̃ n pi ńmɔ yõ pépi tyíyónɔ́ gbõ̀ yõ, pépi tɔ tyɩ́ ye mé n ni álɛ. 
Mé álɛ n ni nɛ pé pól pɩ núkúlɛ. Áwɔ nɛ̂-á Bɔ̌lɛ, áwɔ ànɛ̂ ńmɔ-á mɛ núkúlɛ sɔ̃́, pépi ànɛ̂ ápi tɔ pɩ núkúlɛ sɛ̃́, kèkõyṍ névye kɔ̃ pè pɩ̃ nɛ áwɔ-á tɛkɩ mɔ ńnɛ. 
Wáh gbílkínɔ́ sah ńmɔ yõ sɔ̃́, ńmɔ tɔ ye gbílkínɔ́ sah pé yõ sɛ̃́, nɛ pè pɩ núkúlɛ sɔ̃́-á áwɔ ànɛ̂ ńmɔlɛ núkúlɛ. 
Ńmɔ mɛ pélɛ, áwɔ mɛ ńmɔlɛ. Kǎh pɩ sɛ̃́, pè kɩ pɩ yĩ́ĩ́ núkúlɛ. Képah sɔkɔ, kèkõyṍ névye kɩ pɩ̃ nɛ, áwɔ-á tõ ńmɔlɛ, ànɛ̂ no nɔ́pi tyɩ́-á nɔ á tyɩ́, sɔ̃́-á ńmɔ tyɩ́ nɔ á tyɩ́. 
Bɔ̌, wáh nónónɛ ń gbõ̀ dahbɩ, mé mɛ ké tyɩ́ nɛ pépi tɔ nɛ n mɛ ńnɛ máh kɩ n mɛ n pi ńyãh sɔkɔ, pè mó ń tɔ̃́rɩ́ wɛ. Ńmɔ tyɩ́-á mɛ nɔ́nɔ́lɛ á tyɩ́ kèkõyṍ mɔ́-ńkɛ̃̂ gbĩ́ dùkùlɛ sɛ̃́, képah ye á tɔ̃́rɩ́ mɛ̀nɛ ń kɔ̃. 
Bɔ̌, áwɔ nɛ̂-á mɛ tómm, kèkõyṍ névye náh ǹ pɩ̃, ǹtɛ ńmɔ wɔ ǹ pɩ̃. Tɛ́ no nɔ́pi tɛ̃̀ wɔ pɩ̃ nɛ, áwɔ-á tõ ńnɛ. 
Mé ye á dùkùlɛ pè wɛ̃kɩ, tɛ́ kɩ tɔ̃ á dùkùlɛ pè wɛ̃kɩ n pi, nɛ ńmɔ tyɩ́-á nɔ sɔ̃́ á tyɩ́, kénɛ tyɩ́nɔnɔ dùkù nɛ n mɛ pé tɔ sɔkɔ, ànɛ̂ ńmɔ mó nɛ n mɛ pélɛ.»
