Tyípɩ́nɩ̀.
4.
Képah tɛ̃̀nɛ, Pyɛrɩ ànɛ̂ Nsyɔ̃-á sõ̀ mɛ wɛ́kɩ́nɔ́ sɔkɔ tɔ́wû tyɩ́ sɛ̃́, nósyɔ́ pɔ wil mɔ pé yõ. Sárká syínwìlkìpu yísyɔ́ ye sõ̀, ànɛ̂ Liyel ní-ńsah gbô syɩ́kɩ́pú yõ̀tɛ̃̀nɛ, à tahlɩ Sadusɛ̃ névye yísyɔ́ tyɩ́. 
Tɛ́ kénɛ no fɔkɔ sõ̀ mɛ kah vyãh Pyɛrɩ ànɛ̂ Nsyɔ̃nɛ, pǎh sɔ̃́npɩ́lɛ névyelɛ n kwɔ nɛ, Yesu-á ku tɛ́ yilki mɔ, ànɛ̂ lékyɩ̂ tɔ-á kɩ pɔ n yilki n mɔ n pi. 
Képah sɔkɔ, pè pè tɛ̃. Tɛ́ lékã̂h-á sõ̀ mɛ mɔ n tɛ sɛ̃́, képah ye pè pè dah mɔ wɔ̀kɔ̀npèkè sɔkɔ, pópó à pɔ syɩ dĩndal tyɩ́. 
Ǹtɛ nónó-á Yesu tɛ̃ntõ̀ névye kwɔ́nɔ́ noh, pé náhnáh Liyel wɛ̃̀kɩ̀ tɛ̃. Kǎh pɩ sɛ̃́, Yesu yõ sɛ̃́pú nɔ nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãm (5 000) kénkɔ̃lɛ. 
Képah sɔkɔ, ké dĩndallɛ, Nsyifunɔ yõ̀tãm, ànɛ̂ kwil yèwàhnɩ̀, ànɛ̂ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú wɛ̃ tuke Yerusalɛmɩ sɔkɔ. 
Sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ An sõ̀ mɛ ànɛ̂ ǹ gbô tãn póllɛ. Pépi túkù ye sõ̀ pɩ Kayifilɛ, ànɛ̂ Nsyɔ̃nɛ, ànɛ̂ Alɛkisãdrelɛ. 
Képah tɛ̃̀nɛ, pè Pyɛrɩ ànɛ̂ Nsyɔ̃ tɛ̃ pɔ pé nɩyṍ, à pɔ pè piki yah nɛ, tɛ̃́nwɛnɔ nɛ̂ gbõ̀ yõ se, képah náh pɩ, nɛ̂ yĩn yõ se pè pɩ dɛ́nmɔnɔ mɛ̀nɛ nɩ? 
Képah sɔkɔ, Liyel Mirki gbõ̀ yõ, Pyɛrɩ pè syah nɛ: 
«Kwil yõ̀tãm ànɛ̂ yèwàhnɩ̀, áyáh pèpɔrɩ pɩ sèkùkù dĩ́ yɔ́ tyɩ́, képah yõ sɔkɔ náh yé n piki n yah álɛ kwɛynɛ nɛ, sɔ̃́ se á pɩ, tɛ́ à dɛ mɔ di? 
Yépi pól ànɛ̂ Yisrayel tãn pól yɩ̃nɛ yè kè pɩ̃ nɛ, Nasarɛtɩ Yesu Krista yĩn yõ-á dĩ́ mɛ̀nɛ yĩ́nntɛ̃̀nɔ́lɛ yé yah sɔkɔ lènɔlɛ yɔ̀. Yépi ye syɩ Yesulɛ dahnɩ yõ, tɛ́ Liyel mɛ ǹ lékó yilki mɔ. 
Ǹmɔ Yesu tyɩ́ ye yónɔ́lɛ Liyel wɛy sɔkɔ nɛ ńkɛ̃́: ‹Wɔ̀kɔ̀ mɔ́pú-á yahle dáhkɩ̀ nɛ̂ mɔ́nkõ̀nɔ́lɛ, képah ye pɔ pɩ wɔ̀kɔ̀ dáhgbɩ́lɛ.›» 
Képah náh, Pyɛrɩ tɔ̃ nɛ: «Yesu ó yĩn kɩ fɛ̃ névye pwah mɔ. Nɛ́ ǹmɔ ó yĩn náh, yĩn yɔ́ ńtɛ̃ náh ǹnɩ̂ yɔ́ ńtɛ̃ kɔ̃ kèkõyṍ, nɛ̂-á kɩ fɛ̃ névye min pwah mɔ.» 
Tukey tùkèpu sõ̀ pɩ̃ nɛ Pyɛrɩ ànɛ̂ Nsyɔ̃ náh Nsyifunɔ kǎrn ǹgbɛ̃ pɩ, ànɛ̂ pè náh mɛ n mɔ n kõ nɛ́npɔ́ tĩ̀nnɛ. Tɛ́ pǎh n wɛkɩ nɩ̀sãh ànɛ̂ lékã́mnɛ sɛ̃́, képah mɛ pé gbã̀n du. Ǹtɛ pè mó tɔ̃ sõ̀ pɩ̃ nɛ, pépi-á lésõ Yesu kɔ́mûnɛ. 
Tɛ́ pé yĩ́-á mó tɔ̃ sõ̀ mɛ dĩ́ mɛ̀ yõ nɛ̂-á dɛ́nmɔnɔ wɛ, tyíyóné náh tɔ̃ wɛ pé tyɩ́. 
Kǎh pɩ sɛ̃́, pè Pyɛrɩpi kɔ̃ pè wil tukey tùkèpu tĩ̀nkì wrɔ́, tɛ́ pépi mɛ wɛ̃ wɛ pé yahlɛ 
tɛ́ nɛ, sɔ̃́ se pé kɩ pɩ no nɔ́pilɛ nɩ? Ńkɛ̃́, kɔ̃lɩ kɔ̃lɩ tɛ̃̀ náh se mó tɔ̃ kélɛ tɛ́. Yerusalɛmɩ tãn pól-á pɩ̃ nɛ, Pyɛrɩpi-á pɩ gbǐl mɛ̀nɛ. 
Kàh pɩ sɛ̃́, pé ye kè yahle pé tyɩ́ dírí sɔkɔ nɛ, pè káh tɔ̃ Yesu yĩnnɛ n yo ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́. Képah-à pɩ, yĩ mɛ̀ náh n tɔ̃ n mɔ n yuku n pi yahlɛ. 
Képah tɛ̃̀nɛ, pè tɔ̃ pè ye, tɛ́ kè yahle pé tyɩ́ gbógbó nɛ, pé káh tɔ̃ Yesu yĩn noh n yah pé vyɛ̃y yah, ànɛ̂ pè káh tɔ̃ kwɔ́nɔ́ pɩ ǹ yĩn yõ. 
Ǹtɛ Pyɛrɩpi pè syah nɛ: «Yépi ńtɛ̃ kè sõ n yah kɛ̀: Kè se mɛ yɩ̃́nɛ́nɔ́lɛ Liyel tyɩ́, áyâh ǹmɔ tyɩ́ yɛ, tɛ́ yépi yóré fɛ̃ syi? 
Ápi tɛ̃̀ wɔlɛ, áyáh nɛ̂ wɛ, tɛ́ nɛ̂ noh, á náh fɛ̃ n yɛ n pi sé yã́hnɔ́lɛ.» 
Képah tɛ̃̀nɛ, tukey tùkèpu tɔ̃ dírí gbah wɔ pé tyɩ́, tɛ́ pè fĩn mɔ. Pè náh sõ̀ mɛ vì-ǹsah wɛ pé wɛ fõh fõh tɛ̃̀nɛ, no pól-á sɔ̃́npɩ́lɛ Liyel yĩnnɛ n gbilki Pyɛrɩpi tyípɩ́kɩ̀ mɛ̀ yĩnnɛ. 
Tɛ́ dĩ́ nɛ̂ dɛ́nmɔnɔ-á sõ̀ pɩ gbǐllɛ, à kah ye tikininínɛ. 
Pǎh fyɔ̀ Pyɛrɩ ànɛ̂ Nsyɔ̃ fĩn mɔ, sárká syínwìlkìpu yõ̀tãm ànɛ̂ yèwàhnɩ̀-á nɛ̂ yo pé tyɩ́, pè kyɩ sè pól yãh pé kɔ́mû tyɩ́. 
Pé wɛlɔ nóhntɛnɔ náh, pé pól pé sõnɔ pɩ núkúlɛ, à Liyellɛ n ni nɛ ńkɛ̃́: «Tɛ̃̀ ǹgbɛ̃, áwɔ ye mɔ yĩ̂nyõlɛ, ànɛ̂ sétáhlɛ, ànɛ̂ pnɛ yɔlɛ, à tahlɩ sé sɔkɔ yíyìn pól tyɩ́. 
Ápi náh nɛy Davidi nɛ̂-á á tõ̀npɩlɛ, áwɔ ye à kɔ̃ à kè yo á Mirki gbõ̀ yõ nɛ ńkɛ̃́: ‹Kwâh nɛ̂ yúkúnyahlenɔ mɛ kwlo kwlo tyɩ́? Kwâh nɛ̂ tyíkwɛ̂ sõnɔ mɛ névye sɔkɔ! 
Kèkõyṍ yõ̀tãm wɛ̃ tyɩ́ mɔ, tɛ́ yah tãm tɔ tuke mɔ, à pɩ núkúlɛ, pè Tɛ̃̀ ǹgbɛ̃ Liyellɛ n fwo ànɛ̂ ǹ nɛy yáhntɔkɔnɔlɛ.› 
Gbɩ yõ, kwil mɛ̀ ńtɛ̃ sɔkɔ, yõ̀tɛ̃̀ dĩ́ Yerɔdɩ ànɛ̂ gbɛ̃̀nyah yah tɛ̃́-ò dĩ́ Põse Pilati ye wɛ̃ tyɩ́ mɔ, à tahlɩ Yisrayel tãn ànɛ̂ kwlo kwlo tɛ́lɔ́ tyɩ́, pè á tõ̀npɩ Yesulɛ n fwo, ǹmɔ nɛ̂-á á nɛy yáhntɔkɔnɔlɛ. 
Yah, wáh lésõ á tɛ̃́nwɛnɔ sɔkɔ tyi nónó yah tɔkɔ sah á dyɔ dyɔ tɛ̃̀nɛ, sépi ye pè pɩ. 
Ǹtɛ núkúnúkú, yah pè ápilɛ n kyĩ n kɔ̃ kɛ̀! Nɩ̀sãh ǹgbɛ̃nɛ á kɔ̃, à á wɛynɛ n yah n yo, ápi nónó-á á tõ̀npɩbɩlɛ. 
Tɛ̃́nwɛnɔlɛ á kɔ̃, à yámpúlɛ n dɛ n mɔ, à mó gbǐl tyilɛ n pɩ, á tõ̀npɩ Yesu yĩn yõ.» 
Tɛ́ fɩ̃́ nɛ̂ sɔkɔ-á sõ̀ pélɛ tùkènmɔnɔlɛ, nírí-á fyɔ̀ n tyɛ, nɛ́npɔ́ núkú yike. Képah tɛ̃̀nɛ, Liyel Mirki pé pól tɔkɔ, pè tyah Liyel wɛynɛ n yah n yo n kɔ̃ nɩ̀sãhlɛ. 
Yesu yõ sɛ̃́pú pól sõ̀ mɛ n sõ dùkù núkúlɛ, tɛ́ mɛ ké tyɩ́ nɛ pé nɛ n pɩ dùkù núkúlɛ. Pé sɔkɔ yɔ́ ńtɛ̃ náh sõ̀ mɛ n yo nɛ, pé gbã̀n yîlɛ, pé gblɔ̌y núkúní tyɩ́-á sélɛ. Tɛ́ pé wɛ̃ wrɔ́, pé pól gbã̀n yî sõ̀ pɩ tɔ́wû tyɩ́lɛ. 
Tɛ́ tɛ̃́nwɛnɔ ǹgbɛ̃ sɔkɔ ye sõ̀ Tɛ̃̀ ǹgbɛ̃ Yesu tɛ̃ntõ̀ névye pɩ ǹ tyɩ́ yónwàhpu kègblɔlɛ, à ǹ tyɩ́lɛ n yo n kɔ̃ nɛ, wǎh ku tɛ́ yilki mɔ. Tɛ́ Liyel pèpɔrɩ ǹgbɛ̃ sõ̀ mɛ pé pól yõ. 
Swɛ̃y, képah náh pɩ, gbên-à sõ̀ mɛ nónó tyɩ́, pě sélɛ n yãm, ǹ tɛ́ sé pɔ́lɛ Yesu tɛ̃ntõ̀ névyelɛ n kɔ̃. Pépi n mó sélɛ n dibi, à yɩ̃nɛ pé nɛ́núkù núkú pól fyã̀hnɔ́nɔlɛ. Képah ye, yãm pɩ́-ò náh sõ̀ mɛ pé sɔkɔ. 
Tɛ́ légbĩ́nɛ, Nsyosɛfu yɔ́ sõ̀ mɛ, nɛ̂-á dubi Syipri gbɛ̃̀nyah. Levi yìkì nɛy ye sõ̀ ǹnɛ. Tɛ́ Yesu tɛ̃ntõ̀ névye mɛ yĩn dyɩ à kɔ̃ Barnabasɩlɛ, nɛ̂ kɔ́r-á sṍré mɔ́-òlɛ. 
Ǹmɔ mɛ ǹ swãh yãm, tɛ́ mɛ pɔ ké pɔ́lɛ Yesu tɛ̃ntõ̀ névye gbõ̀ dahbɩ.
