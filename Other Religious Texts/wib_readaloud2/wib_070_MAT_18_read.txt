Matye.
18.
Légbĩ́nɛ, Yesu tõ̀ sã́hpú nɔ de ǹnɛ, tɛ́ à piki yah nɛ, nɛ̂ se nɛ́-ǹgbɛ̃nɛ kal Liyel tɔ̃́rɩ́ nɔlɩ yṍnɔ́ sɔkɔ nɩ? 
Képah tɛ̃̀nɛ, Yesu mɛ wɛ̃́npir tyɩ̃́ɩ̃ yɔ́ ye pɔ sah pé nɩyṍ, tɛ́ yo pé tyɩ́ nɛ: 
«Kègbɩ yõ mé kélɛ n yo yé tyɩ́, nɛ́ yé náh yé nnɔ vi mɔ à pɩ wɛ̃́npì tyɩ́ kɔ̃lɛ, yé náh Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi. 
Névi nɛ̂-à kɩ ǹ gblɔ̌y tiki mɔ pɩ tyɩ̃́ɩ̃ wɛ̃́npir mɛ̀ sɔ̃́, tɛ̃̀ wɔ ye kɩ pɩ nɛ́-ǹgbɛ̃nɛ kal Liyel tɔ̃́rɩ́ nɔlɩ yṍnɔ́ sɔkɔ. 
Névi nɛ̂-à wɛ̃́npir mɛ̀ kénkɔ̃ tɛ̃́nɔ́ plɛ ńmɔ yĩnnɛ, ńmɔ ye tɛ̃̀ tɛ̃ plɛ.» 
«Ǹtɛ wɛ̃́npì nónó-á sɛ̃ ńmɔ yõ yɔ̀, névi nɛ̂-à pé sɔkɔ yɔ́ yõ yɩkɩ mɔ tyípékè pɩ́nɔ́ sɔkɔ, pàh yínɔ́nan ǹgbɛ̃̀kɩ̀ yɔ́ toh tɛ̃̀ yègbɩ tɛ́ à tu dah mɔ pnɛ yɔ tì tɛ̃̀-ǹsah sɔkɔ kè kɔ̃ kè à yõ, képah kɩ fṍnfṍ kal tɛ̃̀nɛ. 
Pèkè mɛ kèkõyṍ névye yõ névye yõ yɩ́kɩ́nmɔnɔ yĩnnɛ tyípékè pɩ́nɔ́ sɔkɔ. Névye yõ tɛ̃̀ wɔ náh fɛ̃ n yɛ yɩ́kɩ́nɔ́lɛ ó, ǹtɛ névi nɛ̂ gbõ̀ yõ sɛ̃́nkɔ̃-à wɛ, pèkè kɩ tɛ̃̀ nɔ. 
Á gbõ̀, képah náh pɩ á gbáhkɩ̀-à kɩ nɛ̂ n dah n mɔ álɛ tyípékè sɔkɔ, kè kãm milki fɩ́ɩ́ álɛ. Wâh tɛkɩ á gbõ̀, képah náh pɩ á gbáhkɩ̀ sɔkɔ tɛ́ tyɛ́-ńkɛ̃̂ min wɛ, képah kɩ fṍnfṍ kal álɛ á mínndahnmɔnɔ yõ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ gbã̀n nínì ànɛ̂ gbyẽ̂h nínìnɛ á tyɩ́. 
Á yĩ́-à kɩ nɛ̂ n dah n mɔ álɛ tyípékè sɔkɔ, kè gbahlɩ n wilki n milki fɩ́ɩ́ álɛ. Wâh tɛkɩ á yĩ́ sɔkɔ tɛ́ tyɛ́-ńkɛ̃̂ min wɛ, képah kɩ fṍnfṍ kal álɛ á mínndahnmɔnɔ yõ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ yɩ̃́ nínɔ́lɛ á tyɩ́.» 
«Yé yé gblɔ̌y tɛ̃ wɛ̃́npì nɔ́pi sɔkɔ yɔ́ yílɔ́nɔ́lɛ. Ǹgbǒ yĩ̂nyõ tõ̀npɩbɩ nónó-á n yah pé yõ, pè mɛ gbĩ́mɛgbĩ́ ń Sú Liyel yah sɔkɔ yĩ̂nyõ dɛ́.  
Yé náh wɛ di, névye nónó-á mɛ mɔ́npunɔlɛ, pépi pwah n mɔ yĩn ye Névi Pídĩ́ pɔ. 
Yépi tɛ̃̀ tyɩ́ à; sɔ̃́ kénkɔ̃-á képyɛ́nɩ̀ lèkwɛ̌-à mɛ dĩ́ yɔ́ tyɩ́ à mɛ n fɛ̃ yɔ̃lɩ yõ, tɛ́ núkú mɔ pu sé sɔkɔ, à náh tikiyẽ́h pɔ́ gbãm pɔ́ kwǎryãh nɔ́pi tɛ́lɔ́ sah n pi nɛ́npɔ́, tɛ́ kyɩ mɔ́npunɔ mɛ̀ yah n kɔ̃ n pi di? 
Tɛ́ gbɩ yõ mé kélɛ n yo yé tyɩ́, wàh à wɛ tɛ̃, ǹ nɩ n vɩ ǹmɔlɛ à kal tikiyẽ́h pɔ́ gbãm pɔ́ kwǎryãh nɔ́pi tɛ́lɔ́lɛ, nónó náh mɔ pu. 
Sɛ̃́ ye yé Sú Liyel nɛ̂-á mɛ yĩ̂nyõ, ǹ tɔ náh mɛ ké tyɩ́ nɛ wɛ̃́npì nɔ́pi sɔkɔ yɔ́ ńtɛ̃ mɔ n pu.» 
Yesu tɔ̃ nɛ: «Á yṹnpi yɔ́-à tyípékè pɩ á tyɩ́, kyɩ n wil ǹ tyɩ́, ǹ ǹ yah yo yé nímí wrɔ́ ó sɔkɔ. Wàh sah á tyɩ́ yõ, yé yṹnpíkì kɩ yuku yahlɛ. 
Wàh yahle, kyɩ nɛ́núkù képah náh pɩ nímí yah n dohnɩ á gblɔ̌y yõ ńtɔ̃, yè kyɩ ké yah wah. Sɛ̃́ ye Liyel wɛy yo kélɛ nɛ, ké yah-á yɩ̃nɛ kè wah tàrfyɔ̀ nɛ́pĩ̂ nímí, képah náh pɩ tɔ̃́mɩ́ yĩ́ yah. 
Tɛ́ wàh yahle sàhnɔ́lɛ pépi tɔ tyɩ́ yõ, kyɩ ké yah yo ńmɔ yõ sɛ̃́pú tĩ̀nkì tyɩ́. Wàh se tɔ̃ yahle sàhnɔ́lɛ pépi tɔ tyɩ́ yõ, ǹnɛ nɛ n mɔ n kõ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú sɔkɔ, képah náh pɩ mɔ́rɩ́ syípú sɔkɔ. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, yâh tir nɛ̂ yahle sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ. Tɛ́ yâh nɛ̂ wɛ̃̀kɩ̀ kɔ̃ sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ. 
Mé kélɛ n yo yé tyɩ́ tɔlɛ, yé sɔkɔ nɛ́pĩ̂ nímí tɛ̃̀-à fyɔ̀ pé yégbékè pɩ núkúlɛ sétáh mɛ̀ yõ, tɛ́ kwáhdúkú nɛ̂ ni ń Sú Liyel tyɩ́, à kɩ kélɛ pè kɔ̃. 
Yé náh wɛ di, nɛ́pĩ̂ nímí képah náh pɩ tɔ̃́mɩ́-à wɛ̃ tuke mɔ ńyãh sɔkɔ ńmɔ yĩn yõ, mè n nɛ mɛ pé nɩyṍ.» 
Képah sɔkɔ, Pyɛrɩ mɛ nɔ de Yesu tnɔ̂ tɛ́ yo ǹ tyɩ́ nɛ, pé Tɛ̃̀ ǹgbɛ̃, pé yṹnpi yɔ́-à n pɩ n de pé tyɩ́, gbáhyɩ̃́ sɔ̃́ se pé yɩ̃nɛ pé yɔ̃ mɔ à kɔ̃ nɩ? Fɔ́ɔ́ gbáhyɩ̃́ kwǎrninɔ se nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Ɔ̃́ɔ̃, mé náh n yo n pi á tyɩ́ nɛ gbáhyɩ̃́ kwǎrninɔ-á dɛ́, ǹtɛ fɔ́ɔ́ gbáhyɩ̃́ kwǎrninɔ sàh-ǹtã̀nnɔ́ tikitɔ̃nɩ́ pɔ́ gbãm ye. 
Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Yõ̀tɛ̃̀ dĩ́ yɔ́ lésõ mɛ ké tyɩ́ nɛ pé pé pɔ́ tyɩ́ yah wah ànɛ̂ pé tõ̀npɩbɩlɛ. 
Pǎh tyah pɔ́ yã́hnɔ́lɛ, pè ǹ tõ̀npɩ yɔ́ tɛ̃ pɔ ǹ tyɩ́, ǹ kɔ̃lɩ ǹgbɛ̃-á sõ̀ mɛ nɛ̂ yõ, fɔ́ɔ́ talã pɔ́ plɔ́ kèyõ kwlɔ́lɛ (10 000). 
Tɛ́ kwâh náh sõ̀ mɛ dĩ́ mɛ̀ tyɩ́ ké dyah dyah tɛ̃̀nɛ. Kǎh pɩ sɛ̃́, yõ̀tɛ̃̀ dĩ́ nɛ, pè à tɛ̃ yãm, ànɛ̂ ǹ kyɩlɛ, ànɛ̂ ǹ wɛ̃́npìlɛ, à tahlɩ ǹ gbã̀n yî pól tyɩ́, pè kɔ̃lɩ mɛ̀ dyah. 
Képah tɛ̃̀nɛ, tõ̀npɩ mɛ̀ mɛ ǹ gbètukunɔ di tɛ́ ǹ yah kwĩnki sétáh yõ̀tɛ̃̀ dĩ́ yah, tɛ́ yo ǹ tyɩ́ nɛ, à ǹ gblɔ̌y yɛ à mó pé syɩkɩ n yah kwéy, páh kɩ pɔ ké pól dyah ǹ tyɩ́. 
Wǎh képah yo sɛ̃́, ǹ yãm yõ̀tɛ̃̀ dĩ́ kah tɛ̃. À mɛ à fĩn mɔ tɛ́ kɔ̃lɩ mɛ̀ yɔ̃ mɔ à kɔ̃. 
Kénɛ dĩ́-á fyɔ̀ wil, à ǹ ní-ò tõ̀npɩ yɔ́ yohnɩ ǹmɔ kɔ̃lɩ kwéy-á sõ̀ mɛ nɛ̂ yõ deniye pɔ́ plɔ́ lèkwɛ̌lɛ. À mɛ à fyãh tɛ̃ ǹ yègbɩ tyɩ́ à ǹnɛ n fɩ̃ kírr, tɛ́ à syah nɛ ńkɛ̃́, pé kɔ̃lɩ nɛ̂-á mɛ ǹ yõ, à pé dyah kélɛ. 
Ǹ ní-ò mɛ̀ mɛ ǹ gbètukunɔ di ǹ yah tɛ́ yo ǹ tyɩ́ nɛ, à ǹ gblɔ̌y yɛ à mó pé syɩkɩ n yah kwéy, páh kɩ pɔ kè dyah ǹ tyɩ́. 
Ǹtɛ à yahle, tɛ́ se tɔ̃ à tɛ̃ kyɩ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ, fɔ́ɔ́ kɩ pɔ nɔ gbĩ́ nɛ̂-á à kɩ kɔ̃lɩ mɛ̀ dyah. 
Pé no tõ̀npɩbɩ-á képah tɛ̃̀ wɛ sɛ̃́, kè pé yah kwɔ kègbɩlɛ. Pè mɛ kyɩ sé pól yo pé yõ̀tɛ̃̀ dĩ́ syah. 
Képah sɔkɔ, yõ̀tɛ̃̀ dĩ́ mɛ pè kɔ̃ pè à ye pɔ, tɛ́ yo ǹ tyɩ́ nɛ: ‹Tõ̀npɩ nɛ́po. Wáh lésõ ń ni, képah ye mé á kɔ̃lɩ yɔ̃ mɔ ǹ kɔ̃. 
Ǹtɛ áwɔ tɔ náh sõ̀ yɩ̃nɛ ǹ á ní-ò yãm yah, ńmɔ-á á yãm yah sɔ̃́ di?› 
Tɛ́ fɔ̀kɔ̀vyã̀hnɩ̀ képékè tɛ̃̀ tyɩ́, yõ̀tɛ̃̀ dĩ́ à dahbɩ wɔ̀kɔ̀npèkè névye fṍhpúpêl gbõ̀, fɔ́ɔ́ kɩ syi pɔ nɔ gbĩ́ nɛ̂-á à kɩ yõ̀tɛ̃̀ dĩ́ pɔ́ póllɛ à dyah. 
Sɛ̃́ ye ń Sú Liyel kɩ n pɩ n pi yé tɔlɛ, yâh n yɔ̃̀nmɔ-ńkɛ̃̂nɛ wɛ̃nɛ n kɔ̃ yé nnɔ póllɛ.»
