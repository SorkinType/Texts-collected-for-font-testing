Likyi-á nɩ̀vɩ̀nɩ̀ tir nɛ̂ wãrkɩ Yesu tyɩ́ sɔkɔ.
1.
Ń dĩ́ Tewofil, tyi nónó-á pɩ á wrɔ́, névye náhnáh tyah mɛ sélɛ n wãrkɩ. 
Pè sè wãrkɩ, sɔ̃́-á tàrfyɔ̀nɔ́ yo sélɛ á tyɩ́, pépi nónó-á sõ̀ wɛ sélɛ sé tyah gbĩ́ dùkùlɛ, tɛ́ sõ̀ mɛ Liyel wɛy yópúlɛ. 
Képah ye, ńmɔ tɔ mɛ tyi nɔ́pi nɩ́y tɛ̃ pallɛ, nónó-á pɩ, à tɔkɔ ké tyah gbĩ́ tyɩ́. Tɛ́ mɛ ké tyɩ́ nɛ, ńmɔ tɔ sè wãrkɩ ǹ kɔ̃ sé wɛ̃̀kɩ̀ yõ, sǎh pɩ syɩ wɛ̃ tyɩ́ sɔ̃́, ń dĩ́ Tewofil. 
Képah sɔkɔ, á kɩ pɩ̃ nɛ, wáh kwɔ́nɔ́ nɛ̂ wɛ, gbɩ-á kélɛ. 
Yerɔdɩ-á sõ̀ mɛ yõ̀tɛ̃̀nɛ Yude kwil yõ sɔkɔ gbĩ́ nɛ̂nɛ, sárká wìlkìpu yõ̀tɛ̃̀ yɔ́ sõ̀ mɛ, pè n ye Nsakarɩlɛ. Sárká wìlkìpu yõ̀tãm nónó-á sõ̀ Abya tĩ̀nkìlɛ, pépi sɔkɔ nɛy yɔ́ ye sõ̀ ǹnɛ. Ǹ kyɩ Yelisabɛtɩ tɔlɛ sõ̀, sárká wìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ Arõ náh nɛy ye ǹnɛ. 
Pé nímí sõ̀ mɛ nɛ́gblɔ́lɛ Liyel yah. Pè sõ̀ mɛ Liyel tyi kõ̀nsàhnɔ́ póllɛ n pɩ, tɛ́ mɛ ǹ tyíyónè pól tɛ̃ yĩ́ĩ́. 
Ǹtɛ, pé nímí sõ̀ dye tɛ, Yelisabɛtɩ náh sõ̀ fɛ̃ pi wɛ n pi, képah ye pi náh sõ̀ mɛ pé tyɩ́. 
Sṍ yɔ́lɛ, Nsakarɩpi tĩ̀nkì tõ̀ pɩ gbĩ́-á pɔ nɔ, à sõ̀ mɛ tõ̀lɛ n pɩ Liyel yah pélɛ. 
Pǎh dar tɛ́ sã́hntɔkɔnɔlɛ n pɩ sɔ̃́, kè sõ̀ pɔ yɩ̃nɛ ǹmɔlɛ. À sõ̀ yɩ̃nɛ à de Liyel wɔ̀kɔ̀ ǹgbɛ̃ sɔkɔ, à sèkukuri pãn pèpɛy sukɛ Tɛ̃̀ ǹgbɛ̃ Liyel yah sɔkɔ. 
Légbĩ́nɛ, tɔ́wû pól sõ̀ mɛ Liyellɛ n ni kégbɔ́ sɔkɔ, wǎh wɔ̀kɔ̀lɛ n yɔ̃ gbĩ́ nɛ̂nɛ. 
Wǎh mɛ képah sɔkɔ, yĩ̂nyõ tɛ̃ntõ̀ yɔ́ wil ǹ yah, tɛ́ mɛ yĩn tɛ̃ wɔ̀kɔ̀ yɔ̃ yɔ̃ kwâh gbõ̀yõ̀ sõ̀lɛ. 
Nsakarɩ-á à wɛ gbĩ́ nɛ̂nɛ, tíkí à tɛ̃, ǹ lékã́m fɛ̃kɩ. 
Ǹtɛ, yĩ̂nyõ tɛ̃ntõ̀ mɛ̀ yo ǹ tyɩ́: «Nsakarɩ, káh tíkí pɩ, á Liyel nínɔ́ ye tɛ̃. Á kyɩ Yelisabɛtɩ kɩ dĩ́npir dubi á tyɩ́, tɛ́ á kɩ ǹ yĩn sah Nsyɔ̃nɛ. 
Ǹ dúbínɔ́ kɩ pɩ nɩ̀vɩ̀nɩ̀ ǹgbɛ̃nɛ áwɔ ànɛ̂ névye náhnáh tyɩ́, 
ǹ nɛ́-ǹgbɛ̃ pɔ́npɩnɔ yĩnnɛ Tɛ̃̀ ǹgbɛ̃ Liyel yah. À náh dìvɛ̃ wɔ n pi, à náh tɔ̃ kònɔ́ yɔ́ ńtɛ̃ wɔ n pi, tɛ́ kɩ mɛ yúkúnɔ́lɛ Liyel Mírkí tǎhkɩ̀lɛ ǹ dúbí-ńkɛ̃̂ gbĩ́ dùkùlɛ. 
À kɩ pɔ Yisrayel névye náhnáh kɔ̃, pè kɩ pé syɩ pɔ pé Tɛ̃̀ ǹgbɛ̃ Liyel tyɩ́. 
À kɩ n kɔ Liyel yah Yeli pɩ dùkù ànɛ̂ ǹ tǎhkɩ̀lɛ ǹ sɔkɔ. À kɩ pɔ súbí ànɛ̂ pé wɛ̃́npì wrɔ́ tɔ̃ sah, tɛ́ kɩ nòhnɩ̀-ǹkɛ̃̂pú vi mɔ pɩ nòhnɩ̀pulɛ. À kɩ névye tɔ̃ sah vi mɔ Tɛ̃̀ ǹgbɛ̃ kɔ̃.» 
Wǎh yo sɛ̃́, Nsakarɩ yo ǹ tyɩ́ nɛ, péwɔ ànɛ̂ pé kyɩlɛ, pé nímí-á dye tɛ. Kǎh pɩ sɛ̃́, sɔ̃́ se pé kɩ fɛ̃ pɩ̃ nɛ, gbɩ-á à yo nɩ? 
Képah sɔkɔ, yĩ̂nyõ tɛ̃ntõ̀ yo ǹ tyɩ́: «Kabrɛlɩ ye ńmɔlɛ, nɛ̂-á n yĩn Liyel yah, à ǹ tõ̀lɛ n pɩ. À ń tɛkɩ mɔ nɛ, mé pɔ nɩ̀vɩ̀nɩ̀ wɛy mɛ̀ yo á tyɩ́. 
Ǹtɛ yah, á náh sɛ̃ ń wɛlɔ yõ. Tɛ́, sè kɩ mó tɔ̃ pɔ n pɩ n tũ n pi sé gbĩ́ sɔkɔ. Képah ye, á vyãh kɩ gbɩ tɛ̃, á náh tɔ̃ fɛ̃ n wɛkɩ n pi fɔ́ɔ́, kɩ pɔ nɔ tyi nɔ́pi pɩ́ntũ sṍ tyɩ́.» 
Légbĩ́nɛ, tɔ́wû sõ̀ mɛ Nsakarɩlɛ n syɩkɩ. Tɛ́ ǹ kàhnmonɔlɛ Liyel wɔ̀kɔ̀ ǹgbɛ̃ sɔkɔ, képah sõ̀ mɛ pé gbã̀n du. 
Ǹtɛ, Nsakarɩ-á pópó n wil, à náh mɛ fɛ̃ n yo n mɔ pé tyɩ́. Kǎh pɩ sɛ̃́, no pɩ̃ nɛ, gbǐl tir yɔ́-á wil ǹ yah Liyel wɔ̀kɔ̀ ǹgbɛ̃ sɔkɔ. À sõ̀ mɛ ǹ gbã̀n ólɛ n pɩ n mɔ pé tyɩ́, tɛ́ mɛ wɛkɩ kwlɔ. 
Képah náh, ǹ tõ̀ pɩ́nɔ́ sèpĩ̀n-á pɔ tũ Liyel tyɩ́, à ǹ syɩ yuku ǹ gbô. 
Kénɛ sèpĩ̀n sɔ́kɔ́ntɛnɔ náh, ǹ kyɩ Yelisabɛtɩ kúrkì tɔkɔ, tɛ́ kè wɔ̃bɩ ńwɔ̃́lɔ́ kwlɔ́. À sõ̀ mɛ n sõ nɛ: 
«Yah, pèpɔrɩ nɛ̂-á Tɛ̃̀ ǹgbɛ̃ Liyel pɩ ń tyɩ́ yɔ̀. À ye ńtyɩ̃́nɔ yah dah mɔ ń tyɩ́ sɔkɔ núkúnúkú, à ń sen tɔkɔ to ń yõ névye yah.» 
Képah náh, ńwɔ̃́lɔ́ kénúkò sɔ́kɔ́ntɛnɔ náh, Liyel tɔ̃ ǹ yĩ̂nyõ tɛ̃ntõ̀ Kabrɛlɩ tõ kwil yɔ́ sɔkɔ, nɛ̂-á pè n ye Nasarɛtɩlɛ, Kalile sétáh yõ. 
À à tõ yɛ̃́ndyɩ́kɩ̀ yɔ́ tyɩ́, nɛ̂-á sõ̀ mɛ gbɩ̃́nɔ́lɛ Davidi náh nɛy Yosɛfu tyɩ́. Yɛ̃́ndyɩ́kɩ̀ mɛ̀ yĩn ye sõ̀ Marilɛ. 
Wǎh kyɩ, à de yuku Mari tnɔ̂, tɛ́ yo ǹ tyɩ́: «Mé wɛkɩ á tyɩ́ Mari. Tɛ̃̀ ǹgbɛ̃ Liyel mɛ pèpɔrɩ yɔ́ tɔ̃ sah pɩ á tyɩ́, à mɛ álɛ.» 
Mari-á kénɛ wɛlɔ noh sɛ̃́, ǹ gbã̀n du. À sõ̀ mɛ n sõ ǹ fɔkɔ nɛ, yõ nɛ̂ yõ se à pɔ wɛkɩ mɔ pé tyɩ́ sɛ̃́ nɩ? 
Képah sɔkɔ, yĩ̂nyõ tɛ̃ntõ̀ yo ǹ tyɩ́: «Mari, pèpɔrɩ ye Liyel n pɩ á tyɩ́ ǹ nɩ̀gbɛ̀kɩ̀ sɔkɔ. Káh n yɛ kè pɩ tíkí tirlɛ á tyɩ́. 
Yah, á kɩ kúrkì tɔkɔ, kɩ dĩ́npir yɔ́ dubi, tɛ́ kɩ ǹ yĩn sah Yesulɛ. 
À kɩ pɩ nɛ́-ǹgbɛ̃nɛ Liyel yah. À kɩ n ye yõ̀tah Liyel pilɛ. Tɛ́ Tɛ̃̀ ǹgbɛ̃ Liyel kɩ à kɔ̃, à kɩ tɛ̃ kwil yõ ǹ náh nɛy Davidi sɔ̃́. 
À kɩ n wũ n pi Yisrayel kwil pól yõ gbĩ́mɛgbĩ́. Ǹ yõ̀tɔ̃rɩ́ náh tyɛ yah n pi póllɛ.» 
Wǎh yo sɛ̃́, Mari à piki yah nɛ, ǹtɛ, pé náh se dĩ́ tyɩ́ pɩ̃. Sɔ̃́ se kè kɩ pɩ, tɛ́ pé kɩ kúrkì tɔkɔ nɩ? 
Yĩ̂nyõ tɛ̃ntõ̀ mɛ tɔ̃ yo ǹ tyɩ́: «Liyel Mírkí kɩ tiki á yõ, tɛ́ yõ̀tah Liyel tǎhkɩ̀ kɩ fɔkɔ tũ á yõ fob sɔ̃́ mɔ. Képah ye, pi nɛ̂-á á kɩ n dubi n pi, à kɩ mɛ pèkè kɛ̃́nɛ ǹ sɔkɔ, tɛ́ kɩ n ye Liyel pilɛ. 
Yah, á wɔ̀kɔ̀ nɛy Yelisabɛtɩ, nɛ̂ tyɩ́-á no yo tũ nɛ, píkã́n sê-á ǹnɛ, à mɛ kúrkì tɔkɔ ǹ dyèntɛnɔ náh, ké ńwɔ̃́lɔ́ kénúkò-ò ye yɔ̀. À kɩ n dubi n pi dĩ́npirlɛ. 
Á náh wɛ di, Liyel tɛ̃ wɛ tyi pól tyɩ́.» 
Wǎh yo tɛ, Mari tɔ̃ ǹ syɩ yo ǹ tyɩ́: «Ńmɔ mɛ Tɛ̃̀ ǹgbɛ̃ Liyel tyɩ́ yõ. Wáh yo sɔ̃́, Liyel kè pɩ sɛ̃́ ń tyɩ́.» Képah náh, yĩ̂nyõ tɛ̃ntõ̀ ǹ syɩ sɔkɔ. 
Kénɛ sèpĩ̀n sɔkɔ, Mari yuku sɔkɔ súú ymɔ yõ, Yude kwil yɔ́ sɔkɔ. 
À kyɩ de Nsakarɩ gbô, tɛ́ wɛkɩ mɔ Yelisabɛtɩ tyɩ́. 
Yelisabɛtɩ-á Mari wɛ́kɩ́nmɔnɔ noh, piri núkú kah werke ǹ fɔkɔ sɔkɔ. Képah sɔkɔ, Yelisabɛtɩ mɛ yuku Liyel Mírkí tǎhkɩ̀lɛ, 
tɛ́ yo ǹ tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: «Sãm mɛ mɔ́nɔ́lɛ á sɔkɔ, à kal syɔ́ póllɛ. Ànɛ̂ sãm mɛ mɔ́nɔ́lɛ á fɔkɔ táh piri tɔ sɔkɔ. 
Nɛ́dúkú nɛ̂ mɛ ńmɔ tɛ̃̀ tɔlɛ, tɛ́ ń Tɛ̃̀ ǹgbɛ̃ yṹ kɩ pɔ n yah ń tyɩ́? 
Á náh wɛ di, máh fyɔ̀ á wɛ́kɩ́nmɔnɔ noh ńyãh, nɩ̀vɩ̀nɩ̀ tɛ̃̀ tyɩ́, piri núkú kah werke ń fɔkɔ sɔkɔ. 
Nɩ̀vɩ̀nɩ̀ mɛ á tyɩ́, áwɔ nɛ̂-á sɛ̃ ké yõ nɛ, Tɛ̃̀ ǹgbɛ̃ Liyel-á nɛ̂ yo, kǎh kɩ pɩ tũ.» 
Képah sɔkɔ, Mari nɛ: «Ńmɔ mɛ ń nɩ dal mɔ Tɛ̃̀ ǹgbɛ̃ Liyellɛ n gbilki, 
Ń Pwáhnmɔ-ò Liyel fɛ̃ syi yah dah mɔ ń tyɩ́ sɔkɔ. Képah ye, ń nɩ mɛ vɩ́nɔ́lɛ ǹ yĩnnɛ. Ńmɔ nɛ̂-á mɛ ǹ tyɩ́ yõ, mɛ ń gblɔ̌ynɛ n tiki n mɔ ǹ yah sɔkɔ. Gbɩ yõ, à tɔkɔ núkúnúkú tyɩ́, knɔ knɔ pól kɩ pɔ n yo n pi nɛ, máh yèvɩnɩ wɛ. 
Tyi pól tyɩ́ tɛ̃́nwɛ-ò Liyel ye tyí-ǹgbnɔ pɩ ńmɔ tyɩ́, ǹ yĩn ye Kètɔ̀kɛ̃́ Liyellɛ. 
Ǹ nɩ̀gbɛ̀kɩ̀ kɩ mɛ tɔ̃̀nmɔnyukunɔlɛ knɔ knɔ pól yõ tyɛ́-ńkɛ̃̂nɛ, nónó-á ǹ yah tíkílɛ n pɩ. 
À ǹ gbõ̀ tǎhkɩ̀lɛ tõ̀ pɩ: Gblɔ̌y gbílkínɔ́ sõnɔ-á mɛ nónó sɔkɔ, à pè karkɩ. 
À yõ̀tãm tiki mɔ pé tɔ̃́rɩ́ klǒ yõ, tɛ́ gblɔ̌y tíkínmɔpu dɔkɔ mɔ. 
À yõ yõ yîlɛ kwɛ́ndébé kɔ̃ yĩ́ĩ́, tɛ́ tahbɩ fĩn mɔ pè sɔkɔ pé gbã̀n kúkɔ́lɛ. 
À Yisrayel névye yohnɩ mɔ, pépi nónó-á mɛ ǹ tyɩ́ yõ. Ǹ dyɔ náh fwɔ ǹ nɩ̀gbɛ̀kɩ̀ pɩ́nɔ́lɛ, 
à yuku Abrahmɩ ànɛ̂ ǹ náh tãn tĩ̀nnɛ tyɛ́-ńkɛ̃̂nɛ, wǎh sõ̀ ǹ vyãh kɔ̃ sɔ̃́ á súbí ǹgbò tãn tyɩ́.» 
Képah sɔkɔ, Mari tɛ̃ Yelisabɛtɩlɛ, à ńwɔ̃́lɔ́ tɔ̃́nɔ́ kénkɔ̃ pɩ. Képah náh, à ǹ syɩ sɔkɔ ǹ tĩ̀nnɛ. 
Képah sɔkɔ, Yelisabɛtɩ dubi gbĩ́-á pɔ nɔ, à dubi dĩ́npirlɛ. 
Ǹ tnɔ̂ tɛ̃̀pu, ànɛ̂ ǹ wɔ̀kɔ̀ tãn-á noh nɛ, Tɛ̃̀ ǹgbɛ̃ Liyel-á pèpɔrɩ ǹgbɛ̃ pɩ ǹ tyɩ́, pè à tahlɩ pè ké nɩ̀vɩ̀nɩ̀ pɩ. 
Ǹ dúbínɔ́ sèpĩ̀n kwǎrtɔ̃́-ò sṍlɛ, pè pɔ piri kõ̀-ǹsah, tɛ́ mɛ n yah n kɔ̃ ǹ sú yĩn dyɩ́-ńsahlɛ ǹ tyɩ́, nɛ̂-á Nsakarɩlɛ. 
Ǹtɛ, ǹ yṹ mɛ yo pé tyɩ́ nɛ: «Ɔ̃́ɔ̃, à kɩ n ye Nsyɔ̃nɛ.» 
Pè mɛ yo ǹ tyɩ́ nɛ, ǹgbǒ ǹ wɔ̀kɔ̀ nɛy yɔ́ ńtɛ̃ yĩn náh mɛ sɛ̃́ dɛ́. 
Tɛ́ mɛ kwásɩ́ pɩ piri sú tyɩ́ nɛ, yĩn nɛ̂ se à kɩ sah à kɔ̃ nɩ? 
Képah sɔkɔ, à pè kɔ̃, pè sésár sɛ́nɔ́ yɔ́ tɔkɔ pɔ à kɔ̃, à wãrkɩ mɔ ké yõ nɛ: «Nsyɔ̃ ye ǹ yĩnnɛ.» Képah pé pól gbã̀n du. 
Nɛ́npɔ́ swɔ, Nsakarɩ nɛ́npɔ́rɩ́ dɛ mɔ, ǹ vyãh gbɩ́nɔ́ tyɛ, à tyah n wɛkɩ, à Liyel yĩnnɛ n gbilki. 
Kǎh pɩ sɛ̃́, tíkí de pé tnɔ̂ tɛ̃̀pu pól sɔkɔ, tɛ́ no sõ̀ mɛ tyi nɔ́pi póllɛ n yãh n kɔ̃ Yude ymɔ yõ kwlo pól sɔkɔ. 
Tɛ́ Tɛ̃̀ ǹgbɛ̃ Liyel sõ̀ mɛ ǹnɛ. Képah ye, nónó pól-à sõ̀ tyi nɔ́pi noh, pě sè tɛ̃ n sah pé nnɔ sɔkɔ, ǹ tɛ́ nɛ mɛ wɛ̃nɛ n piki n yah nɛ ńkɛ̃́, piri mɛ̀ se kɩ pɔ n pɩ n pi nɛ́dúkú nɛ̂nɛ nɩ? 
Képah sɔkɔ, piri sú Nsakarɩ Liyel Mírkí tǎhkɩ̀ wɛ, tɛ́ mɛ Liyel tɛ̃ntõ̀ wɛlɔlɛ n syi n yo nɛ: 
«Tɛ̃̀ ǹgbɛ̃ nɛ̂-á Yisrayel Liyellɛ, ǹ yĩn nɛ n gbilki, ǹmɔ nɛ̂-á pɔ ǹ no yohnɩ mɔ, à pè wilki yɔ̃rɩ sɔkɔ, 
tɛ́ mɛ Pwáhnmɔ-ò tǎh wilki mɔ á kɔ̃, à wil ǹ tõ̀npɩ dĩ́ Davidi náh tãn sɔkɔ. 
Képah ye lésõ à ǹ tɛ̃ntõ̀ syínyopu kɔ̃, pè yo sah di mónɔ́lɛ nɛ, 
páh kɩ á syi pwah mɔ á pěltãn tyɩ́, tɛ́ kɩ á wilki á dyɔ yɔ́pú gbõ̀. 
À á súbí ǹgbò tãn nɩ gbɛ yĩ́ĩ́. Ǹ dyɔ náh fwɔ ǹ yégbékè kwrɔ́nmɔnsàhnɔ́ tyɩ́lɛ pélɛ. 
À sõ̀ ǹ wo á ǹgbò sú Abrahmɩ tyɩ́ nɛ, 
páh kɩ á tɛ̃ syi á pěltãn gbõ̀, á kɔ̃ á fɛ̃ tõ̀lɛ n pɩ pé tyɩ́ tíkí kɛ̃́nɛ, 
á mó pɩ nɛ́gblɔ́lɛ yályál pé yah, á min pól sɔkɔ. 
Tɛ́ áwɔ tɛ̃̀ wɔlɛ, ńmɔ pi, á kɩ pɔ n kɔ Tɛ̃̀ ǹgbɛ̃ yah sɔkɔ, kɩ ǹ wã̀l tɔ̃ n sah n pi. Képah ye, pè kɩ álɛ n ye ‹Yõ̀tah Liyel tɛ̃ntõ̀ syínyo-òlɛ›. 
Tɛ́ kɩ ǹ no kɔ̃ pè kɩ pwáhnmɔnɔ dùkù pɩ̃, pé tyípêl yɔ̃̀nmɔnɔ gbõ̀ yõ. 
Ápi Liyel nɩ̀gbɛ̀kɩ̀, ànɛ̂ ǹ tɔ̃́nsàhnpɩnɔ gbõ̀ yõ, à kɩ kyòo mɔ á yõ kɩ wil yĩ̂nyõ gbĩ́ sɔ̃́ gbo, 
kɩ pɔ kyòolɛ névye kɔ̃, nónó-á mɛ kũ fɛ̃́ntũ̂ tíkí sɔkɔ, tɛ́ kɩ á yey vi mɔ yèvɩnɩ wɛ̃̀kɩ̀ yõ.» 
Tɛ́ Nsyɔ̃ tɛ̃̀ wɔ sõ̀ mɛ n kyãh, ànɛ̂ ǹ lékã́m mɛ n yɩkɩ n mɔ n yuku. À sõ̀ mɛ tɛ̃ gbɛ̃́nnɩ sɔkɔ pópó, à pɔ syɩ ǹ gblɔ̌y wɛ̃́kɩ́nɔ́ gbĩ̀yĩ̀kì tyɩ́ Yisrayel névye yah.
