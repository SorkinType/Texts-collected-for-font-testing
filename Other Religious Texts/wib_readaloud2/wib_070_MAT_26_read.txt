Matye.
26.
Yesu-á tyi nɔ́pi pól yah yo tɛ gbĩ́ nɛ̂nɛ, à yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: 
«Yé pɩ̃ nɛ ńkɛ̃́ sèpĩ̀n nínì-á wũ, tɛ́ Pakɩ syáh kɩ syah. Névi Pídĩ́ kɩ tɛ̃ pè kɔ̃ pè kɩ à syɩ dahnɩ yõ.» 
Légbĩ́nɛ, sárká syínwìlkìpu yõ̀tãm ànɛ̂ kwil yèwàhnɩ̀ wɛ̃ tuke mɔ sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ tĩ̀nnɛ, nɛ̂-á pè n ye Kayifilɛ. 
Tɛ́ yo sah núkú yõ nɛ pé ye Yesu tɛ̃ lékã́mkãm sɔkɔ, pé mó à ko mɔ. 
Pè sõ̀ mɛ yo tũ wɛ̃ tyɩ́ tɔlɛ nɛ, tir mɛ̀ náh yɩ̃nɛ kè pɩ syáh syáhnɔ́ sèpĩ̀n sɔkɔ dɛ́. Páh kɩ képah pɩ, névye-á kɩ tyah fɛ̃kɩ. 
Sõ yɔ́lɛ, Yesu lésõ mɛ tótɛ̃ dĩ́ Simo gbô, Betani kwil sɔkɔ. 
Wǎh mɛ yõke yṍnɔ́ sɔkɔ gbĩ́ nɛ̂nɛ, sê yɔ́ mɛ nɔ de ǹ tnɔ̂ nɩrɩ so pèpɛy yɔ́lɛ ǹ tyɩ́ làsèkwěl dɛ́nkɔ́kɔ̀lɛ ké sɔkɔ. Képah tɛ̃̀nɛ, tɛ́ làsèkwěl mɛ̀ mɔ Yesu yõ yõ. 
Yesu tõ̀ sã́hpú-á képah tɛ̃̀ wɛ sɛ̃́, pé fɔkɔ vyãh. Pè nɛ kwâh nɛ̂ se pɩ tɛ́ à làsèkwěl mɛ̀ yɩkɩ sɛ̃́ yɔ̀ nɩ? 
Kǎh náhkɩ sõ̀ fɛ̃ yãm dántǎhlɛ, à ké pɔ́ dibi yãm tãn kɔ̃. 
Yesu-á képah wɛ sɛ̃́, à mɛ yo pé tyɩ́ nɛ: «Sɔ̃́ pɩ tɛ́ yé pi mɛ̀ yahlɛ n vyãh sɛ̃́? Tyípépɛ̌y ye à pɩ ńmɔ tyɩ́. 
Yé yah, yãm tãn kɩ mɛ yé wrɔ́ sɔkɔ gbĩ́mɛgbĩ́. Ǹtɛ ńmɔ wɔ náh kɩ mɛ n pi yé wrɔ́ sɔkɔ gbĩ́mɛgbĩ́. 
Nɛ̂-á pi mɛ̀ pɩ yɔ̀, ńmɔ lékó yɩ̃ sõ yĩn ye à pɩ sah kélɛ. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, kèkõyṍ pól sɔkɔ, Nɩ̀vɩ̀nɩ̀ tir mɛ̀-à n yo fɩ̃́ nɛ̂ sɔkɔ, pi mɛ̀ tyípɩ́kɩ̀ mɛ̀ tɔ kɩ yo, névye dyɔ kɔ̃ kè kɩ n kwnɛ ǹ tyɩ́lɛ.» 
Yesu tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ, nɛ̂-á pè n ye Nsyudasɩ Yisikariyɔtɩlɛ, ǹmɔ mɛ kyɩ wil sárká syínwìlkìpu yõ̀tãm tyɩ́, 
tɛ́ pè syah nɛ, pè se kɩ pé kɔ̃ kwâh nɛ̂nɛ pé mó kè pɩ pè kɔ̃ pè Yesu wɛ tɛ̃ nɩ? Pè mɛ pɔ́ plɔ́ yõgbãmnɛ à kɔ̃. 
À tɔkɔ légbĩ́ tyɩ́, à tyah n yah n kɔ̃ wǎh nɛ pɩ sɔ̃́ tɛ́ pè Yesu wɛ n tɛ̃ n pi. 
Kɔ́kɔ́nɔ́ kã̂n búrn syáh tyah sõlɛ, Yesu tõ̀ sã́hpú pɔ à piki yah nɛ, ńyãh sɔkɔ se à mɛ ké tyɩ́ nɛ pé kyɩ Pakɩ syáh yõke pɩ ǹ tyɩ́ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Yé kyɩ n yo yã̀hnkõ tyɩ́ kwil sɔkɔ nɛ, kwɔ́-ò dĩ́-á yo nɛ ńkɛ̃́ pé gbĩ-á mɛ sɛ n tɛ, ńkɛ̃́ ǹmɔ tĩ̀n-á péwɔ ànɛ̂ pé tõ̀ sã́hpú kɩ pɔ n yõ n pi Pakɩ syáh yõkelɛ.» 
Wǎh kè yo sɔ̃́ ǹ tõ̀ sã́hpú tyɩ́, pè mɛ sɔkɔ ǹ yóré yõ, à kyɩ yõke mɛ̀ pɩ. 
Lékã̂h-á pɔ mɔ, ǹmɔ ànɛ̂ ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí mɛ kɔ̃ tɛ̃ yõke sõ̀. 
Tɛ́ pǎh mɛ yṍnɔ́ sɔkɔ, à yo pé tyɩ́ nɛ: «Kègbɩ yõ mé kélɛ n yo yé tyɩ́, yé sɔkɔ yɔ́ kɩ pɩ pè kɔ̃ pè kɩ ń tɛ̃ n pi.» 
Ǹ tõ̀ sã́hpú-á képah noh, pé yey kwɔ kègbɩlɛ. Képah tɛ̃̀nɛ, nɛ̂ mɛ̀ nɛ̂ mɛ̀, tɛ̃̌ à piki n yah nɛ, Tɛ̃̀ ǹgbɛ̃, péwɔ se à n ye nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Nɛ̂-á mɛ n di ńmɔlɛ dú núkú sɔkɔ yɔ̀, tɛ̃̀ wɔ ye kɩ pɩ pè kɔ̃ pè kɩ ń tɛ̃ n pi. 
Névi Pídĩ́ kɩ n ku n pi à yɩ̃nɛ kǎh wãrkɩ sah Liyel wɛy sɛ́bɛ́y sɔkɔ sɔ̃́ ǹ tyɩ́ sɔkɔ. Ǹtɛ névi nɛ̂-á kɩ pɩ pè kɔ̃ pè kɩ à tɛ̃ n pi, pèkè kɩ pɩ tɛ̃̀ tyɩ́lɛ. Tɛ̃̀-à sõ̀ mɛ dúbí-ńkɛ̃̂nɛ, képah náhkɩ fṍnfṍ ǹnɛ.» 
Nsyudasɩ nɛ̂-á sõ̀ mɛ ké tyɩ́ nɛ pé pɩ pè kɔ̃ pè à tɛ̃, ǹ tɔ mɛ à syah nɛ, kwɔ́-ò dĩ́, péwɔ se à n ye nɩ? À mɛ yo ǹ tyɩ́ nɛ: «Áwɔ ye yo kélɛ.» 
Tɛ́ pǎh mɛ yṍnɔ́ sɔkɔ, Yesu búrn tɔkɔ tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ kè kwɔkɩ à ǹ tõ̀ sã́hpúlɛ n kɔ̃, tɛ́ yo pé tyɩ́ nɛ: «Yé syi n tah kélɛ. Ńyɔlɛ, ńmɔ wil ye.» 
Képah náh, à tɔ̃ wóh tɔkɔ dìvɛ̃nɛ ké sɔkɔ tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ kélɛ pè kɔ̃ tɛ́ yo pé tyɩ́ nɛ: «Yé póllɛ, yé kè wɔ. 
Yé náh wɛ di, ńyɔlɛ, ńmɔ tṍ ye, nɛ̂-á kɩ névye yégbékè kwrɔ mɔ ànɛ̂ Liyellɛ. Kè kɩ n kwɛ n pi nɛ́kwĩ́nkí yĩnnɛ, kɩ pé tyípêl yɔ̃ mɔ. 
Mé kélɛ n yo yé tyɩ́, à tɔkɔ núkúnúkú tyɩ́, mé náh tɔ̃ dìvɛ̃ wɔ n yah n pi, pópó kɩ pɔ nɔ sõ nɛ̂-á mé kɩ tɔ̃ dìvɛ̃ sã́n yɔ́ wɔ yélɛ ń Sú Liyel tɔ̃́rɩ́ sɔkɔ.» 
Képah náh, pè syáh yɩ̀ gbɩ, tɛ́ wil à n yuku Wólívyê yèh yɔ̃lɩ yõ. 
Képah sɔkɔ, Yesu yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Kwɛy lékã̂h mɛ̀ ńtɛ̃ ólɛ, ńmɔ tyɩ́ kɩ yé pól kɔ̃ yé kɩ yal mɔ ń sõ̀ tɛ̃́nɔ́lɛ. Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: ‹Mé kɩ yípɛ́nfɛ̃ ko mɔ, tɛ́ képyɛ́nɩ̀ tũ pól kɩ karkɩ.» 
Yesu tɔ̃ nɛ: «Ǹtɛ ń lékó yílkínmɔntɛnɔ náh, mé kɩ mɔ yé yahlɛ Kalile mara sɔkɔ.» 
Képah tɛ̃̀nɛ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, no tɛ́lɔ́ pól-à kɩ sɔ̃́ yal mɔ ǹ sõ̀ tɛ̃́nɔ́lɛ, péwɔ tɛ̃̀ wɔ náh n yal n mɔ n pi póllɛ. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Kègbɩ yõ mé kélɛ n yo á tyɩ́, kwɛy lékã̂h mɛ̀ ńtɛ̃ sɔkɔ, sénpɛ-à kɩ pópó n gbo, á kɩ mɛ á vyãh kɔ̃lɩ tɛ kɛ tɔ̃́nɔ́ nɛ á náh ńmɔ pɩ̃.» 
Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, kàh kɩ sɔ̃́ nɔ pé kúnɔ́ tyɩ́ ǹnɛ wɛ̃ tyɩ́, pé náh pé vyãh kɔ̃lɩ yah n pi póllɛ nɛ pé náh à pɩ̃. Yesu tõ̀ sã́hpú tɛ́lɔ́ tɔ mɛ képah ó yo. 
Képah sɔkɔ, Yesu ànɛ̂ ǹ tõ̀ sã́hpú kyɩ yĩni tɛ̃ fɩ̃́ yɔ́ sɔkɔ, pè n ye Kɛtɩsemanɩnɛ. À mɛ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè kɔ̃ tɛ̃ ásõ̂ pé kyɩ Liyel ni n pɔ yàhàh. 
Wǎh n sɔkɔ, à Pyɛrɩ ànɛ̂ Nsebede dĩ́npì nɛ́pĩ̂ nímí tɛ̃ sɔkɔ. Yèkwɔrɩ ànɛ̂ tíkí képékè sõ̀ mɛ tyah n mɩ ǹ sɔkɔ. 
Kǎh pɩ sɛ̃́, à mɛ yo pé tyɩ́ nɛ: «Yèkwɔrɩ mɛ ń sɔkɔ, mɛ n pɩ ń ko ko tɛ̃̀nɛ. Yé tɛ̃ ásõ̂, yépi tɔ káh yé dyɔ mɔ n fwɔ n tɛ̃, sɔ̃́-á ńmɔ náh mɛ ń dyɔ mɔ fwɔ tɛ̃.» 
Tɛ́ mɛ yuku yahlɛ kwéy à kyɩ kɔ kwĩnki ǹ yah di sétáh, à Liyellɛ n ni nɛ ńkɛ̃́: «Ńmɔ Sú, kàh kɩ fɛ̃ n pɩ n pi, yèvyãhrɩ mɛ̀ gbɛ mɔ ńmɔlɛ. Ǹtɛ mé náh nɛ á kè pɩ ńmɔ-á mɛ ké tyɩ́ sɔ̃́ dɛ́, kè pɩ áwɔ-á mɛ ké tyɩ́ sɔ̃́.» 
Wǎh ǹ syɩ pɔ ǹ tõ̀ sã́hpú nɛ́pĩ̂ tɔ̃́mɩ́ nɔ́pi tnɔ̂, à pɔ pè wɛ pè mɛ n dɔ. À mɛ yo Pyɛrɩ tyɩ́ nɛ: «Yé kègbɩ yõ, yé náh tɛ̃ wɛ lɛ́rɩ́ núkú ńtɛ̃ fyé pɩ yáhnɔ́ sɔkɔ ńmɔlɛ kɛ̀? 
Yé káh yé dyɔ mɔ n fwɔ n tɛ̃. Yé nírílɛ n pɩ yè káh n pɔ tɔ̃̀nmɔnɔ fɛ̃ n syi. Tyípéplɔ sõnɔ n nɛ n mɛ névi sɔkɔ, tɛ́ sé tɛ̃́nwɛnpɩnɔ tǎhkɩ̀ náh n wɛ ǹ sɔkɔ.» 
Wǎh képah yo, à tɔ̃ gbɛ mɔ pélɛ kɛ ní-òlɛ à kyɩ Liyellɛ n ni nɛ ńkɛ̃́: «Ńmɔ Sú, ń yõ-à kɩ fɛ̃ n wil n pi ńkɛ̃̂nɛ yèvyãhrɩ mɛ̀ sɔkɔ, áwɔ-á mɛ nɛ̂ tyɩ́, képah pɩ.» 
Wǎh tɔ̃ ǹ syɩ pɔ, à pɔ pè wɛ pè tɔ̃ mɛ n dɔ. Dɔy sõ̀ mɛ tɛ̃ wɛ pé tyɩ́. 
À tɔ̃ yal pé tnɔ̂ à gbɛ mɔ kyɩ Liyel ni kɛ tɔ̃́-òlɛ gbro tãn tyɩ́ kɔ̃lɛ. 
Képah náh, à ǹ syɩ pɔ pé tnɔ̂, tɛ́ yo pé tyɩ́ nɛ: «Yé mɛ dɔy sɔkɔ sɛ̃́ mɛ yélɛ n mɩ kɛ̀? Névi Pídĩ́ tɛ̃ n dahbɩ gbĩ́ wɛ nɔ tyípêl pɩ́pú gbõ̀ yɔ̀. 
Yé yuku à sɔkɔ. Névi nɛ̂-á kɩ ń dahbɩ n pi pé gbõ̀, à wɛ n de yɔ̀.» 
Yesu mɛ képah yónɔ́ sɔkɔ, tɛ́ Nsyudasɩ nɛ̂-á ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ yɔ́lɛ, ǹmɔ yĩn nɛ́kwĩ́nkí yísyɔ́ yah pè pɔ de kèkîrn ànɛ̂ ńsninɛ pé tyɩ́. Sárká syínwìlkìpu yõ̀tãm ànɛ̂ kwil yèwàhnɩ̀ ye sõ̀ tɛkɩ mɔ pélɛ. 
Tɛ́ Nsyudasɩ nɛ̂-á sõ̀ n dahbɩ n pi Yesulɛ pé gbõ̀, à sõ̀ mɛ pɩ dùkù yɔ́lɛ pè wɛ̃kɩ sah. À nɛ páh kɩ wɛkɩ névi nɛ̂ tyɩ́ tɛ́ à pɔrɩ tɛ̃, ǹmɔ-á. Ńkɛ̃́ pè à tɛ̃. 
Tɛ́ núkú nɔ de Yesulɛ tɛ́ nɛ, kwɔ́-ò dĩ́, páh n wɛkɩ ǹ tyɩ́! Tɛ́ à pɔrɩ tɛ̃. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Ń kódĩ́, wáh pɔ nɛ̂ pɩ́-ńsah, kè pɩ.» Képah sɔkɔ, no nɔ́pi núkú nɔ de Yesulɛ à à tɛ̃. 
Képah tɛ̃̀nɛ, nónó-á sõ̀ mɛ Yesulɛ, pé sɔkɔ yɔ́ núkú ǹ kèkírkì yɔrɩ dɛ à sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ tõ̀npɩ níkí kõ kãm. 
Ǹtɛ Yesu mɛ yo ǹ tyɩ́ nɛ: «Á kèkírkì dyɩ ké gbô sɔkɔ. Yah, nónó pól-á mâllɛ n pɩ kèkîrnnɛ, kèkîrn ye kɩ ko mɔ pélɛ. 
Á náh pɩ̃ nɛ ńmɔ-á kɩ sõ̀ fɛ̃ yo mɔ ń Sú tyɩ́, à kɩ yĩ̂nyõ tõ̀npɩbɩ kèyõ náhnáh tɛkɩ mɔ ń kɔ̃ nɛ́npɔ́ swɔ, sè kɩ pɔ ń yohnɩ mɔ di? 
Ǹtɛ kàh pɩ sɛ̃́, Liyel wɛy sɛ́bɛ́y sɔkɔ tyi wã́rkɩ́nsàhnɔ́ kɩ wil dùkù nɛ̂nɛ? Kè mó tɔ̃ mɛ kè yo sah nɛ sɛ̃́-á kè yɩ̃nɛ kè pɩ.» 
Képah náh, tɛ́ yo nɛ́kwĩ́nkí nɔ́pi tyɩ́ nɛ: «Yé pɔ ńmɔ tɛ̃́-ńsah kèkîrn ànɛ̂ ńsninɛ yé tyɩ́, á kɩ nɛ kèlékɛ̃ névi sɔ̃́ ńnɛ. Ǹtɛ swɔ̃ swɔ̃ póllɛ, mé sõ̀ ńkɛ̃́nɛ nɛ n mɛ kɔ̃́ntɛ̃̀nɔ́lɛ Liyel ní-ńsah gbô sɔkɔ ǹ kwɔ́nɔ́lɛ n pɩ, tɛ́ yé náh ń tɛ̃. 
Ǹtɛ Liyel tɛ̃ntõ̀ syínyopu tyi wã́rkɩ́nsàhnɔ́ wil yĩn ye tyi nɔ́pi pól kɔ̃ sè pɩ.» Wǎh képah yo, ǹ tõ̀ sã́hpú pól to karkɩ tɛ́ à sah. 
Nónó-á tɛ̃ Yesulɛ, pè yuku ǹnɛ sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ Kayifi tĩ̀nnɛ, ńyãh sɔkɔ-á Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú ànɛ̂ yèwàhnɩ̀ sõ̀ mɛ wɛ̃ tuke mɔ. 
Pyɛrɩ sõ̀ mɛ Yesu sõ̀ tɛ̃ n kɔ fɩ́ɩ́ ǹnɛ, pópó à kyɩ yĩni tɛ̃ sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ gbô tyɩ́. Tɛ́ de yuku à kyɩ kɔ̃ tɛ̃ tõ̀npɩbɩ tnɔ̂, tɛ́ n yah yĩ mɛ̀-á kɩ pɔ n syi n pi dùkù nɛ̂nɛ. 
Sárká syínwìlkìpu yõ̀tãm ànɛ̂ pé no tukey tùkèpu tĩ̀nkì pól sõ̀ mɛ n yah n kɔ̃ nɛ, tàrfyɔ̀nɔ́ yísyɔ́ wɛ, nónó-á kɩ mɩ̀nɩ̀ wɛ n yo n sah Yesu yõ, pé mó à wɛ n ko n mɔ. 
Tàrfyɔ̀ mɩ̀nyobe náhnáh sénsɔ̃ pɔ mɩ̀nɩ̀ yo sah ǹ yõ, ǹtɛ pè náh tir yɔ́ wɛ wǎh yɩ̃nɛ à ko mɔ nɛ̂ yĩnnɛ. Ké tyɛ́-ńsah sɔkɔ, díbí nɛ́pĩ̂ nímí wil yĩn 
tɛ́ nɛ ńkɛ̃́, dĩ́ mɛ̀-á yo nɛ páh kɩ fɛ̃ Liyel ní-ńsah wɔ̀kɔ̀ kètǎr sɔ tɛ́ kɩ kè tɔ̃ mɔ sèpĩ̀n tɔ̃́nɩ̀ wrɔ́. 
Képah sɔkɔ, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ yuku yĩn tɛ́ yo Yesu tyɩ́ nɛ, no nɔ́pi-á tyi nónónɛ n yo ǹ tyɩ́ sɔkɔ, tyíyóné kó mɛ ǹmɔ tyɩ́ sépi tyɩ́ sɔkɔ nɩ? 
Ǹtɛ Yesu náh dahbɩ wilki. Képah tɛ̃̀nɛ, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ mɛ yo ǹ tyɩ́ nɛ, min nɛ́tɛ̃̂ Liyel yĩn yõ, à yo pé syah ǹmɔ-à kɩ nɛ̂ Liyel nɛy yáhntɔkɔnɔlɛ, nɛ̂-á ǹ Pídĩ́nɛ. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Áwɔ ye yo kélɛ. Tɛ́ mé kélɛ n yo yé tyɩ́, à tɔkɔ núkúnúkú tyɩ́, yé kɩ Névi Pídĩ́ wɛ kɔ̃́ntɛ̃̀nɔ́lɛ tyi pól tyɩ́ tɛ̃́nwɛ-ò Liyel gbõ̀yõ̀ sõ̀lɛ. Yé kɩ à wɛ tɔlɛ à kɩ mɛ n pi ńyĩ̂rn sɔkɔ.» 
Képah tɛ̃̀nɛ, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ ǹ gblɔ̌y flɔ yɔrɩ tɔkɩ tɛ́ nɛ, wǎh yo de Liyel tyɩ́. Tàrfyɔ̀ nɛ̂ tɛ̃̀ se tɔ̃ pé kɩ n yah n kɔ̃ nɩ? Pè kó ǹ Liyel tyɩ́ yóndenɔ noh nɩ? 
Pépi se mɛ n sõ sɔ̃́ nɩ? Pè mɛ yo ǹ tyɩ́ nɛ, wǎh yɩ̃nɛ kònmɔnɔlɛ. 
Képah tɛ̃̀nɛ, tɛ́ tyah n sĩn n mɔ ǹ yah sɔkɔ, tɛ́ ǹnɛ n syɩkɩ gbõ̀nkukelɛ, tɛ́ túkù mó mɛ ǹ létáhlɔ́lɛ n kɔ̃lɩ 
tɛ́ mɛ n yo ǹ tyɩ́ nɛ, ǹmɔ-à kɩ nɛ̂ Liyel nɛy yáhntɔkɔnɔlɛ, à sãh pɩ̃ yo pé syah nɛ̂-á n ko ǹnɛ. 
Pyɛrɩ sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ gbô táh. Tõ̀npɩ pi yɔ́ mɛ nɔ de ǹ tnɔ̂ tɛ́ yo ǹ tyɩ́ nɛ, ǹmɔ tɔ-á sõ̀ mɛ Yesulɛ, nɛ̂-á Kalile nɛynɛ. 
Ǹtɛ à mɛ kè kɔ̃lɩ no pól yĩ́ yah nɛ, pé náh mɛ ǹ wɛy mɛ̀ tyɩ́ yah wɛ yɔ̀ dɛ́. 
Képah náh, Pyɛrɩ sɔkɔ gbô yah tĩ̀nnɛ. Tõ̀npɩ pi yɔ́-á tɔ̃ à wɛ, nónó-á sõ̀ mɛ nɛ́npɔ́, à yo pé tyɩ́ nɛ, dĩ́ mɛ̀ tɔ-á sõ̀ mɛ Yesulɛ, nɛ̂-á Nasarɛtɩ nɛynɛ. 
Pyɛrɩ tɔ̃ kè kɔ̃lɩ à ǹ wo nɛ, pé náh dĩ́ mɛ̀nɛ n pnɛ dɛ́. 
Dǒ kwéy, nónó-á sõ̀ mɛ nɛ́npɔ́ pépi nɔ de Pyɛrɩlɛ tɛ́ yo ǹ tyɩ́ nɛ, kègbɩ yõ, no nɔ́pi sɔkɔ yɔ́-á ǹ tɔlɛ. Ǹ wɛy dùkù-á mɛ kélɛ n wɛ̃kɩ dò. 
Képah sɔkɔ Pyɛrɩ tyah ǹ gblɔ̌ynɛ n wo pélpèl nɛ, pé náh dĩ́ mɛ̀ pɩ̃. Nɛ́npɔ́ swɔ sénpɛ yɔ́ núkú gbo. 
Légbĩ́nɛ, Pyɛrɩ dyɔ to kõ Yesu wɛy mɛ̀nɛ, wǎh sõ̀ nɛ̂ yo nɛ ńkɛ̃́: «Sénpɛ-à kɩ pópó n gbo, á kɩ mɛ á vyãh kɔ̃lɩ tɛ kɛ tɔ̃́nɔ́ nɛ á náh ńmɔ pɩ̃.» Képah sɔkɔ, Pyɛrɩ wil kégbɔ́ à kyɩ kah gbo.
