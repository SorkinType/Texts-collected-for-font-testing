Likyi.
8.
Kénɛ tyi sɔ́kɔ́ntɛnɔ náh, Yesu sõ̀ mɛ n kɔ n kɔ̃ kwíl-ǹgbnɔ ànɛ̂ kwílpî, mɛ nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yo Liyel yõ̀tɔ̃rɩ́ tyɩ́ sɔkɔ. Tɛ́ ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sõ̀ mɛ ǹnɛ, 
ànɛ̂ syɔ́ yísyɔ́ tɔlɛ, wǎh sétã̀n yĩni wilki nónó sɔkɔ, tɛ́ pé yamnɔ tyɛ. Makadala kwil Mari sõ̀ mɛ, nɛ̂ sɔkɔ-á sétã̀n kwǎrninɔ wil, 
ànɛ̂ Nsyannɛ, nɛ̂-á Yerɔdɩ tahrɩ kyɔ̀mɩ̀-ò dĩ́ Kusa kyɩlɛ, ànɛ̂ Susanɩnɛ, ànɛ̂ syɔ́ náhnáh yísyɔ́lɛ, nónó-á sõ̀ pé gbã̀n yîlɛ Yesu ànɛ̂ ǹ tõ̀ sã́hpúlɛ n yohnɩ n mɔ. 
Képah sɔkɔ, névye sõ̀ mɛ wil kwlo kwlo pól sɔkɔ, à n pɔ n pɔ Yesu tyɩ́. Tɔ́wû ǹgbɛ̃-á pɔ tuke mɔ sɛ̃́, à mɛ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: 
«Sṍ yɔ́lɛ, sètõ yɔ́ lésõ wil ǹ sèmukɔlɛ n mir. Wǎh mɛ mírnɔ́ sɔkɔ, yísyɔ́ syɩkɩlɩ wɛ̃̀gbalɩ sɔkɔ. Wɛ̃̀kɔbɩ kɔ sɔkɔ sé yõ, kèpye mɛ pɔ sè yɔ. 
Yísyɔ́ mɛ syɩkɩlɩ légbɩ̃́ yõ, à fi, tɛ́ núkú tɔ̃ kɔkɔ vɩ́nɔ́ kɛ̃́ tɛ̃̀ tyɩ́. 
Yísyɔ́ mɛ syɩkɩlɩ de pã̀n nɩyṍ. Sǎh fi wɛ̃ tyɩ́ pã̀nnɛ, pã̀n mɛ fyãh sé yõ. 
Ǹtɛ, yísyɔ́ mɛ syɩkɩlɩ sétáh pèpɛy sɔkɔ, à fi. Sépi wɔ kyãh, à pɔ pyenɔ lèkwɛ̌kwɛ pɩ.» Sèmìrkì mɛ yã́hntɛnɔ náh, Yesu tɔ̃ yo pé tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ ńkɛ̃́: «Níkí-à mɛ nɛ̂ tyɩ́ noh noh tɛ̃̀nɛ, tɛ̃̀ ǹ níkí sah noh.» 
Képah sɔkɔ, Yesu tõ̀ sã́hpú à piki yah sèmìrkì mɛ̀ yõlɛ. 
À mɛ yo pé tyɩ́: «Liyel tɔ̃́rɩ́ tyi nónó-á mɛ wɔ̃̀bɩ̀nɔ́lɛ, sé pnɛ̀nɔ́ ye yépi tɛ̃̀ wɔ kɔ̃. Ǹtɛ, tɛ́lɔ́ tɛ̃̀ tyɩ́, sèmìrn ó sɔkɔ ye sè kɩ n yãh. Képah sɔkɔ, pè kɩ n yah díkídíkí, tɛ́ pè náh kwâh yɔ́ wɛ n pi. Tɛ́ kɩ n nohnɩ, tɛ́ pè náh ké yõ pɩ̃ n pi.» 
Tɛ́ tɔ̃ yo pé tyɩ́: «Sèmìrkì mɛ̀ yõ-á nɛ̂nɛ, kè wɛ yɔ̀: Mir mir sèmukɔ mɛ̀nɛ, Liyel wɛy ye. 
Tɛ́ névye túkù mɛ wɛ̃̀gbalɩ tyɩ́ kɔ̃lɩ: Pépi-à Liyel wɛy noh, sétõ n pɔ kè wilki pé nnɔ sɔkɔ nɛ, pè káh pɔ sɛ̃ ké yõ, pè mó pwáhnmɔnɔ wɛ. 
Túkù mɛ légbɩ̃́ yõ tyɩ́ kɔ̃lɩ: Pépi-à Liyel wɛy noh, pě kè fɛ̃ n syi nɩ̀vɩ̀lɛ. Tɛ́ pè náh n yɛ kè yã̀hnɩ̀nɛ n mɔ pé sɔkɔ. Pě kè fɛ̃ n syi nɛ́npɔ́ swɔ ó sɔkɔ, tɛ́ mɔ́nyahnɔ-à núkú pè yĩni tɛ̃, pě pé náh vi kè kɔ̃. 
Túkù mɛ pã̀n mɛ́-ńsah tyɩ́ kɔ̃lɩ: Pépi-à Liyel wɛy noh, fɔ́kɔ́sɔ̃nɔ, ànɛ̂ tahrɩ yáhnɔ́, ànɛ̂ kèkõyṍ nɩ̀vɩ̀ tyi n pɔ kè fyãh. Kàh pɩ sɛ̃́, pè n wũ nɩyṍ, kè náh tirlɛ n pɛkɩ pé tyɩ́. 
Ǹtɛ, túkù mɛ sétáh pèpɛy tyɩ́ kɔ̃lɩ: Pépi-à Liyel wɛy noh, pě kè syi n sah pallɛ nɩ pèpɛy sɔkɔ, nɛ̂-á mɛ tɔ̃́nsàhnɔ́lɛ. Ǹ tɛ́ ké tɛ̃́nɔ́ plɛ, yõre n wɛ ké tyɩ́, sèmukɔ pɛ́kɩ́nɔ́ tyɩ́ kɔ̃lɛ.» 
Yesu tɔ̃ yo pé tyɩ́: «Névi se kɩ fɔ̀tnɔ̀ soh, tɛ́ kɩ dú kwĩnki mɔ ké yah, képah náh pɩ, kɩ kè dah mɔ sɛ̃ sɛ̃ kwâh sètàhkɩ̀ sõ̀? Tɛ̃̀ náh kè sah n pi fɔ̀tnɔ̀ sahkɩ sɔkɔ, kè n pɩ n pi kyòolɛ wɔ̀kɔ̀ sɔkɔ dépú tyɩ́ di? 
Tir yɔ́ ńtɛ̃ náh mɛ wɔ̃̀bɩ̀nɔ́lɛ, tɛ́ kɩ n wũ n pi wìlkìnmɔ-ńkɛ̃̂nɛ kégbɔ́. Sètĩ̀n sõ̀ tir yɔ́ ńtɛ̃ náh n wũ n pi pɩ̃́-ńkɛ̃̂nɛ, ànɛ̂ wìlkìnmɔ-ńkɛ̃̂nɛ kwèyy. 
Névi nɛ̂ tyɩ́ kwâh-á mɛ, dòhnɩ̀nɔ́ kɩ n pɩ ǹmɔ tyɩ́ yõ. Tɛ́ névi nɛ̂ tyɩ́ kwâh náh mɛ, kwéy nɛ̂ ńtɛ̃-á à mɛ n sõ nɛ, páh mɛ wɛ, ké tɔ kɩ tɛ̃ syi ǹ tyɩ́. Kǎh pɩ sɛ̃́, yé kyɔ̀mɩ̀nɔ́lɛ n pɩ yé nohnɩ dùkù sɔkɔ!» 
Képah sɔkɔ, Yesu-á sõ̀ mɛ fɩ̃́ nɛ̂ sɔkɔ, ǹ yṹnpyé, ànɛ̂ ǹ yṹ pɔ yĩni tɛ̃ nɛ́npɔ́. Ǹtɛ tɔ́wû tɛ̃̀ tyɩ́, pè náh tɛ̃ wɛ nɔ de ǹnɛ. 
Pè mɛ yo mɔ ǹ tyɩ́ nɛ, ǹ yṹ, ànɛ̂ ǹ yṹnpyé-á mɛ yĩ́nnɔ́lɛ kégbɔ́. Pǎh mɛ ké tyɩ́ nɛ, pé à wɛ. 
Képah tɛ̃̀nɛ, à mɛ yo pé tyɩ́: «Nónó-á Liyel wɛynɛ n nohnɩ tɛ́ ké tyilɛ n pɩ, pépi ye ńmɔ yṹ ànɛ̂ ń yṹnpyélɛ.» 
Sṍ yɔ́lɛ, Yesu de krótókè yɔ́ sɔkɔ ànɛ̂ ǹ tõ̀ sã́hpúlɛ, tɛ́ yo pé tyɩ́ nɛ, pè yɛ pé yuku yɔ́-ǹgbɛ̃ kahnɩnɛ. 
Tɛ́ pǎh mɛ krótókèlɛ n to n yuku gbĩ́ nɛ̂nɛ, Yesu dɔ. Dǒ, yã́hpékè yɔ́ kõ pé yõ yɔ sɔkɔ, à ninɛ n tɔkɔ n mɔ krótókè sɔkɔ, kè syi pɔ n yuku ninɛ, dlo n tyah pé nɔ́nɔ́lɛ. 
Pè mɛ nɔ de Yesu tnɔ̂, mɛ ǹnɛ n yilki n mɔ nɛ, pé yõ̀tɛ̃̀, pé yõ̀tɛ̃̀! Yɔ-á mɛ n tyah pé yṍnɔ́lɛ dɛ́! Képah tɛ̃̀nɛ, Yesu mɛ yuku, tɛ́ dírí kõ yã́hkɩ́, ànɛ̂ ni yõ. Yã́hkɩ́ mɛ yĩn, yɔ tiki tɛ̃, tyah pɩ sṹmm. 
Kǎh pɩ sɛ̃́, à yo pé tyɩ́: «Yé Liyel yõ sɛ̃́nɔ́ ńyãh?» Tíkí tɛ̃̀ tyɩ́, pé gbã̀n du, pè tyah n yo wɛ̃ tyɩ́ nɛ, nɛ́dúkú nɛ̂ se dĩ́ mɛ̀nɛ, tɛ́ à fɛ̃ n yo yã́hkɩ́ ànɛ̂ ni ńtɛ̃ tɔ tyɩ́, sè n sah ǹ tyɩ́ yõ nɩ? 
Yesu ànɛ̂ ǹ tõ̀ sã́hpú kyɩ yĩni tɛ̃ yɔ́-ǹgbɛ̃ kahnɩnɛ, Yerasɛnɩ sétáh yõ, Kalile yèsreynɛ. 
Wǎh mɛ n tiki krótókè sɔkɔ gbĩ́ nɛ̂nɛ, dĩ́ yɔ́ wil kwil sɔkɔ, à pɔ à yohnɩ, sétã̀nnɛ ǹ sɔkɔ. Kwil mɛ̀ nɛy ye sõ̀ ǹnɛ. Tɛ́ kè pɩ di mónɔ́lɛ, à náh tɔ̃ sõ̀ mɛ flɔlɛ n mɔ, ànɛ̂ à náh mɛ n tɛ̃ gbô. Ǹ tɛ̃̀-ǹsah ye sõ̀ léyã̂l wrɔ́. 
Tɛ́, sétõ mɛ̀ sõ̀ n yuku dĩ́ mɛ̀ yõ gbáhyɩ̃́ náhnáhlɛ. Pè sõ̀ ńkɛ̃́nɛ à pukubi kɛ̃́lɔ́, ànɛ̂ kârnnɛ nɛ, pé à tɛ̃ sah. Ǹtɛ, ǎ púkúbí-ńtã̂n nɔ́pi kɔ̃lɩ, tɛ́ sétõ mɛ̀ à tɔkɔ n de gbɛ̃́nnɩ tã̀ntã̀n sɔkɔ. Képah sɔkɔ, wǎh Yesu wɛ, Yesu mɛ yo ǹ tyɩ́ nɛ, à wil dĩ́ mɛ̀ sɔkɔ. Wǎh yo sɛ̃́, dĩ́ pɔ kɔ kwĩnki ǹ yah, tɛ́ de syi yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ, kwâh nɛ̂ se à n yah n kɔ̃ pé tyɩ́ nɩ, yõ̀tah Liyel pi Yesu? Páh ǹnɛ n ni, à káh pé fõh. 
Yesu mɛ à piki yah nɛ, ǹ yĩn se sɔ̃́ nɩ? Tɛ́ kǎh pɩ, sétã̀n náhnáh sõ̀ mɛ dénɔ́lɛ ǹ sɔkɔ, à mɛ nɛ, pé yĩn-á Mɛ́fwóbé Kwĩnkilɛ. 
Képah sɔkɔ, sétã̀n nɔ́pi mɛ Yesu ni nɛ, à káh sé yĩni wilki tɛkɩ mɔ yèvyãhrɩ tɛ̃̀-ǹsah. 
Tɛ́ nɛ́npɔ́ sɔkɔ, sel kwĩnki yísyɔ́ sõ̀ mɛ n yah n yõ n kɔ̃ yɔ̃lɩ yõ. Sétã̀n nɔ́pi tɔ̃ Yesu ni nɛ, à yɛ sé kɔ̃ sé kyɩ de sel nɔ́pi sɔkɔ. À mɛ ké wɛ̃̀kɩ̀lɛ sè kɔ̃. 
Képah tɛ̃̀ sɔkɔ, sétã̀n nɔ́pi wil dĩ́ mɛ̀ sɔkɔ tɛ́ kyɩ de sel nɔ́pi sɔkɔ. Sel kwĩnki nɔ́pi núkú kènpĩn to tiki tùkùtùkù, à kyɩ de yɔ́npɛ̃́kɩ̀ sɔkɔ, kè kɔ̃ kè sè yõ. 
Sel nɔ́pi fɛ̃̀pu-á képah wɛ sɛ̃́, pè to kyɩ kè yãh kwil, ànɛ̂ yèpìyey sɔkɔ. 
Képah tɛ̃̀nɛ, tyi nónó-á pɩ, névye wil nɛ, pé kyɩ sé ségblɔ́ yah. Pè pɔ yĩni tɛ̃ Yesu tnɔ̂, tɛ́ dĩ́ nɛ̂-á sõ̀ sétã̀n wil ǹ sɔkɔ, pè à wɛ kɔ̃́ntɛ̃̀nɔ́lɛ Yesu tnɔ̂. À sõ̀ mɛ flɔ mɔ, tɛ́ mɛ tírr. Kǎh pɩ sɛ̃́, tíkí pè tɛ̃. 
Képah sɔkɔ, nónó-á sõ̀ ǹ dɛ́nmɔ dùkù wɛ, pè kè yãh tɛ́lɔ́ syah. 
Kǎh pɩ sɛ̃́, tíkí tɛ̃̀ tyɩ́, Yerasɛnɩ sétáh yõ névye pól kè yah kɔ̃ Yesu tyɩ́ nɛ, à yal gbɛ mɔ pépilɛ. Képah sɔkɔ, Yesu mɛ de krótókè sɔkɔ, à n tyah sɔ́kɔ́nɔ́lɛ. 
Tɛ́ dĩ́ nɛ̂ sɔkɔ-á sõ̀ sétã̀n wil, kénɛ dĩ́ Yesulɛ n ni nɛ, à yɛ pé tɛ̃ ǹnɛ. Ǹtɛ, Yesu náh kè fɛ̃ syi, tɛ́ yo ǹ tyɩ́: 
«Liyel-á tyi nónó pól pɩ á tyɩ́, á syɩ n yuku á gbô, ǹ kyɩ sè yãh.» Wǎh yo sɛ̃́, dĩ́ mɛ ǹ syɩ yuku ǹ tĩ̀nnɛ. Tɛ́ tyi nónó pól-á Yesu pɩ ǹ tyɩ́, à sélɛ n yãh n kɔ̃ kwil pól sɔkɔ. 
Tɔ́wû sõ̀ mɛ Yesu syɩkɩ tɛ̃ sɛ̃́, képah ye, wǎh wil yɔ́-ǹgbɛ̃ kahnɩnɛ, à pɔ yĩni n tɛ̃ pé tnɔ̂ gbĩ́ nɛ̂nɛ, pé pól à yohnɩ nɩ̀vɩ̀ sɔkɔ. 
Légbĩ́ mɛ̀ ó sɔkɔ ye dĩ́ yɔ́ pɔ yĩni tɛ̃, pè n ye Yayirusilɛ. Ǹmɔ ye sõ̀ nɛ́npɔ́ Liyel ní-ńsah wɔ̀kɔ̀ yah yétɛ̃̀nɛ. Wǎh yĩni tɛ̃ Yesu tnɔ̂, à gbètukunɔ di ǹ yah, tɛ́ à ni nɛ ńkɛ̃́, kékwɛ, à pɔ pé yuku péwɔ gbô, 
ńkɛ̃́, písê nɛ̂-á pé tyɩ́ ǹ gblɔ̌y núkúnínɛ, tɛ́ mɛ ye gbɔ̃ninɔ́ kénkɔ̃ sɔkɔ, wǎh mɛ n tyah kúnɔ́lɛ. Képah tɛ̃̀ sɔkɔ, Yesu-á n sɔkɔ nɛ́npɔ́, tɔ́wû à fyãh kã́mm yey yey póllɛ. 
Tɛ́, sê yɔ́ tɔ sõ̀ mɛ nɛ́npɔ́. À sõ̀ mɛ n yɔ tṍ kwɛ́nɔ́ ńyânnɛ pópó, ye gbɔ̃ninɔ́. Ǹ gbõ̀ pól syi pɔ tyɛ syɛy pɩ́nɔ́lɛ sènpɩbɩ tyɩ́, tɛ́ pé sɔkɔ yɔ́ ńtɛ̃ náh tɛ̃ wɛ à dɛ mɔ. 
À mɛ nɔ de Yesulɛ ǹ náh tĩ̀nnɛ, tɛ́ tũ ǹ fwɔ̀mɔnɩ vyãh tyɩ́. Nɛ́npɔ́ swɔ, ǹ ńyân núkú tyɛ. 
Képah sɔkɔ, Yesu mɛ piki yah nɛ, nɛ̂ se tũ péwɔ tyɩ́ nɩ? No pól-á pé vyɛ̃ynɛ n kɔ̃lɩ nɛ, pépi náh sɛ̃́, Pyɛrɩ mɛ nɛ, pé yõ̀tɛ̃̀, névye-á fyãh sah ǹnɛ kã́mm. 
Ǹtɛ, Yesu mɛ nɛ, ǹnɩ̂ yɔ́-á tũ pé tyɩ́. Páh kè pɩ̃ nɛ, tǎhkɩ̀ yɔ́-á wil pé sɔkɔ. 
Képah sɔkɔ, pi mɛ̀-á pɔ pɩ̃ nɛ, tir mɛ̀ náh tɔ̃ fɛ̃ n wɔ̃bɩ, à pɔ ǹ gbètukunɔ di Yesu yah wílsyahrɩ sɔkɔ. Tir nɛ̂ yĩn-á à tũ ǹ tyɩ́, tɛ́ kè yãh no pól yĩ́ yah, ànɛ̂ sɔ̃́-á à dɛ́nmɔnɔ wɛ nɛ́npɔ́ swɔ. 
Képah tɛ̃̀ sɔkɔ, Yesu mɛ yo ǹ tyɩ́: «Pi, á Liyel yõ sɛ̃́nɔ́ ye ǹ dɛ mɔ. Nɛ n yuku yèvɩnɩ sɔkɔ.» 
Gbĩ́ nɛ̂-á Yesu mɛ kénɛ wɛy yónɔ́ sɔkɔ, dĩ́ yɔ́ wil yõ̀tɛ̃̀ dĩ́ Yayirusi gbô, à pɔ yo Yayirusi tyɩ́ nɛ, ǹ písê-á yah kwlɔ, ńkɛ̃́, à káh fwɔ̌tɛ̃̀ kwɔ́-ò dĩ́ kwlɔ. 
Ǹtɛ, Yesu-á képah noh, à mɛ yo Yayirusi tyɩ́ nɛ ńkɛ̃́: «Káh tíkí pɩ! Sɛ̃ ó sɛ̃ ké yõ, tɛ́ á písê kɩ ǹ min wɛ.» 
Tɛ́ pè mɛ sɔkɔ Yayirusi gbô. Pǎh kyɩ yĩni tɛ̃, à náh fɛ̃ syi nɛy de ǹnɛ, à wil Pyɛrɩ, ànɛ̂ Nsyɔ̃, ànɛ̂ Nsyakɩ, ànɛ̂ pípír sú, ànɛ̂ ǹ yṹ tyɩ́ sɔkɔ. 
Légbĩ́nɛ, no pól sõ̀ mɛ pípír kũ pɔ̀lɛ n gbo mɛ yãm yãm tyilɛ n pɩ. Képah sɔkɔ, Yesu nɛ: «Yé káh nɛ n gbo! À náh ku, dɔy sɔkɔ ye ǹnɛ.» 
Ǹtɛ, névye-á sõ̀ mɛ ǹ kúntɛnɔ pɩ̃ tɛ sɛ̃́, pè tyah kõlɛ Yesulɛ n ko. 
Képah tɛ̃̀ sɔkɔ, Yesu mɛ à tɛ̃ ǹ gbõ̀ tyɩ́, tɛ́ à kah ye: «Ń pi, yuku!» 
Nɛ́npɔ́ swɔ, ǹ min núkú ké syɩ de ǹ sɔkɔ à kɔ̃ à yuku. Wǎh yuku, Yesu mɛ kè yah kɔ̃ pé tyɩ́ nɛ, pè yãrlɛ à kɔ̃ à yõ. 
Kǎh pɩ sɛ̃́, ǹ dúbípú gbã̀n du, tɛ́ Yesu kè yah kɔ̃ pé tyɩ́ nɛ, pè káh tir mɛ̀ yãh nɛy tyɩ́.
