Tyípɩ́nɩ̀.
20.
Tɛ́ fɛ̃̀kɩ̀nɔ́ mɛ̀-á pɔ yĩn mɔ gbĩ́ nɛ̂nɛ, Pole mɛ Yesu tõ̀ sã́hpú ye tuke, à sṍré mɔ pé sɔkɔ, tɛ́ wɛ̃̀kɩ̀ ni pé tyɩ́, à sɔkɔ Masedonɩ gbɛ̃̀nyah sɔkɔ. 
À kénɛ gbɛ̃̀nyah pól kɔ kɔ̃, à sṍré yah mɔ kɔ̃ náhnáh Yesu yõ sɛ̃́pú sɔkɔ. Wǎh yal nɛ́npɔ́, à sɔkɔ Krɛsɩ gbɛ̃̀nyah, 
à kyɩ ńwɔ̃́lɔ́ tɔ̃́nɔ́ pɩ nɛ́npɔ́. Képah náh, wǎh pɔ ǹ vi nɛ, pé de krótókè ǹgbɛ̃ sɔkɔ, pé yal n yuku Siri gbɛ̃̀nyah sɔkɔ, à kè wɛ noh nɛ, Nsyifunɔ-á mɛ ǹnɛ n plɔ pèkè yɔ́lɛ. Kǎh pɩ sɛ̃́, à kè yah tɔkɔ nɛ páh kɩ pé syɩ syɔ̃ Masedonɩ gbɛ̃̀nyahlɛ. 
Tɛ́ nónó-á sõ̀ Pole wɛ̃̀kɩ̀ kɔ́mûnɛ, pépi wɛ yɔ̀: Sopatɛrɩ nɛ̂-á n wil Bere kwil, tɛ́ ǹ sú yĩn mɛ Pirusilɛ, ǹmɔ sõ̀ mɛ, à tahlɩ Arisitarkɩ, ànɛ̂ Sekũdusi tyɩ́, pépi nónó-á n wil Tesaloniki kwil, ànɛ̂ Kayusilɛ, nɛ̂-á n wil Dɛrɩbɩ kwil, ànɛ̂ Tisyikilɛ, ànɛ̂ Torofiminɛ, pépi nónó-á n wil Asi gbɛ̃̀nyah sɔkɔ, à tahlɩ Timote tyɩ́. 
Pépi mɛ sɔkɔ ápi yahlɛ, à kyɩ álɛ n syɩkɩ Turasɩ kwil. 
Búrn sɛ̃̀-ǹkɛ̃̂ syáh syáhnsɔkɔntɛnɔ náh, ápi wɔ mó kyɩ de krótókè ǹgbɛ̃ yɔ́ sɔkɔ Filipi kwil, à pé nyáh sõ. Á sèpĩ̀n kwâl pɩ tɛ́ pyě de á no tnɔ̂ Turasɩ kwil. Képah tɛ̃̀nɛ, á tɛ̃ nɛ́npɔ́ à sèpĩ̀n kwǎrninɔ pɩ. 
Yòh wrɔ́ gbĩ̀yĩ̀kì gben sõlɛ, á lésõ mɛ tuke mɔ Tɛ̃̀ ǹgbɛ̃ yõke yõ yĩnnɛ, tɛ́ Pole mɛ yã́hkɩ̀lɛ n yãh tɔ́wûlɛ. À kɩ sõ̀ n yal n pi dĩndallɛ Turasɩ kwil. Kǎh pɩ sɛ̃́, à Liyel wɛy yo mo pé tyɩ́, pópó à pɔ nɔ tyah tɔ́nɔ́ tyɩ́. 
Tɛ́ á pól sõ̀ mɛ tùkènmɔnɔlɛ gbɛ̃́ntĩ̂n yõ wɔ̀kɔ̀ tɔ̃́-ò sɔkɔ, tɛ́ fɔ̀tnɔ̀nɔ́ mɛ tũ̀nsàhnɔ́lɛ pípí ké sɔkɔ. 
Kètɛ̃̀npìkì yɔ́ tɔ sõ̀ mɛ álɛ pè n ye Yetikilɛ. Ǹmɔ sõ̀ mɛ kɔ̃ tɛ̃ fnɛ̀tír vyãh yõ. Dǒ, dɔy pɔ mɩ ǹ sɔkɔ. Ǹtɛ Pole Liyel wɛy yõ-á sõ̀ mɛ ko sɛ̃́, à pɔ dɔ, tɛ́ tɛ̃ syi yĩ̂nyõ wɔ̀kɔ̀ tɔ̃́-ò sɔkɔ, fɔ́ɔ́ à pɔ syɩkɩ sétáh. Pǎh pópó pɔ nɛ pé à yilki sah, pè à wɛ kúnɔ́lɛ. 
Képah tɛ̃̀nɛ, Pole tɔ tiki pɔ, tɛ́ kɔ à yɔ̃ ǹ gbã̀n sɔkɔ, tɛ́ yo pé tyɩ́ nɛ: «Yé káh n yɛ yé yáh-ńsah tyɛ; à mɛ min sɔkɔ.» 
Képah náh, à ǹ syɩ dɔkɔ yuku ńtɔ̃, pè kyɩ Tɛ̃̀ ǹgbɛ̃ yõke yõ wɛ̃ tyɩ́, tɛ́ tɔ̃ yɔ̃ sɔkɔ Liyel wɛy yónɔ́lɛ, pópó à pɔ nɔ tyah dal gbĩ́ tyɩ́, tɛ́ à pyě sɔkɔ. 
Képah sɔkɔ, no mɛ Yetiki tɛ̃ sɔkɔ ǹ vilɛ krémkrém ǹ gbô. Yĩ mɛ̀ pɩ yĩ́ĩ́ nɩ dénɔ́ ǹgbɛ̃nɛ no pól tyɩ́. 
Pole sõ̀ kè yah tɔkɔ, à sɔkɔ Asɔsɩ kwil sɔkɔ, gbáhkɩ̀ yõ. À sõ̀ nɛ ápi pé yah yohnɩ nɛ́npɔ́, á pé twah. Kǎh pɩ sɛ̃́, ápi mɛ de krótókè ǹgbɛ̃ sɔkɔ, à mɔ ǹ yahlɛ à kyɩ ǹnɛ n syɩkɩ nɛ́npɔ́, ǹ twah yĩnnɛ. 
Wǎh pɔ de, á pól de krótókè sɔkɔ, à yɔ̃ sɔkɔ à kyɩ yĩn Mitilɛnɩ kwil. 
Képah náh, á tɔ̃ yɔ̃ sɔkɔ. Ké sèpĩ̀n ní-ò sõlɛ, á kyɩ yĩni tɛ̃ gbɛ̃̀nyah yɔ́ yèsreynɛ pè n ye Kyolɛ, pnɛ yɔ nɩyṍ. Ké sèpĩ̀n tɔ̃́-ò sõlɛ, á kyɩ sɔkɔ gbɛ̃̀nyah yɔ́ gbɔ̀pɔ pè n ye Samoselɛ, pnɛ yɔ nɩyṍ. Ké sèpĩ̀n ńyã́h-ò sõlɛ, á pyě yĩni tɛ̃ Milɛti kwil. 
Tɛ́ Pole sõ̀ mɛ sàlnɔ́ sɔkɔ nɛ kàh kɩ fɛ̃ n pɩ n pi, pé nɛ mɛ Yerusalɛmɩ kwil Pãntekɔtɩ syáh syah sõlɛ. Képah ye à kɔ̃ à yɔ̃ sɔkɔ, à náh syɔ̃ Yefɛsɩ kwillɛ. À sõ̀ pɩ̃ nɛ páh kɩ de nɛ́npɔ́, pé náh wɛ n wil n pi súú Asi gbɛ̃̀nyah sɔkɔ. 
Képah sɔkɔ, Pole tɛ̃ Milɛti kwil, tɛ́ tɛ̃ntõ̀ mɔ Yefɛsɩ kwil Yesu yõ sɛ̃́pú yah tãm tyɩ́ nɛ pè pɔ. 
Pǎh pɔ yĩni tɛ̃ ǹ tnɔ̂, à yo pé tyɩ́ nɛ: «À tɔkɔ ńmɔ pɔ sõ tyɩ́ Asi gbɛ̃̀nyah sɔkɔ, máh mɛ n pɩ sɔ̃́ gbĩ́mɛgbĩ́ yé wrɔ́, yé se kélɛ n pnɛ tɛ́. 
Mé tõ̀ pɩ Tɛ̃̀ ǹgbɛ̃ tyɩ́ gblɔ̌y tíkínmɔnɔ kègbɩ sɔkɔ, à yétĩ kwɛ. Tɛ́ Nsyifunɔ tɔ mɛ vi ń náh gbáhyɩ̃́ náhnáh sɔkɔ, à yèvyãhrɩ náhnáhlɛ ń yĩni tɛ̃. 
Ǹtɛ yé se pɩ̃ nɛ, mé náh tir yɔ́ ńtɛ̃ wɔ̃bɩ yélɛ nɛ̂-á kɩ fɛ̃ yé tyɩ́ pɛkɩ n pi tɛ́. Mé sõ̀ mɛ sé póllɛ n yo yé tyɩ́, tɛ́ mɛ yélɛ n kwɔ tɔ́wû tùkè-ǹtã̀n sɔkɔ ànɛ̂ yé gbên tɔ sɔkɔ. 
Mé kè yo kɔ̃ Nsyifunɔ ànɛ̂ Krɛkɩ névye tɔ tyɩ́ nɛ, pè pé nnɔ vi mɔ, pè mó sɛ̃ ápi Tɛ̃̀ ǹgbɛ̃ Yesu yõ. 
Ǹtɛ núkúnúkú, Liyel Mirki wɛ ń kyɩ̃ mɔ mé n sɔkɔ Yerusalɛmɩ kwil yɔ̀. Mé náh pɩ̃ tir nɛ̂ tɛ̃̀ wɔ-á kɩ kyɩ ń yĩni n tɛ̃ n pi nɛ́npɔ́. 
Máh nɛ̂ tɛ̃̀ wɔlɛ n pnɛ, mâh de kwil nɛ̂ sɔkɔ, Liyel Mirki ńkɛ̃́nɛ kélɛ n wɛ̃kɩ nɛ, wɔ̀kɔ̀npèkè sɔkɔ dénɔ́ ànɛ̂ yèvyãhrɩ-á mɛ ń syɩkɩ tɛ̃. 
Ǹtɛ sépinɔ́ pól sɔkɔ, ń gblɔ̌y min tɛ̃̀ tyɩ́ náh mɛ tah ń tyɩ́ póllɛ. Mé náh mɛ kélɛ n mɔ n kõ kwáhtǎhlɛ. Tir nɛ̂ tyɩ́-á mɛ tah ń tyɩ́, képah ye pɩ ń tõ̀ pɩ́ntɛnɔlɛ; tõ̀ nɛ̂-á Tɛ̃̀ ǹgbɛ̃ dahbɩ ń gbõ̀ sɔkɔ nɛ mé Nɩ̀vɩ̀nɩ̀ tirlɛ n yo n kɔ̃ Liyel pèpɔrɩ tyɩ́ sɔkɔ. 
Ǹtɛ núkúnúkú, mé pɩ̃ nɛ máh syɔ̃ yépi nónó pól tyɩ́ à Liyel tɔ̃́rɩ́ tyɩ́ yo kɔ̃, yé náh tɔ̃ ń wɛ n yah n pi. 
Ǹtɛ mé náh yé wɔ̃bɩ Liyel dyɔ yĩ́ĩ́ tir yɔ́ ńtɛ̃nɛ, mé ye sé pól yo sɔkɔ yé tyɩ́. Képah ye, mé kélɛ n yo n dal n mɔ yé tyɩ́ kwɛynɛ nɛ, yé sɔkɔ nɛy nɛ̂-à kɩ se tɔ̃ mɔ pu, ké kémû náh n pɩ n pi ńmɔ tyɩ́lɛ. 
Yé yé gblɔ̌y tɛ̃́nɔ́ plɛ, yè mó Liyel yípɛ̂y tɛ̃́nɔ́ plɛ, Liyel Mirki-á yé sah sépi nónó yõ nɛ, yé sélɛ n tɛ̃ plɛ. Sépi ye Yesu yõ sɛ̃́pú tĩ̀nkìlɛ, Liyel-á nónó pɩ ǹ tyɩ́lɛ ǹ pídĩ́ tṍ kwɛ́nɔ́ gbõ̀ yõ. 
Mé pɩ̃ nɛ, ńmɔ sɔ́kɔ́ntɛnɔ náh, névye yísyɔ́-á kɩ pɔ de yé wrɔ́ gbɛ̃́nképèl tyɩ́ kɔ̃lɛ, kɩ yényêh pɩ yípɛ̂y sɔkɔ. 
Tɛ́ yépi ńtɛ̃ sɔkɔ, névye túkù kɩ pɔ yuku tyah mɩ̀nɩ̀ wɛlɔlɛ n yo, kɩ Yesu tõ̀ sã́hpú yõ yɩkɩ, pè kɔ̃ pè kɩ pé sõ̀ tɛ̃. 
Kǎh pɩ sɛ̃́, yé yé gblɔ̌ynɛ n kyɔmɩ, yè mó yé dyɔ sah nɛ n kwnɛ nɛ, máh lésõ yé kwɔ ye tɔ̃́nɔ́, ké lékã̂h ké gbĩ̀ntɔ̃̀, à vìnmɔnɔlɛ yé nɛ́núkù núkú pól kɔ̃. Tɛ́ kèyǎhlɛ ń yétĩ ńtɛ̃ n kwɛ ké tyɩ́ sɔkɔ. 
Tɛ́ núkúnúkú, mé yélɛ n dahbɩ Liyel ànɛ̂ ǹ pèpɔrɩ wɛy gbõ̀ sɔkɔ. Tǎhkɩ̀ mɛ ǹmɔ sɔkɔ kɩ fɛ̃ yé kɔ̃ yé kɩ yuku yahlɛ ǹmɔ yõ sɛ̃́nɔ́ sɔkɔ, tɛ́ wǎh ǹ vyãhlɛ yé kɔ̃ ǹ pèpɔrɩ nɛ̂ tyɩ́ sɔkɔ, yé kɩ kè wɛ wɛ̃ tyɩ́ ànɛ̂ nónó pól-á mɛ Liyel tyɩ́ yõ ǹ yõ sɛ̃́nɔ́ yĩnnɛ. 
Mé náh pɔ́, ànɛ̂ sã́n, ànɛ̂ flɔ yah kɔ̃ ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́. 
Yé pól pɩ̃ nɛ, máh tõ̀ pɩ ń gbã̀nnɛ à yĩn ń gblɔ̌y yĩ́nnɔ́lɛ, ànɛ̂ ń kɔ́mû tɔ yĩ́nnɔ́lɛ kɛ̀. 
Mé kélɛ yé wɛ̃kɩ tyi pól sɔkɔ nɛ, sɛ̃́-á névi yɩ̃nɛ à tõ̀lɛ n pɩ, à kwâhlɛ n wɛ à yãm tãnnɛ n yohnɩ n mɔ, à mó ǹ dyɔ sah nɛ n kwnɛ Tɛ̃̀ ǹgbɛ̃ Yesu wɛy mɛ̀ tyɩ́lɛ wǎh nɛ̂ yo nɛ ńkɛ̃́, sãm-á mɛ kɔ̃́nɔ́ sɔkɔ, à kal wɛ́nsyinɔlɛ.» 
Pole-á wɛlɔ nɔ́pi yo pɔ tyɛ, à kuku yĩn pélɛ wɛ̃ tyɩ́, pé pól Liyel ni. 
Tɛ́ Pole sõ̀ yo pé tyɩ́ nɛ, pè náh tɔ̃ pé yèpɔ wɛ n yah n pi póllɛ. Képah sɔkɔ, pé pól tyah ǹnɛ n pɔrɩ n tɛ̃ n pɔ à n wɛkɩ ǹ tyɩ́, tɛ́ n gbo yèkwɔrɩ tɛ̃̀ tyɩ́. Képah náh, pè kyɩ à tɛkɩ mɔ, fɔ́ɔ́ krótókè tɔ́kɔ́-ńsah sɔkɔ.
