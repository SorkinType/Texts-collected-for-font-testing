Markɩ.
10.
Yesu-á yal nɛ́npɔ́, à kyɩ Yude sétáh yõ, tɛ́ yuku Yurdɛ̃ yɔ́nkahnɩnɛ. Képah sɔkɔ, tɔ́wû ǹgbɛ̃ yɔ́ tɔ̃ tuke mɔ ǹ tnɔ̂. À mɛ tyah pélɛ n kwɔ ǹgbò tyɩ́ kɔ̃lɛ. 
Képah tɛ̃̀nɛ, *Farisiyɛ̃nɔ yísyɔ́ nɔ de ǹ tnɔ̂ nɛ, pé kyɩ ǹ sõ̀ mɔ yah. Tɛ́ ǹnɛ n piki n yah nɛ, ké wɛ̃̀kɩ̀ se mɛ dĩ́ tyɩ́ kɩ ǹ kyɩ fĩn mɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ, tir nɛ̂ se sõ̀ *Moyisi yo sah pé tyɩ́ nɩ? 
Pè mɛ nɛ ńkɛ̃, Moyisi-á ké wɛ̃̀kɩ̀ kɔ̃ nɛ, wâh á kyɩlɛ n fĩn mɔ, á ǹ fĩ́nnmɔnɔ tàrfyɔ̀ sɛ́bɛ̃́ pɩ à kɔ̃. À mó à fĩn mɔ. 
Képah tɛ̃̀ sɔkɔ, Yesu yo pé tyɩ́: «Yépi kùkùtǎhkɩ̀ yĩn ye, Moyisi mɛ ké wɛ̃̀kɩ̀lɛ yé kɔ̃ sɛ̃́. 
Ǹtɛ, kèkõyṍ mɔ gbĩ́nɛ, dĩ́ ànɛ̂ sê ye Liyel mɔ. 
Képah ye, dĩ́ kɩ ǹ sú ànɛ̂ ǹ yṹ sah, tɛ́ kɩ kyɩ yégbékè kwrɔ mɔ ànɛ̂ ǹ kyɩlɛ, 
tɛ́ pé nímí kɩ pɩ núkúlɛ. Képah sɔkɔ, pè náh tɔ̃ n pɩ n pi nímínɛ, pè kɩ pɩ núkúlɛ. 
Kǎh pɩ sɛ̃́, Liyel-á nɛ̂ kwrɔ mɔ, névi káh kè gbɛkɩ.» 
Pǎh de wɔ̀kɔ̀, ǹ *tõ̀ sã́hpú tɔ̃ à piki yah wɛy mɛ̀ yõlɛ. 
À mɛ yo pé tyɩ́: «Dĩ́-à ǹ kyɩ fĩn mɔ, tɛ́ tɔ̃ sê yɔ́ tɔkɔ, à tyípékè pɩ ǹ kyɩ gben tɛ̃̀ tyɩ́ fɔ̀fɩ́kɩ̀ sɔkɔ. 
Sɛ̃́ ó ye tɔ̃, sê-à ǹ pɛ yahle, tɛ́ kyɩ de dĩ́ yɔ́ tyɩ́, à tyípékè pɩ fɔ̀fɩ́kɩ̀ sɔkɔ ǹ pɛ gben tɛ̃̀ tyɩ́.» 
Képah tɛ̃̀nɛ, névye pɔ wɛ̃́npìlɛ Yesu tyɩ́ nɛ, à ǹ gbã̀n sah pé yõ. Tɛ́ ǹ *tõ̀ sã́hpú mɛ tyah dírílɛ n kõ pé yõ. 
Wǎh képah wɛ sɛ̃́, à yo ǹ tõ̀ sã́hpú tyɩ́ fɔ̀kɔ̀vyã̀hnɩ̀ sɔkɔ nɛ: «Yé yɛ wɛ̃́npì kɔ̃, pè nɛ n pɔ n pɔ ńmɔ tyɩ́. Yé káh pé yahlɛ n kõ. Nónó dùkù-á mɛ kénɛ wɛ̃́npì sɔ̃́, *Liyel yõ̀tɔ̃rɩ́lɛ pépi tyɩ́ ye. 
Mé kélɛ n yo yé tyɩ́ kègbɩ yõ: Névi nɛ̂-à *Liyel tɔ̃́rɩ́ fɛ̃ syi ńkɛ̃̂nɛ wɛ̃́npir sɔ̃́, à náh kè wɛ n de n pi póllɛ.» 
Képah sɔkɔ, à wɛ̃́npì tɛ̃, tɛ́ ǹ gbã̀n sah pé yõ, à sãm mɔ pé sɔkɔ. 
Képah náh, Yesu-á n yuku wɛ̃̀kɩ̀ yah, dĩ́ yɔ́ to pɔ gbètukunɔ di ǹ yah, tɛ́ à piki yah nɛ, kwɔ́-ò pèpɛy, pé se kɩ pɩ sɔ̃́, tɛ́ kɩ tyɛ́-ńkɛ̃̂ min wɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́: «Sɔ̃́ mɛ, tɛ́ á ń ye pèpɛynɛ? Nɛ pèpɛy yɔ́ ńtɛ̃ náh mɛ. Liyel ó ye pèpɛynɛ. 
Á Liyel tyi kõ̀nsàhnɔ́ pɩ̃ kɛ̀, nɛ̂-á n yo nɛ, káh nɛ́kókè mɔ, káh fɔ̀fɩ́kɩ̀ pɩ, káh yɩ̃be yɩ̃, káh mɩ̀nɩ̀ sah á ní-ò yõ, káh nɛ n syɩkɩ nɛ yõ, á sú ànɛ̂ á yṹnɛ n tɛ̃ plɛ.» 
Wǎh yo sɛ̃́, dĩ́ mɛ à syah nɛ, kwɔ́-ò dĩ́, páh mɛ sépinɔ́ póllɛ n pɩ pé wɛ̃́npir gbĩ́ dùkùlɛ. 
Képah sɔkɔ, Yesu mɛ à yah tyɩ́nɔnɔ sɔkɔ, tɛ́ yo ǹ tyɩ́ nɛ, tyínúkú-á yah kwlɔ ǹ tyɩ́: À kyɩ ǹ gbã̀n yî pól yãm, à mó sé pɔ́ dibi yãmtãm kɔ̃. Képah sɔkɔ, wǎh kɩ yĩ̂nyõ tahrɩ wɛ, à mó pɔ pé sõ̀ tɛ̃. 
Tɛ́, dĩ́ mɛ̀-á képah noh sɛ̃́, ǹ yah yɩkɩ. À sɔkɔ yèkwɔrɩ sɔkɔ. Tahrɩ ǹgbɛ̃ sõ̀ mɛ ǹ tyɩ́, képah ye. 
Kǎh pɩ sɛ̃́, Yesu ǹ gbɔ̀plɔ yah kore vi, tɛ́ yo ǹ *tõ̀ sã́hpú tyɩ́: «Tahbɩ dénɔ́lɛ *Liyel tɔ̃́rɩ́ sɔkɔ, kè mɛ tah dɛ́!» 
Kénɛ wɛy ǹ tõ̀ sã́hpú gbã̀n du kègbɩlɛ. Ǹtɛ, Yesu tɔ̃ ǹ syɩ yo pé tyɩ́: «Ń wɛ̃́npì, Liyel tɔ̃́rɩ́ sɔkɔ dénɔ́ mɛ tah náhnáh. 
Yã̀hmɩ́kɩ̀ dénwìlnɔ́lɛ sèndi pwã̀h sɔkɔ, képah kɩ fàhfàh kal, tàh dénɔ́lɛ Liyel tɔ̃́rɩ́ sɔkɔ.» 
Képah tɛ̃̀ sɔkɔ, pé gbã̀n kah du. Pè wɛ̃nɛ n piki n yah pé tyɩ́ tyɩ́lɛ nɛ, ǹtɛ, nɛ̂ wɔ se kɩ mó pwah nɩ? 
Yesu mɛ pè yah, tɛ́ yo pé tyɩ́: «Névi tɛ̃̀ tyɩ́, kè náh fɛ̃ n pɩ n pi, tɛ́ Liyel wɔ tyɩ́, kè kɩ fɛ̃ pɩ. Ǹmɔ tɛ̃ wɛ tyi pól tyɩ́.» 
Képah sɔkɔ, Pyɛrɩ yo ǹ tyɩ́ nɛ, ǹtɛ, pépi-á tyi pól sah ǹmɔ sõ̀ tɛ̃ yĩnnɛ. 
Yesu mɛ nɛ: «Kègbɩ yõ, mé kélɛ n yo n dal mɔ yé tyɩ́ nɛ, névi nɛ̂-à fɛ̃ syi, ǹ kéte, ànɛ̂ ǹ yṹpyé, ànɛ̂ ǹ yṹ, ànɛ̂ ǹ sú, ànɛ̂ ǹ wɛ̃́npì, ànɛ̂ ǹ swɛ̃y sah ńmɔ yĩnnɛ, ànɛ̂ nɩ̀vɩ̀nɩ̀ wɛy sã́n yĩnnɛ, 
tɛ̃̀ kɩ sɛ̃́nknɔnɔ lèkwɛ̌kwɛ wɛ kèkõyṍ mɛ̀ sɔkɔ. À kɩ kétlo, ànɛ̂ yṹpyé, ànɛ̂ yṹbí, ànɛ̂ wɛ̃́npì, ànɛ̂ swɛ̃y wɛ. À kɩ fṍhnɔ́ tɔ wɛ, tɛ́ kɩ tyɛ́-ńkɛ̃̂ min wɛ ńsõ̂ kèkõyṍ sã́n sɔkɔ. 
Nónó-á mɛ núkúnúkú nɛ gbrolɛ, pé náhnáh kɩ pɔ n pɩ n pi nɛ tɛ́lɔ́lɛ. Tɛ́ nónó-á mɛ núkúnúkú nɛ tɛ́lɔ́lɛ, pé náhnáh kɩ pɔ n pɩ n pi nɛ gbrolɛ.» 
Yesu ànɛ̂ ǹ *tõ̀ sã́hpú sõ̀ mɛ wɛ̃̀kɩ̀ tɛ̃ n dɔkɔ n yuku Yerusalɛmɩ sɔkɔ. À sõ̀ mɛ n kɔ ǹ tõ̀ sã́hpú yahlɛ. Légbĩ́nɛ, pé fɔkɔ mɛ fɛ̃̀kɩ̀nɔ́lɛ. Tɛ́ nónó tɔ-á mɛ ǹ sõ̀ tɛ̃, tíkí sõ̀ mɛ pè tɛ̃. Képah sɔkɔ, à ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye kɛ̃kɩ pé yahlɛ, à kyɩ tir nɛ̂-á kɩ à yĩni n tɛ̃ n pi, ké yah yo pè syah. 
À nɛ: «Yé yah, á wɛ dɔkɔ n yuku Yerusalɛmɩ sɔkɔ yɔ̀. *Névi Pi kɩ dahbɩ *sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú gbõ̀. Pè kɩ à mɔ kũ kɔ̃, tɛ́ kɩ à dahbɩ Liyel yõ sɛ̃́-ńkɛ̃̂pú gbõ̀. 
Pépi kɩ kṍlɛ ǹnɛ n ko, tɛ́ kɩ n sĩn n mɔ ǹ yõ. Pè kɩ à ko yèfõhlɛ, tɛ́ kɩ à ko mɔ. Tɛ́, ké sèpĩ̀n tɔ̃́-ò sṍlɛ, à kɩ yuku.» 
Nsyakɩ ànɛ̂ Nsyɔ̃nɛ, Sebede wɛ̃́npìlɛ, pè nɔ de Yesu tnɔ̂, tɛ́ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, tir nɛ̂-á pé pɔ ní-ńsah ǹ tyɩ́, páh mɛ ké tyɩ́ nɛ, à kè pɩ pé tyɩ́. 
À mɛ pè piki yah nɛ, tir nɛ̂ se pè nɛ, pé pɩ pé tyɩ́ nɩ? 
Pè mó yo ǹ tyɩ́ nɛ, wàh kyɩ yĩni tɛ̃ ǹ yõ̀tɔ̃rɩ́ tyah sɔkɔ, à yɛ pépi sɔkɔ nɛ núkú tɛ̃ ǹ gbõ̀yõ̀ sõ̀, ní-ò mó tɛ̃ ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀. 
Képah sɔkɔ, à yo pé tyɩ́: «Kwâh nɛ̂-á yé n ni, yé náh kè pɩ̃. Yé se kɩ fɛ̃ wɔ ń wɔ wɔ wóh sɔkɔ, kɩ de wil yèvyãhrɩ sɔkɔ ń tyɩ́ kɔ̃lɛ?» 
Pè mɛ à syah nɛ, páh kɩ tɛ̃ wɛ. À mɛ yo pé tyɩ́: «Gbɩ ye, yé kɩ n wɔ n pi ń wɔ wɔ wóh tɛ̃̀ wɔ sɔkɔ, tɛ́ kɩ n de n wil n pi yèvyãhrɩ sɔkɔ ń tyɩ́ kɔ̃lɛ. 
Ǹtɛ, névi nɛ̂-á n tɛ̃ n pi ń gbõ̀yõ̀ ànɛ̂ ń gbɛ̃̀yɩ̃̀nɛ̀ sõ̀, képah náh n wil n pi ńmɔ tyɩ́. Kénɛ tɛ̃̀-ǹtã̀n kɩ pɔ névye kɔ̃ n pi, Liyel-á sè tɔ̃ sah nónó yĩnnɛ.» 
*Tõ̀ sã́hpú nɛ́pĩ̂ gbãm tɛ́lɔ́-á képah noh sɛ̃́, pé fɔkɔ vyãh Nsyakɩ ànɛ̂ Nsyɔ̃nɛ. 
Képah sɔkɔ, Yesu mɛ pé pól ye, tɛ́ yo pé tyɩ́: «Yé pɩ̃ nɛ, sétáh mɛ̀ yõ kwlo yõ̀tãm-á pé kwil tãnnɛ n tɛ̃ n sah pé dyɔ dyɔ tɛ̃̀nɛ, tɛ́ kwil nɛ́-ǹgbnɔ n mó pé tǎhkɩ̀lɛ n sah névye yõ. 
Ǹtɛ, yépi tyɩ́ náh n pɩ n pi sɛ̃́. Gbɛ̃̀ntɛ̃̀ náhlɛ, yépi sɔkɔ nɛ̂-à mɛ ké tyɩ́ nɛ, pé pɩ yõ̀tɛ̃̀nɛ, tɛ̃̀ ǹ gblɔ̌y pɩ yé tõ̀npɩlɛ. 
Tɛ́, yépi sɔkɔ nɛ̂-à mɛ kɩ tyɩ́ nɛ, pé yõ̀tɔ̃rɩ́ yah n kɔ̃, tɛ̃̀ ǹ gblɔ̌y pɩ yé pól yɔ̃̀lɛ. 
Tir yɔ́ náh, *Névi Pi náh pɔ nɛ, névye tõ̀lɛ n pɩ péwɔ tyɩ́. Ǹmɔ ye pɔ tõ̀ pɩ yĩnnɛ névye tyɩ́, tɛ́ kɩ ǹ min kɔ̃ névye náhnáh yõ dwe yĩnnɛ.» 
Képah náh, pè yĩni tɛ̃ Yeriko kwil. Yesu, ànɛ̂ ǹ *tõ̀ sã́hpú, ànɛ̂ tɔ́wû ǹgbɛ̃-á n wil kwil sɔkɔ gbĩ́ nɛ̂nɛ, yɩ̃́tũ̂ dĩ́ yɔ́ sõ̀ mɛ tɛ̃̀nɔ́lɛ wɛ̃̀gbalɩ sɔkɔ, mɛ n ni n syi. Time pídĩ́ ye sõ̀ ǹnɛ, pè n ye Bartimelɛ. 
Wǎh noh nɛ, Nasarɛti Yesu-á n sɔkɔ, à tyah n de n syi nɛ, Yesu, *Davidi pídĩ́, à pé yãm yah. 
Képah sɔkɔ, náhnáh sõ̀ mɛ ǹ vyãhlɛ n tũ. Ǹtɛ, à tɔ̃ n kah n de n syi nɛ, Davidi pídĩ́, à pé yãm yah. 
Képah tɛ̃̀nɛ, Yesu yĩn, tɛ́ nɛ, pè à ye yah. Pè mɛ yɩ̃́tũ̂ dĩ́ ye yah, tɛ́ yo ǹ tyɩ́ nɛ, à ǹ nɩ vɩ, à mó yuku, wǎh mɛ ǹnɛ n ye. 
Dĩ́-á képah tɛ̃̀ noh, à ǹ fwɔ̀mɔ̀-ǹgbɛ̃ milki, tɛ́ yɔkɔ syɩkɩ ǹ gbyẽ̂h yõ, à yuku Yesulɛ n yɩ̃kɩ. 
Yesu mɛ à piki yah: «Kû mɛ á nɛ, mé pɩ á tyɩ́?» Yɩ̃́tũ̂ dĩ́ mɛ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, à kè pɩ, pé tɔ̃ nɛ n yah. 
À mɛ yo ǹ tyɩ́: «Nɛ n yuku, ńmɔ yõ sɛ̃́nɔ́ ye ǹ dɛ mɔ.» Nɛ́npɔ́ swɔ, ǹ yɩ̃́ yɩkɩ mɔ, à mɛ Yesu sõ̀ tɛ̃.
