Tyípɩ́nɩ̀.
10.
Dĩ́ yɔ́ lésõ mɛ Sesarɩ kwil pè n ye Kɔrnɛylɛ. Sràsyíbí yõ̀tɛ̃̀ yɔ́ ye sõ̀ ǹnɛ, Wrome tãn sràsyíbí tĩ̀nkì sɔkɔ, nɛ̂-á pè n ye Yitalɩ gbɛ̃̀nyah sràsyíbí tĩ̀nkìlɛ. 
À sõ̀ mɛ Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ plɛ, mɛ ǹ yah tíkílɛ n pɩ, tɛ́ ǹmɔ ànɛ̂ ǹ gbô tãn pól mɛ ǹnɛ n ni. À sõ̀ mɛ Nsyifunɔ yãm tãnnɛ n yohnɩ n mɔ náhnáh, tɛ́ mɛ Liyellɛ n ni gbĩ́mɛgbĩ́. 
Képah sɔkɔ, sõ yɔ́ gbèkãm gbĩ́ kénkɔ̃ sɔkɔ, à wɛ̃́kɩ́nɔ́ yɔ́ wɛ. À Liyel yĩ̂nyõ tõ̀npɩ yɔ́ wɛ pnɛ́yy à de pɔ ǹ tĩ̀nnɛ, à pɔ à ye ǹ yĩnnɛ nɛ: «Kɔrnɛy!» 
Ǹtɛ Kɔrnɛy sõ̀ mɛ yĩ̂nyõ tõ̀npɩlɛ n yah díkídíkí tíkítìkì sɔkɔ, tɛ́ à piki yah nɛ, kwâh nɛ̂ se nɩ, pé yõ̀tɛ̃̀? Yĩ̂nyõ tõ̀npɩ mɛ yo ǹ tyɩ́ nɛ: «Wáh nírí nónónɛ n pɩ, ànɛ̂ wáh yòhnɩ̀nmɔnɔ nónónɛ n pɩ yãm tãn tyɩ́, Liyel yĩ́ mɛ sé yõ. Ǹ dyɔ mɛ n kwnɛ á tyɩ́lɛ. 
Núkúnúkú tɛ̃̀nɛ, nósyɔ́ wilki n tɛkɩ n mɔ Nsyope kwil, pè kyɩ Simo yɔ́ tɛ̃ n pɔ, nɛ̂-á pè n ye Pyɛrɩlɛ. 
À mɛ twáhntɛ̃̀nɔ́lɛ sénpɩ dĩ́ yɔ́ tyɩ́, nɛ̂ tɔ yĩn-á Simolɛ. Ǹ gbô mɛ pnɛ yɔ gbɔ̀pɔ.» 
Képah sɔkɔ, yĩ̂nyõ tõ̀npɩ-á wɛlɔ nɔ́pi yo tɛ Kɔrnɛy tyɩ́, tɛ́ yal ǹ tnɔ̂, Kɔrnɛy núkú ǹ tõ̀npɩbɩ díbí nɛ́pĩ̂ nímí ye ànɛ̂ sràsyí yɔ́lɛ, nɛ̂-á sõ̀ mɛ tɛ̃̀nɔ́lɛ ǹ tnɔ̂ di mónɔ́lɛ, tɛ́ mɛ Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ plɛ. 
Tɛ́ mɛ tyi nɔ́pi pól yãh pé tyɩ́, tɛ́ pè tɛkɩ mɔ Nsyope kwil. 
Ké dĩndallɛ, pè sõ̀ mɛ wɛ̃̀kɩ̀ yõ sɛ̃́, tɛ́ pǎh pópó n de Nsyope kwil sɔkɔ gbĩ́ nɛ̂nɛ, gbĩ sõ̀ mɛ yõ̀tah sírr kénkɔ̃ sɔkɔ, tɛ́ Pyɛrɩ mɛ dɔkɔ gbɛ̃́ntĩ̂n yõ, mɛ nírílɛ n pɩ. 
Képah sɔkɔ, kwɛkɩ pɔ à tɛ̃; à sõ̀ mɛ ké tyɩ́ nɛ, pé yõke yõ. Tɛ́ pǎh mɛ yõkelɛ n pɩ ǹ tyɩ́ gbĩ́ nɛ̂nɛ, Liyel wɛ̃́kɩ́nɔ́ yɔ́lɛ à wɛ̃kɩ. 
À yĩ̂nyõ wɛ yɩ́kɩ́nmɔnɔlɛ, tɛ́ kwâh yɔ́ mɛ wil ké sɔkɔ, fwɔ̀pĩ̀n-ǹgbɛ̃ yɔ́ sɔ̃́, ké vyɛ̃ynɔ ńyã́hlɛ tɛ̃́nɔ́lɛ, tɛ́ mɛ n tiki n pi sétáh. 
Tɛ́ gbyẽ̂h ǹyẽ̂h yî duke pól sõ̀ mɛ ké sɔkɔ, ànɛ̂ yî nónó-à n til, à tahlɩ kèpye duke pól tyɩ́. 
Képah sɔkɔ, Pyɛrɩ yĩ́npɔ̃́ yɔ́ noh kè mɛ n yo ǹ tyɩ́ nɛ: «Pyɛrɩ, yuku n ko sé sɔkɔ ǹ kɔ.» 
Ǹtɛ Pyɛrɩ mɛ Tɛ̃̀ ǹgbɛ̃ syah nɛ, páh pɩ sɔ̃́ pilɛ, pé náh dêl kwâh yɔ́ ńtɛ̃ yõ yah, képah náh pɩ, kwâh yɔ́ ńtɛ̃nɛ, nɛ̂-á névilɛ n silki Liyel yah. Képah-á, pé náh kè pɩ n pi póllɛ. 
Képah tɛ̃̀nɛ, yĩ́npɔ̃́ mɛ̀ tɔ̃ ké syɩ yo à noh kɛ ní-òlɛ nɛ ńkɛ̃́: «Liyel-á kwâh nɛ̂ wɛ̃kɩ kwáhpépɛ̌ynɛ, áwɔ tyɩ́ náh ké wɛ̃́kɩ́nɔ́lɛ nɛ, kǎh mɛ névilɛ n silki Liyel yah.» 
Képah tɛ̃̀ pɩ sɛ̃́ kɛ tɔ̃́nɔ́, tɛ́ Liyel núkú kénɛ kwâh yɔ̃ dɔkɔ sɔkɔ yĩ̂nyõ. 
Kǎh pɩ sɛ̃́, Pyɛrɩ lékã́m sõ̀ mɛ fɛ̃kɩ, à mɛ ǹ gblɔ̌ynɛ n piki n yah wɛ̃́kɩ́nɔ́ nɛ̂-á à wɛ, ké yõ-á kɩ fɛ̃ n pɩ n pi tir nɛ̂nɛ. À mɛ képah sɔkɔ, tɛ́ Kɔrnɛy tɛ̃ntõ̀ névye mó mɛ sénpɩ dĩ́ Simo gbô tyɩ́ piki yah kɔ̃ pɔ wɛ, mɛ yĩn tɛ̃ ké yah, 
tɛ́ mɛ n piki n yah nɛ, Simo nɛ̂-á pè n ye Pyɛrɩlɛ, à se mɛ twáhntɛ̃̀nɔ́lɛ ásõ̂ sɔkɔ nɩ? 
Ǹtɛ Pyɛrɩ-á sõ̀ mɛ yɔ̃ sɔkɔ ǹ fɔkɔlɛ n sõ wɛ̃́kɩ́nɔ́ mɛ̀ tyɩ́ sɔkɔ sɛ̃́, Liyel Mirki mɛ yo ǹ tyɩ́ nɛ à yah, díbí nɛ́pĩ̂ tɔ̃́mɩ́-á mɛ ǹnɛ n yah n kɔ̃. 
Péwɔ-á tɛkɩ mɔ pélɛ ǹ tyɩ́. Képah-á, à yuku n tiki n sɔkɔ pélɛ, à káh tɔ̀tɔ̀ pɩ. 
Képah sɔkɔ, Pyɛrɩ tiki kyɩ no nɔ́pi tnɔ̂, tɛ́ yo pé tyɩ́ nɛ, nɛ̂-á pè n yah n kɔ̃, ǹmɔ-á péwɔlɛ. Ńkɛ̃́, pè kèyǎh kɔ̃. 
Pè mɛ yo ǹ tyɩ́ nɛ, sràsyíbí yõ̀tɛ̃̀ dĩ́ Kɔrnɛy-á tõ pélɛ ǹ tyɩ́. Nɛ́gbɩ́-á ǹnɛ, wǎh Liyel yah tíkílɛ n pɩ, ànɛ̂ Nsyifunɔ pól-á ǹ yĩ́ngbɩ́lɛ n yo. Yĩ̂nyõ tõ̀npɩ yɔ́-á kélɛ à wɛ̃kɩ nɛ, wǎh yɩ̃nɛ à pɔ ǹ tɛ̃ n kyɩ ǹ gbô sɔkɔ, tyi nónó-á mɛ á tyɩ́ yo yo tɛ̃̀nɛ, á sè yo à syah. 
Képah sɔkɔ, Pyɛrɩ mɛ pé kɔ̃ pè de gbô sɔkɔ, à twah ǹ tyɩ́ à lékã̂h pɩ. Tɛ́ ké dĩndallɛ, à mɛ pé sõ̀ tɛ̃ sɔkɔ. Tɛ́ Yesu yõ sɛ̃́pú yísyɔ́ tɔ sõ̀ sɔkɔ pélɛ, nónó-á n wil Nsyope kwil. 
Pé sɔkɔ sõ dĩndallɛ, pè kyɩ de Sesarɩ kwil; tɛ́ kyɩ nɛ yah Kɔrnɛy mɛ ǹ gbô tãn ànɛ̂ ǹ kódíbí kègblɔ ye tuke, mɛ pɔ pè syɩkɩ tɛ̃. 
Pyɛrɩ-á pɔ n de gbĩ́ nɛ̂nɛ, Kɔrnɛy kyɩ à yohnɩ, tɛ́ kɔ kwĩnki ǹ yah, à à gbilki. 
Ǹtɛ Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ à yuku, tɛ́ à yilki sah, tɛ́ nɛ névi ó-á péwɔ tɔlɛ dò. 
Pè mɛ képah yónɔ́ sɔkɔ wɛ̃ tyɩ́, tɛ́ de yuku gbô à kyɩ no nɔ́pi wɛ tùkèntɛ̃̀nɔ́lɛ pípí. 
Pyɛrɩ mɛ pè syah nɛ, pǎh pɩ̃ nɛ, dêl-á kélɛ Nsyifunɔ tyɩ́, pè náh n pɩ n soh wɛ̃ tyɩ́ kwlókáhkɩ́ névyelɛ, ànɛ̂ pè náh n de pé gbên sɔkɔ kɛ̀. Ǹtɛ Liyel-á kélɛ péwɔ tɛ̃̀ wɛ̃kɩ nɛ, pé náh yɩ̃nɛ pé ǹnɩ̂ yɔ́ ńtɛ̃nɛ n yah nɛ, wǎh mɛ sìlkìnɔ́lɛ Liyel yah, képah náh pɩ, nɛ à náh yɩ̃nɛ pɩ́nsòhnɔ́lɛ névye tyɩ́. 
Képah-á, pé kɔ̃ pǎh pé ye pé pɔ yékã̂b pɩ́-ńkɛ̃̂nɛ. Ńkɛ̃́, pè se kɩ fɛ̃ pé yénɔ́ yõ yo pé syah nɩ? 
Képah sɔkɔ, Kɔrnɛy mɛ yo ǹ tyɩ́ nɛ, ké sèpĩ̀n ǹyẽ̂h-á yɔ̀, gbèkãm gbĩ́ mɛ̀ ó kénkɔ̃ sɔkɔ, páh lésõ mɛ nírílɛ n pɩ pé wɔ̀kɔ̀ sɔkɔ. Tɛ́ dǒ, pé núkú dĩ́ yɔ́ wɛ yĩ́nntɛ̃̀nɔ́lɛ pé yah tĩ̀nnɛ, wil flɔlɛ pěrr ǹ tyɩ́, 
tɛ́ mɛ pé ye pé yĩnnɛ nɛ, Kɔrnɛy, Liyel-á pé nírí tɛ̃; ǹ dyɔ-á mɛ n kwnɛ pé pèpɔrɩ pɩ́nɔ́ tyɩ́lɛ. 
Képah-á, pé névye wilki n tɛkɩ n mɔ Nsyope kwil, pè kyɩ Simo ye n pɔ ásõ̂, nɛ̂-á pè n ye Pyɛrɩlɛ. Wâh mɛ twáhntɛ̃̀nɔ́lɛ sénpɩ dĩ́ Simo gbô sɔkɔ, pnɛ yɔ gbɔ̀pɔ. 
Kǎh pɩ sɛ̃́, pé núkú névye tɛkɩ mɔ ǹ tyɩ́ nɛ, pè kyɩ à ye n pɔ. Ǹ tɔ mó tɔ̃ pɩ plɛ tɛ́ fɛ̃ syi pɔ. Núkúnúkú, pé pól-á ásõ̂ Liyel yah sɔkɔ, ǹ wɛlɔ noh yĩnnɛ, nónó pól-á Tɛ̃̀ ǹgbɛ̃ dahbɩ ǹ gbõ̀, nɛ à yo pé syah. 
Képah sɔkɔ, Pyɛrɩ mɛ wɛy tɔkɔ tɛ́ nɛ: «Núkúnúkú, mé pɩ̃ kègbɩ yõ nɛ, Liyel náh ǹnɩ̂ yɔ́ ńtɛ̃nɛ n wah. 
Névi nɛ̂-à fyɔ̀ ǹ yah tíkílɛ n pɩ, tɛ́ mɛ n kɔ gbɩ wɛ̃̀kɩ̀ yõ, tɛ̃̀-à sɔ̃́ mɛ yìkì nɛ̂ nɛynɛ, ǹ tyɩ́ n nɔ Liyel tyɩ́. 
Ǹmɔ Liyel ye ǹ wɛy tɛkɩ mɔ Yisrayel tãn tyɩ́, à Nɩ̀vɩ̀nɩ̀ tir yo pé tyɩ́ nɛ, yèvɩnɩ-á n wɛ Liyel tyɩ́ Yesu Krista gbõ̀ yõ, ǹmɔ nɛ̂-á no pól Tɛ̃̀ ǹgbɛ̃nɛ. 
Nsyɔ̃-á kwní kɔ̃ tɛ ni sɔkɔ wèlnwìlkìnɔ́ tyɩ́ sɔkɔ, képah náh, nɛ̂-á pɩ Kalile gbɛ̃̀nyah ǹgbò sɔkɔ ànɛ̂ Nsyude gbɛ̃̀nyah pól sɔkɔ, yé se mó kè pɩ̃ tɛ́! 
Tɛ́ Liyel sõ̀ mɛ Nasarɛtɩ kwil Yesulɛ. À ǹ Mirki ànɛ̂ tɛ̃́nwɛnɔlɛ à kɔ̃. Képah ye, à mɛ n kɔ n kɔ̃ tã̀ntã̀n pól sɔkɔ, à pèpɔrɩlɛ n pɩ, tɛ́ nónó pól-á sõ̀ mɛ sétã̀n yõ̀tɛ̃̀ yɔ̃rɩ sɔkɔ, à sõ̀ mɛ pélɛ n wilki ké sɔkɔ. 
Tɛ́ wǎh tyi nónó pól pɩ Yerusalɛmɩ kwil ànɛ̂ Nsyifunɔ kwlo tɛ́lɔ́ sɔkɔ, á pól ye sè wɛ yĩ́nɛ. Ǹtɛ pè mɛ à tɛ̃ syɩ dahnɩ yõ, à à ko mɔ. 
Tɛ́ ké sèpĩ̀n tɔ̃́-ò sõlɛ, Liyel mɛ ǹ lékó yilki mɔ, tɛ́ à kɔ̃ à ǹ gblɔ̌ynɛ no wɛ̃kɩ, 
ǹtɛ à náh ǹ gblɔ̌ynɛ no pól wɛ̃kɩ. À ǹ gblɔ̌ynɛ ápi ó wɛ̃kɩ, ápi nónó-á lésõ Liyel yah tɔkɔ pɩ Yesu lékó yílkínmɔnɔ wɛ́pú ànɛ̂ sé yã́hpúlɛ. Ápi lésõ yõke yõ, tɛ́ wɔ wɔ kwâh wɔ ǹnɛ wɛ̃ tyɩ́, ǹ lékó yílkínmɔntɛnɔ náh. 
Képah sɔkɔ, à mɛ á tõ nɛ: ‹Yé kyɩ Liyel wɛynɛ n yo n kɔ̃ no tyɩ́. Yé kélɛ pè wɛ̃kɩ nɛ, ńmɔ-á Liyel sah vye ànɛ̂ lékyɩ̂ tùkè-òlɛ.› 
Ǹmɔ Yesu tyɩ́ ye tɔ̃ Liyel tɛ̃ntõ̀ syínyopu pól yo nɛ, névi nɛ̂-à sɛ̃ ǹmɔ yõ, tɛ̃̀ tyípêl-á kɩ yɔ̃ mɔ à kɔ̃ ǹmɔ yĩn yõ.» 
Képah sɔkɔ, Pyɛrɩ sõ̀ mɛ wɛ́kɩ́nɔ́ sɔkɔ sɛ̃́, tɛ́ Liyel Mirki tiki ǹ wɛy nòhnɩ̀pu pól yõ. 
Liyel-á Mirki nɛ̂ kɔ̃, kǎh tɔ̃ mɔ névye yõ nónó náh pɩ Nsyifunɔlɛ, pè kɔ̃ pè wɛ́nkáhkɩ́nɔ́lɛ n wɛkɩ nónó-á pè náh sè sãh, tɛ́ Liyellɛ n gbilki sɛ̃́, képah pɩ gbɛ̃̀ndùnìnɛ Yesu yõ sɛ̃́pú Nsyifunɔ tyɩ́, pépi nónó-á sõ̀ pɔ ànɛ̂ Pyɛrɩlɛ. 
Képah sɔkɔ, Pyɛrɩ nɛ: «Névye nónónɛ Liyel Mirki wɛ tɛ ápi tyɩ́ kɔ̃lɛ, á kɩ fɛ̃ yahle sɔ̃́ nɛ, pè káh n wel n wilki ni sɔkɔ?» 
Tɛ́ mɛ pè kɔ̃ pè Kɔrnɛypi wel wilki ni sɔkɔ Yesu Krista yĩn yõ. Képah sɔkɔ, pépi mó Pyɛrɩ ni nɛ, à wũ sèpĩ̀n yísyɔ́ pɩ pélɛ.
