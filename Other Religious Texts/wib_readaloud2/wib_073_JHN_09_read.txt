Nsyɔ̃.
9.
Sõ yɔ́lɛ, Yesu sõ̀ mɛ n kɔ n sɔkɔ tɛ́ dĩ́ yɔ́ wɛ, nɛ̂-á sõ̀ mɛ yɩ̃́tũ̂nɛ ǹ dubi gbĩ́ dùkùlɛ. 
Képah sɔkɔ, Yesu tõ̀ sã́hpú à piki yah nɛ, pé kwɔ́-ò dĩ́, sɔ̃́ se pɩ tɛ́ dĩ́ mɛ̀ dubi yɩ̃́tũ̂nɛ nɩ? Ǹmɔ tyípékè pɩ́nɔ́ se, tĩ̂ ǹ dúbípú tyɩ́ se? 
À mɛ yo pé tyɩ́ nɛ: «Ǹmɔ, képah náh pɩ ǹ dúbípú tyípékè pɩ́nɔ́ náh à kɔ̃ à dubi yɩ̃́tũ̂nɛ dɛ́! Kè ye pɩ Liyel tyípɩ́nɩ̀ wɛ̃kɩ yĩnnɛ ǹ sɔkɔ. 
Nɛ̂-á tɛkɩ mɔ ńmɔlɛ, tyah kyòo sɔkɔ ye á yɩ̃nɛ à tõ̀lɛ n pɩ ǹ tyɩ́. Ńkɛ̃́ lékã̂h mɛ n pi, nɛ̂ sɔkɔ-á nɛy náh fɛ̃ tõ̀ pɩ n pi. 
Máh fyɔ̀ mɛ kèkõyṍ sɔkɔ, ńmɔ ye kèkõyṍ kyòolɛ ń mɛ gbĩ́ pól sɔkɔ.» 
Wǎh képah yo tɛ, à sĩn mɔ sétáh tɛ́ kè fãh à kele yɩ̃́tũ̂ dĩ́ yɩ̃́ yah. 
Tɛ́ mɛ yo ǹ tyɩ́ nɛ à kyɩ ǹ yah wu pɛ́kɩ̀ tũ̀nsàhnɔ́ yɔ́ ni sɔkɔ, nɛ̂-á pè n ye Silowelɛ. Silowe kɔ́r ye Tɛ̃ntõ̀ névilɛ. Yɩ̃́tũ̂ dĩ́ mɛ sɔkɔ à kyɩ ǹ yah wu nɛ́npɔ́. Wǎh ǹ syɩ n pi gbĩ́ nɛ̂nɛ, à sõ̀ mɛ n yah. 
Képah sɔkɔ, ǹ tnɔ̂ tɛ̃̀pu ànɛ̂ nónó-á ńkɛ̃́nɛ ǹnɛ n wɛ à n nɛ mɛ n ni n syi, pǎh à wɛ pè mɛ yo tũ nɛ, dĩ́ mɛ̀ kó ńkɛ̃́nɛ n kɔ̃ n tɛ̃ nɛ n ni n syi yɔ̀ nɩ? 
Túkù nɛ ǹmɔ-á, túkù mó nɛ ǹmɔ náh, yã̀hnkõ-á à tɔ. Pǎh mɛ képah sɔkɔ, dĩ́ mɛ yo pé tyɩ́ nɛ péwɔ-á tɔ́tɔ́. 
Wǎh yo sɛ̃́, pè mɛ à piki yah nɛ, ǹ yɩ̃́ se pɩ sɔ̃́ tɛ́ yɩkɩ mɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ, dĩ́ nɛ̂-á pè n ye Yesulɛ, ǹmɔ-á fã̀hkɩ̀ pɩ, tɛ́ kè kele pé yɩ̃́ yah, tɛ́ nɛ pé kyɩ pé yah wu Silowe yɔ sɔkɔ. Pé mɛ sɔkɔ kyɩ pé yah wu nɛ́npɔ́, pé mɛ fɛ̃ n yah. 
Pè mɛ à syah nɛ, kénɛ dĩ́ se mɛ ńyãh nɩ? À mɛ nɛ pé náh pɩ̃. 
Képah sɔkɔ, yɩ̃́tũ̂ dĩ́ nɛ̂ yɩ̃́-á yɩkɩ mɔ, pè à tɛ̃ pɔ Farisi wɛ̃̀kɩ̀ kɔ́pú tyɩ́. 
Tɛ́ mɩ mɩ yòh sõ ye mó tɔ̃ lésõ Yesu fã̀hkɩ̀ pɩ à dĩ́ yɩ̃́ yɩkɩ mɔ. 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú dĩ́ piki yah nɛ, sɔ̃́ se kè pɩ tɛ́ à fɛ̃ n yah núkúnúkú nɩ? À mɛ yo pé tyɩ́ nɛ, fã̀hkɩ̀-á à pɩ kele pé yɩ̃́ yah, tɛ́ pé mɛ kyɩ pé yah wu, pé mɛ fɛ̃ n yah. 
Képah tɛ̃̀nɛ, Farisi wɛ̃̀kɩ̀ kɔ́pú túkù sõ̀ mɛ n yo nɛ ńkɛ̃́, dĩ́ mɛ̀ náh fyɔ̀ mɛ mɩ mɩ yòh tɛ̃̀ dêllɛ n tɛ̃, képah-á n wɛ̃kɩ nɛ à náh wil Liyel tyɩ́. Ǹtɛ pé túkù mó mɛ n yo nɛ, tyípêl pɩ́-ò se kɩ fɛ̃ gbǐl nɔ́pi dùkù pɩ sɔ̃́ nɩ? Kǎh pɩ sɛ̃́, pé sõnɔ pɩ sé tyɩ́ tyɩ́lɛ. 
Pè mɛ tɔ̃ pé syɩ dĩ́ mɛ̀ syah nɛ, nɛ̂-á yɩkɩ mɔ ǹ yɩ̃́lɛ, ǹmɔ se kɩ yo sɔ̃́ tɛ̃̀ tyɩ́ sɔkɔ nɩ? Dĩ́ mɛ nɛ, Liyel tɛ̃ntõ̀ syínyo-ò-á ǹnɛ. 
Ǹtɛ Nsyifunɔ yõ̀tãm náh sõ̀ mɛ ké tyɩ́ nɛ pé sɛ̃ ké yõ nɛ, yɩ̃́tũ̂-á sõ̀ ǹnɛ, tɛ́ núkúnúkú à mɛ n yah. Kǎh pɩ sɛ̃́, pè ǹ dúbípú ye pɔ, 
à pɔ pè piki yah nɛ, pépi pídĩ́ se dĩ́ mɛ̀nɛ nɩ? À se dubi yɩ̃́tũ̂nɛ nɩ? Ǹtɛ à se pɩ sɔ̃́ tɛ́ fɛ̃ n yah núkúnúkú nɩ? 
Ǹ dúbípú mɛ yo pé tyɩ́ nɛ, páh pɩ̃ nɛ pépi pídĩ́-á ǹnɛ; wǎh dubi yɩ̃́tũ̂nɛ. 
Ǹtɛ tir nɛ̂ tɛ̃̀ wɔ-á pɩ tɛ́ à fɛ̃ n yah núkúnúkú, pépi náh képah wɔ pɩ̃, pé náh se mó tɔ̃ pɩ̃ nɛ̂-á yɩkɩ mɔ ǹ yɩ̃́lɛ. Ńkɛ̃́ pè à piki yah, wɛ̃́npir náh tɔ̃ ǹnɛ; wǎh kɩ fɛ̃ ké yah yo ǹ gblɔ̌y. 
Nsyifunɔ yõ̀tãm tiki sõ̀ mɛ pè tɛ̃, képah ye pè kɔ̃ pè n wɛkɩ sɛ̃́. Nsyifunɔ yõ̀tãm sõ̀ mɛ ké yo sah tɛ nɛ, névi nɛ̂-à kɩ nɛ Liyel nɛy yáhntɔkɔnɔ-á Yesulɛ, tɛ̃̀-á kɩ yĩni wilki pépi wɛ̃̀kɩ̀ sɔkɔ. 
Képah ye, ǹ dúbípú kɔ̃ pè nɛ, wɛ̃́npir náh tɔ̃ ǹnɛ, pè à piki yah. 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú tɔ̃ dĩ́ mɛ̀ ye yah kɛ ní-òlɛ nɛ̂-á lésõ yɩ̃́tũ̂nɛ, tɛ́ yo ǹ tyɩ́ nɛ, ké Liyel wɛ̃̀kɩ̀ tɛ̃̀ yõ, à gbɩ yo pé syah. Pépi-á pɩ̃ nɛ ǹ yɩ̃́ yɩ́kɩ́nmɔ-ò dĩ́ mɛ̀nɛ, tyípêl pɩ́-ò-á ǹnɛ. 
Dĩ́ mɛ nɛ, pé náh pɩ̃ tyɩ́pêl pɩ́-ò-à kɩ nɛ̂ ǹnɛ dɛ́! Tyínúkú ó-á péwɔ n pnɛ: Yɩ̃́tũ̂-á sõ̀ pélɛ, ǹtɛ núkúnúkú páh mɛ n yah. 
Pè mɛ à piki yah nɛ, tir nɛ̂ se Yesu pɩ ǹ tyɩ́ nɩ? Dùkù nɛ̂ se à yɩkɩ mɔ ǹ yɩ̃́lɛ nɩ? 
Dĩ́ mɛ nɛ, páh sõ̀ kè yo tɛ pé tyɩ́, tɛ́ pè náh pé níkí sah kè noh. Ǹtɛ sɔ̃́ se tɛ́ pè tɔ̃ mɛ ké tyɩ́ nɛ, pé kè yo pé noh kɛ ní-òlɛ nɩ? Pépi tɔ́ se pyě mɛ n yah n kɔ̃ nɛ, pé pɩ ǹ tõ̀ sã́hpúlɛ nɩ? 
Képah tɛ̃̀ sɔkɔ, pè tyah ǹnɛ n gbyɔ, tɛ́ nɛ ńkɛ̃́, ǹmɔ-á kénɛ dĩ́ tõ̀ sã́h-òlɛ dɛ́! Pépi tɛ̃̀ wɔlɛ, Moyisi tõ̀ sã́hpú-á pépilɛ. 
Páh pɩ̃ nɛ Liyel-á yo mɔ Moyisi tyɩ́. Ǹtɛ dĩ́ mɛ̀nɛ, pé se ǹ wìl-ǹsah ńtɛ̃nɛ n pnɛ nɩ? 
Pǎh yo sɛ̃́, dĩ́ mɛ yo pé tyɩ́ nɛ, é! Tɛ́ képah-á pyě pɩ gbɛ̃̀ndùnìnɛ yɔ̀ sɔ̃́. Pépi náh ǹ wìl-ǹsah pɩ̃ pè yo. Ǹtɛ wǎh péwɔ yɩ̃́ yɩkɩ mɔ dɛ́! 
Pé pól-á pɩ̃ nɛ, Liyel náh tyípêl pɩ́-ò nírílɛ n tɛ̃. Ǹtɛ névi nɛ̂-à pɩ nɛ́gbɩ́lɛ, tɛ́ Liyel dyɔ yĩ́ĩ́ tyilɛ n pɩ, ǎ tɛ̃̀ wɔ nírílɛ n tɛ̃. 
Ǹtɛ kèkõyṍ-á mɛ sɔ̃́, pépi náh kè noh yah nɛ, ǹnɩ̂-á yɩ̃́tũ̂ yɔ́ yɩ̃́ yɩkɩ mɔ, nɛ̂-á dubi yɩ̃́tũ̂nɛ dɛ́! 
Dĩ́ mɛ̀-à sõ̀ mɛ wìl-ǹkɛ̃̂nɛ Liyel tyɩ́, à náh náhkɩ fɛ̃ tir yɔ́ ńtɛ̃ pɩ n pi dò! 
Képah tɛ̃̀nɛ, pè à syah nɛ, ǹmɔlɛ dubi ǹ tɔ̂ɔ́ tyípékè ó sɔkɔ, tɛ́ ǹmɔ kɩ pɔ nɛ páh pépilɛ n kwɔ nɩ? Tɛ́ núkú à yĩni wilki kégbɔ́. 
Képah sɔkɔ, Yesu-á kè noh nɛ pǎh dĩ́ yĩni, à kyɩ à yohnɩ tɛ́ à piki yah nɛ: «Á se sɛ̃ Névi Pídĩ́ yõ?» 
Dĩ́ mɛ nɛ, pé yõ̀tɛ̃̀, nɛ̂ se Névi Pídĩ́ mɛ̀nɛ nɩ? À ǹ tyɩ́ yo pé syah, pé kɔ̃ pé wɛ sɛ̃ ǹ yõ. 
Képah tɛ̃̀nɛ, Yesu mɛ yo ǹ tyɩ́ nɛ: «Á ye à wɛ tɛ. Ǹmɔ ye n yãh álɛ núkúnúkú yɔ̀.» 
Dĩ́-á képah noh sɛ̃́, à nɛ páh sɛ̃ ké yõ, Tɛ̃̀ ǹgbɛ̃. Tɛ́ mɛ gbètukunɔ di Yesu yah, à à gbilki. 
Képah sɔkɔ, Yesu nɛ: «Gbɩ wah yĩn ye ńmɔ pɔ kèkõyṍ mɛ̀ sɔkɔ; nɛ yɩ̃́tũbi yɩ̃́ yɩkɩ mɔ, tɛ́ nónó-á mɛ n yah pépi pɩ yɩ̃́tũbilɛ.» 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú nónó-á sõ̀ mɛ ànɛ̂ Yesulɛ, pépi-á wɛlɔ nɔ́pi noh, pè Yesu piki yah nɛ, ǹtɛ yɩ̃́tũbi se pyě pépi tɔlɛ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ: «Yâh sõ̀ mɛ yɩ̃́tũbilɛ, tyípékè náh náhkɩ n mɛ n pi yé yõ. Ǹtɛ núkúnúkú yé nɛ yáh mɛ n yah, képah ye yé tyípékè mɛ yé yõ.»
