Nsyɔ̃-á Nɩ̀vɩ̀nɩ̀ tir nɛ̂ wãrkɩ Yesu tyɩ́ sɔkɔ.
1.
Tyi pól tyah gbĩ́nɛ, nɛ̂-á wɛynɛ, à lésõ mɛ. À lésõ mɛ ànɛ̂ Liyellɛ. Liyel ye ǹnɛ. 
Ǹmɔ ye lésõ mɛ ànɛ̂ Liyellɛ tyah gbĩ́nɛ. 
Yíyìn pól mɔ ǹmɔ gbõ̀ yõ sɔkɔ. Yî nónó pól-á mɔ, sé kèyǎh ńtɛ̃ náh mɔ nɛ́ ǹmɔ náh. 
Ǹmɔ sɔkɔ ye min n wɛ. Kénɛ min ye kyòolɛ névye tyɩ́. 
Kè mɛ kõ tɛ̃ kyòo fɛ̃́ntũ̂ sɔkɔ, tɛ́ fɛ̃́ntũ̂ náh tɛ̃ wɛ ké yah kõ. 
Képah sɔkɔ, Liyel pɔ dĩ́ yɔ́ tõ, nɛ̂ yĩn-á Nsyɔ̃nɛ. 
Ǹmɔ ye pɔ kyòo mɛ̀ tyɩ́ yo wah névye tyɩ́, nɛ pé pól wɛ sɛ̃ ké yõ sɔkɔ ǹmɔ gbõ̀ yõ. 
Ǹmɔ náh sõ̀ kyòo mɛ̀nɛ, ǹtɛ ké tyɩ́ yónwàh-ǹsah ye sõ̀ à pɔ. 
Wɛy mɛ̀nɛ, kyòo kègbɩ ye sõ̀ kélɛ, nɛ̂-á pɔ kèkõyṍ sɔkɔ, à pɔ pɩ kyòolɛ no pól tyɩ́. 
Tɛ́ nɛ̂-á wɛy mɛ̀nɛ, à sõ̀ mɛ kèkõyṍ sɔkɔ. Kèkõyṍ mɔ ǹmɔ gbõ̀ yõ sɔkɔ, ǹtɛ képah fyé yõ, kèkõyṍ névye náh tɛ̃ wɛ à yah pɩ̃. 
Ǹ yɛ̃́ngbɛ̃́ tyah sɔkɔ ye à pɔ, tɛ́ ǹ no náh à tɛ̃ plɛ. 
Ǹtɛ nónó tɛ̃̀ wɔ-á à tɛ̃ plɛ tɛ́ sɛ̃ ǹ yõ, à tɛ̃́nwɛnɔlɛ pépi kɔ̃ pè pɩ Liyel wɛ̃́npìlɛ. 
Ǹtɛ tṍ sɔkɔ dubi dùkù, képah náh pɩ wil gbɛ̃́nsê dékè, képah náh pɩ névi dyɔ yĩ́ĩ́ tɛ̃̀ sɔkɔ náh pè pɩ Liyel wɛ̃́npìlɛ dɛ́! Liyel gbõ̀ yõ sɔkɔ ye pè wɛ. 
Wɛy ye vi mɔ pɩ névilɛ à pɔ twah tɛ̃ á wrɔ́. Yõ̀bènɔrɩ pɩ́-ò ye ǹnɛ, ànɛ̂ à mɛ gbɩ ó ǹ tɔ̂ɔ́. Ǹmɔ nɛ̂-á Liyel Pi núkúnínɛ, á ye ǹ tɔ̃́rɩ́ wɛ, tɔ̃́rɩ́ nɛ̂-á à wɛ syi ǹ Sú tyɩ́. 
Ǹmɔ tyɩ́ ye lésõ Nsyɔ̃ yo tũ léyy nɛ ńkɛ̃́: «Ǹmɔ tyɩ́ sɔkɔ ye mé lésõ kè yo nɛ ńkɛ̃́: ‹Ǹnɩ̂ kɩ n pɔ n pi ńmɔ tyɩ́ náh, tɛ́ à mɛ tɛ tɛ́ ńmɔ dubi, képah ye à kal ńnɛ.›» 
Ápi mɛ yõ̀bènɔrɩlɛ n tɔ̃ n wɛ n yuku ǹ nɩ̀gbɛ̀kɩ̀ tnɔ́nkàhnmɔnɔ gbõ̀ yõ, ǹmɔ nɛ̂-á Yesulɛ. 
Moyisi gbõ̀ yõ ye Liyel ǹ tyi kõ̀nsàhnɔ́lɛ á kɔ̃. Ǹtɛ Yesu nɛ̂-á Liyel nɛy yáhntɔkɔnɔlɛ, ǹmɔ wɔ gbõ̀ yõ sɔkɔ ye yõ̀bènɔrɩ ànɛ̂ gbɩ wɔ wɛ. 
Ǹnɩ̂ yɔ́ ńtɛ̃ yĩ́ náh tɛ̃ yah Liyel yõ. Ǹtɛ ǹ Pi núkúní nɛ̂ tyɩ́-á yo sɔkɔ, ǹmɔ nɛ̂-á Liyellɛ, tɛ́ mɛ Sú Liyel tnɔ̂, ǹmɔ ye Liyel dùkùlɛ á wɛ̃kɩ. 
Nsyifunɔ yõ̀tãm lésõ Yerusalɛmɩ kwil sárká syínwìlkìpu, ànɛ̂ Levi yìkì névye yísyɔ́ tõ Nsyɔ̃ tyɩ́, pè kyɩ à piki yah nɛ, ǹmɔ tɛ̃̀ se nɛ nɛ̂ ó-á péwɔlɛ nɩ? Nsyɔ̃ náh yahle ké yónɔ́lɛ pé tyɩ́ dɛ́! À kè yo dal mɔ pé tyɩ́ pnɛ́yy pé pól yĩ́ yah nɛ, péwɔ náh Liyel nɛy yáhntɔkɔnɔlɛ. 
Képah sɔkɔ, pè mɛ nɛ, ǹtɛ nɛ̂ se ǹmɔlɛ nɩ? Ńkɛ̃́ ńmɔ se Yelilɛ nɩ? Nsyɔ̃ mɛ pè syah nɛ, ǹmɔ náh péwɔlɛ. Pè nɛ, ǹtɛ Liyel tɛ̃ntõ̀ syínyo-ò nɛ̂-á kɩ n pɔ n pi, ǹmɔ se ǹnɛ nɩ? À mɛ nɛ ɔ̃́ɔ̃-á. 
Képah tɛ̃̀nɛ, pè nɛ, ǹtɛ ǹmɔ tɛ̃̀ se nɛ nɛ̂ ó-á pélɛ nɩ? À kè yo pé syah pé kɔ̃ pé kyɩ wɛ yo pé tṍpú tyɩ́ kɛ̀. Ńkɛ̃́ sɔ̃́ se à kɩ yo ǹ gblɔ̌y tyɩ́ sɔkɔ nɩ? 
Képah sɔkɔ, Nsyɔ̃ mɛ nɛ: «Ńmɔ tyɩ́ ye lésõ Liyel tɛ̃ntõ̀ syínyo-ò Yesayi yo sah ǹ wɛy sɔkɔ nɛ, ǹnɩ̂ yĩ́npɔ̃́-á mɛ n dɔkɔ gbɛ̃́nnɩ sɔkɔ nɛ ńkɛ̃́: ‹Yé wɛ̃̀kɩ̀ tɔ̃ sah Tɛ̃̀ ǹgbɛ̃ yĩnnɛ!›» 
Ǹtɛ nónó-á sõ̀ pè tõ Nsyɔ̃ tyɩ́, Farisi wɛ̃̀kɩ̀ kɔ́pú ye sõ̀ pélɛ. 
Pè tɔ̃ à piki yah nɛ, ǹmɔ-à ńkɛ̃̂nɛ Liyel nɛy yáhntɔkɔnɔlɛ, képah náh pɩ Yelilɛ, képah náh pɩ Liyel tɛ̃ntõ̀ syínyo-òlɛ, ǹtɛ kwâh nɛ̂ se pɩ tɛ́ à névyelɛ n wel n wilki ni sɔkɔ nɩ? 
Képah sɔkɔ, à yo pé tyɩ́ nɛ: «Ńmɔ tɛ̃̀ wɔ mɛ nolɛ n wel n wilki ni sɔkɔ, ǹtɛ ǹnɩ̂ mɛ yépi wrɔ́, nɛ̂-á yé náh ǹnɛ n pnɛ. 
Ǹmɔ kɩ n pɔ n pi ńmɔ tyɩ́ náh. Tɛ́ ńmɔ ńtɛ̃ fṍrɔɔ kah ǹ lékyêh mnɔ fyé dɛ́nmɔnɔlɛ.» 
Kénɛ tyi pól sõ̀ pɩ Betani kwil sɔkɔ, Nsyurdɛ̃ yɔ́nkahnɩnɛ. Nɛ́npɔ́ sɔkɔ ye sõ̀ Nsyɔ̃ n wel n wilki nolɛ ni sɔkɔ. 
Ké dĩndallɛ, Nsyɔ̃-á Yesu wɛ à mɛ n pi ǹ tnɔ̂, à nɛ ńkɛ̃́: «Yé yah, Liyel-á képɔ́píkì nɛ̂ tɛkɩ mɔ kèkõyṍ névye tyípêl tɛ̃ yõ yĩnnɛ, ǹmɔ ye yɔ̀. 
Ǹmɔ tyɩ́ sɔkɔ ye mé lésõ kè yo nɛ ńkɛ̃́: ‹Ǹnɩ̂ kɩ n pɔ n pi ńmɔ tyɩ́ náh, tɛ́ à mɛ tɛ tɛ́ ńmɔ dubi, képah ye à kal ńnɛ.›» 
Ńmɔ ńtɛ̃ tɔ náh lésõ ǹnɛ n pnɛ. Ǹtɛ yáh wɛ mé pɔ wèlnwìlkìnɔ́lɛ n pɩ ni sɔkɔ, ǹ dùkù ye mé Yisrayel tãnnɛ n wɛ̃kɩ.» 
Nsyɔ̃ tyíyókè mɛ̀ yo Yesu tyɩ́ sɔkɔ nɛ ńkɛ̃́: «Liyel nɛ̂-á tɛkɩ mɔ ńnɛ nɛ mé wèlnwìlkìnɔ́lɛ n pɩ ni sɔkɔ, à lésõ yo ń tyɩ́ nɛ ńkɛ̃́: ‹Wáh kɩ Liyel Mirki wɛ kè tiki pɔ tɛ̃ nɛ̂ yõ, ǹmɔ ye kɩ névyelɛ n wel n wilki Liyel Mirki sɔkɔ.› Ńmɔ ńtɛ̃ náh lésõ ǹnɛ n pnɛ. Ǹtɛ mé pɔ Liyel Mirki wɛ kè wil yĩ̂nyõ à n tiki n pi kèpíkípíkí tyɩ́ kɔ̃lɛ, à pɔ tɛ̃ ǹ yõ. 
Mé ye kè wɛ yĩ́nɛ. Képah ye mé kélɛ n yo n dal mɔ nɛ, Liyel Pídĩ́-á dĩ́ mɛ̀nɛ.» 
Tɛ́ dĩndallɛ, Nsyɔ̃ tɔ̃ sõ̀ mɛ nɛ́npɔ́ ànɛ̂ ǹ tõ̀ sã́hpú sɔkɔ nɛ́pĩ̂ nímí yísyɔ́lɛ. 
Wǎh Yesu wɛ à pɔ n sɔkɔ à nɛ ńkɛ̃́: «Yé yah képɔ́píkìlɛ, nɛ̂-á Liyel tɛkɩ mɔ.» 
Képah tɛ̃̀nɛ, Nsyɔ̃ tõ̀ sã́hpú nɛ́pĩ̂ nímí nɔ́pi-á ǹ wɛy mɛ̀ noh sɛ̃́, pè kõ Yesu náh. 
Yesu-á vi yah tɛ́ pé wɛ pè mɛ ǹ sõ̀ tɛ̃, à pè piki yah nɛ, kwâh nɛ̂ se pè n yah n kɔ̃ nɩ? Pè mɛ à ye Wrabilɛ, nɛ̂ kɔ́r-á kwɔ́-ò dĩ́nɛ. Tɛ́ à piki yah nɛ, ǹ twáhntɛ̃̀-ǹsah se mɛ ńyãh sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ, pè pɔ, pǎh kɩ kyɩ kè wɛ. Pè mɛ ǹ sõ̀ tɛ̃ kyɩ ǹ twáhntɛ̃̀-ǹsah wɛ lɛ́rɩ́nɔ́ gbãm-ò kénkɔ̃ sɔkɔ. Képah sɔkɔ, pè mɛ wũ ǹ tyɩ́, à lésõ gbĩ̀yĩ̀kì tɛ́y pɩ. 
Nsyɔ̃ tõ̀ sã́hpú nɛ́pĩ̂ nímí nónó-á sõ̀ ǹ wɛy noh tɛ́ Yesu sõ̀ tɛ̃, pé sɔkɔ núkú yɔ́ yĩn ye Andrelɛ. Simo-Pyɛrɩ yṹnpi ye sõ̀ ǹnɛ. 
Andre-á yal nɛ́npɔ́, kénɛ Simo-Pyɛrɩ tyɩ́ ye à kyɩ wil gbennɛ, ǹmɔ nɛ̂-á ǹ nɛ́nkyɩ́dĩ́nɛ, tɛ́ yo ǹ tyɩ́ nɛ ńkɛ̃́: «Ápi Krista wɛ.» Krista kɔ́r ye Liyel nɛy yáhntɔkɔnɔlɛ. 
Képah sɔkɔ, à Simo tɛ̃ yuku Yesu tyɩ́. Yesu-á Simo wɛ, à à yah, tɛ́ yo ǹ tyɩ́ nɛ, Nsyɔ̃ pídĩ́ Simo-á ǹnɛ, tɛ́ páh kɩ yĩn yɔ́ dyɩ à kɔ̃ Sefasɩlɛ. Wâh noh nɛ Sefasɩ-á, ké kɔ́r ye légbɩ̃́lɛ. Kénɛ yĩn ye tɔ̃ pè n ye Pyɛrɩlɛ. 
Ké dĩndallɛ, Yesu kè yah tɔkɔ nɛ, páh kɩ yuku Kalile gbɛ̃̀nyah sɔkɔ. Wǎh Filipi wɛ, à yo ǹ tyɩ́ nɛ, à pé sõ̀ tɛ̃. 
Betisayida kwil nɛy ye sõ̀ Filipilɛ. Kénɛ kwil tãn ye sõ̀ Andre ànɛ̂ Pyɛrɩ tɔlɛ. 
Képah náh, Filipi kyɩ wil Natanayɛlɩ tyɩ́, tɛ́ yo ǹ tyɩ́ nɛ ńkɛ̃́: «Moyisi tyi kõ̀nsàhnɔ́ ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu-á nɛ̂ tyɩ́ yo sah, á ye à wɛ. Nsyosɛfu pídĩ́ Yesu ye. À n wil Nasarɛtɩ kwil.» 
Wǎh yo sɛ̃́, Natanayɛlɩ à syah nɛ, kwáhgbɩ́ se kɩ fɛ̃ wil Nasarɛtɩ kwil sɔkɔ nɩ? Filipi mɛ yo ǹ tyɩ́ nɛ, à pɔ n yah. 
Képah sɔkɔ, Yesu-á Natanayɛlɩ wɛ à n pi ǹ tnɔ̂, à nɛ ńkɛ̃́, Yisrayel nɛy tɔ́tɔ́-á yɔ̀, nɛ̂-á yĩ́nvǐ náh mɛ ǹ sɔkɔ póllɛ. 
Natanayɛlɩ mɛ nɛ, ńyãh sɔkɔ se à pɩ̃ pélɛ nɩ? Yesu mɛ yo ǹ tyɩ́ nɛ, Filipi-á pópó ǹnɛ n ye, gbĩ́ nɛ̂-á sõ̀ à mɛ sènsóké séswɔ̂ sõ̀ sɔkɔ, páh sõ̀ mɛ ǹnɛ yah. 
Képah sɔkɔ, Natanayɛlɩ mɛ nɛ: «Kwɔ́-ò dĩ́, Liyel Pídĩ́ ye álɛ. Áwɔ ye Yisrayel tãn yõ̀tɛ̃̀nɛ.» 
Wǎh yo sɛ̃́, Yesu mɛ à syah nɛ, páh yo ǹ tyɩ́ nɛ páh sõ̀ mɛ ǹnɛ n yah, gbĩ́ nɛ̂-á à mɛ sènsóké séswɔ̂ sõ̀ sɔkɔ, képah kó à kɔ̃ à sɛ̃ pé yõ nɩ? Ǹtɛ ǹ yĩ́-á kɩ tɔ̃ n tɛ̃ n pi tyí-ǹgbnɔ yõ kɩ kal nɔ́pilɛ. 
Tɛ́ tɔ̃ ǹ syɩ yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, yé kɩ yĩ̂nyõ wɛ yɩ́kɩ́nmɔnɔlɛ, tɛ́ yĩ̂nyõ tõ̀npɩbɩ kɩ mɛ n dɔkɔ, tɛ́ kɩ n tiki Névi Pídĩ́ yõ̀tah sɔkɔ.»
