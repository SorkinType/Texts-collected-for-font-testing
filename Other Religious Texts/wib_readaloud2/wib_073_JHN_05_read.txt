Nsyɔ̃.
5.
Tyi nɔ́pi sɔ́kɔ́ntɛnɔ náh, Nsyifunɔ syáh yɔ́-á pɔ n syah Yerusalɛmɩ kwil sɔkɔ, Yesu dɔkɔ yuku nɛ́npɔ́. 
Tɛ́ Yerusalɛmɩ kwil sɔkɔ, pɛ́kɩ̀ yɔ́ sõ̀ mɛ tũ̀nsàhnɔ́lɛ ninɛ ké sɔkɔ ké kwil dé-ńsah yɔ́ tnɔ̂, nɛ̂-á pè n ye Képyɛ́nɩ̀ dé-ńsahlɛ. Pè kénɛ pɛ́kɩ̀lɛ n ye Bɛtɩsatalɛ Yebre wɛy sɔkɔ. Kènkã̀lnɔ́ kwlɔ́ sõ̀ mɛ pɩ́nɔ́lɛ ké gbɔ̀pɔ sɔkɔ. 
Yámpú náhnáh sõ̀ mɛ sɛ̃́ntɛ̃̀nɔ́lɛ kènkã̀l nɔ́pi sõ̀. Yɩ̃́tũbi sõ̀ mɛ, ànɛ̂ gbáhkɩ̀ núkú névyelɛ, à tahlɩ sèkukubi tyɩ́. Pè sõ̀ ńkɛ̃́nɛ nɛ mɛ ni dúkúrínmɔnɔ syɩkɩ n tɛ̃. 
Yé náh wɛ di, yĩ̂nyõ tõ̀npɩ yɔ́ sõ̀ ńkɛ̃́nɛ nɛ n de n pɔ ni mɛ̀ sɔkɔ, ǹ kélɛ n dukuri n mɔ, tɛ́ yɔ́nwó nɛ̂ ǹgbò-à de ni mɛ̀ sɔkɔ, ǹ yam-à sɔ̃́ mɛ dùkù nɛ̂nɛ, à n dɛ. 
Tɛ́ nɛ́npɔ́ sɔkɔ, dĩ́ yɔ́ lésõ mɛ sèkùkùlɛ pópó ye yõgbãm pɔ́ kwǎrtɔ̃nɔ. 
Yesu-á à wɛ sɛ̃́, tɛ́ pɩ̃ nɛ ǹ sɛ̃́ntɛ̃̀nɔ́ di-á mo nɛ́npɔ́, à à piki yah nɛ, à se mɛ ké tyɩ́ nɛ pé dɛ nɩ? 
Sèkùkù dĩ́ mɛ nɛ, pé yõ̀tɛ̃̀, ni-à tyah n dukuri n mɔ gbĩ́ nɛ̂nɛ, nɛy náh n wɛ à pélɛ n yohnɩ n mɔ pé n wɛ n de ké sɔkɔ. Péwɔ tɛ̃̀-à pópó nɛ pé wɛ n de, ǹnɩ̂ pé dyàh tɛ̃ n de. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́ nɛ, à yuku ǹ sɛ̃ sɛ̃ kwâh tɔkɔ nɛ n kɔ. 
Nɛ́npɔ́ swɔ, dĩ́ núkú dɛ. À mɛ yuku ǹ sɛ̃ sɛ̃ kwâh tɔkɔ à n kɔ. Tɛ́ lésõ pɔ yɩ̃nɛ Nsyifunɔ mɩ mɩ yòhlɛ. 
Képah sɔkɔ, dĩ́ nɛ̂-á dɛ́nmɔnɔ wɛ, Nsyifunɔ yõ̀tãm à syah nɛ ńkɛ̃́, mɩ mɩ yòh-á kwɛynɛ. À náh yɩ̃nɛ à ǹ sɛ̃ sɛ̃ kwâh tɔkɔ nɛ n kɔ. 
Ǹtɛ dĩ́ mɛ pè syah nɛ, nɛ̂-á dɛ mɔ pélɛ, ǹmɔ-á yo pé tyɩ́ nɛ, pé pé sɛ̃ sɛ̃ kwâh tɔkɔ nɛ n kɔ. 
Pè mɛ à piki yah nɛ, nɛ̂ se yo ǹ tyɩ́ nɛ, à ǹ sɛ̃ sɛ̃ kwâh tɔkɔ nɛ n kɔ nɩ? 
Légbĩ́nɛ, Yesu mó tɔ̃ sõ̀ yal tɛ dĩ́ mɛ̀ tnɔ̂, à de tɔ́wû sɔkɔ nɛ́npɔ́. Tɛ́ dĩ́ nɛ̂-á sõ̀ Yesu dɛ mɔ, kénɛ dĩ́ náh sõ̀ ǹnɛ n pnɛ. 
Képah náh, Yesu pɔ à wɛ Liyel ní-ńsah gbô sɔkɔ tɛ́ nɛ, à yah, wǎh dɛ́nmɔnɔ wɛ yɔ̀, à káh tɔ̃ tyípêllɛ n pɩ, tir yɔ́ kɔ̃ kè pɔ à nɔ nɛ̂-á kal ńyɔlɛ dɛ́! 
Képah tɛ̃̀nɛ, dĩ́ kyɩ ké yah yo Nsyifunɔ yõ̀tãm syah nɛ, Yesu-á sõ̀ dɛ mɔ pélɛ. 
Képah sɔkɔ, Nsyifunɔ yõ̀tãm tyah lékàhlɛ Yesulɛ n kɔ̃, wǎh sɔ̃́npɩ́lɛ dɛ́nmɔnɔ pɩ mɩ mɩ yòh sõlɛ. 
Yesu mɛ nɛ, péwɔ Sú Liyel-á mɛ tõ̀ sɔkɔ kṹmm, péwɔ tɔ́-á mɛ sɛ̃́ntɛ̃̀nɛ n pɩ. 
Yesu-á képah yo sɛ̃́, Nsyifunɔ yõ̀tãm tɔ̃ lékàhlɛ ǹnɛ n gbah n kɔ̃, à ǹ kònmɔ-ńsahlɛ n yah n kɔ̃, tɛ́ nɛ wǎh mɛ mɩ mɩ yòh dêllɛ n pɩ. À wil képah sɔkɔ, wǎh tɔ̃ mɛ n yo nɛ ńkɛ̃́, pé Sú tɔ́tɔ́-á Liyellɛ, à ǹ gblɔ̌ynɛ n pɩ núkúlɛ ǹnɛ. 
Képah sɔkɔ, Yesu mɛ tɔ̃ yo pé tyɩ́ nɛ ńkɛ̃́: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, Pídĩ́ náh fɛ̃ tir yɔ́ ńtɛ̃ pɩ n pi ǹ yah yĩ́ĩ́nɛ, nɛ́ à náh ké pɩ́nɔ́ wɛ yah Sú Liyel tyɩ́. Sú Liyel-á nɛ̂nɛ n pɩ, sɛ̃́nkɔ̃ ye Pídĩ́ tɔ n nɛ n pɩ. 
Pídĩ́ tyɩ́ nɔ Sú Liyel tyɩ́. Képah ye, wàh tyi nónó póllɛ n pɩ, ǎ sé pɩ́nɔ́lɛ ǹnɛ n wɛ̃kɩ. Tɛ́ wǎh tyi nónónɛ à wɛ̃kɩ, à kɩ tɔ̃ pɔ sé lékan pɩ́nɔ́lɛ à wɛ̃kɩ n pi, nónó-á kɩ yé gbã̀n du. 
Sɔ̃́-á Sú Liyel lékyɩ̂lɛ n yilki n mɔ, à minnɛ pélɛ n kɔ̃, sɛ̃́ntɛ̃̀ ó ye névi nɛ̂-à Pídĩ́ dyɔ, ǎ minnɛ tɛ̃̀ kɔ̃. 
Sú Liyel mɛ ké tyɩ́ nɛ, no pól gbílkínɔ́lɛ n sah Pídĩ́ yõ, sɔ̃́-á pè kélɛ n sah ǹmɔ yõ. Képah ye, à náh ǹnɩ̂ yɔ́ ńtɛ̃nɛ n tuke. À ye no pól tukey tyɩ́ dahbɩ ǹ gbõ̀. Tɛ́ névi nɛ̂-à gbílkínɔ́lɛ n sàh-ǹkɛ̃̂nɛ ǹmɔ yõ, ǹ tɛ́kɩ́nmɔ-ò Liyel ye tɛ̃̀ yahle gbílkínɔ́lɛ. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à ń wɛy noh, tɛ́ sɛ̃ ń tɛ́kɩ́nmɔ-ò yõ, tɛ̃̀-á kɩ tyɛ́-ńkɛ̃̂ min wɛ. À náh n tuke n pi. Wǎh wil tɛ kũ sɔkɔ, tɛ́ min wɛ. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, gbĩ yɔ́ mɛ n pi, nɛ̂-á mɛ tɛ núkúnúkú ńtɛ̃nɛ yɔ̀. Képah sɔkɔ ye névye nónó-á mɛ kúnɔ́lɛ Liyel tyɩ́ tĩ̀nnɛ, pè kɩ ńmɔ yĩ́npɔ̃́ noh, ńmɔ nɛ̂-á Liyel Pídĩ́nɛ. Tɛ́ nónó-à kè noh, pè kɩ min wɛ. 
Sɔ̃́-á Sú Liyellɛ min nɛ́tɛ̃̂nɛ, sɛ̃́ ye ńmɔ nɛ̂-á ǹ Pídĩ́nɛ, à ńmɔ tɔ kɔ̃ mé pɩ min nɛ́tɛ̃̂nɛ. 
Tɛ́ máh pɩ Névi Pídĩ́nɛ sɛ̃́, à tukey tùkènɔ́ tɛ̃́nwɛnɔlɛ ń kɔ̃. 
Ǹtɛ yé káh n yɛ tyi nɔ́pi yé gbã̀n du. Gbĩ yɔ́ mɛ n pi, lékyɩ̂ pól kɩ ń yĩ́npɔ̃́ noh, 
tɛ́ kɩ yuku. Képah sɔkɔ, nónó-á tyípéplɔ pɩ, pépi kɩ yuku tyɛ́-ńkɛ̃̂ min wɛ yĩnnɛ. Tɛ́ nónó-á mó tyípêl pɩ, pépi wɔ kɩ yuku fõh yĩnnɛ. 
Ńmɔ náh fɛ̃ tir yɔ́ ńtɛ̃ pɩ n pi ń yah yĩ́ĩ́nɛ. Ń Sú Liyel-à yo sɔ̃́ ń tyɩ́, sɛ̃́ ye mè n nɛ n kãm wɛynɛ tukey sɔkɔ. Mé náh mɛ ń gblɔ̌y nɩ̀vɩ̀nɩ̀ tyi pɩ́-ńsahlɛ n yah n kɔ̃. Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ-á mɛ nɛ̂ tyɩ́, képah ye mè n nɛ n pɩ. Képah ye, ń wɛy kã̀mnɔ́ n pɩ gbɩ wɛ̃̀kɩ̀ yõ. 
Ńmɔ-à kɩ nɛ̂ n yo ń gblɔ̌y tyɩ́lɛ, nɛ̂ kɩ sɛ̃ ń tyíyónɔ́ yõ? 
Ǹtɛ nɛ́káhkɩ́ ye n yo n wah ń tyɩ́lɛ, tɛ́ mé pɩ̃ nɛ tyi nónó-á à n yo ńmɔ tyɩ́ sɔkɔ, gblɔ-á sélɛ. 
Yáh névye nónó tõ Nsyɔ̃ tyɩ́ ńmɔ tyɩ́ sɔkɔ, tɛ́ à tyi nónó yo, gblɔ ye sélɛ. 
Ǹtɛ mé se yɔ́ mɛ n yah n kɔ̃ nɛ, névi tɛ̃̀ wɔ ń dùkùlɛ n yo? Yé pwáhnmɔnɔ wɛ yĩn ye mé tyi nɔ́pilɛ n yo yé tyɩ́. 
Nsyɔ̃ lésõ pɩ kyòolɛ fɔ̀tnɔ̀ tũ̀nsàhnɔ́ tyɩ́ kɔ̃lɛ. Tɛ́ kè yé dyɔ, yé nɩ̀vɩ̀nɩ̀ pɩ ǹ kyòo sɔkɔ wrɔ́ kwéy yɔ́ ó sɔkɔ. 
Ǹtɛ ńmɔ wɔlɛ, Sú Liyel-á tã̀n nónó dahbɩ ń gbõ̀ ń kɔ̃ mé n pɩ, sépi ńtɛ̃ mɛ ń dùkùlɛ n wɛ̃kɩ n wah n kal Nsyɔ̃-á nɛ̂ yo ń tyɩ́ sɔkɔ. Kénɛ tã̀n nɔ́pi pɩ́nɔ́ ye n wɛ̃kɩ nɛ, Sú Liyel-á tɛkɩ mɔ ńnɛ. 
Tɛ́ à wil képah tyɩ́ sɔkɔ, Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹ tɔ ń tyɩ́ yo wah. Ǹtɛ yé náh ǹ wɛlɔ noh yah póllɛ, yé náh se tɔ̃ à wɛ yah yĩ́nɛ. 
Ǹ wɛy náh mɛ n tɛ̃ yé nnɔ sɔkɔ. Yé náh wɛ wǎh ńmɔ tɛkɩ mɔ, yé yahle sɛ̃́nɔ́lɛ ń yõ di? 
Yè n nɛ mɛ n sõ nɛ, yáh kɩ tyɛ́-ńkɛ̃̂ min wɛ Liyel wɛy sɛ́bɛ́y kàrnɔ́ sɔkɔ, ǹ tɛ́ kélɛ n kar fírfír. Ǹtɛ képah ye mó tɔ̃ yo wah tɔ́tɔ́ ńmɔ tyɩ́lɛ. 
Tɛ́ yé náh mó tɔ̃ mɛ ké tyɩ́ nɛ, yé pɔ ńmɔ tyɩ́, yé pɔ tyɛ́-ńkɛ̃̂ min wɛ. 
Mé náh mɛ n yah n kɔ̃ nɛ névye ńnɛ n gbilki. 
Mé yé dùkùlɛ n pnɛ. Liyel tyɩ́ náh nɔ yé tyɩ́. 
Ń Sú Liyel yĩn yõ ye mé pɔ, tɛ́ yé náh ń tɛ̃ plɛ. Ǹtɛ ǹnɩ̂ yɔ́-à núkú pɔ ǹ yah yĩ́ĩ́nɛ yɔ̀, yé kɩ ǹmɔ tɛ̃ plɛ. 
Yé kɩ fɛ̃ sɛ̃ ń yõ sɔ̃́, yépi nónó-á mɛ ké tyɩ́ nɛ yé wɛ̃nɛ n gbilki, tɛ́ yahle gbílkínɔ́ yáhnkɔ̃nɔlɛ, nɛ̂-á n wɛ Liyel núkúní ó tyɩ́? 
Yé káh nɛ mɛ n sõ nɛ, ńmɔ-á kɩ kyɩ yé kɔ̃ n pi lékàhlɛ Sú Liyel yah sɔkɔ dɛ́! Yé gbõ̀ sàhnɔ́-á mɛ Moyisi nɛ̂ yõ, ǹmɔ ye kɩ kyɩ lékàhlɛ yé kɔ̃. 
Ńmɔ tyɩ́ mɛ Moyisi wãrkɩ sah ǹ sɛ́bɛ́ynɔ́ sɔkɔ. Gbɩ-à yé sõ̀ mɛ sɛ̃ ǹmɔ yõ, yé náhkɩ mɛ sɛ̃ ńmɔ tɔ yõ. 
Ǹtɛ yé náh fyɔ̀ mɛ sɛ̃ ǹmɔ tyi wã́rkɩ́nsàhnɔ́ yõ, sɔ̃́ mɛ yé kɩ fɛ̃ sɛ̃ ńmɔ wɛlɔ yõ wɛ́?»
