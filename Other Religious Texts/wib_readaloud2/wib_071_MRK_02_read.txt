Markɩ.
2.
Sèpĩn yísyɔ́ sɔ́kɔ́ntɛnɔ náh, Yesu ǹ syɩ pɔ Kapɛrnayim sɔkɔ. Névye-á kè noh nɛ, wǎh mɛ gbô, 
tɔ́wû yɔ́ tuke pɔ tyah fyãh. Tɛ̃̀-ǹsah ńtɛ̃ náh sõ̀ mɛ kélé yah tɔ sɔkɔ. Képah sɔkɔ, à tyah ǹ wɛynɛ pélɛ n kwɔ. 
Légbĩ́nɛ, nósyɔ́ pɔ ǹ tyɩ́ sèkùkù yɔ́lɛ. Nɛ́pĩ̂ ǹyẽ̂h sõ̀ mɛ à twah. 
Tɛ́ névye tnɔ́nkàhnɔ́ tɛ̃̀ tyɩ́, pè náh sõ̀ fɛ̃ nɔ n de n pi ǹ tnɔ̂. Pè dɔkɔ gbɛ̃́ntĩ̂n yõ, kye ǹ mɛ́-ńsah tírrlɛ, tɛ́ sèkùkù mɛ̀ tiki mɔ kénɛ pwã̀hlɛ ǹ sɛ̃ sɛ̃ kwâh yõ ǹ yah sɔkɔ. 
Wǎh kè wɛ, pè sɛ̃ ǹ yõ sɛ̃́, à yo sèkùkù mɛ̀ tyɩ́ nɛ: «Ń pídĩ́, á tyípêl ye yɔ̃ mɔ ǹ kɔ̃.» 
Képah tɛ̃̀nɛ, *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú nónó-á sõ̀ mɛ tɛ̃̀nɔ́lɛ nɛ́npɔ́ sɔkɔ, pè tyah n sõ nɛ, 
sɔ̃́ se pɩ, tɛ́ dĩ́ mɛ̀ n wɛkɩ sɛ̃́ nɩ? Liyel yĩn-á à n yɩkɩ. Ké névi se mɛ kɩ fɛ̃ tyípêl yɔ̃ mɔ névi kɔ̃, nɛ́ Liyel náh nɩ? 
Képah sɔkɔ, wǎh núkú pé fɔkɔ sõnɔ wɛ sɛ̃́, à yo pé tyɩ́ nɛ: «Sɔ̃́ pɩ, tɛ́ yé fɔkɔ sõnɔlɛ sɛ̃́? 
Nɛ̂ tɛ̃̀ dɔ̀dɔ̀ kal, à yo sèkùkù mɛ̀ tyɩ́ nɛ, ǹ tyípêl-á yɔ̃ mɔ à kɔ̃, képah náh pɩ, à yuku, ǹ sɛ̃ sɛ̃ kwâh tɔkɔ nɛ n kɔ? 
Ǹtɛ, yé pɩ̃ nɛ, tɛ̃́nwɛnɔ-á mɛ *Névi Pi tyɩ́, kɩ fɛ̃ tyípêl yɔ̃ mɔ névye kɔ̃, tɛ́ à sah sétáh yõ.» 
Képah ye, à yo sèkùkù mɛ̀ tyɩ́: «Mé kélɛ n yo á tyɩ́, yuku, á sɛ̃ sɛ̃ kwâh tɔkɔ, ǹ nɛ n yuku á gbô.» 
Nɛ́npɔ́ swɔ, dĩ́ núkú yuku ǹ sɛ̃ sɛ̃ kwâh tɔkɔ sɔkɔ no pól yĩ́ yah. Kè pé pól gbã̀n du. Pè tyah Liyel yĩnnɛ n gbilki, tɛ́ mɛ n yo nɛ, pé náh sɛ̃́nkɔ̃ wɛ yah. 
Képah náh, Yesu tɔ̃ n syɩ yuku Kalile yɔ́-ǹgbɛ̃ gbɔ̀pɔ. Tɔ́wû pól sõ̀ mɛ n pɔ n pɔ ǹ tyɩ́, à pélɛ n kwɔ. 
Wǎh pɔ n sɔkɔ, à Alfe pídĩ́ Levi wɛ tɛ̃̀nɔ́lɛ mɔ́rɩ́ syí-ńsah wɔ̀kɔ̀. À mɛ yo ǹ tyɩ́ nɛ, à yuku pé sõ̀ tɛ̃. Levi mó yuku ǹ sõ̀ tɛ̃. 
Pǎh kyɩ tyah yõkelɛ n yõ Levi tĩ̂nnɛ gbĩ́ nɛ̂nɛ, mɔ́rɩ́ syípú, ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú náhnáh sõ̀ mɛ n yõ ǹnɛ, ànɛ̂ ǹ *tõ̀ sã́hpúlɛ. Kénɛ névye dùkù náhnáh ye sõ̀ mɛ ǹ sõ̀ tɛ̃, képah ye. 
Képah sɔkɔ, *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú nónó-á sõ̀ *Farisiyɛ̃nɔlɛ, pépi-á à wɛ à mɛ n yõ ànɛ̂ kénɛ névye dùkùlɛ, pè yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, kû se pɩ, tɛ́ à n yõ mɔ́rɩ́ syípú, ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂púlɛ fɩ̃́ núkú nɩ? 
Yesu-á kénɛ wɛy noh sɛ̃́, à yo pé tyɩ́: «Névye nónó-á mɛ lènɔlɛ, sènpɩ yétõ̂ se mɛ pé tyɩ́? Yámpú tyɩ́ náh sènpɩ yétõ̂lɛ di? Névye nónó-á n sõ nɛ, névye tír tír-á pélɛ, mé náh pɔ pépi ye yĩnnɛ. Ǹtɛ, nónó-á pɩ̃ nɛ, tyípêl pɩ́pú-á pélɛ, pépi yé-ńsah ye mé pɔ.» 
Sṍ yɔ́lɛ, *Wèlnwìlkì-ò Nsyɔ̃ tõ̀ sã́hpú ànɛ̂ *Farisiyɛ̃nɔ sõ̀ mɛ níkì tɛ̃. Névye mɛ kyɩ Yesu piki yah nɛ, kû se pɩ, tɛ́ Nsyɔ̃ tõ̀ sã́hpú ànɛ̂ Farisiyɛ̃nɔ tõ̀ sã́hpú mɛ níkì tɛ̃, tɛ́ ǹmɔ *tõ̀ sã́hpú náh mɛ sɛ̃́ntɛ̃̀nɛ n pɩ nɩ? 
Wǎh képah noh sɛ̃́, à yo pé tyɩ́: «Yépi tɛ̃̀ sõnɔ sɔkɔ, fúr pɩ́-ò-á nónó ye, pè se kɩ fɛ̃ fúr yõke yahle, tɛ́ kɩ fúr pɩ́-ò sah pé nɩyṍ? Pɛ tɛ̃̀-à fyɔ̀ mɛ pé nɩyṍ, pé náh fɛ̃ yõke yahle n pi. 
Ǹtɛ, gbĩ́ mɛ n pi, pè kɩ pɔ pɛ mɛ̀ wilki pé nɩyṍ. Ké lésõlɛ, pè kɩ pyě níkì tɛ̃. 
Névi yɔ́ ńtɛ̃ náh fɛ̃ fwɔ sã́n tɔkɔ, à kɩ fwɔ dyàh min n pi. Wâh képah pɩ, fwɔ sã́n kɩ fwɔ dyàh kú-ńsah tɛ́y yɔrɩ yɩkɩ ńtɔ̃, ké pwã̀h kɩ gbɛ mɔ. 
Ànɛ̂ tɔ̃, névi náh fɛ̃ dìvɛ̃ sã́n mɔ n pi pley di sɔkɔ. Dìvɛ̃ sã́n-à fũ, kè kɩ pley nɔ́pi sɛkɩ. Dìvɛ̃ kɩ kwɛ, pley tɔ kɩ yɩkɩ. Képah ye, dìvɛ̃ sã́n ye yɩ̃nɛ mɔ́nɔ́lɛ pley snɔ́ sɔkɔ.» 
*Mɩ mɩ yòh sṍ yɔ́lɛ, Yesu ànɛ̂ ǹ *tõ̀ sã́hpú sõ̀ mɛ n kɔ n kah sèmukɔ swɛ̃y yísyɔ́ sɔkɔ. Képah sɔkɔ, ǹ tõ̀ sã́hpú tyah sèmukɔ ywɔ̃lɛ n kɔ̃lɩ n yuku wɛ̃̀kɩ̀ yah. 
*Farisiyɛ̃nɔ-á képah wɛ sɛ̃́, pè tyah Yesulɛ n piki n yah nɛ, à yah, kû se pɩ, tɛ́ tir nɛ̂ náh yɩ̃nɛ pɩ́nɔ́lɛ mɩ mɩ yòh sṍlɛ, ǹ tõ̀ sã́hpú kélɛ n pɩ nɩ? 
À mɛ yo pé tyɩ́: «Kwɛkɩ-á lésõ *Davidi ànɛ̂ ǹ náh névye kah tɛ̃ sṍ nɛ̂nɛ, tɛ́ yõke wɛ́-ńsah náh mɛ pé tyɩ́, pǎh nɛ̂ pɩ, yé náh kè kar yah di? 
À náh sõ̀ de Liyel wɔ̀kɔ̀ sɔkɔ di? Búrn nónó-á sõ̀ sàhnɔ́lɛ Liyel tyɩ́ yõ, à náh sè tɔkɔ tah, tɛ́ ǹ náhtãn tɔ kɔ̃ pè tah di? Ké légbĩ́nɛ, Abyatar ye sõ̀ *sárká wìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃nɛ. Tɛ́ névi yɔ́ ńtɛ̃ náh sõ̀ yɩ̃nɛ búrn nɔ́pi táhnɔ́lɛ, nɛ́ sárká wìlkìpu yõ̀tãm náh.» 
Képah náh, Yesu tɔ̃ yo pé tyɩ́: «Mɩ mɩ yòh ye sah névi yĩnnɛ, ǹtɛ névi náh mɔ mɩ mɩ yòh yĩnnɛ. 
Képah ye, *Névi Pi pɩ mɩ mɩ yòh ńtɛ̃ yõ̀tɛ̃̀nɛ.»
