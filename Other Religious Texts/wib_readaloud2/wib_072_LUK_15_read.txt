Likyi.
15.
Mɔ́rɩ́ syípú ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú pól sõ̀ n pɔ n pɔ Yesu tnɔ̂ ǹ wɛy noh yĩnnɛ. 
Képah tɛ̃̀nɛ, Farisiyɛ̃nɔ, ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú Yesu tyɩ́lɛ n plɔ n yãh nɛ, dĩ́ mɛ̀-á mɛ tyípêl pɩ́púlɛ n tɛ̃ plɛ, tɛ́ mɛ n yõ pélɛ ńtɛ̃nɛ. 
Képah sɔkɔ, Yesu mɛ sèmìrkì yɔ́ yãh pé tyɩ́: 
«Képyɛ́nɩ̀ lèkwɛ̌-à mɛ yépi sɔkɔ yɔ́ tyɩ́, tɛ́ núkú mɔ pu sé sɔkɔ, tɛ̃̀ náh sé tíkíyẽ́h pɔ́ gbãm pɔ́ kwǎryãh nɔ́pi sah n pi gbãn sɔkɔ, tɛ́ kyɩ mɔ́npunɔ mɛ̀ soh kɔ̃, à syi kyɩ à wɛ n pi di? 
Tɛ́, wàh à wɛ, nɩ̀vɩ̀ tɛ̃̀ tyɩ́, ǎ à twah ǹ gbɛ̃̀nkã̀l yõ ǹ syɩ. 
Wàh ǹ syɩ pɔ gbô, ǎ ǹ kódíbí, ànɛ̂ ǹ tnɔ̂ tɛ̃̀pu ye n tuke, ǹ tɛ́ n yo pé tyɩ́ nɛ ńkɛ̃́, pé képɔ́ nɛ̂-á sõ̀ mɔ pu, páh à wɛ tɛ̃. Képah-á, pè pɔ pé nɩ̀vɩ̀ pɩ.» 
Képah yóntɛnɔ náh, Yesu tɔ̃ pè syah nɛ: «Mé kélɛ n yo yé tyɩ́: sɛ̃́ ó ye, tyípêl pɩ́-ò nɛ́núkù fyé-à Liyel wɛ̃̀kɩ̀ tɛ̃, nɩ̀vɩ̀ kɩ pɩ Liyel fṍ kɩ kal nɛ́pĩ̂ tíkíyẽ́h pɔ́ gbãm pɔ́ kwǎryãh nɔ́pi tyɩ́lɛ, pépi nónó-á nɛ ńkɛ̃́, páh mɛ nɛ́gblɔ́lɛ Liyel yah, Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ yétõ̂ náh tɔ̃ mɛ pé tyɩ́.» 
Yesu tɔ̃ yo pé tyɩ́: «Pɔ́ plɔ́ gbãm-à mɛ sê yɔ́ tyɩ́, tɛ́ sé sɔkɔ núkú mɔ pu, à náh fɔ̀tnɔ̀ soh, tɛ́ wɔ̀kɔ̀ yal, à kè gbah soh kɔ̃, à syi kyɩ kè wɛ n pi di? 
Tɛ́ wàh kè wɛ, ǎ ǹ kósyɔ́, ànɛ̂ ǹ tnɔ̂ tɛ̃̀pu ye n tuke, ǹ tɛ́ n yo pé tyɩ́ nɛ ńkɛ̃́, pé pɔ́ nɛ̂-á sõ̀ mɔ pu, páh kè wɛ. Képah-á, pè pɔ pé nɩ̀vɩ̀ pɩ.» 
Képah yóntɛnɔ náh, Yesu tɔ̃ pè syah nɛ: «Mé kélɛ n yo yé tyɩ́: sɛ̃́ ó ye, tyípêl pɩ́-ò nɛ́núkù fyé-à Liyel wɛ̃̀kɩ̀ tɛ̃, yĩ̂nyõ tɛ̃ntõnè kɩ nɩ̀vɩ̀lɛ n pɩ.» 
Képah náh, Yesu tɔ̃ yo pé tyɩ́: «Dĩ́npì nɛ́pĩ̂ nímí lésõ mɛ dĩ́ yɔ́ tyɩ́. 
Sṍ yɔ́lɛ, gbɛ̃̀ tɛ̃̀ yo ǹ sú tyɩ́ nɛ, pé bɔ̌, ǹ gbɛ̃kɩ sɔkɔ, nɛ̂-á pé wɛ wɛ tɛ̃̀nɛ, à kélɛ pé kɔ̃. Wǎh yo sɛ̃́, ǹ sú mɛ ǹ tahrɩ dibi pè kɔ̃. 
Sèpĩ̀n yísyɔ́ sɔ́kɔ́ntɛnɔ náh, gbɛ̃̀ tɛ̃̀ ǹ tyɩ́ yãm, tɛ́ ǹ pɔ́ tɔkɔ sɔkɔ kwil fɩ́ɩ́ yɔ́ sɔkɔ. À kyɩ sè pɩ karkɩ tyíwã̂n yõ. 
Wǎh ǹ pɔ́ pól pɩ karkɩ tɛ gbĩ́ nɛ̂nɛ, kwɛ́npékè yɔ́ de kénɛ kwil. Képah sɔkɔ, yãm à tɛ̃. 
Kǎh pɩ sɛ̃́, à kyɩ tõ̀ yah kɔ̃ kénɛ kwil nɛy yɔ́ tyɩ́. Tɛ̃̀ mɛ̀ à tɛkɩ mɔ ǹ swãh, ǹ sel syɩkɩ yĩnnɛ. 
Ǎ sel yõkelɛ n yah yèwannɛ, ǹtɛ nɛy náh ǹnɛ n kɔ̃ kélɛ. 
Képah sɔkɔ, ǹ yɩ̃́-á pɔ yɩkɩ, à tyah n sõ nɛ, pé bɔ̌ tyɩ́ yõke-á mɛ dò, ǹ tõ̀npɩbɩ pól mɛ n yõ n kwlɔ n sah. Ǹtɛ, péwɔ-á ásõ̂ kwɛkɩ n ko. 
Ńkɛ̃́, páh kɩ sɔkɔ pé bɔ̌ tyɩ́, tɛ́ kɩ yo ǹ tyɩ́ nɛ, pé bɔ̌, páh pɩ de Liyel tyɩ́, tɛ́ pɩ de ǹ tɔ tyɩ́. 
Pé náh tɔ̃ yɩ̃nɛ yénɔ́lɛ ǹ pilɛ. Ńkɛ̃́, à pélɛ n tɛ̃ ǹ tõ̀npɩ sɔ̃́. 
Képah tɛ̃̀nɛ, à sɔkɔ ǹ sú tyɩ́. Wǎh mɛ fɩ́ɩ́ ǹ sú gbôlɛ, ǹ sú à wɛ. Wǎh à wɛ sɛ̃́, ǹ yãm à kah tɛ̃, à mɛ to kyɩ à yohnɩ pɔrɩ tɛ̃. 
Ǹ pídĩ́ mɛ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé bɔ̌, páh pɩ de Liyel tyɩ́, tɛ́ pɩ de ǹ tɔ tyɩ́, pé náh tɔ̃ yɩ̃nɛ yénɔ́lɛ ǹ pilɛ. 
Ǹtɛ, ǹ sú mɛ yo ǹ tõ̀npɩbɩ tyɩ́: ‹Fwɔ̀mɔ̀-ǹgbɛ̃ nɛ̂-á plɛ kal, yé kyɩ ké tɔkɔ súú, yé pɔ mɔ à kɔ̃. Yé gbõ̀ngbɔ tã́hnɩ́ dyɩ à kɔ̃, yé mó lékyêh dyɩ à kɔ̃. 
Ń pídĩ́ nɛ̂-á yɔ̀, kè pɩ nɛ, à sɔ̃́ sõ̀ ku, tɛ́ tɔ̃ ǹ syɩ min wɛ. À lésõ mɔ pu, tɛ́ à ye wɛ. Képah ye, nɩrɩ-á mɛ náhpíkì nɛ̂ sɔkɔ, yé ǹmɔ tɛ̃ n pɔ n ko á nɛ n yõ, á mó nɛ n syah nɩ̀vɩ̀lɛ.› Képah tɛ̃̀ sɔkɔ, pè tyah syáh syáhnɔ́lɛ nɩ̀vɩ̀ sɔkɔ. 
Légbĩ́nɛ, ǹ pídĩ́ lékɔ̃nɩ́ sõ̀ mɛ swãh sɔkɔ. Wǎh ǹ syɩ n pi, tɛ́ nɔ de gbôlɛ, à yímɔ́nɩ̀, ànɛ̂ yínɔ́ yĩ́npɔ̃́ noh. 
À tõ̀npɩ yɔ́ ye piki yah, tir nɛ̂-á n pɩ. 
Ǹmɔ mɛ yo ǹ tyɩ́ nɛ, ǹ gbɛ̃̀ndĩ́-á pɔ de ǹ lènɔlɛ, képah-á nɩrɩ-á mɛ náhpíkì nɛ̂ sɔkɔ, ǹ sú mɛ à tɛ̃ ko. 
Képah tɛ̃̀nɛ, nɛ́nkyɩ́ mɛ̀ fɔkɔ vyãh, à yahle dénɔ́lɛ gbô. Ǹ sú mɛ wil nɛ, pé à ni à de. 
Képah sɔkɔ, à yo ǹ sú tyɩ́ nɛ, à yah, ké ye náhnáh-á yɔ̀, tɛ́ pé mɛ ǹ tõ̀lɛ n pɩ. Pé náh ǹ yóré yahle yah. Ǹtɛ, à náh pũ̀képir ńtɛ̃ fyé tɛ̃ pé kɔ̃ yah, pé syi ko syáh syah pé kódíbílɛ. 
Tɛ́ ǹ pídĩ́ mɛ̀ wɔ-á pɔ, ǹmɔ nɛ̂-á ǹ tahrɩ pɩ karkɩ syɔ́ fɔ̀fɩ́nɩ̀ yõ, à náhpíkì ko ǹmɔ kɔ̃. 
Képah sɔkɔ, ǹ sú yo ǹ tyɩ́: ‹Ń pídĩ́, áwɔ mɛ kṹmm ń tnɔ̂, ń gbã̀n yî póllɛ, áwɔ tyɩ́ ye. 
Á gbɛ̃̀ndĩ́ nɛ̂-á yɔ̀, kè pɩ nɛ à sɔ̃́ sõ̀ ku, tɛ́ tɔ̃ ǹ syɩ min wɛ. Á lésõ mɔ pu, tɛ́ à ye wɛ. Képah ye, mé yɩ̃nɛ mè syáh syah, mé nɩ̀vɩ̀ pɩ.›»
