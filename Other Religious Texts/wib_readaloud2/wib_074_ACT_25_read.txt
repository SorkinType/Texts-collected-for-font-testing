Tyípɩ́nɩ̀.
25.
Képah náh, gbɛ̃̀nyah yah tɛ̃́-ò Fɛsitusi pɔ́nɔ́ sèpĩ̀n tɔ̃́-ò sõlɛ Sesarɩ kwil sɔkɔ, à dɔkɔ yuku Yerusalɛmɩ kwil. 
Képah sɔkɔ, sárká syínwìlkìpu yõ̀tãm ànɛ̂ Nsyifunɔ nɛ́-ǹgbnɔ kyɩ ǹ tyɩ́ nɛ ńkɛ̃́, Pole-á tyítúkúnì pɩ. 
Tɛ́ tyah ǹnɛ n gbah n ni nɛ, à ǹ gblɔ̌y yɛ pé tyɩ́, à mó Pole kɔ̃ à pɔ Yerusalɛmɩ kwil, pé ǹ tyi sohlɩ n yah pallɛ. Ǹtɛ kègbɩ tɛ̃̀ yõ, yĩ́nvǐ ye sõ̀ pè n pɩ nɛ, pé à wɛ ko mɔ wɛ̃̀kɩ̀ yah. 
Ǹtɛ Fɛsitusi mɛ pè syah nɛ, Pole-á mɛ dáhnmɔnɔlɛ wɔ̀kɔ̀npèkè sɔkɔ Sesarɩ kwil. Tɛ́ péwɔ ńtɛ̃ tɔ náh tɔ̃ n mo n pi pé syɩ́nsɔkɔ-ńsah nɛ́npɔ́. 
Ńkɛ̃́, pé yah tãm nónó-à kɩ fɛ̃ n tɛ̃ n pi, pè péwɔ sõ̀ tɛ̃ pé tiki yuku Sesarɩ kwil. Pole-à kɩ se mɛ tyítúkù yɔ́ pɩ pè wɛ, páh kɩ kyɩ nɛ́npɔ́, pè kɩ n tɔkɔ à kɔ̃. 
Képah sɔkɔ, Fɛsitusi náh tɔ̃ tɛ̃ mo nɛ́npɔ́ à kah sèpĩ̀n kwǎrtɔ̃nɔ, képah náh pɩ, gbãm yõ sɔkɔ, tɛ́ ǹ syɩ sɔkɔ Sesarɩ kwil. Ké dĩndallɛ, à mɛ kɔ̃ tɛ̃ tukey tùkè-ǹsah sɔkɔ, tɛ́ tɛ̃ntõ̀ mɔ nɛ pè pɔ Polelɛ. 
Pole-á pɔ gbĩ́ nɛ̂nɛ, Nsyifunɔ nónó-á sõ̀ wil Yerusalɛmɩ kwil, pè pɔ à mɔ gbɩ̃, tɛ́ tyah lékàhlɛ ǹnɛ n kɔ̃ képékèyɔ̂ tyi náhnáh sɔkɔ. Tɛ́ tir yɔ́ ńtɛ̃ náh mɛ pé tyíyónɔ́ nɔ́pi sɔkɔ, nɛ̂-á wɛ̃kɩ nɛ gblɔ-á sélɛ. 
Képah sɔkɔ, Pole mɛ tyah ǹ gblɔ̌y vyãhlɛ n kɔ̃lɩ nɛ, pé náh tyítúkù yɔ́ ńtɛ̃ pɩ à de Nsyifunɔ tyi kõ̀nsàhnɔ́ tyɩ́, képah náh pɩ Liyel ní-ńsah gbô tyɩ́, képah náh pɩ Wrome kwíltãm yõ̀tɛ̃̀ Sesar tyɩ́. 
Ǹtɛ Fɛsitusi wɔ sõ̀ mɛ ké tyɩ́ nɛ, Nsyifunɔ pé yĩ́ngbɩ́lɛ n yo. Kǎh pɩ sɛ̃́, à Pole syah nɛ, à se kɩ fɛ̃ syi kyɩ Yerusalɛmɩ sɔkɔ, ǹ tukey kyɩ n tuke péwɔ yĩ́ yah nɩ? 
Wǎh yo sɛ̃́, Pole mɛ yo ǹ tyɩ́ nɛ, Sesar tukey tùkè-ǹsah yɔ́ sɔkɔ-á pélɛ yɔ̀, ǹmɔ tyɩ́-á pé tukey yɩ̃nɛ kè tuke. Nsyifunɔ-á yɔ̀, pé náh tyítúkù yɔ́ ńtɛ̃ pɩ pé tyɩ́. Yõ̀tɛ̃̀ dĩ́, ǹmɔ ńtɛ̃ tɔ pɩ̃́nyĩnɔ-á mɛ ké sɔkɔ ó. 
Ńkɛ̃́, pé tyɩ́-à kɩ mɛ kah, ànɛ̂ pé mɛ tir yɔ́ pɩ nɛ̂-á yɩ̃nɛ kũnɛ, pé náh mɛ yahle pé kúnɔ́ yõ dò. Ǹtɛ pǎh tɔkɔ péwɔlɛ n kɔ̃ tyi nónó sɔkɔ, gblɔ-à ńkɛ̃̂nɛ sélɛ, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ pé gbàh dɛ mɔ pè kɔ̃ n pi. Páh mɛ ké tyɩ́ nɛ, pé tyɩ́ tukey kyɩ n tuke Sesar tyɩ́. 
Képah sɔkɔ, Fɛsitusi ànɛ̂ ǹ tukey tùkèpu tĩ̀nkì mɛ wɛ̃ wɛ. Pǎh tyɛ, à mɛ yo Pole tyɩ́ nɛ, wǎh fyɔ̀ Sesar yĩn yo tɛ, wǎh kɩ kyɩ ǹmɔ tyɩ́ Wrome kwil sɔkɔ. 
Sèpĩ̀n yísyɔ́ sɔ́kɔ́ntɛnɔ náh, yõ̀tɛ̃̀ dĩ́ Akripa ànɛ̂ ǹ dósê Berenisi pɔ yah gbɛ̃̀nyah yah tɛ̃́-ò dĩ́ Fɛsitusi tyɩ́, Sesarɩ kwil. 
Pè kɩ sõ̀ sèpĩ̀n yísyɔ́ pɩ n pi nɛ́npɔ́. Kǎh pɩ sɛ̃́, Fɛsitusi mó núkú Pole tyɩ́ wɛ kõ mɔ yãh Akripa tyɩ́ nɛ ńkɛ̃́, Felikisi-á dĩ́ yɔ́ dah mɔ sah wɔ̀kɔ̀npèkè sɔkɔ ásõ̂. 
Tɛ́ páh pɔ yuku Yerusalɛmɩ kwil, sárká syínwìlkìpu yõ̀tãm ànɛ̂ Nsyifunɔ yèwàhnɩ̀-á mɛ pɔ n tɔkɔ ǹnɛ n kɔ̃ pé tyɩ́, tɛ́ mɛ kélɛ n yah n kɔ̃ pé tyɩ́ nɛ, pé à mɔ kũ kɔ̃. 
Ǹtɛ páh kè yo pé tyɩ́ nɛ, kè náh n pɩ sɛ̃́ pépi Wrome tãn wɛ̃̀kɩ̀ sɔkɔ. Pé náh wɔ̀kɔ̀npèkè sɔkɔ nɛynɛ n mɔ kũnɛ n kɔ̃, nɛ́ pé náh wɛ̃̀kɩ̀lɛ à kɔ̃, à ǹ yo yo tɛ̃̀ yo névye yĩ́ĩ́ yah, nónó-á tɔkɔ à kɔ̃. 
Képah sɔkɔ, kénɛ no mɛ pɔ ànɛ̂ péwɔlɛ ásõ̂ sɔkɔ. Péwɔ náh sõ̀ mɛ ké tyɩ́ nɛ pé tir mɛ̀ sah mo. Képah-á, ké dĩndal ńtɛ̃nɛ pé núkú yuku tukey tùkè-ǹsah sɔkɔ, tɛ́ nɛ pè pɔ kénɛ dĩ́nɛ. 
Péwɔ tɛ̃̀-á sõ̀ mɛ n sõ nɛ, névye nónó-á tɔkɔ à kɔ̃, pǎh kɩ pɔ tyítúkúnì ǹgbnɔ duke duke kwãh n pi nɛ, sépi-á à pɩ. Ǹtɛ pǎh pɔ, kè náh pɩ sɛ̃́. 
Pǎh mɛ pɔ yékã̂b yísyɔ́ ólɛ n pɩ ǹnɛ pé yɛ̃́ngbɛ̃́ Liyel wɛ̃̀kɩ̀ tyɩ́ sɔkɔ ànɛ̂ Yesu yɔ́ tyɩ́ sɔkɔ nɛ̂-á lésõ ku, tɛ́ Pole tɔ̃ mɛ n yo nɛ, wǎh mɛ min sɔkɔ. 
Kǎh pɩ sɛ̃́, péwɔ náh sõ̀ pɩ̃, páh nɛ yĩ mɛ̀ dùkù wah n pi sɔ̃́. Képah sɔkɔ, pé mɛ Pole piki yah nɛ, à se kɩ kyɩ Yerusalɛmɩ kwil, ǹ tukey kyɩ tuke nɛ́npɔ́ nɩ? 
Ǹtɛ Pole mɛ nɛ, páh mɛ ké tyɩ́ nɛ pé tɛ̃ ásõ̂, pé tyi yah mó pɔ kyɩ wah Wrome kwíltãm yõ̀tɛ̃̀ Sesar tyɩ́. Kǎh pɩ sɛ̃́, pé mɛ nɛ pè à dah mɔ sah wɔ̀kɔ̀npèkè sɔkɔ, fɔ́ɔ́ kè pɔ nɔ ǹ tɛ́kɩ́nmɔnɔ tyɩ́ Sesar tyɩ́. 
Képah sɔkɔ, Akripa mɛ yo Fɛsitusi tyɩ́ nɛ, péwɔ tɔ-á mɛ ké tyɩ́ nɛ dĩ́ mɛ̀ pɔ ǹ tyi yãh pé noh. Képah tɛ̃̀nɛ, Fɛsitusi mɛ nɛ ńkɛ̃́, ńsõ̂ ńtɛ̃ ó sɔkɔ, wǎh kɩ pɔ ǹ tyi yãh à kɩ noh. 
Dĩndallɛ, Akripa ànɛ̂ Berenisi pé mɔ wil yõ̀tɔ̃rɩ́ flɔ sɔkɔ, tɛ́ sràsyíbí yõ̀tãm ǹgbnɔ ànɛ̂ kwil nɛ́-ǹgbnɔ mɛ pé sõ̀ tɛ̃, pè yuku tukey tùkè-ǹsah wɔ̀kɔ̀ sɔkɔ. Képah sɔkɔ, Fɛsitusi pè kɔ̃ pè pɔ Polelɛ. 
Wǎh pɔ, Fɛsitusi mɛ de wɛy sɔkɔ tɛ́ nɛ, yõ̀tɛ̃̀ dĩ́ Akripa, ǹmɔ ànɛ̂ nónó pól-á mɛ pélɛ ásõ̂ sɔkɔ yɔ̀, pé pól-á mɛ dĩ́ mɛ̀nɛ n yah kɛ̀. Nsyifunɔ tɔ́wû pól-á pɔ wil pé tyɩ́ ǹ tyɩ́ sɔkɔ Yerusalɛmɩ, ànɛ̂ fɔ́ɔ́ ásõ̂ tɔ sɔkɔ. Pǎh mɛ gbo mɔ tũ nɛ, à náh yɩ̃nɛ sàhnɔ́lɛ min sɔkɔ póllɛ. 
Ǹtɛ péwɔ tɛ̃̀ wɔ náh tir yɔ́ wɛ ǹ tyípɩ́nɩ̀ sɔkɔ, nɛ̂-á yɩ̃nɛ ǹ kònmɔnɔlɛ. Tɛ́ wǎh fyɔ̀ ǹ gblɔ̌y tyɩ́ dahbɩ Wrome kwíltãm yõ̀tɛ̃̀ Sesar gbõ̀, képah-á pé nɛ, páh kɩ à tɛkɩ mɔ à kɔ̃. 
Tɛ́ tir nɛ̂ tɛ̃̀ tɔ́tɔ́-á pé kɩ wãrkɩ, kɩ à tɛkɩ mɔ n pi Wrome kwíltãm yõ̀tɛ̃̀ Sesar tyɩ́, pé náh mɛ képah pɩ̃. Képah-á pé pɔ ǹnɛ pé tyɩ́ ànɛ̂ ǹmɔ ńtɛ̃ tɔ tyɩ́ ǹ wahlɛ yõ̀tɛ̃̀ dĩ́ Akripa. Páh nɛ, pàh kɩ à piki yah tɛ, pé tɛ̃ wɛ tir yɔ́ wɛ n wãrkɩ ǹ tyɩ́ sɔkɔ, pé à tɛkɩ n mɔ Sesar tyɩ́. 
Péwɔ tɛ̃̀ tyɩ́ ké yah náh, à wɔ̀kɔ̀npèkè sɔkɔ nɛy tɛkɩ mɔ Wrome kwíltãm yõ̀tɛ̃̀ tyɩ́, tɛ́ à náh mó tɔ̃ tɛ̃̀ tyi yah wɛ yo tɛkɩ mɔ ǹ tyɩ́, nónó yĩn-á pè tɔkɔ à kɔ̃.
