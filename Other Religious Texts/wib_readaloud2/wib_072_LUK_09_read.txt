Likyi.
9.
Sṍ yɔ́lɛ, Yesu ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye tuke mɔ, tɛ́ tǎhkɩ̀ ànɛ̂ tɛ̃́nwɛnɔlɛ pè kɔ̃ nɛ, pè sétã̀n póllɛ n yĩni, pè mó yámpúlɛ n dɛ n mɔ. 
Képah sɔkɔ, tɛ́ pè tɛkɩ mɔ nɛ, pè kyɩ Liyel tɔ̃́rɩ́ tyɩ́lɛ n yo n kɔ̃, pè mó yámpúlɛ n dɛ n mɔ. 
Tɛ́ tɔ̃ yo pé tyɩ́ nɛ, pè káh kwâh yɔ́ ńtɛ̃ tɔkɔ wɛ̃̀kɩ̀ kɔ yĩnnɛ. Pè káh ńsĩn tɔkɔ, pè káh pilki tɔkɔ, pè káh yõke tɔkɔ, pè káh pɔ́ tɔkɔ. Ànɛ̂ nɛy káh fwɔ̀mɔlɔ nínɔ́ tɔkɔ. Tɛ́ tɔ̃ nɛ: 
«Yâh kyɩ wɛ twah kéte nɛ̂ sɔkɔ, yé tɛ̃ képah sɔkɔ fɔ́ɔ́, kè pɔ nɔ yé yal gbĩ́ tyɩ́ nɛ́npɔ́. 
Tɛ́, nónó-à kyɩ yahle nɛ, yé káh twah pé tyɩ́, yâh n wil kénɛ kwil sɔkɔ, yé yé gbyẽ̂h nɛ̀ngbɔ̃ kwer n kwɛ nɛ́npɔ́. Képah kɩ pɩ gbáhdyâh dínsàhnɔ́lɛ pé tyɩ́.» 
Képah tɛ̃̀nɛ, pè mɛ sɔkɔ, à kyɩ n kɔ n kɔ̃ kwlo kwlo sɔkɔ, à nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yo n kɔ̃, tɛ́ mɛ yámpúlɛ n dɛ n mɔ ywɔ̃ ywɔ̃ pól sɔkɔ. 
Légbĩ́nɛ, Kalile kwil tãm yõ̀tɛ̃̀ ǹgbɛ̃nɛ, Yerɔdɩ ye sõ̀. Ǹmɔ-á Yesu tyípɩ́nɔ́ nɔ́pi pól noh gbĩ́ nɛ̂nɛ, túkù ńtɛ̃ mɛ n yo nɛ, Nsyɔ̃-á ǹ syɩ min wɛ, à wil lékyɩ̂ sɔkɔ. Túkù mó nɛ, ǹgbò Liyel tɛ̃ntõ̀ syínyo-ò Yeli-á pɔ wil mɔ. Túkù mó tɔ̃ nɛ, ǹgbò Liyel tɛ̃ntõ̀ syínyo-ò yɔ́-á yuku wil lékyɩ̂ sɔkɔ. Pǎh yo sɛ̃́ Yerɔdɩ tyɩ́, ǹ fɔkɔ fɛ̃kɩ. 
Képah sɔkɔ, à nɛ, tɛ́ páh se Nsyɔ̃ yõ kãm. Ǹtɛ, nɛ̂ tɛ̃̀ ó tyípɩ́nɔ́ se tɔ̃ pé n nohnɩ yɔ̀ nɩ? Kǎh pɩ sɛ̃́, à tyah Yesulɛ n yah n kɔ̃ nɛ, pé à wɛ. 
Yesu tɛ̃̀ntõbe-á pé syɩ pɔ ǹ tnɔ̂, pè pé tyípɩ́nɔ́ pól yãh à syah. À mɛ pè tɛ̃ yal pé yahlɛ, à yuku pélɛ kwil yɔ́ gbɔ̀pɔ, pè n ye Besayidalɛ. 
Ǹtɛ, tɔ́wû-á pɔ pé sɔ́kɔ́nɔ́ pɩ̃ sɛ̃́, pè pé nyáh sõ kyɩ nɛ́kwĩ́nkílɛ. Képah sɔkɔ, Yesu pè tɛ̃ pallɛ, mɛ Liyel tɔ̃́rɩ́ tyɩ́lɛ n yãh pé tyɩ́, tɛ́ mɛ yámpúlɛ n dɛ n mɔ, nónó-á sõ̀ mɛ lènɔlɛ n yah n kɔ̃. 
Ǹtɛ, gbĩ-á pɔ n yi, ǹ tõ̀ sã́hpú nɔ de ǹ tnɔ̂, tɛ́ mɛ yo ǹ tyɩ́ nɛ, ǹgbǒ gbɛ̃́nnɩ tyah sɔkɔ-á pélɛ ásõ̂ dɛ́! Ńkɛ̃́, wǎh náhkɩ tɔ́wû fĩn mɔ, pè kyɩ sɛ̃́-ńtã̂n ànɛ̂ yõke yah kɔ̃ gbɔ̀plɔ kwlo ànɛ̂ yèpìyey sɔkɔ. 
Képah sɔkɔ, Yesu yo pé tyɩ́: «Yépi yõke yah pè kɔ̃ pè yõ yé gblɔ̌y kɛ̀!» Pè mɛ à syah nɛ, búrn plɔ́ kwlɔ, ànɛ̂ yɛ̃́nkyɔ plɔ́ nínɔ́ ó-á mɛ pé tyɩ́ dɛ́! Tĩ̂, à se mɛ ké tyɩ́ nɛ, pépi kyɩ yõke dwe pɔ nɛ́kwĩ́nkí nɔ́pi pu nɩ? 
Tɛ́ díbí sõ̀ nɔ nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãm kénkɔ̃lɛ nɛ́npɔ́. Képah sɔkɔ, Yesu mɛ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè no kɔ̃ pè kɔ̃ tɛ̃ tĩ̀nkìnɔ́lɛ nɛ́pĩ̂ tikiníní pɔ́ gbãm gbãmnɛ. 
Pè kè pɩ, wǎh yo sɔ̃́, à no pól kɔ̃ pè kɔ̃ tɛ̃. 
Képah tɛ̃̀nɛ, Yesu mɛ búrn plɔ́ kwlɔ, ànɛ̂ yɛ̃́nkyɔ plɔ́ nínɔ́ nɔ́pi tɔkɔ, tɛ́ ǹ yah yɔ̃ yĩ̂nyõ, à Liyel ni sé yõ, tɛ́ sè kwɔkɩ, à ǹ tõ̀ sã́hpú kɔ̃ nɛ, pè syi tɔ́wû kɔ̃. 
Képah-á pɩ, no pól yõ tɔ̃. Tɛ́ pè pé yṍnkwlɔnɔ tuke mɔ, vyɔ́ gbɔ̃ninɔ́ yĩ́ĩ́nɛ. 
Sṍ yɔ́lɛ, Yesu sõ̀ mɛ ǹ gbɛ kyɩ tɛ̃ nírílɛ n pɩ, ǹ tõ̀ sã́hpúlɛ ǹ tnɔ̂. Képah sɔkɔ, à mɛ pè piki yah nɛ: «Névye tɛ̃̀ tyɩ́, nɛ̂ mɛ ńmɔlɛ?» 
Pè mɛ à syah nɛ, túkù-á nɛ, Wèlnwìlkì-ò Nsyɔ̃-á ǹnɛ. Túkù mɛ nɛ, Yeli-á ǹnɛ. Túkù mó nɛ, ǹgbò gbĩ Liyel tɛ̃ntõ̀ syínyo-ò lékó yílkínmɔnɔ-á. 
Pǎh yo sɛ̃́, à pè syah nɛ: «Tɛ́, yépi tɛ̃̀ tyɩ́, nɛ̂ mɛ ńmɔlɛ?» Képah tɛ̃̀nɛ, Pyɛrɩ mɛ nɛ ńkɛ̃́: «Krista yé álɛ, Liyel-á nɛ, páh kɩ nɛ̂ tɛkɩ mɔ.» 
Wǎh yo sɛ̃́, Yesu pè kah sni nɛ, pé káh kè yo nɛy syah, 
tɛ́ tɔ̃ yɔ̃ sɔkɔ yo pé tyɩ́: «Névi Pi yɩ̃nɛ à yèvyãhrɩ pɩ náhnáh. Yèwàhnɩ̀, ànɛ̂ sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú kɩ à yahle n pi. À kɩ ko mɔ, tɛ́ ké sèpĩ̀n tɔ̃́-ò sṍlɛ, à kɩ ǹ syɩ min wɛ.» 
Képah náh, à yo no pól tyɩ́: «Névi nɛ̂-à mɛ ké tyɩ́ nɛ, pé ńmɔ sõ̀ tɛ̃, tɛ̃̀ ǹ gblɔ̌y dyɔ yĩ́ĩ́ tyi yɛ, sõmɛsṍ à mó nɛ mɛ ǹ syɩ syɩ dahnɩ twah, ń sõ̀ tɛ̃. 
Yah, névi nɛ̂-à nɛ ńkɛ̃́, páh n to n pwah kũ yah, tɛ̃̀ náh min wɛ n pi. Ǹtɛ, nɛ̂-à kũ fɛ̃ syi ńmɔ yĩnnɛ, tɛ̃̀ wɔ min kɩ pwah. 
Névi nɛ̂-á ńmɔ, ànɛ̂ ń wɛy fɛ̃̀nsyinɔ yah sen ku, ńmɔ Névi Pi tɔ kɩ sen ku nɛ, mé náh tɛ̃̀ pɩ̃, mâh sṍ nɛ̂nɛ pɔ ń yõ̀tɔ̃rɩ́ sɔkɔ, ànɛ̂ ń sú tyɩ́, ànɛ̂ yĩ̂nyõ tɛ̃ntõnè tyɩ́ sɔkɔ. Kǎh pɩ sɛ̃́, yõ se mɛ ké tyɩ́, névi-à kèkõyṍ tahrɩ pól wɛ, tɛ́ n tɛkɩ n pi ǹ min sɔkɔ, kɩ ǹ gblɔ̌y tyɩ́ yɩkɩ? 
Gbɩ yõ, mé kélɛ n yo yé tyɩ́ nɛ: névye nónó-á mɛ ásõ̂, pé túkù kɩ Liyel tɔ̃́rɩ́ wɛ yĩ́nɛ, tɛ́ kɩ mɛ min sɔkɔ.» 
Kénɛ wɛlɔ yóntɛnɔ náh, sèpĩ̀n kwǎrtɔ̃nɔ kénkɔ̃-á sɔkɔ, Yesu Pyɛrɩ, ànɛ̂ Nsyɔ̃, ànɛ̂ Nsyakɩ tɛ̃ sɔkɔ yɔ̃lɩ yɔ́ yõ nɛ, pé kyɩ nírí pɩ. 
Wǎh mɛ nírí sɔkɔ gbĩ́ nɛ̂nɛ, ǹ yèpɔ dùkù vi mɔ, tɛ́ ǹ flɔ piki prɛ́yy. 
Nɛ́npɔ́ swɔ, díbí nɛ́pĩ̂ nímí núkú wil, mɛ yã́hkɩ̀lɛ n yãh ànɛ̂ Yesulɛ. 
Moyisi ànɛ̂ Yeli ye sõ̀ pɔ wil mɔ yĩ̂nyõ tɔ̃́rɩ́ sɔkɔ, tɛ́ mɛ n yãh Yesulɛ, wǎh kɩ ǹ tõ̀ pɩ n tũ n pi sɔ̃́, ǹ kúnɔ́ sɔkɔ Yerusalɛmɩ. 
Tɛ́ légbĩ́nɛ, dɔy sõ̀ mɛ Pyɛrɩ ànɛ̂ pé no tyɩ́ fɛ̃. Pǎh dɔ yuku, pè Yesu tɔ̃́rɩ́ dùkù wɛ ànɛ̂ díbí nɛ́pĩ̂ nímí nɔ́pilɛ yĩ́nnɔ́lɛ ǹnɛ. 
Képah náh, nɛ́pĩ̂ nímí nɔ́pi-á n yal Yesu tnɔ̂ gbĩ́ nɛ̂nɛ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, pé tɛ̃̀nɔ́-á kɩ plɛ ásõ̂ dɛ́! Páh kɩ yèpyenɔ tɔ̃́nɔ́ pɩ. Ńkɛ̃́ ǹmɔ tyɩ́-á núkúlɛ, Moyisi tyɩ́ mɛ núkúlɛ, Yeli tyɩ́ mɛ núkúlɛ. Tɛ́, à náh sõ̀ pɩ̃, tir nɛ̂-á à mɛ n yo. 
Wǎh mɛ wɛ́kɩ́nɔ́ sɔkɔ sɛ̃́, ńyĩ́rkì yɔ́ núkú wil fob mɔ pé yõ. Légbĩ́nɛ, tíkí de Yesu tõ̀ sã́hpú sɔkɔ. 
Képah sɔkɔ, yĩ́npɔ̃́ yɔ́ wil Liyel tyɩ́ ńyĩ́rkì mɛ̀ sɔkɔ, mɛ n yo nɛ ńkɛ̃́: «Ǹmɔ nɛ̂-á yɔ̀, ǹmɔ ye ń pídĩ́nɛ, máh nɛ̂ yah wilki. Yé nɛ n nohnɩ ǹ tyɩ́!» 
Pǎh wɛy mɛ̀ noh ńyãh sɔkɔ, nɛy náh tɔ̃ wɛ nɛ́npɔ́, nɛ́ Yesu ó náh. Kǎh pɩ sɛ̃́, ǹ tõ̀ sã́hpú pé vyɛ̃y vike tɛ̃ syɩ́syɩ́. Tɛ́ légbĩ́nɛ, pǎh tyi nónó wɛ, pè náh mɛ sé sɔkɔ yɔ́ ńtɛ̃nɛ n yãh nɛy tyɩ́. 
Dendallɛ, pǎh tiki yɔ̃lɩ yõ, tɔ́wû ǹgbɛ̃ yɔ́ pɔ Yesu yohnɩ. 
Tɛ́ kénɛ tɔ́wû sɔkɔ, dĩ́ yɔ́ sõ̀ mɛ, à tyah n de n syi nɛ, kwɔ́-ò dĩ́, páh ǹnɛ n ni, à yah dah mɔ pé pídĩ́ tyɩ́ sɔkɔ, ǹmɔ núkúní-á pé tyɩ́, 
tɛ́ sétõ yɔ́-á mɛ n yuku n pɔ ǹ yõ. Kàh yuku ǹ yõ, à n de n syi nɛ́npɔ́ swɔ, ànɛ̂ kě à kah n yike pélpèl, tɛ́ gbèfɔ̀kɔ̀píkínì n nɛ n wil ǹ vyãh. Ànɛ̂ kè náh ǹnɛ n fĩn n mɔ súú. Kàh pópó ǹnɛ n fĩn mɔ, kě à fõh n kah n mɔ. 
Képah-á, pé pɔ ǹ tõ̀ sã́hpú ni nɛ, pè sétõ mɛ̀ yĩni, tɛ́ pè náh tɛ̃ wɛ. 
Wǎh yo sɛ̃́, Yesu mɛ yo pé tyɩ́ nɛ, nɛ́pêl, ànɛ̂ Liyel yõ sɛ̃́-ńkɛ̃̂ névye nɔ́pi! Pé se yɩ̃nɛ, pé pé nɩ tɛ̃, pé tɛ̃ pélɛ, pé pɔ syɩ gbĩ́ nɛ̂ tyɩ́ nɩ? Ńkɛ̃́, à pɔ ǹ pídĩ́nɛ ásõ̂. 
Piri-á núkú n nɔ n de Yesu tnɔ̂, sétõ à kwel min sétáh, tɛ́ à yike kègbɩlɛ. Ǹtɛ, Yesu mɛ dírí kõ sétõ yõ, tɛ́ piri dɛ mɔ, tɛ́ ǹnɛ ǹ sú gbõ̀ dahbɩ. 
Képah tɛ̃̀nɛ, no pól gbã̀n du Liyel tɛ̃́npnɛ̀nɔ́ tyɩ́lɛ. Ǹtɛ, no pól-à sõ̀ mɛ gbɛ̃̀ndùnì sɔkɔ sɛ̃́ ǹ tyípɩ́nɔ́ nɔ́pi pól yĩnnɛ, à mɛ yo ǹ tõ̀ sã́hpú tyɩ́: 
«Nɛ̂-á mé n tyah yónɔ́lɛ yé tyɩ́, yé kè noh pallɛ: Névi Pi kɩ n dahbɩ n pi névye gbõ̀.» 
Ǹtɛ, pè náh sõ̀ mɛ wɛy mɛ̀ tyɩ́ yah wɛ. Kè sõ̀ wɛkɩ wɛ́npǐllɛ pé tyɩ́ nɛ, pè káh ké yõ pɩ̃. Tɛ́ ké dùkù píkínyahnɔ tíkí sõ̀ mɛ pè tɛ̃ Yesu tyɩ́. 
Képah sɔkɔ, tõ̀ sã́hpú tyah yékã̂blɛ n pɩ nɛ, pé pɩ̃ nɛ̂ tɛ̃̀-á kal pépi sɔkɔ. 
Tɛ́ Yesu-á pé fɔ́kɔ́sɔ̃nɔ pɩ̃ sɛ̃́, à piri yɔ́ tɛ̃ sah ǹ tnɔ̂, 
tɛ́ yo pé tyɩ́: «Nɛ̂-á tyɩ̃́ɩ̃ kal yépi pól sɔkɔ, ǹmɔ kal tɛ́lɔ́lɛ. Képah ye, névi nɛ̂-à piri mɛ̀ kénkɔ̃ tɛ̃́nɔ́ plɛ ńmɔ yĩnnɛ, ńmɔ ye tɛ̃̀ tɛ̃ plɛ. Ànɛ̂ névi nɛ̂ tɔ-à ńmɔ tɛ̃́nɔ́ plɛ, ń tɛ́kɩ́nmɔ-ò ye tɛ̃̀ tɛ̃ plɛ.» 
Képah sɔkɔ, Nsyɔ̃ mɛ Yesu syah nɛ, pé yõ̀tɛ̃̀, pépi-á ǹnɩ̂ wɛ, à mɛ sétã̀nnɛ n yĩni ǹ yĩn yõ, tɛ́ à náh se sõ̀ mɛ ǹ sõ̀ tɛ̃ pépilɛ wɛ̃ tyɩ́. Képah-á, pé nɛ, pé ǹnɛ n yĩni n mɔ ké pɩ́nɔ́lɛ. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́: «Névi nɛ̂-à mɛ yélɛ n fwó-ńkɛ̃̂nɛ, tɛ̃̀ mɛ yélɛ. Képah ye, yé káh à yĩni n mɔ ké pɩ́nɔ́lɛ.» 
Képah náh, Yesu yɔ̃̀ndɔkɔnsɔkɔ gbĩ́-á pɔ nɔ yĩ̂nyõ, à kè yah tɔkɔ yĩn núkú yõ nɛ ńkɛ̃́, páh kɩ kyɩ Yerusalɛmɩ. 
Kǎh pɩ sɛ̃́, à tɛ̃ntõ̀ névye tɛkɩ mɔ ǹ yahlɛ. Pépi mɛ wɛ̃̀kɩ̀ tɛ̃, à kyɩ de Samarɩ kwil yɔ́ sɔkɔ nɛ, pé kyɩ twáh-ńsah yah sah à kɔ̃. 
Ǹtɛ, wǎh sɔ̃́npɩ́lɛ ǹ yah mɔ n yuku Yerusalɛmɩ tĩ̀nnɛ, kǎh pɩ sɛ̃́, nɛ́npɔ́ tãn mɛ yahle nɛ, à káh twah pé tyɩ́. 
Yesu tõ̀ sã́hpú sɔkɔ Nsyakɩ, ànɛ̂ Nsyɔ̃-á képah noh sɛ̃́, pè nɛ, Tɛ̃̀ ǹgbɛ̃, à se mɛ ké tyɩ́ nɛ, pé yĩ̂nyõ nyàh ye mɔ no nɔ́pi yõ, kè tiki n pɔ pè sukɛ n mukuli nɩ? 
Pǎh yo sɛ̃́, Yesu ǹ yah vi pé tĩ̀nnɛ, à dírí kõ pé yõ. 
Képah náh, pè mɛ wɛ̃̀kɩ̀ tɛ̃, à sɔkɔ kwil yɔ́ sɔkɔ. 
Képah sɔkɔ, pǎh mɛ wɛ̃̀kɩ̀ yõ gbĩ́ nɛ̂nɛ, dĩ́ yɔ́ Yesu syah nɛ ńkɛ̃́, páh kɩ ǹ sõ̀ tɛ̃ n kɔ n kɔ̃ ǹnɛ, ǹ kɔ́-ńtã̂n pól sɔkɔ. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́: «Gbɛ̃́npãhlɔ tyɩ́, pɩ̃̀ mɛ. Kèpye tyɩ́, syɛ̀ mɛ. Ǹtɛ Névi Pi tɛ̃̀ tyɩ́, wɛ́nsɛ̃nmɩ̀-ǹsah ńtɛ̃ náh mɛ.» 
Tɛ́ mɛ pyě yo ǹnɩ̂ tyɩ́: «Ń sõ̀ tɛ̃.» Kénɛ dĩ́ mɛ à syah nɛ, à yɛ, pâh pɔ pé sú lékó ǹgbò yɩ̃ tɛ. 
Wǎh yo sɛ̃́, Yesu mɛ à syah nɛ: «Yɛ lékyɩ̂ kɔ̃, pè pé lékyɩ̂lɛ n yɩ̃, tɛ́ áwɔ mó kyɩ Liyel tɔ̃́rɩ́ tyɩ́lɛ n yo n kɔ̃.» 
Képah sɔkɔ, ǹnɩ̂ yɔ́ mó yo Yesu tyɩ́ nɛ, pé Tɛ̃̀ ǹgbɛ̃, páh kɩ ǹ sõ̀ tɛ̃. Tɛ́ à yɛ pé kɔ̃, pé kyɩ ké wɛ̃̀kɩ̀ ni pé gbô tãn tyɩ́. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́ nɛ: «Névi nɛ̂-à nyeh wɛkɩ tɛ̃ swãhlɛ n wɔ, tɛ́ n yah náhlɛ, tɛ̃̀ náh fɛ̃ Liyel tɔ̃́rɩ́ tõ̀ pɩ n pi.»
