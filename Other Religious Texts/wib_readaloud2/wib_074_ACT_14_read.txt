Tyípɩ́nɩ̀.
14.
Képah sɔkɔ, Pole ànɛ̂ Barnabasɩ-á yĩni tɛ̃ Yikonyumi kwil, pè tɔ̃ de yuku Nsyifunɔ Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ. Tɛ́ tyah Liyel wɛynɛ n yo, fɔ́ɔ́ Nsyifunɔ ànɛ̂ nónó náh pɩ Nsyifunɔlɛ, pé náhnáh vi mɔ pɩ Yesu yõ sɛ̃́púlɛ. 
Ǹtɛ Nsyifunɔ sɔkɔ nónó-á sõ̀ yahle sɛ̃́nɔ́lɛ Yesu wɛy yõ, pépi nɛ́káhbɩ́ kurki mɔ Yesu yõ sɛ̃́pú yõ. 
Ǹtɛ képah fyé yõ, Pole ànɛ̂ Barnabasɩ tɔ̃ tɛ̃ à mo Yikonyumi sɔkɔ. Pé gbõ̀ sàhnyĩnɔ sõ̀ mɛ Tɛ̃̀ ǹgbɛ̃ yõ pè kɔ̃ pè n wɛkɩ nɩ̀sãhlɛ. Tɛ́ à sõ̀ mɛ pé yah tɛ̃ pè kɔ̃ pè gbǐl tyilɛ n pɩ, gbǐl nónó-á n wɛ̃kɩ nɛ, pǎh wɛlɔ nónónɛ n yo Liyel pèpɔrɩ pɩ́nɔ́ tyɩ́ sɔkɔ, gblɔ-á sélɛ. 
Ǹtɛ kwil névye gbɛkɩ. Pé túkù sõ̀ mɛ mɔ Nsyifunɔlɛ, túkù mó mɛ mɔ Yesu tɛ̃ntõ̀ névyelɛ. 
Képah sɔkɔ, Nsyifunɔ ànɛ̂ nónó náh pɩ Nsyifunɔlɛ, à tahlɩ pé yõ̀tãmnɔ́ tyɩ́, pè wɛ̃ tyɩ́ mɔ nɛ, pé Pole ànɛ̂ Barnabasɩ fõh, pé mó pè minki ko dyêhlɛ. 
Ǹtɛ Pole ànɛ̂ Barnabasɩ-á pɔ pɩ̃ ké sɔkɔ, pè to sɔkɔ Lisitir kwil, ànɛ̂ Dɛrɩbɩ kwil, ànɛ̂ sé gbɔ̀plɔ kwlo sɔkɔ, Likawoni gbɛ̃̀nyah. 
Tɛ́ yɔ̃ sɔkɔ, à Nɩ̀vɩ̀nɩ̀ tirlɛ n yo n kɔ̃ nɛ́npɔ́. 
Tɛ́ dĩ́ yɔ́ lésõ mɛ Lisitir kwil, gbyẽ̂hnɛ kúnɔ́lɛ. À sõ̀ dubi sèkùkùlɛ, à náh tɛ̃ wɛ kɔ yah. 
Tɛ́ sõ yɔ́lɛ, Pole-á n wɛkɩ, kénɛ dĩ́ sõ̀ mɛ ǹ wɛynɛ n nohnɩ. Képah tɛ̃̀nɛ, Pole tyah dĩ́nɛ n yah díkídíkí. Tɛ́ wǎh kè wɛ nɛ Liyel yõ sɛ̃́nɔ́-á mɛ dĩ́ mɛ̀ tyɩ́ kɩ fɛ̃ dɛ́nmɔnɔ wɛ Liyel tyɩ́, 
à yo ǹ tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: «Yuku n yĩn tómm á gbyẽ̂h yõ!» Képah sɔkɔ, dĩ́ núkú yɔkɔ syɩkɩ ǹ gbyẽ̂h yõ, à tyah n kɔ. 
Tɔ́wû-á képah tɛ̃̀ pɩ́nɔ́ wɛ sɛ̃́ Pole tyɩ́, pè tyah n yo wòhòh Likawoni tãn wɛy sɔkɔ nɛ ńkɛ̃́, tã̀n-á sé viri pɩ névyelɛ à tiki pɔ pé wrɔ́ dɛ́! 
Képah sɔkɔ, pè tõ̀ nɛ̂-á pè n ye Nsewusilɛ, pè képah yĩn dyɩ Barnabasɩ kɔ̃. Tɛ́ wɛy-á sõ̀ mɛ Pole wɔ gbõ̀ sɔkɔ sɛ̃́, pè kèyǎh yĩn dyɩ ǹmɔ kɔ̃, nɛ̂-á pè n ye Yɛrɛmɛsɩlɛ. 
Tɛ́ tõ̀ nɛ̂-á pè n ye Nsewusilɛ, ké wɔ̀kɔ̀ sõ̀ mɛ kwil dé-ńsah yah sɔkɔ. Ké lékwén nɛ́tɛ̃̂ dĩ́ mɛ nyéhpyɛ́nɩ̀ tɛ̃ pɔ kénɛ tõ̀ wɔ̀kɔ̀ yah ànɛ̂ yî fĩnɛ. Tɛ́ ǹmɔ ànɛ̂ tɔ́wû sõ̀ mɛ ké tyɩ́ nɛ, pé gbètòh mɔ Barnabasɩ ànɛ̂ Pole yõ. 
Ǹtɛ Barnabasɩ ànɛ̂ Pole-á képah tɛ̃̀ noh sɛ̃́, pépi nónó-á Yesu tɛ̃ntõ̀ névyelɛ, pè pé fwɔ̀mɔlɔ yɔrɩ tɔkɩ yèkwɔrɩ tɛ̃̀ tyɩ́, tɛ́ to de tɔ́wû nɩyṍ mɛ yo tũ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: 
«Á no, sɔ̃́ mɛ tɛ́ yé képahlɛ n pɩ? Névye ye ápi tɔlɛ, yépi tyɩ́ kɔ̃lɛ dò! Nɩ̀vɩ̀nɩ̀ tir ye á pɔ yó-ńsah yé tyɩ́ nɛ, yé yõ kɛ̃́ fal fal tã̀n nɔ́pi milki, yé mó vi mɔ Liyel wɛ̃̀kɩ̀ tɛ̃, ǹmɔ nɛ̂-á min nɛ́tɛ̃̂nɛ. Ǹmɔ ye mɔ yĩ̂nyõlɛ, ànɛ̂ sétáhlɛ, ànɛ̂ pnɛ yɔlɛ, à tahlɩ sé sɔkɔ yíyìn pól tyɩ́. 
Ǹgbòlɛ, Liyel mɛ̀ ye sõ̀ yɛ kwlo kwlo névye pól kɔ̃ pè n kɔ pé dyɔ dyɔ wã̀l yõ. 
Ǹtɛ à mɛ ǹ gblɔ̌y dùkùlɛ n wɛ̃kɩ sɛ̃́ gbĩ́mɛgbĩ́ ǹ pèpɔrɩ pɩ́nɔ́ sɔkɔ. À mɛ pũ̀nɛ n pɩ n mɔ yélɛ n kɔ̃ à n wil yĩ̂nyõ, tɛ́ yé sèmukɔ kɔ̃ sè n pɛkɩ, tɛ́ yõke náhnáhlɛ yélɛ n kɔ̃ ànɛ̂ nɩ̀vɩ̀nɩ̀nɛ.» 
Ǹtɛ wɛlɔ nɔ́pi fyé yónɔ́ yõ, Yesu tɛ̃ntõ̀ névye yah vyãh tɔ́wû yĩ́nínmɔ-ńsah nɛ, pè káh gbètòh mɔ pé yõ. 
Ké légbĩ́ sɔkɔ, Nsyifunɔ yísyɔ́ wil Antyɔsyɩ kwil, Pisidi gbɛ̃̀nyah, ànɛ̂ Yikonyumi kwil, à pɔ nɛ́npɔ́. Pépi pɔ tɔ́wû vi tɛ̃ mɔ Polepi yõ, pè kɔ̃ pè yuku Polelɛ n minki dyêhlɛ nɛ páh ǹnɛ n min n ko. Képah náh, pè à yɔrɩ wilki sah kwil náh tĩ̀nnɛ, tɛ́ mɛ n sõ nɛ wǎh ku. 
Ǹtɛ Yesu tõ̀ sã́hpú-á pɔ tuke mɔ ǹ tnɔ̂, à à mɔ gbɩ̃ gbĩ́ nɛ̂nɛ, à mɛ yuku tɛ́ de kwil sɔkɔ. Tɛ́ ké dĩndallɛ, à sɔkɔ ànɛ̂ Barnabasɩlɛ Dɛrɩbɩ kwil. 
Képah tɛ̃̀nɛ, pǎh kyɩ yĩni tɛ̃ Dɛrɩbɩ kwil, pè Nɩ̀vɩ̀nɩ̀ tir yo nɛ́npɔ́ tãn tyɩ́. Pé náhnáh mɛ sɛ̃ Yesu yõ, à pɩ ǹ tõ̀ sã́hpúlɛ. Képah náh, Polepi pé syɩ syɔ̃ Lisitir kwillɛ, ànɛ̂ Yikonyumi kwillɛ, à kyɩ de Antyɔsyɩ kwil, Pisidi gbɛ̃̀nyah sɔkɔ. 
Tɛ́ nɩ̀sãh ànɛ̂ sṍré ye sõ̀ pè n mɔ n kɔ̃ Yesu tõ̀ sã́hpú sɔkɔ nɛ, pè gbah n tɛ̃ pé sõnɔ sah yĩ́ĩ́ Yesu yõ gbĩ́mɛgbĩ́. Pè sõ̀ mɛ n yo pé tyɩ́ nɛ ńkɛ̃́, yèvyãhrɩ náhnáh dénwìlnɔ́ sɔkɔ-á pé kɩ de Liyel tɔ̃́rɩ́ mɛ́-ńsah sɔkɔ. 
Tɛ́ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ nónó-á sõ̀ mɛ kwlo nɔ́pi pól sɔkɔ, Polepi sõ̀ yah tãm yísyɔ́ yah wilki sé sɔkɔ, à sah sé yõ. Tɛ́ níkì tɛ̃ à nírí pɩ pé tyɩ́, à pè dahbɩ Tɛ̃̀ ǹgbɛ̃ Yesu gbõ̀, pǎh sɛ̃ ǹmɔ nɛ̂ yõ. 
Képah náh, pè Pisidi gbɛ̃̀nyah kɔ kah, à kyɩ de Pãnfili gbɛ̃̀nyah sɔkɔ. 
Nɛ́npɔ́ sɔkɔ, pè Liyel wɛy yo Pɛrsyɩ kwil tãn tyɩ́, tɛ́ pyě yuku Atali kwil. 
Pǎh n yal nɛ́npɔ́, pè de krótókè ǹgbɛ̃ sɔkɔ, à yuku Antyɔsyɩ kwil, Siri gbɛ̃̀nyah sɔkɔ. Kénɛ kwil sɔkɔ ye sõ̀ pè pè dahbɩ Liyel gbõ̀ sɔkɔ, tõ̀ mɛ̀ pɩ yĩnnɛ, nɛ̂-á pè pɩ tɛ yɔ̀. 
Pǎh yĩni tɛ̃ Antyɔsyɩ sɔkɔ, pè Yesu yõ sɛ̃́pú tĩ̀nkì ye tuke. Tɛ́ Liyel-á pé yah tɛ̃, à ǹ tõ̀ pɩ dùkù nɛ̂nɛ, pè kè yãh pé tyɩ́; à tahlɩ wǎh ǹ yõ sɛ̃́nɔ́ wɛ̃̀kɩ̀ yɩkɩ mɔ névye tɔ kɔ̃ sɔ̃́, nónó náh pɩ Nsyifunɔlɛ. 
Képah náh, Polepi tɛ̃ pélɛ Antyɔsyɩ sɔkɔ, à mo.
