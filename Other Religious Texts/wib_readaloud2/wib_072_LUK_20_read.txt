Likyi.
20.
Yesu-á sõ̀ mɛ névyelɛ n kwɔ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, mɛ nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yo sɛ̃́, sṍ yɔ́lɛ, sárká wìlkìpu yõ̀tãm, ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú, ànɛ̂ yèwàhnɩ̀ pɔ wil ǹ tyɩ́, 
tɛ́ à syah nɛ, à yo pé syah, nɛ̂ gbõ̀ yõ sɔkɔ se, à n pɩ tyi nɔ́pilɛ, nɛ̂ se à kɔ̃ tɛ̃́nwɛnɔ mɛ̀nɛ nɩ? 
À mɛ yo pé tyɩ́: «Ńmɔ tɔ kɩ píkínyahnɔ núkú pɩ yé tyɩ́: Yé yo ń syah, 
nɛ̂-á tõ Nsyɔ̃nɛ, à pɔ nolɛ n wel n wilki? Liyel se, tĩ̂ névye se?» 
Pè mɛ yo wɛ̃ tyɩ́ nɛ, páh kɩ nɛ ńkɛ̃́, Liyel-á tõ ǹnɛ, wǎh kɩ pé syah nɛ, tɛ́ kwâh nɛ̂ se pɩ, tɛ́ pé náh sɛ̃ ǹ yõ nɩ? 
Ǹtɛ, névye sɛ̃́nyĩnɔ-á mɛ tɛ ké yõ nɛ, Liyel tɛ̃ntõ̀ syínyo-ò-á Nsyɔ̃nɛ, páh kɩ mó tɔ̃ nɛ ńkɛ̃́, névye-á tõ ǹnɛ, no pól-á kɩ dyêhlɛ pé minki ko. 
Kǎh pɩ sɛ̃́, pè à syah nɛ, pé náh pɩ̃ nɛ̂-á tõ ǹnɛ. 
Pǎh yo sɛ̃́, Yesu tɔ mó pè syah nɛ, kàh pɩ sɛ̃́, tɛ̃́nwɛnɔ nɛ̂-á pé kɔ̃ pé tyi nɔ́pilɛ n pɩ, pé náh mó kè yo pè syah n pi. 
Képah náh, Yesu tyah sèmìrkì yɔ́lɛ n yãh névye tyɩ́ nɛ: «Dĩ́ yɔ́ ye lésõ dìvɛ̃ sésân swãh kõ, tɛ́ kélɛ tɛ̃̀ntõbe gbõ̀ dahbɩ, tɛ́ sɔkɔ kɔ́lɔ́ sɔkɔ à kyɩ mo. 
Sé pye gbɛ gbĩ́-á pɔ nɔ, à tõ̀npɩ yɔ́ tɛkɩ mɔ dìvɛ̃ sésân swah tɛ̃̀ntõbe nɔ́pi tyɩ́ nɛ, pè pé tyɩ́ dìvɛ̃ sésân pyelɛ à kɔ̃. Ǹtɛ, tɛ̃̀ntõbe nɔ́pi mɛ tõ̀npɩ mɛ̀ ko, tɛ́ pé syɩ à tɛkɩ mɔ ǹ gbõ̀ kɔ́kɔ́lɛ. 
Képah sɔkɔ, sètɛ̃̀ dĩ́ tɔ̃ ǹ syɩ tõ̀npɩ yɔ́ tɛkɩ mɔ. Pè tɔ̃ ǹmɔ tɔ ko, tɛ́ à gbyɔ, tɛ́ pé syɩ à tɛkɩ mɔ ǹ gbõ̀ kɔ́kɔ́lɛ. 
À tɔ̃ ǹ syɩ tõ̀npɩ tɔ̃́-ò tɛkɩ mɔ. Pè tɔ̃ ǹmɔ tɔ pɩ yɔ, tɛ́ à yĩni. 
Kǎh pɩ sɛ̃́, sètɛ̃̀ dĩ́ nɛ, sɔ̃́ se pé kɩ pɩ nɩ? Páh kɩ pé nɩ̀dɛ̀ pídĩ́ wɔ tɛkɩ mɔ. Ǹmɔ wɔlɛ, gbɔ̀kénpɩ, pǎh kɩ yõ̀yén sah ǹ yõ. 
Ǹtɛ, dìvɛ̃ sésân swãh tɛ̃̀ntõbe nɔ́pi-á fyɔ̀ ǹ pídĩ́ wɛ, pè núkú yo wɛ̃ tyɩ́ nɛ ńkɛ̃́, ǹ gbɛ̃̀yõ-á yɔ̀. Pé ye à ko mɔ, gbɛ̃kɩ mó pɩ pépi tyɩ́lɛ. 
Képah tɛ̃̀nɛ, pè à yɔrɩ wilki mɔ dìvɛ̃ sésân swãh kahnɩnɛ, tɛ́ à ko mɔ. Núkúnúkú tɛ̃̀nɛ, sɔ̃́ mɛ sètɛ̃̀ dĩ́ kɩ pɔ pɩ pélɛ wɛ́? 
À náh pɔ dìvɛ̃ sésân swãh tɛ̃̀ntõbe nɔ́pi ko mɔ, tɛ́ kɩ kè dahbɩ n pi yísyɔ́ gbõ̀ di?» No-á kénɛ wɛlɔ noh sɛ̃́, pè nɛ, póllɛ, képah wɔ náh n pɩ n pi! 
Ǹtɛ, Yesu mɛ pè yah, tɛ́ yo pé tyɩ́: «Liyel wɛy-á nɛ̂ yo nɛ ńkɛ̃́: ‹Wɔ̀kɔ̀ mɔ́pú-á dáhkɩ̀ nɛ̂ milki, képah ye pɔ pɩ wɔ̀kɔ̀ kyɩ̃ yõ dáhgbɩ́lɛ›, képah yõ nɛ̂nɛ yépi tɛ̃̀ tyɩ́? 
Névi nɛ̂-à wil syɩkɩ kénɛ dáhkɩ̀ yõ, tɛ̃̀ kɩ kwɔkɩ. Tɛ́, kàh se mó tɔ̃ syɩkɩ nɛ̂ yõ, kè kɩ tɛ̃̀ pyãhrɩ.» 
Képah sɔkɔ, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú, ànɛ̂ sárká wìlkìpu yõ̀tãm ǹgbnɔ sõ̀ pɩ̃ yĩ́ĩ́ tɛ nɛ, pépi yõ-á Yesu mɩ sèmìrkì mɛ̀nɛ. Képah ye, pè sõ̀ mɛ ǹ tɛ̃́-ńsahlɛ n yah n kɔ̃ nɛ́npɔ́ swɔ, tɛ́ névye tíkí pè tɛ̃. 
Képah sɔkɔ, pè tyah Yesu kyɔ̀mɩ̀nɔ́lɛ. Pè yĩ́nvínpɩbɩ yísyɔ́ tɛkɩ mɔ ǹ tyɩ́. Pè pɔ pé gblɔ̌y pɩ nɛ́gblɔ́lɛ. Pè sõ̀ mɛ ké tyɩ́ nɛ, à pilki ǹ vyãh wɛy sɔkɔ, pé kɔ̃ pé à wɛ tɛ̃ kyɩ dahbɩ kwil yah tɛ̃́-ò gbõ̀, tɔ̃́rɩ́ ànɛ̂ tɛ̃́nwɛnɔ-á mɛ nɛ̂ tyɩ́. 
Képah tɛ̃̀nɛ, pè píkínyahnɔ yɔ́ pɩ Yesu tyɩ́ nɛ, kwɔ́-ò dĩ́, páh pɩ̃ nɛ, wǎh tyi nónónɛ n yo, tɛ́ kwɔ́nɔ́ nónónɛ n pɩ, gblɔ kɔ́kɔ́-á sélɛ. Á náh névyelɛ n wah, ànɛ̂ wǎh Liyel wɛ̃̀kɩ̀lɛ névyelɛ n kwɔ ké gbɩ yõ. 
Ǹtɛ, à yo pé syah, ńkɛ̃́, ké wɛ̃̀kɩ̀ se mɛ kɔ̃́nɔ́lɛ nɛ, pé mɔ́rɩ́lɛ n mɔ Wromɛnɔ yõ̀tɛ̃̀ ǹgbɛ̃ Sesarɩ tyɩ́, tĩ̂ ké wɛ̃̀kɩ̀ náh mɛ kɔ̃́nɔ́lɛ? 
Tɛ́ Yesu-á pé yĩ́nvǐ pɩ̃ sɛ̃́, à yo pé tyɩ́: 
«Yé pɔ pɔ́ pɔ́ núkúlɛ, mé yah ǎtrɛ. Nɛ̂ yah mírkí, ànɛ̂ ǹ yĩn mɛ wã́rkɩ́nmɔnɔlɛ ké yõ yɔ̀?» Pè mɛ yo ǹ tyɩ́ nɛ, Sesarɩ tyɩ́-á. 
Yesu mɛ yo pé tyɩ́: «Kàh pɩ sɛ̃́, yé Sesarɩ tyɩ́lɛ Sesarɩlɛ n kɔ̃, yé mó Liyel tyɩ́lɛ Liyel kɔ̃.» 
Kǎh pɩ sɛ̃́, pè náh pílkí-ńsah wɛ ǹ wɛy sɔkɔ, à à tɛ̃ névye yĩ́ yah. Tɛ́ ǹ tyi yah yónɔ́ pyě mɛ pé gbã̀n du, pè mɛ tɛ̃ syɩ́syɩ́. 
Sadusɛ̃nɔlɛ, nósyɔ́ ye, nónó-á nɛ, lékyɩ̂ náh min sã́n wɛ n pi. Pé nɔ de Yesulɛ, tɛ́ à piki yah nɛ, 
pé kwɔ́-ò dĩ́, Moyisi-á kɩ wãrkɩ sah pé kɔ̃ nɛ ńkɛ̃́, sétɛ̃̂ nɛ̂-à ku pi wɛ́-ńkɛ̃̂nɛ, tɛ́ gbɛ̃̀ mɛ tɛ̃̀ tyɩ́, gbɛ̃̀ mɛ̀-á yɩ̃nɛ à pɛ́kúsè mɛ̀ tɔkɔ, à mó wɛ̃́npì dubi à kɔ̃ ǹ nɛ́nkyɩ́ náhnɛ. 
Ǹtɛ, kèkùrkì núkú dĩ́npì nɛ́pĩ̂ kwǎrninɔ yísyɔ́-á lésõ mɛ. Nɛ́nkyɩ́ tɛ̃̀ mɛ sê tɔkɔ, tɛ́ ku, tɛ́ à sah píkã́nnɛ. 
Ǹ sõ̀ syɩ mɛ pi mɛ̀ tɔkɔ. 
Képah náh, tɔ̃́-ò mɛ à tɔkɔ. Kè n pɩ n yuku sɛ̃́ fɔ́ɔ́, pé nɛ́pĩ̂ kwǎrnímí pól à tɔkɔ. Tɛ́, pé pól pɔ ku, tɛ́ à sah píkã́nnɛ. 
Képah náh, pi mɛ̀ tɔ pɔ ku. 
Ǹtɛ, pi mɛ̀-á de yah sɛ̃́ nɛ́pĩ̂ kwǎrnímí nɔ́pi tyɩ́-à, lékyɩ̂ min sã́n pɔ́nwɛ sṍlɛ, à kɩ n pɩ n pi pé sɔkɔ nɛy nɛ̂ tyɩ́lɛ? 
Pǎh yo sɛ̃́, Yesu yo pé tyɩ́: «Kèkõyṍ mɛ̀ tɛ̃̀ wɔ sɔkɔ, sétɔ́kɔ́kɩ̀ mɛ, ànɛ̂, pɛ́dékè mɛ. 
Ǹtɛ, Liyel-á nónó kyɔmɩ wɛ nɛ, pǎh yɩ̃nɛ dénɔ́lɛ kèkõyṍ sã́n sɔkɔ, tɛ́ kɩ min sã́nnɛ pè kɔ̃ n pi lékyɩ̂ min sã́n pɔ́nwɛ sṍlɛ, pépi náh syɔ́ tɔkɔ n pi, pè náh n de n pi pɛbɩ tyɩ́ póllɛ. 
À kɩ pè wilki kũ sɔkɔ, kɩ min sã́nnɛ pè kɔ̃. Pè kɩ pɩ yĩ̂nyõ tɛ̃ntõnè tyɩ́ kɔ̃lɛ: Pè náh tɔ̃ n ku n yah n pi. Pè ye pɩ Liyel pyelɛ. 
Moyisi tɔ ye kè yo wah yĩ́ĩ́ nɛ ńkɛ̃́, lékyɩ̂-á kɩ pɔ pé syɩ min sã́n wɛ. Ǹ sɛ́bɛ́y fɩ̃́ nɛ̂-á dukuni tyɩ́lɛ n yo nɛ̂-á sõ̀ nyàh lépɔ̃́lɛ ké sɔkɔ, képah sɔkɔ ye à Tɛ̃̀ ǹgbɛ̃ Liyel ye ‹Abrahmɩ Liyellɛ, ànɛ̂ Yisakɩ Liyellɛ, ànɛ̂ tɔ̃ Yakɔbɩ Liyellɛ›. 
Liyel tyɩ́, no pól mɛ vyelɛ. Képah ye, lékyɩ̂ Liyel náh ǹnɛ, vye Liyel ye.» 
Wǎh yo sɛ̃́, nɩ̀sãh náh tɔ̃ sõ̀ mɛ pé tyɩ́, kɩ à piki yah. Képah ye, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú yísyɔ́ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, wǎh wɛkɩ plɛ. 
Képah sɔkɔ, à mɛ yo pé tyɩ́: «Á kɩ yo sɔ̃́ nɛ, Davidi náh nɛy ó-á Kristalɛ nɛ̂-á Liyel nɛ, páh kɩ tɛkɩ mɔ? 
Tɛ́ ǹmɔ Davidi ńtɛ̃ mó tɔ̃ kè yo ǹ yɩ̀gbɩ̀nɩ̀ sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: ‹Tɛ̃̀ ǹgbɛ̃ Liyel ye kè yo ńmɔ Tɛ̃̀ ǹgbɛ̃ tyɩ́ nɛ ńkɛ̃́: «Pɔ n kɔ̃ n tɛ̃ ń gbõ̀yõ̀ sõ̀lɛ, 
fɔ́ɔ́ kè pɔ nɔ gbĩ́ nɛ̂-á mé kɩ á sépɛ́bɩ́ pɩ á gbáhsúkílɛ.›» 
Ǹtɛ, Davidi ńtɛ̃ wɛ à ye ǹ TƐ̃̀ ǸGBƐ̃NƐ à kɩ tɔ̃ pɩ sɔ̃́, tɛ́ kɩ pɔ n pɩ ǹ náh nɛy ólɛ?» 
Képah náh, à yo ǹ tõ̀ sã́hpú tyɩ́ ǹ wɛy nóhpú pól yĩ́ yah nɛ: 
«Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́púlɛ, fwɔ̀mɔ̀ngbnɔ mɔ́nkɔnkɔ̃nɔ tyɩ́ nɔ pé tyɩ́. Pè n nɛ n mɛ ké tyɩ́ nɛ, no nɛ n tɔ̃ n sah n wɛkɩ pé tyɩ́ tɔ́wû tùkè-ǹtã̀n sɔkɔ. Tɛ̃̀-ǹtã̀n gbro tyɩ́ nɔ pé tyɩ́ Liyel ní-ńtã̂n vɩ̀ sɔkɔ, ànɛ̂ yé yṍ-ńtã̂n sɔkɔ. Yé yé gblɔ̌ynɛ n kyɔmɩ pélɛ! 
Tɛ́ pépi ó ye tɔ̃ n vi n tɛ̃ pɛ́kúsyɔlɛ, à pé gbã̀n yîlɛ n syi, ǹ tɛ́ nírílɛ n pɩ yõ̀sɛ̃̀kɩ̀lɛ, ǹ névye yahlɛ n pil nɛ, páh mɛ tómm. Ǹtɛ, pé tukey kɩ n pɩ n pi képékèyɔ̂, à kal no tɛ́lɔ́ tyɩ́lɛ.»
