Nsyɔ̃.
6.
Képah náh, Yesu sɔkɔ Kalile yɔ́-ǹgbɛ̃ kahnɩnɛ. Pè tɔ̃ sõ̀ kélɛ n ye Tiberiyadɩ yɔ́-ǹgbɛ̃nɛ. 
Tɛ́ Yesu-á gbǐl nónó pɩ sɔkɔ à yámpú nónó dɛ mɔ tɔ́wû wɛ, sépi yĩnnɛ, tɔ́wû ǹgbɛ̃ sõ̀ mɛ ǹ sõ̀ tɛ̃. 
Képah sɔkɔ, Yesu mɛ dɔkɔ kyɩ kɔ̃ tɛ̃ yɔ̃lɩ yɔ́ yõ ànɛ̂ ǹ tõ̀ sã́hpúlɛ. 
Légbĩ́nɛ, Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh sõ̀ mɛ de n tɛ. 
Tɛ́ Yesu-á ǹ yah yɔ̃ yah, à tɔ́wû ǹgbɛ̃ wɛ pè mɛ n pi ǹ tnɔ̂. Képah tɛ̃̀nɛ, à Filipi piki yah nɛ, pé se kɩ wɛ ńyãh yõke dwe dwe tɛ̃̀nɛ kɩ tɔ́wû mɛ̀ kɔ̃ pè kɩ yõ nɩ? 
Ǹtɛ fyã̀hnyahnɔ́ ye sõ̀ Yesu n fyãh n yah Filipi fɔkɔlɛ. Ńkɛ̃́, wǎh sõ̀ nɛ̂ pɩ n pi, à sõ̀ kè pɩ̃ tɛ. 
Wǎh yo sɛ̃́, Filipi mɛ à syah nɛ, deniye pɔ́ plɔ́ lègbãm (200) fyé búrn dwénɔ́lɛ, kàh sɔ̃́ yĩki tyɩ̃́ɩ̃ tyɩ̃́ɩ̃, kè náh n yɩ̃nɛ n pi pélɛ. 
Képah sɔkɔ, Yesu tõ̀ sã́hpú sɔkɔ nɛ̂-á Simo-Pyɛrɩ gbɛ̃̀ndĩ́ Andrelɛ, ǹmɔ mɛ yo Yesu tyɩ́ nɛ, 
búrn plɔ́ kwlɔ́ ànɛ̂ yɛ̃́nkyɔ plɔ́ nínɔ́-á mɛ dĩ́npir yɔ́ tyɩ́ ásõ̂. Ǹtɛ sépi se kɩ tir nɛ̂ pɛkɩ nɛ́kwĩ́nkí nɔ́pi dùkù tyɩ́ nɩ? 
Légbĩ́nɛ fílkì sõ̀ mɛ dò nɛ́npɔ́. Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ pè no pól kɔ̃ pè kɔ̃ tɛ̃ sétáh. Díbí nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãm (5 000) kénkɔ̃ ye sõ̀ mɛ pé sɔkɔ nɛ́npɔ́. 
Képah sɔkɔ, Yesu mɛ búrn nɔ́pi tɔkɔ, tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ sè dibi névye tyɩ́, nónó-á sõ̀ mɛ nɛ́npɔ́. Wǎh yɛ̃́nkyɔ tɔ tɔkɔ, sɛ̃́ntɛ̃̀ ó ye tɔ̃ à pɩ, tɛ́ sè dibi no kɔ̃, pé pól wɛ pǎh sõ̀ mɛ ké tyɩ́ sɔ̃́. 
Pǎh yõ tɔ̃ gbĩ́ nɛ̂nɛ, Yesu mɛ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè sé tɛ́lɔ́ tuke n mɔ, pè káh n yɛ kèyǎh ńtɛ̃ pɔ n yɩkɩ. 
Búrn plɔ́ kwlɔ́ táhnkwlɔnsàhnɔ́ nɛ̂ túkù-á sõ̀ mɛ wũ tɛ̃, pè mɛ sè kwel mɔ, tɔkɔnɩ gbɔ̃ninɔ́ yĩ́ĩ́nɛ. 
Névye-á Yesu gbǐl pɩ́nɔ́ wɛ sɛ̃́, pè nɛ: «Liyel tɛ̃ntõ̀ syínyo-ò nɛ̂-á sõ̀ yɩ̃nɛ à pɔ kèkõyṍ, ǹmɔ ye dĩ́ mɛ̀nɛ tɔ́tɔ́.» 
Képah sɔkɔ, Yesu pɩ̃ nɛ pǎh kɩ pɔ nɛ, páh péwɔlɛ n tɛ̃ n kãhlɩ n pɩ yõ̀tɛ̃̀nɛ. Kǎh pɩ sɛ̃́, à tɔ̃ yal pé tnɔ̂ à sɔkɔ yɔ̃lɩ yõ ǹ gblɔ̌y núkúnínɛ. 
Tɛ́ ńvɩ̃́ lísyi-á pɔ nɔ, Yesu tõ̀ sã́hpú tiki yuku yɔ-ǹgbɛ̃ vyãh yah, 
à kyɩ de krótókè yɔ́ sɔkɔ, à tyah n yuku yɔ́nkahnɩnɛ, Kapɛrnayumi kwil tĩ̀nnɛ. Lékã̂h sõ̀ mɛ mɔ tɛ, tɛ́ Yesu náh tɔ̃̌nɩ̀ sõ̀ mɛ pɔ pé tnɔ̂. 
Ǹtɛ yã́hpékè yɔ́ sõ̀ mɛ kõ mɛ ninɛ n fũ n dɔkɔ. 
Képah sɔkɔ, pǎh krótókè to kyɩ yĩni tɛ̃ kílónɔ́ kwlɔ́ kénkɔ̃ tyɩ́, pè Yesu wɛ à mɛ n kɔ n pi yɔ yõ̀tah pé krótókè tĩ̀nnɛ. Kǎh pɩ sɛ̃́, tíkí pè tɛ̃. 
Ǹtɛ Yesu mɛ yo mɔ pé tyɩ́ nɛ: «Ńmɔ ye, yé káh tíkí pɩ.» 
Képah sɔkɔ, pè sõ̀ mɛ ké tyɩ́ nɛ pé à twah krótókè sɔkɔ, tɛ́ kè núkú yĩni tɛ̃ yɔ́ngbalɩ tyɩ́, ńyãh-á sõ̀ pè n yuku. 
Képah sɔkɔ, dĩndallɛ tɔ́wû nɛ̂-á sõ̀ wũ pé ńnɩ́nɩ́ mɛ́-ńsah, pè kè kyɔmɩ wɛ nɛ krótókè núkú ó-á ńnɩ́nɩ́nɛ mɛ. Ǹtɛ Yesu tõ̀ sã́hpú ó-á sõ̀ de ké sɔkɔ à sɔkɔ, Yesu tɔ náh sɔkɔ pélɛ. 
Tɛ́ krótónè yísyɔ́ mó mɛ wil Tiberiyadɩ kwil sɔkɔ, mɛ pɔ n de ńyãh sɔkɔ-á sõ̀ Tɛ̃̀ ǹgbɛ̃ Yesu yõke tɔkɔ tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ kélɛ tɔ́wû mɛ̀ kɔ̃ pè yõ. 
Ǹtɛ tɔ́wû-á pɔ pɩ̃ gbĩ́ nɛ̂nɛ nɛ Yesu náh mɛ, ǹ tõ̀ sã́hpú tɔ náh mɛ nɛ́npɔ́ sɔkɔ, pè núkú de krótónè sɔkɔ, à sɔkɔ Yesu yáhnkɔ̃-ńsah Kapɛrnayumi kwil sɔkɔ. 
Képah sɔkɔ, pǎh kyɩ à wɛ yɔ́-ǹgbɛ̃ kahnɩnɛ, pè à syah nɛ, kwɔ́-ò dĩ́, gbĩ́ nɛ̂ se à pɔ ásõ̂ nɩ? 
Ǹtɛ Yesu mɛ pè syah nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́ nɛ, yáh gbǐl tyi nónó wɛ, sépi-á nɛ̂nɛ n wɛ̃kɩ, képah yĩn náh yé n yah n kɔ̃ ńnɛ dɛ́! Yõke nɛ̂-á yé wɛ yõ tɔ̃, képah yĩn ye yé n yah n kɔ̃ ńmɔlɛ. 
Yé káh tõ̀lɛ n pɩ yõke yĩnnɛ, nɛ̂-á kɩ n yɩkɩ n pi. Yé tõ̀lɛ n pɩ yõke yĩnnɛ nɛ̂-á kɩ n tɛ̃ n pi kṹmm, tɛ́ tyɛ́-ńkɛ̃̂ minnɛ n kɔ̃. Névi Pídĩ́ ye kɩ yé kɔ̃ n pi kénɛ yõkelɛ, ǹmɔ nɛ̂-á Sú Liyel kè wɛ̃kɩ nɛ páh pé tɔ̃́rɩ́lɛ à kɔ̃.» 
Képah tɛ̃̀nɛ, pè à piki yah nɛ, tyi nónó se pé yɩ̃nɛ pé pɩ, pé mó Liyel tõ̀ wɛ pɩ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ: «Liyel-á nɛ̂ tɛkɩ mɔ, yáh kɩ sɛ̃ ǹmɔ yõ, képah ye Liyel tõ̀lɛ.» 
Pè mɛ yo ǹ tyɩ́ nɛ, gbǐl tir nɛ̂ se ǹmɔ kɩ pɩ pé wɛ̃kɩ, pé kɔ̃ pé sɛ̃ ǹ yõ nɩ? Tir nɛ̂ se à kɩ pɩ nɩ? 
Pépi náh tãn-á yõke yõ gbɛ̃́nnɩ sɔkɔ, nɛ̂-á pè n ye mânnɛ, à yɩ̃nɛ sɔ̃́-á kè wãrkɩ sah nɛ: «À ye yõkelɛ pè kɔ̃ pè yõ nɛ̂-á wil yĩ̂nyõ.» 
Ǹtɛ Yesu mɛ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́ nɛ, Moyisi náh kɔ̃ yélɛ yõke mɛ̀nɛ nɛ̂-á wil yĩ̂nyõ dɛ́, ń Sú Liyel ye n kɔ̃ yélɛ yõke gbɩlɛ, nɛ̂-á n wil yĩ̂nyõ. 
Kénɛ yõke nɛ̂-á Liyel n kɔ̃ nɛ̂-á wil yĩ̂nyõ à tiki pɔ, képah ye minnɛ kèkõyṍ névyelɛ n kɔ̃.» 
Képah sɔkɔ, pè nɛ pé yõ̀tɛ̃̀, à yõke mɛ̀nɛ pélɛ n kɔ̃ gbĩ́mɛgbĩ́ kɛ̀. 
Képah tɛ̃̀nɛ, Yesu mɛ yo pé tyɩ́ nɛ: «Ńmɔ ye yõke mɛ̀nɛ, nɛ̂-á minnɛ n kɔ̃. Névi nɛ̂-à pɔ ńmɔ tyɩ́ tɛ́ sɛ̃ ń yõ, kwɛkɩ ànɛ̂ níkì náh tɔ̃ tɛ̃̀ tɛ̃ n yah n pi. 
Tɛ́ mé ye kè yo tɛ yé tyɩ́ nɛ yáh ń wɛ. Ǹtɛ yé náh mɛ sɛ̃ ń yõ. 
Ńmɔ náh wil yĩ̂nyõ à tiki pɔ ń dyɔ yĩ́ĩ́ tyi pɩ yĩnnɛ dɛ́! Nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ-á mɛ nɛ̂ tyɩ́, képah pɩ́-ńsah ye mé pɔ. Képah ye, Sú Liyel-à nónó póllɛ ń kɔ̃, pè kɩ pɔ ń tyɩ́. Tɛ́ nɛ̂-à pɔ ńmɔ tyɩ́, mé náh à milki n pi póllɛ. 
Tɛ́ nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ-á mɛ nɛ̂ tyɩ́, képah ye yɔ̀ nɛ ńkɛ̃́, páh nónónɛ ń kɔ̃ mé káh n yɛ pé sɔkɔ yɔ́ ńtɛ̃ tyɩ́ yɩkɩ. Ǹtɛ mé mó n pɔ pé lékyɩ̂ yilki mɔ gbĩ̀yĩ̀kì tɛ́y sõlɛ. 
Tɛ́ yah, ńmɔ Sú-á mɛ nɛ̂ tyɩ́, képah ye yɔ̀ nɛ, névi nɛ̂-à ǹ yah mɔ ńmɔ tĩ̀nnɛ, ńmɔ nɛ̂-á Pídĩ́nɛ, tɛ́ sɛ̃ ń yõ, tɛ̃̀ tyɛ́-ńkɛ̃̂ min wɛ, tɛ́ ńmɔ mó tɛ̃̀ lékó yilki mɔ gbĩ̀yĩ̀kì tɛ́y sõlɛ.» 
Képah sɔkɔ, Nsyifunɔ tyah Yesu sèyòkèlɛ n yo wɛ̃ tyɩ́, wǎh sɔ̃́npɩ́lɛ sõ̀ yo nɛ péwɔ-á yõkelɛ, nɛ̂-á wil yĩ̂nyõ à tiki pɔ. 
Pè sõ̀ mɛ yo tũ nɛ, Nsyosɛfu pídĩ́ Yesu kó ǹnɛ nɩ? Ǹmɔ nɛ̂-á pé ǹ sú ànɛ̂ yṹnɛ n pnɛ, à se pɩ sɔ̃́ tɛ́ pɔ n yo núkúnúkú nɛ, páh wil yĩ̂nyõ à tiki pɔ nɩ? 
Képah sɔkɔ, Yesu yo pé tyɩ́ nɛ: «Yé káh ń sèyòkèlɛ n yo yé wɛ̃ wrɔ́. 
Ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ n pɔ n pi ńmɔ tyɩ́, nɛ́ Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ náh tɛ̃̀ yah tɛ̃. Tɛ́ ńmɔ kɩ mó tɛ̃̀ lékó yilki mɔ gbĩ̀yĩ̀kì tɛ́y sõlɛ. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel tɛ̃ntõ̀ syínyopu sɛ́bɛ́ynɔ́ sɔkɔ nɛ: ‹Liyel kɩ pé pól kwɔ.› Ǹtɛ névi nɛ̂-à Sú Liyel wɛy noh tɛ́ ǹ kwɔ́nɔ́ fɛ̃ syi, tɛ̃̀ n pɔ ńmɔ tyɩ́. 
Ǹtɛ ǹnɩ̂ yɔ́ ńtɛ̃ yĩ́ náh tɛ̃ yah Sú Liyel yõ, à wil ńmɔ tyɩ́ sɔkɔ, ńmɔ nɛ̂-á wil ǹ tnɔ̂. Ńmɔ ó wɛ ǹnɛ. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, névi nɛ̂-à sɛ̃ ń yõ, tɛ̃̀ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Ńmɔ ye yõkelɛ nɛ̂-á minnɛ n kɔ̃. 
Yé náh tãn ye yõke yõ gbɛ̃́nnɩ sɔkɔ, yõke nɛ̂-á pè n ye mânnɛ. Ǹtɛ képah fyé yõ, pé pól náh pɔ ku di? 
Tɛ́ yõke nɛ̂ tɛ̃̀ wɔ-á n wil yĩ̂nyõ à tiki pɔ, névi nɛ̂-à képah yõ, tɛ̃̀ náh tɔ̃ ku n yah n pi. 
Ńmɔ ye yõkelɛ, nɛ̂-á wil yĩ̂nyõ à tiki pɔ, tɛ́ minnɛ n kɔ̃. Névi nɛ̂-à kénɛ yõke yõ, tɛ̃̀ kɩ tyɛ́-ńkɛ̃̂ min wɛ. Yõke nɛ̂-á mé kɩ n kɔ̃ n pi, ń wil gbɛ̃́nsê ye. Mé ye kélɛ n kɔ̃ nɛ, kèkõyṍ névye min wɛ.» 
Képah tɛ̃̀ sɔkɔ, yékã̂b ǹgbɛ̃ de Nsyifunɔ wrɔ́. Pè sõ̀ mɛ n yo nɛ, dĩ́ mɛ̀ se kɩ fɛ̃ ǹ wil gbɛ̃́nsêlɛ pépi kɔ̃ sɔ̃́ yõkelɛ nɩ? 
Képah sɔkɔ, Yesu yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, yáh kɩ Névi Pídĩ́ wil gbɛ̃́nsê pɩ́-ńkɛ̃̂nɛ yé yõkelɛ, tɛ́ ǹ tṍ pɩ́-ńkɛ̃̂nɛ yé wɔ wɔ kwâhlɛ, yé náh min wɛ n pi dɛ́! 
Yõke kègbɩ ye ńmɔ wil gbɛ̃́nsêlɛ, wɔ wɔ kwâh kègbɩ mɛ ń tṍnɛ. Képah ye névi nɛ̂-à ńmɔ wil gbɛ̃́nsê pɩ ǹ yõkelɛ, tɛ́ ń tṍ pɩ ǹ wɔ wɔ kwâhlɛ, tɛ̃̀ ye tyɛ́-ńkɛ̃̂ min wɛ. Tɛ́ ńmɔ kɩ mó tɛ̃̀ lékó yilki mɔ gbĩ̀yĩ̀kì tɛ́y sõlɛ. 
Névi nɛ̂-à ńmɔ wil gbɛ̃́nsê pɩ ǹ yõkelɛ, tɛ́ ń tṍ pɩ ǹ wɔ wɔ kwâhlɛ, tɛ̃̀ kɩ mɛ ńmɔlɛ, tɛ́ ńmɔ tɔ kɩ mɛ tɛ̃̀nɛ. 
Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ ye min nɛ́tɛ̃̂nɛ. Ńmɔ mɛ min sɔkɔ ǹmɔ gbõ̀ yõ. Sɛ̃́ ye névi nɛ̂-à ń wil gbɛ̃́nsê pɩ ǹ yõkelɛ, tɛ̃̀ kɩ mɛ min sɔkɔ ńmɔ gbõ̀ yõ. 
Kénɛ yõke nɛ̂-á wil yĩ̂nyõ à tiki pɔ yɔ̀, kè náh mɛ núkúlɛ ànɛ̂ yé náh tãn-á nɛ̂ yõ gbɛ̃́nnɩ sɔkɔ, pépi nónó-á ku sɔkɔ. Ǹtɛ névi nɛ̂-à képah yõ, tɛ̃̀ kɩ tyɛ́-ńkɛ̃̂ min wɛ.» 
Yesu kénɛ tyi yo gbĩ́ nɛ̂-á lésõ à mɛ névyelɛ n kwɔ Kapɛrnayumi kwil, Nsyifunɔ Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ. 
Tɛ́ Yesu-á wɛlɔ nɔ́pi yo sɛ̃́, ǹ tõ̀ sã́hpú náhnáh tyah n yo wɛ̃ tyɩ́ nɛ ńkɛ̃́, ǹ tyi yónɔ́ nɔ́pi-á mɛ tah dɛ́! Nɛ̂ se kɩ fɛ̃ sè fɛ̃ syi nɩ? 
Ǹtɛ Yesu-á pɩ̃ nɛ pé tõ̀ sã́hpú-á mɛ pé sèyòkèlɛ n yo wɛy mɛ̀ tyɩ́ sɔkɔ, à pè syah nɛ: «Kè se kǎn pɩ yépi tɔlɛ? 
Tɛ́ yâh pyě pɔ Névi Pídĩ́ wɛ à yɔ̃ dɔkɔ n sɔkɔ wǎh wil ńyãh sɔkɔ wɛ́? 
Liyel Mirki ye min nɛ́tɛ̃̂nɛ, nòtɛ tir nɛ̂ mɛ névi kɩ fɛ̃ pɛkɩ ǹ gblɔ̌y? Ńmɔ-á wɛlɔ nónó yo yé tyɩ́, sè n wil Liyel Mirki tyɩ́, tɛ́ minnɛ n kɔ̃. 
Ǹtɛ túkù mɛ yé sɔkɔ, pè náh mɛ sɛ̃ ń yõ.» Tɛ́ yah, à tɔkɔ ké tyah gbĩ́ dùkù tyɩ́, Yesu sõ̀ ǹ yõ sɛ̃́-ńkɛ̃̂púlɛ n pnɛ, à tahlɩ nɛ̂ tɔ-á nɛ pɔ à dahbɩ n pi gbõ̀ sɔkɔ. 
Képah sɔkɔ, Yesu tɔ̃ nɛ: «Képah ye ń kɔ̃ mé kè yo nɛ ńkɛ̃́, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ n pɔ n pi ńmɔ tyɩ́, nɛ́ Sú Liyel náh ké wɛ̃̀kɩ̀lɛ à kɔ̃.» 
À tɔkɔ légbĩ́ tyɩ́, ǹ sõ̀ tɛ̃́pú náhnáh núkú pé syɩ ǹ náh, pè náh tɔ̃ mɛ ǹ sõ̀lɛ n tɛ̃. 
Képah sɔkɔ, Yesu mɛ yo ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí tyɩ́ nɛ ńkɛ̃́, tɛ́ pépi-á dò, pé tɔ kó pé syɩ n pi nɩ? 
Ǹtɛ Simo-Pyɛrɩ mɛ nɛ ńkɛ̃́: «Tyɛ́-ńkɛ̃̂ min wɛlɔ nɛ́tɛ̃̂ ye áwɔlɛ, nɛ̂ tyɩ́ mɛ ápi kɩ tɔ̃ yuku wɛ́, Tɛ̃̀ ǹgbɛ̃? 
Ápi tɛ̃̀ wɔ sɛ̃́nɔ́ mɛ á yõ, ànɛ̂ á pɩ̃ nɛ áwɔ-á Liyel wilki sah ǹ tyɩ́ yõ.» 
Képah tɛ̃̀nɛ, Yesu tɔ̃ yo pé tyɩ́ nɛ, péwɔ kó yah wilki pélɛ nɛ́pĩ̂ gbɔ̃nimínɛ nɩ? Ǹtɛ sétõ-á pé sɔkɔ nɛ́núkù yɔ́lɛ. 
Tɛ́ Nsyudasɩ ye sõ̀ Yesu n ye, ǹmɔ nɛ̂-á Simo Yisikariyɔtɩ pídĩ́nɛ. Yesu tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ yɔ́ ye sõ̀ ǹnɛ. Tɛ́ ǹmɔ ye kɩ sõ̀ pɔ Yesu dahbɩ n pi gbõ̀ sɔkɔ.
