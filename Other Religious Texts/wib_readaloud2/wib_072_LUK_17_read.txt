Likyi.
17.
Képah náh, Yesu yo ǹ tõ̀ sã́hpú tyɩ́: «Tyi nónó-á névilɛ n mɔ tyípékè pɩ́nɔ́ sɔkɔ, sépi náh fɛ̃ n yɛ wɛ́nɔ́lɛ. Ǹtɛ, névi nɛ̂ tyɩ́-á ké yõ wil, yèvyãhrɩ mɛ tɛ̃̀ yõ. 
Kè kɩ sõ̀ fṍnfṍ kal, pàh yínɔ́nan toh tɛ̃̀ yègbɩ, tɛ́ à min dah mɔ pnɛ yɔ sɔkɔ à kɔ̃ à káh pɔ ńmɔ yõ sɛ̃́-ò tyɩ̃́ɩ̃ yɔ́ ńtɛ̃ dah mɔ tyípékè sɔkɔ. 
Yé yé gblɔ̌y tɛ̃ dɛ́! Á yṹnpi-à pɩ de á tyɩ́, wɛkɩ ǹ yõ! Wàh ké nàhnpɩ̃ gbo, kè yɔ̃ mɔ à kɔ̃. 
Ànɛ̂ wàh pɩ de á tyɩ́ gbáhyɩ̃́ kwǎrninɔ gbĩ̀yĩ̀kì núkú sɔkɔ, tɛ́ ǹnɛ n syɩ n yo n pɔ á tyɩ́ fɔ́ɔ́ gbáhyɩ̃́ kwǎrninɔ nɛ, páh mɛ ké nàhnpɩ̃nɛ n gbo, sélɛ n yɔ̃ n mɔ ǹ tyɩ́.» 
Képah sɔkɔ, Yesu tɛ̃̀ntõbe à syah nɛ, pé Yõ̀tɛ̃̀, à wãn mɔ pépi Liyel yõ sɛ̃́nɔ́ yõ. 
Tɛ̃̀ ǹgbɛ̃ mɛ yo pé tyɩ́: «Gbɩ tɛ̃̀ yõ, yé Liyel yõ sɛ̃́nɔ́-à sõ̀ mɛ tyɩ̃́ɩ̃ ǹsɔ̃̀, mútárdɩ̀ pi fyé sɔ̃́, yé náhkɩ sõ̀ fɛ̃ yo sènsóké mɛ̀ tyɩ́ nɛ ńkɛ̃́, kè ké gblɔ̌y viki kyɩ sõ pnɛ yɔ sɔkɔ, tɛ́ kè náhkɩ kè pɩ yé yóré yõ.» 
Képah náh, Yesu yo pé tyɩ́: «Tõ̀npɩ yɔ́-à mɛ á tyɩ́, mɛ á swãhlɛ n wɔ, képah náh pɩ, á yípɛ̂ynɛ n fɛ̃... wàh wil swãh, à pɔ n de á wɛ, á se kɩ yo ǹ tyɩ́ nɛ, à sal n pɔ yõke yõ? 
Á náh pyě tɔ̃ n yo n pi ǹ tyɩ́ nɛ, à áwɔ yõke ǹgbò pɩ, képah náh, à mó flɔ yísyɔ́ mɔ pɔ, á ǹnɛ n tõ á yõke yõ gbĩ́nɛ, tɛ́ képah náh, à mó pyě pɔ yõ, à mó wɔ di? 
Tɛ́ tõ̀npɩ mɛ̀-á ǹ tõ̀ pɩ ké wɛ̃̀kɩ̀ yõ, á se kɩ ǹ yĩ́ngbɩ́ yo? Ǹ pɩ pɩ tõ̀ ó náh à pɩ di? 
Képah núkú ó ye yépi tɔ tyɩ́: Tyi nónó-á yah kɔ̃ yé tyɩ́, yâh sè pɩ tɛ, yé yo nɛ, tõ̀npɩbɩ ó-á yélɛ. Yáh sõ̀ yɩ̃nɛ yè nɛ̂ pɩ, képah ó-á yé pɩ.» 
Yesu-á sõ̀ mɛ n yuku Yerusalɛmɩ, à syɔ̃ sɔkɔ Samarɩ ànɛ̂ Kalile gbàh yõ. 
Wǎh kyɩ n de kwílpír yɔ́ sɔkɔ, díbí tótãm nɛ́pĩ̂ gbãm yísyɔ́ pɔ à yohnɩ. Pè sõ̀ mɛ yĩn fɩ́ɩ́, 
tɛ́ tyah n yo n mɔ ǹ tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀ Yesu, à pé yãm yah. 
Wǎh pè wɛ sɛ̃́, à yo pé tyɩ́ nɛ, pè kyɩ pé gblɔ̌ynɛ sárká wìlkìpu yõ̀tãm wɛ̃kɩ. Pǎh képah noh sɛ̃́, tɛ́ wɛ̃̀kɩ̀ tɛ̃ n yuku, pè dɛ. 
Pé sɔkɔ nɛ́núkù yɔ́-á kè wɛ à dɛ sɛ̃́, à ǹ syɩ Liyel yĩn gbilki pɔ Yesu tnɔ̂ yĩ́npɔ̃́ ǹgbɛ̃nɛ. 
À pɔ kɔ kwĩnki ǹ yah di sétáh Yesu yah, à wɛkɩ ǹ tyɩ́. Tɛ́ kénɛ dĩ́nɛ, Samarɩ nɛy ye sõ̀ ǹnɛ. 
Yesu-á à wɛ sɛ̃́, à nɛ: «Nɛ́pĩ̂ gbãm nɔ́pi pól náh dɛ di? Ǹtɛ, kwǎryãh nɔ́pi tɛ́lɔ́-á dò, pépi ńyãh? 
Nɛy náh wɛ pépi sɔkɔ à ǹ syɩ pɔ wɛkɩ Liyel tyɩ́, à wil nɛ́kã̂h mɛ̀ tyɩ́ sɔkɔ kɛ̀?» 
Tɛ́ mɛ ǹ syɩ yo kénɛ dĩ́ tyɩ́: «Yuku n nɛ n yuku! Á Liyel yõ sɛ̃́nɔ́ ǹ pwah mɔ.» 
Képah náh, Farisiyɛ̃nɔ Yesu piki yah nɛ, gbĩ́ nɛ̂ se Liyel tɔ̃́rɩ́ kɩ n pɔ n pi nɩ? À mɛ yo pé tyɩ́: «Liyel tɔ̃́rɩ́ náh n pɔ n pi kwâh yɔ́ sɔ̃́ nɛ̂-á n wɛ yĩ́nɛ. 
Ànɛ̂ tɔ̃, nɛy náh n yo n pi nɛ, kè wɛ yɔ̀, képah náh pɩ, kè wɛ yàhàh. Ǹtɛ, yé pɩ̃ nɛ ńkɛ̃́, Liyel tɔ̃́rɩ́-á mɛ yé wrɔ́.» 
Képah náh, Yesu tɔ̃ yo ǹ tõ̀ sã́hpú tyɩ́: «Sṍ yɔ́ mɛ n pi, Névi Pi wɛ dékè kɩ mɛ yé tɛ̃, tɛ́ yé náh à wɛ n pi. 
Pè kɩ yé syah nɛ, wǎh yɔ̀, képah náh pɩ, wǎh yàhàh. Ǹtɛ, yé káh n kyɩ, yé káh nɛ n sal. 
Sɔ̃́-á pũ̀ n yĩkiri yã̌l, à tɔkɔ yĩ̂nyõ yah yɔ́ tyɩ́, à kyɩ syɩ ké yah yɔ́ tyɩ́, sɛ̃́ntɛ̃̀ ye Névi Pi pɔ́nɔ́ kɩ pɩ. 
Ǹtɛ, Névi Pi ǹgbò yɩ̃nɛ à yèvyãhrɩ pɩ náhnáh, ànɛ̂ kwɛy gbĩ névye mó à yahle. 
Kǎh pɩ sɔ̃́ ǹgbòlɛ Nowe gbĩ sɔkɔ, sɛ̃́ tɔ ye kɩ pɩ Névi Pi syɩ́npɔ gbĩ́nɛ. 
Névye lésõ mɛ n yõ, tɛ́ mɛ n wɔ, tɛ́ mɛ syɔ́lɛ n tɔkɔ, tɛ́ mɛ n de pɛbɩ tyɩ́ fɔ́ɔ́, à pɔ syɩ sṍ nɛ̂-á Nowe de krótó-ǹgbɛ̃ sɔkɔ. Képah tɛ̃̀ sɔkɔ, pṹnpã́hkɩ̀ pɩ, ni pé pól yõ. 
Ànɛ̂, kè kɩ pɩ, kǎh pɩ sɔ̃́ ǹgbòlɛ Lɔtɩ gbĩ tɔ sɔkɔ: Névye lésõ mɛ n yõ, tɛ́ mɛ n wɔ, tɛ́ mɛ yídwénènɛ n dwe, tɛ́ mɛ yíyã̌mnɛ n yãm, tɛ́ mɛ swãhlɛ n tõ, tɛ́ mɛ vɩ̀lɛ n mɔ. 
Ǹtɛ, Lɔtɩ-á wil sṍ nɛ̂nɛ Sodɔmɩ sɔkɔ, Liyel nyàh lépɔ̃́ ànɛ̂ nyàh lépúkú tiki mɔ pé yõ pũ̀ sɔ̃́ n pɩ, à pé sukɛ ko. 
Sɛ̃́ntɛ̃̀ ó ye kè kɩ pɩ n sɔkɔ n pi Névi Pi syɩ́npɔ sṍlɛ. 
Ké lésõ-à pɔ, névi nɛ̂ wɛ gbɛ̃́ntĩ̂n yõ, tɛ́ vyɔ́ mɛ tɛ̃̀ tyɩ́ wɔ̀kɔ̀ sɔkɔ, à káh nɛ, páh n tiki n yuku sé tɔ́kɔ́-ńsah. Tɛ́ nɛ̂ tɔ-à mɛ swãh sɔkɔ, tɛ̃̀ káh nɛ, páh pélɛ n syɩ n yuku gbô. 
Yé yé dyɔ to kõ Lɔtɩ kyɩ tyɩ́lɛ kɛ̀! 
Sɛ̃́ ye, nɛ̂-à nɛ, páh pé minnɛ n koh n sah, tɛ̃̀ kɩ tɛkɩ ké sɔkɔ, ǹtɛ nɛ̂-à tɛkɩ ǹ min sɔkɔ, tɛ̃̀ wɔ min kɩ pwah. 
Mé kélɛ n yo yé tyɩ́ nɛ: kénɛ lékã̂hnɛ, nɛ́pĩ̂ nímí kɩ mɛ sɛ̃ sɛ̃ kwâh núkú yõ, núkú kɩ tɛ̃ sɔkɔ, tɛ́ ǹ ní-ò kɩ wũ. 
Syɔ́ nɛ́pĩ̂ nímí kɩ mɛ yîlɛ n nɔ wɛ̃ tyɩ́, núkú kɩ tɛ̃ sɔkɔ, tɛ́ ǹ ní-ò kɩ wũ.  
Díbí nɛ́pĩ̂ nímí kɩ mɛ swãh núkú sɔkɔ, núkú kɩ tɛ̃ sɔkɔ, tɛ́ ǹ ní-ò kɩ wũ.» 
Képah sɔkɔ, Yesu tõ̀ sã́hpú à piki yah nɛ, pé Yõ̀tɛ̃̀, ńyãh sɔkɔ se képah kɩ n pɩ n pi nɩ? À mɛ yo pé tyɩ́: «Kókè-à mɛ ńyãh, nɛ́npɔ́ sɔkɔ náh ǹkyɔ̀nkyɔ̀nɩ̀ wɛ̃ tuke n mɔ di?»
