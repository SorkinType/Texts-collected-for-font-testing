Likyi.
10.
Képah náh, Tɛ̃̀ ǹgbɛ̃ Yesu mɛ tõ̀ sã́hpú nɛ́pĩ̂ tikitɔ̃nɩ́ pɔ́ gbãm pɔ́ nínɔ́ yísyɔ́ yah wah, tɛ́ pè tɛkɩ mɔ nímí nímínɛ ǹ yah tĩ̀nnɛ, kwlo ànɛ̂ wley wley pól sɔkɔ, ǹmɔ-á sõ̀ mɛ ké tyɩ́ nɛ, pé kyɩ pé gblɔ̌y nónó sɔkɔ, 
tɛ́ yo pé tyɩ́ nɛ: «Sèmukɔ tɔkɔ tɔkɔ tɛ̃̀ mɛ dò, tɛ́ ké tõ̀npɩbɩlɛ kwéyy. Yé sètɛ̃̀ ni, à tõ̀npɩbɩ náhnáh tɛkɩ mɔ, ǹ sèmukɔ tɔkɔ yĩnnɛ. 
Yé nɛ n yuku! Tɛ́, yé mɛ á kɩ nɛ, képɔ́pî sɔ̃́, mé n tɛkɩ n mɔ gbɛ̃́npãhlɔ nɩyṍ. 
Ǹtɛ, yé káh pɔ́, ànɛ̂ pilki, ànɛ̂ lékyêh tɔkɔ. Yé káh yĩn wɛkɩ nɛy tyɩ́ wɛ̃̀kɩ̀ yah. 
Ànɛ̂, yâh kyɩ n de kéte nɛ̂ sɔkɔ, yé wɛy gbennɛ, yé yo nɛ ńkɛ̃́: ‹Yèvɩnɩ nɛ n mɛ kéte mɛ̀ sɔkɔ.› 
Kàh pɔ yɩ̃nɛ nɛy mɛ yèvɩnɩnɛ n yah n kɔ̃ kénɛ kéte sɔkɔ, yé yèvɩnɩ yénmɔnɔ mɛ̀ kɩ pɩ tɛ̃̀ wɔ tyɩ́lɛ, yé mó tɛ̃ nɛ́npɔ́. Képah-à mó pɩ́-ńkɛ̃̂nɛ, yé yèvɩnɩ yénmɔnɔ kɩ ké syɩ pɔ yé tyɩ́. 
Tõ̀npɩ ye yɩ̃nɛ ǹ tõ̀ pɩ́nɔ́ yõre wɛ́nɔ́lɛ. Képah ye, yâh twah kéte nɛ̂ sɔkɔ, pàh yé kɔ̃ yõke nɛ̂nɛ, yé kè yõ, ànɛ̂ pàh yé kɔ̃ wɔ wɔ kwâh nɛ̂nɛ, yé kè wɔ. Tɛ́, yé káh nɛ n vi n mɔ n twah n kɔ̃ gbéngbèn sɔkɔ. 
Yâh kyɩ de kwil nɛ̂ sɔkɔ, tɛ́ pè yé tɛ̃́nɔ́ plɛ, pàh kwâh nɛ̂nɛ yé kɔ̃, yé kè yõ. 
Yé mó yámpúlɛ n dɛ n mɔ, nónó-á mɛ nɛ́npɔ́. Yé mó nɛ n yo kénɛ kwil tãn tyɩ́ nɛ ńkɛ̃́, Liyel tɔ̃́rɩ́-á yĩni tɛ̃ tɛ pé tyɩ́. 
Ǹtɛ, yâh kyɩ de kwil nɛ̂ sɔkɔ, tɛ́ pè nɛ, yé káh twah pé tyɩ́, yé wil kyɩ pé tɔ́wû tùkè-ǹtã̀n sɔkɔ, yé mó yo nɛ, 
pé kwil nɛ̀ngbɔ̃ nɛ̂ ńtɛ̃-á tahlɩ yé gbyẽ̂h tyɩ́, yáh kɩ sè kwer kwɛ pè kɔ̃. Ǹtɛ, pè pɩ̃ nɛ, Liyel tɔ̃́rɩ́-á yĩni tɛ̃ tɛ. 
Mé kélɛ n yo yé tyɩ́ nɛ: Liyel tukey tuke sṍlɛ, Sodɔmɩ kwil fṍhnɔ́ ńtɛ̃ kɩ fàhfàh kal kènɛ kwil tyɩ́lɛ.» 
Yesu tɔ̃ yɔ̃ sɔkɔ yo kwlo nɔ́pi tyɩ́ sɔkɔ nɛ: «Yèvyãhrɩ mɛ yé yõ, Korasini ànɛ̂ Betisayida tãn! Gbǐl tyi nónó-á pɩ yé tyɩ́, sɛ̃́nkɔ̃-à sõ̀ mɛ pɩ Tiri ànɛ̂ Sidõ tãn tyɩ́, pé Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ di náhkɩ mɛ mo. Pè náhkɩ mɛ yèkwɔrɩ flɔ mɔ, tɛ́ náhkɩ mɛ pé piki syɩ̃lɛ pé tyípêl nàhnpɩ̃ gbónɔ́ sɔkɔ. 
Ǹtɛ, tukey tuke sṍlɛ, Tiri ànɛ̂ Sidõ fṍhnɔ́ kɩ pɩ fàhfàh kal yépi tyɩ́lɛ. 
Tɛ́, áwɔ-á, Kapɛrnayumi, á se wɛ nɛ, wáh kɩ yɔ̃ dɔkɔ yĩ̂nyõ? Póllɛ! Á kɩ tiki yuku fɔ́ɔ́ lékókwíl nyàh tɛ̃̀-ǹsah.» 
Tɛ́ tɔ̃ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Névi nɛ̂-à n nohnɩ yé tyɩ́, ńmɔ tyɩ́ ye tɛ̃̀ n nohnɩ, tɛ́ nɛ̂-à yé yahle, ńmɔ ye tɛ̃̀ yahle ànɛ̂ ń tɛkɩ mɔ́-ò tɔlɛ.» 
Képah náh, tõ̀ sã́hpú nɛ́pĩ̂ tikitɔ̃nɩ́ pɔ́ gbãm pɔ́ nínɔ́ nɔ́pi pé syɩ pɔ de nɩ̀vɩ̀nɩ̀ sɔkɔ, mɛ n yo Yesu tyɩ́ nɛ, pé Yõ̀tɛ̃̀, ǹmɔ yĩn yõ, sétã̀n ńtɛ̃-á mɛ n sah pépi yóré yõ! 
Képah tɛ̃̀nɛ, à mɛ yo pé tyɩ́: «Mé sõ̀ sétã̀n yõ̀tɛ̃̀ wɛ, à wil yĩ̂nyõ, à tiki kɛ núkú, à pɔ syɩkɩ pũ̀ sɔ̃́ yikiri yã̌l. 
Ǹtɛ, yé yah, mé tɛ̃́nwɛnɔlɛ yé kɔ̃: Yé kɩ n kɔ vyeh, ànɛ̂ mìntyɩ̃ yõ. Ànɛ̂ yé kɩ fɛ̃ sépɛ tǎhkɩ̀lɛ n yɩkɩ, tɛ́ tir yɔ́ ńtɛ̃ náh yé nɔ n pi. 
Tɛ́ yé káh yɛ yé nɩ vɩ sétã̀n sàhnɔ́ ó tyɩ́lɛ yé yóré yõ. Ǹtɛ, yé yé nɩ vɩ nɛ, yé yni-á mɛ wã́rkɩ́nsàhnɔ́lɛ yĩ̂nyõ.» 
Légbĩ́-ó sɔkɔ, Yesu nɩ kah vɩ Liyel Mírkí gbõ̀ yõ. Képah sɔkɔ, à mɛ nɛ: «Bɔ̌, áwɔ nɛ̂-á yĩ̂nyõ, ànɛ̂ sétáh nɛ́tɛ̃̂nɛ, mé álɛ n gbilki, áwɔ nɛ̂-á tyi nɔ́pi wɔ̃bɩ yɩ̃ pɩ̃́pú, ànɛ̂ lékã́mpu yah, tɛ́ sélɛ wɛ̃́npì wɔ wɛ̃kɩ. Àwe, bɔ̌, áwɔ dyɔ yĩ́ĩ́ tɛ̃̀ ye nɛ kè pɩ sɛ̃́. 
Ń sú ye sé póllɛ ń gbõ̀ dahbɩ. Ǹtɛ, ǹnɩ̂ yɔ́ ńtɛ̃ náh Liyel pilɛ n pnɛ, nɛ́ sú ó náh. Sɛ̃́ ó ye, nɛy tɔ náh súlɛ n pnɛ, nɛ́ pídĩ́ ó náh, ànɛ̂ wǎh mɛ ké tyɩ́ nɛ, pé pé súlɛ nɛ̂ wɛ̃kɩ.» 
Képah náh, Yesu ǹ yah vi ǹ tõ̀ sã́hpú tĩ̀nnɛ, tɛ́ yo pépi ó tyɩ́: «Yèvɩnɩ mɛ yé tyɩ́, yépi nónó-á tyi nɔ́pi wɛ. 
Mé kélɛ n yo yé tyɩ́ nɛ: Liyel tɛ̃ntõnè náhnáh, ànɛ̂ yõ̀tãm náhnáh sõ̀ mɛ ké tyɩ́ nɛ, pé sè wɛ, tɛ́ pè náh sè wɛ. Tɛ́ mɛ ké tyɩ́ nɛ, yáh nónó noh, pé sè noh, tɛ́ pè náh sé tɔ wɛ noh.» 
Légbĩ́nɛ, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́-ò yɔ́ sõ̀ nɛ, pé Yesu mɔ yah, tɛ́ yuku yĩn, à yo ǹ tyɩ́ nɛ ńkɛ̃́, kwɔ́-ò dĩ́, pé se kɩ pɩ sɔ̃́, tɛ́ kɩ tyɛ́-ńkɛ̃̂ min wɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ, tir nɛ̂ se wã́rkɩ́nsàhnɔ́lɛ Liyel tyi sàhnɔ́ sɛ́bɛ́y sɔkɔ nɩ? Tir nɛ̂ se à mɛ n kar ké sɔkɔ nɩ? 
Dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh mɛ wã́rkɩ́nsàhnɔ́lɛ nɛ ńkɛ̃́, á Tɛ̃̀ ǹgbɛ̃ Liyel tyɩ́-á yɩ̃nɛ kè nɔ á tyɩ́ á nɩ póllɛ, ànɛ̂ á min póllɛ, ànɛ̂ á tǎhkɩ̀ póllɛ, ànɛ̂ á sõnɔ póllɛ. Á ní-ò tɔ tyɩ́ mó nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́: «Á yo plɛ. Képahlɛ n pɩ, á kɩ tyɛ́-ńkɛ̃̂ min wɛ.» 
Ǹtɛ, dĩ́ mɛ̀ sõ̀ mɛ ké tyɩ́ nɛ, pé gbɩ wɛ. Kǎh pɩ sɛ̃́, à Yesu syah nɛ, nɛ̂ se péwɔ ní-òlɛ nɩ? 
Yesu mɛ à syah nɛ: «Dĩ́ yɔ́ ye lésõ wil Yerusalɛmɩ, à mɛ n tiki n yuku Yeriko sɔkɔ, tɛ́ kyɩ wil mɔ yɩ̃́pú yõ. Pè mɛ à tɛ̃ ǹ gbã̀n yî pól tɛ̃ syi, tɛ́ à ko markɩ sah, tɛ́ sɔkɔ. 
Tɛ́, kè pɔ yɩ̃nɛ, sárká wìlkìpu yõ̀tɛ̃̀ yɔ́ mó mɛ n tiki n pi kénɛ wɛ̃̀kɩ̀lɛ. Wǎh pɔ dĩ́ mɛ̀ wɛ, à gbɔ sɔkɔ fɩ́ɩ́. 
Képah náh, Levi yìkì nɛy yɔ́ tɔ tɔ̃ pɔ yĩni tɛ̃ nɛ́npɔ́. Ǹmɔ tɔ-á à wɛ sɛ̃́, à gbɔ sɔkɔ fɩ́ɩ́. 
Ǹtɛ, Samarɩ nɛy yɔ́ sõ̀ mɛ n yuku kɔ́lɔ́ sɔkɔ, tɛ́ kyɩ dĩ́ yõ dɔkɔ. Ǹmɔ wɔ-á à wɛ sɛ̃́, ǹ yãm à tɛ̃. 
À mɛ nɔ de ǹ tnɔ̂, tɛ́ dìvɛ̃ ànɛ̂ nɩrɩ mɔ ǹ nɩ́y yah, tɛ́ sè pukubi. Tɛ́ mɛ à dɔkɔ mɔ ǹ yɛ̃́ngbɛ̃́ ńkwɛ̃́ndyɩ́kɩ̀ yõ, à yuku ǹnɛ nɛ́pân twáh-ńsah yɔ́ sɔkɔ, à kyɩ yah dah mɔ ǹ tyɩ́ sɔkɔ. 
Ké dendallɛ, à mɛ pɔ́ plɔ́ nínɔ́ wilki nɛ́pân twáh-ńsah yõ̀tɛ̃̀ dĩ́ kɔ̃, tɛ́ nɛ, à nɛ n yah n dah n mɔ dĩ́ mɛ̀ tyɩ́ sɔkɔ. Ńkɛ̃́, wàh kɩ sɔ̃́ gbõ̀ wilki à kah mɔ képah yõ, pâh pé syɩ pɔ gbĩ́ nɛ̂nɛ, páh kɩ à dyah kélɛ.» 
Képah yóntɛnɔ náh, Yesu dĩ́ piki yah nɛ, ǹmɔ tɛ̃̀ sõnɔ sɔkɔ, yɩ̃́pú-á dĩ́ nɛ̂ pɩ yɔ, ǹmɔ ní-ò se nɛ̂ tɛ̃̀nɛ nɛ́pĩ̂ tɔ̃́mɩ́ nɔ́pi sɔkɔ nɩ? 
Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́-ò mɛ à syah nɛ, nɛ̂-á ǹ yãm yah, ǹmɔ-á. Képah tɛ̃̀nɛ, Yesu mɛ yo ǹ tyɩ́: «Á tɔ kyɩ sɛ̃́nkɔ̃lɛ n pɩ.» 
Tɛ́ Yesu ànɛ̂ ǹ tõ̀ sã́hpú-á sõ̀ mɛ wɛ̃̀kɩ̀ yõ sɛ̃́, à kyɩ de kwílpír yɔ́ sɔkɔ. Tɛ́ kénɛ kwil sê yɔ́ sõ̀ mɛ, pè n ye Martɩlɛ. Ǹmɔ mɛ Yesu tɛ̃ plɛ ǹ gbô. 
Tɛ́ gbɛ̃̀ sê yɔ́ sõ̀ mɛ ǹ tyɩ́, pè n ye Marilɛ. Ǹmɔ sõ̀ mɛ pɔ tɛ̃ Tɛ̃̀ ǹgbɛ̃ Yesu tnɔ̂, à ǹ wɛlɔlɛ n nohnɩ. 
Légbĩ́nɛ, Martɩ yah mɛ n vyãh Yesu fèn pɩ́nɔ́ sɔkɔ. À mɛ pɔ yo Yesu tyɩ́ nɛ, pé Yõ̀tɛ̃̀, pé gbɛ̃̀ sê-á tõ̀ pól sah péwɔ ó kɔ̃, tɛ́ pɔ tɛ̃ ǹ tnɔ̂, kè kó mɛ tirlɛ ǹ yah nɩ? À kó n yo ǹ tyɩ́, à pélɛ n yohnɩ n mɔ nɩ? 
Ǹtɛ, à mɛ yo ǹ tyɩ́ nɛ: «Martɩ, áwɔ mɛ á gblɔ̌ynɛ n kwlɔ, mɛ á fɔkɔlɛ n sõ tyi náhnáh tyɩ́lɛ. 
Ǹtɛ, tir núkú ó tyɩ́ ye yõrelɛ. Mari ye ké pèpɛy yah tɔkɔ, nɛ̂-á kè náh n tɛ̃ n syi n yah n pi ǹ tyɩ́.»
