Markɩ.
15.
Képah sɔkɔ, séngbɔ̃́ púpúlɛ, *sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú, ànɛ̂ yèwàhnɩ̀nɛ, pépi pól ye sõ̀ pɩ *Nsyifunɔ tukey tùkèpu yõ̀tãmnɛ. Pépi wɛ̃ tuke mɔ Yesu tyɩ́ sɔkɔ. Képah tɛ̃̀nɛ, pè à tɛ̃ pukubi, tɛ́ kyɩ ǹnɛ *Pilati gbõ̀ dahbɩ. 
Pilati mɛ à piki yah nɛ, ǹmɔ se Nsyifunɔ yõ̀tɛ̃̀nɛ nɩ? À mɛ yo ǹ tyɩ́: «Áwɔ ye yo kélɛ.» 
Képah sɔkɔ, sárká wìlkìpu yõ̀tãm ǹgbnɔ mɛ lékàhlɛ à kɔ̃ tyi náhnáh sɔkɔ. 
Képah náh, Pilati mɛ tɔ̃ à piki yah nɛ, à kó tir yɔ́ yo n pi nɩ? Tyi nónó pól-á n yo n sah ǹ yõ, à kó sè noh nɩ? 
Ǹtɛ, à náh tɔ̃ vi mɔ. Képah Pilati yáh-ńsah tyɛ. 
Ye ye póllɛ, *Pakɩ syáh-à n syah, *Pilati ńkɛ̃́nɛ wɔ̀kɔ̀npèkè sɔkɔ nɛ núkú yɔ́ fĩn mɔ, nɛ̂ tyɩ́-á nɔ tɔ́wû tyɩ́. 
Pǎh Yesu tɛ̃ kyɩ Pilati tĩ̂nnɛ gbĩ́ nɛ̃̂nɛ, dĩ́ yɔ́ sõ̀ mɛ wɔ̀kɔ̀npèkè sɔkɔ, pè n ye Barabasilɛ, ànɛ̂ pé nolɛ. Pè sõ̀ nɛ́kókè mɔ gbĩ́ nɛ̂-á sõ̀ pè mɛ yahle sàhnɔ́lɛ *Wromɛnɔ tyɩ́ yõ. 
Képah tɛ̃̀nɛ, tɔ́wû yuku kyɩ de Pilati tĩ̂nnɛ, à kyɩ tyah ǹnɛ n ni nɛ, wǎh dar, tɛ́ tir nɛ̂nɛ n pɩ pé tyɩ́ Pakɩ syáhnɔ́ sɔkɔ, wǎh náh kè pɩ pé tyɩ́. 
Pilati mó yo pé tyɩ́ nɛ, pè se nɛ, pé *Nsyifunɔ yõ̀tɛ̃̀ ǹgbɛ̃ fĩn mɔ nɩ? 
À sõ̀ kè pɩ̃ nɛ ńkɛ̃, dyɔ yɔ́nɔ́-á *sárká wìlkìpu yõ̀tãm ǹgbnɔ kɔ̃ pè Yesu dahbɩ péwɔ gbõ̀. 
Tɛ́ sárká wìlkìpu yõ̀tãm ǹgbnɔ sõ̀ mɛ gbah tɛ̃ tɔ́wûlɛ n kurki n mɔ nɛ, pè yo nɛ, à Barabasi wilki fĩn mɔ pé kɔ̃. 
Pilati pè piki yah tɔ̃ ńtɔ̃ nɛ, sɔ̃́ se pé kɩ pɩ dĩ́ mɛ̀nɛ, nɛ̂-á pè n ye Nsyifunɔ yõ̀tɛ̃̀nɛ nɩ? 
Pè tɔ̃ gbo mɔ, tɛ́ mɛ yo: «Yé à syɩ *dahnɩ yõ!» 
Pilati yo pé tyɩ́ nɛ, tyípékè nɛ̂ se à pɩ nɩ? Pè tɔ̃ tɔ̃ gbo mɔ náhnáh ńkɛ̃, à syɩ dahnɩ yõ. 
Képah sɔkɔ, Pilati Barabasi fĩn mɔ. À sõ̀ mɛ tɔ́wû nɩ̀vɩ̀lɛ n yah n kɔ̃, képah ye. À pè kɔ̃ pè Yesu ko yèfõhlɛ, tɛ́ ǹnɛ sràhsyíbí kɔ̃ nɛ, pè kyɩ à syɩ dahnɩ yõ. 
Képah tɛ̃̀nɛ, sràhsyíbí Yesu tɛ̃ yuku gbô nɩyṍ yõ̀tɛ̃̀ dĩ́ tɛ̃̀-ǹsah, tɛ́ pé no pól ye tuke mɔ. 
Pǎh kɩ pɩ sɔ̃́, tɛ́ kɩ kṍlɛ à wɛ n ko n pi, pè fwɔ̀mɔn syã̂h pèpɛy yɔ́ mɔ à kɔ̃ tɛ́, yõ̀kɔ̃ pɩ pã̀n yîlɛ, à dyɩ à kɔ̃ tɔ̃́rɩ́ yṍkúrí swɔ. 
Tɛ́ tyah n yo ǹ tyɩ́ nɛ, páh n wɛkɩ ǹ tyɩ́, *Nsyifunɔ yõ̀tɛ̃̀ ǹgbɛ̃, 
tɛ́ ǹnɛ n mɩkɩ ǹ yõ sɔkɔ gbè-ńsĩnnɛ, tɛ́ n sĩn n mɔ n pɔ ǹ yõ. Pè gbètukunɔ di, à kɔ kwĩnki sétáh ǹ yah sɔkɔ. 
Pǎh kṍlɛ à ko tɛ, pè fwɔ̀mɔn syã̂h mɛ̀ dɛ ǹ wil. Tɛ́ tɔ̃ ǹ flɔ tɔkɔ mɔ à kɔ̃. Képah náh, pè wil ǹnɛ kégbɔ́ sɔkɔ ǹ syɩ yĩnnɛ *dahnɩ yõ. 
Sirɛni nɛ yɔ́ sõ̀ mɛ, pè n ye Simolɛ, Alɛkisãdre ànɛ̂ Wurfu súlɛ. Légbĩ́nɛ, ǹmɔ sõ̀ mɛ wil swãh, mɛ pɔ n sɔkɔ. Sràhsyíbí à kyɩ̃ mɔ, à Yesu syɩ syɩ *dahnɩ mɛ̀ twah. 
Pè yuku Yesulɛ tyah yɔ́ sɔkɔ, ńyãh-á pè n ye «Kolkɔtalɛ», yõ̀gbèkè tyahlɛ. 
Tɛ́ pè mɛ syɩbɩ yɔ́ fɛ̃kɩ mɔ dìvɛ̃ sɔkɔ à kɔ̃ nɛ, à wɔ, tɛ́ à yahle. 
Képah náh, sràhsyíbí à syɩ dahnɩ yõ, tɛ́ mɛ ǹdan mɔ à ǹ flɔ dibi wɛ̃ tyɩ́. 
Pǎh à syɩ gbĩ́ nɛ̂nɛ, gbĩ mɛ pɔ̃́ntɔ̃kɩ sɔkɔ. 
Pǎh à syɩ yõ nɛ̂ yõ, pè kè wãrkɩ syɩ ǹ yõ̀tah sésár sɛ́nɔ́ yõ nɛ: «*Nsyifunɔ yõ̀tɛ̃̀ ǹgbɛ̃». 
Tɛ́ mɛ tɔ̃ kèlékɛ̃ névye nɛ́pĩ̂ nímí syɩ dahlɔ yõ ǹ gbɔ̀plɔ, ǹ gbõ̀yõ̀ ànɛ̂ ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀.  
Képah sɔkɔ, nɛ̂-á yo sah *Liyel wɛy sɛ́bɛ̃́nɔ́ sɔkɔ nɛ: «Pè à mɔ kõ tyípêl pɩ́pú sɔkɔ», kè ńtyɩ̃́nɔ pɩ sɛ̃́. 
Képah tɛ̃̀nɛ, wɛ̃̀kɔbɩ sõ̀ mɛ ǹnɛ n gbyɔ n sɔkɔ, tɛ́ mɛ pé yõlɛ n kal, mɛ yo tũ ǹ tyɩ́ nɛ, ɔ̃̀ɔ̃́, ǹmɔ nɛ̂-á sõ̀ nɛ, páh kɩ *Liyel wɔ̀kɔ̀ ǹgbɛ̃ sɔ, tɛ́ kɩ kè tɔ̃ mɔ sèpĩ̀n tɔ̃́nɩ̀ wrɔ́, 
à ǹ gblɔ̌y pwah n mɔ, à mó tiki dahnɩ yõ! 
Képah núkú-ó ye, *sárká wìlkìpu yõ̀tãm ǹgbnɔ ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú sõ̀ mɛ kṍlɛ ǹnɛ n ko, tɛ́ mɛ n yo wɛ̃ tyɩ́ nɛ, wǎh névye túkù pwah mɔ, tɛ́ à náh fɛ̃ ǹ gblɔ̌y pwah n mɔ! 
Ǹmɔ-à kɩ nɛ Liyel-á nɛ, páh kɩ nɛ tɛkɩ mɔ Yisrayɛl yõ̀tɛ̃̀nɛ, à tiki dahnɩ mɛ̀ yõ! Páh kɩ képah wɛ, páh kɩ sɛ̃ ǹ yõ. Tɛ́ nónó-á sõ̀ mɛ syɩ́nɔ́lɛ dahlɔ yõ ǹ gbɔ̀plɔlɛ, pépi tɔ mɛ ǹnɛ n gbyɔ. 
Gbĩ̀ntɔ̃̀ pɔ̃́lɛ, lékã̂h mɔ sétáh pól yõ sɔkɔ, fɔ́ɔ́ à pɔ nɔ gbèkãm gbĩ́ tyɩ́. 
Gbèkãm gbĩ́-á nɔ, Yesu kah de syi, tɛ́ yo pé kwil wɛy sɔkɔ: «Eloyi, Eloyi, lama sabaktani?» Képah n yo nɛ: «Ńmɔ Liyel, ńmɔ Liyel, kû pɩ, tɛ́ á ń fĩn mɔ?» 
Nónó-á sõ̀ mɛ nɛ́npɔ́ tɛ́ kè noh, pé túkù sõ̀ mɛ n yo nɛ, pè tɔkɔ noh, wǎh mɛ *Yelilɛ n ye. 
Képah sɔkɔ, pé sɔkɔ nɛ yɔ́ mɛ to kyɩ sènsúkúrí tɔkɔ dahbɩ wilki yõ̀yõ̀kɩ̀ ni sɔkɔ, à dyɩ gbey yɔ́ vyãh tyɩ́, à pɔ nɔ ǹ tyɩ́ nɛ, à wɔ, tɛ́ mɛ yo tũ nɛ, pé ye yah ǹkàh Yeli-á kɩ ńtyɩ̃́nɔ pɔ à tiki mɔ n pi *dahnɩ mɛ̀ yõ nɩ? 
Képah náh, Yesu mɛ de syi, tɛ́ pyě ku. 
Nɛ́npɔ́ swɔ, fwɔ nɛ̂-á *Liyel wɔ̀kɔ̀ ǹgbɛ̃ fɔkɔ táh dibi nínɔ́, kè tɔ, à tɔkɔ ké yõ̀tah, à pɔ syɩ sétáh tyɩ́. 
Sràsyíbí lèkwɛ̌ yõ̀tɛ̃̀ dĩ́ nɛ̂-á sõ̀ mɛ yĩ́nntɛ̃̀nɔ́lɛ, tɛ́ ǹ ku dùkù wɛ sɛ̃́, à nɛ: «Gbɩ yõ, dĩ́ mɛ̀nɛ, Liyel pi ye ńtyɩ̃́nɔ sõ̀ ǹnɛ.» 
Syɔ́ yísyɔ́ tɔ sõ̀ mɛ yĩn tɛ̃ fɩ́ɩ́, mɛ n yah. Pépi wrɔ́ sɔkɔ Makadala kwil Mari, ànɛ̂ Mari nɛ̂-á Nsyakɩ pir ànɛ̂ Yose yṹnɛ, ànɛ̂ Salome sõ̀ mɛ. 
Pépi ye sõ̀ Yesu sõ̀ tɛ̃, ǹ tõ̀lɛ n pɩ Kalile sɔkɔ. Syɔ́ kwĩnki yísyɔ́ tɔ sõ̀ yuku ǹnɛ Yerusalɛmɩ sɔkɔ. 
Pǎh Yesu ko mɔ sṍ nɛ̂ ńvɩ̃́nɛ, ké dèndal ye sõ̀ pɩ *Nsyifunɔ *mɩ mɩ yòhlɛ. 
Arimatɩ kwil nɛ yɔ́ sõ̀ mɛ, pè n ye Yosɛfulɛ. Ǹmɔlɛ, Nsyifunɔ tukey tùkèpu sɔkɔ nɛ pèpɛy yɔ́ ye sõ̀ ǹnɛ. Ǹmɔ tɔ sõ̀ mɛ *Liyel tɔ̃́rɩ́ pɔ́nɔ́lɛ n syɩkɩ. Ǹmɔ ǹ nɩ syãh, à kyɩ Yesu lékó ni *Pilati tyɩ́ nɛ, à ǹnɛ pé kɔ̃, pé kyɩ à yɩ̃. 
Pilati-á noh nɛ, Yesu-á núkú ku tɛ, kè ǹ yáh-ńsah tyɛ. À sràsyíbí yõ̀tɛ̃̀ dĩ́ ye pɔ piki yah nɛ, ǹ kúnɔ́ di se mo nɩ? 
Wǎh ké gbɩ noh tɛ sràsyíbí yõ̀tɛ̃̀ dĩ́ tyɩ́, à mó wɛ̃̀kɩ̀lɛ Yosɛfu kɔ̃ nɛ, à kyɩ Yesu lékó tɔkɔ. 
Képah sɔkɔ, Yosɛfu kyɩ fwɔ̀pĩn dwe. À kyɩ Yesu lékó dɛ *dahnɩ yõ, tɛ́ à plo fwɔ̀pĩn mɛ̀ sɔkɔ, à dah mɔ diki sɔkɔ, nɛ̂-á tũ tɛkɩ sɔkɔ, tɛ́ dáh-ǹgbɛ̃ yɔ́ pil pɔ vike ké yah. 
Légbĩ́nɛ, Makadala Mari ànɛ̂ Yose yṹ Mari mó Yesu dáhnmɔ-ńsah yah pɔkɔ mɔ sah.
