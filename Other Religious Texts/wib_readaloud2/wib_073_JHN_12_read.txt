Nsyɔ̃.
12.
Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh sõ̀ mɛ wũ sèpĩ̀n kénúkòlɛ, tɛ́ Yesu mɛ kyɩ Betani kwil sɔkɔ, ńyãh sɔkɔ-á sõ̀ Lasarɩlɛ tɛ̃̀nɔ́lɛ, wǎh nɛ̂ lékó yilki mɔ. 
Pè mɛ yõke yah Yesu kɔ̃ nɛ́npɔ́. Martɩ sõ̀ mɛ yõkelɛ n dibi, tɛ́ yõke mɛ̀ yṍpú sɔkɔ yɔ́ ye sõ̀ Lasarɩlɛ. 
Képah sɔkɔ, Mari mɛ làsèkwěl dɛ́nkɔ́kɔ̀ yɔ́ lítír tɔ́nɩ́ tɔkɔ, nɛ̂-á pè nardɩ kɔ́kɔ́lɛ kè pɩ. Tɛ́ pɔ kè kwɛ mɔ Yesu gbɛ̃́náhlɔ́ yõ, tɛ́ kè kele ǹ yõ̀yénnɛ. Làsèkwěl mɛ̀ pãn mɛ wɔ̀kɔ̀ pól tɔkɔ. 
Képah tɛ̃̀nɛ, Nsyudasɩ Yisikariyɔtɩ nɛ̂-á Yesu tõ̀ sã́hpú sɔkɔ yɔ́lɛ, tɛ́ kɩ sõ̀ pɔ n vi n pi Yesu náh, ǹmɔ nɛ, 
kwâh nɛ̂ se pɩ tɛ́ làsèkwěl mɛ̀ náh yãm deniye pɔ́ plɔ́ kyɛtɔ̃nɩ́nɛ, à ké pɔ́ dibi yãm tãn kɔ̃ nɩ? 
Ǹtɛ yãm tãn yãm náh sõ̀ tɛ̃ ǹnɛ dɛ́, yɛ̃́nwó ye sõ̀ ǹnɛ képah ye à kɔ̃ à képahlɛ n yo. Pɔ́ dú sõ̀ mɛ ǹmɔ tyɩ́. Tɛ́ pǎh nónónɛ n dah n mɔ n pɔ, ǎ sélɛ n yɩ̃. 
Képah sɔkɔ, Yesu nɛ: «Pi yah vɩ́nɔ́ sah kénòh! À ye képah pɩ sah ńmɔ lékó yɩ̃ sõ yĩnnɛ. 
Yãm tãn kɩ mɛ ànɛ̂ yélɛ sétáh yõ sõmɛsṍ. Ǹtɛ ńmɔ wɔlɛ, ńmɔ náh n mɛ n pi yélɛ sétáh yõ sõmɛsṍ.» 
Tɛ́ tɔ́wû ǹgbɛ̃-á pɔ noh nɛ Yesu-á mɛ Betani kwil sɔkɔ gbĩ́ nɛ̂nɛ, pè kyɩ nɛ́npɔ́. Ǹtɛ Yesu ó yĩn náh sõ̀ pè kyɩ dɛ́, Lasarɩ nɛ̂ lékó-á Yesu yilki mɔ, ǹmɔ tɔ wɛ yĩn ye sõ̀ pè kyɩ. 
Ǹtɛ Lasarɩ mɛ̀ yĩnnɛ, névye náhnáh sõ̀ mɛ pélɛ n syɩ sárká syínwìlkìpu yõ̀tãm náh à n vi n mɔ n pɩ Yesu yõ sɛ̃́púlɛ. Képah ye, sárká syínwìlkìpu yõ̀tãm mɛ Lasarɩ tyɩ́ yo syɩ nɛ, páh kɩ ǹ tɔ ko mɔ. 
Ké dĩndallɛ, tɔ́wû ǹgbɛ̃ nɛ̂-á sõ̀ mɛ pɔ Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh sɔkɔ, pè kè noh nɛ Yesu-á mɛ n pi Yerusalɛmɩ sɔkɔ. 
Pè mɛ sãm yɩ̃̀ngbã̀n kɔ̃lɩ tɛ̃ yĩn, tɛ́ wil kwil sɔkɔ à kyɩ à yohnɩ. Pè sõ̀ mɛ gbo mɔ tũ nɛ: «Wosana! Nɛ̂-á n pi Tɛ̃̀ ǹgbɛ̃ Liyel yĩn yõ, Liyel sãm mɔ ǹ sɔkɔ, ǹmɔ nɛ̂-á Yisrayel yõ̀tɛ̃̀ dĩ́nɛ.» 
Képah sɔkɔ, Yesu mɛ ńkwɛ̃́ndyɩ́píkì yɔ́ wɛ, tɛ́ dɔkɔ kɔ̃ tɛ̃ ǹ yõ, à yɩ̃nɛ sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: 
«Yerusalɛmɩ tãn, yé káh tíkí pɩ. Yé yah, yé yõ̀tɛ̃̀ mɛ n pi. À mɛ dɔ́kɔ́ntɛ̃̀nɔ́lɛ ńkwɛ̃́ndyɩ́píkì yɔ́ yõ sɔkɔ.» 
Tɛ́ légbĩ́nɛ, Yesu tõ̀ sã́hpú náh sõ̀ mɛ tyi nɔ́pi pɩ́nɔ́ yõ pɩ̃. Ǹtɛ wǎh pɔ dɔkɔ ǹ tɔ̃́rɩ́ yõ gbĩ́ nɛ̂nɛ, pé dyɔ to kõ nɛ, kǎh lésõ wãrkɩ sah sɔ̃́ ǹ tyɩ́ sɔkɔ Liyel wɛy sɛ́bɛ́y sɔkɔ, sɛ̃́-á kè pɩ ǹ tyɩ́. 
Képah tɛ̃̀nɛ, Yesu-á lésõ Lasarɩ lékó ye yah, à à yilki mɔ wilki diki sɔkɔ nónó yĩ́ yah, pépi mɛ tyah kélɛ n yah n yãh n kɔ̃. 
Névye-á kénɛ gbǐl tir pɩ́nɔ́ noh sɛ̃́, pè wil pɔ Yesu yohnɩ. 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú tyah n yo wɛ̃ tyɩ́ nɛ, pè kó kè wɛ nɩ, pépi náh tɔ̃ fɛ̃ n tɛ̃ n pi dò. No pól-á mɛ ǹ sõ̀ tɛ̃. 
Névye nónó-á sõ̀ mɛ pɔ syáh mɛ̀ sɔkɔ Yerusalɛmɩ, nɛ pé pɔ Liyel yĩn gbilki, Nsyifunɔ náh sõ̀ pé túkùlɛ. 
Kénɛ névye nónó-á Nsyifunɔ náh pélɛ, pépi mɛ nɔ de Filipi tnɔ̂ ǹmɔ nɛ̂-á n wil Betisayida kwil, Kalile gbɛ̃̀nyah sɔkɔ. Tɛ́ kè yah kɔ̃ ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, páh mɛ ké tyɩ́ nɛ pé Yesu wɛ. 
Kǎh pɩ sɛ̃́, Filipi mɛ kè yo Andre tyɩ́. Andre mɛ tahlɩ ǹ tyɩ́ pè kyɩ kè yo Yesu tyɩ́. 
Képah tɛ̃̀nɛ, Yesu mɛ yo pé tyɩ́ nɛ: «Gbĩ nɛ̂ sɔkɔ-á Névi Pídĩ́ kɩ n dɔkɔ n pi ǹ tɔ̃́rɩ́ yõ, képah ye sɛ núkúnúkú yɔ̀. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, nɛ́ sèmukɔ pi náh duku mɔ sétáh sɔkɔ kè fi tɛ́ ké pi ku, kè n tɛ̃ ké gblɔ̌y. Ǹtɛ kàh duku mɔ à fi tɛ́ ké pi ku, kè n kwel. 
Névi nɛ̂-à ǹ gblɔ̌y ó minnɛ n mɔ n kõ, tɛ̃̀ min tyɩ́ kɩ yɩkɩ. Ǹtɛ nɛ̂-à ǹ minnɛ n mɔ́nkõ̀-ǹkɛ̃̂nɛ kèkõyṍ mɛ̀ sɔkɔ Liyel tɔ̃́rɩ́ yĩnnɛ, tɛ̃̀ wɔ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Névi nɛ̂-à mɛ ké tyɩ́ nɛ, pé tõ̀lɛ n pɩ ńmɔ tyɩ́, tɛ̃̀ ń sõ̀ tɛ̃. Képah sɔkɔ, máh mɛ ńyãh sɔkɔ, nɛ̂-á tõ̀lɛ n pɩ ń tyɩ́, ǹ tɔ kɩ mɛ nɛ́npɔ́. Névi nɛ̂-à tõ̀lɛ n pɩ ńmɔ tyɩ́, ń Sú Liyel kɩ ǹ yõ yɔ̃. 
Tɛ́ núkúnúkú, ń fɔkɔ mɛ fɛ̃kɩ. Ǹtɛ mè yo sɔ̃́? Mé se n yo Sú Liyel tyɩ́ nɛ, à káh n yɛ yèvyãhrɩ mɛ̀ ń wɛ? Ǹtɛ ɔ̃́ɔ̃ dɛ́, yèvyãhrɩ mɛ̀ yĩn ye mé pɔ tɔ́tɔ́.» 
Képah tɛ̃̀nɛ, à nɛ: «Bɔ̌, gbílkínɔ́ sah á yĩn yõ.» Wǎh yo sɛ̃́, yĩ́npɔ̃́ yɔ́ wil yĩ̂nyõ sɔkɔ nɛ ńkɛ̃́: «Mé ń yĩn gbilki tɛ, tɛ́ mé kɩ kè gbilki n pi ńtɔ̃.» 
Tɔ́wû nɛ̂-á sõ̀ mɛ nɛ́npɔ́, tɛ́ yĩ́npɔ̃́ mɛ̀ noh, pè nɛ gbɛ̃́nséntã́hkɩ̀-á gboke. Túkù mó nɛ, yĩ̂nyõ tõ̀npɩ yɔ́-á yo mɔ Yesu tyɩ́. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ: «Wɛy nɛ̂-á yé noh yɔ̀, kè náh yo ńmɔ yĩnnɛ, kè yo yépi yĩnnɛ. 
Liyel kɩ lékàhlɛ kèkõyṍ névye kɔ̃ n pi núkúnúkú sɛ̃́ yɔ̀. Núkúnúkú, à kɩ kèkõyṍ névye fnɛ̀-ò yĩni yal. 
Ǹtɛ mâh pɔ dɔkɔ mɔ gbĩ́ nɛ̂nɛ, mé kɩ névye póllɛ pɔ ń gblɔ̌y tyɩ́.» 
Yesu-á sõ̀ nɛ n ku n pi dùkù nɛ̂nɛ, képah ye à wɛ̃kɩ ǹ wɛy mɛ̀ sɔkɔ yɔ̀. 
Képah sɔkɔ, tɔ́wû mɛ̀ névye mɛ yo ǹ tyɩ́ nɛ, pépi-á kè wɛ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ nɛ, Liyel nɛy yáhntɔkɔnɔ náh n ku n yah n pi. Tɛ́ ǹmɔ se sɔ̃́npɩ́lɛ pɔ n yo nɛ, Névi Pídĩ́-á yɩ̃nɛ à dɔkɔ mɔ nɩ? Ǹtɛ nɛ̂ tɛ̃̀ ó se mó Névi Pídĩ́ mɛ̀nɛ nɩ? 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ: «Kyòo mɛ̀nɛ, kè mɛ sɛ̃́ yé wrɔ́ kɩ wrɔ́ kwéy yɔ́ pɩ n pi. Tɛ́ névi nɛ̂-à n kɔ fɛ̃́ntũ̂ sɔkɔ, tɛ̃̀ náh ǹ yúkú-ńsahlɛ n pnɛ. Képah ye, kyòo mɛ̀-á mɛ yé wrɔ́ sɔkɔ yɔ̀, yé nɛ n kɔ ké sɔkɔ, fɛ̃́ntũ̂ káh pɔ yé kwɔkɩ tɛ̃. 
Kyòo mɛ̀nɛ, kǎh mɛ yé wrɔ́ gbĩ́ nɛ̂nɛ, yé sɛ̃ képah yõ. Képah sɔkɔ, yé kɩ pɩ kyòolɛ képah tyɩ́ kɔ̃lɛ.» Yesu-á képah yo tɛ, à yal pé tnɔ̂ à kyɩ wɔ̃bɩ fɩ́ɩ́ pélɛ. 
Yesu sõ̀ gbǐl tyi náhnáh pɩ pé yĩ́ yah, ǹtɛ képah fyé yõ, pè náh sɛ̃ ǹ yõ. 
Liyel tɛ̃ntõ̀ syínyo-ò Yesayi tyíyónɔ́ nɔ́pi ye wil ásõ̂ sɔkɔ yɔ̀. À lésõ nɛ ńkɛ̃́: «Tɛ̃̀ ǹgbɛ̃, nɛ̂ sɛ̃ ápi tyíyónɔ́ yõ? Á á tɛ̃́npnɛ̀nɔ́lɛ nɛ̂ wɛ̃kɩ?» 
Tɛ́ tir nɛ̂ yĩn-á pè náh sõ̀ fɛ̃ n sɛ̃ n pi ké yõ, képah yah ye Yesayi tɔ̃ yo nɛ ńkɛ̃́: 
«Liyel pé yɩ̃́ wɔ, tɛ́ pé lékã́m yah vike, nɛ pé yɩ̃́ káh fɛ̃ nɛ n yah, pè káh tir yɔ́ tyɩ́ yah wɛ, pè mó vi mɔ Liyel wɛ̃̀kɩ̀ yõ, à kɔ̃ à pè wɛ dɛ mɔ.» 
Yáh wɛ Liyel tɛ̃ntõ̀ syínyo-ò Yesayi képahlɛ n yo, à lésõ kwãh tɛ̃ Yesu tɔ̃́rɩ́ wɛ, képah ye à kɔ̃ à ǹ tyɩ́lɛ n yo. 
Tɛ́ képah fyé yõ, yõ̀tãm náhnáh sõ̀ mɛ sɛ̃ Yesu yõ. Ǹtɛ Farisi wɛ̃̀kɩ̀ kɔ́pú tíkí sõ̀ mɛ pè tɛ̃ nɛ, pè káh pɔ pé yĩni wilki pé wɛ̃̀kɩ̀ sɔkɔ. Képah ye, pè náh kè wilki mɔ nɛ, páh sɛ̃ ǹ yõ. 
Yáh wɛ pè képah pɩ, névye-á mɛ gbílkínɔ́ nɛ̂nɛ n sah pé yõ, képah tyɩ́ sõ̀ mɛ pé tyɩ́ kal Liyel tyɩ́lɛ. 
Képah sɔkɔ, Yesu wɛkɩ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ ńkɛ̃́: «Névi nɛ̂-à sɛ̃ ńmɔ yõ, ńmɔ ó yõ náh tɛ̃ sɛ̃ dɛ́, Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ tɔ yõ ye tɛ̃̀ sɛ̃. 
Tɛ́ névi nɛ̂-à ńmɔ wɛ, ń tɛ́kɩ́nmɔ-ò ye tɛ̃̀ wɛ. 
Ńmɔ pɔ kèkõyṍ à pɔ pɩ kyòolɛ, nɛ névi nɛ̂-à sɛ̃ ńmɔ yõ, tɛ̃̀ káh tɛ̃ fɛ̃́ntũ̂ sɔkɔ. 
Ńmɔ náh pɔ kèkõyṍ sɔkɔ, nɛ mé pɔ lékàhlɛ névye kɔ̃. Mé pɔ nɛ kèkõyṍ névye pwáhnmɔnɔ wɛ. Képah ye, névi nɛ̂-à ń wɛlɔ noh, tɛ́ à náh mɛ n kɔ sé wã̀l yõ, ńmɔ náh kɩ n kɔ̃ n pi tɛ̃̀nɛ lékàhlɛ dɛ́. 
Wɛy nɛ̂-á mé yo yɔ̀, képah ńtɛ̃ ye kɩ lékàhlɛ tɛ̃̀ kɔ̃ n pi gbĩ̀yĩ̀kì tɛ́y sõlɛ. Ń gblɔ̌y yah yĩ́ĩ́ náh mé yo wɛlɔ nɔ́pilɛ dɛ́! Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, máh yɩ̃nɛ mè nɛ̂ yo, ànɛ̂ máh yɩ̃nɛ mè kwɔ́nɔ́ nɛ̂ pɩ, ǹmɔ ye yo sélɛ ń tyɩ́ nɛ mé pɩ. Képah ye, névi nɛ̂-à ń yahle, tɛ́ à náh mɛ ń wɛynɛ n mɔ n kõ, nɛy mɛ tɛ, nɛ̂-á kɩ lékàhlɛ tɛ̃̀ kɔ̃ n pi. 
Tɛ́ ńmɔ mó tɔ̃ pɩ̃ nɛ Liyel-á nɛ̂ kõ sah, képah gbõ̀ yõ sɔkɔ-á névye kɩ fɛ̃ tyɛ́-ńkɛ̃̂ min wɛ. Sú Liyel-á wɛy mɛ̀ yo sɔ̃́ ń tyɩ́, sɛ̃́ ye mé n yo kélɛ yé tyɩ́.»
