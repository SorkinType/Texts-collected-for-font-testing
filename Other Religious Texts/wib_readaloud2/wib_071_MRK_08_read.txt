Markɩ.
8.
Kénɛ sèpĩ̀n sɔkɔ, tɔ́wû ǹgbɛ̃ yɔ́ tɔ̃ sõ̀ mɛ tùkènmɔnɔlɛ Yesu tnɔ̂. Tɛ́ yõ yõ kwâh náh sõ̀ mɛ pé tyɩ́ sɛ̃́, à ǹ *tõ̀ sã́hpú ye, tɛ́ mɛ yo pé tyɩ́ nɛ, 
tɔ́wû mɛ̀ yãm-á mɛ pé tɛ̃. Pé sèpĩ̀n tɔ̃́nɩ̀-á ànɛ̂ péwɔlɛ yɔ̀, tɛ́ yõ yõ kwâh náh mɛ pé tyɩ́. 
Páh kɩ pè fĩn mɔ, pè sɔkɔ kwɛkɩlɛ, pǎh kɩ kah kwlɔ wɛ̃̀kɩ̀ yah. Pé túkù-á n wil fɩ́ɩ́ sɔkɔ. 
Képah tɛ̃̀ sɔkɔ, ǹ tõ̀ sã́hpú yo ǹ tyɩ́ nɛ, pé se kɩ fɛ̃ wɛ ńyãh yõkelɛ gbɛ̃́nnɩ mɛ̀ sɔkɔ, kɩ pè kɔ̃, pè kɩ yõ tɔ̃ nɩ? 
À mɛ pè piki yah nɛ, búrn plɔ́ sɔ̃́ se mɛ pé tyɩ́ nɩ? Pè mɛ nɛ, kwǎrnínɔ́-á. 
Képah sɔkɔ, à tɔ́wû kɔ̃ pè kɔ̃ tɛ̃ sétáh, tɛ́ búrn plɔ́ kwǎrninɔ nɔ́pi tɔkɔ, à wɛynɛ Liyel kɔ̃, tɛ́ sè kwɔkɩ ǹ tõ̀ sã́hpú kɔ̃ nɛ, pè syi dibi névye kɔ̃. Pè mó sè syi dibi pè kɔ̃. 
Yɛ̃́nkyɔ́pì yísyɔ́ tɔ sõ̀ mɛ pé tyɩ́. À wɛkɩ Liyel tyɩ́ sé tyɩ́ yõ, tɛ́ nɛ, pè sé tɔ tɔkɔ dibi pè kɔ̃. 
Képah tɛ̃̀nɛ, pé pól yõ tɔ̃, tɛ́ ǹ tõ̀ sã́hpú yṍnkwlɔnɔ tɛ́lɔ́ tuke mɔ tɔkɔ yɩ̃́ kwǎrninɔ. 
Tɛ́ nónó-á sõ̀ mɛ nɛ́npɔ́, pè nɔ nɛ́pĩ̂ kèyõ nínɔ́ kénkɔ̃lɛ. Képah náh, à pè fĩn mɔ, 
tɛ́ mó núkú de krótókè sɔkɔ ànɛ̂ ǹ tõ̀ sã́hpúlɛ, à yuku Dalmanuta kwil sétáh yõ. 
Képah tɛ̃̀ sɔkɔ, *Farisiyɛ̃nɔ pɔ tyah yékã̂blɛ n pɩ Yesulɛ, tɛ́ ǹ sõ̀lɛ n mɔ n yah nɛ, à gbǐl yɔ́ pɩ pé yah, nɛ̂-á kè wɛ̃kɩ n pi nɛ, wǎh n wil Liyel tyɩ́. 
Képah sɔkɔ, Yesu mɩ dɔkɔ náhnáh, tɛ́ yo pé tyɩ́: «Kû pɩ, tɛ́ kwɛy gbĩ névye gbǐlnɛ n yah n kɔ̃? Kègbɩ yõ, mé kélɛ n yo yé tyɩ́, gbǐl yɔ́ ńtɛ̃ náh pè wɛ̃kɩ n pi!» 
Tɛ́ yal pé tnɔ̂, à de krótókè, à yuku yɔ́-ǹgbɛ̃ kahnɩnɛ. 
Légbĩ́nɛ, Yesu *tõ̀ sã́hpú sõ̀ pé dyɔ fwɔ yõke tɔ́kɔ́nɔ́lɛ. Búrn pɔ́ núkú ó sõ̀ mɛ pé tyɩ́ krótókè sɔkɔ. 
Képah sɔkɔ, Yesu mɛ kè gbah yo pé tyɩ́: «Yé yé gblɔ̌ynɛ n kyɔmɩ *Farisiyɛ̃nɔ ànɛ̂ *Yerɔdi búrn dɔkɔ dɔkɔ kwâhlɛ.» 
Wǎh yo sɛ̃́, ǹ tõ̀ sã́hpú yékã̂b mɔ nɛ, búrn kɛ̃́ yĩn-á à kɔ̃, à képah yo. 
Wǎh pè kyɔmɩ wɛ, à pè piki yah nɛ, kû se pɩ, tɛ́ pè yékã̂blɛ n pɩ búrn kɛ̃́ yĩnnɛ nɩ? Pè kó tɔ̃̌nɩ̀ mɛ ké tyɩ́ yah wɛ sɛ̃́ nɩ? Pè kó kè pɩ̃ nɩ? Pé lékã́m se mɛ víkéntũnɔlɛ nɩ? 
Yɩ̃́-á mɛ pé tyɩ́, pè kó mɛ n yah nɩ? Ní mɛ pé tyɩ́, pè kó mɛ n nohnɩ nɩ? Pé dyɔ kó mɛ n kwnɛ nɩ? 
Páh lésõ búrn plɔ́ kwlɔ́ kwɔkɩ dibi nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãm kɔ̃, gbétɔ́lɔ́ yɩ̃́ sɔ̃́ se pè kwel yṍnkwlɔnɔlɛ nɩ? Pè mɛ nɛ, gbɔ̃ninɔ́-á. 
À tɔ̃ pè piki yah nɛ, tɛ́ páh lésõ búrn plɔ́ kwǎrninɔ kwɔkɩ dibi nɛ́pĩ̂ kèyõ nínɔ́ kɔ̃, tɔkɔ yɩ̃́ sɔ̃́ se pè kwel yṍnkwlɔnɔlɛ nɩ? Pè mɛ nɛ, kwǎrninɔ-á. 
Képah sɔkɔ, à yo pé tyɩ́ nɛ, tɛ́, pè náh tɔ̃̌nɩ̀ mɛ ké tyɩ́ yah wɛ sɛ̃́ nɩ? 
Yesu ànɛ̂ ǹ *tõ̀ sã́hpú-á kyɩ yĩni tɛ̃ Besayida kwil, nósyɔ́ pɔ yɩ̃́tũ̂ yɔ́lɛ ǹ tyɩ́, tɛ́ à ni nɛ, à tũ ǹ tyɩ́. 
Yesu mɛ yɩ̃́tũ̂ mɛ̀ gbõ̀ sɔkɔ tɛ̃, à wil ǹnɛ kwil gbɔ̀pɔ, tɛ́ sĩn mɔ ǹ yɩ̃́ sɔkɔ, tɛ́ ǹ gbã̀n sah ǹ yõ, tɛ́ à piki yah nɛ, à se mɛ n yah nɩ? 
Yɩ̃́tũ̂ dĩ́ ǹ yah yɔ̃, tɛ́ nɛ, páh mɛ névyelɛ n yah yìrkɔ̀ɔ̀, sésân sɔ̃́. Tɛ́, pǎh mɛ n kɔ. 
Yesu tɔ̃ ǹ syɩ ǹ gbã̀n sah dĩ́ yɩ̃́ yah. Kǎh pɩ sɛ̃́, à fɛ̃ n yah pnɛpnɛ, à dɛ. À yíyìn póllɛ n wah n pnɛ wɛ̃ sɔkɔ. 
Képah sɔkɔ, Yesu yo dĩ́ mɛ̀ tyɩ́ nɛ, à sɔkɔ ǹ gbô, tɛ́ à káh syɔ̃ kwil sɔkɔ. 
Képah náh, Yesu ànɛ̂ ǹ *tõ̀ sã́hpú yuku Filipe Sesarɩ gbɔ̀plɔ kwlo tĩ̂nnɛ. Pǎh mɛ wɛ̃̀kɩ̀ yõ gbĩ́ nɛ̂nɛ, à pè piki yah: «Névye tɛ̃̀ tyɩ́, ńmɔlɛ, nɛ̂ mɛ?» 
Ǹ tõ̀ sã́hpú nɛ, túkù-á nɛ, *Wèlnwìlkì-ò Nsyɔ̃-á ǹnɛ. Túkù mó nɛ, *Yeli-á. Túkù tɔ̃ nɛ, *Liyel tɛ̃ntõ̀ syínyo-ò yɔ́-á ǹnɛ. 
À mɛ tɔ̃ pè piki yah: «Tɛ́ yépi tɛ̃̀ tyɩ́, ńmɔlɛ, nɛ̂ mɛ?» Képah tɛ̃̀nɛ, Pyɛrɩ yo ǹ tyɩ́ nɛ: «*Krista ye álɛ, Liyel-á nɛ, páh kɩ nɛ̂ tɛkɩ mɔ.» 
Képah sɔkɔ, Yesu pè sni nɛ, pè káh kè yo nɛ syah. 
Képah sɔkɔ, Yesu tyah ǹ *tõ̀ sã́hpúlɛ n kwɔ nɛ, *Névi Pi-á yɩ̃nɛ à de yèvyãhrɩ sɔkɔ náhnáh. Yèwàhnɩ̀, ànɛ̂ *sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú-á kɩ à yahle, tɛ́ kɩ à ko n mɔ n pi. Tɛ́, sèpĩ̀n tɔ̃́-ò sṍlɛ, wǎh kɩ yuku. 
Wǎh kénɛ wɛy yo dal mɔ pè syah sɛ̃́, Pyɛrɩ mɛ à ye kɛ̃kɩ, à kyɩ tyah lékàhlɛ ǹnɛ n kɔ̃. 
Képah sɔkɔ, Yesu ǹ yah vi ǹ tõ̀ sã́hpú yah, tɛ́ lékàhlɛ Pyɛrɩ kɔ̃, tɛ́ yo ǹ tyɩ́: «To ń tnɔ̂, *sétã̀n yõ̀tɛ̃̀! Á fɔkɔ sõnɔ náh mɛ núkúlɛ Liyel tyɩ́lɛ. Névye tyɩ́ kɔ̃ ye áwɔ n sõ.» 
Képah náh, à mɛ tɔ́wû ye pɔ ànɛ̂ ǹ *tõ̀ sã́hpúlɛ, tɛ́ yo pé tyɩ́: «Névi nɛ̂-á mɛ ké tyɩ́ nɛ, pé ńmɔ sõ̀ tɛ̃, tɛ̃̀ ǹ gblɔ̌y dyɔ́yĩ tyi yɛ, à mó ǹ *dahnɩ twah, à mó ńmɔ sõ̀ tɛ̃. 
Sɛ̃́ ó ye: Névi nɛ̂-á nɛ, páh pé minnɛ n pwah n mɔ, à kɩ tɛkɩ ké sɔkɔ. Tɛ́ névi nɛ̂-á tɛkɩ ǹ min sɔkɔ ńmɔ ànɛ̂ nɩ̀vɩ̀nɩ̀ wɛy sã́n yĩnnɛ, ǹmɔ min kɩ pwah. 
Kè kɩ tir nɛ̂ pɛkɩ névi tyɩ́, wàh kèkõyṍ yî pól wɛ, tɛ́ kɩ n tɛkɩ n pi tyɛ́-ńkɛ̃̂ min sɔkɔ? 
Kwâh nɛ̂ tɛ̃̀ mɛ névi kɩ fɛ̃ kɔ̃, kɩ ǹ min dwe syi sah? 
Kwɛy gbĩ́ névyelɛ, Liyel dêl ànɛ̂ tyípêl pɩ́pú ye pélɛ. Ǹtɛ, névi nɛ̂-á sen ku ńmɔ tyɩ́, ànɛ̂ ń wɛy tyɩ́ yónɔ́lɛ pépi wrɔ́, *Névi Pi tɔ kɩ sen ku tɛ̃̀ yah, wàh sṍ nɛ̂nɛ pɔ ǹ sú gblò sɔkɔ, ànɛ̂ *yĩ̂nyõ tɛ̃̀ntõbelɛ.»
