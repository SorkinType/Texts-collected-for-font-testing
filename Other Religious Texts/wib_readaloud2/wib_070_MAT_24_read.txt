Matye.
24.
Yesu-á wil Liyel ní-ńsah gbô sɔkɔ tɛ́ n sɔkɔ gbĩ́ nɛ̂nɛ, ǹ tõ̀ sã́hpú mɛ nɔ de ǹ tnɔ̂ à Liyel ní-ńsah gbô mɔ dùkùlɛ n sõ ǹ tyɩ́. 
Ǹtɛ à mɛ pè syah nɛ, kǎh pè n yah yɔ̀ kɛ̀; kègbɩ yõ páh kélɛ n yo pé tyɩ́, ké pól-á kɩ pɔ sɔ. Dáhkɩ̀ núkú yɔ́ ńtɛ̃ náh tɔ̃ n wũ n pi ké ní-ò yõ. 
Wǎh kyɩ kɔ̃ tɛ̃ gbĩ́ nɛ̂nɛ Wólívyê yèh yɔ̃lɩ yõ, ǹ tõ̀ sã́hpú mɛ nɔ de ǹnɛ pé yahlɛ, tɛ́ à piki yah nɛ, à yo pé syah, gbĩ́ nɛ̂ se képah kɩ n pɩ n pi nɩ? Tir nɛ̂ pɩ́nɔ́ se kɩ ǹ syɩ́npɔnɔ ànɛ̂ kèkõyṍ tyɛ́nɔ́ wɛ̃kɩ n pi nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Yé yah, névye náhnáh kɩ pɔ ńmɔ yĩn tɔkɔ n pi pé yah yĩ́ĩ́nɛ, tɛ́ kɩ nɛ pépi-á ńmɔlɛ, nɛ̂-á Liyel nɛy yáhntɔkɔnɔlɛ. Pè kɩ névye náhnáh mɔ n pi pílkínɔ́ sɔkɔ. Ǹtɛ yépi tɛ̃̀ wɔ yé gblɔ̌ynɛ n kyɔmɩ, nɛy káh n fɛ̃ yé pilki. 
Yé kɩ mâl pɩ́nɔ́ tyɩ́ yónɔ́ noh n pi, ànɛ̂ mâl tyɩ́ dírí wɔ́nsàhnɔ́lɛ. Ǹtɛ yé káh n yɛ tíkí yé tɛ̃. Kè yɩ̃nɛ kè pɩ sɛ̃́. Ǹtɛ képah náh tɔ̃̌nɩ̀ n pɩ n pi kèkõyṍ tyɛ́nɔ́lɛ. 
Yé náh wɛ di, kwil yɔ́ kɩ pɔ yuku kwil yɔ́lɛ n fwo, tɔ̃́rɩ́ yɔ́ kɩ pɔ yuku tɔ̃́rɩ́ yɔ́lɛ n fwo. Tɛ́ kwɛkɩ ànɛ̂ sétáh syahrɩ kɩ n wɛ n pi tã̀ntã̀n náhnáh sɔkɔ. 
Ǹtɛ sépi pól kɩ mɛ á kɩ nɛ kúrkì tɛ̃ gbĩ́ yɔ́nɔ́ sõ tyàhnɔ́ sɔ̃́. 
Pè kɩ yélɛ n tɛ̃ pè kɔ̃ pè kɩ n fõh, tɛ́ kɩ yélɛ n ko mɔ. Kèkõyṍ névye pól kɩ yé wɛ mɔ ńmɔ yĩnnɛ. 
Képah tɛ̃̀ sɔkɔ, névye náhnáh kɩ Liyel wɛ̃̀kɩ̀ fĩn mɔ. Pè kɩ n wil wɛ̃ náh, tɛ́ kɩ wɛ̃ dyɔlɛ n yɔ. 
Tɛ́ Liyel yĩn yõ mɩ̀nyobe kɩ wil mɔ pí, kɩ névye náhnáh mɔ pílkínɔ́ sɔkɔ. 
Pèkè pɩ́nɔ́ kɩ yah syi, fɔ́ɔ́ tyɩ́nɔnɔ náh tɔ̃ n wɛ n pi fwɔ́ɔ névye náhnáh sɔkɔ. 
Ǹtɛ nɛ̂-à ǹ tah sah à pɔ nɔ ké tyɛ́nɔ́ tyɩ́, tɛ̃̀ kɩ pwáhnmɔnɔ wɛ. 
Képah sɔkɔ, Nɩ̀vɩ̀nɩ̀ tir kɩ yo Liyel tɔ̃́rɩ́ tyɩ́ yõ kèkõyṍ pól sɔkɔ, névye pól kɔ̃ pè kɩ noh. Képah náh, kèkõyṍ kɩ pyě tyɛ.» 
Képah sɔkɔ, Yesu tɔ̃ nɛ: «Liyel tɛ̃ntõ̀ syínyo-ò Danyɛlɩ-á lésõ yɩ́kɩ́nɔ́ pɩ pɩ kwâh képékè nɛ̂ tyɩ́ yo sah, yé kɩ kè wɛ n pi Liyel ní-ńsah gbô tyah kètǎr sɔkɔ.» — Nónó-à kɩ wɛy mɛ̀nɛ n kar, pè gbah n tɛ̃ ké dùkù pɩ̃. — 
Yesu tɔ̃ yɔ̃ sɔkɔ ǹ wɛynɛ nɛ ńkɛ̃́: «Yâh képah wɛ gbĩ́ nɛ̂nɛ, nónó-à kɩ mɛ Nsyude mara sɔkɔ, pè to sɔkɔ ymɔ yõ. 
Névi nɛ̂-à kɩ mɛ gbɛ̃́ntĩ̂n yõ, tɛ̃̀ káh nɛ páh n tiki pé vyɔ́lɛ n tɔkɔ pé wɔ̀kɔ̀ sɔkɔ. 
Nɛ̂-à kɩ se mó mɛ ǹ swãh sɔkɔ, tɛ̃̀ káh nɛ páh pélɛ n syɩ n yuku pé fwɔ̀mɔ̀-ǹgbɛ̃ tɔ́kɔ́-ńsah. 
Kénɛ sèpĩ̀n-à pɔ syɔ́ nónó wɛ kûrn nátãnnɛ, ànɛ̂ nónó-à mɛ yɔ̃kɩ syɔ́lɛ, kè kɩ pɩ yèwàhtàhpolɛ pé tyɩ́. 
Képah ye, yé yɩ̃nɛ yè Liyellɛ n ni nɛ tólɔ́ mɛ̀ káh n pɩ vɩrɩ gbĩ́nɛ, képah náh pɩ mɩ mɩ yòh sõlɛ. 
Kè kɩ pɩ yèvyãhrɩ ǹgbɛ̃nɛ, nɛ̂ kénkɔ̃ náh wɛ yah kèkõyṍ mɔ́mɔ́nɩ̀ à pɔ syɩ kwɛy tyɩ́, ànɛ̂ nɛ̂ kénkɔ̃ náh tɔ̃ n wɛ n yah n pi póllɛ. 
Liyel-à náhkɩ sõ̀ wìlkìnɔ́ pɩ́-ńkɛ̃̂nɛ kénɛ sèpĩ̀n sɔkɔ, ǹnɩ̂ yɔ́ ńtɛ̃ yõ náh náhkɩ pɔ n wil n pi ké sɔkɔ. Ǹtɛ wǎh nónó yah wilki pɩ ǹ tyɩ́lɛ, pépi yĩnnɛ, à wìlkìnɔ́ pɩ sé sɔkɔ. 
Névi nɛ̂-à yé syah nɛ, Liyel nɛy yáhntɔkɔnɔ-á mɛ ásõ̂, képah náh pɩ wǎh mɛ yàhàh, yé káh sɛ̃ tɛ̃̀ yõ. 
Yé yah, mɩ̀nyobe kɩ pɔ wil mɔ pí. Pé túkù kɩ nɛ Liyel nɛy yáhntɔkɔnɔnɔ-á pépilɛ, túkù kɩ mó nɛ Liyel tɛ̃ntõ̀ syínyopu-á pépilɛ. Pè kɩ gbǐlgbìl tyi ǹgbnɔlɛ n pɩ, fɔ́ɔ́ kɩ n yah n kɔ̃ Liyel névye yáhnwìlkìnɔ́ ńtɛ̃ tɔ yõ yɩ́kɩ́-ńsahlɛ kàh sõ̀ fɛ̃ n pɩ n pi. 
Tyi nɔ́pi ye mé kwãh tɛ̃ yo sah yé tyɩ́ yɔ̀. 
Képah sɔkɔ, névi nɛ̂-à yé syah nɛ Névi Pídĩ́-á mɛ gbɛ̃́nnɩ sɔkɔ, yé káh nɛ yáh ǹ nyáhlɛ n sõ. Nɛ̂-à se nɛ wǎh mɛ wɔ̃̀bɩ̀ntɛ̃̀nɔ́lɛ, yé káh sɛ̃ tɛ̃̀ yõ. 
Sɔ̃́-á pũ̀ n yĩkiri yã̌l yè n wɛ, à tɔkɔ gbĩ̀ngboke tyɩ́, fɔ́ɔ́ n kyɩ n dal n mɔ gbĩ̀nyiki tyɩ́, sɛ̃́ ye Névi Pídĩ́ syɩ́npɔnɔ tɔ kɩ n wɛ n pi. 
Kwáhkókè se kɩ fɛ̃ wɔ̃bɩ ǹkyɔ̀nkyɔ̀nɩ̀ yah?» 
«Kénɛ sèpĩ̀n nɔ́pi yèvyãhrɩ-à fyɔ̀ sɔkɔ n tɛ ǹsɔ̃̀, gbĩ kɩ pɩ syìì, ńwɔ̃́ náh tɔ̃ n piki n pi, ńwɔ̃́ kèpye kɩ wil yĩ̂nyõ kɩ kwɛ, tɛ́ yĩ̂nyõ yíyìn pól kɩ yike. 
Légbĩ́nɛ, Névi Pídĩ́ syɩ́npɔnɔ kwní kɩ yõ mɔ yĩ̂nyõ. Képah tɛ̃̀nɛ, sétáh yõ yìkìnì pól névye kɩ n gbo kɩ yãm yãm tyilɛ n pɩ. Tɛ́ pè kɩ Névi Pídĩ́ wɛ à kɩ mɛ n pi ńyĩ̂rn sɔkɔ, tǎhkɩ̀ ó ǹ tɔ̂ɔ́ ànɛ̂ tɔ̃́rɩ́lɛ. 
Kèpɔ̃ ǹgbɛ̃ yɔ́ kɩ fɩ̃bɩ, tɛ́ à kɩ ǹ yĩ̂nyõ tõ̀npɩbɩ tɛkɩ mɔ ywɔ̃ ywɔ̃ pól sɔkɔ, à tɔkɔ kèkõyṍ yah mɛ̀ tyɩ́, à kyɩ sah ké yah mɛ̀ tyɩ́. Pè kɩ kyɩ névye tuke mɔ, Liyel-á nónó yah wilki. 
Yé yah sènsóké kèdùkù-á nɛ̂nɛ álɛ n wɛ̃kɩ. Yâh fyɔ̀ ké yɩ̃̀ngbã̀n wɛ sè n gboke n wil à mɩ̀snɔlɛ n pɩ gbĩ́ nɛ̂nɛ, yè n pɩ̃ nɛ ki de gbĩ́-á mɛ yúkɛy. 
Sɛ̃́ ó ye, yépi tɔ-à pópó kénɛ tyi wɛ sè tyah n pɩ gbĩ́ nɛ̂nɛ, yé pɩ̃ nɛ Névi Pídĩ́ syɩ́npɔnɔ-á mɛ yúkɛy. Wǎh kɩ pɔ sɛ̃́ yɔ̀ sɛ̃́ yɔ̀. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, kwɛy gbĩ névye pól náh n ku n tɛ n pi, tɛ́ kénɛ tyi nɔ́pi kɩ mɛ n pɩ. 
Yĩ̂nyõ ànɛ̂ sétáh kɩ pɔ tyɛ, ǹtɛ ńmɔ wɛlɔ tɛ̃̀ wɔ náh n tyɛ n yah n pi póllɛ. 
Ǹtɛ ké lésõ ànɛ̂ ké gbĩ́ tɛ̃̀ wɔlɛ, ǹnɩ̂ yɔ́ ńtɛ̃ náh sépilɛ n pnɛ. Yĩ̂nyõ tõ̀npɩbɩ náh sélɛ n pnɛ, Liyel Pídĩ́ tɔ náh sélɛ n pnɛ. Ǹnɩ̂ yɔ́ ńtɛ̃ náh sélɛ n pnɛ à wil Sú Liyel tyɩ́ sɔkɔ. Ǹmɔ ó ye n pnɛ sélɛ. 
Nɛ̂-á pɩ sɔkɔ Nowe gbĩ sɔkɔ, sɛ̃́nkɔ̃ ye kɩ n pɩ n pi Névi Pídĩ́ syɩ́npɔnɔ sɔkɔ. 
Yé náh wɛ di, gbĩ́ nɛ̂-á lésõ pṹnpã́hkɩ̀ náh tɔ̃̌nɩ̀ mɛ pɩ à ninɛ kèkõyṍ pól yõ, névye lésõ mɛ n yõ, tɛ́ mɛ n wɔ, tɛ́ mɛ syɔ́lɛ n tɔkɔ, képah náh pɩ mɛ pé pyelɛ pɛbɩlɛ n kɔ̃, pópó à pɔ syɩ Nowe de sõ tyɩ́ krótó-ǹgbɛ̃ sɔkɔ. 
Pè náh sõ̀ mɛ n sõ tir yɔ́ ńtɛ̃ tyɩ́ sɔkɔ, fɔ́ɔ́ à pɔ nɔ pṹnpã́hkɩ̀ pɩ sõ tyɩ́, ni kɔ̃ kè pɔ pé pól yõ. Sɛ̃́nkɔ̃ ye kɩ n pɩ n pi Névi Pídĩ́ syɩ́npɔnɔ sɔkɔ. 
Díbí nɛ́pĩ̂ nímí kɩ mɛ swãh sɔkɔ, núkú kɩ tɛ̃ sɔkɔ, tɛ́ ǹ ní-ò kɩ wũ. 
Syɔ́ nɛ́pĩ̂ nímí kɩ mɛ nannɛ n nɔ, núkú kɩ tɛ̃ sɔkɔ, tɛ́ ǹ ní-ò kɩ wũ. 
Yé Tɛ̃̀ ǹgbɛ̃-á ǹ syɩ n pɔ n pi sõ nɛ̂nɛ, yé náh mɛ kè pɩ̃. Képah ye, yé káh yé dyɔ mɔ n fwɔ n tɛ̃. 
Yépi ńtɛ̃ pɩ̃ nɛ, gbóntɛ̃̀ dĩ́-à mɛ pɩ̃ gbĩ́ nɛ̂-á yɛ̃́nwó kɩ n pɔ n pi lékã̂hnɛ, à kɩ yah sõ, à náh yɛ à kɔ̃ à wɛ kwɔ n de n pi ǹ gbô sɔkɔ. 
Képah ye, yépi tɔ nɛ mɛ yé vi sah. Ǹgbǒ yé náh mɛ gbĩ́ nɛ̂ tyɩ́lɛ n sõ, légbĩ́ ye Névi Pídĩ́ kɩ ǹ syɩ n pɔ n pi dɛ́.» 
«Tõ̀npɩ nɛ́gbɩ́ ànɛ̂ lékã́m névi nɛ̂ mɛ yõ̀tɛ̃̀ dĩ́ kɩ tɔkɔ sah ǹ tõ̀npɩbɩ tɛ́lɔ́ yõ, à kɔ̃ à kɩ yõkelɛ pélɛ n kɔ̃ ké gbĩ́ pèpɛy sɔkɔ, tɛ́ kɩ sɔkɔ kɔ́lɔ́ sɔkɔ? 
Yõ̀tɛ̃̀ dĩ́-à ǹ syɩ pɔ à wɛ à mɛ kénɛ tõ̀lɛ n pɩ ké yah yõ, tõ̀npɩ mɛ̀ tyɩ́ nɔ. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, yõ̀tɛ̃̀ dĩ́ kɩ à pɩ ǹ gbã̀n yî pól kyɔ̀mɩ̀-òlɛ. 
Ǹtɛ kàh pɩ tõ̀npɩ túkùlɛ, tɛ́ à tyah n sõ nɛ pé yõ̀tɛ̃̀ dĩ́-á mɛ mo ǹ syɩ́npɔ-ńsah, 
tɛ́ tyah pé no tõ̀npɩbɩ tɛ́lɔ́lɛ n ko, tɛ́ n yõ tɛ́ n wɔ sètɛ̃̀nɩ̀ pɩ́púlɛ wɛ̃ tyɩ́, 
ǹ yõ̀tɛ̃̀ dĩ́ kɩ ǹ syɩ pɔ ǹ yõ dɔkɔ sõ yɔ́lɛ, ànɛ̂ wɔ̂ yɔ́ sɔkɔ, à náh mɛ nɛ̂ tyɩ́lɛ n sõ. 
À kɩ tõ̀npɩ mɛ̀ yĩni, tɛ́ kɩ à tɛkɩ mɔ yĩ́nvǐnpɩbɩ mɛ́-ńsah sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ.»
