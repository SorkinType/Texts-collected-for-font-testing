Likyi.
7.
Képah sɔkɔ, Yesu-á wɛlɔ nɔ́pi pól yo pɔ tyɛ no tyɩ́, à de Kapɛrnayumi. 
Tɛ́ nɛ́npɔ́ sɔkɔ, tõ̀npɩ yɔ́ sõ̀ mɛ sràsyíbí yõ̀tɛ̃̀ dĩ́ yɔ́ tyɩ́. Á sõ̀ mɛ n yɔ képékèyɔ̂, mɛ n tyah kúnɔ́lɛ. Tɛ́ ǹ tyɩ́ sõ̀ tah yĩ́ĩ́ ǹ yõ̀tɛ̃̀ dĩ́ tyɩ́. 
Wǎh Yesu tyɩ́ noh, à mɛ Nsyifunɔ yèwàhnɩ̀ yísyɔ́ tɛkɩ mɔ nɛ, pè kyɩ à ni, à pɔ pé tõ̀npɩ dɛ mɔ. 
Pǎh kyɩ yĩni tɛ̃ Yesu tnɔ̂, pè tyah ǹnɛ n gbah n ni, mɛ n yo ǹ tyɩ́ nɛ, pépi Nsyifunɔ tyɩ́-á kah nɔ yõ̀tɛ̃̀ dĩ́ mɛ̀ tyɩ́. Ànɛ̂, ǹmɔ-á tɔ̃ yĩn pé Liyel ní-ńsah wɔ̀kɔ̀ mɔ́nɔ́lɛ. Képah-á, wǎh yɩ̃nɛ yòhnɩ̀nmɔnɔ mɛ̀ pɩ́nɔ́lɛ ǹ tyɩ́. 
Képah sɔkɔ, Yesu mɛ sɔkɔ pélɛ. Wǎh de n tɛ sràsyíbí yõ̀tɛ̃̀ dĩ́ mɛ̀ kételɛ, kénɛ dĩ́ tɔ̃ ǹ kódíbí yísyɔ́ tõ nɛ ńkɛ̃́, pè pɔ yo Yesu tyɩ́ nɛ, pé Yõ̀tɛ̃̀, péwɔ náh yɩ̃nɛ ǹ dénɔ́lɛ pé kéte sɔkɔ. Képah-á, à káh ǹ gblɔ̌y kwlɔ pɔ. 
Ńkɛ̃́, képah tɔ-á pé kɔ̃ pé kè yah nɛ, pé náh yɩ̃nɛ pɔ́nwìlnɔ́lɛ ǹ tyɩ́ pé gblɔ̌y. Ǹtɛ, à wɛy núkú ó yo, pé tõ̀npɩ-á kɩ dɛ. 
Ńkɛ̃́, péwɔ ńtɛ̃ tɔ tyɩ́ yõ̀tãm-á mɛ, péwɔ tɔ mó mɛ sràsyíbí yísyɔ́ yõ. Pâh yo nɛ̂ tyɩ́: «Sɔkɔ!», tɛ̃̀ n sɔkɔ. Pâh yo nɛ̂ tyɩ́: «Pɔ!», tɛ̃̀ n pɔ. Ànɛ̂, pâh yo pé tõ̀npɩ tyɩ́: «Tir mɛ̀ pɩ!», ǎ kè pɩ. 
Képah sɔkɔ, Yesu-á kénɛ wɛlɔ noh sɛ̃́, à dĩ́ mɛ̀ sõ kègbɩlɛ. Tɛ́ ǹ yah vi yo ǹ sõ̀ tɛ̃́pú tɔ́wû tyɩ́ nɛ: «Mé kélɛ n yo yé tyɩ́ nɛ: Yisrayel ńtɛ̃ sɔkɔ, mé náh sɛ̃́nkɔ̃ Liyel yõ sɛ̃́nɔ́ névi wɛ yah.» 
Képah sɔkɔ, tɛ̃ntõ̀ névye-á pé syɩ kyɩ sràsyíbí yõ̀tɛ̃̀ kéte, pè kyɩ nɛ n yah, ǹ tõ̀npɩ mɛ̀ dɛ. 
Sṍ yɔ́lɛ, Yesu mɛ n yuku kwil yɔ́ sɔkɔ, nɛ̂-á pè n ye Nayinnɛ. Tɛ́ ǹ tõ̀ sã́hpú, ànɛ̂ tɔ́wû ǹgbɛ̃ sõ̀ mɛ ǹ sõ̀ tɛ̃. 
Wǎh yĩni tɛ̃ kwil dé-ńsahlɛ yúkɛy, à nósyɔ́ yohnɩ, pè mɛ lékó yɔ́ twah n yuku yɩ̃́-ńsah. Pɛ́kúsè yɔ́ pídĩ́ ye sõ̀, ǹ gblɔ̌y núkúnínɛ ǹ yṹ tyɩ́. Tɛ́, kénɛ kwil névye sõ̀ mɛ wil náhnáh, mɛ pi mɛ̀ sõ̀ tɛ̃. 
Tɛ̃̀ ǹgbɛ̃ Yesu-á pi mɛ̀ wɛ sɛ̃́, ǹ yãm à kah tɛ̃, à mɛ yo ǹ tyɩ́: «Káh n tɔ̃ nɛ n gbo!» 
Tɛ́ mɛ nɔ de, à tũ lékó twáhkɩ̀ tyɩ́, ké twáhpú núkú yĩn. Yesu mɛ yo nɛ: «Dĩ́npir, mé kélɛ n yo á tyɩ́: yuku!» 
Wǎh yo sɛ̃́, à núkú yuku kɔ̃ tɛ̃, tɛ́ tyah n yãh. Yesu mɛ ǹnɛ ǹ yṹ gbõ̀ dahbɩ. 
Képah tɛ̃̀nɛ, tíkí no pól tɛ̃. Pè tyah Liyellɛ n gbilki, mɛ yo tũ nɛ, Liyel tɛ̃ntõ̀ syínyo-ò ǹgbɛ̃ yɔ́-á wil mɔ pé wrɔ́. Ànɛ̂, Liyel-á yah dah mɔ ǹ no tyɩ́ sɔkɔ. 
Képah sɔkɔ, Yesu tyípɩ́nɔ́ tyɩ́ yónɔ́ tɔ̃ mɔ Yude tã̀ntã̀n pól sɔkɔ, à kyɩ sɔkɔ ké gbɔ̀plɔ kwlo tɔ sɔkɔ. 
Képah sɔkɔ, Nsyɔ̃ tõ̀ sã́hpú tyi nɔ́pi pól yãh Nsyɔ̃ tyɩ́. À mɛ pé sɔkɔ nɛ́pĩ̂ nímí ye, 
à tõ Tɛ̃̀ ǹgbɛ̃ Yesu tyɩ́. Ńkɛ̃́, pè kyɩ à piki yah nɛ, ǹmɔ se Liyel nɛy tɛ́kɩ́nmɔnɔlɛ, nɛ̂-á kɩ sõ̀ n pɔ n pi, tĩ̂ pé se yɩ̃nɛ pé ǹnɩ̂nɛ n syɩkɩ nɩ? 
Pǎh pɔ yĩni tɛ̃ Yesu tnɔ̂, pè yo ǹ tyɩ́ nɛ, Wèlnwìlkì-ò Nsyɔ̃-á pépi tɛkɩ mɔ ǹ tyɩ́ nɛ, pé pɔ à piki yah nɛ ńkɛ̃́, ǹmɔ se Liyel nɛy tɛ́kɩ́nmɔnɔlɛ, nɛ̂-á kɩ sõ̀ n pɔ n pi, tĩ̂ pé se yɩ̃nɛ pé ǹnɩ̂nɛ n syɩkɩ nɩ? 
Légbĩ́nɛ, Yesu mɛ yámpú náhnáhlɛ n dɛ n mɔ pé yamnɔ, ànɛ̂ pé yèvyãhrɩnɔ sɔkɔ. À sétã̀n yĩni wilki pé sɔkɔ, tɛ́ yɩ̃́tũbi náhnáh yɩ̃́ yɩkɩ mɔ. 
Tɛ́ mɛ yo Nsyɔ̃ tɛ̃ntõ̀ névye tyɩ́: «Yáh nɛ̂ wɛ, tɛ́ nɛ̂ noh, yé kyɩ kè yãh Nsyɔ̃ tyɩ́: Yɩ̃́tũbi yɩ̃́ mɛ n yɩkɩ n mɔ, sèkukubi mɛ n kɔ tómtóm, tótãm mɛ n dɛ, níkítũbi mɛ n nohnɩ, lékyɩ̂ mɛ n yilki n mɔ, ànɛ̂ nɩ̀vɩ̀nɩ̀ wɛy sã́n mɛ n yo n kɔ̃ yãm tãn tyɩ́. 
Ǹtɛ, nɛ̂-á Liyel yõ sɛ̃́nɔ́ yáhle-ńkɛ̃̂nɛ ńmɔ yĩnnɛ, yèvɩnɩ mɛ tɛ̃̀ tyɩ́.» 
Képah sɔkɔ, Nsyɔ̃ tɛ̃ntõ̀ névye-á sɔkɔ tɛ, Yesu tyah ǹ tyɩ́lɛ n yãh tɔ́wû tyɩ́ mɛ n yo: «Kwâh nɛ̂ mɛ yé kyɩ yah gbɛ̃́nnɩ sɔkɔ? Fílkì yɔ́ se yã́hkɩ́ mɛ n fĩ n kɔ̃? 
Tĩ̂ kwâh nɛ̂ mɛ yé kyɩ yah wɛ? Dĩ́ yɔ́ se, nɛ̂-á mɛ fwɔ̀pèplɔ mɔ? Tɛ́ nónó-á mó tɔ̃ fwɔ̀tǎhnɩ̀nɛ n mɔ, tɛ́ tɔ̃́rɩ́ yélɛ n yõ, pépi náh mɛ yõ̀tãm fnɛ́ sɔkɔ di? 
Ǹtɛ, kwâh nɛ̂ mɛ yé kyɩ yah wɛ? Liyel tɛ̃ntõ̀ syínyo-ò yɔ́lɛ nɩ? Mé kélɛ n yo yé tyɩ́ nɛ: sɛ̃́-á. Tɛ́ ǹmɔ ńtɛ̃-á kal Liyel tɛ̃ntõ̀ syínyo-òlɛ. 
Ńkɛ̃́, ǹmɔ tyɩ́ sɔkɔ-á kè wãrkɩ sah nɛ ńkɛ̃́: ‹Yáh, ńmɔ Liyel kɩ ń tɛ̃ntõ̀ yó-ò tɛkɩ mɔ á yahlɛ, à kɩ pɔ á wɛ̃̀kɩ̀ tɔ̃ sah.› 
Mé kélɛ n yo yé tyɩ́ nɛ: sê fɔkɔ táh pi yɔ́ ńtɛ̃ náh mɛ nɛ̂-á kal Nsyɔ̃nɛ. Ǹtɛ, nɛ̂-á tyɩ̃́ɩ̃ kal Liyel yõ̀tɔ̃rɩ́ sɔkɔ, ǹmɔ ńtɛ̃ gblò kal ǹnɛ. 
Képah sɔkɔ, nónó pól-á sõ̀ mɛ n nohnɩ ǹ tyɩ́, à tahlɩ mɔ́rɩ́ syípú tɔlɛ, pé pól nɛ, Liyel-á mɛ tómm, tɛ́ Nsyɔ̃ kɔ̃, à pè wel wilki. 
Ǹtɛ, Liyel-á nɛ̂ yah kɔ̃ Farisiyɛ̃nɔ ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú tyɩ́, pè pé náh vi kè kɔ̃, tɛ́ yahle pé wèlnwìlkìnɔ́lɛ Nsyɔ̃ tyɩ́.» 
Tɛ́ Yesu tɔ̃ nɛ: «Nɛ̂ tɛ̃̀ mɛ mé kɩ tɔkɔ sãh kwɛy gbĩ névyelɛ? Nɛ̂ tɛ̃̀ mɛ pè tɔ? 
Pè mɛ, á kɩ nɛ wɛ̃́npì sɔ̃́ kɔ̃ tɛ̃ n syah tɔ́wû tùkè-ǹsah sɔkɔ, tɛ́ mɛ n yo wɛ̃ tyɩ́ nɛ, pépi-á yɩkɩ fɩ̃bɩ pé tyɩ́, tɛ́ pé náh se yi. Pé tɔ̃ kũ yɩ̀kɩ̀ gbɩ mɔ, pè náh se tɔ̃ gbo. 
Ǹtɛ, Wèlnwìlkì-ò Nsyɔ̃ pɔ, à náh mɛ n yõ, à náh mɛ dìvɛ̃nɛ n wɔ, tɛ́ yé nɛ, ǹ yõ̀gbèkè náh mɛ. 
Névi Pi-á se pɔ, à n yõ tɛ́ n wɔ, yé tɔ̃ nɛ, pè yah yèwònpɩ, ànɛ̂ sètɛ̃-á ǹnɛ, mɔ́rɩ́ syípú ànɛ̂ tyípêl pɩ́pú kódĩ́-á. 
Ǹtɛ, Liyel lékã́m tyilɛ, gbɩ ye sélɛ sé fɛ̃̀nsyipu tyɩ́.» 
Farisiyɛ̃ dĩ́ yɔ́ sõ̀ mɛ, pè n ye Simɔlɛ. Sṍ yɔ́lɛ, à Yesu ye nɛ, à pɔ yõke yõ pé tyɩ́. À mɛ kyɩ de kénɛ dĩ́ gbô, à tɛ̃ yõke yah. 
Tɛ́ kénɛ kwil sɔkɔ, tyípêl pɩ́-ò pi yɔ́ sõ̀ mɛ. Wǎh noh nɛ, Yesu-á mɛ yõkelɛ n yõ Farisiyɛ̃ dĩ́ gbô, à pɔ nɩrɩ pãn pènɛ̀kɩ̀ yɔ́lɛ, so pèpɛy yɔ́ sɔkɔ. 
Tɛ́ mɛ vi Yesu náh tĩ̀nnɛ, à n gbo ǹ yétĩnɛ n kwɛ Yesu gbyẽ̂h yõ, tɛ́ sélɛ n kele ǹ yõ̀yénnɛ, tɛ́ sè pɔrɩ tɛ̃, tɛ́ nɩrɩ mɛ̀nɛ n mɔ sé yõ. 
Farisiyɛ̃ dĩ́ nɛ̂-á ye Yesulɛ, ǹmɔ-á képah wɛ sɛ̃́, à tyah n sõ ǹ fɔkɔ sɔkɔ nɛ, dĩ́ mɛ̀-à sõ̀ mɛ Liyel tɛ̃ntõ̀ syínyo-òlɛ, pi nɛ̂-á n tũ ǹ tyɩ́ yɔ̀, wǎh náhkɩ ǹ dùkù pɩ̃ nɛ ńkɛ̃́, tyípêl pɩ́-ò ǹgbɛ̃-á ǹnɛ. 
Yesu mɛ wɛy tɔkɔ, tɛ́ yo Simɔ tyɩ́ nɛ, tir yɔ́-á mɛ pé tyɩ́ yo yo tɛ̃̀nɛ ǹ tyɩ́. Simɔ mɛ nɛ, à kè yo, kwɔ́-ò dĩ́. 
Képah sɔkɔ, à mɛ yo ǹ tyɩ́: «Kmɔ mɔ́-ò yɔ́ kmɔ sõ̀ mɛ nɛ́pĩ̂ nímí yísyɔ́ yõ. Núkú sõ̀ yɩ̃nɛ à à dyah pɔ́ plɔ́ kyɛkwɛ́lɛ, ní-ò mɛ yɩ̃nɛ à à dyah plɔ́ tikiníní pɔ́ gbãmnɛ. 
Ǹtɛ, pé sɔkɔ nɛy yɔ́ ńtɛ̃ náh sõ̀ mɛ fɛ̃ ǹ tyɩ́ dyah n pi sɛ̃́, dĩ́ pé nímí tyɩ́ yɔ̃ mɔ pè kɔ̃. Kǎh pɩ sɛ̃́, ǹ tyɩ́ kɩ nɔ kal nɛ̂ tɛ̃̀ tyɩ́ pépi nímí sɔkɔ?» 
Simɔ mɛ à syah nɛ, péwɔ tɛ̃̀ sõnɔ sɔkɔ, nɛ̂ kɔ̃lɩ-á kal, ǹmɔ-á. Yesu mɛ nɛ, gbɩ-á à yo. 
Képah sɔkɔ, tɛ́ ǹ yah vi pi tĩ̀nnɛ, tɛ́ yo Simɔ tyɩ́: «Á pi mɛ̀ wɛ kɛ̀? Ńmɔ-á de á gbô, á náh gbyẽ̂h wu wu ninɛ ń kɔ̃. Ǹtɛ ǹmɔ wɔ ǹ yétĩ kwɛ ń gbyẽ̂h yõ, tɛ́ sè kele ǹ yõ̀yénnɛ. 
Áwɔ náh ń pɔrɩ tɛ̃. Ǹtɛ, ǹmɔ tɛ̃̀nɛ, wǎh de pɔ sɔ̃́, à náh mɛ yal mɔ ǹ gbyẽ̂h pɔ́rɩ́ntɛ̃nɔlɛ. 
Á náh mó tɔ̃ nɩrɩ mɔ ń yõ, ǹtɛ ǹmɔ tɛ̃̀ wɔ nɩrɩ pãn pènɛ̀kɩ̀ mɔ ń gbyẽ̂h yõ. 
Mé kélɛ n yo á tyɩ́ nɛ: pi mɛ̀ tyípêl náhnáh nɔ́pi pól-á yɔ̃ mɔ à kɔ̃. Képah ye, à tyɩ́nɔnɔ ǹgbɛ̃ tir mɛ̀ pɩ. Ǹtɛ, yɔ̃̀nmɔnɔ-à pɩ nɛ̂ tyɩ́ kwéyy, ǹmɔ tyɩ́nɔnɔ n mó nɛ kwéyy.» 
Tɛ́ tɔ̃ ǹ syɩ yo pi mɛ̀ tyɩ́: «Á tyípêl ye yɔ̃ mɔ ǹ kɔ̃.» 
Képah sɔkɔ, wǎh sõ̀ mɛ n yõ ànɛ̂ nónónɛ, pè tyah n sõ pé fɔkɔnɩ sɔkɔ nɛ, nɛ́dúkú nɛ̂ se dĩ́ mɛ̀nɛ, tɛ́ à kɩ pɔ tyípêl tɔlɛ n yɔ̃ mɔ nɩ? 
Kǎh pɩ sɛ̃́, Yesu mɛ yo pi tyɩ́: «Ńmɔ yõ sɛ̃́nɔ́ ye ǹ pwah mɔ. Nɛ n yuku yèvɩnɩ sɔkɔ.»
