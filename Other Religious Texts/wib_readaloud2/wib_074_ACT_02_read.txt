Tyípɩ́nɩ̀.
2.
Syáh sõ-á pɔ nɔ, nɛ̂-á pè n ye Pãntekɔtɩ syáhlɛ, Yesu yõ sɛ̃́pú pól sõ̀ mɛ tùkènmɔnɔlɛ fɩ̃́ núkú sɔkɔ. 
Dǒ, yĩ́npɔ̃́ yɔ́ núkú n wil líyel sɔkɔ á kɩ nɛ yã́hpékè yɔ́ sɔ̃́ kõ, à pé mɛ́-ńsah wɔ̀kɔ̀ pól yuku. 
Tɛ́ kwâh yɔ́ núkú wil mɔ á kɩ nɛ nyàh lépɔ̃́ sɔ̃́, tɛ́ kɔ̃lɩ tɛ̃ tɛ̃ kɔ̃ pé nɛ́núkù núkú pól yõ. 
Liyel Mirki mɛ pé pól tɔkɔ. Pè tyah wɛ́nsnɔ́lɛ n wɛkɩ Liyel Mirki-á nónó dahbɩ pé vyɛ̃y. 
Tɛ́ légbĩ́nɛ, Nsyifunɔ nónó-á pé Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ plɛ, pè sõ̀ mɛ wil kèkõyṍ kwlo kwlo pól sɔkɔ, à pɔ twah Yerusalɛmɩ kwil. 
Yĩ́npɔ̃́ mɛ̀-á wil sɛ̃́, pé pól tuke mɔ tɔ́wûlɛ. Tɛ́ pé pól yáh-ńsah sõ̀ mɛ tyɛ, pé nɛ́núkù núkú pól-á sɔ̃́npɩ́lɛ pé kwlo wɛlɔ wɛ́kɩ́nɔ́lɛ n nohnɩ Yesu yõ sɛ̃́pú tyɩ́. 
Kǎh pɩ sɛ̃́, pé gbã̀n du, pé lékã́m fɛ̃kɩ, pè mɛ tyah n yo wɛ̃ tyɩ́ nɛ: «Nónó pól-á n wɛkɩ yɔ̀, pé póllɛ Kalile gbɛ̃̀nyah tãn náh di? 
Ǹtɛ kè pɩ sɔ̃́ tɛ́ ápi nɛ́núkù núkú pól á kwlo wɛlɔ wɛ́kɩ́nɔ́lɛ n nohnɩ pé tyɩ́, 
ápi nónó-á n wil gbɛ̃̀nyah nɔ́pi sɔkɔ, nónó-á pɩ: Partɛselɛ, ànɛ̂ Medilɛ, ànɛ̂ Yelamnɛ, ànɛ̂ Mesopotami tɔlɛ, ànɛ̂ Nsyudelɛ, ànɛ̂ Kapadɔsɩlɛ, ànɛ̂ Põtelɛ, ànɛ̂ Asilɛ, 
ànɛ̂ Frisyilɛ, ànɛ̂ Pãnfililɛ, ànɛ̂ Yesyiptilɛ, à tahlɩ Sirɛnɩ kwil tãn tyɩ́, Libi sɔkɔ, ànɛ̂ Wrome kwil tãn nónó-á mɛ tɛ̃̀nɔ́lɛ ásõ̂, 
ànɛ̂ Krɛtɩ tãnnɛ, ànɛ̂ Arabuninɛ. Ápi túkù sõ̀ dubi Nsyifunɔ Liyel nínɔ́ wɛ̃̀kɩ̀ sɔkɔ, tɛ́ túkù mó vi mɔ de álɛ ké sɔkɔ. Tɛ́ á pól mɛ Liyel tyí-ǹgbnɔ pɩ́nɔ́ tyɩ́ yónɔ́lɛ n nohnɩ pé tyɩ́ á kwlo wɛlɔ sɔkɔ.» 
Képah sõ̀ mɛ pé pól yáh-ńsah tyɛ, tɛ́ gbɛ̃̀ndùnì tɛ̃̀ tyɩ́, pè mɛ tyah n yo wɛ̃ tyɩ́ nɛ, tir nɛ̂ se képah n wɛ̃kɩ nɩ? 
Tɛ́ pé túkù mó sõ̀ mɛ kṍlɛ Yesu yõ sɛ̃́púlɛ n ko nɛ, dìvɛ̃ pènɛ̀kɩ̀-á tɛ̃ pélɛ. 
Képah sɔkɔ, Pyɛrɩ yuku yĩn pé no tɛ̃ntõ̀ névye nɛ́pĩ̂ gbɔ̃núkú tɛ́lɔ́ nɩyṍ, tɛ́ tyah n yo yĩ́npɔ̃́ ǹgbɛ̃nɛ tɔ́wû tyɩ́ nɛ: «Ápi Nsyifunɔ póllɛ ànɛ̂ nónó pól-á mɛ tɛ̃̀nɔ́lɛ Yerusalɛmɩ kwil, yé yé ní kwɔ noh pallɛ ń tyɩ́, yè mó ń wɛy syɩ́kɩ́-ńsah pɩ̃. 
Mé pɩ̃ nɛ yáh mɛ n sõ nɛ, sètɛ̃̀nɩ̀ sɔkɔ-á no nɔ́pilɛ, tɛ́ sɛ̃́ náh dɛ́! Pɔ̃̂y tɔ̃ gbĩ́ ó sɔkɔ náh álɛ yɔ̀ di? 
Ǹtɛ Liyel tɛ̃ntõ̀ syínyo-ò Nsyowɛlɩ-á nɛ̂ yo sah, képah ye pɩ yɔ̀. 
À lésõ nɛ, Liyel-á nɛ ńkɛ̃́: ‹Kèkõyṍ-à pópó n tyɛ, mé kɩ ń Mirki tiki mɔ no pól yõ. Yé dĩ́npì ànɛ̂ yé syɔ́npì kɩ ń tɛ̃ntõnènɛ n syi n yo. Mé kɩ wɛ̃́kɩ́nɔ́lɛ yé kètɛ̃̀npìlɛ n wɛ̃kɩ, tɛ́ yé yèwàhnɩ̀ kɩ dlɛ́ynɛ n dɔ. 
Gbɩ yõ, kénɛ sèpĩ̀n sɔkɔ, mé kɩ ń Mirki tiki mɔ ń tõ̀npɩbɩ díbí ànɛ̂ syɔ́ yõ. Tɛ́ pè kɩ ń tɛ̃ntõnènɛ n syi n yo. 
Mé kɩ gbǐl tyi pɩ yĩ̂nyõ, tɛ́ kɩ gbǐl tyi pɩ sétáh yõ, nɛ̂-á kɩ pɩ tṍ, ànɛ̂ nyàh, ànɛ̂ gbìsyikilɛ, 
gbĩ kɩ vi mɔ pɩ fɛ̃́ntũ̂nɛ, tɛ́ ńwɔ̃́ kɩ syãh tṍ sɔ̃́, tɛ́ Tɛ̃̀ ǹgbɛ̃ tukey tùkènɔ́ yòh ǹgbɛ̃ kɩ pyě pɔ, kénɛ tɔ̃́rɩ́ yòh ǹgbɛ̃nɛ. 
Képah sɔkɔ, nónó pól-à Tɛ̃̀ ǹgbɛ̃ ye gbo, pépi wɔ kɩ pwah.›» 
Pyɛrɩ-á Nsyowɛlɩ wɛlɔ nɔ́pi yo tɛ, à tɔ̃ ǹ syɩ yo pé tyɩ́ nɛ ńkɛ̃́: «Yisrayel tãn, yé noh ń tyɩ́, máh kɩ nɛ̂ yo n pi: Nasarɛtɩ Yesulɛ, ǹmɔ ye pɩ névi yɔ́lɛ, nɛ̂-á Liyel syɔ̃ ǹ tyɩ́, à ǹ gblɔ̌y tɔ̃́rɩ́lɛ yé wɛ̃kɩ ǹ gbǐl ànɛ̂ ǹ gbɛ̃̀ndùnì tyi pɩ́nɔ́ sɔkɔ yé wrɔ́. Yépi ńtɛ̃ ye kè pɩ̃ ó! 
Liyel-á ǹ tyi yah tɔkɔ sah sɔ̃́, sè pɩ sɛ̃́. À kénɛ dĩ́ gbàh dɛ mɔ yé kɔ̃, yé mɛ ǹnɛ Liyel dêl pɩ́pú kɔ̃ pè à syɩ dahnɩ yõ, à à ko mɔ. 
Ǹtɛ kũ náh sõ̀ fɛ̃ à tɛ̃ n sah n pi. Képah ye, Liyel mɛ tɔ̃ ǹ lékó yilki mɔ, à à wilki kũ yèvyãhrɩ sɔkɔ. 
Ǹmɔ tyɩ́ ye lésõ yõ̀tɛ̃̀ dĩ́ Davidi yo nɛ ńkɛ̃́: ‹Péwɔ Tɛ̃̀ ǹgbɛ̃ Liyel náh mɛ ké tyɩ́ nɛ pé tɛ̃ n kwlɔ. Képah-á, à mɛ pélɛ gbĩ́mɛgbĩ́ pé yohnɩ mɔ yĩnnɛ. 
Páh tɔ̃ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ, ànɛ̂ nɩ̀vɩ̀nɩ̀ wɛlɔ ó-á pè n nɛ n wɛkɩ. Pé gbõ̀ sàhnɔ́-á mɛ ké yõ nɛ, pâh pɔ ku, pé min-á kɩ mɛ tɛ̃ yèvɩnɩ sɔkɔ. 
À náh pé lékó sah kũ kɔ̃ n pi, ànɛ̂ à náh n yɛ pé lékó kɔ̃ kè n pyãh n pi, péwɔ nɛ̂-á ǹ tõ̀npɩ pèpɛynɛ. 
Wǎh min wã̀llɛ pé wɛ̃kɩ. Tɛ́ kɩ pé kɔ̃, pé kɩ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ kṹmm ǹ yah.›» 
Képah yóntɛnɔ náh, Pyɛrɩ tɔ̃ yo pé tyɩ́ nɛ: «Á no, yé yɛ mè á náh nɛy Davidi tyi yo n wah yé syah. À lésõ ku pè yɩ̃. Tɛ́ kwɛy mɛ̀ sɔkɔ, ǹ diki ńtɛ̃ mɛ ápi kwil. 
Ǹtɛ Liyel tɛ̃ntõ̀ syínyo-ò ye sõ̀ ǹnɛ. Ǹ gbõ̀ sàhnɔ́ sõ̀ mɛ Liyel vyãh kɔ̃́nɔ́ mɛ̀ yõ, Liyel-á ǹ wo nɛ̂ sɔkɔ nɛ, ǹ pétóbé sɔkɔ yɔ́-á kɩ pɔ pɩ yõ̀tɛ̃̀nɛ ǹ swɔ. 
Tɛ̃̀ wɔ ye pɔ pɩ Kristalɛ, nɛ̂ kɔ́r-á Liyel nɛy yáhntɔkɔnɔlɛ. Tɛ́ wǎh nɛ n yilki n mɔ n pi ǹ kúntɛnɔ náh, képah ye Davidi kwãh tɛ̃ wɛ, à kɔ̃ à nɛ ńkɛ̃́: ‹À náh n wũ n pi lékókwíl, ànɛ̂ ǹ lékó náh n pyãh n pi.› 
Núkúnúkú Yesu mɛ̀nɛ, á pól ye kè wɛ yĩ́nɛ nɛ wǎh ku, tɛ́ Liyel mɛ ǹ lékó yilki mɔ. 
À yɔ̃ dɔkɔ sɔkɔ yĩ̂nyõ, mɛ kyɩ tɛ̃ Liyel gbõ̀yõ̀ sõ̀lɛ. Tɛ́ Sú Liyel-á ǹ vyãh kɔ̃ ǹ Mirki nɛ̂ tyɩ́ sɔkɔ, à mɛ kélɛ Yesu kɔ̃ à tiki mɔ á yõ. Képah ye á n yah, tɛ́ n nohnɩ yɔ̀. 
Ǹtɛ Davidi náh dɔkɔ yuku yĩ̂nyõ, tɛ́ wǎh kè yo nɛ ńkɛ̃́: ‹Tɛ̃̀ ǹgbɛ̃ Liyel kè yo ńmɔ yõ̀tɛ̃̀ dĩ́ tyɩ́ nɛ, pɔ n kɔ̃ n tɛ̃ ń gbõ̀yõ̀ sõ̀lɛ, 
fɔ́ɔ́ kè pɔ nɔ gbĩ́ nɛ̂-á mé kɩ á sépɛ́bɩ́ pɩ á gbáhkɩ̀ súnsàh-ǹsahlɛ.›» 
Képah yóntɛnɔ náh, Pyɛrɩ tɔ̃ pè syah nɛ: «Yisrayel tãn pól yɩ̃nɛ yè kè pɩ̃ gbɩ yõ nɛ, Yesu nɛ̂-á yépi tɛ̃ syɩ dahnɩ yõ, ǹmɔ-á Liyel pɩ TƐ̃̀ ǸGBƐ̃NƐ tɛ́ tɔ̃ à pɩ Kristalɛ, nɛ̂ kɔ́r-á Liyel nɛy yáhntɔkɔnɔlɛ.» 
Pyɛrɩ-á wɛy mɛ̀ yo pɔ tyɛ sɛ̃́, kè névye nɩ kah tɛ̃, pè mɛ Pyɛrɩ ànɛ̂ Yesu tɛ̃ntõ̀ névye tɛ́lɔ́ piki yah nɛ, pé no, sɔ̃́ se pé yɩ̃nɛ pé pɩ nɩ? 
Képah sɔkɔ, Pyɛrɩ mɛ yo pé tyɩ́ nɛ: «Yé yé nnɔ vi mɔ, yé nɛ́núkù núkú pól mó wèlnwìlkìnɔ́ wɛ ni sɔkɔ Yesu Krista yĩn yõ, yé tyípêl yɔ̃̀nmɔnɔ yĩnnɛ. Képah sɔkɔ, Liyel Mirki kɩ yé kɔ̃. 
Liyel-á ǹ vyãh kɔ̃ ǹ Mirki tyɩ́ sɔkɔ nónó yĩnnɛ, yépi ye, ànɛ̂ yé wɛ̃́npìlɛ, à tahlɩ nónó pól-á mɛ kwlo fɩ́fɩ́ sɔkɔ, ànɛ̂ nónó pól-á Tɛ̃̀ ǹgbɛ̃ Liyel kɩ n ye n pi.» 
Tɛ́ Pyɛrɩ tɔ̃ sõ̀ mɛ wɛlɔ náhnáhlɛ n gbah n yo pé tyɩ́, à sṍrélɛ n mɔ pé sɔkɔ. À sõ̀ mɛ n yo pé tyɩ́ nɛ: «Yé Liyel pwáhnmɔnɔ fɛ̃ syi, yè mó wil kwɛy gbĩ névye nɛ́pêl wrɔ́.» 
Képah sɔkɔ, nónó-á Pyɛrɩ wɛlɔ fɛ̃ syi, pè wèlnwìlkìnɔ́ wɛ ni sɔkɔ. Ké lésõlɛ, nɛ́pĩ̂ kèyõ núkú kyɛgbãm (3 000) kénkɔ̃ dohnɩ Yesu yõ sɛ̃́pú yõ. 
Képah tɛ̃̀nɛ, Yesu tɛ̃ntõ̀ névye-á kwɔ́nɔ́lɛ nónónɛ n kɔ̃, pé pól sõ̀ mɛ pé gblɔ̌y kɔ̃, mɛ n nohnɩ pé tyɩ́, tɛ́ mɛ kwrɔ mɔ pé sɔkɔ, à pɩ pélɛ núkúlɛ. Pè sõ̀ mɛ yõkelɛ n yõ pélɛ wɛ̃ tyɩ́, tɛ́ mɛ nírílɛ n pɩ pélɛ. 
Liyel sõ̀ mɛ n syɔ̃ Yesu tɛ̃ntõ̀ névye tyɩ́, à gbǐl tyi náhnáhlɛ n pɩ. Kǎh pɩ sɛ̃́, kè tíkílɛ no pól tɛ̃. 
Nónó pól-á sõ̀ vi mɔ pɩ Yesu yõ sɛ̃́púlɛ, pè sõ̀ mɛ kwrɔ mɔ tɛ̃ wɛ̃ tyɩ́, tɛ́ mɛ pé gbã̀n yî póllɛ n sah wɛ̃ yõ. 
Pè sõ̀ mɛ pé swɛ̃y ànɛ̂ pé gbã̀n yîlɛ n yãm. Tɛ́ mɛ sé yã̀mnɔ́ pɔ́nɔ́lɛ n dibi wɛ̃ wrɔ́, à yɩ̃nɛ pé nɛ́núkù núkú pól fyã̀hnɔ́nɔlɛ. 
Tɛ́ pè sõ̀ mɛ pé sõnɔ pɩ núkúlɛ, à wɛ̃nɛ n tuke n mɔ sõmɛsṍ Liyel ní-ńsah gbô sɔkɔ. Pè sõ̀ pé fɔkɔ dal wɛ̃nɛ, à yõkenɔlɛ n yõ wɛ̃ yõ nɩ̀vɩ̀nɩ̀nɛ, pé gbéngbèn sɔkɔ. 
Pè sõ̀ mɛ Liyellɛ n gbilki, tɛ́ no pól mɛ pélɛ n sõ. Tɛ́ Tɛ̃̀ ǹgbɛ̃ sõ̀ mɛ névyelɛ n pwah n mɔ n dohnɩ pé yõ sõmɛsṍ.
