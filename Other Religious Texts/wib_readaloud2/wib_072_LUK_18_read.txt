Likyi.
18.
Képah náh, Yesu sèmìrkì yɔ́ yãh ǹ tõ̀ sã́hpú tyɩ́, à pè wɛ̃kɩ nɛ, pǎh yɩ̃nɛ pè nírílɛ n pɩ kṹmm wílkúnì kɛ̃́nɛ ké sɔkɔ. 
À nɛ: «Tukey tùkè-ò dĩ́ yɔ́ lésõ mɛ kwil yɔ́ sɔkɔ, à náh Liyel yah tíkílɛ n pɩ, à náh yõ̀yénnɛ n sah névye yõ. 
Tɛ́ pɛ́kúsè yɔ́ tɔ sõ̀ mɛ kénɛ kwil sɔkɔ. Ǹmɔ sõ̀ mɛ n yo n pɔ ǹ tyɩ́ gbĩ́mɛgbĩ́ nɛ, nɛ̂-á mɛ péwɔ ànɛ̂ pé fwó-ò wrɔ́, à kè tɔ̃ sah pé kɔ̃. 
Ǹtɛ, kè pɩ di mónɔ́lɛ, tɛ́ dĩ́ náh sõ̀ mɛ fɛ̃ syi. Képah náh, à pɔ yo n gblɔ̌y tyɩ́ nɛ, gbɩ tɛ̃̀ wɔ-á, pé náh Liyel yah tíkílɛ n pɩ, pé náh yõ̀yénnɛ n sah névye yõ. 
Ǹtɛ, pɛ́kúsè mɛ̀-á mɛ pé gblɔ́lɛ n vyãh n pɔ, páh kɩ ǹ tir mɛ̀ yah wah à kɔ̃, pé yah mó vɩ.» 
Tɛ̃̀ ǹgbɛ̃ Yesu-á képah yãh tɛ, à tɔ̃ yo pé tyɩ́: «Yé tukey tùkè-ò dĩ́ mɛ̀ wɛy noh kɛ̀, ǹmɔ nɛ̂-á fĩ̀n yṍ-òlɛ. 
Ǹtɛ Liyel wɔ-á nónó yah tɔkɔ, tɛ́ pè mɛ ǹnɛ n ye n gbo lékã̂h ké gbĩ̀ntɔ̃̀, à náh pé tyi yah wah pè kɔ̃ n pi di? À se kɩ mo pé yòhnɩ̀nmɔ-ńsah? 
Mé kélɛ n yo yé tyɩ́ nɛ: wǎh kɩ pé tyi yah wah pè kɔ̃ súú. Ǹtɛ, Névi Pi-à sṍ nɛ̂nɛ pɔ sétáh yõ, à se kɩ pɔ Liyel yõ sɛ̃́nɔ́ wɛ?» 
Névye nónó-á sõ̀ mɛ pé gblɔ̌ynɛ n yah nɛ ńkɛ̃́, páh mɛ nɛ́gbɔ́lɛ Liyel yah, tɛ́ mɛ tɛ́lɔ́lɛ n yilɔ, Yesu sèmìrkì yɔ́ yãh pépi tyɩ́ sɔkɔ nɛ: 
«Díbí nɛ́pĩ̂ nímí yísyɔ́ sõ̀ kyɩ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, à kyɩ Liyel ni. Nɛ́núkùlɛ, Farisiyɛ̃ ye sõ̀ ǹnɛ, tɛ́ ní-òlɛ, mɔ́rɩ́ syí-òlɛ. 
Képah sɔkɔ, Farisiyɛ̃ mɛ̀ mɛ yuku yĩn, à Liyellɛ n ni ǹ fɔkɔ táh, mɛ n yo nɛ, èè Liyel, páh n wɛkɩ ǹ tyɩ́, wǎh sɔ̃́npɩ́ péwɔ pɩ pé gblɔ̌y. Pé náh mɛ no tɛ́lɔ́ tyɩ́ kɔ̃lɛ, nónó-á yɩ̃́púlɛ, ànɛ̂ tyítúkúnì pɩ́púlɛ, ànɛ̂ fɔ̀fɩ́kɩ̀ pɩ́púlɛ. Páh n wɛkɩ ǹ tyɩ́, pé náh sɔ̃́npɩ́lɛ mɛ mɔ́rɩ́ syí-ò mɛ̀ sɔ̃́. 
Péwɔ-á níkìlɛ n tɛ̃ sèpĩ̀n nínì yòh núkú wrɔ́. Páh pé gbã̀n yî wɛ́nɔ́ gbãm-ònɔ́lɛ ǹnɛ n kɔ̃. 
Ǹtɛ mɔ́rɩ́ syí-ò wɔ mó sõ̀ mɛ yĩn tɛ̃ fɩ́ɩ́, à náh fɛ̃ syi ǹ yah ńtɛ̃ yɔ̃ yĩ̂nyõ. Tɛ́ mɛ ǹ gbõ̀lɛ n sah ǹ nɩ̀vɔ sɔkɔ mɛ n yo nɛ: ‹Liyel, ń yãm yah, ńmɔ nɛ̂-á tyípékè pɩ́-òlɛ.›» 
Képah tɛ̃̀nɛ, Yesu mɛ nɛ: «Mé kélɛ n yo yé tyɩ́ nɛ: dĩ́ mɛ̀ pɩ nɛ́gbɩ́lɛ Liyel yah, wǎh ǹ syɩ n yuku ǹ tĩ̀nnɛ gbĩ́ nɛ̂nɛ. Tɛ́ Farisiyɛ̃ wɔ náh sɛ̃́nkɔ̃ wɛ. Gbɩ yõ, névi nɛ̂-à ǹ gblɔ̌ynɛ n gbilki, à kɩ tiki mɔ. Tɛ́ nɛ̂-à ǹ gblɔ̌ynɛ n tiki n mɔ, tɛ̃̀ wɔ kɩ mó dɔkɔ mɔ.» 
Névye sõ̀ mɛ n pɔ n pɔ wɛ̃́npì yrɔ̃́ tɔlɛ Yesu tyɩ́ nɛ, à tũ pé tyɩ́. Ǹ tõ̀ sã́hpú-á képah wɛ, pè tyah dírílɛ n kõ pé yõ. 
Ǹtɛ, Yesu mɛ wɛ̃́npì kɔ̃ pè nɔ de ǹ tnɔ̂, tɛ́ nɛ: «Nónó dùkù-á mɛ wɛ̃́npì nɔ́pi sɔ̃́, Liyel tɔ̃́rɩ́lɛ, pépi tyɩ́ ye. Képah ye, yé yɛ wɛ̃́npì kɔ̃ pè nɛ n pɔ n pɔ ńmɔ tyɩ́. Yé káh pé yahlɛ n kõ. 
Gbɩ yõ, mé kélɛ n yo n dal n mɔ yé tyɩ́ nɛ: névi nɛ̂-à Liyel tɔ̃́rɩ́ fɛ̃̀nsyi-ńkɛ̃̂nɛ wɛ̃́npir tyɩ́ kɔ̃lɛ, à náh n wɛ n de n pi ké sɔkɔ póllɛ.» 
Képah náh, Nsyifunɔ yõ̀tɛ̃̀ yɔ́ Yesu piki yah nɛ, kwɔ́-ò pèpɛy, pé se yɩ̃nɛ pé pɩ sɔ̃́, pé mó tyɛ́-ńkɛ̃̂ min wɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́: «Kwâh nɛ̂ pɩ, tɛ́ á ńnɛ n ye ‹pèpɛynɛ›? Pèpɛy yɔ́ ńtɛ̃ náh mɛ, nɛ́ Liyel ó náh. 
Á Liyel tyi kõ̀nsàhnɔ́ pɩ̃ kɛ̀, nɛ̂-á n yo nɛ ńkɛ̃́: ‹Káh fɔ̀fɩ́kɩ̀ pɩ, káh nɛ́kókè mɔ, káh yɩ̃be yɩ̃, káh mɩ̀nɩ̀ sah á ní-ò yõ, á sú ànɛ̂ á yṹnɛ n tɛ̃ plɛ.›» 
Wǎh yo sɛ̃́, dĩ́ mɛ à syah nɛ, páh mɛ sépinɔ́ póllɛ n pɩ fɔ́ɔ́, pé wɛ̃́npir gbĩ́ dùkùlɛ. 
Yesu-á képah noh sɛ̃́, à yo ǹ tyɩ́: «Tyínúkú ó tɔ̃ yah kwlɔ á tyɩ́: Gbã̀n yî nónó pól-á mɛ á tyɩ́, kyɩ sè yãm, ǹ tɛ́ sé pɔ́ dibi yãm tãn kɔ̃, ǹ tɛ́ n pɔ ńmɔ sõ̀ tɛ̃. Képah sɔkɔ, á kɩ Liyel fṍ tahrɩ wɛ.» 
Tɛ́ dĩ́ tahrɩ-á sõ̀ kah tnɔ sɛ̃́, képah ye, wǎh wɛy mɛ̀ noh, ǹ yah kwɔ yɩkɩ. 
Képah sɔkɔ, Yesu-á ǹ yah wɛ kè kwɔ sɛ̃́, à nɛ, tahbɩ dénɔ́-á mɛ tah Liyel tɔ̃́rɩ́ sɔkɔ dɛ́! 
Ńkɛ̃́, yã̀hmɩ́kɩ̀ dénwìlnɔ́lɛ sèndi pwã̀h sɔkɔ, képah ńtɛ̃-á kɩ fàhfàh kal tàh dénɔ́lɛ Liyel tɔ̃́rɩ́ sɔkɔ. 
Képah sɔkɔ, nónó-á sõ̀ mɛ ǹ wɛynɛ n nohnɩ, pè nɛ, ǹtɛ, nɛ̂ wɔ se kɩ mó fɛ̃ pwah nɩ? 
À mɛ yo pé tyɩ́: «Névi náh fɛ̃ tir nɛ̂ pɩ n pi, Liyel kɩ fɛ̃ képah pɩ.» 
Képah sɔkɔ, Pyɛrɩ mɛ nɛ, pépi tɛ̃̀ wɔlɛ, páh pé yɛ̃́ngbɛ̃́ yî sah, tɛ́ ǹmɔ sõ̀ tɛ̃. 
Yesu mɛ yo pé tyɩ́: «Gbɩ yõ, mé kélɛ n yo n dal mɔ yé tyɩ́ nɛ: névi nɛ̂-à Liyel tɔ̃́rɩ́ yĩnnɛ ǹ kéte sah, képah náh pɩ, ǹ kyɩlɛ, képah náh pɩ, ǹ yṹnpyélɛ, képah náh pɩ, ǹ tyɩ́ névyelɛ, képah náh pɩ, ǹ wɛ̃́npìlɛ, 
tɛ̃̀ kɩ pé lékan wɛ kèkõyṍ mɛ̀ sɔkɔ, tɛ́ kɩ tyɛ́-ńkɛ̃̂ min wɛ Liyel fṍ sɔkɔ ńsõ̂nɛ.» 
Képah náh, Yesu ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye pé yahlɛ, tɛ́ yo pé tyɩ́: «Á wɛ n yuku Yerusalɛmɩ sɔkɔ yɔ̀. Tɛ́ Liyel tɛ̃ntõ̀ syínyopu-á lésõ tyi nónó pól wãrkɩ sah Névi Pi tyɩ́ sɔkɔ, sè kɩ n pɩ n tũ n pi. 
Pè kɩ à dahbɩ Liyel pɩ̃́-ńkɛ̃̂pú gbõ̀. Pè kɩ kṍlɛ ǹnɛ n ko, kɩ ǹnɛ gbyɔ, tɛ́ kɩ n sĩn n mɔ ǹ yõ. 
Pè kɩ à ko yèfõhlɛ, tɛ́ kɩ à ko mɔ. Ǹtɛ, ké sèpĩ̀n tɔ̃́-ò sṍlɛ, à kɩ ǹ syɩ min sãn wɛ.» 
Ǹtɛ, Yesu wɛy mɛ̀ sõ̀ mɛ wɛ́npǐllɛ pé tyɩ́. Képah ye pè náh ké syɩ́kɩ́-ńsah pɩ̃. 
Képah sɔkɔ, Yesu-á de n tɛ Yeriko kwillɛ yúkɛy, yɩ̃́tũ̂ yɔ́ sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ wɛ̃̀gbalɩ sɔkɔ, mɛ n ni n syi. 
Wǎh tɔ́wû sɔ́kɔ́nɔ́ yĩ́ noh, à piki yah nɛ, tir nɛ̂ se nɩ? 
Pè mɛ à syah nɛ, Nasarɛtɩ Yesu-á pɔ n sɔkɔ. 
Képah tɛ̃̀nɛ, à tyah n yo n mɔ léyy nɛ, Yesu, Davidi Pídĩ́, à pé yãm yah. 
Tɛ́ nónó-á mɛ n kɔ yah tĩ̀nnɛ, pépi sõ̀ mɛ dírílɛ n kõ ǹ yõ nɛ, à ǹ vyãh vike. Ǹtɛ, ǹmɔ tɛ̃̀ sõ̀ mɛ n tɔ̃ n gbah n yo n mɔ nɛ ńkɛ̃́, Davidi Pídĩ́, à pé yãm yah. 
Képah sɔkɔ, Yesu mɛ yĩn, tɛ́ nɛ, pè pɔ ǹnɛ. Wǎh nɔ de, Yesu mɛ à piki yah: 
«Tir nɛ̂ mɛ á mɛ ké tyɩ́ nɛ, mé pɩ á tyɩ́?» Dĩ́ mɛ à syah nɛ, pé Yõ̀tɛ̃̀, à kè pɩ, pé fɛ̃ nɛ n yah ńtɔ̃. 
Yesu mɛ yo ǹ tyɩ́: «Á yɩ̃́ ye yɩkɩ mɔ! Á Liyel yõ sɛ̃́nɔ́ ye ǹ dɛ mɔ.» 
Nɛ́npɔ́ swɔ, ǹ yɩ̃́ núkú yɩkɩ mɔ, tɛ́ à mɛ Yesu sõ̀ tɛ̃, à Liyellɛ n gbilki. No pól-á képah wɛ sɛ̃́, pé tɔ tyah Liyellɛ n gbilki.
