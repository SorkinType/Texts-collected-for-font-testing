Nsyɔ̃.
13.
Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh kɩ sõ̀ n pɩ n pi dĩndallɛ. Yesu sõ̀ pɩ̃ nɛ pé yálnɔ́ gbĩ-á sɛ kèkõyṍ, páh kɩ n sɔkɔ n pi Sú Liyel tyɩ́. Ǹ sõ̀ tɛ̃́pú nónó-á sõ̀ mɛ kèkõyṍ, pé tyɩ́ sõ̀ mɛ nɔ́nɔ́lɛ ǹ tyɩ́. Pé tyɩ́ nɔ kah mɔ ǹ tyɩ́. 
Yesu ànɛ̂ ǹ tõ̀ sã́hpú sõ̀ mɛ yõkelɛ n yõ. Tɛ́ sétã̀n yõ̀tɛ̃̀ sõ̀ mɛ kè dahbɩ tɛ Simo Yisikariyɔtɩ pídĩ́ Nsyudasɩ nɩ sɔkɔ nɛ, à vi Yesu náh. 
Yesu sõ̀ pɩ̃ nɛ Sú Liyel-á tyi pól dahbɩ péwɔ gbõ̀. Ǹmɔ tyɩ́-á pé wil à pɔ, tɛ́ páh kɩ pé syɩ sɔkɔ ǹmɔ tyɩ́. 
Képah sɔkɔ, pǎh mɛ yõke yṍnɔ́ sɔkɔ sɛ̃́, à yuku yĩn, tɛ́ ǹ fwɔ̀mɔ̀-ǹgbɛ̃ dɛ sah, tɛ́ fwɔ yɔ́ tɔkɔ toh ǹ syɩ̃ sɔkɔ. 
Képah náh, à ni du mɔ dú yɔ́ sɔkɔ, à tyah ǹ tõ̀ sã́hpú gbyẽ̂hnɛ n wu kélɛ, tɛ́ sélɛ n tuke fwɔ mɛ̀nɛ, nɛ̂-á à toh ǹ syɩ̃ sɔkɔ. 
Kǎh pɔ syɩ Simo-Pyɛrɩ tyɩ́, à Yesu syah nɛ ńkɛ̃́, ǹmɔ nɛ̂-á TƐ̃̀ ǸGBƐ̃NƐ ǹmɔ kɩ n yah n kɔ̃ péwɔ gbyẽ̂h wú-ńsahlɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ, nɛ̂-á péwɔ n pɩ yɔ̀, ǹmɔ náh fɛ̃ ké tyɩ́ yah wɛ n pi núkúnúkú lɔ́! Ǹtɛ náh-á à kɩ pɔ ké tyɩ́ yah wɛ n pi. 
Pyɛrɩ tɔ̃ yuku à syah nɛ, póllɛ, à náh pé gbyẽ̂h wu n pi. Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ: «Wáh kɩ yɛ ń kɔ̃ mé á gbyẽ̂h wú-ńkɛ̃̂nɛ, á vyãh kɩ dɛ ńmɔ tyi sɔkɔ.» 
Képah tɛ̃̀nɛ, Simo-Pyɛrɩ yo ǹ tyɩ́ nɛ, Tɛ̃̀ ǹgbɛ̃, kàh pɩ sɛ̃́ kè káh yĩn pé gbyẽ̂h ó tyɩ́, ńkɛ̃́ à pé gbã̀n ànɛ̂ pé yõ tɔ wu. 
À mɛ yo ǹ tyɩ́ nɛ: «Névi nɛ̂-à ni wel tɛ, tɛ̃̀ n nɛ n mɛ yályál. Képah ye, à wil ǹ gbyẽ̂h tyɩ́ sɔkɔ, yõ náh tɔ̃ ńkɛ̃́nɛ nɛ mɛ tɛ̃̀ wil pól tɔ̃́nwunɔ tyɩ́. Yépi mɛ yályál. Ǹtɛ yé pól tɛ̃̀ wɔ náh mé n ye dɛ́!» 
Névi nɛ̂-á kɩ sõ̀ pɔ n vi n pi Yesu náh, Yesu sõ̀ mɛ à pɩ̃ tɛ. Képah ye à kɔ̃ à nɛ, pé pól tɛ̃̀ wɔ náh pé nɛ pǎh mɛ yályál dɛ́! 
Wǎh pé gbyẽ̂h wu tɛ, à tɔ̃ ǹ fwɔ̀mɔ̀-ǹgbɛ̃ tɔkɔ mɔ pɔ kɔ̃ tɛ̃ yõke yṍ-ńsah ńtɔ̃. Tɛ́ yo pé tyɩ́ nɛ: «Máh nɛ̂ pɩ yé tyɩ́ sɛ̃́ yɔ̀, yé se ké yõ pɩ̃? 
Yâh pɩ yě ńnɛ n ye kwɔ́-ò dĩ́nɛ, képah náh pɩ, Tɛ̃̀-ǹgbɛ̃nɛ, yé mɛ n yo plɛ tɛ̃̀ wɔlɛ. Ńmɔ ye ńtyɩ̃́nɔ ǹnɛ. 
Ǹtɛ ńmɔ nɛ̂-á Tɛ̃̀ ǹgbɛ̃ ànɛ̂ kwɔ́-òlɛ, yáh wɛ mé yé gbyẽ̂h wu, yépi tɔ yɩ̃nɛ yè wɛ̃ gbyẽ̂hnɛ n wu. 
Képah dùkù ye mé pɩ yé wɛ̃kɩ yɔ̀, nɛ máh nɛ̂ pɩ yé tyɩ́, yé tɔ sɛ̃́nknɔlɛ n pɩ. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, tõ̀npɩ náh fɛ̃ n kal ǹ yõ̀tɛ̃̀nɛ. Sɛ̃́ ó ye tɛ̃ntõ̀ névi tɔ náh fɛ̃ n kal ǹ tṍ-òlɛ. 
Ǹtɛ núkúnúkú, yé se kénɛ tyilɛ n pnɛ tɛ́! Yâh fɛ̃ syi n kɔ sé wã̀l yõ, yé kɩ nɩ̀vɩ̀nɩ̀ wɛ. 
Ń wɛy mɛ̀ náh mɛ ké yah mɔ yé pól tyɩ́ dɛ́! Máh névye nónó yah wilki, mé pélɛ n pnɛ. Ǹtɛ tir nɛ̂-á wãrkɩ sah Liyel wɛy sɛ́bɛ́y sɔkɔ, kè yɩ̃nɛ kè wil ó wil. Kè mɛ wã́rkɩ́nsàhnɔ́lɛ ké sɔkɔ nɛ ńkɛ̃́: ‹Máh lésõ ń yõkelɛ n yõ ànɛ̂ nɛ̂nɛ, tɛ̃̀ mɛ yuku ń yõ.› 
Ké pɩ́-ńkɛ̃̂ sɔkɔ ye mé kélɛ n yo n sah yé tyɩ́ núkúnúkú yɔ̀. Képah sɔkɔ kàh pɔ pɩ, yé kɩ sɛ̃ ké yõ nɛ, Ńmɔ-á. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, névi nɛ̂-à ń tɛ̃ntõ̀ névi tɛ̃ plɛ, ńmɔ ye tɛ̃̀ tɛ̃ plɛ. Nɛ̂ tɔ-à ńmɔ tɛ̃ plɛ, nɛ̂-á tõ ńnɛ, ǹmɔ ye tɛ̃̀ tɛ̃ plɛ.» 
Képah tɛ̃̀nɛ, kénɛ wɛlɔ yóntɛnɔ náh, Yesu fɔkɔ kah fɛ̃kɩ. À mɛ kè yo dal mɔ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, yé sɔkɔ nɛy yɔ́ kɩ n vi n pi ń náh.» 
Wǎh képah yo sɛ̃́, ǹ tõ̀ sã́hpú tyah n yɔ̃ wɛ̃nɛ n yah núkú núkúlɛ. Pè náh sõ̀ mɛ pɩ̃ nɛ̂ tyɩ́-á à n yo. 
Tɛ́ Yesu tõ̀ sã́hpú sɔkɔ nɛ̂ tyɩ́-á nɔ yĩ́ĩ́ ǹ tyɩ́, ǹmɔ sõ̀ mɛ tɛ̃̀ntelnɔlɛ Yesu tyɩ́. 
Képah sɔkɔ, Simo-Pyɛrɩ pɩ mɔ ǹmɔ tyɩ́ nɛ à Yesu piki yah, nɛ̂ tyɩ́-á à n yo. 
Ǹmɔ mó ǹ gbalɩ kɛ̃kɩ Yesu tĩ̀nnɛ, tɛ́ à piki yah nɛ, Tɛ̃̀ ǹgbɛ̃, nɛ̂ se à n ye nɩ? 
À mɛ yo ǹ tyɩ́ nɛ, páh kɩ búrn yĩ di nɛ̂ kɔ̃, tɛ̃̀ wɔ-á. Tɛ́ núkú búrn yĩ́nɔ́ tɔkɔ, tɛ́ kè di à Simo Yisikariyɔtɩ pídĩ́ Nsyudasɩ kɔ̃. 
Nsyudasɩ-á búrn mɛ̀ syi, sétã̀n yõ̀tɛ̃̀ mɛ de ǹ sɔkɔ. Yesu mɛ yo ǹ tyɩ́ nɛ, nɛ̂-á ǹ tyɩ́ pɩ pɩ tɛ̃̀nɛ, à kè pɩ súú. 
Ǹtɛ nónó pól-á sõ̀ mɛ nɛ́npɔ́, pé sɔkɔ yɔ́ ńtɛ̃ náh sõ̀ pɩ̃ tir nɛ̂ yĩn-á Yesu yo wɛy mɛ̀nɛ ǹ tyɩ́. 
Tɛ́ pɔ́ dú-á sõ̀ mɛ Nsyudasɩ tyɩ́ sɛ̃́, túkù sõ̀ mɛ n sõ nɛ, Yesu-á yo ǹ tyɩ́ nɛ à kyɩ syáh yídwénè yah dwe, képah náh pɩ, à pɔ́ yɩ̃ yãm tãn kɔ̃. 
Ǹtɛ Nsyudasɩ-á fyɔ̀ búrn yĩ́nɔ́ mɛ̀ syi, à núkú yuku wil. Légbĩ́nɛ lékã̂h sõ̀ mɛ mɔ. 
Nsyudasɩ-á fyɔ̀ wil tɛ, Yesu nɛ: «Núkúnúkú tɛ̃̀nɛ, gbílkínɔ́ ye sah Névi Pídĩ́ yõ, ànɛ̂ Sú Liyel yĩn kɩ se tɔ̃ gbilki ǹmɔ yĩnnɛ. 
Gbílkínɔ́-à kɩ sah Liyel yõ ǹmɔ yĩnnɛ, Liyel tɔ kɩ à gbilki ǹ gblɔ̌y yĩnnɛ. Dǒ kwéy ńtɛ̃nɛ, à kɩ kè pɩ n pi. 
Ń wɛ̃́npì, mé kɩ tɔ̃ n mɛ n pi yélɛ kɩ wrɔ́ kwéy ó pɩ. Yé kɩ ń yah n kɔ̃ n pi. Ǹtɛ nɛ̂-á lésõ mé yo Nsyifunɔ tyɩ́, képah núkú ye tɔ̃ mé kɩ yo yépi tɔ tyɩ́ núkúnúkú nɛ, yé náh fɛ̃ n kyɩ n pi ńmɔ yúkú-ńsah sɔkɔ. 
Mé kɩ tir sã́n yɔ́ kõ sah yé kɔ̃ nɛ, yé tyɩ́ nɔ wɛ̃ tyɩ́. Yé tyɩ́-á nɔ sɔ̃́ ńmɔ tyɩ́, yépi tɔ tyɩ́ yɩ̃nɛ kè nɔ sɛ̃́ wɛ̃ tyɩ́. 
Yé tyɩ́-à kɩ nɔ wɛ̃ tyɩ́, képah sɔkɔ ye no pól kɩ pɩ̃ nɛ ńmɔ tõ̀ sã́hpú-á yélɛ.» 
Képah sɔkɔ, Simo-Pyɛrɩ à piki yah nɛ, Tɛ̃̀ ǹgbɛ̃, ńyãh se à n yuku nɩ? À mɛ yo ǹ tyɩ́ nɛ, ńyãh-á pé n yuku, à náh fɛ̃ n kyɩ n pi pélɛ nɛ́npɔ́ sɔkɔ núkúnúkú. Ǹtɛ náh tĩ̀nnɛ, wǎh kɩ pɔ pé sõ tɛ̃ n pi. 
Pyɛrɩ mɛ nɛ, Tɛ̃̀ ǹgbɛ̃, sɔ̃́ se pɩ tɛ́ pé náh fɛ̃ ǹ sõ̀ tɛ̃ n pi núkúnúkú nɩ? Péwɔ-á kɩ fɛ̃ syi pé min kɔ̃ n pi ǹmɔ yĩnnɛ dò. 
À mɛ yo ǹ tyɩ́ nɛ: «Á kɩ á min kɔ̃ n pi ńmɔ yĩnnɛ lɛ́? Ǹtɛ kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo á tyɩ́, sénpɛ-à kɩ pópó n gbo, á kɩ mɛ á vyãh kɔ̃lɩ tɛ kɛ tɔ̃́nɔ́, nɛ á náh ńmɔ pɩ̃.»
