Matye.
20.
«Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Sètɛ̃̀ dĩ́ yɔ́ lésõ wil séngbɔ̃́ púpúlɛ nɛ pé tõ̀npɩbɩ yah n tɔkɔ n tɛkɩ mɔ pé yɛ̀rɛsɛ̃̂ swãh sɔkɔ. 
Wǎh nónó wɛ, à yo sah núkú yõ pélɛ nɛ, nɛ́núkú gbĩ̀yĩ̀kì núkú tõ̀ pɩ́nɔ́ sǎrlɛ, deniye pɔ́ pɔ́ núkú-á. Képah sɔkɔ, tɛ́ pè tɛkɩ mɔ ǹ swãh sɔkɔ. 
Pɔ̃̂y tɔ̃ gbĩ́nɛ, à tɔ̃ wil tɛ́ kyɩ nósyɔ́ wɛ tɛ̃̀nɔ́lɛ tɔ́wû tùkè-ǹsah sɔkɔ tõ̀ kɛ̃́nɛ pé tyɩ́. 
À mɛ yo pé tyɩ́ nɛ, pépi tɔ sɔkɔ péwɔ swãh sɔkɔ, páh kɩ pè sar ké yah yõ. 
Pè mɛ sɔkɔ. Gbĩ-á mɛ yõ̀tah sírr sɔkɔ, à tɔ̃ wil à sɛ̃́ntɛ̃̀ pɩ. Gbèkãm gbĩ́-á nɔ, à tɔ̃ wil à képah núkú ó pɩ. 
Ńvɩ̃́ lésyilɛ, à tɔ̃ wil tɛ́ kyɩ nósyɔ́ wɛ ńtɔ̃ kénɛ tɔ́wû tùkè-ǹsah sɔkɔ. À mɛ pè piki yah nɛ, sɔ̃́ se pɩ tɛ́ pélɛ ásõ̂ gbĩ̀yĩ̀kìlɛ tɔ́tɔ́ tõ̀ pɩ́-ńkɛ̃̂nɛ nɩ? 
Pè mɛ yo ǹ tyɩ́ nɛ, nɛy náh pé tɔkɔ képah-á. À mɛ nɛ pépi tɔ sɔkɔ péwɔ swãh sɔkɔ. 
Tõ̀-á tiki ńvɩ̃́nɛ, sètɛ̃̀ dĩ́ yo ǹ pɔ́ yah wàh-ò dĩ́ tyɩ́ nɛ, à tõ̀npɩbɩ nɔ́pi ye n pɔ pè dyah pé pɔ́lɛ. Ńkɛ̃́ nónó-á kyɩ swãh sɔkɔ nɛ́tɛ́lɔ́lɛ, à kè tyah pépi tyɩ́ à pɔ n dal n mɔ nɛ́gbro tyɩ́. 
Nónó-á tɛ́lɔ́lɛ, pǎh pɔ, à pè kɔ̃ deniye pɔ́ pɔ́ núkú núkúlɛ. 
Kǎh syɩ nɛ́gbro tyɩ́, pépi tɛ̃̀ sõ̀ mɛ n sõ nɛ wãn-á kɩ wɛ pépi tyɩ́ yõ. Ǹtɛ deniye pɔ́ pɔ́ núkú núkú ye pépi tɔ wɛ. 
Pǎh pɔ́ nɔ́pilɛ n syi gbĩ́ nɛ̂nɛ, pè tyah sètɛ̃̀ dĩ́ sèyòkèlɛ n yo 
nɛ ńkɛ̃́, nónó-á à tɔkɔ pɔ tɛ́lɔ́lɛ, lɛ́rɩ́ núkú ó sɔkɔ-á pè wɛ pɩ tõ̀lɛ. Ǹtɛ kè se pɩ sɔ̃́ tɛ́ à pɔ pélɛ n sar núkúlɛ pépilɛ, pépi nónó-á kwlɔ́nɔ́ fɛ̃ syi à tõ̀ pɩ kwlɔ gbĩ̀yĩ̀kìlɛ tɔ́tɔ́ gbĩ syɔ̌kɩ̀ yah sɔkɔ nɩ? 
Sètɛ̃̀ dĩ́ mɛ yo pé sɔkɔ yɔ́ tyɩ́ nɛ: ‹Ń kódĩ́, mé náh syɩkɩ á yõ sɔkɔ dɛ́. Ńmɔ ànɛ̂ áwɔ náh sõ̀ wɛkɩ sah núkú yõ nɛ, á gbĩ̀yĩ̀kì núkú tõ̀ pɩ́nɔ́ sǎr-á kɩ pɩ deniye pɔ́ pɔ́ núkúlɛ di? 
Nɛ n yuku á pɔ́lɛ. Mé kɩ nɛ́tɛ́y mɛ̀ kɔ̃ dan núkúlɛ áwɔlɛ ń dyɔ ó. 
Ké wɛ̃̀kɩ̀ náh mɛ ńmɔ tyɩ́, kɩ fɛ̃ ń dyɔ dyɔ tyi pɩ ń pɔ́lɛ di? Tĩ̂, ń nɩ nɔ́nɔ́ se n yɔ álɛ?» 
Képah yã́hntɛnɔ náh, Yesu tɔ̃ yo pé tyɩ́ nɛ: «Sɛ̃́ ó ye, nɛ́tɛ́lɔ́ kɩ pɩ nɛ́gbrolɛ, tɛ́ nɛ́gbro kɩ pɩ nɛ́tɛ́lɔ́lɛ.» 
Yesupi-á n sɔkɔ Yerusalɛmɩ kwil, pǎh mɛ wɛ̃̀kɩ̀ yõ sɔkɔ, à ǹ tõ̀ sã́hpú ye pé yahlɛ, tɛ́ yo pé tyɩ́ nɛ: 
«Á wɛ wɛ̃̀kɩ̀ yõ à n dɔkɔ n yuku Yerusalɛmɩ sɔkɔ yɔ̀. Névi Pídĩ́ kɩ tɛ̃ n dahbɩ n pi sárká syínwìlkìpu yõ̀tãm ànɛ̂ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú gbõ̀ nɛ́npɔ́. Pè kɩ kè yah tɔkɔ nɛ wǎh yɩ̃nɛ kònmɔnɔlɛ, 
tɛ́ kɩ à dahbɩ névye gbõ̀ nónó náh pɩ Nsyifunɔlɛ, pè kɩ kṍlɛ à ko, tɛ́ kɩ à ko yèfõhlɛ, tɛ́ kɩ à syɩ dahnɩ yõ. Ǹtɛ ké sèpĩ̀n tɔ̃́-ò sõlɛ, ǹ lékó kɩ yilki mɔ.» 
Képah sɔkɔ, Nsebede kyɩ mɛ ǹ dĩ́npì tɛ̃ nɔ de Yesulɛ, tɛ́ kɔ kwĩnki ǹ yah nɛ pé yáhnkɔ̃nɔ yɔ́ pɩ ǹ tyɩ́. 
Yesu mɛ à piki yah nɛ, à se mɛ kwâh nɛ̂ tyɩ́ nɩ? Pi mɛ yo ǹ tyɩ́ nɛ, pé dĩ́npì nónó-á yɔ̀, à ǹ vyãhlɛ péwɔ kɔ̃ nɛ wàh pɔ dɔkɔ ǹ yõ̀tɔ̃rɩ́ yõ, pé sɔkɔ núkú kɔ̃ tɛ̃ ǹ gbõ̀yõ̀ sõ̀lɛ, ní-ò mó kɔ̃ tɛ̃ ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀lɛ. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Ǹpɩ̃̀-ǹkɛ̃̂ sɔkɔ ye yé n ni kélɛ. Máh kɩ n de n pi yèvyãhrɩ nɛ̂ sɔkɔ, yé tɔ se kɩ fɛ̃ de ké sɔkɔ?» Pè mɛ à syah nɛ páh kɩ fɛ̃ de ké sɔkɔ. 
Yesu mɛ yo pé tyɩ́ nɛ: «Máh kɩ n de n pi yèvyãhrɩ nɛ̂ sɔkɔ, yé tɔ kɩ de képah wɔ sɔkɔ ó, ǹtɛ ń gbõ̀yõ̀ sõ̀ ànɛ̂ ń gbɛ̃̀yɩ̃̀nɛ̀ sõ̀ tɛ̃̀nɔ́ tɛ̃̀ wɔlɛ, ńmɔ tyɩ́ náh képah wɛ̃̀kɩ̀ kɔ̃́nɔ́lɛ. Ń Sú Liyel-á kè yah sah nónó yĩnnɛ, pépi ye à kɩ kɔ̃ kélɛ.» 
Yesu tõ̀ sã́hpú nɛ́pĩ̂ gbãm nɔ́pi tɛ́lɔ́-á képah noh sɛ̃́, pé fɔkɔ vyãh nímí nɔ́pilɛ. 
Képah sɔkɔ, Yesu mɛ pé pól ye yah tɛ́ yo pé tyɩ́ nɛ: «Tir nɛ̂-á mɛ n pɩ kwlo kwlo sɔkɔ, yé kè pɩ̃ ó. Yõ̀tãm náh ńkɛ̃́nɛ pé kwlo névyelɛ nɛ n fnɛ, tɛ́ nɛ́-ǹgbnɔ tɔ n mó nɛ n fyãh pé yõ pé tɛ̃́nwɛnɔ yĩnnɛ di? 
Ǹtɛ kè náh yɩ̃nɛ kè nɛ n pɩ sɛ̃́ yépi wrɔ́. Gbɛ̃̀ntɛ̃̀náhlɛ, yépi sɔkɔ nɛ̂-à mɛ ké tyɩ́ nɛ pé yĩ́n-ńsah ǹgbɛ̃ wɛ, tɛ̃̀ yɩ̃nɛ à pɩ tõ̀npɩlɛ tɛ́lɔ́ tyɩ́. 
Tɛ́ nɛ̂-à mɛ ké tyɩ́ nɛ pé pɩ yah névilɛ yé sɔkɔ, tɛ̃̀ yɩ̃nɛ à pɩ yé yɔ̃̀lɛ. 
Sɛ̃́ ó ye, Névi Pídĩ́ náh pɔ nɛ névye tõ̀lɛ n pɩ péwɔ tyɩ́ dɛ́. Ǹmɔ ye pɔ tõ̀ pɩ yĩnnɛ névye tyɩ́, tɛ́ kɩ ǹ min kɔ̃ névye náhnáh yõ dwe yĩnnɛ. 
Yesu ànɛ̂ ǹ tõ̀ sã́hpú-á n wil Nsyeriko kwil sɔkɔ gbĩ́ nɛ̂nɛ, tɔ́wû ǹgbɛ̃ yɔ́ mɛ ǹ sõ̀ tɛ̃. 
Tɛ́ yɩ̃́tũbi nɛ́pĩ̂ nímí yísyɔ́ sõ̀ mɛ tɛ̃̀nɔ́lɛ wɛ̃̀gbalɩ sɔkɔ. Pǎh noh nɛ Yesu-á pɔ n sɔkɔ, pè tyah n yo n mɔ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ, pé Tɛ̃̀ ǹgbɛ̃, ǹmɔ nɛ̂-á Davidi Pídĩ́nɛ, ńkɛ̃́ à pé yãm yah. 
Képah tɛ̃̀nɛ, tɔ́wû dírí kõ pé yõ nɛ pè pé vyɛ̃y vike. Ǹtɛ pè tɔ̃ wãn mɔ pé yónmɔnɔ yõ nɛ ńkɛ̃́, pé Tɛ̃̀ ǹgbɛ̃, ǹmɔ nɛ̂-á Davidi Pídĩ́nɛ, ńkɛ̃́ à pé yãm yah. 
Yesu mɛ yĩn tɛ́ pè ye yah, tɛ́ yo pé tyɩ́ nɛ: «Kwâh nɛ̂ mɛ yé mɛ ké tyɩ́ nɛ mé pɩ yé tyɩ́?» 
Pè mɛ yo ǹ tyɩ́ nɛ, pé Tɛ̃̀ ǹgbɛ̃, à pé yɩ̃́ yɩkɩ mɔ. 
Képah tɛ̃̀nɛ, pé yãm Yesu tɛ̃. À mɛ tũ pé yɩ̃́ tyɩ́. Nɛ́npɔ́ swɔ, pé yɩ̃́ núkú yɩkɩ mɔ. Pè mɛ ǹ sõ̀ tɛ̃.
