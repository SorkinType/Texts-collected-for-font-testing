Nsyɔ̃.
11.
Dĩ́ yɔ́ lésõ mɛ ńyân syɩkɩ, pè n ye Lasarɩlɛ. À sõ̀ mɛ Betani kwil sɔkɔ, képah nɛ̂ sɔkɔ-á sõ̀ ǹ dósyɔ́lɛ, pè n ye Mari ànɛ̂ Martɩlɛ. 
Kénɛ Mari mɛ̀ wɔ ye lésõ làsèkwěl yɔ́ kwɛ mɔ Yesu gbyẽ̂h yõ, tɛ́ sè kele ǹ yõ̀yénnɛ. Ǹmɔ dódĩ́ Lasarɩ ye sõ̀ n yɔ. 
Képah sɔkɔ, ǹ dósyɔ́ mɛ tɛ̃ntõ̀ mɔ Yesu tyɩ́ nɛ, pé yõ̀tɛ̃̀, ǹ kódĩ́ Lasarɩ náh mɛ lènɔlɛ. 
Yesu-á képah noh, à nɛ ńkɛ̃́, Lasarɩ ńyân mɛ̀ náh pɩ ǹ ko ko ńyânnɛ. Ǹtɛ kǎh kɩ Liyel yõ̀tɔ̃rɩ́ wɛ̃kɩ, tɛ́ kɩ tɔ̃ ǹ Pídĩ́ yõ̀tɔ̃rɩ́ wɛ̃kɩ. 
Tɛ́ Martɩ, ànɛ̂ Mari, ànɛ̂ pé dódĩ́ Lasarɩ tyɩ́ sõ̀ nɔ yĩ́ĩ́ Yesu tyɩ́. 
Ǹtɛ ńyãh sɔkɔ-á sõ̀ Yesulɛ tɛ́ pè pɔ Lasarɩ ńyân tyɩ́ yo ǹ tyɩ́, à tɔ̃ wũ nɛ́npɔ́ à sèpĩ̀n nínì pɩ. 
Képah náh, à yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè yɛ pé pé syɩ n yuku Nsyude gbɛ̃̀nyah sɔkɔ. 
Wǎh yo sɛ̃́, pè mɛ nɛ, kwɔ́-ò dĩ́, nɛ́npɔ́ sɔkɔ kó lésõ pè n yah n kɔ̃ ǹ mínkò-ǹsahlɛ dyêhlɛ sɛ̃́ yɔ̀ sɛ̃́ yɔ̀, tɛ́ à tɔ̃ n yah n kɔ̃ ǹ syɩ́nyuku-ńsahlɛ nɛ́npɔ́ sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ, lɛ́rɩ́nɔ́ gbɔ̃ninɔ́ kó gbĩ̀yĩ̀kì sɔkɔ nɩ? Névi nɛ̂-à n kɔ tyah kyòolɛ, kèkõyṍ kyòo-á à n nɛ n yah. Képah-á, tɛ̃̀ gbáhkɩ̀ mɩ́nɔ́ náh nɔ. 
Ǹtɛ névi nɛ̂-à n kɔ lékã̂hnɛ kyòo kɛ̃́nɛ, tɛ̃̌ ǹ gbáhkɩ̀lɛ n mɩ. 
Wɛlɔ nɔ́pi yóntɛnɔ náh, Yesu tɔ̃ yo pé tyɩ́ nɛ, pépi kódĩ́-á mɛ dɔy sɔkɔ, ǹtɛ páh kɩ kyɩ à yilki mɔ. 
Képah sɔkɔ, ǹ tõ̀ sã́hpú mó nɛ Tɛ̃̀ ǹgbɛ̃, dɔy sɔkɔ-à kɩ nɛ̂ Lasarɩlɛ, wǎh kɩ dɛ. 
Ǹtɛ Yesu wɛy sɔkɔ, Lasarɩ kúnɔ́ tyɩ́ ye à yo, tɛ́ ǹ tõ̀ sã́hpú wɔ mó wɛ nɛ dɔy gbɔy tyɩ́-á à yo. 
Kǎh pɩ sɛ̃́, Yesu kè yo yɩkɩ mɔ pé tyɩ́ nɛ Lasarɩ-á ku. 
Ǹtɛ péwɔ mɛ́-ńkɛ̃̂nɛ sõ̀ nɛ́npɔ́, képah-á mɛ nɔ yĩ́ĩ́ péwɔ tyɩ́ pépi yĩnnɛ. Kǎh pɩ sɛ̃́, pǎh kɩ sɛ̃ yĩ́ĩ́ péwɔ yõ. Tɛ́ núkúnúkú, pè yɛ pé sɔkɔ ǹ tnɔ̂. 
Képah sɔkɔ, Toma nɛ̂-á pè n ye Didiminɛ, ǹmɔ pé no syah nɛ, pè yɛ pépi tɔ kyɩ ku ǹnɛ wɛ̃ tyɩ́. 
Yesu-á yĩni tɛ̃ nɛ́npɔ́ gbĩ́ nɛ̂nɛ, à kyɩ nɛ yah Lasarɩ lékó sõ̀ sèpĩ̀n ǹyẽ̂h pɩ tɛ diki sɔkɔ. 
Yé yah, Betani ànɛ̂ Yerusalɛmɩ wrɔ́ náh sõ̀ mɛ kah gbɛ mɔ. Kè sõ̀ mɛ kílónɔ́ tɔ̃́nɔ́ kénkɔ̃lɛ. 
Képah ye, Yerusalɛmɩ tãn náhnáh sõ̀ mɛ pɔ Martɩ ànɛ̂ Mari yétĩ kélé-ńsah pé dódĩ́ yáhnkwlɔnɔ sɔkɔ. 
Martɩ-á fyɔ̀ noh nɛ Yesu-á mɛ n pi, à kyɩ à yohnɩ. Tɛ́ Mari wɔ sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ gbô sɔkɔ. 
Képah sɔkɔ, Martɩ mɛ yo Yesu tyɩ́ nɛ, pé yõ̀tɛ̃̀, wàh sõ̀ mɛ ásõ̂ sɔkɔ, pé dódĩ́ náh náhkɩ n ku n pi. 
Ǹtɛ núkúnúkú ńtɛ̃ fyé sɔkɔ, páh pɩ̃ nɛ wàh kɩ sɔ̃́ tir dùkù nɛ̂ yah kɔ̃ Liyel tyɩ́, wǎh kɩ kè pɩ ǹ tyɩ́. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ: «Á dódĩ́ lékó kɩ yilki mɔ.» 
Wǎh yo sɛ̃́, Martɩ nɛ, páh pɩ̃ nɛ lékyɩ̂-à kɩ pɔ n yilki n mɔ gbĩ̀yĩ̀kì tɛ́y sõlɛ, ǹ tɔ lékó-á kɩ yilki mɔ. 
Képah tɛ̃̀nɛ, Yesu yo ǹ tyɩ́ nɛ: «Ńmɔ ye lékyɩ̂ yílkínmɔ-ò ànɛ̂ minnɛ. Névi nɛ̂-à sɛ̃ ńmɔ yõ, tɛ̃̀-à sɔ̃́ ku, à kɩ tyɛ́-ńkɛ̃̂ min wɛ; 
ànɛ̂ névi nɛ̂-à mɛ min sɔkɔ tɛ́ sɛ̃ ńmɔ yõ, tɛ̃̀ náh tɔ̃ n ku n yah n pi Liyel tyɩ́ tĩ̀nnɛ. Á se sɛ̃ képah yõ, Martɩ?» 
Martɩ mɛ nɛ, àwe-á Tɛ̃̀ ǹgbɛ̃, páh sɛ̃ ké yõ nɛ ǹmɔ-á Liyel nɛy yáhntɔkɔnɔlɛ, nɛ̂-á ǹ Pídĩ́nɛ, ǹmɔ nɛ̂-á sõ̀ yɩ̃nɛ à pɔ kèkõyṍ. 
Képah yóntɛnɔ náh, à ǹ syɩ kyɩ plɔ yo ǹ gbɛ̃̀nsê Mari tyɩ́ nɛ, kwɔ́-ò dĩ́-á mɛ de n tɛ. Tɛ́ wǎh mɛ ǹnɛ n ye. 
Mari-á fyɔ̀ képah noh, à núkú yuku à sɔkɔ Yesu tnɔ̂. 
Yesu náh tɔ̃̌nɩ̀ sõ̀ mɛ de kwil sɔkɔ. À sõ̀ mɛ fɩ̃́ nɛ̂ sɔkɔ-á sõ̀ Martɩ pɔ à yohnɩ. 
Tɛ́ Yerusalɛmɩ tãn nónó-á sõ̀ mɛ Mari tnɔ̂ gbô sɔkɔ, mɛ ǹ yétĩnɛ n kele, pépi-á à wɛ à yuku wil súú sɛ̃́, pè ǹ sõ̀ tɛ̃. Pè sõ̀ mɛ n sõ nɛ, pɔ̀ gbó-ńsah-á à n yuku diki yõ. 
Ǹtɛ Mari-á kyɩ yĩni tɛ̃ Yesu mɛ́-ńsah sɔkɔ, tɛ́ à wɛ, à gbètukunɔ di Yesu yah sɔkɔ tɛ́ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, wàh sõ̀ mɛ ásõ̂ sɔkɔ, pé dódĩ́ náh náhkɩ n ku n pi. 
Képah sɔkɔ, Yesu-á à wɛ à mɛ n gbo à tahlɩ ǹ tɛ́kɩ́nmɔpu tyɩ́, kè ǹ nɩ kah tɛ̃ képékèyɔ̂, ǹ yah kwɔ. 
À mɛ nɛ, ńyãh sɔkɔ se pè yɩ̃ ǹ lékólɛ nɩ? Pè mɛ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, à pɔ kyɩ yah. 
Képah sɔkɔ, Yesu yétĩ kwɛ. 
Képah tɛ̃̀nɛ, ǹ tɛ́kɩ́nmɔpu nɛ, pè yah, ǹ tyɩ́-á sõ̀ mɛ nɔ yĩ́ĩ́ ǹ tyɩ́ dɛ́! 
Tɛ́ tɔ̃ mɛ n yo nɛ, ǹmɔ nɛ̂-á yɩ̃́tũ̂ dĩ́ yɩ̃́ yɩkɩ, à kó sõ̀ fɛ̃ kũ yah kõ n pi Lasarɩ tyɩ́ nɩ? 
Képah sɔkɔ, kè tɔ̃ Yesu nɩ kah tɛ̃ képékèyɔ̂, à mɛ sɔkɔ diki yõ. Tɛkɩ diki ye sõ̀ kélɛ, tɛ́ ké yah mɛ víkénɔ́lɛ dáh-ǹgbɛ̃ yɔ́lɛ. 
Yesu mɛ yo no tyɩ́ nɛ, pè dáhkɩ̀ mɛ̀ tɔkɔ tɛ̃ yõ. Lékókè dósê Martɩ mɛ à syah nɛ, Tɛ̃̀ ǹgbɛ̃, ǹ yɩ̃́nɔ́ sèpĩ̀n ǹyẽ̂h-á yɔ̀. Wǎh kɩ mɛ n põ dò. 
Képah sɔkɔ, Yesu yo ǹ tyɩ́ nɛ: «Mé náh kè yo á tyɩ́ nɛ, wáh kɩ sɛ̃ ń yõ, á yĩ́-á kɩ tɛ̃ Liyel yõ̀tɔ̃rɩ́ yõ di?» 
Képah tɛ̃̀nɛ, no mɛ dáhkɩ̀ tɛ̃ yõ. Yesu mɛ ǹ yah yɔ̃ yĩ̂nyõ tɛ́ nɛ: «Bɔ̌, mé n wɛkɩ á tyɩ́ wáh sɔ̃́npɩ́ ń nírí tɛ̃. 
Mé pɩ̃ nɛ wáh mɛ ń nírílɛ n tɛ̃ gbĩ́mɛgbĩ́. Ǹtɛ mé kélɛ n yo tɔ́wû mɛ̀ yĩnnɛ nɛ̂-á ń mɔ gbɩ̃ sah yɔ̀, nɛ pè sɛ̃ ké yõ nɛ, áwɔ-á tɛkɩ mɔ ńnɛ.» 
Képah yóntɛnɔ náh, à kah yo mɔ nɛ ńkɛ̃́: «Lasarɩ, wil nɛ́npɔ́!» 
Képah sɔkɔ, Lasarɩ nɛ̂-á sõ̀ ku, à yuku wil ǹ gbyẽ̂h ànɛ̂ ǹ gbã̀nnɛ púkúbínɔ́lɛ fwɔ gbrólɛ; tɛ́ ǹ yèpɔ mɛ fɔ́kɔ́nɔ́lɛ fwɔlɛ. Yesu mɛ yo pé tyɩ́ nɛ, pè à wah mɔ, pè mó yɛ à kɔ̃ à nɛ n yuku. 
Névye nónó-á sõ̀ mɛ pɔ Mari tnɔ̂, tɛ́ pɔ Yesu tyípɩ́nɩ́ mɛ̀ wɛ sɛ̃́, pé náhnáh sɛ̃ Yesu yõ. 
Ǹtɛ pé túkù wɔ mó kyɩ ǹ tyípɩ́nɩ́ mɛ̀ sèsahrɩ yah sah Farisi wɛ̃̀kɩ̀ kɔ́pú tyɩ́. 
Kǎh mó pɩ sɛ̃́, Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ sárká syínwìlkìpu yõ̀tãm mɛ pé no tukey tùkèpu tĩ̀nkì ye, pè kɔ̃ tɛ̃ ké tyɩ́ sɔkɔ. Pè wɛ̃ syah nɛ, dĩ́ mɛ̀-á mɛ gbǐl tyilɛ n kah n pɩ dɛ́, sɔ̃́ se pé kɩ pɩ nɩ? 
Páh kɩ à sah à yɔ̃ sɔkɔ sélɛ n pɩ sɛ̃́ yɔ̀, no pól-á kɩ sɛ̃ ǹ yõ. Ǹtɛ képah-à pɩ, Wrome tãn-á kɩ yuku pépi Liyel ní-ńsah gbô sɔ, tɛ́ kɩ pépi yìkì tyɩ́ yɩkɩ. 
Ǹtɛ pé sɔkɔ yɔ́ lésõ mɛ, pè n ye Kayifilɛ. Léyílɛ, ǹmɔ ye lésõ sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃nɛ. Ǹmɔ mɛ pé no syah nɛ: «Yé náh kwâh yɔ́lɛ n pnɛ dɛ́! 
Yé náh pɩ̃ nɛ, dĩ́ nɛ́núkù-à kɩ ko mɔ ápi yìkì swɔ kè kɔ̃ kè wɛ pwah, képah-á kɩ nɛkɩ kal ápilɛ ǹ tɛ̃̀nɔ́lɛ min sɔkɔ, tɛ́ á yìkì pól tyɩ́ mó pɔ yɩkɩ di?» 
Ǹtɛ ǹmɔ yah yĩ́ĩ́ náh sõ̀ à yo tyi nɔ́pilɛ dɛ́! Wǎh sõ̀ mɛ sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃nɛ léyílɛ sɛ̃́, képah ye Liyel mɛ tɛ̃ntõ̀lɛ à kɔ̃ à syi yo sah nɛ, Yesu-á yɩ̃nɛ à ku ǹ yìkì névye yĩnnɛ. 
Tɛ́ ǹ yìkì névye ó yĩn náh à kɩ n ku n pi dɛ́, ǹ kúnɔ́ kɩ Liyel wɛ̃́npì pól tɔ tuke mɔ pɩ núkúlɛ, pépi nónó-á mɛ karkɩ tɛ̃ tɛ̃ kɔ̃. 
Kayifi-á fyɔ̀ képah yo, à tɔkɔ lésõ tyɩ́, pè núkú Yesu kònmɔnɔ sah gbáhkɩ̀ yõ. 
Kǎh pɩ sɛ̃́, Yesu náh tɔ̃ mɛ n gblɔ̌ynɛ n wilki n mɔ kwèyy Nsyifunɔ yah. À yal ànɛ̂ ǹ tõ̀ sã́hpúlɛ, pè kyɩ tɛ̃ kwil yɔ́ sɔkɔ, pè n ye Yefrayiminɛ. Képah sõ̀ mɛ gbɛ̃̀nyah yɔ́ sɔkɔ, nɛ̂-á mɛ gbɛ̃́nnɩ tnɔ̂. 
Ǹtɛ Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh-á sõ̀ mɛ de n tɛ sɛ̃́, névye náhnáh sõ̀ mɛ n wil gbnɔ sɔkɔ, tɛ́ mɛ n dɔkɔ n yuku Yerusalɛmɩ kwil sɔkɔ. Pè sõ̀ mɛ syáh mɛ̀ dyàh tɛ̃, nɛ pé kyɩ pé gblɔ̌y tɔ̃ n sah, à yɩ̃nɛ pé tyi kõ̀nsàhnɔ́ wã̀llɛ. 
Pǎh sõ̀ mɛ Liyel ní-ńsah gbô sɔkɔ gbĩ́ nɛ̂nɛ, pè sõ̀ mɛ Yesulɛ n yah n kɔ̃, tɛ́ mɛ wɛ̃nɛ n piki n yah nɛ ńkɛ̃́, kè se mɛ sɔ̃́ nɩ? À se kɩ pyě yɛ wũ syáh mɛ̀ sɔkɔ pɔ́nɔ́lɛ nɩ? 
Tɛ́ sárká syínwìlkìpu yõ̀tãm ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́pú lésõ mɛ ké gbah yo sah tɛ névye tyɩ́ nɛ, nɛ̂-à fyɔ̀ Yesu mɛ́-ńsah pɩ̃, tɛ̃̀ pɔ kè yo pé syah, pé kɔ̃ pé à wɛ tɛ̃.
