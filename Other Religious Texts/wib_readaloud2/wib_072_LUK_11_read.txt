Likyi.
11.
Sṍ yɔ́lɛ, Yesu sõ̀ mɛ Liyellɛ n ni fɩ̃́ yɔ́ sɔkɔ. Wǎh ni tɛ, ǹ tõ̀ sã́hpú sɔkɔ yɔ́ à syah nɛ, pé Yõ̀tɛ̃̀, à pépi kwɔ Liyel ni dùkùlɛ, sɔ̃́-á Wèlnwìlkì-ò Nsyɔ̃ sõ̀ kélɛ ǹ tõ̀ sã́hpú kwɔ. 
Képah sɔkɔ, à yo pé tyɩ́: «Yâh Liyellɛ n ni, yé nɛ n yo nɛ ńkɛ̃́: ‹Sú Liyel, no pól kɔ̃ pè ǹ pɩ̃ Liyellɛ. Á tɔ̃́rɩ́ kɔ̃ kè pɔ. 
Á sõmɛsṍ gbĩ̀yĩ̀kì yõkelɛ álɛ n kɔ̃, nɛ̂-á kɩ n yɩ̃nɛ n pi álɛ. 
Nónó-á n pɩ n de ápi tyɩ́, á ye mɛ sélɛ n yɔ̃ n mɔ pé tyɩ́. Képah ye, ápi tɔ tyípêl yɔ̃ mɔ á kɔ̃. Ànɛ̂, káh n yɛ à sétõ tɔ̃̀nmɔnɔ fɛ̃ n syi.›» 
Képah náh, Yesu tɔ̃ yɔ̃ sɔkɔ yo pé tyɩ́ nɛ: «Yépi sɔkɔ yɔ́ tyɩ́ kódĩ́-à mɛ, tɛ́ à pɔ wil ǹ tyɩ́ lénɩlɛ nɛ ńkɛ̃́, pé kódĩ́ yɔ́-á mɛ wil kɔ́lɔ́ sɔkɔ, à pɔ pé tyɩ́, tɛ́ yõ yõ kwâh náh mɛ pé tyɩ́ kɩ à kɔ̃. Ńkɛ̃́, à dɛkɩ pé kɔ̃ búrn plɔ́ tɔ̃́nɔ́lɛ. 
Ǹtɛ, ní-ò mɛ̀-à ńtyɩ̃́nɔ tɛ̃ wɔ̀kɔ̀ sɔkɔ táh, tɛ́ yo mɔ ǹ tyɩ́ nɛ, à káh pélɛ n kwlɔ, wɔ̀kɔ̀ yah-á vike tɛ núkúnúkú, péwɔ ànɛ̂ pé wɛ̃́npì-á sɛ̃, pé náh fɛ̃ yuku búrnnɛ à kɔ̃ n pi. 
Mé kélɛ n yo yé tyɩ́ nɛ: dĩ́ mɛ̀-à sɔ̃́ yuku sélɛ à kɔ̃́-ńkɛ̃̂nɛ pé kódírí yĩnnɛ, à kɩ yuku ǹ yétõ̂ yî póllɛ à kɔ̃ ǹ nínɔ́ gblɔ́vyã́hrɩ́ tɛ̃̀ wɔ yĩnnɛ. 
Ǹtɛ, ńmɔ tɛ̃̀ wɔ kélɛ n yo yé tyɩ́ nɛ: yé nɛ n ni Liyel tyɩ́, à kɩ yélɛ n kɔ̃. Yé yáhnkɔ̃nɔlɛ n pɩ, yé kɩ n wɛ. Yé wɔ̀kɔ̀ yah mɩ, kè kɩ yɩkɩ mɔ yé kɔ̃. 
Sɛ̃́ ó ye, nɛ̂-à ni Liyel tyɩ́, tɛ̃̀ n wɛ n syi. Nɛ̂-à yah kɔ̃, tɛ̃̀ n wɛ. Ànɛ̂ nɛ̂-à wɔ̀kɔ̀ yah mɩ, kè n yɩkɩ n mɔ tɛ̃̀ kɔ̃. 
Sú nɛ̂ tɛ̃̀ mɛ yépi sɔkɔ, ǹ pi-à yɛ̃́nkɔ̂ ni ǹ tyɩ́, à kɩ wahynɛ à kɔ̃? 
Képah náh pɩ, wàh séngbɩ̃́ ni ǹ tyɩ́, à kɩ mìnkɔ̃nɛ à kɔ̃? 
Ǹtɛ, yépi nónó-á nɛ́pêllɛ, yâh yé wɛ̃́npì kɔ̃́nɔ́ yah wɛ yípéplɔlɛ, sú Liyel nɛ̂-á yĩ̂nyõ, ǹmɔ náh n pɩ plɛ n kal, à ǹ Mírkílɛ ké yáhnkɔ̃pu kɔ̃ n pi di?» 
Sṍ yɔ́lɛ, Yesu sõ̀ mɛ sétõ yɔ́lɛ n yĩni n wilki dĩ́ yɔ́ sɔkɔ, nɛ̂-á à kɔ̃ à wɛkɩ kwlɔ. Sétõ-á fyɔ̀ wil dĩ́ mɛ̀ sɔkɔ, dĩ́ tyah n wɛkɩ, kè pɩ gbɛ̃̀ndùnìnɛ tɔ́wû tyɩ́. 
Tɛ́ túkù mó sõ̀ mɛ n yo nɛ ńkɛ̃́, sétã̀n yõ̀tɛ̃̀ Belsebuli gbõ̀ yõ-á à n yĩni sétã̀nnɛ. 
Túkù mó ǹ mɔ́nyah yĩnnɛ, mɛ n yo ǹ tyɩ́ nɛ, à gbǐl yɔ́ pɩ pé yah, nɛ̂-á n wil yĩ̂nyõ. 
Ǹtɛ, Yesu-á pé fɔ́kɔ́sõnɔ pɩ̃ sɛ̃́, à yo pé tyɩ́: «Kwil nɛ̂ névye-à gbɛkɩ tyah wɛ̃nɛ n fwo, kénɛ kwil kɩ pɔ yɩkɩ, ké vɩ̀ kɩ syɩkɩlɩ dohnɩ wɛ̃ yõ. 
Ǹtɛ, yáh nɛ ńkɛ̃́, sétã̀n yõ̀tɛ̃̀ gbõ̀ yõ-á ńmɔ sétã̀nnɛ n yĩni-à; sétõ-à tyah ǹ gblɔ̌ynɛ n fwo, ǹ yõ̀tɔ̃rɩ́ se kɩ n wũ n pi? 
Ǹtɛ, ńmɔ-à mɛ sélɛ n yĩni Belsebuli gbõ̀ yõ, tɛ́ yépi pye wɔ-á dò, nɛ̂ gbõ̀ yõ mɛ pépi n yĩni sélɛ wɛ́? Pépi ńtɛ̃ ye kɩ lékàhlɛ yé kɔ̃. 
Ǹtɛ, Liyel gbõ̀ yõ-à mé n yĩni sétã̀nnɛ, képah n wɛ̃kɩ nɛ ńkɛ̃́, Liyel tɔ̃́rɩ́-á yĩni tɛ̃ yé tyɩ́. 
Tǎhkɩ̀-á mɛ dĩ́ nɛ̂ sɔkɔ, ànɛ̂ mɛ́fwóyîlɛ ǹ tyɩ́, wàh ǹ kételɛ n syɩkɩ, tir yɔ́ ńtɛ̃ náh fɛ̃ ǹ gbã̀n yî nɔ n pi. 
Ǹtɛ, nɛ̂ tǎhkɩ̀-á kal ǹ tyɩ́lɛ, ǹmɔ-à pɔ, tɛ́ tɛ̃ wɛ ǹ tyɩ́, à kɩ ǹ mɛ́fwó yî tɛ̃ syi, wǎh sõ̀ mɛ ǹ gbõ̀ sah nónó yõ, tɛ́ kɩ ǹ wɔ̀kɔ̀ yî kwel dibi. 
Képah ye, névi nɛ̂ náh mɛ ń pèpɛy nɛynɛ, ń pělnɛy ye tɛ̃̀nɛ. Névi nɛ̂ náh ńnɛ n yohnɩ n mɔ tùkènmɔnɔ sɔkɔ, tɛ̃̀ mɛ kárkɩ́nɔ́lɛ n pɩ.» 
«Sétõ-à wil névi sɔkɔ, à n kyɩ gbɛ̃́nnɩ kɔ n kɔ̃, ǹ mɩ̀-ǹsah yah n kɔ̃. Wàh wɛ́-ńkɛ̃̂nɛ, à n yo ǹ gblɔ̌y tyɩ́ nɛ, páh sõ̀ wil ńyãh sɔkɔ, páh kɩ pé syɩ yuku nɛ́npɔ́. 
Wàh ǹ syɩ, à n kyɩ ǹ tɛ̃̀-ǹsah dyàh wɛ yàlntɔ̃nsàhnɔ́lɛ, plɛ́ntnɔɔ. 
Képah tɛ̃̀nɛ, sétõ mɛ̀ n kyɩ sé no sétã̀nnɔ́ kwǎrninɔ syah, nónó-á peke kal ǹmɔlɛ. Sè n kyɩ n de n tɛ̃ tɛ̃̀ sɔkɔ. Kàh pɩ sɛ̃́, tɛ̃̀ mɛ̀ dùkù ké syɩ n peke n kal ǹ gben tɛ̃̀nɛ.» 
Képah tɛ̃̀nɛ, Yesu-á mɛ képah yónɔ́ sɔkɔ, pi yɔ́ tɛ̃ tɔ́wû nɩyṍ sɔkɔ, tɛ́ yo mɔ ǹ tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ, sê nɛ̂ fɔkɔ sɔkɔ-á à wil, tɛ́ ǹ yĩ́nɔ́ yĩ, yèvɩnɩ-á mɛ ǹ tyɩ́. 
Ǹtɛ, Yesu mɛ yo pi tyɩ́: «Yèvɩnɩ ye pyě mɛ névye tyɩ́, nónó-á Liyel wɛynɛ n nohnɩ, tɛ́ ké tyilɛ n pɩ ké wɛ̃̀kɩ̀ yõ.» 
Képah sɔkɔ, névye-á pɔ n tuke n mɔ Yesu tnɔ̂, à tyah n yo nɛ: «Kwɛy gbĩ névyelɛ, nɛ́pêl ye pélɛ, pè mɛ gbǐl yɔ́lɛ n yah n kɔ̃. Ǹtɛ, kèyǎh ńtɛ̃ náh n pɩ pè wɛ̃kɩ n pi, à wil Liyel tɛ̃ntõ̀ syínyo-ò Yonasɩ tyɩ́ sɔkɔ. 
Sɔ̃́-á Yonasɩ lésõ pɩ kwní yṍnmɔnɔlɛ Ninivi tãn tyɩ́, sɛ̃́ ó ye, Névi Pi tɔ kɩ pɩ kwní yṍnmɔnɔlɛ kwɛy gbĩ névye tyɩ́. 
Pi nɛ̂-á lésõ yõ̀tɛ̃̀nɛ Saba sétáh yõ, à lésõ wil fɩ́ɩ́ sétáh yĩ̀n tyɩ́, à pɔ yõ̀tɛ̃̀ dĩ́ Salomo lékã́m wɛlɔ noh. Ǹtɛ gbɩ yõ, nɛy mɛ ásõ̂, nɛ̂-á kal Salomolɛ. Kǎh pɩ sɛ̃́, Liyel fṍ tukey tuke sṍlɛ, kénɛ yõ̀tɛ̃̀ pi kɩ yuku kwɛy gbĩ́ névye tyɩ́, kɩ lékàhlɛ pè kɔ̃. 
Ànɛ̂ tɔ̃, Ninivi kwil tãn lésõ fɛ̃ syi Liyel wɛ̃̀kɩ̀ tɛ̃, Yonasɩ-á Liyel tɛ̃ntõ̀ yo pé tyɩ́. Ǹtɛ, nɛy mɛ ásõ̂, nɛ̂-á kal Yonasɩlɛ. Kǎh pɩ sɛ̃́, Liyel fṍ tukey tuke sṍlɛ, Ninivi kwil tãn kɩ yuku kwɛy gbĩ névye tyɩ́, kɩ lékàhlɛ pè kɔ̃.» 
«Névi se kɩ fɔ̀tnɔ̀ soh wɔ̃bɩ, képah náh pɩ, kɩ dú kwĩnki mɔ ké yah? À náh kè sah n pi fɔ̀tnɔ̀ sahkɩ sɔkɔ, kyòo wɛ yĩnnɛ wɔ̀kɔ̀ sɔkɔ dépú tyɩ́ di? 
Sɛ̃́ ó ye, á yɩ̃́ ye á wil kyòolɛ. Á yɩ̃́-à mɛ plɛ, à wil tɔ pól n nɛ mɛ kyòo sɔkɔ. Tɛ́, á yɩ̃́-à mɛ plɛ́-ńkɛ̃̂nɛ, á wil pól n nɛ mɛ fɛ̃́ntũ̂ sɔkɔ. 
Á gblɔ̌y kyɔmɩ yĩ́ĩ́, kyòo nɛ̂-á á sɔkɔ, kè káh nɛ mɛ fɛ̃́ntũ̂nɛ. 
Tɛ́ á wil pól-à mɛ kyòolɛ, fɛ̃́ntũ̂ tyah kɛ̃́nɛ ké sɔkɔ, á kɩ mɛ kyòolɛ, sɔ̃́-á fɔ̀tnɔ̀lɛ kyòo á tyɩ́.» 
Yesu-á wɛkɩ tɛ, Farisiyɛ̃ dĩ́ yɔ́ à ye yõke yõ yĩnnɛ ǹ gbô. Képah tɛ̃̀nɛ, pè de à tɛ̃ yõke yah. 
Farisiyɛ̃ mɛ̀-á Yesu wɛ, à tyah n yõ tɛ́ à náh ǹ gbã̀n wu Nsyifunɔ wu dùkùlɛ, kè ǹ gbã̀n du. 
Képah tɛ̃̀ sɔkɔ, Yesu mɛ yo ǹ tyɩ́: «Sɛ̃́ ó ye, yépi Farisiyɛ̃nɔlɛ, yě wóhlɔ́ ànɛ̂ yɛ̃́yṍvyɔ náhlɔ́ wu n tɔ̃ n sah, tɛ́ yé fɔkɔ táhlɔ́ n nɛ n mɛ yúkúnɔ́lɛ pèkè, ànɛ̂ yɩ̃be dékè tyilɛ. 
Lékã́m kɛ̃́ névye nɔ́pi, nɛ̂-á mɔ vyɔ́ náhlɔ́lɛ, à náh sé sɔkɔ táhlɔ́ tɔ mɔ di? 
Yé sé sɔkɔ táhlɔ́ yî yɩ̃ yãm tãn kɔ̃! Képah-à pɩ, sé pól kɩ pɩ pallɛ yé tyɩ́. 
Yépi Farisiyɛ̃nɔlɛ, pèkè mɛ yé yõ! Yě yímɩ̂ ànɛ̂ swãh sɔkɔ pɛ̃́nfěnè pól gbãm-ònɔ́lɛ Liyellɛ n kɔ̃. Ǹtɛ, yě n yahle tɔ̃́nsàhnpɩnɔlɛ névye tyɩ́, ànɛ̂ Liyel tyɩ́ náh nɔ yé tyɩ́. Képah nɛ̂ tɔ-á sõ̀ yé yɩ̃nɛ yè nɛ n pɩ gbõ̀nwɔ̀kɔ̀ kɛ̃́nɛ. 
Yépi Farisiyɛ̃nɔlɛ, pèkè mɛ yé yõ! Yě tɛ̃̀-ǹtã̀n gbrolɛ n yah n kɔ̃ Liyel ní-ńtã̂n vɩ̀ sɔkɔ, ǹ tɛ́ nɛ mɛ ké tyɩ́ nɛ, névye nɛ n tɔ̃ n sah n wɛkɩ n pɔ yé tyɩ́ tɔ́wû tɛ̃̀-ǹtã̀n sɔkɔ. 
Pèkè mɛ yé yõ, yépi nónó-á mɛ á kɩ nɛ, léyã̂l nírínɔ́ sɔ̃́, névye n nɛ n kɔ sé yõ, pɩ̃́-ńkɛ̃̂nɛ.» 
Yesu-á yo sɛ̃́, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́-ò yɔ́ à syah nɛ, kwɔ́-ò dĩ́, wǎh mɛ n wɛkɩ sɔ̃́, pépi tɔ-á à n gbyɔ dɛ́! 
À mɛ yo ǹ tyɩ́: «Yépi Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú tɔlɛ, pèkè mɛ yé yõ! Yépi nónó-á ńkɛ̃́nɛ no twah n mɔ tɔ́kɔ́pêllɛ, tɛ́ yé náh n fɛ̃ n syi n so n mɔ pé tyɩ́ kèngbɔ núkú fyélɛ. 
Pèkè mɛ yé yõ, yépi nónó-á Liyel tɛ̃ntõ̀ syínyopu léyã̂llɛ n tɔ̃ n sah n mɔ, tɛ́ yépi náh tãn ye mó tɔ̃ ko mɔ pélɛ. 
Képah n wɛ̃kɩ nɛ, yáh yé súbí tyípɩ́nɔ́ fɛ̃ syi. Pépi-á Liyel tɛ̃ntõ̀ syínyopu ko mɔ, tɛ́ yépi sɔ̃́npɩ́ pé léyã̂llɛ n tɔ̃ n sah n mɔ. 
Képah ye sõ̀ Liyel yo lékã́m sɔkɔ nɛ ńkɛ̃́: ‹Mé kɩ tɛ̃ntõ̀ syínyopu tɛkɩ mɔ pè kɔ̃, ànɛ̂ tɛ̃̀ntõbelɛ. Pè kɩ pé túkù ko mɔ, tɛ́ kɩ túkù fõh.› 
Kǎh pɩ sɛ̃́, Liyel kɩ kwɛy gbĩ névye fõh n pi Liyel tɛ̃ntõ̀ syínyopu nɔ́pi yĩnnɛ, nónó-á ko mɔ kèkõyṍ mɔ gbĩ́ dukulɛ, 
à tɔkɔ Abɛlɩ kònmɔnɔ tyɩ́, à pɔ syɩ Nsakarɩ kúnɔ́ tyɩ́, nɛ̂-á ko mɔ sárká wìlkì-ǹsah, ànɛ̂ Liyel ní-ńsah wɔ̀kɔ̀ ǹgbɛ̃ wrɔ́. Gbɩ yõ, mé kélɛ n yo yé tyɩ́ nɛ: kwɛy gbĩ névye kɩ n fõh n pi képah yĩnnɛ. 
Pèkè mɛ yépi Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú yõ, yépi nónó-á Liyel wɛ̃̀kɩ̀ wɛ́npɩ̃nɔ yah kõ névye tyɩ́. Yépi náh mɛ n kɔ kélɛ, tɛ́ nónó-á se nɛ, pé nɛ n kɔ kélɛ, yé pépi yĩni mɔ.» 
Képah sɔkɔ, Yesu wìlntɛnɔ náh nɛ́npɔ́, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú, ànɛ̂ Farisiyɛ̃nɔ pé fɔkɔ vyãh ǹnɛ kègbɩlɛ, tɛ́ tyah ǹnɛ n piki n yah tyi duke duke pól yõ. 
Pè sõ̀ mɛ ǹnɛ n mɔ n yah nɛ, à pilki ǹ vyãh wɛlɔ sɔkɔ pé kɔ̃ pé à wɛ tɛ̃.
