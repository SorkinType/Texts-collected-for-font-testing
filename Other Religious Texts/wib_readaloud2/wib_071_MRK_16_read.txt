Markɩ.
16.
*Mɩ mɩ yòh-á sɔkɔ, Makadala Mari, ànɛ̂ Nsyakɩ yṹ Mari, ànɛ̂ Salome kyɩ làsèkwěl dwe Yesu yõ mɔ mɔ tɛ̃̀nɛ. 
Dimasyi sṍ séngbɔ̃́ púsɔ́, gbĩ gbo gbĩ́nɛ, pè yuku diki yõ. 
Syɔ́ nɔ́pi mɛ n yo wɛ̃ tyɩ́ nɛ, nɛ̂ se kɩ dáh-ǹgbɛ̃ mɛ̀ yɩkɩ mɔ pé kɔ̃ nɩ? 
Tɛ́ pǎh nɛ, pé yah, pè dáhkɩ̀ mɛ̀ wɛ pílntonɔlɛ diki yah sɔkɔ. 
Pè de yuku diki sɔkɔ, tɛ́ kyɩ kètɛ̃̀npìkì yɔ́ wɛ, à mɛ fwɔ̀mɔ̀npíkí mɔ kɔ̃ tɛ̃ gbõ̀yõ̀ sõ̀lɛ. Tíkí pè kah tɛ̃. 
Tɛ́ kètɛ̃̀npìkì mɛ̀ mɛ yo pé tyɩ́: «Yé káh tíkí pɩ! Nasarɛti Yesu ye yé n yah n kɔ̃, nɛ̂-á pè syɩ *dahnɩ yõ. À yuku wil diki sɔkɔ. À náh mɛ ásõ̂. Yé pɔ yah ǹ sɛ̃́-ńsahlɛ. 
Yé kyɩ yo Pyɛrɩ ànɛ̂ pé no tɛ́lɔ́ tyɩ́ ńkɛ̃, wǎh mɛ pélɛ n syɩkɩ Kalile sɔkɔ, sɔ̃́-á lésõ à kè yo sah pé tyɩ́.» 
Syɔ́ nɔ́pi-á képah noh sɛ̃́, pè wil diki, à to sɔkɔ wil syahrɩ sɔkɔ tíkí tɛ̃̀ tyɩ́. Pè náh mɛ n yo n mɔ nɛ tyɩ́, tíkí kàhntɛ̃nɔ tɛ̃̀ tyɩ́.  
Dimasyi sṍ séngbɔ̃́ púsɔ́, Yesu yuku wil diki sɔkɔ. À ǹ gblɔ̌ynɛ Makadala Mari ǹgbò wɛ̃kɩ, wǎh lésõ *sétã̀n kwǎrninɔ yĩni wilki pi nɛ̂ sɔkɔ. 
Makadala Mari kyɩ kè yo Yesu *tõ̀ sã́hpú tyɩ́. À kyɩ pé wɛ kũ yèkwɔrɩ gbní sɔkɔ. 
Pǎh noh, Mari n yo pé tyɩ́ nɛ: «Yesu mɛ ǹ vilɛ! Ńmɔ ye à wɛ!», pè náh sɛ̃ ǹ yõ. 
Képah náh, Yesu *tõ̀ sã́hpú nɛ́pĩ̂ nímí sõ̀ mɛ kwílpír yɔ́ wɛ̃̀kɩ̀ tɛ̃. Pǎh mɛ kénɛ wɛ̃̀kɩ̀ yõ, Yesu ǹ gblɔ̌ynɛ pè wɛ̃kɩ dùkù yɔ́lɛ. 
Pè pé syɩ kyɩ ké yah yo pé no tɛ́lɔ́ tyɩ́, tɛ́ pépi tɔ náh tɔ̃ sɛ̃ pé yõ. 
Sépi pól sɔ́kɔ́ntɛnɔ náh, Yesu ǹ gblɔ̌ynɛ ǹ *tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃núkú wɛ̃kɩ pé yõke yõ gbĩ́nɛ. À pé yahlɛ n yo pé níkí sékélkè, ànɛ̂ pé tir yõ sɛ̃́-ńkɛ̃̂ yĩnnɛ. Pǎh sɔ̃́npɩ yahle pé vi wɛ́pú yõ sɛ̃́nɔ́lɛ pé kṹntɛnɔ náh. 
Képah sɔkɔ, à yo pé tyɩ́: «Yé kyɩ nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yah n yo n kɔ̃ kèkõyṍ névye pól tyɩ́. 
Névi nɛ̂-à sɛ̃ ké yõ tɛ́ *wèlnwìlkìnɔ́ wɛ, ǹmɔ kɩ pwah. Tɛ́ nɛ̂-à sɛ̃ ńkɛ̃̂nɛ ké yõ, ǹmɔ kɩ wũ ké sɔkɔ. 
Tɛ́ nónó-á sɛ̃ ké yõ, yé kɩ pè pɩ̃ pé tyípɩ́nɔ́ sɔkɔ: Pè kɩ *sétã̀nnɛ n yĩni n wilki névye sɔkɔ ńmɔ yĩn yõ, tɛ́ kɩ wɛlɔ snɔ́lɛ n wɛkɩ, 
tɛ́ kɩ fɛ̃ wahy tɛ̃ pé gbõ̀lɛ, tɛ́ kɩ fɛ̃ kwâh pèkè wɔ, tɛ́ kè náh tir yɔ́ pɩ n pi pé sɔkɔ. Ànɛ̂ pàh pé gbã̀n sah yámpú yõ, pè kɩ dɛ.» 
Tɛ̃̀-ǹgbɛ̃ Yesu-á yo tɛ sɛ̃́, à yɔ̃ dɔkɔ sɔkɔ yĩ̂nyõ, à kyɩ kɔ̃ tɛ̃ Liyel gbõ̀yõ̀ sõ̀. 
Képah náh, tõ̀ sã́hpú mɛ kyɩ tyah nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yo n kɔ̃ ywɔ̃ ywɔ̃ pól sɔkɔ. Tɛ̃̀-ǹgbɛ̃ sõ̀ mɛ pélɛ n yohnɩ mɔ kénɛ tõ̀ sɔkɔ, mɛ gbǐl tyilɛ n pɩ, tyi nónó-á n wɛ̃kɩ nɛ gblɔ-á pé wɛlɔlɛ.
