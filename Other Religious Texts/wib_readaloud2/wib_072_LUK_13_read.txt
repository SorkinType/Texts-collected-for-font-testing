Likyi.
13.
Légbĩ́ mɛ̀-ó sɔkɔ, nósyɔ́ pɔ yo Yesu tyɩ́ nɛ, yõ̀tɛ̃̀ Pilatɩ-á lésõ Kalile tãn yísyɔ́ ko mɔ, tɛ́ pè sah sárká wìlkìnɔ́ sɔkɔ, pé tṍ kwɛ fɛ̃kɩ sárká tṍ sɔkɔ. 
Képah sɔkɔ, à mɛ pè syah nɛ: «Yé se mɛ n sõ nɛ ńkɛ̃́, Kalile tãn nɔ́pi-á tyípêl pɩ kal pé kwil tãn póllɛ, tɛ́ yĩ mɛ̀ pè nɔ? 
Ɔ̃́ɔ̃ dɛ́! Mé kélɛ n yo yé tyɩ́ nɛ: yâh Liyel wɛ̃̀kɩ̀ tɛ̃́-ńkɛ̃̂nɛ, yé pól tyɩ́ kɩ n yɩkɩ n pi sɛ̃́. 
Ànɛ̂, Silowe wɔ̀kɔ̀ kèyy-á syɩkɩ tũ nɛ́pĩ̂ gbɔ̃nkwǎrtɔ̃mɩ nónó yõ, à pè ko, yé se mɛ n sõ nɛ ńkɛ̃́, pépi pèkè pɩ́nɔ́-á kal Yerusalɛmɩ kwil tãn pól tyɩ́lɛ? 
Ɔ̃́ɔ̃ dɛ́! Mé kélɛ n yo yé tyɩ́ nɛ: yâh Liyel wɛ̃̀kɩ̀ tɛ̃́-ńkɛ̃̂nɛ, yé pól tyɩ́ kɩ n yɩkɩ n pi sɛ̃́.» 
Képah náh, Yesu tɔ̃ sèmìrkì yɔ́ yãh nɛ: «Dĩ́ yɔ́ tyɩ́ sènsóké séswɔ̂ sõ̀ mɛ ǹ dìvɛ̃ sésân swãh sɔkɔ. Wǎh pɔ ké pye yáhnkɔ̃-ńsah, à náh kwâh yɔ́ wɛ ké sɔkɔ. 
À mɛ yo ǹ swãh tɔ̃́-ò dĩ́ tyɩ́ nɛ, à yah, ké ye tɔ̃́-ò-á yɔ̀, pé mɛ pyelɛ n yah n kɔ̃ n pɔ sènsóké mɛ̀ sɔkɔ, tɛ́ pé náh mɛ kwâh yɔ́ ńtɛ̃nɛ n wɛ ké sɔkɔ. Ńkɛ̃́, à kè kõ fɛ̃! Yõ se tɔ̃ mɛ ké tyɩ́ kè tɛ̃ sétáhlɛ n wal nɩ? 
Wǎh yo sɛ̃́, dĩ́ mɛ̀ à syah nɛ, pé yõ̀tɛ̃̀ dĩ́, à yɛ ké yõ ńtɔ̃, kè kwɛtyɩ́ pɩ. Páh kɩ swɔ gbɔ kè kore vi, tɛ́ kɩ tũ̀ mɔ ké sõ̀. 
Kàh pɩ sɛ̃́, gbɔ̀kénpɩ gbĩ́ yɔ́lɛ, kǎh kɩ syi. Képah-à pɩ́-ńkɛ̃̂nɛ, à kè fɛ̃.» 
Mɩ mɩ yòh sṍ yɔ́lɛ, Yesu sõ̀ mɛ névyelɛ n kwɔ Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ. 
Tɛ́, pi yɔ́ sõ̀ mɛ nɛ́npɔ́ sétõlɛ ǹ sɔkɔ, mɛ à pɩ nɛ́dôlɛ fɔ́ɔ́, ye gbɔ̃nkwǎrtɔ̃nɔ. À sõ̀ mɛ kɔ̀ntɛ̃̀nɔ́lɛ, tɛ́ à náh fɛ̃ ǹ tɔrɩ n tɛ n pi. 
Yesu-á pi mɛ̀ wɛ sɛ̃́, à à ye tɛ́ yo ǹ tyɩ́: «Pi, á ye dɛ́nmɔnɔ wɛ á dónɔ́ yam sɔkɔ.» 
Tɛ́ mɛ ǹ gbã̀n sah ǹ yõ. Nɛ́npɔ́ swɔ, à núkú ǹ tɔrɩ tómm tɛ́ tyah Liyellɛ n gbilki. 
Képah sɔkɔ, pi mɛ̀-á dɛ́nmɔnɔ wɛ sɛ̃́ mɩ mɩ yòh sṍlɛ, kè Liyel ní-ńsah wɔ̀kɔ̀ yõ̀tɛ̃̀ fɔkɔ vyãh. À mɛ yo tɔ́wû tyɩ́ nɛ, sèpĩ̀n kénúkò-á tõ̀ pɩ pɩ sèpĩ̀nnɛ. Pǎh sõ̀ yɩ̃nɛ pè pɔ à kɔ̃ à pè dɛ mɔ kénɛ sèpĩ̀n ó sɔkɔ. Ńkɛ̃́, pè náh yɩ̃nɛ pè pɔ mɩ mɩ yòh sṍ tɔlɛ. 
Wǎh yo sɛ̃́, Tɛ̃̀ ǹgbɛ̃ yo ǹ tyɩ́: «Yĩ́nvǐ névye nɔ́pi! Mɩ mɩ yòh sṍlɛ, yé nɛ́núkù núkú pól náh ńkɛ̃́nɛ yé nyeh, képah náh pɩ, yé ńkwɛ̃́ndyɩ́nɩ̀ dɛ sé yõke krótónè tnɔ̂, ǹ kyɩ sè kɔ̃ sě ni wɔ di? 
Tɛ́ pi mɛ̀nɛ, nɛ̂-á Abrahmɩ náh nɛynɛ, sétõ pukubi sah pópó, ye gbɔ̃nkwǎrtɔ̃nɔ, ǹmɔ náh sõ̀ yɩ̃nɛ dɛ́nmɔnɔlɛ kénɛ púkúbínɔ́ sɔkɔ mɩ mɩ yòh sṍlɛ kɛ̀!» 
Képah tɛ̃̀nɛ, Yesu wɛy mɛ̀ sennɛ ǹ fwópú pól tɛ̃, tɛ́ tɔ́wû pól sõ̀ mɛ nɩ̀vɩ̀ sɔkɔ Yesu tyísnɔ́ nɔ́pi pól pɩ́nɔ́ yĩnnɛ. 
Képah náh, Yesu tɔ̃ nɛ: «Liyel tɔ̃́rɩ́lɛ, kè mɛ kwâh nɛ̂ tɔ? Mé kɩ kè tɔkɔ sãh kwâh nɛ̂nɛ? 
Kè mɛ tyɩ̃́ɩ̃, á kɩ nɛ séswɔ̂ yɔ́ pi sɔ̃́, nɛ̂-á pè n ye mútárdɩ̀lɛ: Dĩ́ yɔ́ tɔkɔ, à duku ǹ swãh. Kè fi pɩ séswɔ̂lɛ, tɛ́ kèpye pɔ sé syɛ̀ pɩ ké yɩ̃̀ngbã̀n tyɩ́.» 
Tɛ́, tɔ̃ nɛ: «Mé kɩ Liyel tɔ̃́rɩ́ tɔkɔ sãh kwâh nɛ̂nɛ? 
Kè mɛ á kɩ nɛ kɔ́kɔ́nɔ́ sɔ̃́, sê yɔ́ tɔkɔ mɔ fãh dú yɩ̃́ tɔ̃́nɔ́ mi sɔkɔ ké pól kɔ̃ kè dɔkɔ.» 
Képah náh, Yesu mɛ yɔ̃ sɔkɔ n yuku Yerusalɛmɩ, tɛ́ névyelɛ n kwɔ n yuku, wǎh n kah kwlo ǹgbnɔ ànɛ̂ kwlo pî nónó sɔkɔ. 
Képah sɔkɔ, ǹnɩ̂ yɔ́ mɛ à piki yah nɛ, pé Yõ̀tɛ̃̀, névye kwéy ó se kɩ de Liyel tɔ̃́rɩ́ sɔkɔ nɩ? À mɛ pè syah nɛ: 
«Mé kélɛ n yo yé tyɩ́ nɛ: névye náhnáh kɩ syɔ̃̀-ǹsah yah kɔ̃ nɛ, pé wɛ n de Liyel tɔ̃́rɩ́ tyah sɔkɔ nɛ̂ dé-ńsah-á mɛ fɩ̃́ɩ̃, tɛ́ pè náh fɛ̃ n tɛ̃. Képah ye, yé gbah n tɛ̃ n wɛ n de ké sɔkɔ! 
Gbóntɛ̃̀ dĩ́-à pópó pɔ yuku wɔ̀kɔ̀ yah tɛ̃ mɔ, tɛ́ yé mɛ kégbɔ́, yé kɩ tyah wɔ̀kɔ̀ yahlɛ n mɩ, tɛ́ kɩ mɛ yo tũ nɛ, yé Yõ̀tɛ̃̀, à yɩkɩ mɔ yé kɔ̃. Ǹtɛ, à kɩ yo yé tyɩ́ nɛ, pé náh yélɛ n pnɛ. 
Képah sɔkɔ, yé kɩ tyah n yo ǹ tyɩ́ nɛ ńkɛ̃́, yáh yõke yõ, tɛ́ ni wɔ wɛ̃ tyɩ́ ǹnɛ, ànɛ̂ yépi tɔ́wû tùkè-ǹtã̀n sɔkɔ-á à kwɔ́nɔ́ pɩ. 
Képah sɔkɔ, à kɩ yo yé tyɩ́ nɛ, pé náh yélɛ n pnɛ. Ńkɛ̃́, yé gbɛ mɔ pélɛ, yépi nónó pól-á tyípêllɛ n pɩ. 
Képah tɛ̃̀ sɔkɔ, yâh pópó Abrahmɩ, ànɛ̂ Yisakɩ, ànɛ̂ tɔ̃ Liyel tɛ̃ntõ̀ syínyopu pól wɛ, pè mɛ Liyel tɔ̃́rɩ́ sɔkɔ, tɛ́ yépi mɛ yĩni mɔ dénɔ́lɛ, légbĩ́nɛ, gbní ànɛ̂ yní táhnɔ́ ye kɩ pɩ yé tyɩ́lɛ. 
Tɛ́ névye kɩ wil kèkõyṍ ywɔ̃ ywɔ̃ pól sɔkɔ, kɩ pɔ yõke ǹgbɛ̃ yõ Liyel tɔ̃́rɩ́ sɔkɔ. 
Kǎh pɩ sɛ̃́, nɛ́tɛ́lɔ́ túkù kɩ pɩ nɛ́gbrolɛ, tɛ́ nɛ́gbro túkù kɩ pɩ nɛ́tɛ́lɔ́lɛ.» 
Légbĩ́nɛ, Farisiyɛ̃nɔ yísyɔ́ nɔ de, tɛ́ yo Yesu tyɩ́ nɛ, yõ̀tɛ̃̀ Yerɔdɩ-á mɛ n yah n kɔ̃ ǹ kònmɔ-ńsahlɛ. Képah-á, à yal n to ásõ̂. 
Képah sɔkɔ, à mɛ yo pé tyɩ́: «Yé kyɩ yo kénɛ áswɔ̃́ plɛ́y mɛ̀ tyɩ́ nɛ, máh kɩ sétã̀n yĩni, tɛ́ kɩ dɛ́nmɔnɔ pɩ kwɛynɛ, ànɛ̂ ńsõ̂nɛ, tɛ́ ńsõ̂ náhlɛ, kǎh kɩ tyɛ. 
Ǹtɛ, Liyel tɛ̃ntõ̀ syínyo-ò náh yɩ̃nɛ kúnɔ́lɛ tyah yɔ́, nɛ́ Yerusalɛmɩ sɔkɔ náh. Képah ye, mé yɩ̃nɛ mè n yɔ̃ n sɔkɔ ń wɛ̃̀kɩ̀ kɔ́lɔ́lɛ, kwɛy, ànɛ̂ ńsõ̂, ànɛ̂ tɔ̃ ńsõ̂ náhlɛ. 
Yerusalɛmɩ tãn, Yerusalɛmɩ tãn, yépi nónó-á Liyel tɛ̃ntõ̀ syínyopulɛ n ko n mɔ, tɛ́ Liyel-á nónó tɛkɩ mɔ yé kɔ̃, yé pépilɛ n minki n ko dyêhlɛ. Gbáhyɩ̃́ sɔ̃́ pól mɛ mé kè yah kɔ̃ nɛ, mé yé tuke mɔ, sɔ̃́-á sĩ̌n n tuke n mɔ ǹ pyelɛ ǹ kâm sõ̀ sɔkɔ, tɛ́ yé náh fɛ̃ syi. 
Kǎh pɩ sɛ̃́, Liyel kɩ ǹ náh vi yé kwil kɔ̃. Tɛ́ mé kélɛ n yo yé tyɩ́: yé náh tɔ̃ ńmɔ wɛ n yah n pi fɔ́ɔ́, kɩ pɔ syɩ gbĩ́ nɛ̂-á yé kɩ n yo nɛ ńkɛ̃́: ‹Nɛ̂-á n pi Tɛ̃̀ ǹgbɛ̃ Liyel yĩn yõ, sãm mɛ ǹ sɔkɔ.›»
