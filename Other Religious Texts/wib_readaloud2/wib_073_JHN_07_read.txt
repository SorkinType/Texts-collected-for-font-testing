Nsyɔ̃.
7.
Képah náh, Yesu sõ̀ mɛ yɔ̃ sɔkɔ n kɔ n kɔ̃ Kalile gbɛ̃̀nyah sɔkɔ. Tɛ́ Nsyifunɔ yõ̀tãm-á sõ̀ mɛ n yah n kɔ̃ ǹ kònmɔ-ńsahlɛ Nsyude gbɛ̃̀nyah sɔkɔ sɛ̃́, képah ye à náh tɔ̃ sõ̀ mɛ ké tyɩ́ nɛ, pé kɔ kɔ̃ nɛ́npɔ́. 
Tɛ́ Nsyifunɔ kènkã̀l sõ̀ syáh sõ̀ mɛ de n tɛ. 
Yesu yṹnpyé mɛ yo ǹ tyɩ́ nɛ ńkɛ̃́, à yal ásõ̂, à mó yuku Nsyude gbɛ̃̀nyah sɔkɔ, ǹ sõ̀ tɛ̃́pú tɔ kɔ̃ pè ǹ tyípɩ́nɔ́ wɛ. 
Ńkɛ̃́ névi nɛ̂-à mɛ ké tyɩ́ nɛ no pé dùkù pɩ̃, tɛ̃̀ náh ǹ tyilɛ n pɩ sètĩ̀n sõ̀ sɔkɔ. Tɛ́ ǹmɔ-á tyi nónónɛ n pɩ yɔ̀, wǎh yɩ̃nɛ à sélɛ n pɩ n wilki n mɔ, no kɔ̃ pè à wɛ. 
Ǹtɛ Yesu yṹnpyé ńtɛ̃ náh sõ̀ mɛ sɛ̃ ǹ yõ. 
Képah sɔkɔ, Yesu mɛ nɛ ké gbĩ náh tɔ̃̌nɩ̀ sɛ péwɔ tyɩ́. Gbĩ́mɛgbĩ́-á se mɛ yɩ̃́nɛ́nɔ́lɛ pépi tyɩ́ tɛ́. 
Kèkõyṍ névye náh fɛ̃ pépi dyɔ yɔ n pi. Ǹtɛ pǎh mɛ péwɔ wɔ dyɔlɛ n yɔ, páh sɔ̃́npɩ́lɛ kélɛ n yo n wah nɛ pé tyípɩ́nɩ̀ náh mɛ plɛ. 
Pépi dɔkɔ n yuku kénɛ syáh sɔkɔ. Péwɔ tɛ̃̀ wɔ tyɩ́, ké gbĩ náh tɔ̃̌nɩ̀ sɛ. Képah-á, pé náh n kyɩ n pi. 
Wǎh képah yo tɛ, à wũ Kalile sɔkɔ. 
Tɛ́ ǹ yṹnpyé-á sɔkɔ tɛ syáh mɛ̀ sɔkɔ, ǹmɔ tɔ yuku nɛ́npɔ́ nyáh sõ. Tɛ́ à náh sõ̀ yɛ névye kɔ̃ pè à wɛ. À kè pɩ wɔ̃̀bɩ̀nsɔkɔnɔlɛ kannɛ. 
Tɛ́ Nsyifunɔ yõ̀tãm mó mɛ ǹnɛ n yah n kɔ̃ syáh mɛ̀ sɔkɔ, tɛ́ mɛ n piki n yah nɛ, ńyãh se ǹnɛ nɩ? 
Képah sɔkɔ, tɔ́wû mɛ̀ névye tyah Yesu tyɩ́lɛ n yãh tõ̀tõ̀. Pé túkù sõ̀ mɛ n yo nɛ nɛ́gbɩ́-á ǹnɛ, túkù mó mɛ n yo nɛ sɛ̃́ náh, ǹmɔ-á pyě n pilki névyelɛ ńkɛ̃́. 
Ǹtɛ Nsyifunɔ yõ̀tãm tíkí sõ̀ mɛ pè tɛ̃. Kǎh pɩ sɛ̃́, pé sɔkɔ yɔ́ ńtɛ̃ náh tɛ̃ wɛ ǹ yõ wilki mɔ à yo Yesu tyɩ́ sɔkɔ. 
Syáh mɛ̀ sõ̀ mɛ ké nɩyṍ sɔkɔ, tɛ́ Yesu dɔkɔ pɔ Liyel ní-ńsah gbô sɔkɔ, à pɔ kwɔ́nɔ́lɛ n pɩ. 
Tɛ́ wǎh kwɔ́nɔ́ nɛ̂nɛ n pɩ, kè Nsyifunɔ yõ̀tãm gbã̀n du. Pè mɛ tyah n yo wɛ̃ tyɩ́ nɛ, dĩ́ mɛ̀ se wɛ ńyãh pnɛ̀nɔ́ mɛ̀ dùkùlɛ nɩ? Tɛ́ à náh se mó tɔ̃ kǎrn ǹgbɛ̃ pɩ. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ ńkɛ̃́: «Máh kwɔ́nɔ́lɛ n pɩ tyi nónó yõ yɔ̀, ńmɔ náh sõ tɛ̃ sélɛ dɛ́, Liyel nɛ̂-à tɛkɩ mɔ ńnɛ, ǹmɔ ye n wɛ̃kɩ sélɛ. 
Névi nɛ̂-à kè yah tɔkɔ nɛ pé Liyel dyɔ dyɔ tɛ̃̀nɛ n pɩ, tɛ̃̀ wɔ ye kɩ fɛ̃ ké gbɩ pɩ̃, Liyel-à kɩ nɛ̂ wɛ̃kɩ ńnɛ sélɛ, képah náh pɩ, ń yah yĩ́ĩ́ kwɔ́nɔ́-à kɩ nɛ mé n pɩ. 
Névi nɛ̂-à ǹ yah yĩ́ĩ́ tyilɛ n yo, ǹ gblɔ̌y yĩn gbílkí-ńsah ye à n yah n kɔ̃. Ǹtɛ névi nɛ̂-à mɛ ké tyɩ́ nɛ pé tyi pól pɩ, pé tṍ-ò yĩn mó gbilki, nɛ́gbɩ́ ye tɛ̃̀nɛ. Pílkí-ńsah náh mɛ tɛ̃̀ tyɩ́ póllɛ. 
Moyisi náh Liyel tyi kõ̀nsàhnɔ́lɛ yé kɔ̃ di? Ǹtɛ yé sɔkɔ yɔ́ ńtɛ̃ náh se mɛ n kɔ ké wɛ̃̀kɩ̀ yõ nɩ? Sɔ̃́ pɩ tɛ́ yé n yah n kɔ̃ ńmɔ kònmɔ-ńsahlɛ wɛ́?» 
Képah tɛ̃̀nɛ, tɔ́wû à syah nɛ sétõ-á mɛ ǹ sɔkɔ, à kɔ̃ à képahlɛ n yo, nòtɛ nɛ̂ se nɛ páh ǹnɛ n ko n mɔ nɩ? 
Yesu mɛ tɔ̃ yo pé tyɩ́ nɛ: «Tyi núkú yɔ́ ó náh mé pɩ, tɛ́ yé pól gbã̀n du yɔ̀ di? 
Moyisi náh mɔ yélɛ kõ̀kè kõ̀nɔ́ sɔkɔ di? Ǹtɛ ǹmɔ ńtɛ̃ náh tyah ké gbennɛ dò, ǹ náh tãn ye tyah kélɛ. Tɛ́ yé náh mɛ píyĩ́lɛ n kõ mɩ mɩ yòh sõlɛ di? 
Ǹtɛ yépi-à kõ̀kèlɛ n kõ mɩ mɩ yòh sõlɛ, nɛ yé káh pɔ Moyisi tyi kõ̀nsàhnɔ́ dêl pɩ, sɔ̃́ pɩ tɛ́ ńmɔ-á névi yõ̀tàhsyi pól dɛ mɔ mɩ mɩ yòh sõlɛ, yé ń dyɔlɛ n yɔ wɛ́? 
Yé káh tyilɛ n wah à yɩ̃nɛ yɩ̃́ yáhnɔ́lɛ. Yé sélɛ n wah gbɩ wɛ̃̀kɩ̀ yõ.» 
Képah sɔkɔ, Yerusalɛmɩ tãn túkù sõ̀ mɛ n yo nɛ ńkɛ̃́, dĩ́ nɛ̂-á sõ̀ pè n yah n kɔ̃ nɛ pé ko mɔ, ǹmɔ kó yɔ̀ nɩ? 
Ǹtɛ pè yah à n wɛkɩ tɔ́wû nɩyṍ, tɛ́ pè náh mɛ n vi n mɔ ǹ tyɩ́ lɛ́! Yõ̀tãm se kɩ pyě mɛ kè fɛ̃ syi nɛ, ǹmɔ-á Liyel nɛy yáhntɔkɔnɔlɛ nɩ? 
Tɛ́ Liyel nɛy yáhntɔkɔnɔ-à sõ nɛ̂nɛ pɔ, névi náh ǹ wìl-ǹsah pɩ̃ n pi, ǹtɛ pépi-á dĩ́ mɛ̀ wìl-ǹsah pɩ̃ dò. 
Képah ye, Yesu-á mɛ névyelɛ n kwɔ Liyel ní-ńsah gbô sɔkɔ, à yo pé tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: «Yé ńnɛ n pnɛ, tɛ́ ń wìl-ǹsah pɩ̃ yé yo kɛ̀? Ǹtɛ ńmɔ náh pɔ ń gblɔ̌y yõ sɛ̃́ dɛ́! Liyel nɛ̂-á tɛkɩ mɔ ńmɔlɛ, à mɛ gbɩ ó ǹ tɔ̂ɔ́, ǹmɔ nɛ̂-á yé náh ǹnɛ n pnɛ. 
Ǹtɛ ńmɔ wɔ wil ǹ tnɔ̂. Mé ǹnɛ n pnɛ, ànɛ̂ ǹmɔ ye tɛkɩ mɔ ńnɛ.» 
Képah tɛ̃̀ sɔkɔ, pè tyah Yesu tɛ̃́-ńsahlɛ n yah n kɔ̃. Tɛ́ kǎh pɩ ǹ gbĩ náh tɔ̃̌nɩ̀ sõ̀ mɛ sɛ sɛ̃́, ǹnɩ̂ yɔ́ ńtɛ̃ náh tɛ̃ wɛ gbõ̀ tũ ǹ tyɩ́. 
Ǹtɛ tɔ́wû mɛ̀ sɔkɔ névye náhnáh sõ̀ mɛ sɛ̃ ǹ yõ, tɛ́ mɛ n yo nɛ, tɛ́ Liyel nɛy yáhntɔkɔnɔ-à sõ nɛ̂nɛ pɔ, à se kɩ pɔ gbǐl tyi pɩ kɩ kal dĩ́ mɛ̀ tyɩ́lɛ yɔ̀ nɩ? 
Tɛ́ tɔ́wû-á sõ̀ mɛ tyi nónónɛ n yãh tõ̀tõ̀ Yesu tyɩ́ sɔkɔ, sè pɔ wɛ de Farisi wɛ̃̀kɩ̀ kɔ́pú níkí sɔkɔ. Képah sɔkɔ, pépi ànɛ̂ sárká syínwìlkìpu yõ̀tãm mɛ Liyel ní-ńsah gbô syɩ́kɩ́pú tɛkɩ mɔ nɛ, pè kyɩ Yesu tɛ̃. 
Pǎh pɔ, Yesu mɛ nɛ: «Ńmɔ kɩ mɛ yélɛ kɩ wrɔ́ kwéy ó pɩ ńtɔ̃, tɛ́ kɩ sɔkɔ ń tɛ́kɩ́nmɔ-ò tnɔ̂. 
Ńmɔ-á kɩ n mɛ n pi ńyãh sɔkɔ, yépi náh fɛ̃ n kyɩ nɛ́npɔ́. Képah ye, yé kɩ ń yah kɔ̃ tɛ́ yé náh ń wɛ n pi.» 
Wǎh yo sɛ̃́, Nsyifunɔ núkú tyah n yo wɛ̃ tyɩ́ nɛ, ńyãh se ǹmɔ kɩ n yuku n pi, tɛ́ pépi kɩ à yah kwlɔ nɩ? Ńkɛ̃́ pé no Nsyifunɔ nónó-á mɛ karkɩ tɛ̃ tɛ̃ kɔ̃ nɛ́káhbɩ́ wrɔ́, pépi tyɩ́ se pyě à kɩ yuku, kɩ kyɩ nɛ́káhbɩ́lɛ n kwɔ nɩ? 
Ńkɛ̃́ wɛlɔ nónó-á à yo yɔ̀ nɛ ńkɛ̃́: «Ńmɔ-á kɩ n mɛ n pi ńyãh sɔkɔ, yépi náh fɛ̃ n kyɩ nɛ́npɔ́, képah ye, yé kɩ ń yah kɔ̃, tɛ́ yé náh ń wɛ n pi.» Sépi kɔ́r se nɛ̂nɛ nɩ? 
Syáh gbĩ̀yĩ̀kì tɛ́y sõlɛ, nɛ̂-á pɩ ké gbĩ̀yĩ̀kì ǹgbɛ̃nɛ, Yesu yuku yĩn tɔ́wû sɔkɔ, tɛ́ yo yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: «Níkì-à mɛ nɛ̂ tɛ̃, tɛ̃̀ pɔ ńmɔ tyɩ́, à pɔ wɔ. 
Névi nɛ̂-à sɛ̃ ńmɔ yõ, min yrɛy kɩ n fwo n wil tɛ̃̀ sɔkɔ, sɔ̃́-á Liyel wɛy sɛ́bɛ́y yo kélɛ.» 
Liyel Mirki nɛ̂-á sõ̀ Yesu yõ sɛ̃́pú kɩ pɔ n wɛ n pi, képah tyɩ́ ye sõ̀ Yesu n yo. Tɛ́ légbĩ́nɛ, Yesu náh tɔ̃̌nɩ̀ sõ̀ mɛ dɔkɔ ǹ tɔ̃́rɩ́ yõ, képah ye Liyel náh tɔ̃̌nɩ̀ sõ̀ mɛ ǹ Mirkilɛ pè kɔ̃. 
Tɔ́wû-á Yesu wɛy mɛ̀ noh tɛ, túkù tyah n yo nɛ ńkɛ̃́, Liyel tɛ̃ntõ̀ syínyo-ò nɛ̂-á pé syɩkɩ tɛ̃, ǹmɔ-á dĩ́ mɛ̀nɛ tɔ́tɔ́. 
Túkù mó nɛ, Liyel-á nɛ̂ yah tɔkɔ, ǹmɔ-á. Tɛ́ túkù tɛ̃̀ wɔ mó nɛ, Liyel nɛy yáhntɔkɔnɔ se kɩ fɛ̃ wil Kalile gbɛ̃̀nyah sɔkɔ nɩ? 
Kè kó mɛ yónɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́, wǎh kɩ pɩ yõ̀tɛ̃̀ dĩ́ Davidi pétóbé sɔkɔ yɔ́lɛ, tɛ́ kɩ n dubi n pi Betlɛmɩ kwil, képah nɛ̂ sɔkɔ-á Davidi dubi di? 
Képah sɔkɔ, névye sõnɔ pɩ sé tyɩ́ tyɩ́lɛ Yesu tyɩ́ sɔkɔ. 
Túkù sõ̀ mɛ n yah n kɔ̃ ǹ tɛ̃́-ńsahlɛ, ǹtɛ ǹnɩ̂ yɔ́ ńtɛ̃ náh tɛ̃ wɛ gbõ̀ tũ ǹ tyɩ́. 
Képah sɔkɔ, Liyel ní-ńsah gbô syɩ́kɩ́pú pé syɩ sɔkɔ sárká syínwìlkìpu yõ̀tãm ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́pú tnɔ̂. Pèpi mɛ pè piki yah nɛ, sɔ̃́ se pɩ tɛ́ pè náh Yesu tɛ̃ pɔ nɩ? 
Pè mɛ pè syah nɛ, dĩ́ mɛ̀ wɛy kénkɔ̃ náh mɛ póllɛ. 
Képah tɛ̃̀nɛ, Farisi wɛ̃̀kɩ̀ kɔ́pú nɛ, è, pè se pyě yɛ à kɔ̃ à pé tɔ yõ yɩkɩ nɩ? 
Pè se yõ̀tɛ̃̀ yɔ́, képah náh pɩ, Farisi wɛ̃̀kɩ̀ kɔ́-ò yɔ́ wɛ yah à sɛ̃ Yesu yõ nɩ? 
Tɛ́ kàh tɔ́wû mɛ̀ névye tɛ̃ wɔ lésyi, pè náh Moyisi tyi kõ̀nsàhnɔ́lɛ n pnɛ dò. Pé pól tyɩ́-á yɩkɩ. 
Tɛ́ Farisi wɛ̃̀kɩ̀ kɔ́pú nɔ́pi sɔkɔ yɔ́ sõ̀ mɛ pè n ye Nikodɛmɩnɛ, nɛ̂-á kyɩ wil yah Yesu tyɩ́ sõ yɔ́lɛ. Ǹmɔ mɛ yo pé no tyɩ́ nɛ: 
«À yɩ̃nɛ ápi tyi kõ̀nsàhnɔ́lɛ, á se yɩ̃nɛ à lékàhlɛ névi kɔ̃, nɛ́ á náh ǹ vyãh wɛy noh, tɛ́ pɩ̃ tyi nónó-á à pɩ?» 
Képah sɔkɔ, pé no à syah nɛ, è, Kalile gbɛ̃̀nyah nɛy se ǹ tɔlɛ tɛ́ à n wɛkɩ sɛ̃́ nɩ? Ńkɛ̃́, à Liyel wɛy sɛ́bɛ́y tɔkɔ n sohlɩ n yah, wǎh kɩ kè wɛ nɛ, Liyel tɛ̃ntõ̀ syínyo-ò yɔ́ ńtɛ̃ náh n wil Kalile gbɛ̃̀nyah sɔkɔ. 
Képah náh, pé pól karkɩ sɔkɔ pé fnɛ́ sɔkɔ.
