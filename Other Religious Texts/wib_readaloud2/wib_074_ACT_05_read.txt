Tyípɩ́nɩ̀.
5.
Ǹtɛ dĩ́ yɔ́ lésõ mɛ pè n ye Ananiyasilɛ. Ǹmɔ ànɛ̂ ǹ kyɩ Safira mɛ pé sétáh yɔ́ yãm, tɛ́ yo sah núkú yõ, 
à ké pɔ́ túkù kãm sah. Képah náh, Ananiyasi mɛ kyɩ ké pɔ́ tɛ́ynɛ Yesu tɛ̃ntõ̀ névye kɔ̃. 
Képah sɔkɔ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, Ananiyasi, sétã̀n yõ̀tɛ̃̀ se pɩ sɔ̃́ tɛ́ wɛ de yĩ́ĩ́ ǹ nɩ sɔkɔ sɛ̃́, fɔ́ɔ́ à pɔ mɩ̀nɩ̀ yo Liyel Mirki tyɩ́, tɛ́ sétáh yã̀mnɔ́ pɔ́ túkù kãm sah nɩ? 
Sétáh mɛ̀ yã̀m-ǹkɛ̃̂ gbĩ́nɛ, ǹ yɛ̃́ngbɛ̃́ tyɩ́ kó sõ̀ kélɛ nɩ? Wǎh tɔ̃ pɔ kè yãm, ké pɔ́ kó pɩ ǹmɔ tyɩ́lɛ nɩ? Ǹtɛ yĩ mɛ̀ dùkù pɩ́nɔ́ sõnɔ se pɩ sɔ̃́, tɛ́ de ǹ nɩ sɔkɔ nɩ? Ńkɛ̃́, névye ó tyɩ́ náh à yo mɩ̀nɩ̀nɛ dɛ́, Liyel tɔ tyɩ́-á. 
Ananiyasi-á fyɔ̀ kénɛ wɛlɔ noh, à núkú syɩkɩ ku. Kǎh pɩ sɛ̃́, nónó pól-á yĩ mɛ̀ noh, tíkí pè kah tɛ̃. 
Wǎh ku sɛ̃́, kètɛ̃̀npì mɛ pɔ ǹ lékó fɔkɔ, tɛ́ à tɔkɔ kyɩ yɩ̃. 
Képah náh, lɛ́rɩ́nɔ́ tɔ̃́nɔ́ kénkɔ̃ sɔ́kɔ́ntɛnɔ náh, ǹ kyɩ pɔ de nɛ́npɔ́, tɛ́ tir nɛ̂-á pɩ sɔkɔ, à náh sõ̀ mɛ kè pɩ̃. 
Pyɛrɩ mɛ à syah nɛ, à yo pé syah, dan mɛ̀ se ńtyɩ̃́nɔ pè yãm pé sétáh mɛ̀nɛ nɩ? À mɛ nɛ, àwe, kénɛ dan-á. 
Képah sɔkɔ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, pè se pɩ sɔ̃́, tɛ́ yo sah núkú yõ ǹ pɛlɛ, à pɔ Tɛ̃̀ ǹgbɛ̃ Mirki vyãh sõ̀lɛ n noh nɩ? À yah, nónó-á kyɩ yɩ̃ ǹ pɛ lékólɛ, pǎh mɛ pɔ n de yɔ̀. Tɛ́ pǎh kɩ ǹ tɔ lékó tɔkɔ n sɔkɔ n pi. 
Pyɛrɩ-á yo sɛ̃́, nɛ́npɔ́ swɔ, pi núkú syɩkɩ ǹ yah tĩ̀nnɛ, à ku. Kètɛ̃̀npì-á de kyɩ, pè kyɩ à wɛ kúnɔ́lɛ. Pè mɛ à tɔkɔ kyɩ yɩ̃ ǹ pɛ tnɔ̂. 
Képah sɔkɔ, tíkí Yesu yõ sɛ̃́pú tĩ̀nkì pól kah tɛ̃, à tahlɩ tyi nɔ́pi nóhpú pól tyɩ́. 
Liyel sõ̀ mɛ n syɔ̃ Yesu tɛ̃ntõ̀ névye tyɩ́, à gbǐl tyi náhnáhlɛ n pɩ névye nɩyṍ. Pè sõ̀ mɛ pé sõnɔ pɩ núkúlɛ ànɛ̂ Yesu yõ sɛ̃́pú tɛ́lɔ́ póllɛ, tɛ́ wɛ̃nɛ n tuke n mɔ Liyel ní-ńsah gbô párn yɔ́ sõ̀, nɛ̂-á pè n ye Salomo párnnɛ. 
Ǹtɛ no tɛ́lɔ́ sɔkɔ nɛy yɔ́ ńtɛ̃ náh sõ̀ mɛ ǹ nɩlɛ n syãh n kwrɔ n mɔ pé sɔkɔ. Tɛ́ no pól mó tɔ̃ mɛ pélɛ n kah n sõ. 
Képah sɔkɔ, díbí ànɛ̂ syɔ́ náhnáh sõ̀ mɛ n sɛ̃ Tɛ̃̀ ǹgbɛ̃ Yesu yõ, dòhnɩ̀nɔ́ kɔ̃ kè n pɩ n yuku ǹ yõ sɛ̃́pú yõ. 
Kǎh pɩ sɛ̃́, névye tyah n wil pé yámpúlɛ, à pélɛ n sah n pɔ pàl yõ, képah náh pɩ, sɛ̃ sɛ̃ yî yõ wã̀l yey sɔkɔ, nɛ Pyɛrɩ-à pɔ n sɔkɔ, tɛ́ ǹ mírkí wɛ mɔ pé sɔkɔ nɛy nɛ̂ yõ, tɛ̃̀ wɛ dɛ. 
Yerusalɛmɩ gbɔ̀plɔ kwlo névye náhnáh ńtɛ̃ tɔ sõ̀ mɛ pé yámpúlɛ n tɛ̃ n pɔ n pɔ, à tahlɩ névye tyɩ́, sétã̀n-á sõ̀ mɛ nónónɛ n kwlɔ. Tɛ́ pé pól sõ̀ mɛ dɛ́nmɔnɔlɛ n wɛ. 
Képah tɛ̃̀nɛ, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ ànɛ̂ ǹ náh névye póllɛ nónó-á Sadusɛ̃ névyelɛ, pè tyah dyɔ́yɔ́kɩ̀lɛ n pɩ Yesu tɛ̃ntõ̀ névyelɛ, 
tɛ́ pè tɛ̃ kyɩ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Ǹtɛ lékã̂h-á pɔ mɔ, Tɛ̃̀ ǹgbɛ̃ yĩ̂nyõ tõ̀npɩ yɔ́ pɔ wɔ̀kɔ̀npèkè mɛ̀ yah yɩkɩ mɔ ànɛ̂ ké sɔkɔ táh vɩ̀ yeynɛ, à Yesu tɛ̃ntõ̀ névye wilki, tɛ́ yo pé tyɩ́ nɛ: 
«Yé kyɩ Liyel ní-ńsah gbô sɔkɔ, yè kyɩ min sã́n tyi nɔ́pi pól yo névye tyɩ́.» 
Wɛy mɛ̀-á yo pè noh sɛ̃́, séngbɔ̃́ púpúlɛ, pè mɛ yuku kyɩ Liyel ní-ńsah gbô sɔkɔ, à kyɩ tyah kwɔ́nɔ́lɛ n pɩ. Légbĩ́nɛ, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ ànɛ̂ ǹ náh névye mɛ kyɩ tukey tùkèpu tĩ̀nkì ye ànɛ̂ Yisrayel gbɛ̃̀nyah yèwàhnɩ̀ póllɛ. Tɛ́ mɛ sràsyíbí tõ nɛ, pè kyɩ Yesu tɛ̃ntõ̀ névye wilki n pɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Ǹtɛ pǎh kyɩ yĩni tɛ̃, pè náh pè wɛ wɔ̀kɔ̀npèkè sɔkɔ. Pè mɛ pé syɩ pɔ ké yah yo pé yõ̀tãm tyɩ́ nɛ ńkɛ̃́, 
páh kyɩ wɔ̀kɔ̀npèkè yah wɛ tɛ̃́nmɔnɔlɛ pallɛ ànɛ̂ ké sɔkɔ táh vɩ̀ yey tɔlɛ, sé syɩ́kɩ́púlɛ tɛ̃̀nɔ́lɛ pé tɛ̃̀-ǹtã̀n sɔkɔ. Ǹtɛ páh se mó tɔ̃ sè yɩkɩ mɔ de, pé náh nɛy wɛ sé sɔkɔ. 
Pǎh yo sɛ̃́, yĩ mɛ̀ nóhnɔ́ Liyel ní-ńsah gbô syɩ́kɩ́pú yõ̀tɛ̃̀ dĩ́ ànɛ̂ sárká syínwìlkìpu yõ̀tãm fɔkɔ fil. Pè mɛ wɛ̃nɛ n piki n yah Yesu tɛ̃ntõ̀ névye tyɩ́-á pɩ dùkù nɛ̂nɛ. 
Pǎh mɛ képah sɔkɔ, ǹnɩ̂ yɔ́ pɔ yo pé tyɩ́ nɛ, névye nónó-á pè dah mɔ wɔ̀kɔ̀npèkè sɔkɔ, pǎh mɛ névyelɛ n kwɔ Liyel ní-ńsah gbô sɔkɔ. 
Képah tɛ̃̀nɛ, Liyel ní-ńsah gbô syɩ́kɩ́pú yõ̀tɛ̃̀ dĩ́ ǹ no tɛ̃, à pé nyáh sõ. Pǎh kyɩ, tíkí pè tɛ̃ nɛ páh kɩ nɛ pé pè tɛ̃ gbùgbù, névye káh pɔ pé minki ko dyêhlɛ. Kǎh pɩ sɛ̃́, pè pè ye yárr, à sɔkɔ. 
Pǎh kyɩ yĩni tɛ̃ pélɛ tukey tùkèpu tĩ̀nkì tyɩ́, sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ mɛ pè syah nɛ, 
pé kó lésõ kè tũ pé tyɩ́ nɛ, pè káh tɔ̃ kwɔ́nɔ́ pɩ n yah dĩ́ mɛ̀ yĩn yõ nɩ? Tɛ́ pè yah, pǎh pé kwɔ́nɔ́ tɔ̃ mɔ Yerusalɛmɩ kwil pól sɔkɔ, tɛ́ tɔ̃ mɛ ǹ kũ plɔlɛ n twah pépi yõ kɛ̀! 
Ǹtɛ Pyɛrɩ ànɛ̂ Yesu tɛ̃ntõ̀ névye tɛ́lɔ́ mɛ yo pé tyɩ́ nɛ: «Á kɩ Liyel dyɔ yĩ́ĩ́ tɛ̃̀ yɛ sɔ̃́, tɛ́ kɩ névye tyɩ́lɛ n pɩ? 
Yesu nɛ̂-á yé ko mɔ dahnɩ yõ, á náh tãn Liyel ye ǹ lékó yilki mɔ, 
tɛ́ à yɔ̃ dɔkɔ sɔkɔ, à kyɩ à sah ǹ gbõ̀yõ̀ sõ̀lɛ. Tɛ́ à pɩ yõ̀tɛ̃̀nɛ, ànɛ̂ pwáhnmɔ-òlɛ nɛ ńkɛ̃́, ǹmɔ gbõ̀ yõ Yisrayel tãn fɛ̃ pé nnɔ vi mɔ, pè mó pé tyípêl yɔ̃̀nmɔnɔ wɛ. 
Kénɛ tyi nónó-á Liyel pɩ, ápi ànɛ̂ Liyel Mirki ye sé yónwàhpu kègblɔlɛ, wǎh képah nɛ̂nɛ névye kɔ̃, nónó-á sah ǹ tyɩ́ yõ.» 
Képah tɛ̃̀nɛ, tukey tùkèpu tĩ̀nkì-á wɛlɔ nɔ́pi noh sɛ̃́, pé fɔkɔ kah vyãh, fɔ́ɔ́ pè sõ̀ mɛ ké tyɩ́ nɛ, pé Yesu tɛ̃ntõ̀ névye ko mɔ. 
Ǹtɛ Farisiyɛ̃ nɛy yɔ́ sõ̀ mɛ pé sɔkɔ pè n ye Kamalyɛlɩlɛ. Liyel tyi kõ̀nsàhnɔ́ kwɔ́-ò yɔ́ ye sõ̀ ǹnɛ, nɛ̂-á névye pól mɛ yõ̀yénnɛ n sah ǹ yõ. Ǹmɔ mɛ yuku yĩn tukey tùkèpu tĩ̀nkì nɩyṍ tɛ́ nɛ, pè Yesu tɛ̃ntõ̀ névye wilki mɔ kégbɔ́ kwéy átrɛ. 
Pǎh wil, à ǹ syɩ yo pé no tyɩ́ nɛ: «Á no Yisrayel tãn, yé yé gblɔ̌y kyɔmɩ yĩ́ĩ́ yé tyípɩ́nɔ́ sɔkɔ, à yuku díbí nɔ́pi tĩ̀nnɛ dɛ́! 
Ké náh tɔ̃̌nɩ̀ pɩ di mónɔ́lɛ kah, gbĩ́ nɛ̂-á Tedasɩ mɛ yuku ǹ gblɔ̌y pɩ nɛ, nɛ́-ǹgbɛ̃-á péwɔlɛ, tɛ́ díbí nɛ́pĩ̂ kyɛyẽ́h (400) sõnɔ vi mɔ pè kɔ̃ pè ǹ sõ̀ tɛ̃. Ǹtɛ wǎh pɔ ko mɔ, ǹ sõ̀ tɛ̃́pú pól náh karkɩ di? Nɛy se tɔ̃ wũ? 
Ǹmɔ tyɩ́ sɔ́kɔ́ntɛnɔ náh, Kalile gbɛ̃̀nyah dĩ́ Nsyudasɩ mɛ tɔ̃ yuku kénɛ gbɛ̃̀nyah névye yãh gbĩ́nɛ, à pé náhnáh yõ yɩkɩ pè kɔ̃ pè ǹ sõ̀ tɛ̃. Ǹtɛ ǹ tɔ náh pɔ ko mɔ, tɛ́ ǹ sõ̀ tɛ̃́pú mɛ karkɩ di? 
Képah ye núkúnúkú tɛ̃̀nɛ, mé kélɛ n yo yé tyɩ́, yé yé vyɛ̃y dɛ no nɔ́pi tyɩ́ sɔkɔ, yè mó pè fĩn mɔ pè nɛ n yuku. Pé tir yáhntɔkɔnɔ, képah náh pɩ, pé tyípɩ́nɩ́ mɛ̀-à kɩ mɛ wil névye tɛ̃̀ tyɩ́, kè kɩ tyɛ ké gblɔ̌y. 
Ǹtɛ kàh kɩ mɛ wil Liyel tɛ̃̀ wɔ tyɩ́ tɔ́tɔ́, yépi náh fɛ̃ kè yɩkɩ n pi. Yé kyɔ̀mɩ̀nɔ́ pɩ, yè káh pɔ tyah Liyel wɔlɛ n fwo dɛ́!» Kamalyɛlɩ-á yo sɛ̃́, pé no sah ǹ tyɩ́ yõ. 
Pè mɛ tɔ̃ Yesu tɛ̃ntõ̀ névye ye de, à pɔ pè ko yèfõhlɛ. Tɛ́ kè yahle pé tyɩ́ nɛ, pè káh tɔ̃ Yesu tyɩ́lɛ n yãh, tɛ́ pè fĩn mɔ. 
Ǹtɛ Yesu tɛ̃ntõ̀ névye sõ̀ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ kègbɩlɛ, pǎh n yal tukey tùkèpu tĩ̀nkì wrɔ́ gbĩ́ nɛ̂nɛ, pǎh sɔ̃́npɩ́lɛ kè kyɔmɩ wɛ nɛ, páh yɩ̃nɛ fṍhnɔ́lɛ Yesu yĩnnɛ. 
Képah sɔkɔ, sõmɛsṍ pè n nɛ n mɛ Liyel ní-ńsah gbô ànɛ̂ kétlo sɔkɔ, ǹ névyelɛ n kwɔ, ǹ tɛ́ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo pé tyɩ́ Yesu Krista tyɩ́ yõ.
