Markɩ.
6.
Yesu-á yal ké nɛ́npɔ́, à kyɩ ànɛ̂ ǹ *tõ̀ sã́hpúlɛ, wǎh sõ̀ kyãh kwil nɛ̂ sɔkɔ. 
*Mɩ mɩ yòh sṍ-á nɔ, à kyɩ de *Liyel ní-ńsah wɔ̀kɔ̀, à tyah névyelɛ n kwɔ. Nónó-á sõ̀ mɛ n nohnɩ ǹ tyɩ́, pé náhnáh gbã̀n du. Pè tyah n yo wɛ̃ tyɩ́ nɛ, ńyãh se à wɛ kélɛ, nɛ̂ se à kɔ̃ kénɛ lékã́m ànɛ̂ tɛ̃́nwɛnɔlɛ, à fɛ̃ gbǐl mɛ̀ dùkùlɛ n pɩ nɩ? 
Ńkɛ̃, ǹmɔ kó sésân sɛ́-ò mɛ̀nɛ, Mari pídĩ́nɛ nɩ? Ǹ yṹpyé kó Nsyakɩ, ànɛ̂ Yose, ànɛ̂ Yude, ànɛ̂ Simolɛ nɩ? Ǹmɔ dósyɔ́ kó pé pól wrɔ́ sɔkɔ ásõ̂ nɩ? Képah sõnɔ ye pè kɔ̃, pè náh sɛ̃ ǹ yõ. 
Képah sɔkɔ, à mɛ yo pé tyɩ́: «*Liyel tɛ̃ntõ̀ syínyo-ò mɛ gbílkínɔ́lɛ névye pól tyɩ́, à wil ǹ súfṍ, ànɛ̂ ǹ no, ànɛ̂ ǹ kétetãn tyɩ́ sɔkɔ. Pépi ye ǹnɛ n yilɔ.» 
Képah ye Yesu kɔ̃, à náh fɛ̃ gbǐl yɔ́ ńtɛ̃ pɩ pé tyɩ́, nɛ́ yámpú yísyɔ́ ó yõ náh. À ǹ gbõ̀ sah pé yõ pè kɔ̃ pè dɛ. 
Tɛ́, ǹ kwil névye-á yahle sɛ̃́nɔ́lɛ ǹ yõ sɛ̃́, kè ǹ gbã̀n du. Képah náh, à yuku gbɔ̀plɔ kwlo sɔkɔ, à kyɩ névyelɛ n kwɔ n kɔ̃. 
Képah sɔkɔ, à ǹ *tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye, tɛ́ tyah pélɛ n tõ nímí nímínɛ. Tɛ́ tɛ̃́nwɛnɔlɛ pè kɔ̃ *sétã̀n yõ. 
Tɛ́ mɛ kè gbah yo pé tyɩ́ nɛ, pàh n sɔkɔ pé kɔ́lɔ́ sɔkɔ, pè káh kwâh yɔ́ ńtɛ̃ tɔkɔ, à wil pé kɔ kɔ lépló tyɩ́ sɔkɔ. Pè káh yõke tɔkɔ, pè káh pilki tɔkɔ, pè káh pɔ́ tɔkɔ dah mɔ pé yõ. 
Pè lékyêh wɔ dyɩ, tɛ́ pè káh fwɔ̀mɔlɔnɔ nínɔ́ nínɔ́ mɔ. 
Tɛ́ tɔ̃ yo pé tyɩ́: «Yâh kyɩ kwil yɔ́ sɔkɔ, tɛ́ twah nɛ̂ tyɩ́, yé tɛ̃ nɛ́npɔ́, kè pɔ nɔ yé yálnɔ́ tyɩ́. 
Kwil nɛ̂ névye-à kyɩ yahle nɛ, yé káh twah pé tyɩ́, tɛ́ yahle yé wɛy nóhnɔ́lɛ, yâh n wil kénɛ kwil sɔkɔ, yé yé gbyẽ̂h nɛ̀ngbɔ̃ kwer n kwɛ nɛ́npɔ́. Képah kɩ pɩ gbáhdyâh dínsàhnɔ́lɛ pé tyɩ́.» 
Képah sɔkɔ, ǹ tõ̀ sã́hpú mɛ sɔkɔ, à kyɩ tyah n yo n kɔ̃ no tyɩ́ nɛ, pè pé nnɔ vi mɔ. 
Pè sõ̀ mɛ sétã̀n náhnáhlɛ n yĩni n wilki névye sɔkɔ, tɛ́ mɛ yámpú náhnáhlɛ n dɛ n mɔ, pàh nɩrɩ tuke tɛ tɛ̃̀ tyɩ́. 
Légbĩ́nɛ, yõ̀tɛ̃̀ dĩ́ *Yerɔdi sõ̀ mɛ Yesu tyípɩ́nɔ́lɛ n nohnɩ, tir yɔ́ náh, ǹ yĩn sõ̀ mɛ n gbilki ywɔ̃ ywɔ̃ pól sɔkɔ. Túkù nɛ ńkɛ̃, *Wèlnwìlkì-ò Nsyɔ̃-á ǹnɛ, wǎh ǹ syɩ wil lékyɩ̂ wrɔ́, à pɔ. Képah-á à kɔ̃, à tɛ̃ wɛ gbǐlnɔ́lɛ n pɩ. 
Tɛ́, túkù mó nɛ, *Yeli-á. Túkù tɔ̃ nɛ, *Liyel tɛ̃ntõ̀ syínyo-ò yɔ́-á, ǹgbò Liyel tɛ̃ntõ̀ syínyo-ò sɔ̃́. 
Yerɔdi-á sépi pól yónɔ́ noh sɛ̃́, à nɛ, Nsyɔ̃-á, páh sõ̀ pè kɔ̃ pè ǹ yõ kãm, tɛ́ wǎh ǹ syɩ yuku gbáh! 
Kègbɩ yõ, Yerɔdi sõ̀ pè kɔ̃ nɛ, pè Nsyɔ̃ tɛ̃ pukubi dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. Yerɔdyadi yĩn ye à kɔ̃, à képah pɩ. Yerɔdi sõ̀ pi mɛ̀ tɔkɔ kyɩlɛ. Tɛ́ ǹ gbɛ̃̀ndĩ́ Filipe kyɩ ye lésõ ǹnɛ. 
Ǹtɛ, Nsyɔ̃ ye sõ̀ ké gbɩlɛ n yo Yerɔdi tyɩ́ nɛ, à náh yɩ̃nɛ à ǹ gbɛ̃̀ndĩ́ kyɩ tɔkɔ. 
Kǎh pɩ sɛ̃́, Yerɔdyadi sõ̀ mɛ Nsyɔ̃ dyɔlɛ n yɔ, mɛ ǹ kònmɔ-ńsahlɛ n yah n kɔ̃. Ǹtɛ, à náh mɛ ké syɔ̃̀-ǹsah wɛ. 
Yerɔdi sõ̀ mɛ Nsyɔ̃ yõlɛ n syɩkɩ, képah ye. À sõ̀ mɛ ǹ tíkílɛ n pɩ. À pɩ̃ nɛ, wǎh mɛ tírr pi kɛ̃́nɛ Liyel yah. Wàh sõ̀ ǹ wɛlɔlɛ n nohnɩ, kě ǹ fɔkɔ fɛ̃kɩ. Tɛ́, ǹ wɛlɔ tyɩ́ sõ̀ kah nɔ ǹ tyɩ́. 
Képah sɔkɔ, Yerɔdyadi pɔ syɔ̃̀-ǹsah yɔ́ wɛ, Yerɔdi dúbínɔ́ yòh vìnpɔnɔ syáh syah sṍlɛ. À sõ̀ ǹ tɔ̃́rɩ́ sɔkɔ nɛ́-ǹgbnɔ ye, ànɛ̂ ǹ mɛ́fwóbé yõ̀tãmnɛ, à tahlɩ Kalile yõ̀tãm pól tyɩ́, tɛ́ yõke ǹgbɛ̃ pɩ pé tyɩ́. 
Képah sɔkɔ, Yerɔdyadi písê de kyɩ yɔ̃ yi pé mɛ́-ńsah wɔ̀kɔ̀. Kǎh pɩ sɛ̃́, ǹ tyɩ́ nɔ Yerɔdi ànɛ̂ ǹ náh névye tyɩ́. À mɛ yo pípír tyɩ́ nɛ, wàh kɩ mɛ kwáhdúkú nɛ̂ tyɩ́, à kè yo, páh kɩ kélɛ à kɔ̃. 
Tɛ́ ǹ wo ǹ tyɩ́ nɛ, wàh kɩ kwáhdúkú nɛ̂ tyɩ́ yo, páh kɩ kélɛ à kɔ̃, kàh kɩ sɔ̃́ pɩ pé yõ̀tɔ̃rɩ́ tɔ́nɩ́nɛ. 
Wǎh yo sɛ̃́, pípír mɛ wil kyɩ ǹ yṹ piki yah nɛ, pé se yɩ̃nɛ pé kwâh nɛ̂ tyɩ́ yo nɩ? Ǹ yṹ mó núkú à syah nɛ, à kyɩ yo ǹ tyɩ́ nɛ, à Wèlnwìlkì-ò Nsyɔ̃ yõlɛ pé kɔ̃. 
Képah tɛ̃̀nɛ, pípír núkú ǹ syɩ súú, à pɔ yo yõ̀tɛ̃̀ dĩ́ tyɩ́ nɛ, à pɔ pé kɔ̃ Wèlnwìlkì-ò Nsyɔ̃ yõlɛ yɛ̃́víkì sɔkɔ núkúnúkú yɔ̀. 
Wǎh yo sɛ̃́, kè pɩ yèkwɔrɩlɛ yõ̀tɛ̃̀ dĩ́ Yerɔdi tyɩ́ kègbɩlɛ. Tɛ́, à náh sõ̀ mɛ ké tyɩ́ nɛ, pé pé vyãh vi pé wónɔ́ yĩnnɛ pé náh névye yĩ́ yah. 
Nɛ́npɔ́ swɔ, yõ̀tɛ̃̀ dĩ́ núkú ǹ tyah syɩ́kɩ́pú sɔkɔ nɛ yɔ́ tɛkɩ mɔ nɛ, à kyɩ Nsyɔ̃ yõ kãm pɔ. Kénɛ dĩ́ mó kyɩ Nsyɔ̃ yõ kãm wɔ̀kɔ̀npèkè sɔkɔ, 
tɛ́ pɔ kélɛ pípír kɔ̃ yɛ̃́víkì sɔkɔ. À mó kyɩ kélɛ ǹ yṹ kɔ̃. 
Nsyɔ̃ *tõ̀ sã́hpú-á képah noh sɛ̃́, pè pɔ ǹ lékó tɔkɔ, à kyɩ yɩ̃ léyɛ̃́kɩ̀ sɔkɔ. 
Képah náh, Yesu tõ̀npɩbɩ pé syɩ pɔ ǹ tnɔ̂, à pɔ pé tyípɩ́nɩ̀ ànɛ̂ pé névye kwɔ dùkù pól yo à wɛ̃kɩ. 
Tɛ́ légbĩ́nɛ, pɔ́pú ànɛ̂ sɔ́kɔ́pú sõ̀ mɛ dò. Yõke yõ yõ wrɔ́ ńtɛ̃ náh sõ̀ mɛ Yesu ànɛ̂ ǹ *tõ̀ sã́hpú tyɩ́. Kǎh pɩ sɛ̃́, à yo pé tyɩ́ nɛ: «Yé pɔ, á yuku gbɔ̀pɔlɛ, yé kyɩ mɩ yah kwéy.» 
Pè mɛ de pé yahlɛ krótókè yɔ́ sɔkɔ, à gbɛ mɔ névyelɛ. 
Tɛ́ névye náhnáh-á pè wɛ sɔ́kɔ́-ńsah, pè n gbɛ n mɔ sɛ̃́, pè mɛ pɩ̃, fɩ̃́ nɛ̂-á pè n yuku. Pè núkú wil ywɔ̃ ywɔ̃ pól sɔkɔ tólɔ́lɛ gbáhkɩ̀ yõ, à Yesupi dyàh tɛ̃ kyɩ de nɛ́npɔ́. 
Yesu-á wil krótókè sɔkɔ, tɛ́ tɔ́wû ǹgbɛ̃ mɛ̀ wɛ sɛ̃́, pé yãm à tɛ̃. Tir yɔ́ náh, pè sõ̀ mɛ, á kɩ nɛ yípɛ̃̂ sɔ̃́, fɛ̃̀-ò kɛ̃́nɛ sé tyɩ́. Képah sɔkɔ, à tyah pélɛ ǹ kwɔ tyi náhnáhlɛ. 
Ńvɩ̃́-á pɔ nɔ, ǹ tõ̀ sã́hpú nɔ de ǹ tnɔ̂, tɛ́ yo ǹ tyɩ́ nɛ, ńvɩ̃́-á syi pɔ nɔ, tɛ́ tyah mɛ̀-á gbɛ̀nmɔnɔlɛ. 
Wǎh náh ké wɛ̃̀kɩ̀lɛ névye kɔ̃, pè kyɩ yõke yah dwe gbɔ̀plɔ gbnɔ ànɛ̂ kwlo sɔkɔ. 
À mɛ yo pé tyɩ́ nɛ, pépi yõkelɛ pè kɔ̃, pè yõ. Pè mɛ à piki yah nɛ, à se nɛ, pépi yõke dwe pè kɔ̃ pɔ́ plɔ́ lègbãmnɛ nɩ? 
À mɛ pè piki yah nɛ, búrn plɔ́ sɔ̃́ se mɛ pé tyɩ́ nɩ? Ńkɛ̃, pè kyɩ yah tɛ. Pǎh kyɩ piki yah, pè pɔ yo ǹ tyɩ́ nɛ, búrn plɔ́ kwlɔ́-á mɛ pé tyɩ́, ànɛ̂ yɛ̃́nkyɔ plɔ́ nínɔ́lɛ. 
Képah tɛ̃̀nɛ, Yesu nɛ, pè névye pól kɔ̃ pè kɔ̃ tɛ̃ tĩ̀n tĩ̀nnɛ fílkì yõ. 
Névye mó kɔ̃ tɛ̃ tĩ̀n tĩ̀nnɛ, lèkwɛ̌kwɛ ànɛ̂ tíkíníní pɔ́ gbãm gbãmnɛ. 
Pǎh kɔ̃ tɛ̃ tɛ, Yesu búrn plɔ́ kwlɔ́ ànɛ̂ yɛ̃́nkyɔ plɔ́ nínɔ́ nɔ́pi tɔkɔ, à ǹ yah yɔ̃ yĩ̂nyõ, tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ búrn kwɔkɩ pè kɔ̃ nɛ, pè dibi névye kɔ̃. Tɛ́ yɛ̃́nkyɔ nɔ́pi tɔ dibi névye pól kɔ̃. 
Képah tɛ̃̀nɛ, pé pól yõ tɔ̃, 
tɛ́ pè tɔ̃ pé búrn ànɛ̂ yɛ̃́nkyɔ yṍnkwlɔnɔ tɛ́lɔ́ tuke mɔ gbétɔ́lɔ́ yɩ̃́ gbɔ̃ninɔ́lɛ. 
Tɛ́ nónó-á sõ̀ yõke mɛ̀ yõ, pè sõ̀ mɛ díbí nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãmnɛ. 
Nɛ́npɔ́ swɔ, Yesu ǹ *tõ̀ sã́hpú kyɩ̃ mɔ, pè de krótókè sɔkɔ nɛ, pè yuku pé yahlɛ yɔ́-ǹgbɛ̃ kahnɩnɛ, Besayida kwil sɔkɔ. Péwɔ mó tɔ́wû fĩn mɔ. 
Wǎh pè fĩn mɔ tɛ, à yuku yɔ̃lɩ yɔ́ yõ, à kyɩ Liyellɛ n ni. 
Lékã̂h-á mɔ, krótókè sõ̀ mɛ yɔ nɩyṍ, tɛ́ ǹmɔ ó mɛ kènkwã̀h yõ. 
Dǒ, à ǹ tõ̀ sã́hpú wɛ, yã́hkɩ́ mɛ pélɛ n yohnɩ, pè mɛ n kwlɔ krótókè vyɔ́nmɔnto-ńsah. Tyah dal gbĩ́nɛ, à tyah n kɔ n yuku pé tírrlɛ yɔ yõ̀tah, à n yuku pé kàl-ǹsah. 
Pǎh à wɛ, à n kɔ sɛ̃́ yɔ yõ̀tah, pè wɛ nɛ, lékó mírkí yɔ́-á, tɛ́ tyah n de n syi. 
Pé pól-á à wɛ sɛ̃́, pé sãm yɔkɔ. Ǹtɛ, nɛ́npɔ́ swɔ, à yo mɔ pé tyɩ́ nɛ: «Yé yé nɩ syãh, ńmɔ ye. Yé káh tíkí pɩ.» 
Tɛ́ de krótókè sɔkɔ pé tnɔ̂, yã́hkɩ́ mó yĩn. Kè pɩ gbɛ̃̀ndùnìnɛ náhnáh pé tyɩ́. 
Wǎh búrn pɩ dò gbǐl nɛ̂ sɔkɔ, pè náh sõ̀ ké dùkù pɩ̃, képah ye. Pé lékã́m sõ̀ kah tɛkɩ ké pɩ̃́nɔ́lɛ. 
Pǎh yɔ́-ǹgbɛ̃ kah, pè yĩni tɛ̃ Yenesarɛti sétáh yõ, tɛ́ pé krótókè yĩn mɔ yɔ vyãh tyɩ́. 
Pǎh fyɔ̀ n tiki krótókè sɔkɔ, névye núkú Yesu yah pɩ̃. 
Pàh noh nɛ, wǎh mɛ ńyãh sɔkɔ, pè n tyah pé yámpúlɛ n twah pé sɛ̃ sɛ̃ yî sɔkɔ ywɔ̃ ywɔ̃ póllɛ, nɛ n pɔ n pɔ ǹ tyɩ́. 
Yesu-à de ńyãh dùkù nɛ̂ sɔkɔ, kwil ǹgbnɔ ànɛ̂ kwílpyé ànɛ̂ gbnɔ, névye pé yámpúnɔlɛ n twah n pɔ ǹ tyɩ́ pé tɔ́wû tùkè-ǹtã̀n sɔkɔ. Pè n pɔ n tyah ǹnɛ n fal nɛ, à yɛ pé kɔ̃, pé wɛ tũ ǹ fwɔ̀mɔn vyãh ó tyɩ́. Nónó pól-à fyɔ̀ wɛ tũ ǹ fwɔ̀mɔn vyãh tyɩ́, pè n mó n dɛ.
