Likyi.
16.
Képah náh, Yesu yo ǹ tõ̀ sã́hpú tyɩ́: «Tàh dĩ́ yɔ́ ye lésõ, à ǹnɩ̂ tɔkɔ sah ǹ tahrɩ yõ, tɛ́ pè pɔ yo ǹ tyɩ́ nɛ, tɛ̃̀-á mɛ sélɛ n pɩ n karkɩ. 
Ǹ yõ̀tɛ̃̀ dĩ́ mɛ à ye, tɛ́ yo ǹ tyɩ́ nɛ, pé náh mɛ tyípéplɔlɛ n nohnɩ ǹ tyɩ́ sɔkɔ. Kǎh pɩ sɛ̃́, à náh tɔ̃ fɛ̃ n tɛ̃ n pi pé tahrɩ yõ. Képah-á, à ǹ tõ̀ pɩ dùkù yah yo pé syah. 
Képah sɔkɔ, tahrɩ yõ tɛ̃̀-ò dĩ́ mɛ̀ mɛ yo ǹ gblɔ̌y tyɩ́ nɛ, pé yõ̀tɛ̃̀ dĩ́-á kɩ pé yal n pi ǹ tahrɩ yõ. Sɔ̃́ se pé kɩ pɩ nɩ? Swãh tõ tõ tǎhkɩ̀ náh se mɛ pé sɔkɔ. Tɛ́, nínsyinɔ tɛ̃̀nɛ, ké sen-á mɛ pé sɔkɔ. 
Ǹtɛ, páh pɩ̃, páh nɛ nɛ̂ pɩ n pi, tɛ́ névye kɩ pé tɛ̃́nɔ́ plɛ pé gbên sɔkɔ, pàh pɔ pé fĩn mɔ pé tõ̀. 
Képah tɛ̃̀nɛ, à ǹ yõ̀tɛ̃̀ dĩ́ kmɔ nɛ́tãnnɛ n ye núkú núkúlɛ, tɛ́ nɛ́gben piki yah nɛ, dan nɛ̂ se pé yõ̀tɛ̃̀ dĩ́ kɔ̃lɩlɛ ǹ yõ nɩ? 
À mɛ nɛ, nɩrɩ pní yɩ̃́ lèkwɛ̌-á. Tahrɩ yõ tɛ̃̀-ò dĩ́ mɛ à syah nɛ, ǹ kɔ̃lɩ sɛ́bɛ́y-á yɔ̀, à kɔ̃ n tɛ̃ súú, à wãrkɩ mɔ ké sɔkɔ nɛ ńkɛ̃́, nɩrɩ pní tikiníní pɔ́ gbãm-á. 
Képah náh, à yo kèyǎh tyɩ́ nɛ, tɛ́ ǹmɔ-á dò, dan nɛ̂ se à yɩ̃nɛ à kɔ̃ nɩ? Ǹmɔ mɛ à syah nɛ, sèmukɔ gbɔ̀tɔ́nɩ̀ yɩ̃́ lèkwɛ̌-á. À mɛ yo ǹmɔ tyɩ́ nɛ, ǹ sɛ́bɛ́y-á yɔ̀, à wãrkɩ mɔ ké sɔkɔ nɛ ńkɛ̃́, sèmukɔ gbɔ̀tɔ́nɩ̀ yɩ̃́ tíkíyẽ́h-á. 
Képah tɛ̃̀ sɔkɔ, ǹ yõ̀tɛ̃̀ dĩ́ à sõ ké yah wɛ́npɩnɔ yĩnnɛ. Kègbɩ yõ, kèkõyṍ névye tyi yahlɛ n wɛ n pɩ pé wɛ̃ wrɔ́ lékã́mkãmnɛ, à kal kyòo sɔkɔ névyelɛ.» 
Képah náh, Yesu tɔ̃ yo pé tyɩ́: «Ńmɔ wɔ kélɛ n yo yé tyɩ́ nɛ: pɔ́lɛ, gbɛ̃̀ntɛ̃̀náh kwâh tɛ̃̀ wɔ ye kélɛ. Ǹtɛ, yé kélɛ kódíbí yah kɔ̃. Kàh pópó pɔ n knɛ, yé kɩ tɛ̃ plɛ tyɛ́-ńkɛ̃̂ tyah sɔkɔ. 
Ńkɛ̃́, névi nɛ̂-à pɩ nɛ́gbɩ́lɛ tir tyɩ̃́ɩ̃ sɔkɔ, à kɩ pɩ nɛ́gbɩ́lɛ tir gblò tɔ sɔkɔ. Tɛ́ nɛ̂-à yĩ́nvǐ pɩ tir tyɩ̃́ɩ̃ sɔkɔ, à kɩ yĩ́nvǐ pɩ tir gblò tɔ sɔkɔ. 
Képah ye, pɔ́ nɛ̂-á gbɛ̃̀ntɛ̃̀náh kwâhlɛ, yâh pɩ́-ńkɛ̃̂nɛ nɛ́gblɔ́lɛ képah tyɩ́ sɔkɔ, nɛ̂ kɩ dahbɩ yé gbõ̀ yĩ̂nyõ tahrɩ kègbɩlɛ? 
Tɛ́, yâh pɩ́-ńkɛ̃̂nɛ nɛ́gblɔ́lɛ nɛ́káhbɩ́ gbã̀n yî tyɩ́ sɔkɔ, nɛ̂ kɩ yé yɛ̃́ngbɛ̃́ tyɩ́nɔlɛ yé kɔ̃? 
Tõ̀npɩ yɔ́ ńtɛ̃ náh fɛ̃ tõ̀ kwrɔ n pɩ yõ̀tãm nɛ́pĩ̂ nímí tyɩ́. À kɩ núkú dyɔ yɔ, tɛ́ kɩ nɔ ǹ ní-òlɛ, képah náh pɩ, à kɩ mɔ núkúlɛ, tɛ́ kɩ ǹ ní-ò wɛ mɔ. Sɛ̃́ ó ye, yé náh fɛ̃ tõ̀ kwrɔ n mɔ n pɩ n pi, Liyel ànɛ̂ pɔ́ tyɩ́.» 
Képah sɔkɔ, pɔ́ tyɩ́-á nɔ Farisiyɛ̃nɔ tyɩ́ sɛ̃́, pépi-á wɛlɔ nɔ́pi pól noh, pè tyah kṍlɛ Yesulɛ n ko. 
Képah tɛ̃̀nɛ, à mɛ yo pé tyɩ́: «Yépilɛ, yě yé gblɔ̌ynɛ n wɛ̃kɩ nɛ́gblɔ́lɛ névye yah sɔkɔ. Ǹtɛ, Liyel yé nnɔlɛ n pnɛ. Ǹgbǒ névye-á tir nɛ̂nɛ n gbilki, Liyel dêl ye képahlɛ dɛ́. 
Moyisi tyi kõ̀nsàhnɔ́ ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu sɛ́bɛ́ynɔ́ sõ̀ mɛ n mɔ n kõ n yĩ́ĩ́ fɔ́ɔ́, à pɔ syɩ Wèlnwìlkì-ò Nsyɔ̃ mɛ gbĩ́ tyɩ́. Tɛ́ à tɔkɔ légbĩ́ tyɩ́, nɩ̀vɩ̀nɩ̀ wɛy sã́n mɛ n yo Liyel tɔ̃́rɩ́ tyɩ́ sɔkɔ, tɛ́ táhnsàhnɔ́ ó sɔkɔ ye no pól kɩ n de n pi ké sɔkɔ. 
Ǹtɛ, yĩ̂nyõ ànɛ̂ sétáh kɩ yal sé slɔ sɔkɔ sɛ́rɔɔ kal tir kyɩ̃rɩ́ wìlkìnɔ́lɛ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ. 
Sɔ̃́-á kè yo nɛ, dĩ́ nɛ̂-à ǹ kyɩ wilki, tɛ́ ǹ syɩ kèyǎh tɔkɔ, fɔ̀fɩ́npɩ-á ǹnɛ. Ànɛ̂, sê nɛ̂ pɛ-à à wilki, tɛ́ névi nɛ̂ à tɔkɔ, fɔ̀fɩ́npɩ-á tɛ̃̀nɛ.» 
«Tàh dĩ́ yɔ́ lésõ mɛ, à fwɔ̀tǎhnɩ̀ dɛ́nkɔ́kɔ́nɩ̀nɛ n mɔ, tɛ́ kwã́nkwã́n yélɛ n yõ sõmɛsṍ, mɛ nɩ̀vɩ̀lɛ n pɩ. 
Tɛ́ yãm nɛy yɔ́ tɔ sõ̀ mɛ, pè n ye Lasarɩlɛ. À sõ̀ mɛ sɛ̃́ntɛ̃̀nɔ́lɛ tàh dĩ́ mɛ̀ gbô yah. Ǹ wil pól sõ̀ mɛ nɩ́ynɛ. 
À sõ̀ ńkɛ̃́nɛ nɛ mɛ ké tyɩ́ nɛ, pé yɛ̃́nmîrnnɛ n tõ n yõ, tàh dĩ́-á nónónɛ n yõ n kwɛ. Ǹtɛ, pãhlɔ ye sõ̀ n dɔ ǹ nɩ́ynɛ. 
Sṍ yɔ́lɛ, yãm nɛy mɛ̀ ku, yĩ̂nyõ tɛ̃ntõnè mɛ yuku ǹnɛ Abrahmɩ tnɔ̂. Tàh dĩ́ tɔ mɛ ku, pè kɔ̃ pè yɩ̃. 
Ǹtɛ, wǎh kyɩ lékókwíl sɔkɔ, à kyɩ fṍhnɔ́lɛ n wɛ. Wǎh ǹ yah yɔ̃ yah, à Abrahmɩ wɛ fɩ́ɩ́ sɔkɔ, Lasarɩlɛ ǹ gbɔ̀pɔ. 
Képah sɔkɔ, à ye yah fléyy nɛ, pé sú Abrahmɩ, ńkɛ̃́ páh mɛ n kwlɔ képékèyɔ̂ nyàh mɛ̀ sɔkɔ. Képah-á, à pé yãm yah, à mó Lasarɩ tɛkɩ mɔ, à pɔ ǹ kèngbɔ vyãh dahbɩ wilki ni sɔkɔ, à sah pé nɛ́npɔ́rɩ́ yõ kè kɔ̃ kè vɩ. 
Ǹtɛ, Abrahmɩ mɛ yo ǹ tyɩ́: ‹Ńmɔ pi, á dyɔ to n kõ nɛ, wáh yèvɩnɩ wɛ sɔkɔ, wáh sõ̀ mɛ min sɔkɔ gbĩ́ nɛ̂nɛ, tɛ́ Lasarɩ wɔ mɛ yèvyãhrɩ wɛ. Ǹtɛ núkúnúkú, ǹmɔ mɛ yèvɩnɩ sɔkɔ, tɛ́ áwɔ mɛ yèvyãhrɩ sɔkɔ. 
À wil képah sɔkɔ, pwã̀h ǹgbɛ̃ mɛ yépi ànɛ̂ ápi wrɔ́. Képah yĩnnɛ, ásõ̂ névye nónó-á mɛ ké tyɩ́ nɛ, pé kyɩ yé tyɩ́, pè káh tɛ̃ wɛ, ànɛ̂ nɛ́npɔ́ névye nónó-á mɛ ké tyɩ́ nɛ, pé pɔ ápi tyɩ́, pé tɔ káh fɛ̃ tɛ̃.› 
Képah sɔkɔ, tàh dĩ́ mɛ à syah nɛ, pé sú Abrahmɩ, kàh pɩ sɛ̃́, yṹnpyé nɛ́pĩ̂ kwâl-á mɛ pé tyɩ́, páh ǹnɛ n ni, à Lasarɩ tɛkɩ mɔ pé sú kéte sɔkɔ, à kyɩ kè yo sah pé tyɩ́, pé tɔ káh pɔ de yèvyãhrɩ tɛ̃̀-ǹsah mɛ̀ sɔkɔ. 
Abrahmɩ mó yo ǹ tyɩ́: ‹Moyisi tyi sàhnɔ́, ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu wɛlɔ mɛ pé tyɩ́. Pè sépi tyíyónɔ́lɛ n pɩ!› 
Wǎh yo sɛ̃́, dĩ́ mɛ̀ tɔ̃ à syah nɛ, képah náh n yɩ̃nɛ n pi, pé sú Abrahmɩ. Ǹtɛ, névi-à kɩ wil lékyɩ̂ sɔkɔ, à kyɩ yo pé tyɩ́, pǎh kɩ Liyel wɛ̃̀kɩ̀ tɛ̃. 
Képah tɛ̃̀nɛ, Abrahmɩ mɛ tɔ̃ yo ǹ tyɩ́: ‹Pàh kɩ yahle Moyisi ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu wɛlɔ nóhnɔ́lɛ, névi-à sɔ̃́ wil lékókwíl à pɔ kèkõyṍ, à kyɩ yo pé tyɩ́, pè náh n sɛ̃ n pi ké yõ tɔ̃ ńtɔ̃.›
