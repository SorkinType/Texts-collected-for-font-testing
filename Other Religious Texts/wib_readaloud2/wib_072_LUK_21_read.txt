Likyi.
21.
Sṍ yɔ́lɛ, Yesu sõ̀ mɛ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, tɛ́ wǎh ǹ yah yɔ̃, à tahbɩ wɛ, pè mɛ pé yɩ̃́nkɔ̃ kɔ̃ pɔ́lɛ n dah n mɔ n pɔ pɔ́ dáhnmɔ-ńsah kwâh sɔkɔ. 
À pɛ́kúsè yãm nɛy yɔ́ tɔ wɛ, à pɔ pɔ́pî plɔ́ nínɔ́ dah mɔ. 
Képah sɔkɔ, à nɛ: «No póllɛ, pɔ́ nónó yétõ̂ náh mɛ pè tɛ̃, sépi ye pè tɔkɔ pɔ dah mɔ. Ǹtɛ, pɛ́kúsè yãm nɛy mɛ̀nɛ, ǹmɔ ye ǹ yãm fõh, tɛ́ ǹ yõ yõ pɔ́ pól tɔkɔ pɔ dah mɔ. Képah ye, mé kélɛ n yo n dal n mɔ yé tyɩ́ kègbɩ yõ nɛ: ǹmɔ-á dah mɔ kal no póllɛ.» 
Képah náh, névye túkù sõ̀ mɛ n yo nɛ, Liyel wɔ̀kɔ̀ ǹgbɛ̃-á mɛ plɛ: Ńkɛ̃́, ké mɔ mɔ dyêh, ànɛ̂ ké vyɔ́ nónó-á névye yɩ̃ Liyel kɔ̃, sǎh mɛ kè syi yĩ́ĩ́. Ǹtɛ, Yesu mɛ yo pé tyɩ́: 
«Nɛ̂-á yé n yah yɔ̀ kɛ̀, sèpĩ̀n yísyɔ́ mɛ n pi, dáhkɩ̀ yɔ́ ńtɛ̃ náh n wũ n pi ké ní-ò yõ. Sé pól kɩ pirki.» 
Képah sɔkɔ, pè mɛ Yesu piki yah nɛ ńkɛ̃́, kwɔ́-ò dĩ́, gbĩ́ nɛ̂ se képah kɩ pɩ nɩ? Ànɛ̂, kwní yṍnmɔnɔ nɛ̂ se kɩ wɛ̃kɩ kélɛ nɛ, kǎh kɩ n pɩ n pi nɩ? 
À mɛ yo pé tyɩ́: «Névye náhnáh kɩ pɔ ńmɔ yĩn tɔkɔ n pi pé yah yĩ́ĩ́nɛ, tɛ́ kɩ n yo nɛ ńkɛ̃́, pépi-á Ńmɔlɛ. Ńkɛ̃́, ké gbĩ́-á nɔ tɛ! Ǹtɛ, yé káh pé sõ̀ tɛ̃! Yé yé gblɔ̌ynɛ n kyɔmɩ, yé káh yɛ nɛy pɔ yé pilki. 
Yâh pɔ mâl, ànɛ̂ névye yúkúnyahlenɔ tyɩ́ yónɔ́lɛ n nohnɩ, yé káh yɛ tíkí yé tɛ̃! Képah ǹgbò ye yɩ̃nɛ kè pɩ. Ǹtɛ, képah náh núkú n pɩ n pi kèkõyṍ mɛ̀ tyɛ́nɔ́lɛ.» 
Képah náh, à tɔ̃ ǹ syɩ yo pé tyɩ́: «Kwil yɔ́ kɩ yuku kwil yɔ́lɛ n fwo. Tɔ̃́rɩ́ yɔ́ kɩ yuku tɔ̃́rɩ́ yɔ́lɛ n fwo. 
Sétáh syahrɩ ǹgbnɔ kɩ mɛ n pɩ, tɛ́ tã̀ntã̀n náhnáh sɔkɔ, ńyân pèl ànɛ̂ kwɛkɩ kɩ mɛ de. Tíkítìkì tyi, ànɛ̂ kwní yṍnmɔnɔ ǹgbnɔ kɩ n wɛ yĩ̂nyõ. 
Ǹtɛ, kénɛ tyi-à pópó n pɩ, pè kɩ yé tɛ̃ fõh, tɛ́ kɩ yé gbàh dɛ mɔ pè kɔ̃ pè kɩ tuke Liyel ní-ńtã̂n vɩ̀ sɔkɔ. Pè kɩ yé dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. Pé kɩ yé tɛ̃ kyɩ yõ̀tãm ànɛ̂ kwlo yah tɛ̃́pú tyɩ́ ńmɔ yĩnnɛ. 
Képah gbõ̀ yõ yé kɩ ńmɔ tyɩ́ wɛ yo pé tyɩ́. 
Ǹtɛ, yé yɛ, yé dyɔ nɛ mɛ n kwnɛ nɛ, ńmɔ-á kɩ wɛy wɛkɩ dùkù ànɛ̂ lékã́mnɛ yé kɔ̃, nɛ̂-á kɩ yé fṍhpú kɔ̃, pè náh fɛ̃ n wɛkɩ yélɛ, ànɛ̂ pè náh fɛ̃ yé kɔ̃lɩ. Képah ye, yépi tyɩ́ náh yé gblɔ̌y vìnɔ́lɛ, yé vyɛ̃y syi yĩnnɛ. 
Ǹtɛ, nónó ńtɛ̃-á kɩ yé mɔ kũ kɔ̃ n pi, yé dúbípú ye, ànɛ̂ yé yṹnpyelɛ, ànɛ̂ yé tyɩ́ névyelɛ, à tahlɩ yé kódíbí tyɩ́. Képah sɔkɔ, yé náhnáh kɩ ko mɔ. 
No pól kɩ yé dyɔlɛ n yɔ ńmɔ yĩnnɛ. 
Ǹtɛ, yé yõ̀yén sɔkɔ yɔ́ ńtɛ̃ náh n yah n kwlɔ n pi. 
Gbáhntɛ̃nɔ sɔkɔ ye yé kɩ tyɛ́-ńkɛ̃̂ min wɛ.» 
«Yâh mɛ́gbã̀n wɛ sè Yerusalɛmɩ mɔ gbɩ gbĩ́ nɛ̂nɛ, yé pɩ̃ nɛ, ké sɔ gbĩ́-á nɔ. 
Kénɛ sèpĩ̀n kɩ pɩ pèkè wɔ dɛ́nɔ́lɛ, Liyel sɛ́bɛ́y sɔkɔ tyi wã́rkɩ́nsàhnɔ́ pól kɔ̃ sè kɩ pɩ tũ. Képah ye, nónó-à mɛ Yude kwil sɔkɔ, pépi to de ymɔ sɔkɔ. Nónó-à mɛ Yerusalɛmɩ kwil táh sɔkɔ, pépi wil. Nónó-à mɛ gbnɔ sɔkɔ, pépi káh de kwil táh. 
Kè kɩ pɩ kénɛ sèpĩ̀n sɔkɔ yèvyãhrɩ ǹgbɛ̃nɛ kwil sɔkɔ. Tɛ́ Liyel kɩ yuku kwil mɛ̀ névye yõ fɔ̀kɔ̀vyã̀hnɩ̀nɛ. Pè kɩ ko mɔ kèkîrnnɛ, kɩ tɛ̃ yuku yɔ̃bɩlɛ kèkõyṍ kwlo pól sɔkɔ. Tɛ́ Liyel pɩ̃́-ńkɛ̃̂ névye kɩ Yerusalɛmɩ yɩkɩ pól tɛ̃̀nɛ fɔ́ɔ́, kɩ pɔ syɩ pé gbĩ tyɛ́nɔ́ tyɩ́. Képah ye, kénɛ sèpĩ̀n sɔkɔ, kè kɩ pɩ yĩ́ĩ́ yèvyãhrɩlɛ syɔ́ kûrn nɛ́tãn, ànɛ̂ yɔ̃kɩ nɛ́tãn tyɩ́. 
Képah sɔkɔ, kwní yṍnmɔnɔ kɩ wɛ gbĩ sɔkɔ, ànɛ̂ ńwɔ̃́ sɔkɔ, ànɛ̂ ńwɔ̃́ kèpye sɔkɔ. Tɛ́ kèkõyṍ névye pól yáh-ńsah kɩ mɛ tyɛ képékèyɔ̂. Pnɛ yɔ gbní képékè ànɛ̂ ké fṹndɔkɔnɔ tíkí kɩ mɛ pè kah tɛ̃. 
Yĩ̂nyõ tǎhkɩ̀nɔ́ kɩ yike. Képah ye, tyi nónó-á kɩ pɔ n pɩ n pi kèkõyṍ, sépi tyɩ́ sõnɔ tíkí kɩ névye ko. 
Képah sɔkɔ, pè kɩ Névi Pi wɛ à kɩ mɛ n pi ńyĩ́rkì sɔkɔ ǹ tǎhkɩ̀ póllɛ, ànɛ̂ ǹ gblò sɔkɔ. 
Kénɛ tyi-à pópó tyah n pɩ gbĩ́ nɛ̂nɛ, yé pwáhnmɔnɔ mɛ nɔ n tɛ. Kàh pɩ sɛ̃́, yé yuku yĩn, yé mó sṍré mɔ yé gblɔ̌y sɔkɔ.» 
Képah náh, Yesu tɔ̃ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: «Yé yah sènsóké séswɔ̂, ànɛ̂ sésân tɛ́lɔ́ póllɛ kɛ̀: 
Yâh sè wɛ sè tyah n tyɛ mɩ̀snɔlɛ, yé n pɩ̃ nɛ, ki de gbĩ́-á mɛ yúkɛy. 
Sɛ̃́ ó ye, yâh pópó kénɛ tyi wɛ sè tyah n pɩ gbĩ́ nɛ̂nɛ: Yé pɩ̃ nɛ ńkɛ̃́, Liyel tɔ̃́rɩ́-á mɛ yúkɛy. 
Kègbɩ yõ, mé kélɛ n yo n dal n mɔ yé tyɩ́ nɛ: kwɛy gbĩ névye pól náh n ku n tɛ n pi, tɛ́ kénɛ tyi pól kɩ yĩni tɛ̃. 
Yĩ̂nyõ ànɛ̂ sétáh kɩ pɔ tyɛ, ǹtɛ, ńmɔ wɛlɔ náh n tyɛ n yah n pi póllɛ.» 
Yesu tɔ̃ yo pé tyɩ́: «Yé yé gblɔ̌y kyɔmɩ syáh syáhnɔ́lɛ, ànɛ̂ sètɛ̃̀nɩ̀ pɩ́nɔ́lɛ, ànɛ̂ kèkõyṍ mɛ̀ tyi sõnɔlɛ. Sé káh n pɔ yé lékã́m pɩ tĩ́ntĩ́. Képah-à pɩ, kèkõyṍ tyɛ sṍ kɩ yé kwɔkɩ tɛ̃. 
Sɛ̃́ tɔ ye, ké lésõ kɩ kwɔkɩ tɛ̃ sétáh yõ névye póllɛ, sɔ̃́-á yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔ n tɛ̃ yɛ̃́nkyɔlɛ. 
Ǹtɛ, yé káh yé dyɔ mɔ n fwɔ, yé nírílɛ n pɩ gbĩ́mɛgbĩ́. Képah sɔkɔ, tyi nónó pól-á n pi yɔ̀, yé kɩ tɛ̃́nwɛnɔ wɛ sé yõ, tɛ́ kɩ fɛ̃ pɔ yuku yĩn Névi Pi yah tukey sɔkɔ.» 
Ké légbĩ́nɛ, tyah kyòo yah sɔkɔ, Yesu ńkɛ̃́nɛ n kyɩ kwɔ́nɔ́ pɩ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, ǹ tɛ́ n kyɩ lékã̂h pɩ yɔ̃lɛ yɔ́ yõ nɛ̂-á pè n ye Wólívî yèh yɔ̃lɩlɛ. 
Tɛ́ à tɔkɔ séngbɔ̃́ púsɔ́ tyɩ́, no pól n pɔ nóhnɔ́ pɩ ǹ tyɩ́ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ.
