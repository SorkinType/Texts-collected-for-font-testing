Matye.
25.
Yesu tɔ̃ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Névi Pídĩ́ syɩ n pɔ gbĩ́nɛ, Liyel tɔ̃́rɩ́ kɩ fɛ̃ tɔkɔ sãh syɔ́npì nɛ́pĩ̂ gbãm yísyɔ́ sèmìrkìlɛ, nónó-á pé fɔ̀tnɔ̀nɔ́ tɔkɔ sɔkɔ pɛ yòhnɩ̀-ǹsah. 
Syɔ́npì nɔ́pi sɔkɔ, nɛ́pĩ̂ kwâl sõ̀ mɛ lékã́m kɛ̃́púlɛ, tɛ́ kwlɔ́ mó mɛ lékã́mpulɛ. 
Nónó-á lékã́m kɛ̃́púlɛ, pépi-á sõ̀ pé fɔ̀tnɔ̀nɔ́ tɔkɔ n sɔkɔ, pè náh nɩrɩ wãn tɔkɔ. 
Ǹtɛ lékã́mpu nɔ́pi wɔ mó pé fɔ̀tnɔ̀nɔ́ tɔkɔ, tɛ́ nɩrɩ mɔ sɔkɔ sonɔ́ sɔkɔ. 
Pɛ-á mo pɔ́-ńsah, dɔy mɩ pé sɔkɔ, pè dɔ sɔkɔ. 
Lénɩ wérrlɛ, yĩ́npɔ̃́ yɔ́ n dɔkɔ nɛ ńkɛ̃́, pɛ-á oo, pè wil à yohnɩ. 
Képah tɛ̃̀nɛ, syɔ́npì nɔ́pi pól dɔ yuku, tɛ́ yah dah mɔ pé fɔ̀tnɔ̀nɔ́ tyɩ́ sɔkɔ. 
Nónó-á lékã́m kɛ̃́púlɛ, pépi mɛ lékã́mpu nɔ́pi syah nɛ, pè yɩ̃ pé kɔ̃ nɩrɩlɛ, ńkɛ̃́ pé fɔ̀tnɔ̀nɔ́-á mɛ tɛ̃ syi n ni pé tyɩ́. 
Pè mɛ yo pé tyɩ́ nɛ, sè náh fɛ̃ n yɩ̃nɛ n pi pépi tyɩ́nɔlɛ tɛ́ pé tɔ̃ n yɩ̃ n kɔ̃ n pi. Ńkɛ̃́ pè kyɩ n yah dwe yã̀mpu tyɩ́ pè mɔ pé tyɩ́nɔ sɔkɔ. 
Pǎh sɔkɔ yáhndwe-ńsah, pɛ pɔ de. Syɔ́npì nónó-á sõ̀ mɛ pé vi sah, pépi mɛ de ǹnɛ fúr yõke yṍ-ńsah wɔ̀kɔ̀ sɔkɔ, tɛ́ pè mɛ wùkah vike. 
Syɔ́npì nɔ́pi tɛ́lɔ́-á pópó pé syɩ pɔ, pè pɔ wùkahlɛ n mɩ tɛ́ mɛ yo mɔ tũ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, pé yõ̀tɛ̃̀, à yɩkɩ mɔ pé kɔ̃. 
Ǹtɛ à mɛ yo pé tyɩ́ nɛ, kègbɩ yõ páh kélɛ n yo pé tyɩ́, pé náh pélɛ n pnɛ. 
Képah ye, yépi káh yé dyɔ mɔ n fwɔ n tɛ̃ dɛ́. Yé náh se yɔ́ lésõ ànɛ̂ ké gbĩ́nɛ n pnɛ tɛ́.» 
«Liyel tɔ̃́rɩ́-á mɛ sèmìrkì nɛ̂ kètɔ̀lɛ, kè wɛ yɔ̀: Dĩ́ yɔ́ lésõ n sɔkɔ kɔ́lɔ́ sɔkɔ, tɛ́ ǹ tõ̀npɩbɩ ye pɔ ǹ tahrɩlɛ pè kɔ̃ sah, 
à yɩ̃nɛ pé nɛ́núkù núkú pól-á fɛ̃ nɛ̂ pɩ n pi. À talã pɔ́ plɔ́ kwlɔ́lɛ ǹnɩ̂ kɔ̃, tɛ́ nínɔ́lɛ ǹnɩ̂ kɔ̃, tɛ́ mɛ núkúlɛ ǹnɩ̂ kɔ̃. Képah náh, tɛ́ sɔkɔ ǹ kɔ́lɔ́ sɔkɔ. 
Nɛ̂-á wɛ talã pɔ́ plɔ́ kwlɔ́lɛ, ǹmɔ núkú kyɩ ǹ gbõ̀ vi ǹ tyɩ́lɛ à kwlɔ́ yísyɔ́ wɛ dohnɩ sé yõ. 
Sɛ̃́-ó ye nɛ̂-á wɛ talã pɔ́ plɔ́ nínɔ́lɛ, ǹ tɔ mɛ nínɔ́ wɛ dohnɩ sé yõ. 
Ǹtɛ nɛ̂-á wɛ talã pɔ́ pɔ́ núkúlɛ, ǹmɔ mɛ sɔkɔ kyɩ swɔ tũ à ǹ yõ̀tɛ̃̀ dĩ́ pɔ́ wɔ̃bɩ. 
Ké di-á mo náhnáh, pé yõ̀tɛ̃̀ dĩ́ ǹ syɩ pɔ. Wǎh pɔ, à pè ye nɛ pè pɔ pé tahrɩ tyɩ́ dùkù yo pé noh. 
Képah tɛ̃̀nɛ, nɛ̂-á sõ̀ wɛ talã pɔ́ plɔ́ kwlɔ́lɛ, ǹmɔ mɛ ǹ tyɩ́ tɔkɔ nɔ de ǹ yõ̀tɛ̃̀ dĩ́nɛ tɛ́ kwlɔ́ yísyɔ́ tɔ́lɛ à wɛ̃kɩ, tɛ́ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, talã pɔ́ plɔ́ kwlɔ́-á sõ̀ à pé kɔ̃ sah, ǹtɛ à yah, páh kwlɔ́ yísyɔ́ wɛ dohnɩ sé yõ ǹ tyɩ́. 
Ǹ yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh plɛ, tõ̀npɩ pèpɛy ànɛ̂ nɛ́gbɩ́-á ǹnɛ. Wǎh fyɔ̀ pɩ nɛ́gbɩ́lɛ tir tyɩ̃́ɩ̃ sɔkɔ, páh kɩ tyí-ǹgbnɔ dahbɩ ǹ gbõ̀. À pɔ pé nɩ̀vɩ̀nɩ̀nɛ n pɩ wɛ̃ tyɩ́. / Ǹ yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh plɛ, tõ̀npɩ pèpɛy-á ǹnɛ nɛ̂-á tyilɛ n pɩ sé wã̀l yõ. Wǎh fyɔ̀ tir tyɩ̃́ɩ̃ pɩ ké wɛ̃̀kɩ̀ yõ, páh kɩ tyí-ǹgbnɔ dahbɩ ǹ gbõ̀. À pɔ pé nɩ̀vɩ̀nɩ̀nɛ n pɩ wɛ̃ tyɩ́. 
Nɛ̂-á sõ̀ wɛ talã pɔ́ plɔ́ nínɔ́lɛ, ǹmɔ tɔ mɛ ǹ tyɩ́ tɔkɔ nɔ de ǹ yõ̀tɛ̃̀ dĩ́nɛ tɛ́ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, talã pɔ́ plɔ́ nínɔ́-á sõ̀ à pé kɔ̃ sah, ǹtɛ à yah, páh nínɔ́ yísyɔ́ wɛ dohnɩ sé yõ ǹ tyɩ́. 
Ǹ yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh plɛ, tõ̀npɩ pèpɛy ànɛ̂ nɛ́gbɩ́-á ǹnɛ. Wǎh fyɔ̀ pɩ nɛ́gbɩ́lɛ tir tyɩ̃́ɩ̃ sɔkɔ, páh kɩ tyí-ǹgbnɔ dahbɩ ǹ gbõ̀. À pɔ pé nɩ̀vɩ̀nɩ̀nɛ n pɩ wɛ̃ tyɩ́. 
Képah náh, nɛ̂-á sõ̀ wɛ talã pɔ́ pɔ́ núkúlɛ, ǹmɔ-á nɔ de, à ǹ yõ̀tɛ̃̀ dĩ́ syah nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, páh pɩ̃ nɛ ńkɛ̃́ ǹ tõ̀-á tah. À náh sèmukɔ nɛ̂ duku mɔ, ǎ kè tɔkɔ. À náh nɛ̂ mir mɔ, ǎ kè kõ. 
Képah-á tíkí pé tɛ̃, pé mɛ kyɩ ǹ pɔ́ wɔ̃bɩ sétáh sɔkɔ. Kǎh yɔ̀, à tɔkɔ ǹ pɔ́lɛ. 
Ǹ yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, tõ̀npɩ nɛ́túkù, gbãndal névi-á ǹnɛ. Wǎh sõ̀ pɩ̃ nɛ ńkɛ̃́, pé náh sèmukɔ nɛ̂ duku mɔ, pě kè tɔkɔ, pé náh nɛ̂ mir mɔ, pě kè kõ. 
Kǎh pɩ sɛ̃́, wǎh náhkɩ mó sõ̀ mɛ kyɩ pé pɔ́ dah mɔ sah pɔ́ sàh-ǹsah wɔ̀kɔ̀ sɔkɔ kɛ̀. Páh pé syɩ pɔ sɔ̃́ yɔ̀, páh náhkɩ mó sè wɛ wilki dòhnɩ̀nɔ́lɛ sé yõ kɛ̀. 
Ńkɛ̃́ pè ǹ tyɩ́ mɛ̀ tɛ̃ syi ǹ tyɩ́, pè kélɛ talã pɔ́ plɔ́ gbãm nɛ́tɛ̃̂ kɔ̃. 
Pè kó wɛ nɩ, névi nɛ̂ tyɩ́ kwâh-á mɛ, dòhnɩ̀nɔ́-á kɩ n pɩ ǹmɔ tyɩ́ yõ yrɔ̌yrɔ̌. Tɛ́ névi nɛ̂ tyɩ́ kwâh náh mɛ, kwéy nɛ̂ ńtɛ̃-á mɛ ǹ tyɩ́, kǎh kɩ tɛ̃ syi ǹ tyɩ́. 
Yõre kɛ̃́ tõ̀npɩ mɛ̀ tɛ̃̀ wɔlɛ, ńkɛ̃́ pè à wilki mɔ kégbɔ́ fɛ̃́ntũ̂ sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ.» 
«Névi Pídĩ́-à ǹ syɩ pɔ ǹ tɔ̃́rɩ́ sɔkɔ ànɛ̂ yĩ̂nyõ tõ̀npɩbɩ póllɛ gbĩ́ nɛ̂nɛ, à kɩ kɔ̃ tɛ̃ ǹ tɔ̃́rɩ́ kǒ yõ. 
Kwlo kwlo pól névye kɩ pɔ tuke mɔ ǹ yah sɔkɔ. Tɛ́ à kɩ névye wah wilki wɛ̃ sɔkɔ, sɔ̃́-á yípɛ́nfɛ̃ n wah n wilki képyɛ́nɩ̀ ànɛ̂ perlɛ wɛ̃ sɔkɔ. 
À kɩ nɛ́gblɔ́ sah ǹ gbõ̀yõ̀ sõ̀lɛ, tɛ́ kɩ nɛ́pêl sah ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀lɛ. 
Képah sɔkɔ, nónó-à kɩ mɛ ǹ gbõ̀yõ̀ sõ̀lɛ, à kɩ yo pé tyɩ́ nɛ: ‹Ń Sú Liyel-á sãm mɔ yépi nónó sɔkɔ, tɛ́ tɔ̃́rɩ́ nɛ̂ yah sah yé kɔ̃ kèkõyṍ mɔ gbĩ́ dùkùlɛ, yé pɔ ké nɔlɩlɛ n yõ. 
Yé náh wɛ di, kwɛkɩ-á sõ̀ ń tɛ̃, yé yõkelɛ ń kɔ̃ mé yõ. Níkì-à sõ̀ ń tɛ̃, yé ninɛ ń kɔ̃ mé wɔ. Máh sõ̀ mɛ nɛ́pɔ̂lɛ, yé ń kɔ̃ mé twah yé tyɩ́. 
Máh sõ̀ mɛ flɔ kɛ̃́nɛ, yé flɔlɛ ń kɔ̃ mé mɔ. Máh sõ̀ mɛ n yɔ, yé yah dah mɔ ń tyɩ́ sɔkɔ. Máh sõ̀ mɛ wɔ̀kɔ̀npèkè sɔkɔ, yé pɔ yah ń tyɩ́.› 
Képah sɔkɔ, nɛ́gblɔ́ nɔ́pi kɩ à piki yah nɛ, Tɛ̃̀ ǹgbɛ̃, gbĩ́ nɛ̂ se pé wɛ ǹnɛ kwɛkɩ sɔkɔ, tɛ́ à kɔ̃ à yõ nɩ? Ànɛ̂ gbĩ́ nɛ̂ se pé wɛ ǹnɛ níkì sɔkɔ, tɛ́ à kɔ̃ à wɔ nɩ? 
Gbĩ́ nɛ̂ se pé wɛ ǹnɛ nɛ́pɔ̂lɛ, tɛ́ à kɔ̃ à twah pé tyɩ́ nɩ? Ànɛ̂ gbĩ́ nɛ̂ se pé wɛ ǹnɛ flɔ kɛ̃́nɛ, tɛ́ flɔlɛ à kɔ̃ à mɔ nɩ? 
Gbĩ́ nɛ̂ se pé wɛ ǹnɛ à mɛ n yɔ, képah náh pɩ à mɛ wɔ̀kɔ̀npèkè sɔkɔ, tɛ́ pɔ yah ǹ tyɩ́ nɩ? 
Képah sɔkɔ, yõ̀tɛ̃̀ dĩ́ kɩ yo pé tyɩ́ nɛ: ‹Kègbɩ yõ mé kélɛ n yo yé tyɩ́, ń yṹnpyé nónó-á yɔ̀ kɛ̀, yáh ńkɛ̃́nɛ sɛ̃́nknɔlɛ n pɩ pé sɔkɔ tyɩ̃̂tyɩ̃̂ nónó fyé tyɩ́, ńmɔ tyɩ́ ye ńkɛ̃́nɛ yè n nɛ n pɩ sélɛ.› 
Képah náh, nónó-à kɩ mɛ ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀lɛ à kɩ yo pépi tyɩ́ nɛ: ‹Yé to ń tnɔ̂, yépi nónó tyɩ́-á yɩkɩ. Yé nɛ n yuku ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ, képah nɛ̂-á mɛ mɔ́nsàhnɔ́lɛ sétã̀n ànɛ̂ sé yõ̀tɛ̃̀ yĩnnɛ. 
Yé náh wɛ di, kwɛkɩ-á sõ̀ ń tɛ̃, yé náh ń kɔ̃ mé yõ. Níkì-á sõ̀ ń tɛ̃, yé náh ń kɔ̃ mé wɔ. 
Mé sõ̀ mɛ nɛ́pɔ̂lɛ, tɛ́ yé náh fɛ̃ syi ń kɔ̃ mé twah yé tyɩ́. Mé sõ̀ mɛ flɔ kɛ̃́nɛ, tɛ́ yé náh flɔlɛ ń kɔ̃ mé mɔ. Mé sõ̀ mɛ n yɔ, tɛ́ mɛ wɔ̀kɔ̀npèkè sɔkɔ, tɛ́ yé náh pɔ yah ń tyɩ́.› 
Pépi tɔ kɩ à piki yah nɛ, Tɛ̃̀ ǹgbɛ̃, gbĩ́ nɛ̂ se pé wɛ ǹnɛ kwɛkɩ sɔkɔ, képah náh pɩ níkì sɔkɔ, képah náh pɩ nɛ́pɔ̂lɛ, képah náh pɩ flɔ kɛ̃́nɛ, képah náh pɩ mɛ n yɔ, képah náh pɩ mɛ wɔ̀kɔ̀npèkè sɔkɔ, tɛ́ pé náh à yohnɩ mɔ nɩ? 
Képah sɔkɔ, à kɩ yo pé tyɩ́ nɛ: ‹Kègbɩ yõ mé kélɛ n yo yé tyɩ́, ń yṹnpyé nónó-á yɔ̀ kɛ̀, yáh ńkɛ̃́nɛ n yahle sɛ̃́nknɔ pɩ́nɔ́lɛ pé sɔkɔ tyɩ̃̂tyɩ̃̂ nónó fyé tyɩ́, ńmɔ tyɩ́ ye ńkɛ̃́nɛ yè n yahle pɩ́nɔ́lɛ sélɛ.› 
Tyɛ́-ńkɛ̃̂ yèvyãhrɩ sɔkɔ ye nɛ́túkúnì nɔ́pi kɩ kyɩ n de n pi, tɛ́ nɛ́gblɔ́ nɔ́pi wɔ kɩ kyɩ tyɛ́-ńkɛ̃̂ min wɛ.
