Markɩ.
12.
Képah náh, Yesu tyah n yãh pé tyɩ́ sèmìrn sɔkɔ nɛ ńkɛ̃: «Dĩ́ yɔ́ ye lésõ dìvɛ̃ sésân sõ, tɛ́ kɛ̃kɩ mɔ kè kore vi. Tɛ́ kyɩ tɛ́dáhkɩ̀ yɔ́ yõ swɔ yɔ́ tũ, dìvɛ̃ sésân pye yĩ́kí-ńsahlɛ. Tɛ́ mɛ duke syɩ́kɩ́pú tɛ̃̀-ǹsah mɔ. Képah náh, à swãh tɔ̃́pú tɔkɔ, à pɔ swãh mɛ̀ dahbɩ pé gbõ̀, tɛ́ pyě sɔkɔ kɔ́lɔ́. 
Sé pye gbɛ gbĩ́-á pɔ nɔ, à ǹ tõ̀npɩ yɔ́ tɛkɩ mɔ dìvɛ̃ sésân swãh tɔ̃́pú nɔ́pi tyɩ́ nɛ, pè pé tyɩ́ dìvɛ̃ sésân pyelɛ à kɔ̃, à pɔ pé kɔ̃. 
Képah tɛ̃̀nɛ, dìvɛ̃ sésân swãh tɔ̃́pú nɔ́pi tɛ̃ntõ̀ dĩ́ tɛ̃ ko, tɛ́ à fĩn mɔ, à sɔkɔ ǹ gbõ̀ kɔ́kɔ́lɛ. 
Swãh nɛ́tɛ̃̂ tɔ̃ tõ̀npɩ yɔ́ tõ. Pé tɔ̃ kyɩ ǹmɔ tɔ ko, tɛ́ à mɩkɩ ǹ yõ sɔkɔ, tɛ́ ǹnɛ n gbyɔ. 
Swãh nɛ́tɛ̃̂ tɔ̃ ǹ syɩ ǹnɩ̂ yɔ́ tɛkɩ mɔ n tɔ̃. Ǹmɔlɛ, pè ǹmɔ ko mɔ, tɛ́ sɛ̃́nkɔ̃ pɩ pé náhnáh tyɩ́: Pè pé túkù ko, tɛ́ pé túkù ko mɔ. 
Nɛ núkú nɛ̂-á sõ̀ wũ sètɛ̃̀ dĩ́ tnɔ̂, ǹmɔlɛ, ǹ nɩ̀dɛ̀ Pídĩ́ ye. À ǹmɔ tɛkɩ mɔ pé tyɩ́ tɛ́ynɛ, tɛ́ mɛ n sõ nɛ, pǎh kɩ pé pi wɔ yah sen ku. 
Ǹtɛ, swãh tɔ̃́pú nɔ́pi-á à wɛ, pè tyah n yo wɛ̃ tyɩ́ nɛ, pè yah, ǹ gbɛ̃̀yõ̀pi-á n pi yɔ̀. Pè pɔ, pé à ko mɔ. Gbɛ̃kɩ mó pɩ pépi tyɩ́lɛ. 
Tɛ́ dĩ́ pídĩ́ tɛ̃ ko mɔ, tɛ́ à tɔkɔ kyɩ milki dìvɛ̃ sésân swãh kahnɩnɛ.» 
Képah sɔkɔ, Yesu pè piki yah: «Sɔ̃́ mɛ dìvɛ̃ sésân swãh nɛ́tɛ̃̂ kɩ pɔ n pɩ n pi? À náh pɔ dìvɛ̃ sésân swãh tɔ̃́pú nɔ́pi pól ko mɔ, tɛ́ kè dahbɩ n pi nósyɔ́ gbõ̀ di? 
Yé náh kè kar yah *Liyel wɛy sɛ́bɛ̃́ sɔkɔ di? Kè mɛ wã́rkɩ́nsàhnɔ́lɛ nɛ: ‹Wɔ̀kɔ̀ mɔ́pú-á dáhkɩ̀ nɛ̂ yahle, képah pɔ pɩ dáhgbɩ́lɛ, wɔ̀kɔ̀ dáhgbennɛ. 
Képah n wil Tɛ̃̀-ǹgbɛ̃ tyɩ́. Tɛ́ ápi tyɩ́, gbɛ̃̀ndùnì tir ye kélɛ!›» 
Képah sɔkɔ, *Nsyifunɔ yõ̀tãm tyah Yesu tɛ̃́-ńsahlɛ n yah n kɔ̃. Tir yɔ́ náh, pè sõ̀ pɩ̃ nɛ, pépi yõ-á à mɩ sèmìrkì mɛ̀nɛ. Tɛ́, tɔ́wû tíkí sõ̀ mɛ pè tɛ̃. Pè mɛ yɛ ǹ yõ, tɛ́ sɔkɔ. 
Képah náh, pè *Farisiyɛ̃nɔ ànɛ̂ *Yerɔdi náh névye túkù wilki tɛkɩ mɔ Yesu tyɩ́ nɛ, pè kyɩ à mɔ yah píkínyahnɔ sɔkɔ. 
Pè pɔ yo ǹ tyɩ́ nɛ, pé kwɔ́-ò dĩ́, páh pɩ̃ nɛ, gbɩ ó kɔ́kɔ́-á ǹmɔ n nɛ n yo. À náh tíkílɛ n pɩ névi yah. À náh névilɛ n wah névi sɔkɔ. Wǎh gbɩlɛ névyelɛ n kwɔ Liyel wɛ̃̀kɩ̀ yõ. Képah-á pé kɔ̃, pé nɛ, pé à piki yah: Kè se yɩ̃nɛ pé mɔ́rɩ́lɛ *Wromɛnɔ yõ̀tɛ̃̀ ǹgbɛ̃nɛ n kɔ̃, tĩ̂ kè náh yɩ̃nɛ? Pé se sélɛ n mɔ, tĩ̂ pé káh sè mɔ? 
Ǹtɛ, Yesu sõ̀ pɩ̃ nɛ, nɛ̂-á pè n sõ, képah náh pè n yo. Kǎh pɩ sɛ̃́, à yo pé tyɩ́ nɛ, sɔ̃́ se pɩ, tɛ́ pè pélɛ n mɔ n yah nɩ? Pè pɔ pɔ́ mɛ̀ núkúlɛ, pé yah. 
Pè mɛ pɔ́ núkúlɛ à kɔ̃. À mɛ pè piki yah nɛ, nɛ̂ yah se, ànɛ̂ ǹ yĩnnɛ wã́rkɩ́nmɔnɔlɛ ké yõ nɩ? Pè à syah nɛ, Wromɛnɔ yõ̀tɛ̃̀ ǹgbɛ̃-á. 
Képah tɛ̃̀nɛ, à yo pé tyɩ́ nɛ, pè Wromɛnɔ yõ̀tɛ̃̀ ǹgbɛ̃nɛ n kɔ̃ ǹ tyɩ́lɛ, pè mó Liyellɛ n kɔ̃ ǹ tyɩ́lɛ. Képah sɔkɔ, pé gbã̀n du ǹ wɛy mɛ̀ dùkùlɛ. 
Sadusɛ̃nɔlɛ, nósyɔ́ ye nónó-á nɛ, lékyɩ̂ náh n yuku n wil n pi. Pé túkù pɔ Yesu tnɔ̂, tɛ́ à piki yah nɛ ńkɛ̃, 
kwɔ́-ò dĩ́, *Moyisi-á Liyel tir kõ̀nsàhnɔ́ mɛ̀ wãrkɩ sah pé kɔ̃ nɛ, dĩ́ nɛ̂ tyɩ́ gbɛ̃̀ndĩ́-à mɛ, tɛ́ tɛ̃̀ ku, tɛ́ ǹ kyɩ sah pi kɛ̃́nɛ, ǹ gbɛ̃̀ndĩ́ mɛ̀-á yɩ̃nɛ à pɛ́kúsê mɛ̀ tɔkɔ. À wɛ̃́npì dubi à kɔ̃, nónó-á kɩ pɔ n pɩ n pi ǹ nɛ́nkyɩ́dĩ́ náhnɛ. 
Tɛ́, dĩ́npì yísyɔ́-á sõ̀ mɛ nɛ́pĩ̂ kwǎrnimi pé yṹ tyɩ́. Nɛ́nkyɩ́ tɛ̃̀-á sê tɔkɔ, tɛ́ ku, tɛ́ ǹ kyɩ sah pi kɛ̃́nɛ. 
Ǹ sõ̀ syɩ mɛ pɛ́kúsê mɛ̀ tɔkɔ. Ǹ tɔ pɔ ku, tɛ́ à sah pi kɛ̃́nɛ. Tɔ̃́-ò tɔ tyɩ́ pɔ pɩ sɛ̃́. 
Sɛ̃́ntɛ̃̀ ó-á pɩ pé no tɛ́lɔ́ pól tyɩ́. Tɛ́ nɛ́pĩ̂ kwǎrnimi nɔ́pi pól-á ku pi kɛ̃́nɛ. Pé kúntɛnɔ náh, pi mɛ̀ tɔ pɔ ku. 
Kàh pɩ sɛ̃́, lékyɩ̂ yúkúnwìl sṍlɛ, pàh yuku, pi mɛ̀ se kɩ pɩ nɛ̂ tyɩ́lɛ nɩ? Tir yɔ́ náh, pé nɛ́pĩ̂ kwǎrnimi pól-á à pɩ̃ sêlɛ. 
Képah sɔkɔ, Yesu yo pé tyɩ́: «Liyel wɛy pɩ̃́-ńkɛ̃̂, ànɛ̂ Liyel tɛ̃́nwɛnɔ pɩ̃́-ńkɛ̃̂ yĩn náh yélɛ pílkínɔ́ sɔkɔ di? 
Lékyɩ̂ yúkúnwìl sṍlɛ, sê tɔ́kɔ́kɩ̀ náh mɛ, pɛ dékè náh mɛ. Névye kɩ mɛ, á kɩ nɛ *yĩ̂nyõ tɛ̃̀ntõbe sɔ̃́ pal tɛ̃̀-ǹsah. 
À yuku lékyɩ̂ yúkúnwìlnɔ́ tyɩ́ sɔkɔ: Yé náh kè kar yah Moyisi sɛ́bɛ̃́ sɔkɔ, fɩ̃́ nɛ̂-á dukuni tyɩ́lɛ n yo, nɛ̂-á sõ̀ nyàh lépɔ̃́lɛ ké sɔkɔ di? Kè mɛ wã́rkɩ́nsàhnɔ́lɛ nɛ́npɔ́ nɛ, Liyel-á yo Moyisi tyɩ́: ‹Ńmɔ ye Abrahmɩ Liyellɛ, ànɛ̂ Yisakɩ Liyellɛ, ànɛ̂ Yakɔbɩ Liyellɛ.› 
Vye Liyel ye ǹnɛ, lékyɩ̂ Liyel náh. Sɛ̃́ ye yélɛ kàhnpilkintɛ̃̀nɔ́lɛ yɔ̀.» 
Képah sɔkɔ, *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́-ò dĩ́ yɔ́ kénɛ yékâb noh. À kè wɛ, Yesu tɛ̃ wɛ kè yo plɛ Sadusɛ̃ nɔ́pi tyɩ́. À mɛ nɔ de ǹ tnɔ̂, tɛ́ à piki yah nɛ, Liyel tir kõ̀nsàhnɔ́ nɛ̂ tɛ̃̀ se kal sé póllɛ nɩ? 
À mɛ yo ǹ tyɩ́: «Liyel tir kõ̀nsàhnɔ́ nɛ̂-á kal sé póllɛ, képah wɛ yɔ̀: ‹Á níkí kwɔ, Yisrayɛl: Ápi Tɛ̃̀-ǹgbɛ̃ Liyellɛ, ǹmɔ núkúní ó ye Tɛ̃̀-ǹgbɛ̃ Liyellɛ. 
Ǹ tyɩ́ yɩ̃nɛ kè nɔ á tyɩ́ á nɩ póllɛ, ànɛ̂ á min póllɛ, ànɛ̂ á fɔkɔ sõnɔ póllɛ, ànɛ̂ á tǎhkɩ̀ póllɛ.› 
Ké ní-ò wɛ yɔ̀: ‹Á ní-ò tyɩ́ yɩ̃nɛ kè nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ.› Liyel tir kõ̀nsàhnɔ́ yɔ́ ńtɛ̃ náh tɔ̃ kal sépi nínɔ́lɛ.» 
Wǎh yo sɛ̃́, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́-ò dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh plɛ, pé kwɔ́-ò dĩ́! Tyi nónó-á à yo, gbɩ-á sélɛ. Tɛ̃̀-ǹgbɛ̃nɛ, ǹmɔ núkúní ó-á Liyellɛ. Liyel yɔ́ náh tɔ̃ mɛ ǹmɔ tyɩ́ náh. 
Ǹmɔ tyɩ́-á yɩ̃nɛ kè nɔ á tyɩ́ á nɩ póllɛ, á fɔkɔ sõnɔ póllɛ, á tǎhkɩ̀ póllɛ. Ànɛ̂ á ní-ò tyɩ́ mó nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ. Képah-á kɩ plɛ kal á yípɛ̃̂ ko sukɛ Liyel kɔ̃ *sárkálɛ, ànɛ̂ sárká duke tɛ́lɔ́ póllɛ. 
Képah sɔkɔ, Yesu kè wɛ nɛ, ǹ lékã́m-á nɔ, à ké yah wɛ tɔ̃ sah yo. À yo ǹ tyɩ́ nɛ, à náh mɛ fɩ́ɩ́ *Liyel tɔ̃́rɩ́lɛ. Képah náh, névi náh tɔ̃ fɛ̃ syi à piki yah tir yɔ́lɛ. 
Yesu sõ̀ mɛ névyelɛ n kwɔ *Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, tɛ́ mɛ n yo nɛ, sɔ̃́ se pɩ, tɛ́ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú n nɛ n yo nɛ, *Kristalɛ, nɛ̂-á Liyel nɛ, páh kɩ tɛkɩ mɔ, ǹmɔ-á kɩ pɩ *Davidi náh nɛnɛ nɩ? 
Tir yɔ́ náh, Davidi ńtɛ̃ sõ̀ kè yo *Liyel Mírkí gbõ̀ yõ nɛ, Tɛ̃̀-ǹgbɛ̃ Liyel-á kè yo péwɔ Tɛ̃̀-ǹgbɛ̃ tyɩ́ nɛ: «Kɔ̃ tɛ̃ ńmɔ gbõ̀yõ̀ sõ̀lɛ, mé kɩ á sépɛ́bɩ́ pɩ á gbáhsúkílɛ.» 
Davidi ńtɛ̃-à ǹnɛ n ye ǹ Tɛ̃̀-ǹgbɛ̃nɛ, kè kɩ pɩ sɔ̃́, tɛ́ à kɩ pɔ n pɩ Davidi náh nɛnɛ? Légbĩ́nɛ, tɔ́wû ǹgbɛ̃ sõ̀ mɛ wɛy mɛ̀nɛ n nohnɩ Yesu tyɩ́ nɩ̀vɩ̀nɩ̀ sɔkɔ. 
Wǎh mɛ névye kwɔ́nɔ́ sɔkɔ gbĩ́ nɛ̂nɛ, à yo pé tyɩ́: «Yé yé gblɔ̌ynɛ n kyɔmɩ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́púlɛ. Fwɔ̀mɔn ǹgbnɔ mɔ́nkɔnkɔ̃nɔ tyɩ́ nɔ pé tyɩ́. Pé n nɛ mɛ ké tyɩ́ nɛ, névye nɛ n tɔ̃ n sah n wɛkɩ pé tyɩ́ tɔ́wû tùkè-ǹtã̀n sɔkɔ. 
Pě tɛ̃̀-ǹtã̀n ǹgbnɔlɛ n yah n kɔ̃ *Liyel ní-ńtã̂n vɩ̀ sɔkɔ, ǹ tɛ́ nɛ́-ǹgbnɔ tɛ̃̀-ǹtã̀nnɛ n yah n kɔ̃ yõke ǹgbnɔ yṍ-ńtã̂n sɔkɔ. 
Ǹtɛ nɛ n syɩkɩ pɛ́kúsyɔ yõ, pé gbã̀n yîlɛ n syi yĩ́nvǐ sɔkɔ. Ǹtɛ nírílɛ n pɩ yõ̀sɛ̃̀kɩ̀lɛ nɛ, névye pé wɛ. Sɛ̃́ ye pé tukey kɩ tuke képékè yɔ́.» 
Légbĩ́nɛ, Yesu sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ Liyel wɔ̀kɔ̀ yah, pɔ́ dáhnmɔ-ńsah kwâh gbɔ̀pɔ, tɛ́ névye pɔ́ dáhnmɔ dùkùlɛ n yah. Tahbɩ náhnáh sõ̀ mɛ pɔ́ náhnáhlɛ n dah n mɔ n pɔ. 
Dǒ, pɛ́kúsê yãm nɛ yɔ́ pɔ, pɔ́ plɔ́ nínɔ́ dah mɔ. Sè sõ̀ mɛ núkú núkúnɔ nínɔ́ ólɛ. 
Képah sɔkɔ, Yesu ǹ *tõ̀ sã́hpú ye, tɛ́ yo pé tyɩ́: «Kègbɩ yõ, mé kélɛ n yo yé tyɩ, pɛ́kúsê yãm nɛ mɛ̀-á pɔ́ nɛ̂ dah mɔ kwâh sɔkɔ, kè kal névye pól tyɩ́lɛ. 
Tir yɔ́ náh, no pól pɔ́ yõ dòhnɔ̀nɔ́ ye, pè dah mɔ. Ǹtɛ, ǹmɔ ànɛ̂ ǹ yãm tɔ́lɛ, ǹ sahkɩ pól ye à dah mɔ, wǎh nɛ pɔ nɛ̂ tɔ yõ n pi.»
