Matye.
22.
Yesu tɔ̃ tyah sèmìrkì yɔ́lɛ n yãh pé tyɩ́ nɛ: 
«Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Yõ̀tɛ̃̀ dĩ́ yɔ́ lésõ ǹ pídĩ́ fúr fèn pɩ. 
Tɛ́ mɛ ǹ tõ̀npɩbɩ wilki tõ nɛ, pè kyɩ nɛ́yénɔ́ ye n pɔ fúr yõke yṍ-ńsah. Ǹtɛ nɛ́yénɔ́ nɔ́pi náh sõ̀ mɛ ké tyɩ́ nɛ pé pɔ. 
Képah tɛ̃̀nɛ, à tɔ̃ tõ̀npɩbɩ yísyɔ́ wilki tõ pé tyɩ́ nɛ, pĩ́n-á pɩ tɛ. Páh nyéhpyɛ́nɩ̀ ànɛ̂ yípɛ̂y ko tɛ nɩrɩ-á mɛ nónó sɔkɔ. Tyi pól-á pɩ tɛ, pè pɔ fúr sɔkɔ. 
Ǹtɛ nɛ́yénɔ́ nɔ́pi náh kè mɔ kõ póllɛ. Ńyɔ-à yuku à n sɔkɔ ǹ swãh, ńyɔ-à yuku à n sɔkɔ ǹ gbɔ̀ yɩ̃́-ńsah. 
Pé túkù mɛ yuku mɩ yõ̀tɛ̃̀ dĩ́ tõ̀npɩbɩ sɔkɔ, à pè fõh pélpèl, tɛ́ pè ko mɔ. 
Képah sɔkɔ, yõ̀tɛ̃̀ dĩ́ fɔkɔ kah vyãh. À mɛ ǹ sràsyíbí tɛkɩ mɔ pè kyɩ kénɛ nɛ́kókè mɔ́pú ko mɔ, tɛ́ pé kwil sukɛ. 
Képah náh, tɛ́ yo ǹ tõ̀npɩbɩ tyɩ́ nɛ, fúr yõke-á pɩ tɛ. Ǹtɛ nɛ́yénɔ́ nɔ́pi náh sõ̀ yɩ̃nɛ yõke mɛ̀ yṍnɔ́lɛ. 
Ńkɛ̃́ pè kyɩ wɛ̃̀tlo yey sɔkɔ, pàh kɩ fyɔ̀ névye nónó wɛ, pè pé pól ye n pɔ fúr yõke yṍ-ńsah. 
Tõ̀npɩbɩ nɔ́pi mɛ sɔkɔ wã̀l yey sɔkɔ. Pǎh kyɩ tɛ́ névye nónó wɛ, nɛ́túkúnì tɔkɔ dohnɩ nɛ́péplɔ yõ, pè pé pól ye pɔ tuke mɔ, fɔ́ɔ́ fúr pɩ́-ńsah wɔ̀kɔ̀ pól yuku yõke yṍpúlɛ. 
Yõ̀tɛ̃̀ dĩ́-á de yuku yáh-ńsah pé tyɩ́, à kyɩ dĩ́ yɔ́ wɛ pé sɔkɔ nɛ̂ náh sõ̀ mɛ fúr flɔ mɔ. 
À mɛ yo kénɛ dĩ́ tyɩ́ nɛ, pé kódĩ́, ǹmɔ se de sɔ̃́ ásõ̂ sɔkɔ fúr flɔ kɛ̃́nɛ ǹ wil sɔkɔ nɩ? Ǹtɛ yó-ńsah náh wɛ dĩ́ tyɩ́. 
Képah sɔkɔ, yõ̀tɛ̃̀ dĩ́ yo tõ̀npɩbɩ tyɩ́ nɛ pè ǹ gbyẽ̂h ànɛ̂ ǹ gbã̀n pukubi, pè mó à min mɔ kégbɔ́ fɛ̃́ntũ̂ sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ. 
Yé náh wɛ di, névye náhnáh mɛ yénɔ́lɛ. Ǹtɛ nónó-á yah wilki pé sɔkɔ, pè náh mɛ fwɔ́ɔ.» 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú kyɩ wɛ̃ wɛ à yo sah núkú yõ, sɔ̃́-á pè kɩ pɩ tɛ́ kɩ Yesu wɛ n tɛ̃ n pi ǹ vyãh wɛy sɔkɔ. 
Tɛ́ mɛ pé tõ̀ sã́hpú yísyɔ́ ànɛ̂ yõ̀tɛ̃̀ dĩ́ Yerɔdɩ náh névye túkù wilki tõ Yesu tyɩ́ nɛ pè kyɩ yo ǹ tyɩ́ nɛ, pé kwɔ́-ò dĩ́, páh pɩ̃ nɛ gbɩ yó-ò-á ǹnɛ, ànɛ̂ wǎh Liyel wɛ̃̀kɩ̀lɛ névyelɛ n kwɔ gbɩ wɛ̃̀kɩ̀ yõ. À náh névye yah tíkílɛ n pɩ. À náh névyelɛ n wah. 
Ǹtɛ à ǹ sõnɔ yo pé syah. Ńkɛ̃́ ké wɛ̃̀kɩ̀ se mɛ kɔ̃́nɔ́lɛ nɛ pé mɔ́rɩ́lɛ n mɔ Wrome kwíltãm yõ̀tɛ̃̀ ǹgbɛ̃ Sesar tyɩ́, tĩ̂ ké wɛ̃̀kɩ̀ náh mɛ kɔ̃́nɔ́lɛ? 
Ǹtɛ Yesu-á pé fɔkɔ sõnɔ túkúnì pɩ̃ sɛ̃́, à mɛ yo pé tyɩ́ nɛ, fɔ̀kɔ̀pǐl ànɛ̂ vyã́hsesyɩ̃́rɩ́ névye nɔ́pi, sɔ̃́ se pɩ tɛ́ pè pé pílkíntɛ̃-ńsahlɛ n yah n kɔ̃ nɩ? 
Pè pɔ mɔ́rɩ́ mɔ mɔ pɔ́ pɔ́ núkúlɛ pé yah ǎtrɛ. Pè mɛ pɔ ké núkúlɛ à kɔ̃. 
Yesu mɛ pè piki yah nɛ, nɛ̂ yah mírkí ànɛ̂ ǹ yĩn se wã́rkɩ́nmɔnɔlɛ ké yõ yɔ̀ nɩ? 
Pè mɛ yo ǹ tyɩ́ nɛ, Sesar tyɩ́-á. Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ, pè Sesar tyɩ́lɛ Sesarlɛ n kɔ̃, pè mó Liyel tyɩ́lɛ Liyellɛ n kɔ̃. 
Pǎh képah tɛ̃̀ noh sɛ̃́, pé gbã̀n du. Pè mɛ yal ǹ tnɔ̂ à sɔkɔ. 
Lésõ mɛ̀ ólɛ, Sadusi wɛ̃̀kɩ̀ kɔ́pú nónó-á ńkɛ̃́nɛ nɛ n yo nɛ lékyɩ̂ yílkínmɔnɔ náh mɛ, pé túkù nɔ de Yesu tnɔ̂ tɛ́ à piki yah nɛ, 
kwɔ́-ò dĩ́, Moyisi-á kè yo ǹ tyi kõ̀nsàhnɔ́ sɔkɔ nɛ, dĩ́ nɛ̂-à ku pi kɛ̃́nɛ ǹ tyɩ́, ǹ gbɛ̃̀ndĩ́-á yɩ̃nɛ à ǹ kyɩ pɛ́kúsè mɛ̀ tɔkɔ, à pi dubi à kɔ̃, nɛ̂-á kɩ pɔ n pɩ n pi ǹ nɛ́nkyɩ́ dĩ́ lékókè náhnɛ. 
Ǹtɛ sú núkú dĩ́npì nɛ́pĩ̂ kwǎrnimi yísyɔ́-á lésõ mɛ pépi wrɔ́ sɔkɔ. Gben tɛ̃̀ mɛ pɔ sê tɔkɔ, tɛ́ pɔ ku pi wɛ́-ńkɛ̃̂nɛ tɛ́ ǹ kyɩ sah ǹ sõ̀ syɩ kɔ̃. 
Ǹtɛ ǹmɔ tɔ tyɩ́ tɔ̃ pɔ pɩ sɛ̃́, tɔ̃́-ò tɔ tyɩ́ pɔ pɩ képah núkú ólɛ, fɔ́ɔ́ à pɔ syɩ kwǎrní-ò tyɩ́. 
Pépi pól tyɩ́ náh, pi mɛ̀ tɔ-á pɔ ku. 
Ǹtɛ pi mɛ̀-á de yah pé pól tyɩ́ sɛ̃́ à, lékyɩ̂ yilki n mɔ sõlɛ, à se kɩ n pɩ n pi pé nɛ́pĩ̂ kwǎrnimi nɔ́pi sɔkɔ nɛ̂ tɛ̃̀ tyɩ́lɛ nɩ? 
Képah sɔkɔ, Yesu yo pé tyɩ́ nɛ: «Liyel wɛy sɛ́bɛ́y pɩ̃́-ńkɛ̃̂ ànɛ̂ Liyel tɛ̃́nwɛnɔ pɩ̃́-ńkɛ̃̂ ye mɔ sah yélɛ pílkínɔ́ sɔkɔ. 
Yé yah, lékyɩ̂-à pɔ yilki mɔ, pɛ́dékè náh kɩ n mɛ n pi, sétɔ́kɔ́kɩ̀ tɔ náh kɩ n mɛ n pi. Ǹtɛ névye kɩ mɛ yĩ̂nyõ tõ̀npɩbɩ tyɩ́ kɔ̃lɛ yĩ̂nyõ. 
Tɛ́ à yuku lékyɩ̂ yílkínmɔnɔ wɔ tyɩ́ sɔkɔ, Liyel-á nɛ̂ yo yé tyɩ́, yé náh kè kar yah di? À lésõ nɛ: 
‹Ńmɔ ye Abrahmɩ Liyellɛ, ànɛ̂ Yisakɩ Liyellɛ, ànɛ̂ Nsyakɔbɩ Liyellɛ.› Liyellɛ, lékyɩ̂ Liyel náh ǹnɛ dò, vye Liyel ye. Képah n wɛ̃kɩ nɛ á náh tãn Abrahmɩpi lékyɩ̂-á yilki mɔ.» 
Képah tɛ̃̀nɛ, tɔ́wû nɛ̂-á sõ̀ mɛ ǹ wɛynɛ n nohnɩ, pé gbã̀n du ǹ névye kwɔ dùkù tyɩ́lɛ. 
Farisi wɛ̃̀kɩ̀ kɔ́pú-á noh nɛ Yesu-á wɛynɛ Sadusi wɛ̃̀kɩ̀ kɔ́pú tɛ̃, pè wɛ̃ ye tuke mɔ. 
Tɛ́ pé Liyel tyi kõ̀nsàhnɔ́ pɩ̃́pú sɔkɔ yɔ́ mɛ píkínyahnɔ yɔ́ pɩ Yesu tyɩ́ ǹ pilki n tɛ̃ yĩnnɛ nɛ, 
kwɔ́-ò dĩ́, Liyel tir kõ̀nsàhnɔ́ nɛ̂ tɛ̃̀ tyɩ́ se tah kal sé tɛ́lɔ́lɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Á Tɛ̃̀ ǹgbɛ̃ Liyel tyɩ́ yɩ̃nɛ kɩ nɔ á tyɩ́ á nɩ póllɛ, ànɛ̂ á min póllɛ, ànɛ̂ á sõnɔ póllɛ.› 
Képah ye tir kõ̀nsàhnɔ́ gbennɛ, ànɛ̂ nɛ̂ tyɩ́-á tah kal. 
Ké ní-ò nɛ̂-á tɔ̃ mɛ ké tyɩ́ kɔ̃lɛ, képah ye yɔ̀ nɛ: ‹Yɛ á ní-ò tyɩ́ nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ.› 
Moyisi tyi kõ̀nsàhnɔ́ ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu sɛ́bɛ́ynɔ́ sɔkɔ tyi pól mɛ tyi kõ̀nsàhnɔ́nɔ nínɔ́ nɔ́pi sɔkɔ.» 
Farisi wɛ̃̀kɩ̀ kɔ́pú-á sõ̀ mɛ tùkènmɔnɔlɛ, Yesu mɛ píkínyahnɔ yɔ́ pɩ pé tyɩ́ nɛ: 
«Yépi tɛ̃̀ sõnɔ nɛ̂nɛ Krista tyɩ́ sɔkɔ, nɛ̂-á Liyel nɛy yáhntɔkɔnɔlɛ? Nɛ̂ pídĩ́ mɛ ǹnɛ?» Pè mɛ yo ǹ tyɩ́ nɛ, Davidi Pídĩ́-á. 
Képah sɔkɔ, à mɛ yo pé tyɩ́ nɛ: «Ǹtɛ kwâh nɛ̂ pɩ tɛ́ Liyel Mirki-á Davidi yah tɛ̃, Davidi mɛ Krista ye Tɛ̃̀ ǹgbɛ̃nɛ? Yé náh wɛ di, à lésõ nɛ ńkɛ̃́, 
nɛ̂-á péwɔ TƐ̃̀ ǸGBƐ̃NƐ Tɛ̃̀ ǹgbɛ̃ Liyel-á kè yo ǹ tyɩ́ nɛ ńkɛ̃́, à pɔ n kɔ̃ n tɛ̃ pé gbõ̀yõ̀ sõ̀lɛ, fɔ́ɔ́ kè pɔ nɔ gbĩ́ nɛ̂-á pé kɩ ǹ sépɛ́bɩ́ pɩ ǹ gbáhkɩ̀ súnsàh-ǹsahlɛ. 
Ǹtɛ Davidi-á Krista ye TƐ̃̀ ǸGBƐ̃NƐ à kɩ tɔ̃ pɩ sɔ̃́ tɛ́ kɩ pɔ n pɩ ǹ Pídĩ́nɛ?» 
Yesu-á sépi yo, tyíyóné náh tɔ̃ fɛ̃ wɛ pé sɔkɔ yɔ́ ńtɛ̃ tyɩ́. Tɛ́ à tɔkɔ lésõ tyɩ́, ǹnɩ̂ yɔ́ ńtɛ̃ náh tɔ̃ ǹ nɩ syãh à piki yah yah tir yɔ́ ńtɛ̃nɛ.
