Markɩ.
5.
Képah náh, pè kyɩ yĩni tɛ̃ Kalile yɔ́-ǹgbɛ̃ kahnɩnɛ, Yerasa tãn sétáh yõ. 
Yesu-á fyɔ̀ n tiki krótókè sɔkɔ, nɛ́npɔ́ swɔ, dĩ́ yɔ́ wil léyã̂l wrɔ́, à pɔ à yohnɩ. *Sétõ sõ̀ mɛ kénɛ dĩ́ sɔkɔ. 
À sõ̀ mɛ tɛ̃̀nɔ́lɛ léyã̂l wrɔ́. Névi náh sõ̀ fɛ̃ à pukubi n sah n pi kɛ̃́y fyélɛ póllɛ. 
Kèyáhlɛ, pè ńkɛ̃́nɛ kârn mɔ ǹ gbyẽ̂h tyɩ́, ǹ tɛ́ kɛ̃́lɔ́ mɔ ǹ gbã̀n tyɩ́, tɛ́ à kârn kwɔkɩ, ǹ tɛ́ kɛ̃́lɔ́ kɔ̃lɩ. Kǎh pɩ sɛ̃́, ké tǎhkɩ̀ náh sõ̀ mɛ ǹnɩ̂ yɔ́ ńtɛ̃ sɔkɔ, kɩ fɛ̃ à tɛ̃ sah. 
Ké lékã̂h, ké gbĩ̀ntɔ̃̀, à sõ̀ ńkɛ̃́nɛ n nɛ n kɔ n kɔ̃ léyã̂l wrɔ́ ànɛ̂ ymɔ sɔkɔ, n nɛ n de n syi, ǹ gblɔ̌ynɛ n pɩ n yɔ n kɔ̃ dyêhlɛ. 
Tɛ́, wǎh Yesu wɛ fɩ́ɩ́ sɔkɔ, à to pɔ gbètukunɔ di ǹ yah. 
Tɛ́ de syi yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ, Yesu, pé yõ̀tah Liyel pi, kwâh nɛ̂ se à n yah n kɔ̃ pé tyɩ́ nɩ? Páh ǹnɛ n ni Liyel yĩn yõ, à káh pé fõh. 
Yesu sõ̀ mɛ n yo sétõ mɛ̀ tyɩ́ nɛ, à wil dĩ́ mɛ̀ sɔkɔ, képah ye. 
Tɛ́ mɛ à piki yah nɛ, ǹ yĩn se sɔ̃́? À mɛ nɛ, pé yĩn-á Mɛ́fwóbé Kwĩnkilɛ. Páh mɛ dò, képah-á. 
Tɛ́ mɛ à gbah ni nɛ, à káh pé yĩni wilki kwil mɛ̀ sɔkɔ. 
Ké nɛ́npɔ́ sɔkɔ, sel kwĩnki yísyɔ́ sõ̀ mɛ n yah n yõ n kɔ̃ yɔ̃lɩ yõ. 
Sétã̀n nɔ́pi Yesu ni nɛ, páh kɩ wil dĩ́ mɛ̀ sɔkɔ, à yɛ pé kɔ̃, pé de sel nɔ́pi sɔkɔ. 
À mɛ ké wɛ̃̀kɩ̀lɛ sè kɔ̃. Légbĩ́nɛ, sè mɛ wil dĩ́ mɛ̀ sɔkɔ, tɛ́ kyɩ de sel nɔ́pi sɔkɔ. Sel kwĩnki ye sõ̀ sélɛ, sè kɩ fɛ̃ nɔ sel kèyõ kénkɔ̃lɛ, tɛ́ mɛ tɛ́yõ. Sè núkú to tiki tùkùtùkù, à kyɩ de yɔ́-ǹgbɛ̃ sɔkɔ, kè kɔ̃ kè sè yõ. 
Kǎh pɩ sɛ̃́, sel nɔ́pi fɛ̃̀pu to sɔkɔ kwil nɩyṍ, ànɛ̂ gbnɔ sɔkɔ, à kyɩ ké dùkù yo kɔ̃ no tyɩ́. Képah tɛ̃̀nɛ, névye pɔ ké nɛ́npɔ́, à pɔ ké dùkù yah, sɔ̃́-á kè pɩ. 
Pè kyɩ Yesu tnɔ̂, tɛ́ kyɩ dĩ́ mɛ̀ wɛ, nɛ̂-á sõ̀ sétã̀n kwĩnkilɛ déntɛ̃̀nɔ́lɛ ǹ sɔkɔ. À sõ̀ mɛ flɔ mɔ kɔ̃ tɛ̃, tɛ́ mɛ tírr névye tyɩ́ kɔ̃lɛ. Kǎh pɩ sɛ̃́, tíkí pè tɛ̃. 
Tir nɛ̂-á pɩ sétã̀n nɛ́déné sɔkɔ ànɛ̂ sel tyɩ́, sé wɛ́pú tyah sélɛ n yãh pé tyɩ́. 
Képah sɔkɔ, kwil tãn tyah Yesulɛ n ni nɛ, à yal pé kwil sɔkɔ. 
Kǎh pɩ sɛ̃́, Yesu-á n dɔkɔ krótókè sɔkɔ gbĩ́ nɛ̂nɛ, sétã̀n-á wil dĩ́ nɛ̂ sɔkɔ, à tyah ǹnɛ n ni nɛ, à yɛ pé kɔ̃ pé ǹ sõ̀ tɛ̃. 
Ǹtɛ, à náh ké wɛ̃̀kɩ̀lɛ à kɔ̃, tɛ́ mɛ yo ǹ tyɩ́: «Á syɩ n kyɩ gbô! Tɛ̃̀-ǹgbɛ̃-á ǹ nɩ̀gbɛ̀kɩ̀ sɔkɔ pèpɔrɩ ǹgbɛ̃ nɛ̂ pɩ á tyɩ́, ǹ kyɩ ké yah yo pé tyɩ́!» 
Dĩ́ mó ńtyɩ̃́nɔ ǹ syɩ kyɩ kélɛ n yo n kɔ̃ Kwlogbãm kwil sétáh yõ, tir nɛ̂-á Yesu pɩ ǹ tyɩ́. Nónó pól-á kélɛ n nohnɩ ǹ tyɩ́, pé gbã̀n du. 
Képah sɔkɔ, Yesu-á yĩni tɛ̃ yɔ́-ǹgbɛ̃ kahnɩnɛ krótókè sɔkɔ, à sõ̀ mɛ yɔ́ngbalɩ sɔkɔ, tɛ́ tɔ́wû ǹgbɛ̃ yɔ́ pɔ à mɔ gbɩ̃. 
Légbĩ́nɛ, *Liyel ní-ńsah wɔ̀kɔ̀ yõ̀tɛ̃̀ yɔ́ yĩni tɛ̃ nɛ́npɔ́, nɛ̂ yĩn-á Yayirusilɛ. Wǎh Yesu wɛ, à gbètukunɔ di ǹ yah, 
tɛ́ à gbah ni nɛ, pé písê lépír-á mɛ n tyah kúnɔ́lɛ. Páh ǹnɛ n ni, à pɔ kyɩ ǹ gbã̀n sah ǹ yõ. Képah-à pɩ, wǎh kɩ dɛ, tɛ́ kɩ min wɛ. 
Képah sɔkɔ, Yesu mɛ yuku ǹnɛ. Tɛ́ tɔ́wû ǹgbɛ̃ sõ̀ mɛ ǹ sõ̀ tɛ̃, mɛ à fyãh sah ywɔ̃ ywɔ̃ póllɛ. 
Nɛ́npɔ́ sɔkɔ, sê yɔ́ sõ̀ mɛ náhlɛ n yɔ tyɛ́-ńkɛ̃̂nɛ ye gbɔ̃ninɔ́. 
À sõ̀ kah kwlɔ kègbɩlɛ ǹ gblɔ̌y mɔ́nɔ́ sɔkɔ sènpɩbɩ náhnáh tyɩ́. Ǹ gbã̀n yî pól sõ̀ tyɛ, tɛ́ kè náh kwâh wilki ǹ yam sɔkɔ. Kè pyě mɛ gbɛ ólɛ n gbɛ mɔ. 
Pi mɛ̀-á Yesu tyɩ́ noh ńyãh sɔkɔ, à de tɔ́wû nɩyṍ ǹ náh tĩ̂nnɛ, à kyɩ tũ ǹ fwɔ̀mɔn tyɩ́. 
À sõ̀ mɛ n sõ ǹ fɔkɔ sɔkɔ nɛ, páh náhkɩ wɛ tũ ǹ fwɔ̀mɔn ó tyɩ́, pé ńyân-á náhkɩ tyɛ. 
Wǎh mó núkú tũ ǹ fwɔ̀mɔn tyɩ́, nɛ́npɔ́ swɔ, ǹ náh yɔ́nɔ́ tyɛ. À kè kyɔmɩ wɛ ǹ wil sɔkɔ nɛ, páh dɛ. 
Nɛ́npɔ́ swɔ, Yesu pɩ̃ nɛ, tǎhkɩ̀ yɔ́-á wil pé sɔkɔ. À mɛ ǹ yah vi tɔ́wû nɩyṍ, tɛ́ piki yah nɛ, nɛ̂ se tũ pé fwɔ̀mɔn tyɩ́ nɩ? 
Ǹ *tõ̀ sã́hpú mɛ yo ǹ tyɩ́ nɛ, à kó mɛ n yah tɔ́wû à mɔ gbɩ̃, mɛ à fyãh sah ywɔ̃ ywɔ̃ póllɛ, tɛ́ à tɔ̃ mɛ n piki n yah nɛ, nɛ̂ se tũ pé tyɩ́ nɩ? 
Ǹtɛ, Yesu mɛ vi ǹ gbɔ̀plɔ névye yah. À sõ̀ mɛ ké tyɩ́ nɛ, pé ké pɩ́-ò wɛ. 
Képah tɛ̃̀nɛ, pi mɛ̀ wil n syah tíkí tyɩ́. Ǹmɔ sõ̀ pɩ̃ tir nɛ̂-á pɩ ǹ sɔkɔ. À yuku kyɩ gbètukunɔ di Yesu yah, tɛ́ ké gbɩ yo dal mɔ à syah. 
À mó yo pi mɛ̀ tyɩ́: «Ń písê, ńmɔ yõ sɛ̃́nɔ́ ye ǹ dɛ mɔ. Nɛ n yuku yèvɩnɩ sɔkɔ, á ńyân tyɛ.» 
Wǎh mɛ képah yónɔ́ sɔkɔ sɛ̃́, tɛ̃ntõ̀ yɔ́ wil Liyel ní-ńsah wɔ̀kɔ̀ yõ̀tɛ̃̀ dĩ́ gbô, mɛ pɔ n yo kénɛ dĩ́ tyɩ́ nɛ, ǹ písê-á ku. Ńkɛ̃, yõ se tɔ̃ mɛ ké tyɩ́, à tɔ̃ kwɔ́-ò dĩ́nɛ n kwlɔ nɩ? 
Tɛ́ Yesu náh ǹ níkí sah képah tɛ̃̀ noh pé tyɩ́, tɛ́ mɛ yo Yayirusi tyɩ́: «Káh tíkí pɩ, sɛ̃ ó sɛ̃ ń yõ!» 
Képah tɛ̃̀nɛ, pǎh n sɔkɔ, à náh fɛ̃ syi nɛ ǹ sõ̀ tɛ̃, à wil Pyɛrɩ ànɛ̂ Nsyakɩ ànɛ̂ ǹ gbɛ̃̀ndĩ́ Nsyɔ̃ tyɩ́ sɔkɔ. 
Pǎh kyɩ de Liyel ní-ńsah wɔ̀kɔ̀ yõ̀tɛ̃̀ dĩ́ gbô, Yesu kyɩ névye wɛ, pé mɛ n gbo n mir n kɔ̃, mɛ n de n syi. 
À mɛ de wɔ̀kɔ̀, tɛ́ yo pé tyɩ́: «Kû pɩ, tɛ́ yé n gbo n mir n kɔ̃ sɛ̃́? Piri náh ku. Dɔ́nɔ́ ó ye à n dɔ.» 
Névye-á képah noh, pè tyah kṍlɛ ǹnɛ n ko. Képah sɔkɔ, à pé pól wilki wɔ̀kɔ̀ sɔkɔ. Tɛ́ piri sú, ànɛ̂ ǹ yṹ, ànɛ̂ ǹ tõ̀ sã́hpú nɛ́pĩ̂ tɔ̃́mɩ́ ó de ǹnɛ piri mɛ́-ńsah wɔ̀kɔ̀. 
À mɛ piri gbõ̀ tɛ̃, tɛ́ yo ǹ tyɩ́ pé kwil wɛy sɔkɔ: «Talita kumi!», nɛ̂-á pè n ye: «Pípír, mé kélɛ n yo á tyɩ́, yuku!» 
Ké nɛ́npɔ́ swɔ, piri yuku tyah n kɔ. Kǎh pɩ sɛ̃́, kè névye pól gbã̀n kah du. Légbĩ́nɛ, ǹ ye gbɔ̃ninɔ́ ye sõ̀. 
Tɛ́ Yesu mɛ ké gbah yo pé tyɩ́ nɛ, pé káh yɛ névye pɩ̃ ké sɔkɔ, ńkɛ̃, pè mó yõkelɛ à kɔ̃ à yõ.
