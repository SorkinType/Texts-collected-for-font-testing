Tyípɩ́nɩ̀.
6.
Légbĩ́nɛ, wãn sõ̀ mɛ n mɔ Yesu tõ̀ sã́hpú nɛ́dán yõ. Képah sɔkɔ, Krɛkɩ wɛy wɛ́kɩ́pú tyah n wɛkɩ nɛ, sõmɛsṍ pé no Yebre wɛy wɛ́kɩ́pú-á mɛ n syɩkɩ pépi pɛ́kúsyɔ yõ yõke dibi gbĩ́nɛ. 
Kǎh pɩ sɛ̃́, Yesu tɛ̃ntõ̀ névye nɛ́pĩ̂ gbɔ̃nimí mɛ Yesu tõ̀ sã́hpú tɔ́wû pól ye tuke, tɛ́ yo pé tyɩ́ nɛ, kè náh yɩ̃nɛ pépi Liyel wɛy yónɔ́ yal n mɔ, pé mó de tɛ̃ yõke díbínɔ́ sɔkɔ. 
Képah-á, pé yṹnpyé, pè díbí nɛ́pĩ̂ kwǎrnimi yah n wilki pé sɔkɔ, nónó yĩ́ngbɩ́-á mɛ n yo, tɛ́ Liyel Mirki mɛ pè tɔkɔ, ànɛ̂ lékã́m mɛ pé sɔkɔ. Páh kɩ tõ̀ mɛ̀ dahbɩ pépi gbõ̀. 
Kàh kɩ pɩ sɛ̃́, pépi-á kɩ mó yɔ̃ sɔkɔ nírí pɩ́nɔ́ ànɛ̂ Liyel wɛy yónɔ́ tõ̀lɛ. 
Képah sɔkɔ, pǎh nɛ̂ yo, tɔ́wû pól kè fɛ̃ syi. Pè mɛ Yetyɛ̃nɩ yah wilki, ǹmɔ nɛ̂ Liyel yõ sɛ̃́nɔ́-á sõ̀ mɛ gblò, tɛ́ Liyel Mirki mɛ à tɔkɔ. À tahlɩ Filipi tyɩ́, ànɛ̂ Prokɔrɩlɛ, ànɛ̂ Nikanɔrɩlɛ, ànɛ̂ Timolɛ, ànɛ̂ Parmenasɩlɛ, ànɛ̂ Antyɔsyɩ kwil Nikolasɩlɛ, ǹmɔ nɛ̂-á lésõ vi mɔ de Nsyifunɔ Liyel nínɔ́ wɛ̃̀kɩ̀ sɔkɔ. 
Tɛ́ mɛ pépilɛ Yesu tɛ̃ntõ̀ névye wɛ̃kɩ. Pépi mɛ pé gbã̀n sah pé yõ nírí pɩ́nɔ́ sɔkɔ, à sãm mɔ pé sɔkɔ. 
Képah tɛ̃̀nɛ, Liyel wɛy sõ̀ mɛ n tɔ̃ n mɔ n yuku, tɛ́ wãn mɛ n mɔ n yuku náhnáh Yesu tõ̀ sã́hpú nɛ́dán yõ Yerusalɛmɩ kwil. Sárká syínwìlkìpu tɔ náhnáh sõ̀ mɛ fɛ̃ syi sɛ̃ Yesu yõ. 
Yetyɛ̃nɩ sõ̀ Liyel pèpɔrɩ náhnáh wɛ ànɛ̂ ǹ tɛ̃́nwɛnɔlɛ. À sõ̀ mɛ à tɔkɔ à gbǐl tyilɛ n pɩ névye nɩyṍ. 
Képah tɛ̃̀nɛ, Nsyifunɔ tyah yékã̂blɛ n pɩ Yetyɛ̃nɩnɛ. Pé túkùlɛ, Liyel ní-ńsah wɔ̀kɔ̀ nɛ̂-á pè n ye yɔ̃rɩ sɔkɔ wìlpu wɔ̀kɔ̀lɛ, képah névye ye. Pè n wil Sirɛnɩ kwil ànɛ̂ Alɛkisãdri kwil. Pé túkù mɛ n wil Silisi gbɛ̃̀nyah ànɛ̂ Asi gbɛ̃̀nyah sɔkɔ. 
Tɛ́ Liyel Mirki gbõ̀ yõ, Yetyɛ̃nɩ sõ̀ mɛ n wɛkɩ lékã́mnɛ. Képah ye, no nɔ́pi náh mɛ tɛ̃ wɛ ǹ wɛlɔlɛ n kɔ̃lɩ. 
Képah tɛ̃̀nɛ, pè kyɩ nósyɔ́ tɔkɔ pɔ́lɛ nɛ pè pɔ yo nɛ, páh à wɛ à mɛ Moyisi ànɛ̂ Liyel tyɩ́lɛ n yo n yɩkɩ. 
Pè képah pɩ, à tɔ́wû, ànɛ̂ yèwàhnɩ̀, ànɛ̂ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú mɔ Yetyɛ̃nɩ yõ, pè kɔ̃ pè à fyãh tɛ̃ yuku ǹnɛ tukey tùkèpu tĩ̀nkì tyɩ́. 
Tɛ́ tɔ̃ tàrfyɔ̀ mɩ̀nyobe yísyɔ́ yah, à yuku pélɛ nɛ pè kyɩ yo nɛ, dĩ́ mɛ̀ náh mɛ n yal n mɔ Liyel ní-ńsah gbô ànɛ̂ Moyisi tyi kõ̀nsàhnɔ́ tyɩ́ yónyɩkɩnɔlɛ. 
Páh kè noh ǹ vyãh yah à mɛ n yo ńtɛ̃nɛ nɛ, kénɛ Nasarɛtɩ Yesu mɛ̀-á kɩ pɔ Liyel ní-ńsah gbô sɔ n pi, tɛ́ Moyisi-á pé kwɔ wã̀l nónónɛ, wǎh kɩ sé tɔ vi mɔ. 
Képah sɔkɔ, tukey tùkèpu tĩ̀nkì pól tyah Yetyɛ̃nɩnɛ n yah díkídíkí, tɛ́ ǹ yah wɛ kè mɛ yĩ̂nyõ tõ̀npɩ tyɩ́ kɔ̃lɛ.
