Markɩ.
4.
Sṍ yɔ́lɛ, Yesu tɔ̃ mɛ tyah névyelɛ n kwɔ Kalile yɔ́-ǹgbɛ̃ gbɔ̀pɔ sɔkɔ. Tɔ́wû ǹgbɛ̃ yɔ́ pɔ à mɔ gbɩ̃. Pè tnɔ́nkàhnɔ́ tɛ̃̀ tyɩ́, à de kɔ̃ tɛ̃ krótókè yɔ́ sɔkɔ. Krótókè mɛ̀ sõ̀ mɛ yɔ sɔkɔ, tɛ́ tɔ́wû pól mɛ tɛ̃̀nɔ́lɛ yɔ́ngbalɩ. 
À sõ̀ mɛ pélɛ n kwɔ tyi náhnáhlɛ sèmìrn sɔkɔ, mɛ n yo pé tyɩ́ ǹ kwɔ́nɔ́ sɔkɔ nɛ: 
«Yé noh! Sṍ yɔ́lɛ, dĩ́ yɔ́ sõ̀ kyɩ ǹ swãh, sèmukɔ mír-ńsah. 
Wǎh mɛ mírnɔ́ sɔkɔ, yísyɔ́ syɩkɩlɩ wɛ̃̀gbalɩ sɔkɔ, kèpye pɔ sè yɔ. 
Yísyɔ́ mɛ syɩkɩlɩ yen kwéy tɛ̃̀-ǹsah légbɩ̃́ yõ, tɛ́ fi súú. Yen gbɩ náh sõ̀ mɛ nɛ́npɔ́, képah ye. 
Gbĩ-á pɔ vyãh sè tɛ̃, sè kɔkɔ. Yã̀hnɩ̀ gbɩ náh sõ̀ mɛ sé tyɩ́, képah ye. 
Yísyɔ́ mɛ kyɩ syɩkɩlɩ pã̀n nɩyṍ. Pã̀n nɔ́pi mɛ yuku fyãh sé yõ, sè náh tɛ̃ wɛ pye pɩ. 
Ǹtɛ, yísyɔ́ kyɩ syɩkɩlɩ sétáh pèpɛy sɔkɔ. Sépi mɛ fi, à kyãh pallɛ. Túkù pyenɔ yõgbãm pɩ, túkù tikitɔ̃nɩ́ pɩ, sé túkù mó lèkwɛ̌ pɩ.» 
Képah náh, à tɔ̃ yo pé tyɩ́: «Níkí-á kɩ mɛ nɛ̂ tyɩ́ noh noh tɛ̃̀nɛ, tɛ̃̀ ǹ níkí sah noh.» 
Yesu-á yal tɔ́wû tnɔ̂ gbĩ́ nɛ̂nɛ, nónó-á ńkɛ̃́nɛ nɛ mɛ ǹnɛ, pépi ànɛ̂ ǹ *tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí mɛ ǹnɛ n piki n yah sèmìrkì mɛ̀ yõlɛ. 
À mó yo pé tyɩ́: «*Liyel tɔ̃́rɩ́ tyi nónó-á wɔ̃̀bɩ̀nɔ́lɛ, sé pnɛ̀nɔ́ ye yépi tɛ̃̀ wɔ kɔ̃, ǹtɛ kégbɔ́ névye tyɩ́, tyi pól n yo sèmìrn ó sɔkɔ. 
Képah sɔkɔ, pè kɩ n yah díkídíkí, tɛ́ pè náh kwâh yɔ́ wɛ n pi. Pè kɩ n nohnɩ, tɛ́ pè náh tir yɔ́ pɩ̃ n pi. Nɛ́ képah náh pɩ, pè náhkɩ pé pɩ dùkù vi mɔ, à pé gblɔ̌ynɛ Liyel kɔ̃, tɛ́ náhkɩ pé tyípêl yɔ̃̀nmɔnɔ wɛ.» 
Yesu tɔ̃ yɔ̃ sɔkɔ yo pé tyɩ́: «Yé náh sèmìrkì mɛ̀ yõ pɩ̃ di? Ǹtɛ, yé kɩ sèmìrn tɛ́lɔ́ yõ pɩ̃ sɔ̃́? 
Sèmukɔ mír-òlɛ, Liyel wɛy ye à n mir. 
Tɛ́ névye túkù mɛ, á kɩ nɛ wɛ̃̀gbalɩ sɔ̃́, wɛy-á mir mɔ ńyãh sɔkɔ: Pépi-à wɛy mɛ̀ noh, nɛ́npɔ́ swɔ, *sétã̀n yõ̀tɛ̃̀ n pɔ wɛy mɛ̀ wilki pé fɔkɔ. 
Túkù tɔ̃ mɛ légbɩ̃́ tɛ̃̀-ǹsah sɔ̃́. Pàh wɛy mɛ̀ noh, pě kè fɛ̃ n syi nɩ̀vɩ̀nɩ̀nɛ. 
Tɛ́ pè náh n yɛ wɛy mɛ̀ yã̀hnɩ̀nɛ n mɔ pé sɔkɔ. Nɛ́npɔ́ swɔ névye ye pélɛ. Képah ye, kwlɔ́nɔ́-à pè yĩni tɛ̃, képah náh pɩ, névi-à pè fõh Liyel wɛy yĩnnɛ, pě pé náh vi súú Liyel wɛy kɔ̃. 
Túkù mó tɔ̃ mɛ, á kɩ nɛ pã̀n tɛ̃̀-ǹsah sɔ̃́. Pàh wɛy mɛ̀ noh, 
kèkõyṍ tyi fɔkɔ sõnɔ, ànɛ̂ tahrɩ ànɛ̂ tyi náhnáh dékè n de pé sɔkɔ. Sépi n pɔ wɛy mɛ̀ fyãh, kè náh n tɛ̃ n wɛ pyelɛ n pɩ. 
Névye túkù mó mɛ, á kɩ nɛ sétáh pèpɛy sɔ̃́: Pépi-à wɛy mɛ̀ noh, pě kè fɛ̃ n syi, ǹ tɛ́ pye pɩ. Túkù pyenɔ yõgbãm pɩ, túkù tikitɔ̃nɩ́ pɩ, túkù n mó lèkwɛ̌ pɩ.» 
Képah náh, à tɔ̃ yo pé tyɩ́: «Névi se kɩ fɛ̃ fɔ̀tnɔ̀ soh, tɛ́ kɩ dú kwĩnki mɔ ké yah, képah náh pɩ, kɩ kè dah mɔ sɛ̃ sɛ̃ kwâh sètàhkɩ̀ sõ̀? À náh kè dɔkɔ n mɔ n pi ké sàh-ǹsah di? 
Tir yɔ́ ńtɛ̃ náh mɛ wɔ̃̀bɩ̀nɔ́lɛ, nɛ̂ náh n wilki n mɔ n pi kégbɔ́. Sètĩ̀n sõ̀ tir yɔ́ ńtɛ̃ náh n wũ n pi wɛ̃́kɩ́-ńkɛ̃̂nɛ. 
Níkí-à mɛ nɛ̂ tyɩ́ noh noh tɛ̃̀nɛ, tɛ̃̀ ǹ níkí sah noh!» 
Tɛ́ tɔ̃ ǹ syɩ yo pé tyɩ́: «Yáh tyi nónónɛ n nohnɩ, yé sélɛ n kyɔmɩ pallɛ. Yáh no tɛ́lɔ́ tyɩ́lɛ n sãh dú nɛ̂nɛ, Liyel kɩ yépi tɔ tyɩ́ sãh n pi kénɛ dú núkúlɛ, tɛ́ dòhnɩ̀nɔ́ ńtɛ̃ kɩ wɛ yépi tyɩ́ yõ. 
Kwâh-á mɛ névi nɛ̂ tyɩ́, dòhnɩ̀nɔ́ kɩ pɩ ǹ tyɩ́ yõ ńtɔ̃. Tɛ́, névi nɛ̂ tyɩ́ kwâh náh mɛ, kwéy nɛ̂-á mɛ tɛ̃̀ tyɩ́, képah tɔ kɩ tɛ̃ syi ǹ tyɩ́.» 
Képah náh, à tɔ̃ yo pé tyɩ́: «Yé yah, kwâh nɛ̂-á *Liyel tɔ̃́rɩ́ kètɔ̀lɛ: Kè mɛ, á kɩ nɛ dĩ́ yɔ́ sɔ̃́ nɛ̂-á sèmukɔ mir ǹ swãh sɔkɔ. 
À mɛ n dɔ ò, à mɛ n yah ò, lékã̂h ànɛ̂ ké gbĩ̀ntɔ̃̀, sèmukɔ n nɛ mɛ fi, n nɛ n kyãh, à náh ké dùkùlɛ n pnɛ. 
Sétáh ye sèmukɔ mɛ̀ kɔ̃ kè fi ké gblɔ̌y. Sè kètuke wil à pɔ dubi pɩ ywɔ̃lɛ, à yɩ̃́ pɩ. 
Sǎh fyɔ̀ ko tɛ, sèmukɔ tɔkɔ gbĩ́-á nɔ, à yɛ̃́nkwɛ̃́kɩ̀ tɔkɔ kyɩ tyah sélɛ n kõ.» 
À tɔ̃ yo pé tyɩ́: «Á kɩ fɛ̃ kwâh nɛ̂ tɔkɔ kɩ *Liyel tɔ̃́rɩ́ dùkù wɛ̃kɩ? Képah náh pɩ, sèmìrkì nɛ̂ mɛ mé kɩ yãh kɩ yɩ̃nɛ kélɛ? 
Kè mɛ, á kɩ nɛ mútárdì pi sɔ̃́ tyɩ̃́ɩ̃. Kè yilɔ kal sèmukɔ duke póllɛ. 
Tɛ́, pàh kè duku, kè n yuku n pɔ n pɩ yòhkɩ̀ ǹgbɛ̃nɛ n kal sèmukɔ duke póllɛ swãh sɔkɔ. Ǹtɛ, yɩ̃̀ngbã̀n gbɛkɩ dò, kèpye kɩ fɛ̃ pɔ syɛ̀ pɩ ké sɔkɔ.» 
Sɛ̃́ ye Yesu n kwɔ névyelɛ sèmìrkì mɛ̀ duke náhnáh sɔkɔ. À sõ̀ mɛ sélɛ n yãh n yɩ̃nɛ sé nòhnɩ̀pu pɩ̃ pɩ̃ tɛ̃̀nɛ. 
À náh sõ̀ n yãh pé tyɩ́ sèmìrn mɩ́-ńkɛ̃̂nɛ. Tɛ́, pàh mɛ pé yahlɛ, ǹmɔ ànɛ̂ ǹ *tõ̀ sã́hpúlɛ, ǎ sé pól yah yo n dal n mɔ pé tyɩ́. 
Ké lésõ ńvɩ̃́nɛ, Yesu yo ǹ *tõ̀ sã́hpú tyɩ́ nɛ, pè yɛ, pé yuku yɔ́-ǹgbɛ̃ kahnɩnɛ. 
Képah sɔkɔ, pè mɛ yal tɔ́wû tnɔ̂. Wǎh sõ̀ mɛ krótókè nɛ̂, ǹ tõ̀ sã́hpú sɔkɔ ǹnɛ képah sɔkɔ. Tɛ́ krótónènɔ́ yísyɔ́ tɔ sõ̀ mɛ ǹnɛ. 
Dǒ, yã́hpékè yɔ́ kõ, à ninɛ n tɔkɔ n mɔ krótókè sɔkɔ pópó, kè pɔ n tyah yúkúnɔ́lɛ ninɛ. 
Tɛ́ légbĩ́nɛ, Yesu sõ̀ mɛ sɛ̃ tɛ̃ n dɔ krótókè sɔkɔ náh tĩ̀nnɛ, tɛ́ mɛ ǹ yõ sah yõ̀sah yõ. Képah sɔkɔ, ǹ tõ̀ sã́hpú kyɩ à yilki mɔ, tɛ́ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, yɔ-á n tyah pé yṍnɔ́lɛ, kè kó mɛ tirlɛ ǹmɔ tyɩ́ nɩ? 
Wǎh dɔ yuku, à dírí kõ yã́hkɩ́ yõ, tɛ́ yo yɔ tyɩ́: «Yĩn! Tɛ̃ syɩ́syɩ́!» Yã́hkɩ́ núkú yĩn, tyah pɩ sṹmm. 
À mɛ yo pé tyɩ́: «Kû pɩ, tɛ́ tíkí yé tɛ̃ sɛ̃́? Yé náh tɔ̃̌nɩ̀ sɛ̃ ń yõ di?» 
Képah tɛ̃̀nɛ, tíkí mɛ pè kah tɛ̃. Pè tyah wɛ̃nɛ n piki n yah nɛ, nɛ dùkù nɛ̂ se dĩ́ mɛ̀nɛ, tɛ́ yã́hkɩ́ ànɛ̂ yɔ tɔ n sah ǹ tyɩ́ yõ nɩ?
