Tyípɩ́nɩ̀.
16.
Képah náh, Polepi-á wil Siri ànɛ̂ Silisi gbɛ̃̀nyah sɔkɔ, pè yɔ̃ sɔkɔ Dɛrɩbɩ kwil. Pǎh wil nɛ́npɔ́, pè yuku Lisitir kwil. Pè kyɩ Yesu tõ̀ sã́h-ò yɔ́ wɛ nɛ́npɔ́ pè n ye Timotelɛ. Ǹ yṹnɛ, Nsyifu nɛy ye sõ̀ ǹnɛ, nɛ̂-á mɛ Yesu wɛ̃̀kɩ̀lɛ n kɔ, tɛ́ ǹ sú mɛ Krɛkɩ nɛynɛ. 
Tɛ́ Yesu yõ sɛ̃́pú nónó-á sõ̀ mɛ Lisitir sɔkɔ ànɛ̂ Yikonyumi kwil, pè sõ̀ mɛ Timote yĩ́ngbɩ́lɛ n yo. 
Képah sɔkɔ, Pole sõ̀ mɛ ké tyɩ́ nɛ pé à tɛ̃ sɔkɔ pé kɔ́lɔ́ sɔkɔ. Tɛ́ Nsyifunɔ nónó-á sõ̀ mɛ kénɛ kwlo sɔkɔ, pè sõ̀ pɩ̃ nɛ, Krɛkɩ nɛy-á Timote súlɛ. Pépi yĩnnɛ, Pole mɛ Timote kõ. 
Képah sɔkɔ, Polepi-à fyɔ̀ n sɔkɔ kwlo nónó sɔkɔ, Yesu tɛ̃ntõ̀ névye ànɛ̂ Yerusalɛmɩ kwil Yesu yõ sɛ̃́pú yah tãm-á tyi nónó yah tɔkɔ, pě sélɛ n yãh kénɛ kwlo Yesu yõ sɛ̃́pú tyɩ́ nɛ, pè nɛ n kɔ sé wã̀l yõ. 
Kǎh pɩ sɛ̃́, Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ sõ̀ mɛ n gbah n sɛ̃ Liyel yõ, tɛ́ wãn tɔ̃ mɛ n mɔ pé nɛ́dán tɔ yõ sõmɛsṍ. 
Képah náh, Polepi sõ̀ mɛ ké tyɩ́ nɛ pé kyɩ Liyel wɛy yo Asi gbɛ̃̀nyah sɔkɔ, ǹtɛ Liyel Mirki kè yahle pé tyɩ́. Kǎh pɩ sɛ̃́, pè mɛ kah syɔ̃ Frisyi gbɛ̃̀nyah ànɛ̂ Kalasi gbɛ̃̀nyahlɛ. 
Pǎh kyɩ yĩni tɛ̃ n tɛ Misi gbɛ̃̀nyahlɛ, pè tɔ̃ kè yah tɔkɔ nɛ, páh kɩ yuku Bitini gbɛ̃̀nyah. Ǹtɛ Yesu Mirki kè yahle pé tyɩ́. 
Kǎh pɩ sɛ̃́, pè Misi kɔ kah, à tiki yuku Turasɩ kwil. 
Tɛ́ lékã̂hnɛ, Pole wɛ̃́kɩ́nɔ́ yɔ́ wɛ. Wɛ̃́kɩ́nɔ́ mɛ̀ sɔkɔ, à Masedonɩ gbɛ̃̀nyah dĩ́ yɔ́ wɛ yĩ́nntɛ̃̀nɔ́lɛ, tɛ́ tɛ̃̀ mɛ ǹnɛ n ni nɛ ńkɛ̃́, à pɔ Masedonɩ sɔkɔ, à pɔ pépi yohnɩ n mɔ. 
Pole-á fyɔ̀ wɛ̃́kɩ́nɔ́ mɛ̀ wɛ sɛ̃́, á mɛ kè pɩ̃ yĩ́ĩ́ nɛ, Liyel-á ye álɛ nɛ, á kyɩ pé Nɩ̀vɩ̀nɩ̀ tir yo Masedonɩ tãn tyɩ́. Kǎh pɩ sɛ̃́, á núkú á vi nɛ áyáh kɩ sɔkɔ nɛ́npɔ́. 
Képah sɔkɔ, á de krótókè ǹgbɛ̃ yɔ́ sɔkɔ Turasɩ kwil, à yɔ̃ sɔkɔ gbɛ̃̀nyah yɔ́ sɔkɔ, nɛ̂-á mɛ pnɛ yɔ nɩyṍ pè n ye Samotrasɩlɛ, á náh kwɔ dah fɩ̃́ yɔ́ ńtɛ̃ sɔkɔ. Tɛ́ ké dĩndallɛ, á yuku Neyapolisi kwil. 
Áyáh yal nɛ́npɔ́, á yuku kwíl-ǹgbɛ̃ yɔ́ sɔkɔ pè n ye Filipilɛ, Masedonɩ gbɛ̃̀nyah sɔkɔ. Kénɛ kwil yõ̀tɔ̃rɩ́ sõ̀ mɛ Wrome tãn gbõ̀. Á mɛ tɛ̃ nɛ́npɔ́, à sèpĩ̀n yísyɔ́ pɩ. 
Nsyifunɔ mɩ mɩ yòh sõ-á pɔ nɔ, á kè sõ nɛ, pé Liyel ní-ńsah-á kɩ wɛ yɔ́ngbalɩ sɔkɔ. Kǎh pɩ sɛ̃́, á wil kwil nɩyṍ, à nɛ́npɔ́ nyáh sõ. Á kyɩ syɔ́ yísyɔ́ wɛ tùkènmɔntɛ̃̀nɔ́lɛ nɛ́npɔ́. Á mɛ kɔ̃ tɛ̃ pé tnɔ̂, à yã́hkɩ̀ yãh pélɛ Liyel wɛy tyɩ́ yõ. 
Tɛ́ pi yɔ́ sõ̀ mɛ syɔ́ nɔ́pi sɔkɔ pè n ye Lidilɛ. Fwɔ̀sãhlɔ dɛ́nkɔ́kɔ́nɩ̀ gbɔ̀ yɩ̃́-ò ye sõ̀ ǹnɛ. À n wil Tyatiri kwil, tɛ́ à sõ̀ mɛ Nsyifunɔ Liyel wɛ̃̀kɩ̀lɛ n kɔ. Tɛ́ Tɛ̃̀ ǹgbɛ̃ mɛ wɛkɩ pi mɛ̀ nɩ sɔkɔ à kɔ̃ à ǹ níkí kwɔ, à Pole wɛlɔ noh pallɛ. 
Kǎh pɩ sɛ̃́, ǹmɔ ànɛ̂ ǹ kéte tãn pól mɛ wèlnwìlkìnɔ́ wɛ ni sɔkɔ. Képah sɔkɔ, pi mɛ kè yah kɔ̃ ápi tyɩ́ nɛ, áyâh kɩ mɛ sɛ̃ ké yõ nɛ péwɔ-á pɩ Tɛ̃̀ ǹgbɛ̃ Yesu yõ sɛ̃́-òlɛ, á pɔ n kyɩ n twah pé tyɩ́ pé gbô. Tɛ́ á gbah tɛ̃ pópó, á pɔ sah ǹ tyɩ́ yõ. 
Képah sɔkɔ, sõ yɔ́lɛ á mɛ n yuku nírí pɩ́-ńsah, tɛ́ tõ̀npɩ pípír yɔ́ pɔ á yohnɩ. Sétõ sõ̀ mɛ kénɛ pípír gbil à kɔ̃ à tyísnɔ́lɛ n wɛ n yo. À sõ̀ mɛ pɔ́ náhnáhlɛ n wɛ ǹ yõ̀tãmnɛ n kɔ̃ kénɛ tyísnɔ́ wɛ́nyonɔ sɔkɔ. 
Kénɛ pípír mɛ yuku, à ápi ànɛ̂ Pole sõ̀ tɛ̃, tɛ́ mɛ yo tũ léyy no tyɩ́ nɛ ńkɛ̃́, díbí nɔ́pilɛ, yõ̀tah Liyel tõ̀npɩbɩ-á pélɛ. Pwáhnmɔnɔ wɛ̃̀kɩ̀ tyɩ́-á pè pɔ n yo pé tyɩ́. 
À sõ̀ mɛ képah tɛ̃̀nɛ n pɩ sɛ̃́, sèpĩ̀n náhnáh sɔkɔ. Kǎh pɔ Pole gblɔ́lɛ n vyãh, à ǹ yah vi yo sétõ mɛ̀ tyɩ́ nɛ ńkɛ̃́, Yesu Krista yĩn yõ, à pípír mɛ̀ fĩn mɔ. Wǎh yo sɛ̃́, nɛ́npɔ́ swɔ sétõ núkú pípír fĩn mɔ. 
Képah sɔkɔ, pípír yõ̀tãm-á kè wɛ nɛ, pé sõnɔ-á sõ̀ mɛ pɔ́ nɛ̂ wɛ́nɔ́ yõ ǹmɔ gbõ̀ yõ kǎh kɩ yĩn mɔ, pè Pole ànɛ̂ Silasɩ tɛ̃, tɛ́ yuku pélɛ yõ̀tãm tyɩ́, tɔ́wû tùkè-ǹsah sɔkɔ. 
Képah náh, pè mɛ kyɩ pélɛ Wrome tãn yísyɔ́ tyɩ́ nónó-á pé yõ̀tãmnɛ, tɛ́ kyɩ tyah n yo nɛ ńkɛ̃́, díbí nɔ́pi-á mɛ fɛ̃̀kɩ̀nɔ́lɛ n pɩ pépi kwil. Nsyifunɔ-á pélɛ, 
tɛ́ pǎh mɛ pɔ wã̀l yísyɔ́lɛ névyelɛ n kwɔ nónó-á sé wã̀l náh mɛ pépi Wrome tãn ńtɛ̃ tyɩ́ kɩ sè fɛ̃ syi, képah náh pɩ, kɩ sè pɩ n pi. 
Képah tɛ̃̀nɛ, tɔ́wû tɔ núkú yuku Polepi yõ. Yõ̀tãm nɔ́pi mɛ pè kɔ̃ pè Polepi flɔ dɛ, tɛ́ nɛ pè pè ko yèfõhlɛ. 
Képah sɔkɔ, pǎh yèfõh yɩ̃́ náhnáhlɛ pè ko tɛ, pè pè dah mɔ wɔ̀kɔ̀npèkè sɔkɔ, tɛ́ yo ké syɩ́kɩ́-ò dĩ́ tyɩ́ nɛ, à pélɛ n kyɔmɩ pallɛ. 
Dĩ́-á fyɔ̀ képah noh sɛ̃́, à mó pè tɛ̃ dah mɔ wɔ̀kɔ̀npèkè nɩyṍ wɔ̀kɔ̀ sɔkɔ, tɛ́ sésár kârn mɔ pé gbyẽ̂h tyɩ́. 
Ǹtɛ lénɩ mɔ́nɔ́ wrɔ́ sɔkɔ, Pole ànɛ̂ Silasɩ sõ̀ mɛ nírílɛ n pɩ, tɛ́ mɛ yɩ̀lɛ n gbɩ à Liyel yĩnnɛ n gbilki, tɛ́ wɔ̀kɔ̀npèkè sɔkɔ névye tɛ́lɔ́ mó mɛ n nohnɩ pé tyɩ́. 
Képah tɛ̃̀nɛ, sétáh syahrɩ ǹgbɛ̃ núkú pɩ nɛ́npɔ́ swɔ, à wɔ̀kɔ̀npèkè sètyah yike. Ké vɩ̀ yey pól núkú yɩkɩ mɔ, ké sɔkɔ névye pól gbyẽ̂h sɔkɔ kârnnɔ́ pól kɔ̃lɩ kwɛ. 
Wɔ̀kɔ̀npèkè syɩ́kɩ́-ò dĩ́-á pópó dɔ yuku, tɛ́ ké vɩ̀ yey wɛ yɩ́kɩ́nmɔnɔlɛ sɛ̃́, à wɛ nɛ ké sɔkɔ névye-á to wil. Képah tɛ̃̀nɛ, à núkú ǹ kèkírkì yɔrɩ dɛ nɛ, pé pé gblɔ̌y ko mɔ. 
Ǹtɛ Pole mɛ yo mɔ ǹ tyɩ́ yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ, à káh dlo yɔ́ ńtɛ̃nɛ ǹ gblɔ̌y nɔ, pé pól-á mɛ ásõ̂. 
Képah sɔkɔ, wɔ̀kɔ̀npèkè syɩ́kɩ́-ò dĩ́ pè kɔ̃ pè pɔ fɔ̀tnɔ̀lɛ à kɔ̃, à de yuku súú wɔ̀kɔ̀npèkè nɩyṍ wɔ̀kɔ̀ sɔkɔ, tɛ́ ǹ gbètukunɔ di Pole ànɛ̂ Silasɩ yah wílsyahrɩ sɔkɔ. 
Tɛ́ mɛ pè tɛ̃ wil kégbɔ́, tɛ́ pè piki yah nɛ, pé nɛ́nkyɩ́bɩ́, sɔ̃́ se pé yɩ̃nɛ pé pɩ pé mó pwáhnmɔnɔ wɛ nɩ? 
Pè mɛ yo ǹ tyɩ́ nɛ: «Sɛ̃ Tɛ̃̀ ǹgbɛ̃ Yesu yõ, á kɩ pwáhnmɔnɔ wɛ, áwɔ ànɛ̂ á nolɛ.» 
Képah sɔkɔ, Polepi Tɛ̃̀ ǹgbɛ̃ Yesu wɛy yo ǹmɔ ànɛ̂ ǹ gbô tãn pól tyɩ́. 
Dĩ́ mɛ Polepi tɛ̃ légbĩ́ mɛ̀ ó sɔkɔ lékã̂hnɛ, à kyɩ pé nɩ́y wu; tɛ́ nɛ́npɔ́ swɔ, ǹmɔ ànɛ̂ ǹ gbô tãn pól mɛ wèlnwìlkìnɔ́ wɛ ni sɔkɔ. 
Képah náh, à Pole ànɛ̂ Silasɩ tɛ̃ yuku ǹ gbô sɔkɔ, à kyɩ yõkelɛ pè kɔ̃ pè yõ. Tɛ́ ǹmɔ ànɛ̂ ǹ no pól nɩ sõ̀ mɛ vɩ yĩ́ĩ́ pé wɛ́nsɛ̃nɔ yĩnnɛ Liyel yõ sɔkɔ. 
Képah sɔkɔ, tyah-á pɔ dal, yõ̀tãm nɔ́pi tɛ̃ntõ̀ névye tɛkɩ mɔ nɛ, pè kyɩ yo wɔ̀kɔ̀npèkè syɩ́kɩ́-ò dĩ́ tyɩ́ nɛ, à díbí nɔ́pi fĩn mɔ. 
Dĩ́ mɛ kyɩ kè yo Pole tyɩ́ nɛ ńkɛ̃́, yõ̀tãm-á tɛ̃ntõ̀ mɔ pé tyɩ́ nɛ, pé pè fĩn mɔ. Kǎh pɩ sɛ̃́, núkúnúkú pǎh kɩ fɛ̃ wil kɩ n yuku yèvɩnɩ sɔkɔ. 
Ǹtɛ Pole mɛ pè syah nɛ, pépi nónó-á gbõ̀ mɔ yõ Wrome tãn tyɩ́, pépi kó pè tɛ̃ ko tɔ́wû yĩ́ yah pé tùkè-ǹkɛ̃̂nɛ; tɛ́ pé dah mɔ wɔ̀kɔ̀npèkè sɔkɔ nɩ? Tɛ́ núkúnúkú, pè tɔ̃ mɛ n yah n kɔ̃ pé wìlkìnfĩnnmɔ-ńsahlɛ sètĩ̀n sõ̀ sɔkɔ nɩ? Képah náh fɛ̃ n pɩ n pi dɛ́! Ńkɛ̃́, fɔ́ɔ́ pé dáhnmɔpu-á kɩ pɔ pé fĩn mɔ pé gblɔ̌y. 
Képah tɛ̃̀nɛ, tɛ̃ntõ̀ névye nɔ́pi mɛ pé syɩ kyɩ wɛlɔ nɔ́pi yo yõ̀tãm nɔ́pi tyɩ́. Pépi-á képah tɛ̃̀ noh sɛ̃́ nɛ, Pole ànɛ̂ Silasɩlɛ, Wrome tãn gbõ̀nmɔ̀nyõbe-á pélɛ, tíkí pè tɛ̃. 
Pè pɔ gblɔ̌y yɛ yah kɔ̃ Polepi tyɩ́, tɛ́ pè fĩn mɔ, tɛ́ mɛ pè ni nɛ pè wil kwil sɔkɔ. 
Polepi-á wil wɔ̀kɔ̀npèkè sɔkɔ, pè yuku Lidi tĩ̀nnɛ. Pǎh kyɩ Yesu yõ sɛ̃́pú wɛ, pè sṍré mɔ pé sɔkɔ tɛ́ sɔkɔ.
