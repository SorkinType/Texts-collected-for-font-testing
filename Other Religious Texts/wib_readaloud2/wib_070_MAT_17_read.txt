Matye.
17.
Sèpĩ̀n kénúkò sɔ́kɔ́ntɛnɔ náh, Yesu mɛ Pyɛrɩ, ànɛ̂ Nsyakɩ, ànɛ̂ Nsyakɩ gbɛ̃̀ndĩ́ Nsyɔ̃ tɛ̃ pè yuku yɔ̃lɩ kèyy yɔ́ yõ, à kyɩ tɛ̃ pé tyɩ́lɛ. 
Dǒ, Yesu dùkù vi mɔ pé yĩ́ yah. Ǹ yèpɔ tyah n tɛ̃ gbĩ sɔ̃́, tɛ́ ǹ flɔ mɛ piki prɛ́yy. 
Moyisi ànɛ̂ Yeli núkú wil mɔ pé yah, tɛ́ mɛ n yãh ànɛ̂ Yesulɛ. 
Képah sɔkɔ, Pyɛrɩ mɛ yo Yesu tyɩ́ nɛ, Tɛ̃̀ ǹgbɛ̃, pé pól tɛ̃̀nɔ́-á mɛ plɛ ásõ̂ dɛ́! Wàh kɩ mɛ ké tyɩ́, páh kɩ yèpyenɔ tɔ̃́nɔ́ pɩ: Núkú-á kɩ pɩ ǹmɔ tyɩ́lɛ, núkú kɩ pɩ Moyisi tyɩ́lɛ, núkú mó pɩ Yeli tyɩ́lɛ. 
Wǎh mɛ képah yónɔ́ sɔkɔ, ńyĩ́rkì yɔ́ núkú wil ńvyɔ̃́ɔ̃ à fob mɔ pé yõ. Tɛ́ yĩ́npɔ̃́ yɔ́ mɛ wil ńyĩ́rkì mɛ̀ sɔkɔ nɛ: «Nɛ̂-á yɔ̀, ńmɔ nɩ̀dɛ̀ Pídĩ́ ye. Ǹmɔ ye ń nɩ̀vɩ̀nɩ̀ yõlɛ. Yé nɛ n nohnɩ ǹ tyɩ́.» 
Yesu tõ̀ sã́hpú nɔ́pi-á képah tɛ̃̀ noh ńyãh sɔkɔ, pè syɩkɩlɩ pé yah di sétáh tíkí tɛ̃̀ tyɩ́. 
Képah sɔkɔ, Yesu mɛ nɔ de pé tnɔ̂ à tũ pé tyɩ́, tɛ́ yo pé tyɩ́ nɛ: «Yé yuku, yé káh n yɛ tíkí yé tɛ̃.» 
Pǎh pé yey yɔ̃, pè Yesu ó wɛ ǹ gblɔ̌y núkúnínɛ. 
Pǎh pé syɩ yɔ̃lɩlɛ n tiki, Yesu mɛ pè sni nɛ: «Yáh nɛ̂ wɛ, yé káh kè yãh nɛy tyɩ́, fɔ́ɔ́ kè syi n pɔ n nɔ Névi Pídĩ́ lékó yílkínmɔnɔ tyɩ́.» 
Képah sɔkɔ, ǹ tõ̀ sã́hpú mɛ à piki yah nɛ, ǹtɛ sɔ̃́ se pɩ tɛ́ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú wɔ nɛ ńkɛ̃́ Yeli ǹgbò-á yɩ̃nɛ à pɔ Liyel nɛy yáhntɔkɔnɔ yah nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Gbɩ ye Yeli ye yɩ̃nɛ à pɔ tyi pól tɔ̃ sah. 
Ǹtɛ mé kélɛ n yo yé tyɩ́, Yeli pɔ tɛ tɛ́ pè náh à yah pɩ̃. Pè pé dyɔ pɩ ǹ sɔkɔ. Sɛ̃́ ye Névi Pídĩ́ tɔ kɩ fṍhnɔ́ wɛ n pi pé gbõ̀.» 
Képah sɔkɔ, ǹ tõ̀ sã́hpú núkú pɩ̃ nɛ Wèlnwìlkì-ò Nsyɔ̃-á à n ye. 
Yesu ànɛ̂ ǹ tõ̀ sã́hpú nɔ́pi-á pɔ yĩni tɛ̃ tɔ́wû sɔkɔ gbĩ́ nɛ̂nɛ, dĩ́ yɔ́ nɔ de Yesulɛ à gbètukunɔ di ǹ yah, tɛ́ yo ǹ tyɩ́ nɛ, 
pé Tɛ̃̀ ǹgbɛ̃, à pé pídĩ́ yãm yah. Ǹdyɩ̀kɩ̀npɩ-á ǹnɛ, wǎh mɛ n kwlɔ kégbɩlɛ. Kèyǎhlɛ à n syɩkɩ n de nyàh sɔkɔ, képah náh pɩ ni sɔkɔ. 
Páh pɔ ǹnɛ ǹ tõ̀ sã́hpú tyɩ́, tɛ́ pè náh tɛ̃ wɛ à dɛ mɔ. 
Képah tɛ̃̀nɛ, Yesu wɛkɩ pé yõ nɛ: «Nɛ́pêl yísyɔ́lɛ n yah dɛ́, Liyel yõ sɛ̃́-ńkɛ̃̂pú nɔ́pi. Mé kɩ tɛ̃ yélɛ, pópó kɩ pɔ syɩ gbĩ́ nɛ̂ tɛ̃̀ tyɩ́? Mé kɩ yé sṍmnɔ́lɛ n pɩ, pópó kɩ pɔ syɩ gbĩ́ nɛ̂ tɛ̃̀ tyɩ́? Yé pɔ ǹnɛ ń kɔ̃ ásõ̂.» 
Pǎh pɔ ǹnɛ, Yesu dírí kõ sétõ yõ nɛ̂-á mɔ ǹ sɔkɔ ǹdyɩ̀kɩ̀nɩ̀nɛ, à mɛ wil dípír sɔkɔ. Nɛ́npɔ́ swɔ, dípír núkú dɛ. 
Képah tɛ̃̀nɛ, Yesu tõ̀ sã́hpú nɔ de Yesu tnɔ̂ pé yahlɛ tɛ́ à piki yah nɛ, sɔ̃́ se pɩ tɛ́ pépi wɔ sõ̀ tɛ̃ sétõ mɛ̀ yĩni wilki kwlɔ dípír mɛ̀ sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Yé Liyel yõ sɛ̃́nɔ́ yilɔ kah mɔ képah ye. Kègbɩ yõ mé kélɛ n yo yé tyɩ́, yé Liyel yõ sɛ̃́nɔ́-à mɛ nɔ mútárdɩ̀ pi fyélɛ, yé náhkɩ fɛ̃ yo yɔ̃lɩ mɛ̀ tyɩ́ nɛ kè yal ásõ̂ kè kyɩ n tɛ̃ yàhàh, tɛ́ kè náhkɩ mó tɔ̃ kè pɩ. Yé náh náhkɩ tir yɔ́ ńtɛ̃ pɩ n kwlɔ n pi dò.  
Ǹtɛ sétõ mɛ̀ dùkùlɛ, nɛ́ nírí pɩ́nɔ́ ànɛ̂ níkì tɛ̃́nɔ́ ó sɔkɔ náh, à náh fɛ̃ n yĩni.» 
Sõ yɔ́lɛ, Yesu ànɛ̂ ǹ tõ̀ sã́hpú sõ̀ mɛ tùkènmɔnɔlɛ Kalile mara sɔkɔ. À mɛ yo pé tyɩ́ nɛ: «Pè kɩ Névi Pídĩ́ tɛ̃ dahbɩ névye gbõ̀, 
pè kɩ à ko mɔ. Ǹtɛ ké sèpĩ̀n tɔ̃́-ò sõlɛ, ǹ lékó kɩ yilki mɔ.» Wɛlɔ nɔ́pi nóhnɔ́ ǹ tõ̀ sã́hpú yah kwɔ kègbɩlɛ. 
Yesupi-á kyɩ yĩni tɛ̃ Kapɛrnayumi kwil sɔkɔ, Liyel ní-ńsah gbô mɔ́rɩ́ syípú nɔ de Pyɛrɩlɛ tɛ́ à piki yah nɛ, pé kwɔ́-ò dĩ́ kó mɛ Liyel ní-ńsah gbô mɔ́rɩ́lɛ n mɔ nɩ? 
À mɛ nɛ àwe-á, wǎh mɛ kélɛ n mɔ kɛ̀. Pyɛrɩ-á kyɩ yĩni tɛ̃ gbô, Yesu ǹgbò à piki yah nɛ: «Simo-Pyɛrɩ, áwɔ sõnɔ tɛ̃̀ nɛ̂nɛ à yuku mɔ́rɩ́ mɔ́nɔ́ tyɩ́ sɔkɔ? Kàh mɛ mɔ́rɩ́lɛ nɛ̂-á kɩ kèkõyṍ yõ̀tãmnɛ n kɔ̃, kwil wɛ̃́npì se kɩ n mɔ kélɛ, tĩ̂, nɛ́pân se?» 
Pyɛrɩ mɛ nɛ nɛ́pân-á. Yesu mɛ nɛ: «Kàh pɩ sɛ̃́, kwil wɛ̃́npì wɔ vyãh náh mó mɛ ké mɔ́nɔ́ sɔkɔ kɛ̀. 
Ǹtɛ mé náh mɛ ké tyɩ́ nɛ no nɔ́pi kélɛ nɛ n sõ kétúkùyɔ̂. Képah ye, kyɩ dlóy mɔ Kalile yɔ́-ǹgbɛ̃ sɔkɔ, yɛ̃́nkɔ̂ nɛ̂-á kɩ pɔ tɛ̃ gbennɛ, ǹ vyãh yɩkɩ n mɔ n yah, á kɩ pɔ́ wɛ ké sɔkɔ. Kè wilki n kyɩ mɔ́rɩ́ mɔ pé tyɩ́, ńmɔ ànɛ̂ áwɔ tyɩ́lɛ.»
