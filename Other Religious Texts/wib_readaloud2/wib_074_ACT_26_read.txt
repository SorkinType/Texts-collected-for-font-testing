Tyípɩ́nɩ̀.
26.
Képah sɔkɔ, yõ̀tɛ̃̀ dĩ́ Akripa yo Pole tyɩ́ nɛ, wɛy-á mɛ ǹ tyɩ́, wǎh kɩ fɛ̃ ǹ vyãh kɔ̃lɩ. Wǎh yo sɛ̃́, Pole mɛ ǹ gbõ̀ yɔ̃, tɛ́ tyah ǹ vyãh kɔ̃̀lɩ̀nɔ́lɛ pé tyɩ́ nɛ ńkɛ̃́, 
yõ̀tɛ̃̀ dĩ́ Akripa, ǹmɔ-á pépi Nsyifunɔ tyídî póllɛ n pnɛ ànɛ̂ tyi nónó yõ-á pě yékã̂blɛ n pɩ. Képah-á, nɩ̀vɩ̀nɩ̀ ǹgbɛ̃-á kélɛ péwɔ tyɩ́ kwɛynɛ, kɩ pé vyãh kɔ̃lɩ tyi nɔ́pi pól sɔkɔ, Nsyifunɔ-á n tɔkɔ pélɛ n kɔ̃ tyi nónó sɔkɔ. Páh ǹnɛ n ni, à ǹ gblɔ̌y yɛ, à mó noh pé tyɩ́. 
À tɔkɔ pé kètɛ̃̀npìkì gbĩ́ dùkù tyɩ́, páh sõ̀ mɛ pé minnɛ n pɩ dùkù nɛ̂nɛ pé kwil tãn wrɔ́ ànɛ̂ Yerusalɛmɩ sɔkɔ, Nsyifunɔ pól-á képahlɛ n pnɛ. 
Pǎh pélɛ n pnɛ di mónɔ́lɛ. Pàh kɩ mɛ ké tyɩ́, pǎh kɩ fɛ̃ pé kèdùkù yo ńtɛ̃nɛ nɛ, Farisiyɛ̃ névye tĩ̀nkì sɔkɔ nɛy-á pélɛ, tĩ̀nkì nɛ̂ Liyel wɛ̃̀kɩ̀ tyɩ́-á kah tah. 
Ǹtɛ Liyel-á ǹ vyãh tɔkɔ pé náh tãn tyɩ́ nɛ̂ tyɩ́ sɔkɔ, péwɔ gbõ̀ sàhnɔ́-á mɛ ké yõ, képah yĩn-á pélɛ tukey sɔkɔ núkúnúkú yɔ̀. 
Tɛ́ pépi yìkìnì gbɔ̃ninɔ́ gbõ̀ sàhnɔ́-á mɛ ké yõ nɛ, kǎh kɩ n pɩ n dal n mɔ n pi, tɛ́ mɛ gbah tɛ̃ à Liyellɛ n gbilki ké lékã̂h, ké gbĩ̀ntɔ̃̀. Képah yõ gbõ̀ sàhnɔ́ yĩn-á Nsyifunɔ tɔkɔ péwɔ kɔ̃ yɔ̀, yõ̀tɛ̃̀ dĩ́ Akripa. 
Ǹtɛ kwâh nɛ̂ se pɩ, tɛ́ ké fɛ̃̀nsyinɔ tɔ̃ kah tah pé no Nsyifunɔ tyɩ́ nɛ, Liyel-á lékyɩ̂lɛ n yilki n mɔ nɩ? 
Péwɔ ńtɛ̃ tɔ-á sõ̀ mɛ n sõ nɛ, páh yɩ̃nɛ pé tyi pól pɩ, pé Nasarɛtɩ kwil Yesu yĩn yõ sɛ̃́nɔ́ yah kõ. 
Sárká syínwìlkìpu yõ̀tãm-á ké wɛ̃̀kɩ̀lɛ pé kɔ̃ tɛ gbĩ́ nɛ̂nɛ, páh Yesu yõ sɛ̃́pú náhnáh tɛ̃ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. Tɛ́ pàh sɔ̃́ sõ̀ pélɛ n ko n mɔ fyélɛ, péwɔ kè fɛ̃ n syi. Képah-á pé pɩ Yerusalɛmɩ kwil. 
Páh sõ̀ mɛ n de n kɔ̃ pépi Nsyifunɔ Liyel ní-ńtã̂n vɩ̀ sɔkɔ, à Yesu yõ sɛ̃́púlɛ n tɛ̃ n fõh, tɛ́ mɛ ké tyɩ́ nɛ pé pè kyɩ̃ mɔ, pè Yesu yõ sɛ̃́nɔ́ yal mɔ. Tɛ́ pé fɔkɔ vyã́hnkàhnmɔnɔ tɛ̃̀ tyɩ́ pélɛ, páh ńkɛ̃́nɛ pé nyáh sõ ǹ kyɩ pélɛ n fõh kwlókáhkɩ́nɔ tɔ sɔkɔ. 
Sɛ̃́-á sõ yɔ́lɛ, sárká syínwìlkìpu yõ̀tãm mɛ tɛ̃́nwɛnɔ ànɛ̂ wɛ̃̀kɩ̀lɛ pé kɔ̃, pé yuku à n yuku Damasɩ kwil. 
Ńkɛ̃́, yõ̀tɛ̃̀ dĩ́ Akripa, páh sõ̀ mɛ wɛ̃̀kɩ̀ yõ gbĩ̀ntɔ̃̀ pɔ̃́lɛ, tɛ́ mɛ kyòo yɔ́ wɛ kè wil yĩ̂nyõ, nɛ̂-á piki kal gbĩnɛ, à pɔ kwɛ péwɔ ànɛ̂ pé wɛ̃̀kɩ̀ kɔ́mû yõ pěrr, à pé mɔ gbɩ̃. 
Képah tɛ̃̀nɛ, pé pól syɩkɩlɩ sétáh tɛ́ péwɔ mɛ yĩ́npɔ̃́ yɔ́ noh kè mɛ n yo pé tyɩ́ Yebre wɛy sɔkɔ nɛ ńkɛ̃́: «Sole, Sole, kwâh nɛ̂ pɩ tɛ́ á ńnɛ n fõh sɛ̃́? Ǹgbǒ gbõ̀ nɛ̂-à yahle náh vìnɔ́lɛ, képah ye n kwɔ dɛ́!» 
Kǎh pɩ sɛ̃́, pé mɛ piki yah nɛ, nɛ̂ se ǹmɔlɛ nɩ, Tɛ̃̀ ǹgbɛ̃? À mɛ yo pé tyɩ́ nɛ: «Yesu ye ńmɔlɛ, nɛ̂-á á n fõh. 
Ǹtɛ yuku n yĩn á gbyẽ̂h yõ. Máh ń gblɔ̌ynɛ ǹ wɛ̃kɩ tir nɛ̂ yĩnnɛ, képah wɛ yɔ̀: Mé nɛ á pɩ ń tõ̀npɩlɛ. Sɔ̃́-á á wɛ ńnɛ yɔ̀, á kɩ kélɛ n yah n yo n kɔ̃, à tahlɩ tyi nónó tɔ-á mé kɩ pɔ ǹ wɛ̃kɩ n pi náhlɛ. 
Mé kɩ ǹ pwah mɔ wilki Nsyifunɔ ànɛ̂ kwlókáhkɩ́nɔ névye gbõ̀, pépi nónó tyɩ́-á mé n tɛkɩ n mɔ álɛ. 
Mé álɛ n tɛkɩ n mɔ pé tyɩ́ nɛ, á kyɩ pé yɩ̃́ yɩkɩ, pè kɔ̃ pè wil fɛ̃́ntũ̂ sɔkɔ pè pɔ kyòo yah. Pè mó wil sétã̀n yõ̀tɛ̃̀ yɔ̃rɩ sɔkɔ pè pɔ Liyel tyɩ́, pé tyípêl kɔ̃ sè yɔ̃ mɔ pè kɔ̃, pè mó kyɩ tɛ̃̀-ǹsah wɛ Liyel wɛ̃́npì wrɔ́, nónó-á mɛ Liyel tyɩ́ yõ ńmɔ yõ sɛ̃́nɔ́ yĩnnɛ.» 
Pole-á Tɛ̃̀ ǹgbɛ̃ Yesu wɛlɔ nɔ́pi yo tɛ, à ǹ syɩ yo Akripa tyɩ́ nɛ ńkɛ̃́, yõ̀tɛ̃̀ dĩ́, páh tyi nónó noh, tɛ́ tyi nónó wɛ wɛ̃́kɩ́nɔ́ mɛ̀ sɔkɔ nɛ̂-á n wil Liyel tyɩ́, pé náh yahle sé pɩ́nɔ́lɛ. 
Tɛ́ Damasɩ tãn ǹgbò tyɩ́-á pé yo sélɛ. Képah náh, pé mɛ sè yo Yerusalɛmɩ tãn tyɩ́, ànɛ̂ Nsyude gbɛ̃̀nyah tãn póllɛ, à tahlɩ kwlo kwlo tɛ́lɔ́ tyɩ́. Páh kè yo pé pól tyɩ́ nɛ, pè pé nnɔ vi mɔ, pè mó Liyel wɛ̃̀kɩ̀ tɛ̃. Ńkɛ̃́, pè tyípéplɔlɛ n pɩ, nónó-á n wɛ̃kɩ nɛ gbɩ yõ, pé nnɔ-á vi mɔ. 
Tɛ́ képah yõ sɔkɔ-á Nsyifunɔ tɛ̃ pélɛ, gbĩ́ nɛ̂-á sõ̀ pé mɛ Liyel ní-ńsah gbô sɔkɔ. Pǎh sõ̀ mɛ n yah n kɔ̃ pé kònmɔ-ńsahlɛ. 
Ǹtɛ Liyel yòhnɩ̀nmɔnɔ gbõ̀ yõ, páh mɛ min sɔkɔ à pɔ syɩ kwɛy tyɩ́. Tɛ́ páh tɔ̃ ásõ̂ sɔkɔ yɔ̀, gblɔ yo yĩnnɛ tahbɩ ànɛ̂ yãm tãn tyɩ́. Liyel tɛ̃ntõ̀ syínyopu ànɛ̂ Moyisi-á tyi nónó tyɩ́ yo nɛ, sǎh kɩ pɔ n pɩ n pi, pé náh tir yɔ́ ńtɛ̃ yo kah mɔ sépi yõ, sépi nónó-á yo nɛ ńkɛ̃́: 
Liyel nɛy yáhntɔkɔnɔ-á kɩ yèvyãhrɩ pɩ kɩ ku. Tɛ́ ǹmɔ-á kɩ n pɩ n pi nɛ́gbennɛ, kɩ yilki mɔ lékyɩ̂ sɔkɔ. Wǎh kɩ pɔ pwáhnmɔnɔ tyɩ́ yo wah kyòo pépi Nsyifunɔ tyɩ́ ànɛ̂ névye tɛ́lɔ́ tyɩ́. 
Tɛ́ Pole-á mɛ ǹ tyi yah yónɔ́ sɔkɔ sɛ̃́ mɛ ǹ vyãhlɛ n kɔ̃lɩ, Fɛsitusi wɛkɩ páhpáh ǹ yõ nɛ ńkɛ̃́, ǹ yõ̀gbèkè náh mɛ, ǹ kǎrn pɩ́nɔ́ képékè-á ǹ yõ̀gbèkè yuku. 
Ǹtɛ Pole mɛ yo ǹ tyɩ́ nɛ ńkɛ̃́, yõ̀tɛ̃̀ pèpɛy, pé yõ̀gbèkè náh yuku dɛ́! Gblɔ ànɛ̂ yèsrey wɛlɔ ó kɔ́kɔ́-á pé n wɛkɩ. 
Ńkɛ̃́, tyi nɔ́pi sɔkɔ yɔ́ ńtɛ̃ náh pɩ sètĩ̀n sõ̀ sɔkɔ, pé sɛ̃́nɔ́-á mɛ ké yõ nɛ, yõ̀tɛ̃̀ dĩ́ Akripa ńtɛ̃ pɩ̃́nɔ́-á mɛ sé sɔkɔ dò. Képah-á, pé kɔ̃ pé kélɛ n yo n dal n mɔ ǹ yah sɔkɔ nɩ̀sãhlɛ. 
Ǹtɛ yõ̀tɛ̃̀ dĩ́ Akripa, à kó sɛ̃ Liyel tɛ̃ntõ̀ syínyopu wɛlɔ yõ nɩ? Péwɔ tɛ̃̀-á pɩ̃ nɛ wǎh sɛ̃ sé yõ. 
Képah sɔkɔ, Akripa mɛ yo Pole tyɩ́ nɛ, kàh náh kɩ nɛ pé mo kwéy, wǎh náhkɩ núkú péwɔ vi mɔ pɩ Yesu yõ sɛ̃́-òlɛ. 
Pole mɛ yo ǹ tyɩ́ nɛ, kàh kɩ se pɩ núkúnúkú ò, kàh kɩ se mo ò, péwɔ tɛ̃̀-á mɛ Liyellɛ n ni nɛ, kè káh yĩn ǹmɔ ńtɛ̃ ó tyɩ́, ńkɛ̃́ nónó pól tɔ-á mɛ pé wɛlɔlɛ n nohnɩ kwɛynɛ yɔ̀, pé pól pɩ péwɔ tyɩ́ kɔ̃lɛ, Yesu yõ sɛ̃́púlɛ, tɛ́ kârn wɔ káh mɔ pépi tyɩ́. 
Képah náh, yõ̀tɛ̃̀ dĩ́ Akripa yuku, ànɛ̂ gbɛ̃̀nyah yah tɛ̃́-ò dĩ́ Fɛsitusilɛ, ànɛ̂ Berenisilɛ, à tahlɩ nónó pól-á sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ pélɛ. 
Ǹtɛ pǎh wil, pè tyah n yo wɛ̃ tyɩ́ nɛ, dĩ́ mɛ̀ náh tir yɔ́ ńtɛ̃ pɩ, nɛ̂-á yɩ̃nɛ ǹ kònmɔnɔlɛ, képah náh pɩ, ǹ dáhnmɔnɔlɛ wɔ̀kɔ̀npèkè sɔkɔ. 
Képah sɔkɔ, Akripa mɛ yo Fɛsitusi tyɩ́ nɛ, dĩ́ mɛ̀ à sõ̀ mɛ ǹ tukey tyɩ́ dáhbɩ́-ńkɛ̃̂nɛ Wrome kwíltãm yõ̀tɛ̃̀ Sesar gbõ̀ sɔkɔ, wǎh yɩ̃nɛ fĩ́nmɔnɔlɛ.
