Nsyɔ̃.
4.
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú sõ̀ noh nɛ Yesu-á mɛ tõ̀ sã́hpúlɛ n wɛ, mɛ pélɛ n wel n wilki ni sɔkɔ à kal Nsyɔ̃nɛ. Tɛ́ Yesu ńtɛ̃ náh sõ̀ n wel n wilki névyelɛ ni sɔkɔ ǹ gblɔ̌y dɛ́, ǹ tõ̀ sã́hpú ye sõ̀ n pɩ ké tõ̀lɛ. Képah sɔkɔ, Yesu-á pɔ képah yónɔ́ noh gbĩ́ nɛ̂nɛ, à yal Nsyude gbɛ̃̀nyah sɔkɔ ànɛ̂ ǹ tõ̀ sã́hpúlɛ, à ǹ syɩ n yuku Kalile gbɛ̃̀nyah sɔkɔ. 
Tɛ́ képah wɔ sɔkɔ, à sõ̀ yɩ̃nɛ à Samari gbɛ̃̀nyah kɔ kah. 
Wǎh n yuku, à kyɩ yĩni tɛ̃ Samari kwil yɔ́ sɔkɔ, pè n ye Sikarlɛ. Kénɛ kwil mɛ Nsyosɛfu swãh tnɔ̂, nɛ̂-á lésõ ǹ sú Nsyakɔbɩ à kɔ̃. 
Tɛ́ Nsyakɔbɩ kwlɛ̀y sõ̀ mɛ nɛ́npɔ́. Yesu-á wil kɔ́lɔ́ sɔkɔ tɛ́ kwlɔ, à sõ̀ mɛ kyɩ kɔ̃ tɛ̃ kénɛ kwlɛ̀y yõ. Tɛ́ gbĩ sõ̀ mɛ yõ̀tah sírr kénkɔ̃ sɔkɔ. 
Képah sɔkɔ, Samari pi yɔ́-á pɔ yɔ dí-ńsah, Yesu mɛ à syah nɛ à ninɛ pé kɔ̃ pé wɔ. 
Tɛ́ légbĩ́nɛ, Yesu tõ̀ sã́hpú sõ̀ mɛ sɔkɔ yõke yáhndwe-ńsah kwil nɩyṍ. 
Képah sɔkɔ, Samari pi mɛ à syah nɛ, kè se pɩ sɔ̃́ tɛ́ ǹmɔ nɛ̂-á Nsyifulɛ, à kɩ pɔ wɔ wɔ ninɛ n ni péwɔ Samari nɛy tyɩ́ nɩ? Kègbɩ tɛ̃̀ yõ Nsyifunɔ náh ńtyɩ̃́nɔ sõ̀ n pɩ n soh wɛ̃ tyɩ́ ànɛ̂ Samari tãnnɛ. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́ nɛ: «Liyel-á nɛ̂nɛ névilɛ n kɔ̃, wâh sõ̀ mɛ kélɛ n pnɛ, tɛ́ pɩ̃ nɛ́dúkú nɛ̂-á n yo n mɔ á tyɩ́ yɔ̀, áwɔ ye náhkɩ sõ̀ ni wɔ wɔ ninɛ ǹ tyɩ́, tɛ́ à náhkɩ ninɛ ǹ kɔ̃ nɛ̂-á minnɛ névilɛ n kɔ̃.» 
Képah sɔkɔ, pi nɛ pé yõ̀tɛ̃̀, kwlɛ̀ndìkì náh mɛ ǹmɔ tyɩ́, tɛ́ kwlɛ̀y-á mó tɔ̃ mɛ súkúlínɔ́lɛ, à se náhkɩ sõ̀ wɛ ńyãh kénɛ ninɛ nɩ? 
Ńkɛ̃́ pépi náh nɛy Nsyakɔbɩ nɛ̂-á pé kɔ̃ kwlɛ̀y mɛ̀nɛ, tɛ́ ké ni wɔ sɔkɔ ànɛ̂ ǹ wɛ̃́npìlɛ, à tahlɩ ǹ yípɛ̂y tyɩ́, ǹmɔ se kal Nsyakɔbɩ mɛ̀nɛ nɩ? 
Yesu mɛ nɛ: «Névi nɛ̂-à kwlɛ̀y mɛ̀ ni wɔ, níkì kɩ tɔ̃ pɔ tɛ̃̀ tɛ̃. 
Ǹtɛ ńmɔ-à ninɛ névi nɛ̂ kɔ̃, tɛ̃̀-à kè wɔ, níkì náh tɔ̃ tɛ̃̀ tɛ̃ n yah n pi póllɛ. Kè kɩ n pɔ n mɔ lègbèr tyɩ́ kɔ̃lɛ tɛ̃̀ sɔkɔ, kɩ tyɛ́-ńkɛ̃̂ minnɛ n kɔ̃.» 
Képah sɔkɔ, pi à syah nɛ pé yõ̀tɛ̃̀, à pé kɔ̃ kénɛ ninɛ, níkì kɔ̃ kè káh tɔ̃ pélɛ n tɛ̃, ànɛ̂ pé káh tɔ̃ pɔ ásõ̂ yɔ tɛ̃̀ di yĩnnɛ. 
Wǎh yo sɛ̃́, Yesu mɛ nɛ, à kyɩ ǹ pɛ ye pè pɔ ásõ̂. 
Képah sɔkɔ, pi nɛ pɛ náh mɛ pé tyɩ́. Yesu mɛ nɛ, gbɩ-á à yo nɛ pɛ náh mɛ pé tyɩ́. 
Pɛbɩ nɛ́pĩ̂ kwâl tyɩ́-á à tɛ̃ yah sɔkɔ, tɛ́ nɛ̂ tyɩ́-á tɔ̃ ǹnɛ núkúnúkú yɔ̀, gbɩ-á à yo, ǹ pɛ náh ǹnɛ. 
Képah tɛ̃̀nɛ, pi mɛ yo Yesu tyɩ́ nɛ pé yõ̀tɛ̃̀, páh kè kyɔmɩ wɛ nɛ, Liyel tɛ̃ntõ̀ syínyo-ò-á ǹnɛ. 
Ńkɛ̃́ pépi náh tãn-á Liyellɛ n gbilki ńyãh sɔkɔ, nɛ́npɔ́-á mɛ yɔ̃lɩ mɛ̀ yõ yàhàh. Ǹtɛ pépi Nsyifunɔ wɔlɛ, pépi n nɛ n yo nɛ, Liyel-á yɩ̃nɛ à nɛ n gbilki ńyãh sɔkɔ, nɛ́npɔ́-á mɛ Yerusalɛmɩ sɔkɔ. 
Képah sɔkɔ, Yesu mɛ yo pi tyɩ́ nɛ: «Sɛ̃ ń yõ. Sõ yɔ́ mɛ n pi, nɛy náh tɔ̃ pɔ Sú Liyel gbilki n pi yɔ̃lɩ mɛ̀ yõ, képah náh pɩ, Yerusalɛmɩ ńtɛ̃ sɔkɔ. 
Yépi Samari tãnnɛ, yě Liyellɛ n gbilki tɛ́ yé náh ǹnɛ n pnɛ. Ǹtɛ ápi Nsyifunɔ tyɩ́ ye Liyel syɔ̃ à pwáhnmɔnɔlɛ n kɔ̃. Képah ye á ǹnɛ n pnɛ, tɛ́ ǹnɛ gbilki. 
Ǹtɛ gbĩ yɔ́ mɛ n pi, nɛ̂-á mɛ tɛ núkúnúkú ńtɛ̃nɛ yɔ̀. Képah sɔkɔ ye Sú Liyel gbílkípú kègblɔ kɩ ǹnɛ n gbilki ǹ Mirki gbõ̀ yõ sɔkɔ ànɛ̂ gbɩ wɛ̃̀kɩ̀ yõ. Gbílkínɔ́ mɛ̀ dùkù pɩ́pú névye ye Sú Liyel n nɛ n yah n kɔ̃. 
Liyellɛ, Mirki ye ǹnɛ. Képah ye, nónó-á ǹnɛ n gbilki, pè yɩ̃nɛ pè ǹnɛ n gbilki ǹ Mirki gbõ̀ yõ, ànɛ̂ gbɩ wɛ̃̀kɩ̀ yõ.» 
Képah sɔkɔ, pi à syah nɛ, páh pɩ̃ nɛ Liyel nɛy yáhntɔkɔnɔ-á kɩ n pɔ n pi, nɛ̂-á pè n ye Kristalɛ. Ǹmɔ-à pɔ gbĩ́ nɛ̂nɛ, wǎh kɩ pɔ tyi pól dùkù yo wah pé tyɩ́. 
Yesu mɛ nɛ: «Ǹmɔ ye ńmɔlɛ, nɛ̂-á n yo n mɔ á tyɩ́ yɔ̀.» 
Wǎh mɛ képah yónɔ́ sɔkɔ, ǹ tõ̀ sã́hpú mɛ pé syɩ pɔ de. Tɛ́ pǎh pɔ à wɛ à mɛ n yãh ànɛ̂ pilɛ sɛ̃́, kè pé gbã̀n du. Ǹtɛ pé sɔkɔ yɔ́ ńtɛ̃ náh fɛ̃ syi à piki yah nɛ, kwâh nɛ̂ se à n yah n kɔ̃ pi mɛ̀ tyɩ́, képah náh pɩ, tir nɛ̂ se à n yo ǹ tyɩ́ nɩ? 
Képah tɛ̃̀nɛ, pi ǹ ni pĩ́n sah nɛ́npɔ́, tɛ́ to kyɩ kwil nɩyṍ, à kyɩ tyah n yo no tyɩ́ nɛ, 
pè pɔ nɛ n yah dĩ́ yɔ́lɛ, à péwɔ tyípɩ́nɩ̀ pól yo pé tyɩ́ lɛ́! Ǹmɔ káh kɩ nɛ̂ Liyel nɛy yáhntɔkɔnɔlɛ dɛ́? 
Képah sɔkɔ, névye mɛ wil kwil nɩyṍ mɛ n yuku Yesu tnɔ̂. 
Tɛ́ légbĩ́nɛ, Yesu tõ̀ sã́hpú sõ̀ mɛ Yesulɛ n ni nɛ, kwɔ́-ò dĩ́, à yõke yõ kɛ̀! 
Ǹtɛ Yesu mɛ pè syah nɛ, yõke yɔ́-á mɛ péwɔ tyɩ́ yõ yõ tɛ̃̀nɛ, nɛ̂-à pépi náh kélɛ n pnɛ. 
Képah sɔkɔ, ǹ tõ̀ sã́hpú sõ̀ mɛ wɛ̃nɛ n piki n yah nɛ, ǹnɩ̂ se kɩ mɛ pɔ yõkelɛ à kɔ̃ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Liyel nɛ̂-á tɛkɩ mɔ ńmɔlɛ, wǎh mɛ nɛ̂ tyɩ́, képah pɩ́nɔ́ ànɛ̂ ǹ tõ̀ pɩ́ntɛnɔ ye pɩ ńmɔ yõkelɛ. 
Yépi náh ńkɛ̃́nɛ nɛ n yo nɛ, ńwɔ̃́lɔ́ ńyã́h-á tɔ̃ wũ tɛ́ sèmukɔ tɔkɔ gbĩ́ kɩ nɔ di? Ǹtɛ ńmɔ wɔ kɩ yo yé tyɩ́ nɛ, yé yé yey yɔ̃ n yah sèmukɔlɛ kòntɛ̃̀nɔ́lɛ swãh sɔkɔ, tɔkɔ tɔkɔ tɛ̃̀nɛ. 
Névi nɛ̂-à mɛ sèmukɔlɛ n tɔkɔ, ǹ tõ̀ pɩ́nɔ́ lédyáhnɩ́ mɛ tɛ ǹ tyɩ́; sèmukɔ pɔ́rɩ́ nɛ̂-á à n tuke n mɔ, névye ye nónó-á kɩ tyɛ́-ńkɛ̃̂ min wɛ n pi. Képah sɔkɔ, névi nɛ̂-á mɔ sélɛ, ànɛ̂ névi nɛ̂-á sè tɔkɔ, pé pól kɩ nɩ̀vɩ̀nɩ̀ pɩ wɛ̃ tyɩ́. 
Képah n wɛ̃kɩ nɛ, gbɩ-á sèmìrkì mɛ̀nɛ, nɛ̂-á n yo nɛ ńkɛ̃́: ‹Ǹnɩ̂ kɩ sèmukɔ mɔ, tɛ́ ǹnɩ̂ kɩ sè tɔkɔ.› 
Yépi náh swãh nɛ̂ tõ, ńmɔ yé tɛkɩ mɔ nɛ, yé kyɩ képah sèmukɔ tɔkɔ. Nósyɔ́ ye tõ̀ pɩ ké sɔkɔ, yépi kɔ̃ yé pɔ wɛ de pé tõ̀ pɩ́nɔ́ yén sõ̀.» 
Tɛ́ pi mɛ̀-á sõ̀ wɛy nɛ̂ yo nɛ, dĩ́ mɛ̀-á péwɔ tyípɩ́nɩ̀ pól yo pé tyɩ́, képah yĩnnɛ Samari tãn náhnáh sõ̀ sɛ̃ Yesu yõ kwil mɛ̀ sɔkɔ. 
Képah ye Samari tãn nɔ́pi-á kyɩ Yesu tyɩ́, pè à ni nɛ à wũ pé tyɩ́. Yesu mɛ wũ nɛ́npɔ́ à sèpĩ̀n nínì pɩ. 
Képah tɛ̃̀nɛ, névye náhnáh tɔ̃ sɛ̃ Yesu yõ wǎh wɛlɔ nónó yo pè noh sépi yĩnnɛ. 
Tɛ́ mɛ tyah n yo pi mɛ̀ tyɩ́ nɛ, ǹmɔ yóré ó yĩn náh tɔ̃ pépi sɛ̃ Yesu yõ dɛ́! Ńkɛ̃́ pépi ńtɛ̃ tɔ-á ǹ wɛy wɛ noh níkílɛ, tɛ́ pɩ̃ nɛ kègbɩ tɛ̃̀ yõ, ǹmɔ-á ńtyɩ̃́nɔ kèkõyṍ pwáhnmɔ-òlɛ. 
Yesu-á sèpĩ̀n nínì nɔ́pi pɩ tɛ, à yal nɛ́npɔ́ à yuku Kalile gbɛ̃̀nyah sɔkɔ. 
Tɛ́ à mó tɔ̃ lésõ kè yo nɛ, Liyel tɛ̃ntõ̀ syínyo-ò náh n tɛ̃ plɛ ǹ sú kwil sɔkɔ. 
Ǹtɛ Kalile tãn tɔ-á lésõ kyɩ Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh sɔkɔ Yerusalɛmɩ sɔkɔ, tɛ́ Yesu tyípɩ́nɔ́ wɛ sɛ̃́, képah ye wǎh pɔ pɔ pé gbɛ̃̀nyah sɔkɔ, pè mɛ à tɔ̃ sah tɛ̃. 
Képah sɔkɔ, Yesu sɔkɔ kénɛ gbɛ̃̀nyah kwil mɛ̀ sɔkɔ, nɛ̂-á pè n ye Kanalɛ. Nɛ́npɔ́ sɔkɔ ye lésõ à ni vi mɔ pɩ dìvɛ̃nɛ. Tɛ́ nɛ́-ǹgbɛ̃ yɔ́ lésõ mɛ yõ̀tɛ̃̀ dĩ́ yɔ́ tyɩ́ Kapɛrnayumi kwil sɔkɔ. Ǹ pídĩ́ náh lésõ mɛ lènɔlɛ. 
Wǎh pɔ noh nɛ, Yesu-á mɛ wil Nsyude gbɛ̃̀nyah sɔkɔ à pɔ Kalile sɔkɔ, à mɛ ǹ nyáh sõ kyɩ à ni nɛ, à tiki pɔ Kapɛrnayumi sɔkɔ, à pɔ pé pídĩ́ dɛ mɔ pé kɔ̃; wǎh mɛ n tyah kúnɔ́lɛ. 
Yesu mɛ yo ǹ tyɩ́ nɛ, pè se yɔ́ kɩ fɛ̃ sɛ̃ yah péwɔ yõ; nɛ́ gbǐl ànɛ̂ gbɛ̃̀ndùnì tyi náh pɩ pè wɛ nɩ? 
Ǹtɛ dĩ́ mɛ tɔ̃ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, à tiki pɔ súú péwɔ tĩ̀nnɛ pé dípír káh pɔ ku. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ, à ǹ syɩ sɔkɔ ǹ tĩ̀nnɛ, ǹ pídĩ́-á dɛ. Dĩ́ mɛ sɛ̃ ǹ wɛy mɛ̀ yõ, tɛ́ ǹ syɩ. 
Wǎh mɛ wɛ̃̀kɩ̀ yõ, ǹ tõ̀npɩbɩ pɔ à yohnɩ, tɛ́ yo ǹ tyɩ́ nɛ, ǹ pídĩ́-á dɛ. 
À mɛ pè piki yah nɛ, gbĩ́ nɛ̂ se fàhfàh de ǹ tyɩ́ nɩ? Pè mɛ yo ǹ tyɩ́ nɛ, ńnɩ́nɩ́ gbĩ̀ntɔ̃̀ pɔ̃́ sɔkɔ-á wílvyã́hnɩ̀ à fĩn mɔ. 
Képah sɔkɔ, dípír sú kè kyɔmɩ wɛ nɛ légbĩ́ mɛ̀-ó sɔkɔ-á Yesu yo pé tyɩ́ nɛ, pé nɛ n yuku, pé pídĩ́-á dɛ. À tɔkɔ légbĩ́ tyɩ́, ǹmɔ ànɛ̂ ǹ gbô tãn pól sɛ̃ Yesu yõ. 
Tɛ́ Yesu-á wil Nsyude gbɛ̃̀nyah sɔkɔ, à pɔ Kalile gbɛ̃̀nyah sɔkɔ, képah ye à pɩ gbǐl ní-òlɛ.
