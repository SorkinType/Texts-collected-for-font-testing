Nsyɔ̃.
10.
Képah náh, Yesu tɔ̃ yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, névi nɛ̂-à n dé-ńkɛ̃̂nɛ képyɛ́nɩ̀ tũ̀ yahlɛ, tɛ́ n dɔkɔ n de ké sɔkɔ ké gbɔ̀pɔ yɔ́lɛ, yɛ̃́nwó ye tɛ̃̀nɛ, névi yõ syɩ́kɩ́-ò tɔ̃ ǹnɛ. 
Ǹtɛ nɛ̂-à n de ké yahlɛ, képyɛ́nfɛ̃ ye tɛ̃̀ wɔlɛ. 
Tũ̀ syɩ́kɩ́-ò dĩ́ ké yah yɩkɩ n mɔ ǹmɔ kɔ̃. Ǹ képyɛ́nɩ̀ ǹ yĩ́nɛ n pnɛ. Wàh de, ǹ tyɩ́-á képyɛ́nɩ̀ nónónɛ, ǎ sé ye n wil sé yninɛ kégbɔ́. 
Ǹ képyɛ́nɩ̀-á ǹ yĩ́nɛ n pnɛ sɛ̃́, képah ye wàh sé pól wilki mɔ tɛ, à n yĩn sé yah, tɛ́ sě ǹ sõ̀ tɛ̃. 
Ǹtɛ sàh névi nɛ̂ yĩ́nɛ n pnɛ̀-ǹkɛ̃̂nɛ, sè náh tɛ̃̀ sõ̀lɛ n tɛ̃ póllɛ. Sè n to n gbɛ n mɔ fɩ́ɩ́ ǹnɛ ńtɛ̃nɛ.» 
Yesu kénɛ sèmìrkì yãh tɔ́wû tyɩ́, ǹtɛ pè náh ké kɔ́r pɩ̃. 
Képah sɔkɔ, Yesu tɔ̃ yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, ńmɔ ye képyɛ́nɩ̀ tũ̀ yahlɛ. 
Nónó pól-á ń dyàh tɛ̃ pɔ sɔkɔ tɛ, yɛ̃́nwóbé ànɛ̂ névye yõ syɩ́kɩ́pú ye pélɛ. Ǹtɛ, képyɛ́nɩ̀ náh mó fɛ̃ syi noh pé tyɩ́. 
Ńmɔ ye dénsahlɛ. Névi nɛ̂-à de képahlɛ, tɛ̃̀ kɩ pwah. Tɛ̃̀ kɩ fɛ̃ n de, tɛ́ kɩ ǹnɛ n syɩ n wil kɩ ǹ yõkelɛ n wɛ. 
Tɛ́ yɛ̃́nwó wɔ-à n pi, yɩ̃be, ànɛ̂ nɛ́kókè mɔ́nɔ́, ànɛ̂ yényêh pɩ yĩn ó ye à n pɔ. Ǹtɛ ńmɔ pɔ nɛ névye min wɛ, nɛ̂-á plɛ yĩ́ĩ́. 
Képyɛ́nfɛ̃ pèpɛy ye ńmɔlɛ. Kàh sɔ̃́ n nɔ kũ tyɩ́, képyɛ́nfɛ̃ pèpɛy n fɛ̃ n syi ǹ min kɔ̃ ǹ képyɛ́nɩ̀ yĩnnɛ. 
Ǹtɛ pɔ́ yáhnkɔ̃-ò képyɛ́nfɛ̃ tɛ̃̀ wɔlɛ, képyɛ́nfɛ̃ kègbɩ náh ǹmɔlɛ dɛ́! Ǹmɔ yɛ̃́ngbɛ̃́ tyɩ́ náh képyɛ́nɩ̀ nɔ́pilɛ. Képah ye, wàh fyɔ̀ gbɛ̃́npã̂h wɛ à n pi, à n to ǹ tɛ́ képyɛ́nɩ̀ milki, à kɔ̃ à n pɔ n tɛ̃ sé sɔkɔ, sé tɛ́lɔ́ n mó n to n karkɩ. 
Pɔ́ ó kɔ́kɔ́ ye ǹmɔ n yah n kɔ̃ dò. Yɩ́kɩ́nɔ́-à sɔ̃́ n pɩ képyɛ́nɩ̀ sɔkɔ, kè náh n pɩ tirlɛ ǹmɔ yah. 
Képyɛ́nfɛ̃ pèpɛy ye ńmɔlɛ. Sɔ̃́-á Sú Liyel ńnɛ n pnɛ, tɛ́ ń tɔ ǹnɛ n pnɛ, sɛ̃́ ó ye, mé ń képyɛ́nɩ̀nɛ n pnɛ, sé tɔ mó ńnɛ n pnɛ. Ńmɔ kɩ ń min kɔ̃ ń képyɛ́nɩ̀ yĩnnɛ. 
Képyɛ́nɩ̀ túkù mɛ ń tyɩ́, nónó-á sè náh mɛ tũ̀ mɛ̀ sɔkɔ. Sè kɩ n noh n pi ń tyɩ́. Mé yɩ̃nɛ mè n kyɩ sè yah n fɛ̃ fɛ̃ n pɔ n kwrɔ n mɔ sè pɩ tũ núkúlɛ, képyɛ́nfɛ̃ núkú ó mó sélɛ n fɛ̃. 
Ńmɔ kɩ ń min kɔ̃ kũ sɔkɔ, tɛ́ kɩ tɔ̃ pɔ kè syi n pi. Képah ye, ń tyɩ́ nɔ Sú Liyel tyɩ́. 
Nɛy náh fɛ̃ ń ko n mɔ n pi dɛ́! Ńmɔ ye n kɔ̃ ń minnɛ ń gblɔ̌y. Tɛ̃́nwɛnɔ mɛ ń tyɩ́ kɩ fɛ̃ kè kɔ̃, tɛ̃́nwɛnɔ tɔ̃ mɛ ń tyɩ́ kɩ fɛ̃ kè syi ńtɔ̃. Sɛ̃́ ye ń Sú Liyel nɛ mé pɩ.» 
Képah tɛ̃̀nɛ, Nsyifunɔ-á wɛlɔ nɔ́pi noh sɛ̃́, pé sõnɔ tɔ̃ kah pɩ sé tyɩ́ tyɩ́lɛ. 
Pé náhnáh tyah n yo nɛ ńkɛ̃́, sétõ yɔ́-á mɛ ǹ sɔkɔ, ǹ yõ̀gbèkè kɔ̃ kè yuku. Yõ se tɔ̃ mɛ ké tyɩ́, pè ǹ wɛlɔlɛ n nohnɩ nɩ? 
Ǹtɛ túkù mó nɛ ńkɛ̃́, yõ̀gbèkè yúkú-ò wɛlɔ náh wɛlɔ nɔ́pilɛ dɛ́! Yõ̀gbèkè yúkú-ò se kɩ fɛ̃ yɩ̃́tũ̂ yɩ̃́ yɩkɩ mɔ nɩ? 
Képah sɔkɔ, kwɔ̀vɩrɩ gbĩ́nɛ, syáh yɔ́ sõ̀ mɛ n syah Yerusalɛmɩ sɔkɔ, nɛ̂-á Nsyifunɔ dyɔlɛ n to n kõ Liyel ní-ńsah gbô tɔ̃́nsàhnsɔkɔnɔ tyɩ́lɛ. 
Képah sɔkɔ, Yesu sõ̀ mɛ n kɔ n kɔ̃ Liyel ní-ńsah gbô párn yɔ́ sõ̀, nɛ̂-á pè n ye Salomo párnnɛ. 
Nsyifunɔ yísyɔ́ mɛ pɔ à mɔ gbɩ̃ tɛ́ yo ǹ tyɩ́ nɛ, à se kɩ pépi mɔ sah tɔ̀tɔ̀ sɔkɔ, pópó kɩ pɔ nɔ gbĩ́ nɛ̂ tyɩ́ nɩ? Ǹmɔ-à kɩ nɛ̂ Liyel nɛy yáhntɔkɔnɔlɛ, à kè yo n dal n mɔ pé syah kɛ̀! 
À mɛ yo pé tyɩ́ nɛ: «Mé kè yo tɛ yé tyɩ́, ǹtɛ yé náh sɛ̃ ké yõ. Tyi nónó-á mé mɛ n pɩ ń Sú Liyel yĩn yõ, sépi ye mɛ ń tyɩ́lɛ n yo n wah. 
Ǹtɛ yé náh pɩ ńmɔ képyɛ́nɩ̀nɛ sɛ̃́, képah ye yé kɔ̃ yé náh sɛ̃ ké yõ. 
Ńmɔ képyɛ́nɩ̀ tɛ̃̀ wɔ ń wɛynɛ n nohnɩ, mé sélɛ n pnɛ, ànɛ̂ sě ń sõ̀ tɛ̃. 
Mé kɩ tyɛ́-ńkɛ̃̂ minnɛ sè kɔ̃. Sé tyɩ́ náh n yɩkɩ n yah n pi, ànɛ̂ ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ sè tɛ̃ n syi ń gbõ̀. 
Ńmɔ Sú nɛ̂-á sélɛ ń gbõ̀ dahbɩ, ǹmɔ kal yíyìn póllɛ. Képah ye, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ sè tɛ̃ n syi ǹmɔ gbõ̀ sɔkɔ. 
Ńmɔ ànɛ̂ Sú Liyel mɛ núkúlɛ.» 
Képah tɛ̃̀nɛ, Nsyifunɔ tɔ̃ dyêh tõ nɛ páh Yesulɛ n min n ko. 
Ǹtɛ à mɛ yo pé tyɩ́ nɛ, Sú Liyel gbõ̀ yõ, páh tyípéplɔ náhnáh pɩ pè wɛ. Sépi sɔkɔ nɛ̂ tɛ̃̀ yĩn se pè n yah n kɔ̃ pé mínnkò-ǹsahlɛ nɩ? 
Nsyifunɔ mɛ à syah nɛ, tyípépɛ̌y pɩ́nɔ́ yĩn náh pé nɛ páh ǹnɛ n min n ko dɛ́! Ǹtɛ wǎh yo de Liyel tyɩ́ képah yĩn-á. Ǹmɔ névi gbɔy kɩ pɔ ǹ gblɔ̌ynɛ n pɩ Liyellɛ nɩ? 
Képah sɔkɔ, à yo pé tyɩ́ nɛ: «Kè náh mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ, Liyel-á nɛ líyelnɔ́-á yélɛ di? 
Liyel-á yo mɔ névye nónó tyɩ́, à pè ye líyelnɔ́lɛ. Ǹtɛ yé se pɩ̃ nɛ Liyel wɛy sɛ́bɛ́ynɔ́-á nɛ̂ yo, kè náh fɛ̃ n milki n pi tɛ́! 
Tɛ́ Sú Liyel-á ńmɔ nɛ̂ wilki sah ǹ tyɩ́ yõ, tɛ́ ń tɛkɩ mɔ kèkõyṍ, yé nɛ máh yo de Liyel tyɩ́, máh sɔ̃́npɩ́lɛ yo nɛ Liyel Pídĩ́-á ńnɛ nɩ? 
Máh kɩ mɛ ń Sú Liyel tyípɩ́nɩ̀nɛ n pɩ́-ńkɛ̃̂nɛ, yé káh sɛ̃ ń yõ. 
Ǹtɛ máh kɩ mɛ sélɛ n pɩ, yáh kɩ sɔ̃́ mɛ n sɛ̃ n pi ńkɛ̃̂nɛ ńmɔ yõ, yé sɛ̃ sépi tɛ̃̀ wɔ yõ; képah sɔkɔ yé kɩ pɩ̃ yĩ́ĩ́ nɛ, máh mɛ núkúlɛ Sú Liyellɛ, ànɛ̂ Sú Liyel-á mɛ núkúlɛ ńnɛ.» 
Képah tɛ̃̀nɛ, pè tɔ̃ nɛ páh Yesulɛ n tɛ̃. Ǹtɛ à pwah pé tyɩ́. 
Képah náh, Yesu tɔ̃ ǹ syɩ yuku Nsyurdɛ̃ yɔ́nkahnɩnɛ, à kyɩ tɛ̃ ńyãh sɔkɔ-á Nsyɔ̃ ǹgbò tyah névye wèlnwílkìnɔ́lɛ. 
Névye náhnáh mɛ kyɩ ǹ tnɔ̂, tɛ́ mɛ yo tũ nɛ, Nsyɔ̃ náh gbǐl tir yɔ́ ńtɛ̃ pɩ. Ǹtɛ wǎh tyi nónó pól yo dĩ́ mɛ̀ tyɩ́ sɔkɔ, gblɔ-á sélɛ. 
Képah sɔkɔ, névye náhnáh mɛ sɛ̃ Yesu yõ nɛ́npɔ́.
