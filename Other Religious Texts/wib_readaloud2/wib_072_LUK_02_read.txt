Likyi.
2.
Kénɛ tyi pɩ gbĩ́nɛ, Wromɛnɔ yõ̀tɛ̃̀ Wokusti kwní kɔ̃ nɛ, pè pé tɔ̃́rɩ́ mɛ́-ńtã̂n névye pól yãh pé dan pɩ̃. 
Kénɛ névye yã́hnɔ́ gben pɔ Krinisi wɛ Siri kwlo pól yah yétɛ̃̀nɛ. 
Képah tɛ̃̀nɛ, no pól sõ̀ mɛ pé yninɛ n wãrkɩ n pɔ pé súbí kwlo sɔkɔ. 
Tɛ́ Yosɛfulɛ, yõ̀tɛ̃̀ dĩ́ Davidi náh nɛy ye sõ̀ ǹnɛ. Kǎh pɩ sɛ̃́, ǹ tɔ mó wil Nasarɛtɩ kwil, Kalile sétáh yõ, à yuku Yude kwil yɔ́ sɔkɔ, pè n ye Betlɛmɩnɛ, yõ̀tɛ̃̀ dĩ́ Davidi dúbí-ńsah. 
Tɛ́ Yosɛfu kyɩ Mari sõ̀ mɛ kúrkìlɛ, pǎh n yuku pé yni wã́rkɩ́-ńsah gbĩ́ nɛ̂nɛ. 
Pǎh kyɩ yĩni tɛ̃, pè sõ̀ mɛ sɛ̃́ nɛ́npɔ́, tɛ́ Mari dubi sṍ nɔ. 
Képah sɔkɔ, pè tɛ̃̀-ǹsah yah kwlɔ nɛ́pân twáh-ńsah, tɛ́ kyɩ tɛ̃ yípɛ̂y sɛ̃́-ńsah. À ǹ pígben dubi dĩ́npirlɛ, tɛ́ à plo fwɔ gbrólɛ, à snɛkɩ mɔ yípɛ̂y yõ yõ dú yɔ́ sɔkɔ. 
Légbĩ́nɛ, Betlɛmɩ kwil gbɔ̀pɔ yɔ́lɛ, képyɛ́nfɛ̃bɩ yísyɔ́ sõ̀ mɛ n sõ gbãn sɔkɔ, à pé képyɛ́nɩ̀nɛ n syɩkɩ. 
Pè mɛ tɛ̃̀nɔ́lɛ, tɛ́ yĩ̂nyõ tɛ̃ntõ̀ yɔ́ pɔ wil mɔ pé yõ. Nɛ́npɔ́ swɔ, Liyel tɔ̃́rɩ́ núkú tyah pól piki pěrr pé yõ. Tíkí mɛ pè kah tɛ̃. 
Yĩ̂nyõ tɛ̃ntõ̀ mɛ yo pé tyɩ́: «Yé káh tíkí pɩ! Kwil pól nɩ̀vɩ̀ tir pèpɛy yɔ́ ye mé pɔ yó-ńsah yé tyɩ́. 
Piri yɔ́ mɛ dubi yé tyɩ́ kwɛy lékã̂hnɛ Davidi kwil sɔkɔ. Ǹmɔ ye TƐ̃̀ ǸGBƐ̃NƐ pè n ye Kristalɛ, Liyel-á nɛ, páh kɩ nɛ̂ tɛkɩ mɔ. Ǹmɔ ye tɔ̃ yé pwáhnmɔ-òlɛ. 
Tɛ́ yáh nɛ pɩ sɔ̃́, tɛ́ à yah n pɩ̃ n pi, kè wɛ yɔ̀: Yáh kɩ kyɩ píyĩ́rí nɛ̂ wɛ plónɔ́lɛ flɔlɛ, tɛ́ mɛ snɛ́kɩ́nmɔnɔlɛ yípɛ̂y yõ yõ dú sɔkɔ, ǹmɔ ye.» 
Nɛ́npɔ́ swɔ, yĩ̂nyõ tɛ̃ntõnè yísyɔ́ wil kwĩnkilɛ yĩ̂nyõ, à pɔ gben tɛ̃̀ nɔ, pè tyah Liyel yĩnnɛ n gbilki yɩ̀gbɩ̀nɩ̀ sɔkɔ, mɛ yo tũ nɛ ńkɛ̃́: 
«Liyel yĩn pɩ gblò yĩ̂nyõ. Ǹmɔ ye ǹ yèvɩnɩnɛ ǹ nɩ̀vɩ̀ névye kɔ̃ sétáh yõ.» 
Képah náh, yĩ̂nyõ tɛ̃ntõnè-á pé syɩ sɔkɔ yĩ̂nyõ, képyɛ́nfɛ̃bɩ nɔ́pi mó n yo wɛ̃ tyɩ́ nɛ, pé no, pè yâwɔ pé kyɩ yah Betlɛmɩ. Tir nɛ̂-á pɩ nɛ́npɔ́, tɛ́ Liyel ǹ tɛ̃ntõ̀ tõ pé tyɩ́, páh yɩ̃nɛ pé kè wɛ. 
Pǎh wɛkɩ wɛ̃ kɔ̃ sɛ̃́, pè pé kyãh yuku Betlɛmɩ. Pè píyĩ́rí mɛ̀ soh kyɩ wɛ snɛ́kɩ́nmɔnɔlɛ yípɛ̂y yõ yõ dú sɔkɔ, Mari ànɛ̂ Yosɛfulɛ ǹ gbɔ̀pɔ. 
Pǎh à wɛ sɛ̃́, yĩ̂nyõ tɛ̃ntõ̀-á tyi nónó pól yo pé tyɩ́ ǹ tyɩ́ sɔkɔ, pè sè yãh pè syah. 
Nónó pól-á képyɛ́nfɛ̃bɩ wɛlɔ nɔ́pi noh, kè pé gbã̀n du. 
Mari tɛ̃̀ mó tyi nɔ́pi pól syi sah ǹ fɔkɔlɛ n sõ. 
Képah náh, képyɛ́nfɛ̃bɩ pé syɩ. Yĩ̂nyõ tɛ̃ntõ̀-á sõ̀ yo sɔ̃́ pé tyɩ́, kè mó ńtyɩ̃́nɔ pɩ sɛ̃́. Képah ye, pǎh tyi nónó pól noh tɛ́ tyi nónó pól wɛ, sépi pè kɔ̃ pè Liyel yĩnnɛ n gbilki, ǹ pal tyɩ́lɛ n yãh n yuku wɛ̃̀kɩ̀ yah. 
Képah náh, piri dúbínɔ́ sèpĩ̀n kwǎrtɔ̃́-ò sṍlɛ, ǹ kõ gbĩ́ nɔ. Pè mɛ ǹ yĩn sah Yesulɛ, sɔ̃́-á sõ̀ yĩ̂nyõ tɛ̃ntõ̀ yo sah kélɛ ǹ kúrkì tɔ́kɔ́-ńkɛ̃̂ gbĩ́nɛ. 
Képah sɔkɔ, sédám pɛ̀kɩ̀ kṹnɔ́ nɛ̂-á yɩ̃nɛ kè pɩ Moyisi tyi kõ̀nsàhnɔ́-á kè yo sɔ̃́, ké sèpĩ̀n-á tũ, Yosɛfu ànɛ̂ Mari Yesu tɛ̃ yuku Yerusalɛmɩ, à kyɩ ǹnɛ Liyel gbõ̀ dahbɩ. 
Kè ye mɛ wã́rkɩ́nsàhnɔ́lɛ Tɛ̃̀ ǹgbɛ̃ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ nɛ ńkɛ̃́, pye gbro dĩ́npì pól-á yɩ̃nɛ pè Tɛ̃̀ ǹgbɛ̃ Liyel gbõ̀ dahbɩ. 
Ànɛ̂ tɔ̃ ǹkànpĩ̀nnɔ́ nínɔ́lɛ, pɛ̀kɩ̀ ànɛ̂ yũ̌kìlɛ Liyel kɔ̃ sárkálɛ, képah náh pɩ, kèpíkípíkínɔ nínɔ́lɛ, sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Tɛ̃̀ ǹgbɛ̃ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ. 
Tɛ́ légbĩ́nɛ, dĩ́ yɔ́ sõ̀ mɛ Yerusalɛmɩ sɔkɔ, pè n ye Simeyɔ̃lɛ. À sõ̀ mɛ nɛ́gbɩ́lɛ Liyel yah, tɛ́ mɛ n sah ǹ yóré yõ. À sõ̀ mɛ Yisrayel pwáhnmɔ-ò pɔ́nɔ́lɛ n syɩkɩ, tɛ́ Liyel Mírkí sõ̀ mɛ ǹ sɔkɔ, 
tɛ́ kélɛ à wɛ̃kɩ nɛ, à náh n ku n pi, nɛ́ à náh Krista wɛ yĩ́nɛ, nɛ̂-á Tɛ̃̀ ǹgbɛ̃ Liyel kɩ tɛkɩ mɔ n pi. 
Képah sɔkɔ, Liyel Mírkí ǹ yah tɛ̃ yuku Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ. Tɛ́ Yesu dúbípú sõ̀ mɛ à tɛ̃ pɔ tyi kõ̀nsàhnɔ́ pɩ́ntũ yĩnnɛ ǹ tyɩ́, kǎh yo sah sɔ̃́. 
Képah tɛ̃̀nɛ, Simeyɔ̃ mɛ piri mɛ̀ tɛ̃ ǹ gbõ̀, tɛ́ Liyel yĩnnɛ n gbilki, mɛ yo tũ nɛ: 
«Ń Tɛ̃̀ ǹgbɛ̃ Liyel, mé pwáhnmɔ-ò wɛ ń yĩ́nɛ, wáh nɛ̂ kɔ̃ kèkõyṍ névye pól yĩ́ yah. Képah ye, á kɩ fɛ̃ á tõ̀npɩ min syi núkúnúkú à kɔ̃ à nɛ n yuku yèvɩnɩ sɔkɔ, sɔ̃́-á á wɛy kè yo. 
Pwáhnmɔ-ò kɩ pɩ kyòolɛ kwlo kwlo névye tyɩ́, kɩ á dùkùlɛ pè wɛ̃kɩ, tɛ́ kɩ pɩ yõ yɔ̃̀nɔ́lɛ áwɔ no Yisrayel tãn tyɩ́.» 
Kǎh pɩ sɛ̃́, tyi nónó-á à yo Yesu tyɩ́ sɔkɔ, sè pɩ gbɛ̃̀ndùnìnɛ ǹ sú ànɛ̂ ǹ yṹ tyɩ́. 
Képah sɔkɔ, Simeyɔ̃ mɛ sãm mɔ pé sɔkɔ, tɛ́ yo ǹ yṹ Mari tyɩ́ nɛ: «Yah, piri mɛ̀ mɛ Yisrayel névye náhnáh syɩ́kɩ́nmɔnɔ ànɛ̂ pé náhnáh yílkínsàhnɔ́ yĩnnɛ, ànɛ̂ yékã̂b dénɔ́ yõlɛ. 
Kǎh pɩ sɛ̃́, névye náhnáh fɔ́kɔ́sɔ̃nɔ kɩ wilki mɔ kwèyy. Tɛ́ áwɔ tɛ̃̀ wɔlɛ, á kɩ yèvyãhrɩ pɩ, á kɩ nɛ pè sɔ̃́ mɛ á nɩlɛ n sukɛ nyàhlɛ.» 
Tɛ́ Liyel tɛ̃ntõ̀ syínyo-ò pi yɔ́ sõ̀ mɛ, pè n ye Annɛ. Fanuwel písê ye sõ̀, nɛ̂-á Asɛrɩ yìkì nɛynɛ. Ǹ yah sõ̀ wah náhnáh. Ǹ sépíl gbĩ́nɛ, wǎh de pɛ tyɩ́, ǹ pɛ yah kwlɔ ké ye kwǎrní-ò sɔkɔ. 
À mɛ tɛ̃ pɛ́kúsèlɛ, à pɔ nɔ ye tíkíyẽ́h pɔ́ ńyã́h tyɩ́. À náh sõ̀ mɛ n gbɛ n mɔ Liyel wɔ̀kɔ̀ ǹgbɛ̃nɛ, tɛ́ mɛ Liyel yĩnnɛ n gbilki gbĩ́mɛgbĩ́, ké lékã̂h ké gbĩ̀ntɔ̃̀, níkì tɛ̃́nɔ́ ànɛ̂ nírí sɔkɔ. 
Ǹmɔ tɔ mɛ de kyɩ yɩ̃nɛ Yesu ànɛ̂ ǹ dúbípúlɛ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ. À tyah Liyel yĩnnɛ n gbilki. Tɛ́ nónó-á sõ̀ mɛ Yerusalɛmɩ pwáhnmɔnɔ syɩkɩ tɛ̃, à mɛ piri tyɩ́lɛ n yãh pé pól tyɩ́. 
Yesu dúbípú-á Liyel tyi sàhnɔ́ pɩ tũ tɛ, képah náh, pè pé syɩ sɔkɔ Kalile sétáh yõ, pé kwil Nasarɛtɩ sɔkɔ. 
Tɛ́ piri wɔ mó mɛ n kyãh, mɛ tǎhkɩ̀lɛ n wɛ. Lékã́m ǹgbɛ̃ sõ̀ mɛ ǹ sɔkɔ, tɛ́ Liyel mɛ ǹ nɩlɛ n gbɛ. 
Ye ye póllɛ, Yesu dúbípú ńkɛ̃́nɛ nɛ n kyɩ n pɔ Yerusalɛmɩ, Pakɩ syáh sɔkɔ. 
Tɛ́ Yesu-á pɔ nɔ ye gbɔ̃ninɔ́lɛ gbĩ́ nɛ̂nɛ, ǹ dúbípú dɔkɔ yuku ǹnɛ Yerusalɛmɩ, Nsyifunɔ-á sõ̀ kélɛ n pɩ sɔ̃́. 
Syáh-á pɔ tyɛ sṍ nɛ̂nɛ, pè pé syɩ sɔkɔ, tɛ́ pé pídĩ́ Yesu mɛ wũ Yerusalɛmɩ, ǹ dúbípú pɩ̃́-ńkɛ̃̂ sɔkɔ. 
Pè sõ̀ mɛ n sõ nɛ, wǎh mɛ ànɛ̂ pé no wɛ̃̀kɔbɩ yísyɔ́lɛ. Kǎh pɩ sɛ̃́, pè gbĩ̀yĩ̀kì núkú kɔ́lɔ́ kɔ, tɛ́ pyě tyah ǹnɛ n yah n kɔ̃ pé wɔ̀kɔ̀ tãn ànɛ̂ pé nɛ́pɩ̃́nɔ́ wrɔ́. 
Ǹtɛ, pè náh à wɛ. Pǎh à yah kwlɔ sɛ̃́, pè pé syɩ à yah kɔ̃ yuku Yerusalɛmɩ sɔkɔ. 
Ké sèpĩ̀n tɔ̃́-ò sṍlɛ, pè pyě kyɩ à wɛ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ. À mɛ kɔ̃́ntɛ̃̀nɔ́lɛ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú wrɔ́, mɛ n nohnɩ pé tyɩ́, tɛ́ mɛ pélɛ n piki n yah tyilɛ. 
Nónó pól-á sõ̀ mɛ n nohnɩ ǹ tyɩ́, ǹ lékã́m ànɛ̂ ǹ tyi yah yónɔ́ dùkù pé gbã̀n du. 
Ǹ dúbípú-á à wɛ sɛ̃́, kè pépi tɔ gbã̀n kah du. Ǹ yṹ mɛ yo ǹ tyɩ́ nɛ, pé pídĩ́, sɔ̃́ se pɩ, tɛ́ à pɩ sɛ̃́ pé tyɩ́ nɩ? À yah, péwɔ ànɛ̂ ǹ sú yáh-ńsah-á tyɛ kègbɩlɛ ǹ yáhnkɔ̃nɔ sɔkɔ. 
Ǹtɛ, à mɛ yo pé tyɩ́: «Sɔ̃́ pɩ, tɛ́ yé ńnɛ n yah n kɔ̃? Yé náh pɩ̃ nɛ, máh yɩ̃nɛ mé nɛ n mɛ ń sú tyah sɔkɔ di?» 
Ǹtɛ, pè náh mɛ ǹ wɛy mɛ̀ tyɩ́ yah wɛ. 
Képah náh, à mɛ yuku pélɛ Nasarɛtɩ kwil sɔkɔ, tɛ́ mɛ ǹ gblɔ̌ynɛ n tiki n mɔ pé tyɩ́. Ǹ yṹ wɔ mó tyi nɔ́pi pól syi sah ǹ fɔkɔ. 
Tɛ́ Yesu sõ̀ mɛ n kyãh ǹ gbɛ̃́nsê ànɛ̂ ǹ lékã́m mɛ n yɩkɩ n mɔ n yuku. Tɛ́ wãn tɔ mɛ mɔ ǹ nɩ̀gbɛ̀kɩ̀ yõ Liyel ànɛ̂ névye tyɩ́.
