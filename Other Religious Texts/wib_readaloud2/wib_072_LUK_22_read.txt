Likyi.
22.
Búrn sɛ̃̀-ǹkɛ̃̂ syáh nɛ̂-á pè n ye Pakɩlɛ, képah déntɛnɔ sɔkɔ, 
sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú sõ̀ mɛ syɔ̃̀-ǹsahlɛ n yah n kɔ̃, pǎh kɩ pɩ sɔ̃́, tɛ́ Yesu ko mɔ n pi. Tɛ́ névye yah tíkí sõ̀ mɛ pè tɛ̃. 
Tɛ́ légbĩ́nɛ, sétã̀n yõ̀tɛ̃̀ de Yudasɩ sɔkɔ nɛ̂-á pè n ye Yisikariyɔtɩlɛ. Yesu tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ nɛy yɔ́ ye sõ̀ ǹnɛ. 
À mɛ sɔkɔ, à kyɩ yo sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kyɔ̀mɩ̀pu yõ̀tãm tyɩ́, wǎh nɛ Yesu tɛ̃ pè kɔ̃ n pi dùkù nɛ̂nɛ. 
Képah sɔkɔ, kè nɔ pé tyɩ́. Pè mɛ yo sah núkú yõ nɛ ńkɛ̃́, páh kɩ pɔ́lɛ à kɔ̃. 
Yudasɩ mɛ fɛ̃ syi, tɛ́ tyah ké wɛ́npɩ-ńsahlɛ n yah n kɔ̃, wǎh pɩ sɔ̃́, tɛ́ Yesu dahbɩ n pi pé gbõ̀, névye pɩ̃́-ńkɛ̃̂ sɔkɔ. 
Búrn sɛ̃̀-ǹkɛ̃̂ syáh sṍ-á pɔ nɔ, nɛ̂ sɔkɔ-á sõ̀ Pakɩ képyɛ́nɩ̀ kɩ n ko n pi, 
Yesu mɛ Pyɛrɩ ànɛ̂ Nsyɔ̃ tɛkɩ mɔ nɛ, pè kyɩ Pakɩ syáh yõke tɛ̃̀-ǹsah yah sah yõ yõ tɛ̃̀nɛ. 
Pè mɛ à piki yah nɛ, ńyãh sɔkɔ se à mɛ ké tyɩ́ nɛ, pé kyɩ kè pɩ nɩ? 
À mɛ yo pé tyɩ́: «Yé yah, yáh kɩ n de kwil sɔkɔ gbĩ́ nɛ̂nɛ, yé kɩ dĩ́ yɔ́ yohnɩ, à kɩ mɛ ni pĩ́n twah. Yé ǹ sõ̀ tɛ̃. Wàh kɩ kyɩ de kéte nɛ̂ sɔkɔ, yé de nɛ́npɔ́. 
Yé mó yo kététɛ̃̀ tyɩ́ nɛ, kwɔ́-ò-á yo nɛ ńkɛ̃́, wɔ̀kɔ̀ nɛ̂ sɔkɔ se pé kɩ pɔ yõke yõ n pi ànɛ̂ pé tõ̀ sã́hpúlɛ nɩ? 
Képah sɔkɔ, à kɩ yĩ̂nyõ wɔ̀kɔ̀ sɔkɔ táh ǹgbɛ̃ yɔ́lɛ yé wɛ̃kɩ, gbõ̀ vi vyɔ́ póllɛ kɩ sɔkɔ. Nɛ́npɔ́ sɔkɔ ye, yé kɩ pɩ Pakɩ syáh yõkelɛ.» 
Képah tɛ̃̀nɛ, pè mɛ sɔkɔ. Yesu-á kè yo sɔ̃́ pé tyɩ́, pè mó ńtyɩ̃́nɔ kyɩ ké pól wɛ sɛ̃́. Pè mɛ Pakɩ syáh yõke pɩ nɛ́npɔ́. 
Yõke yõ gbĩ́-á pɔ nɔ, Yesu ànɛ̂ ǹ tɛ̃̀ntõbe-á pɔ tɛ̃ yõke yah sɔkɔ gbĩ́ nɛ̂nɛ, 
à mɛ yo pé tyɩ́: «Mé sõ̀ mɛ ké tyɩ́ náhnáh nɛ, mé Pakɩ yõke mɛ̀ yõ yélɛ, mé mó pyě de yèvyãhrɩ sɔkɔ. 
Ńkɛ̃́, mé kélɛ n yo yé tyɩ́ nɛ: mé náh tɔ̃ Pakɩ yõke mɛ̀ yõ n yah n pi fɔ́ɔ́, kɩ kyɩ syɩ ké tyi pɩ́ntũnɔ tyɩ́ Liyel yõ̀tɔ̃rɩ́ sɔkɔ.» 
Képah sɔkɔ, à mɛ dìvɛ̃ wɔ wɔ kwâh tɔkɔ dìvɛ̃nɛ ké sɔkɔ, à wɛkɩ Liyel tyɩ́, tɛ́ yo pé tyɩ́: «Mé kélɛ n yo yé tyɩ́ nɛ: à tɔkɔ núkúnúkú tyɩ́, mé náh tɔ̃ dìvɛ̃ wɔ n yah n pi fɔ́ɔ́, kɩ pɔ syɩ Liyel tɔ̃́rɩ́ pɔ́nɔ́ tyɩ́. Képah ye, yé syi kélɛ, yé wɔ dibi wɛ̃ tyɩ́.» 
Képah náh, à mɛ búrn tɔkɔ, tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ kè kwɔkɩ dibi pè kɔ̃, tɛ́ yo pé tyɩ́: «Ńyɔlɛ, ńmɔ wil ye, nɛ̂-á kɩ kɔ̃ yépi yĩnnɛ. Yé sɛ̃́ntɛ̃̀nɛ n pɩ, sɔ̃́-á mé n pɩ kélɛ. Képah sɔkɔ, yé dyɔ kɩ n kwnɛ ńmɔ tyɩ́lɛ.» 
Yõke yṍntɛnɔ náh, à tɔ̃ dìvɛ̃ tɔkɔ, à pè kɔ̃ sɛ̃́ntɛ̃̀ ólɛ, tɛ́ yo pé tyɩ́: «Ńyɔlɛ, ńmɔ tṍ ye, nɛ̂-á kɩ kwɛ yépi yĩnnɛ, kɩ pɩ yégbékè kwrɔ́nmɔnɔ késã́nnɛ Liyel ànɛ̂ névye wrɔ́. 
Ǹtɛ, yé yah, névi nɛ̂-á kɩ n vi n pi ńmɔ náh, ǹmɔ tɔ mɛ n yõ ńnɛ fɩ̃́ núkú. 
Névi Pi kɩ ku, sɔ̃́-á Liyel yah tɔkɔ kélɛ. Ǹtɛ, névi nɛ̂-á kɩ n vi n pi ǹ náh, pèkè kɩ pɩ ǹmɔ tyɩ́lɛ.» 
Képah tɛ̃̀ sɔkɔ, pè tyah wɛ̃nɛ n piki n yah nɛ, pépi tɛ̃̀ sɔkɔ nɛ̂ se kɩ n pɩ n pi képahlɛ nɩ? 
Képah náh, pè mɛ tyah, à yékã̂blɛ n pɩ. Pè sõ̀ mɛ ké tyɩ́ nɛ, pé pɩ̃ nɛ̂ tɛ̃̀-á kal pépi sɔkɔ. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́: «Kèkõyṍ kwlo yõ̀tãm ńkɛ̃́nɛ pé tɔ̃́rɩ́lɛ n sah sé yõ, ànɛ̂ tɔ̃́rɩ́-á mɛ nónó gbõ̀, pépi n nɛ mɛ ké tyɩ́ nɛ, no pé yĩ́ngbɩ́lɛ n yo. 
Ǹtɛ, yépi tɛ̃̀ tyɩ́ náh yɩ̃nɛ kè pɩ sɛ̃́. Nɛ̂-á yah tɛ̃̀nɛ yépi sɔkɔ, ǹmɔ pyě yɩ̃nɛ à pɩ yé pól gbɛ̃̀ sɔ̃́. Yõ̀tɔ̃rɩ́-á mɛ nɛ̂ gbõ̀, ǹmɔ pɩ yé pól tõ̀npɩ sɔ̃́. 
Yépi tɛ̃̀ tyɩ́, névi nɛ̂-á tɛ̃ yõke yah, ànɛ̂ nɛ̂-á n tɔkɔ n kɔ̃ kélɛ, nɛ̂ tɛ̃̀ kal pépi sɔkɔ? Nɛ̂-á tɛ̃ yõke yah, ǹmɔ náh di? Ǹtɛ, képah fyé yõ, ńmɔ mɛ yépi wrɔ́ sɔkɔ tõ̀npɩ sɔ̃́. 
Yépi tɛ̃̀ wɔlɛ, yépi ye sõ̀ mɛ ńmɔlɛ ń yèvyãhrɩ pɩ́-ńtã̂n sɔkɔ. 
Képah ye, ńmɔ sú-á tɔ̃́rɩ́lɛ ń kɔ̃ sɔ̃́, ńmɔ tɔ kɩ kélɛ yé kɔ̃ sɛ̃́. 
Képah sɔkɔ, yé kɩ pɔ n yõ, tɛ́ kɩ n wɔ ńmɔlɛ fɩ̃́ núkú ń yõ̀tɔ̃rɩ́ sɔkɔ. Ànɛ̂ yé kɩ pɔ kɔ̃ tɛ̃ yõ̀tɔ̃rɩ́ klǒ yõ, kɩ tukey tuke Yisrayel névye tĩ̀nnɔ gbɔ̃ninɔ́ yõ.» 
Képah náh, Yesu yo Simɔ Pyɛrɩ tyɩ́: «Simɔ, Simɔ! Sétã̀n yõ̀tɛ̃̀ ye ké wɛ̃̀kɩ̀ yah kɔ̃ nɛ, pé yé fɛ, sɔ̃́-á sèmukɔlɛ pè n fɛ gbɛ́y sɔkɔ. 
Tɛ́ mé mɛ Liyel ni á tyɩ́ nɛ, á Liyel yõ sɛ̃́nɔ́ káh tɛ̃ syi á tyɩ́. Tɛ́, wâh á syɩ pɔ ńmɔ tyɩ́ gbĩ́ nɛ̂nɛ, sṍrélɛ n mɔ yé no sɔkɔ.» 
Wǎh yo sɛ̃́, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, péwɔ tɛ̃̀ wɔlɛ, kàh kɩ sɔ̃́ n pɩ wɔ̀kɔ̀npèkè sɔkɔ dénɔ́lɛ, képah náh pɩ, kũnɛ, péwɔ-á mɛ pé vi sah kɩ de ǹnɛ sépi pól sɔkɔ. 
Ǹtɛ, Yesu mɛ yo ǹ tyɩ́: «Pyɛrɩ, mé kélɛ n yo á tyɩ́ nɛ: kwɛy lékã̂h sɔkɔ, sénpɛ náh tɔ̃̌nɩ̀ mɛ n gbo n pi, tɛ́ á kɩ mɛ á vyãh kɔ̃lɩ tɛ gbáhyɩ̃́ tɔ̃́nɔ́ nɛ ńkɛ̃́, á náh ńmɔ pɩ̃.» 
Képah náh, Yesu mɛ yo pé tyɩ́: «Ńmɔ-á sõ̀ yé tõ gbĩ́ nɛ̂nɛ, tɛ́ yo yé tyɩ́ nɛ, yé sɔkɔ pɔ́, ànɛ̂ pilki, ànɛ̂ lékyêh kɛ̃́nɛ, kwâh yɔ́ se kyɩ tɛkɩ yé tyɩ́?» Pè mɛ yo ǹ tyɩ́ nɛ, kwâh yɔ́ náh tɛkɩ. 
Képah sɔkɔ, à mɛ yo pé tyɩ́: «Mé kélɛ n yo yé tyɩ́ nɛ: wɛy nɛ̂-á wãrkɩ ńmɔ tyɩ́ sɔkɔ nɛ ńkɛ̃́, pè à mɔ kõ tyípêl pɩ́pú wrɔ́, képah kɩ n pɩ n tũ n pi. Tyi nónó-á yo sah ńmɔ tyɩ́ sɔkɔ, sé pɩ gbĩ́ mɛ nɔ n tɛ. Képah ye núkúnúkú tɛ̃̀ wɔlɛ, pɔ́-à kɩ mɛ nɛ̂ tyɩ́, tɛ̃̀ kè tɔkɔ. Ànɛ̂ képah núkú-ó ye: Pilki-à kɩ mɛ nɛ̂ tyɩ́, tɛ̃̀ kè tɔkɔ. Tɛ́ kèkírkì-à kɩ n knɛ nɛ̂ tyɩ́, tɛ̃̀ ǹ fwɔ̀mɔ̀-ǹgbɛ̃ yãm kèyǎh dwe.» 
Képah sɔkɔ, tõ̀ sã́hpú nɛ, à yah, kèkîrnnɔ nínɔ́-á yɔ̀. À mɛ nɛ, kǎh yɩ̃nɛ. 
Képah náh, Yesu mɛ wil sɔkɔ Wólívî yèh yɔ̃lɩ yõ, wǎh dar tɛ n kyɩ n pɔ sɔ̃́. Tɛ́ ǹ tõ̀ sã́hpú mɛ ǹ sõ̀ tɛ̃. 
Wǎh yĩni tɛ̃ nɛ́npɔ́ gbĩ́ nɛ̂nɛ, à yo pé tyɩ́ nɛ, pè Liyellɛ n ni, mɔ́nyahnɔ káh pɔ tɛ̃ wɛ pé tyɩ́. 
Képah náh, à gbɛ mɔ tɛ̃ pélɛ fɩ́ɩ́ kwéy, dáhkɩ̀ mínnɔ́ kɛ núkú yĩ́níntɛ̃-ńsah kénkɔ̃lɛ. Tɛ́ gbètukunɔ di, à nírílɛ n pɩ, 
mɛ n yo nɛ: «Bɔ̌, wáh kɩ mɛ ké tyɩ́, yèvyãhrɩ mɛ̀ wrɔ́ gbɛ mɔ ńmɔlɛ. Ǹtɛ, kè káh pɩ ńmɔ dyɔ dyɔ tɛ̃̀nɛ. Áwɔ dyɔ dyɔ tɛ̃̀ ó pɩ.»  
Légbĩ́nɛ, yĩ̂nyõ tɛ̃ntõ̀ yɔ́ wil ǹ yah, à pɔ tǎhkɩ̀lɛ à kɔ̃. 
Tɛ́ Yesu min sõ̀ mɛ yèvyã̀hnpèkè sɔkɔ. À tɔ̃ wãn mɔ Liyel nínɔ́ yõ. Ǹ wil myèh pɔ pɩ tṍkwîl sɔ̃́, à n kwɛ sétáh. 
Wǎh nírí pɩ tɛ gbĩ́ nɛ̂nɛ, à yuku kyɩ ǹ tõ̀ sã́hpú tnɔ̂. À kyɩ nɛ yah, pè mɛ n dɔ yèkwɔrɩ kwlɔ́nɔ́ tɛ̃̀ tyɩ́. 
À yo pé tyɩ́: «Kwâh nɛ̂ pɩ, tɛ́ yé n dɔ sɛ̃́? Yé yuku Liyellɛ n ni, mɔ́nyahnɔ káh pɔ tɛ̃ wɛ yé tyɩ́.» 
Yesu mɛ wɛ́kɩ́nɔ́ sɔkɔ sɛ̃́, tɛ́ Yudasɩ nɛ̂-á ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ yɔ́lɛ, ǹmɔ tɔ́wû yɔ́ yah tɛ̃ pɔ wil mɔ ǹ yõ. Tɛ́ nɔ de ǹnɛ nɛ, pé à pɔrɩ tɛ̃, pé wɛkɩ ǹ tyɩ́. 
Ǹtɛ, Yesu mɛ yo ǹ tyɩ́: «Yudasɩ, pɔ́rɩ́ntɛ̃nɔ sɔkɔ se á kɩ dɛ mɔ Névi Pi gbàhlɛ?» 
Nónó-á sõ̀ mɛ Yesulɛ, pǎh pɔ kè kyɔmɩ à pɩ̃ tir nɛ̂-á kɩ pɔ n pɩ n pi, pè yo Yesu tyɩ́ nɛ, pépi Tɛ̃̀ ǹgbɛ̃, pé se kèkîrnnɛ n dɛ no tyɩ́ nɩ? 
Tɛ́ sárká wìlkìpu yõ̀tɛ̃̀ tõ̀npɩbɩ nónó-á sõ̀ mɛ, Yesu tõ̀ sã́hpú sɔkɔ nɛy yɔ́ núkú ǹ kèkírkìlɛ pépi sɔkɔ yɔ́ gbõ̀yõ̀ sõ̀ níkí wɛ kõ kãm. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́: «Yé yĩn sɛ̃́.» Tɛ́ tũ dĩ́ mɛ̀ níkí tyɩ́, kè mɛ tyɛ. 
Képah sɔkɔ, sárká wìlkìpu yõ̀tãm, ànɛ̂ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kyɔ̀mɩ̀pu yõ̀tãm, ànɛ̂ yèwàhnɩ̀ nónó-á sõ̀ pɔ Yesu tɛ̃́-ńsah, à yo pé tyɩ́: «Yé pɔ ńmɔ tyɩ́ kèkîrn ànɛ̂ léplólɛ kèlékɛ̃ névi sɔ̃́ ńnɛ nɩ? 
Ǹtɛ, swɔ̃ swɔ̃ póllɛ, mé sõ̀ ńkɛ̃́nɛ n nɛ n mɛ yélɛ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ, yé náh ń tɛ̃. Tɛ́ yépi gbĩ ye sɛ núkúnúkú, ànɛ̂ sétõ tǎhkɩ̀ pɩ gbĩ́nɛ fɛ̃́ntũ̂ sɔkɔ.» 
Képah náh, pè Yesu tɛ̃, à sɔkɔ ǹnɛ sárká wìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ kéte. Pyɛrɩ sõ̀ mɛ pé sõ̀ tɛ̃ n kɔ pélɛ fɩ́ɩ́. 
Pè sõ̀ mɛ nyàh mɔ kéte nɩyṍ, mɛ kɔ̃ tɛ̃ wɛ̃ tyɩ́, tɛ́ Pyɛrɩ tɔ sõ̀ mɛ pélɛ nyàh yah sɔkɔ. 
Képah sɔkɔ, tõ̀npɩ pi yɔ́ à wɛ kɔ̃́ntɛ̃̀nɔ́lɛ nyàh yah, tɛ́ à gbah yah díkídíkí, tɛ́ nɛ, dĩ́ mɛ̀ tɔ-á sõ̀ mɛ Yesulɛ. 
Wǎh yo sɛ̃́, Pyɛrɩ mɛ ǹ vyãh kɔ̃lɩ pi tyɩ́ nɛ, pé náh à pɩ̃ dɛ́. 
Dǒ kwéy, dĩ́ yɔ́ tɔ̃ à wɛ, tɛ́ nɛ, ǹmɔ tɔlɛ, kénɛ no sɔkɔ nɛy yɔ́-á ǹnɛ. Képah tɛ̃̀nɛ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé sɔkɔ nɛy náh péwɔlɛ. 
Képah náh, lɛ́rɩ́ núkú kénkɔ̃ sɔ́kɔ́ntɛnɔ náh, dĩ́ yɔ́ tɔ̃ kè gbah yo nɛ ńkɛ̃́, yékã̂b náh mɛ ké sɔkɔ, dĩ́ mɛ̀ tɔ-á sõ̀ mɛ ǹnɛ, Kalile nɛy-á ǹnɛ dò. 
Képah sɔkɔ, Pyɛrɩ mɛ ǹ syɩ yo ǹ tyɩ́ nɛ ńkɛ̃́, péwɔ náh mɛ ǹ wɛy mɛ̀ syɩ́kɩ́-ńsah wɛ dɛ́ dĩ́. Nɛ́npɔ́ swɔ, Pyɛrɩ mɛ képah yónɔ́ sɔkɔ sɛ̃́, tɛ́ sénpɛ gbo. 
Tɛ̃̀ ǹgbɛ̃ Yesu mɛ ǹ yah vi Pyɛrɩ yah díkídíkí. Képah sɔkɔ, Pyɛrɩ dyɔ núkú to kõ ǹ wɛy mɛ̀nɛ, wǎh sõ̀ nɛ̂ yo ǹ tyɩ́ nɛ, sénpɛ-à kɩ pópó n gbo kwɛynɛ, wǎh kɩ mɛ ǹ vyãh kɔ̃lɩ tɛ kɛ tɔ̃́nɔ́ nɛ ńkɛ̃́, pé náh péwɔ pɩ̃. 
Kǎh pɩ sɛ̃́, à mɛ wil kégbɔ́, à kyɩ nàhnpɩ̃ gbní kah gbo. 
Képah sɔkɔ, Yesu-á sõ̀ mɛ díbí nónó gbõ̀, pè sõ̀ mɛ kṍlɛ ǹnɛ n ko, tɛ́ mɛ ǹnɛ n syɩkɩ gbõ̀nkukelɛ. 
Pè sõ̀ mɛ ǹ yah pukubi, tɛ́ mɛ n yo ǹ tyɩ́ nɛ, à sãh pɩ̃, névi nɛ̂-á mɩ ǹnɛ. 
Tɛ́ tɔ̃ mɛ ǹnɛ n gbyɔ wɛ̀ntùkùnì náhnáhlɛ. 
Képah náh, tyah-á dal, Nsyifunɔ yèwàhnɩ̀, ànɛ̂ sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ ǹgbò wɛ̃̀kɩ̀ kwɔ́pú wɛ̃ tuke mɔ. Pè mɛ pɔ Yesulɛ pé tukey tùkè-ǹsah ǹgbɛ̃ sɔkɔ, 
tɛ́ yo ǹ tyɩ́ nɛ, ǹmɔ-à kɩ nɛ Kristalɛ, Liyel nɛy tɛ́kɩ́nmɔnɔlɛ, à kè yo pé syah. À mɛ yo pé tyɩ́: «Máh kɩ kè yo yé syah, yé náh n sɛ̃ n pi ké yõ. 
Ànɛ̂ máh kɩ se yélɛ n piki n yah, yé náh vi mɔ n pi ń tyɩ́. 
Ǹtɛ, à tɔkɔ núkúnúkú tyɩ́, Névi Pi kɩ kɔ̃ tɛ̃ tyi pól Tɛ̃̀ ǹgbɛ̃ Liyel gbõ̀yõ̀ sõ̀lɛ.» 
Képah sɔkɔ, pé pól núkú yo nɛ, ǹmɔ tɛ̃̀ tyɩ́, ǹmɔ-á Liyel pilɛ kɛ̀? Yesu mɛ yo pé tyɩ́: «Yépi ńtɛ̃ ye kè yo tɛ nɛ ńkɛ̃́, ńmɔ-á dò.» 
Képah tɛ̃̀nɛ, pè nɛ, tàrfyɔ̀ nɛ̂ tɛ̃̀ se tɔ̃ pé kɩ n yah n kɔ̃ nɩ? Pépi ńtɛ̃-á kɩ noh, à kè yo ǹ vyãhlɛ.
