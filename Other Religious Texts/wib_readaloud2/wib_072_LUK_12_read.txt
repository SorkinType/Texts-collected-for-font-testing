Likyi.
12.
Képah sɔkɔ, nɛ́kwĩ́nkí-á tuke mɔ tnɔ́nɔ́lɛ gbĩ́ nɛ̂nɛ fɔ́ɔ́, mɛ wɛ̃ gbyẽ̂hnɛ n suki, Yesu tyah n yo ǹ tõ̀ sã́hpú ǹgbò tyɩ́: «Yé yé gblɔ̌y tɛ̃ Farisiyɛ̃nɔ vyɛ̃́nnínì wɛ́kɩ́nɔ́lɛ, képah nɛ̂-á n tɔ̃ n mɔ kɔ́kɔ́nɔ́ tyɩ́ kɔ̃lɛ mífã́hkɩ̀ sɔkɔ. 
Tyi nónó pól-á yé n yo fɛ̃́ntũ̂ sɔkɔ, sè kɩ pɔ noh gbĩ̀ntɔ̃̀lɛ kyòo yah. Ànɛ̂ nónó pól-á yé n plɔ n yo wɛ̃ níkílɔ́ yey, wɔ̀kɔ̀ sɔkɔ táh, sé pól kwní kɩ pɔ kɔ̃ gbɛ̃́ntĩ̂n yõ. Képah ye, tyípǐl yɔ́ ńtɛ̃ náh mɛ, nɛ̂-á kɩ n wũ n pi wìlkìnmɔ-ńkɛ̃̂nɛ. Sètĩ̀n sõ̀ tir yɔ́ ńtɛ̃ náh mɛ, nɛ̂-á kɩ n wũ n pi pɩ̃́-ńkɛ̃̂nɛ.» 
«Mé kélɛ n yo yé tyɩ́ nɛ, yépi nónó-á ń kódíbílɛ: yé káh névye yah tíkílɛ n pɩ, nónó-á wil kɔ̀ ólɛ n ko n mɔ n pnɛ, tɛ́ pè náh tɔ̃ tir yɔ́lɛ n pɩ n kah n mɔ n pnɛ képah yõ. 
Nɛ̂ yah tíkí-á yɩ̃nɛ pɩ́nɔ́lɛ, mé kɩ ǹnɛ yé wɛ̃kɩ: Liyel nɛ̂-á kɩ fɛ̃ névi wil kɔ̀ kònmɔntɛnɔ náh, kɩ ǹ min dah mɔ tyɛ́-ńkɛ̃̂ nyàh sɔkɔ, yé ǹmɔ tíkílɛ n pɩ. Mé kélɛ n yo yé tyɩ́ nɛ ńkɛ̃́: yé tɛ̃̀ wɔ tíkílɛ n pɩ. 
Ǹtɛ̃̀npînɔ́ kwlɔ náh n yãm pɔ́píkí plɔ́ nínɔ́lɛ di? Ǹtɛ, Liyel dyɔ náh n fwɔ sépi núkú yɔ́ ńtɛ̃ tyɩ́lɛ. 
À wil képah sɔkɔ, yé yõ̀yén ńtɛ̃ pól dan mɛ pɩ̃́nɔ́lɛ Liyel tyɩ́. Kǎh pɩ sɛ̃́, yé káh tíkí pɩ. Yépi tyɩ́ náh tah kal ǹtɛ̃̀npî kwĩnkilɛ di?» 
Képah náh, Yesu tɔ̃ yo pé tyɩ́: «Mé kélɛ n yo yé tyɩ́ nɛ: nɛ̂-à kè yo dal mɔ névye yĩ́ yah nɛ ńkɛ̃́, páh mɛ ńmɔ tyɩ́ yõ, Névi Pi tɔ kɩ kè yo Liyel yĩ̂nyõ tɛ̃ntõnè yĩ́ yah nɛ ńkɛ̃́, wǎh mɛ pé tyɩ́ yõ. 
Ǹtɛ, nɛ̂-à ń yahle névye yĩ́ yah, Névi Pi tɔ kɩ à yahle Liyel yĩ̂nyõ tɛ̃ntõnè yĩ́ yah. 
Névi nɛ̂-à Névi Pi yĩ́npo yo, kè kɩ yɔ̃ mɔ tɛ̃̀ kɔ̃. Tɛ́ nɛ̂ tɛ̃̀ wɔ-à Liyel Mírkí gbyɔ, kè náh n yɔ̃ n mɔ n pi ǹ tyɩ́. 
Pàh yé tɛ̃ kyɩ tùkè-ǹsah Liyel ní-ńtã̂n vɩ̀ sɔkɔ, képah náh pɩ, yah tãm tyɩ́, képah náh pɩ, yõ̀tãm tyɩ́, Liyel Mírkí kɩ yé kwɔ nɛ́npɔ́ swɔ, yáh nɛ nɛ̂ yo n pi. Képah ye, yé káh yɛ yé yáh-ńsah tyɛ, yáh nɛ n yo n wil n pi sɔ̃́.» 
Tɔ́wû nɩyṍ sɔkɔ, ǹnɩ̂ Yesu syah nɛ, pé kwɔ́-ò dĩ́, à yo pé nɛ́nkyɩ́ dĩ́ tyɩ́ nɛ, à yɛ pé pé sú gbɛ̃kɩ dibi. 
Képah tɛ̃̀nɛ, Yesu mɛ yo ǹ tyɩ́: «Ń dĩ́, nɛ̂ sah ńnɛ nɛ, mé yélɛ n tuke, tĩ̂ nɛ, mé yé gbã̀n yîlɛ n dibi?» 
Képah sɔkɔ, tɛ́ yo pé tyɩ́: «Yé yé kyɔmɩ tahrɩ dékè képékèlɛ! Tahrɩ-à sɔ̃́ mɛ sɔ̃́ névi tyɩ́, képah náh fɛ̃ minnɛ à kɔ̃ n pi.» 
Képah sɔkɔ, à tɔ̃ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: «Tàh dĩ́ yɔ́ lésõ mɛ, ǹ swãh pɛkɩ yĩ́ĩ́ ǹ tyɩ́. 
Képah tɛ̃̀nɛ, à tyah n sõ ǹ fɔkɔ sɔkɔ nɛ, ǹtɛ, sèmukɔ mɔ́-ńsah náh se mɛ fwɔ́ɔ pé tyɩ́, sɔ̃́ se pé kɩ pɩ nɩ? 
Dǒ, à nɛ, páh kɩ nɛ̂ pɩ n pi, képah á yɔ̀: Páh kɩ pé díké sɔkɩ, tɛ́ kɩ sè tɔ̃ mɔ gblò gblò, kɩ pé gbã̀n yî pól mɔ sé sɔkɔ, 
tɛ́ kɩ yo pé gblɔ̌y tyɩ́ nɛ, páh yɔ̀, yõke dò-á mɛ pé tyɩ́ pé gbõ̀ sõ̀ ye náhnáh yĩnnɛ. Núkúnúkú, páh kɩ n mɩ, tɛ́ kɩ n yõ, tɛ́ kɩ n wɔ, tɛ́ kɩ nɩ̀vɩ̀lɛ n pɩ pé dyɔlɛ. 
Ǹtɛ, Liyel mɛ yo ǹ tyɩ́: ‹Lékã́m kɛ̃́ névi! Kwɛy lékã̂h ńtɛ̃ sɔkɔ á min kɩ n syi n pi á tyɩ́. Tɛ́ yî nónó-á á tuke mɔ sah, sè kɩ pɩ nɛ̂ tyɩ́lɛ?› 
Képah sɔkɔ, Yesu nɛ: «Névi nɛ̂-à tahrɩlɛ n yah n koh n sah ǹ gblɔ̌y tyɩ́lɛ, tɛ́ Liyel fṍ tyɩ́ yáhnɔ́ yahle, sɛ̃́ntɛ̃̀ ye n nɔ tɛ̃̀nɛ!» 
Képah náh, Yesu yo ǹ tõ̀ sã́hpú tyɩ́: «Névi min tyɩ́ náh tah kal yõkelɛ, wil tyɩ́ mɛ tah kal flɔlɛ di? Képah ye, mé kélɛ n yo yé tyɩ́ nɛ: yé káh yé fɔkɔlɛ n sõ, yáh nɛ wɛ n yõ n pi sɔ̃́, ànɛ̂ yáh nɛ yé wil flɔ wɛ n mɔ n pi sɔ̃́. 
Yé yah ǹgbɔ̃lɔlɛ kɛ̀: Sè náh sèmukɔlɛ n duku, sè náh sèmukɔlɛ n tɔkɔ. Díké ànɛ̂ sèmukɔ kóhnsàh-ǹtã̀n tɔ náh mɛ sé tyɩ́, tɛ́ Liyel sélɛ n pu. Ǹtɛ, yépi tyɩ́ náh tah kal kèpyelɛ di? 
Yépi sɔkɔ, nɛ̂ tɛ̃̀ kɩ fɛ̃ kwéyy ńtɛ̃ fyé dohnɩ ǹ min wɛ́nɔ́ yõ ǹ fɔ́kɔ́sõnɔ gbõ̀ yõ? 
Tɛ̃́nwɛnɔ-à mɛ́-ńkɛ̃̂nɛ yé tyɩ́, à fɛ̃ sɛ̃́ntɛ̃̀ tir tyɩ̃́ɩ̃ ńtɛ̃ fyé pɩ n pi, sɔ̃́ pɩ, tɛ́ yé yé fɔkɔlɛ n sõ yî tɛ́lɔ́ tyɩ́ sɔkɔ? 
Yé fílfĩ pɩ dùkùlɛ n yah kɛ̀: Sè náh tõ̀lɛ n pɩ, sè náh flɔlɛ n tĩ. Ǹtɛ, mé kélɛ n yo yé tyɩ́ nɛ: yõ̀tɛ̃̀ Salomo ńtɛ̃nɛ ǹ tahrɩ pól sɔkɔ, à náh flɔ pèplɔ wɛ mɔ yah fílfĩ nɔ́pi sɔkɔ yɔ́ ńtɛ̃ kètɔ̀lɛ. 
Fílkì nɛ̂ wɔ-á gbɛ̃́nnɩ sɔkɔ kwɛynɛ, tɛ́ kɩ pɔ n sukɛ n pi ńsõ̂nɛ, Liyel-á képah fĩ pɩ pallɛ sɛ̃́, à náh n yah n dah n mɔ n pi yépi flɔ tyɩ́ sɔkɔ à kal sépilɛ di, Liyel yõ sɛ̃́nyĩ-ńkɛ̃̂pú nɔ́pi? 
Kǎh pɩ sɛ̃́, yé káh yé gblɔ̌ynɛ n kwlɔ yé yõke, ànɛ̂ yé wɔ wɔ yî yáhnkɔ̃nɔ tyɩ́ sɔkɔ. 
Kèkõyṍ mɛ̀ sɔkɔ, Liyel pɩ̃́-ńkɛ̃̂ névye ye ńkɛ̃́nɛ nɛ n yah n kɔ̃ sépi póllɛ tyɛ́-ńkɛ̃̂nɛ. Ǹtɛ, yépi tɛ̃̀ wɔlɛ, yé sú Liyel pɩ̃ nɛ, sé yétõ̂-á mɛ yé tɛ̃. 
Kǎh pɩ sɛ̃́, yé Liyel tɔ̃́rɩ́ ǹgbò yah n kɔ̃! Képah sɔkɔ, à kɩ yî tɛ́lɔ́ dohnɩ yé kɔ̃. 
Kè ye nɔ yé sú tyɩ́, à ǹ tɔ̃́rɩ́lɛ yé kɔ̃. Képah ye, yé káh tíkí pɩ, tũ̀npir yípɛ̂y! 
Yé yé gbã̀n yî yãm, yé mó sé yɩ̃ yãm tãn kɔ̃. Yé tahrɩlɛ n yah n sah Liyel fṍ, képah nɛ̂-á kè náh n yɩkɩ, kè náh se n tyɛ. Yɩ̃́pú náh fɛ̃ n kyɩ nɛ́npɔ́, sóhrɩ́ tɔ náh fɛ̃ sè yõ. 
Sɛ̃́ ó ye, á tahrɩ-à mɛ ńyãh, á sõnɔ tɔ n nɛ n mɛ nɛ́npɔ́.» 
«Yé nɛ mɛ yé vi sah tõ̀ pɩ pɩ tɛ̃̀nɛ, yé mó yé knɔ fɩ̃ mɔ, yé fɔ̀tnɔ̀nɔ́ mó nɛ mɛ kõ̀ntɛ̃̀nɔ́lɛ. 
Yé nɛ mɛ á kɩ nɛ tõ̀npɩbɩ sɔ̃́, nónó-á ńkɛ̃́nɛ mɛ pé yõ̀tɛ̃̀ syɩ́kɩ́ntɛ̃̀nɔ́ sɔkɔ nɛ, wàh kɩ wil fúr pɩ́-ńsah, à pɔ n de, tɛ́ kɩ fyɔ̀ wɔ̀kɔ̀ yah mɩ, pé núkú kè yɩkɩ mɔ à kɔ̃. 
Yèvɩnɩ mɛ pé yõ, tõ̀npɩbɩ nónó yõ̀tɛ̃̀-à ǹ syɩ pɔ pè wɛ, pè mɛ yáhnɔ́ sɔkɔ. Kègbɩ yõ, mé kélɛ n yo yé tyɩ́ nɛ: yõ̀tɛ̃̀ mɛ̀ kɩ ǹ tõ̀ pɩ pɩ flɔ mɔ, tɛ́ kɩ yõkelɛ pè kɔ̃ pè kɩ yõ. 
Tɛ́ wàh se mɛ n syɩ pɔ lénɩlɛ, képah náh pɩ, tyah dal gbĩ́nɛ, tɛ́ pè mɛ ǹ syɩ́kɩ́nɔ́ sɔkɔ, yèvɩnɩ mɛ pé yõ! 
Yépi ńtɛ̃ pɩ̃ nɛ, gbóntɛ̃̀ dĩ́-à pɩ̃ gbĩ́ nɛ̂-á yɛ̃́nwó kɩ n pɔ n pi ǹ tyɩ́, à náh n yɛ à kɔ̃ à n wɛ n de n pi ǹ gbô. 
Névi Pi kɩ n pɔ n pi gbĩ́ nɛ̂-á yé náh mɛ ǹ pɔ́nɔ́ tyɩ́lɛ n sõ. Képah ye, yépi tɔ yé gblɔ̌y vi sah.» 
Képah sɔkɔ, Pyɛrɩ Yesu syah nɛ, pé Yõ̀tɛ̃̀, pépi ó yĩn se à n yãh sèmìrkì mɛ̀nɛ, tĩ̂ no pól yĩn se nɩ? 
À mɛ yo ǹ tyɩ́: «Gbô tahrɩ kyɔ̀mɩ̀-ò nɛ́gbɩ́ lékã́m névilɛ, nɛ̂ mɛ? Ǹmɔ pɩ névilɛ, nɛ̂-á ǹ yõ̀tɛ̃̀ dĩ́ kɩ à pɩ ǹ kéte kyɔ̀mɩ̀-òlɛ. Tɛ́ à kɩ tõ̀npɩbɩ tɛ́lɔ́ yõkelɛ pélɛ n kɔ̃ ké kɔ̃ gbĩ́nɛ. 
Yèvɩnɩ mɛ ǹ tõ̀npɩ mɛ̀ tyɩ́, yõ̀tɛ̃̀ dĩ́-à ǹ syɩ pɔ à wɛ à mɛ kénɛ tõ̀lɛ n pɩ. 
Kègbɩ yõ, mé kélɛ n yo yé tyɩ́ nɛ: wǎh kɩ à pɩ ǹ gbã̀n yî pól kyɔ̀mɩ̀-òlɛ. 
Ǹtɛ, tõ̀npɩ mɛ̀-à tyah n sõ ǹ fɔkɔ sɔkɔ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀ dĩ́-á kɩ mo pɔ́-ńsah, tɛ́ tyah pé no tõ̀npɩbɩlɛ n ko, pé syɔ́, pé díbí, tɛ́ mɛ n yõ, mɛ wɔ wɔ yîlɛ n wɔ, à sètɛ̃̀nɩ̀nɛ n pɩ, 
yõ̀tɛ̃̀ dĩ́ kɩ pɔ ǹ yõ dɔkɔ sṍ yɔ́lɛ, à náh mɛ nɛ̂ sõ sah, ànɛ̂ gbĩ́ yɔ́ sɔkɔ, à náh nɛ̂nɛ n pnɛ. À kɩ pɔ à fõh képékèyɔ̂, tɛ́ kɩ à dyah Liyel tyɩ́ yáhlepu dyah dùkùlɛ. 
Ǹmɔ nɛ̂-á ǹ yõ̀tɛ̃̀ dyɔ yĩ́ĩ́nɛ n pnɛ, tɛ́ à náh ǹ gblɔ̌y vi à képahlɛ n pɩ, à kɩ ko náhnáh wɛ. 
Ǹtɛ, névi nɛ̂-á sõ̀ mɛ kè pɩ ǹpɩ̃́-ńkɛ̃̂ sɔkɔ, tɛ́ yɩ̃nɛ ko syínɔ́lɛ, ǹmɔ kɩ ko kwéyy wɛ. Pàh névi nɛ̂ kɔ̃ náhnáh, náhnáh kɩ yah kɔ̃ tɛ̃̀ tyɩ́. Ànɛ̂ náhnáh-à névi nɛ̂ gbõ̀ dahbɩ, náhnáh tyɩ́ kɩ piki yah kal tɛ̃̀ tyɩ́.» 
Képah náh, Yesu tɔ̃ nɛ: «Nyàh ye mé pɔ sétáh yõ. Tɛ́ mé sõ̀ mɛ ké tyɩ́ nɛ, kè nɛ mɛ kõ tɛ! 
Ńmɔ yɩ̃nɛ mè n de n wil yèvyãhrɩ yɔ́ sɔkɔ, tɛ́ kè náh se mɛ n pi! 
Yé se mɛ n sõ nɛ, yèvɩnɩ-á mé pɔ mɔ́-ńsah sétáh yõ? Ɔ̃́ɔ̃ dɛ́! Mé kélɛ n yo yé tyɩ́ nɛ: gbɛ́kɩ́nɔ́ wɔ-á pyě. 
Képah ye, à tɔkɔ núkúnúkú tyɩ́, nɛ́pĩ̂ kwâl-à mɛ kéte núkú sɔkɔ, pè kɩ gbɛkɩ: Tɔ̃́mɩ́ kɩ tyah nímínɛ n fwo, nímí kɩ tyah tɔ̃́mɩ́nɛ n fwo. 
Sú yɔ́ kɩ tyah ǹ pídĩ́nɛ n fwo, pídĩ́ yɔ́ kɩ tyah ǹ súlɛ n fwo. Yṹ yɔ́ kɩ tyah ǹ písêlɛ n fwo, písê yɔ́ kɩ tyah ǹ yṹnɛ n fwo. Yṹ yɔ́ kɩ tyah ǹ pídĩ́ kyɩlɛ n fwo, pídĩ́ kyɩ yɔ́ kɩ tyah ǹ pɛ yṹnɛ n fwo.» 
Yesu tɔ̃ sõ̀ mɛ n yo tɔ́wû tyɩ́: «Yĩ̂nyõ yah yɔ́ mɛ, yâh ńyĩ́rkì wɛ ké nɛ́npɔ́lɛ, yé n yo nɛ́npɔ́ swɔ nɛ, pũ̀-á kɩ pɩ. Tɛ́, kè n mó n pɔ n pɩ sɛ̃́. 
Ànɛ̂ yah yɔ́ mɛ, yâh fyɔ̀ yã́hkɩ́ wɛ kè kõ wil nɛ́npɔ́lɛ, yé n nɛ, tyah vyã́hnɔ́ képékè-á kɩ tyah. Tɛ́, kè n mó n pɔ n pɩ sɛ̃́. 
Yĩ́nvǐ névye nɔ́pi! Yé fɛ̃ sétáh ànɛ̂ yĩ̂nyõ dùkù wɔlɛ n pnɛ. Tɛ́ sɔ̃́ pɩ, tyi nónó-á n pɩ kwɛy gbĩnɛ, yé náh fɛ̃ sépi pɩ̃?» 
«Kwâh nɛ̂ pɩ, tɛ́ yé náh pɩ dùkù pèpɛynɛ n sõ n tɛ̃ n pnɛ yé gblɔ̌y? 
Sɔ̃́-á, tir-à mɛ yé wrɔ́ ànɛ̂ á ní-òlɛ, tɛ́ à ǹ tɛ̃ n yuku tùkè-ǹsah tukey tùkè-ò tyɩ́, gbah n tɛ̃ tir mɛ̀ yah wah wɛ̃̀kɩ̀ yah. Képah-à pɩ́-ńkɛ̃̂nɛ, tɛ́ à kyɩ álɛ tukey tùkè-ò gbõ̀ dahbɩ, tukey tùkè-ò mɛ kyɩ álɛ sràsyíbí tyɩ́, tɛ́ pépi mɛ ǹ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Mé kɩ kè yo dal mɔ á tyɩ́ nɛ, á náh n wil n pi nɛ́npɔ́, nɛ́ á náh kɔ̃lɩ mɛ̀ pól dyah tɛ pnɛ́pnɛ́.»
