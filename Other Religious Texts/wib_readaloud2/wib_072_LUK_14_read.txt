Likyi.
14.
Mɩ mɩ yòh sṍ yɔ́lɛ, Yesu kyɩ yõke yṍ-ńsah Farisiyɛ̃nɔ yõ̀tɛ̃̀ dĩ́ yɔ́ gbô. Tɛ́ nónó-á sõ̀ mɛ nɛ́npɔ́, pè mɛ Yesulɛ n kyɔmɩ. 
Tɛ́ kè pɔ yɩ̃nɛ, wílsɛ̃́nɩ̀ yɔ́nwó yɔ́ sõ̀ mɛ ǹ tnɔ̂. 
Képah sɔkɔ, Yesu mɛ píkínyahnɔ pɩ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú ànɛ̂ Farisiyɛ̃nɔ tyɩ́: «À yɔ́nwó dɛ mɔ mɩ mɩ yòh sṍlɛ, ké wɛ̃̀kɩ̀ se mɛ kɔ̃́nɔ́lɛ, tĩ̂ kè se mɛ yáhlenɔlɛ?» 
Ǹtɛ, pè náh vi mɔ ǹ tyɩ́. Képah sɔkɔ, Yesu mɛ yɔ́nwó mɛ̀ tɛ̃, à à dɛ mɔ, tɛ́ à kɔ̃ à sɔkɔ, 
tɛ́ mɛ yo pé tyɩ́: «Yépi sɔkɔ, nɛ̂ tɛ̃̀ pi, képah náh pɩ, ǹ nahy kɩ sɔkɔ de kwlɛ̀y sɔkɔ mɩ mɩ yòh sṍlɛ, tɛ́ à náh à wilki n pi súú?» 
Ǹtɛ, pè náh tɛ̃ wɛ tir yo ké tyɩ́ sɔkɔ. 
Képah yóntɛnɔ náh, Yesu-á nɛ́yénɔ́ kyɔmɩ wɛ, pè mɛ tɛ̃̀-ǹtã̀n ǹgbnɔlɛ n yah n kɔ̃, à vìnmɔnɔlɛ pè kɔ̃ nɛ: 
«Pàh ǹ ye fúr yõke yṍ-ńsah, káh n kyɩ n tɛ̃ tɛ̃̀-ǹsah ǹgbɛ̃ sɔkɔ. Wâh képah pɩ, tɛ́ pè mɛ ǹnɩ̂ ye, nɛ̂-á kal áwɔlɛ, 
yé yé-ò káh pɔ ǹ syah nɛ ńkɛ̃́, á yuku, á mó tɛ̃̀-ǹsahlɛ tɛ̃̀ kɔ̃. Légbĩ́nɛ, á kɩ yuku sen sɔkɔ, kɩ kyɩ tɛ̃ tɛ̃̀-ǹsah tyɩ̃́ɩ̃ sɔkɔ. 
Ǹtɛ, pàh ǹ ye, kyɩ n tɛ̃ tɛ̃̀-ǹsah tyɩ̃́ɩ̃ sɔkɔ. Á yé-ò-à pɔ ǹ wɛ sɛ̃́, à kɩ yo á tyɩ́ nɛ, pé kódĩ́, á nɔ pɔ tɛ̃̀-ǹsah ǹgbɛ̃ sɔkɔ. Képah sɔkɔ, kè kɩ pɩ gbílkínɔ́lɛ á tyɩ́ yõke yṍpú pól yĩ́ yah. 
Ńkɛ̃́, névi nɛ̂-à ǹ gblɔ̌y dɔkɔ sah, tɛ̃̀ kɩ tiki mɔ, tɛ́ nɛ̂-à ǹ gblɔ̌y tiki mɔ, tɛ̃̀ kɩ dɔkɔ mɔ.» 
Tɛ́ ǹ syɩ yo ǹ yé-ò dĩ́ tyɩ́: «Wâh névyelɛ n ye yõkelɛ gbĩ̀ntɔ̃̀lɛ, képah náh pɩ, lékã̂hnɛ, káh á kódíbí ye, káh á yṹnpyé ye, káh á tyɩ́ névye ye, káh á tnɔ̂ tɛ̃̀pu tahbɩ ye. Pépi tɔ kɩ fɛ̃ pɔ ǹ ye yõke ǹgbɛ̃ yṍnɔ́lɛ, kè kɔ̃ kè kɩ pɩ á yénɔ́ mɛ̀ lédyáhnɩ́nɛ. 
Ǹtɛ, wâh névyelɛ n ye yõkelɛ, yãm tãn, ànɛ̂ nɛ́dónè, ànɛ̂ gbáhkɩ̀ núkú névye, ànɛ̂ yɩ̃́tũbi ye. 
Pépi náh tɛ̃ wɛ ǹ dyah n pi ké sɔ̃́nɛ. Kàh pɩ sɛ̃́, á kɩ yèvɩnɩ wɛ Liyel tyɩ́, tyípéplɔ pɩ́pú lékyɩ̂-à yilki n mɔ sṍ nɛ̂nɛ.» 
Képah sɔkɔ, kénɛ yõke yṍpú sɔkɔ nɛ́núkù yɔ́-á kénɛ wɛlɔ noh sɛ̃́, à mɛ Yesu syah nɛ, névi nɛ̂-á kɩ yõke yõ n pi Liyel tɔ̃́rɩ́ sɔkɔ, yèvɩnɩ-á kélɛ tɛ̃̀ tyɩ́ dɛ́! 
Yesu mó yo ǹ tyɩ́: «Dĩ́ yɔ́ lésõ yõke ǹgbɛ̃ pɩ, tɛ́ névye náhnáh ye. 
Yõke yõ gbĩ́-á pɔ nɔ, à ǹ tõ̀npɩ dĩ́ tɛkɩ mɔ nɛ, à kyɩ yo nɛ́yénɔ́ tyɩ́, pè pɔ, páh tyɛ núkúnúkú. 
Ǹtɛ, pé pól tyah gblɔ̌y yɛlɛ n yah n kɔ̃ dùkù núkúlɛ. Nɛ́gben à syah nɛ, páh swãh yɔ́ dwe núkúnúkú, ńkɛ̃́ páh yɩ̃nɛ pé kyɩ kè yah. Páh ǹnɛ n ni, à ǹ gblɔ̌y yɛ pé tyɩ́. 
Ǹnɩ̂ mɛ nɛ, páh sètõ̀nyehnɔ gbãm dwe núkúnúkú, tɛ́ páh n yuku sé pɩ́nyah-ńsah. Páh ǹnɛ n ni, à ǹ gblɔ̌y yɛ pé tyɩ́. 
Ǹnɩ̂ mɛ nɛ, páh sê tɔkɔ núkúnúkú, képah-á pé kɔ̃ pé náh n wɛ n pɔ n pi. 
Tõ̀npɩ-á ǹ syɩ pɔ, à pɔ tyi nɔ́pi pól yãh ǹ yõ̀tɛ̃̀ dĩ́ syah. Képah sɔkɔ, kététɛ̃̀ dĩ́ fɔkɔ vyãh, à mɛ yo ǹ tõ̀npɩ tyɩ́ nɛ, à kyɩ súú kwil tɔ́wû tùkè-ǹtãn ànɛ̂ wã̀l yey sɔkɔ. À kyɩ yãm tãn, ànɛ̂ nɛ́dónè, ànɛ̂ yɩ̃́tũbi, ànɛ̂ gbáhkɩ̀ núkú névye tɛ̃ pɔ ásõ̂. 
Képah náh, tõ̀npɩ ǹ syɩ pɔ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, wǎh nɛ̂ yo, páh kè pɩ, tɛ́ tɛ̃̀-ǹtã̀n túkù-á tɔ̃ mɛ. 
Kǎh pɩ sɛ̃́, yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, à kyɩ wɛ̃̀-ǹgbnɔ yey, ànɛ̂ névye fwɔ́rɩ́nkɔ̃-ńtã̂n sɔkɔ, à kyɩ névye kãhlɩ tɛ̃ pɔ, pé wɔ̀kɔ̀ kɔ̃ kè yuku. 
Páh kélɛ n yo pé tyɩ́ nɛ: nónó-á sõ̀ yénɔ́lɛ, pé sɔkɔ yɔ́ ńtɛ̃ náh pé yõke dɔ n yah n pi.» 
Képah náh, Yesu-á n sɔkɔ, tɔ́wû ǹgbɛ̃ sõ̀ mɛ ǹ sõ̀ tɛ̃. À mɛ ǹ yah vi, tɛ́ yo pé tyɩ́: 
«Nɛ̂-à pɔ ńmɔ tyɩ́, tɛ́ ń tyɩ́ náh nɔ kal ǹ tyɩ́ ǹ súlɛ, ànɛ̂ ǹ yṹnɛ, ànɛ̂ ǹ kyɩlɛ, ànɛ̂ ǹ wɛ̃́npìlɛ ànɛ̂ ǹ yṹnpyelɛ, à tahlɩ ǹ gblɔ̌y tɔ tyɩ́, tɛ̃̀ náh fɛ̃ n pɩ n pi ńmɔ tõ̀ sã́h-òlɛ. 
Névi nɛ̂ náh ǹ syɩ syɩ dahnɩ twah, à ńmɔ sõ̀ tɛ̃, tɛ̃̀ náh fɛ̃ n pɩ n pi ńmɔ tõ̀ sã́h-òlɛ. 
Ǹtɛ, yépi sɔkɔ nɛ̂ tɛ̃̀ kɩ n yah n kɔ̃ wɔ̀kɔ̀ ǹgbɛ̃ mɔ́-ńsahlɛ, nɛ́ à náh tɛ̃, à ké gbõ̀ wilki kõ yah, tɛ́ kè yah n pi, ǹ pɔ́-à kɩ n yɩ̃nɛ n pi ké mɔ́nsyìnɔ́lɛ? 
Nɛ́ tɛ̃̀ náh képah pɩ, tɛ́ wɔ̀kɔ̀ sètyah wilki, tɛ́ tɛ̃ kwlɔ ké mɔ́nsyìnɔ́ tyɩ́, no-à kè wɛ, pè kɩ tyah kṍlɛ ǹnɛ n ko, 
tɛ́ kɩ n yo nɛ, pè yah dĩ́ yɔ́lɛ, à wɔ̀kɔ̀ mɔ tyah mɔ, tɛ́ à náh tɛ̃ wɛ kè syi. 
Sɛ̃́ ó ye tɔ̃, yõ̀tɛ̃̀ nɛ̂ tɛ̃̀ kɩ n yuku mɛ́kɩ̀ fwó-ńsah ǹ ní-ò yõ̀tɛ̃̀nɛ, nɛ́ à náh tɛ̃ ǹ fɔkɔ mɔ n yah n pi, wàh kɩ fɛ̃ kyɩ mɛ́fwóbé nɛ́pĩ̂ kèyõ kwlɔ́lɛ ǹ fwó-ò swah n pi, nɛ̂-á mɛ ǹnɛ n yohnɩ mɛ́fwóbé kèyõ gbãmnɛ? 
Wàh pɩ̃ nɛ, pé náh fɛ̃ n tɛ̃ n pi, à kɩ ǹ ní-ò sah fɩ́ɩ́, tɛ́ kɩ nírí pɩ́-ò tɛkɩ mɔ kɩ mɛ́kɩ̀ yálnmɔnɔ yah kɔ̃ ǹ tyɩ́. 
Sɛ̃́ ó ye, nɛ̂-à ǹ gbã̀n yî pól wɛ́nyɛ̀-ǹkɛ̃̂nɛ, tɛ̃̀ náh fɛ̃ n pɩ n pi ńmɔ tõ̀ sã́h-òlɛ.» 
«Pnɛnɛ, kwâh pèpɛy ye kélɛ. Ǹtɛ, ké dêndên-à wal, sɔ̃́ mɛ kè kɩ tɔ̃ fɛ̃ pɩ dêndên? 
Kàh pɩ sɛ̃́, kè náh fɛ̃ sétáh tyɩ́ pɛkɩ, ké náh se tɔ̃ fɛ̃ n pɩ pilɛ. Képah ye, kè n milki kégbɔ́. Níkí-à mɛ nɛ̂ tyɩ́ noh noh tɛ̃̀nɛ, tɛ̃̀ ǹ níkí sah noh.»
