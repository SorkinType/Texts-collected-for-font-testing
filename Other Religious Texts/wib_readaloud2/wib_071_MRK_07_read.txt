Markɩ.
7.
Sṍ yɔ́lɛ, *Farisiyɛ̃nɔ ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú túkù wil Yerusalɛmɩ, à tuke mɔ pɔ Yesu tnɔ̂. 
Pè pɔ kè kyɔmɩ wɛ nɛ, ǹ *tõ̀ sã́hpú túkù-á mɛ n yõ gbɛ̃̀nsìllɛ. Pè náh mɛ pé gbã̀nnɛ n kãm n wu pé ǹgbòtãn tyɩ́ kɔ̃lɛ. 
Tɛ́, Farisiyɛ̃nɔ ànɛ̂ *Nsyifunɔ tɛ́lɔ́ pól sõ̀ ǹgbòtãn wɛ̃̀kɩ̀ mɛ̀ sàhnɔ́lɛ n kɔ. Pè náh sõ̀ yõkelɛ n yõ, nɛ́ pè náh pé gbã̀n kãm wu pallɛ. 
Pàh wil pɔ yòh sɔkɔ, pè náh yõkelɛ n yõ pé kã̀mnwèl-ǹkɛ̃̂nɛ. Pè sõ̀ pé ǹgbòtãn pɩ dùkùnɔ́ náhnáh yísyɔ́ tɔ tɛ̃ yĩ́ĩ́ kègbɩlɛ, sɔ̃́-á wóhnɔ, ànɛ̂ pní, ànɛ̂ kɛ́sɛ̂n ànɛ̂ sɛ̃ sɛ̃ yînɔ́ wúnɔ́ dùkù pèpɛynɛ. 
Képah ye, Farisiyɛ̃nɔ ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú mɛ Yesu piki yah nɛ, kû se pɩ, tɛ́ ǹ tõ̀ sã́hpú náh mɛ ǹgbòtãn tyi sàhnɔ́lɛ n pɩ, tɛ́ n yõ pé gbã̀n kã̀mnwu-ńkɛ̃̂nɛ nɩ? 
Képah sɔkɔ, à mɛ yo pé tyɩ́: «Gbɩ lésõ mɛ *Yesayi tyɩ́, wǎh Liyel wɛy nɛ̂ yo yépi vyɛ̃y nini wɛ́kɩ́pú tyɩ́ sɔkɔ. Sɛ̃́ ye sõ̀ à wãrkɩ sah kélɛ: ‹Kénɛ névyelɛ, pè ńnɛ n gbilki vyãh ó sɔkɔ, tɛ́ pé nɩ mɛ fɩ́ɩ́ ńnɛ. 
Nírí nɛ̂-á pè n nɛ n pɩ ńmɔ tyɩ́, nɔ̀nɔ́ ye kélɛ. Tir yɔ́ náh, kwɔ́nɔ́ nɛ̂-á pè n nɛ n kɔ̃, névye sàhnɔ́ ye kélɛ, ńmɔ tyɩ́ náh.›» 
Yesu tɔ̃ yɔ̃ sɔkɔ yo pé tyɩ́: «Yépǐ Liyel tyi kõ̀nsàhnɔ́lɛ n milki, ǹ tɛ́ névye tyɩ́ sàhnɔ́lɛ n pɩ. 
Yě n gbah n tɛ̃ Liyel tyi kõ̀nsàhnɔ́ yahle, ǹ tɛ́ yé ǹgbò wã̀llɛ n kɔ! 
Sɔ̃́-á sõ̀ *Moyisi yo nɛ, névi-á yɩ̃nɛ à ǹ sú ànɛ̂ ǹ yṹnɛ n gbilki. Képah náh pɩ, névi nɛ̂-à ǹ sú ànɛ̂ ǹ yṹ yilɔ, tɛ̃̀-á yɩ̃nɛ kònmɔnɔlɛ. 
Ǹtɛ, yépǐ névyelɛ n kwɔ nɛ, névi-à yo ǹ sú tyɩ́, képah náh pɩ, ǹ yṹ tyɩ́ nɛ, páh sõ̀ nɛ, pé kwâh nɛ̂nɛ à kɔ̃, páh kélɛ Liyel kɔ̃ tɛ. 
Yě ké gbɩ syi tɛ̃̀ kɔ̃ nɛ, à náh tɔ̃ yɩ̃nɛ à tir yɔ́ pɩ ǹ sú tyɩ́, képah náh pɩ ǹ yṹ tyɩ́. 
Kénɛ pɩ dùkù sɔkɔ ye, yě Liyel wɛy pɩ nɔ̀nɔ́ tirlɛ yé ǹgbò wã̀l kwɔ́nɔ́ sɔkɔ. Ànɛ̂ yě n tɔ̃ sɛ̃́nknɔ náhnáhlɛ n pɩ.» 
Képah náh, Yesu ǹ syɩ tɔ́wû ye tuke, tɛ́ yo pé tyɩ́: «Yé póllɛ, yé noh ń tyɩ́, yé mó pɩ̃ nɛ, 
kwâh yɔ́ ńtɛ̃ náh n wil kégbɔ́, à n de névi sɔkɔ, tɛ́ kɩ fɛ̃ tɛ̃̀ silki. Tɛ́, tyi nónó-á n wil névi sɔkɔ, sépi-á n silki ǹnɛ.  
Níkí-à mɛ nɛ̂ tyɩ́ noh noh tɛ̃̀nɛ, tɛ̃̀ ǹ níkí sah noh!» 
Yesu-á yal tɔ́wû tnɔ̂, tɛ́ kyɩ de gbô gbĩ́ nɛ̂nɛ, ǹ *tõ̀ sã́hpú ǹnɛ n piki n yah sèmìrkì mɛ̀ yõlɛ. 
À mɛ pè syah nɛ, lékã́m kɛ̃́ névye se pépi tɔlɛ nɩ? Pè kó pɩ̃ nɛ, névi-á kwâh nɛ̂nɛ n dah n mɔ ǹ vyãh, képah náh fɛ̃ tɛ̃̀ silki n pi nɩ? 
Kè náh n de ǹ nɩ sɔkɔ. Kǎh ńkɛ̃́nɛ n de ǹ kèkùrkì sɔkɔ, à kɔ̃ à n kyɩ n wilki n kwɛ, wàh de fílkì. Kénɛ wɛy sɔkɔ ye à yo pé tyɩ́ nɛ, yõkenɔ pól-á kɩ fɛ̃ yõ. 
Tɛ́ tɔ̃ yo pé tyɩ́: «Tyi nónó-á n wil névi fɔkɔ, sépi ye ǹnɛ n silki Liyel yah. 
Fɔkɔ sõnɔ túkúnì nónó-á n wil névi sɔkɔ, sépi ye à kɔ̃, ǎ ǹ minnɛ n pɩ fɔ̀fɩ́kɩ̀ sɔkɔ, ànɛ̂ yɩ̃be, ànɛ̂ kònmɔnɔ, 
ànɛ̂ syɔ́ yõ yɩ́kɩ́nɔ́, ànɛ̂ yĩ́ yúkú-ńkɛ̃̂ sɔkɔ, ànɛ̂ nɛ́pékè, ànɛ̂ yĩ́vî, ànɛ̂ sen kú-ńkɛ̃̂ sɔkɔ, ànɛ̂ dyɔ́ yɔ́nɔ́, ànɛ̂ névye yĩn yɩ́kɩ́nɔ́, ànɛ̂ gblɔ̌y gbílkínɔ́, ànɛ̂ tɔ̃ lékã́m kɛ̃́ sɔkɔ. 
Kénɛ tyítúkúnì pól n wil névi sɔkɔ, à ǹnɛ n silki Liyel yah.» 
Képah náh, Yesu yal nɛ́npɔ́, à yuku Tiri sétáh yõ. À kyɩ wɔ̃bɩ twah gbô yɔ́ sɔkɔ. À náh sõ̀ mɛ ké tyɩ́ nɛ, nɛ pɩ̃ nɛ, páh mɛ nɛ́npɔ́. Tɛ́, à náh tɛ̃ wɛ wɔ̃bɩ. 
Nɛ́npɔ́ swɔ, pi yɔ́ sõ̀ mɛ, *sétõlɛ ǹ písê sɔkɔ. Wǎh Yesu tyɩ́ noh sɛ̃́, à núkú pɔ gbètukunɔ di ǹ yah. 
Pi mɛ̀nɛ, *Nsyifu nɛ náh sõ̀ ǹnɛ. À sõ̀ dubi Siri sétáh yõ, Fenisi sɔkɔ. Wǎh pɔ, à sõ̀ mɛ Yesulɛ n ni nɛ, à sétõ yĩni wilki pé písê sɔkɔ. 
Képah tɛ̃̀nɛ, Yesu mɛ yo ǹ tyɩ́ nɛ, à yɛ wɛ̃́npì ǹgbò kɔ̃, pè yõ tɔ̃. Wɛ̃́npì yõke náh yɩ̃nɛ kè tɔkɔ mɔ pã̀hnpye kɔ̃. 
Pi mɛ̀ mó yo ǹ tyɩ́ nɛ, gbɩ-á, pé Tɛ̃̀-ǹgbɛ̃. Ǹtɛ, wɛ̃́npì-á yõkelɛ n yõ, yɛ̃́nmîr nónó-à n syɩkɩlɩ sétáh, pã̀hnpye kó ńkɛ̃́nɛ sépilɛ n tõ n yõ nɩ? 
Képah sɔkɔ, à yo pi mɛ̀ tyɩ́: «Á wɛy mɛ̀ yĩnnɛ, á syɩ n to n yuku. Sétõ ye wil á písê sɔkɔ.» 
Kǎh pɩ sɛ̃́, à mɛ ǹ syɩ to yuku gbô, à kyɩ ǹ písê wɛ sɛ̃́nɔ́lɛ sɛ̃ sɛ̃ kwâh yõ. Sétõ sõ̀ wil tɛ ǹ sɔkɔ. 
Képah náh, Yesu yal Tiri sétáh yõ, à kyɩ sɔkɔ Sidõ kwillɛ, tɛ́ pyě Kwlogbãm kwil sétáh kah, à yuku Kalile yɔ tĩ̂nnɛ. 
Nósyɔ́ mɛ níkítũ̂ yɔ́ tɛ̃ pɔ ǹ tyɩ́, màhmàhkɩ̀ wɛ́kɩ́-òlɛ, tɛ́ mɛ à ni nɛ, à ǹ gbõ̀ sah ǹ yõ. 
Képah sɔkɔ, à dĩ́ mɛ̀ tɛ̃ wil, à gbɛ mɔ tɔ́wûlɛ, à kyɩ ǹ kèngblɔ dah mɔ dĩ́ ní sɔkɔ, tɛ́ sĩn mɔ, à tuke ǹ nɛ́npɔ́rɩ́ tyɩ́. 
Tɛ́ ǹ yah yɔ̃ yĩ̂nyõ, à mɩ dɔkɔ, tɛ́ yo dĩ́ tyɩ́ pé kwil wɛy sɔkɔ: «Efata», nɛ̂-á pè n ye: «Yɩkɩ n mɔ.» 
Nɛ́npɔ́ swɔ, dĩ́ ní yɩkɩ mɔ, ǹ nɛ́npɔ́rɩ́ dɛ mɔ, à tyah n wɛkɩ tírr. 
Tɛ́, Yesu mɛ pè sni nɛ, pè káh kélɛ n yo n kɔ̃ névye tyɩ́. Tɛ́, wǎh pélɛ n sni sɔ̃́, pè pyě wãnnɛ n mɔ ké yónkɔ̃nɔ yõ. 
Pé gbã̀n kàhndunɔ tɛ̃̀ tyɩ́, pè tyah n yo wɛ̃ tyɩ́ nɛ, gbɩ yõ, ǹ tyípɩ́nɔ́ pól-á mɛ kah plɛ! Wǎh níkítũbi nínɛ n yɩkɩ, tɛ́ kèdyɔ̀nɩ̀ nɛ́npɔ́rɩ́lɛ n dɛ n mɔ.
