Nsyɔ̃.
20.
Dimansyi sõ séngbɔ̃́lɛ, Makadala kwil Mari kwãh kyɩ diki yõ tósõ yɔ̃ gbĩ́nɛ. À kyɩ nɛ n yah diki yah dáhkɩ̀ náh tɔ̃ mɛ ké yah. 
Képah sɔkɔ, à to sɔkɔ à kyɩ wil Simo-Pyɛrɩ ànɛ̂ ǹ ní-ò Yesu tõ̀ sã́h-ò yɔ́ tyɩ́ nɛ̂ tyɩ́-á nɔ yĩ́ĩ́ Yesu tyɩ́, tɛ́ yo pé tyɩ́ nɛ, pǎh Tɛ̃̀ ǹgbɛ̃ lékó wilki diki sɔkɔ. Tɛ́ pépi náh pɩ̃ pǎh à sah ńyãh sɔkɔ. 
Képah sɔkɔ, Pyɛrɩ ànɛ̂ ǹ ní-ò Yesu tõ̀ sã́h-ò mɛ̀ mɛ yuku à diki yõ nyáh sõ. 
Pé nímí sõ̀ mɛ n to. Tɛ́ ǹmɔ-á sõ̀ mɛ n to súú kal Pyɛrɩlɛ, ǹmɔ kyɩ yĩni tɛ̃ gbennɛ diki yõ. 
Wǎh kɔ yah diki sɔkɔ, à fwɔ gbró wɛ sàhnɔ́lɛ ké sɔkɔ, tɛ́ à náh de. 
Ǹtɛ Simo-Pyɛrɩ nɛ̂ wɔ-á sõ̀ mɛ n pi ǹ náhlɛ, ǹmɔ wɔ-á pɔ yĩni tɛ̃, à de diki sɔkɔ. Tɛ́ kyɩ fwɔ gbró wɛ sàhnɔ́lɛ, 
ànɛ̂ fwɔ yɔ́lɛ fíkénsàhnɔ́lɛ ké tyɩ́lɛ, pǎh sõ̀ nɛ̂ mɔ ǹ yõ yõ. Képah náh sõ̀ mɛ fwɔ gbró tɛ́lɔ́lɛ fɩ̃́ núkú. 
Képah sɔkɔ, Yesu tõ̀ sã́h-ò nɛ̂-á sõ̀ pɔ de diki yõ gbennɛ, ǹ tɔ mɛ de diki sɔkɔ. Wǎh kè wɛ, à sɛ̃ ké yõ. 
Tɛ́ à pɔ nɔ légbĩ́ tyɩ́, wɛy nɛ̂-á wãrkɩ sah Liyel wɛy sɔkɔ nɛ, Yesu lékó-á yɩ̃nɛ à pɔ n yilki mɔ, ǹ tõ̀ sã́hpú náh tɔ̃̌nɩ̀ sõ̀ mɛ képah dùkù pɩ̃. 
Képah náh, Yesu tõ̀ sã́hpú nɛ́pĩ̂ nímí nɔ́pi pé syɩ to yuku pé tĩ̀nnɛ. 
Légbĩ́nɛ, Mari sõ̀ mɛ yĩ́nntɛ̃̀nɔ́lɛ kégbɔ́ sɔkɔ diki tnɔ̂ mɛ n gbo. Wǎh mɛ gbní sɔkɔ gbĩ́ nɛ̂nɛ tɛ́ kɔ yah diki sɔkɔ, 
à yĩ̂nyõ tõ̀npɩbɩ nɛ́pĩ̂ nímí wɛ fwɔ̀pikinɔ sɔkɔ diki sɔkɔ. Pè sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ Yesu lékó snɛ́kɩ́nmɔ-ńsah sɔkɔ. Núkú sõ̀ mɛ tɛ̃ ǹ yõ mɔ́-ńsah tĩ̀nnɛ, tɛ́ ní-ò mɛ tɛ̃ ǹ gbyẽ̂h mɔ́-ńsah tĩ̀nnɛ. 
Pè mɛ nɛ, pi, kwâh nɛ̂ se pɩ tɛ́ à n gbo nɩ? À mɛ nɛ pǎh pé Tɛ̃̀ ǹgbɛ̃ lékó wilki. Ǹtɛ́ pé náh pɩ̃ pǎh à sah ńyãh sɔkɔ, képah-á pé kɔ̃ pé n gbo. 
Wǎh mɛ képah yónɔ́ sɔkɔ tɛ́ ǹ yah vi, à Yesu wɛ yĩ́nnɔ́lɛ nɛ́npɔ́. Ǹtɛ à náh sõ̀ mɛ pɩ̃ nɛ ǹmɔ-á. 
Yesu mɛ à piki yah nɛ, pi, kwâh nɛ̂ se pɩ tɛ́ à n gbo nɩ? Nɛ̂ se à n yah n kɔ̃ nɩ? Ǹtɛ Mari wɔ sõ̀ mɛ n sõ nɛ sètɛ̃̀ dĩ́-á, tɛ́ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, ǹmɔ-à kɩ nɛ̂ wilki ǹnɛ, à ǹ sàh-ǹsahlɛ pé wɛ̃kɩ pé kyɩ à tɔkɔ. 
Képah tɛ̃̀nɛ, Yesu mɛ à ye nɛ, Mari! Mari mɛ ǹ yah vi ǹ tĩ̀nnɛ tɛ́ yo ǹ tyɩ́ Yebre wɛy sɔkɔ nɛ, Wrabuni, nɛ̂ kɔ́r-á kwɔ́-ò dĩ́nɛ. 
Ǹtɛ Yesu mɛ yo ǹ tyɩ́ nɛ: «Mé náh tɔ̃̌nɩ̀ mɛ dɔkɔ yuku Sú Liyel tnɔ̂, képah ye, káh ń tɛ̃ n sah. Tɛ́ kyɩ n yo ńmɔ yṹnpyé tyɩ́ nɛ, máh kɩ dɔkɔ yuku ń Sú tyɩ́, nɛ̂-á yépi tɔ Súlɛ. Ǹmɔ nɛ̂-á ń Liyellɛ ànɛ̂ yépi tɔ Liyellɛ, máh kɩ dɔkɔ yuku ǹ tyɩ́.» 
Képah sɔkɔ, Makadala kwil Mari mɛ kyɩ kè yo Yesu tõ̀ sã́hpú tyɩ́ nɛ, páh Tɛ̃̀ ǹgbɛ̃ wɛ. Tɛ́ wǎh tyi nónó yo ǹ tyɩ́, à mɛ sè yãh pè tyɩ́. 
Kénɛ dimansyi mɛ̀ ó sõ ńvɩ̃́nɛ, Yesu tõ̀ sã́hpú sõ̀ mɛ tùkènmɔntɛ̃̀nɔ́lɛ wɔ̀kɔ̀ yɔ́ sɔkɔ. Tɛ́ Nsyifunɔ yõ̀tãm yah tíkí tɛ̃̀ tyɩ́, pè sõ̀ mɛ ké vɩ̀ yey tɛ̃ n mɔ sah. Ǹtɛ dǒ, Yesu núkú pɔ n yĩn pé nɩyṍ, tɛ́ yo pé tyɩ́ nɛ: «Yé nɛ n mɛ yèvɩnɩ sɔkɔ.» 
Képah yóntɛnɔ náh, à ǹ gbã̀n táhlɔ́ ànɛ̂ ǹ gbɔ̀pɔlɛ pè wɛ̃kɩ. Pǎh Tɛ̃̀ ǹgbɛ̃ wɛ sɛ̃́, kè pɩ yĩ́ĩ́ nɩ̀vɩ̀nɩ̀nɛ pé tyɩ́. 
À mɛ tɔ̃ yo pé tyɩ́ nɛ: «Yé nɛ n mɛ yèvɩnɩ sɔkɔ. Sú Liyel-á ńmɔ tõ sɔ̃́, sɛ̃́ ye ńmɔ tɔ yélɛ n tõ.» 
Kénɛ wɛlɔ yóntɛnɔ náh, à fɩ̃bɩ mɔ pé yõ tɛ́ nɛ: «Liyel Mirki ye mé yélɛ n kɔ̃. 
Yâh nónó tyípêl yɔ̃ mɔ pè kɔ̃, sè kɩ yɔ̃ mɔ pé tyɩ́. Tɛ́ yâh nónó tyɩ́ yɔ̃̀nmɔ-ńkɛ̃̂nɛ, sè náh n yɔ̃ n mɔ n pi pé tyɩ́.» 
Ǹtɛ Yesu-á pɔ ǹ tõ̀ sã́hpú tnɔ̂ sõ nɛ̂nɛ, Toma nɛ̂-á pè yĩn dyɩ à kɔ̃ kènìnìnɛ, ǹmɔ wɔ náh sõ̀ mɛ. Yesu tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí sɔkɔ yɔ́ ye sõ̀ ǹ tɔlɛ. 
Kǎh pɩ sɛ̃́, pé no tɛ́lɔ́ mɛ pɔ kè yo ǹ tyɩ́ nɛ, pépi-á Tɛ̃̀ ǹgbɛ̃ Yesu wɛ. Ǹtɛ à mɛ pè syah nɛ ńkɛ̃́, nɛ́ péwɔ tɛ̃̀ náh kwèrkè nɩ́y wɛ ǹ gbã̀n táhlɔ́ sɔkɔ, tɛ́ pé kèngbɔ dyɩ yah sé sɔkɔ, ànɛ̂ nɛ́ pé náh pé gbõ̀ dyɩ yah ǹ gbɔ̀pɔ tũ̀-ǹsah sɔkɔ, pé náh n sɛ̃ n yah n pi ké yõ póllɛ. 
Képah náh, ké dimansyi kèyâhlɛ ńtɔ̃, Yesu tõ̀ sã́hpú sõ̀ mɛ wɛ̃ tuke mɔ tɛ̃ wɔ̀kɔ̀ sɔkɔ, tɛ́ mɛ ké vɩ̀ yey pól tɛ̃ mɔ. Képah wɔ sɔkɔ, Toma tɔ sõ̀ mɛ pélɛ. Dǒ, Yesu núkú pɔ n yĩn pé nɩyṍ, tɛ́ yo pé tyɩ́ nɛ: «Yé nɛ n mɛ yèvɩnɩ sɔkɔ.» 
Képah yóntɛnɔ náh, à yo Toma tyɩ́ nɛ: «Á kèngbɔlɛ pɔ ásõ̂ sɔkɔ yɔ̀, ǹ tɛ́ n yah ń gbã̀n táhlɔ́lɛ. Á gbõ̀ nɔ n dyɩ ń gbɔ̀pɔ sɔkɔ. Yal n mɔ tɔ̀tɔ̀ pɩ́nɔ́lɛ, ǹ tɛ́ n sɛ̃ ké yõ.» 
Képah tɛ̃̀nɛ, Toma nɛ ńkɛ̃́, ǹmɔ-á pé Tɛ̃̀ ǹgbɛ̃ ànɛ̂ pé Liyel tɔlɛ. 
Képah sɔkɔ, Yesu mɛ à syah nɛ: «Wáh ń wɛ, képah ye á sɛ̃ ń yõ kɛ̀. Ǹtɛ nónó-à sɛ̃ ńmɔ yõ ń wɛ́nyah-ńkɛ̃̂nɛ, pè kɩ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ.» 
À wil képah sɔkɔ, Yesu tɔ̃ gbǐl tyi náhnáh pɩ ǹ tõ̀ sã́hpú yĩ́ yah, nónó-á sè náh wãrkɩ sɛ́bɛ́y mɛ̀ sɔkɔ. 
Ǹtɛ nónó-á wãrkɩ ké sɔkɔ, sè ye wãrkɩ nɛ sépi gbõ̀ yõ, yé sɛ̃ ké yõ nɛ Yesu-á Liyel Pídĩ́nɛ, ànɛ̂ Liyel nɛy yáhntɔkɔnɔlɛ. Tɛ́ yáh kɩ sɛ̃ ǹmɔ yõ, yé mó tyɛ́-ńkɛ̃̂ min wɛ.
