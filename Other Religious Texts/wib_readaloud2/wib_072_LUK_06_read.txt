Likyi.
6.
Képah náh, Nsyifunɔ mɩ mɩ yòh sṍ yɔ́lɛ, Yesu sõ̀ mɛ wɛ̃̀kɩ̀ tɛ̃ n kɔ n kah sèmukɔ swɛ̃y yísyɔ́ sɔkɔ. Tɛ́ ǹ tõ̀ sã́hpú mɛ sèmukɔ ywɔ̃lɛ n kɔ̃lɩ n wɔkɔlɩ pé gbã̀n sɔkɔ, à n tah. 
Farisiyɛ̃nɔ-á képah wɛ sɛ̃́, pé túkù mɛ nɛ, tõ̀ pɩ́nɔ́ wɛ̃̀kɩ̀ náh mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sṍlɛ. Tɛ́ kwâh nɛ̂ se, tɛ́ pè kélɛ n pɩ nɩ? 
Képah sɔkɔ, Yesu yo pé tyɩ́: «Kwɛkɩ-á lésõ yõ̀tɛ̃̀ Davidi ànɛ̂ ǹ náh névye tɛ̃ sṍ nɛ̂nɛ, wǎh nɛ̂ pɩ, yé náh kè kar yah kɛ̀? 
À náh sõ̀ de Liyel wɔ̀kɔ̀ sɔkɔ, à búrn tɔkɔ, nónó-á sõ̀ mɛ sàhnɔ́lɛ Liyel tyɩ́ yõ, à tah, tɛ́ ǹ náh névye tɔ kɔ̃ di? Tɛ́ nɛy náh sõ̀ yɩ̃nɛ búrn nɔ́pi táhnɔ́lɛ, nɛ́ sárká wìlkìpu yõ̀tãm ó náh.» 
Tɛ́ tɔ̃ yo pé tyɩ́: «Névi Pilɛ, mɩ mɩ yòh yõ̀tɛ̃̀ ye ǹnɛ.» 
Képah náh, mɩ mɩ yòh sṍ yɔ́lɛ ńtɔ̃, Yesu de Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ, mɛ tyah névyelɛ n kwɔ. Tɛ́ dĩ́ yɔ́ sõ̀ mɛ nɛ́npɔ́ sɔkɔ, gbõ̀yõ̀lɛ kúnɔ́lɛ. 
Képah tɛ̃̀nɛ, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú, ànɛ̂ Farisiyɛ̃nɔ mó mɛ Yesulɛ n kyɔmɩ n yah, wàh kɩ ńtyɩ̃́nɔ nɛy dɛ n mɔ n pi mɩ mɩ yòh sṍlɛ, pè mó lékàh wɛ à kɔ̃. 
Ǹtɛ, ǹmɔ tɛ̃̀ wɔ sõ̀ mɛ pé fɔ́kɔ́sɔ̃nɔlɛ n pnɛ. Kǎh pɩ sɛ̃́, à mɛ yo gbõ̀nkù dĩ́ mɛ̀ tyɩ́: «Yuku n pɔ n yĩn névye nɩyṍ ásõ̂.» Dĩ́ mɛ yuku yĩn. 
Képah tɛ̃̀nɛ, Yesu yo pé tyɩ́: «Ńmɔ yélɛ n piki n yah, tir nɛ̂ pɩ́nɔ́ wɛ̃̀kɩ̀ mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sṍlɛ? Pèpɔrɩ se, tĩ̂ pèkè se? À nɛy min pwah mɔ, tĩ̂ à tɛ̃̀ ko mɔ?» 
Képah sɔkɔ, wǎh pè yah kore vi, à yo dĩ́ tyɩ́: «Á gbõ̀ tɔ̃!» À mɛ ǹ gbõ̀ tɔ̃, kè núkú pɩ ké vilɛ. 
Képah tɛ̃̀nɛ, pé fɔkɔ kah vyãh Yesulɛ, pè tyah wɛ̃nɛ n vi n mɔ, sɔ̃́-á pè kɩ n pɩ n pi ǹnɛ. 
Sṍ yɔ́lɛ, Yesu dɔkɔ yuku yɔ̃lɩ yɔ́ yõ, à kyɩ nírílɛ n pɩ. À lékã̂h pól pɩ Liyel nínɔ́ sɔkɔ. 
Tyah-á pɔ dal, à ǹ tõ̀ sã́hpú ye, tɛ́ nɛ́pĩ̂ gbɔ̃nimí yah wilki pé sɔkɔ, tɛ́ pépi yĩn sah ǹ tɛ̃̀ntõbelɛ. 
Wǎh nónó yah wilki, pé yni wɛ yɔ̀: Simɔ wɔlɛ, Yesu ǹmɔ yĩn sah Pyɛrɩlɛ, tɛ́ ǹ gbɛ̃̀ndĩ́ Andre sõ̀ mɛ, ànɛ̂ Nsyakɩ, ànɛ̂ Nsyɔ̃, ànɛ̂ Filipi, ànɛ̂ Bartelemi, 
ànɛ̂ Matye, ànɛ̂ Toma. À tahlɩ Alfe pídĩ́ Nsyakɩ tyɩ́, ànɛ̂ Simɔ nɛ̂-á sõ̀ pè n ye «Sú fṍ vyãh syí-òlɛ», 
ànɛ̂ Nsyakɩ pídĩ́ Yudelɛ, ànɛ̂ Yisikariyɔtɩ Yudasɩlɛ, nɛ̂-á pɔ vi Yesu náh. 
Képah náh, Yesu tiki yɔ̃lɩ yõ ànɛ̂ pélɛ, à pɔ yĩn kéntáhkɩ̀ támm yɔ́ sɔkɔ. Tɛ́ ǹ tõ̀ sã́hpú náhnáh sõ̀ mɛ, à tahlɩ tɔ́wû ǹgbɛ̃ yɔ́ tɔ tyɩ́, nónó-á wil Yude tã̀ntã̀n pól sɔkɔ, ànɛ̂ Yerusalɛmɩ, à tahlɩ pnɛ yɔ gbɔ̀pɔ kwlo tyɩ́, Tiri ànɛ̂ Sidõnɛ. 
Pè sõ̀ pɔ Yesu wɛlɔ nóh-ńsah, ànɛ̂ dɛ́nmɔnɔ wɛ yĩnnɛ pé yamnɔ sɔkɔ. Sétã̀n-á sõ̀ mɛ nónó sɔkɔ mɛ pélɛ n kwlɔ, pé tɔ dɛ́nmɔnɔ wɛ. 
Tɛ́ tǎhkɩ̀ yɔ́ sõ̀ mɛ n wil ǹ sɔkɔ, à pé póllɛ n dɛ n mɔ. Képah ye tɔ́wû pól kɔ̃, pè n yah n kɔ̃ wɛ́ntũ̀-ǹsahlɛ ǹ tyɩ́. 
Képah sɔkɔ, Yesu mɛ ǹ yah yɔ̃ ǹ tõ̀ sã́hpú yah, tɛ́ nɛ: «Yèvɩnɩ mɛ yé tyɩ́, yépi nónó-á mɛ yãm sɔkɔ. Liyel tɔ̃́rɩ́ mɛ yépi tyɩ́ yõ. 
Yèvɩnɩ mɛ yé tyɩ́, yépi nónó-á mɛ núkúnúkú kwɛkɩ sɔkɔ. Yé kɩ pɔ n tɔ̃ n pi. Yèvɩnɩ mɛ yé tyɩ́, yépi nónó-á mɛ núkúnúkú gbní sɔkɔ. Yé kɩ pɔ yɩ̀ syɔ n pi. 
Yèvɩnɩ mɛ yé tyɩ́, no-à yé dyɔlɛ n yɔ, tɛ́ yé milki, tɛ́ yélɛ n gbyɔ, tɛ́ yé yĩnnɛ n yɩkɩ Névi Pi yõ sɛ̃́nɔ́ yĩnnɛ. 
Ǹtɛ, ké yõre ǹgbɛ̃ kɩ pɔ yé dyah Liyel fṍ. Képah ye, yé nɛ n mɛ nɩ̀vɩ̀ sɔkɔ lésõlɛ, yé nɛ n yɔkɔ n kwãh nɩ̀vɩ̀ tɛ̃̀ tyɩ́. Sɛ̃́ ye lésõ pé náh tãn tɔ n fõh Liyel tɛ̃ntõ̀ syínyopulɛ. 
Ǹtɛ, yépi nónó-á tahbɩlɛ, yèvyãhrɩ mɛ yé yõ. Yé yé pèpɔrɩ wɛ sɔkɔ tɛ. 
Yépi nónó-á mɛ núkúnúkú tɛ̃kɩ sɔkɔ, yèvyãhrɩ mɛ yé yõ. Kwɛkɩ kɩ pɔ yé tɛ̃ n pi. Yépi nónó-á mɛ núkúnúkú yɩ̀syɔ̀nɩ̀ sɔkɔ, yèvyãhrɩ mɛ yé yõ. Yé kɩ pɔ yèkwɔrɩ gbní gbo n pi. 
Pàh yépi nónónɛ n gbilki, yèvyãhrɩ mɛ yé yõ. Sɛ̃́ ye lésõ pé náh tãn tɔ n gbilki Liyel yĩn yõ mɩ̀nyobelɛ.» 
«Ǹtɛ, yépi nónó-á mɛ n nohnɩ ń tyɩ́, mé kélɛ n yo yé tyɩ́ nɛ: yé yé sépɛ́bɩ́lɛ n tɛ̃ plɛ. Yé pèpɔrɩlɛ n pɩ yé dyɔ yɔ́pú tyɩ́. 
Nónó-á plɔlɛ n ye n mɔ yé yõ, yé sãmnɛ n mɔ pé sɔkɔ. Yé mó Liyellɛ n ni yé fṍhpú tyɩ́. 
Névi-à á létáh núkú mɩ, ké ní-ò tɔ vi n mɔ à kɔ̃. Névi-à á fwɔ̀mɔ̀-ǹgbɛ̃ tɔkɔ, yɛ tɛ̃̀ kɔ̃ à á fwɔ̀mɔ̀npir tɔ tɔkɔ. 
Névi nɛ̂-à kwâh ni á tyɩ́, kè yɩ̃ tɛ̃̀ kɔ̃. Ànɛ̂, névi-à á gbõ̀ kwâh tɔkɔ, káh nɛ, tɛ̃̀ n syɩ n pɔ kélɛ. 
Ànɛ̂ tɔ̃, yáh mɛ ké tyɩ́ nɛ, no nɛ̂nɛ n pɩ yé tyɩ́, yépi tɔ sɛ̃́nkɔ̃lɛ n pɩ pé tyɩ́. 
Yépi tyɩ́-á nɔ nónó tyɩ́, pépi ó tyɩ́-à nɔ yé tyɩ́, yĩ́ngbɩ́ nɛ̂ mɛ yé kɩ n wɛ n pi képah sɔkɔ? Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú ńtɛ̃ tɔ nɛ́nɔ́nɔ́ tyɩ́ náh nɔ pé tyɩ́ di? 
Nónó-á n tɔ̃ n sah n pɩ yé tyɩ́, yé tɔ-à n tɔ̃ n sah n pɩ pépi ó tyɩ́, yĩ́ngbɩ́ nɛ̂ mɛ yé kɩ n wɛ n pi képah sɔkɔ? Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú ńtɛ̃ tɔ náh sɛ̃́nkɔ̃lɛ n pɩ di? 
Ànɛ̂, yâh n dɛkɩ névyelɛ n kɔ̃, yáh sɛ̃ ké yõ nɛ, nónó ó-á kɩ pɔ sé lédyáhnɩ́nɛ yé kɔ̃ n pi, yĩ́ngbɩ́ nɛ̂ mɛ yé kɩ n wɛ n pi képah sɔkɔ? Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú tɔ náh n dɛkɩ wɛ̃nɛ n kɔ̃, sé lédyáhnɩ́ pɔ́nwɛ yĩnnɛ di? 
Pèpɔrɩ pɩ̃́-ńkɛ̃̂pú, ànɛ̂ nɛ́pêl nɩ̀gbɛ̀kɩ̀ mɛ Liyel sɔkɔ. Képah ye, yépi tɔ yé sépɛ́bɩ́lɛ n tɛ̃ plɛ. Yé pèpɔrɩlɛ n pɩ pé tyɩ́. Yé nɛ n dɛkɩ n kɔ̃, ké lédyáhnɩ́ tyɩ́ sõ̀-ǹkɛ̃̂nɛ. Képah sɔkɔ, yé kɩ kyɩ ké lédyáhnɩ́ ǹgbɛ̃ wɛ Liyel fṍ, tɛ́ kɩ pɩ yõ̀tah Liyel wɛ̃́npìlɛ. 
Yé pɩ nɩ̀gbɛ̀kɩ̀ névyelɛ, sɔ̃́-á yé sú Liyellɛ nɩ̀gbɛ̀kɩ̀ pɩ́-òlɛ.» 
Képah sɔkɔ, Yesu tɔ̃ yo pé tyɩ́: «Yé káh nolɛ n tuke. Képah sɔkɔ, Liyel náh yé tɔ tuke n pi. Yé káh lékàhlɛ nolɛ n kɔ̃. Képah sɔkɔ, Liyel náh lékàhlɛ yé tɔ kɔ̃ n pi. Ǹtɛ, yé nɛ n yɔ̃ n mɔ nolɛ n kɔ̃. Képah sɔkɔ, Liyel kɩ n yɔ̃ n mɔ yé tɔlɛ n kɔ̃. 
Yáh no tɛ́lɔ́ tyɩ́lɛ n sãh dú nɛ̂nɛ, kénɛ dú núkú ó kɩ yé tɔ tyɩ́ sãh n pi. Képah ye, yé nɛ n yɩ̃ n kɔ̃. Képah sɔkɔ, Liyel kɩ yɩ̃ yé tɔ kɔ̃. À kɩ dú tɔkɔ, kɩ sã́hnɔ́ pèpɛy sãh, kɩ sni, kɩ yike yike, kè kɩ yuku kwɛ, kɩ vi mɔ yé kɔ̃ yé fwɔ̀mɔlɔ vyɛ̃y tɛ̃́nkũnmɔnɔ sɔkɔ.» 
Tɛ́ tɔ̃ sèmìrkì yɔ́ mɩ pé tyɩ́ nɛ ńkɛ̃́: «Yɩ̃́tũ̂ se kɩ fɛ̃ pɩ ǹ ní-ò yɩ̃́tũ̂ yah tɛ̃́-òlɛ? Pé nímí náh n syɩkɩ n de n pi pwã̀h yɔ́ sɔkɔ di? 
Tir sã́h-ò náh n kal n pnɛ ǹ kwɔ́-òlɛ. Tɛ́ sã́h-ò nɛ̂-à kwɔ́nɔ́ wɛ yĩ́ĩ́, ǹmɔ n pɩ ǹ kwɔ́-ò tyɩ́ kɔ̃lɛ. 
Á pɩ sɔ̃́, tɛ́ ǹdan kyɔmɩ wɛ á yṹnpi yĩ́ sɔkɔ, tɛ́ dàhnpɛlɛ áwɔ yĩ́ sɔkɔ, á náh mɛ képah wɔ wɛ? 
Á kɩ pɩ sɔ̃́, tɛ́ kɩ yo á yṹnpi tyɩ́ nɛ, à yɛ á ǹdan wilki à kɔ̃ ǹ yĩ́ sɔkɔ, áwɔ nɛ̂-á dàhnpɛlɛ á yĩ́ sɔkɔ, tɛ́ á náh mɛ képah wɔ wɛ? Vɩ̃́nnínì wɛ́kɩ́-ò! Á yĩ́ sɔkɔ dàhnpɛ ǹgbò wilki! Képah-à pɩ, á kɩ fɛ̃ n yah pallɛ, kɩ á ní-ò yĩ́ sɔkɔ ǹdan wilki.» 
«Séswɔ̂ pèpɛy yɔ́ ńtɛ̃ náh n syi pye túkúnìnɛ. Séswɔ̂ túkù tɔ náh mó n syi pye pèplɔlɛ. 
Sésân pól n pnɛ sé pye sɔkɔ. Sènsóké pye náh fɛ̃ n wɛ kahrɩ sɔkɔ. Yrɛnsɛ̃ náh mó tɔ̃ fɛ̃ n wɛ n gbɛ gbɩ́yɛ̃́kɩ̀ sɔkɔ. 
Nɛ́pépɛ̌y yípéplɔ n wil ǹ nɩ sɔkɔ, wǎh nónó koh sah. Sɛ̃́ ó ye, nɛ́túkù tɔ tyítúkúnì n wil ǹ nɩ sɔkɔ, wǎh nónó koh sah. Névi nɩ-à yuku tir nɛ̂nɛ, képah ye n wil ǹ vyãh.» 
Képah sɔkɔ, Yesu tɔ̃ yo pé tyɩ́: «Sɔ̃́ pɩ, tɛ́ yě ńnɛ n ye ‹Tɛ̃̀ ǹgbɛ̃, Tɛ̃̀ ǹgbɛ̃nɛ›, tɛ́ mâh se nɛ̂ yo, yé náh képahlɛ n pɩ? 
Névi nɛ̂-à pɔ ńmɔ tyɩ́, tɛ́ ń wɛlɔ noh, tɛ́ mɛ sélɛ n pɩ, tɛ̃̀-á mɛ nɛ̂ kètɔ̀lɛ, mé kɩ ǹ tyɩ́ yo yé syah: 
À mɛ, á kɩ nɛ wɔ̀kɔ̀ mɔ́-ò yɔ́ sɔ̃́, nɛ̂-á swɔ tũ tiki náhnáh, tɛ́ wɔ̀kɔ̀ sõ légbɩ̃́ yõ. Tɛ́ ké sõ tyàhnɔ́-á sõ̀ plɛ sɛ̃́, képah ye, yɔ-á pɔ yuku, tɛ́ kwɔ, kè fwo kyɩ syɩ kénɛ wɔ̀kɔ̀ tyɩ́, tɛ́ kè náh tɛ̃ wɛ kè fɛ̃. 
Tɛ́, névi nɛ̂-à ń wɛlɔ noh, tɛ́ à náh sélɛ n pɩ, tɛ̃̀ wɔ mɛ, á kɩ nɛ wɔ̀kɔ̀ mɔ́-ò yɔ́ sɔ̃́, nɛ̂-á wɔ̀kɔ̀ yɔ̃ sɔkɔ mɔ sétáh kɔ́kɔ́ yõ swɔ tũ̀-ǹkɛ̃̂nɛ. Yɔ-á pɔ yuku, à fwo kyɩ syɩ kénɛ wɔ̀kɔ̀ tyɩ́, nɛ́npɔ́ swɔ, ké pól pirki kwɛ, à núkú mukuli pnɛ́pnɛ́.»
