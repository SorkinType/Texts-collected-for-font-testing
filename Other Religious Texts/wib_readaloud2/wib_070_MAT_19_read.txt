Matye.
19.
Yesu-á tyɛ ǹ wɛynɛ gbĩ́ nɛ̂nɛ, à yal Kalile mara sɔkɔ à yuku Nsyude mara tyah yɔ́ sɔkɔ, nɛ̂-á mɛ Nsyurdɛ̃ yɔ́nkahnɩnɛ. 
Tɔ́wû ǹgbɛ̃ yɔ́ mɛ ǹ sõ̀ tɛ̃ yuku, tɛ́ à mɛ pé yámpú dɛ mɔ nɛ́npɔ́. 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú yísyɔ́ nɔ de ǹ tnɔ̂ nɛ pé à mɔ yah, tɛ́ à piki yah nɛ, ké wɛ̃̀kɩ̀ se mɛ dĩ́ tyɩ́ à kɩ ǹ kyɩ fĩn mɔ, kàh sɔ̃́ mɛ tyídúkú nɛ̂ yõ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ: «Yé náh kè kar yah Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ, kèkõyṍ mɔ gbĩ́nɛ, Kèkõyṍ mɔ́-ò-á pè mɔ dĩ́ ànɛ̂ sêlɛ di? 
Wǎh nɛ, képah-á dĩ́-á kɩ yal ǹ sú ànɛ̂ ǹ yṹ tnɔ̂, tɛ́ kɩ kwrɔ mɔ ǹ kyɩlɛ, pé nímí kɩ pɩ wil gbɛ̃́nsê núkúlɛ. 
Kǎh pɩ sɛ̃́, nímí náh tɔ̃ pélɛ dɛ́, núkú-á pélɛ. Liyel-á nónó kwrɔ mɔ, nɛy káh pè gbɛkɩ.» 
Képah tɛ̃̀nɛ, Farisi wɛ̃̀kɩ̀ kɔ́pú à syah nɛ, ǹtɛ sɔ̃́ se pɩ tɛ́ Moyisi kè kõ sah nɛ, névi nɛ̂-à ǹ kyɩlɛ n fĩn mɔ, tɛ̃̀ ǹ fĩn mɔ mɔ tàrfyɔ̀ sɛ́bɛ́y pɩ à kɔ̃ à mó pyě à fĩn mɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Yépi nòhnɩ̀-ǹkɛ̃̂ yĩn ye Moyisi kɔ̃ à képah wɛ̃̀kɩ̀lɛ yé kɔ̃ kɛ̀. Ńkɛ̃́ sɛ̃́ náh se lésõ kélɛ ké tyàh-ǹsah sɔkɔ tɛ́. 
Tɛ́ ńmɔ kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à ǹ kyɩ fĩn mɔ nɛ́ fɔ̀fɩ́kɩ̀ tɛ̃̀ tyɩ́ yõ náh, tɛ́ sékáhkɩ́ tɔkɔ, fɔ̀fɩ́kɩ̀-á tɛ̃̀ n pɩ.» 
Képah sɔkɔ, Yesu tõ̀ sã́hpú yo ǹ tyɩ́ nɛ, dĩ́ yĩ́n-ńsah-à mó mɛ pɔ sah képah yõ à yuku ǹ kyɩ tyɩ́ tĩ̀nnɛ, sê tɔ́kɔ́-ńkɛ̃̂-á fṍnfṍ dò. 
Yesu mɛ yo pé tyɩ́ nɛ: «Wɛy nɛ̂-á mé kɩ n yo n pi yé tyɩ́ yɔ̀, no pól náh fɛ̃ ké syɩ́kɩ́-ńsah wɛ n pi. Ǹtɛ Liyel-á ké fɛ̃̀npɩ̃nɔ lékã́mnɛ nónó kɔ̃, pépi wɔ kɩ fɛ̃ ké syɩ́kɩ́-ńsah wɛ. 
Yé náh wɛ di, tyi yísyɔ́ mɛ nónó-á díbí kɔ̃ pè náh syɔ́lɛ n tɔkɔ: Túkù mɛ pé dúbí-ńsah dùkù sɔkɔ, pè náh mɛ díbí sɔkɔ. Túkù mɛ névye ye mɩ pélɛ pè kɔ̃ pè náh mɛ díbí sɔkɔ. Túkù mó sétɔ́kɔ́kɩ̀ yahle Liyel yõ̀tɔ̃rɩ́ yah yúkúnɔ́ yĩnnɛ. Nɛ̂-à kɩ fɛ̃ wɛy mɛ̀ syɩ́kɩ́-ńsah wɛ n pi, tɛ̃̀ kɩ ké syɩ́kɩ́-ńsah wɛ.» 
Képah tɛ̃̀nɛ, névye pɔ wɛ̃́npì yrɔ̃́lɛ Yesu tyɩ́ nɛ à ǹ gbã̀n sah pé yõ à Liyel ni pé tyɩ́. Ǹtɛ ǹ tõ̀ sã́hpú mɛ tyah dírílɛ n kõ pé yõ. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ: «Yé yɛ wɛ̃́npì kɔ̃ pè nɛ n pɔ n pɔ ńmɔ tyɩ́. Yé káh pé yahlɛ n kõ. Yé náh wɛ di, névye nónó dùkù-á mɛ wɛ̃́npì nɔ́pi tyɩ́ kɔ̃lɛ, pè kɩ Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi.» 
Képah tɛ̃̀nɛ, wǎh ǹ gbã̀n sah tɛ pé yõ, à yal nɛ́npɔ́. 
Képah tɛ̃̀nɛ, kètɛ̃̀npìkì yɔ́ nɔ de Yesulɛ tɛ́ à piki yah nɛ, kwɔ́-ò dĩ́, tir nɛ̂ sé pé yɩ̃nɛ pé pɩ tyípépɛ̌ynɛ pé mó tyɛ́-ńkɛ̃̂ min wɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Sɔ̃́ pɩ tɛ́ á ńnɛ n piki n yah pèpɛy tyɩ́ yõ? Pèpɛy núkú ó mɛ. Wáh kɩ mɛ ké tyɩ́ nɛ á min wɛ, Liyel tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ.» 
À mɛ nɛ Liyel tyi kõ̀nsàhnɔ́ nónó tãn se nɩ? Yesu mɛ yo ǹ tyɩ́ nɛ: «Káh nɛ́kókè mɔ, káh fɔ̀fɩ́kɩ̀ pɩ, káh yɩ̃be yɩ̃, káh mɩ̀nɩ̀ sah nɛy yõ. 
Yõ̀yénnɛ n sah á sú ànɛ̂ á yṹ yõ. Yɛ á ní-ò tyɩ́ nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ.» 
Képah tɛ̃̀nɛ, kénɛ kètɛ̃̀npìkì mɛ yo ǹ tyɩ́ nɛ, páh mɛ sépi tɛ̃̀ póllɛ n pɩ dò, kèyǎh se tɔ̃ mɛ wũ pé tyɩ́ pɩ pɩ tɛ̃̀nɛ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Wáh kɩ mɛ ké tyɩ́ nɛ á tyi pól tũ, gbã̀n yî nónó-á mɛ á tyɩ́, kyɩ sè yãm ǹ tɛ́ sé pɔ́ dibi yãm tãn kɔ̃. Képah sɔkɔ, á kɩ tahrɩ wɛ Liyel fṍ sɔkɔ. Képah-à pɩ, ǹ pɔ ńmɔ sõ̀ tɛ̃.» 
Wǎh képah tɛ̃̀ noh sɛ̃́, à sɔkɔ yahlɛ yɩ́kɩ́nsàhnɔ́lɛ. Kwâh nɛ́tɛ̃̂ ǹgbɛ̃ ye sõ̀ ǹnɛ képah ye. 
Képah sɔkɔ, Yesu mɛ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Kègbɩ yõ mé kélɛ n yo yé tyɩ́, tàh Liyel tɔ̃́rɩ́ nɔlɩ wɛ́nyõnɔlɛ, kè kɩ tah dɛ́. 
Mé kélɛ n tɔ̃ n yo yé tyɩ́, yã̀hmɩ́kɩ̀ dénwìlnɔ́lɛ sèndi pwã̀h sɔkɔ, képah ńtɛ̃ kɩ dɔ̀dɔ̀ kal tàh Liyel tɔ̃́rɩ́ nɔlɩ wɛ́nyõnɔlɛ.» 
Yesu tõ̀ sã́hpú-á képah noh sɛ̃́, pé gbã̀n kah du. Pè mɛ nɛ, ǹtɛ nɛ̂ wɔ se kɩ mó fɛ̃ pwah nɩ? 
Yesu mɛ yɔ̃ pè yah, tɛ́ yo pé tyɩ́ nɛ: «Névye tɛ̃̀ wɔlɛ, pè náh fɛ̃ kè pɩ n pi. Ǹtɛ Liyel kɩ fɛ̃ tyi pól pɩ.» 
Képah sɔkɔ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, à yah, pépi-á pé tyi pól sah tɛ́ ǹmɔ sõ̀ tɛ̃. Pépi tyɩ́ se kɩ n pɩ n pi sɔ̃́ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Kègbɩ yõ mé kélɛ n yo yé tyɩ́, Névi Pídĩ́-à pɔ kɔ̃ tɛ̃ ǹ tɔ̃́rɩ́ kǒ yõ kèkõyṍ késã́n sɔkɔ gbĩ́ nɛ̂nɛ, yépi nónó-á ńmɔ sõ̀ tɛ̃, yé tɔ kɩ kɔ̃ tɛ̃ tɔ̃́rɩ́ klǒ gbɔ̃ninɔ́ yõ kɩ Yisrayel no yìkìnìnɔ́ gbɔ̃ninɔ́ yah tɛ̃ n pi. 
Ńmɔ yĩn yõ, névi nɛ̂ tɛ̃̀-à fyɔ̀ mɛ yal ǹ gbô sɔkɔ ò, ǹ yṹnpyé tnɔ̂ ò, ǹ dósyɔ́ tnɔ̂ ò, ǹ sú tnɔ̂ ò, ǹ yṹ tnɔ̂ ò, ǹ wɛ̃́npì tnɔ̂ ò, ǹ swɛ̃y tnɔ̂ ò, tɛ̃̀ kɩ sɛ̃́nknɔnɔ lèkwɛ̌ wɛ, tɛ́ kɩ se tɔ̃ tyɛ́-ńkɛ̃̂ min wɛ. 
Ǹtɛ nɛ́gbro náhnáh kɩ n pɩ n pi nɛ́tɛ́lɔ́lɛ, tɛ́ nɛ́tɛ́lɔ́lɛ náhnáh kɩ n pɩ n pi nɛ́gbrolɛ.»
