Markɩ.
9.
Tɛ́ tɔ̃ ǹ syɩ yo pé tyɩ́: «Mé kélɛ n yo n dal n mɔ yé tyɩ́ kègbɩ yõ: Yépi nónó-á ásõ̂, yé túkù náh n ku n pi, tɛ́ kɩ *Liyel tɔ̃́rɩ́ wɛ yĩ́nɛ, pɔ́ntɛnɔlɛ. Kè kɩ n pi tɛ̃́nwɛnɔ ǹgbɛ̃nɛ.» 
Sèpĩ̀n kénúkò-ò sɔ́kɔ́ntɛnɔ náh, Yesu mɛ Pyɛrɩ ànɛ̂ Nsyakɩ ànɛ̂ Nsyɔ̃ tɛ̃, à yuku pélɛ yɔ̃lɩ ǹgbɛ̃ yɔ́ yõ. Pépi ó ye sõ̀ pé yahlɛ nɛ́npɔ́ sɔkɔ. Tɛ́, ǹ dùkù mɛ vi mɔ pé yah sɔkɔ. 
Ǹ flɔ piki, à n tɛ̃ pérpér. Ké névi náh mɛ kèkõyṍ kɩ fɛ̃ kwâh yɔ́ ńtɛ̃ piki n pi sɛ̃́ntɛ̃̀nɛ. 
Képah sɔkɔ, *Yeli ànɛ̂ *Moyisi wil pé yah, mɛ n yãh ànɛ̂ Yesulɛ. 
Képah tɛ̃̀nɛ, Pyɛrɩ mɛ yo Yesu tyɩ́ nɛ, kwɔ́-ò dĩ́, kǎh yɩ̃nɛ pé tɛ̃ ásõ̂. Páh kɩ yèpyenɔ tɔ̃́nɔ́ pɩ ásõ̂, núkúlɛ ǹmɔ tyɩ́lɛ, núkúlɛ Moyisi tyɩ́lɛ, núkú mó pɩ Yeli tyɩ́lɛ. 
Pyɛrɩ náh sõ̀ pɩ̃, wǎh nɛ̂ yo n pi, képah ye. Tíkí sõ̀ ǹmɔ ànɛ̂ ǹ kɔ́pú nɛ́pĩ̂ nímí kah tɛ̃. 
Pǎh mɛ képah sɔkɔ, ńyĩ́rkì yɔ́ pɔ fob mɔ pé yõ. Kénɛ ńyĩ́rkì sɔkɔ, yĩ́npɔ̃́ yɔ́ n wil pè n nohnɩ nɛ: «Ǹmɔ nɛ̂-á yɔ̀, ńmɔ nɩ̀dɛ̀ Pídĩ́ ye ǹnɛ. Yé nɛ n nohnɩ ǹ tyɩ́!» 
Kénɛ nɩ́ yõ, *tõ̀ sã́hpú-á yah kɔ̃ pé gbɔ̀plɔ sɔkɔ, pè náh tɔ̃ névi wɛ. Yesu núkúní ó sõ̀ wũ ànɛ̂ pélɛ. 
Pǎh mɛ yɔ̃lɩlɛ n tiki gbĩ́ nɛ̂nɛ, Yesu pè sni nɛ, pǎh tir nɛ̂ wɛ, pè káh kè yãh névye syah, kè pɔ nɔ *Névi Pi yúkúnwìlnɔ́ tyɩ́ lékyɩ̂ wrɔ́. 
Pè mɛ snínɔ́ mɛ̀ tɛ̃ yĩn, tɛ́ mó tɔ̃ mɛ wɛ̃nɛ n piki n yah nɛ, kû se à n ye lékyɩ̂ wrɔ́ yúkúnwìlnɔ́lɛ nɩ? 
Képah tɛ̃̀nɛ, pè tyah ǹnɛ n piki n yah nɛ, kû se pɩ, tɛ́ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú nɛ, Yeli ǹgbò-á yɩ̃nɛ à pɔ? 
À yo pé tyɩ́: «Yeli ǹgbò ye yɩ̃nɛ à pɔ tyi pól tɔ̃ sah. Ǹtɛ, kû pɩ kè wãrkɩ sah *Liyel wɛy sɛ́bɛ̃́nɔ́ sɔkɔ nɛ, Névi Pi-á kɩ yèvyãhrɩ pɩ náhnáh, tɛ́ pè kɩ ǹnɛ n yilɔ nɩ? 
Ǹtɛ, mé kélɛ n yo n dal n mɔ yé tyɩ́ nɛ, Yeli-á pɔ tɛ. Tɛ́, névye-á à pɩ pé dyɔ dyɔ tɛ̃̀nɛ. Sɛ̃́ ye sõ̀ kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ̃́nɔ́ sɔkɔ ǹ tyɩ́ yõ.» 
Képah sɔkɔ, pǎh yĩni tɛ̃ *tõ̀ sã́hpú tɛ́lɔ́ tnɔ̂, pè tɔ́wû ǹgbɛ̃ yɔ́ wɛ, pè mɛ pè kore vi, tɛ́ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú mɛ yékã̂blɛ n pɩ tõ̀ sã́hpú nɔ́pilɛ. 
Névye-á núkú Yesu wɛ gbĩ́ nɛ̂nɛ, kè pɩ pé tyɩ́ gbɛ̃̀ndùnìnɛ. Pè to kyɩ wɛkɩ ǹ tyɩ́. 
Yesu mɛ ǹ tõ̀ sã́hpú piki yah nɛ, kwâh nɛ̂ yékã̂b se pè n pɩ pélɛ nɩ? 
Ǹtɛ, dĩ́ yɔ́ sõ̀ mɛ tɔ́wû sɔkɔ, ǹmɔ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, páh pɔ pé pídĩ́nɛ ǹ tyɩ́. *Sétõ yɔ́-á mɛ ǹ sɔkɔ, mɛ à pɩ kèdɔ̀kɩ̀lɛ. 
Kě ǹnɛ n kwel n min n kɔ̃ tã̀ntã̀n pól sɔkɔ sétáh. Gbèfɔ̀kɔ̀píkínì n nɛ n wil ǹ vyãh, tɛ́ ǎ ǹ yĩ́nílɛ n tah, ànɛ̂ ǹ wil n kɔkɔ. Képah-á, pé pɔ ǹnɛ ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè sétõ mɛ̀ yĩni wilki ǹ sɔkɔ, tɛ́ pè náh tɛ̃ wɛ. 
Képah tɛ̃̀nɛ, Yesu mɛ yo pé tyɩ́ nɛ, Liyel yõ sɛ̃́-ńkɛ̃̂ névye, pé se kɩ tɛ̃ pélɛ pópó, kɩ pɔ syɩ gbĩ́ nɛ̂ tyɩ́ nɩ? Pé se kɩ pè so sah, kɩ pɔ syɩ gbĩ́ nɛ̂ tyɩ́ nɩ? Ńkɛ̃, pè pɔ ǹnɛ pé kɔ̃. 
Pè mó pɔ ǹnɛ à kɔ̃. Sétõ-á fyɔ̀ Yesu wɛ, à piri yike kègbɩlɛ, à syɩkɩ sétáh, à n pil n kɔ̃ gbèfɔ̀kɔ̀píkínìnɛ ǹ vyãh. 
Képah sɔkɔ, Yesu mɛ piri sú piki yah nɛ, gbĩ́ nɛ se kè tyah ǹ tyɩ́ nɩ? À mɛ yo ǹ tyɩ́ nɛ, ǹ wɛ̃́npir gbĩ́ dùkù-á. 
Kèyahlɛ, sétõ-á ńkɛ̃́nɛ à tɔkɔ n min n dah n mɔ nyàh, képah náh pɩ, ni sɔkɔ, ǹ kò-ǹsahlɛ n yah n kɔ̃. Ǹtɛ, wàh kɩ fɛ̃ n tɛ̃ n pi, à pé yãm yah, à mó pé yohnɩ mɔ. 
Képah sɔkɔ, à mɛ yo dĩ́ mɛ̀ tyɩ́: «Máh kɩ fɛ̃ n tɛ̃ n pi wɔ̀...! Tyi pól kɩ fɛ̃ pɩ Liyel yõ sɛ̃́-ò tyɩ́.» 
Nɛ́npɔ́ swɔ, piri sú gbo mɔ nɛ: «Mé sɛ̃ ké yõ, tɛ́ kè mɛ tah ń tyɩ́. Ń yohnɩ mɔ ké sɔkɔ.» 
Képah sɔkɔ, Yesu-á névye wɛ, pè n to n tuke pé yõ sɛ̃́, à dírí kõ *sétõ mɛ̀ yõ, tɛ́ yo ǹ tyɩ́ nɛ, ǹmɔ nɛ̂-á névilɛ n pɩ kèdɔ̀kɩ̀ ànɛ̂ níkítũ̂nɛ, à wil piri mɛ̀ sɔkɔ! Tɛ́ à káh tɔ̃ de yah ǹ sɔkɔ! 
Wǎh yo sɛ̃́, sétõ mɛ gbo mɔ, tɛ́ piri yike kègbɩlɛ, tɛ́ mó wil ǹ sɔkɔ. Piri núkú pɩ tɛ̃ kúnɔ́ sɔ̃́, fɔ́ɔ́ névye náhnáh n yo nɛ, wǎh ku. 
Ǹtɛ, Yesu mɛ à tɛ̃ ǹ gbõ̀ tyɩ́, à yilki sah. Piri yuku yĩn. 
Yesu-á de wɔ̀kɔ̀ gbĩ́ nɛ̂nɛ, ǹmɔ ànɛ̂ ǹ tõ̀ sã́hpú ye sõ̀ wũ pé yahlɛ, tɛ́ pè mɛ à piki yah nɛ, kû se pɩ, tɛ́ pépi náh tɛ̃ wɛ sétõ mɛ̀ yĩni wilki piri sɔkɔ nɩ? 
À mɛ yo pé tyɩ́: «Nírí ó sɔkɔ ye kénɛ sétã̀n dùkù kɩ fɛ̃ yĩni.» 
Pǎh yal nɛ́npɔ́, pè mɛ Kalile kwlolɛ n kah, tɛ́ Yesu náh sõ̀ mɛ ké tyɩ́ nɛ, pè pé mɛ́-ńsah pɩ̃. 
Tir yɔ́ náh, à sõ̀ mɛ ǹ *tõ̀ sã́hpúlɛ n kwɔ, tɛ́ mɛ n yo pé tyɩ́: «*Névi Pi kɩ n dahbɩ n pi névye gbõ̀. Pè kɩ à ko mɔ, tɛ́ sèpĩ̀n tɔ̃́-ò sṍlɛ, à kɩ yuku.» 
Ǹtɛ, ǹ tõ̀ sã́hpú náh sõ̀ mɛ wɛy mɛ̀ tyɩ́ yah wɛ, tɛ́ tíkí sõ̀ mɛ pè tɛ̃ ǹ píkínyahnɔlɛ. 
Képah náh, pè yĩni tɛ̃ Kapɛrnayim kwil sɔkɔ. Pǎh mɛ wɔ̀kɔ̀ sɔkɔ gbĩ́ nɛ̂nɛ, Yesu ǹ *tõ̀ sã́hpú piki yah nɛ tir nɛ̂ yékã̂b se sõ̀ pè n pɩ wɛ̃̀kɩ̀ yah nɩ? 
Ǹtɛ, pè náh tir wɛ yo. Tir yɔ́ náh, pè sõ̀ mɛ yékã̂blɛ n pɩ wɛ̃̀kɩ̀ yah nɛ, pé pɩ̃ pépi wrɔ́ nɛ̂ tɛ̃̀-á kal. 
Képah sɔkɔ, Yesu kɔ̃ tɛ̃, tɛ́ ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye, tɛ́ yo pé tyɩ́: «Nɛ̂-à mɛ ké tyɩ́ nɛ, pé pɩ yah yétɛ̃̀nɛ, ǹmɔ yɩ̃nɛ à pɩ pé no náhtũnɛ, à mó pɩ tõ̀npɩlɛ pé tyɩ́.» 
Képah tɛ̃̀nɛ, à wɛ̃́npir yɔ́ tɛ̃ pɔ pé nɩyṍ. Tɛ́ à tɛ̃ kõhlɩ ǹ nɩ yõ, tɛ́ yo ǹ tõ̀ sã́hpú tyɩ́: 
«Névi nɛ̂-à wɛ̃́npir mɛ̀ kénkɔ̃ tɛ̃́nɔ́ plɛ́ ńmɔ yĩn yõ, ńmɔ ye tɛ̃̀ tɛ̃ plɛ́. Névi nɛ̂ tɔ-à ńmɔ tɛ̃́nɔ́ plɛ́, ńmɔ ó náh tɛ̃̀ tɛ̃ plɛ́: Ǹtɛ nɛ̂-á tɛkɩ mɔ ńmɔlɛ, ǹmɔ ye à tɛ̃ plɛ́.» 
Képah sɔkɔ, Nsyɔ̃ mɛ yo Yesu tyɩ́ nɛ, pé kwɔ́-ò dĩ́, pépi-á dĩ́ yɔ́ wɛ, à mɛ *sétã̀nnɛ n yĩni n wilki ǹ yĩn yõ. Tɛ́, à náh sõ̀ mɛ pépilɛ. Képah-á pé kɔ̃, pé nɛ pé à yĩni mɔ ké pɩ́nɔ́lɛ. 
Yesu mɛ yo pé tyɩ́: «Yé káh à yĩni mɔ. Névi náh fɛ̃ gbǐl pɩ n pi ńmɔ yĩn yõ, tɛ́ kɩ tɔ̃ ńmɔ yĩnnɛ n yɩkɩ nɛ́npɔ́ swɔ. 
Tir yɔ́ náh, á sépɛ náh nɛ̂nɛ, á pèpɛy nɛ ye ǹnɛ. 
Kègbɩ yõ, mé kélɛ n yo yé tyɩ́: Yáh pɩ *Krista tyɩ́lɛ, névi nɛ̂-à ni ńgbíyĩ́nɛ yé kɔ̃ yé wɔ képah yĩnnɛ, ké pèpɔrɩ kɩ dyah ólɛ à dyah. 
Wɛ̃́npì nónó-á sɛ̃ ńmɔ yõ, névi nɛ̂-à pé sɔkɔ nɛ yɔ́ yõ yɩkɩ dah mɔ tyípêl sɔkɔ, kè kɩ sõ̀ fṍnfṍ pè dáh-ǹgbɛ̃ toh tɛ̃̀ yègbɩ sɔkɔ, pè kyɩ min dah mɔ yɔ́ntî ǹgbɛ̃ sɔkɔ. 
Á gbõ̀-à n dah n mɔ álɛ tyípékè sɔkɔ, kè kãm n milki, ǹ tɛ́ n kyɩ n de gbõ̀ núkúlɛ tyɛ́-ńkɛ̃̂ min sɔkɔ. Kèpah kɩ fṍnfṍ kal á gbã̀n nínì kyɩ́ndenɔlɛ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ.  
Nɛ́npɔ́ sɔkɔ, kèfyɔ̃ nónó-á willɛ n yõ, sè náh n ku, ànɛ̂ ké nyàhnpèkè náh n ni. 
Á gbáhkɩ̀-à n dah n mɔ álɛ tyípékè sɔkɔ, kè kãm n milki, ǹ tɛ́ n kyɩ n de gbáhkɩ̀ núkúlɛ tyɛ́-ńkɛ̃̂ min sɔkɔ. Képah kɩ fṍnfṍ kal á gbyẽ̂h nínì mínndahnmɔnɔlɛ nyàhnpèkè sɔkɔ.  
Nɛ́npɔ́ sɔkɔ, kèfyɔ̃ nónó-á willɛ n yõ, sè náh n ku, ànɛ̂ ké nyàhnpèkè náh n ni. 
Á yĩ́-à n dah n mɔ álɛ tyípékè sɔkɔ, kè wilki n milki, ǹ tɛ́ n kyɩ n de yĩ́ núkúlɛ *Liyel yõ̀tɔ̃rɩ́ sɔkɔ. Képah kɩ fṍnfṍ kal á mínndahnmɔnɔ yõ yɩ̃́ nínɔ́lɛ nyàhnpèkè sɔkɔ. 
Nɛ́npɔ́ sɔkɔ, kèfyɔ̃ nónó-á willɛ n yõ, sè náh n ku, ànɛ̂ ké nyàhnpèkè náh n ni. 
Képah ye, no pól kɩ tɔ̃ sah nyàhlɛ, sɔ̃́-á pnɛ n pɩ kélɛ. 
Pnɛnɛ, kwâh pèpɛy ye kélɛ. Tɛ́ ké syɩ̃́nsyɩ̃́-à wil ké sɔkɔ, sɔ̃́ mɛ yé kɩ tɔ̃ fɛ̃ pɩ kélɛ, tɛ́ kè kɩ tɔ̃ nɔ? Yé pnɛ mɔ yé gblɔ̌y sɔkɔ, yé tɛ̃ wɛ̃ tyɩ́ yèvɩnɩ sɔkɔ.»
