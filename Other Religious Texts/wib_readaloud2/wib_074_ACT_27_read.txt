Tyípɩ́nɩ̀.
27.
Képah sɔkɔ, kǎh pɔ sah núkú yõ nɛ, áyáh kɩ yuku n pi Wrome kwíltãm yõ̀tɛ̃̀ ǹgbɛ̃ Sesar tyɩ́ Yitalɩ gbɛ̃̀nyah sɔkɔ, pè mɛ Pole ànɛ̂ wɔ̀kɔ̀npèkè sɔkɔ névye yísyɔ́ dahbɩ sràsyíbí yõ̀tɛ̃̀ yɔ́ gbõ̀, nɛ à sɔkɔ álɛ. Dĩ́ mɛ̀ yĩn ye sõ̀ Nsyulyusilɛ. Ǹmɔ ye sõ̀ Wrome tãn sràsyíbí tĩ̀nkì yɔ́ yõ̀tɛ̃̀nɛ nɛ̂-á pè n ye Wokustilɛ. 
Képah tɛ̃̀nɛ, á de Adramiti kwil krótókè ǹgbɛ̃ yɔ́ sɔkɔ, à sɔkɔ. Krótókè mɛ̀ kɩ sõ̀ n syɔ̃ n pi Asi gbɛ̃̀nyah krótónè yĩ́n-ńtã̂n sɔkɔ. Arisitarkɩ sõ̀ mɛ álɛ, nɛ̂-á n wil Masedonɩ gbɛ̃̀nyah kwil yɔ́ sɔkɔ pè n ye Tesalonikilɛ. 
Ké dĩndallɛ, á kyɩ de Sidõ kwil. Ǹtɛ Nsyulyusi sõ̀ mɛ Pole tɛ̃́nɔ́ plɛ. Kǎh pɩ sɛ̃́, à wɛ̃̀kɩ̀lɛ Pole kɔ̃ nɛ, à kyɩ yah ǹ kódíbí tyɩ́, pè kɔ̃ pè à wɛ n yohnɩ n mɔ. 
Képah náh, áyáh yal nɛ́npɔ́, yã́hkɩ́ sõ̀ mɛ álɛ n fwo. Kǎh pɩ sɛ̃́, á syɔ̃ gbɛ̃̀nyah yɔ́ gbɔ̀pɔ pè n ye Syiprilɛ, pnɛ yɔ nɩyṍ. Képah sõ̀ mɛ yã́hkɩ́ yah kõ sah á tyɩ́ kwéy. 
Tɛ́ yɔ̃ sɔkɔ pnɛ yɔ sɔkɔ, à syɔ̃ Silisi gbɛ̃̀nyah ànɛ̂ Pãnfili gbɛ̃̀nyah gbɔ̀pɔ, à kyɩ tiki Mira kwil, Lisi gbɛ̃̀nyah sɔkɔ. 
Áyáh mɛ nɛ́npɔ́, Wrome tãn sràsyíbí yõ̀tɛ̃̀ mɛ Alɛkisãdri kwil krótókè ǹgbɛ̃ yɔ́ wɛ, nɛ̂-á kɩ sõ̀ n yuku n pi Yitalɩ gbɛ̃̀nyah sɔkɔ. À mɛ á dah mɔ képah sɔkɔ. 
Á sèpĩ̀n náhnáh pɩ, tɛ́ á náh sõ̀ mɛ fɛ̃ n to súú. Á yèvyãhrɩ kah pɩ, tɛ́ pyě yĩni tɛ̃ Knidi kwil dé-ńsahlɛ yúkɛy. Ǹtɛ yã́hkɩ́-á sõ̀ mɛ álɛ n fwo nɛ́npɔ́ tĩ̀n yúkúnɔ́lɛ sɛ̃́, gbɛ̃̀nyah nɛ̂-á pè n ye Krɛtɩlɛ pnɛ yɔ nɩyṍ, á syɔ̃ képah dónmɔ-ńsah yɔ́ gbɔ̀pɔ pè n ye Salmonenɛ. Nɛ́npɔ́ sõ̀ mɛ yã́hkɩ́ yah kõ sah á tyɩ́ kwéy. 
Á yèvyãhrɩ pɩ nɛ́npɔ́ sɔ́kɔ́-ńsah sɔkɔ, à kyɩ yĩni tɛ̃ Krɛtɩ fɩ̃́ yɔ́ sɔkɔ pè n ye Krótónè yĩ́n-ńsah pèpɛynɛ. Lasɩ kwil mɛ nɛ́npɔ́ tnɔ̂. 
Légbĩ́nɛ, wrɔ́ náhnáh sõ̀ mɛ sɔkɔ tɛ. Tɛ́ Nsyifunɔ fílsã̀hkɩ̀ sõ̀ níkì tɛ̃ gbĩ́-á sõ̀ mɛ sɔkɔ tɛ sɛ̃́, yã́hkɩ́ sõ̀ mɛ kõ. Kǎh pɩ sɛ̃́, krótókè tónɔ́ sõ̀ mɛ pɩ dlolɛ. Képah ye, Pole mɛ ké gbáhdyâh di pé tyɩ́ nɛ, 
pé kódíbí, péwɔ-á kè wɛ tɛ nɛ, krótókè tónɔ́ náh tɔ̃ fɛ̃ n yɔ̃ n sɔkɔ n pi, nɛ́ dlo náh wɛ ké sɔkɔ. Tɛ́ kè náh n yĩn n pi krótókè ànɛ̂ ké sɔkɔ vyɔ́ yɩ́kɩ́nɔ́ képékè ó tyɩ́ dɛ́, pépi tɔ min-á kɩ n syah n wũ ké sɔkɔ. 
Ǹtɛ sràsyíbí yõ̀tɛ̃̀ dĩ́ sõ̀ mɛ sɛ̃ krótókè tó-ò dĩ́ ànɛ̂ ké yõ̀tɛ̃̀ wɛy yõ sɔkɔ, à kal Pole tyɩ́lɛ. 
Tɛ́ Krɛtɩ gbɛ̃̀nyah krótónè yĩ́n-ńsah mɛ̀ náh sõ̀ mɛ plɛ kwɔ̀vɩrɩ gbĩ́nɛ. Képah ye, névye kan sõ̀ mɛ ké tyɩ́ nɛ, kàh kɩ fɛ̃ n pɩ n pi, pé yĩni tɛ̃ ké krótónè yĩ́n-ńsah yɔ́ sɔkɔ pè n ye Fenikisilɛ, pé kyɩ kwɔ̀vɩrɩ gbĩ́ pɩ nɛ́npɔ́. Nɛ́npɔ́ sõ̀ mɛ Krɛtɩ gbĩ̀yiki tĩ̀nnɛ. 
Képah sɔkɔ, yã́hkɩ́ yɔ́ tyah n kõ n wil fálfál yɔ nɩyṍ tĩ̀nnɛ. Képah tɛ̃̀nɛ, pè sõ̀ mɛ n sõ nɛ páh nɛ̂ yah tɔkɔ, kǎh kɩ fɛ̃ pɩ, tɛ́ mɛ krótókè yĩn mɔ mɔ vyɔ́ngbílkì dɔkɔ mɔ, krótókè kɔ̃ kè tyah n sɔkɔ, à kyɩ tɔrɩ sɔkɔ Krɛtɩ gbɛ̃̀nyah gbɔ̀pɔlɛ. 
Dǒ kwéy, yã́hpékè yɔ́ kõ wil ymɔ yõ à tiki pɔ mɩ á sɔkɔ. Yã́hkɩ́ mɛ̀ yĩn ye sõ̀ Yerakilõnɛ. 
Kè pɔ á krótókè tɔkɔ n yuku gbɛ̃̀ntɛ̃̀náhlɛ, á náh tɛ̃ wɛ ké yah kõ. Á mɛ á gblɔ̌y fĩn mɔ kè kɔ̃. 
Kè kyɩ sɔkɔ álɛ gbɛ̃̀nyahpír yɔ́ tnɔ̂ pè n ye Kodalɛ, pnɛ yɔ nɩyṍ. Kénɛ gbɛ̃̀nyah sõ̀ mɛ yã́hkɩ́ yah kõ sah á tyɩ́ kwéy. Kǎh pɩ sɛ̃́, krótópír nɛ̂-á sõ̀ mɛ tóhnmɔnsàhnɔ́lɛ krótókè ǹgbɛ̃ tyɩ́, á gbah tɛ̃, à képah wɛ yɔrɩ dah mɔ ké sɔkɔ yèvyãhrɩlɛ. 
Ké dáhnmɔntɛnɔ náh, á mnɔlɛ krótókè ǹgbɛ̃ pukubi kore vi nɛ kè káh kwɔkɩ. Tɛ́ tíkí-á sõ̀ mɛ á tɛ̃ nɛ á káh kyɩ n nir Libi gbɛ̃̀nyah sèmìyã́hrɩ́ sɔkɔ sɛ̃́, képah ye á krótókè yõ̀tah flɔ kũ, sépi nónó gbõ̀ yõ-á ńkɛ̃́nɛ yã́hkɩ́ kélɛ n tu n mɔ súú, tɛ́ mɛ á gblɔ̌y fĩn mɔ yã́hkɩ́ kɔ̃ kè n yuku álɛ. 
Dĩndallɛ, yã́hkɩ́ sõ̀ mɛ yɔ̃ sɔkɔ álɛ n kah fwo sɛ̃́, kǎh pɩ sɛ̃́, pè krótókè tɔ́kɔ́bɩ́ wilki min dah mɔ pnɛ yɔ sɔkɔ. 
Sèpĩ̀n tɔ̃́-ò sõlɛ, krótókè tópú tɔ mɛ ké tõ̀npɩ̀vyɔ́ túkù wilki ké sɔkɔ, à min dah mɔ pnɛ yɔ sɔkɔ pé gblɔ̌y. 
Kè sõ̀ mɛ pɩ sèpĩ̀n náhnáhlɛ, tɛ́ á náh gbĩ wɛ yah yĩ́nɛ, képah náh pɩ, ńwɔ̃́ kèpyelɛ. Yã́hkɩ́ sõ̀ mɛ peke tɛ̃ sɛ̃́, fɔ́ɔ́ á náh tɔ̃ mɛ pɔ n sõ nɛ, áyáh kɩ pwah. 
Kǎh pɩ sɛ̃́, á sõ̀ mɛ sèpĩ̀n náhnáh pɩ yõke yṍ-ńkɛ̃̂nɛ. Képah sɔkɔ, Pole mɛ yuku yĩn no pól nɩyṍ, tɛ́ nɛ ńkɛ̃́, pé kódíbí, pǎh sõ̀ yɩ̃nɛ pè sah péwɔ yóré yõ, pé mó wũ Krɛtɩ gbɛ̃̀nyah. Képah-à sõ̀ mɛ pɩ, páh dlo nónó gbal, ànɛ̂ yɩ́kɩ́nɔ́ nónó-á pɩ pé tyɩ́ yɔ̀, sè náh náhkɩ pé yĩni n tɛ̃ n pi dò! 
Ǹtɛ núkúnúkú, páh kélɛ n yah n kɔ̃ pé tyɩ́ nɛ, pè pé nɩ syãh, ńkɛ̃́ ǹnɩ̂ yɔ́ ńtɛ̃ yõ náh yɩ̃ n pi à wil krótókè tyɩ́ sɔkɔ. 
Ńkɛ̃́, Liyel nɛ̂ tyɩ́ yõ-á péwɔlɛ pé tõ̀lɛ n pɩ ǹ tyɩ́, ǹmɔ yĩ̂nyõ tõ̀npɩ yɔ́-á wil pé yah lékã̂h mɛ̀ sɔkɔ, 
tɛ́ yo pé tyɩ́ nɛ: «Káh tíkí pɩ, Pole! Á kɩ n yĩn n pi Wrome kwíltãm yõ̀tɛ̃̀ Sesar yah sɔkɔ, kɩ á yṍtyí yo ǹ tyɩ́. Tɛ́ névye nónó pól-á mɛ n yuku álɛ krótókè sɔkɔ, áwɔ gbõ̀ yõ, pé sɔkɔ yɔ́ ńtɛ̃ yõ náh n yɩ̃ n pi.» 
Képah-á, péwɔ Pole sɛ̃́nɔ́-á mɛ Liyel yõ nɛ, kǎh kɩ n pɩ n pi sɔ̃́-á ǹ yĩ̂nyõ tõ̀npɩ yo kélɛ pé tyɩ́. Képah-á, pè pé nɩ syãh pé kódíbí. 
Ǹtɛ krótókè tɛ̃̀ wɔ-á kɩ kyɩ n yɩkɩ n pi pé tyɩ́ yɔ nɩyṍ kwil yɔ́ sɔkɔ dɛ́! 
Légbĩ́nɛ, yã́hkɩ́ sõ̀ mɛ yɔ̃ sɔkɔ álɛ n fwo n yuku sɛ̃́ pnɛ yɔ yõ, nɛ̂-á pè n ye Mediteranɩnɛ, fɔ́ɔ́ à pɔ syɩ ké sèpĩ̀n gbɔ̃nyã́h-ò sõ lékã̂h tyɩ́. Tɛ́ lénɩ mɔ́nɔ́ wrɔ́ sɔkɔ, krótókè tópú sõ̀ kè sõ nɛ, áyáh mɛ de n tɛ kènkwã̀h yõlɛ, 
tɛ́ yɔ́ntî sãh sãh kwâh min dah mɔ yɔ sɔkɔ, tɛ́ kè wɛ ké súkúlínɔ́ mɛ mɛ́tɛ́rnɔ́ yõgbãm pɔ́ kwǎrninɔlɛ. Áyáh yuku yahlɛ kwéy, pè tɔ̃ kè min dah mɔ, tɛ́ kè wɛ ké súkúlínɔ́ mɛ mɛ́tɛ́rnɔ́ túkúrí pɔ́ kwǎrtɔ̃nɔlɛ. 
Pè sõ̀ mɛ tíkílɛ n pɩ nɛ, krótókè káh pɔ kyɩ syɩ dyêh yísyɔ́ tyɩ́, kè yɩkɩ. Képah ye, pè krótókè yĩn mɔ mɔ vyɔ́ngbîl ǹyẽ̂h dah mɔ krótókè náh tĩ̀nnɛ, tɛ́ mɛ tyah dálnɔ́ ólɛ n dɛkɩ. 
Ǹtɛ krótókè tópú sõ̀ mɛ n yah n kɔ̃ tó-ńsahlɛ. Kǎh pɩ sɛ̃́, pè krótópír tiki mɔ yɔ yõ, tɛ́ nɛ páh nɛ pé krótókè yĩn mɔ mɔ vyɔ́ngbîl dah mɔ krótókè yah tĩ̀nnɛ. 
Képah sɔkɔ, Pole mɛ yo sràsyíbí ànɛ̂ pé yõ̀tɛ̃̀ dĩ́ tyɩ́ nɛ ńkɛ̃́, díbí nɔ́pi-à kɩ wṹ-ńkɛ̃̂nɛ krótókè sɔkɔ, pépi náh tɛ̃ wɛ n pwah n pi dɛ́! 
Képah sɔkɔ, mnɔ nónó-á sõ̀ tɛ̃ sah krótópírlɛ, sràsyíbí núkú sépi kɔ̃lɩ mɔ kè kɔ̃ kè sɔkɔ. 
Képah náh, tyah-á pópó n dal, Pole kè yah kɔ̃ no pól tyɩ́ nɛ ńkɛ̃́, pè yõke yõ. À nɛ ńkɛ̃́, ké sèpĩ̀n gbɔ̃nyã́h-ò-á kwɛynɛ yɔ̀, pé yáh-ńsah mɛ tyɛ tɛ̃ pópó, pè náh mɛ fɛ̃ yõkelɛ n yõ. 
Ǹtɛ yõke vyãh-á mó tɔ̃ mɛ pé min pwáhnɔ́ tyɩ́ sɔkɔ. Képah-á, páh kélɛ n yah n kɔ̃ pé tyɩ́ nɛ, pè yõke yõ. Ńkɛ̃́, pé sɔkɔ nɛy yɔ́ ńtɛ̃ yõ̀yén pɔ́ núkú fyé náh n mɔ n pu n pi. 
Képah yóntɛnɔ náh, Pole búrn tɔkɔ, tɛ́ wɛkɩ Liyel tyɩ́ no pól yĩ́ yah, tɛ́ kè kwɔ à tyah n tah. 
Képah sɔkɔ, no pól nɩ sũ, pé tɔ mɛ yõke yõ. 
Névye nónó-á sõ̀ mɛ krótókè mɛ̀ sɔkɔ, á pól sõ̀ mɛ nɛ́pĩ̂ lègbãm tikitɔ̃nɩ́ pɔ́ gbãm pɔ́ kénúkòlɛ (276). 
Á pól-á yõ tɔ̃ tɛ gbĩ́ nɛ̂nɛ, á sèmukɔ wilki min dah mɔ yɔ sɔkɔ krótókè kɔ̃ kè pɩ fáhfáh. 
Képah tɛ̃̀nɛ, tyah-á pɔ dal krótókè tópú náh sõ̀ mɛ kènkwã̀h yõ yah pɩ̃. Pǎh pɔ yɔ kéngbárkɩ̀ yɔ́ kyɔmɩ wɛ, nɛ́npɔ́ mɛ wɛ́nyĩ́n-ńsahlɛ, pè nɛ páh kɩ yuku yah nɛ́npɔ́ krótókèlɛ, kàh kɩ fɛ̃ n pɩ n pi. 
Képah sɔkɔ, pè krótókè yĩn mɔ mɔ vyɔ́ngbîl dɛ mɔ sè kɔ̃ sè wũ pnɛ yɔ sɔkɔ. Tɛ́ núkú yɔ̃ sɔkɔ krótókè yah tɛ̃ tɛ̃ yî tɔ wah mɔ, ké yah tɛ̃ yĩnnɛ. Tɛ́ mɛ fwɔ toh yĩ̂nyõ, krótókè yah tĩ̀nnɛ, yã́hkɩ́ kɔ̃ kè kélɛ n tu n mɔ, tɛ́ pé yah mɔ à yɔ kéngbárkɩ̀ mɛ̀nɛ n yɩ̃kɩ. 
Ǹtɛ pè kyɩ yĩni tɛ̃ yɔ́ntáh sèmìyã́hrɩ́ fìì yɔ́ tyɩ́, à nir ké sɔkɔ. Krótókè yah tĩ̀n sõ̀ mɛ kyɩ di sèmìyã́hrɩ́ sɔkɔ, fɔ́ɔ́ kè náh mɛ fɛ̃ n yike, tɛ́ níkwîl mɛ ké náh tĩ̀nnɛ n mɩ n kwɔkɩ n pɔ. 
Képah sɔkɔ, sràsyíbí sõ̀ mɛ n yah n kɔ̃ wɔ̀kɔ̀npèkè sɔkɔ névye kònmɔ-ńsahlɛ. Tíkí sõ̀ mɛ pè tɛ̃ nɛ, pè káh pɔ n yɔ̃ n wil n to n pwah. 
Ǹtɛ sràsyíbí yõ̀tɛ̃̀ dĩ́-á sõ̀ mɛ ké tyɩ́ nɛ, pé Pole pwah mɔ sɛ̃́, à pè yĩni mɔ képah pɩ́nɔ́lɛ. Tɛ́ nɛ, yɔ yɔ̃́npnɛ̀pu ǹgbò yɔkɔ de yɔ sɔkɔ, pè kènkwã̀h yõlɛ n yɩ̃kɩ. 
Tɛ́ mɛ nɛ, no tɛ́lɔ́ sésárnɔ tɔkɔ, képah náh pɩ, krótókè kwɔ́kɩ́nɔ́lɛ, pè dɔkɔ sépi yõ, pè nɛ n yɔ̃ n yuku. Sɛ̃́ ye kè pɩ, tɛ́ no pól yĩni tɛ̃ kènkwã̀h yõ sɔkɔ krómkróm pé vyelɛ.
