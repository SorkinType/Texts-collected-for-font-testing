Nsyɔ̃.
3.
Nsyifunɔ sɔkɔ, yɛ̃́ntɛ̃̂ yɔ́ lésõ mɛ pè n ye Nikodɛmɩnɛ. À sõ̀ mɛ Farisi wɛ̃̀kɩ̀ kɔ́pú sɔkɔ. 
Sõ yɔ́ lékã̂hnɛ, à kyɩ wil Yesu tyɩ́, tɛ́ yo ǹ tyɩ́ nɛ kwɔ́-ò dĩ́, gbǐl tyi nónó-á ǹmɔ n pɩ yɔ̀, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ sɛ̃́nknɔlɛ nɛ n pɩ n pi, nɛ́ Liyel náh mɛ tɛ̃̀nɛ. Képah-á, páh pɩ̃ nɛ ǹmɔ-á kwɔ́-òlɛ, nɛ̂-á Liyel tɛkɩ mɔ. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo á tyɩ́, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ Liyel tɔ̃́rɩ́ wɛ n pi, nɛ́ tɛ̃̀ náh tɔ̃ dubi késã́nnɛ.» 
Képah sɔkɔ, Nikodɛmɩ nɛ, ǹtɛ névi wàhnmɔntɛnɔ náh, sɔ̃́ se képah kɩ fɛ̃ pɩ nɩ? À se kɩ fɛ̃ ǹ syɩ de ǹ yṹ fɔkɔ sɔkɔ, kɩ tɔ̃ dubi kɛ ní-òlɛ nɩ? 
Yesu mɛ tɔ̃ yo ǹ tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo á tyɩ́, ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ n de n pi Liyel tɔ̃́rɩ́ sɔkɔ, nɛ́ tɛ̃̀ náh tɔ̃ dubi ni, ànɛ̂ Liyel Mirki gbõ̀ yõ. 
Névi nɛ̂-à dubi névi tyɩ́, névi ye tɛ̃̀nɛ. Tɛ́ névi nɛ̂-à dubi Liyel Mirki gbõ̀ yõ, Liyel Mirki n nɛ n mɛ tɛ̃̀ sɔkɔ. 
Káh n yɛ kè á gbã̀n du, máh sɔ̃́npɩlɛ yo á tyɩ́ nɛ, yáh yɩ̃nɛ yé tɔ̃ n dubi késã́nnɛ. 
Ńyãh-à yã́hkɩ́ dyɔ, nɛ́npɔ́ ye kè n nɛ n pɩ n yuku. Á kɩ ké pɩ́nɔ́ yĩ́nɛ n nohnɩ, tɛ́ á náh n pɩ̃ n pi ńyãh sɔkɔ-á kè wil, ànɛ̂ kǎh n yuku ńyãh. Névi nɛ̂-à dubi Liyel Mirki gbõ̀ yõ, tɛ̃̀ tɔ tyɩ́ n nɛ n mɛ sɛ̃́ntɛ̃̀nɛ.» 
Wǎh yo sɛ̃́, Nikodɛmɩ tɔ̃ nɛ, sɔ̃́ se képah kɩ fɛ̃ pɩ nɩ? 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Áwɔlɛ tyi kõ̀nsàhnɔ́ kwɔ́-ò ǹgbɛ̃nɛ Yisrayel sɔkɔ, tɛ́ á náh képahlɛ n pnɛ nɩ? 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo á tyɩ́ nɛ, áyáh nɛ̂nɛ n pnɛ, képah tyɩ́-á à n nɛ n yo. Áyáh nɛ̂ wɛ, à n pɩ képah tyɩ́ yónwàhpulɛ. Ǹtɛ yé náh mɛ sélɛ n fɛ̃ n syi. 
Mâh sétáh yõ tyilɛ n yo yé tyɩ́ tɛ́ yé náh mɛ sɛ̃ sé yõ, mâh pɔ yĩ̂nyõ tyilɛ n yo yé tyɩ́, yé kɩ pyě fɛ̃ sɛ̃ sépi yõ sɔ̃́? 
Ǹnɩ̂ yɔ́ ńtɛ̃ náh dɔkɔ kyɩ yah yĩ̂nyõ sɔkɔ, à wil Névi Pídĩ́ tyɩ́ sɔkɔ, ǹmɔ nɛ̂-á wil yĩ̂nyõ à tiki pɔ. 
Sɔ̃́-á lésõ Moyisi wahy kètɔ̀ pɩ yésãhrɩlɛ gbɛ̃́nnɩ sɔkɔ, à dɔkɔ mɔ sah dahnɩ yõ, Névi Pídĩ́ tɔ yɩ̃nɛ à dɔkɔ mɔ sɛ̃́ntɛ̃̀nɛ. 
Képah sɔkɔ, névi nɛ̂-à sɛ̃ ǹ yõ, tɛ̃̀ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Yah, kèkõyṍ névye tyɩ́ ye kah nɔ Liyel tyɩ́, képah ye à ǹ Pi núkúní tɛkɩ mɔ, nɛ névi nɛ̂-à sɛ̃ ǹmɔ yõ, tɛ̃̀ tyɩ́ káh yɩkɩ, ǹtɛ tɛ̃̀ tyɛ́-ńkɛ̃̂ min wɛ. 
Liyel tɛ̃̀ wɔ náh ǹ Pídĩ́ tɛkɩ mɔ kèkõyṍ sɔkɔ nɛ, à pɔ lékàhlɛ névye kɔ̃ pé tyípêl yĩnnɛ dɛ́! À ye à tɛkɩ mɔ nɛ, kèkõyṍ névye pwáhnmɔnɔ wɛ ǹmɔ gbõ̀ yõ. 
Névi nɛ̂-à sɛ̃ ǹmɔ yõ, tɛ̃̀ wɔ náh tɔ̃ n tuke n pi. Ǹtɛ nɛ̂-à yahle sɛ̃́nɔ́lɛ ǹmɔ yõ, tukey syɩkɩ tɛ tɛ̃̀ yõ, wǎh sɔ̃́npɩ́ yahle sɛ̃́nɔ́lɛ Liyel Pi núkúní yõ. 
Tukey mɛ̀ wɛ yɔ̀: Kyòo ye pɔ kèkõyṍ, tɛ́ névye tyípɩ́nɔ́-á pɩ tyítúkúnìnɛ sɛ̃́, fɛ̃́ntũ̂ tyɩ́ nɔ kal pé tyɩ́ kyòolɛ. 
Névi nɛ̂-à tyítúkúnìnɛ n pɩ, kyòo tyɩ́ náh n nɔ tɛ̃̀ tyɩ́ gbógbó. Tɛ̃̀ n pɩ̃ nɛ páh kɩ pɔ kyòo yah, pé tyítúkúnì pɩ́nɔ́-á kɩ wil mɔ kwèyy. Képah ye, à náh n pɔ kyòo yah sɔkɔ. 
Ǹtɛ névi nɛ̂-à gbɩ wɛ̃̀kɩ̀lɛ n kɔ, ǹmɔ n pɔ kyòo yah sɔkɔ. À n nɛ n mɛ ké tyɩ́ nɛ, no pé tyípɩ́nɔ́ wɛ pnɛ́yy sépi nónó-á mɛ n pɩ Liyel dyɔ dyɔ tɛ̃̀nɛ.» 
Képah náh, Yesu ànɛ̂ ǹ tõ̀ sã́hpú pɔ tɛ̃ yah Nsyude gbɛ̃̀nyah sɔkɔ. À sõ̀ mɛ névyelɛ n wel n wilki ni sɔkɔ. 
Tɛ́ ni-á sõ̀ mɛ dò Yɛno kwil sɔkɔ sɛ̃́, képah ye Nsyɔ̃ tɔ mó mɛ wèlwìlkìnɔ́lɛ n pɩ nɛ́npɔ́. Névye sõ̀ mɛ n pɔ n pɔ ǹ tyɩ́, à kɔ̃ à pélɛ n wel n wilki ni sɔkɔ. Nɛ́npɔ́ sõ̀ mɛ yúkɛy Salimi kwillɛ. 
Légbĩ́nɛ, pè náh sõ̀ tɔ̃̌nɩ̀ Nsyɔ̃ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Képah sɔkɔ sõ yɔ́lɛ, yékã̂b yɔ́ de Nsyɔ̃ tõ̀ sã́hpú ànɛ̂ Nsyifu yɔ́ wrɔ́, pé gblɔ̌y tɔ̃ n sah dùkù tyɩ́ yõ, à yɩ̃nɛ pé tyi kõ̀nsàhnɔ́lɛ. 
Kǎh pɩ sɛ̃́, Nsyɔ̃ tõ̀ sã́hpú mɛ Nsyɔ̃ nyáh sõ kyɩ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé kwɔ́-ò dĩ́, dĩ́ nɛ̂-á lésõ ǹnɛ Nsyurdɛ̃ yɔ́nkahnɩnɛ, tɛ́ à ǹ tyɩ́lɛ n yo pé tyɩ́ kɛ̀; à yah, ǹ tɔ-á mɛ yuku névyelɛ n wel n wilki ni sɔkɔ. Tɛ́ no pól-á mɛ pé yah mɔ ǹmɔ tĩ̀nnɛ. 
Ǹtɛ Nsyɔ̃ mɛ yo pé tyɩ́ nɛ ńkɛ̃́: «Nɛ́ Liyel náh yĩ́n-ńsah nɛ̂nɛ névi kɔ̃, tɛ̃̀ se kɩ fɛ̃ kè wɛ? 
Yépi ńtɛ̃ ye ńmɔ tàrfyɔ̀lɛ nɛ máh lésõ kè yo dal mɔ yé tyɩ́ nɛ ńmɔ náh Liyel nɛy yáhntɔkɔnɔlɛ. Ǹ tɛ̃ntõ̀ ó-á mé pɔ pɩ́-ńsah ǹ yahlɛ. 
Yé yah, nɛ̂ tyɩ́-á sépíllɛ, ǹ pɛ ye. Tɛ́ pɛ mɛ̀ kódĩ́ n mó nɛ n mɛ pɛ mɛ̀ tnɔ̂. Kódĩ́ mɛ̀-à pɛ mɛ̀ wɛ nɩ̀vɩ̀nɩ̀ sɔkɔ, kè n pɩ nɩ̀vɩ̀nɩ̀ ǹgbɛ̃nɛ ǹ tyɩ́. Sɛ̃́nkɔ̃ nɩ̀vɩ̀nɩ̀ ye ńmɔ sɔkɔ. Ń nɩ mɛ vɩ syɩ ké yĩ̀n tyɩ́. 
Wãn yɩ̃nɛ kè nɛ n mɔ ǹmɔ yĩn wìlnɔ́ yõ, kwâh mó nɛ n wilki ńmɔ tyɩ́ sɔkɔ. 
Névi nɛ̂-á n wil yĩ̂nyõ, ǹmɔ ye no pól yõ. Tɛ́ névi nɛ̂-á mɛ sétáh mɛ̀ yõ, sétáh mɛ̀ névi ye ǹnɛ. Sétáh mɛ̀ yõ tyi ó ye ǹmɔ kɩ fɛ̃ n yãh dò. Ǹtɛ, nɛ̂-á n wil yĩ̂nyõ, ǹmɔ mɛ no pól yõ. 
Wǎh tyi nónó wɛ, tɛ́ tyi nónó noh, sépi tyɩ́ ye à n yo n wah, tɛ́ nɛy náh mɛ ǹ wɛy fɛ̃ syi. 
Ǹtɛ nɛ̂-à ǹ wɛy fɛ̃ syi, tɛ̃̀ mɛ kélɛ n wɛ̃kɩ tɔ́tɔ́ nɛ gbɩ-á Liyel wɛynɛ. 
Ǹmɔ nɛ̂-á Liyel tɛkɩ mɔ, à ǹ Mirkilɛ à kɔ̃ yĩ́ĩ́. Liyel wɛy ye à n nɛ n yo. 
Sú Liyellɛ, ǹ Pídĩ́ tyɩ́ nɔ ǹ tyɩ́. Képah ye à tyi pól dahbɩ ǹmɔ gbõ̀. 
Névi nɛ̂-à sɛ̃ Liyel Pídĩ́ yõ, tɛ̃̀ kɩ tyɛ́-ńkɛ̃̂ min wɛ. Tɛ́ névi nɛ̂-à yahle sɛ̃́nɔ́lɛ ǹ yõ, tɛ̃̀ náh kénɛ min wɛ n pi. Liyel pèkè kɩ mɛ tɛ̃̀ yõ gbĩ́mɛgbĩ́.»
