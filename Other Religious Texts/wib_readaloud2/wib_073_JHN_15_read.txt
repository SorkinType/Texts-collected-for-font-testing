Nsyɔ̃.
15.
Képah sɔkɔ, Yesu tɔ̃ nɛ: «Ńmɔ ye yɛrɛsɛ̃ séswɔ̂ yèsyɩ̀kɩ̀lɛ, ń Sú mɛ ké nɛ́tɛ̃̀nɛ. 
Yɩ̃̀ngbã̀n nónó pól-á mɛ ńmɔ sɔkɔ tɛ́ sè náh mɛ n syi, à kɩ sè kɔ̃lɩ tɛ̃ yõ. Tɛ́ nónó pól-á mɛ n syi, à kɩ sépilɛ n tɔ̃ sè kɔ̃ sè kɩ n gbah n syi. 
Wɛy nɛ̂-á mé yo yé tyɩ́ yɔ̀, képah gbõ̀ yõ, yé pɩ tɛ yályál. 
Yé nɛ n mɛ ńmɔlɛ, sɔ̃́-á ńmɔ mɛ yélɛ. Sɔ̃́-á yɩ̃̀ngbõ̀ náh fɛ̃ n syi n pi ké gblɔ̌y nɛ́ kè náh mɛ séswɔ̂ sɔkɔ, sɛ̃́ ó ye yépi tɔ náh fɛ̃ tir yɔ́ pɩ n pi nɛ́ yé náh mɛ ńmɔlɛ. 
Ńmɔ ye yɛrɛsɛ̃ séswɔ̂lɛ, yépi mɛ ké yɩ̃̀ngbã̀nnɛ. Yé náh fɛ̃ tir yɔ́ ńtɛ̃ pɩ n pi ńmɔ kɛ̃́nɛ. Képah ye, nɛ̂-à mɛ ńmɔlɛ, tɛ́ mé mɛ tɛ̃̀nɛ, tɛ̃̀ kɩ tyípéplɔ náhnáhlɛ n pɩ Liyel tyɩ́. 
Névi nɛ̂-à mɛ́-ńkɛ̃̂nɛ ńmɔlɛ, tɛ̃̀ mɛ yɩ̃̀ngbõ̀ yɔ́ tyɩ́ kɔ̃lɛ, nɛ̂-á pè kãm milki kégbɔ́, tɛ́ kè kɩ n kɔkɔ n pi. Tɛ́ yɩ̃̀ngbã̀n kɔ́kɔ́rɩ́lɛ, sépi kɩ pɔ kwel mɔ n sukɛ n pi nyàh sɔkɔ. 
Ǹtɛ yâh mɛ ńmɔlɛ, tɛ́ ń wɛlɔ mɛ n tɛ̃ yé fɔkɔnɩ sɔkɔ, kwâh nɛ̂ tyɩ́-à nɔ yé tyɩ́, yé kè ni, yé kɩ kè wɛ. 
Yâh tyípéplɔ náhnáhlɛ n pɩ Liyel tyɩ́, képah ye kɩ kè wɛ̃kɩ nɛ yáh pɩ ńmɔ tõ̀ sã́hpúlɛ. Tɛ́ ńmɔ Sú Liyel yĩn kɩ gbilki. 
Ńmɔ tyɩ́-á nɔ Sú Liyel tyɩ́ sɔ̃́, yépi tɔ tyɩ́ nɔ ńmɔ tyɩ́ sɛ̃́. Yé yɛ yé tyɩ́ nɛ mɛ nɔ́ntɛ̃̀nɔ́lɛ ń tyɩ́ sɛ̃́. 
Yâh ń tyi kõ̀nsàhnɔ́lɛ n pɩ, yé tyɩ́ kɩ mɛ nɔ́ntɛ̃̀nɔ́lɛ ń tyɩ́, sɔ̃́-á ńmɔ ń Sú tyi kõ̀nsàhnɔ́lɛ n pɩ, tɛ́ ń tyɩ́lɛ nɔ́ntɛ̃̀nɔ́lɛ ǹ tyɩ́. 
Mé mɛ ké tyɩ́ nɛ nɩ̀vɩ̀nɩ̀ nɛ̂-á mɛ ńmɔ sɔkɔ, kè nɛ mɛ yépi tɔ sɔkɔ, tɛ́ yé nɩ̀vɩ̀nɩ̀ mɛ̀ mó syɩ ké yĩ̀n tyɩ́. Képah ye, mé tyi nɔ́pi yo yé tyɩ́. 
Máh nɛ̂ kõ sah yé kɔ̃, kè wɛ yɔ̀: Yé tyɩ́ yɩ̃nɛ kè nɔ wɛ̃ tyɩ́, yé tyɩ́-á nɔ sɔ̃́ ńmɔ tyɩ́. 
Névi-à ǹ min kɔ̃ ǹ kódíbí yĩnnɛ, tyɩ́nɔnɔ ǹgbɛ̃ yɔ́ náh tɔ̃ fɛ̃ n kah n mɔ n pi képah yõ. 
Ńmɔ-á nɛ̂ kõ sah yé kɔ̃, yáh kɩ mɛ kélɛ n pɩ, ń kódíbí ye yélɛ. 
Mé náh tɔ̃ yé ye n pi tõ̀npɩbɩlɛ. Ǹgbǒ tõ̀npɩ náh fɛ̃ ǹ yõ̀tɛ̃̀ dĩ́ tyípɩ́nɔ́ pɩ̃ dɛ́! Ǹtɛ máh tyi nónó noh ń Sú Liyel tyɩ́, mé ye sé póllɛ yépi kwɔ. Képah ye, mé yélɛ n ye ń kódíbílɛ. 
Yépi náh yah tɔkɔ ńmɔlɛ dɛ́, ńmɔ ye yah tɔkɔ yélɛ, tɛ́ yé tɛkɩ mɔ nɛ, yé kyɩ tyípéplɔlɛ n pɩ Liyel tyɩ́, nónó-á sé yõre náh n tyɛ n yah n pi. Képah sɔkɔ, yâh kwáhdúkú nɛ̂ ni Sú Liyel tyɩ́ ńmɔ yĩn yõ, à kɩ kélɛ yé kɔ̃. 
Ǹtɛ nɛ̂-á mé n kõ n sah yé tyɩ́, kè wɛ yɔ̀: Yé tyɩ́ yɩ̃nɛ kè nɔ wɛ̃ tyɩ́.» 
Képah sɔkɔ, Yesu tɔ̃ nɛ: «Kèkõyṍ névye-à yé dyɔlɛ n yɔ, yé pɩ̃ nɛ ńmɔ ǹgbò dyɔ-á pè tyah n yɔ. 
Yâh sõ̀ mɛ kèkõyṍ névye tyɩ́ yõ sɔkɔ, yáh pɩ pé tyɩ́lɛ sɛ̃́, yé tyɩ́ náhkɩ nɔ pé tyɩ́. Ǹtɛ kǎh pɩ ńmɔ yé yah wilki kèkõyṍ névye sɔkɔ, tɛ́ yé náh tɔ̃ mɛ pépi tyɩ́ yõ, képah ye pè kɔ̃ pè yé dyɔlɛ n yɔ. 
Yé yé dyɔ sah nɛ n kwnɛ wɛy mɛ̀ tyɩ́lɛ, máh nɛ̂ yo yé tyɩ́ nɛ: ‹Tõ̀npɩ náh fɛ̃ n kal ǹ yõ̀tɛ̃̀nɛ.› Pàh kɩ mɛ ńmɔ fõh, pè kɩ yé tɔ fõh. Pàh kɩ mɛ ń wɛy fɛ̃ syi, pè kɩ yé tɔ tyɩ́ fɛ̃ syi. 
Ǹtɛ pè náh sɔ̃́npɩ́lɛ mɛ ńmɔ tṍ-òlɛ n pnɛ, képah ye pè kɩ tyi nɔ́pi pól pɩ yé sɔkɔ ńmɔ yĩnnɛ. 
Mâh sõ̀ mɛ pɔ́-ńkɛ̃̂nɛ pé tyɩ́, ànɛ̂ mâh sõ̀ mɛ tir yɔ́ yó-ńkɛ̃̂nɛ pé tyɩ́, pé tyítúkúnì pɩ́nɔ́ lékàh náh náhkɩ n mɛ n pi pé yõ. Ǹtɛ núkúnúkú, gbɩ wɛ́-ńsah yɔ́ ńtɛ̃ náh n wɛ n pi pé tyɩ́ pé tyítúkúnì pɩ́nɔ́ tyɩ́ sɔkɔ. 
Névi nɛ̂-à ńmɔ dyɔlɛ n yɔ, tɛ̃̀ mɛ ń Sú tɔ dyɔlɛ n yɔ. 
Tyi nónó-á ǹnɩ̂ yɔ́ ńtɛ̃ náh sɛ̃́nknɔ pɩ yah, mâh sõ̀ mɛ sépi pɩ́-ńkɛ̃̂nɛ pé wrɔ́ sɔkɔ, pé tyítúkúnì pɩ́nɔ́ lékàh náh náhkɩ n mɛ n pi pé yõ. Ǹtɛ núkúnúkú, pè ye ńmɔ tyípɩ́nɩ̀ wɛ. Tɛ́ képah fyé yõ, pè mɛ ńmɔ ànɛ̂ ń Sú dyɔlɛ n yɔ. 
Ǹtɛ nɛ̂-á wã́rkɩ́nsàhnɔ́lɛ pé tyi kõ̀nsàhnɔ́ sɔkɔ nɛ: ‹Pè ń dyɔ yɔ yõ kɛ̃́nɛ.› képah wil yĩn ye pè kɔ̃ pè képahlɛ n pɩ. 
Tɛ́ yòhnɩ̀nmɔ-ò nɛ̂-á ńmɔ kɩ n tɛkɩ n mɔ n pi kɩ wil Sú Liyel tyɩ́, Liyel Mirki ye. Képah ye gbɩlɛ névyelɛ n wɛ̃kɩ Liyel tyɩ́ sɔkɔ. Képah-à wil Sú Liyel tyɩ́ à pɔ, kè kɩ pɔ ńmɔ tyɩ́ yo wah yé tyɩ́. 
Ǹtɛ yépi-á sõ̀ mɛ ànɛ̂ ńmɔlɛ ǹgbò dùkùlɛ sɛ̃́, képah ye yé tɔ kɩ ń tyɩ́lɛ n yo n wah névye tyɩ́.»
