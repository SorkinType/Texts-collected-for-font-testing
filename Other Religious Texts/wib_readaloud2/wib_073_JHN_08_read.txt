Nsyɔ̃.
8.
Ǹtɛ, Yesu wɔ mó sɔkɔ Wolivi yèh yɔ̃lɩ yõ. 
Tɛ́ dĩndal séngbɔ̃́ púpúlɛ, Yesu tɔ̃ ǹ syɩ pɔ Liyel ní-ńsah gbô sɔkɔ, tɛ́ no pól mɛ pɔ ǹ tnɔ̂. À mɛ kɔ̃ tɛ̃ à tyah pélɛ n kwɔ. 
Wǎh mɛ kwɔ́nɔ́ sɔkɔ, Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́pú mɛ pɔ sê fɔ̀fɩ́npɩ yɔ́lɛ ǹ tyɩ́. Pè pɔ pi mɛ̀ sah tɔ́wû nɩyṍ Yesu yah, 
tɛ́ Yesu syah nɛ, kwɔ́-ò dĩ́, páh pi mɛ̀ wɛ tɛ̃ fɔ̀fɩ́kɩ̀ pɩ́nɔ́ sɔkɔ tɔ́tɔ́. 
Tɛ́ Moyisi-á kè yo sah ǹ tyi kõ̀nsàhnɔ́ sɔkɔ nɛ, sɛ̃́nkɔ̃ sê-á yɩ̃nɛ mínkònɔ́lɛ dyêhlɛ. Tɛ́ ǹmɔ-á dò, ǹmɔ se nɛ sɔ̃́-á nɩ? 
Pè sõ̀ mɛ képah tɛ̃̀nɛ n yo sɛ̃́, nɛ wàh kɩ pilki pé lékàh wɛ à kɔ̃. Ǹtɛ Yesu mɛ ǹ yõ sĩnki, tɛ́ tyah n wãrkɩ sétáh sɔkɔ ǹ kèngbɔlɛ. 
Kǎh pɩ sɛ̃́, pè tɔ̃ mɛ yɔ̃ sɔkɔ ǹnɛ n gbah n piki n yah. Képah tɛ̃̀nɛ, à mɛ ǹ yõ yɔ̃ tɛ́ yo pé tyɩ́ nɛ: «Yé sɔkɔ nɛy nɛ̂-à kɩ mɛ à náh tyípékè pɩ yah póllɛ, tɛ̃̀ wɔ dáhkɩ̀lɛ pi mɛ̀ min gbennɛ.» 
Képah yóntɛnɔ náh, à tɔ̃ ǹ yõ sĩnki à tyah n wãrkɩ sétáh sɔkɔ. 
Pǎh képah tɛ̃̀ noh sɛ̃́, pè tyah n sɔkɔ núkú núkúlɛ, à tɔkɔ lékɔ̃lɔ́ ǹgbò tyɩ́, pópó à pɔ gbàh sah Yesu ànɛ̂ pi mɛ̀ ó tyɩ́ nɛ́npɔ́. 
Képah sɔkɔ, Yesu mɛ ǹ yõ yɔ̃ tɛ́ yo pi tyɩ́ nɛ, nónó-á pɔ tɔkɔ à kɔ̃, ńyãh se pélɛ nɩ? Pé sɔkɔ yɔ́ ńtɛ̃ kó à mɔ kũ kɔ̃ nɩ? 
Pi mɛ nɛ, pé yõ̀tɛ̃̀, nɛy náh pé mɔ kũ kɔ̃. Képah sɔkɔ, Yesu mɛ nɛ: «Ńmɔ tɔ náh ǹ mɔ kũ kɔ̃ n pi. Nɛ n yuku; tɛ́ à tɔkɔ núkúnúkú tyɩ́, káh n tɔ̃ tyípékè pɩ n yah.» 
Képah náh, Yesu tɔ̃ yo névye tyɩ́ nɛ: «Ńmɔ ye kèkõyṍ kyòolɛ. Névi nɛ̂-à ńmɔ sõ̀ tɛ̃, tɛ̃̀ náh tɔ̃ n kɔ n pi fɛ̃́ntũ̂ sɔkɔ. À kɩ kyòo wɛ, nɛ̂-á kɩ minnɛ à kɔ̃. 
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú à syah nɛ, nɛ̂ se kɩ sɛ̃ ǹ tyíyónɔ́ yõ nɩ? Ǹmɔ ó kó ǹ gblɔ̌y tàrfyɔ̀lɛ yɔ̀ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ ńkɛ̃́: «Gbɩ tɛ̃̀ wɔ ye ńmɔ ye ń gblɔ̌y tàrfyɔ̀lɛ ó, ǹtɛ ńmɔ ń wìl-ǹsahlɛ n pnɛ, tɛ́ ń yúkú-ńsahlɛ n pnɛ. Képah ye, máh tyi nónó yo ń gblɔ̌y dùkù tyɩ́ sɔkɔ, gblɔ ye sélɛ. Tɛ́ yépi náh ń wìl-ǹsah pɩ̃, yé náh se mó tɔ̃ ń yúkú-ńsah pɩ̃. 
Yépi tɛ̃̀ wɔ lékàhlɛ névyelɛ n kɔ̃ à yɩ̃nɛ névi yõ̀tàhsyi sõnɔlɛ. Ǹtɛ ńmɔ wɔ náh lékàhlɛ ǹnɩ̂ yɔ́ ńtɛ̃nɛ n kɔ̃. 
Tɛ́ kàh se pɔ pɩ mé yɩ̃nɛ mè lékàhlɛ ǹnɩ̂ kɔ̃, mé náh kélɛ à kɔ̃ n pi ń gblɔ̌y. Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, á kɩ kélɛ à kɔ̃ wɛ̃ tyɩ́ ǹmɔlɛ. Képah ye, ń lékàh kɔ̃́nɔ́ kɩ pɩ gbɩ wɛ̃̀kɩ̀ yõ. 
Kè náh mɛ wã́rkɩ́nsàhnɔ́lɛ yé tyi kõ̀nsàhnɔ́ sɔkɔ nɛ, nɛ́pĩ̂ nímí-à pɩ tàrfyɔ̀lɛ tir nɛ̂ sɔkɔ, gbɩ-á kélɛ di? 
Ńmɔ mɛ ń gblɔ̌y tyɩ́lɛ n yo, tɛ́ Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ tɔ mɛ ń tyɩ́lɛ n yo.» 
Képah sɔkɔ, pè Yesu syah nɛ, ńyãh se ǹ súlɛ nɩ? Yesu mɛ nɛ: «Yé náh ńmɔlɛ n pnɛ, yé náh ń Sú tɔlɛ n pnɛ. Yâh sõ̀ mɛ ńnɛ pnɛ, yé náhkɩ ń Sú tɔ pɩ̃.» 
Yesu sõ̀ mɛ kwɔ́nɔ́lɛ n pɩ Liyel ní-ńsah gbô fɩ̃́ yɔ́ sɔkɔ, ńyãh-á ké pɔ́ kyɛ́súkùlɛ, tɛ́ tyi nɔ́pi yo. Ǹtɛ ǹ gbĩ náh tɔ̃̌nɩ̀ sõ̀ mɛ sɛ, képah ye ǹnɩ̂ yɔ́ ńtɛ̃ náh tɛ̃ wɛ à tɛ̃. 
Képah náh, Yesu tɔ̃ yo tɔ́wû tyɩ́ nɛ: «Ńmɔ ye n sɔkɔ yɔ̀. Yé kɩ ń yah n kɔ̃ n pi, ǹtɛ yé kɩ ku yé tyípékè sɔkɔ. Ńmɔ-á n yuku ńyãh, yépi náh fɛ̃ n kyɩ n pi nɛ́npɔ́.» 
Képah sɔkɔ, Nsyifunɔ nɛ, ǹtɛ ǹ gblɔ̌y ko mɔ dékè se kɩ pyě mɛ à tɛ̃ nɩ? À kɔ̃ à n yo nɛ, pépi náh fɛ̃ n kyɩ n pi péwɔ yúkú-ńsah sɔkɔ. 
Yesu mɛ yo pé tyɩ́ nɛ: «Yépilɛ, sétáh yõ névye ye yélɛ, tɛ́ ńmɔ wɔlɛ, ńmɔ wil yĩ̂nyõ. Kèkõyṍ mɛ̀ névye ye yépilɛ, ǹtɛ ké névi náh ńmɔ wɔlɛ. 
Yáh wɛ mé kélɛ n yo yé tyɩ́ nɛ yáh kɩ ku yé tyípêl sɔkɔ, yé náh mɛ sɛ̃ ké yõ nɛ, Ńmɔ-á, képah ye ń kɔ̃ mé nɛ yáh kɩ ku yé tyípêl sɔkɔ.» 
Képah sɔkɔ, pè à piki yah nɛ, nɛ̂ tɛ̃̀ ó se ǹmɔlɛ nɩ? Yesu mɛ yo pé tyɩ́ nɛ: «Mé sõ̀ kè yo tɛ yé tyɩ́ ké tyah gbĩ́ dùkùlɛ. 
Tyi náhnáh mɛ ń tyɩ́ yo yo tɛ̃̀nɛ yé tyɩ́ sɔkɔ ànɛ̂ kɩ lékàhlɛ yé kɔ̃. Ǹtɛ máh nɛ̂ noh ń tɛ́kɩ́nmɔ-ò tyɩ́, képah ó ye mé n yo yé tyɩ́. Ǹmɔ ye gbɩ nɛ́tɛ̃̂nɛ.» 
Ǹtɛ pépi náh sõ̀ mɛ pɩ̃ nɛ, Sú Liyel tyɩ́-á Yesu n yo pé tyɩ́. 
Képah sɔkɔ, Yesu yo pé tyɩ́ nɛ: «Yâh pɔ Névi Pídĩ́ yɔ̃ dɔkɔ gbĩ́ nɛ̂nɛ, yé kɩ pɩ̃ nɛ Ńmɔ-á. Tɛ́ kɩ tɔ̃ pɩ̃ nɛ mé náh tir yɔ́ ńtɛ̃nɛ n pɩ ń yah yĩ́ĩ́nɛ. Sú Liyel-á tyi nónónɛ ń kwɔ, sépi ó ye mè n nɛ n yo. 
Nɛ̂-á tɛkɩ mɔ ńmɔlɛ, mé mɛ ǹ dyɔ yĩ́ĩ́ tyilɛ n pɩ gbĩ́mɛgbĩ́, képah ye à mɛ ńnɛ, à náh ń sah n pi ń gblɔ̌y.» 
Yesu-á mɛ képah yónɔ́ sɔkɔ gbĩ́ nɛ̂nɛ, névye náhnáh sɛ̃ ǹ yõ. 
Képah tɛ̃̀nɛ, Nsyifunɔ nónó-á sõ̀ sɛ̃ Yesu yõ, à mɛ yo pé tyɩ́ nɛ: «Yáh kɩ mɛ ńmɔ tyíyónɔ́ wɛ̃̀kɩ̀lɛ n kɔ, ń tõ̀ sã́hpú kègblɔ ye yélɛ. 
Yé kɩ gbɩlɛ n pnɛ, tɛ́ kénɛ gbɩ kɩ yé wilki yɔ̃rɩ sɔkɔ.» 
Képah sɔkɔ, pè mɛ Yesu syah nɛ, Abrahmɩ pétóbé-á pépilɛ. Pépi náh pɩ yah ǹnɩ̂ yɔ́ ńtɛ̃ yɔ̃bɩlɛ póllɛ. Ǹtɛ à se sɔ̃́npɩ́lɛ n yo nɛ, pépi-á kɩ wil yɔ̃rɩ sɔkɔ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, névi nɛ̂-à tyípékèlɛ n pɩ, tyípékè yɔ̃̀ ye tɛ̃̀nɛ. 
Yɔ̃̀lɛ, ǹ yõ̀tɛ̃̀ dĩ́ gbô sɔkɔ tɛ̃̀-ò kṹmm náh ǹnɛ. Ǹtɛ yõ̀tɛ̃̀ dĩ́ pídĩ́ tɛ̃̀ wɔlɛ, ǹ sú gbô sɔkɔ tɛ̃̀-ò kṹmm ye ǹmɔlɛ. 
Képah ye, Liyel Pídĩ́-à yé wilki yɔ̃rɩ sɔkɔ, yé kɩ mɛ yé gblɔ̌y yõ kègbɩ tɛ̃̀ yõ. 
Mé pɩ̃ nɛ Abrahmɩ pétóbé-á yélɛ. Ǹtɛ yáh ńmɔ wɛlɔ yahle, képah ye yé kɔ̃ yé n yah n kɔ̃ ń kònmɔ-ńsahlɛ. 
Ńmɔ-á nɛ̂ wɛ ń Sú Liyel tyɩ́, képah ye mé n yo. Yépi tɔ-á mó tyi nónó noh yé sú tyɩ́, sépi ye yé n pɩ.» 
Pǎh Yesu wɛy noh sɛ̃́, pè à syah nɛ, Abrahmɩ-á pépi súlɛ. Képah sɔkɔ, Yesu nɛ: «Yâh sõ̀ mɛ Abrahmɩ pétóbélɛ, yé náhkɩ mɛ Abrahmɩ tyípɩ́nɩ̀nɛ n pɩ. 
Ǹtɛ núkúnúkú, ńmɔ nɛ̂-á gbɩ noh ń Sú Liyel tyɩ́ à yo yé tyɩ́, yé n yah n kɔ̃ ńmɔ kònmɔ-ńsahlɛ. Abrahmɩ náh sɛ̃́nkɔ̃ pɩ dò. 
Yépilɛ, yé sú tyípɩ́nɔ́ ye yé n pɩ.» Képah tɛ̃̀nɛ, pè à syah nɛ, fɔ̀fɩ́kɩ̀ pɩ́nɔ́ sɔkɔ náh pépi dubi dɛ́! Ńkɛ̃́ Sú núkú ó-á pépi tyɩ́, nɛ̂-á Liyellɛ. 
Yesu mɛ yo pé tyɩ́ nɛ: «Liyel tyɩ́ ye ńmɔ wil à pɔ. Ńmɔ náh pɔ ń gblɔ̌y yĩn yõ dɛ́! Ǹmɔ ye tɛkɩ mɔ ńnɛ. Ǹtɛ Liyel-à sõ̀ mɛ pɩ yé Súlɛ kègbɩ yõ sɔkɔ, ńmɔ tyɩ́ náhkɩ mɛ nɔ yé tyɩ́. 
Yé yahle ń wɛlɔ nóhnɔ́lɛ, képah ye yé náh ń wɛlɔ syɩ́kɩ́-ńtã̂n pɩ̃. 
Yépi súlɛ, sétã̀n yõ̀tɛ̃̀ ye ǹnɛ. Ǹmɔ dyɔ yĩ́ĩ́ tyi ye yé mɛ ké tyɩ́ nɛ yé nɛ n pɩ. Nɛ́kókè mɔ́-ò ye ǹnɛ ǹgbò dùkùlɛ. Wǎh pɩ gbɩ náh mɛ ǹ sɔkɔ, à náh mó gbɩ wɛ̃̀kɩ̀lɛ n kɔ póllɛ. Wǎh pɩ mɩ̀nɩ̀ yó-òlɛ, wàh mɩ̀nɩ̀nɛ n yo, ǎ kélɛ n yo à yɩ̃nɛ ǹ gblɔ̌y kèdùkùlɛ. Mɩ̀nɩ̀ yónɔ́ yõ wil ǹmɔ tyɩ́. 
Ǹtɛ ńmɔ tɛ̃̀ wɔlɛ, ńmɔ gbɩlɛ n yo, képah ye yé náh mɛ sɛ̃ ń yõ. 
Yépi sɔkɔ nɛ̂ tɛ̃̀ kɩ fɛ̃ kè wɛ̃kɩ nɛ máh tyípékè pɩ? Ǹtɛ máh gbɩlɛ n yo, kwâh nɛ̂ pɩ tɛ́ yé náh sɛ̃ ń yõ? 
Liyel tyɩ́ névi-à névi nɛ̂nɛ, ǹmɔ Liyel wɛynɛ n noh n tɛ̃ n yĩn. Ǹtɛ Liyel tyɩ́ névye náh yépilɛ, képah ye yé kɔ̃ yé yahle sé nóhnɔ́lɛ.» 
Képah sɔkɔ, Nsyifunɔ Yesu syah nɛ, gbɩ kó mɛ pé tyɩ́ pé yo nɛ, Samari nɛy-á ǹnɛ, ànɛ̂ sétõ-á mɛ ǹ sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Sétõ náh mɛ ńmɔ sɔkɔ. Ń Sú Liyel ye mé n gbilki, tɛ́ yépi mɛ ńmɔlɛ n yilɔ. 
Ńmɔ náh mɛ n yah n kɔ̃ ń gblɔ̌y gbílkí-ńsahlɛ dɛ́. Ń gbílkí-ò ǹ gblɔ̌y ńmɔlɛ. Ǹmɔ ye gbɩ syínkɔ̃-òlɛ. 
Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, névi nɛ̂-à n sah ńmɔ yóré yõ, tɛ̃̀ náh kũ dɔ n yah n pi póllɛ.» 
Képah sɔkɔ, Nsyifunɔ à syah nɛ, páh pyě kè pɩ̃ yĩ́ĩ́ núkúnúkú tɛ̃̀nɛ nɛ sétõ-á mɛ ǹ sɔkɔ. Abrahmɩ-á ku, Liyel tɛ̃ntõ̀ syínyopu tɔ mɛ ku. Ǹtɛ ǹmɔ pɔ n yo nɛ, névi nɛ̂-à n sah péwɔ yóré yõ, tɛ̃̀ náh kũ dɔ n yah n pi nɩ? 
Pépi sú Abrahmɩ-á ku. Ǹtɛ ǹmɔ se mɛ n sõ nɛ páh kal Abrahmɩnɛ nɩ? Liyel tɛ̃ntõ̀ syínyopu tɔ mɛ ku. Tɛ́ ǹmɔ tɛ̃̀ se mɛ ǹ gblɔ̌ynɛ n yah nɛ́dúkú nɛ̂nɛ nɩ? 
Képah sɔkɔ, Yesu yo pé tyɩ́ nɛ: «Mâh náhkɩ sõ̀ mɛ ń gblɔ̌ynɛ n gbilki, yõ náh mɛ gbílkínɔ́ mɛ̀ tyɩ́. Ǹtɛ ń Sú Liyel ye n gbilki ńnɛ, ǹmɔ nɛ̂-á yé nɛ, yé Liyel-á ǹnɛ, 
tɛ́ yé náh se mó tɔ̃ mɛ ǹnɛ n pnɛ. Ǹtɛ ńmɔ wɔ ǹnɛ n pnɛ. Mâh sõ̀ mɛ yo nɛ, mé náh ǹnɛ pnɛ, mé náhkɩ pɩ mɩ̀nyolɛ yé tyɩ́ kɔ̃lɛ. Ǹtɛ ńmɔ ǹnɛ n pnɛ, tɛ́ mɛ n sah ǹ yóré yõ. 
Yé sú Abrahmɩ sõ̀ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ nɛ, pé ńmɔ pɔ sõ gbĩ̀yĩ̀kì wɛ. À mɛ pɔ kè wɛ, kè pɩ nɩ̀vɩ̀nɩ̀nɛ ǹ tyɩ́.» 
Képah sɔkɔ, Nsyifunɔ mɛ à syah nɛ, ǹmɔ náh tɔ̃̌nɩ̀ ye tikininí pɔ́ gbãm wɛ, tɛ́ nɛ páh Abrahmɩ wɛ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Kègbɩgbɩ tɛ̃̀ yõ, mé kélɛ n yo yé tyɩ́, Abrahmɩ-á pópó n dubi, Ńmɔ mɛ.» 
Wǎh képah yo sɛ̃́, pè núkú dyêh kwel ǹ tyɩ́ nɛ páh ǹnɛ n minki. Ǹtɛ à mɛ wɔ̃bɩ à wil Liyel ní-ńsah gbô sɔkɔ à sɔkɔ.
