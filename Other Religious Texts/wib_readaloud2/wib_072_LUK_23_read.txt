Likyi.
23.
Képah náh, pé pól mɛ yuku, à Yesu tɛ̃ kyɩ Pilatɩ tyɩ́. 
Nɛ́npɔ́ sɔkɔ, pè tyah lékàhlɛ ǹnɛ n kɔ̃ nɛ, páh dĩ́ mɛ̀ kyɔmɩ wɛ, à mɛ pé kwil névye yõlɛ n yɩkɩ, mɛ pélɛ n yĩni n mɔ mɔ́rɩ́ mɔ́nɔ́lɛ yõ̀tɛ̃̀ ǹgbɛ̃ Sesarɩ tyɩ́. Tɛ́ ǹmɔ ńtɛ̃-á mɛ n yo nɛ ńkɛ̃́, péwɔ-á yõ̀tɛ̃̀nɛ, nɛ̂-á sõ̀ Liyel nɛ, páh kɩ tɛkɩ mɔ. 
Képah sɔkɔ, Pilatɩ mɛ à piki yah nɛ, ǹmɔ se Nsyifunɔ yõ̀tɛ̃̀nɛ nɩ? À mɛ yo ǹ tyɩ́: «Áwɔ ye yo kélɛ.» 
Pilatɩ mɛ yo sárká wìlkìpu yõ̀tãm ànɛ̂ no pól tyɩ́ nɛ, péwɔ náh tir yɔ́ wɛ dĩ́ mɛ̀ tyɩ́ póllɛ, nɛ̂-á yɩ̃nɛ lékàhlɛ ǹ kɔ̃́nɔ́lɛ. 
Ǹtɛ, pè tɔ̃ mɛ kè gbah yo tũ nɛ ńkɛ̃́, wǎh mɛ névyelɛ n kwɔ nòhnɩ̀-ǹkɛ̃̂nɛ, Yude tã̀ntã̀n pól sɔkɔ. Kalile kwil sɔkɔ-á à tyah kélɛ fɔ́ɔ́, à pɔ de ásõ̂. 
Pilatɩ-á képah noh ńyãh sɔkɔ, à píkínyahnɔ pɩ nɛ, Kalile nɛy se dĩ́ mɛ̀nɛ nɩ? 
Wǎh fyɔ̀ ǹ wìl-ǹsah tyɩ́ noh, tɛ́ pɩ̃ nɛ, Yerɔdɩ-á nɛ́npɔ́ yõ̀tɛ̃̀nɛ, à núkú à tɛkɩ mɔ ǹmɔ kɔ̃. Tɛ́ kè mó pɔ yɩ̃nɛ Yerɔdɩ sõ̀ mɛ Yerusalɛmɩ kénɛ sèpĩ̀n sɔkɔ. 
Kè pɩ di mónɔ́lɛ, Yerɔdɩ sõ̀ mɛ Yesu tyɩ́ yónɔ́lɛ n nohnɩ, tɛ́ mɛ ké tyɩ́ nɛ, pé à wɛ à nɛ n mɛ gbǐl yɔ́lɛ n pɩ. Képah ye, wǎh à wɛ, kè pɩ nɩ̀vɩ̀ ǹgbɛ̃nɛ ǹ tyɩ́. 
À mɛ à gbah piki yah náhnáh, tɛ́ Yesu náh dahbɩ wilki ǹ tyɩ́. 
Képah sɔkɔ, sárká wìlkìpu yõ̀tãm ǹgbnɔ ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú sõ̀ mɛ nɛ́npɔ́, tɛ́ mɛ lékàhlɛ ǹnɛ n kɔ̃ fɔ́ɔ́ képékèyɔ̂. 
Yerɔdɩ ànɛ̂ ǹ sràsyíbí-á kṍlɛ à ko, tɛ́ mɛ à yilɔ tɛ gbĩ́ nɛ̂nɛ, pè mɛ yõ̀tɔ̃rɩ́ fwɔ yɔ́ mɔ à kɔ̃, tɛ́ pé syɩ à tɛkɩ mɔ Pilatɩ kɔ̃. 
Pilatɩ ànɛ̂ Yerɔdɩ sõ̀ mɛ wɛ̃ sépɛ́bɩ́lɛ ǹgbòlɛ, tɛ́ lésõlɛ, pè kódírí toh. 
Képah sɔkɔ, Pilatɩ sárká wìlkìpu yõ̀tãm ǹgbnɔ, ànɛ̂ kwil yahtãm, ànɛ̂ névye ye tuke, 
tɛ́ yo pé tyɩ́: «Yé dĩ́ mɛ̀ tɛ̃ pɔ, tɛ́ nɛ, wǎh mɛ névye yõlɛ n yɩkɩ. Ǹtɛ yah, yépi tɔ yĩ́ yah ye mé piki yah ǹnɛ. Mé náh tir yɔ́ ńtɛ̃ wɛ ǹ tyɩ́, yáh nónónɛ n yo n sah ǹ yõ, nɛ̂-á yɩ̃nɛ lékàhlɛ ǹ kɔ̃́nɔ́lɛ. 
Yerɔdɩ ńtɛ̃ tɔ náh tir yɔ́ wɛ ǹ sɔkɔ, wǎh sɔ̃́npɩ́ ǹ syɩ à tɛkɩ mɔ ápi kɔ̃. Kǎh pɩ sɛ̃́, dĩ́ mɛ̀ náh tir yɔ́ ńtɛ̃ pɩ nɛ̂-á yɩ̃nɛ kũnɛ. 
Képah ye, mé kɩ à ko yèfõhlɛ, tɛ́ kɩ à fĩn mɔ.»  
Tɛ́ Pakɩ syáhnɔ́ pól sɔkɔ, Pilatɩ sõ̀ yɩ̃nɛ à wɔ̀kɔ̀npèkè sɔkɔ névi yɔ́ fĩn mɔ pè kɔ̃. 
Képah tɛ̃̀nɛ, pé pól tyah n kwrɔ n gbo n mɔ nɛ, à dĩ́ mɛ̀ ko mɔ, à mó Barabasɩ fĩn mɔ. 
Tɛ́ Barabasɩ sõ̀ mɛ dáhnmɔnɔlɛ wɔ̀kɔ̀npèkè sɔkɔ kèlékɛ̃ tir yɔ́ yõ, nɛ̂-á pɩ kwil sɔkɔ, ànɛ̂ nɛ́kókè mɔ́nɔ́lɛ. 
Tɛ́ Yesu fĩ́nnmɔ dékè-á sõ̀ mɛ Pilatɩ tɛ̃, à tɔ̃ ǹ syɩ kè tɔ̃ yo pé tyɩ́. 
Ǹtɛ, pè tɔ̃ gbo mɔ nɛ ńkɛ̃́: «À syɩ dahnɩ yõ! À syɩ dahnɩ yõ!» 
Pilatɩ mɛ tɔ̃ wɛy tɔkɔ kɛ tɔ̃́-òlɛ, à yo pé tyɩ́ nɛ, tyítúkù nɛ̂ se à pɩ nɩ? Péwɔ náh tir yɔ́ ńtɛ̃ wɛ ǹ sɔkɔ, nɛ̂-á yɩ̃nɛ kũnɛ. Képah-á, páh kɩ à ko yèfõhlɛ, tɛ́ kɩ à fĩn mɔ. 
Ǹtɛ, pè mɛ yɔ̃ sɔkɔ kélɛ n kah n yo náhnáh nɛ ńkɛ̃́: «À syɩ dahnɩ yõ.» Képah tɛ̃̀nɛ, pé kàhngbonmɔnɔ pɔ tɛ̃ wɛ Pilatɩ tyɩ́. 
Képah sɔkɔ, à mɛ kè yah tɔkɔ nɛ, nɛ̂-á pè n yah n kɔ̃, képah-á kɩ pɩ pé tyɩ́. 
Dĩ́ nɛ̂-á sõ̀ pè dah mɔ wɔ̀kɔ̀npèkè sɔkɔ, kèlékɛ̃ yɔ́ ànɛ̂ nɛ́kókè mɔ́nɔ́ yĩnnɛ, ǹmɔ nɛ̂-á sõ̀ pè n yah n kɔ̃ ǹ tyɩ́, à ǹmɔ fĩn mɔ, tɛ́ Yesu pɩ pépi dyɔ yĩ́ĩ́ tɛ̃̀nɛ. 
Képah sɔkɔ, pǎh Yesu tɛ̃ n sɔkɔ syɩ́-ńsah gbĩ́ nɛ̂nɛ, pè Sirɛnɩ kwil dĩ́ yɔ́ yohnɩ, pè n ye Simɔlɛ. À mɛ wil swãh, mɛ n de Yerusalɛmɩ. Pè mɛ ǹmɔ tɛ̃ Yesu dahnɩnɛ, à twah mɔ nɛ, à ǹ sõ̀ tɛ̃. 
Tɛ́ tɔ́wû ǹgbɛ̃ sõ̀ mɛ n yuku ǹ náh, à tahlɩ syɔ́ tyɩ́ nónó-á sõ̀ mɛ n gbo pé gbã̀nnɛ n kõhlɩ pé nɩ yõ yèkwɔrɩ tɛ̃̀ tyɩ́ Yesu yĩnnɛ. 
Képah sɔkɔ, Yesu mɛ ǹ yah vi pé tĩ̀nnɛ, tɛ́ yo pé tyɩ́: «Sèpĩ̀n yísyɔ́ mɛ n pi, névye kɩ pɔ tyah n yo nɛ, yõ̀bènɔrɩ-á mɛ píkã́n syɔ́ tyɩ́, ànɛ̂ nónó-á pè náh pi dubi yah, ànɛ̂ pè náh yã́nnɛ piri kɔ̃ yah. Képah ye Yerusalɛmɩ syɔ́, yé káh nɛ n gbo ńmɔ yĩnnɛ. Yé nɛ n gbo yé gblɔ̌y, ànɛ̂ yé wɛ̃́npì yĩnnɛ. 
Ké légbĩ́nɛ, névye kɩ pɔ tyah n yo tlɛy tyɩ́ nɛ, sè syɩkɩlɩ pé yõ. Tɛ́ kɩ yo ymɔ tyɩ́ nɛ, sè vike tũ pé yah. 
Ǹtɛ, pàh séswɔ́npõ̀lɛ n pɩ sɛ̃́, séswɔ́nkɔ̀kɔ̀ wɔ tyɩ́ kɩ pɔ n pɩ n pi sɔ̃́?» 
Tɛ́ pè sõ̀ mɛ tyípêl pɩ́pú nɛ́pĩ̂ nímí tɔ tɛ̃ kyɩ, nónó-á kɩ kyɩ n ko n mɔ n pi ànɛ̂ Yesulɛ wɛ̃ tyɩ́. 
Pǎh kyɩ yĩni tɛ̃ fɩ̃́ yɔ́, ńyãh-á pè n ye «Yõ̀gbèkè tyahlɛ», pè à syɩ dahnɩ yõ nɛ́npɔ́ ànɛ̂ tyípêl pɩ́pú nɛ́pĩ̂ nímí nɔ́pilɛ pé tyɩ́ tyɩ́ dahlɔ yõ. Núkú sõ̀ mɛ syɩ́nɔ́lɛ ǹ gbõ̀yõ̀ sõ̀, ní-ò mɛ syɩ́nɔ́lɛ ǹ gbɛ̃̀yɩ̃̀nɛ̀ sõ̀. 
Légbĩ́nɛ, Yesu mɛ nɛ: «Bɔ̌, pǎh nɛ̂nɛ n pɩ, pè náh kè pɩ̃. Képah ye, á gblɔ̌y yɛ pé tyɩ́.» Képah náh, sràsyíbí mɛ ǹdan mɔ, à ǹ flɔ dibi wɛ̃ wrɔ́. 
Névye sõ̀ mɛ yĩ́nntɛ̃̀nɔ́lɛ nɛ́npɔ́ mɛ n yah. Tɛ́ Nsyifunɔ yõ̀tãm wɔ sõ̀ mɛ kṍlɛ ǹnɛ n ko, mɛ yo tũ nɛ, wǎh névye túkù pwah mɔ, ǹtɛ à ǹ gblɔ̌y pwah mɔ mó kɛ̀, ǹmɔ-à kɩ nɛ̂ Kristalɛ, Liyel yáhntɔkɔnɔlɛ. 
Sràsyíbí tɔ sõ̀ mɛ kṍlɛ ǹnɛ n ko, tɛ́ nɔ de ǹ tnɔ̂, à dìvɛ̃ yṍrénɔ́lɛ à kɔ̃, tɛ́ nɛ, 
ǹmɔ-à kɩ nɛ̂ Nsyifunɔ yõ̀tɛ̃̀nɛ, à ǹ gblɔ̌y pwah mɔ kɛ̀! 
Tɛ́ kè sõ̀ mɛ wã́rkɩ́nsyɩnɔlɛ Yesu yõ̀tah nɛ: «Ńyɔlɛ, Nsyifunɔ yõ̀tɛ̃̀ ye.» 
Képah sɔkɔ, tyípêl pɩ́pú nónó-á syɩ dahlɔ yõ, pé sɔkɔ núkú sõ̀ mɛ Yesulɛ n gbyɔ nɛ, ǹmɔ kó Kristalɛ nɩ? À ǹ gblɔ̌y pwah mɔ, à mó pépi tɔ pwah mɔ! 
Ǹtɛ, ǹ ní-ò mɛ wɛkɩ ǹ yõ, tɛ́ yo ǹ tyɩ́ nɛ, à kó Liyel yah tíkílɛ n pɩ nɩ, ǹmɔ nɛ̂-á mɛ kénɛ yèvyãhrɩ núkú mɛ̀ sɔkɔ? 
Pépi tɛ̃̀ wɔ tyɩ́lɛ, pé yèvyãhrɩ pɩ́nɔ́ yah-á. Pé tyípɩ́nɔ́ kɔ̃lɩ-á n dyah pélɛ. Ǹtɛ, ǹmɔ tɛ̃̀ náh tir yɔ́ ńtɛ̃ pɩ pèkèlɛ. 
Tɛ́ tɔ̃ ǹ syɩ yo Yesu tyɩ́: «Yesu, á dyɔ to n kõ ńmɔ tyɩ́lɛ, wâh pɔ á yõ̀tɔ̃rɩ́ sɔkɔ gbĩ́ nɛ̂nɛ.» 
Yesu mɛ yo ǹ tyɩ́: «Kègbɩ yõ, mé kélɛ n yo n dal n mɔ á tyɩ́ nɛ: kwɛy mɛ̀ ńtɛ̃ sɔkɔ, á kɩ mɛ ńmɔlɛ Liyel fṍ sɔkɔ pal tɛ̃̀-ǹsah.» 
Gbĩ̀ntɔ̃̀ pɔ̃́lɛ, gbĩ yah kwlɔ, fɛ̃́ntũ̂ mɔ kèkõyṍ pól sɔkɔ pópó, à pɔ syɩ gbèkãm gbĩ́ tyɩ́. 
Nɛ́npɔ́ swɔ, fwɔ̀gbrǒkè nɛ̂-á Liyel wɔ̀kɔ̀ ǹgbɛ̃ fɔkɔ táh kõ kah, kè tɔ ké nɩyṍ. 
Yesu mɛ kah de syi nɛ ńkɛ̃́: «Bɔ̌, mé ń minnɛ n dahbɩ á gbõ̀.» Képah yóntɛnɔ náh, à ku. 
Sràsyíbí lèkwɛ̌ yõ̀tɛ̃̀ dĩ́-á képah wɛ ńyãh sɔkɔ, à tyah Liyel yĩnnɛ n gbilki mɛ n yo nɛ, kègbɩ yõ, nɛ́gbɩ́-á sõ̀ dĩ́ mɛ̀nɛ. 
Tɛ́ nónó pól-á sõ̀ mɛ pɔ yĩn tɛ̃ kélɛ n yah, pǎh kénɛ tyípɩ́nɔ́ wɛ sɛ̃́, pé pól sõ̀ mɛ pé gbã̀n kõhlɩ pé nɩ yõ mɛ n sɔkɔ. 
Ànɛ̂ nónó pól-á sõ̀ Yesulɛ n pnɛ, à tahlɩ syɔ́ nónó pól-á sõ̀ wil Kalile kwil sɔkɔ, à pɔ ǹnɛ wɛ̃ tyɩ́, pé pól sõ̀ mɛ yĩn tɛ̃ fɩ́ɩ́, tɛ́ mɛ n yah. 
Tɛ́ dĩ́ yɔ́ sõ̀ mɛ, pè n ye Yosɛfulɛ. À sõ̀ n wil Nsyifunɔ kwil yɔ́ sɔkɔ, nɛ̂ yĩn-á Arimatɩlɛ. Nsyifunɔ tukey tùkèpu sɔkɔ nɛy yɔ́ ye sõ̀. Tɛ́ pèpɔrɩ pɩ́-ò nɛ́gbɩ́ ye sõ̀ ǹnɛ. À náh sõ̀ n de pé no tɛ́lɔ́ tyi yáhntɔkɔnɔ, ànɛ̂ sé pɩ́nɔ́ sɔkɔ. Ànɛ̂, à sõ̀ mɛ Liyel tɔ̃́rɩ́ pɔ́nɔ́lɛ n syɩkɩ. 
Ǹmɔ mɛ kyɩ wil Pilatɩ tyɩ́, à Yesu lékó ni ǹ tyɩ́. 
Tɛ́ mɛ kyɩ à tiki mɔ dahnɩ yõ, tɛ́ fwɔlɛ à plo, à kyɩ dah mɔ diki yɔ́ sɔkɔ. Kénɛ diki sõ̀ tũ tɛkɩ sɔkɔ, tɛ́ lékó yɔ́ ńtɛ̃ náh tɔ̃̌nɩ̀ sõ̀ dah mɔ yah ké sɔkɔ. 
Tɛ́ lésõlɛ, gblɔ̌y vìnsàhnɔ́ yòh ye sõ̀ Nsyifunɔ tyɩ́, dendal mɩ mɩ yòh yĩnnɛ. 
Syɔ́ nónó-á sõ̀ wil Yesulɛ, Kalile kwil sɔkɔ, pè Yosɛfu sõ̀ tɛ̃ kyɩ diki yõ. Pè kyɩ diki yah mɔ, ànɛ̂ Yesu dáhnmɔ dùkùlɛ ké sɔkɔ. 
Képah náh, pè pé syɩ to yuku kwil sɔkɔ, à kyɩ nɩrɩnɔ́, ànɛ̂ làsèkwěl pèplɔ yah sah lékó yĩnnɛ. Tɛ́ mɛ mɩ, pé mɩ mɩ yòh sṍlɛ, sɔ̃́-á kélɛ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ.
