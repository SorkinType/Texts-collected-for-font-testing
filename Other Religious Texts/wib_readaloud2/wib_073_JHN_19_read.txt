Nsyɔ̃.
19.
Képah sɔkɔ, Pilati mɛ pè kɔ̃ pè Yesu tɛ̃ kyɩ ko yèfõhlɛ. 
Sràsyíbí mɛ pã̀n yõ̀kɔ̃ pɩ dyɩ à kɔ̃ yõ̀tɔ̃rɩ́ yṍkúrílɛ, tɛ́ mɛ fwɔ̀mɔ̀nsãh ǹgbɛ̃ yɔ́ mɔ à kɔ̃. 
Képah tɛ̃̀nɛ, pè nɔ de ǹnɛ tɛ́ n yo ǹ tyɩ́ nɛ, Nsyifunɔ yõ̀tɛ̃̀ dĩ́, páh n wɛkɩ ǹ tyɩ́! Tɛ́ mɛ ǹ létáhlɔ́lɛ n kɔ̃lɩ. 
Képah sɔkɔ, Pilati tɔ̃ ǹ syɩ wil pɔ yo tɔ́wû tyɩ́ nɛ, pè yah yɔ̀, páh kɩ Yesu wilki pɔ kégbɔ́ sɔkɔ, képah sɔkɔ pǎh kɩ pɩ̃ nɛ pé náh pílkí-ńsah yɔ́ ńtɛ̃ wɛ ǹ tyɩ́. 
Yesu mɛ wil pɔ kégbɔ́ pã̀n yõ̀kɔ̃lɛ ǹ yõ sɔkɔ, ànɛ̂ fwɔ̀mɔ̀nsãh ǹgbɛ̃nɛ mɔ́nɔ́lɛ ǹ tyɩ́. Képah tɛ̃̀nɛ, Pilati mɛ yo pé tyɩ́ nɛ, pè yah névi mɛ̀nɛ kɛ̀. 
Sárká syínwìlkìpu yõ̀tãm ànɛ̂ Liyel ní-ńsah gbô syɩ́kɩ́pú-á fyɔ̀ à wɛ, pè gbo mɔ tũ nɛ, à à syɩ dahnɩ yõ! À à syɩ dahnɩ yõ! Pilati mɛ yo pé tyɩ́ nɛ, péwɔ tɛ̃̀ wɔ náh pílkí-ńsah yɔ́ ńtɛ̃ wɛ ǹ tyɩ́. Pépi kyɩ à syɩ dahnɩ yõ sɔkɔ pé gblɔ̌y. 
Wǎh yo sɛ̃́, Nsyifunɔ à syah nɛ, tir kõ̀nsàhnɔ́ yɔ́-á mɛ pépi tyɩ́, ńkɛ̃́ à yɩ̃nɛ képah sɔkɔ wɛynɛ, dĩ́ mɛ̀-á yɩ̃nɛ kònmɔnɔlɛ, wǎh sɔ̃́npɩ́lɛ ǹ gblɔ̌ynɛ n pɩ Liyel Pídĩ́nɛ. 
Pilati-á kénɛ wɛy noh sɛ̃́, tíkí tɔ̃ à gbah tɛ̃. 
À mɛ ǹ syɩ de ǹ gbô sɔkɔ à kyɩ Yesu piki yah nɛ, ńyãh se à n wil nɩ? Ǹtɛ Yesu náh dahbɩ wilki ǹ tyɩ́. 
Képah sɔkɔ, Pilati mɛ à syah nɛ, péwɔ tyɩ́ se à yahle vìnmɔnɔlɛ nɩ? À kó pɩ̃ nɛ, tɛ̃́nwɛnɔ-á mɛ péwɔ tyɩ́ kɩ fɛ̃ à fĩn mɔ, képah náh pɩ kɩ fɛ̃ à syɩ dahnɩ yõ nɩ? 
Ǹtɛ Yesu mɛ yo ǹ tyɩ́ nɛ: «Tɛ̃́nwɛnɔ yɔ́ ńtɛ̃ náh n wɛ n pi á tyɩ́ ńmɔ yõ sɔkɔ, nɛ́ Liyel náh kélɛ ǹ kɔ̃. Képah ye, nɛ̂-á dahbɩ ńmɔlɛ á gbõ̀, ǹmɔ tyípékè kal áwɔ tyɩ́lɛ.» 
Wǎh képah yo, Pilati sõ̀ mɛ ǹ wɛ́nfĩnnmɔ-ńsahlɛ n yah n kɔ̃. Ǹtɛ Nsyifunɔ sõ̀ mɛ kah gbo mɔ tũ nɛ, wàh kɩ à fĩn mɔ, Wrome kwíltãm yõ̀tɛ̃̀ Sesar kódĩ́ tɛ̃̀ wɔ náh ǹnɛ dɛ́! Ńkɛ̃́ névi nɛ̂-á fyɔ̀ ǹ gblɔ̌ynɛ n pɩ yõ̀tɛ̃̀nɛ, Sesar sépɛ-á tɛ̃̀nɛ dò. 
Pilati-á fyɔ̀ wɛlɔ nɔ́pi noh, à pè kɔ̃ pè Yesu tɛ̃ wil kégbɔ́. Tɛ́ mɛ kɔ̃ tɛ̃ tukey tuke tuke kǒ yõ fɩ̃́ yɔ́ sɔkɔ, ńyãh-á mɛ mɔ́nɔ́lɛ dyêhlɛ. Pè nɛ́npɔ́lɛ n ye Kabatalɛ Yebre wɛy sɔkɔ. 
Ké dĩndal kɩ sõ̀ n pɩ n pi Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáhlɛ. Tɛ́ gbĩ-á mɛ yõ̀tah sírr kénkɔ̃ sɔkɔ, Pilati mɛ yo Nsyifunɔ tyɩ́ nɛ, pè yah pé yõ̀tɛ̃̀nɛ kɛ̀. 
Ǹtɛ pè tɔ̃ kah gbo mɔ tũ nɛ, à à ko mɔ! à à ko mɔ! À à syɩ dahnɩ yõ! Pilati mɛ nɛ, pé yõ̀tɛ̃̀nɛ pè nɛ pé syɩ dahnɩ yõ lɛ́? Sárká syínwìlkìpu yõ̀tãm mɛ yo ǹ tyɩ́ nɛ, yõ̀tɛ̃̀ yɔ́ ńtɛ̃ náh mɛ pépi tyɩ́, à wil Sesar tyɩ́ sɔkɔ. 
Képah sɔkɔ, Pilati mɛ ǹ gbàh dɛ mɔ nɛ à syɩ dahnɩ yõ. Sràsyíbí mɛ Yesu tɛ̃ sɔkɔ. 
Pǎh n sɔkɔ, pè à kɔ̃ à ǹ syɩ syɩ dahnɩ twah ǹ gblɔ̌y pè wil kwil sɔkɔ, à yuku fɩ̃́ yɔ́ sɔkɔ ńyãh-á pè n ye yõ̀gbèkè tyahlɛ, tɛ́ pè kélɛ n ye Kɔlkotalɛ Yebre wɛy sɔkɔ. 
Nɛ́npɔ́ sɔkɔ ye pè syɩ ǹnɛ dahnɩ yõ, tɛ́ mɛ nɛ́pĩ̂ nímí yísyɔ́ tɔ syɩ dahlɔ yõ ǹ gbɔ̀plɔlɛ, tɛ́ ǹmɔlɛ pé nɩyṍ. 
Pilati mɛ wɛy yɔ́ wãrkɩ à syɩ Yesu syɩ syɩ dahnɩ tyɩ́ ǹ yõ̀tah, nɛ̂ yõ-á kè mɛ wã́rkɩ́nsàhnɔ́lɛ nɛ ńkɛ̃́: «Nasarɛtɩ kwil Yesu, Nsyifunɔ yõ̀tɛ̃̀.» 
Pǎh Yesu syɩ dahnɩ yõ, tɛ́ wɛy nɛ̂ wãrkɩ syɩ kénɛ dahnɩ tyɩ́, ǹ yõ̀tah, kè sõ̀ mɛ Yebre wɛy sɔkɔ, ànɛ̂ Latɛ̃ wɛy sɔkɔ, ànɛ̂ Krɛkɩ wɛy sɔkɔ. Tɛ́ ńyãh sɔkɔ-á sõ̀ pè syɩ ǹnɛ dahnɩ yõ, nɛ́npɔ́ sõ̀ mɛ yúkɛy kwillɛ. Kǎh pɩ sɛ̃́, Nsyifunɔ náhnáh kè wɛ kar. 
Képah sɔkɔ, Nsyifunɔ sárká syínwìlkìpu yõ̀tãm yo Pilati tyɩ́ nɛ, à káh yɛ wã́rkɩ́nɔ́ mɛ̀ tɛ̃ sɛ̃́ nɛ, Nsyifunɔ yõ̀tɛ̃̀-á. Ńkɛ̃́ à kè wãrkɩ mɔ nɛ, dĩ́ mɛ̀-á yo nɛ ńkɛ̃́, Nsyifunɔ yõ̀tɛ̃̀-á péwɔlɛ. 
Ǹtɛ Pilati mɛ yo pé tyɩ́ nɛ, páh fyɔ̀ nɛ̂ wãrkɩ tɛ, kǎh kɩ tɛ̃ sɛ̃́. 
Képah tɛ̃̀nɛ, sràsyíbí-á Yesu syɩ tɛ dahnɩ yõ, pè ǹ flɔ tɔkɔ à dibi tã̀ntã̀n ǹyẽ̂h à pɔ gbàh sah ǹ fwɔ̀mɔ̀-ǹgbɛ̃ tyɩ́. Pé nɛ́núkù núkú pól mɛ tɔkɔ núkú núkúlɛ. Tɛ́ fwɔ̀mɔ̀-ǹgbɛ̃ mɛ̀ sõ̀ mɛ kóhntel-ńsah kɛ̃́nɛ ké tyɩ́. Kè sõ̀ mɛ fwɔ núkúlɛ à tɔkɔ ké yõ̀tah tyɩ́, fɔ́ɔ́ à pɔ syɩ ké sètàhkɩ̀ sõ̀ tyɩ́. 
Pè mɛ yo wɛ̃ tyɩ́ nɛ, pé káh képah wɔ yɔrɩ n tɔkɩ. Pé ye ǹdan mɔ n yah, kàh kɩ pɩ nɛ̂ tyɩ́lɛ. Tɛ́ nɛ̂-á lésõ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Pè ń flɔ dibi pé wɛ̃ wrɔ́, tɛ́ ǹdan mɔ à ń fwɔ̀mɔ̀-ǹgbɛ̃ tɔkɔ.» Képah wil yĩn ye kè kɔ̃ kè pɩ sɛ̃́ yɔ̀. Sɛ̃́ ye sõ̀ sràsyíbí pɩ. 
Tɛ́ Yesu yṹ sõ̀ mɛ yĩ́nntɛ̃̀nɔ́lɛ Yesu syɩ syɩ dahnɩ tnɔ̂, ànɛ̂ ǹ yṹ gbɛ̃̀nsêlɛ, ànɛ̂ Klopasɩ kyɩ Marilɛ, à tahlɩ Makadala kwil Mari tyɩ́. 
Yesu tõ̀ sã́h-ò nɛ̂ tyɩ́-á nɔ yĩ́ĩ́ ǹ tyɩ́, ǹmɔ tɔ sõ̀ mɛ yĩ́nnɔ́lɛ Yesu yṹ tnɔ̂ sɔkɔ. Yesu-á pè wɛ, à yo ǹ yṹ tyɩ́ nɛ: «Á pídĩ́ wɛ yɔ̀.» 
Képah náh, tɛ́ yo ǹ tõ̀ sã́h-ò mɛ̀ tyɩ́ nɛ: «Á nɔ̂ wɛ yɔ̀.» À tɔkɔ légbĩ́ tyɩ́, tõ̀ sã́h-ò mɛ̀ mɛ Yesu yṹ tɛ̃ à kyɩ sah ǹ gblɔ̌y tyɩ́. 
Képah náh, Yesu-á pɩ̃ nɛ tyi pól-á pɩ dal mɔ sɛ̃́, à nɛ: «Níkì mɛ ń tɛ̃.» Képah sɔkɔ, nɛ̂-á wãrkɩ sah Liyel wɛy sɛ́bɛ́y sɔkɔ, kè wil. 
Tɛ́ pĩ́n yɔ́ sõ̀ mɛ nɛ́npɔ́ yõ̀yõ̀kè ninɛ ké yĩ́ĩ́nɛ. Sràsyíbí mɛ sènsúkúrí tɔkɔ à dahbɩ wilki kénɛ yõ̀yõ̀kè ni sɔkɔ, à dyɩ yisɔpɩ ǹdan tyɩ́, tɛ́ kè nɔ à sah Yesu vyãh yah. 
Wǎh kè wɔ, à nɛ: «Tyi pól ye wɛ pɩ dal mɔ.» Wǎh fyɔ̀ képah yo, à ǹ yõ sĩnki tɛ́ ku. 
Nsyifunɔ mɩ mɩ yòh kɩ sõ̀ n tyah n pi lékã̂h mɔ́nɔ́ sɔkɔ. Tɛ́ mɩ mɩ yòh mɛ̀ sõ̀ kal sé no tɛ́lɔ́lɛ. Képah ye, Nsyifunɔ yõ̀tãm náh sõ̀ mɛ ké tyɩ́ nɛ nónó-á mɛ syɩ́nsàhnɔ́lɛ dahlɔ yõ, pé lékyɩ̂ pɔ mɩ mɩ yòh sõ nɔ dahlɔ yõ. Kǎh pɩ sɛ̃́, pè kè yah kɔ̃ Pilati tyɩ́ nɛ, à kénɛ no nɔ́pi gbyẽ̂h kwɔkɩ pè kɔ̃ pè ku súú, pé lékyɩ̂ mó tiki n mɔ n to dahlɔ yõ. 
Képah sɔkɔ, nɛ́pĩ̂ nímí nónó-á sõ̀ pè syɩ dahlɔ yõ ànɛ̂ Yesulɛ, sràsyíbí mɛ pɔ nɛ́gben gbyẽ̂h kwɔkɩ tɛ́ ní-ò tɔ tyɩ́ kwɔkɩ. 
Tɛ́ pǎh yĩni tɛ̃ Yesu tyɩ́, pè kè wɛ ǹmɔ mɛ ku tɛ. Kǎh pɩ sɛ̃́, pè náh fwɔ̌tɛ̃̀ ǹmɔ gbyẽ̂h kwɔkɩ. 
Ǹtɛ képah fyé yõ, sràsyíbí nɔ́pi sɔkɔ yɔ́ Yesu gbɔ̀pɔ tũ mɛ̀mɛ̀kɩ̀lɛ. Nɛ́npɔ́ swɔ, tṍ ànɛ̂ ni n fwɔ n wil nóré mɛ̀ sɔkɔ. 
Névi nɛ̂-á n yo tyi nɔ́pilɛ, ǹ tɔ yĩ́ yah tɔ́tɔ́ sɔkɔ ye sè pɩ. Gblɔ ye ǹ tyíyónɔ́lɛ. Ǹmɔ ńtɛ̃ tɔ pɩ̃ nɛ gblɔ-á pé n yo. À tyi nɔ́pi yo nɛ yépi tɔ sɛ̃ Yesu yõ. 
Ǹtɛ yah, tyi nɔ́pi pól ye pɩ Liyel wɛy sɛ́bɛ́y sɔkɔ tir wã́rkínsàhnɔ́ mɛ̀ kɔ̃ kè wil. Kè lésõ mɛ wã́rkínsàhnɔ́lɛ nɛ ńkɛ̃́: «Ǹ gbèkè yɔ́ ńtɛ̃ náh n kwɔ n pi.» 
Tɛ́ kè tɔ̃ mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y fɩ̃́ yɔ́ sɔkɔ nɛ ńkɛ̃́: «Pǎh névi nɛ̂ tũ, pé yɩ̃́plɔ́ pye kɩ tɛ̃ ǹ yõ.» 
Tɛ́ dĩ́ yɔ́ lésõ mɛ pè n ye Nsyosɛfulɛ. À sõ̀ n wil Arimatɩ kwil sɔkɔ. Yesu tõ̀ sã́h-ò yɔ́ ye sõ̀ ǹ tɔlɛ. Tɛ́ Nsyifunɔ yõ̀tãm tíkí-á sõ̀ mɛ à tɛ̃ sɛ̃́, képah ye à kè wɔ̃bɩ pélɛ. Kénɛ tyi pɩ́nsɔkɔntɛnɔ náh, ǹmɔ mɛ pɔ Pilati ni nɛ, à yɛ pé kɔ̃ pé Yesu lékó dɛ dahnɩ yõ pé kyɩ à yɩ̃. À mɛ ké wɛ̃̀kɩ̀lɛ à kɔ̃. Képah sɔkɔ, à mɛ kyɩ ǹ lékó dɛ. 
Tɛ́ Nikodɛmɩ nɛ̂-á lésõ kyɩ wil Yesu tyɩ́ sõ yɔ́ lékã̂hnɛ, ǹ tɔ mɛ pɔ pútúr pãn pènàl dukenɔ nínɔ́ fɛ̃̀kɩ̀nmɔnɔlɛ wɛ̃ sɔkɔ. Núkú yĩn ye mirlɛ, ní-ò yĩn mɛ alowɛsɩlɛ. Pútúr mɛ̀ sõ̀ nɔ kílónɔ́ yõgbãm kénkɔ̃lɛ. 
Pé nímí mɛ ǹ lékó tɔkɔ, tɛ́ à plo fwɔ gbrólɛ. Pè pãn pènàl yî nɔ́pi mɔ ǹ yõ, à yɩ̃nɛ Nsyifunɔ lékó yɩ̃ dùkùlɛ. 
Pǎh Yesu syɩ dahnɩ yõ ńyãh sɔkɔ, swãh yɔ́ lésõ mɛ nɛ́npɔ́ tnɔ̂ diki sã́n yɔ́lɛ ké sɔkɔ, diki nɛ̂-á nɛy náh yɩ̃ yah ké sɔkɔ póllɛ. 
Tɛ́ Nsyifunɔ mɩ mɩ yòh-á kɩ sõ̀ n tyah n pi lékã̂h mɔ́nɔ́ sɔkɔ sɛ̃́, képah ye pè mɛ Yesu lékó yɩ̃ diki mɛ̀ sɔkɔ, képah nɛ̂-á sõ̀ mɛ yúkɛy.
