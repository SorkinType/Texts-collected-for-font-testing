Markɩ.
3.
Képah náh, Yesu ǹ syɩ de *Liyel ní-ńsah wɔ̀kɔ̀ yɔ́ sɔkɔ. Ké nɛ́npɔ́ sɔkɔ, dĩ́ yɔ́ sõ̀ mɛ gbõ̀lɛ kúnɔ́lɛ. 
*Farisiyɛ̃nɔ mó tyah Yesulɛ n yah, wàh kɩ ńtyɩ̃́nɔ à dɛ mɔ n pi *mɩ mɩ yòh sṍlɛ, pè mó lékàh wɛ à kɔ̃. 
Képah sɔkɔ, Yesu mɛ yo gbõ̀nkù dĩ́ mɛ̀ tyɩ́ nɛ, à yuku n pɔ yĩn névye nɩyṍ. 
Tɛ́ no piki yah: «Tir nɛ̂ wɛ̃̀kɩ̀ mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sṍlɛ: Pèpɔrɩ se yɩ̃nɛ pɩ́nɔ́lɛ, tĩ̂ pèkè se? À névi wilki kũ sɔkɔ, tĩ̂ à tɛ̃̀ ko mɔ?» Ǹtɛ, pè náh tir yɔ́ yo. 
Képah sɔkɔ, wǎh pè yah yah yésyẽ̂hlɛ fɔ̀kɔ̀vyã̀hnɩ̀ sɔkɔ, ǹ yah kwɔ pé kùkùtǎhkɩ̀ tɛ̃̀ tyɩ́. À mɛ yo dĩ́ mɛ̀ tyɩ́: «Á gbõ̀ tɔ̃!» À mɛ ǹ gbõ̀ tɔ̃, kè núkú pɩ ké vilɛ. 
Farisiyɛ̃nɔ-á képah wɛ sɛ̃́, pè yuku wil kénɛ Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ, à núkú kyɩ wɛ̃ tuke mɔ ànɛ̂ yõ̀tɛ̃̀ dĩ́ *Yerɔdi náh névyelɛ Yesu tyɩ́ sɔkɔ, pǎh kɩ pɩ sɔ̃́, tɛ́ à wɛ n ko n mɔ n pi. 
Képah náh, Yesu ànɛ̂ ǹ *tõ̀ sã́hpú yuku Kalile yɔ́-ǹgbɛ̃ gbɔ̀pɔ, tɛ́ tɔ́wû ǹgbɛ̃ yɔ́ sõ̀ mɛ ǹ sõ̀ tɛ̃. Pé túkù n wil Kalile, túkù mó n wil Yude, 
ànɛ̂ Yerusalɛmɩ. Túkù n wil Yidume, túkù n wil Yurdɛ̃ yɔ́nkahnɩnɛ. Túkù mó n wil Tiri ànɛ̂ Sidõ gbɔ̀plɔ kwlo sɔkɔ. Kénɛ tɔ́wû ǹgbɛ̃ ye kyɩ wil ǹ tyɩ́. Pè sõ̀ ǹ tyípɩ́nɔ́ noh, képah yĩn ye. 
Kǎh pɩ sɛ̃́, Yesu yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, pè krótókè yɔ́ tɔ̃ sah pé kɔ̃, tɔ́wû káh pɔ pé fyãh. 
Wǎh sõ̀ mɛ névye náhnáhlɛ n dɛ n mɔ, yámpú pól sõ̀ mɛ n to n tuke ǹ yõ nɛ, pé wɛ tũ ǹ tyɩ́. 
Tɛ́ *sétã̀n-á sõ̀ mɛ nónó sɔkɔ, pǎh à wɛ, pè pé gbètukunɔlɛ n di n pɔ ǹ yah, tɛ́ n yo ǹ tyɩ́ dénsyinɔ sɔkɔ nɛ, Liyel pi-á ǹnɛ. 
Képah sɔkɔ, à dírí kõ pé yõ, tɛ́ nɛ, pè káh pé dùkùlɛ n yo. 
Sṍ yɔ́lɛ, Yesu dɔkɔ yɔ̃lɩ yɔ́ yõ, tɛ́ wǎh sõ̀ mɛ nónó tyɩ́, à pè ye. Pè pɔ ǹ tyɩ́. 
Sɛ̃́ ye sõ̀, à yah wilki kènɛ nɛ́pĩ̂ gbɔ̃nimínɛ, à pɩ ǹ tõ̀npɩbɩlɛ. Nɛ, pè nɛ n mɛ pé tnɔ̂, pé nɛ n wɛ n tõ pé wɛy pèpɛy yo n kɔ̃ yĩnnɛ. 
Tɛ́ tɛ̃́nwɛnɔlɛ pè kɔ̃ nɛ̂-á kɩ fɛ̃ *sétã̀n yĩni n wilki n pi. 
Nɛ́pĩ̂ gbɔ̃nimí nɔ́pi yĩni wɛ yɔ̀: Simolɛ, Yesu ǹmɔ yĩn sah Pyɛrɩlɛ. 
Nsyakɩ ànɛ̂ ǹ gbɛ̃̀ndĩ́ Nsyɔ̃nɛ, pépilɛ, Sebede wɛ̃́npì ye sõ̀. À pépi yĩn sah Gbanɛrsyɛsilɛ, ké yõ ye «nónó-á gbɛ̃́nséntã́hkɩ̀ sɔ̃́». 
Andre tɔ sõ̀ mɛ, ànɛ̂ Filipe, ànɛ̂ Bartelemi, ànɛ̂ Matye, ànɛ̂ Toma, ànɛ̂ Nsyakɩ, nɛ̂-á Alfe pídĩ́nɛ. À tahlɩ Tade, ànɛ̂ Simolɛ, nɛ̂-á sõ̀ ǹ sú fṍ yõ yɔ̃̀pu sɔkɔ yɔ́lɛ, 
ànɛ̂ tɔ̃ Yisikaryɔti Yudasilɛ, nɛ̂-á pɔ vi Yesu náh. 
Képah náh, Yesu de gbô. Tɔ́wû tɔ̃ tuke ǹ yõ. Wrɔ́ ńtɛ̃ náh sõ̀ mɛ ǹmɔ ànɛ̂ ǹ *tõ̀ sã́hpú tyɩ́ yõke yõ yõ tɛ̃̀nɛ. 
Ǹ gbôtãn-á képah noh sɛ̃́, pè yuku wɛ̃̀kɩ̀ tɛ̃ nɛ, páh kɩ kyɩ à tɛ̃, tɛ́ n yo wɛ̃ tyɩ́ nɛ, ǹ lékã́m-á fɛ̃kɩ, wǎh kah mɔ. 
Légbĩ́nɛ, *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú nónó-á wil Yerusalɛmɩ sɔkɔ, pépi nɛ ńkɛ̃, *sétã̀n yõ̀tɛ̃̀ Belsebul-á mɛ Yesu sɔkɔ. Ǹmɔ tǎhkɩ̀-á à kɔ̃, à tɛ̃ wɛ *sétã̀nnɛ n yĩni n wilki névye sɔkɔ. 
Képah tɛ̃̀nɛ, à pé ye, tɛ́ tyah n wɛkɩ pé tyɩ́ sèmìrn sɔkɔ: «Sɔ̃́ mɛ sétã̀n yõ̀tɛ̃̀ kɩ fɛ̃ ǹ gblɔ̌y yĩni wilki névi sɔkɔ? 
Yõ̀tɔ̃rɩ́ núkú névye-à tyah wɛ̃nɛ n fwo, kénɛ yõ̀tɔ̃rɩ́ náh n mo n pi. 
Kéte nɛ̂ névye-à tyah wɛ̃nɛ n fwo, kénɛ kéte tɔ náh n mo n pi. 
Sɛ̃́ntɛ̃̀ ó ye, sétã̀n yõ̀tɛ̃̀-à yuku, à ǹ tǎhkɩ̀ tɔkɩ, à tyah wɛ̃nɛ n fwo, ǹ yõ̀tɔ̃rɩ́ kɩ yɩkɩ, kè náh n mo n pi. 
Névi yɔ́ ńtɛ̃ náh fɛ̃ de n pi kètɛ̃̀npɛ tǎhkɩ̀ wɔ̀kɔ̀, kɩ ǹ gbã̀n yî tɔkɔ, nɛ́ à náh à tɛ̃ pukubi. Wàh à tɛ̃ pukubi tɛ gbĩ́ nɛ̂nɛ, à kɩ pyě ǹ wɔ̀kɔ̀ yî kwel. 
Mé kélɛ n yo yé tyɩ́ kègbɩ yõ: Névye-á tyípêl nónó póllɛ n pɩ, ànɛ̂ Liyel gbyɔ̀nɔ́ póllɛ, sè kɩ fɛ̃ yɔ̃ mɔ pè kɔ̃. 
Ǹtɛ, névi nɛ̂-à *Liyel Mírkí gbyɔ, kè náh n yɔ̃ n mɔ n pi ǹ tyɩ́ póllɛ. À kɩ wũ ké sɔkɔ.» 
À képah yo sɛ̃́ pé tyɩ́, pǎh sɔ̃́npɩ́ yo nɛ, sétõ-á mɛ ǹ sɔkɔ. 
Képah náh, Yesu-á sõ̀ mɛ ńyãh, ǹ yṹpyé ànɛ̂ ǹ yṹ pɔ yĩni tɛ̃ kénɛ gbô yah, tɛ́ nɛ tɛkɩ mɔ à kyɩ à ye. 
Tɔ́wû sõ̀ mɛ kɔ̃ tɛ̃ Yesu mɔ gbɩ̃, tɛ́ pè yo ǹ tyɩ́ nɛ, à noh yɔ̀, ǹ yṹ, ànɛ̂ ǹ yṹpyé-á mɛ kégbɔ́. Pǎh mɛ ké tyɩ́ nɛ, pé à wɛ. 
À mɛ yo pé tyɩ́: «Nɛ̂ mɛ ńmɔ yṹnɛ, ànɛ̂ nónó mɛ ń yṹpyélɛ?» 
Képah náh, névye nónó-á sõ̀ mɛ tɛ̃ à mɔ gbɩ̃, à yɔ̃ pè yah kore vi, tɛ́ yo: «Yé yah, ńmɔ yṹ, ànɛ̂ ń yṹpyé mɛ ásõ̂. 
Névi nɛ̂-á Liyel nɩ̀vɩ̀lɛ n pɩ, ǹmɔ ye ńmɔ yṹpi, ànɛ̂ ń yṹnɛ.»
