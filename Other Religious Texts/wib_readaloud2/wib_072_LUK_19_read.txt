Likyi.
19.
Yesu-á de Yeriko kwil, à sõ̀ mɛ n kah ké sɔkɔ, 
tɛ́ dĩ́ yɔ́ sõ̀ mɛ kénɛ kwil, pè n ye Nsasyɩlɛ. Mɔ́rɩ́ syípú yõ̀tãm sɔkɔ yɔ́ ye sõ̀ ǹnɛ, tɛ́ à pɩ tàhlɛ. 
À sõ̀ mɛ n yah n kɔ̃ nɛ, pé Yesu wɛ, nɛ́dúkú nɛ̂-á ǹnɛ. Tɛ́ ǹmɔ-á pɩ nɛ́túkúrí nɛynɛ, tɔ́wû tnɔ́nkàhnɔ́ tɛ̃̀ tyɩ́, à náh sõ̀ fɛ̃ à wɛ n pi. 
Kǎh pɩ sɛ̃́, à to yuku yah tĩ̀nnɛ, à kyɩ dɔkɔ séswɔ̂ yɔ́ sɔkɔ Yesu wɛ yĩnnɛ. Tɛ́ à kɩ sõ̀ pɔ n sɔkɔ n pi nɛ́npɔ́. 
Yesu-á yĩni tɛ̃ nɛ́npɔ́, à ǹ yah yɔ̃ tɛ́ yo Nsasyɩ tyɩ́ nɛ ńkɛ̃́, à tiki súú Nsasyɩ, páh yɩ̃nɛ pé twah ǹ kéte sɔkɔ kwɛynɛ. 
Képah tɛ̃̀nɛ, Nsasyɩ núkú tiki súú, à kyɩ Yesu tɔ̃ sah tɛ̃ ǹ tĩ̀nnɛ nɩ̀vɩ̀ sɔkɔ. 
Névye-á képah wɛ sɛ̃́, pé pól tyah ǹ tyɩ́lɛ n plɔ n yãh nɛ, tyípêl pɩ́-ò tyɩ́-á à kyɩ twah yɔ̀ dɛ́! 
Ǹtɛ, Nsasyɩ mɛ nɔ de Tɛ̃̀ ǹgbɛ̃ Yesulɛ, tɛ́ yo ǹ tyɩ́ nɛ, à noh pé tyɩ́, pé yõ̀tɛ̃̀. Páh kɩ pé gbã̀n yî tɔ́nɩ́ núkú yɩ̃ yãm tãn kɔ̃. Ànɛ̂, pâh mɛ syɩkɩ nɛ̂ yõ, páh kɩ sɛ̃́nknɔnɔ ńyã́hlɛ tɛ̃̀ dyah. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́: «Áwɔ tɔ-á pɩ Abrahmɩ náh nɛynɛ, képah ye, pwáhnmɔnɔ de kéte mɛ̀ sɔkɔ kwɛynɛ. 
Yah, nónó-á sõ̀ mɛ mɔ́npunɔlɛ, pépi yáhnkɔ̃-ńsah ye Névi Pi pɔ kɩ pè pwah mɔ.» 
Képah tɛ̃̀nɛ, Yesu-á de n tɛ Yerusalɛmɩ sɔkɔ, ǹ wɛlɔ nóhpú sõ̀ mɛ n sõ nɛ, Liyel tɔ̃́rɩ́-á kɩ n pɔ n pi nɛ́npɔ́ swɔ. Képah sɔkɔ, Yesu mɛ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: 
«Yõ̀tɛ̃̀ yɔ́ pídĩ́ ye lésõ. Sṍ yɔ́lɛ, à yuku sɔkɔ kwil fɩ́ɩ́ yɔ́ sɔkɔ nɛ, pé kyɩ yõ̀tɔ̃rɩ́ yah n kɔ̃, képah náh, pé mó pé syɩ pɔ. 
Wǎh n sɔkɔ, à ǹ tõ̀npɩbɩ nɛ́pĩ̂ gbãm ye pɔ, pɔ́ dan núkúlɛ pé nɛ́núkù núkú pól kɔ̃, tɛ́ yo pé tyɩ́ nɛ, pè gbɔ̀lɛ n yɩ̃ sépilɛ fɔ́ɔ́, kè pɔ syɩ pé syɩ́npɔnɔ tyɩ́. 
Tɛ́, ǹ kwil névye sõ̀ mɛ ǹ dyɔlɛ n yɔ. Pè mɛ tɛ̃ntõ̀ tɛkɩ mɔ ǹ náh, à kyɩ yo nɛ ńkɛ̃́, pépi náh mɛ ké tyɩ́ nɛ, dĩ́ mɛ̀ tɛ̃̀ pɔ pɩ pépi yõ̀tɛ̃̀nɛ. 
Ǹtɛ, à mó tɔ̃ yõ̀tɔ̃rɩ́ mɛ̀ wɛ, tɛ́ ǹ syɩ pɔ. Wǎh pɔ, à ǹ tõ̀npɩbɩ nɔ́pi ye, wǎh sõ̀ pɔ́ dibi nónó kɔ̃. Pǎh gbɔ̀ yɩ̃, tɛ́ pɔ́ dòhnɔ̀ nɛ̂ wɛ, à sõ̀ mɛ ké tyɩ́ nɛ, pé kè pɩ̃. 
Képah sɔkɔ, nɛ́gben pɔ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, wǎh pɔ́ dan nɛ̂nɛ pé kɔ̃, páh ké kénknɔnɔ gbãm yah dohnɩ. 
Yõ̀tɛ̃̀ dĩ́ mɛ yo ǹ tyɩ́ nɛ, kǎh plɛ, tõ̀npɩ pèpɛy-á ǹnɛ. Wǎh fyɔ̀ pɩ nɛ́gbɩ́lɛ tir tyɩ̃́ɩ̃ sɔkɔ, wǎh kɩ pɩ kwlonɔ gbãm yah tɛ̃́-òlɛ. 
Ní-ò mɛ pɔ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, wǎh pɔ́ dan nɛ̂nɛ pé kɔ̃, páh ké kénknɔnɔ kwlɔ yah dohnɩ. 
Yõ̀tɛ̃̀ dĩ́ mɛ képah núkú ó yo ǹ tɔ tyɩ́ nɛ, ǹmɔ pɩ kwlonɔ kwlɔ yah tɛ̃́-òlɛ. 
Kèyǎh mɛ pɔ yo ǹ tyɩ́ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, ǹ pɔ́-á yɔ̀. Páh sõ̀ mɛ kè dah mɔ sah fwɔ yɔ́ sɔkɔ ké tyɩ́lɛ. 
Ńkɛ̃́, ǹ tõ̀-á kah tah. Képah-á, ǹ tíkí mɛ pé tɛ̃. Ǹmɔ náh sah kwâh nɛ̂nɛ, ǎ kè tɔkɔ. À náh sèmukɔ nɛ̂ duku mɔ, ǎ képah tɔkɔ. 
Képah tɛ̃̀nɛ, yõ̀tɛ̃̀ dĩ́ mɛ à syah nɛ, tõ̀npɩ túkù, ǹ vyãh wɛy mɛ̀ sɔkɔ-á pé kɩ tuke ǹnɛ. Ǹmɔ-á sõ̀ pɩ̃ nɛ ńkɛ̃́, pé tõ̀-á tah, pé náh kwâh nɛ̂ sah, pě kè tɔkɔ, ànɛ̂ pé náh sèmukɔ nɛ̂ duku, pě kɩ tɔkɔ. 
Ǹtɛ, kwâh nɛ̂ se pɩ, tɛ́ à náh pé pɔ́ dah mɔ sah pɔ́ dáhnmɔ-ńsah wɔ̀kɔ̀ sɔkɔ nɩ? Páh pé syɩ pɔ, páh náhkɩ sè wɛ wilki dòhnɩ̀nɔ́lɛ sé yõ. 
Képah sɔkɔ, nónó-á sõ̀ mɛ nɛ́npɔ́, à ǹ syɩ yo pé tyɩ́ nɛ, pè pɔ́ mɛ̀ syi ǹ tyɩ́, pè kélɛ ké kénknɔnɔ gbãm nɛ́tɛ̃̂ kɔ̃. 
Pè mɛ yo ǹ tyɩ́ nɛ, pé yõ̀tɛ̃̀, sé gbãm-á mɛ tɛ ǹmɔ tyɩ́ dò! 
Ǹtɛ, à mɛ pè syah nɛ: «Mé kélɛ n yo yé tyɩ́ nɛ: névi nɛ̂ tyɩ́ kwâh-á mɛ, pè kɩ tɔ̃ ǹmɔ kɔ̃. Tɛ́ névi nɛ̂ tyɩ́ kwâh náh mɛ, kwéyy nɛ̂ ńtɛ̃-á mɛ ǹ gbõ̀, ké tɔ kɩ tɛ̃ syi ǹ tyɩ́. 
Ǹtɛ, ń sépɛ́bɩ́ tɛ̃̀ wɔlɛ, pépi nónó náh sõ̀ mɛ ké tyɩ́ nɛ, mé pɩ pé yõ̀tɛ̃̀nɛ, yé pɔ pélɛ ásõ̂, yé pɔ pé yɛ̀sõ̀ kõ ń yĩ́ yah.» 
Képah yóntɛnɔ náh, Yesu yĩn pé yah, pè n dɔkɔ n yuku Yerusalɛmɩ. 
Tɛ́ wǎh yĩni tɛ̃ n tɛ gbĩ́ nɛ̂nɛ Betefasyɩ ànɛ̂ Betanɩlɛ yɔ̃lɩ yɔ́ tnɔ̂, pè n ye Wólívî yèh yɔ̃lɩlɛ, à ǹ tõ̀ sã́hpú nɛ́pĩ̂ nímí tõ nɛ: 
«Kwil nɛ̂-á yé yahlɛ, yé kyɩ ké sɔkɔ. Yáh kɩ n de, yé kɩ ńkwɛ̃́ndyɩ́pìkì yɔ́ wɛ tóhnmɔnɔlɛ nɛ̂-á nɛy náh dɔkɔ yah ǹ yõ póllɛ. Yé à dɛ n pɔ ásõ̂. 
Tɛ́ nɛ̂-à yé piki yah nɛ ńkɛ̃́, sɔ̃́ se, tɛ́ yé ǹnɛ n dɛ nɩ, yé yo tɛ̃̀ tyɩ́ nɛ, ǹ yétõ̂-á mɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́, képah-á. 
Wǎh yo sɛ̃́, tɛ̃ntõ̀ névye nɔ́pi sɔkɔ. Pè ńtyɩ̃́nɔ kyɩ tyi nɔ́pi wɛ, sɔ̃́-á sõ̀ à yo kélɛ pé tyɩ́. 
Pǎh mɛ ǹ dɛ́nɔ́ sɔkɔ gbĩ́ nɛ̂nɛ, ǹ nɛ́tãm pè piki yah nɛ, sɔ̃́ se, tɛ́ pè ńkwɛ̃́ndyɩ́pìkì mɛ̀nɛ n dɛ nɩ? 
Pè mɛ yo pé tyɩ́ nɛ ńkɛ̃́, ǹ yétõ̂-á mɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́, képah-á. 
Tɛ́ mɛ yuku ǹnɛ Yesu tyɩ́. Pè pé flɔ mɔ ǹ yõ, à Yesu kɔ̃ à dɔkɔ. 
Képah tɛ̃̀nɛ, wǎh mɛ n yuku sɔ̃́, névye mɛ pé flɔlɛ n mɔ n yuku sɛ̃́ wɛ̃̀kɩ̀ yõ. 
Pǎh n tyah Wólívî yèh yɔ̃lɩ tíkíntɛnɔlɛ, nɩ̀vɩ̀ de ǹ tõ̀ sã́hpú kwĩnki pól sɔkɔ. Pǎh gbǐl tyi nónó pól wɛ, sépi yĩnnɛ, pè tyah Liyel yĩnnɛ n gbilki yĩ́npɔ̃́ ǹgbɛ̃nɛ, 
mɛ yo tũ nɛ ńkɛ̃́: «Yõ̀tɛ̃̀ nɛ̂-á n pi Liyel yĩn yõ, Liyel sãm mɔ ǹ sɔkɔ! Yèvɩnɩ nɛ mɛ yĩ̂nyõ, gbílkínɔ́ mó nɛ n sah Liyel yõ, ǹmɔ nɛ̂-á yĩ̂nyõ fléyy tɛ̃̀-ǹsah.» 
Képah sɔkɔ, tɔ́wû nɩyṍ sɔkɔ Farisiyɛ̃nɔ yísyɔ́ Yesu syah nɛ, pé kwɔ́-ò dĩ́, à ǹ tõ̀ sã́hpú vyɛ̃y tũ kɛ̀! 
Képah tɛ̃̀nɛ, Yesu pè syah nɛ: «Mé kélɛ n yo yé tyɩ́ nɛ: pàh kɩ sɔ̃́ pé vyɛ̃y vike ò, dyêh ye kɩ n gbo n mɔ pé swɔ.» 
Yesu-á de n tɛ Yerusalɛmɩ, tɛ́ kè wɛ gbĩ́ nɛ̂nɛ, à n gbo ké yĩnnɛ nɛ: 
«É, Yerusalɛmɩ, wáh náhkɩ sõ̀ mɛ fɛ̃ syi yèvɩnɩ wɛ dùkù pɩ̃ kwɛy mɛ̀nɛ! Ǹtɛ wéèè, kè ye wɔ̃bɩ á yah! 
Yah, sèpĩ̀n yísyɔ́ mɛ n pi, á sépɛ́bɩ́ kɩ wɔ̃̀bɩ̀-ǹtã̀n pɩ á kɛ̃kɩ kore vi, tɛ́ kɩ ǹ fyãh sah yey yey póllɛ. 
Liyel-á pɔ á yòhnɩ̀nmɔ-ńsah gbĩ́ nɛ̂nɛ, á náh fɛ̃ syi kè pɩ̃. Képah ye, pè kɩ á sɔkɔ névye pól ko mɔ, tɛ́ kɩ ǹ mukuli. Á dáhkɩ̀ yɔ́ ńtɛ̃ náh n wũ n pi ké ní-ò yõ.» 
Yesu-á kyɩ de Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀, tɛ́ yíyã̌m yã̀mpu wɛ ké sɔkɔ, à tyah pélɛ n yĩni n wilki. 
Tɛ́ mɛ n yo pé tyɩ́: «Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɔkɔ nɛ ńkɛ̃́: ‹Ń wɔ̀kɔ̀ kɩ pɩ nírí pɩ́-ńsah wɔ̀kɔ̀lɛ›. Tɛ́ yépilɛ, yé kè tɔkɔ pɩ névye yõ syɩ́kɩ́pú tɛ̃̀-ǹsahlɛ.» 
Tɛ́ à sõ̀ mɛ névyelɛ n kwɔ sõmɛsṍ Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ. Képah sɔkɔ, sárká wìlkìpu yõ̀tãm, ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú sõ̀ mɛ ǹ kònmɔ-ńsahlɛ n yah n kɔ̃, kwil yah tãm tɔlɛ pélɛ. 
Tɛ́ no pól-á sõ̀ mɛ pé ní kwɔ n tɔ̃ n sah n nohnɩ n yĩ́ĩ́ ǹ tyɩ́ sɛ̃́, képah ye, pè náh pɩ̃ pǎh kɩ n pɩ n pi sɔ̃́.
