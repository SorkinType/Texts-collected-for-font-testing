Likyi.
5.
Sṍ yɔ́lɛ, Yesu-á sõ̀ mɛ Yenesarɛtɩ yɔ́-ǹgbɛ̃ gbɔ̀pɔ, tɔ́wû pɔ à mɔ gbɩ̃ sah pílpíl, Liyel wɛy noh yĩnnɛ. 
Yesu mɛ krótónè nínì wɛ yɔ tnɔ̂ sɔkɔ, tɛ́ yɛ̃́nkyɔ tɛ̃́pú mɛ wil sé sɔkɔ, mɛ pé yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔlɛ n wu. 
À mɛ de krótónè nɔ́pi núkú sɔkɔ, nɛ̂-á sõ̀ Simɔ tyɩ́lɛ. Tɛ́ kè yah kɔ̃ ǹ tyɩ́ nɛ, à yal kélɛ yɔ vyãh tyɩ́, à de yuku yɔ sɔkɔ kwéyy. Képah-á pɩ, à kɔ̃ tɛ̃ krótókè sɔkɔ, à yɔ̃ sɔkɔ tɔ́wûlɛ n kwɔ. 
Wǎh wɛkɩ tɛ gbĩ́ nɛ̂nɛ, à yo Simɔ tyɩ́ nɛ, à de yuku yahlɛ yɔ tì tɛ̃̀-ǹsah. Pè mó pé mnɔ min dah mɔ yɔ sɔkɔ, pè yɛ̃́nkyɔ tɛ̃. 
Simɔ mɛ à syah nɛ, pé yõ̀tɛ̃̀, pépi-á tõ̀ pɩ sõ, tɛ́ pé náh kwâh yɔ́ ńtɛ̃ wɛ tɛ̃. Ǹtɛ, ǹmɔ yóré yõ, páh kɩ mnɔ min dah mɔ. 
Képah sɔkɔ, pè mɛ pé mnɔ min dah mɔ yɔ sɔkɔ, tɛ́ yɛ̃́nkyɔ wɛ tɛ̃ tnɔ́nɔ́lɛ fɔ́ɔ́, pé mnɔ tyah n kɔ̃lɩ. 
Pè mɛ pɩ mɔ pé no tyɩ́, nónó-á sõ̀ mɛ krótókè ní-ò sɔkɔ nɛ, pè pɔ pé yohnɩ mɔ. Pè mɛ pɔ pè yohnɩ mɔ, pè yɛ̃́nkyɔ wilki mɔ krótónè nínì nɔ́pi yĩ́ĩ́nɛ fɔ́ɔ́, sè n tyah tíkíndenɔlɛ ni sɔkɔ. 
Simɔ Pyɛrɩ-á képah wɛ sɛ̃́, à ǹ gbètukunɔ di Yesu yah, tɛ́ nɛ, pé Yõ̀tɛ̃̀, tyípékè pɩ́-ò-á pélɛ. Képah yĩnnɛ, à gbɛ mɔ péwɔlɛ. 
Pǎh yɛ̃́nkyɔ nɔ́pi wɛ tɛ̃ pípí sɛ̃́, tíkí mɛ Simɔ ànɛ̂ pé no pól kah tɛ̃, 
à tahlɩ Sebede dĩ́npì tyɩ́, Nsyakɩ ànɛ̂ Nsyɔ̃nɛ, pépi nónó-á sõ̀ Simɔ tõ̀ pɩ mûnɛ. Képah tɛ̃̀nɛ, Yesu mɛ yo Simɔ tyɩ́: «Káh tíkí pɩ! À tɔkɔ núkúnúkú tyɩ́, á kɩ pɩ névye yénpɔ-òlɛ Liyel tyɩ́.» 
Képah náh, pè wil pɔ krótónè sah kènkwã̀h yõ ànɛ̂ pé vyɔ́ póllɛ, tɛ́ mɛ Yesu sõ̀ tɛ̃. 
Sṍ yɔ́lɛ, Yesu sõ̀ mɛ kwil yɔ́ sɔkɔ, tɛ́ dĩ́ yɔ́ tɔ sõ̀ mɛ nɛ́npɔ́, dú ńyânnɛ ǹ wil pól sɔkɔ. Wǎh Yesu wɛ sɛ̃́, à kɔ kwĩnki, ǹ yah di sétáh Yesu yah, tɛ́ à ni nɛ, pé Yõ̀tɛ̃̀, wàh mɛ ké tyɩ́, wǎh kɩ fɛ̃ pé dɛ mɔ. 
Wǎh yo sɛ̃́, Yesu núkú ǹ gbõ̀ nɔ, à tũ ǹ tyɩ́, tɛ́ nɛ: «Mé mɛ ké tyɩ́, á kɩ dɛ!» Nɛ́npɔ́ swɔ, dĩ́ núkú dɛ. 
Képah sɔkɔ, Yesu kè gbah yo ǹ tyɩ́ nɛ, à káh yĩ mɛ̀ yãh nɛy tyɩ́ póllɛ. Tɛ́ à kyɩ ǹ gblɔ̌ynɛ sárká wìlkìpu yõ̀tɛ̃̀ wɛ̃kɩ. Moyisi-á sõ̀ sárká nɛ̂ tyɩ́ yo sah tótãm dɛ́nmɔnɔ tyɩ́ sɔkɔ, à mó kè pɩ n tũ. Képah-á kɩ ǹ dɛ́nmɔnɔlɛ pè wɛ̃kɩ. 
Képah fyé yõ, wãn sõ̀ mɛ n dohnɩ Yesu yĩn wìlnɔ́ yõ, tɛ́ nɛ́kwĩ́nkí sõ̀ mɛ n tuke n mɔ ǹ wɛy noh yĩn, ànɛ̂ pé yamnɔ mɔ́ndɛnmɔ yĩnnɛ. 
Tɛ́ Yesu wɔ sõ̀ ńkɛ̃́nɛ n gbɛ n mɔ n tɛ̃ ǹ gblɔ̌y gbɛ̃́nnɩ sɔkɔ, ǹ Liyellɛ n ni. 
Sṍ yɔ́lɛ ńtɔ̃, Yesu sõ̀ mɛ kwɔ́nɔ́lɛ n pɩ, tɛ́ Farisiyɛ̃nɔ ànɛ̂ Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú tɔ sõ̀ mɛ kɔ̃́ntɛ̃̀nɔ́lɛ nɛ́npɔ́. Pè wil Kalile ànɛ̂ Yude kwlo pól sɔkɔ, ànɛ̂ Yerusalɛmɩ tɔ sɔkɔ. Tɛ́ Tɛ̃̀ ǹgbɛ̃ Liyel tǎhkɩ̀ sõ̀ mɛ Yesu sɔkɔ. Képah ye à kɔ̃, à yámpúlɛ n dɛ n mɔ. 
Légbĩ́nɛ, nósyɔ́ sèkùkù dĩ́ yɔ́ twah pɔ ǹ sɛ̃ sɛ̃ kwâh yõ, mɛ n yah n kɔ̃ nɛ, pé de ǹnɛ wɔ̀kɔ̀, pé kyɩ à tiki sah Yesu yah. 
Tɛ́ névye tnɔ́nkàhnɔ́ tɛ̃̀ tyɩ́, pè dé-ńsah yah kwlɔ. Kǎh pɩ sɛ̃́, pè dɔkɔ gbɛ̃́ntĩ̂n yõ kye, à sèkùkù mɛ̀ tiki mɔ ànɛ̂ ǹ sɛ̃ sɛ̃ kwâh tɔlɛ, névye nɩyṍ Yesu yah. 
Yesu-á wɛ pè sɛ̃ ǹ yõ sɛ̃́, à mɛ yo sèkùkù mɛ̀ tyɩ́: «Dĩ́, á tyípêl ye yɔ̃ mɔ ǹ kɔ̃.» 
Képah tɛ̃̀ sɔkɔ, Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú ànɛ̂ Farisiyɛ̃nɔ tyah n sõ nɛ, nɛ̂ tɛ̃̀ ó se dĩ́ mɛ̀nɛ, tɛ́ à Liyel yĩnnɛ n yɩkɩ sɛ̃́ nɩ? Nɛ̂ se kɩ fɛ̃ tyípêl yɔ̃ mɔ névi kɔ̃, nɛ́ Liyel ó náh nɩ? 
Ǹtɛ, Yesu-á pé fɔ́kɔ́sɔ̃nɔ pɩ̃ sɛ̃́, à pé syah nɛ: «Kwâh nɛ̂ pɩ, tɛ́ yé n sõ sɛ̃́? 
Nɛ̂ tɛ̃̀ dɔ̀dɔ̀ kal, à yo sèkùkù mɛ̀ tyɩ́ nɛ, ǹ tyípêl-á yɔ̃ mɔ à kɔ̃, képah náh pɩ, à yo ǹ tyɩ́ nɛ, à yuku nɛ n kɔ? 
Ǹtɛ, mé mɛ ké tyɩ́ nɛ, yé pɩ̃ nɛ, tɛ̃́nwɛnɔ-á mɛ Névi Pi tyɩ́ sétáh yõ, kɩ fɛ̃ névi tyípêl yɔ̃ mɔ à kɔ̃», tɛ́ ǹ syɩ yo sèkùkù mɛ̀ tyɩ́: «Mé kélɛ n yo á tyɩ́: yuku, á sɛ̃ sɛ̃ kwâh tɔkɔ, ǹ nɛ n yuku á gbô!» 
Nɛ́npɔ́ swɔ, dĩ́ núkú yuku no pól yĩ́ yah, à sɛ̃ sɛ̃ kwâh tɔkɔ, à sɔkɔ ǹ gbô, mɛ Liyel yĩnnɛ n gbilki n yuku. 
Képah tɛ̃̀ sɔkɔ, no pól gbã̀n du. Pè Liyel yĩnnɛ n gbilki tíkí tɛ̃̀ tyɩ́, mɛ yo tũ nɛ, kwɛynɛ, páh tyi snɔ́ wɛ dɛ́! 
Képah náh, Yesu wil nɛ́npɔ́, tɛ́ mɔ́rɩ́ syí-ò Levi wɛ kɔ̃́ntɛ̃̀nɔ́lɛ mɔ́rɩ́ syí-ńsah wɔ̀kɔ̀ sɔkɔ. À yo ǹ tyɩ́ nɛ: «Ń sõ̀ tɛ̃!» 
Levi núkú ǹ tõ̀ sah, tɛ́ yuku ǹ sõ̀ tɛ̃. 
Képah tɛ̃̀nɛ, Levi yõke ǹgbɛ̃ pɩ ǹ gbô Yesu yĩnnɛ, tɛ́ mɔ́rɩ́ syípú náhnáh ànɛ̂ nɛ́káhbɩ́ kwĩ́nkí sõ̀ mɛ pélɛ kénɛ yõke yṍ-ńsah. 
Képah sɔkɔ, Farisiyɛ̃nɔ ànɛ̂ pé Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú sõ̀ mɛ lékàhlɛ Yesulɛ n kɔ̃ mɛ n yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, kwâh nɛ̂ se, tɛ́ pè n yõ tɛ́ n wɔ mɔ́rɩ́ syípú ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂púlɛ wɛ̃ tyɩ́ nɩ? 
Képah tɛ̃̀nɛ, Yesu yo pé tyɩ́ nɛ: «Névye nónó-á mɛ lènɔlɛ, sènpɩ yétõ̂ se mɛ pé tyɩ́? Yámpú tyɩ́ náh sènpɩ yétõ̂lɛ di? 
Névye nónó-á n sõ nɛ, nɛ́gblɔ́-á pélɛ Liyel yah, mé náh pɔ pépi ye yĩnnɛ. Ǹtɛ, nónó-á pɩ̃ nɛ, tyípêl pɩ́pú-á pélɛ, pépi yé-ńsah ye mé pɔ nɛ, pè Liyel wɛ̃̀kɩ̀ tɛ̃.» 
Képah náh, Farisiyɛ̃nɔ yo ǹ tyɩ́ nɛ, Nsyɔ̃ tõ̀ sã́hpú ànɛ̂ pépi tõ̀ sã́hpú-á ńkɛ̃́nɛ níkìlɛ n tɛ̃, ǹ tɛ́ nírílɛ n pɩ. Ǹtɛ, ǹmɔ tyɩ́nɔ tɛ̃̀ wɔ ninɛ n wɔ, tɛ́ yõkelɛ n yõ kɛ̀. 
Wǎh képah noh sɛ̃́, à mɛ yo pé tyɩ́: «Fúr pɩ́-ò-á nónó ye ǹ fúr yõke yõ yĩnnɛ, yépi se kɩ fɛ̃ pè kɔ̃ pè kɩ níkì tɛ̃ pé yé-ò yĩ́ yah? 
Ǹtɛ, sṍ yɔ́ mɛ n pi, pɛ mɛ̀ kɩ pɔ n wilki n pi pé wrɔ́. Légbĩ́nɛ, pè kɩ pyě níkìlɛ n tɛ̃.» 
Képah náh, Yesu mɛ tɔ̃ sèmìrkì yɔ́ mɩ pé tyɩ́ nɛ ńkɛ̃́: «Névi yɔ́ ńtɛ̃ náh fwɔ̀sãn kãm, à fwɔ̀dyàh min n pi. Wâh képah pɩ, fwɔ̀sãn kɩ yɩkɩ, ké gbó mɛ̀ tɔ náh se tɔ̃ fwɔ̀dyàh syi n pi. 
Ànɛ̂ tɔ̃, névi yɔ́ ńtɛ̃ náh dìvɛ̃ sã́n mɔ n pi pley dî sɔkɔ. Wâh képah pɩ, dìvɛ̃ sã́n mɛ̀-à fũ, kè kɩ pley nɔ́pi tɔkɩ, dìvɛ̃ kɩ kwɛ, pley kɩ yɩkɩ. 
Kàh pɩ dìvɛ̃ sã́nnɛ, kè yɩ̃nɛ kè mɔ pley snɔ́. 
Névi nɛ̂-á dar dìvɛ̃ kònɔ́ wɔ́nɔ́lɛ, à náh tɔ̃ n fɛ̃ n syi ké kò-ǹkɛ̃̂nɛ n wɔ. Tɛ̃̀ n nɛ n yo nɛ, dìvɛ̃ kònɔ́-á nɔ kal.»
