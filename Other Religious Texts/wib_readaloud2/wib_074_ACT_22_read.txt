Tyípɩ́nɩ̀.
22.
«Ń súbí ànɛ̂ ń yṹnpyé, yé yé gblɔ̌y yɛ, yè mó noh ń tyɩ́, nɛ̂-á mé kɩ yo kɩ ń vyãh kɔ̃lɩ n pi yé tyɩ́.» 
Képah sɔkɔ, pǎh Pole wɛ à n wɛkɩ pé tyɩ́ Yebre wɛy sɔkɔ sɛ̃́, tyah tɔ̃ gbah pɩ sṹmm. 
À mɛ yo pé tyɩ́ nɛ: «Nsyifu nɛy ye ńmɔlɛ. Mé dubi Tarsɩ kwil, Silisi gbɛ̃̀nyah sɔkɔ. Ǹtɛ Yerusalɛmɩ kwil mɛ̀ sɔkɔ ye mé kyãh yɔ̀. Tɛ́ Kamalyɛlɩ ye sõ̀ kwɔ ńnɛ. À ń kwɔ yĩ́ĩ́ á náh tãn tyi kõ̀nsàhnɔ́ wã̀llɛ. Mé sõ̀ mɛ gbah tɛ̃ yĩ́ĩ́ Liyel yõ sɛ̃́nɔ́ sɔkɔ, sɔ̃́-á yélɛ ké sɔkɔ kwɛynɛ yɔ̀. 
Mé Yesu yõ sɛ̃́pú fõh, fɔ́ɔ́ à pé túkù ko mɔ. Mé pé díbí ànɛ̂ pé syɔ́ túkùlɛ n tɛ̃ n pukubi n dah n mɔ n pɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Sárká syínwìlkìpu yõ̀tɛ̃̀ ǹgbɛ̃ ànɛ̂ Nsyifunɔ yèwàhnɩ̀ pól kɩ fɛ̃ pɩ ń tàrfyɔ̀lɛ ké sɔkɔ. Pépi ńtɛ̃ tyɩ́ ye lésõ mé wɛ syi sɛ́bɛ́ynɔ́lɛ nɛ mé kyɩ á no Nsyifunɔ kɔ̃ Damasɩ kwil, mé mó Yesu yõ sɛ̃́pú wɛ n tɛ̃ n pukubi nónó-á mɛ nɛ́npɔ́, mé pɔ pélɛ Yerusalɛmɩ sɔkɔ, mé pɔ pè fõh. 
Képah sɔkɔ, máh mɛ wɛ̃̀kɩ̀ yõ, tɛ́ mɛ de n tɛ Damasɩ kwillɛ, kyòo yɔ́ núkú wil yĩ̂nyõ sɔkɔ gbĩ̀ntɔ̃̀lɛ, à pɔ kwɛ ń yõ pěrr à ń mɔ gbɩ̃. 
Kǎh pɩ sɛ̃́, mé yɔ̃ syɩkɩ sétáh, tɛ́ yĩ́npɔ̃́ yɔ́ noh nɛ̂-á mɛ n yo ń tyɩ́ nɛ: ‹Sole, Sole, kwâh nɛ̂ pɩ tɛ́ á ńnɛ n fõh sɛ̃́?› 
Mé mɛ piki yah nɛ, nɛ̂ se ǹmɔlɛ nɩ, Tɛ̃̀ ǹgbɛ̃? Kénɛ yĩ́npɔ̃́ tɔ̃ ké syɩ yo ń tyɩ́ nɛ: ‹Nasarɛtɩ kwil Yesu ye ńmɔlɛ, nɛ̂-á á n fõh.› 
Képah sɔkɔ, máh sõ̀ mɛ wɛ̃̀kɩ̀ yõ ànɛ̂ nónónɛ, pépi tɔ sõ̀ kyòo mɛ̀ wɛ; tɛ́ nɛ̂-á n yo n mɔ ń tyɩ́, pè náh tɛ̃̀ wɔ wɛy noh. 
Mé mɛ piki yah nɛ, tir nɛ̂ se mé yɩ̃nɛ mé pɩ nɩ, Tɛ̃̀ ǹgbɛ̃? À mɛ yo ń tyɩ́ nɛ: ‹Yuku n de Damasɩ kwil. Tɛ́ wáh yɩ̃nɛ ǹ tyi nónó pól pɩ, pè kɩ sè yo ǹ syah.› 
Ǹtɛ kyòo mɛ̀ píkínɔ́ sõ̀ mɛ ń yɩ̃́ wɔ. Kǎh pɩ sɛ̃́, áyáh mɛ wɛ̃̀kɩ̀ yõ ànɛ̂ nónónɛ, pépi mɛ ń gbõ̀ sɔkɔ tɛ̃, à de Damasɩ kwil. 
Tɛ́ dĩ́ yɔ́ lésõ mɛ nɛ́npɔ́ pè n ye Ananiyasilɛ. Kénɛ dĩ́ sõ̀ mɛ Liyel wɛ̃̀kɩ̀ tɛ̃́nɔ́ plɛ, mɛ ǹ tyi kõ̀nsàhnɔ́lɛ n pɩ sé wã̀l yõ. Nsyifunɔ nónó pól-á sõ̀ mɛ nɛ́npɔ́, pé pól sõ̀ mɛ ǹ yĩ́ngbɩ́lɛ n yo. 
Ǹmɔ mɛ pɔ ńmɔ tnɔ̂, tɛ́ yo ń tyɩ́ nɛ: ‹Ń yṹnpi Sole, fɛ̃ nɛ n yah ńtɔ̃.› Wǎh yo sɛ̃́, nɛ́npɔ́ swɔ ń yɩ̃́ yɩkɩ, mé mɛ à wɛ. 
Képah sɔkɔ, à mɛ yo ń tyɩ́ nɛ: ‹Liyel nɛ̂-á á náh tãn n gbilki, ǹmɔ ye ǹ yah tɔkɔ nɛ á pé dyɔ yĩ́ĩ́nɛ n pnɛ, ànɛ̂ á pé tõ̀npɩ nɛ́gbɩ́ wɛ yĩ́nɛ, á mó ǹ vyãh wɛy noh. 
Képah ye, wáh tyi nónó wɛ, tɛ́ tyi nónó noh Liyel tõ̀npɩ nɛ́gbɩ́ tyɩ́ sɔkɔ, á kɩ n pɩ n pi sé yónwàh-òlɛ no pól yĩ́ yah. 
Kǎh pɩ sɛ̃́, núkúnúkú káh n tɔ̃ n tɛ̃ n mo. Yuku, á yɩ̃nɛ wèlnwìlkìnɔ́lɛ ni sɔkɔ, á tyípêl mó yɔ̃ mɔ ǹ kɔ̃, Tɛ̃̀ ǹgbɛ̃ Yesu yĩn yéngbonɔ sɔkɔ.› 
Képah náh, mé ń syɩ sɔkɔ Yerusalɛmɩ kwil. Tɛ́ sõ yɔ́lɛ, máh mɛ nírílɛ n pɩ Liyel ní-ńsah gbô sɔkɔ, Liyel wɛ̃́kɩ́nɔ́ yɔ́lɛ ń wɛ̃kɩ. 
Mé Tɛ̃̀ ǹgbɛ̃ wɛ à mɛ n yo ń tyɩ́ nɛ ńkɛ̃́, Yerusalɛmɩ kwil tãn náh ń tyíyónɔ́ fɛ̃ n syi n pi péwɔ tyɩ́ sɔkɔ. Képah-á, mé sal n wil súú Yerusalɛmɩ kwil sɔkɔ. 
Ǹtɛ mé mɛ yo ǹ tyɩ́ nɛ, á no Nsyifunɔ-á kè pɩ̃ yĩ́ĩ́ nɛ, máh lésõ mɛ n kyɩ n pɔ á Liyel ní-ńtã̂n vɩ̀ sɔkɔ, à ǹmɔ yõ sɛ̃́púlɛ n tɛ̃ n dah n mɔ n pɔ wɔ̀kɔ̀npèkè sɔkɔ, tɛ́ pélɛ n ko yèfõhlɛ; 
à tahlɩ pǎh ǹ tyɩ́ yónwàh-ò Yetyɛ̃nɩ ko mɔ gbĩ́ nɛ̂nɛ, ńmɔ tɔ-á sõ̀ mɛ dò. Máh sõ̀ mɛ mɔ ǹ kònmɔpulɛ, tɛ́ pɩ pé fwɔ̀mɔ̀-ǹgbnɔ syɩ́kɩ́-òlɛ dò. 
Képah sɔkɔ, Tɛ̃̀ ǹgbɛ̃ mɛ yo ń tyɩ́ nɛ: ‹Fɩ́ɩ́ sɔkɔ ye mé n tɛkɩ n mɔ álɛ kwlókáhkɩ́ névye tyɩ́. Nɛ n yuku!›» 
Tɔ́wû sõ̀ mɛ Pole wɛlɔlɛ n tɔ̃ n sah n nohnɩ, ǹtɛ wǎh fyɔ̀ pɔ wɛlɔ nɔ́pi yo nɛ, kwlókáhkɩ́ névye tyɩ́-á Tɛ̃̀ ǹgbɛ̃ n tɛkɩ n mɔ pélɛ, pè núkú gbo mɔ wòhòh nɛ, ǹsɔ̃̀ntɛ̃̂ névi-á yɩ̃nɛ kònmɔntonɔlɛ sétáh yõ. À náh yɩ̃nɛ à tɛ̃ min sɔkɔ. 
Pè sõ̀ mɛ kah gbo mɔ tũ, mɛ pé flɔlɛ n dɛ n milki, tɛ́ mɛ nɛ̀ngbɔ̃nɛ n yilki n mɔ. 
Kǎh pɩ sɛ̃́, sràsyíbí yõ̀tɛ̃̀ ǹgbɛ̃ nɛ, sràsyíbí Pole tɛ̃ de pé wɔ̀kɔ̀ ǹgbɛ̃ sɔkɔ, pè mó à ko yèfõhlɛ. Képah sɔkɔ, tir nɛ̂ yĩn-á tɔ́wû n gbo n mɔ ǹ yõ sɔkɔ, páh kɩ ké yõ pɩ̃. 
Ǹtɛ pǎh Pole pukubi mnɔlɛ nɛ páh ǹnɛ n ko gbĩ́ nɛ̂nɛ, sràsyíbí yõ̀tɛ̃̀ dĩ́ nɛ̂-á sõ̀ mɛ nɛ́npɔ́, à yo ǹmɔ tyɩ́ nɛ: «Ké wɛ̃̀kɩ̀ se mɛ yépi tyɩ́ kɩ fɛ̃ Wrome tãn gbõ̀ mɔ yõ ko ǹ tùkè-ǹkɛ̃̂ ńtɛ̃nɛ?» 
Dĩ́-á képah noh sɛ̃́, à kyɩ kè yo sràsyíbí yõ̀tɛ̃̀ ǹgbɛ̃ tyɩ́ nɛ, tyítúkù nɛ̂ se à n tyah pɩ́nɔ́lɛ sɛ̃́ nɩ? Ǹgbǒ Wrome tãn gbõ̀ mɔ yõ-á dĩ́ mɛ̀nɛ dɛ́! 
Képah sɔkɔ, sràsyíbí yõ̀tɛ̃̀ ǹgbɛ̃ pɔ Pole piki yah nɛ, à yo pé syah, Wrome tãn gbõ̀ mɔ yõ se ńtyɩ̃́nɔ ǹnɛ nɩ? Pole mɛ nɛ àwe-á. 
Pole-á yo sɛ̃́, dĩ́ à syah nɛ, tɛ́ pɔ́ ǹgbɛ̃-á péwɔ tɛ̃̀ wɔ wilki, tɛ́ pɩ Wrome tãn gbõ̀ mɔ yõlɛ dɛ́! Pole mɛ nɛ, ǹtɛ péwɔ tɛ̃̀ wɔ-á dubi kélɛ. 
Képah tɛ̃̀nɛ, nónó-á kɩ sõ̀ yɩ̃nɛ pè Pole ko n pi nɛ à wɛkɩ, pè gbɛ mɔ ǹnɛ. Tɛ́ sràsyíbí yõ̀tɛ̃̀ ǹgbɛ̃-á kè wɛ pɩ̃ nɛ, Wrome tãn gbõ̀ mɔ yõ-á Polelɛ, tɛ́ pé mɛ pè kɔ̃ pè à pukubi, ké tíkí de ǹ sɔkɔ. 
Képah sɔkɔ, ké dĩndallɛ sràsyíbí yõ̀tɛ̃̀ ǹgbɛ̃ sõ̀ mɛ ké tyɩ́ nɛ pé pɩ̃ tɔ́tɔ́, tir nɛ̂ yõ-á Nsyifunɔ tɔkɔ Pole kɔ̃. Képah tɛ̃̀nɛ, à pè kɔ̃ pè kyɩ Pole wah mɔ, tɛ́ mɛ tukey tùkèpu tĩ̀nkì névye pól ye tuke, pépi nónó sɔkɔ-á sárká syínwìlkìpu yõ̀tãmnɛ, tɛ́ pè kɔ̃ pè Pole tɛ̃ pɔ pé nɩyṍ.
