Markɩ.
13.
Yesu-á n wil *Liyel wɔ̀kɔ̀ ǹgbɛ̃ kéntáhkɩ̀ sɔkɔ gbĩ́ nɛ̂nɛ, ǹ *tõ̀ sã́hpú sɔkɔ nɛ yɔ́ mɛ à syah nɛ, kwɔ́-ò dĩ́, à vɩ̀ ǹgbnɔ nɔ́pi mɔ dùkù tɛ̃̀nɛ n yah, dyêh pèplɔlɛ dɛ́! 
Yesu mɛ yo ǹ tyɩ́: «Á vɩ̀ ǹgbnɔ nɔ́pi mɔ́nɔ́ wɛ kɛ̀! Sé pól kɩ pɔ sɛkɩ. Dáhkɩ̀ núkú ńtɛ̃ náh n wũ n pi ké ní-ò yõ.» 
Képah náh, Yesu kyɩ kɔ̃ tɛ̃ Wólívyê sésân yɔ̃lɩ yõ, Liyel wɔ̀kɔ̀ ǹgbɛ̃ yèsreynɛ. Wǎh mɛ nɛ́npɔ́, Pyɛrɩ, ànɛ̂ Nsyahkɩ, ànɛ̂ Nsyɔ̃, ànɛ̂ Andrepi pɔ pé yahlɛ, tɛ́ à piki yah nɛ, 
gbĩ́ nɛ̂ se tyi nɔ́pi pól kɩ n pɩ n pi nɩ? Ńkɛ̃, à yo pé syah. Tyi nónó se kɩ sé pɩ gbĩ́nɛ pé wɛ̃kɩ nɩ? 
Képah sɔkɔ, Yesu tyah n yo pé tyɩ́: «Yé yé gblɔ̌ynɛ n kyɔmɩ, nɛ káh yé pilki. 
Névye náhnáh kɩ pɔ ńmɔ yĩn tɔkɔ n pi, tɛ́ kɩ nɛ, pépi-á *Kristalɛ. Pè kɩ névye náhnáh pilki. 
Yâh mâl tyɩ́lɛ n nohnɩ gbĩ́ nɛ̂nɛ, yé gbɔ̀plɔ ànɛ̂ fɩ́ɩ́ sɔkɔ, yé káh yɛ tíkí yé tɛ̃. Sépi ye yɩ̃nɛ sè pɩ. Ǹtɛ, képah náh tɔ̃̌nɩ̀ kèkõyṍ tyɛ́nɔ́lɛ. 
Kwil yɔ́ kɩ yuku kwil yɔ́lɛ n fwo, yõ̀tɛ̃̀ yɔ́ kɩ yuku yõ̀tɛ̃̀ yɔ́lɛ n fwo. Tɛ́, sétáh syahrɩ kɩ mɛ n pɩ tyahnɔ́ túkù sɔkɔ, ànɛ̂ kwɛkɩlɛ. Kè kɩ mɛ, á kɩ nɛ sê kúrkì tɛ̃ gbĩ́ yɔ́nɔ́ sɔ̃́. 
Képah ye, yépi tɛ̃̀ wɔlɛ, yé yé gblɔ̌ynɛ n kyɔmɩ. Tir yɔ́ náh, névye kɩ yélɛ n tɛ̃ n kyɩ n pɔ tukey tùkè-ǹtã̀n sɔkɔ. Pè kɩ yélɛ n ko *Liyel ní-ńtã̂n vɩ̀ sɔkɔ, tɛ́ kɩ yélɛ n tɛ̃ n kyɩ n pɔ yõ̀tãm ànɛ̂ yõ̀tãm ǹgbnɔ tyɩ́ ńmɔ yĩnnɛ nɛ, yé pɩ ńmɔ tàrfyɔ̀nɔ́lɛ pé yah sɔkɔ. 
Tɛ́ sépi-à pópó nɛ n pɩ, nɩ̀vɩ̀nɩ̀ wɛy sã́n ǹgbò yɩ̃nɛ ké nɛ mɛ yo kɔ̃ kèkõyṍ kwlo kwlo pól névye tyɩ́. 
Ǹtɛ, pàh yé tɛ̃ n yuku pé gbõ̀ dáhbɩ́-ńsah, yé káh tɔ̃̌nɩ̀ yé fɔkɔlɛ n sõ, yáh nɛ tyi nónó yo n pi. Tyi nónó-à de yé fɔkɔ légbĩ́nɛ, yé kɩ sépi yo. Tir yɔ́ náh, tyi nónó-á yé kɩ n yo n pi, sè náh n wil n pi yépi tyɩ́, sè n wil *Liyel Mírkí tyɩ́. 
Névye kɩ pé yṹpyé gbàhlɛ n dɛ n mɔ nɛ, pè pè ko mɔ, súbí tɔ kɩ n pɩ sɛ̃́ntɛ̃̀nɛ pé wɛ̃́npì tyɩ́. Wɛ̃́npì tɔ kɩ pé dúbípúlɛ n fwo, kɩ pélɛ n dah n mɔ kũ sɔkɔ, pé ku ku tɛ̃̀nɛ. 
Névye pól kɩ yé wɛ mɔ ńmɔ yĩnnɛ. Ǹtɛ, névi nɛ̂-à ǹ tah sah, à pɔ nɔ kɩ tyɛ gbĩ́ tyɩ́, tɛ̃̀ kɩ pwáhnmɔnɔ wɛ.» 
Képah sɔkɔ, Yesu tɔ̃ yɔ̃ sɔkɔ ǹ wɛynɛ - ǹtɛ, wɛy mɛ̀ kàrpu yɩ̃nɛ pè ké dùkù pɩ̃. À nɛ: «Yé kɩ yɩ́kɩ́nɔ́ pɩ́-ò képékè wɛ fɩ̃́ yɔ́, à náh sõ̀ yɩ̃nɛ tɛ̃̀nɔ́lɛ ńyãh sɔkɔ. Légbĩ́nɛ, nónó-à mɛ Yude, pè yɩ̃nɛ pè to de ymɔ sɔkɔ. 
Névi nɛ̂-à mɛ gbɛ̃́ntĩ̂n yõ, tɛ̃̀-à tiki, à káh nɛ, páh n de kwâhlɛ n tɔkɔ pé wɔ̀kɔ̀. 
Nɛ̂-à se mɛ swãh sɔkɔ, à káh nɛ, páh pé syɩ n yuku fwɔ̀mɔ̀-ǹgbɛ̃ tɔ́kɔ́-ńsah pé tĩ̂nnɛ. 
Tɛ́ kénɛ sèpĩ̀n kɩ pɩ tyípékèlɛ syɔ́ kûrn nɛ́tãn, ànɛ̂ yɔ̃kɩ nɛ́tãn tyɩ́. 
Képah ye, yé Liyellɛ n ni, kè káh pɩ ki sɔkɔ. 
Tir yɔ́ náh, kénɛ sèpĩ̀n kɩ pɩ yèvyãhrɩ ǹgbɛ̃nɛ, nɛ̂ kénkɔ̃ náh wɛ yah, Liyel-á kèkõyṍ mɔ sɔ̃́, à pɔ syɩ núkúnúkú tyɩ́. Ànɛ̂ sɛ̃́nkɔ̃ náh tɔ̃ wɛ n yah n pi. 
Ǹtɛ, Liyel ńtɛ̃-à mɛ ké yáhntɔkɔ-ńkɛ̃̂nɛ nɛ, páh kɩ wìlkìnɔ́ pɩ kénɛ sèpĩ̀n sɔkɔ, ǹnɩ̂ yɔ́ ńtɛ̃ min náh náh n wil n pi ké sɔkɔ. Ǹtɛ, wǎh nónó yah tɔkɔ pɩ ǹ tyɩ́lɛ, pépi yĩn ye à wìlkìnɔ́ pɩ kénɛ sèpĩ̀n sɔkɔ. 
Képah ye, névi-à yo yé tyɩ́ nɛ, yé yah, *Krista-á mɛ ásõ̂, képah náh pɩ, wǎh mɛ wrɔ́ mɛ̀, yé káh sɛ̃ tɛ̃̀ yõ. 
Tir yɔ́ náh, mɩ̀nyobe túkù kɩ n wil n pi, tɛ́ kɩ nɛ, pépi-á Kristalɛ. Túkù kɩ mó tɔ̃ wil nɛ, pépi-á *Liyel tɛ̃ntõ̀ syínyopulɛ. Pè kɩ gbǐl duke dukelɛ n pɩ, kɩ n yah n kɔ̃ Liyel névye yáhntɔkɔnɔ ńtɛ̃ pílkí-ńsahlɛ, pàh náh fɛ̃ n tɛ̃ n pi. 
Képah ye, yépi tɛ̃̀ wɔlɛ, yé yé gblɔ̌ynɛ n kyɔmɩ. Mé ye sé pól kwãh yo sah yé tyɩ́. 
Kénɛ sépĩ́n sɔkɔ, yèvyãhrɩ mɛ̀ sɔ́kɔ́ntɛnɔ náh, gbĩ kɩ pɩ fɛ̃́ntũ̂nɛ, ńwɔ̃́ náh tɔ̃ n piki n pi, 
ńwɔ̃́ képye kɩ wil yĩ̂nyõ kɩ syɩkɩlɩ, yĩ̂nyõ tǎhkɩ̀nɔ́ kɩ yike. 
Ké légbĩ́nɛ, yé kɩ *Névi Pi wɛ, à kɩ n pi ńyîrn wrɔ́ ǹ tǎhkɩ̀ ǹgbɛ̃ ànɛ̂ ǹ gblò sɔkɔ. 
À kɩ *yĩ̂nyõ tɛ̃̀ntõbe tɛkɩ mɔ kèkõyṍ yey yey pól sɔkɔ, pè kɩ ǹ névye yáhntɔkɔnɔ tuke mɔ, à tɔkɔ yõ mɛ̀ tyɩ́, á pɔ syɩ yõ mɛ̀ tyɩ́. 
Yé kè pɩ̃ kwɔ́nɔ́ nɛ̂-á sènsóké n kɔ̃: Ni-à mɛ n kɔ ké yɩ̃̀ngbã̀n sɔkɔ, kě mɩ pɩ, yé n pɩ̃ nɛ, ki de gbĩ́-á nɔ. 
Sɛ̃́nkɔ̃-ó ye: Yáh tyi nɔ́pi wɛ, sè n pɩ, yé pɩ̃ nɛ, *Névi Pi pɔ́nɔ́-á de tɛ, wǎh mɛ yúkɛy. 
Kègbɩ yõ, mé kélɛ n yo yé tyɩ́, kwɛy gbĩ́ névye pól náh n ku n tɛ n pi, nɛ́ tyi nɔ́pi pól náh pɩ. 
Yĩ̂nyõ ànɛ̂ sétáh kɩ pɔ sɔkɔ, ǹtɛ ńmɔ wɛlɔ náh n sɔkɔ n pi póllɛ. 
Tɛ́ ké lésõlɛ ànɛ̂ ké gbĩ́nɛ, ǹnɩ̂ yɔ́ ńtɛ̃ náh kè pɩ̃: *Yĩ̂nyõ tɛ̃̀ntõbe náh kè pɩ̃, Liyel Pi ńtɛ̃ tɔ náh kè pɩ̃. Sú núkúní ó ye kè pɩ̃. 
Yé yé gblɔ̌ynɛ n kyɔmɩ, yé mó nɛ n yah. Tir yɔ́ náh, yé náh ké pɔ gbĩ́ pɩ̃. 
Kè mɛ, á kɩ nɛ dĩ́ yɔ́ sɔ̃́ n sɔkɔ kɔ́lɔ́, tɛ́ ǹ fṍ dahbɩ ǹ tɛ̃̀ntõbe gbõ̀. À pé pól nɛ núkú núkú wɛ̃kɩ pé tã̀nnɛ, tɛ́ yo gbô syɩ́kɩ́-ò tyɩ́ nɛ, à káh nɛ n dɔ, à nɛ n yah. 
Képah ye pɩ yépi tɔ nɛ n yah, tir yɔ́ náh, yé náh gbóntɛ̃̀ pɔ sṍ pɩ̃: Kè kɩ pɩ ńvɩ̃́nɛ nɩ? Kè kɩ pɩ lénɩlɛ nɩ? Kè kɩ pɩ snǐ gbo gbĩ́nɛ nɩ? Kè kɩ pɩ séngbɔ̃́lɛ nɩ? 
Wàh n pi, à káh pɔ yé wɛ tɛ̃ dɔy sɔkɔ. 
Ǹtɛ, nɛ̂-á mé n yo yé tyɩ́, mé kélɛ n yo no pól tyɩ́: Yé nɛ mɛ n yah.»
