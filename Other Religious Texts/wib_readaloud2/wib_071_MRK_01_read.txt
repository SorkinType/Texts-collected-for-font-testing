Markɩ-á nɩ̀vɩ̀nɩ̀ tir nɛ̂ wãrkɩ Yesu tyɩ́ sɔkɔ.
1.
Liyel pi Yesu *Krista nɩ̀vɩ̀nɩ̀ wɛy sã́n tyàh-ǹsah ye yɔ̀. 
Sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ *Liyel tɛ̃ntõ̀ syínyo-ò *Yesayi sɛ́bɛ̃́ sɔkɔ nɛ: «Yah, ńmɔ Liyel kɩ ń tɛ̃ntõ̀ névi tɛkɩ mɔ á yahlɛ, à kɩ pɔ á wɛ̃̀kɩ̀ tɔ̃ sah á tyɩ́.» 
Ǹnɩ̂ yɔ́ yĩ́npɔ̃́ ye n dɔkɔ gbɛ̃́nnɩ sɔkɔ nɛ ńkɛ̃: «Yé Tɛ̃̀-ǹgbɛ̃ wɛ̃̀kɩ̀ tɔ̃ sah, yé kè pɩ tómm!» 
Kǎh wãrkɩ sah sɔ̃́, sɛ̃́ ye *Wèlnwìlkì-ò Nsyɔ̃ pɔ wil mɔ gbɛ̃́nnɩ sɔkɔ. À tyah n yo n kɔ̃ névye tyɩ́ nɛ, pè pé nnɔ vi mɔ, pè mó *wèlnwìlkìnɔ́ wɛ. Képah sɔkɔ, Liyel-á kɩ pé tyípêl yɔ̃ mɔ pè kɔ̃. 
Képah tɛ̃̀nɛ, Yude kwlo névye pól, ànɛ̂ tɔ̃ Yerusalɛmɩ névye pól sõ̀ mɛ n pɔ n pɔ ǹ tyɩ́, n pɔ pé tyípêl yo névye pól yĩ́ yah, tɛ́ ǎ pè wel n wilki Yurdɛ̃ yɔ sɔkɔ. 
Tɛ́, Nsyɔ̃ sõ̀ mɛ yã̀hmɩ́kɩ̀ yɛ̀ fwɔ̀mɔn mɔ, tɛ́ ǹ syɩ̃ tɛ̃ sêl kɔ̃lɛ. Ǹ yõkelɛ sõ̀ sègblo ànɛ̂ syɩ̃rɩ ye. 
À sõ̀ mɛ n yo tɔ́wû tyɩ́: «Névi nɛ̂-á kɩ n pɔ n pi ńmɔ náh, ǹmɔ ye gblò kal ńmɔlɛ. Ńmɔ tɛ̃̀ náh yɩ̃nɛ mé kɔ yĩn, ǹ gbáhkɩ̀ lékyêh sro dɛ. 
Ńmɔ ye yélɛ n wel n wilki ni sɔkɔ, ǹtɛ ǹmɔ kɩ pɔ yé wel n wilki n pi *Liyel Mírkí sɔkɔ.» 
Légbĩ́nɛ, Yesu wil Nasarɛtɩ kwil, Kalile sétáh yõ, à pɔ Nsyɔ̃ kɔ̃ à à *wel wilki Yurdɛ̃ yɔ sɔkɔ. 
Tɛ́, wǎh n wil ni sɔkɔ gbĩ́ nɛ̂nɛ, à yĩ̂nyõ wɛ kè n yɩkɩ n mɔ, tɛ́ *Liyel Mírkí tiki ǹ yõ, dàhyõ̀kèpil sɔ̃́. 
Légbĩ́nɛ, yĩ́npɔ̃́ yɔ́ n wil Liyel sɔkɔ nɛ ńkɛ̃: «Áwɔ ye ńmɔ nɩ̀dɛ̀ Pi núkúnínɛ. Á tyɩ́ nɔ yĩ́ĩ́ ń tyɩ́ kègbɩlɛ.» 
Ké nɛ́npɔ́ swɔ, Liyel Mírkí à tɛ̃ yuku gbɛ̃́nnɩ sɔkɔ. 
À sèpĩ̀n tíkíníní pɩ nɛ́npɔ́ sɔkɔ. Tɛ́ *sétã̀n yõ̀tɛ̃̀ pɔ ǹnɛ n tɔ̃ n mɔ. À sõ̀ mɛ gbɛ̃́nkyɔ nɩyṍ sɔkɔ, tɛ́ *yĩ̂nyõ tɛ̃̀ntõbe mɛ ǹ tɛ̃̀-ǹsahlɛ n yah. 
Képah náh, pè pɔ *Wèlnwìlkì-ò Nsyɔ̃ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. Ǹ déntɛnɔ náh wɔ̀kɔ̀npèkè, Yesu yuku Kalile sɔkɔ, à kyɩ tyah nɩ̀vɩ̀nɩ̀ wɛy sã́nnɛ n yo n kɔ̃, nɛ̂-á wil Liyel tyɩ́. 
À nɛ: «Gbĩ́ nɛ̂ tyɩ́-á sõ̀ yo sah, kè mɛ de tɛ. *Liyel tɔ̃́rɩ́ mɛ yĩni tɛ̃ tɛ. Yé yé nnɔ vi mɔ, yé mó sɛ̃ nɩ̀vɩ̀nɩ̀ wɛy sã́n yõ!» 
Wǎh sõ̀ mɛ n kɔ Kalile yɔ́-ǹgbɛ̃ gbɔ̀pɔ gbĩ́ nɛ̂nɛ, à yɛ̃́nkyɔ tɛ̃́pú nɛ́pĩ̂ nímí wɛ, Simo, ànɛ̂ ǹ gbɛ̃̀ndĩ́ Andrelɛ. Pè sõ̀ mɛ yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔlɛ n min yɛ̃́nkyɔlɛ n tɛ̃ yɔ sɔkɔ. 
À yo pé tyɩ́ nɛ: «Yé pɔ ńmɔ náh. Mé kɩ yé pɩ névye tɛ̃́púlɛ Liyel tyɩ́.» 
Nɛ́npɔ́ swɔ, pè núkú pé yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔ sah, tɛ́ ǹ sõ̀ tɛ̃. 
Wǎh tɔ̃ kɔ yuku yahlɛ kwéy, à kyɩ Sebede wɛ̃́npì wɛ, Nsyakɩ ànɛ̂ Nsyɔ̃nɛ. Pè sõ̀ mɛ pé krótókè sɔkɔ, mɛ yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔlɛ n tɔ̃ n sah. 
Ké nɛ́npɔ́ swɔ, à pépi tɔ ye yah. Pè núkú yal pé sú ànɛ̂ ǹ tõ̀npɩbɩ tnɔ̂ krótókè sɔkɔ, tɛ́ ǹ sõ̀ tɛ̃. 
Képah náh, wǎh nónó ye, à yuku pélɛ Kapɛrnayim kwil sɔkɔ. *Mɩ mɩ yòh sṍlɛ, à kyɩ de *Liyel ní-ńsah wɔ̀kɔ̀, à tyah névyelɛ n kwɔ. 
Nónó-á sõ̀ mɛ ǹ wɛynɛ n nohnɩ, pé gbã̀n du ǹ névye kwɔ́nɔ́ dùkùlɛ. Tir yɔ́ náh, kè náh sõ̀ mɛ núkúlɛ ànɛ̂ *Liyel ǹgbò wɛ̃̀kɩ̀ kwɔ́pú tyɩ́lɛ. Ǹmɔ kwɔ dùkù sõ̀ mɛ yõ̀tɔ̃rɩ́ kwɔ́nɔ́lɛ, képah ye. 
Ké légbĩ́nɛ, kè pɔ yɩ̃nɛ, dĩ́ yɔ́ sõ̀ mɛ Liyel ní-ńsah wɔ̀kɔ̀, *sétõlɛ ǹ sɔkɔ. À mɛ yuku gbo mɔ nɛ, 
kwâh nɛ̂ se à n yah n kɔ̃ pépi tyɩ́ nɩ, Nasarɛti Yesu? À se pɔ pépi yɩ́kɩ́-ńsah nɩ? Páh pɩ̃ nɛ, pi kɛ̃́-á ǹnɛ, ǹmɔ nɛ̂-á wil Liyel tyɩ́. 
Képah tɛ̃̀nɛ, Yesu dírí kõ sétõ mɛ̀ yõ, tɛ́ yo ǹ tyɩ́: «Á vyãh vike, ǹ tɛ́ n wil dĩ́ mɛ̀ sɔkɔ!» 
Képah sɔkɔ, sétõ dĩ́ mɛ̀ yike kègbɩlɛ, tɛ́ de syi náhnáh, tɛ́ wil ǹ sɔkɔ. 
Képah névye gbã̀n du. Pè tyah wɛ̃nɛ n piki n yah nɛ, kwâh nɛ̂ se sɛ̃́ nɩ? Kwɔ́nɔ́ késã́n yɔ́lɛ à n pɩ yõ̀tɔ̃rɩ́ sɔkɔ. À fɛ̃ n wɛkɩ sétã̀n tɔ yõ, tɛ́ sè n sah ǹ tyɩ́ yõ. 
Kénɛ nɩ́ yõ, Yesu yĩn gbílkínɔ́ tɔ̃ mɔ Kalile kwlo pól sɔkɔ. 
Pǎh yal *Liyel ní-ńsah wɔ̀kɔ̀, pè núkú yɔ̃ sɔkɔ de Simo ànɛ̂ Andrepi gbô. Nsyakɩ ànɛ̂ Nsyɔ̃ tɔ sõ̀ mɛ pélɛ. 
Tɛ́ Simo dɛ́ysê sõ̀ mɛ sɛ̃́nɔ́lɛ wílvyã́hnɩ̀nɛ. Nɛ́npɔ́ swɔ, pè mɛ kè yo Yesu syah. 
Wǎh kè noh sɛ̃́, à nɔ de pi mɛ̀ tnɔ̂, tɛ́ ǹ gbõ̀ tɛ̃ à yilki sah. Ǹ wílvyã́hnɩ̀ núkú tyɛ, à mɛ yuku tyah pĩ́nnɛ n pɩ pé tyɩ́. 
Gbĩ́yĩ́-á de, névye pé yámpúnɔ pól twah pɔ Yesu tyɩ́, ànɛ̂ *sétã̀n-á mɛ dénɔ́lɛ nónó sɔkɔ. 
Kwil névye pól pɔ tuke mɔ kénɛ gbô yah sɔkɔ. 
À mɛ yámpú duke duke náhnáh dɛ mɔ, tɛ́ sétã̀n náhnáh yĩni wilki, sǎh mɛ de nónó sɔkɔ. Sè sõ̀ ǹnɛ n pnɛ, nɛ dùkù nɛ̂-á ǹnɛ. Képah ye, à yahle nɛ, sè káh nɛ n wɛkɩ. 
Séngbɔ̃́ púsɔ́, tyah syì syì sɔkɔ, Yesu yuku wil wɔ̀kɔ̀. À yuku gbɛ̃́nnɩ sɔkɔ, à kyɩ tyah Liyellɛ n ni. 
Képah sɔkɔ, Simopi yuku tyah ǹnɛ n yah n kɔ̃. 
Pǎh kyɩ à wɛ, pè à syah nɛ, névye pól-á mɛ ǹnɛ n yah n kɔ̃. 
À mɛ yo pé tyɩ́: «Á ye yuku gbɔ̀plɔ kwlo sɔkɔ. Mé yɩ̃nɛ mé nɩ̀vɩ̀nɩ̀ wɛy sã́n yo nɛ́npɔ́ tɔ sɔkɔ. Ń pɔ́nɔ́ yõ ye képahlɛ.» 
Képah sɔkɔ, à kyɩ Kalile tã̀ntã̀n pól sɔkɔ, mɛ Liyel wɛynɛ n yo n kɔ̃ pé *Liyel ní-ńtã̂n vɩ̀ sɔkɔ, tɛ́ mɛ *sétã̀nnɛ n yĩni n wilki névye sɔkɔ. 
Sṍ yɔ́lɛ, *tótɛ̃ yɔ́ nɔ de Yesu tnɔ̂, à gbètukunɔ di ǹ yah, tɛ́ à ni nɛ, wàh mɛ ké tyɩ́, wǎh kɩ fɛ̃ pé dɛ mɔ. 
Wǎh yo sɛ̃́, ǹ yãm Yesu tɛ̃. À mɛ ǹ gbõ̀ tɔ̃ ǹ yõ, à tũ ǹ tyɩ́, tɛ́ nɛ: «Mé mɛ ké tyɩ́, á kɩ dɛ!» 
Nɛ́npɔ́ swɔ, *dú núkú à fĩn mɔ, à mɛ dɛ. 
Képah sɔkɔ, Yesu núkú à tɛkɩ mɔ, tɛ́ kè gbah yo ǹ tyɩ́ nɛ, 
à noh yɔ̀, à káh kè yãh nɛ tyɩ́ dɛ́! À kyɩ ǹ gblɔ̌ynɛ *sárká wìlkìpu yõ̀tɛ̃̀ wɛ̃kɩ. Ńkɛ̃, *Moyisi-á sõ̀ *sárká nɛ̂ tyɩ́ yo, tótãm dɛ́nmɔnɔ tyɩ́ sɔkɔ, à mó képah wilki, nɛ̂-á kɩ no pól wɛ̃kɩ n pi nɛ, wǎh dɛ. 
Ǹtɛ, dĩ́-á yal ǹ tnɔ̂, tir nɛ̂-á pɩ ǹ tyɩ́, à kyɩ tyah kélɛ n yãh n kɔ̃ no pól tyɩ́. Kǎh pɩ sɛ̃́, Yesu náh tɔ̃ sõ̀ fɛ̃ ǹ gblɔ̌y wɛ̃kɩ n pi kwil yɔ́ ńtɛ̃ sɔkɔ. À sõ̀ mɛ gbɛ mɔ tɛ̃ kwlolɛ, kégbɔ́ sɔkɔ, tɛ́ névye n wil tã̀ntã̀n pól sɔkɔ, à n pɔ n pɔ ǹ tyɩ́.
