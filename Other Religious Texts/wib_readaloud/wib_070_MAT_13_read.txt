Matye.
13.
Lésõ mɛ̀ ólɛ, Yesu wil gbô sɔkɔ à kyɩ kɔ̃ tɛ̃ yɔ́-ǹgbɛ̃ gbɔ̀pɔ. 
Tɔ́wû ǹgbɛ̃ yɔ́ mɛ pɔ à mɔ gbɩ̃. Pé tnɔ́nkàhnɔ́ tɛ̃̀ tyɩ́, à de kɔ̃ tɛ̃ krótókè yɔ́ sɔkɔ. Tɔ́wû pól sõ̀ mɛ yĩ́nntɛ̃̀nɔ́lɛ yɔ́ngbalɩ sɔkɔ. 
À sõ̀ mɛ tyi náhnáhlɛ n yãh pé tyɩ́ sèmìrn sɔkɔ, mɛ n yo pé tyɩ́ nɛ: «Sõ yɔ́lɛ, dĩ́ yɔ́ lésõ kyɩ sèmukɔ mír-ńsah ǹ swãh sɔkɔ. 
Wǎh mɛ mírnɔ́ sɔkɔ, yísyɔ́ syɩkɩlɩ wɛ̃̀gbalɩ sɔkɔ, kèpye mɛ pɔ sépi yɔ. 
Yísyɔ́ mɛ syɩkɩlɩ yen kwéy tɛ̃̀-ǹsah légbɩ̃́ yõ. Tɛ́ kǎh pɩ yen gbɩ náh sõ̀ mɛ nɛ́npɔ́ sɛ̃́, sè núkú fi súú. 
Tɛ́ sè náh tɛ̃ wɛ yã̀hnɩ̀ gbɩ mɔ. Kǎh pɩ sɛ̃́, gbĩ-á pɔ vyãh sè tɛ̃, sè kɔkɔ. 
Yísyɔ́ mɛ syɩkɩlɩ de pã̀n nɩyṍ, pã̀n mɛ yuku fyãh sé yõ. 
Ǹtɛ yísyɔ́ mɛ syɩkɩlɩ sétáh pèpɛy sɔkɔ à pɔ ywɔ̃ pɩ. Túkù pyenɔ lèkwɛ̌kwɛ pɩ, túkù tikitɔ̃nɩ́tɔ̃nɩ pɩ, túkù yõgbãmgbãm pɩ. 
Nɛ̂-à kɩ mɛ kè noh, tɛ̃̀ kè tɛ̃ yĩn.» 
Képah sɔkɔ, ǹ tõ̀ sã́hpú mɛ nɔ de ǹ tnɔ̂ tɛ́ à piki yah nɛ, kwâh nɛ̂ se pɩ tɛ́ à n yãh no nɔ́pi tyɩ́ sèmìrn sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Liyel tɔ̃́rɩ́ tyi nónó-á sõ̀ mɛ wɔ̃̀bɩ̀nsàhnɔ́lɛ, sé pnɛ̀nɔ́ ye yépi kɔ̃. Ǹtɛ kè náh pépi wɔ kɔ̃. 
Névi nɛ̂ tyɩ́ kwâh-á mɛ, dòhnɩ̀nɔ́ kɩ n pɩ ǹmɔ tyɩ́ yõ yrɔ̌yrɔ̌. Tɛ́ névi nɛ̂ tyɩ́ kwâh náh mɛ, kwéy nɛ̂ ńtɛ̃-á mɛ ǹ tyɩ́, kè kɩ tɛ̃ syi ǹ tyɩ́. 
No nɔ́pi mɛ n yah, tɛ́ pè náh mɛ kwâh yɔ́lɛ n wɛ, képah n wɛ̃kɩ nɛ pè náh mɛ ké tyɩ́ yahlɛ n wɛ. Pè mɛ pé nínɛ n kwɔ, tɛ́ pè náh mɛ kwâh yɔ́lɛ n nohnɩ, képah n wɛ̃kɩ nɛ pè náh mɛ ké tyɩ́ yahlɛ n wɛ. Képah ye ń kɔ̃ mé n yãh pé tyɩ́ sèmìrn sɔkɔ. 
Liyel tɛ̃ntõ̀ syínyo-ò Yesayi-á lésõ nɛ̂ yo sah, képah ye n nɔ pélɛ yɔ̀. À lésõ nɛ ńkɛ̃́: ‹Yé kɩ yé nínɛ n kwɔ n nohnɩ, tɛ́ yé náh ké tyɩ́ yah wɛ n pi. Yé kɩ n yah, tɛ́ yé náh kwâh yɔ́ wɛ n pi, képah n wɛ̃kɩ nɛ yé náh mɛ ké tyɩ́ yahlɛ n wɛ. 
Nikisekɔ́kɔ́ névye ye no nɔ́pilɛ. Pè mɛ pé ní tah tũ, tɛ́ pé yɩ̃́ vike sah nɛ sè káh kwâh yɔ́ ńtɛ̃ wɛ, pé ní káh tir yɔ́ ńtɛ̃ noh pé kɔ̃ pé káh fɛ̃ ké tyɩ́ yah wɛ, pé mó vi mɔ Liyel wɛ̃̀kɩ̀ yõ ń kɔ̃ mé pè wɛ n dɛ n mɔ.› 
Wɛlɔ nɔ́pi yóntɛnɔ náh, Yesu tɔ̃ yo pé tyɩ́ nɛ: «Ǹtɛ yépi tɛ̃̀ wɔ tyɩ́ nɔ, yé yɩ̃́ mɛ n yah, yé ní mɛ n nohnɩ. 
Kégbɩ yõ mé kélɛ n yo yé tyɩ́, yáh tyi nónó wɛ, Liyel tɛ̃ntõ̀ syínyopu ànɛ̂ névye nónó-á nɛ́gblɔ́lɛ Liyel yah, pé náhnáh sõ̀ mɛ ké tyɩ́ nɛ pé sè wɛ, tɛ́ pè náh sè wɛ. Yáh tyi nónó noh, pè sõ̀ mɛ ké tyɩ́ nɛ pé sé tɔ noh, tɛ́ pè náh sè wɛ noh.» 
«Sèmukɔ mír-ò sèmìrkì yõ-á nɛ̂nɛ, yé yé ní sah kè noh: 
Névye nónó-à Liyel tɔ̃́rɩ́ tyɩ́ yónɔ́ noh tɛ́ pè náh ké dùkù pɩ̃, sétã̀n yõ̀tɛ̃̀ n pɔ wɛy mɛ̀ wilki pé fɔkɔnɩ sɔkɔ. Pè mɛ wɛ̃̀gbalɩ tyɩ́ kɔ̃lɛ ńyãh sɔkɔ-á sèmukɔ syɩkɩlɩ. 
Túkù mɛ légbɩ̃́ yõ tyɩ́ kɔ̃lɛ, ńyãh sɔkɔ-á sèmukɔ mir mɔ. Pépi-à wɛy mɛ̀ noh, nɛ́npɔ́ swɔ pě kè fɛ̃ n syi nɩ̀vɩ̀nɩ̀ sɔkɔ. 
Tɛ́ pè náh n yɛ kè yã̀hnɩ̀nɛ n mɔ pé sɔkɔ. Pě kè fɛ̃ n syi nɛ́npɔ́ swɔ ó sɔkɔ. Tɛ́ yèvyãhrɩ, képah náh pɩ fṍhnɔ́-à núkú pè yĩni tɛ̃ kénɛ wɛy yĩnnɛ, pě pé náh vi kè kɔ̃ súú. 
Túkù mɛ pã̀n tɛ̃̀-ǹsah tyɩ́ kɔ̃lɛ, pàh wɛy mɛ̀ noh, kèkõyṍ tyi fɔkɔ sõnɔ ànɛ̂ tahrɩ dékè n pɔ ké yah kõ, kè náh n tɛ̃ n wɛ tirlɛ n pɛkɩ pé tyɩ́. 
Túkù wɔ mó mɛ sétáh pèpɛy tyɩ́ kɔ̃lɛ, pépi-à wɛy mɛ̀ noh, pě ké dùkù pɩ̃, yõre n wɛ ké tyɩ́ sèmukɔ tyɩ́ kɔ̃lɛ, nónó-á pyenɔ lèkwɛ̌kwɛ pɩ, ànɛ̂ nónó-á tikitɔ̃nɩ́tɔ̃nɩ pɩ, ànɛ̂ nónó-á yõgbãmgbãm pɩ.» 
Yesu tɔ̃ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ ńkɛ̃́: «Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Dĩ́ yɔ́ lésõ sèmukɔ duku pèpɛy mir mɔ ǹ swãh sɔkɔ. 
Tɛ́ névye-á pɔ dɔ gbĩ́ nɛ̂nɛ, ǹ sépɛ kyɩ fílpyé mir dah mɔ kɔ̃ ké sɔkɔ nɛ̂-á mɛ ké kètɔ̀lɛ, tɛ́ ǹ náh vi sɔkɔ. 
Kǎh pɔ kyãh tɛ́ n dubi gbĩ́ nɛ̂nɛ, fílkì tɔ ké yõ wilki mɔ kwèy. 
Sètɛ̃̀ dĩ́ tõ̀npɩbɩ-á kè kyɔmɩ wɛ sɛ̃́, pè mɛ kyɩ kè yo pé yõ̀tɛ̃̀ dĩ́ tyɩ́ nɛ ńkɛ̃́, pé yõ̀tɛ̃̀, sèmukɔ duku pèpɛy-á se sõ̀ à mir mɔ ǹ swãh sɔkɔ; tɛ́ kè se pɩ sɔ̃́ tɛ́ fílkì tɔ pɔ n wɛ ké sɔkɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ, sépɛ yɔ́ tõ̀ vyãh-á képahlɛ. Pè mɛ nɛ, à se mɛ ké tyɩ́ nɛ pé kyɩ sè viki nɩ? 
À mɛ nɛ ɔ̃́ɔ̃-á. Ńkɛ̃́ pàh kɩ nɛ páh sélɛ n viki, sèmukɔ tɔ-á kɩ syah n kwrɔ n mɔ n viki sélɛ. 
Pè yɛ sé yõ sè nɛ n kyãh wɛ̃ tyɩ́ kè syi pɔ n nɔ sèmukɔ tɔkɔ gbĩ́ tyɩ́. Tɛ́ sèmukɔ tɔkɔ gbĩ́-à kɩ pɔ nɔ, páh kɩ yo sèmukɔ tɔ́kɔ́pú tyɩ́ nɛ pè fílkì ǹgbò kõ, pè mó sè pukubi pbɔ pbɔlɛ pè kyɩ n sukɛ. Tɛ́ sèmukɔ wɔlɛ, pè képah tɔkɔ n mɔ pé dúkúrí sɔkɔ.» 
Yesu tɔ̃ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: «Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Dĩ́ yɔ́ lésõ mútárdɩ̀ pi tɔkɔ duku mɔ ǹ swãh sɔkɔ. 
Képah yilɔ kal yípyé póllɛ. Tɛ́ kàh fi, kè n pɩ gblò kal sèmukɔ duke póllɛ. Kè n pɩ séswɔ̂ sɔ̃́, kèpye n pɔ syɛ̀ pɩ ké yɩ̃̀ngbã̀n tyɩ́.» 
Képah náh Yesu tɔ̃ sèmìrkì yɔ́ yãh pé tyɩ́ nɛ: «Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Sê yɔ́ lésõ kɔ́kɔ́nɔ́ tɔkɔ mɔ tɔkɔ yɩ̃́ tɔ̃́nɔ́ mífã́hkɩ̀ sɔkɔ, ké pól kɔ̃ kè dɔkɔ.» 
Yesu sõ̀ mɛ tyi nɔ́pi póllɛ n yãh tɔ́wû tyɩ́ sèmìrn ó sɔkɔ. À náh sõ̀ mɛ tir yɔ́ ńtɛ̃nɛ n yo pé tyɩ́ nɛ́ sèmìrn sɔkɔ náh. 
À képah pɩ Liyel tɛ̃ntõ̀ syínyo-ò tir yónsàhnɔ́ kɔ̃ kè wil. À lésõ nɛ ńkɛ̃́: «Mé kɩ tyilɛ n yo sèmìrn sɔkɔ. Tyi nónó-á sõ̀ mɛ wɔ̃̀bɩ̀nsàhnɔ́lɛ kèkõyṍ mɔ gbĩ́ dùkùlɛ, mé kɩ névye níkí yɩkɩ sélɛ.» 
Képah tɛ̃̀nɛ, Yesu yal tɔ́wû tnɔ̂ à sɔkɔ gbô. Ǹ tõ̀ sã́hpú mɛ nɔ de ǹ tnɔ̂ tɛ́ yo ǹ tyɩ́ nɛ, swãh sɔkɔ fílkì sèmìrkì nɛ̂-á sõ̀ à yãh, à ké yõ yo pé syah. 
À mɛ yo pé tyɩ́ nɛ: «Sèmukɔ duku pèpɛy mírnmɔ-òlɛ, Névi Pídĩ́ ye. 
Swãhlɛ, kèkõyṍ ye. Sèmukɔ duku pèpɛynɛ, Liyel yõ̀tɔ̃rɩ́ sɔkɔ wɛ̃́npì ye. Fílkìlɛ, sétã̀n yõ̀tɛ̃̀ náh névye ye. 
Sépɛ nɛ̂-á mir dah mɔ fílpyélɛ, sétã̀n yõ̀tɛ̃̀ ye. Sèmukɔ tɔ́kɔ́nɔ́lɛ, kèkõyṍ tyɛ́nɔ́ ye. Sèmukɔ tɔ́kɔ́púlɛ, yĩ̂nyõ tõ̀npɩbɩ ye. 
Sɔ̃́-á pě fílkì viki n sukɛ nyàh sɔkɔ, sɛ̃́nkɔ̃ ye kɩ n pɩ n pi kèkõyṍ tyɛ́nɔ́ tɔ sɔkɔ. 
Névye nónó pól-á névye yõlɛ n yɩkɩ n mɔ pèkè pɩ́nɔ́ sɔkɔ, ànɛ̂ nónó pól-á tyítúkúnì pɩ́púlɛ, Névi Pídĩ́ kɩ ǹ yĩ̂nyõ tõ̀npɩbɩ tɛkɩ n mɔ n pi, pè kɩ pɔ pè wilki ǹ tɔ̃́rɩ́ tyah sɔkɔ. 
Tɛ́ kɩ pè min dah mɔ sĩnki nyàh sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ. 
Ǹtɛ névye nónó-á mɛ nɛ́gblɔ́lɛ Liyel yah, pépi wɔ kɩ n tɛ̃ pé Sú Liyel tɔ̃́rɩ́ sɔkɔ, gbĩ tyɩ́ kɔ̃lɛ. Nɛ̂-à kɩ mɛ kè noh, tɛ̃̀ kè tɛ̃ yĩn.» 
Képah náh, Yesu tɔ̃ yo pé tyɩ́ nɛ: «Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Dĩ́ yɔ́ lésõ kyɩ wil mɔ tahrɩ wɔ̃̀bɩ̀nsàhnɔ́ yɔ́ yõ swãh yɔ́ sɔkɔ, tɛ́ kè tɔkɔ tɔ̃ wɔ̃bɩ. Ké wɛ́nɔ́ nɩ̀vɩ̀nɩ̀ tɛ̃̀ tyɩ́, à sɔkɔ à kyɩ ǹ gbã̀n yî pól yãm à pɔ swãh mɛ̀ dwe. 
Yé yah, Liyel tɔ̃́rɩ́-á kɩ tɔ̃ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Gbɔ̀ yɩ̃́-ò yɔ́ lésõ mɛ, nɛ̂-á yî kètǎrnɔ́ yísyɔ́lɛ n yah n kɔ̃, pè n ye pɛ́rlɩ̀nɔ́lɛ. 
Wǎh pɔ ké dɛ́nkɔ́kɔ̀ yɔ́ wɛ, à kyɩ ǹ gbã̀n yî pól yãm à pɔ kè dwe.» 
«Yé yah, Liyel tɔ̃́rɩ́-á kɩ fɛ̃ tɔkɔ sãh sèmìrkì nɛ̂nɛ, kè wɛ yɔ̀: Nósyɔ́ lésõ yɛ̃́nkyɔ tɛ̃ tɛ̃ mnɔ min dah mɔ yɔ sɔkɔ, à yɛ̃́nkyɔ duke pól tɛ̃. 
Kǎh yuku, pè kè yɔrɩ wilki pɔ kènkwã̀h yõ, tɛ́ kɔ̃ tɛ̃ à yɛ̃́nkyɔ gblɔ wah mɔ dú sɔkɔ, tɛ́ sé tɛ́lɔ́ milki. 
Sɛ̃́ ye kɩ n pɩ n pi kèkõyṍ tyɛ́nɔ́ tɔ sɔkɔ. Yĩ̂nyõ tõ̀npɩbɩ kɩ pɔ nɛ́túkúnì wah wilki nɛ́gblɔ́ sɔkɔ, 
kɩ kyɩ pè min dah mɔ sĩnki nyàh sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ.» 
Képah náh Yesu tɔ̃ yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Yé se tyi nɔ́pi pól noh wah?» Pè mɛ nɛ àwe-á. 
Képah sɔkɔ, à mɛ yo pé tyɩ́ nɛ: «Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú nónó-á sõ̀ kwɔ́nɔ́ wɛ sɔkɔ Liyel tyi kõ̀nsàhnɔ́ tyɩ́ yõ ǹgbòlɛ, tɛ́ tɔ̃ pɔ kwɔ́nɔ́ wɛ Liyel tɔ̃́rɩ́ tyɩ́ yõ núkúnúkú, tɛ́ névyelɛ n kwɔ sélɛ, pépi mɛ kéte tɛ̃̀ yɔ́ sɔ̃́, nɛ̂-á yídî sõ̀ mɛ ǹ tyɩ́ ǹ dúkúrí sɔkɔ, tɛ́ à tɔ̃ pɔ snɔ́ wɛ dohnɩ sé yõ, tɛ́ mɛ sélɛ névyelɛ n kɔ̃.» 
Tɛ́ Yesu-á sèmìrn nɔ́pi yãh tɛ pé tyɩ́ gbĩ́ nɛ̂nɛ, à yal nɛ́npɔ́ 
tɛ́ sɔkɔ ǹ kwil sɔkɔ. Wǎh kyɩ nɛ́npɔ́, à kyɩ kwɔ́nɔ́lɛ n pɩ pé Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ, fɔ́ɔ́ no pól gbã̀n du ǹ tyɩ́lɛ. Pè sõ̀ mɛ yo tũ nɛ, à se wɛ ńyãh lékã́m ànɛ̂ tɛ̃́nwɛnɔ mɛ̀nɛ, à fɛ̃ gbǐl nɔ́pilɛ n pɩ nɩ? 
Ǹmɔ kó sésár sɛ́-ò pídĩ́nɛ nɩ? Ǹ yṹ yĩn kó Marilɛ, tɛ́ ǹ gbɛ̃̀ndĩ́pi mɛ Nsyakɩ, ànɛ̂ Nsyosɛfu, ànɛ̂ Simo, ànɛ̂ Nsyudelɛ nɩ? 
Tɛ́ ǹ dósyɔ́ pól kó mɛ pépilɛ ásõ̂ yɔ̀ nɩ? Ǹtɛ ńyãh tɛ̃̀ ó se ǹmɔ wɛ lékã́m ànɛ̂ tɛ̃́nwɛnɔ mɛ̀nɛ nɩ? 
Képah sõnɔ sõ̀ mɛ pé yõ yɩkɩ pè kɔ̃ pè tɛ̃ sɛ̃ kwlɔ Yesu yõ. Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ: «Liyel tɛ̃ntõ̀ syínyo-ò kwil ànɛ̂ ǹ kéte sɔkɔ ó ye pè náh ǹnɛ n mɔ n kõ Liyel tɛ̃ntõ̀ syínyo-òlɛ.» 
Pǎh yahle sɛ̃́nɔ́lɛ Yesu yõ sɛ̃́, à náh mó gbǐlnɔ́ náhnáh pɩ nɛ́npɔ́.
