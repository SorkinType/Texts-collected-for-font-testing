Matye.
7.
Yé káh nolɛ n tuke. Képah sɔkɔ, Liyel náh yé tɔ tuke n pi. 
Yé náh wɛ di, yáh no tɛ́lɔ́lɛ n tuke sɔ̃́, sɛ̃́ ye Liyel kɩ n tuke n pi yé tɔlɛ. Yáh no tɛ́lɔ́ tyɩ́lɛ n sãh dú nɛ̂nɛ, yé tɔ tyɩ́ kɩ n sãh n pi kénɛ dú ólɛ. 
Á pɩ sɔ̃́ tɛ́ ǹdan wɛ á yṹnpi yĩ́ sɔkɔ, tɛ́ dàhnpɛlɛ áwɔ yĩ́ sɔkɔ, á náh mɛ képah wɔ pɩ̃? 
Á kɩ pɩ sɔ̃́, tɛ́ kɩ yo á yṹnpi tyɩ́ nɛ, à yɛ á ǹdan wilki à kɔ̃ ǹ yĩ́ sɔkɔ, tɛ́ dàhnpɛ mó tɔ̃ mɛ áwɔ yĩ́ sɔkɔ? 
Fɔ̀kɔ̀pǐl névi. Á yĩ́ sɔkɔ dàhnpɛ ǹgbò wilki. Képah-à pɩ, á kɩ fɛ̃ n yah pallɛ kɩ á yṹnpi yĩ́ sɔkɔ ǹdan wilki. 
Yî nónó-á mɛ Liyel tyɩ́ yõ, yé káh sélɛ névyelɛ n kɔ̃ nónó-á mɛ pãhlɔ tyɩ́ kɔ̃lɛ, pè kɔ̃ pè káh n pɔ pé yey vi yé tyɩ́ képékèyɔ̂. Yé káh mó tɔ̃ yî kètǎrnɔlɛ névyelɛ n kɔ nónó-á mɛ slo tyɩ́ kɔ̃lɛ, pè kɔ̃ pè káh n pɔ n suki n sɔkɔ sé yõ. 
Yé nɛ n ni, yé kɩ n wɛ n syi. Yé nɛ n yah n kɔ̃, yé kɩ n wɛ. Yé wùkahlɛ n mɩ, kè kɩ n yɩkɩ n mɔ yélɛ n kɔ̃. 
Yé náh wɛ di, nɛ̂-à ni Liyel tyɩ́, tɛ̃̀ n wɛ n syi. Nɛ̂-à yah kɔ̃, tɛ̃̀ n wɛ. Nɛ̂-à wùkah mɩ, kè n yɩkɩ n mɔ tɛ̃̀ kɔ̃. 
Nɛ̂ tɛ̃̀ mɛ yépi sɔkɔ, ǹ pídĩ́-à búrn ni ǹ tyɩ́ à kɩ dáhkɩ̀lɛ à kɔ̃? 
Képah náh pɩ, wàh yɛ̃́nkɔ̂ ni ǹ tyɩ́, à kɩ wahynɛ à kɔ̃? 
Yépi nónó-á nɛ́pêllɛ, yé yé wɛ̃́npì kɔ̃́nɔ́ yah wɛ yípéplɔlɛ. Tɔ̀tɔ̀ náh mɛ ké sɔkɔ, yé Sú Liyel nɛ̂-á mɛ yĩ̂nyõ sɔkɔ, ǹmɔ kɩ yîlɛ sé nípú kɔ̃ nónó-á plɛ kal yépi tyɩ́lɛ dò. 
Yáh mɛ ké tyɩ́ nɛ no tyi nónó póllɛ n pɩ yé tyɩ́, yépi tɔ sɛ̃́nknɔlɛ n pɩ pé tyɩ́. Sɛ̃́ ye tyi kõ̀nsàhnɔ́ ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu sɛ́bɛ́ynɔ́ yo kélɛ. 
Yé nɛ n de dé-ńsah fɩ̃́ɩ̃lɛ. Yé náh wɛ di, wɛ̃̀kɩ̀ nɛ̂-á n yuku tyɩ́ yɩ́kɩ́nɔ́ tɛ̃̀-ǹsah, ké kɔ́lɔ́ mɛ fàhfàh, ké dé-ńsah mɛ wòo. Képah sɔkɔ dépú mɛ dò. 
Ǹtɛ wɛ̃̀kɩ̀ nɛ̂-á n yuku tyɛ́-ńkɛ̃̂ min tɛ̃̀-ǹsah, ké kɔ́lɔ́ mɛ táhnɔ́lɛ, ké dé-ńsah mɛ fɩ̃́ɩ̃. Ké wɛ́ndepu náh mɛ fwɔ́ɔ. 
Yé yé gblɔ̌ynɛ n kyɔmɩ Liyel yĩn yõ mɩ̀nyobelɛ. Pàh n pi yé tyɩ́, pě pé gblɔ̌y pɩ mámm képyɛ́nɩ̀ tyɩ́ kɔ̃lɛ, tɛ́ ké kègbɩ tɛ̃̀ yõ sɔkɔ, pè mɛ pékénɔ́lɛ gbɛ̃́npãhlɔ sɔ̃́. 
Yé kɩ pè pɩ̃ n pi pé tyípɩ́nɔ́ sɔkɔ. Yɛ̀rɛsɛ̃̂ náh fɛ̃ n wɛ n gbɛ kahrɩ sɔkɔ. Sènsóké pye tɔ náh mó fɛ̃ n wɛ n gbɛ gbɩ́yɛ̃́kɩ̀ sɔkɔ. 
Sésân pèplɔ pól n syi pye pèplɔlɛ. Tɛ́ séswɔ́ntùkù wɔ mó n syi pye túkúnìnɛ. 
Séswɔ̂ pèpɛy náh fɛ̃ n syi pye túkúnìnɛ. Séswɔ́ntùkù tɔ náh mó fɛ̃ n syi pye pèplɔlɛ. 
Séswɔ̂ nɛ̂-à n syí-ńkɛ̃̂nɛ pye pèplɔlɛ, kè n kãm ǹ min n dah n mɔ nyàh sɔkɔ. 
Sɛ̃́ ó ye yé kɩ n pɩ̃ n pi Liyel yĩn yõ mɩ̀nyobelɛ pé tyípɩ́nɔ́ sɔkɔ. 
Kè náh pɩ nɛ nónó pól-á ńmɔlɛ n ye Tɛ̃̀ ǹgbɛ̃, TƐ̃̀ ǸGBƐ̃NƐ pé pól sɔ̃́ kɩ Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi dɛ́. Ń Sú Liyel nɛ̂-á mɛ yĩ̂nyõ, ǹmɔ-á mɛ nɛ̂ tyɩ́, képah pɩ́pú ó ye kɩ yõ ké nɔlɩlɛ. 
Liyel tukey tuke sõlɛ, névye náhnáh kɩ n yo n pi ń tyɩ́ nɛ, Tɛ̃̀ ǹgbɛ̃, Tɛ̃̀ ǹgbɛ̃, ńmɔ yĩn yõ kó lésõ pé n syi n yo Liyel tɛ̃ntõnènɛ nɩ? Ńmɔ yĩn yõ kó lésõ pè n nɛ n yĩni n wilki sétã̀nnɛ névye sɔkɔ nɩ? Ńmɔ yĩn yõ kó lésõ pè n nɛ n pɩ gbǐlnɔ́ náhnáhlɛ nɩ? 
Ǹtɛ mé kɩ yo pé tyɩ́ nɛ, mé náh pélɛ n pnɛ póllɛ. Pè gbɛ mɔ ńnɛ, Liyel tyi kõ̀nsàhnɔ́ dêl pɩ́pú nɔ́pi. 
Képah ye, névi nɛ̂-á ń wɛlɔ nɔ́pi noh, tɛ́ mɛ sé tyíyónɔ́lɛ n pɩ, tɛ̃̀ mɛ á kɩ nɛ lékã́m névi yɔ́ sɔ̃́, nɛ̂-á ǹ wɔ̀kɔ̀ mɔ légbɩ̃́ yõ. 
Pũ̀ mɛ pɔ pɩ, yrɛy yuku à kwɔkɩ kyɩ syɩ kénɛ wɔ̀kɔ̀ tyɩ́, pṹnyã́hkɩ́ tɔ mɛ kõ kyɩ mɩ ké sɔkɔ, ǹtɛ kǎh sõ̀ sõ légbɩ̃́ yõ sɛ̃́, képah ye kè náh syɩkɩ. 
Ǹtɛ névi nɛ̂-á ń wɛlɔ nɔ́pi noh tɛ́ à náh mɛ sé tyíyónɔ́lɛ n pɩ, tɛ̃̀ mɛ á kɩ nɛ lékã́m kɛ̃́ névi yɔ́ sɔ̃́, nɛ̂-á ǹ wɔ̀kɔ̀ mɔ sèmìyã́hrɩ́ yõ. 
Pũ̀ mɛ pɔ pɩ, yrɛy yuku à kwɔkɩ kyɩ syɩ kénɛ wɔ̀kɔ̀ tyɩ́, pṹnyã́hkɩ́ tɔ mɛ kõ kyɩ mɩ ké sɔkɔ, kè mɛ syɩkɩ à pirki kwɛ pnɛ́pnɛ́. 
Képah sɔkɔ, Yesu-á kwɔ́nɔ́ nɔ́pi kɔ̃ tɛ gbĩ́ nɛ̂nɛ, tɔ́wû gbã̀n du ǹ névye kwɔ́nɔ́ dùkùlɛ. 
Yé náh wɛ di, ǹ névye kwɔ́nɔ́ dùkù náh sõ̀ mɛ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú tyɩ́ kɔ̃lɛ dɛ́, kè sõ̀ mɛ yõ̀tɔ̃rɩ́ kwɔ́nɔ́lɛ képah ye.
