Matye.
16.
Farisi wɛ̃̀kɩ̀ kɔ́pú yísyɔ́ ànɛ̂ Sadusi wɛ̃̀kɩ̀ kɔ́pú yísyɔ́ nɔ de Yesu tnɔ̂ nɛ pé à mɔ yah. Tɛ́ à syah nɛ à gbǐl yɔ́ pɩ pé yah, nɛ̂-á n wɛ̃kɩ nɛ wǎh n wil Liyel tyɩ́. 
Képah sɔkɔ, Yesu mɛ yo pé tyɩ́ nɛ: «Gbĩ́yĩ́ de gbĩ́nɛ, yĩ̂nyõ-à mɛ gbɔ̃̌lɩ̀ɩ̀, yè n nɛ tyah yah-á kɩ yɩkɩ mɔ. 
Tɛ́ séngbɔ̃́lɛ, yĩ̂nyõ-à mɛ fĩ̌nmɔ̀ɔ̀, yè n nɛ kwɛynɛ tyah yah náh n yɩkɩ n mɔ n pi. Yé mɛ fɛ̃ tyah dùkù wɔlɛ n pnɛ, tɛ́ tyi nónó wɔ-á mɛ n pɩ kwɛy gbĩ sɔkɔ, sǎh mɛ tyi nónónɛ n wɛ̃kɩ, yé tɛ̃ sépi wɔ dùkù pɩ̃ kwlɔ kɛ̀? 
Kwɛy gbĩ névye náh plɛ. Pè náh mɛ tɛ̃ Liyel ó tyɩ́ yõ. Pè mɛ ké tyɩ́ nɛ mé gbǐl yɔ́ pɩ pé yah. Ǹtɛ kèyǎh ńtɛ̃ náh pɩ pè wɛ̃kɩ n pi, à wil Liyel tɛ̃ntõ̀ syínyo-ò Nsyonasɩ tyɩ́ sɔkɔ.» Képah yóntɛnɔ náh, à pè sah nɛ́npɔ́ tɛ́ sɔkɔ. 
Képah náh, Yesu ànɛ̂ ǹ tõ̀ sã́hpú-á n sɔkɔ yɔ́nkahnɩnɛ, ǹ tõ̀ sã́hpú dyɔ sõ̀ fwɔ búrn tɔ́kɔ́nsɔkɔnɔlɛ. 
Yesu mɛ yo pé tyɩ́ nɛ: «Yé kyɔ̀mɩ̀nɔ́ pɩ. Yé yé gblɔ̌ynɛ n tɛ̃ Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ Sadusi wɛ̃̀kɩ̀ kɔ́pú kɔ́kɔ́nɔ́lɛ.» 
Képah sɔkɔ, ǹ tõ̀ sã́hpú tyah n sõ nɛ pépi-á pɔ búrn kɛ̃́nɛ, képah-á à kɔ̃ à n yo sɛ̃́ yɔ̀. 
Ǹtɛ Yesu mɛ pé sõnɔ pɩ̃ tɛ́ yo pé tyɩ́ nɛ: «Liyel yõ sɛ̃́nyĩ-ńkɛ̃̂pú nɔ́pi. Sɔ̃́ pɩ tɛ́ yé n sõ sɛ̃́ nɛ, yé náh pɔ búrnnɛ képah-á? 
Yé náh tɔ̃̌nɩ̀ mɛ sé dùkù pɩ̃ sɛ̃́ nɩ? Yé dyɔ náh mɛ n kwnɛ tɔlɛ máh búrn plɔ́ kwlɔ́lɛ díbí nɛ́pĩ̂ kèyõ nínɔ́ kyɛgbãm (5 000) pu, ànɛ̂ yáh yṍnkwlɔnɔ tɛ́lɔ́ tuke mɔ vyɔ́ yɩ̃́ dan nɛ̂nɛ di? 
À tahlɩ máh búrn plɔ́ kwǎrninɔlɛ díbí nɛ́pĩ̂ kèyõ nínɔ́ (4 000) pu, tɛ́ yáh yṍnkwlɔnɔ tɛ́lɔ́ tuke mɔ tɔkɔ yɩ̃́ dan nɛ̂nɛ, yé dyɔ náh mɛ n kwnɛ kélɛ di? 
Tɛ́ sɔ̃́ pɩ tɛ́ yé náh tɛ̃ wɛ pɩ̃ nɛ búrn tyɩ́ náh mé n yo yé tyɩ́, máh sõ̀ n yo yé tyɩ́ nɛ: ‹Yé yé gblɔ̌ynɛ n tɛ̃ Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ Sadusi wɛ̃̀kɩ̀ kɔ́pú kɔ́kɔ́nɔ́lɛ?» 
Képah sɔkɔ, pè pyě pɩ̃ nɛ mífã́hkɩ̀ sɔkɔ mɔ mɔ kɔ́kɔ́nɔ́ tyɩ́ náh sõ̀ Yesu n yo pé tyɩ́ nɛ pé pé gblɔ̌ynɛ n tɛ̃ kélɛ. Ǹtɛ Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ Sadusi wɛ̃̀kɩ̀ kɔ́pú-á kwɔ́nɔ́ nɛ̂nɛ n kɔ̃, képah tyɩ́-á sõ̀ à n yo pé tyɩ́ nɛ pé pé gblɔ̌ynɛ n tɛ̃ kélɛ. 
Yesu-á kyɩ yĩni tɛ̃ Filipi-Sesarɩ gbɛ̃̀nyah tĩ̀nnɛ, à ǹ tõ̀ sã́hpú piki yah nɛ: «Névye tɛ̃̀ tyɩ́ nɛ̂ mɛ ńmɔlɛ, ńmɔ nɛ̂-á Névi Pídĩ́nɛ?» 
Pè mɛ nɛ, túkù-á nɛ Wèlnwìlkì-ò Nsyɔ̃-á ǹnɛ. Túkù nɛ Liyel tɛ̃ntõ̀ syínyo-ò Yeli-á ǹnɛ. Túkù mó nɛ Liyel tɛ̃ntõ̀ syínyo-ò Nsyeremi-á, képah náh pɩ Liyel tɛ̃ntõ̀ syínyo-ò yɔ́-á ǹnɛ. 
Képah sɔkɔ, à tɔ̃ nɛ: «Ǹtɛ yépi-á dò, yépi tɛ̃̀ tyɩ́ nɛ̂ mɛ ńmɔlɛ?» 
Simo Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ: «Áwɔ ye Liyel nɛy yáhntɔkɔnɔlɛ. Liyel nɛ̂-á min nɛ́tɛ̃̂nɛ, ǹmɔ Pídĩ́ ye álɛ.» 
Képah sɔkɔ, Yesu yo ǹ tyɩ́ nɛ: «Áwɔ tyɩ́ nɔ, Nsyonasɩ pídĩ́ Simo. Á náh wɛ di, névi náh wɛ̃kɩ álɛ tyi nɔ́pilɛ dɛ́, ń Sú Liyel nɛ̂-á mɛ yĩ̂nyõ sɔkɔ, ǹmɔ ye wɛ̃kɩ álɛ sélɛ. 
Ǹtɛ ńmɔ kɩ yo á tyɩ́ nɛ: ‹Pyɛrɩ ye álɛ, (nɛ̂ kɔ́r-á légbɩ̃́lɛ). Tɛ́ kénɛ légbɩ̃́ yõ sɔkɔ ye mé kɩ ń yõ sɛ̃́pú tĩ̀nkì sah. Tɛ́ kũ tǎhkɩ̀ ńtɛ̃ fyé náh tɛ̃ wɛ tir yɔ́ pɩ n pi pé sɔkɔ. 
Mé kɩ Liyel tɔ̃́rɩ́ tyah yah gbɩ̃́nɩ́ kéníkɔ́pyélɛ ǹ kɔ̃. Wâh tir nɛ̂ yahle sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ. Tɛ́ wâh nɛ̂ wɛ̃̀kɩ̀ kɔ̃ sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ.» Mé kɩ áwɔ kɔ̃ á kɩ fɛ̃ Liyel tɔ̃́rɩ́ nɔlɩ yṍnɔ́ wɛ̃̀kɩ̀lɛ n kɔ̃. Wâh tir nɛ̂ yahle sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ. Tɛ́ wâh nɛ̂ wɛ̃̀kɩ̀ kɔ̃ sétáh mɛ̀ yõ sɔkɔ, Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, ǹ tɔ kɩ sah ké yõ sɔkɔ.» 
Képah náh, Yesu ǹ tõ̀ sã́hpú gbah sni nɛ, pè káh kè yãh névi syah nɛ péwɔ-á Liyel nɛy yáhntɔkɔnɔlɛ dɛ́. 
À tɔkɔ légbĩ́ tyɩ́, Yesu tyah ké yahlɛ n yo ǹ tõ̀ sã́hpú tyɩ́ nɛ, páh yɩ̃nɛ pé yuku Yerusalɛmɩ kwil sɔkɔ. Yèwàhnɩ̀, ànɛ̂ sárká syínwìlkìpu yõ̀tãm, ànɛ̂ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú-á kɩ yèvyãhrɩ náhnáh sah n pi péwɔ yõ. Pǎh kɩ pé ko mɔ, tɛ́ ké sèpĩ̀n tɔ̃́-ò sõlɛ, pé lékó-á kɩ yilki mɔ. 
Képah tɛ̃̀nɛ, Pyɛrɩ mɛ à ye kɛ̃kɩ tɛ́ tyah n wɛkɩ ǹ yõ nɛ ńkɛ̃́, póllɛ pé Tɛ̃̀ ǹgbɛ̃, képah náh à yĩni n tɛ̃ n pi. Liyel-á kɩ n tũ n mɔ ǹ yõ. 
Ǹtɛ Yesu mɛ ǹ yah vi tɛ́ yo ǹ tyɩ́ nɛ: «To ń yah sétã̀n yõ̀tɛ̃̀ mɛ̀. Ńmɔ fɛ̃̀kɩ̀-ǹsah ye á n yah n kɔ̃. Yah, á sõnɔ náh mɛ núkúlɛ Liyel tyɩ́lɛ. Á sõnɔ mɛ núkúlɛ névi tyɩ́lɛ.» 
Képah náh, Yesu yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Névi nɛ̂-à mɛ ké tyɩ́ nɛ pé ńmɔ sõ̀ tɛ̃, tɛ̃̀ yɩ̃nɛ à ǹ gblɔ̌y tyi wɛ n yɛ, à mó ǹ syɩ syɩ dahnɩ twah ńmɔ sõ̀ tɛ̃. 
Névi nɛ̂-à nɛ páh n to n pwah kũ yah, tɛ̃̀ náh tyɛ́-ńkɛ̃̂ min wɛ n pi. Tɛ́ nɛ̂-á tɛkɩ ǹ min sɔkɔ ńmɔ yĩnnɛ, tɛ̃̀ wɔ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Névi-à nɛ kèkõyṍ yî pól wɛ n pi, tɛ́ nɛ n tɛkɩ n pi ǹ min sɔkɔ, yõ se mɛ ké tyɩ́? Képah náh pɩ, kwâh se kɩ fɛ̃ wɛ névi tyɩ́ à kɩ tɔkɔ kɔ̃, ǹ min sɔ̃́nɛ? 
Yé yah, Névi Pídĩ́ kɩ ǹ syɩ n pɔ n pi ànɛ̂ ǹ yĩ̂nyõ tõ̀npɩbɩlɛ ǹ Sú Liyel tɔ̃́rɩ́lɛ ǹ sɔkɔ. À kɩ pɔ nɛ́núkù núkú pól dyah à yɩ̃nɛ pé tyípɩ́nɩ̀nɛ. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, névye nónó-á ásõ̂, pé túkù náh tɔ̃̌nɩ̀ mɛ n ku n pi, tɛ́ kɩ Névi Pídĩ́ wɛ à kɩ mɛ n pi ǹ yõ̀tɔ̃rɩ́ sɔkɔ.»
