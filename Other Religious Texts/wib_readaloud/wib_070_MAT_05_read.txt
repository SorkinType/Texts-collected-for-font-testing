Matye.
5.
Yesu-á tɔ́wû wɛ sɛ̃́, à dɔkɔ yuku yɔ̃lɩ yõ. Wǎh kɔ̃ tɛ̃, ǹ tõ̀ sã́hpú mɛ nɔ de ǹ tnɔ̂. 
Képah sɔkɔ, à mɛ tyah pélɛ n kwɔ nɛ ńkɛ̃́: 
«Nónó-á pɩ̃ nɛ Liyel yétõ̂-á mɛ pé tɛ̃, pépi tyɩ́ nɔ. Pè kɩ Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi. 
Nónó-á mɛ gbní sɔkɔ, pépi tyɩ́ nɔ. Liyel kɩ pé yétĩ kele n pi. 
Gblɔ̌y tíkínmɔnɔ-á mɛ nónó sɔkɔ, pépi tyɩ́ nɔ. Sétáh kɩ pɔ n pɩ n pi pépi tyɩ́lɛ. 
Gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ dékè-á mɛ nónó sɔkɔ, á kɩ nɛ kwɛkɩ ànɛ̂ níkì sɔ̃́ tɛ̃ pélɛ, pépi tyɩ́ nɔ. Pǎh nɛ̂nɛ n yah n kɔ̃, pè kɩ kè wɛ n dal n mɔ n pi. 
Nónó-á névye yãmnɛ n yah, pépi tyɩ́ nɔ. Liyel kɩ pé tɔ yãm yah n pi. 
Nónó nnɔ-á mɛ yályál Liyel yah, pépi tyɩ́ nɔ. Pè kɩ Liyel wɛ n pi yĩ́nɛ. 
Nónó gbõ̀ yõ-á yèvɩnɩ n wɛ, pépi tyɩ́ nɔ. Pè kɩ n ye n pi Liyel wɛ̃́npìlɛ. 
Nónó-á mɛ fṍhnɔ́lɛ n wɛ pé nɛ́gblɔ́ pɩ́nɔ́ yĩnnɛ Liyel yah, pépi tyɩ́ nɔ. Pè kɩ Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi. 
Névye-à yélɛ n gbyɔ gbĩ́ nɛ̂nɛ, tɛ́ yélɛ n fõh, tɛ́ yé tyɩ́lɛ n yo n ko à mɩ̀nɩ̀ duke duke póllɛ n sah yé yõ ńmɔ yĩnnɛ, yépi tyɩ́ nɔ. 
Sɛ̃́ ye lésõ pè fõh Liyel tɛ̃ntõ̀ syínyopulɛ nónó-á yé dyàh tɛ̃ pɔ sɔkɔ. Yé kɩ ké yõre ǹgbɛ̃ wɛ n pi yĩ̂nyõ sɔkɔ. Képah ye, yé nɛ n mɛ nɩ̀vɩ̀nɩ̀ ǹgbɛ̃ sɔkɔ yèsyɩ̀kɩ̀ tɛ̃̀nɛ. 
Yépi ye n nɔ kèkõyṍlɛ, sɔ̃́-á pnɛ pãnnɛ n nɔ. Ǹtɛ pnɛ dêndên-à wal ké sɔkɔ, sɔ̃́ mɛ kè kɩ tɔ̃ fɛ̃ pɩ dêndên? Kè náh tɔ̃ fɛ̃ tir yɔ́ pɛkɩ n pi, à wil mílkínɔ́ tyɩ́ sɔkɔ kégbɔ́, tɛ́ névye kɩ n kɔ n sɔkɔ ké yõ. 
Yépi ye kèkõyṍ kyòolɛ. Kwil nɛ̂-à mɛ mɔ́nɔ́lɛ yɔ̃lɩ yõ, kè náh fɛ̃ n wɔ̃bɩ n pi dò. 
Pè náh fɔ̀tnɔ̀lɛ n soh tɛ́ dúlɛ n kwĩnki n mɔ ké yah. Pě kè sah fɔ̀tnɔ̀ sahkɩ sɔkɔ, kè kɔ̃ kè n pɩ kyòolɛ wɔ̀kɔ̀ sɔkɔ névye pól tyɩ́. 
Sɛ̃́ ye yé yɩ̃nɛ yè pɩ kyòolɛ névye tyɩ́, pè kɔ̃ pè yé tyípéplɔ pɩ́nɔ́ wɛ, pè mó yé Súlɛ n gbilki, nɛ̂-á mɛ yĩ̂nyõ sɔkɔ. 
Yé káh wɛ nɛ ńmɔ-á pɔ tyi kõ̀nsàhnɔ́ duku kò-ǹsah, képah náh pɩ Liyel tɛ̃ntõ̀ syínyopu-á kwɔ́nɔ́ nónó kɔ̃, sépi duku kò-ǹsah dɛ́. Mé náh pɔ sé duku kò-ǹsah, ǹtɛ mé pɔ sé tyi pɩ́ndalnmɔ-ńsah. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, yĩ̂nyõ ànɛ̂ sétáh tɛ̃̀-à fyɔ̀ mɛ, kwâh tyɩ̃́ɩ̃ yɔ́, képah náh pɩ kwáhkyɩ̃rɩ́ yɔ́ ńtɛ̃ náh n wilki n pi tyi kõ̀nsàhnɔ́ sɔkɔ, fɔ́ɔ́ kɩ pɔ syɩ ké sɔkɔ tyi pól pɩ́ntɛnɔ tyɩ́. 
Képah ye, névi nɛ̂-à n sàh-ǹkɛ̃̂nɛ tyi kõ̀nsàhnɔ́ sɔkɔ tir kyɩ̃rɩ́ fyé yõ, tɛ́ no tɛ́lɔ́ tɔ yõ yɩkɩ mɔ sɛ̃́ntɛ̃̀ pɩ́nɔ́ sɔkɔ, tɛ̃̀ kɩ fṍrɔɔ kal Liyel tɔ̃́rɩ́ nɔlɩ yṍpú póllɛ. Ǹtɛ névi nɛ̂ wɔ-à n sah sé sɔkɔ tyíyónɔ́ yõ, tɛ́ sélɛ nolɛ n kwɔ, tɛ̃̀ wɔ kɩ yĩ́n-ńsah ǹgbɛ̃ wɛ Liyel tɔ̃́rɩ́ nɔlɩ yṍ-ńsah sɔkɔ. 
Mé kélɛ n yo yé tyɩ́, yáh kɩ pɩ́-ńkɛ̃̂nɛ gbɩ wɛ̃̀kɩ̀ kɔ́púlɛ Liyel yah à kal Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́púlɛ, yé náh Liyel tɔ̃́rɩ́ nɔlɩ yõ n pi. 
Yé kè noh sɔkɔ nɛ kǎh yo á náh tãn tyɩ́ nɛ, nɛy káh nɛ́kókè mɔ. Névi nɛ̂-à nɛ́kókè mɔ, tɛ̃̀-á yɩ̃nɛ tùkènɔ́lɛ. 
Ǹtɛ ńmɔ wɔ kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à ǹ fɔkɔ vyãh ǹ ní-òlɛ, tɛ̃̀-á yɩ̃nɛ tùkènɔ́lɛ. Nɛ̂-à ǹ ní-ò ye Nɛ́wõ̂lɛ, tɛ̃̀-á yɩ̃nɛ tùkènɔ́lɛ tukey tùkèpu tĩ̀nkì tyɩ́. Tɛ́ nɛ̂-à à ye Vínpɩlɛ, tɛ̃̀ wɔ-á yɩ̃nɛ dáhnmɔnɔlɛ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ. 
Wâh pɔ á yɩ̃ n kɔ̃ kɔ̃ kwâhlɛ Liyel kɔ̃́-ńsah, tɛ́ á dyɔ to kõ nɛ́npɔ́ sɔkɔ nɛ wáh pɩ de á ní-ò yɔ́ tyɩ́, 
á yɩ̃ n kɔ̃ kɔ̃ kwâh sah nɛ́npɔ́, ǹ tɛ́ n kyɩ á wrɔ́ tɔ̃ n sah á ní-ò mɛ̀nɛ, ǹ tɛ́ n pyě n pɔ kélɛ Liyel kɔ̃. 
Tir yɔ́ kénkɔ̃-à mɛ yé wrɔ́ ànɛ̂ ǹnɩ̂ yɔ́lɛ, tɛ́ à kyɩ ǹ ye tukey tùkè-ò tyɩ́, gbah n tɛ̃ ké yah wah yé yĩ́níntɛ̃-ńkɛ̃̂ sɔkɔ nɛ́npɔ́. Képah-à pɩ́-ńkɛ̃̂nɛ, à kɩ kyɩ álɛ tukey tùkè-ò tyɩ́ nɛ à ké yah wah, tukey tùkè-ò mɛ̀ kɩ kyɩ álɛ sràsyíbí kɔ̃, tɛ́ pépi kɩ mó ǹ dah mɔ wɔ̀kɔ̀npèkè sɔkɔ. 
Kègbɩ yõ mé kélɛ n yo á tyɩ́, á náh n wil n pi nɛ́npɔ́, nɛ́ á náh kɔ̃lɩ mɛ̀ pól dyah tɛ pnɛ́pnɛ́. 
Yé kè noh sɔkɔ nɛ kǎh mɛ yónɔ́lɛ nɛ: «Káh fɔ̀fɩ́kɩ̀lɛ n pɩ.» 
Ǹtɛ ńmɔ kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à sê yɔ́ yah, tɛ́ yĩ́ wil ǹ tyɩ́, tɛ̃̀-á fɔ̀fɩ́kɩ̀ pɩ tɛ ǹnɛ ǹ nɩ sɔkɔ. 
Á gbõ̀yõ̀ sõ̀ yĩ́-à kɩ nɛ̂ ǹ dah n mɔ n pi tyípékè pɩ́nɔ́ sɔkɔ, kè gbahlɩ n wilki n milki álɛ fɩ́ɩ́. Á tɛ́kɩ́nɔ́lɛ á wil gbɛ̃́nsê fɩ̃́ núkú sɔkɔ, képah kɩ fṍnfṍ kal á wil pól mínndahnmɔnɔ yõ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ. 
Á gbõ̀yõ̀ sõ̀ gbõ̀-à kɩ nɛ̂ ǹ dah n mɔ n pi tyípékè pɩ́nɔ́ sɔkɔ, kè kãm n milki álɛ fɩ́ɩ́. Á tɛ́kɩ́nɔ́lɛ á wil gbɛ̃́nsê fɩ̃́ núkú sɔkɔ, képah kɩ fṍnfṍ kal á wil pól kyɩ́ndenɔ yõ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ. 
Kè mɛ yónɔ́ tɔlɛ nɛ, névi nɛ̂-à ǹ kyɩlɛ n fĩn n mɔ, tɛ̃̀-á yɩ̃nɛ à ǹ fĩ́nnmɔnɔ tàrfyɔ̀ sɛ́bɛ́y pɩ à kɔ̃. 
Ǹtɛ ńmɔ tɛ̃̀ wɔ kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à ǹ kyɩ fĩn mɔ nɛ́ fɔ̀fɩ́kɩ̀ tɛ̃̀ tyɩ́ yõ náh, tɛ̃̀-á mɛ ǹnɛ n mɔ fɔ̀fɩ́kɩ̀ pɩ́nɔ́ wɛ̃̀kɩ̀ yõ. Névi nɛ̂ tɔ-à mó sê fĩ́nnmɔnɔ tɔkɔ, fɔ̀fɩ́kɩ̀-á tɛ̃̀ n pɩ. 
Yé kè noh sɔkɔ tɔlɛ nɛ kǎh yo á náh tãn tyɩ́ nɛ: «Káh álɛ n wo mɩ̀nɩ̀ tyi sɔkɔ. Wâh á wo tir nɛ̂ sɔkɔ Tɛ̃̀ ǹgbɛ̃ yah, á yɩ̃nɛ ǹ kè pɩ.» 
Ǹtɛ ńmɔ tɛ̃̀ wɔ kélɛ n yo yé tyɩ́ nɛ, yé káh yélɛ n wo póllɛ. Yé káh yélɛ n wo yĩ̂nyõlɛ. Liyel tɔ̃́rɩ́ kǒ ye kélɛ, képah ye yé káh yélɛ n wo kélɛ. 
Yé káh yélɛ n wo sétáhlɛ. Liyel gbyẽ̂h súnsàh-ǹsah kǒ ye kélɛ, képah ye yé káh yélɛ n wo kélɛ. Yé káh yélɛ n wo Yerusalɛmɩnɛ. Yõ̀tɛ̃̀ ǹgbɛ̃ Liyel kwil ye kélɛ, képah ye yé káh yélɛ n wo kélɛ. 
Káh álɛ n wo á yõlɛ. Á náh fɛ̃ á yõ̀yén pɔ́ núkú ńtɛ̃ fyé piki, képah náh pɩ à kè silki n pi, képah ye káh álɛ n wo kélɛ. 
Kàh àwelɛ, yo nɛ àwe-á. Kàh ɔ̃́ɔ̃lɛ, yo nɛ ɔ̃́ɔ̃-á. Tɛ́ wâh nɛ̂ yo dohnɩ sépi yõ, pèkè pɩ́-ò tõ̀ vyãh ye. 
Yé kè noh sɔkɔ nɛ kǎh mɛ yónɔ́lɛ nɛ: «Wâh yĩ́ sɔ, á yĩ́ kɩ sɔ. Wâh yĩ́n dal, á yĩ́n kɩ dal.» 
Ǹtɛ ńmɔ kélɛ n yo yé tyɩ́ nɛ, névi-à tyítúkù pɩ yé tyɩ́, yé káh nɛ yáh ké wɔ́lɛ n dɛ. Tɛ́ névi-à á gbõ̀yõ̀ sõ̀ létáh mɩ, ké ní-ò tɔ vi n mɔ à kɔ̃. 
Névi-à mɛ ké tyɩ́ nɛ pé ǹ ye tukey tùkèpu tyɩ́ á fwɔ̀mɔ̀npír syi yĩnnɛ, á fwɔ̀mɔ̀-ǹgbɛ̃ tɔlɛ tɛ̃̀ kɔ̃. 
Névi-à ǹ kyɩ̃ mɔ nɛ á kíló núkú kɔ pélɛ, kílónɔ́ nínɔ́ kɔ tɛ̃̀nɛ. 
Névi nɛ̂-à ni á tyɩ́, yɩ̃ tɛ̃̀ kɔ̃. Nɛ̂-à nɛ pé dɛkɩ syi á tyɩ́, káh n yahle. 
Yé kè noh sɔkɔ nɛ kǎh mɛ yónɔ́lɛ nɛ, á ní-ò tyɩ́-á yɩ̃nɛ kè nɔ á tyɩ́, á mó á sépɛ dyɔlɛ n yɔ. 
Ǹtɛ ńmɔ kélɛ n yo yé tyɩ́ nɛ, yé yɛ yé sépɛ́bɩ́ tyɩ́ nɔ yé tyɩ́, yé mó Liyellɛ n ni yé fṍhpú tyɩ́. 
Yâh képahlɛ n pɩ, yé Sú Liyel nɛ̂-á yĩ̂nyõ sɔkɔ, yé mɛ kélɛ n wɛ̃kɩ nɛ ǹmɔ wɛ̃́npì-á yélɛ. Ǹmɔ ye n sɛ ǹ gbĩnɛ nɛ́pêl ànɛ̂ nɛ́péplɔ tyɩ́, tɛ́ pũ̀nɛ n pɩ n mɔ nɛ́gblɔ́ ànɛ̂ nɛ́túkúnì tyɩ́. 
Tɛ́ yépi tyɩ́-á nɔ nónó tyɩ́, pépi ó tyɩ́-à nɔ yé tyɩ́, yõre nɛ̂ mɛ yé kɩ n wɛ n pi képah sɔkɔ Liyel tyɩ́? Mɔ́rɩ́ syípú ńtɛ̃ tɔ náh mɛ sɛ̃́ntɛ̃̀nɛ n pɩ di? 
Yâh n wɛkɩ n mɔ yé no ó tyɩ́, gbɛ̃̀ndùnì tir nɛ̂ mɛ képahlɛ? Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú ńtɛ̃ tɔ náh mɛ sɛ̃́ntɛ̃̀nɛ n pɩ di? 
Yépi tɛ̃̀ wɔlɛ, yé tyípéplɔ ó kɔ́kɔ́lɛ n pɩ, sɔ̃́-á yé Sú tyípéplɔ ó kɔ́kɔ́lɛ n pɩ, ǹmɔ nɛ̂-á mɛ yĩ̂nyõ sɔkɔ.
