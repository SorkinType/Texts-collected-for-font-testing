Matye.
8.
Yesu-á wil yɔ̃lɩ yõ à tiki pɔ gbĩ́ nɛ̂nɛ, tɔ́wû ǹgbɛ̃ ǹ sõ̀ tɛ̃. 
Tótɛ̃ yɔ́ mɛ nɔ de à gbètukunɔ di ǹ yah sɔkɔ, tɛ́ yo ǹ tyɩ́ nɛ, pé Tɛ̃̀ ǹgbɛ̃, wàh mɛ ké tyɩ́ wǎh kɩ fɛ̃ pé dɛ mɔ pé kɔ̃ pé pɩ yályál Liyel yah. 
Wǎh yo sɛ̃́, Yesu mɛ ǹ gbõ̀ nɔ à tũ ǹ tyɩ́, tɛ́ yo ǹ tyɩ́ nɛ: «Mé mɛ ké tyɩ́, á kɩ dɛ.» Nɛ́npɔ́ swɔ, dĩ́ dú ńyân núkú tyɛ. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ: «Káh yĩ mɛ̀ yãh nɛy tyɩ́ dɛ́. Tɛ́ kyɩ á gblɔ̌ynɛ sárká syínwìlkì-ò wɛ̃kɩ. Moyisi-á sõ̀ sárká nɛ̂ tyɩ́ yo sah tótãm dɛ́nmɔnɔ tyɩ́ sɔkɔ, kyɩ képah wilki, nɛ̂-á kɩ kélɛ pè wɛ̃kɩ nɛ wáh dɛ.» 
Tɛ́ Yesu-á pɔ de Kapɛrnayumi sɔkɔ gbĩ́ nɛ̂nɛ, Wrome sràsyíbí yõ̀tɛ̃̀ dĩ́ yɔ́ pɔ ǹ tnɔ̂ tɛ́ à ni 
nɛ, pé yõ̀tɛ̃̀, pé tõ̀npɩ dĩ́-á mɛ sɛ̃́nɔ́lɛ gbô sɔkɔ wil kúnɔ́ ńyânnɛ ǹ sɔkɔ. Wǎh mɛ n kwlɔ képékèyɔ̂. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ páh kɩ ǹ sõ̀ tɛ̃ kyɩ à dɛ mɔ. 
Ǹtɛ kénɛ dĩ́ mɛ yo Yesu tyɩ́ nɛ, pé yõ̀tɛ̃̀, péwɔ-á fṍrɔɔ kah ǹ dénɔ́lɛ pé gbô sɔkɔ. Ńkɛ̃́ à wɛy núkú ó yo, tɛ́ pé tõ̀npɩ-á kɩ dɛ. 
À kó wɛ nɩ, yõ̀tãm-á mɛ péwɔ tyɩ́, tɛ́ péwɔ tɔ mó mɛ sràsyíbí yísyɔ́ yõ̀tɛ̃̀nɛ. Pâh yo nɛ̂ tyɩ́ nɛ, à sɔkɔ! Tɛ̃̀ n sɔkɔ. Pâh yo nɛ̂ tyɩ́ nɛ, à pɔ! Tɛ̃̀ n pɔ. Pâh yo pé yɔ̃̀ tyɩ́ nɛ, à tir mɛ̀ pɩ! Ǎ kè pɩ. 
Képah sɔkɔ, Yesu-á kénɛ wɛlɔ noh sɛ̃́, sè ǹ gbã̀n kah du. À mɛ yo ǹ sõ̀ tɛ̃́pú tyɩ́ nɛ: «Kègbɩ yõ mé kélɛ n yo yé tyɩ́, mé náh Liyel yõ sɛ̃́nɔ́ ǹgbɛ̃ mɛ̀ dùkù névi wɛ yah Yisrayel no ńtɛ̃ fyé sɔkɔ. 
Tɛ́ mé kélɛ n yo yé tyɩ́ nɛ, névye náhnáh-á kɩ wil ywɔ̃ ywɔ̃ póllɛ, kɩ pɔ yõke yõ wɛ̃ tyɩ́ Abrahmɩnɛ, ànɛ̂ Yisakɩlɛ, ànɛ̂ Nsyakɔbɩlɛ Liyel tɔ̃́rɩ́ tyah sɔkɔ. 
Ǹtɛ Yisrayel no nónó-á sõ̀ yɩ̃nɛ dénɔ́lɛ Liyel tɔ̃́rɩ́ tyah sɔkɔ, pépi kɩ wũ kégbɔ́ fɛ̃́ntũ̂ sɔkɔ, ńyãh sɔkɔ-á gbní ànɛ̂ yní táhnɔ́ kɩ mɛ.» 
Képah tɛ̃̀nɛ, Yesu ǹ syɩ yo sràsyíbí yõ̀tɛ̃̀ dĩ́ mɛ̀ tyɩ́ nɛ: «Á syɩ nɛ n yuku á tĩ̀nnɛ. Wáh sɛ̃ ńmɔ yõ sɛ̃́, wáh nɛ̂ yah kɔ̃, kè kɩ pɩ á tyɩ́.» Nɛ́npɔ́ swɔ, ǹ tõ̀npɩ dĩ́ núkú dɛ. 
Képah náh, Yesu kyɩ Pyɛrɩ gbô, tɛ́ kyɩ Pyɛrɩ dɛ́nsê wɛ sɛ̃́nɔ́lɛ wílvyã́hnɩ̀nɛ. 
Yesu mɛ tũ pi gbõ̀ tyɩ́, wílvyã́hnɩ̀ núkú à fĩn mɔ. Képah tɛ̃̀nɛ, pi mɛ yuku tyah n yah n dah n mɔ Yesu tyɩ́ sɔkɔ. 
Gbĩ́yĩ́-á de, no pɔ névye náhnáhlɛ Yesu tyɩ́ sétã̀n-á sõ̀ mɛ dénɔ́lɛ nónó sɔkɔ. À mɛ ǹ vyãh wɛynɛ sétã̀n nɔ́pi yĩni wilki pé sɔkɔ, tɛ́ yámpú tɛ́lɔ́ pól dɛ mɔ. 
À képah pɩ Liyel tɛ̃ntõ̀ syínyo-ò Yesayi tir yónsàhnɔ́ mɛ̀ kɔ̃ kè wil. À lésõ nɛ ńkɛ̃́: «À ye á yèvyãhrɩnɔ pól tɔkɔ to á yõ, tɛ́ á yamnɔ pól tyɛ.» 
Yesu-á tɔ́wû ǹgbɛ̃ wɛ pè pɔ à mɔ gbɩ̃ sɛ̃́, à nɛ pé tõ̀ sã́hpú àwɔ pé yuku yɔ́-ǹgbɛ̃ kahnɩnɛ. 
Képah sɔkɔ, Liyel tyi kõ̀nsàhnɔ́ kwɔ́-ò yɔ́ nɔ de ǹ tnɔ̂ tɛ́ yo ǹ tyɩ́ nɛ, kwɔ́-ò dĩ́, páh kɩ ǹ sõ̀ tɛ̃ n kɔ n kɔ̃ ǹnɛ ǹ yúkú-ńtã̂n pól sɔkɔ. 
Wǎh yo sɛ̃́, Yesu mɛ yo ǹ tyɩ́ nɛ: «Gbɛ̃́npãhlɔ tyɩ́ pɩ̃̀ mɛ. Kèpye tyɩ́ syɛ̀ mɛ. Ǹtɛ Névi Pídĩ́ tɛ̃̀ tyɩ́, wɛ́nsɛ̃nmɩ̀-ǹsah ńtɛ̃ náh mɛ.» 
Képah tɛ̃̀nɛ, ǹ tõ̀ sã́hpú sɔkɔ yɔ́ tɔ̃ à syah nɛ, kwɔ́-ò dĩ́, à yɛ pâh pɔ pé sú lékó ǹgbò yɩ̃ tɛ. 
Yesu mɛ yo ǹ tyɩ́ nɛ: «Ńmɔ sõ̀ tɛ̃, ǹ tɛ́ n yɛ nɛ́kyɩ̂ kɔ̃ pè pé lékyɩ̂lɛ n yɩ̃.» 
Képah sɔkɔ, Yesu de krótókè sɔkɔ ànɛ̂ ǹ tõ̀ sã́hpúlɛ pè sɔkɔ. 
Dǒ yã́hpékè yɔ́ núkú kõ yɔ́-ǹgbɛ̃ yõ sɔkɔ, fɔ́ɔ́ krótókè yõ n tyah yɩ̃́nɔ́lɛ ni sɔkɔ. Ǹtɛ Yesu wɔ sõ̀ mɛ n dɔ. 
Képah sɔkɔ, ǹ tõ̀ sã́hpú nɔ de ǹ tnɔ̂ mɛ kyɩ ǹnɛ n yilki n mɔ nɛ ńkɛ̃́, Tɛ̃̀ ǹgbɛ̃, à pé pwah mɔ oo! Yɔ-á mɛ n tyah pé yṍnɔ́lɛ dɛ́! 
Yesu mɛ yo pé tyɩ́ nɛ: «Kwâh nɛ̂ pɩ tɛ́ tíkí yé tɛ̃? Liyel yõ sɛ̃́nyĩ-ńkɛ̃̂pú nɔ́pi.» Képah tɛ̃̀nɛ, à yuku à dírí kõ yã́hkɩ́ ànɛ̂ ni yõ, tyah núkú pɩ sṹmm. 
Képah tɛ̃̀nɛ, kè kah pɩ gbɛ̃̀ndùnìnɛ pé tyɩ́. Pè mɛ tyah yo tũ wɛ̃ tyɩ́ nɛ, nɛ́dúkú nɛ̂ se dĩ́ mɛ̀nɛ tɛ́ yã́hkɩ́ ànɛ̂ yɔ tɔ n sah ǹ tyɩ́ yõ nɩ? 
Képah sɔkɔ, Yesu-á kyɩ yĩni tɛ̃ yɔ́-ǹgbɛ̃ kahnɩnɛ Kadarɩ tãn sétáh yõ gbĩ́ nɛ̂nɛ, díbí nɛ́pĩ̂ nímí mɛ wil léyã̂l wrɔ́ sɔkɔ à pɔ à yohnɩ sétã̀nnɛ pé sɔkɔ. Pé pékénkàhnɔ́ tɛ̃̀ tyɩ́, nɛy náh sõ̀ mɛ fɛ̃ n sɔkɔ nɛ́npɔ́ wɛ̃̀kɩ̀lɛ. 
Pǎh pɔ à yohnɩ, pè tyah n de n syi nɛ, Liyel Pídĩ́, kwâh nɛ̂ se à n yah n kɔ̃ pé tyɩ́ nɩ? À se pɔ ásõ̂ pépi fõh yĩnnɛ ké gbĩ́ nɔ́-ńkɛ̃̂ sɔkɔ nɩ? 
Tɛ́ kè sõ̀ pɔ yɩ̃nɛ sel kwĩnki yísyɔ́ mɛ n yah n yõ n kɔ̃ fɩ́ɩ́ kwéy pélɛ. 
Sétã̀n nɔ́pi mɛ Yesu ni nɛ, wàh kɩ sé yĩni wilki díbí nɔ́pi sɔkɔ, à yɛ sé kɔ̃ sé kyɩ n de sel nɔ́pi sɔkɔ. 
Yesu mɛ nɛ, sè nɛ n yuku. Sè núkú wil díbí nɔ́pi sɔkɔ à kyɩ de sel nɔ́pi sɔkɔ. Sel nɔ́pi pól núkú wil yɔ̃lɩ yõ̀tah, à to tiki tùkùtùkù à kyɩ de yɔ sɔkɔ à ku. 
Képah tɛ̃̀nɛ, sel nɔ́pi fɛ̃̀pu mɛ to de kwil sɔkɔ, à kyɩ tyi nɔ́pi pól yãh no tyɩ́, à tahlɩ tir nɛ̂-á pɩ díbí nɛ́pĩ̂ nímí nɔ́pi tyɩ́ sétã̀n-á sõ̀ nónó sɔkɔ. 
Képah sɔkɔ, kwil tãn pól wil kyɩ Yesu yohnɩ. Pǎh fyɔ̀ à wɛ, pè à ni nɛ à yal pépi sétáh yõ.
