Matye.
6.
Yé yé gblɔ̌ynɛ n tɛ̃ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́lɛ yĩ́nyáhkɩ̀ sɔkɔ dɛ́. Yâh képahlɛ n pɩ, yé náh ké yõre wɛ n pi Sú Liyel tyɩ́, ǹmɔ nɛ̂-á yĩ̂nyõ. 
Képah ye, wâh kwâhlɛ n yɩ̃ yãm nɛynɛ n kɔ̃, káh ké kwnínɛ n kɔ̃ sɔ̃́-á névye túkù n nɛ n pɩ kélɛ Liyel ní-ńtã̂n vɩ̀ sɔkɔ ànɛ̂ wã̀l yey sɔkɔ, névye kɔ̃ pě pélɛ n gbilki. Kègbɩ yõ mé kélɛ n yo yé tyɩ́, pè ye ké yõre wɛ tɛ. 
Ǹtɛ áwɔ-à kwâhlɛ n yɩ̃ yãm nɛynɛ n kɔ̃, káh n yɛ ǹnɩ̂ yɔ́ ńtɛ̃ pɩ̃ ké sɔkɔ. 
Képah sɔkɔ, yɩ̃́nkɔ̃nɔ nɛ̂-á á pɩ, kè kɩ tɛ̃̀ sètĩ̀n sõ̀ sɔkɔ. Tɛ́ Sú Liyel nɛ̂-á sètĩ̀n sõ̀ tyilɛ n pnɛ, ǹmɔ kɩ ǹ dyah ké yõrelɛ. 
Yâh Liyellɛ n ni, yé káh névye túkùlɛ n yah n pɩ. Liyel ní-ńtã̂n vɩ̀ ànɛ̂ wɛ̃̀tlo yey sɔkɔ yúkúnyĩnntɛ̃̀nɔ́ tyɩ́ ye nɔ pépi tyɩ́ pàh Liyellɛ n ni. Pé gblɔ̌y wɛ̃kɩ yĩn ye pè n nɛ n pɩ képahlɛ. Kègbɩ yõ mé kélɛ n yo yé tyɩ́, pè ye ké yõre wɛ tɛ. 
Ǹtɛ áwɔ-à mɛ ké tyɩ́ nɛ á Liyel ni, wâh pɩ ǹ de á wɔ̀kɔ̀ sɔkɔ, ǹ tɛ́ á yah vike, ǹ tɛ́ á Sú Liyellɛ n ni. Tɛ́ Sú Liyel nɛ̂-á sètĩ̀n sõ̀ tyilɛ n pnɛ, ǹmɔ kɩ ǹ dyah ké yõrelɛ. 
Yâh nírílɛ n pɩ, yé káh tyilɛ n tɔ̃ n yo tyɛ́-ńkɛ̃̂nɛ Liyel pɩ̃́-ńkɛ̃̂pú tyɩ́ kɔ̃lɛ. Pépi tɛ̃̀ sõnɔ sɔkɔ, tyi tɔ̃́nyonɔ kwlěyy nírí ye Liyel n tɛ̃. 
Ǹtɛ yî nónó yétõ̂-á mɛ yépi tɛ̃, yé Sú Liyel sè pɩ̃ tɛ sé ní-ńkɛ̃̂ gbĩ́ ńtɛ̃nɛ ǹ tyɩ́. Képah ye, yé káh Liyel pɩ̃́-ńkɛ̃̂púlɛ n yah n pɩ. 
Yáh yɩ̃nɛ yè Liyellɛ n ni dùkù nɛ̂nɛ, kè wɛ yɔ̀: Ápi Sú Liyel nɛ̂-á yĩ̂nyõ, á yĩn nɛ n gbilki. 
Á yõ̀tɔ̃rɩ́ wɛ̃kɩ. Wáh mɛ nɛ̂ tyɩ́, képah nɛ n pɩ sétáh yõ, sɔ̃́-á kélɛ yĩ̂nyõ. 
Á kwɛy gbĩ̀yĩ̀kì yõkelɛ á kɔ̃. 
Ápi-á pɩ de áwɔ tyɩ́ tyi nónó sɔkɔ, sè yɔ̃ n mɔ á kɔ̃, sɔ̃́-á ápi tɔ yɔ̃ mɔ névye kɔ̃, nónó-á pɩ de á tyɩ́. 
Káh n yɛ à sétã̀n yõ̀tɛ̃̀ tɔ̃̀nmɔnɔlɛ n fɛ̃ n syi. Ǹtɛ á pwah n mɔ ǹ gbõ̀, ǹmɔ nɛ̂ tyɩ́-á pèkè pɩ́nɔ́ sõ n wil. Yõ̀tɔ̃rɩ́, ànɛ̂ tɛ̃́nwɛnɔ, ànɛ̂ gbílkínɔ́lɛ, áwɔ tyɩ́ ye gbĩ́mɛgbĩ́. Amiina! 
Névye nónó-á n pɩ n de yé tyɩ́, yâh n yɔ̃ n mɔ pélɛ n kɔ̃, yé Sú Liyel nɛ̂-á yĩ̂nyõ, à kɩ n yɔ̃ n mɔ yé tɔlɛ n kɔ̃. 
Ǹtɛ yépi-à névye tyípêllɛ n yɔ̃ n mɔ pélɛ n kɔ̃́-ńkɛ̃̂nɛ, yé Sú Liyel náh yépi tɔ tyɩ́ yɔ̃ n mɔ yé kɔ̃ n pi. 
Yâh níkì tɛ̃, yé káh yé yey kwɔ n sah névye túkù tyɩ́ kɔ̃lɛ. Pépi ńkɛ̃́nɛ pé yey yõre n sah nɛ névye pɩ̃ nɛ páh mɛ níkì tɛ̃. Kègbɩ yõ mé kélɛ n yo yé tyɩ́, pè ye ké yõre wɛ tɛ. 
Ǹtɛ áwɔ-à níkì tɛ̃, á yah wu, tɛ́ káh á yõ pɩ n sah yákárá, 
névye kɔ̃ pè pɩ̃ nɛ wáh mɛ níkì tɛ̃. Tɛ́ á Sú Liyel nɛ̂-á sètĩ̀n sõ̀ tyilɛ n pnɛ, ǹmɔ kɩ ǹ dyah ké yõrelɛ. 
Yé káh tahrɩlɛ n yah n koh n sah kèkõyṍ mɛ̀ sɔkɔ, képah nɛ̂ sɔkɔ-á sóhrɩ́ ànɛ̂ syɩ̃́mɩ́ kɩ fɛ̃ sélɛ n yõ, képah náh pɩ yɩ̃́pú kɩ fɛ̃ kye de vɩ̀ sɔkɔ kɩ sè yɩ̃ tɔkɔ. 
Ǹtɛ yé tahrɩlɛ n yah n koh n sah yĩ̂nyõ, képah nɛ̂ sɔkɔ-á sóhrɩ́ ànɛ̂ syɩ̃́mɩ́ náh mɛ sé yõ yõ tɛ̃̀nɛ, yɩ̃́pú tɔ náh mɛ kɩ fɛ̃ kye de vɩ̀ sɔkɔ, képah náh pɩ kɩ yɩ̃be yɩ̃. 
Á náh wɛ di, á tahrɩ-à mɛ ńyãh, á sõnɔ tɔ kɩ mɛ nɛ́npɔ́. 
Wil kyòolɛ, yɩ̃́ ye. Á yɩ̃́-à mɛ vyelɛ, á wil pól kɩ mɛ kyòo sɔkɔ. 
Ǹtɛ á yɩ̃́-à yɩkɩ, á wil pól kɩ mɛ fɛ̃́ntũ̂ sɔkɔ. Kyòo nɛ̂-á á sɔkɔ, kàh kɩ mɛ fɛ̃́ntũ̂nɛ, á mɛ yĩ́ĩ́ fɛ̃́ntũ̂ pbèy sɔkɔ dɛ́. 
Névi yɔ́ ńtɛ̃ náh fɛ̃ tõ̀ kwrɔ n pɩ n pi yõ̀tãm nɛ́pĩ̂ nímí tyɩ́. À kɩ núkú dyɔlɛ n yɔ, tɛ́ kɩ nɔ ǹ ní-òlɛ. Képah náh pɩ, à kɩ núkúlɛ n tĩnki, tɛ́ kɩ ǹ ní-ò wɛ mɔ. Sɛ̃́ ó ye, yé náh fɛ̃ tõ̀ kwrɔ n pɩ n pi Liyel ànɛ̂ pɔ́ tyɩ́. 
Képah ye ń kɔ̃ mé kélɛ n yo yé tyɩ́ nɛ, yé káh yé fɔkɔlɛ n sõ yáh nɛ wɛ n yõ tɛ́ wɛ n wɔ n pi sɔ̃́, tɛ́ kɩ mɛ min sɔkɔ, ànɛ̂ yáh nɛ yé wil flɔ wɛ n mɔ n pi sɔ̃́. Névi min tyɩ́ náh tah kal yõkelɛ di? Wil tɔ tyɩ́ náh tah kal flɔlɛ di? 
Yé yah kèpyelɛ yĩ̂nyõ sɔkɔ kɛ̀; sè náh sèmukɔlɛ n duku, sè náh sèmukɔlɛ n tɔkɔ; sè náh se mó tɔ̃ sèmukɔlɛ n koh n sah díké sɔkɔ; tɛ́ yé Sú Liyel sélɛ n pu. Ǹtɛ yépi náh fṍnfṍ kal kèpye yõ náhnáh di? 
Yépi sɔkɔ nɛ̂ tɛ̃̀ kɩ fɛ̃ kwéy ńtɛ̃ fyé dohnɩ ǹ gblɔ̌y min yõ, ǹ fɔ́kɔ́sõnɔ gbõ̀ yõ? 
Ǹtɛ kwâh nɛ̂ fɔkɔ sõnɔ mɛ yé tyɩ́ yé mɔ mɔ flɔ tyɩ́ sɔkɔ wɛ́? Yé yah fílkì fĩ pɩ dùkùlɛ swɛ̃y sɔkɔ kɛ̀: Sè náh tõ̀lɛ n pɩ, sè náh se mó tɔ̃ flɔlɛ n tĩ. 
Ǹtɛ mé kélɛ n yo yé tyɩ́ nɛ, yõ̀tɛ̃̀ dĩ́ Salomo tàh fyé yah yõ, à náh flɔ wɛ mɔ yah nónó-á plɛ nɔ fílfĩ nɔ́pi sɔkɔ yɔ́ ńtɛ̃nɛ. 
Liyel yõ sɛ̃́nyĩ-ńkɛ̃̂pú nɔ́pi. Fílkì nɛ̂-á swãh sɔkɔ kwɛynɛ, tɛ́ kɩ pɔ n sukɛ n pi ńsõ̂nɛ, Liyel-á képah fĩ pɩ pallɛ sɛ̃́ à, yépi wɔ se pyě à kɩ yɛ kóhnmɔnɔlɛ kal plɛ́ntnɔɔ? 
Yé káh yé fɔkɔlɛ n sõ yè pɔ nɛ n yo nɛ, kwâh nɛ̂ se yé kɩ wɛ yõ nɩ? Kwâh nɛ̂ se yé kɩ wɛ wɔ nɩ? Kwâh nɛ̂ se yé kɩ wɛ mɔ yé wil sɔkɔ nɩ? 
Yé náh wɛ di, Liyel pɩ̃́-ńkɛ̃̂pú ye ńkɛ̃́nɛ nɛ n yah n kɔ̃ sépi póllɛ tyɛ́tyɛ́nɩ́ kɛ̃́nɛ. Ǹtɛ yé Sú nɛ̂-á yĩ̂nyõ, à pɩ̃ nɛ yî nɔ́pi pól yétõ̂-á mɛ yé tɛ̃ dò. 
Yé Liyel tɔ̃́rɩ́ ǹgbò yah n kɔ̃, ànɛ̂ wǎh mɛ nɛ̂ tyɩ́. Képah sɔkɔ, à kɩ yî tɛ́lɔ́ tɔ pól dohnɩ yé kɔ̃. 
Yé káh yé fɔkɔlɛ n sõ ńsõ̂ tyɩ́ sɔkɔ. Yé ńsõ̂ tyi sah ńsõ̂ yah. Swɔ̃ swɔ̃ pól ànɛ̂ sé yṍtyí ye.
