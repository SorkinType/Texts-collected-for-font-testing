Matye.
10.
Sõ yɔ́lɛ, Yesu ǹ tõ̀ sã́hpú nɛ́pĩ̂ gbɔ̃nimí ye pɔ, tɛ́ tɛ̃́nwɛnɔlɛ pè kɔ̃ nɛ pè sétã̀nnɛ n yĩni, pè mó yámpú póllɛ n dɛ mɔ, ànɛ̂ nɛ́fukulɔ́ póllɛ. 
Kénɛ nɛ́pĩ̂ gbɔ̃nimí nɔ́pi ye à pɩ ǹ tɛ̃ntõ̀ névyelɛ. Pé yni wɛ yɔ̀: Kè tyah Simo tyɩ́, nɛ̂-á pè yĩn dyɩ à kɔ̃ Pyɛrɩlɛ, ànɛ̂ ǹ gbɛ̃̀ndĩ́ Andrelɛ, à tahlɩ Nsyakɩ ànɛ̂ ǹ gbɛ̃̀ndĩ́ Nsyɔ̃ tyɩ́, pépi nónó-á Nsebede dĩ́npìlɛ, 
à tahlɩ Filipi ànɛ̂ Bartelemi tyɩ́. Toma sõ̀ mɛ pé sɔkɔ ànɛ̂ Matyelɛ, nɛ̂-á sõ̀ mɔ́rɩ́ syí-òlɛ, à tahlɩ Alfe pídĩ́ Nsyakɩ tyɩ́ ànɛ̂ Tadɩlɛ, 
ànɛ̂ Simolɛ, nɛ̂-á sõ̀ sú fṍ vyãh syípú sɔkɔ yɔ́lɛ, à tahlɩ Nsyudasɩ Yisikariyɔtɩ tyɩ́ nɛ̂-á pɔ pɩ pè kɔ̃ pè Yesu tɛ̃. 
Nɛ́pĩ̂ gbɔ̃nimí nɔ́pi ye Yesu tõ tɛ́ kè gbah yo pé tyɩ́ nɛ: «Yé káh kyɩ névye tĩ̀nnɛ nónó náh pɩ Nsyifunɔlɛ, ànɛ̂ yé káh de Samari tãn kwil yɔ́ ńtɛ̃ sɔkɔ. 
Yé nɛ n yuku Yisrayel no wɔ tĩ̀nnɛ, pépi nónó-á mɛ képyɛ́nɩ̀ mɔ́npunɔ tyɩ́ kɔ̃lɛ. 
Yáh kɩ n yuku wɛ̃̀kɩ̀ yõ, yé nɛ n yo n yuku nɛ, Liyel tɔ̃́rɩ́-á mɛ yĩni tɛ̃ tɛ. 
Yé yámpúlɛ n dɛ n mɔ. Yé lékyɩ̂lɛ n yilki n mɔ. Yé tótãmnɛ n dɛ n mɔ pé dú ńyân sɔkɔ, yè mó sétã̀nnɛ n yĩni n wilki névye sɔkɔ. Yáh wɛ syi nɔ̀nɔ́lɛ sɔ̃́, yé tɔ nɛ n kɔ̃ nɔ̀nɔ́lɛ sɛ̃́. 
Yé káh pɔ́ tɔkɔ dah mɔ yé yõ. 
Yé káh pley, ànɛ̂ fwɔ̀mɔlɔ ní-ònɔ́, ànɛ̂ lékyêh, ànɛ̂ lépló tɔkɔ kɔ́lɔ́ yĩnnɛ. Yé náh wɛ di, tõ̀npɩ yɩ̃nɛ ǹ tõ̀ pɩ́nɔ́ yõke yṍnɔ́lɛ. 
Yâh kyɩ de kwíl-ǹgbɛ̃ képah náh pɩ kwílpír nɛ̂ sɔkɔ, yé nɛ́gbɩ́ yah kɔ̃ yè twah tɛ̃̀ tyɩ́. Yé mó tɛ̃ tɛ̃̀ tyɩ́, fɔ́ɔ́ kè pɔ nɔ yé yálnɔ́ tyɩ́ kénɛ kwil sɔkɔ. 
Tɛ́ yâh kyɩ n de kéte nɛ̂ sɔkɔ, yé yo pé tyɩ́ nɛ yèvɩnɩ nɛ mɛ pé yõ. 
Kénɛ kéte tãn-à fɛ̃ syi yé twah pé tyɩ́, yèvɩnɩ mɛ̀ kɩ mɛ pé yõ. Ǹtɛ pàh yahle, yèvɩnɩ mɛ̀ kɩ ké syɩ pɩ yé tyɩ́lɛ. 
Yâh de gbô nɛ̂ sɔkɔ képah náh pɩ kwil nɛ̂ sɔkɔ, tɛ́ pè yahle nɛ yé káh twah pé tyɩ́, tɛ́ yahle yé wɛy nóhnɔ́lɛ, yâh n yal, yé yé gbyẽ̂h nɛ̀ngbɔ̃ kwer n kwɛ nɛ́npɔ́. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́, Liyel tukey tuke sõlɛ, Sodɔmɩ kwil ànɛ̂ Gbomɔrɩ kwil fṍhnɔ́ ńtɛ̃ náh n peke n nɔ n pi kénɛ kwil tyɩ́lɛ.» 
«Yé yah, yé mɛ á kɩ nɛ képyɛ́nɩ̀ sɔ̃́ ńmɔ n tɛkɩ n mɔ gbɛ̃́npãhlɔ nɩyṍ. Yé nɛ mɛ lékã́m névyelɛ áswɔ̃́ plɛ́y tyɩ́ kɔ̃lɛ, yé fɔkɔ mó nɛ mɛ pérpér tìtí sɔ̃́. 
Yé yé gblɔ̌ynɛ n kyɔmɩ névyelɛ dɛ́. Pè kɩ yélɛ n tɛ̃ n tuke n pɔ tukey tùkè-ǹtã̀n sɔkɔ, tɛ́ kɩ yélɛ n ko yèfõhlɛ Liyel ní-ńtã̂n vɩ̀ sɔkɔ. 
Pè kɩ yélɛ n tɛ̃ n kyɩ n pɔ kwíltãm yõ̀tãm ànɛ̂ yõ̀tãm tyɩ́ ńmɔ yĩnnɛ. Képah sɔkɔ, yé kɩ ńmɔ tyɩ́lɛ n wɛ n yo pé tyɩ́, à tahlɩ névye tyɩ́ nónó náh pɩ Nsyifunɔlɛ. 
Ǹtɛ pàh yé tɛ̃ n yuku tùkè-ǹsah, yé káh yɛ yé yáh-ńsah tyɛ yáh nɛ n yo n wil n pi sɔ̃́, ànɛ̂ yáh nɛ nɛ̂ yo n pi. Yáh nɛ nɛ̂ yo n pi, ké sõnɔ kɩ de yé sɔkɔ nɛ́npɔ́ swɔ. 
Yé náh wɛ di, yé yah yĩ́ĩ́ wɛlɔ náh yé kɩ n wɛkɩ dɛ́, Sú Liyel Mirki ye kɩ tɛ̃ yé yahlɛ yé kɩ n wɛkɩ. 
Névye kɩ pè kɔ̃ pè kɩ pé yṹnpyélɛ n tɛ̃ n ko n mɔ. Súbí tɔ kɩ sɛ̃́ntɛ̃̀nɛ n pɩ pé wɛ̃́npì tyɩ́. Wɛ̃́npì tɔ kɩ pé dúbípú tyɩ́ yahle, tɛ́ kɩ pè kɔ̃ pè kɩ pélɛ n ko n mɔ. 
Névye pól kɩ yé wɛ mɔ ńmɔ yĩnnɛ. Ǹtɛ nɛ̂-à ǹ tah sah à pɔ nɔ ké tyɛ́nɔ́ tyɩ́, tɛ̃̀ kɩ pwáhnmɔnɔ wɛ. 
Pàh yélɛ n fõh kwil yɔ́ sɔkɔ, yé sɔkɔ kèyǎh sɔkɔ. Kègbɩ yõ mé kélɛ n yo yé tyɩ́, ńmɔ nɛ̂-á Névi Pídĩ́nɛ, yé náh tɔ̃̌nɩ̀ Yisrayel kwlo pól kɔ n kɔ̃ n tɛ n pi tɛ́ mé kɩ pɔ. 
Tir sã́h-ò náh n kal n pnɛ ǹ kwɔ́-òlɛ. Tõ̀npɩ tɔ náh n kal n pnɛ ǹ yõ̀tɛ̃̀nɛ. 
Tir sã́h-ò-à pɩ ǹ kwɔ́-ò tyɩ́ kɔ̃lɛ, kè kɩ yɩ̃nɛ. Tõ̀npɩ tɔ-à pɩ ǹ yõ̀tɛ̃̀ tyɩ́ kɔ̃lɛ, kè kɩ yɩ̃nɛ. Névye-á fyɔ̀ mɛ kéte tɛ̃̀ dĩ́ ńtɛ̃nɛ n gbyɔ Bɛlsebulilɛ, (nɛ̂ kɔ́r-á sétã̀n yõ̀tɛ̃̀nɛ), ǹ kéte tãn wɔ se pyě pè kɩ yɛ gbyɔ̀npekenɔlɛ? 
Kǎh pɩ sɛ̃́, yé káh névye yah tíkílɛ n pɩ. Yé náh wɛ di, tyípǐl yɔ́ ńtɛ̃ náh n wũ n pi wìlkìnmɔ-ńkɛ̃̂nɛ. Sètĩ̀n sõ̀ tir yɔ́ ńtɛ̃ náh n wũ n pi pɩ̃́-ńkɛ̃̂nɛ. 
Máh tyi nónónɛ n yo yé tyɩ́ fɛ̃́ntũ̂ sɔkɔ, yé sélɛ n yo n wilki n mɔ tyah kyòo yah. Tyi nónó-á n plɔ n yo yé tyɩ́ yé níkílɔ́ yey, yé sé kwnínɛ n kɔ̃ gbɛ̃́ntĩ̂n yõ. 
Yé káh névye yah tíkílɛ n pɩ, nónó-á névyelɛ n ko mɔ n pnɛ, tɛ́ pè náh fɛ̃ ǹ min ko mɔ n pi. Ǹtɛ Liyel nɛ̂-á kɩ fɛ̃ névi wil gbɛ̃́nsê ànɛ̂ ǹ min tɔ min dah mɔ ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ, yé ǹmɔ wɔ tíkílɛ nɛ n pɩ. 
Ǹtɛ̃̀npînɔ́ nínɔ́ n yãm pɔ́píkí pɔ́ núkú ólɛ. Ǹtɛ sé sɔkɔ yɔ́ ńtɛ̃ náh n syɩkɩ sétáh, nɛ́ yé Sú Liyel náh ké wɛ̃̀kɩ̀ kɔ̃. 
Ǹtɛ yépi tɛ̃̀ wɔlɛ, yé yõ̀yén ńtɛ̃ pól dan mɛ pɩ̃́nɔ́lɛ Liyel tyɩ́. 
Kǎh pɩ sɛ̃́, yé káh tíkílɛ n pɩ, yépi tyɩ́ náh tah kal ǹtɛ̃̀npî kwĩnki tyɩ́lɛ di? 
Névi nɛ̂-à kè yo dal mɔ névye yĩ́ yah nɛ páh mɛ ńmɔ tyɩ́ yõ, ńmɔ tɔ kɩ kè yo dal mɔ ń Sú yĩ́ yah nɛ̂-á mɛ yĩ̂nyõ, nɛ tɛ̃̀-á mɛ ńmɔ tyɩ́ yõ. 
Ǹtɛ nɛ̂-à yahle névye yĩ́ yah nɛ pé náh mɛ ńmɔ tyɩ́ yõ, ńmɔ tɔ kɩ yahle ń Sú yĩ́ yah nɛ̂-á mɛ yĩ̂nyõ, nɛ tɛ̃̀ náh mɛ ńmɔ tyɩ́ yõ.» 
«Yé káh wɛ nɛ yèvɩnɩ-á mé pɔ mɔ́-ńsah sétáh yõ dɛ́. Yèvɩnɩ náh mé pɔ mɔ́-ńsah sétáh yõ, gbɛ́kɩ́nɔ́ wɔ ye pyě. 
Mé pɔ dĩ́ ànɛ̂ ǹ sú wrɔ́ gbɛ́kɩ́-ńsah; sênpir ànɛ̂ ǹ yṹ wrɔ́ gbɛ́kɩ́-ńsah; sê ànɛ̂ ǹ pɛ yṹ wrɔ́ gbɛ́kɩ́-ńsah. 
Névi gbô tãn ye kɩ pɩ ǹ sépɛ́bɩ́lɛ. 
Névi nɛ̂ sú, képah náh pɩ ǹ yṹ tyɩ́-à nɔ ǹ tyɩ́ à kal ńmɔlɛ, tɛ̃̀ náh yɩ̃nɛ à nɛ n mɛ ńmɔ tyɩ́ yõ. Névi nɛ̂ pídĩ́, képah náh pɩ ǹ písê tyɩ́-à nɔ ǹ tyɩ́ à kal ńmɔlɛ, tɛ̃̀ náh yɩ̃nɛ à nɛ n mɛ ńmɔ tyɩ́ yõ. 
Névi nɛ̂-à ǹ syɩ syɩ dahnɩ twah ń sõ̀ tɛ̃́-ńkɛ̃̂nɛ, tɛ̃̀ náh yɩ̃nɛ à nɛ n mɛ ńmɔ tyɩ́ yõ. 
Névi nɛ̂-á nɛ páh pé minnɛ n koh n sah, tɛ̃̀ kɩ tɛkɩ tyɛ́-ńkɛ̃̂ min sɔkɔ. Tɛ́ nɛ̂-á tɛkɩ ǹ min sɔkɔ ńmɔ yĩnnɛ, tɛ̃̀ wɔ kɩ tyɛ́-ńkɛ̃̂ min wɛ.» 
«Névi nɛ̂-à yé tɛ̃ plɛ, ńmɔ ye tɛ̃̀ tɛ̃ plɛ. Nɛ̂ tɔ-à ńmɔ tɛ̃ plɛ, Sú Liyel nɛ̂-á tɛkɩ mɔ ńnɛ, ǹmɔ ye tɛ̃̀ tɛ̃ plɛ. 
Névi nɛ̂-à Liyel tɛ̃ntõ̀ syínyo-ò tɛ̃ plɛ ǹ pɩ́nɔ́ yĩnnɛ Liyel tɛ̃ntõ̀ syínyo-òlɛ, Liyel tɛ̃ntõ̀ syínyo-ò dyah dyah kwâh kénkɔ̃ ye kɩ dyah tɛ̃̀nɛ. Névi nɛ̂ tɔ-à nɛ́gbɩ́ tɛ̃ plɛ ǹ pɩ́nɔ́ yĩnnɛ nɛ́gbɩ́lɛ, nɛ́gbɩ́ dyah dyah kwâh kénkɔ̃ ye kɩ dyah tɛ̃̀nɛ. 
Kègbɩ yõ mé kélɛ n yo yé tyɩ́ nɛ, névi nɛ̂-à nívɩ́nɔ́ ńgbíyĩ́nɛ ńmɔ tõ̀ sã́hpú sɔkɔ yɔ́ ńtɛ̃ fyé kɔ̃ à wɔ ǹ pɩ́nɔ́ yĩnnɛ ńmɔ tõ̀ sã́h-òlɛ, tɛ̃̀ náh n tɛkɩ n pi ké lédyáhnɩ́ sɔkɔ póllɛ.»
