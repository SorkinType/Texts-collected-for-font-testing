Matye.
3.
Légbĩ́nɛ, Wèlnwìlkì-ò Nsyɔ̃ pɔ Nsyude mara gbɛ̃́nnɩ sɔkɔ, à pɔ tyah Liyel wɛynɛ n yo. 
À sõ̀ mɛ yo tũ nɛ: «Yé yé tyípêl pɩ́nɔ́ yal mɔ. Yé náh wɛ di, Liyel tɔ̃́rɩ́ mɛ yĩni tɛ̃ tɛ.» 
Liyel tɛ̃ntõ̀ syínyo-ò Yesayi-á lésõ nɛ̂ yo sah nɛ ńkɛ̃́: «Ǹnɩ̂ yɔ́ yĩ́npɔ̃́ mɛ n dɔkɔ gbɛ̃́nnɩ sɔkɔ nɛ ńkɛ̃́: ‹Yé Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ tɔ̃ sah. Yé ǹ sɔ́kɔ́-ńtã̂n pɩ tómtóm.» Nsyɔ̃ yĩ́npɔ̃́ tyɩ́ ye lésõ à n yo. 
Yã̀hmɩ́kɩ̀ yɛ̀ fwɔ̀mɔnɩ ye sõ̀ Nsyɔ̃ wil, tɛ́ à mɛ ǹ syɩ̃ tɛ̃ sêl kɔ̃lɛ. Sègblo ànɛ̂ syɩ̃rɩ ye sõ̀ ǹ yõkelɛ. 
Képah tɛ̃̀nɛ, Yerusalɛmɩ kwil tãn, ànɛ̂ Nsyude mara pól névye, ànɛ̂ Nsyurdɛ̃ yɔ gbɛ̃̀nyah pól névye sõ̀ mɛ n wil n pɔ ǹ tyɩ́. 
Pè sõ̀ mɛ pé tyípêl pɩ́nɔ́lɛ n yo n pɔ no pól yĩ́ yah, tɛ́ Nsyɔ̃ mɛ pélɛ n wel n wilki Nsyurdɛ̃ yɔ sɔkɔ. 
Képah sɔkɔ, Nsyɔ̃-á kè wɛ Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ Sadusi wɛ̃̀kɩ̀ kɔ́pú náhnáh mɛ n pɔ n pɔ à kɔ̃ à pélɛ n wel n wilki sɛ̃́, à mɛ yo pé tyɩ́ nɛ, wáhnmɩ́rɔ́ dùkù nɔ́pi! Nɛ̂ se kwɔ pélɛ nɛ pǎh kɩ fɛ̃ to pwah Liyel fɔ̀kɔ̀vyã̀hnɩ̀ yah, nɛ̂-á n pi nɩ? 
Pè tyípéplɔlɛ n pɩ, nónó-á n wɛ̃kɩ n pi nɛ pǎh pé tyípêl pɩ́nɔ́ yal mɔ. 
Tɛ́ pè káh wɛ nɛ páh kɩ n sõ pé fɔkɔnɩ sɔkɔ nɛ Abrahmɩ-á pé súlɛ, képah-á kɩ yɩ̃nɛ dɛ́. Páh kélɛ n yo pé tyɩ́, dyêh nónó-á yɔ̀, Liyel-á kɩ fɛ̃ sè vi mɔ pɩ Abrahmɩ pyelɛ. 
Tɛ́ man ńtɛ̃-á mɛ yáhnsàhntɛnɔlɛ sésân fɛ̃ yĩnnɛ sé sémîn sɔkɔ. Séswɔ̂ nɛ̂-à n syí-ńkɛ̃̂nɛ pye pèplɔlɛ, kǎh kɩ ko min dah mɔ nyàh sɔkɔ. 
Péwɔ tɛ̃̀ wɔ-á pélɛ n wel n wilki ni sɔkɔ, nɛ̂-á kɩ kè wɛ̃kɩ nɛ pǎh pé tyípêl pɩ́nɔ́ yal mɔ. Ǹtɛ névi nɛ̂-á kɩ n pɔ n pi péwɔ náhlɛ, tɛ̃̀ wɔ tɛ̃́nwɛnɔ-á kal péwɔ tyɩ́lɛ. Páh fṍrɔɔ kah ǹ lékyêh ńtɛ̃ dɛ́nɔ́lɛ. Tɛ̃̀ wɔ-á kɩ pɔ pè wel wilki Liyel Mirki ànɛ̂ nyàh sɔkɔ. 
Ǹ sèmukɔ fwo fwo kwâh-á mɛ ǹ gbõ̀. Wǎh kɩ ǹ táh tɔ̃ sah, kɩ ǹ sèmukɔ fwo, tɛ́ kɩ ké pɔ́rɩ́ kwel mɔ dúkúrí sɔkɔ. Ǹtɛ sé yésyɩ̂ wɔlɛ, wǎh kɩ sépi sukɛ ní-ńkɛ̃̂ nyàh sɔkɔ. 
Képah tɛ̃̀nɛ, Yesu wil Kalile mara sɔkɔ à pɔ Nsyɔ̃ tyɩ́ Nsyurdɛ̃ yɔ́ngbalɩ sɔkɔ, nɛ pé pɔ à kɔ̃ à pé wel wilki ni sɔkɔ. 
Ǹtɛ Nsyɔ̃ mɛ yahle tɛ́ nɛ, péwɔ-á yɩ̃nɛ pé pɔ ǹmɔ kɔ̃ à pé wel wilki ni sɔkɔ, tɛ́ ǹmɔ pyě pɔ péwɔ tyɩ́ nɩ? 
Ǹtɛ Yesu mɛ yo ǹ tyɩ́ nɛ: «Yɛ kè pɩ sɛ̃́ ǎtrɛ. Á náh wɛ di, ké yah ye Liyel-á nɛ̂ yah kɔ̃, ápi kè pɩ.» Wǎh képah yo sɛ̃́, Nsyɔ̃ mɛ fɛ̃ syi. 
Yesu wèlnwìlkìntɛnɔ ǹsɔ̃̀, à wil ni sɔkɔ. Nɛ́npɔ́ swɔ yĩ̂nyõ yɩkɩ mɔ, tɛ́ à mɛ Liyel Mirki wɛ kè mɛ n tiki n pi kèpíkípíkí tyɩ́ kɔ̃lɛ, à pɔ tɛ̃ ǹ yõ. 
Képah sɔkɔ, yĩ́npɔ̃́ yɔ́ núkú n wil yĩ̂nyõ nɛ ńkɛ̃́: «Ńmɔ nɩ̀dɛ̀ Pídĩ́ ye yɔ̀. Ǹmɔ ye ń nɩ̀vɩ̀nɩ̀ yõlɛ.»
