Matye.
12.
Légbĩ́ mɛ̀ ó sɔkɔ, mɩ mɩ yòh sõ yɔ́lɛ Yesu ànɛ̂ ǹ tõ̀ sã́hpú sõ̀ mɛ n kah sèmukɔ swɛ̃y yísyɔ́ sɔkɔ. Tɛ́ kwɛkɩ sõ̀ mɛ ǹ tõ̀ sã́hpú tɛ̃. Kǎh pɩ sɛ̃́, pè mɩ de sèmukɔ ywɔ̃ kɔ̃̀lɩ̀ntahnɔ sɔkɔ. 
Farisi wɛ̃̀kɩ̀ kɔ́pú-á képah wɛ sɛ̃́, pè mɛ yo Yesu tyɩ́ nɛ, tir nɛ̂ pɩ́nɔ́ wɛ̃̀kɩ̀ náh mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sõlɛ, à nɛ n yah ǹ tõ̀ sã́hpú kélɛ n pɩ lɛ́. 
Ǹtɛ Yesu mɛ yo pé tyɩ́ nɛ: «Kwɛkɩ-á lésõ yõ̀tɛ̃̀ dĩ́ Davidi ànɛ̂ ǹ náh névye tɛ̃, wǎh nɛ̂ pɩ, yé náh kè kar yah di? 
Búrn nónó-á lésõ mɛ sàhnɔ́lɛ Liyel tyɩ́ yõ Liyel wɔ̀kɔ̀ sɔkɔ, à náh lésõ de ké sɔkɔ à sè tɔkɔ tah ànɛ̂ ǹ náh névyelɛ di? Tɛ́ à wil sárká syínwìlkìpu tyɩ́ sɔkɔ, nɛy náh lésõ yɩ̃nɛ sé táhnɔ́lɛ. 
Yé se kɩ nɛ yé náh kè kar yah tɔlɛ nɛ, sárká syínwìlkìpu-á ńkɛ̃́nɛ mɩ mɩ yòh dêllɛ n pɩ Liyel ní-ńsah gbô sɔkɔ mɩ mɩ yòh sõnɔ́lɛ, tɛ́ lékàh náh mɛ pé tyɩ́? 
Ǹtɛ mé kélɛ n yo yé tyɩ́, névi mɛ ásõ̂ nɛ̂-á kal Liyel ní-ńsah gbôlɛ. 
Liyel kè yo nɛ ńkɛ̃́: ‹Sárká súkɛ́nkɔ̃nɔ náh mé n yah n kɔ̃ dɛ́, wɛ̃ yãm yáhnɔ́ ye mé n yah n kɔ̃.› Yâh sõ̀ mɛ képah kɔ́rlɛ n pnɛ, yé náh náhkɩ lékàhlɛ lékàh kɛ̃́ névye kɔ̃ n pi. 
Yé náh wɛ di, Névi Pídĩ́ ye mɩ mɩ yòh yõ̀tɛ̃̀nɛ dò.» 
Yesu-á yal nɛ́npɔ́, à sɔkɔ pé Liyel ní-ńsah wɔ̀kɔ̀ yɔ́ sɔkɔ. 
Gbõ̀nkù dĩ́ yɔ́ sõ̀ mɛ nɛ́npɔ́. Tɛ́ pè sõ̀ mɛ ké tyɩ́ nɛ pé lékàh wɛ Yesu kɔ̃. Kǎh pɩ sɛ̃́, pè Yesu piki yah nɛ ńkɛ̃́, yɔ́nwó dɛ́nmɔnɔ wɛ̃̀kɩ̀ se mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sõlɛ nɩ? 
Ǹtɛ à mɛ yo pé tyɩ́ nɛ: «Yé sɔkɔ yɔ́ tyɩ́ képɔ́ núkú ó-à mɛ, tɛ́ à sɔkɔ de pwã̀hsǔkù yɔ́ sɔkɔ mɩ mɩ yòh sõlɛ, tɛ̃̀ se kɩ yɛ ǹ kyɩ́nwìlkìnɔ́lɛ lésõlɛ? 
Ǹtɛ névi náh fṍnfṍ kal náhnáh képɔ́ yõ di? Kǎh pɩ sɛ̃́, pèpɔrɩ pɩ́nɔ́ wɛ̃̀kɩ̀ mɛ kɔ̃́nɔ́lɛ mɩ mɩ yòh sõlɛ.» 
Képah tɛ̃̀nɛ, tɛ́ yo gbõ̀nkù dĩ́ mɛ̀ tyɩ́ nɛ: «Á gbõ̀ tɔ̃.» À mɛ kè tɔ̃, kè núkú pɩ vilɛ ké ní-ò mɛ̀ sɔ̃́. 
Ǹtɛ Farisi wɛ̃̀kɩ̀ kɔ́pú-á fyɔ̀ wil Liyel ní-ńsah wɔ̀kɔ̀ sɔkɔ, pè kyɩ wɛ̃ wɛ à vì-ǹsah yah Yesu wɛ n ko n mɔ yĩnnɛ. 
Yesu-á pɔ pɩ̃ nɛ Farisi wɛ̃̀kɩ̀ kɔ́pú-á mɛ pé kònmɔ-ńsahlɛ n yah n kɔ̃, à yal nɛ́npɔ́. Névye náhnáh mɛ ǹ sõ̀ tɛ̃. À mɛ pé sɔkɔ yámpú pól dɛ mɔ, 
tɛ́ pè gbah sni nɛ pè káh pé tyɩ́ yãh nɛy tyɩ́. 
Yesu-á képah pɩ, Liyel-á lésõ nɛ̂ yo sah ǹ tɛ̃ntõ̀ syínyo-ò Yesayi tyɩ́, kè wil. À lésõ nɛ ńkɛ̃́: 
«Máh nɛ̂ yah wilki nɛ à ń tõ̀lɛ n pɩ, à wɛ yɔ̀. Ń nɩ̀dɛ̀ nɛy ye ǹnɛ. Ǹmɔ ye ń nɩ̀vɩ̀nɩ̀ yõlɛ. Mé kɩ ń Mirki tiki mɔ ǹ yõ, à kɔ̃ à kɩ gbɩ wɛ̃̀kɩ̀ tyɩ́ yo yìkìnì pól névye tyɩ́. 
À náh yékã̂b pɩ n pi nɛynɛ, à náh n gbo n mɔ n tũ n pi, nɛy náh ǹ yĩ́npɔ̃́ noh n pi tɔ́wû tùkè-ǹtã̀n sɔkɔ. 
Fílkì nɛ̂-á mɛ kwɔ́nmɔntɛ̃̀nɔ́lɛ, à náh ké tɛ́y kwɔ n pi. Fɔ̀tnɔ̀ nɛ̂-á mɛ n tyah nínɔ́lɛ, à náh ké tɛ́y ni n mɔ n pi. Sɛ̃́ ye à kɩ n pɩ, pópó kɩ pɔ nɔ gbĩ́ nɛ̂-á gbɩ wɛ̃̀kɩ̀ kɩ yah syi. 
Tɛ́ yìkìnì pól névye gbõ̀ sàhnɔ́ kɩ mɛ ǹmɔ yõ.» 
Képah náh, nósyɔ́ dĩ́ yɔ́ tɛ̃ pɔ Yesu tyɩ́. Sétõ sõ̀ mɛ ǹ sɔkɔ à kɔ̃ à pɩ yɩ̃́tũ̂ ànɛ̂ kèdɔ̀kɩ̀lɛ. Yesu mɛ à dɛ mɔ, à kɔ̃ tɛ̃ wɛ n wɛkɩ, tɛ́ fɛ̃ n yah. 
Képah tɛ̃̀nɛ, kè pɩ gbɛ̃̀ndùnìnɛ tɔ́wû pól tyɩ́, pè mɛ tyah n yo wɛ̃ tyɩ́ nɛ: «Tɛ́ Davidi Pídĩ́ náh yɔ̀ lɛ́?» 
Ǹtɛ Farisi wɛ̃̀kɩ̀ kɔ́pú-á képah tɛ̃̀ noh sɛ̃́, pè nɛ sétã̀n yõ̀tɛ̃̀ Bɛlsebuli ó gbõ̀ yõ-á à n yĩni sétã̀nnɛ. 
Tɛ́ Yesu-á pé fɔkɔ sõnɔ wɛ sɛ̃́, à yo pé tyɩ́ nɛ: «Yõ̀tɔ̃rɩ́ nɛ̂ tɛ̃̀ névye-à fyɔ̀ yuku wɛ̃nɛ n fwo, kénɛ yõ̀tɔ̃rɩ́ kɩ yɩkɩ. Kwil nɛ̂ névye-à yuku wɛ̃nɛ n fwo, kè náh n wũ n pi. Sɛ̃́ ó ye, kéte nɛ̂ névye-à yuku wɛ̃nɛ n fwo, ké tɔ náh n wũ n pi. 
Tɛ́ sétã̀n yõ̀tɛ̃̀-à tyah ǹ nolɛ n yĩni, ǹ gblɔ̌y náh à n fwo di? Ǹ yõ̀tɔ̃rɩ́ se kɩ wũ? 
Tɛ́ ńmɔ-à mɛ sétã̀nnɛ n yĩni Bɛlsebuli gbõ̀ yõ yé yo à; yépi tĩ̀nkì névye wɔ-á dò, nɛ̂ gbõ̀ yõ mɛ pépi n yĩni sélɛ wɛ́? Pépi ńtɛ̃ ye kɩ lékàhlɛ yé kɔ̃. 
Liyel Mirki gbõ̀ yõ-à mé n yĩni sélɛ, képah n wɛ̃kɩ nɛ ńkɛ̃́, Liyel tɔ̃́rɩ́-á yĩni tɛ̃ yé tyɩ́. 
Tǎhkɩ̀-á mɛ dĩ́ nɛ̂ sɔkɔ, nɛ̂ kɩ fɛ̃ ǹ púkúbí-ńkɛ̃̂ sɔkɔ, kɩ de ǹ wɔ̀kɔ̀ sɔkɔ kɩ ǹ gbã̀n yî kwel? Máhah wǎh kɩ à pukubi tɛ́ kɩ pyě de ǹ wɔ̀kɔ̀ yî kwel. 
Névi nɛ̂-à mɔ́-ńkɛ̃̂nɛ ńmɔlɛ, tɛ̃̀ mɛ ńnɛ n fwo. Névi nɛ̂ náh mɛ ńnɛ n yohnɩ n mɔ tùkènmɔnɔ sɔkɔ, tɛ̃̀ mɛ kárkɩ́nɔ́lɛ n pɩ. 
Képah ye, mé kélɛ n yo yé tyɩ́ nɛ: ‹Névye tyípêl pól ànɛ̂ pé Liyel gbyɔ̀nɔ́ pól kɩ fɛ̃ yɔ̃ mɔ pè kɔ̃. Ǹtɛ Liyel Mirki gbyɔ̀nɔ́ tɛ̃̀ wɔ náh n yɔ̃ mɔ n pi pé tyɩ́ póllɛ. 
Névi nɛ̂-à yo de Névi Pídĩ́ tyɩ́, kè kɩ fɛ̃ yɔ̃ mɔ tɛ̃̀ kɔ̃. Ǹtɛ névi nɛ̂-à yo de Liyel Mirki tɛ̃̀ wɔ tyɩ́, kè náh yɔ̃ mɔ tɛ̃̀ kɔ̃ n pi kwɛynɛ, kè náh yɔ̃ mɔ tɛ̃̀ kɔ̃ n pi ńsõ̂ tɔlɛ.› 
Yé yah, séswɔ̂ n pnɛ ké pye dùkù sɔkɔ. Yâh nɛ séswɔ̂ nɛ̂-á mɛ plɛ, yé nɛ ké pye tɔ-á mɛ plɛ. Yâh nɛ séswɔ̂ nɛ̂ náh mɛ plɛ, yé nɛ ké pye tɔ-á mɛ kétúkùyɔ̂. 
Wáhnmɩ́rɔ́ nɔ́pi, tyípéplɔ se kɩ fɛ̃ n wil yé vyɛ̃y sɔkɔ tɛ́ yé nnɔ mó tɔ̃ mɛ yúkúnɔ́lɛ pèkèlɛ? Névi nɩ-á mɛ yúkúnɔ́lɛ tyi nónónɛ, sépi ye n wil tɛ̃̀ vyãh sɔkɔ dò. 
Nɛ́pépɛ̌y tyípéplɔ n wil ǹ nɩ sɔkɔ. Nɛ́túkù tyítúkúnì tɔ mó n wil ǹ nɩ sɔkɔ. 
Ǹtɛ mé kélɛ n yo yé tyɩ́, yah kɛ̃́ wɛlɔ nónó pól-á névye n wɛkɩ, sé ywɔ̃ kɩ piki n yah n pi pé tyɩ́ Liyel tukey tuke sõlɛ. 
Tukey kɩ tuke á yõ à yɩ̃nɛ á vyãh wɛynɛ kɩ gbɩlɛ ǹ kɔ̃, képah náh pɩ kɩ lékàhlɛ ǹ kɔ̃. 
Képah sɔkɔ, Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́pú yísyɔ́ mɛ Yesu syah nɛ, pé kwɔ́-ò dĩ́, páh mɛ ké tyɩ́ nɛ à gbǐl yɔ́ pɩ pé yah, nɛ̂-á n wɛ̃kɩ nɛ wǎh n wil Liyel tyɩ́. 
À mɛ yo pé tyɩ́ nɛ: «Kwɛy gbĩ névye náh plɛ. Pè náh mɛ tɛ̃ Liyel ó tyɩ́ yõ. Pè mɛ ké tyɩ́ nɛ mé gbǐl yɔ́ pɩ pé yah. Ǹtɛ kèyǎh ńtɛ̃ náh pɩ pè wɛ̃kɩ n pi, à wil Liyel tɛ̃ntõ̀ syínyo-ò Nsyonasɩ tyɩ́ sɔkɔ. 
Yé yah, sɔ̃́-á lésõ Nsyonasɩ sèpĩ̀n tɔ̃́nɩ̀ pɩ yɛ̃́nkɔ̂-ǹgbɛ̃̀kɩ̀ yɔ́ fɔkɔ sɔkɔ, sɛ̃́ ye Névi Pídĩ́ tɔ kɩ sèpĩ̀n tɔ̃́nɩ̀ pɩ sétáh sõwãh. 
Nsyonasɩ-á lésõ Liyel tɛ̃ntõ̀ yo Ninivi kwil tãn tyɩ́, pè fɛ̃ syi pé tyípêl pɩ́nɔ́ yal mɔ. Ǹtɛ nɛy mɛ ásõ̂ nɛ̂-á kal Nsyonasɩlɛ. Kǎh pɩ sɛ̃́, Liyel tukey tuke sõlɛ, Ninivi kwil tãn kɩ yuku kwɛy gbĩ névye yõ, kɩ lékàhlɛ pè kɔ̃. 
Pi nɛ̂-á lésõ yõ̀tɛ̃̀nɛ Saba sétáh yõ, à lésõ wil fɩ́ɩ́ sétáh yĩ̀n tyɩ́, à pɔ yõ̀tɛ̃̀ dĩ́ Salomo lékã́m wɛlɔ noh. Ǹtɛ nɛy mɛ ásõ̂, nɛ̂-á kal Salomolɛ. Kǎh pɩ sɛ̃́, Liyel tukey tuke sõlɛ, kénɛ pi kɩ yuku kwɛy gbĩ névye yõ, kɩ lékàhlɛ pè kɔ̃. 
Sétõ-à wil névi sɔkɔ, à n kyɩ gbɛ̃́nnɩ kɔ n kɔ̃, ǹ mɩ̀-ǹsah yah n kɔ̃. Ǹtɛ wàh wɛ́-ńkɛ̃̂nɛ, 
à n yo ǹ gblɔ̌y tyɩ́ nɛ, páh sõ̀ wil ńyãh sɔkɔ, páh kɩ pé syɩ yuku nɛ́npɔ́. Wàh ǹ syɩ, tɛ́ kyɩ ǹ tɛ̃̀-ǹsah dyàh wɛ ké kɔ́kɔ́lɛ yàlntɔ̃nsàhnɔ́lɛ, 
à n kyɩ sé no sétã̀nnɔ́ kwǎrninɔ syah, nónó-á peke kal ǹmɔlɛ. Sé tɔ̂ɔ́ n kyɩ n de n tɛ̃ tɛ̃̀ sɔkɔ. Kàh pɩ sɛ̃́, tɛ̃̀ mɛ̀ dùkù ké syɩ n peke n kal ǹ gben tɛ̃̀nɛ. Sɛ̃́ ye kwɛy gbĩ nɛ́pêl tɔ tyɩ́ kɩ n pɩ n pi. 
Yesu-á mɛ képah yónɔ́ sɔkɔ tɔ́wû tyɩ́, ǹ yṹ ànɛ̂ ǹ yṹnpyé pɔ yĩn kégbɔ́ tɛ́ mɛ ké tyɩ́ nɛ pé yo mɔ ǹ tyɩ́. 
Ǹnɩ̂ mɛ yo ǹ tyɩ́ nɛ à noh yɔ̀, ǹ yṹ ànɛ̂ ǹ yṹnpyé-á mɛ yĩ́ntɛ̃̀nɔ́lɛ kégbɔ́. Pǎh mɛ ké tyɩ́ nɛ pé yo mɔ ǹ tyɩ́. 
Yesu mɛ yo tɛ̃̀ tyɩ́ nɛ: «Nɛ̂ mɛ ńmɔ nɔ̂lɛ, ànɛ̂ nónó mɛ ń yṹnpyélɛ?» 
Tɛ́ ǹ gbõ̀ nɔ mɔ ǹ tõ̀ sã́hpú tyɩ́ tɛ́ nɛ ńkɛ̃́: «Ńmɔ nɔ̂ ànɛ̂ ń yṹnpyé wɛ yɔ̀. 
Yé yah, ńmɔ Sú nɛ̂-á yĩ̂nyõ, wǎh mɛ nɛ̂ tyɩ́, névi nɛ̂-à képahlɛ n pɩ, tɛ̃̀ wɔ ye ńmɔ yṹnpilɛ, képah náh pɩ ń nɔ̂lɛ.»
