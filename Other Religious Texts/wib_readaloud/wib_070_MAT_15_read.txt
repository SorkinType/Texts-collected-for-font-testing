Matye.
15.
Képah sɔkɔ, Farisi wɛ̃̀kɩ̀ kɔ́pú ànɛ̂ Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú wil Yerusalɛmɩ kwil sɔkɔ à pɔ Yesu tnɔ̂, tɛ́ à syah nɛ ńkɛ̃́, 
sɔ̃́ se pɩ tɛ́ ǹ tõ̀ sã́hpú náh mɛ n kɔ lékɔ̃lɔ́ wã̀ldì yõ nɩ? À nɛ n yah pè yãrlɛ n yõ, tɛ́ pè náh se mó tɔ̃ pé gbã̀n wu à yɩ̃nɛ pé wã̀ldìlɛ lɛ́. 
À mɛ yo pé tyɩ́ nɛ: «Tɛ́ yépi wɔ-á dò, sɔ̃́ pɩ tɛ́ yé wã̀ldì yĩnnɛ, yé náh mɛ n kɔ Liyel tyi kõ̀nsàhnɔ́ wã̀l yõ? 
Yé yah, Liyel kè yo nɛ ńkɛ̃́: ‹Yõ̀yénnɛ n sah á sú ànɛ̂ á yṹ yõ.› Tɛ́ tɔ̃ kè yo nɛ ńkɛ̃́: ‹Névi nɛ̂-à wɛ̀ntùkù wɛkɩ ǹ sú tyɩ́, képah náh pɩ ǹ yṹ tyɩ́, tɛ̃̀ yɩ̃nɛ kònmɔnɔlɛ.› 
Ǹtɛ yépi wɔ n mó nɛ, névi nɛ̂-à yo ǹ sú képah náh pɩ ǹ yṹ tyɩ́ nɛ, kwâh nɛ̂-á sõ̀ mɛ pé tyɩ́ ǹ yohnɩ n mɔ mɔ tɛ̃̀nɛ, páh képahlɛ Liyel kɔ̃ tɛ, 
ǹ dúbí-ò mɛ̀ yòhnɩ̀nmɔnɔ kɔ̃lɩ náh tɔ̃ mɛ ǹ yõ. Kàh pɩ sɛ̃́ à, Liyel tyi kõ̀nsàhnɔ́ náh yé yɩkɩ yé wã̀ldì yĩnnɛ di? 
Vyã́hsesyɩ̃́rɩ́ névye nɔ́pi, Liyel-á lésõ Yesayi kɔ̃ à nɛ̂ yo sah, képah ye yépi n pɩ yɔ̀. À lésõ nɛ ńkɛ̃́: 
«No nɔ́pilɛ, vyãh ó sɔkɔ ye pè n gbilki ńmɔlɛ. Ǹtɛ pé nnɔ mɛ fɩ́ɩ́ ńnɛ. 
Pǎh kwɔ́nɔ́lɛ n pɩ tyi nónó yõ, névye yah yĩ́ĩ́ tyi kõ̀nsàhnɔ́ ye sélɛ. Képah ye, gbílkínɔ́ nɛ̂-á pè n gbilki n pɔ ńnɛ pè yo, nɔ̀nɔ́ ye kélɛ.» 
Képah sɔkɔ, Yesu tɔ́wû mɛ̀ ye tuke tɛ́ yo pé tyɩ́ nɛ: «Yé yé ní kwɔ n noh pallɛ, yè wɛlɔ nɔ́pi syɩ́kɩ́-ńsah pɩ̃. 
Yî nónó-á n de névi vyãh sɔkɔ, sépi náh n silki ǹnɛ Liyel yah dɛ́. Wɛlɔ nónó-á n wil ǹ vyãh sɔkɔ, sépi ye n silki ǹnɛ.» 
Képah tɛ̃̀nɛ, ǹ tõ̀ sã́hpú mɛ nɔ de ǹ tnɔ̂ tɛ́ yo ǹ tyɩ́ nɛ, à kó pɩ̃ nɛ ǹ wɛlɔ nónó-á sõ̀ sé yey mɔ Farisi wɛ̃̀kɩ̀ kɔ́pú tĩ̀nnɛ, sǎh pè kah yɔ nɩ? 
À mɛ yo pé tyɩ́ nɛ: «Yèh nónó pól tɛ̃̀-á fyɔ̀ mɛ, tɛ́ ń Sú nɛ̂-á yĩ̂nyõ ǹmɔ sṍnɔ́ náh sélɛ, sè kɩ viki to. 
Yé yɛ pé yõ. Yɩ̃́tũbi ye pélɛ, nónó-á mɛ pé no yɩ̃́tũbi yahlɛ n tɛ̃. Tɛ́ yɩ̃́tũ̂-à mó tɔ̃ ǹ ní-ò yɩ̃́tũ̂ yah tɛ̃, pé nímí kɩ syɩkɩ de pwã̀h yɔ́ sɔkɔ.» 
Képah sɔkɔ, Pyɛrɩ mɛ yo ǹ tyɩ́ nɛ, wǎh sõ̀ nɛ pé pé ní kwɔ wɛlɔ nónó noh pallɛ, à sé kɔ́r yo pé syah. 
À mɛ pè syah nɛ: «Lékã́m kɛ̃́ névye se yépi tɔlɛ? 
Yé náh pɩ̃ nɛ yî nónó pól-á n de vyãh sɔkɔ, sǎh ńkɛ̃́nɛ n syɔ̃ kèkùrkì sɔkɔ, ǹ tɛ́ n pyě n pɔ n wilki n kwɛ kèdúkúdahkɩ sɔkɔ di? 
Ǹtɛ wɛlɔ nónó-á n wil vyãh sɔkɔ, sé sõ n wil sõnɔ sɔkɔ. Sépi ye n silki névilɛ Liyel yah. 
Névi lékã́m sɔkɔ ye sõnɔ túkúnì n wil, ànɛ̂ nɛ́kókè mɔ́nɔ́lɛ, ànɛ̂ fɔ̀fɩ́kɩ̀ pɩ́nɔ́lɛ, ànɛ̂ prɔ̀prɔ̀ pɩ́nɔ́lɛ, ànɛ̂ yɩ̃belɛ, ànɛ̂ mɩ̀nɩ̀ yónɔ́lɛ, ànɛ̂ gbyɔlɛ. 
Kénɛ tyi wɔ ye n silki névilɛ Liyel yah. Ǹtɛ névi-à ǹ gbã̀n wú-ńkɛ̃̂nɛ à yɩ̃nɛ wã̀ldì-á kè yo sɔ̃́ tɛ́ yõke yõ, képah náh n silki tɛ̃̀nɛ.» 
Képah náh, Yesu yal nɛ́npɔ́ tɛ́ sɔkɔ Tiri ànɛ̂ Sidõ sétáh yõ. 
Tɛ́ pi yɔ́ sõ̀ mɛ nɛ́npɔ́ nɛ̂-á Kanah nɛynɛ. Ǹmɔ mɛ pɔ Yesulɛ yúkɛy tɛ́ mɛ n de n syi, mɛ yo tũ ǹ tyɩ́ nɛ pé Tɛ̃̀ ǹgbɛ̃, ǹmɔ nɛ̂-á Davidi Pídĩ́nɛ, à pé yãm yah. Ńkɛ̃́ sétõ yɔ́-á mɛ pé písê sɔkɔ mɛ ǹnɛ n kwlɔ képékèyɔ̂. 
Ǹtɛ Yesu náh dahbɩ wilki ǹ tyɩ́. Ǹ tõ̀ sã́hpú mɛ nɔ de ǹnɛ tɛ́ kè yah kɔ̃ ǹ tyɩ́ nɛ, à pi mɛ̀ kɔ̃ à ǹ syɩ pé náh, à káh pé sõ̀ tɛ̃ nɛ n de n syi n yuku sɛ̃́. 
Yesu mɛ nɛ, Liyel-á pé tɛkɩ mɔ Yisrayel no ó yĩnnɛ, pépi nónó-á mɛ képyɛ́nɩ̀ mɔ́npunɔ sɔ̃́. 
Ǹtɛ pi mɛ nɔ pɔ ǹ gbètukunɔ di ǹ yah, tɛ́ yo ǹ tyɩ́ nɛ pé Tɛ̃̀ ǹgbɛ̃, à pé yohnɩ n mɔ. 
Yesu mɛ à syah nɛ, ké yah náh pè wɛ̃́npì yõke tɔkɔ n mɔ pã̀hnpye kɔ̃. 
Pi mɛ yo ǹ tyɩ́ nɛ, gbɩ-á à yo pé yõ̀tɛ̃̀. Ǹtɛ pã̀hnpye wɔ-á n yõ sé nátãm yɛ̃́nmîrn yṍnkwɛnɔlɛ. 
Képah sɔkɔ, Yesu mɛ yo ǹ tyɩ́ nɛ: «Pi, á sɛ̃ yĩ́ĩ́ ńmɔ yõ dɛ́. Wáh nɛ̂ yah kɔ̃, Liyel kɩ kè pɩ á tyɩ́.» Légbĩ́ mɛ̀ ó sɔkɔ, pi písê núkú dɛ. 
Képah náh, Yesu yal nɛ́npɔ́ à yuku Kalile yɔ́-ǹgbɛ̃ gbɔ̀pɔ. Tɛ́ dɔkɔ yuku yɔ̃lɩ yõ à kyɩ kɔ̃ tɛ̃. 
Képah tɛ̃̀nɛ, nɛ́kwĩ́nkí pé yámpú tɛ̃ pɔ ǹ tyɩ́ à pɔ pè sah ǹ yah sɔkɔ. Túkù sõ̀ mɛ gbáhkɩ̀ núkú névyelɛ, túkù mɛ yɩ̃́tũbilɛ, túkù mɛ nɛ́dónènɛ, túkù mɛ kèdyɔ̀nɩ̀nɛ, à tahlɩ yámpú túkù tyɩ́. Tɛ́ Yesu mɛ pè dɛ mɔ. 
Kè pɩ gbɛ̃̀ndùnìnɛ tɔ́wû tyɩ́, pǎh kèdyɔ̀nɩ̀ wɛ pè mɛ fɛ̃ n wɛkɩ, nɛ́dónè dónɔ́nɔ mɛ tyɛ, gbáhkɩ̀ núkú névye mɛ fɛ̃ n kɔ tómtóm, tɛ́ yɩ̃́tũbi mɛ fɛ̃ n yah. Képah tɛ̃̀nɛ, pè gbílkínɔ́ sah Yisrayel Liyel yõ. 
Yesu ǹ tõ̀ sã́hpú ye, tɛ́ yo pé tyɩ́ nɛ, tɔ́wû mɛ̀ yãm-á mɛ péwɔ tɛ̃. Pé sèpĩ̀n tɔ̃́nɩ̀-á péwɔ náh sɔkɔ kwɛynɛ yɔ̀, tɛ́ yõ yõ kwâh náh se pé tyɩ́. Pé náh mɛ ké tyɩ́ nɛ pé pè fĩn n mɔ pè sɔkɔ kwɛkɩlɛ. Pé túkù-á kɩ kyɩ n syah n tɛ̃ n kwlɔ wɛ̃̀kɩ̀ yah. 
Képah tɛ̃̀nɛ, ǹ tõ̀ sã́hpú à syah nɛ, pé se kɩ wɛ ńyãh yõkelɛ gbɛ̃́nnɩ mɛ̀ sɔkɔ, kɩ fɛ̃ tɔ́wû mɛ̀ dùkù kɔ̃ pè kɩ yõ tɔ̃ nɩ? 
Yesu mɛ pé piki yah nɛ, búrn plɔ́ sɔ̃́ se mɛ pé tyɩ́ nɩ? Pè mɛ nɛ kwǎrninɔ-á, à tahlɩ yɛ̃́nképì yísyɔ́ tyɩ́. 
Képah sɔkɔ, à tɔ́wû kɔ̃ pè kɔ̃ tɛ̃ sétáh. 
Tɛ́ búrn plɔ́ kwǎrninɔ ànɛ̂ yɛ̃́nkyɔ nɔ́pi tɔkɔ, tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ sè kwɔkɩ à ǹ tõ̀ sã́hpú kɔ̃ pè syi tɔ́wû kɔ̃. 
Pé pól mɛ yõ tɔ̃, tɛ́ pè mɛ pé yṍnkwlɔnɔ tuke mɔ tɔkɔ yɩ̃́ kwǎrninɔlɛ. 
Tɛ́ nónó-á sõ̀ yõ yõke mɛ̀nɛ, pè sõ̀ mɛ díbí nɛ́pĩ̂ kèyõ nínɔ́lɛ (4 000), syɔ́ ànɛ̂ wɛ̃́npì vyãh kɛ̃́nɛ. 
Képah náh, Yesu tɔ́wû fĩn mɔ pè sɔkɔ, tɛ́ mɛ de krótókè sɔkɔ à sɔkɔ Makadã sétáh yõ.
