Matye.
9.
Képah sɔkɔ, Yesu de krótókè sɔkɔ à ǹ syɩ yɔ́-ǹgbɛ̃ kah à sɔkɔ ǹ kwil sɔkɔ. 
Nósyɔ́ mɛ sèkùkù dĩ́ yɔ́ twah sɛ̃ sɛ̃ kwâh yɔ́ yõ à pɔ ǹ tyɩ́. Wǎh kè wɛ pè sɛ̃ ǹ yõ sɛ̃́, à yo sèkùkù dĩ́ mɛ̀ tyɩ́ nɛ: «Káh n yɛ á yáh-ńsah tyɛ ń pídĩ́. Á tyípêl pɩ́nɔ́ ye yɔ̃ mɔ ǹ kɔ̃.» 
Képah tɛ̃̀nɛ, Liyel tyi kõ̀nsàhnɔ́ kwɔ́pú yísyɔ́ tyah n sõ nɛ, dĩ́ mɛ̀-á mɛ n yo n de Liyel tyɩ́ dɛ́. 
Ǹtɛ Yesu-á pé sõnɔ pɩ̃ sɛ̃́, à yo pé tyɩ́ nɛ: «Kwâh nɛ̂ pɩ tɛ́ sõnɔ túkúnìnɛ yé sɔkɔ sɛ̃́? 
Mâh yo sèkùkù dĩ́ mɛ̀ tyɩ́ nɛ: ‹Á tyípêl ye yɔ̃ mɔ.› Képah náh pɩ, mâh yo ǹ tyɩ́ nɛ: ‹Yuku nɛ n kɔ.› Dlo mɛ nɛ̂ tɛ̃̀ yónɔ́ sɔkɔ kal? 
Ǹtɛ mé kɩ tir yɔ́ pɩ yé kɔ̃ yé kɩ pɩ̃ nɛ, tɛ̃́nwɛnɔ-á mɛ Névi Pídĩ́ tyɩ́ sétáh yõ kɩ fɛ̃ tyípêl yɔ̃ mɔ.» Tɛ́ ǹ syɩ yo sèkùkù mɛ̀ tyɩ́ nɛ: «Yuku, á sɛ̃ sɛ̃ kwâh tɔkɔ ǹ nɛ n yuku á gbô.» 
Dĩ́ mɛ yuku à sɔkɔ ǹ gbô. 
Tɔ́wû-á képah tɛ̃̀ wɛ sɛ̃́, tíkí pè tɛ̃. Pè tyah Liyel yĩnnɛ n gbilki wǎh sɔ̃́npɩ́lɛ tɛ̃́nwɛnɔ mɛ̀ dùkùlɛ névye kɔ̃. 
Yesu-á yal nɛ́npɔ́ tɛ́ n sɔkɔ, à dĩ́ yɔ́ wɛ à mɛ kɔ̃ tɛ̃ mɔ́rɩ́lɛ n syi mɔ́rɩ́ syí-ńsah sɔkɔ, pè n ye Matyelɛ. Yesu mɛ yo ǹ tyɩ́ nɛ: «Ń sõ̀ tɛ̃.» À mɛ yuku ǹ sõ̀ tɛ̃. 
Képah náh, Yesu-á sõ̀ mɛ yõke yṍnɔ́ sɔkɔ Matye gbô, mɔ́rɩ́ syípú náhnáh ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂pú tɔ pɔ n yõ ǹnɛ wɛ̃ tyɩ́ ànɛ̂ ǹ tõ̀ sã́hpúlɛ. 
Farisi wɛ̃̀kɩ̀ kɔ́pú-á képah wɛ sɛ̃́, pè yo Yesu tõ̀ sã́hpú tyɩ́ nɛ, kwâh nɛ̂ se pɩ tɛ́ pé kwɔ́-ò dĩ́ n yõ mɔ́rɩ́ syípú ànɛ̂ Liyel wɛ̃̀kɩ̀ kɔ́-ńkɛ̃̂púlɛ wɛ̃ tyɩ́ nɩ? 
Ǹtɛ Yesu-á pé wɛy noh, à yo pé tyɩ́ nɛ: «Névye nónó-á mɛ lènɔlɛ, sènpɩ yétõ̂ se mɛ pé tyɩ́? Yámpú tyɩ́ náh ǹ yétõ̂lɛ di? 
Liyel-á nɛ̂ yo nɛ ńkɛ̃́: ‹Yâh wɛ̃ yãmnɛ n yah, képah tyɩ́ nɔ ńmɔ tyɩ́ kal yáh sárká nɛ̂nɛ n sukɛ ńnɛ n kɔ̃.› Yé kyɩ yé fɔkɔnɩ mɔ yah képah tyɩ́ sɔkɔ, yè ké kɔ́r pɩ̃. Yé náh wɛ di, névye nónó-á mɛ n sõ nɛ nɛ́gblɔ́-á pélɛ Liyel yah, mé náh pɔ pépi yé-ńsah dɛ́. Ǹtɛ nónó-á pɩ̃ nɛ tyípêl pɩ́pú-á pélɛ, pépi yé-ńsah ye mé pɔ.» 
Képah sɔkɔ, Wèlnwìlkì-ò Nsyɔ̃ tõ̀ sã́hpú nɔ de Yesu tnɔ̂ tɛ́ à syah nɛ, kè se pɩ sɔ̃́ tɛ́ pépi ànɛ̂ Farisi wɛ̃̀kɩ̀ kɔ́pú níkìlɛ n tɛ̃ tɛ́ ǹmɔ tõ̀ sã́hpú náh mɛ níkìlɛ n tɛ̃ nɩ? 
Yesu mɛ yo pé tyɩ́ nɛ: «Pǎh nónó ye fúr yõke yõ yĩnnɛ, pépi se kɩ fɛ̃ pé yey kwɔ sah pɛ mɛ gbĩ́nɛ pé nɩyṍ? Ǹtɛ sèpĩ̀n yísyɔ́ mɛ n pi, pɛ mɛ̀ kɩ pɔ n tɛ̃ n wilki n pi pé nɩyṍ. Légbĩ́nɛ, pè kɩ níkìlɛ n tɛ̃. 
Ǹnɩ̂ yɔ́ ńtɛ̃ náh yɩ̃nɛ à fwɔ̀sãn kãm à fwɔ̀dyàh min. Yé náh wɛ di, fwɔ̀sãn mɛ̀ kɩ fwɔ̀dyàh mɛ̀ yɔrɩ tɔ, wãn ó kɔ̃ kè kɩ tɔ̃ mɔ ké tɔ́nɔ́ yõ. 
Dìvɛ̃ sã́n tɔ náh yɩ̃nɛ mɔ́nɔ́lɛ pley dî sɔkɔ. Wâh képah pɩ, dìvɛ̃ mɛ̀-à fũ, kè kɩ pley nɔ́pi sɔkɩ, dìvɛ̃ kɩ kwɛ, pley tɔ tyɩ́ kɩ yɩkɩ. Ǹtɛ kàh dìvɛ̃ sã́nnɛ, pley snɔ́ sɔkɔ ye kè n mɔ. Képah sɔkɔ, kèyǎh ńtɛ̃ náh n yɩkɩ.» 
Yesu-á mɛ kénɛ wɛlɔlɛ n yo pé tyɩ́ gbĩ́ nɛ̂nɛ, Nsyifunɔ yõ̀tɛ̃̀ dĩ́ yɔ́ nɔ de ǹ tnɔ̂, tɛ́ gbètukunɔ di ǹ yah, tɛ́ yo ǹ tyɩ́ nɛ pé písê-á yah kwlɔ núkúnúkú sɛ̃́ yɔ̀. Ǹtɛ à pɔ kyɩ ǹ gbõ̀ sah ǹ yõ, wǎh kɩ min wɛ. 
Képah sɔkɔ, Yesu ànɛ̂ ǹ tõ̀ sã́hpú mɛ yuku dĩ́ sõ̀ tɛ̃. 
Tɛ́ kè pɔ yɩ̃nɛ sê yɔ́ sõ̀ mɛ náhlɛ n yɔ tyɛ́-ńkɛ̃̂nɛ ye gbɔ̃ninɔ́. Pi mɛ̀ mɛ nɔ de Yesulɛ ǹ náh tĩ̀nnɛ, à pɔ tũ ǹ fwɔ̀mɔnɩ vyãh tyɩ́. 
Yé náh wɛ di, à sõ̀ mɛ n sõ ǹ fɔkɔ sɔkɔ nɛ páh kɩ tɛ̃ wɛ tũ ǹ fwɔ̀mɔnɩ ńtɛ̃ ó tyɩ́, páh kɩ dɛ. 
Yesu-á ǹ yah vi tɛ́ à wɛ, à yo ǹ tyɩ́ nɛ: «Káh n yɛ á yáh-ńsah tyɛ, ń písê. Á Liyel yõ sɛ̃́nɔ́ ye ǹ dɛ mɔ.» Nɛ́npɔ́ swɔ, pi núkú dɛ. 
Tɛ́ Yesu-á pɔ yĩni tɛ̃ Nsyifunɔ yõ̀tɛ̃̀ dĩ́ gbô, à yikɔ fɩ̃́bɩ́pú wɛ tɛ́ tɔ́wû wɛ pè mɛ n pɩ wrǒwrǒ. Képah sɔkɔ, à mɛ yo pé tyɩ́ nɛ: 
«Yé wil ásõ̂. Pípír náh ku, dɔy sɔkɔ ye ǹnɛ.» Ǹtɛ pè sõ̀ mɛ kṍlɛ Yesulɛ n ko. 
Pǎh tɔ́wû wilki mɔ tɛ kégbɔ́ sɔkɔ, Yesu mɛ de yuku wɔ̀kɔ̀ sɔkɔ, tɛ́ pípír gbõ̀ sɔkɔ tɛ̃, pípír yuku. 
Yĩ mɛ̀ pɩ́nɔ́ tyɩ́ tyah n yãh kénɛ mara ywɔ̃ ywɔ̃ pól sɔkɔ. 
Yesu-á yal nɛ́npɔ́ à n sɔkɔ gbĩ́ nɛ̂nɛ, yɩ̃́tũbi nɛ́pĩ̂ nímí yuku ǹ sõ̀ tɛ̃ tɛ́ mɛ yo tũ léyy nɛ, Davidi Pídĩ́, à pé yãm yah. 
Yesu-á pɔ yĩni tɛ̃ gbô sɔkɔ, yɩ̃́tũbi nɔ́pi mɛ nɔ de ǹ tnɔ̂. À mɛ pè piki yah nɛ, pè se sɛ̃ ké yõ nɛ péwɔ-á kɩ fɛ̃ pé yɩ̃́ yɩkɩ mɔ nɩ? Pè mɛ nɛ, àwe-á, Tɛ̃̀ ǹgbɛ̃. 
Képah sɔkɔ, Yesu tũ pé yɩ̃́ tyɩ́ tɛ́ nɛ: «Yáh sɛ̃ ńmɔ yõ sɔ̃́, tɛ́ nɛ̂ yah kɔ̃ ń tyɩ́, Liyel kè pɩ yé tyɩ́ sɛ̃́.» 
Képah tɛ̃̀nɛ, pé yɩ̃́ yɩkɩ mɔ. Yesu mɛ pè gbah sni nɛ, pè pé gblɔ̌y tɛ̃, pè káh n yɛ nɛy pɩ̃ ké sɔkɔ dɛ́! 
Ǹtɛ pé wìlntɛnɔ ǹsɔ̃̀, pè kyɩ tyah Yesu yĩ́ngbɩ́lɛ n yo n kɔ̃ kénɛ mara pól sɔkɔ. 
No nɔ́pi-á wil n sɔkɔ gbĩ́ nɛ̂nɛ, nósyɔ́ pɔ dĩ́ yɔ́lɛ Yesu tyɩ́, nɛ̂-á sétõ yɔ́ mɛ ǹ sɔkɔ à kɔ̃ à pɩ kèdɔ̀kɩ̀lɛ. 
Yesu-á núkú sétõ mɛ̀ yĩni wilki dĩ́ mɛ̀ sɔkɔ, dĩ́ tyah n wɛkɩ. Kè pɩ gbɛ̃̀ndùnìnɛ tɔ́wû pól tyɩ́, pè mɛ yo tũ nɛ ńkɛ̃́, pépi náh ǹsɔ̃̀ntɛ̃̂ dùkù wɛ yah Yisrayel sɔkɔ póllɛ. 
Ǹtɛ Farisi wɛ̃̀kɩ̀ kɔ́pú wɔ sõ̀ mɛ yo tũ nɛ, sétã̀n yõ̀tɛ̃̀ gbõ̀ yõ-á à n yĩni sétã̀nnɛ. 
Yesu lésõ mɛ kwló-ǹgbnɔ ànɛ̂ kwlópî póllɛ n kɔ n kɔ̃, mɛ nolɛ n kwɔ Liyel ní-ńtã̂n vɩ̀ sɔkɔ, mɛ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo Liyel tɔ̃́rɩ́ tyɩ́ yõ, tɛ́ mɛ yámpú pól ànɛ̂ nɛ́fukulɔ́ póllɛ n dɛ n mɔ. 
Yesu-á tɔ́wû nónó wɛ kwlɔ́nɔ́ sɔkɔ, tɛ́ pé yáh-ńsah mɛ tyɛ á kɩ nɛ yípɛ̂y sɔ̃́ fɛ̃̀-ò kɛ̃́nɛ sé tyɩ́, pé yãm à tɛ̃. 
Képah sɔkɔ, à yo ǹ tõ̀ sã́hpú tyɩ́ nɛ: «Sèmukɔ tɔkɔ tɔkɔ tɛ̃̀ mɛ dò, tɛ́ ké tõ̀npɩbɩlɛ kwéy. 
Kǎh pɩ sɛ̃́, Tɛ̃̀ ǹgbɛ̃ Liyel nɛ̂-á sèmukɔ nɛ́tɛ̃̂nɛ, yé à ni à tõ̀npɩbɩ tɛkɩ n mɔ sèmukɔ tɔkɔ yĩnnɛ.»
