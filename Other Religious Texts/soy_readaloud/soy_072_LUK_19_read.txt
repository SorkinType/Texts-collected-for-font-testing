Luki.
19.
Yeesu uu Yeriko-i lompɔ uu uncee n tɔ́su. 
Utisi unyinɛ kɑpi yɛ n sée rɛ Sɑsee un kuyu kpɛ-i we. U pɛɛ pilɑmpooyɔɔ́ uwɛ́ɛ́sɛ lɛ un kɔ umɑ́mɑ́. 
Uu nɑ́ɑ́si ukɛ́ uyɛ̃ kɑpi yɛ n sée rɛ Yeesu yɛ̃, ɑmɑ́ úu fe. Pisoi yɛ pɛɛ riwúí cɑ́pinɛlɛnlɛ, uyɛ̃ úu kɔ pɛɛ mɛcɑ́ɑ we. 
Uu pɛɛ kɛ́yu-mɛ̃ wuru uú hɑ kuléé kunyinɛ kɑpi yɛ n sée rɛ Sikomɔɔ yopoi ukɛ́ Yeesu yɛɛ kei n tíkinɛ yɛ̃. 
Kɛ Yeesu uu kɛlõ kɛ-i n tu, uu uinipɛɛ síkɑ uu Sɑsee mɑɑ rɛ: A mɛkɛɛkɛɛ súimɛ Sɑsee, li pisɛ rɛ kɛ́ nɛni kɛpɔ́yɔ-i tonɛ. 
Sɑsee uu mɛkɛɛkɛɛ súiri uú hɑ kusɑ́nɛ u yɔ́su nɛ mpɔ́ɔnɑrɛ. 
Ai pɛ̃ nnɛ́í pɛɛ lɛ̃ n yɛ̃́ pírí wɑi, ɑpi pɛɛ́ n tee rɛ: Ukópɛkɔɔ́ kɛyɔ kɛ usoi nkó uu nní lõ. 
Sɑsee un Upíimɑ kɛyúrí nyɛnu uu rɛ: A kutu ricɔ, Sɑ́ɑ nɛ́ mɛnɛ́mɑ́ rikénɛ ɑm kuwɛ́lɛ́ píwɛkɔɔ́ pɑ. Nɛn kɔ úye mɛnípɛɛ n li ɑm usiwóó yɔ́su, nɛ́ liute si hɛ́ɛ́lɛ mɛpehẽ mɛnɑ. 
Yeesu uu u mɑɑ rɛ: Píyulɑlɛ yɛ nɛni kɛyɑɑ nkɛ́ kɛ́yɔ nkɛ́-i lommɛlɛ, ɑ́i líkɑ nnyɑ, Sɑsee ricuruu yɛ kɔ Apirɑhɑm kɛpirɛ ukɔ́ lɛ. 
Likumúŋɛ́ rɛ Usoi Kɛpipi yɛ kɑlɛ kɛkɛ́ pɛpɛɛ m pɔlɑɑlɛ̃ tiyu lɔ. 
Pisoi pɛɛ Yeesu kutu rińcɔ yɛ pɛɛ́ musí rɛ kuu nní nɛ Yerusɑlɛm tińkɔ nnyɑ, Uléécɑɑ yɛ lɛ̃ tuimɛlɛ ukɛ́ iyɔ́ɔpi n le pin u nyɑ́ni. Lɛ̃ nnyɑ kuu kɛnyɑ́rũ pi mɑɑ rɛ: 
Kɛ́yɔ kɛcirɛ usoi unyinɛ yɛɛ kuyu tɑɑ-tɑɑ kunyinɛ-i hɑ rɛ pikɛ́ iyɔ́ɔpi u tĩ ukɛ́ pɛɛ limɛmɑ́ɑ́ pɛɛmɛ. 
Ukɛ́ kɛlenɛ ń tɔsí, uu upikɛikɔ́ kɛfi pinyinɛ séi uu ḿpɑ́ úye wurɑ kɛwóó múísɛ uu pi mɑɑ rɛ: Ani siwóó nsí nɛ kpéénsɛ hɑ́i nɛ mɛnɛ́pɛɛmɛ. 
Amɑ́ ukuyu pisoi ɑpi uipuri yulu, ɑpi pitumɛ yukusɛ pikɛ́ ukɛpirɛ ritikipɔ pikɛ́ rɛ: Ári lɑ rɛ usoi nkó ukɛ́ urɔ́yɔ́ɔpi. 
Api iyɔ́ɔpi u tini uu ukuyu pɛlɛ. Uu pɛɛ pikɛikɔ́ pɛ̃ kuú pɛɛ siwóó m mulɛisɛ séi rɛ ukɛ́ yɛ̃́ kulɑ́ɑ kpɛ̃ kɑpi n yɛ̃́. 
Ufoí uú hɑpɔ uú mɑɑ rɛ: Sɑ́ɑ, wurɑ kɛwóó kɑɑ nɛ́ rimmúísɛ kulɑɑ yɛ wɑlɛ wurɑ siwóó kɛfi. 
Uyɔ́ɔpi uu u mɑɑ rɛ: Li nyɑmlɛ, ukɛikɔ́ sɔnɛ. Kɑɑ nní likɛi síńsɑ́ pimúlɛ́ ń céri ńsɔnɛ nnyɑ, nɛ́ ɑyu kɛfi pɔ́ pɑ ɑɑ ɑkɛcɑ́ɑ́ m pɑílɛ̃. 
Ukɛikɔ́ lírũ uú hɑpɔ uu rɛ: Sɑ́ɑ, wurɑ kɛwóó kɑɑ nɛ́ rimmúísɛ kulɑɑ yɛ wɑlɛ wurɑ siwóó sinupũ. 
Uyɔ́ɔpi uú uyɛ̃ mɑɑ rɛ: Nɛ́ pɔ̃́ ɑyu ɑnupũ pɑ ɑɑ ɑkɛcɑ́ɑ́ m pɑílɛ̃. 
Ukɛikɔ́ ucɔ uú hɑpɔ uu rɛ: Sɑ́ɑ, upɔ́wurɑ kɛwóó yɛ nkɛ́. Kɛnírípi-i kɑm kɛ pɑɑsi ɑm pésu. 
Ipɔ́wɑmɛ yɛ nɛ́ wɑlɛ, kɑm mɛpɔ́kɔ́ n nyu rɛ ɑ́ɑ yɛ nkɑ̃́ usoi wɑ nnyɑ. Lɛ̃ kɑ́ɑ n yekei kɑɑ yɛ kpísi, lɛ̃ kɑ́ɑ n lukɛsi kɑɑ yɛ kpɑsi. 
Uyɔ́ɔpi uu u mɑɑ rɛ: Pɔ́ ukɛikɔ́ kópɛ nkó, ɑpɔ́nɔ́ɔ cirɛ kɑḿ nɛ kɛtɑhɑi pɔ́ pɛpirɛnɛ. Pɔ nyu rɛ ɑ́m yɛ usoi nkɑ̃́ wɑ, tɛ nɛ yɛ lɛ̃ kɑ́m n yekei kpísilɛ, ɑm lɛ̃ kɑ́m n lukɛsi kpɑsi. 
Yo nnyɑ kɑ́ɑ pɛɛ siwóó kɛyekei-i yekei? Nɛ́ pɛɛ fe ɑm mɛnɛ́pɛɛmɛ mmɛ́-i si yɔ́su nɛ kulɑ́ɑ. 
Uu pɛɛ pɛpɛɛ kei ń we mɑɑ rɛ: Ani wurɑ kɛwóó kɛ̃ yɔsí ɑni uyɛɛ mɛwóópipi kɛfi m mɑ́ múísɛ. 
Api u mɑɑ rɛ: Uyɛ̃ nɛ mɛwóópipi kɛfi mɑ́lɛ. 
Asei kɑm nɔ́ símisi, uyɛɛ m mɑ́ kɑpi yɛ pípɑ rikpɑ́, ɑmɑ́ úye unsɑ́ m mɑ́, nkɑ́ripi kuu m mɑ́ kɑpi u yɔ́sunɛ. 
Amɑ́ pinɛ́lɑ́ɑrɔ pɛɛ ɑ́pi nɛ iyɔ́ɔpi nɛ́ n lɑ pɔ̃́ nkɔ́ yɛ mmú: Ani nɛ nté pi kɑm ɑni kɛnɛ́yu-i pi kóni. 
Kɛ Yeesu uu lɛ̃ pisímɛ́ ḿ mɑsí, uu risoiwuí kɛkpeẽ wɑi un Yerusɑlɛm simpɔ. 
Kumúŋɛ́ kpɛ-i kuu Pɛtifɑsee nɛ Petɑnii ɑyu kuwɛ́lɛ́-mɛ̃ tikúú tɛ̃ kɑpi yɛ n sée rɛ Olifiyee rikúú kɛyúrí ń we, uu upipirɛtiki pitɛ́ tum 
un pi tee rɛ: Ani kuyupi kpɛɛ nní kɛrɔ́yu ń wepɔ kɛ́mɛɛ hɑ. Nɔn kei n tuipɔ, nɔ́ kɛminɑɑpipi kɛnyinɛ lɛɛpɔ kɛn torɑɑlɛ̃. Usoi úkɑ úu kɛ tonɛlɛ̃. Ani kɛ fénnɛ ɑní nɛ nɛ́ kɛ weri. 
Úye un nɔ́ m pisɛ rɛ yo nnyɑ kɑni kɛ fénnɛ, ɑni liute mɑɑ rɛ Upíimɑ yɛɛ kɛ lɑ. 
Pitumɛ ɑpi sĩ́ ɑpí hɑ lɛɛpɔ yɑrɛ lɛ̃ kɛ Yeesu uu pi n símisi. 
Kumúŋɛ́ kpɛ-i kɑpi kɛminɑɑpipi n fénnɛ, kɛpite ɑpi pi mɑɑ rɛ: Yo nnyɑ kɑni kɛminɑɑpipi kɛ̃ toriyɛ? 
Api rinɔ́ɔ yɔ́su rɛ: Upíimɑ yɛɛ kɛ lɑ. 
Api kɛminɑɑpipi kɛ̃ nɛ sĩ, ɑpi kɛkɛcɑ́ɑ́ piilũ lɑɑi ɑpi Yeesu kɛkɛcɑ́ɑ́ tónsɛ. 
Kɑpi kɛ́yu-mɛ̃ n sĩ́, pisoi ɑpi piilũ ncée kɛcɑ́ɑ́ n tɑ́ŋɛntɛ. 
Kɑpi Yerusɑlɛm n nyɑhɑipɔ pin ncée mmɛɛ Olifiyee rikúú ń cɛ́pilɛ̃ tikilɛ̃ pipirɛtiki kulúi pɛɛ u n tíkilɛ̃, ɑpi Uléécɑɑ mɛyɔɔpi n yom lin cɑ́lɛ̃, mɛwɑisɑŋɑ mɛ̃ nnɛ́í kɑpi n yɛ́nti nnyɑ. 
Pi pɛɛ mɑ́ikɛɛ rɛ: Uléécɑɑ ukɛ́ Uyɔ́ɔpi uyɛɛ nɛ Upíimɑ rinyiri n sɔ́nti rikpɑ́. Nkíŋniŋɛ nkɛ́ kɛyómɛcɑɑ-pɔ n we, Uléécɑɑ rinyiri tikɛ́ kei-pɔ n tɑɑ́lɛ̃. 
Pifɑrisi pinyinɛ pɛɛ risoiwuí tɛ-i ń we ɑpi Yeesu mɑɑ rɛ: A pipɔ́pirɛtiki rinɔ́ɔ hɛ rɛ pikɛ́ risɛ́ɛ́. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ ḿpɑ́ pin tinsɛ́ɛ́, ɑpɑrɛ yɛ́ ricɑ́ɑ́i. 
Kɛ Yeesu uu Yerusɑlɛm kuyu n nyɑhɑipɔ, uu ku yɛnu, uu kpɛ̃ nnyɑ téni 
un tee rɛ: Yerusɑlɛm, pɔn pɛɛ nɛni kɛyɑɑ nkɛ́ n ceri lɛlɛɛ yɛ́ nɛ nkíŋniŋɛ pɔ́ n kɑm! Amɑ́ li nkpéni pékɑɑlɛnlɛ, ɑ́ɑ yɛ́ fe pɔkɛ́ li yɛ̃. 
Siyɑ́ɑ sɛ-i kɛ pipɔ́lɑ́ɑrɔ ɑpi itɔpilũ kɛcirɛ nɛ kɛkɑ́ípi pɔ́ n wɑinɛ yɛ sɔ́ntilɛ, ɑpi ɑwɛ́lɛ́ nnɛ́í pɔ́ n cɔ́ŋlɛ̃. 
Pi yɛ́ nɔ́ nɛ pipɔ́kɔ́ nnɛ́í nɑ̃́ kpɑ́tɑ́-kpɑ́tɑ́, ɑ́pi ripɔ́pɑrɛ ríkɑ pɔ́ yɑ́nɛ tikɛ́ ricɔ kɛcɑ́ɑ́ n tɔsilɛ̃. Kɑ́ɑ kumúŋɛ́ kpɛ-i kɛ Uléécɑɑ uu picomɛ pɔ́ ń kɑ n ceri nnyɑ kɑi lɛ̃ pɔ́ wɑinɛ. 
Yeesu uu Uléécɑɑ kɛyɔ loni uu pikpéensɛ pilɑ́kɑsɛ kɑ́pɑ́ɑ́ 
un pi tee rɛ: Li wɔlɑɑlɛ̃ tɛ: Pi yɛ́ kɛnɛ́yɔ sée rɛ kɛyómɛyɑ́hɑɑyɔ. Amɑ́ nɔ̃́ nɛ kɛ ripɔ́lɔ́sɛlɛ piyɑɑlukɛ kuhorɛ! 
Ḿpɑ́ kɛyɑ́ɑ kɛ́ye kɛ Yeesu uu yɛ Uléécɑɑ kɛyɔ-i n céési. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ nɛ kuyu pisoi kɛcirɛ ɑpí n wɛ́ɛ́si pikɛ́ Yeesu kpu. 
Amɑ́ ɑ́pi ceri lɛ̃ kɑpi yɛ́ n wɑ likɛ́ nɛ lɛ̃, likumúŋɛ́ rɛ kuyu nnɛ́í yɛ pɛɛ kutu u cɔlɛnlɛ ńsɔnɛ.
