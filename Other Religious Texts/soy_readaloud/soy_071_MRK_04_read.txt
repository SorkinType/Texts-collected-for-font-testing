Mɑriki.
4.
Kɛ Yeesu uu icélɑɑ Kɑlilee kupiyɛ ritimɛ-i n kpɑ́lɛ̃, pisoi kulúi ɑpi u kɑ́lisi ɑi tɔ́su uu mɛsɛ hɑ kúninɔi kɛcɑ́ɑ́ tonɛ. Míni kɛcɑ́ɑ́ kɛ kúninɔi kpɛ̃ ɑku tɔ́silɛ̃, pisoi pin mɛritimɛ nyɛnu. 
Sinyɑ́rũ kuú nɛ icélɑɑ mɛyɑ̃ pi céesi. Uu icélɑɑ iyɛ̃ kɛ́mɛɛ pi mɑɑ rɛ: 
Ani kutu ricɔ ɑni kóm! Utisi unyinɛ yɛɛ ilukɛpuri pifómɛ leemɛ. 
Kuu n fóm, ilukɛpuri icɔ ii ricéetimɛ-i lólu, sinúipi ɑsi i le. 
Icɔ ii kɛpɑrɛ kɛcɑ́ɑ́ kei kɛ ntɑɑi ńn m piyɛ lólu. Kɛ kɛtẽ ɑ́kɛ n címú nnyɑ, ipuri iyɛ̃ ii mɛsɛ nɛ mɛsɛ lɛlu. 
Ntuŋɛ nn tɑpu. Kɛ iníŋí íi m pɑpisi nnyɑ, mɛlɛi ɑmɛ kóósi. 
Icɔ ii sinípɛleépi kɛpɑ́ɑ́pɑ́ɑ́ lóipɔ. Sinípɛleépi sɛ̃ ɑsi mɛlɛi hilɑ, ɑ́mɛ pɛɛ ɑpipi torɛsi. 
Amɑ́ ipuri icɔ ii kɛtẽ sɔnɛ nɛ sɑ́, ii lɛlu ii pɑpisi ii kɔ ɑpipi torɛsi, iwéhe icɔ ɑfɛɛtɑɑni, icɔ kuwóó nɛ kɛfi, icɔ pílɛ. 
Yeesu uu pi mɑɑ rɛ: Úye un ukómɛ ukɛ́ kṍ. 
Kɛ Yeesu uu lɛ̃ mɛmɑ́ɑ́ iyɑɑ n léepɔ, pɛpɛɛ nɛ u n kɔ́lɛ̃ nɛ upipirɛtiki kɛfi nɛ pitɛ́ ɑpi u pisɛ rɛ ukɛ́ sinyɑ́rũ sɛ̃ ɑsei pi símisi. 
Yeesu uu rinɔ́ɔ pi pɛsɛ rɛ: Nɔ̃́ nɛ Uléécɑɑ iyɔɔpi nsímɛ́ mɛɛ m pékɑɑlɛ̃ picére mɑsilɛ, ɑmɑ́ picɔ yɛ ḿpɑ́ yo kómɛilɛ yɑrɛ sinyɑ́rũ. 
Pi yɛ n wiilɛnlɛ, ɑmɑ́ ɑ́pi kɔ pɛɛ nyɑ́ni. Pi yɛ kutu n cɔlɛnlɛ, ɑmɑ́ ɑ́pi kɔ pɛɛ kómɛi. Lɛlɛɛ rɛ ɑ́pi yɛ ŋmurɛi pikɛ́ piisoi consɛ, Uléécɑɑ ukɛ́ piɑkópɛ pi sɑ́rɛi. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Áni kɛnyɑ́rũ kɛ̃ kõo? Nɔnsɑ́ kɛ n kõ, íye kɑni yɛ́ pɛɛ́ nɛ sitɔ́rɔɔ nsí nnɛ́í kṍ mɑ́? 
Kei kɛ Yeesu uu pɛɛ kɛnyɑ́rũ kɛ̃ ɑsei pi lesɛ rɛ: Li nkpɑ́ni yɑrɛ nléécɑɑsimɛ́ kɛ utisi uyɛ̃ uu fóm. 
Pisoi picɔ isoi yɛ ricéetimɛ tɛ̃ mɛcɔ welɛ. Pin nléécɑɑsimɛ́ n kõ, Setɑni uu mɛsɛ nɛ mɛsɛ nléécɑɑsimɛ́ kɑpi pikɛmɛɛ n fṍ kpóupɔ. 
Kɛpɑrɛtẽ kɛɛ we yɑrɛ pɛpɛɛ nléécɑɑsimɛ́ n kṍ ɑpi mɛsɛ nɛ mɛsɛ piɑkiŋ kɛ́mɛɛ n yɔ́su nɛ mpɔ́ɔnɑrɛ. 
Amɑ́ ɑ́pi yɛ tíyɛsɛ nkɛ́ pikɛmɛɛ iníŋí wɑ. Ńn yɛ pikɛmɛɛ nɑ́ŋɑi. Ncɔ́ŋ ń-ye nɛ́ɛ íwɛ íye in n kɑ nléécɑɑsimɛ́ nnyɑ, nfɑtɛnɛ nń nɛ pi kpi mɛsɛ nɛ mɛsɛ. 
Pisoi picɔ yɛ we yɑrɛ kɛtẽ kɛɛ sinípɛleépi m mɑ́. Pi yɛ nléécɑɑsimɛ́ kónlɛ, 
ɑmɑ́ kɛtẽ kɛcɑ́ɑ́ likɔ́ nfɑsimɛ́ nɛ lílɑlɑ lɛɛ rɔ́ n kírɑɑsɛntɛ nɛ mɛmɑ́ ɑi nléécɑɑsimɛ́ hílɑ, ńn yɛ pɛɛ kɛrɔ́mɛɛ líkɑ yoriyɛ. 
Pisoi picɔ pin we yɑrɛ kɛtẽ sɔnɛ. Pi yɛ nléécɑɑsimɛ́ kónlɛ ɑpi n yɔ́su nn pikɛmɛɛ pelu nn linyinɛ yoriyɛ. Nléécɑɑsimɛ́ nn picɔ kɛ́mɛɛ nkɑ́ripi wɑi, nn picɔ kɛ́mɛɛ nɛ kumúŋɛ́ wɑi, nn picɔ kɛ́mɛɛ mɛyɑ̃́ wɑi. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ pi pisɛ rɛ: Unyinɛ yɛɛ kɛ́firɑ́ɑ risɛ́lɛ uu kucɑ́ri kɛ kipi nɛ́ɛ uu ukɛfinɛ mɛtene kɛ pésu mɑ́? Nɛ́ntɛ kɛlõ cɑ́cɑ́ kɑpi yɛ kɛ ritɔsi ɑkɛ pɛɛ ḿpɑ́ úye n kpɑ̃́iilɛ̃? 
Lɛ̃ nnɛ́í lɛɛ pɛɛ m pékɑɑlɛ̃ ɑi kúyɛnɛ wɑi, lɛ̃ kɛ ḿpɑ́ úye úu pɛɛ kúyɛnɛ ń we ɑi mɛtɛ́í kɛ́mɛɛ léeri. 
Úye un ukómɛ ukɛ́ kṍ. 
Yeesu uu pimɑ́ɑ pi kpɑ́ rɛ: Ani nɔ́mɛnɛ́círɛ tĩ nɛ lɛ̃ nnɛ́í kɑni n kómɛi. Kulũ kpɛ̃ kɑni yɛ nɛ ḿ musí kɛ Uléécɑɑ uú nɛ nɔ́ músunɛ. Ticuruu u yɛ́ kusɑ́ nɔ́ rikpɑ́. 
Likumúŋɛ́ rɛ uyɛɛ m mɑ́ kɑpi yɛ pípɑ rikpɑ́ ɑmɑ́ úye unsɑ́ m mɑ́, ḿpɑ́ nkɑ́ripi ń-ye kuu mɑ́, pi n yɔ́sunɛ. 
Yeesu uu pimɑ́ɑ kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ usoi unyinɛ yɛɛ ilukɛpuri ukɛcɑrɛ-i n fṍ. 
Lɛ̃ mɛmɑ́ɑ́, ḿpɑ́ un kɛsinɛ nɛ́ɛ kɛtúŋɛ́ n lɔni nɛ́ɛ un n nyɑ́ni, ilukɛpuri yɛ lɛlulɛ in pelu úu nyu lɛlɛɛ i m peisɛ. 
Kɛtẽ ticuruu kɛɛ ilukɛ peisɛ. Kuhɛ́rɛ kpɛɛ yɛ mɛfoí pi, lɛ̃ mɛmɑ́ɑ́ ripipi ɑri tórɛsi, ɑri mɛtɔ́rɔɔ pelu. 
Tin m pi, usoi uu ritɛ́hɛ múlú. Kúkpɑɑ kpɛɛ lɛ̃ tu. 
Yo kɑri yɛ́ kɔ nɛ Uléécɑɑ iyɔɔpi múŋɛisɛnɛ? Kɛnyɑ́rũ kɛ́ye kɑri yɛ́ nɛ linsímɛ́ wɑ mɑ́? 
Uléécɑɑ iyɔɔpi yɛ we yɑrɛ mpuri yukɛ-yukɛ nnyinɛ. Kɛlukɑɑ-i, mmɛ mɛɛ yɛ kɛtẽ nnɛ́í kɛcɑ́ɑ́ ipuri kɛcɔpɛ nsíńsɑ́. 
Amɑ́ mɛlukɑɑ-mɑɑ́, n yɛ lɛlɛ nn pɑpisi nn wɑi kuléé kpɛɛ kɛcɑrɛ-i ɑcɔ mpiyɛ n fe. Nkuléé yɛɛ ilésɛ kɛcirɛ wɑlɛ, sinúipi ɑsi ikɛcɑ́ɑ́ ɑyɑ́mɛ́ wɑpisi. 
Sinyɑ́rũ nsí sicɔ kɛ Yeesu uu yɛ nɛ ḿpɑ́ úye céesi. Pɛpɛɛ kutu u n cɔ́lɛ̃ mɛkomɛ kuu yɛ ripɑí uú nɛ céési. 
Úu yɛ píkɑi símisi kúu kɛnyɑ́rũ n wɑ, ɑmɑ́ lin u nɛ upipirɛtiki ŋmɑnɛ rin-yɑ́, ḿpɑ́ yo ɑsei kuu yɛ pi lesɛ. 
Kɛyɑ́ɑ kɛ̃ cirɛ nnyɔ́ɔ́, Yeesu uu upipirɛtiki mɑɑ rɛ: Tɔkɛ́ míni kuwɛ́lɛ́ kucɔ-mɛ̃ hɑ. 
Api risoiwuí yɑ́, upipirɛtiki ɑpi kúninɔi kpɛ-i kuú pɛɛ n tṹ nɛ n tɛ́ŋ́lɛ̃. Áninɔi ɑcɔ ɑn kɔ pikɛyúrí we. 
Kei kɛ mɛyɔ píimɑ mɛnyinɛ ɑmɛ pipépɛ kori ɑ́niwɑlɛ́ ɑɑ kúninɔi-i n lééripɔ míni ɑmɛ pɛɛ ku n yipu. 
Kumúŋɛ́ kpɛ-i, Yeesu un kúninɔi kɛ́mɛɛ kɛpirɛ-pɔ we, un lɔni, tíyu tin tíyulɑɑ kɛcɑ́ɑ́ lɑ́ɑlɛ̃. Upipirɛtiki ɑpi pɛɛ u yóŋsɛ ɑpi u mɑɑ rɛ: Sɑ́ɑ, pɔ sɛ́ɛ́lɛnlɛ tɔkɛ́ kpii? 
Yeesu uu yɑ́kɑ-yɑkɑ uu kuyɔ cɛ́si uu míni mɑɑ rɛ: Sɛ́ɛ́, ɑ niŋɛsi. Kuyɔ ɑku kéu ɑi níŋɛsi. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ upipirɛtiki pisɛ rɛ: Yo nnyɑ kɛ iwɑmɛ ii nɔ́ wɑ? Nɔn nɛ nfɑtɛnɛ mɑ́ɑ? 
Amɑ́ ɑi iwɑmɛ píimɑ pi wɑi ɑpí m mɑ́ikɛɛnɛ rɛ: Usoi úye kɛcirɛ yɛ nkó kɑí nɛ kuyɔ nɛ míni ripɛ́nɛ lin u pɑkɑrɛlɛ̃?
