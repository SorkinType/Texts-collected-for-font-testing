Mɑriki.
8.
Pisoi kulúi ɑpi kɔ kumúŋɛ́ kpɛ-i picɑ́pinɛ kpɑ́ ńsɔnɛ ɑ́pi kɔ líkɑ mɑ́ pikɛ́ li. Yeesu uu upipirɛtiki séi uu pi mɑɑ rɛ: 
Pisoi mpí icɔ yɛ íwɛ nɛ́ welɛ. Siyɑ́ɑ sitɑɑni yɛ nsí kɑpi nɛ́ tíkilɛ̃ ɑ́pi kɔ líkɑ mɑ́ pikɛ́ li. 
Pimɛyɑ̃ yɛ kɛtɑɑ leerilɛ. Lɛ̃ nnyɑ, nɛn nkpéni nɛ nkṹ mmú pi m mɑɑ rɛ pikɛ́ kúi, pi yɛ́ hɑ nsẽ́ pɔɔnɛ. 
Kei kɛ upipirɛtiki ɑpi rinɔ́ɔ u yɔsí rɛ: Yei kɑri yɛ́ pisoi píimɑ mpí nnɛ́í ilukɛ yɛ̃ kɛlõ kóimɑ nkɛ́-i? 
Yeesu uu pi pisɛ rɛ: Akpɔ́nɔ́ ɑlɛ́ kɑni mɑ́? Api rinɔ́ɔ yɔ́su rɛ: Asɛɛi. 
Uu limɛmɑ́ɑ́ pisoi pɛ̃ tinɔ́ɔ hɛ rɛ pikɛ́ kɛteni-i tonɛ, uu ɑkpɔ́nɔ́ ɑsɛɛi nyɛ̃ yɔ́su, uu ɑkɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu pɛɛ ɑ kpɔ́kɔrinɛ uu upipirɛtiki ɑ pɑ pikɛ́ hɔ́ɔ́nɛntɛ, ɑpi pisoi ɑ hɔɔnɛ. 
Pi kɔ pɛɛ ikpíntomɛ́ inyinɛ mɑ́lɛ. Yeesu uu kɔ iyɛ̃ kɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu rɛ pikɛ́ kɔ iyɛ̃ hɔɔnɛ. 
Api le ɑpi lɛpu, ɑpi ilukɛ pɔɔnɛ-pɔɔnɛ kɔ́ɔ́i ɑnɛ́rɛ ɑsɛɛi. 
Pisoi pɛɛ tu yɑrɛ ɑ́kotokú ɑnɑ (4.000). Kɑpi ḿ mɑsí, Yeesu uu pɛɛ rɛ pikɛ́ kúi. 
U nɛ upipirɛtiki ɑpi mɛsɛ nɛ mɛsɛ kúninɔi loni ɑpi Tɑlimɑnutɑ kɛtẽ sĩ. 
Pifɑrisi ɑpi Yeesu lɛɛpɔ ɑpí nɛ íkɛŋɛnɛ u kɑ́pɑ́ɑ́ ɑpi pɛɛ ripinɛ u wɑi rɛ ukɛ́ nyísɛ rɛ Uléécɑɑ kɛ́mɛɛ kɛ unnɑŋɛ nn léeri. 
Yeesu uu mɛyɑ̃́ nfɑ́ɑ tɔ́ɔ́ri, uu ukɛfɑ-i mɑɑ rɛ yo nnyɑ kɛ nfɑ́ɑni pisoi mpí ɑpi mɛwɑisɑŋɑ piyɛ́nɛ lɑ? Asei kɑm nɔ́ símisi, nfɑ́ɑni pisoi mpí ɑ́pi mɛwɑisɑŋɑ mɛ́kɑ yɛnunɛ. 
Uu lɛ̃ mɛmɑ́ɑ́ kei pi yɑ́ uu kúninɔi loni uu rikóitimɛ ricɔ sĩ. 
Upipirɛtiki yɛ pɛɛ ɑkpɔ́nɔ́ kɛcɑ́ɑ́ pɑlɛilɛ. Kusɛ ŋmɑnɛ kɑpi múílɛ̃ kúninɔi-i. 
Yeesu uu rinɔ́ɔ pi hɛ rɛ: Ani nɔ́mɛnɛ́círɛ tĩ ńsɔnɛ nɛ Pifɑrisi nɛ Erooti nkpɔ́nɔ́tɔ pitɛ. 
Pipirɛtiki yɛ pimɛcɔpɛcirɛ símisi rɛ ɑ́pi ɑkpɔ́nɔ́ mɑ́. 
Kɛ Yeesu uu lɛ̃ n céri, uu pi pisɛ rɛ: Yo nnyɑ kɑni mɑ́ikɛɛ rɛ ɑ́ni ɑkpɔ́nɔ́ mɑ́? Ái kɑhɑnɛ nɔ́ rikpɑ̃́ii nɛ́ɛ ɑ́ni kɑhɑnɛ n ceri? Nɔ́ɑnɛ́tu yɛ tɑkɑsilɛɛ? 
Nɔ inípɛɛ mɑ́lɛ, ɑ́ni kɔ pɛɛ nyɑ́ni, nɔ ɑtu mɑ́lɛ ɑ́ni kɔ pɛɛ kómɛi. Áni léisɛɛ? 
Kumúŋɛ́ kpɛ-i kɑm ɑkpɔ́nɔ́ n kpɔ́kɔrinɛ ɑm pisoi ɑ́kotokú ɑnupũ (5.000) hɔɔnɛ, ɑnɛ́rɛ ɑlɛ́ kɑni ilukɛ tɔrɔɔ yípũ ɑní nɛ tɔ́mpɔ? Api u mɑɑ rɛ ɑnɛ́rɛ kɛfi nɛ ɑtɛ́. 
Yeesu uu kɔ pi pisɛ rɛ: Kɑm ɑkpɔ́nɔ́ ɑsɛɛi pisoi ɑ́kotokú ɑnɑ (4.000) n kpɔ́kɔrinɛ ní, ɑnɛ́rɛ ɑlɛ́ kɑni ilukɛ tɔrɔɔ yi? Api rɛ: Asɛɛi. 
Uu pɛɛ pi mɑɑ rɛ: Ái kɑhɑnɛ nɔ́ n hemnɛɛ? 
Kɛ Yeesu nɛ upipirɛtiki ɑpi Petisɑyitɑ n tu, pisoi ɑpí nɛ unyíyɛ unyinɛ u lɛɛpɔ ɑpí nɛ u téni rɛ ukɛ́ u ricɑ. 
Yeesu uu unyíyɛ kunípɛ tini uu u tɔrɔi uú nɛ kuyu ilúkɛ́ sĩ́ uu pɛɛ mɛtɑ inípɛɛ-i u wɑi, uu ɑnípɛ ukɛcɑ́ɑ́ lɑ́ɑ uu u pisɛ rɛ: Pɔ linyinɛ nyɑ́nii? 
Unyíyɛ uú weríí uu rɛ: Nɛ pisoi nyɑ́nilɛ, ɑmɑ́ pi pɛɛ́ welɛ yɑrɛ ɑléé, pin sɔ́nɛ. 
Yeesu uu uinipɛɛ pícɑ kpɑ́, kɛ unyíyɛ uu n nyɑ́nɛi tíkí-tíkí, uu pélu uu ḿpɑ́ yo n nyɑ́ni cɑ́í-cɑ́í. 
Yeesu uu pɛɛ u mɑɑ rɛ ukɛ́ kúi, ɑmɑ́ úu kɑpɛ kuyu lõ. 
Yeesu nɛ upipirɛtiki ɑpi Sesɑree Filipu siyupi kɛ́mɛɛ sĩ́. Uu ncée kɛ́mɛɛ pi pisɛ rɛ: Pisoi yɛ mɑ́ikɛɛ rɛ nɛ́ɛ úye? 
Pi mɑ́ikɛɛ rɛ pɔ́ɔ Yohɑni yɛɛ míni pisoi ń wolɛ, picɔ rɛ pɔ́ɔ Elii, picɔ rɛ pɔ ɑntepuyɛ kɛcɔpɛ unyinɛ lɛ. 
Yeesu uu pi pisɛ rɛ: Nɔ̃́ ni? Nɔ mpíí mɑ́ikɛɛ rɛ nɛ́ɛ wóo? Piyɛɛ uu rinɔ́ɔ u yɔ́su rɛ pɔ̃́ kɛ Uléécɑɑ uu mɛkɛɛ-mɛ wɛ́ɛ. 
Yeesu uu pi nyi rɛ ɑ́pi kɑpɛ linkɑ úkɑ símisi. 
Uu mɛncirɛ picélɑɑ pi koru rɛ li pisɛ rɛ Usoi Kɛpipi kɛkɛ́ mɛyɑ́nsei íwɛ li. Pisuifi piwɛ́ɛ́sɛ nɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ pikɛ́ kɛ ŋmɛriyɛ. Li kɔ pisɛ rɛ pikɛ́ kɛ kpu ɑkɛ kɔ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ yisi. 
U símisilɛ ńsɔnɛ kpɑ̃́ii. Kei kɛ Piyɛɛ uu iyɑɑ u sée uu rɛ ɑ́i lɛ̃ kuu yɛ́ pɛɛ wɑ. 
Yeesu uú ŋmɛɛlú uu upipirɛtiki pɑí uú nɛ Piyɛɛ cɛ́si rɛ: Setɑni, ɑ nɛ kɛtɑɑ nɛ́ wɑ, likumúŋɛ́ rɛ sipɔ́múŋɛ́ ɑ́si Uléécɑɑ sikɔ́, ɑmɑ́ sisoipipi sikɔ́ yɛ si. 
Yeesu uu pɛɛ lɛ̃ mɛmɑ́ɑ́ pisoi nɛ upipirɛtiki séi uu pi mɑɑ rɛ: Uɑye un n lɑ ukɛ́ nɛ́ ritiki, ukɛ́ umɛcirɛ pípɑí riyɑ́, uu ukunɑpɛ́ɛ́kɔɔ́ yɑ uu pɛɛ nɛ̃́ tiki. 
Ái líkɑ nnyɑ, úye un n lɑ ukɛ́ unfɑɑ lɔ, u n fómnilɛ, ɑmɑ́ úye un unfɑɑ n fóm nɛ̃́ nɛ Nsímɛ́ Kɛcirɛ nnyɑ, liute yɛɛ yɛ́ unfɑɑ lɔ. 
Usoi kulɑɑ yɛ kúye in tɛ u kɛtẽ nté mɛmɑ́ nnɛ́í yɛ̃́ uu pɛɛ kpi. 
Linyinɛ yɛ we kɛ usoi uu yɛ́ n hɛ́ɛ́lɛ ukɛ́ nɛ unfɑɑ n lɔɔ? 
Úye un nɛni inɛ́sɛi nɛ nnɛ́símɛ́ isɛi n wɑ, piwɑ́sɑ́nkɑí nɛ pikópɛkɔɔ́ mpí kɛcɔpɛ, Usoi Kɛpipi yɛ́ kɔ uisɛi wɑ, kɛn píyei m pɛɛri nɛ kɛusɑɑ mɛyɔɔpi nɛ piléécɑɑtumɛ.
