Luki.
9.
Yeesu uu upitumɛ kɛfi nɛ pitɛ́ cɑ́pinɛ uu ńnɑŋɛ pi hɛ rɛ pikɛ́ ɑníri n fe ɑpi kɔ itói m pɔisɛntɛ. 
Uu pɛɛ pi tum tɛ pikɛ́ hɑ Uléécɑɑ iyɔɔpi nsímɛ́ símisi ɑpi pitóikɔ́ pɔisɛntɛ. 
Uu pi mɑɑ rɛ: Áni kɑpɛ linyinɛ rimúlú yɑrɛ kunɑɑpi nɛ́ɛ kulɔ́ɔ nɛ́ɛ ilukɛ nɛ́ɛ kɛwóó nɔkɛ́ nɛ ncée hɑ. Áni kɑpɛ sitúkɑnkɑ sitɛ́ sitɛ́ rimúlú. 
Nɔn kɛ́yɔ kɛ́ye-i n topilɛ, ɑni kei n tũ hɑ́i nɛ nɔ́kɛnɛ́tɔ́ŋɛ́ tuŋɛ́. 
Pin yei kusɑ́nɛ piyɔ́ɔ́ nɔ́ n yɛ̀, ɑni pikuyu le, ɑni nɔ́ɑnɛ́nɑ nkoŋo péi likɛ́ nyísɛ rɛ pi Uléécɑɑ pipɑkɑrɛ yɛlɛ. 
Upitumɛ ɑpi tɔ́su, ɑpi ɑyu nnɛ́í kɛ́mɛɛ sɔ́nɛ ɑpi Nsímɛ́ Kɛcirɛ símisi pin ḿpɑ́ yei pitóikɔ́ pɔisɛntɛ. 
Uyɔ́ɔpi Erooti uu lɛ̃ nnɛ́í lɛɛ n wɑi nsímɛ́ kóm. Úu ceri lɛ̃ kuu yɛ́ m musí, likumúŋɛ́ rɛ pinyinɛ yɛ pɛɛ tee rɛ: Yohɑni Úniwolɛ yɛɛ pikpɔkpɔ-i yisi. 
Picɔ pin tee rɛ: Elii yɛɛ leemɛ. Picɔ pin kɔ tee rɛ: Pɛɛ mɛkɛɛ ɑntepu unyinɛ yɛɛ nkpɔ-i yisi. 
Amɑ́ Erooti uu rɛ: Nɛ tíyɛsɛlɛ ɑpi Yohɑni ríyu tɔlu. Usoi úye mpuri nsímɛ́ kɑm pɛɛ nní kómɛi mɑ́? Uú n wɛ́ɛ́si ukɛ́ Yeesu yɛ̃. 
Pitumɛ ɑpi pɛɛri ɑpi lɛ̃ nɛ lɛ̃ kɑpi n wɑpisi Yeesu kɛɛnkɛɛ. Uú nɛ pimɛcirɛ iyɑɑ lelu kuyu kpɛ̃ kɑpi yɛ n sée rɛ Petisɑyitɑ kɛkúrí. 
Amɑ́ pisoi pɛɛ n cɑ́pinɛlɛ̃ ɑpi kóm ɑpi u tiki. Yeesu uu Uléécɑɑ iyɔɔpi nsímɛ́ pi símisi uu kɔ pɛpɛɛ m pɔ́ɔ́lú pɔisɛntɛ. 
Kɛ ituŋɛ ii n tɛnɛ, upitumɛ kɛfi nɛ pitɛ́ ɑpí nɛ u kɔ ɑpi u mɑɑ rɛ: A pisoi mɑɑ rɛ pikɛ́ hɑ ɑyu nɛ siyupi sɛɛ nɛ nní rɔ́ ń kɔ́lɛ̃ kɛ́mɛɛ ilukɛ nɛ sifinɛ wɛ́ɛ́si, likumúŋɛ́ rɛ kucɛsi kɛcɔpɛ kɑri nté we. 
Amɑ́ Yeesu uu pi mɑɑ rɛ: Ani nɔ́rinɛ́cúruu lilukɛ-lukɛ pi hɛ. Api rinɔ́ɔ yɔ́su rɛ: Akpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́ ŋmɑnɛ kɑri nté mɑ́. Nɛ́ɛ pɔ lɑ rɛ tɔkɛ́ hɑ pisoi mpí nnɛ́í ilukɛ lɔ? 
Pisoi ɑ́kotokú ɑnupũ (5.000) pɛɛ pɛɛ kei we. Yeesu uu upitumɛ mɑɑ rɛ: Ani pi tonsɛntɛ pisoi kuwóó kuwóó ɑnɑi. 
Pitumɛ ɑpi lɛ̃ kuu pi n forii wɑi, ɑpi pisoi tonsɛntɛ. 
Yeesu uu ɑkpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́ kpísi, uu kɛyómɛ weríí, uu ilukɛ iyɛ̃ kɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ. Uu kpɔ́kɔrinɛ uu upitumɛ pɑ rɛ pikɛ́ pisoi pɛɛ n cɑ́pinɛlɛ̃ hɔɔnɛ. 
Pinnɛ́í ɑpi le ɑpi lɛpu. Upitumɛ ɑpi ɑkéri nyɛɛ m pɔɔnɛ cɑ́pinɛ ɑi ɑnɛ́rɛ kɛfi nɛ ɑtɛ́ yipu. 
Kɛyɑ́ɑ kɛnyinɛ kɛ Yeesu uu umɛcirɛ kɛyómɛ yɑ́ɑ́si, upipirɛtiki ɑpi u lɛɛpɔ, uu pi pisɛ rɛ: Pisoi yɛ mɑ́ikɛɛ rɛ nɛ́ɛ úye? 
Api rinɔ́ɔ yɔ́su rɛ: Pinyinɛ yɛ tee rɛ pɔ Yohɑni Úniwolɛ lɛ, picɔ rɛ pɔ Elii lɛ, picɔ pin kɔ tee rɛ pɔ pɛɛ mɛkɛɛ ɑntepu unyinɛ lɛ yɛɛ nkpɔ kɛ́mɛɛ n yisi. 
Yeesu uu pi pisɛ rɛ: Nɔ̃́ ní, nɔ mpíí rɛ nɛ́ɛ úye? Piyɛɛ uu rinɔ́ɔ yɔ́su rɛ: Pɔ́ɔ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ. 
Yeesu uu pi nyi rɛ ɑ́pi kɑpɛ úkɑ nsímɛ́ mmɛ̃ símisi. 
Uu pimɑ́ɑ kpɑ́ rɛ: Li pisɛ rɛ nɛ̃́ Usoi Kɛpipi kɛ́ mɛyɑ́nsei íwɛ li. Pisuifi piwɛ́ɛ́sɛ nɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ yɛ́ kɛ kɛsi. Li kɔ pisɛ rɛ pikɛ́ kɛ kpu ɑkɛ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ yisi. 
Uu pɛɛ pinnɛ́í mɑɑ rɛ: Úye un n lɑ ukɛ́ nɛ́ ritiki, ukɛ́ umɛcirɛ pípɑí riyɑ́ uu ḿpɑ́ kɛyɑ́ɑ kɛ́ye ukunɑpɛ́ɛ́kɔɔ́ n yɑlɛ̃, uu pɛɛ nɛ́ tíki. 
Ái líkɑ nnyɑ, úye un n lɑ ukɛ́ unfɑɑ lɔ, u n fómnilɛ. Amɑ́ úye un nɛ̃́ nnyɑ unfɑɑ n fóm, liute yɛɛ yɛ́ unfɑɑ lɔ. 
Usoi kulɑɑ yɛ kúye in tɛ u kɛtẽ nté likɔ́ nnɛ́í yɛ̃́, uu pɛɛ kpi nɛ́ɛ uu umɛcirɛ ɑ́sɑ́lɑ́ú tɑni. 
Úye un nɛni inɛ́sɛi nɛ nnɛ́símɛ́ isɛi n wɑi, Usoi Kɛpipi yɛ́ kɔ uisɛi wɑ kɛn píyei kɛmɛyɔɔpi nɛ kɛusɑɑ nɛ piléécɑɑtumɛ kɛ Uléécɑɑ uu n wɛ́ɛ mɛyɔɔpi-i m pɛɛri. 
Asei kɑm nɔ́ símisi, pinyinɛ yɛ nté welɛ pɛɛ yɛ́ Uléécɑɑ mɛyɔɔpi n yɛ̃́ ɑpi kɛlenɛ kpíni. 
Kɛ Yeesu uu lɛ̃ pisímɛ́ ḿ mɑsí, ɑi wɑi yɑrɛ siyɑ́ɑ sipɑhɑ kumúŋɛ́, uu Yohɑni nɛ Yɑkupu kpísi uú nɛ rikúú tɑɑ́ pikɛ́ kɛyómɛ yɑ́ɑ́si. 
Kuu kɛyómɛ n yɑ́ɑ́si, ukɛyu ɑkɛ mɛ́wee consɛ, uilũ ii wɑ́rɑrɑ póí-póí. 
Mɛsɛ nɛ mɛsɛ, pisoi pitɛ́ pinyinɛ ɑpi Uléécɑɑ mɛyɔɔpi-i léeri, pi nɛ Yeesu pin yɔ́i. Pikɛi kɑi m pisɛ rɛ ukɛ́ wɑ nɛ unkpɔ Yerusɑlɛm-pɔ ifɑɑci kɑpi lɛ̃ wɑi. Pɛpɛɛ Moisi nɛ Elii. 
Piyɛɛ nɛ upicɔ pin nɛ nnɔ̃ kɛtɑɑ tu, ɑmɑ́ ɑpi yɑ́kɑ-yɑkɑ ɑpi Yeesu mɛyɔɔpi nɛ pisoi pitɛ́ pɛɛ ukɛkúrí ń nyɛnu yɛnu. 
Kɛ pisoi pɛ̃ ɑpi Yeesu rin-yɑ́, Piyɛɛ uu u mɑɑ rɛ: Sɑ́ɑ, li nyɑmlɛ rɛ tɔkɛ́ nté n we. Tɔ lɑ rɛ tɔkɛ́ sicɑ́ŋíípi sitɑɑni kɑrii, pɔ̃́ kɛsɛ, Moisi kɛsɛ, Elii kɛsɛ. Úu nkpɑ́ni nyu mmɛ̃ kuu n símisi. 
Kumúŋɛ́ kpɛ-i kuu lɛ̃ n símisi, kuhopɛ kunyinɛ ɑkú weri ɑku pi hilɑ. Kɛ pipirɛtiki ɑpi ń yɛ̃́ kuhopɛ kun lɛ̃ pi hilɑlɛ̃, iwɑmɛ ii pi loni. 
Tinɔ́ɔ rinyinɛ ɑri kuhopɛ kpɛ-i léeri rɛ: Nkó yɛɛ Kɛnɛ́pipi kɑm n wɛ́ɛ, ɑni kutu kɛ ricɔ! 
Kɑpi rinɔ́ɔ tɛ̃ lɛ̃ n kómɛi Yeesu ŋmɑnɛ kɑpi pɛɛ nyɑ́ni. Pipirɛtiki ɑpi sɛ́ɛ́, ɑ́pi kumúŋɛ́ kpɛ-i úkɑ lɛ̃ kɑpi n yɛ̃́ símisi. 
Kɑi kóso n weesi, ɑpi rikúú cɛ́pimɛ, risoiwuí píimɑ rinyinɛ ɑri Yeesu tɛ́pɛipɔ. 
Utisi unyinɛ uu riwúí tɛ̃ kɛcɔpɛ sicɑ́ɑ́ilɑ kɑ́pɑ́ɑ́ rɛ: Sɑ́ɑ, nɛ nɛ pɔ́ téni rɛ ɑ kɛnɛ́wɑ̃́ kɛcɑ́ɑ́ ripɑí, kɛnɛ́círɛníŋɛ yɛ kɛ. 
Kuníri kunyinɛ kpɛɛ yɛ kɛ yisi, ɑkɛ mɛsɛ nɛ mɛsɛ pipupukɛɛ loni, ɑku kɛ kpɑpɑmnisɛ mɛyɑ̃́, ɑku ɑtɑlɑlɑɑ́ kɛ wɔ́isɛ. Áku yɛ mɛsɛ nɛ mɛsɛ kɛ riyɑ́, ku yɛ íwɛ kɛ wɑlɛ kɛlenɛ. 
Aḿ nɛ pipɔ́pirɛtiki téni rɛ pikɛ́ kuníri kpɛ̃ lɑkɑsɛ, ɑmɑ́ ɑ́pi fe. 
Yeesu uu pɛɛ rɛ: Nɔ́ nfɑtɛnɛcirɛ́ nɛ pikpɑ́ɑ́ree mpuri mmú! Ituŋɛ ilɛ́ kɑm nɔ́kɛnɛ́mɛɛ pitónɛ kpɑ́nɛ? Ituŋɛ ilɛ́ kɑm yɛ́ lɛ̃ nɛ nɔ́ n kɑhɑri? Ani nɛ kɛwɑ̃́ kɛ̃ nté nɛ́ kɑm. 
Kɛ kɛwɑ̃́ ɑkɛ n nyɑhɑipɔ, kuníri ɑku kɛ pɛpirɛ ɑku kɛ lémesi mɛyɑ̃́. Amɑ́ Yeesu uú nɛ kuníri cɛ́si hɑ́i, uu kɛwɑ̃́ pɔisɛ uu kɛusɑɑ múísɛ. 
Uléécɑɑ nnɑŋɛ píimɑ nn ḿpɑ́ úye pírí wɑi. Yeesu mɛwɑi mɛ̃ mɛn kɔ lɛ̃ ḿpɑ́ úye pírí we, uu upipirɛtiki mɑɑ rɛ: 
Ani nkpéni ɑsei nyɛ̃ kɑm ń lɑ kɛ́ nɔ́ símisi rimúlú ńsɔnɛ. Pi yɛ́ Usoi Kɛpipi pisoi ɑnipɛ-i wɑ pikɛ́ kɛ kpu. 
Amɑ́ ɑ́pi nsímɛ́ mmɛ̃ kõ. Pi pi m pesilɛ rɛ ɑ́pi kɑpɛ n kõ, iwɑmɛ íi kɔ ŋmurɛi pikɛ́ nɛ Yeesu m pisɛ. 
Pipirɛtiki ɑpi pimɛcɔpɛcirɛ kɛyɑ́ɑ kɛnyinɛ n kɛŋɛnɛ pikɛ́ céri rɛ úye yɛɛ pikɛcɔpɛ uwɛ́ɛ́sɛ. 
Yeesu uu pinfɑsimɛ́ céru. Uu pɛɛ kɛwɑ̃́ kɛnyinɛ kpísi uu ukɛkúrí nyɛrɛsɛ. 
Uu pi mɑɑ rɛ: Úye un nɛ̃́ nnyɑ kɛwɑ̃́ nkɛ́ kusɑ́nɛ n yɔsí, tinɛ́cúruu kuu kusɑ́nɛ yɔsí. Úye un kɔ kusɑ́nɛ nɛ́ n yɔsí, uyɛɛ nɛ́ n tummɛ kuu kɔ kusɑ́nɛ yɔsí. Likumúŋɛ́ rɛ úye un nɔ́kɛnɛ́cɔpɛ usíńsɑ́, uyɛɛ uwɛ́ɛ́sɛ. 
Yohɑni uu rinɔ́ɔ yɔ́su rɛ: Sɑ́ɑ, tɔ unyinɛ yɛnlɛ yɛɛ ɑníri n lɑ́kɑsɛ nɛ ripɔ́nyíri nnɑŋɛ ɑrí n lɑ tɔkɛ́ nɛ u yɛ̀, kúu kɛrɔ́cɔpɛ ń we tɔn nɛ pɔ́ tíkilɛ̃ nnyɑ. 
Amɑ́ Yeesu uu rinɔ́ɔ u pɛsɛ rɛ: Áni kɑpɛ nɛ u yɛ̀, liriyíkí rɛ úye un nɔ́kɛnɛ́cɑ́ɑ́ n kpɑ́lɛ̃, uyɛ̃ úu nɔ́unɛ́lɑ́ɑrɔ. 
Kɛ kumúŋɛ́ kpɛ̃ kɛ Yeesu uu kɛyómɛ-mɛ̃ ń tɑɑ́nɛ ɑku n tu, Yeesu uu hɔ́ rɛ ukɛ́ Yerusɑlɛm hɑ. 
Uu pisoi pinyinɛ kɛkpéẽ tɔ́ŋsɛ, ɑpi Sɑmɑrii kuyu kunyinɛ sĩ tɛ pikɛ́ lɛ̃ nɛ lɛ̃ kɑi u m pisɛ u wɛ́ɛ́si ukɛ́ kɛlenɛ tuipɔ. 
Amɑ́ kuyu kpɛ̃ pikɔ́ ɑ́pi ŋmurɛi pikɛ́ kusɑ́nɛ u yɔsí, kuu Yerusɑlɛm-mɛ̃ n símpɔ nnyɑ. 
Kɛ upipirɛtiki Yɑkupu nɛ Yohɑni ɑpi lɛ̃ n yɛ̃́ ɑpi u mɑɑ rɛ: Upíimɑ, pɔ lɑlɛ rɛ tɔkɛ́ nnɑ rinɔ́ɔ hɛ nkɛ́ pikɛcɑ́ɑ́ cuunu nkɛ́ pi torɛɛ? 
Yeesu uu pikuwɛ́lɛ́-mɛ̃ pɑnsɛpɔ uú nɛ pi cɛ́si uu pɛɛ pi mɑɑ rɛ: Áni nyu kɛmúŋɛ́ kɛkɛɛ nɔ́kɛnɛ́mɛɛ ń we. Usoi Kɛpipi ɑ́kɛ kɑ rɛ kɛkɛ́ pisoi fóm, ɑmɑ́ kɛ kɑlɛ rɛ kɛkɛ́ piɑyu lɔ. 
Api pɛɛ yisi ɑpi kuyu féé-mɛ̃ sĩ. 
Pin ncée-i we kɛ utisi unyinɛ uu Yeesu mɑɑ rɛ: Nɛ pɔ́ tíkinɛ kei kɑɑ n símpɔnɛ nnɛ́í. 
Yeesu uu u mɑɑ rɛ: Sisɑ́ŋɑ́ yɛ siɑlééri mɑ́lɛ, sinúipi sin siɑyɑ́mɛ́ mɑ́, ɑmɑ́ nɛ̃́, Usoi Kɛpipi ɑ́m tíkɑ mɑ́ kɑm yɛ́ ríyu n tɑmpɔ kɛ́ wéntɛ. 
Uu usoi ucɔ mɑɑ rɛ: A nɛ́ ritiki. Amɑ́ usoi uu rɛ: Sɑ́ɑ, ɑ ncée nɛ́ hɛ kɛ́ hɑ unɛ́sɑ́ɑ kulɛsi kɛ́ pɛɛ kɛlenɛ pɔ́ ritiki. 
Yeesu uu rinɔ́ɔ u pɛsɛ rɛ: A tíyɛ pikpɔkpɔ pikɛ́ pikpɔkpɔ kulɛsi, pɔ́ɔkɛ́ hɑ Uléécɑɑ iyɔɔpi nsímɛ́ símisi. 
Usoi ucɔ uu u mɑɑ rɛ: Nɛ pɔ́ tíkinɛ Upíimɑ, ɑmɑ́ ɑ ncée nɛ́ hɛ kɛ́ hɑ mɛnɛ́tɔ́ŋɛ́ pinɛ́kɔ́ símisi. 
Yeesu uu pɛɛ u pɛsɛ rɛ: Uyɛɛ nɛ inɑ́ɑ n kɔ́lɔi un pɛɛ kɛpirɛ ŋmɛ́ɛ́ni úu nɛ sɑ́ rɛ ukɛ́ Uléécɑɑ iyɔɔpi kɛ́mɛɛ ukɛikɔ́.
