Uléécɑɑ Pitumɛ Mɛwɑi.
4.
Piyɛɛ nɛ Yohɑni pin kei lɛ̃ pisoi símisi, mɛsɛ nɛ mɛsɛ, pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ nɛ Uléécɑɑ kɛyɔ umɛ́rɛ́ wɛ́ɛ́sɛ nɛ Pisɑtusee pin tuipɔ. 
Li pɛɛ mɛyɑ̃́ pi pɔ́ɔ́lúlɛ rɛ pitumɛ pɛ̃ kɛtɛ́ yɛ we pin pisoi céési pin kɔ pi símisi rɛ Yeesu yɛ pikpɔkpɔ kɛ́mɛɛ yisi, lin pɛɛ nyísɛlɛ̃ tɛ pikpɔkpɔ yɛ́ fe ɑpi yisi. 
Lɛ̃ nnyɑ kɑpi pitumɛ pɛ̃ kɛtɛ́ tĩ́ ɑpi hɑ́nɛsi ɑí nɛ weesi rɛ ituŋɛ yɛ pilólɛ́ mɑsí nnyɑ. 
Amɑ́ ḿpɑ́ nɛ lɛ̃, pɛpɛɛ pitumɛ pɛ̃ icélɑɑ n kṍ mɛyɑ̃ nɛ pɛɛ́ nɛ kɛfɑ i tɛnɛlɛ, ɑpi tulu yɑrɛ pitisi ɑ́kotokú ɑnupũ kumúŋɛ́. 
Kɑi n weesi, Pisuifi piyukɔɔ́ nɛ pipiwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpí hɑ Yerusɑlɛm-pɔ cɑ́pinɛ. 
Uléécɑɑ usinɑ Anɑ nɛ ukɛyɔ pikɔ́ nnɛ́í nɛ Kɑyifu nɛ Yohɑni nɛ Alɛkisɑntiri pɛɛ kɔ pɛɛ kei we. 
Api Piyɛɛ nɛ Yohɑni lesɛri pikɛ́ nɛ pi túhɑɑnɛ, ɑpi kɛcɔpɛ pi nyɛrɛsɛ ɑpi pi pisɛ rɛ: Ńnɑŋɛ ń-ye nɛ́ɛ úye kɛcirɛ rinyiri kɑní nɛ lɛ̃ wɑ? 
Nfɑ́ɑsɔnɛ nn Piyɛɛ yipu uu pɛɛ pi mɑɑ rɛ: Nɔ̃́ Isirɑyɛɛli kuyu piyɔɔpi nɛ piwɛ́ɛ́sɛ! 
Mɛwɑi sɔnɛ kɑri utisi nkó wɑ nnyɑ kɑpi nɛni kɛyɑɑ nkɛ́ rɔ́ písɛi rɛ íye kɑi wɑ kuú nɛ peí? 
Lɛ̃ nnyɑ, nɔ́nnɛ́nɛ́í mmú nɛ Isirɑyɛɛli pikɔ́ picɔ nnɛ́í ɑni kõ tɛ Nɑsɑrɛti ukɔ́ Yeesu Kirisi uyɛ̃ kɑni kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ ń kɑrii, Uléécɑɑ uu pikpɔkpɔ kɛ́mɛɛ u yukusɛ rinyíri nnɑŋɛ kɑrí nɛ utisi nkó péisɛ, kuu nní isɑrɛ lɑ́ɑ́rú un nɔ́kɛnɛ́yu-i nyɛnu. 
Yeesu uyɛ̃ kɛcɑ́ɑ́ kɑi wɔ́lɑɑlɛ̃ tɛ: Tipɑrɛ tɛ̃ kɛ nɔ̃́ pimɔ́mɛ ɑni n fómpɔ, tɛɛ nkpéni pɑnsɛ tikɛcirɛ tɛɛ kɛnui-i ń we. 
Uyɛ̃ mɛmɑ́ɑ́, úyulɑlɛ úkɑ ucɔ úu we. Uléécɑɑ úu rinyíri ríkɑ ricɔ kɛtẽ nkɛ́ kɛcɑ́ɑ́ pɑ kɑri yɛ́ n sée likɛ́ nɛ tirɔ́yu n lɔ. 
Piyɛɛ nɛ Yohɑni ɑ́pi mɛnyuwɛ píimɑ mɛ́kɑ mɑ́ ɑ́pi kɔ píyukɔɔ́ píkɑ. Amɑ́ piripɔɔ kɑpi pɛɛ́ nɛ símisi nɛ itisi. Kɛ pitúhɑɑnɛ ɑpi lɛ̃ n yɛ̃́, ɑi pi muuni simm, ɑpi kɔ ceru rɛ Piyɛɛ nɛ Yohɑni yɛ pɛɛ́ nɛ Yeesu sɔ́nɛlɛ. 
Kɑpi usoi yɛɛ ḿ peí n nyɑ́ni un pikɛkúrí nyɛnu nnyɑ, pitúhɑɑnɛ pɛ̃ ɑ́pi pɛɛ nyu ń-ye kɑpi yɛ́ mɑlɛ. 
Api pɛɛ Piyɛɛ nɛ Yohɑni kɛtúhɑɑnɛ kɛ-i lɑ́kɑsɛ, ɑpi pɛɛ kei n we pin pimɛcɔpɛcirɛ nsímɛ́ píŋirisɛ. 
Api pɛɛ m písɛinɛ rɛ: Íye kɑri pisoi mpí wɑinɛ? Yerusɑlɛm kuyu nnɛ́í yɛ pikómɛ mɑsí rɛ pi mɛwɑisɑŋɑ píimɑ wɑ. Ári yɛ́ fe tɔkɛ́ kɛ́si. 
Lɛ̃ nnyɑ, nsímɛ́ mmɛ̃ ńn kɑpɛ nɛ pikɔ́ɔ́nú rinkpɑ́, tɔkɛ́ nɛ pi yɛ ɑri pi cɛ́si rɛ ɑ́pi kɑpɛ úkɑ mpuri nɛ Yeesu uyɛ̃ tinyiri ńkɑ pisímɛ́ rikpɑ́. 
Kei kɑpi pɛɛ pi séepɔ ɑpí nɛ pi yulu rɛ ɑ́pi kɑpɛ Yeesu nnɑŋɛ nɛ picélɑɑ rikpɑ́. Ápi kɑpɛ yɛ ricuruu urinyiri sée. 
Piyɛɛ nɛ Yohɑni ɑpi pi pɛsɛ rɛ: Nɔ̃́ ticuruu ɑni ripɑí nkɛ̃́ tɛ li nɛ Uléécɑɑ kɛ́mɛɛ sɑ́ rɛ usoi ukɛ́ nɔ́ pɑkɑrɛ nɛ́ɛ Uléécɑɑ kuu yɛ́ pɑkɑrɛ? 
Nɔ́ɔ yɛ́ nkpéni mmɛ̃ ceri. Tɔ̃́ ɑ́ri yɛ́ fe tɔkɛ́ lɛ̃ kɑri n yɛ̃́ ɑri kɔ kom pisímɛ́ riyɑ́. 
Kei kɛ pitúhɑɑnɛ pɛ̃ ɑpi kɔ picɛ́ pi rikpɑ́ ɑpi pi yɑ́ ɑpi tɔ́mpɔ. Ápi ńkɑ yɛ̃ kɑpi yɛ́ nɛ rinwɑ́ɑ́nɑ́ pikɛ́ nɛ kutu pi rimpɑ́sɑ. Pisoi mɛyɑ̃ yɛ pɛɛ́ welɛ pin mɛwɑi mɛɛ nní n wɑ nnyɑ Uléécɑɑ pɑkɑrɛntɛ. 
Utisi uyɛ̃ kɑpi nní nɛ Uléécɑɑ nnɑŋɛ m pɔisɛ ɑŋmɛ̃ yɛ ɑfɛɛnɑ rifɑɑulɛ. 
Kɑpi Piyɛɛ nɛ Yohɑni rin-yɑ́, ɑpi yisi ɑpi pipisoi lɛɛpɔ ɑpi lɛ̃ kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Isirɑyɛɛli piwɛ́ɛ́sɛ ɑpi pi ḿ mɑ pi kɛɛni. 
Kɛ pipisoi pɛ̃ ɑpi lɛ̃ n kṍ, ɑpi rinɔ́ɔ risɛ wɑi ɑpi Uléécɑɑ yɑ́ɑ́si rɛ: Upíimɑ Utɛnɛcirɛ́, pɔ́ɔ kɛtẽ nɛ kɛyómɛ nɛ mínimɑɑ nɛ lɛ̃ nnɛ́í lɛɛ ḿpɑ́ yei ń we wɑ. 
Mpɔ́fɑ́ɑsɔnɛ kɑɑ́ nɛ ritiki ɑɑ́ nɛ urɔ́sɑ́ɑyɑhɑ Tɑfiti upɔ́kɛikɔ́ mɑɑsɛ rɛ: Yo nnyɑ kɛ mpuri sɑnɛ pikɔ́ ɑpi kɛtẽ kɛcɑ́ɑ́ pɑnsɛntɛ? Yo kɛcirɛ nnyɑ kɛ sisoipipi ɑsi simúŋɛ́ kópɛ wɑi? 
Kɛtẽ kɛcɑ́ɑ́ piyɔɔpi yɛ pimɛcirɛ cɑ́pinɛ rɛ pikɛ́ ɑtɔpi leemɛ. Piyɔ́ɔpi ɑpi rinɔ́ɔ risɛ wɑi rɛ pikɛ́ nɛ Upíimɑ nɛ uyɛ̃ kuu n wɛ́ɛ tɔ. 
Mɛyíkíyiki kɛ Erooti nɛ Pɔnsi Pilɑti nɛ Isirɑyɛɛli pikɔ́ nɛ mpuri sɑnɛ pikɔ́ ɑpi Yerusɑlɛm nté rinɔ́ɔ risɛ wɑ rɛ pikɛ́ nɛ upɔ́kɛikɔ́ Yeesu kɑɑ n wɛ́ɛ tɔ. 
Lɛ̃ kɑɑ mɛkɛɛ-mɛ m musí, ɑɑ́ nɛ mpɔ́nɑŋɛ yekei kɑpi nkpéni lɛ̃ wɑ. 
Lɛ̃ nnyɑ, ɑ nkpéni pipicɛ́ ńsɔnɛ kutu ricɔ, Upíimɑ ɑɑ tíyɛsɛ pipɔ́kɛikɔ́ pikɛ́ itisi nɛ mpɔ́símɛ́ riyóó. 
A mpɔ́nɑŋɛ nyísɛ pitóikɔ́ pikɛ́ pélesi, ɑɑ mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ upɔ́kɛikɔ́ Yeesu kɑɑ n wɛ́ɛ rinyiri nɛ wɑpisi. 
Kɑpi lɛ̃ piyómɛyɑ́hɑɑ ḿ mɑsí, kɛlõ kɛ-i kɑpi n cɑ́pinɛlɛ̃ ɑkɛ yéŋesi. Nfɑ́ɑsɔnɛ nn pinnɛ́í yipu ɑpi pɛɛ itisi nɛ Uléécɑɑ nsímɛ́ n yóólɛ̃. 
Pisoi kulúi kpɛ̃ nnɛ́í pɛɛ nɛ Yeesu kɛfɑ n tɛnɛ yɛ pɛɛ kɛfɑ pɛ́nɛlɛnlɛ pin kɔ kɛmúŋɛ́ kɛsɛ mɑ́. Úkɑ úu yɛ pɛɛ mɑɑ rɛ u linyinɛ umɛcirɛ te. Ḿpɑ́ yo kɑpí pɛɛ pɛ́nɛlɛ̃ pin te. 
Uléécɑɑ yɛ pɛɛ pitumɛ ńnɑŋɛ hɛlɛ pin nɛ nyísɛ rɛ Upíimɑ Yeesu yɛ nkpɔ kɛ́mɛɛ yisilɛ, Uléécɑɑ un pɛɛ Kirisi pikɔ́ nnɛ́í mɛyɑ̃́ kpɑ́lɛ̃. 
Líkɑ ɑ́i pɛɛ piukɑ pɑ́rílɛ̃. Pɛpɛɛ sicɑrɛ nɛ́ɛ síyɔ m mɑ́ yɛ̀ɛ̀ pɛɛ si yɑ́ɑlɛ ɑpí nɛ lisiwóó cɑ́pinɛpɔ 
ɑpi pitumɛ pɑ. Pɛ̃ pɔ́ɔ ḿpɑ́ úye ncɔ́ŋ pɑí ɑpí nɛ si hɔɔnɛntɛ. 
Pi pɛɛ Lefi mpuri ukɔ́ unyinɛ Siipuru kɛtẽ-pɔ mɑrilɛ. Pi yɛ u sée rɛ Yosɛfi. Pitumɛ pɔ́ɔ rinyíri u hɛ rɛ Pɑrinɑpɑsi. Liɑsei rɛ uyɛɛ yɛ picɔ rinŋmɔ́ɔ́púsɛ. 
Uyɛ̃ uu kɔ ukɛcɑrɛ yɑ́i uú nɛ lisiwóó hɑpɔ uu pitumɛ pɑ.
