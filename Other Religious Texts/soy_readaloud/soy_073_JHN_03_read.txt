Yohɑni.
3.
Utisi unyinɛ yɛɛ pɛɛ́ we, ɑpi yɛ u séi rɛ Nikotɛm. U pɛɛ Pifɑrisi kunɑi kɛ́mɛɛ welɛ un kɔ Pisuifi uwɛ́ɛ́sɛ usɛ unyinɛ. 
Uu kɛsinɛ kɛnyinɛ Yeesu-i sĩ́ uú hɑ u mɑɑ rɛ: Ucélɑɑ, tɔ nyulɛ rɛ Uléécɑɑ yɛɛ pɔ́ tummɛ rɛ ɑ rɔ́ céési. Uléécɑɑ unsɑ́ usoi kɛpirɛ n we, úu yɛ́ fe ukɛ́ mɛwɑisɑŋɑ mmɛ́ kɑɑ nní n wɑi mɛcɔ n wɑi. 
Yeesu uu u pɛsɛ rɛ: Kɛ́ pɔ́ símisi ɑsei kɛcɑ́ɑ́ rɛ pinsɑ́ úye pimɑ́rɛ rinkpɑ́, úu Uléécɑɑ iyɔɔpi yɛnunɛ. 
Kei kɛ Nikotɛm uu pɛɛ u pisɛ rɛ: Íye kɑpi yɛ́ kɔ fe ɑpi usoi yɛɛ pikpurũ ḿ mɑsí pimɑ́rɛ kpɑ́? Yɑrɛ u yɛ́ fe uu kɔ úni kɛfɑ-i pɛɛpɔ ɑpi kɔ pimɑ́rɛ u kpɑ́ nɛ́ɛ? 
Yeesu uu u pɛsɛ rɛ: Kɛ́ pɔ́ símisi ɑsei kɛcɑ́ɑ́ rɛ pinsɑ́ míni nɛ Uléécɑɑ Nfɑɑ kɛcɑ́ɑ́ rintiki pikɛ́ nɛ usoi pimɑ́rɛ rikpɑ́, úu Uléécɑɑ iyɔɔpi yɛnunɛ. 
Lɛ̃ kɛ kɛsoipipi ɑkɛ m mɑ́ri yɛ lisoikɔ́ lɛ. Lɛ̃ kɛ Uléécɑɑ Nfɑɑ nn nfɑ́ɑ n hɛ yɛ Nfɑ́ɑsɔnɛ likɔ́ lɛ. 
Ái kɑpɛ pírí pɔ́ wɑ rɛ nɛ mɑɑ rɛ li pisɛ rɛ pikɛ́ pimɑ́rɛ nɔ́ rikpɑ́ nnyɑ. 
Kuyɔ kun yei n lɑ, kei kɑku yɛ m pepu, ɑɑ kunnɛ̃́ n kómɛi, ɑmɑ́ ɑ́ɑ pɛɛ nyu mɛ́ye kɑku léeri, mɛ́ye kɑku sĩ́. Limɛcɔ kɛ uyɛ̃ kɛ Uléécɑɑ Nfɑɑ nn nfɑ́ɑ n hɛ pɔ́ɔ kɔ we. 
Kei kɛ Nikotɛm uu kɔ pipisɛ u rikpɑ́ rɛ: Íye kɑi yɛ́ fe ɑi lɛ̃ wɑi? 
Yeesu uu u pɛsɛ rɛ: Pɔ Isirɑyɛɛli kɛtẽ kɛ́mɛɛ ucélɑɑ wɛ́ɛ́sɛ lɛ. Yo nnyɑ kɑ́ɑ pɛɛ liɑsei kómɛi? 
Kei kɛ Yeesu uu pɛɛ u mɑɑ rɛ: Kɛ́ pɔ́ símisi ɑsei kɛcɑ́ɑ́ rɛ lɛ̃ kɑri n nyu nkɔ́ kɑri símisi. Lɛ̃ kɑri n yɛ̃́ kɛcɑ́ɑ́ kɑri ɑseérɑ lesɛ. Amɑ́ nɔ̃́ ɑ́ni lɑ nɔkɛ́ ɑrɔ́seérɑ nyɛ̃ ŋmurɛi. 
Nɛ yɛ nɛn lɛlɛɛ kɛtẽ nté n wɑi nsímɛ́ nɔ́ n símisi, ɑ́ni yɛ ŋmurɛi. Íye kɑni yɛ́ pɛɛ fe ɑni ŋmurɛi in tɛ nɛ lɛlɛɛ kɛyómɛcɑɑ-pɔ n wɑi nsímɛ́ nɔ́ símisi? 
Úkɑ úu píkɑi kɛyómɛcɑɑ hɑ́pɑɑlɛ̃, insɑ́ Usoi Kɛpipi kɛɛ kɛyómɛcɑɑ-pɔ n léeri mɛcirɛ. 
Moisi yɛ pɛɛ nweni ŋmɔhɔ nɛ iwɑ́ɑ wɑlɛ uu kucɛsi kóimɑ-i kunɑ kɛcɑ́ɑ́ kɔ́lɔ. Limɛcɔ kɑpi sɔ́nti pikɛ́ Usoi Kɛpipi kunɑ kɛcɑ́ɑ́ rikɔlɔ. 
Úye un pɛɛ́ nɛ kɛfɑ u n tɛnɛ, liute ukɛ́ nɛ nfɑ́ɑ tɛnɛcirɛ́ n yɛ̃́ nnyɑ. 
Li we rɛ Uléécɑɑ yɛ pisoi mɛyíkíyiki lɑlɛ hɑ́i uú nɛ Ukɛpipi Cirɛníŋɛ pɑmɛ rɛ úye un nɛ kɛfɑ kɛ n tɛnɛ, liute úu kɑpɛ pɔ, ɑmɑ́ ukɛ́ nfɑ́ɑ tɛnɛcirɛ́ yɛ̃́. 
Asei kɛcɑ́ɑ́, Uléécɑɑ úu Ukɛpipi kɛtẽ kɛcɑ́ɑ́ tummɛ rɛ kɛkɛ́ nɛ pisoi túhɑɑnɛ likɛ́ nɛ pi kpu. Amɑ́ u kɛ tummɛlɛ rɛ kɛkɛ́ piriyu lɔ. 
Úye un nɛ kɛfɑ kɛ n tɛnɛ, Uléécɑɑ úu yɛ́ uyɛ̃ nɛ túhɑɑnɛ likɛ́ nɛ u kpu, ɑmɑ́ úye unsɑ́ nɛ kɛfɑ kɛ n tɛnɛ, Uléécɑɑ yɛ nɛ uyɛ̃ pitúhɑɑnɛ mɑsilɛ rɛ úu Uléécɑɑ Kɛpipi Cirɛníŋɛ rinyiri nɛ kɛfɑ tɛnɛ nnyɑ. 
Lɛlɛɛ n tíyɛsɛ kɛ Uléécɑɑ uú nɛ pisoi n túhɑɑnɛnɛ yɛɛ rɛ mɛtɛ́í yɛ kɛtẽ kɛcɑ́ɑ́ léemɛlɛ, kɛ pisoi ɑpi mɛwɑi kópɛ n wɑi nnyɑ, ɑpi mɛtɛ́í yúlu ɑpi pɛɛ kuŋmɑhɑ mɛyɑ̃́ n lɑ. 
Li we rɛ nkó yɛɛ mɛwɑi kópɛ n wɑi yɛ mɛtɛ́í kɔ́hɔlɛ, úu yɛ nɛ mɛ rikɔ, un wuru rɛ ɑ́i kɑpɛ umɛwɑi kópɛ yɑ́mnɛ. 
Amɑ́ nkó yɛɛ nsímɛ́sei n tíkilɛ̃ yɛ̀ɛ̀ mɛtɛ́í kɛ́mɛɛ léepɔlɛ likɛ́ nyísɛ rɛ Uléécɑɑ kɛ́mɛɛ kɛ umɛwɑi nnɛ́í ɑmɛ léeri. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu yisi u nɛ upipirɛtiki ɑpi Yutee kɛtẽ-mɛ̃ tɔ́mpɔ. U nɛ pɛ̃ ɑpi kei siyɑ́ɑ sinyinɛ wɑi un pisoi míni wolɛ. 
Yohɑni pɔ̃́ nɛ kɔ pɛɛ Enɔɔ kuyu-i welɛ uu yɛ pisoi míni wolɛ. Ái kei nɛ Sɑlim kuyu kɛtɑɑ we. Míni yɛ pɛɛ kei mɛyɑ̃́ welɛ. Pisoi pin pɛɛ mɛsɛ́rɛ ukɛmɛɛ sɔ́nɑɑpɔ un míni pi wolɛ. 
Ápi pɛɛ kɑhɑnɛ likumúŋɛ́ kpɛ-i u n tĩ pikɛ́ kpɑnii. 
Yohɑni pipirɛtiki pinyinɛ nɛ Usuifi unyinɛ ɑpi pɛɛ kɛyɑ́ɑ inyɛ́kii kɑpi yɛ nɛ usoi n nyɔ́ɔnsɛ kɛcɑ́ɑ́ nsímɛ́ pɑ́sɑinɛ. 
Apí nɛ Yohɑni lɛɛpɔ ɑpi u mɑɑ rɛ: Ucélɑɑ, pɔ nɔ́ nɛ utisi uyɛ̃ nɔ́ɔ pɛɛ Yuritɛɛ nkoi mɛtɛ́ŋ́-mɛ̃ ń we, ɑɑ ukɛcɑ́ɑ́ ɑseérɑ lesɛ léisɛɛ? Uyɛɛ nní we un míni pisoi wolɛ, pinnɛ́í pin ukɛmɛɛ sɔ́nɑɑpɔ! 
Yohɑni uu pi pɛsɛ rɛ: Uléécɑɑ un ńnɑŋɛ pɔ́ n hɛ rɛ ɑ nɛ linyinɛ wɑ mɛcirɛ kɑɑ yɛ fe ɑɑ li wɑi. 
Nɔ́rinɛ́cúruu yɛ́ fe ɑni ɑseérɑ kɛnɛ́cɑ́ɑ́ lesɛ rɛ nɛ pɛɛ́ mɑɑ rɛ: Ái nɛ́ɛ Kirisi kɛ Uléécɑɑ uu n wɛ́ɛ. Amɑ́ Uléécɑɑ yɛ nɛ́ tummɛlɛ rɛ kɛ́ nɛ uyɛ̃ kuu n wɛ́ɛ kɛkpéẽ wɑimɛ. 
Nkó yɛɛ unɔ́si n kpísi yɛɛ unɔ́sikɔ́. Amɑ́ unɔ́sikɔ́ uyɛ̃ usɑnɛ pɔ̃́ nɛ yɛ ukɛkúrí n welɛ un unɔ́sikɔ́ mɛtẽ kómɛi lin u lɑ́ɑ́rú. Limɛcɔ ŋmɑɑ kɑi nní mɛyíkíyiki nɛ́ lɑ́ɑ́rú, uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ nnyɑ. 
Li nɛ sɑ́ rɛ uyɛ̃ tinyiri tikɛ́ n tɑɑ́lɛ̃, tinɛ́kɔ́ tikɛ́ n cɛpilɛ̃. 
Nkó yɛɛ kɛyómɛcɑɑ-pɔ n léeri yɛ ḿpɑ́ úye felɛ. Nkó yɛɛ kɛtẽ nté ń we yɛ kɛtẽ nté ukɔ́ lɛ. Kɛtẽ nté likɔ́ nsímɛ́ kuu yɛ n símisi. Nkó yɛɛ kɛyómɛcɑɑ-pɔ n léeri yɛ ḿpɑ́ úye felɛ. 
Lɛ̃ kuu n yɛ̃́ uu kɔ kom ɑseérɑ kuu lésɛ, ɑmɑ́ úkɑ úu kɔ pɛɛ lɛ̃ kuu ḿ mɑ ŋmurɛi. 
Uyɛɛ lɛ̃ kuu ḿ mɑ n ŋmurɛi yɛ mɑlɛ uu wééri rɛ Uléécɑɑ yɛ ɑsei lɛ. 
Li we rɛ Uléécɑɑ nsímɛ́ kuu ḿ mɑ kɛ uutumɛ uu yɛ símisi. Lɛ̃ nnyɑ kɛ Uléécɑɑ uu yɛ uyɛ̃ kuu n tũ Nfɑ́ɑsɔnɛ hɛ ɑ́i kumúŋɛ́ mɑ́. 
Sɑ́ɑ Uléécɑɑ yɛ Ukɛpipi lɑlɛ uu ḿpɑ́ yo kɛɑnipɛ-i wɑi. 
Nkó yɛɛ Uléécɑɑ Kɛpipi nɛ kɛfɑ n tɛnɛ yɛ nfɑ́ɑ tɛnɛcirɛ́ mɑ́lɛ. Nkó yɛɛ kɛnkɔ́ n kɛ́su úu yɛ́ nfɑ́ɑ mmɛ̃ yɛ̃. Amɑ́ Uléécɑɑ yɛ nɛ u wɔ́lɑɑlɛnlɛ.
