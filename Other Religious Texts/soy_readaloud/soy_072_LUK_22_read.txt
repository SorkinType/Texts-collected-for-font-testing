Luki.
22.
Kɛ ɑnyɑ̃́ nyɛ-i kɑpi yɛ ɑkpɔ́nɔ́ nyɛɛ ɑ́ɑ pitɛ m mɑ́ n li ɑɑ n nyɑhɑimɛ, 
pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpi ncée n wɛ́ɛ́si kɑpi yɛ́ nɛ Yeesu ń kpu, ɑmɑ́ pin pɛɛ kuyu pikɔ́ wuru. 
Setɑni uu Yutɑsi kɑpi yɛ n sée rɛ Isikɑriyɔɔti loni. U pɛɛ pipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ usɛ lɛ. 
Yutɑsi nɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ kuyómɛyɑ́hɑɑlee pimɛ́rɛ́ piwɛ́ɛ́sɛ ɑpi lɛ̃ kuu yɛ́ Yeesu n tĩ́ ukɛ́ piɑnipɛ-i n wɑ kɛcɑ́ɑ́ kómɛinɛ. 
Ai pi lɑrisi ńsɔnɛ ɑpi rinɔ́ɔ yekei rɛ pi yɛ́ siwóó u hɛ́ɛ́lɛ. 
Yutɑsi uu ŋmurɛi uu icée sɔnɛ n wɛ́ɛ́si ukɛ́ nɛ Yeesu pi rimúísɛ risoiwuí ɑ́ri kɑpɛ n nyu. 
Kɛyɑ́ɑ kɛ̃ kɑpi yɛ ɑkpɔ́nɔ́ nyɛɛ ɑ́ɑ pitɛ m mɑ́ n li, kɛyɑ́ɑ kɛ̃ kɑpi yɛ kɔ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ isɑŋ n kṍ ɑkɛ tulu. 
Yeesu uu pɛɛ Piyɛɛ nɛ Yohɑni tum uu rɛ: Ani hɑ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ rɔ́ wɑ tɔkɛ́ li. 
Api u pisɛ rɛ: Yei kɑɑ́ lɑ tɔkɛ́ hɑ liilukɛ wɑ? 
Uu rinɔ́ɔ pi yɔ́su rɛ: Ani kutu ricɔ: Nɔn hɑ píyei kuyu n lompɔnɛ, utisi unyinɛ yɛɛ kɛcɔ́rípi nɛ míni n topori yɛ́ nɛ nɔ́ risɑ́nɛ. Ani u ritiki hɑ́i nɛ kɛ́yɔ kɛ-i kuú hɑ n loninɛ. 
Ani úyɔte mɑɑ rɛ: Urɔ́sɑ́ɑ yɛ pɔ́ pisɛ rɛ yei yɛ kulee kpɛ-i kɛ u nɛ upipirɛtiki ɑpi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ n lenɛ? 
U yɛ́ kɛ́yɔ kɛcɑ́ɑ́ kulee mɑɑ kunyinɛ kɑpi n nyɔ́ɔnsɛ nɔ́ nyísɛ. Kei kɑni yɛ́ ɑnyɑ̃́ ilukɛ wɑ. 
Api símpɔ ɑpi ḿpɑ́ yo lɛɛpɔ yɑrɛ kɛ Yeesu uú pɛɛ pi ḿ mɑ, ɑpi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ wɑi. 
Kɛ ituŋɛ ii n tu, Yeesu nɛ upitumɛ ɑpí tonɛ rɛ pikɛ́ li. 
Uu pi mɑɑ rɛ: Li pɛɛ mɛyɑ́nsei ripɔ́ɔ nɛ́ wɑ rɛ tɔkɛ́ kɛsẽ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ nnyí li kɛ́ kɛlenɛ íwɛ li! 
Asei kɑm nɔ́ símisi rɛ ɑ́m pilukɛ i kpɑ́nɛ hɑ́i nɛ kumúŋɛ́ kpɛ-i kɛ Uléécɑɑ uu yɛ́ uiyɔɔpi kɛ́mɛɛ rɔ́ i n wɑ ɑri pɛɛ iɑsei nnɛ́í kóm. 
Uu pɛɛ kɛpóripi nɛ pítɑ kpísi uu Uléécɑɑ pɔɔnɛsɛ uu rɛ ɑni yɔsí ɑni hɔɔnɛ. 
Asei kɑm nɔ́ símisi rɛ ɑ́m píkɑi ɑléépipi pitɑ mpí pinírɛ́ kpɑ́nɛ hɑ́i nɛ kɛyɑ́ɑ kɛ̃ kɑm yɛ́ Uléécɑɑ iyɔɔpi kɛ́mɛɛ pi ń ntí. 
Uu kukpɔ́nɔ́ kpísi, kuu Uléécɑɑ m pɔɔnɛsɛ mɛmɑ́ɑ́, uu ku kpɔkɔ́ri uu pi ku pɑ uu rɛ: Inɛ́piŋɛ yɛ nnyí kɑm nɔ̃́ nnyɑ m pɑ́lɛ̃. Ani yɛ lɛ̃ wɑ ɑní nɛ kɛnɛ́cɑ́ɑ́ léisɛ. 
Ilukɛ mɛlukɛ-mɑɑ́, uu kɔ kɛ́tɑporipi pi pɑ uu rɛ: Mɛnɛ́nyɛ yɛ mmɛ́, mɛ́nyɛ mɛɛ nɔ̃́ nnyɑ n kɔɔnu Uléécɑɑ uú nɛ nɔ́ séi rɛ nɔ́ nɛ uyɛ̃ ɑni kómɛinɛ. 
Amɑ́ ɑni ripɑí, uyɛɛ nɛ́ n yɑ́inɛ yɛ nté welɛ, tɔ́ nɛ uyɛ̃ tɔn kɛsẽ́ le. 
Mɛsei yɛ mɛ rɛ Usoi Kɛpipi yɛ tɔ́sulɛ yɑrɛ kɑi kɛkɛcɑ́ɑ́ n wɔ́lɑɑmɛlɛ̃. Amɑ́ usoi nkó yɛɛ yɛ́ Usoi Kɛpipi n yɑ́ɑ́, íwɛ kɛ liute uu topori. 
Api pɛɛ pimɛcɔpɛcirɛ pipísɛinɛ koru rɛ pikɛcɔpɛ úye yɛɛ yɛ́ mɛwɑi mɛ̃ wɑ. 
Pipirɛtiki ɑpi íkɛŋɛnɛ kɑ́pɑ́ɑ́ rɛ pikɛ́ yɛ̃́ tɛ pikɛcɔpɛ úye yɛɛ yɛ́ uwɛ́ɛ́sɛ. 
Yeesu uu pi mɑɑ rɛ: Sitẽ piyɔɔpi yɛ sikɛcɑ́ɑ́ túnlɛ, pɛpɛɛ sikɛcɑ́ɑ́ iyɔ́ɔpi n le yɛ̀ɛ̀ tíyɛlɛ ɑpi pi n sélei rɛ “lisɔnɛ piwɑi.” 
Amɑ́ ɑ́i kɑpɛ lɛ̃ nɔ́kɛnɛ́mɛɛ́ n we. Nɔ́kɛnɛ́cɔpɛ úyukɔɔ́ kɑi pisɛ rɛ ukɛ́ usíńsɑ́. Li pisɛ rɛ úyukɔɔ́ ukɛ́ n we yɑrɛ ukɛikɔ́. 
Ukɛikɔ́ yɛɛ ilukɛ n wɑi nɛ uyɛ̃ kuu ilukɛ n wɑi kɛcɔpɛ úye yɛɛ úyukɔɔ́? Ái uyɛ̃ kuu i n wɑii? Too, nɛ̃́ nɛ nɔ́kɛnɛ́cɔpɛ we yɑrɛ ukɛikɔ́ uyɛɛ ilukɛ n wɑi. 
Nɔ nɛ́ ritikilɛ nɛ ɑsei, nɔn kɑhɑri tɔ́ nɛ nnɔ̃́ tɔn nɛ ituŋɛ iyɛ-i kɑi nɛ́ ḿ peikɛɛ kɛ́mɛɛ we. 
Lɛ̃ nnyɑ kɑm Uléécɑɑ iyɔɔpi nɔ́ pɑnɛ yɑrɛ kɛ Unɛ́sɑ́ɑ uu nɛ́ i ḿ pɑ mɛcɔ. 
Tɔ́ nɛ nnɔ̃́ yɛ́ inɛ́yɔ́ɔpi-i kɛsẽ́ tonɛ ɑri le ɑri kɔ níru, ɑni sinɛ́yɔ́ɔpitonɛ-i tonɛ ɑni Isirɑyɛɛli ɑmɑrɛ kɛfi nɛ ɑtɛ́ nyɛ̃ túhɑɑnɛ. 
Simɔɔ, Simɔɔ, ɑ kutu ricɔ, Setɑni yɛ ncée we rɛ ukɛ́ pɔ́ pɑ́ŋɑi yɑrɛ kɑpi yɛ ɑsírɛ́ m pɑ́ŋɑi ɑpi ɑcɔ́lɛ́ lésɛ. 
Amɑ́ nɛ nɛ Uléécɑɑ pɔ́ tẽ́ tɛ ɑ́i kɑpɛ wɑ rɛ mpɔ́fɑtɛnɛ yɛ nɛ pɔ́ kpu. Amɑ́ pɔn píyei kɛnɛ́mɛɛ m pɛɛmɛ, ɑ pipɔ́mɑ́rɛcɔ itisi rikpɑ́sɛ. 
Piyɛɛ uu u mɑɑ rɛ: Upíimɑ, nɛ ŋmurɛi rɛ ḿpɑ́ in kukpɑniilee yɛ ku kɛ́ pɔ́ ritiki, nɛ́ɛ ḿpɑ́ nkpɔ yɛ mu tɔkɛ́ kɛsẽ́ kpu. 
Yeesu uu rɛ: Piyɛɛ, nɛ pɔ́ mɑɑ rɛ icɑ̃́ ikɛ́ kɛlenɛ n kooi, pɔ́ mɛpehẽ mɛtɑɑni kɛsi rɛ ɑ́ɑ nɛ́ nyu. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Kɑm nɔ́ n tũ, ɑ́ni kɛwóó nɛ kulɔ́ɔ nɛ ɑnɛ́ɛ́ri rimmúlú, linyinɛ yɛ nɔ́ ripɑ́rílɛɛ? Api rɛ: Líkɑ ɑ́i tɔ́ ripɑ́rí. 
Uu pɛɛ pi mɑɑ rɛ: Amɑ́ úye un nkpéni kɛwóó m mɑ́, li pisɛ rɛ ukɛ́ kɛ rimúlú, úye un kɔ kulɔ́ɔ m mɑ́, li pisɛ rɛ ukɛ́ ku kpísi. Úye unsɑ́ kɔ titɔpitɛ́hɛ m mɑ́, li pisɛ rɛ ukɛ́ uilũ yɑ́ɑ́ uú nɛ risɛ lɔlu. 
Likumúŋɛ́ rɛ rinɔ́ɔ tɛ̃ kɑi kɛnɛ́cɑ́ɑ́ rítɛlɛ́-i n wɔ́lɑɑlɛ̃ tɛ: Pi piwɑikópɛkɔɔ́ kɛcɔpɛ u yekei yɛ wɑlɛ. Lɛlɛɛ kɛnɛ́cɑ́ɑ́ n wɔ́lɑɑmɛlɛ̃ yɛ tulɛ. 
Pipirɛtiki ɑpi u mɑɑ rɛ: Upíimɑ, ɑtɔpitɛ́hɛ ɑtɛ́ yɛ nnyɛ́. Uu rinɔ́ɔ pi yɔ́su rɛ: Li wɑlɛ. 
Yeesu uu yisi uu Olifiyee rikúú kɛcɑ́ɑ́ sĩ́ yɑrɛ kuu yɛ pɛɛ kei n sɔ́nɛ mɛcɔ. Upipirɛtiki ɑpi u tiki. 
Kɑpi kɛlõ kɛ̃ n tuipɔ, uu pi mɑɑ rɛ: Ani yɛ kɛyómɛ yɑ́ɑ́si ɑ́ni kɑpɛ nɛ pipeikɛɛ kɛ́mɛɛ ń loó nnyɑ. 
Uu lɛ̃ mɛmɑ́ɑ́ nɛ kɛtɑɑ pi wɑi, tinɑ̃ tɛɛ pikɛcɔpɛ ń we yɛ tu yɑrɛ kunípɛ ripɑrɛ mɛfomɛ kumúŋɛ́, uú hɑ wulɑ uu kɛyómɛ yɑ́ɑ́si rɛ: 
Sɑ́ɑ, in tɛ pɔ ŋmurɛi, ɑ tíyɛsɛ ntóósi kɛporipi nkɛ́ kɛkɛ́ nɛ kɛtɑɑ nɛ́ wɑ. Ḿpɑ́ nɛ lɛ̃, ɑ́i kɑpɛ mɛnɛ́lɑ mɛɛ yɛ́ wɑ, ɑmɑ́ likɛ́ mɛpɔ́kɔ́. 
Uléécɑɑtumɛ uu pɛɛ kɛyómɛcɑɑ-pɔ kpɑ́fúmɛ rɛ ukɛ́ itisi u hɛ. 
Mpɔ́ɔcɑɑi píimɑ nn Yeesu tini, uú nɛ piyómɛyɑ́hɑɑ kɛ́yu m mɑhɑnlɛ̃, urikɑhɑ ɑri pɔ́lɔ́ yɑrɛ mɛ́nyɛ, tin cɔlɔi. 
Kuu piyómɛyɑ́hɑɑ ḿ mɑsí, uu upipirɛtiki-mɛ̃ pɛlɛ uu pi lɛɛpɔ pin lɔni nɛ mpɔ́ɔcɑɑi. 
Uu pi mɑɑ rɛ: Yo nnyɑ kɑni lɔni? Ani yisi ɑni kɛyómɛ yɑ́ɑ́si ɑ́i kɑpɛ wɑ rɛ nɔ pipeikɛɛ-i lóipɔ. 
Un kei lɛ̃ símisi risoiwuí ɑri tuipɔ, upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ usɛ kɑpi yɛ n sée rɛ Yutɑsi un pikɛyu we. Uú nɛ Yeesu kɔ uu u pirɑ. 
Amɑ́ Yeesu uu u mɑɑ rɛ: Pɔ́ Usoi Kɛpipi ripirɑ ɑɑ́ nɛ pilɑ́ɑrɔ ɑnipɛ-i kɛ wɑi, nɛ́ɛ yo? 
Kɛ Yeesu pisencɔ ɑpi n yɛ̃́ lɛlɛɛ n wɑinɛ, ɑpi u pisɛ rɛ: Upíimɑ, tɔkɛ́ ɑtɛ́hɛ kpísii? 
Piusɛ uu Uléécɑɑ usinɑ ukɛikɔ́ kutu lukɛ-lukɛ seḿ. 
Amɑ́ Yeesu uu rɛ: Áni kɑpɛ lɛ̃ tikpɑ́, li wɑlɛ. Uu usoi uyɛ̃ kutu cɑ uu u pɔisɛ. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ pimɛ́rɛ́ uwɛ́ɛ́sɛ nɛ kuyu piwɛ́ɛ́sɛ pɛɛ pitínɛ u n hɑ́pɔ mɑɑ rɛ: Li pisɛ rɛ nɔkɛ́ pitínɛ nɛ́ kɑm nɔn ɑtɛ́hɛ nɛ ɑpúnípi mulɛilɛ̃ yɑrɛ usoi kpɑ́ɑ́ree? 
Siyɑ́ɑ nnɛ́í kɛ tɔ́ nɛ nnɔ̃́ ɑri yɛ kɛsẽ́ Uléécɑɑ kɛyɔ-i n we, ɑ́ni kei ncée wɛ́ɛsi nɔkɛ́ nɛ nɛ́ tĩ́. Amɑ́ ituŋɛ nnyí yɛ nɔ́ nɛ kuŋmɑhɑ uyɔɔpi ikɔ́ lɛ. 
Api Yeesu tini ɑpí nɛ Uléécɑɑ usinɑ kɛyɔ sĩ, Piyɛɛ un kɛtɑɑ-pɔ pi tikilɛ̃. 
Api kɛ́yɔcɔpɛ nnɑ wénesi ɑpí tonɛ, Piyɛɛ uu pɛpɛɛ nnɑ n wéni kɛcɔpɛ tonɛ. 
Unɔ́si kɛikɔ́ unyinɛ uu u yɛnu un nnɑ kɛkúrí tṹ, uu u nyɑ́nɛi tíḿḿ uu rɛ: Usoi nkó ricuruu yɛ pɛɛ Yeesu tikilɛnlɛ. 
Piyɛɛ uu kɛ́su, uu unɔ́si uyɛ̃ mɑɑ rɛ: Ám u nyu, unɔ́si. 
Kɑi rinkpɑ́sɛ, unyinɛ ucɔ uu u yɛnu uu rɛ: Pɔ pisoi pɛ̃ usɛ lɛ! Amɑ́ Piyɛɛ uu rinɔ́ɔ yɔ́su rɛ: Ám pikɛcɔpɛ we. 
Likɛ́ nɛ mɛsɛ pipépɛ rinkpɑ́, unyinɛ ucɔ uu kɔ ń-yupɑpɛ nɛ pimɑ́ɑ kpɑ́ rɛ: Asei kɛcɑ́ɑ́, usoi nkó yɛ pɛɛ u tikilɛnlɛ, likumúŋɛ́ rɛ u Kɑlilee ukɔ́ lɛ. 
Amɑ́ Piyɛɛ uu rinɔ́ɔ yɔ́su rɛ: Ám nyu lɛ̃ kɑɑ ń lɑ pɔkɛ́ mɑlɛ. Kumúŋɛ́ kpɛ-i kuu nní n símisi, icɑ̃́ ii kooi. 
Upíimɑ uu pɛɛ́ ŋmɛɛlú uu Piyɛɛ weríí. Piyɛɛ uu léisɛ rɛ Upíimɑ yɛ u mɑɑ rɛ: Icɑ̃́ ikɛ́ kɛlenɛ n kooi, pɔ́ kɛsi mɛtɑɑni rɛ ɑ́ɑ nɛ́ nyu. 
Kei kɛ Piyɛɛ uu léepɔ uú nɛ mpɔ́ɔcɑɑi téni. 
Pisoi pɛɛ pɛɛ Yeesu m mɛ̃́ ɑpi kɛcɑ́ɑ́ u nyɑ́nɛi ɑpi u kɑii. 
Api kɛ́yu u pi, ɑpi u mɑɑ rɛ: Wóo pɔ́ kɑ́ii, ɑ ŋmɔinu. 
Api kɔ ɑlɛ̃́ u kpɑ́ mɛyɑ̃́. 
Kɑi n weesi, Pisuifi piwɛ́ɛ́sɛ nɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpi cɑ́pinɛ. Api nɛ Yeesu pikɛtúhɑɑnɛ píimɑ-i sĩ́. 
Api u pisɛ rɛ: Pɔ́ɔ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ? A rɔ́ símisi. Uu rinɔ́ɔ yɔ́su rɛ: Nɛn nɔ́ m mɑɑ rɛ lɛɛ ni, ɑ́ni ŋmurɛinɛ. 
Nɛn kɔ nnyinɛ nɔ́ m pisɛ, ɑ́ni rinɔ́ɔ nɛ́ yɔ́sunɛ. 
Amɑ́ kɑí nɛ nkpéni rimpɑ́, nɔ́ Usoi Kɛpipi yɛ̃ kɛn Uléécɑɑ, ńnɑŋɛ nnɛ́í ute kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tṹ. 
Pinnɛ́í ɑpi pɛɛ rɛ: Lɛ̃ nnyɑ, pɔ Uléécɑɑ Kɛpipi lɛɛ? Uu rinɔ́ɔ yɔ́su rɛ: Lɛ̃ kɑni nní mɑ, nɛ Uléécɑɑ Kɛpipi lɛ. 
Api pɛɛ rɛ: Píseérɑkɔɔ́ píye picɔ kɑri nkpéni wɛ́ɛ́si? Tirɔ́cúruu yɛ uɑnɔɔlempi konlɛ!
