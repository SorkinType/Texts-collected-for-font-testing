Mɑriki.
12.
Yeesu uu pɛɛ lɛ̃ mɛmɑ́ɑ́ sinyɑ́rũ pi loni, uu rɛ: Utisi unyinɛ yɛɛ we, uu ɑléé nyɛ̃ kɑpi yɛ n sée rɛ fiinyi kɛcɑrɛ tɑmɛsi. Uu mmɛlɛ wɑi uú nɛ kɛ kɔ́ɔ́nú. Uu kɔ kɛcɑrɛ kɛ-i kuhórɛ túni uu kɛ́tɑcɛrii mɔm kei kɑpi yɛ́ pítɑ n cɛ́rii. Uu kɔ limɛmɑ́ɑ́ rikúsi cɑ́cɑ́ rinyinɛ mɔm uu pɛɛ pikɛikɔ́ wɛ́ɛ́si rɛ pikɛ́ kɛcɑrɛ kɛ̃ kɛcɑ́ɑ́ u m pɑílɛ̃ ukɛ́ ncée hɑ ukɛ́ kɑm. 
Kɑi ɑléépipi nyɛ̃ kutúũ n tu, uu ulɑ́si unyinɛ upikɛikɔ́ pɛ̃ kɛ́mɛɛ tumpɔ rɛ pikɛ́ ɑléépipi ɑnyinɛ u yɔ́ɔmɛ. 
Api u tini ɑpi kɑ́ii, ɑpi mɛnípɛ ŋmɑnɛ u lɑkɑsɛ. 
Uu kɔ ulɑ́si ucɔ pikɛmɛɛ tumpɔ, ɑpi uyɛ̃ tíyu kɛcɑ́ɑ́ kɑ́ii ɑpi u lɑ́misi. 
Uu utumɛ tɑ́ɑ́nũ tum, ɑpi uyɛ̃ kopu. Uu lɛ̃ mɛmɑ́ɑ́ pitumɛ mɛyɑ̃ túmɛipɔ, ɑpi picɔ kɑiinkɛɛ ɑpi kɔ picɔ kóni. 
Ucɑrɛte uyɛ̃ uŋmɑ́nɛ lɑlɑ mɛcirɛ kɑí pɛɛ u pɔɔnɛ. Uu pɛɛ mɛtɔ́rɔɔ uyɛ̃ tumpɔ un tee rɛ: Pi yɛ́ unɛ́ŋmɑ́nɛ wuri. 
Amɑ́ pikɛikɔ́ pɛ̃ ɑpi símɑɑnɛ rɛ: Nkó yɛɛ mɑsí usɑ́ɑ ikulɑnlũ nnɛ́í te. Ani kɑm tɔkɛ́ u kpu ɑri pɛɛ uikulɑnlũ iyɛ̃ n te. 
Kuu n tuipɔ ɑpi u tini, ɑpi kópu ɑpi kɛcɑrɛ ilúkɛ́ u sɑ́. 
Íye kɛ ucɑrɛte uu wɑinɛ? U werinɛ ukɛ́ pikɛikɔ́ pɛ̃ kṍ uu ɑléé nyɛ̃ picɔ pɑ. 
Áni Uléécɑɑ Nsímɛ́ ritɛlɛ́ kɛ́mɛɛ kɛ́ɛ́nlɛ̃ tɛ: Tipɑrɛ kɛ pimɔ́mɛ ɑpi n fóm tɛɛ pɑnsɛ ticírɛ-cirɛ tɛɛ kɛnui-i ń wee? 
Upíimɑ yɛɛ lɛ̃ wɑ. Mɛwɑisɑŋɑ kuu rɔ́ n yɛ́nɛisɛ yɛ mɛ̃. 
Pisuifi piwɛ́ɛ́sɛ ɑpi céru rɛ pɛ̃ kɛ Yeesu kɛnyɑ́rũ ɑkɛ́ nɛ mɑ́nɛ. Api pɛɛ lɛ̃ kɑpi yɛ́ nɛ u n tĩ́ wɛ́ɛ́si, ɑmɑ́ risoiwuí ɑri iwɑmɛ pi wɑi. Api u tíyɛ ɑpi tɔ́su. 
Api Yeesu kɛ́mɛɛ Pifɑrisi pinyinɛ nɛ pɛpɛɛ Erooti kɛpirɛ ń we tumpɔ rɛ pikɛ́ ripinɛ u wɑ ɑpí nɛ uɑnɔɔlempi cirɛ u tini. 
Apí weri ɑpi Yeesu pisɛ rɛ: Sɑ́ɑ, tɔ yɛ̃ tɛ pɔ́ɔ ɑsei ute ɑ́ɑ kɔ úkɑ kɛwuu túúni. Likumúŋɛ́ rɛ ɑ́i pisoi mɛwee kɑɑ́ nɛ músu pɔn nɛ símisi. Pɔ yɛ kɔ Uléécɑɑ ncee pisoi céesilɛ nɛ ɑsei. Ncée yɛ we rɛ tɔkɛ́ yɛ Sesɑɑ lɑmpoo hɛ́ɛ́lɛɛ? Li pisɛ rɛ tɔkɛ́ yɛ hɛ́ɛ́lɛ nɛ́ɛ ɑ́ri kɑpɛ yɛ hɛ́ɛ́lɛ? 
Yeesu yɛɛ pikɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi n nyu uu rinɔ́ɔ pi yɔ́su rɛ: Yo nnyɑ kɑni ripinɛ nɛ́ wɑi? Ani nɛ mɛwóópipi mɛnyinɛ nɛ́ kɑm kɛ́ mɛ yɛ̃. 
Apí nɛ mɛnyinɛ u hɑpɔ. Yeesu uu pɛɛ pi pisɛ rɛ: Úye riyu nɛ úye rinyiri yɛɛ kɛwóó nkɛ́ kɛcɑ́ɑ́ we? Api rinɔ́ɔ yɔ́su rɛ Sesɑɑ lo. 
Uu pɛɛ pi mɑɑ rɛ: Ani Sesɑɑ likɔ́ Sesɑɑ pɑ, ɑni Uléécɑɑ likɔ́ Uléécɑɑ pɑ. Urinɔɔ mɛyɔ́ɔ́ ɑmɛ mɛyɑ̃́ nnɔ́ɔ pi yipu. 
Pisɑtusee pɛɛ n tee rɛ pikpɔkpɔ ɑ́pi yɛ nkpɔ-i yisi, ɑpi Yeesu kɛ́mɛɛ sĩ́ ɑpi u pisɛ rɛ: 
Sɑ́ɑ, lɛ̃ kɛ Moisi uu isé ń wɔ́i yɛ nní: Úye umɑɑ un n kpu uu unɔ́si tíyɛ úu nɛ kɛwɑ̃́, uuwɑ̃ yɛɛ yɛ́ unɔ́si uyɛ̃ kukúmɑnnɔsi kpísi uu umɑ́ɑ mpuri wɑi. 
Pimɑ́rɛcɔ pisɛɛi pinyinɛ pɛɛ pɛɛ́ we. Ufoí uu unɔ́si kpísi uu kpi úu nɛ kɛwɑ̃́. 
Ulírũ uu ukúmɑnnɔsi kpísi, uu kpi úu nɛ kɛwɑ̃́. Lɛ̃ mɛcɔ kɑi kɔ utɑ́ɑ́nũ titiki. 
Pikɛsɛɛi kɛ̃, úkɑ úu kɛwɑ̃́ nɛ u mɑri. Pɛ̃ mɛmɑ́ɑ́ unɔ́si uyɛ̃ pɔ́ɔ kɔ kpi. 
Pikpɔkpɔ kɛyisɛyɑɑ kɛtúŋɛ́, pikɛcɔpɛ úye yɛɛ yɛ́ unɔ́si n te, likumúŋɛ́ rɛ pikɛsɛɛi kɑpi kunɔ́si u kpíkɛsi. 
Yeesu uu pi mɑɑ rɛ: Lɛ̃ nnyɑ kɑni nɔ́mɛnɛ́círɛ kírɑɑsɛntɛ: Áni Nléécɑɑsimɛ́ ɑsei kómɛi, ɑ́ni kɔ Uléécɑɑ nnɑŋɛ nyu. 
Mɛseisei kɛ̃ kɛ pisoi ɑpi pikpɔkpɔ kɛ́mɛɛ n yisinɛ, pitisi ɑ́pi pinɔ́si kpíkɛsinɛ, pinɔ́si pɔ̃́ ɑ́pi kɔ pilɑ sɔ́ɔnkɛɛnɛ, ɑmɑ́ pimɛfinɛ yɛ́ n we yɑrɛ piléécɑɑtumɛ kɛyómɛcɑɑ-pɔ. 
Yo nnyɑ kɑni tee rɛ pikpɔkpɔ ɑ́pi yɛ́ yisi? Áni kɛlõ kɛ-i kɑi Moisi ritɛlɛ́-i n wɔ́lɑɑlɛ̃ tɛ Uléécɑɑ yɛ kuhííhíí kpɛɛ ń torɛ kɛ́mɛɛ we un nɛ Moisi símisi kɛɛnlɛ̃ɛ? Li kei wɔ́lɑɑlɛ̃ tɛ, Uléécɑɑ yɛ Moisi mɑɑ rɛ: Nɛ́ɛ Apirɑhɑm Uleecɑɑ nɛ Isɑɑki Uleecɑɑ nɛ Yɑkupu Uleecɑɑ. 
Uléécɑɑ úu pikpɔkpɔ Uleecɑɑ, ɑmɑ́ u pinyɑ́nnyɑ̃́ Uleecɑɑ lɛ. Nɔ́nnɛ́púnnɛ yɛ piyɛlɛ. 
Uléécɑɑ isé ucélɑɑ unyinɛ yɛɛ pɛɛ Pisɑtusee nɛ Yeesu ikɛŋɛnɛ n kṍ, uu yɛ́nu rɛ Yeesu yɛ rinɔ́ɔ pi yɔsilɛ ńsɔnɛ, uú nɛ u kɔ, uu u pisɛ rɛ: Isé nnɛ́í kɛ́mɛɛ íye yɛɛ imɑ́ɑ? 
Yeesu uu rinɔ́ɔ yɔ́su rɛ: Isé mɑɑ yɛ nnyí: Isirɑyɛɛli, ɑ kutu ricɔ, Upíimɑ, Urɔ́léécɑɑ, Upíimɑ yɛ ucírɛníŋɛ lɛ. 
A Upíimɑ, Upɔ́léécɑɑ n lɑ nɛ kɛpɔ́fɑ nnɛ́í, nɛ ipɔ́soi nnɛ́í nɛ kɛpɔ́múŋɛ́ nnɛ́í nɛ mpɔ́nɑŋɛ nnɛ́í kɔ. 
Ilírũ yɛ nnyí: A upɔ́cɔ n lɑ yɑrɛ kɑɑ ripɔ́cúruu ń lɑ mɛcɔ. Isé íkɑ íi we yɛɛ nnyí n fe. 
Isé ucélɑɑ uyɛ̃ uu u mɑɑ rɛ: Li nyɑmlɛ, sɑ́ɑ. Asei yɛ nyɛ kɑɑ́ mɑɑ rɛ Uléécɑɑ yɛ ucírɛníŋɛ lɛ úu kɔ ucɔ mɑ́. 
Ḿpɑ́ úye ukɛ́ u n lɑ nɛ kɛfɑ nnɛ́í, nɛ mɛsɔhɔ nnɛ́í nɛ ńnɑŋɛ nnɛ́í kɔ. Ḿpɑ́ úye ukɛ́ kɔ uucɔ n lɑ yɑrɛ kuu uricuruu ń lɑ. Lɛ̃ nɛ isɛɛ kɑpi yɛ nɛ Uléécɑɑ n nyɔ́ɔnsɛ ɑpi tóroisɛ múkú-múkú nɛ inyɔ́ɔnsɛ icɔ nnɛ́í tɔsilɛ. 
Kɛ Yeesu uu n yɛ̃́ tɛ mɛsɔhɔ kuú nɛ rinɔ́ɔ yɔsí, uu u mɑɑ rɛ: Pɔ mɛpɔ́círɛ ńsɔnɛ cɑ́pinɛlɛ. Áɑ nɛ Uléécɑɑ kuyɔɔpiyu kɛtɑɑ we. Úkɑ úu pɛɛ ikɑri piwɑi rikpɑ́ ukɛ́ nnyinɛ u pisɛ. 
Yeesu yɛ Uléécɑɑ kɛyɔ-i céésilɛ uu pi pisɛ rɛ: Íye kɛ isé picélɑɑ ɑpi yɛ́ fe ɑpí mɑɑ rɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ yɛ Tɑfiti kɛpipi lɛ. 
Nfɑ́ɑsɔnɛ yɛ Tɑfiti ricuruu mɑɑsɛ rɛ: Upíimɑ yɛ Unɛ́píimɑ mɑɑ rɛ: A kunɛ́lukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tonɛ, hɑ́i nɛ kumúŋɛ́ kpɛ-i kɑm yɛ́ pipɔ́lɑ́ɑrɔ kɛpɔ́nyɑ́lɛ-i n cɔ. 
Tɑfiti ricuruu yɛɛ u sée rɛ "Upíimɑ". Íye kuu yɛ́ pɛɛ́ nɛ Tɑfiti kɛpipi? Pisoi kulúi yɛ pɛɛ kutu u cɔlɛnlɛ lin pi lɑ́ɑ́rú. 
Uicélɑɑ kɛ́mɛɛ kuu pi símisi rɛ: Ani nɛ isé picélɑɑ nɛ nɔ́mɛnɛ́círɛ tĩ. Pikɛ́ nɛ ɑtúkɑnkɑ cɑɑ-cɑɑ n kɔɔnii nɛ pisoi pikɛ́ pi n yɑ́hɑɑnkɛɛ ɑyɑ́ɑ kɛ́mɛɛ yɛɛ ripɔ́ɔ pi we. 
Ayómɛyɑ́hɑɑlee sitonɛ foí nɛ kɛlukɛ kɛ́mɛɛ sitonɛ foí yɛɛ pilikɑ́ipi-kɑ́ipi. 
Pi yɛ lɛ̃ kɛ pikúmɑnnɔsi ɑpi m mɑ́ nnɛ́í yɔ́ɔilɛ ɑpi yɛ pɛɛ kɛyómɛ yɑ́ɑ́si mɛcɑ́ɑ yɑrɛ pi pisoi sɔnɛ lɛ. Uléécɑɑ yɛ́ nɛ pi túhɑɑnɛ uu íwɛ pi wɑi ɑ́i kumúŋɛ́ mɑ́. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ Uléécɑɑ kɛyɔ-i, kei kɑpi yɛ ihɛɛ n sɑriipɔ kɛyúrí tonɛ. Uu lɛ̃ kɛ pisoi ɑpi siwóó n sɑriipɔ m pɑílɛ̃. Pimɑ́mɑ́ mɛyɑ̃ ɑpi mɛyɑ̃́ hɛkɛsi. 
Ukúmɑnnɔsi unyinɛ pɔ́ɔ kɔ weri uu sitɑnkɑ́ sitɛ́ sɑ́pɔ. 
Yeesu uu pɛɛ upipirɛtiki séi uu pi mɑɑ rɛ: Asei kɑm nɔ́ símisi, ukúmɑnnɔsi wɛkɔɔ́ nkó yɛ hɛlɛ ɑi ḿpɑ́ úye tɔ́su. 
Likumúŋɛ́ rɛ mpí nnɛ́í, lɛ̃ kɑ́pi nɛ líkɑ n wɑi kɑpi hɛ, ɑmɑ́ uyɛ̃ pɔ̃́, ulikɔ́ kɛcirɛ, lɛ̃ kuu m mɑ́ nnɛ́í ukɛ́ nɛ ilukɛ n lɔ kuu hɛ.
