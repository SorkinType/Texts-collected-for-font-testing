Mɑriki.
14.
Li riyɑ́lɛ siyɑ́ɑ sitɛ́ Isirɑyɛɛli pikɔ́ pikɛ́ nɛ nkpɔ mɛlɔ́ɔ́ nɛ ɑkpɔ́nɔ́ nyɛɛ ɑ́ɑ pitɛ m mɑ́ ɑnyɑ̃́ li. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpi ncée n wɛ́ɛ́si kɑpi yɛ́ nɛ Yeesu n tĩ́ nɛ mɛsɔhɔ pikɛ́ u kpu. 
Api símɑɑnɛ rɛ: Ái kɑpɛ ɑnyɑ̃́ kumúŋɛ́-i rɛ pisoi yɛ́ yisi ɑpi pitele wɑi nnyɑ. 
Kɛ Yeesu uu Petɑnii kuyu-i uyɔ̃́ Simɔɔ kɛyɔ kɛ́mɛɛ ń we un ilukɛ le, unɔ́si unyinɛ uu lompɔ. Unɔ́si uyɛ̃ un kɛnúḿpi sɔnɛ kɛɛ tulɑɑli pɑpɛ-pɑpɛ yɛɛ nnɛ́í píimɑ m mɑ́ múílɛ̃. Uu kɛnúḿpi kɛ̃ foru uu tulɑɑli Yeesu riyu kɛcɑ́ɑ́ kɔɔni. 
Mɛwɑi mɛ̃ ɑ́mɛ nɛ picɔ risɑ́, ɑpi pɛɛ pimɛcɔpɛcirɛ m mɑ́ikɛɛ rɛ: Linsɔnɛ yɛ ń-ye kuu nní tulɑɑli nkó n cɑɑi? 
Tɔ́ pɛɛ fe ɑri u yɑ́i nɛ nnɛ́í píimɑ ɑri siwóó sɛ̃ píwɛkɔɔ́ hɛ. Apí nɛ unɔ́si uyɛ̃ wɔ́ɔ́si. 
Amɑ́ Yeesu uu pi mɑɑ rɛ: Ani u riyɑ́. Yo nnyɑ kɑni u cɔ́ŋlɛ̃? Mɛwɑi sɔnɛ kuu nɛ́ wɑ. 
Likumúŋɛ́ rɛ ḿpɑ́ píyei kɛ nɔ́ nɛ píwɛkɔɔ́ ɑní we, nɔ́ fe ɑni lisɔnɛ pi wɑi pɛɛ́ uyɛ-i kɑni ń lɑ, ɑmɑ́ nɛ̃́, ɑ́i ḿpɑ́ píyei kɑm yɛ́ nɛ nɔ́ n we. 
Lɛ̃ kɛ unɔ́si nkó uu n fenɛ kuu wɑ. Inɛ́piŋɛ kuu mɛncirɛ tulɑɑli fṍ pikɛ́ kɛlenɛ nhórɛ i tɑ̃. 
Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ ḿpɑ́ yei kɑpi kɛtẽ nnɛ́í kɛcɑ́ɑ́ Nsímɛ́ Kɛcirɛ yóólɛ̃, pi yɛ́ɛ́ unɔ́si nkó mɛwɑi mmɛ́ símisi ɑpí nɛ u n léisɛlɛ̃. 
Yeesu pipirɛtiki kɛfi nɛ pitɛ́ kɛcɔpɛ usɛ kɑpi yɛ n sée rɛ Yutɑsi Isikɑriyɔɔti uú hɑ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ lɛɛpɔ rɛ ukɛ́ Yeesu tĩ ukɛ́ pi rimúísɛ. 
Nkíŋniŋɛ kɑpí nɛ kutu u ricɔ ɑpi rinɔ́ɔ yekei rɛ pi yɛ́ siwóó u hɛ. Yutɑsi uu icée sɔnɛ wɛ́ɛ́si ukɛ́ nɛ Yeesu piɑnipɛ-i n wɑ. 
Kɛnyɑ̃́yɑɑ foí kɛ̃ kɑpi yɛ ɑkpɔ́nɔ́ nyɛɛ ɑ́ɑ pitɛ m mɑ́ n li, kɛyɑ́ɑ kɛ̃ kɑpi yɛ kɔ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ isɑŋ n kṍ, Yeesu pipirɛtiki ɑpi u pisɛ rɛ: Yei kɑɑ́ lɑ tɔkɛ́ hɑ ɑnyɑ̃́ ilukɛ pɔ́ wɑ? 
Uu upipirɛtiki pitɛ́ tum uu pi mɑɑ rɛ: Ani kuyu-i hɑ, utisi unyinɛ yɛɛ kɛcɔ́rípi nɛ míni n topori yɛ́ nɛ nɔ́ risɑ́nɛ, ɑni u ritiki. 
Un yei n lõ ɑni úyɔte mɑɑ rɛ: Urɔ́sɑ́ɑ yɛ pisɛ rɛ yei yɛ kulee kpɛ-i kɛ u nɛ upipirɛtiki ɑpi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ n lenɛ? 
U yɛ́ kɛ́yɔ kɛcɑ́ɑ́ kulee mɑɑ kunyinɛ kɑpi ńsɔnɛ n nyɔ́ɔnsɛ nɔ́ nyísɛ. Ilũ kɑpi yɛ nɛ ɑnyɑ̃́ n li nnɛ́í yɛ kukɛmɛɛ́ welɛ. Kei kɑni yɛ́ ɑnyɑ̃́ ilukɛ wɑ. 
Yeesu pipirɛtiki ɑpi tɔ́su ɑpi kuyu tulu, ɑpi ḿpɑ́ yo lɛɛpɔ yɑrɛ kɛ Yeesu uú pɛɛ pi ḿ mɑ, ɑpi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ wɑi. 
Kɑi nnyɔ́ɔ́ n wɑ, Yeesu nɛ upipirɛtiki kɛfi nɛ pitɛ́ ɑpí weri. 
Kumúŋɛ́ kpɛ-i kɑpi ilukɛ n le, Yeesu uú mɑɑ rɛ: Asei kɑm nɔ́ símisi, nɔ́kɛnɛ́cɔpɛ unyinɛ yɛɛ nɛ ilukɛ nɛ́ n le yɛ́ nɛ́ yɑ́ɑ́. 
Piɑpɔɔ ɑɑ cɑɑi ɑpi pɛɛ usɛ usɛ u m písɛi rɛ: Yɑrɛ nɛ̃́ nɛ́ɛ? 
Uu rinɔ́ɔ pi yɔ́su rɛ: Nɔ́ kɛfi nɛ pitɛ́ pɛ̃ usɛ lo, tɔ́ nɛ uyɛ̃ tɔɔ ricɑ́ripi risɛ-i n le. 
Usoi Kɛpipi yɛ tɔ́sulɛ yɑrɛ lɛ̃ kɑi kɛkɛcɑ́ɑ́ n wɔ́lɑɑmɛlɛ̃. Amɑ́ usoi nkó yɛɛ yɛ́ Usoi Kɛpipi n yɑ́ɑ́, íwɛ kɛ liute uu topori. Li kutɔsi we ɑ́pi kɑpɛ pɛɛ liute m mɑri. 
Kɑpi n le, Yeesu uu kukpɔ́nɔ́ kpísi, uu kukɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu limɛmɑ́ɑ́ ku kpɔkɔ́ri uu pi ku pɑ uu rɛ: Ani yɔsí, inɛ́piŋɛ yɛ nnyí. 
Uu lɛ̃ mɛmɑ́ɑ́ kɛpóripi kpísi uu kɛkɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ. Uu pi kɛ pɑ ɑpi pinnɛ́í níru. 
Uu pi mɑɑ rɛ: Mɛnɛ́nyɛ yɛ mmɛ́, mɛ́nyɛ mɛmɛɛ pisoi mɛyɑ̃ nnyɑ kɔɔnu, Uléécɑɑ uú nɛ pi séi rɛ u nɛ pɛ̃ pikɛ́ kómɛinɛ. 
Asei kɑm nɔ́ símisi, ɑ́m píkɑi ɑléépipi pitɑ mpí pinírɛ́ kpɑ́nɛ hɑ́i nɛ kɛyɑ́ɑ kɛ̃ kɑm yɛ́ pifɑ́lɛ ń ntí Uléécɑɑ iyɔɔpi kɛ́mɛɛ. 
Kuu lɛ̃ pisímɛ́ pi ḿ mɑsí, ɑpi pɛɛ ipɑkɑrɛ siyomɛ n yom pin pɑkɑrɛntɛ ɑpi wɑi ɑpi Olifiyee rikúú sĩ́. 
Yeesu uu pi mɑɑ rɛ: Nɔ́ nɔ́nnɛ́nɛ́í kɛpirɛ nɛ́ ritɔ, likumúŋɛ́ rɛ li Nléécɑɑsimɛ́ ritɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Nɛ́ usɛɛsẽ́ kpu, isɑ́ŋ pɔ́ɔ kpíi. 
Amɑ́ nɛn píyei nkpɔ kɛ́mɛɛ piyisɛ m mɑsí, nɛ́ hɑ Kɑlilee kɛtẽ-pɔ nɔ́ m mɛ̃́. 
Piyɛɛ uu u mɑɑ rɛ: Ḿpɑ́ pinnɛ́í pin kɛpirɛ pɔ́ rintɔ, ɑ́i nɛ nɛ̃́. 
Yeesu uu u mɑɑ rɛ: Asei kɑm pɔ́ símisi, nɛni kɛsinɛ nkɛ́ cirɛ icɑ̃́ ikɛ́ kɛlenɛ mɛtɛ́ n kooi, pɔ́ kɛ́si nnɔ́ɔ mɛtɑɑni rɛ ɑ́ɑ nɛ́ nyu. 
Amɑ́ Piyɛɛ uú n kɛŋɑɑlɛ̃ tɛ: Ḿpɑ́ lin nkpɔ yɛ mu, tɔ́ nɛ mpɔ̃́ tɔkɛ́ kɛsẽ́ kpu, ɑ́m kuŋmɛriyɛ pɔ́ we. Pinnɛ́í ɑpi kɔ nsɛ mmɛ̃ m mɑ́ikɛɛ. 
Yeesu nɛ upipirɛtiki ɑpi lɛ̃ mɛmɑ́ɑ́ kɛlõ kɛnyinɛ kɑpi yɛ n sée rɛ Kɛtisemɑnee sĩ. Uu upipirɛtiki mɑɑ rɛ: Ani nté n tũ, nɛ́ɛkɛ́ hɑ kɛyómɛ n yɑ́hɑɑpɔ. 
Uú nɛ Piyɛɛ nɛ Yɑkupu nɛ Yohɑni pɛ́nɛ. Kei kɛ iwɑmɛ nɛ mpɔ́ɔcɑɑi yíkíyiki nn u tĩ. 
Uu pi mɑɑ rɛ: Tinɛ́pɔ́ɔ yɛ cɑɑilɛnlɛ hɑ́i lin nɛ nkpɔ mɛnyínɛ we. Ani nté tonɛ ɑ́ni kɑpɛ lɔpilɛ. 
Uu lɛ̃ mɛmɑ́ɑ́ kɛyu-mɛ̃ nyɔsɔ́pɔ nkɑ́ripi, uu wúlɑ uu Uléécɑɑ yɑ́ɑ́si rɛ lin íye n we, ɑ tíyɛsɛ ḿpɔlɛ ituŋɛ iyɛ̃ íi kɑpɛ kɑ. 
Uu kɛyómɛ yɑ́ɑ́si rɛ: Unɛ́sɑ́ɑ, pɔ́ fe ɑɑ ḿpɑ́ yo wɑi, ɑ nɛ ntóósi kɛporipi nkɛ́ kɛtɑɑ nɛ́ wɑ. Ḿpɑ́ nɛ lɛ̃, kɑpɛ mɛnɛ́pɔ́ɔlɑ wɑ, ɑmɑ́ ɑ mɛpɔ́kɔ́ wɑ. 
Uu lɛ̃ mɛmɑ́ɑ́ upipirɛtiki-mɛ̃ pɛlɛ, uu pi lɛɛpɔ pin lɔni. Uu Piyɛɛ mɑɑ rɛ: Simɔɔ, pɔ lɔnilɛɛ? Áɑ nkpɑ́ni fe pɔkɛ́ n nyɑ́ni kɛnɑmpi kɛsɛɛ? 
Ani n tũ nɔn nyɑ́ni nɔn kɛyómɛ yɑ́ɑ́si, ɑ́ni kɑpɛ nɛ pipeikɛɛ kɛ́mɛɛ ń loó nnyɑ. Li kɛsoipipi ripɔ́ɔ we rɛ kɛkɛ́ yɛ lisɔnɛ wɑ, ɑmɑ́ kɛ pɛɛ rinɑ́rɛi mɑ́lɛ. 
Uú nɛ kɛtɑɑ piwɑi pi kpɑ́, uú hɑ kɛyómɛ yɑ́ɑ́si, uu umɛwelɛ foí mɛ̃ mɛcɔ welu. 
Uu pɛɛri uu kɔ pi lɛɛri pin lɔni, likumúŋɛ́ rɛ piinipɛɛ yɛ mulɛsilɛ. Ápi ceri mmɛ̃ kɑpi yɛ́ rinɔ́ɔ u n yɔsí. 
Uu mɛtɑ́ɑ́nũ pɛɛri uu pi mɑɑ rɛ: Nɛ́ntɛ nɔ nkpéni lɔnilɛ nɔn wéntɛlɛ̃? Li lɛ̃ tulɛ, ituŋɛ yɛ tulɛ, lɛɛ nní, pi Usoi Kɛpipi pikópɛkɔɔ́ ɑnipɛ-i wɑlɛ. 
Ani yisi tɔkɛ́ tɔsí, uyɛɛ nɛ́ n yɑ́inɛ yɛ nyɑhɑimɛlɛ. 
Yeesu úu kɑhɑnɛ nsímɛ́ n tɛnɛ kɛ upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ usɛ kɑpi yɛ n sée rɛ Yutɑsi nɛ risoiwuí ɑpi mɛsɛ nɛ mɛsɛ rikpɑ́fúmɛ, pin ɑtɛ́hɛ nɛ ɑpúnípi mulɛilɛ̃. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ nɛ Pisuifi piwɛ́ɛ́sɛ yɛɛ pi tumpɔ. 
Yutɑsi yɛɛ pɛɛ piɑnipɛ-i Yeesu n wɑinɛ, yɛ pɛɛ risoiwuí mɛsɔhɔ mɛ̃ kuú nɛ u ń tininɛ céesilɛ. U pɛɛ pi céesi rɛ: Nɛn hɑ úye rimpirɑ ɑm u yɑ́ɑ́si, uyɛɛ lo, ɑni u tĩ, ɑni u múlú ńsɔnɛ ɑní nɛ tɔ́su. 
Kuu n tuipɔ uú nɛ mɛsɛ nɛ mɛsɛ Yeesu kɔ, uu rɛ: Sɑ́ɑ! Uu pɛɛ u pirɑ uu u yɑ́ɑ́si. 
Pisoi pɛ̃ ɑpi Yeesu kɛcɑ́ɑ́ kunípɛ wɑi ɑpi u tini. 
Kei kɛ Yeesu upirɛtiki usɛ uu risɑɑpɔ uu uritɛ́hɛ kpeéu, uu Uléécɑɑ usinɑ ukɛikɔ́ kutu seḿ. 
Yeesu uu rinɔ́ɔ yɔ́su uu pi mɑɑ rɛ: Yo nnyɑ kɑni pitínɛ nɛ́ kɑ nɛ ɑtɛ́hɛ nɛ ɑpúnípi yɑrɛ usoi kpɑ́ɑ́ree? 
Ḿpɑ́ kɛyɑ́ɑ kɛ́ye kɑm yɛ nɔ́kɛnɛ́cɔpɛ n we, nɛn Uléécɑɑ kɛyɔ-i céési, ɑ́ni nɛ́ tĩ́. Amɑ́ lɛ̃ kɑi Nléécɑɑsimɛ́ ritɛlɛ́-i n wɔ́lɑɑmɛlɛ̃ likɛ́ nɛ n wɑ nnyɑ kɑi lɛ̃ wɑi. 
Yeesu pipirɛtiki nnɛ́í ɑpi u ŋmɛriyɛ ɑpi wuru. 
Uŋmɑ́nɛ unyinɛ yɛ̀ɛ̀ pɛɛ Yeesu n tikilɛnlɛ un kusɑ́ŋɑ́ɑ́ ŋmɑnɛ yɑriilɛ̃. Api rɛ pikɛ́ u tĩ, 
uu kusɑ́ŋɑ́ɑ́ tɔpíri uu mɛŋmɑnɛ wuru. 
Apí nɛ Yeesu Uléécɑɑ usinɑ kɛ́mɛɛ sĩ́. Kei kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpi pinnɛ́í cɑ́pinɛlɛ̃. 
Piyɛɛ uu kɛtɑɑ-pɔ Yeesu n tikilɛ̃ hɑ́i nɛ Uléécɑɑ usinɑ kɛyɔcɔpɛ-pɔ. U nɛ pimɛ́rɛ́ ɑpi kei kɛsẽ́ n tũ un nnɑ wéni. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ pitúhɑɑnɛ píimɑ nnɛ́í ɑpi pisoi wɛ́ɛ́si pɛɛ yɛ́ Yeesu nɛ nsímɛ́ rinkpɑ́rɑ́. Pitúhɑɑnɛ pɛ̃ ɑ́pi nkópɛ ńkɑ yɛ̃ kɑpi yɛ́ nɛ u n kpu. 
Likumúŋɛ́ rɛ mɛyɑ̃́ mɛɛ nɛ nsímɛ́ u kpɑrɑi, ɑmɑ́ mmɛ̃ kɑpi n símisi ńn ncée nsɛ ritiki. 
Picɔ ɑpi yɛ yisi ɑpí nɛ u wɑ́ɑ́nɑ́ rɛ: 
Tɔ kõ un tee rɛ: Nɛ̃́ nɛ́ Uléécɑɑ kɛyɔ nkɛ́ kɛ usoi kunipɛ ɑku n wɑ fori, ɑm siyɑ́ɑ sitɑɑni kɛcɔpɛ kɛféé mɔm kɛ usoi kunipɛ ɑ́ku yɛ́ n wɑ. 
Ḿpɑ́ nɛ kei ricuruu, pinsímɛ́ ńn ncée nsɛ ritiki. 
Uléécɑɑ usinɑ uu icɑ́pinɛ iyɛ̃ kɛcɔpɛ yisi uu Yeesu pisɛ rɛ: Áɑ rinɔ́ɔ ríkɑ yɔ́sunɛɛ? Mpɔ́kópɛ ń-ye kɛ pisoi mpí ɑpi nní símisi? 
Yeesu uu sɛ́ɛ́, úu rinɔ́ɔ ríkɑ yɔsí. Uléécɑɑ usinɑ uu pipisɛ u kpɑ́ rɛ: Pɔ́ɔ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ? Uléécɑɑ yɛɛ ipɑkɑrɛ n te Kɛpipii? 
Yeesu uu rinɔ́ɔ yɔ́su rɛ: Nɛ́ɛ lo. Nɔ́ Usoi Kɛpipi yɛ̃ kɛn Uléécɑɑ, ńnɑŋɛ nnɛ́í ute kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tṹ. Nɔ́ kɔ kɛyómɛcɑɑ-pɔ ɑhopɛ kɛ́mɛɛ u yɛ̃ un nɛ wemɛ. 
Kei kɛ Uléécɑɑ usinɑ uu pɛɛ uricuruu ilũ kuu n tɑnɑɑlɛ̃ cɛ́ɛ́rii ɑi nyísɛ rɛ li u tóósilɛ hɑ́i uu rɛ: Píseérɑkɔɔ́ píye picɔ kɑri nkpéni wɛ́ɛ́si? 
Nɔ konlɛ kuu Uléécɑɑ kɛcɑ́ɑ́ ɑnɔ́ɔ kópɛ n símisi. Íye kɑni músu? Pinnɛ́í ɑpí mɑɑ rɛ li nɛ sɑ́ rɛ pikɛ́ u kpu. 
Picɔ ɑpi ukɛcɑ́ɑ́ mɛtɑ pitúkɛntɛ loni, ɑpi ukɛyu pɑɑsi ɑpi ɑkpini pinɛ́i u kɑ́pɑ́ɑ́ pin u tee rɛ: A uyɛɛ lɛ̃ pɔ́ n wɑi ŋmɔinu! Pimɛ́rɛ́ ɑpi u tini ɑpí nɛ sipɛsɛ u tɛ́pɛi. 
Kumúŋɛ́ kpɛ-i kɛ Piyɛɛ uu mɛtene ripoo-i ń we, unɔ́si unyinɛ yɛɛ Uléécɑɑ usinɑ n kɛisi uú weri. 
Uu Piyɛɛ yɛnu un nnɑ wéni, uu ukɛyu-i nyɑ́nɛi uu u mɑɑ rɛ: Pɔ̃́ nɛ kɔ pɛɛ Nɑsɑrɛti ukɔ́ Yeesu uyɛ̃ tikilɛnlɛ. 
Uu kɛ́su rɛ: Ám nyu, ɑ́m mmɛ̃ kɑɑ ń lɑ pɔkɛ́ símisi kómɛi. Uu pɛɛ kei yisi uu kupookpɑŋɑ́-i sĩ́. 
Unɔ́si uyɛ̃ uu piyɛ́nɛ u kpɑ́ uu pɛpɛɛ kei ń we mɑɑ rɛ: U pisoi pɛ̃ usɛ lɛ. 
Piyɛɛ uu kɔ pikɛ́ŋɛ kpɑ́. Kɑi nkɑ́ripi rinkpɑ́sɛ, pɛpɛɛ kei ń we ɑpi pimɑ́ɑ u kpɑ́ rɛ: Mɛsei yɛ mɛ, pɔ pisoi pɛ̃ usɛ lɛ. Likumúŋɛ́ rɛ pɔ Kɑlilee ukɔ́ lɛ. 
Uu kɛ́su ŋmɑɑ uú nɛ kɛtẽ kpɛɛni uu wééri rɛ: Nɛ Uléécɑɑ, ɑ́m usoi nkó nyu. 
Icɑ̃́ ii mɛsɛ nɛ mɛsɛ mɛlírũ kooi, Piyɛɛ uu léisɛ rɛ Yeesu yɛ u mɑɑ rɛ: Icɑ̃́ ikɛ́ kɛlenɛ mɛtɛ́ n kooi, pɔ́ kɛ́si mɛtɑɑni rɛ ɑ́ɑ nɛ́ nyu. Kei kuu pɛɛ mɛsɛ nɛ mɛsɛ itu ripɛ.
