Mɑriki.
10.
Yeesu uu ncée pɔlɔ uu Yutee kɛtẽ loni uu kɔ Yuritɛɛ rikoitimɛ ricɔ kuwɛ́lɛ́-mɛ̃ sĩ. Pisoi kulúi ɑpi kɔ pikɑ́lɑɑ u kpɑ́ uu kɔ pɛ̃ picélɑɑ loni yɑrɛ kuu yɛ n céesi. 
Pifɑrisi pinyinɛ ɑpí nɛ u kɔ pikɛ́ nɛ ripinɛ u n wɑ nnyɑ, ɑpi u pisɛ rɛ in tɛ ncée yɛ we rɛ utisi ukɛ́ uunɔsi yɛ. 
Uu rinɔ́ɔ pi pɛsɛ rɛ: Íye kɛ Moisi uu nɔ́ pisɛ rɛ ɑni wɑ? 
Api rinɔ́ɔ yɔ́su rɛ: Moisi yɛ utisi ncée hɛlɛ rɛ u yɛ́ fe uu pitúunnɛ ritɛlɛ́ unɔ́si wɔlu uu pɛɛ u yulu. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Nɔ́ɑnɛ́kíŋ ntɑkɛ nnyɑ kɛ Moisi uu isé nnyí nɔ́ wɔ́i. 
Amɑ́ ḿpɑ́ yo mɛwɑi kɛkorɑɑnɛ, Uléécɑɑ yɛ utisi nɛ unɔ́si wɑlɛ. 
Lɛ̃ nnyɑ kɛ utisi uu usɑ́ɑ nɛ úni yisinɛ uu unɔ́si kpísi ɑpi pɛ́nɛ, 
u nɛ uyɛ̃ ɑpi pɑnsɛ ipiŋɛ isɛ. Kɑi nní lɛ̃ n wɑ, ɑ́pi nkpéni pitɛ́, ɑmɑ́ piipiŋɛ yɛ isɛ lɛ. 
Lɛ̃ nnyɑ usoi úu kɑpɛ lɛ̃ kɛ Uléécɑɑ uu rimpɛ́nɛ kóólɛnɛ. 
Kɑpi kɛ́yɔ ń hɑ, Yeesu pipirɛtiki ɑpi kɔ nsímɛ́ mmɛ̃ kɛcɑ́ɑ́ pipisɛ u kpɑ́. 
Uu pɛɛ pi mɑɑ rɛ: Úye un uunɔsi n yɛ̀ un uféé n kpísi, u nɛ unɔ́si foí púnnɛlɛ, iwɑ́sɑ́nkɑí kuu lɛ̃ wɑ. 
Unɔ́si un uulɑ n yɛ̀ un uféé n sɔ́ɔ̃, iwɑ́sɑ́nkɑí kuu lɛ̃ wɑ. 
Pisoi ɑpí nɛ Yeesu siwɑ̃́ lɛɛpɔ rɛ ukɛ́ si ricɑ. Amɑ́ upipirɛtiki ɑpí nɛ pi cɛ́si. 
Kɛ Yeesu uu lɛ̃ n yɛ̃́, ɑ́i nɛ u risɑ́, uu pi mɑɑ rɛ: Ani riyɑ́ siwɑ̃́ sikɛ́ kɛnɛ́mɛɛ́ kɑm, ɑ́ni kɑpɛ si tɑ́pisi, likumúŋɛ́ rɛ mpí pɛɛ simɛcɔ ń we pɛɛ yɛ́ Uléécɑɑ iyɔɔpi kɛ́mɛɛ lõ. 
Asei kɑm nɔ́ símisi, úye unsɑ́ Uléécɑɑ iyɔɔpi n ŋmurɛi yɑrɛ kɛ kɛwɑ̃́ ɑkɛ yɛ n ŋmurɛi, úu uiyɔɔpi kɛ́mɛɛ loninɛ. 
Uu pɛɛ lɛ̃ mɛmɑ́ɑ́ si sɑkɑɑ uu ɑnípɛ si lɑɑ uú nɛ rinɔ́ɔ sɔnɛ si tikisɛ. 
Kɛ Yeesu uu ncée m pɔ́lɔlɛ̃, utisi unyinɛ uu u tulu nɛ ntóó, uú hɑ ukɛyu-i wúlɑ uu pɛɛ u pisɛ rɛ: Sɑ́ɑ sɔnɛ, íye kɑm wɑinɛ kɛ́ nɛ nfɑ́ɑ tɛnɛcirɛ́ yɛ̃́? 
Yeesu uu u mɑɑ rɛ: Yo nnyɑ kɑɑ nɛ́ sée rɛ Usɔnɛ? Úkɑ úu usɔnɛ, insɑ́ Uléécɑɑ mɛcirɛ. 
Ḿpɑ́ likɛ́ íye wɑ, pɔ Uléécɑɑ isé nyulɛ rɛ: Kɑpɛ usoi kpu, kɑpɛ iwɑ́sɑ́ wɑ, kɑpɛ yɑ́ɑi, kɑpɛ nnɔ́ɔmɛ itɑnsei nyɛrɛ, kɑpɛ unyinɛ ncɑɑi ńkɑ wɑ, ɑ upɔ́sɑ́ɑ nɛ upɔ́ni ríyu wɑisɛ. 
Uu rinɔ́ɔ u yɔ́su rɛ: Sɑ́ɑ, hɑ́i inɛ́wɑ̃́-mɛ kɑm lɛ̃ múílɛ̃. 
Yeesu uu u nyɑ́nɛi nɛ ńlɑ, uu pɛɛ u mɑɑ rɛ: Likɛi lisɛ lɛɛ pɔ́ pɑ́rílɛ̃, ɑ n ŋme ɑ lɛ̃ nnɛ́í kɑɑ m mɑ́ yɑ́ɑ́, ɑɑ siwóó sɛ̃ píwɛkɔɔ́ hɔ́ɔ́nɛ, pɔ́ pɛɛ mɛmɑ́ yɛ̃́ kɛyómɛcɑɑ-pɔ. Pɔn lɛ̃ piwɑi m mɑsí, ɑɑ pɛɛ́ weri ɑɑ nɛ́ tíki. 
Kɛ utisi uyɛ̃ uu ɑnɔ́ɔ nyɛ̃ n kṍ, kɛ ukunipɛ ɑku nní ḿ mulú nnyɑ, ukɛyu ɑkɛ cɔ́hɔnɛ uú nɛ mpɔ́ɔcɑɑi tɔ́mpɔ. 
Yeesu uu upipirɛtiki pɛɛ u n kɑ́lɑɑlɛ̃ nyɑ́nɛi, uu pi mɑɑ rɛ: Uléécɑɑ iyɔɔpi kɛ́mɛɛ pilonɛ yɛ nɛ pimɑ́mɑ́ pɑ́ɑ́pú kpɑ! 
Anɔ́ɔ nnyɛ́ ɑɑ Yeesu pipirɛtiki pírí wɑi. Amɑ́ Yeesu uu pimɑ́ɑ pi kpɑ́ rɛ: Sinɛ́pipi, ɑ tíyɛ Uléécɑɑ iyɔɔpi kɛ́mɛɛ pilonɛ yɛ pɔ́ɔ́lú kpɑ! 
Kuyooyoo kun n lɑ kukɛ́ kɛhímpi kupolɛ-i lompɔ, li yɛ́ kutɔsi m purú nɛ kɛ umɑ́mɑ́ uu yɛ́ ń lɑ ukɛ́ Uléécɑɑ iyɔɔpi kɛ́mɛɛ n lompɔ. 
Ai kɔ mɛyɑ́nsei pírí Yeesu pipirɛtiki kpɑ́, ɑpi pɛɛ́ m píseinɛ rɛ: In lɛɛ ni, wóo yɛ pɛɛ píyulɑlɛ yɛ̃? 
Yeesu uu pi nyɑ́nɛi uu rɛ: Usoi kɛ́mɛɛ, lɛ̃ ɑ́i kuwɑi we, ɑmɑ́ ɑ́i Uléécɑɑ kɛ́mɛɛ. Likumúŋɛ́ rɛ ḿpɑ́ yo yɛ kuwɑi welɛ Uléécɑɑ kɛ́mɛɛ. 
Piyɛɛ uu pɛɛ u mɑɑ rɛ: A ripɑí kɑri lirɔ́kɔ́ nnɛ́í n tíyɛ tɔn pɔ́ tikilɛ̃. 
Yeesu uu rinɔ́ɔ yɔ́su rɛ: Asei kɑm nɔ́ símisi, úkɑ úu we yɛɛ yɛ́ nɛ̃́ nɛ Nsímɛ́ kɛcirɛ nnyɑ, kɛ́yɔ nɛ pimɑ́ɑ nɛ piwɑ̃́, nɛ piyɛ́i nɛ píni nɛ pisɑ́ɑ nɛ siwɑ̃́ nɛ sicɑrɛ n tíyɛ 
úu yɛ́ pɛɛ nɛni mɛwee mmɛ́-i síyɔ nɛ pimɑ́ɑ nɛ piwɑ̃́ nɛ piyɛ́i nɛ píni nɛ siwɑ̃́ nɛ sicɑrɛ mɛpehẽ pílɛ n yɛ̃́. Úu yɛ́ kɔ Yeesu pitiki iwɛ pɑɑ. U yɛ́ kɔ ɑŋmɛ̃ nyɛɛ n wémɛ kɛ́mɛɛ nfɑ́ɑ tɛnɛcirɛ́ yɛ̃́. 
Pifoí mɛyɑ̃ yɛ́ pitɔ́rɔɔ pɑnsɛ, pitɔ́rɔɔ mɛyɑ̃ ɑpi pifoí. 
Ncée kɛ́mɛɛ kɑpí pɛɛ́ we pin Yerusɑlɛm-mɛ̃ sĩ. Yeesu un pikɛkpeẽ sɔ́nɛ, upipirɛtiki pin kɛwuu pirɑlɛ̃, iwɑmɛ in pɛpɛɛ pi n tíkilɛ̃ we. Yeesu uu kɔ upipirɛtiki kɛfi nɛ pitɛ́ kpísi uu lɛlɛɛ u n lɛɛrinɛ pi símisi 
uu rɛ: Nɔ nyɑ́nii, Yerusɑlɛm kɑri nní símpɔ pikɛ́ hɑ kei Usoi Kɛpipi tĩ pikɛ́ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑnipɛ-i wɑ, pɛ̃ pikɛ́ nɛ u túhɑɑnɛ ɑpi yekei rɛ pikɛ́ u kpu. Pi yɛ́ pɛpɛɛ ɑ́pi Uléécɑɑ m pɑkɑrɛlɛ̃ ɑnipɛ-i u wɑ. 
Pi yɛ́ mɛ́woo u nyɑ́nɛi ɑpi ukɛcɑ́ɑ́ mɛtɑ tukɛ, ɑpi isɛ́í u fopii ɑpi pɛɛ u kopu. Amɑ́ siyɑ́ɑ sitɑɑni mɛmɑ́ɑ́, u yɛ́ nkpɔ kɛ́mɛɛ yisi. 
Sepetee sipipi kɛtɛ́, Yɑkupu nɛ Yohɑni ɑpí nɛ Yeesu kɔ ɑpi u mɑɑ rɛ: Sɑ́ɑ, tɔ lɑ rɛ ɑ lɛ̃ kɑri pɔ́ n welunɛ rɔ́ wɑ. 
Uu pi mɑɑ rɛ: Yo kɑni mpíí lɑ rɛ kɛ́ nɔ́ wɑ? 
Api u mɑɑ rɛ: Pɔn píyei ipɔ́yɔ́ɔpi kɛ́mɛɛ n tonɛ, tɔ lɑ rɛ ɑ tíyɛsɛ usɛ ukɛ́ kupɔ́lukɛ-lukɛ-mɛ̃ n tũ, ucɔ kupɔ́míi-mɛ̃. 
Yeesu uu pi mɑɑ rɛ: Áni nyu lɛ̃ kɑni nní n welu. Nɔ́ fe ɑni ntóósi kɛporipi kɑm n nírunɛ níruu? Nɛ́ɛ nɔ́ fe ɑni íwɛ tɑkɑi yɑrɛ kɑm i n tɑkɑinɛ? 
Api rinɔ́ɔ yɔ́su rɛ: Tɔ́ li fe. Yeesu uu rinɔ́ɔ pi pɛsɛ rɛ: Mɛsei yɛ mɛ rɛ nɔ́ ntóósi kɛporipi kɑm n nírunɛ n ntí, tɛ íwɛ kɑm n tɑkɑinɛ kɑni tɑkɑinɛ. 
Amɑ́ in kunɛ́lukɛ-lukɛ nɛ́ɛ kunɛ́míi-mɛ̃ mɛtonɛ nkɔ́ mmú ɑ́i nɛ́ɛ yɛ́ tíyɛsɛ likɛ́ nɛ lɛ̃. Pɛ̃ nnyɑ kɛ sitónɛ sɛ̃ ɑsi n nyɔ́ɔnsɛlɛ̃ kuu si pɑnɛ. 
Kɛ Yeesu pipirɛtiki kɛfi tɔrɔɔ ɑpi lɛ̃ n kṍ, piɑpɔɔ ɑɑ́ nɛ Yɑkupu nɛ Yohɑni cɑɑi. 
Yeesu uu pɛɛ pi séi uu pi mɑɑ rɛ: Nɔ nyulɛ rɛ pɛ̃ kɑpi n kpíilɛ̃ tɛ ɑyu piwɛ́ɛ́sɛ yɛ pinnɑŋɛ nɛ ɑ wéékusɛlɛnlɛ, pisoi kɛcirɛ pin pinnɑŋɛ ɑ nyísɛlɛ̃. 
Amɑ́ ɑ́i li mɛcɔ kɑi nɔ̃́ kɛ́mɛɛ we. Uɑye un n lɑ ukɛ́ nɔ́kɛnɛ́cɔpɛ uwɛ́ɛ́sɛ, ukɛ́ pɑnsɛ nɔ́unɛ́kɛikɔ́. 
Úye un kɔ nɔ́kɛnɛ́cɔpɛ ikpéẽ n lɑ, ukɛ́ nɔ́nnɛ́nɛ́í ulɑsi. 
Limɛcɔ kɛ Usoi Kɛpipi ɑ́kɛ kɑ rɛ pikɛ́ kɛ kɛisi, ɑmɑ́ kɛ kɑlɛ rɛ kɛkɛ́ picɔ kɛisi, ɑkɛ kɔ kɛnfɑɑ nɛ riwómɛ hɛ́ɛ́lɛ mɛyɑ̃́ mɛkɛ́ nɛ píyulɑlɛ ń yɛ̃́ nnyɑ. 
Api Yeriko tuipɔ. Kumúŋɛ́ kpɛ-i kɛ Yeesu nɛ upipirɛtiki nɛ risoiwuí ɑpi kuyu ń lelu, unyíyɛ unyinɛ yɛɛ n wélei kɑpi yɛ n sée rɛ Pɑtimee, Timee uŋmɑ́nɛ un ticéetimɛ tũ. 
Uu kóm tɛ Nɑsɑrɛti ukɔ́ Yeesu yɛɛ tɔ́su, uu sicɑ́ɑ́ilɑ kɑ́pɑ́ɑ́ rɛ: Tɑfiti kɛpipi, Yeesu, ɑ inɛ́wɛ ripɑí! 
Mɛyɑ̃́ ɑmɛ nɛ u cɛ́si rɛ ukɛ́ risɛ́ɛ́, ɑmɑ́ uu mɛyɑ̃́ cɑ́sɛ rɛ: Tɑfiti kɛpipi, ɑ inɛ́wɛ ripɑí! 
Yeesu uu nyɛrɛ uu rɛ: Ani u séemɛ. Api unyíyɛ séipɔ, pin u mɑ́ikɛɛ rɛ: A n kɑhɑri, ɑ yisi, u pɔ́ séilɛ. 
Unyíyɛ uu ukɛtukɑnkɑ sɑ́ uu wɑ́kɑsi uu Yeesu-mɛ̃ sĩ, 
Yeesu uu u pisɛ rɛ: Yo kɑɑ́ lɑ kɛ́ pɔ́ wɑ? Unyíyɛ uu rɛ: Unɛ́sɑ́ɑ, nɛ lɑ rɛ inɛ́nípɛɛ ikɛ́ yɛ yɛ̃. 
Yeesu uu u mɑɑ rɛ: A n ŋme, mpɔ́fɑtɛnɛ yɛ pɔ́ yóriyɛ. Mɛsɛ nɛ mɛsɛ inípɛɛ ii wúkulɛ uu Yeesu tiki.
