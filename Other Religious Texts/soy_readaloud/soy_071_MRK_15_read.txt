Mɑriki.
15.
Piwɛ́ɛ́sɛ nnɛ́í ɑpi mmɑrɛ mɑrɛ cɑ́pinɛ: pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ nɛ isé picélɑɑ nɛ pitúhɑɑnɛ píimɑ nnɛ́í. Kei kɑpi pɛɛ Yeesu pɑɑsi ɑpí nɛ u tɔ́su ɑpi Pilɑti ɑnipɛ-i u wɑi. 
Pilɑti uu u pisɛ rɛ: Pɔ́ɔ Pisuifi uyɔɔpii? Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Mmɛ̃ kɑɑ nní mɑ. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpí nɛ nsímɛ́ mɛyɑ̃ u kpɑrɑi. 
Pilɑti uu kɔ pipisɛ u kpɑ́ rɛ: Áɑ rinɔ́ɔ yɔ́sunɛɛ? A ripɑí mmɛ̃ nɛ mmɛ̃ kɑpi kɛpɔ́cɑ́ɑ́ n símisi. 
Yeesu úu rinɔ́ɔ ríkɑ piyɔ́ɔ́ rikpɑ́. Ai Pilɑti pírí wɑi. 
Ḿpɑ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ɑ́ye, Pilɑti yɛɛ pɛɛ ukpɑniikɔ́ uyɛ̃ kɛ pisoi ɑpi n wɛ́ɛ ɑkpɑnii tɔlɛ. 
Utisi unyinɛ kɑpi yɛ n sée rɛ Pɑrɑpɑsi yɛ kukpɑniilee-i welɛ nɛ pinyinɛ pɛɛ ɑ́pi pɛɛ uyɔ́ɔpi kɛcɑ́ɑ́ n kpɑ́lɛ̃. Pitele pinyinɛ-i kɑpi pɛɛ usoi kpu. 
Tisoiwuí ɑri Pilɑti-mɛ̃ tɔ́mpɔ, ɑpi lɛ̃ kuu yɛ ḿpɑ́ ɑnyɑ̃́ ɑ́ye pi n wɑ pisɛ. 
Pilɑti uu pi pisɛ rɛ: Nɔ́ lɑ kɛ́ Pisuifi uyɔɔpi ɑkpɑnii nɔ́ tɔɔ? 
Likumúŋɛ́ rɛ Pilɑti yɛ ceri rɛ nfɑpɔlɛ nnyɑ kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpi uɑnipɛ-i Yeesu wɑ. 
Amɑ́ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpi pisoi Pilɑti-i teriipɔ pikɛ́ u ricɔ́ŋ ukɛ́ nɛ Pɑrɑpɑsi ɑkpɑnii tɔ. 
Pilɑti uu pipisɛ pi kpɑ́ rɛ: Íye kɑni mpíí lɑ ŋmɑɑ kɛ́ uyɛ̃ kɑni nní n sélei rɛ Pisuifi uyɔɔpi wɑ? 
Api pipupɛi kpɑ́ rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii. 
Pilɑti uu pi mɑɑ rɛ: Ncɑɑi ń-ye kuu mpíí wɑ? Api mɛyɑ̃́ pipupɛi kpɑ́ rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii. 
Pilɑti uu risoiwuí mɛpɔɔlɑ tiki uu Pɑrɑpɑsi ɑkpɑnii tɔlu. Kɑpi Yeesu nɛ isɛ́í pikɑ́ii ḿ mɑsí, ɑpi pisoi ɑnipɛ-i u wɑi rɛ pikɛ́ hɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii. 
Pisɔ́ɔ́cɑ ɑpí nɛ kɛ́yɔcɔpɛ, kei kɑpi yɛ n túhɑɑnɛ u sĩ ɑpi pisɔ́ɔ́cɑ tɔrɔɔ nnɛ́í séi. 
Api kuyɔ́ɔpikpɛlɛnku wɛɛ́ u tɑni ɑpi sinípɛleépi nɛ riyɔ́ɔpikɔŋ pɑrii ɑpi u kipisɛ. 
Api pɛɛ lɛ̃ mɛmɑ́ɑ́ u yɑ́hɑɑnkɛɛ rɛ: Tɔ pɔ́ yɑ́ɑsi, Pisuifi uyɔɔpi. 
Api ríyu kɛcɑ́ɑ́ nɛ kɛnɑɑpi u pepu ɑpi mɛtɑ ukɛcɑ́ɑ́ tukɛ, ɑpi ukɛyu-i ńsɔnɛ wulɑ ɑpi u yɑ́ɑ́si. 
Kɑpi kɛcɑ́ɑ́ pinyɑ́nɛi u ḿ mɑsí, ɑpi kuyɔ́ɔpikpɛlɛnku wɛɛ́ kpɛ̃ u lɑpilɛ ɑpi uilũ pitɑnɛ u pɛsɛ ɑpí nɛ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ pikɑrii u tɔ́mpɔ. 
Api ucéetɔŋɛ́ unyinɛ yɛɛ sicɑrɛ-mɛ̃ n lééri ńnɑŋɛ Yeesu kunɑpɛ́ɛ́kɔɔ́ cɔnsɛ. Pi yɛ utisi uyɛ̃ sée rɛ Simɔɔ, Sirɛɛni kuyu ukɔ́ lo. Uyɛɛ Alɛkisɑntiri nɛ Rufuusi usɑɑ. 
Apí nɛ kɛlõ kɛ̃ kɑpi yɛ n sée rɛ Kɔlikɔtɑ Yeesu sĩ. Kɔlikɔtɑ ɑsei rɛ Kúyu Koŋoo kɛlõ. 
Api pítɑ kɑpí nɛ pitɛ kɑpi yɛ n sée rɛ miiri n cɔ́hɔnɛ u pɑ rɛ ukɛ́ n ntí, ɑmɑ́ úu n ntí. 
Api kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii ɑpi uilũ hɔɔnɛ. Pi pɑŋɑilɛ ḿpɑ́ úye uú nɛ uikpíkɛ́kpikɛ́ céru. 
Ituŋɛ in mɛsɛ ɑ́mɛ we kɛfi kumúŋɛ́ pepu kɑpi u kɑrii kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́. 
Api lɛ̃ nnyɑ kɑpi Yeesu ń kpu wɔlu rɛ: Pisuifi uyɔɔpi. 
U nɛ pisoi kpɑ́ɑ́ree pitɛ́ pinyinɛ kɑpi kɛsẽ́ kɑrii, ucɔ ukulukɛ-lukɛ-mɛ̃, ucɔ ukumii-mɛ̃. 
Lɛ̃ kɑi Nléécɑɑsimɛ́ kɛ́mɛɛ rintíki yɛ lɛ̃: Piwɑikópɛkɔɔ́ kunɑi-i kɑpi u wɑ. 
Picéetɔŋɛ́ ɑpi u lɑ́mɑɑnkɛɛ pin tíyu pɔuli pin mɑ́ikɛɛ rɛ: Yéee! Pɔ́ɔ yɛ Uléécɑɑ kɛyɔ n fori ɑɑ siyɑ́ɑ sitɑɑni nɛ pimɔ́mɛ kɛ pɛsɛ, 
ɑ ripɔ́cúruu yoriyɛ ɑɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ súiri. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ricuruu nɛ isé picélɑɑ ɑpi kɛcɑ́ɑ́ u nyɑ́nɛi pin pimɛcɔpɛcirɛ tee rɛ: U picɔ yóriyɛntɛlɛ ɑmɑ́ úu pɛɛ fe ukɛ́ uricuruu yoriyɛ. 
Ukirisi, Isirɑyɛɛli uyɔɔpi ukɛ́ nkpéni kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ súimɛ tɔkɛ́ yɛ̃́ ɑrí nɛ kɛfɑ tɛnɛ. U nɛ pɛ̃ kɑpi kɛsẽ́ n kɑrii pɔ́ɔ kɔ u lɑ́mɑɑnkɛɛ. 
Ituŋɛ kɛcírɛ kumúŋɛ́, kuŋmɑhɑ ɑku kɛtẽ nnɛ́í cɔni hɑ́i nɛ ituŋɛ mɛtɑɑni kumúŋɛ́-pɔ. 
Kɛtúŋɛ́ mɛtɑɑni kumúŋɛ́ kpɛ-i, Yeesu uu kɛcɑ́ɑ́ilɑ píimɑ kɛnyinɛ súúni rɛ: Eloi, Eloi, lɑmɑ sɑpɑtɑni? Lɛlɛɛ rɛ Uléécɑɑ, Uléécɑɑ, yo nnyɑ kɑɑ nɛ́ ŋmɛriyɛ? 
Pisoi pɛɛ kei n cɑ́pinɛlɛ̃ kɛcɔpɛ pinyinɛ ɑpi lɛ̃ kom ɑpi rɛ: Ani ripɑí, Elii kuu nní sélei. 
Kei kɛ pikɛcɔpɛ unyinɛ uu wuri uú hɑ pítɑ nyɛpɛ-nyɛpɛ pinyinɛ-i sinɛ́ɛ̃ fɛ uu kɛnɑɑpi tuhu uu u pɑ rɛ ukɛ́ n ntí un tee rɛ: Tɔkɛ́ yɛ̃́ nkɛ̃́, in tɛ Elii yɛ́ kɑm uu u súisɛri. 
Amɑ́ Yeesu uu mɛyɑ̃́ kɛpupɛilɑ seḿ uu pɛɛ kpi. 
Kusɑ́ŋɑ́ɑ́ kɑpí nɛ Uléécɑɑ kɛyɔ kɛ́mɛɛ n kénɛlɛ̃ ɑkú cɛɛ́rinɛ mɛtɛ́, kɛcɑ́ɑ́ nɛ mɛtene-pɔ. 
Kɛ Rɔm usɔ́ɔ́cɑ kpéẽ yɛɛ pɛɛ Yeesu kɛyu-i n nyɛnu uu unkpɔ mmɛ̃ n yɛ̃́ uu rɛ: Usoi nkó yɛ pɛɛ mɛsei Uléécɑɑ Kɛpipi lɛ. 
Pinɔ́si pinyinɛ yɛ kɔ pɛɛ welɛ pin kɛtɑɑ nyɑ́nɛi. Pinɔ́si pɛ̃ kɛcɔpɛ kɛ Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri nɛ Mɑɑri Yɑkupu Síńsɑ́ nɛ Yosee uni nɛ Sɑlomee ɑpí pɛɛ we. 
Kɛ Yeesu uú pɛɛ Kɑlilee-pɔ ń we, pinɔ́si pɛ̃ pɛɛ yɛ pɛɛ u n tikilɛ̃ pin u kɛisi. Pinɔ́si picɔ yɛ kɔ pɛɛ u tikilɛnlɛ ɑpí nɛ Yerusɑlɛm sĩ. 
Nnyɔ́ɔ́ nn pitulɛ mɑsí kɛ Yosɛfi, Arimɑtee kuyu ukɔ́ uú kɑ. Yosɛfi yɛ pɛɛ pitúhɑɑnɛ pɛ̃ kɛ́mɛɛ usoi kɛcirɛ lɛ. Uyɛ̃ ticuruu yɛ kɔ pɛɛ Uléécɑɑ iyɔɔpi mɛkɑmɛ mɛ́nlɛ. Kɛyɑ́ɑ kɛ̃ ɑí nɛ sɑ́ ɑnyɑ̃́ isɔɔlú kɛyɑɑ yɛ kɛ, likɛ́ nɛ kóso weesi kɛwéntɛyɑɑ. Lɛ̃ nnyɑ kɛ Yosɛfi uu ikɑri wɑ uu Pilɑti kɛ́mɛɛ sĩ́ uú hɑ u pisɛ rɛ ukɛ́ Yeesu u pɑ. 
Ai Pilɑti pírí wɑi kuú nɛ n kṍ tɛ Yeesu yɛ nkpɔ mɑsí. Uu Rɔm usɔ́ɔ́cɑ kpéẽ séi uu u pisɛ rɛ in tɛ li nɑ́ŋɑi kɛ Yeesu uú nɛ n kpu. 
Kɛ Rɔm usɔ́ɔ́cɑ kpéẽ uu pisímɑɑ u ḿ mɑsí lɛ̃ kɑi ń we, Pilɑti uu ncée hɛ rɛ pikɛ́ Yeesu kpísi pikɛ́ Yosɛfi pɑ. 
Uyɛ̃ uu kusɑ́ŋɑ́ɑ́ lɔlu uu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ Yeesu súisɛri uu kusɑ́ŋɑ́ɑ́ u pílɛsi uú hɑ rikpíí kɑpi kupɑrɛcɑŋɑɑ-i n wɑ kɛ́mɛɛ u yekei, uu pɛɛ lɛ̃ mɛmɑ́ɑ́ mɛpɑrɛ píimɑ mɛnyinɛ pímiisɛ uu rinɔnɔɔ fi. 
Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri nɛ Mɑɑri Yosee uni ɑpi kei kɑpi u n wɑipɔ n nyɑ́nɛi.
