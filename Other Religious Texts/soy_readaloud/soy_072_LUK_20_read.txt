Luki.
20.
Kɛ Yeesu uu kɛyɑ́ɑ kɛnyinɛ Uléécɑɑ kɛyɔ-i n céési un Nsímɛ́ Kɛcirɛ símisi, pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ nɛ kuyu piwɛ́ɛ́sɛ ɑpi kpɑ́fúpɔ 
ɑpi u pisɛ rɛ: A rɔ́ símisi, ńnɑŋɛ ń-ye kɑɑ́ nɛ nní wɑi? Wóo ncée pɔ́ hɛ rɛ ɑ lɛ̃ wɑ? 
Yeesu uu pɛɛ pi pɛsɛ rɛ: Nɛ lɑ kɛ́ nsɛ nɔ́ pisɛ. Ani nɛ́ símisi: 
Nɛ lɑ kɛ́ nɔ́ pisɛ rɛ Uléécɑɑ nɛ́ɛ pisoi pɛɛ Yohɑni tummɛ rɛ ukɛ́ pisoi míni wolɛ? 
Api pimɛcɔpɛcirɛ n kɛŋɛnɛ rɛ: Tɔn nkpéni m mɑɑ rɛ Uléécɑɑ yɛɛ u tummɛ, u yɛ́ rɔ́ pisɛ rɛ: Kɑi pɛɛ íye wɑ kɑ́ri nɛ Yohɑni nsímɛ́ ŋmurɛi? 
Amɑ́ tɔn kɔ m mɑɑ rɛ: Pisoi pɛɛ u tummɛ, kuyu nnɛ́í pikɔ́ yɛ ɑpɑrɛ rɔ́ tɔ́ɔ́kɛɛnɛ ɑpi rɔ́ kóni, likumúŋɛ́ rɛ pi nɛ kɛfɑ tɛnɛlɛ rɛ Yohɑni yɛ ɑntepu lɛ. 
Api pɛɛ rinɔ́ɔ u yɔ́su rɛ: Ári nyu uyɛɛ píniwolɛ u n tummɛ. 
Yeesu uu rinɔ́ɔ pi pɛsɛ rɛ: Nɛ̃́ ɑ́m nɔ́ símisinɛ ńnɑŋɛ mmɛ̃ kɑḿ nɛ lɛ̃ n wɑi. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ kɛnyɑ́rũ nkɛ́ pisoi mɑɑ rɛ: Utisi unyinɛ yɛɛ we, uu ɑléé nyɛ̃ kɑpi yɛ n sée rɛ fiinyi kɛcɑrɛ tɑmɛsi uu pikɛikɔ́ wɛ́ɛsi rɛ pikɛ́ kɛkɛcɑ́ɑ́ m pɑílɛ̃, uu pɛɛ ncée tɑɑ-tɑɑ sĩ ukɛ́ hɑ nɑ́ŋɑipɔ. 
Kɑi ɑléépipi nyɛ̃ kutúũ n tu, uu ukɛikɔ́ unyinɛ pɛpɛɛ ɑléé nyɛ̃ kɛcɑrɛ kɛcɑ́ɑ́ ḿ pɑílɛ̃ kɛ́mɛɛ tum tɛ ukɛ́ hɑ ɑléépipi ɑnyinɛ u yɔ́ɔmɛ. Amɑ́ ɑpi u kɑii ɑpi mɛnípɛ ŋmɑnɛ u lɑkɑsɛ. 
Ucɑrɛte uu kɔ ukɛikɔ́ féé tumpɔ, ɑmɑ́ pɛpɛɛ ɑléé nyɛ̃ kɛcɑ́ɑ́ ḿ pɑílɛ̃ ɑpi kɔ uyɛ̃ ticuruu kɑii ɑpi u lɑ́misi ɑpi mɛnípɛ ŋmɑnɛ u lɑkɑsɛ. 
Uu kɔ ukɛikɔ́ tɑ́ɑ́nũ pitumɛ kpɑ́, ɑpi uyɛ̃ pɔ̃́ itói wɑi ɑpi lɑ́kɑsɛ. 
Ucɑrɛte uu pɛɛ rɛ: Íye kɑm nkpéni wɑinɛ? Nɛ́ unɛ́ŋmɑ́nɛ lɑlɑ tũ. In n lɑ, pi yɛ́ u wuri. 
Amɑ́ kɛ pɛpɛɛ ɑléé nyɛ̃ kɛcɑ́ɑ́ ḿ pɑílɛ̃ ɑpi u n yɛ̃́, ɑpi símɑɑnɛ rɛ: Nkó yɛɛ yɛ́ mɑsí usɑ́ɑ ikulɑnlũ nnɛ́í n te. Ani kɑm tɔkɛ́ u kpu ɑri pɛɛ ikúlɑnlũ iyɛ̃ n té. 
Api kɛcɑrɛ kɛ̃ ilúkɛ́ u lesɛpɔ, ɑpi u kopu. Íye kɛ ucɑrɛte uu wɑinɛ? 
U sɔ́ntilɛ ukɛ́ pi kṍ, uu pisoi féé kɛcɑrɛ múísɛ. Kɛ pisoi ɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, ɑpi rɛ: Asei kɛcɑ́ɑ́, ɑ́i lɛ̃ wɑinɛ. 
Amɑ́ Yeesu uu pi nyɑ́nɛi uu rɛ: In lɛɛ ni, lɛ̃ kɑpi nní Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́i ɑsei rɛ íye? Ái pi wɔi rɛ: Tipɑrɛ kɛ pimɔ́mɛ ɑpi n fóm tɛɛ pɑnsɛ ricírɛ-cirɛ tɛɛ kɛnui-i ń wee? 
Lin úye n fóm un tipɑrɛ tɛ̃ tinkpɔu, u yɛ́ selesi. Lin úye n fóm tin ukɛcɑ́ɑ́ n loó, ti yɛ́ u nɑ̃. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑpi kumúŋɛ́ kpɛ-i n wɛ́ɛ́si pikɛ́ Yeesu tĩ, ɑmɑ́ pin pɛɛ kuyu pikɔ́ wuru. Pi ceri rɛ pɛ̃ kɛ Yeesu kɛnyɑ́rũ ɑkɛ́ nɛ mɑ́nɛ. 
Api pɛɛ Yeesu pimúŋɑɑ loni. Api lɛ̃ nnyɑ pisoi pinyinɛ ukɛmɛɛ tumpɔ, ɑpí n wɑi yɑrɛ pisoi pɛɛ ɑsei n wɛ́ɛ́si. Li pɛɛ lɛ̃ wɑilɛ pikɛ́ nɛ n fe pikɛ́ u n tĩ́ pikɛ́ uyɔ́ɔpi yɛɛ ńnɑŋɛ m mɑ́ ukɛ́ nɛ u n túhɑɑnɛ ɑnipɛ-i u n wɑ nnyɑ. 
Api pɛɛ u pisɛ rɛ: Sɑ́ɑ, tɔ yɛ̃ tɛ lɛ̃ kɑɑ n símisi pɔn céesi yɛ ɑsei lɛ, ɑ́ɑ nkpɑ́ni usoi úkɑ kɛwuu túúni. Pɔ yɛ Uléécɑɑ ncee pisoi céesilɛ nɛ ɑsei. 
Lɛ̃ nnyɑ, ɑ rɔ́ símisi in tɛ irɔ́sé yɛ ncée hɛ rɛ pikɛ́ Sesɑɑ lɑmpoo hɛ́ɛlɛ nɛ́ɛ ɑ́pi kɑpɛ hɛ́ɛ́lɛ? 
Amɑ́ Yeesu uu pimɛsɔhɔ ceru, uu pi mɑɑ rɛ: 
Ani kɛwóó kɛnyinɛ nɛ́ nyísɛ nkɛ̃́. Úye ríyu nɛ úye rinyiri tɛɛ kɛkɛcɑ́ɑ́ we? Api rinɔ́ɔ yɔ́su rɛ Sesɑɑ lo. 
Yeesu uu pɛɛ pi mɑɑ rɛ: In lɛ̃, ɑni Sesɑɑ likɔ́ Sesɑɑ pɑ, ɑni Uléécɑɑ likɔ́ Uléécɑɑ pɑ. 
Ápi fe pikɛ́ u tĩ, mmɛ̃ kuu nní kuyu pikɔ́ kɛyu-i n símisi nnyɑ. Amɑ́ umɛyɔ́ɔ́ mɛ̃ ɑmɛ nnɔ́ɔ pi yipu ɑpi sɛ́ɛ́. 
Pisɑtusee pinyinɛ ɑpi Yeesu kɛ́mɛɛ hɑpɔ. Pɛpɛɛ múílɛ̃ tɛ pikpɔkpɔ ɑ́pi nkpɔ-i yisinɛ. 
Api nní Yeesu pisɛ rɛ: Sɑ́ɑ, lɛ̃ kɛ Moisi uu isé ń wɔ́i yɛ nní: Úye umɑɑ un n kpu uu unɔ́si tíyɛ úu nɛ kɛwɑ̃́, uuwɑ̃ yɛɛ yɛ́ unɔ́si uyɛ̃ kukúmɑnnɔsi kpísi uu umɑ́ɑ mpuri wɑi. 
Amɑ́ pimɑ́rɛcɔ pisɛɛi pinyinɛ pɛɛ pɛɛ́ we. Ufoí uu unɔ́si kpísi, uu kpi úu nɛ kɛwɑ̃́. 
Ulírũ uu ukúmɑnnɔsi kpísi, 
utɑ́ɑ́nũ pɔ́ɔ kɔ pikpíkɛ́ u kpɑ́, uyɛ̃ uu kɔ kpi. Pikɛsɛɛi kɛ̃ ɑpi lɛ̃ kukúmɑnnɔsi u kpíkɛsi ɑpi kpíni úkɑ úu kɔ pɛɛ́ nɛ kɛwɑ̃́ u mɑri. 
Unɔ́si pɔ́ɔ kɔ mɛtɔ́rɔɔ kpi. 
Kɛyɑ́ɑ kɛ̃ kɛ pikpɔkpɔ ɑpi nkpɔ kɛ́mɛɛ n yisinɛ, pitisi pɛ̃ kɛsɛɛi kɛcɔpɛ úye yɛɛ yɛ́ hɑ uulɑ? Liriyíkí rɛ pinnɛ́í pɛɛ kunɔ́si u kpíkɛsi! 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Kɛtẽ nté pitisi yɛ pinɔ́si kpíkɛsilɛ, pinɔ́si pɔ́ɔn kɔ pilɑ sɔ́ɔnkɛɛ. 
Amɑ́ pitisi nɛ pinɔ́si pɛɛ nɛ n sɑ́ rɛ pikɛ́ pikpɔkpɔ kɛ́mɛɛ yisi pikɛ́ hɑ kɛtẽ fɑlɛ-i n we ɑ́pi pinɔ́si kpíkɛsinɛ, ɑ́pi kɔ pilɑ sɔ́ɔnkɛɛnɛ. 
Ápi píkɑi kpinɛ, piléécɑɑtumɛ kɑpí nɛ mɛnyínɛ we. Pi Uléécɑɑ sipipi lɛ, liriyíkí rɛ pi kpulɛ ɑpi kɔ nkpɔ kɛ́mɛɛ yisi. 
Moisi yɛ pɛɛ pikpɔkpɔ mɛyisɛ nkpɔ kɛ́mɛɛ nkɔ́ pisímɛ́ mɑsilɛ uritɛlɛ́ tɛ-i kuu kuhííhíí kpɛɛ n torɛ nsímɛ́ ń símisi. U pɛɛ́ mɑɑ rɛ: Upíimɑ yɛ Apirɑhɑm Uleecɑɑ nɛ Isɑɑki Uleecɑɑ nɛ Yɑkupu Uleecɑɑ lɛ. 
Yeesu uu kpɑ́ rɛ: Uléécɑɑ úu pikpɔkpɔ Uleecɑɑ, ɑmɑ́ u pinyɑ́nnyɑ̃́ Uleecɑɑ lɛ. Liriyíkí rɛ uyɛ̃ nnyɑ kɛ pinnɛ́í ɑpi nyɑ́ni. 
Isé picélɑɑ pinyinɛ ɑpi rinɔ́ɔ yɔ́su ɑpí mɑɑ rɛ: Sɑ́ɑ, pɔ símisilɛ lin nyɑḿ. 
Lɛ̃ mɛmɑ́ɑ́, ɑ́pi ńkɑ ncɔ pipisɛ u peenu. 
Yeesu uu pi mɑɑ rɛ íye kɛ usoi uu yɛ́ fe uú mɑɑ rɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ yɛ Tɑfiti kɛpipi lɛ. 
Likumúŋɛ́ rɛ Tɑfiti yɛ siyómɛ ritɛlɛ́-i rihɑɑ rɛ: Upíimɑ yɛ Unɛ́píimɑ mɑɑ rɛ: A kunɛ́lukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tonɛ 
hɑ́i nɛ kumúŋɛ́ kpɛ-i kɑm yɛ́ pipɔ́lɑ́ɑrɔ kɛpɔ́nyɑ́lɛ-i n cɔ. 
Tɑfiti yɛ̀ɛ̀ u sée rɛ Upíimɑ. Íye kɛ uyɛ̃ cirɛ uu yɛ́ kɔ pɛɛ Tɑfiti Kɛpipi? 
Kɛ risoiwuí nnɛ́í ɑri kutu n cɔ́lɛ̃, Yeesu uu upipirɛtiki mɑɑ rɛ: 
Ani isé picélɑɑ nɛ nɔ́mɛnɛ́círɛ tĩ. Pikɛ́ nɛ ɑtúkɑnkɑ cɑɑ-cɑɑ n kɔɔnii nɛ pisoi pikɛ́ ɑyɑ́ɑ kɛ́mɛɛ pi n yɑ́hɑɑnkɛɛ yɛɛ ripɔ́ɔ pi we. Ayómɛyɑ́hɑɑlee sitonɛ foí nɛ kɛlukɛ kɛ́mɛɛ sitonɛ foí kɑpí lɑ. 
Pi yɛ lɛ̃ kɛ pikúmɑnnɔsi ɑpi m mɑ́ nnɛ́í yɔ́ɔilɛ ɑpi yɛ pɛɛ kɛyómɛ yɑ́ɑ́si mɛcɑ́ɑ yɑrɛ pi pisoi sɔnɛ lɛ. Uléécɑɑ yɛ́ nɛ pi túhɑɑnɛ uu íwɛ pi wɑi ɑ́i kumúŋɛ́ mɑ́.
