Mɑtiyee.
17.
Siyɑ́ɑ sikpuulũ mɛmɑ́ɑ́, Yeesu uu Piyɛɛ nɛ Yɑkupu nɛ uyɛ̃ uwɑ̃ Yohɑni kpísi ɑpí nɛ riyɔ́pɛ cɑ́cɑ́ rinyinɛ tɑɑ́ ɑpí hɑ kei pimɛcirɛ n we. 
Kei kɛ uipiŋɛ ii piinipɛɛ iyɛ-i consɛ. Ukɛyu ɑkɛ́ n tɛ́lu yɑrɛ ituŋɛ. Uilũ ii wɑ́rɑrɑ yɑrɛ mɛtɛ́í. 
Moisi nɛ Elii ɑpi pɛɛ kpɑ́fúmɛ, pi nɛ Yeesu pin yɔ́i. 
Kei kɛ Piyɛɛ uu pɛɛ Yeesu mɑɑ rɛ: Upíimɑ, li nyɑmlɛ rɛ tɔkɛ́ nté n we. Pɔn n lɑ, nɛ́ sicɑ́ŋíípi sitɑɑni nté kɑrii, pɔ́ɔ kɛsɛ kpísi, Moisi kɛsɛ, Elii kɛsɛ. 
Un kei lɛ̃ símisi, kuhopɛ pei-pei kunyinɛ ɑkú weri ɑku pi hilɑ. Tinɔ́ɔ rinyinɛ ɑri kuhopɛ kpɛ-i léeri rɛ: Nkó yɛɛ Kɛnɛ́pipi lɑlɑ kɛɛ nnɛ́pɔ́ɔnɑrɛ nnɛ́í ń te. Ani kutu kɛ ricɔ! 
Kɛ Yeesu pipirɛtiki ɑpi rinɔ́ɔ tɛ̃ n kṍ, iwɑmɛ píimɑ ii pi loni ɑpi lólu ɑpi kɛtẽ kɛ́yu tuhu. 
Yeesu uu pɛɛ́ nyɔsɔ́pɔ uú nɛ pi kɔ uu rɛ: Ani yisi, iwɑmɛ íi kɑpɛ nɔ́ wɑ. 
Kɑpi inípɛɛ rinsíkɑ, Yeesu mɛcirɛ kɑpi yɛ̃́. 
Kɑpi riyɔ́pɛ n cɛ́pilɛ̃, Yeesu uu rinɔ́ɔ ntí pi hɛ: Ani n sɛ́ɛ́lɛ̃, ɑ́ni kɑpɛ úkɑ lɛ̃ kɑni n yɛ̃́ símisi likɛ́ hɑ nɛ kɛyɑ́ɑ kɛ̃ kɛ Usoi Kɛpipi ɑkɛ yɛ́ nkpɔ kɛ́mɛɛ n yisi tu. 
Upipirɛtiki ɑpi pɛɛ u pisɛ rɛ: Yo nnyɑ kɛ isé picélɑɑ ɑpi rɛ Elii yɛɛ mɛfoí sɔ́nti likɛ́ kɛlenɛ kɛtẽ kɛtɔ tu? 
Uu pi pɛsɛ rɛ: Li pisɛ ɑsei kɛcɑ́ɑ́ rɛ Elii ukɛ́ mɛfoí pɛɛmɛ ukɛ́ ḿpɑ́ yo ripɔhɔ-i li yekei. 
Nɛ̃́ nɛ lɑ kɛ́ nɔ́ símisi rɛ Elii yɛ pipɛɛmɛ mɑsilɛ pisoi ɑ́pi nkpɑ́ni u ceri, ɑmɑ́ ɑpí nɛ u wɑi yɑrɛ kɑpi ń lɑ. Limɛcɔ kɛ Usoi Kɛpipi ɑkɛ kɔ piɑnipɛ-i íwɛ lenɛ. 
Kei kɛ upipirɛtiki ɑpi pɛɛ kṍ tɛ Yohɑni Úniwolɛ nsímɛ́ kuu pi símisi. 
Kɛ Yeesu nɛ upipirɛtiki ɑpi risoiwuí nɛ rińkɔ, utisi unyinɛ uu ukɛkúrí nyɔsɔ́pɔ uu uɑnɑ-i wúlɑ. 
Uu rɛ: Upíimɑ, ɑ kɛnɛ́wɑ̃́ nkɛ́ íwɛ tẽ́! Kɛ rikpínkpĩ́ pɔ́ɔ́lúlɛ ɑkɛ yɛ mɛsɛ́rɛ lóirɛntɛ ɑkɛ́ nɛ míni-i nɛ́ɛ nnɑ-i lóipɔ. 
Nɛ nɛ pipɔ́pirɛtiki kɛ lɛɛpɔlɛ ɑ́pi fe pikɛ́ kɛ pɔisɛ. 
Yeesu uu pɛɛ rɛ: Nɔ́ nfɑtɛnɛcirɛ́ nɛ pikpɑ́ɑ́ree mpuri mmú! Ituŋɛ ilɛ́ kɑm nɔ́kɛnɛ́mɛɛ pitónɛ kpɑ́nɛ? Ituŋɛ ilɛ́ kɑm yɛ́ lɛ̃ nɛ nɔ́ n kɑhɑri? Ani nɛ kɛwɑ̃́ kɛ̃ nté nɛ́ kɑm. 
Yeesu uu kuníri sɛnkɛɛ ɑku kɛŋmɑ́nɛpi kɛ-i lelu ɑkɛ mɛyéne mɛsɛ kei nní pélu. 
Kei kɛ upipirɛtiki ɑpi pɛɛ ukɛkúrí hɑ́pɔ ɑpi iyɑɑ u pisɛ rɛ: Yo nnyɑ kɑ́ri fe tɔkɛ́ kɛŋmɑ́nɛpi kɛ̃ ɑníri pɛsɛ? 
Yeesu uu pi pɛsɛ rɛ: Kɛ nɔ́nnɛ́fɑtɛnɛ nn n fíílɛ̃ nnyɑ yɛ ni. Asei kɑm nɔ́ símisi rɛ nɔ́nnɛ́fɑtɛnɛ nn pɛɛ mpuri yukɛ-yukɛ sei nnyinɛ n tulɑɑlɛ̃, nɔ́ yɛ pɛɛ fe ɑni riyɔ́pɛ ntí mɑɑ rɛ: A nté yisi ɑ hɑ nté-pɔ nyɛrɛ, ɑri mɛsei lɛ̃ wɑi. Líkɑ ɑ́i yɛ́ pɛɛ n we kɑni yɛ́ piwɑi m pɔɔnɛ. 
Piyómɛyɑ́hɑɑ nɛ pinɔ́ɔpɑhɑɑ kɛ usoi uu yɛ́ nɛ ɑníri nnyɛ́ mpuri pɛsɛ. 
Kumúŋɛ́ kpɛ-i kɑpi Kɑlilee kɛtẽ n kɔ́ɔ́núlɛ̃, Yeesu uu pɛɛ pi mɑɑ rɛ: Pi yɛ́ Usoi Kɛpipi tĩ ɑpi pisoi ɑnipɛ-i wɑi 
ɑpi kɛ kopu, ɑkɛ pɛɛ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ nkpɔ kɛ́mɛɛ yisi. Kuu lɛ̃ ḿ mɑ, upipirɛtiki ɑpɔɔ ɑɑ cɑɑi hɑ́i. 
Kɛ Yeesu nɛ upipirɛtiki ɑpi Kɑpɛrinɑwum n tuipɔ, pɛpɛɛ Uléécɑɑ kɛyɔ kɛ́mɛɛ lɑmpoo n yɔ́su ɑpi Piyɛɛ kɛkúrí sĩ́ ɑpí hɑ u pisɛ rɛ: Nɔ́unɛ́wɛ́ɛ́sɛ úu yɛ Uléécɑɑ kɛyɔ ulɑmpoo hɛ́ɛlɛ nɛ́ɛ? 
Piyɛɛ uu pi pɛsɛ rɛ: U yɛ hɛ́ɛlɛlɛ. Kuu kɛ́yɔ-i m pɛɛpɔ, Yeesu uu u pisɛ rɛ: Íye kɑɑ músu Simɔɔ? Piyupipi nɛ́ɛ pisɑ́nɛ kɑi pisɛ rɛ pikɛ́ kɛtẽ nté piyɔɔpi lɑmpoo nɛ ítɔhɔ m pɑsii? 
Piyɛɛ uu u pɛsɛ rɛ: Pisɑ́nɛ yɛ pi. Kei kɛ Yeesu uu pɛɛ rɛ: In lɛɛ ni, ɑ́i pisɛ rɛ piyupipi pikɛ́ hɛ́ɛ́lɛ. 
Amɑ́ ɑ́ri lɑ mɛrɔ́wɑi mɛnyinɛ mɛkɛ́ pi ricɔ́ŋ. Lɛ̃ nnyɑ, ɑ hɑ kupiyɛ kɛ́mɛɛ kɛhímpi ripɔlɔ ɑɑ nkpíntomɛ́ foí kɑɑ ń tininɛ lesɛri ɑɑ nnɔ́ɔ n hɑɑsɛ. Pɔ́ kɛ́mɛɛ siwóó sɛ̃ kɑi rɔ́ m pisɛ lɛɛpɔ, ɑɑ si kpísi ɑɑ́ nɛ tɔ́ nɛ mpɔ̃́ ulɑmpoo hɛ́ɛ́lɛ.
