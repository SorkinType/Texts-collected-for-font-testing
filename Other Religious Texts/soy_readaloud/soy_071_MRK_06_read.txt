Mɑriki.
6.
Kɛ Yeesu uu kɛlõ kɛ-i n yisi uu kuyu kpɛ-i kuu iwɑ̃́ n li kɛ́mɛɛ sĩ́, upipirɛtiki ɑpi u tiki. 
Kɛ kɛwéntɛyɑɑ kɛtúŋɛ́ ɑkɛ n tu, uu Pisuifi kuyomɛyɑ́hɑɑlee-i lompɔ uu pisoi n céési. Nsímɛ́ nn nnɔ́ɔ pisoi píimɑ pɛ̃ nnɛ́í pɛɛ kutu u n cɔ́lɛ̃ yipu. Apí m písɛinɛ rɛ: Yei kɛ nní yɛ nnɛ́í ɑi mpíí u léenti mɑ́? Mɛwɛ́ɛ́sɛsɔhɔ mɛ́ye mpuri kɑi nkó nní hɛ mɑ́? Íye kuú nɛ mɛwɑisɑŋɑ mmɛ́ mpuri wɑpisi mɑ́? 
In nɛ Mɑɑri kɛpipi usɛ́rɛ́ɛ? Nɛ́ntɛ upiwɑ̃ pɛɛ Yɑkupu nɛ Yosee nɛ Yuuti nɛ Simɔɔ? Ái upiyɛi pɛɛ nní kɛrɔ́cɔpɛ wee? Nní nnɛ́í ɑi pi tɑpisi ɑ́pi nɛ Yeesu kɛfɑ tɛnɛ. 
Yeesu uu pɛɛ rinɔ́ɔ pi pɛsɛ rɛ: Antepu kuyu cirɛ-i kɑpi yɛ u lómiisɛ. Umpuri pikɔ́ nɛ upiyɔkɔ́ pɛɛ ɑ́pi yɛ u wɑisɛ. 
Yeesu úu lɛ̃ mɛmɑ́ɑ́ fe ukɛ́ mɛwɑisɑŋɑ piwɑi rikpɑ́, insɑ́ pitóikɔ́ pinyinɛ kɛcɑ́ɑ́ kuu ɑnípɛ rinlɑ́ɑ uu pi pɔisɛ. 
Ai Yeesu pírí wɑi kɑ́pi nɛ kɛfɑ u n tɛnɛ. Lɛ̃ mɛmɑ́ɑ́, Yeesu uu ɑyu ɑcɔ nyɛɛ nɛ kei n kɔ́lɛ̃ kɛ́mɛɛ sĩ́ uu ɑpisoi nléécɑɑsimɛ́ céési. 
Uu pɛɛ upipirɛtiki kɛfi nɛ pitɛ́ séi uu pi tum pitɛ́ pitɛ́, uu ńnɑŋɛ pi hɛ rɛ pikɛ́ ɑníri n lɑkɑsɛ. 
Uu rinɔ́ɔ pi hɛ rɛ: Áni kɑpɛ kunɑɑpi mɛmɑ́ɑ́ líkɑ rimúlú yɑrɛ ilukɛ nɛ́ɛ kulɔ́ɔ nɛ́ɛ kɛwóó rɛ ncée kɑni nɛ sĩ. 
Amɑ́ uu rɛ: Ani ɑnɛ́ɛ́ri mɛtɑnɛ mɛsɛ mɛsɛ tɑ́nti, ɑmɑ́ ɑ́ni kɑpɛ sitúkɑnkɑ sitɛ́ sitɛ́ tɑ́nti. 
Uu pimɑ́ɑ pi kpɑ́ rɛ: Nɔn kɛ́yɔ kɛ́ye-i n topilɛ, ɑni kei n tũ hɑ́i nɛ kumúŋɛ́ kpɛ-i kɑni kuyu kpɛ̃ n yisinɛ. 
Pisoi pin yei kusɑ́nɛ piyɔ́ɔ́ nɔ́ n yɛ̀ nɛ́ɛ kutu kɑ́pi nɔ́ ricɔ, nɔn n tɔ́su, ɑni nɔ́ɑnɛ́nɑ nkoŋo péi. Likɛ́ pi nyísɛ rɛ pi Uléécɑɑ nsímɛ́ pipɑkɑrɛ yɛ̀. 
Pipirɛtiki pɛ̃ ɑpi ncée pɔlɔ ɑpi pisoi nléécɑɑsimɛ́ n símisi pin tee rɛ pikɛ́ pimɛfinɛ consɛ. 
Aníri mɛyɑ̃ kɑpi lɑ́kɑsɛ, ɑpi pitóikɔ́ mɛyɑ̃ kɛcɑ́ɑ́ mɛ́kpɔ kɔɔni itói ii tɛnɛntɛ. 
Lɛ̃ kɑi wɑ Yeesu rinyiri ɑri ḿpɑ́ yei kɔ́ɔ́nú, uyɔ́ɔpi Erooti uú nɛ Yeesu nnɛ̃́ kóm. Pisoi ɑpi pɛɛ́ n símɑɑnkɛɛ rɛ: Yohɑni Úniwolɛ yɛ pikpɔkpɔ kɛ́mɛɛ yisilɛ, lɛ̃ nnyɑ kuu nní ńnɑŋɛ mɑ́ un mɛwɑisɑŋɑ wɑpisi. 
Picɔ pin mɑ́ikɛɛ rɛ Elii lo. Picɔ pɔ̃́ tɛ mɛkɛɛ ɑntepuyɛ kɛcɔpɛ usɛ unyinɛ lo. 
Amɑ́ kɛ Erooti uu lɛ̃ n kṍ, uu rɛ: Yohɑni uyɛ̃ tiyu kɑm n tíyɛsɛ ɑpi tɔ́lu yɛɛ yisi. 
Likumúŋɛ́ rɛ Erooti ricuruu yɛɛ rinɔ́ɔ hɛ ɑpi Yohɑni pɑɑsi ɑpi kukpɑniilee-i sɑ́pɔ. Ái líkɑ nnyɑ kɛ Erooti uu lɛ̃ wɑ. Erooti yɛ pɛɛ uumɑɑ Filipu unɔsi Erootiyɑti yɔsilɛ. 
Kɑi lɛ̃ n wɑ, Yohɑni uu pɛɛ u mɑɑ rɛ ɑ́i nɛ risɑ́ rɛ ukɛ́ uumɑɑ unɔsi yɔsí. 
Erootiyɑti uu lɛ̃ nnyɑ Yohɑni ukɛfɑ-i wɑi uu ncée wɛ́ɛ́si ukɛ́ nɛ u kpu. Amɑ́ úu fe. 
Kɛ Erooti uú pɛɛ Yohɑni m pɑkɑrɛlɛ̃ un kɔ nyu rɛ u ɑsei tikilɛnlɛ, uisoi in kɔ kɛcirɛ we nnyɑ, Erooti úu yɛ pɛɛ Yohɑni kɛcɑ́ɑ́ kṍ, un unsímɛ́ kutu n cɔlɛ̃ n yɛ u ricɔ́ŋlɛ, ḿpɑ́ nɛ lɛ̃ u kɔ pɛɛ mpikomɛ lɑlɛ. 
Amɑ́ kɛyɑ́ɑ sɔnɛ ɑkɛ Erootiyɑti leeri. Kɛ Erooti uu ukɛmɑrɛyɑɑ ɑnyɑ̃́ n wɑi, uu lɛ̃ nnyɑ sɔ́ɔ́ni uu ukuyu pinɑŋɛ-nɑŋɛ nɛ pisɔ́ɔ́cɑ piwɛ́ɛ́sɛ nɛ Kɑlilee kɛtẽ pisoi kɛcirɛ sélei. 
Kei kɛ Erootiyɑti ukpére uu kɛnyɑ́nkɑlɛ-i lompɔ uu yésu ɑi Erooti nɛ upisɑnɛ nnɛ́í lɑrisi. Uyɔ́ɔpi Erooti uu ukpére uyɛ̃ mɑɑ rɛ: A lɛlɛɛ ripɔ́ɔ pɔ́ ń we nɛ́ we, kɛ́ pɔ́ li hɛ. 
Ai wɑi uú hɑ nɛ wééri rɛ: Pɔn yo nɛ́ n we, nɛ́ pɔ́ li hɛ, ḿpɑ́ in kɛnɛ́yɔ́ɔpitẽ kuwɛ́lɛ́, nɛ́ pɔ́ hɛ. 
Kɛ ukpére uyɛ̃ uu kɛkɑ́lɛ-i n léepɔ, uú hɑ úni pisɛ rɛ: Yo kɑm yɛ́ pisɛ? Uyɛ̃ uu u pɛsɛ rɛ: A Yohɑni Úniwolɛ riyu pisɛ. 
Uu mɛkɛɛkɛɛ uyɔ́ɔpi-mɛ̃ pɛlɛ uu u mɑɑ rɛ: Nɛ lɑ rɛ ɑ nfɑ́ɑni Yohɑni Úniwolɛ riyu kɛcɑ́ripi-i nɛ́ ritɔsi. 
Uyɔ́ɔpi ripɔɔ ɑri cɑɑi, ɑmɑ́ uɑwééri nɛ pisɑ́nɛ isɛi nnyɑ, ɑ́i kúyulɛ we. 
Uu mɛsɛ nɛ mɛsɛ uumɛ́rɛ́ usɛ yukusɛ rɛ ukɛ́ nɛ Yohɑni riyu u kɑm. Umɛ́rɛ́ uú hɑ uriyu tɔlu kukpɑniilee-i. 
Uu ríyu kɛ́pɛlɛntɛ kɛcɑ́ɑ́ tɔ́si uú hɑ ukpére uyɛ̃ ti múísɛ, ukpére pɔ́ɔ úni ri pɑ. 
Kɛ Yohɑni pipirɛtiki ɑpi lɛ̃ n kṍ, ɑpí weri ɑpi u kpísi ɑpí hɑ kulɛsi. 
Yeesu pitumɛ ɑpi Yeesu kɛyúrí cɑ́pinɛ, ɑpi lɛ̃ nɛ lɛ̃ kɑpi n wɑpisi nɛ lɛ̃ kɑpi pisoi n céesi u kɛɛnkɛɛ. 
Pisoi pɛɛ Yeesu-i n sɔ́nɑɑpɔ nɛ pɛpɛɛ ukɛmɛɛ n léeri yɛ kulúi tɔsilɛ. U nɛ upipirɛtiki ɑ́pi yɛ pɛɛ nkpɑ́ni kɛ́yu yɛ̃ pikɛ́ ilukɛ li. Lɛ̃ nnyɑ kɛ Yeesu uu pipirɛtiki mɑɑ rɛ: Tɔkɛ́ iyɑɑ le, kei kɛ pisoi ɑ́pi ń we ɑní hɑ nkɑ́ripi wéntɛ. 
Api kúninɔi loni pikɛ́ kei kɛ úkɑ úu ń we hɑ. 
Pisoi mɛyɑ̃ ɑpi pimɛtɔ́ŋɛ́ yɛ́nu ɑpi kɔ pi ceru. Ayu nyɛɛ nɛ pi n kɔ́lɛ̃ nnɛ́í pisoi ɑpi ntóó wɑi ɑpi kei kɑpi n sĩ́ pi fɔ́su. 
Kɛ Yeesu uu kúninɔi-i n léemɛ, uu pisoi kulúi yɛnu ɑi íwɛ u wɑi rɛ pi pɛɛ we yɑrɛ isɛɛ yɛɛ íi usẽ́ m mɑ́ nnyɑ. Uu mɛsɛ picélɑɑ pi loni ɑi nɑ́ŋɑi hɑ́i. 
Kɛ ituŋɛ ii n tɛnɛ, upipirɛtiki ɑpí nɛ u kɔ ɑpi uu mɑɑ rɛ: Líkɑ ɑ́i kɛlõ nkɛ́-i we. Lɛ̃ mɛmɑ́ɑ́, ituŋɛ yɛ kɔ tɛnɛlɛ. 
A pi mɑɑ rɛ pikɛ́ hɑ siyupi nɛ ɑyu nyɛɛ nɛ rɔ́ n kɔ́lɛ̃ kɛ́mɛɛ lilukɛ-lukɛ lɛ́ɛ́ri. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Ani nɔ́rinɛ́cúruu lilukɛ-lukɛ pi hɛ. Api u pisɛ rɛ: Tɔ nkpéni sínlɛ tɔkɛ́ nwóóweni mɛpipi pílɛ pílɛ mɛtɛ́ nɛ ɑkpɔ́nɔ́ lɔ ɑri pi pɑ pikɛ́ li nɛ́ɛ? 
Yeesu uu pi pisɛ rɛ: Akpɔ́nɔ́ ɑlɛ́ kɑni mɑ́? Ani hɑ ripɑí. Api písɛinɛ ɑpi limɛmɑ́ɑ́ Yeesu rinɔ́ɔ yɔ́su rɛ: Akpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́. 
Uu pɛɛ rinɔ́ɔ pi hɛ rɛ pikɛ́ mɛyúi pɛ́ni kɛcɑ́ɑ́ ɑwúí ɑwúí pi tonsɛ 
ɑpi ɑnɑ́i ɑnɑ́i túnti, ɑcɔ pisoi pílɛ pílɛ, ɑcɔ kuwóó kuwóó. 
Uu ɑkpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́ iyɛ̃ kpísi, uu kɛyómɛ weríí, uu ilukɛ iyɛ̃ kɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ. Uu lɛ̃ mɛmɑ́ɑ́ ɑkpɔ́nɔ́ kpɔ́kɔrinɛ uu upipirɛtiki pɑ rɛ pikɛ́ pisoi hɔɔnɛ. Uu kɔ ikpíntomɛ́ itɛ́ iyɛ̃ pinnɛ́í hɔ́ɔ́nɛ. 
Api pinnɛ́í le ɑpi lɛpu. 
Upipirɛtiki ɑpi ɑkpɔ́nɔ́ nɛ ikpíntomɛ́ keri cɑ́pinɛntɛ ɑi ɑnɛ́rɛ kɛfi nɛ ɑtɛ́ yipu. 
Pɛpɛɛ ilukɛ iyɛ̃ n li kɛ́mɛɛ pitisi yɛ lelɛ ɑ́kotokú ɑnupũ (5.000). 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ mɛsɛ nɛ mɛsɛ upipirɛtiki rinɔ́ɔ hɛ rɛ pikɛ́ kúninɔi lõ pikɛ́ hɑ rikóitimɛ ricɔ-pɔ u m mɛ̃́, Petisɑyitɑ kuyu kuwɛ́ɛ́-mɛ̃, uyɛ̃ ukɛ́ pɛɛ kumúŋɛ́ kpɛ-i pisoi n kúúnsɛ. 
Kuu pisoi rin-yɑ́, uu riyɔ́pɛ tɑɑ́ ukɛ́ kɛyómɛ yɑ́ɑ́si. 
Kɑi nnyɔ́ɔ́ ń wɑ, kúninɔi kun mínimɑɑ kɛcɔpɛ we. Yeesu pɔ́ɔn umɛcirɛ kɛtẽ mmɛ́ we. 
Uu yɛ́nu rɛ kúninɔi pitíkíí yɛ íwɛ pi wɑilɛ, likumúŋɛ́ rɛ mɛ̃ kɑpi n sĩ́ kɛ kuyɔ píimɑ kunyinɛ ɑku pépuri. Tiweeki kumúŋɛ́, uu pikuwɛ́lɛ́-mɛ̃ n sɔ́nɛpɔ míni kɛcɑ́ɑ́, un lɑ ukɛ́ pi rifɑɑu. 
Kɑpi u n yɛ̃́ un míni kɛcɑ́ɑ́ sɔ́nɛ, ɑpi músu rɛ kɛ́pirii-pirii kɛnyinɛ yɛ kɛ, ɑpi pipupukɛɛ loni. 
Pinnɛ́í pɛɛ mɛyíkíyiki u yɛ̃ ɑi pi túítúísɛ. Yeesu uu pɛɛ rinɔ́ɔ pi mɑɑ rɛ: Iwɑmɛ íi kɑpɛ nɔ́ wɑ, nɛ̃́ ŋmɑɑ lo. 
Uu limɛmɑ́ɑ́ kúninɔi-i pi lɛɛpɔ, kuyɔ ɑku kéu. Ai pírí pi wɑi hɑ́i 
likumúŋɛ́ rɛ piɑkiŋ ntɑkɛ nnyɑ, ɑ́pi pɛɛ ɑkpɔ́nɔ́ mɛwɑisɑŋɑ mɛ̃ kõ. 
Mɛtɛ́ŋ́-mɑɑ́ ɑpi Kenesɑrɛti kɛtẽ tulu, ɑpi rikóitimɛ pikuninɔi kɑrii. 
Kɑpi kúninɔi-i n léemɛ, pisoi ɑpi mɛsɛ nɛ mɛsɛ Yeesu ceru. 
Api kɛtẽ kɛ̃ nnɛ́í kɔ́ɔ́nú, ɑpí yɛ pin n kõ tɛ u yei we, ɑpi kei nnɛ́í nɛ ɑsɑ́ŋɑ́ɑ́pɔrɛ́ kɛ́mɛɛ pitóikɔ́ u n sɔ́nɑɑpɔ. 
Pi yɛ kɛ́yɑɑ́lɑ-i pitóikɔ́ nɑ́ŋɑisɛlɛ siyupi nɛ ɑyu nɛ ɑwosoo nyɛ-i kuu n sɔ́nɛ ɑpí nɛ u n téni rɛ ukɛ́ tíyɛsɛ pitóikɔ́ pikɛ́ ḿpɑ́ ukɛtukɑnkɑ nnɔɔ ricɑ. U yɛ pɛ̃ nnɛ́í pɛɛ u rińcɑ pɔisɛntɛlɛ.
