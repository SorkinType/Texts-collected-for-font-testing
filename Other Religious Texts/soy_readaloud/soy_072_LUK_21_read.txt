Luki.
21.
Yeesu uu inípɛɛ síkɑ, uú yɛ́nu pin kei kɑpi yɛ ihɛɛ Uléécɑɑ kɛyɔ-i n wɑ piihɛɛ sɑriipɔ. 
Uu kɔ ukúmɑnnɔsi wɛkɔɔ́ unyinɛ yɛnu un sítɑnkɑ́ sitɛ́ sɑ́pɔlɛ̃. 
Uu pɛɛ́ mɑɑ rɛ: Asei kɑm nɔ́ símisi, unɔ́si nkó yɛ hɛlɛ ɑi pisoi picɔ mpí tɔ́su. 
Likumúŋɛ́ rɛ lɛ̃ kɑ́pi nɛ líkɑ n wɑi kɛ mpí nnɛ́í ɑpi kei risɑ́pɔ, ɑmɑ́ ulikɔ́ kɛcirɛ, lɛ̃ kuu m mɑ́ nnɛ́í ukɛ́ nɛ ilukɛ n lɔ kɛ uyɛ̃ uu wɑipɔ. 
Pisoi pinyinɛ ɑpi Uléécɑɑ kɛyɔ nsímɛ́ n símisi pin tee rɛ kɛ nyɑmlɛ hɑ́i, nɛ kɛɑpɑrɛ kɛcirɛ nɛ ilũ kɑpi Uléécɑɑ n hɛkɛsi. Amɑ́ Yeesu uú mɑɑ rɛ: 
Siyɑ́ɑ yɛ sɔ́ntilɛ kɛ ɑpɑrɛ nyɛɛ nní ń tɔ́siinɛlɛ̃ tikɑ ɑ́ri m mɛ́suninɛ. Lɛ̃ kɑni nní n nyɑ́ni yɛ sɔ́ntilɛ likɛ́ fori tɛ́kɛ́-tɛ́kɛ́. 
Api pɛɛ u pisɛ rɛ: Sɑ́ɑ, píyei kɛ lɛ̃ nnɛ́í ɑí hɑ sɔ́nti? Yo yɛɛ yɛ́ kumúŋɛ́ kpɛ-i kɛ lɛ̃ nnɛ́í ɑí hɑ n sɔ́nti rɔ́ nyísɛ? 
Yeesu uu rinɔ́ɔ yɔ́su rɛ: Ani nɔ́mɛnɛ́círɛ tĩ ńsɔnɛ, ɑ́ni kɑpɛ tíyɛ unyinɛ ukɛ́ kuyúi nɔ́ tɑ̃. Likumúŋɛ́ rɛ pisoi mɛyɑ̃́ yɛ́ sɔ́nɑɑmɛ nɛ rinɛ́nyíri pin tee rɛ: “Nɛ́ɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ”. Likumúŋɛ́ yɛ kɔ nyɑhɑimɛlɛ, ɑmɑ́ ɑ́ni kɑpɛ pi ritiki. 
Nɔn píyei ɑtɔpi nɛ pitele nnɛ̃́ n kõ, nɔ́sinɛ́wuu ɑ́si kɑpɛ tɔ. Li pisɛ rɛ lɛ̃ nnɛ́í likɛ́ kɑm kɛlenɛ, ɑmɑ́ kɛtẽ kɛtɔ ɑ́kɛ kɑhɑnɛ n tu. 
Uu pɛɛ lɛ̃ mɛmɑ́ɑ́ kpɑ́ rɛ: Nsoipuri nnyinɛ yɛ́ yisi nn nsoipuri ncɔ nɛ tɔpu, kɛyɔ́ɔpitẽ ɑkɛ kɛyɔ́ɔpitẽ kɛcɔ nɛ tɔpu. 
Kɛtẽ yɛ́ yéŋesi ɑ́i nyɑḿ, itói yɛɛ n tɛŋni nɛ nkṹ nn silõ silõ leeri, mɛwɑi mɛwɑi mɛɛ iwɑmɛ ń we yɛ́ kɔ wɑpisi, nɛ linyísɛ-nyísɛ lɛɛ kɛyómɛcɑɑ-pɔ n léeri yɛ́ kɔ wɑpisi. 
Amɑ́ likɛ́ kɛlenɛ lɛ̃ n wɑ, pi yɛ́ nɔ́ cɔ́pii ɑpi íwɛ nɔ́ wɑi piyómɛyɑ́hɑɑ nnyɑ, ɑpi ɑyómɛyɑ́hɑɑlee-i nɛ piyɔ́ɔpi nɛ kuyu piwɛ́ɛ́sɛ kɛ́mɛɛ nɛ nɔ́ túhɑɑnɛ ɑpi nɛ̃́ nnyɑ kukpɑniilee nɔ́ wɑi. 
Lɛlɛɛ yɛ́ kɛpolɛ nɔ́ hɛ nɔkɛ́ nɛ ɑnɛ́seérɑ n lesɛ. 
Lɛ̃ nnyɑ, ɑni m múílɛ̃ ńsɔnɛ. Áni kɑpɛ mmɛ̃ kɑni yɛ́ ḿ mɑlɛ nɔkɛ́ nɛ nsímɛ́ kɛ́mɛɛ ń le kɛcɑ́ɑ́ m músu. 
Nɛ́ rinɛ́cúruu ɑnɔ́ɔ nyɛ̃ kɑi m pisɛ rɛ nɔkɛ́ mɑlɛ nɛ mɛwɛ́ɛ́sɛsɔhɔ nɔ́ hɛ, ɑi tíyɛsɛ nɔ́unɛ́lɑ́ɑrɔ úkɑ úu yɛ́ fe ukɛ́ nɔ́nnɛ́kɔ́ rikósɛ. 
Nɔ́pinɛ́sɑ́ɑ nɛ nɔ́pinɛ́ni nɛ nɔ́pinɛ́wɑ̃́ nɛ nɔ́pinɛ́mɑ́ɑ nɛ nɔ́pinɛ́mɑ́rɛcɔ nɛ nɔ́pinɛ́sɑ́nɛ yɛ́ nɔ́pinɛ́lɑ́ɑrɔ ɑnipɛ-i nɔ́ wɑ ɑpi nɔ́mɛnɛ́yɑ̃́ kóni. 
Tinɛ́nyíri nnyɑ, ḿpɑ́ úye yɛ nɔ́inɛ́puri yulunɛ. 
Amɑ́ ḿpɑ́ nɔ́nnɛ́nyúpi nsɛ ńn pɔlunɛ. 
Ani n ŋmɔ́ɔ́púlɛ̃, lɛlɛɛ yɛ́ tíyɛsɛ ɑní nɛ nɔ́nnɛ́fɑ́ɑ lɔlu. 
Nɔn hɑ píyei n yɛ̃ ɑtɔpilee ɑn Yerusɑlɛm kɛkɑ́ípi wɑilɛ̃, ɑni ceri rɛ umɛforɛ yɛ nnyɑ́ɑ̃ wɑlɛ. 
Pɛɛ́ uyɛ-i, pɛpɛɛ Yutee-i ń we yɛ́ hɑ wúrumpɔ, ɑpi ɑkúú-mɛ̃ tɑɑ́, pɛpɛɛ Yerusɑlɛm kuyu kɛcɔpɛ ń we, ɑpi kɛtɑɑ sĩ, pɛpɛɛ sicɑrɛ-mɛ̃ ń we ɑ́pi fenɛ pikɛ́ kuyu-mɛ̃ kuimɛ. 
Likumúŋɛ́ rɛ siyɑ́ɑ sɛ̃ kɛ́mɛɛ kɛ Uléécɑɑ uu sisoipipi mɛwɑi kópɛ si cɛɛinɛ. Sikɛmɛɛ́ kɛ lɛ̃ nnɛ́í lɛɛ ɑ́tɛlɛ́-i n wɔ́lɑɑlɛ̃ ɑi wɑinɛ. 
Íwɛ kɛ pifɑni nɛ pimɑ́rɛni ɑpí hɑ siyɑ́ɑ sɛ-i topori. Mpɔ́ɔcɑɑi píimɑ yɛ hɑ kuyu kpɛ-i wenɛ, Uléécɑɑ yɛ́ kuyu kpɛ̃ pikɔ́ ukuwɔi nyísɛ. 
Pi yɛ́ nɛ ɑtɛ́hɛ cɑɑcɑɑ pi kṍ, ɑpi sitẽ nnɛ́í kɛ́mɛɛ ilɑ́si pi cɔ́pii, piyómɛyɑ́hɑɑcirɛ́ ɑpi Yerusɑlɛm cɔpisi hɑ́i nɛ siyɑ́ɑ sɛ̃ kɛtɔ-pɔ. 
Li yɛ́ ituŋɛ nɛ iwɑ́rɛ nɛ ɑwɑ́rɛpi-i linyinɛ nyísɛ. Mpɔ́ɔcɑɑi yɛ́ kɛtẽ kɛcɑ́ɑ́ pisoi nnɛ́í lõ, mínimɑɑ iwɔɔ píimɑ ii nfɑsimɛ́ pi tɑni. 
Pisoi yɛ́ nɛ iwɑmɛ kpí pin lɛlɛɛ kɛtẽ kɛcɑ́ɑ́ n sɔ́nti nfɑsimɛ́ wɑi, likumúŋɛ́ rɛ li yɛ́ kɛyómɛcɑɑ-pɔ linɑŋɛ-nɑŋɛ lémesi. 
Pi yɛ́ pɛɛ Usoi Kɛpipi yɛ̃ kɛn kuhopɛ kɛcɑ́ɑ́ wemɛ nɛ kɛnnɑŋɛ nɛ kɛmɛyɔɔpi. 
Mɛwɑi mɛ̃ mɛn hɑ piwɑi n korɑɑnɛnɛ, ɑni kɛcírɛ nyɛrɛ ɑni ɑ́yu síkɑ, likumúŋɛ́ rɛ nɔ́kɛnɛ́ŋmɛ́hɛrɛɛ yɛ lɛ̃ tulɛ. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ kɛnyɑ́rũ nkɛ́ pi wɑi rɛ: Ani kúfikiyee nkú nɛ ɑléécɔ nnɛ́í ripɑí. 
Nɔn píyei n yɛ̃ ɑpɛ́i fɑlɛ ɑn n torɑɑlɛ̃, nɔ yɛ nɔ́rinɛ́cúruu cerilɛ rɛ kupii yɛ nyɑhɑilɛ. 
Lɛ̃ cirɛ yɛ ni, nɔn hɑ píyei n yɛ̃ mɛwɑi mɛ̃ mɛn n wɑpisi, ɑni ceri rɛ Uléécɑɑ iyɔɔpi yɛ nyɑhɑilɛ. 
Asei kɑm símisi: Pisoi kunɑi nkú ɑ́ku tɔ́sunɛ kɛlenɛ nní nnɛ́í likɛ́ nɛ ń kɑm. 
Kɛtẽ nɛ kɛyómɛ yɛ́ tɔsí, ɑmɑ́ ɑnɛ́nɔ́ɔ ɑ́ɑ mɛtɔ́ŋɛ́ mɑ́. 
Ani nɔ́mɛnɛ́círɛ tĩ, ɑ́ni kɑpɛ tíyɛsɛ nɔ́nnɛ́fɑsimɛ́ nkɛ́ mɛyɑ́nsei ɑnyɑ̃́ nɛ pítɑnirɛ́-mɛ̃ pɛlɛ. Áni kɑpɛ kɔ mɛyɑ́nsei kɛtẽ kɛcɑ́ɑ́ likɔ́ nfɑsimɛ́ n wɑi. Insɑ́ lɛ̃, kɛtúhɑɑnɛyɑɑ yɛ́ nɔ́ rikɑikɑi 
yɑrɛ ripinɛ. Likumúŋɛ́ rɛ kɛtẽ nté pisoi nnɛ́í kɛ kɛyɑ́ɑ kɛ̃ ɑkɛ lɛɛrinɛ. 
Áni kɑpɛ lɔpilɛ, ɑni yɛ ḿpɑ́ píyei Uléécɑɑ we nɔkɛ́ nɛ ńnɑŋɛ n yɛ̃́, ɑní nɛ lɛ̃ nnɛ́í lɛɛ n wɑpisinɛ kɛ́mɛɛ ŋmɛ́hɛrɛɛ ɑni kɔ hɑ nɛ Usoi Kɛpipi kɛyu-i nyɛrɛ. 
Kɛtúŋɛ́ kɛ Yeesu uu yɛ pɛɛ Uléécɑɑ kɛyɔ-i céesi, ɑmɑ́ lin kɛsinɛ n wɑ, uú hɑ kɛkúúpi kɑpi yɛ n sée rɛ Olifiyee rikúú kɛcɑ́ɑ́ n we. 
Kuyu pikɔ́ nnɛ́í ɑpi yɛ mmɑrɛ mɑrɛ Uléécɑɑ kɛyɔ-i weri pikɛ́ kutu u ricɔ.
