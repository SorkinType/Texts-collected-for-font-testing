Luki.
17.
Yeesu uu upipirɛtiki mɑɑ rɛ: Mɛwɑi mɛɛ yɛ́ n tíyɛsɛ pisoi pikɛ́ nɛ ɑkópɛ n wɑ yɛ́ mɛsɛ́rɛ n welɛ. Amɑ́ úye un n tíyɛsɛ unyinɛ uú nɛ ɑkópɛ kɛ́mɛɛ loni, íwɛ kuu topori. 
Li kutɔsi we pikɛ́ rinɑ́ipɑrɛ mulɛ-mulɛ ńŋmɛ wɑ ɑpi ripɔ́ɔ u lɛsi ɑpi mínimɑɑ-i u sɑ́pɔ nɛ kuu yɛ́ siwɑ̃́ sɛ̃ kɛnyinɛ ɑkópɛ n wɑisɛ. 
Ani ńsɔnɛ nɔ́mɛnɛ́círɛ tĩ! Upɔ́mɑ́rɛcɔ un nɛ pɔ́ m púnnɛ, ɑ ńsɔnɛ nɛ u símisi. Un n ceri rɛ u cɑɑilɛ, ɑ uɑkópɛ u sɑ́rɛi. 
Ḿpɑ́ un nɛ pɔ́ m púnnɛntɛ mɛpehẽ mɛsɛɛi kɛyɑ́ɑ kɛsɛ kɛ́mɛɛ, uu yɛ ḿpɑ́ mɛpúnnɛ mɛ́ye nɛ pɔ́ téni rɛ u cɑɑilɛ, ɑ uɑkópɛ u sɑ́rɛi. 
Pitumɛ ɑpi Upíimɑ mɑɑ rɛ: A nfɑtɛnɛ tɔ́ rikpɑ́sɛ. 
Upíimɑ uu rinɔ́ɔ yɔ́su rɛ: Nɔn pɛɛ nfɑtɛnɛ m mɑ́ nkɛ́ m piyɛ yɑrɛ riléépipi síńsɑ́pi rinyinɛ, nɔ́ pɛɛ fe ɑni kuléé nkú mɑɑ rɛ: “A wɛhɛ ɑɑ́ hɑ mínimɑɑ-i nyɛrɛ” ɑku nɔ́rinɛ́nɔ́ɔ tiki. 
Tɔkɛ́ wɑisɛ rɛ nɔ́ unyinɛ yɛ ukɛikɔ́ mɑ́ un u cɑ́kɑripɔ nɛ́ɛ un uisɛɛ sémpɔ. Un píyei u n yɛmmɛ un kɛcɑrɛ n leeri, liute yɛ́ ukɛikɔ́ uyɛ̃ mɑɑ rɛ ɑ mɛkɛɛkɛɛ kɑm ɑ ilukɛ lii? 
Aɑí, ɑ́i yɛ́ lɛ̃ wɑ. Amɑ́ u mɛníŋɛ u mɑɑnɛ rɛ: A ilukɛ nɛ́ wɑ ɑɑ ilũ consɛ ɑɑ nɛ́ i yekei kɛ́ li, pɔ́ pɛɛ lɛ̃ mɛmɑ́ɑ́ fe ɑɑ ipɔ́kɔ́ le. 
U yɛ́ fe uu uukɛikɔ́ yɑ́ɑ́si rɛ nɛ upikɛi kuú nɛ pikɛi pɛ̃ kɑí pɛɛ m pisɛ rɛ ukɛ́ wɑ n wɑɑ? 
Lɛ̃ cirɛ yɛ ni nɛ nɔ̃́ ticuruu. Nɔn píyei lɛ̃ kɑi m pisɛ rɛ nɔkɛ́ wɑ n wɑ, ɑni yɛ mɑɑ rɛ: Tɔ̃́ nɛ pikɛikɔ́ lɛ, ɑ́i pisɛ rɛ pikɛ́ rɔ́ pɔɔnɛsɛ. Lɛ̃ kɑi m pisɛ rɛ tɔkɛ́ wɑ kɑri wɑ. 
Kɛ Yeesu uu Yerusɑlɛm-mɛ̃ n sĩ́, uu Sɑmɑrii nɛ Kɑlilee ɑyu kɛpɑ́ɑ́pɑ́ɑ́ n tikilɛ̃. 
Uu kuyu kunyinɛ-i lompɔ, piyɔ̃́ kɛfi pinyinɛ ɑpi u tɛ́pɛipɔ ɑpi kɛtɑɑ-pɔ nyɛrɛ 
ɑpí m pupukɛɛ rɛ: Yeesu, Sɑ́ɑ, ɑ íwɛ rɔ́ tẽ́! 
Yeesu uu pi yɛnu uu rɛ: Ani hɑ nɔ́mɛnɛ́círɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ nyísɛ. Kɑpi kei-pɔ n sĩ́, ɑpi pelu. 
Kɛ piusɛ uu n céri rɛ u peilɛ, uu uɑnɑ kɛpirɛ pɛlɛ un Uléécɑɑ mɛyɔɔpi yom lin cɑ́lɛ̃. 
Uu Yeesu ɑnɑ mɛtene wulɑpɔ uu kɛ́yu kɛtẽ kɑm, uu u yɑ́ɑ́si rɛ nɛ upikɛi. Utisi uyɛɛ pɛɛ Sɑmɑrii ukɔ́ lɛ. 
Yeesu uu pi mɑɑ rɛ: Nɛ́ntɛ pikɛfi kɑm pɔisɛ? Yei yɛ pɛɛ usɛ úu we kɛfi tɔrɔɔ mpí? 
Piukɑ úu nfɑsimɛ́ wɑ rɛ ukɛ́ pɛɛmɛ ukɛ́ Uléécɑɑ ríyu wɑisɛ rɛ nɛ upikɛi, insɑ́ usɑ́nɛ nkó mɛcirɛ. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ u mɑɑ rɛ ɑ yisi, ɑ n ŋme, mpɔ́fɑtɛnɛ yɛ pɔ́ yóriyɛlɛ. 
Pifɑrisi ɑpi Yeesu pisɛ rɛ: Píyei kɛ Uléécɑɑ iyɔɔpi ii sɔ́nti? Uu rinɔ́ɔ pi yɔ́su rɛ: Uléécɑɑ iyɔɔpi íi we rɛ nɔ́ fe ɑni imɛkɑmɛ yɛnu. 
Ái rɛ pi yɛ́ nɔ́ mɑɑ rɛ: Ani ripɑí, nté kií we nɛ́ɛ nté-pɔ le. Ani ceri rɛ Uléécɑɑ iyɔɔpi yɛ nɔ́kɛnɛ́cɔpɛ welɛ. 
Uu pɛɛ pipirɛtiki mɑɑ rɛ: Kɛyɑ́ɑ yɛ sɔ́ntilɛ kɑni yɛ́ ń lɑ rɛ nɔkɛ́ ḿpɑ́ kɛyɑ́ɑ kɛsɛ Usoi Kɛpipi yɛ̃, ɑmɑ́ ɑ́ni yɛ́ kɔ kɛ yɛ̃. 
Pi yɛ́ nɔ́ mɑɑ rɛ: Nté nní kuú we nɛ́ɛ nté-pɔ kuú we, ɑmɑ́ ɑ́ni kɑpɛ kei hɑ, ɑ́ni kɑpɛ kɔ mɛ̃ wurupɔ. 
Yɑrɛ kɛ kɛkónɛsɛípi ɑkɛ yɛ nní mɛyéne mɛsɛ riń-yɛɛ́ri kɛyómɛ kɛtɔ nɛ kɛ́tɔ, ɑi kpɑ̃́ii mɛcɔ kɛ Usoi Kɛpipi ɑkɛ sɔ́nti. 
Amɑ́ li pisɛ rɛ kɛkɛ́ mɛyɑ̃́ íwɛ li, nfɑ́ɑni pisoi mpí pikɛ́ kɛ ŋmɛriyɛ. 
Lɛlɛɛ pɛɛ Nowee kumúŋɛ́ n wɑ licɔ lɛɛ Usoi Kɛpipi kumúŋɛ́ wɑinɛ. 
Pisoi yɛ pɛɛ lelɛ pin níru pin sɔ́ɔnnɛ pin pilɑ nɛ pinɔ́si pɑsiinɛ hɑ́i nɛ kɛyɑ́ɑ kɛ̃ kɛ Nowee uu kutɑ́kɑi kɛ́mɛɛ n lompɔ, ikónɛ mɑɑ ií nɛ weri, míni ɑmɛ pisoi kpísi. 
Li yɛ́ hɑ n we yɑrɛ Lɔti kumúŋɛ́: Pisoi yɛ pɛɛ lelɛ pin níru, pin lɛ́ɛ́ri pin kɔ yɑ́lisi, pin ɑléé tɑ́mɑɑnkɛɛ pin kɔ mɔ́mti. 
Amɑ́ kɛyɑ́ɑ kɛ̃ kɛ Lɔti uu Sotɔm-i ń le kɛ nnɑ nɛ linyinɛ yɑrɛ nfúrɔ̃ toi ɑi kɛyómɛcɑɑ-pɔ leemɛ ɑi pisoi kóni. 
Mɛsɛ mɛ̃ kɑí hɑ kɛyɑ́ɑ kɛ̃ kɛ Usoi Kɛpipi ɑkɛ́ hɑ n léerinɛ pikɛ́ kɛ yɛ̃ wɑinɛ. 
Úye un kɛyɑ́ɑ kɛ̃ ukulee ipirɑɑ kɛcɑ́ɑ́ n we, úu kɑpɛ rɛ ukɛ́ mɛtene ricɛpi ukɛ́ uilũ iyɛɛ kei ń we kpísi nɛ́ɛ, úye un sicɑrɛ-mɛ̃ n we, úu kɑpɛ rɛ ukɛ́ kɛ́yɔ pɛlɛ. 
Ani Lɔti unɔsi nkɔ́ kɛcɑ́ɑ́ léisɛ. 
Úye un n lɑ ukɛ́ nfɑ́ɑ mɛsɛ́rɛ m mɑ́, u yɛ́ n fóm. Amɑ́ úye un unfɑɑ n fóm, u yɛ́ n yɛ̃. 
Nɛ nɔ́ símisi rɛ kɛsinɛ kɛ̃, pisoi pitɛ́ yɛ́ kuhɑ̃́ kusɛ kɛcɑ́ɑ́ n finu, ɑí usɛ kpíípɔ ɑi ucɔ yɑ́. 
Pinɔ́si pitɛ́ yɛ́ n we pin nɑm ɑi usɛ kpíípɔ ɑi ucɔ yɑ́. 
Pitisi pitɛ́ yɛ́ likumúŋɛ́ kpɛ-i kɛcɑrɛ-i n we, ɑi usɛ kpíípɔ ɑi ucɔ yɑ́. 
Pipirɛtiki ɑpi u pisɛ rɛ: Upíimɑ, yei kɑí hɑ lɛ̃ wɑinɛ mɑ́? Uu rinɔ́ɔ yɔ́su rɛ: Kei kɛ isɑrɛ kópɛ ii ń we kɛ ɑ́yemeekú ɑɑ yɛ cɑ́pinɛ.
