Mɑtiyee.
27.
Kɑi kóso n weesi mmɑrɛ mɑrɛ, pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nnɛ́í nɛ Pisuifi piwɛ́ɛ́sɛ ɑpi rinɔ́ɔ risɛ wɑi rɛ pikɛ́ Yeesu kpu. 
Kei kɑpi pɛɛ u pɑɑsi ɑpí nɛ Rɔm uyɔɔpi Pilɑti lɛɛpɔ. 
Yutɑsi yɛɛ Yeesu n kpísi uu pilɑ́ɑrɔ ɑnipɛ-i wɑi uu yɛ́nu rɛ pi lɑ pikɛ́ Yeesu kpu. Ai u cɔ́ŋ, uu ukɛfɑ-i téni uu nwóóweni mɛpipi ɑfɛɛtɑɑni mɛ̃ kpísi uú nɛ pɛlɛ uú hɑ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ pɛ̃ pɑ. 
Uu pi mɑɑ rɛ: Nɛ ɑkópɛ wɑlɛ. Nɛ usoi yɛɛ úu líkɑ n cɑɑi yɑ́ɑ́ ɑḿ pɑ ɑpi kópu. Pɛ̃ ɑpi u pɛsɛ rɛ: Mpɔ́kɔ́ yɛ mmɛ̃! Ái nɛ tɔ̃́ yɛ̃! 
Uu pɛɛ siwóó sɛ̃ kpísi uu Uléécɑɑ kɛyɔ-i sɑ́, uu sĩ́ uú hɑ umɛcirɛ ńŋmɛ wɑi. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpi siwóó sɛ̃ híni ɑpi rɛ: Kɑi nní mɛ́nyɛ kɛwóó nnyɑ, ncée ńn irɔ́sé kɛ́mɛɛ we rɛ pikɛ́ Uléécɑɑ kɛyɔ kuwóótɑkɑi kɛ́mɛɛ kɛ wɑipɔ. 
Api kómɛinɛ ɑpi pɛɛ kɛ kpísi ɑpí nɛ utúhumɔmɛ kɛ́mɛɛ kɛtẽ lɔlu rɛ pikɛ́ yɛ kei pisɑ́nɛ kulɛsi. 
Lɛ̃ nnyɑ kɑí nɛ kei n kpíimɛ nɛ nɛni-mɛ̃, pi kɛtẽ kɛ̃ sélei rɛ: mɛ́nyɛ kɛcɑrɛ. 
Lɛ̃ kɛ ɑntepu Yeremii uú pɛɛ Yeesu kɛcɑ́ɑ́ ḿ mɑ uu Uléécɑɑ nsímɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lu kɑi wɑ rɛ: Pi nwóóweni mɛpipi ɑfɛɛtɑɑni mɛ̃ kpísilɛ. Mɛ̃ kɛ Isirɑyɛɛli pikɔ́ ɑpí musí rɛ pi yɛ́ ukɛcɑ́ɑ́ hɛ́ɛ́lɛ. 
Apí nɛ utúhumɔmɛ kɛ́mɛɛ kɛtẽ lɔlu. Lɛ̃ kɛ Upíimɑ uu rɛ kɛ́ pi símisi. 
Kɑpi uyɔɔpi Pilɑti kɛ́mɛɛ nɛ Yeesu pitúhɑɑnɛ ń hɑ, uyɔ́ɔpi uyɛ̃ uu u pisɛ rɛ: Pɔ́ɔ Pisuifi uyɔɔpii? Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Mmɛ̃ kɑɑ nní mɑ. 
Amɑ́ kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ ɑpí nɛ nsímɛ́ u rinkpɑ́rɑ́, úu ríkɑ pi pɛsɛ. 
Kei kɛ Pilɑti uu pɛɛ u pisɛ rɛ: Áɑ mmɛ̃ kɑpi nní nɛ pɔ́ n kpɑrɑi kómɛi, nɛ́ɛ? 
Uu kɔ lɛ̃ n sɛ́ɛ́lɛ̃ úu ríkɑ pɛsɛ. Ai pɛɛ Pilɑti pírí wɑi. 
Ḿpɑ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ɑ́ye, Pilɑti yɛ̀ɛ̀ ukpɑniikɔ́ uyɛ̃ kɛ risoiwuí ɑri ń lɑ ukɛ́ lesɛ ɑkpɑnii kɛ́mɛɛ lésɛlɛ. 
Utisi unyinɛ kɑpi ḿpɑ́ yei n nyu yɛ pɛɛ ɑnyɑ̃́ kumúŋɛ́ kpɛ-i ɑkpɑnii kɛ́mɛɛ welɛ. Pi yɛ u sée rɛ Yeesu Pɑrɑpɑsi. 
Pilɑti uu pɛɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ pisɛ rɛ: Yeesu Pɑrɑpɑsi nɛ Yeesu kɑpi n sélei rɛ Kirisi kɛcɔpɛ, úye kɑní lɑ kɛ́ ɑkpɑnii kɛ́mɛɛ lesɛ? 
U pɛɛ nyulɛ ŋmɑɑ ńsɔnɛ rɛ nfɑpɔlɛ nnyɑ kɑpi Yeesu uɑnipɛ-i wɑ. 
Pilɑti un kei lɛ̃ kɛtúhɑɑnɛ kɛ-i we, likɛ́ nɛ rinsɑ́sɛ, uunɔsi uu pisoi tumpɔ rɛ pikɛ́ u mɑɑ rɛ: Usoi nkó úu ncɑɑi ńkɑ wɑ. A unsímɛ́ kɛ́mɛɛ ɑpɔ́nípɛ mɑhɑ̃. Nɛ kɛsinɛ nkɛ́ uyɛ̃ nnyɑ kɛlɑ́rɛ́ kópɛ kɛnyinɛ kɛ́mɛɛ íwɛ lilɛ hɑ́i. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ pin likumúŋɛ́ kpɛ-i pisoi pitɑ́lɑɑnkɛɛ mɑsí rɛ pikɛ́ Pilɑti símisi rɛ ukɛ́ Pɑrɑpɑsi ɑkpɑnii kɛ́mɛɛ lesɛ uu Yeesu pɔ̃́ kopu. 
Uyɔ́ɔpi uu pipisɛ pi kpɑ́ rɛ: Pikɛtɛ́ kɛcɔpɛ, úye kɑní lɑ kɛ́ lesɛ? Api u pɛsɛ rɛ Pɑrɑpɑsi lo. 
Kɛ́ íye Yeesu kɑpi n sélei rɛ Kirisi pɔ̃́ wɑ? Api pinnɛ́í u pɛsɛ rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii! 
Kei kuu kɔ pipisɛ pi rikpɑ́ rɛ: Ncɑɑi ń-ye kuu mpíí wɑ? Api pɛɛ sicɑ́ɑ́ilɑ nɑ́kíísɛ ŋmɑɑ rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii! 
Kɛ Pilɑti uu n yɛ̃́ tɛ ḿpɑ́ un nɛ n símisi, ɑ́i nkpɑ́ni líkɑ yoriyɛnɛ, iwɔ́ɔ in mɛníŋɛ mɛyɑ̃́ cɑ́lɛ̃, uu míni kpísi uu pisoi nnɛ́í inipɛɛ-i ɑnípɛ nɑlɛ uu rɛ: Ái nɛ́ɛ ncée hɛ rɛ pikɛ́ usoi nkó yɛɛ úu ncɑɑi ńkɑ n wɑ kpu. Nɔ́nnɛ́kɔ́ yɛ mmɛ̃! 
Pisoi pɛɛ n cɑ́pinɛlɛ̃ nnɛ́í ɑpi u pɛsɛ rɛ: Ɛɛɛ! Tɔ́ nɛ sirɔ́wɑ̃́ tɔkɛ́ unkpɔ ncɔni n te! 
Kei kɛ Pilɑti uu pɛɛ ɑkpɑnii kɛ́mɛɛ Pɑrɑpɑsi pi lesɛ, uu tíyɛsɛ ɑpi Yeesu isɛ́í súúkɛɛ, uu pi u pɑ rɛ pikɛ́ hɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii. 
Uyɔ́ɔpi pisɔ́ɔ́cɑ ɑpi Yeesu kpísi ɑpí nɛ kei kɑpi yɛ n túhɑɑnɛ lompɔ, pisɔ́ɔ́cɑ nnɛ́í ɑpi kei u kɑ́lɑɑpɔ. 
Api ilũ u mɑhɑni ɑpi kukpɛ́lɛnku wɛɛ́ u tɑni. 
Api pɛɛ sinípɛleépi rikɔŋ pɑrii ɑpi u kipisɛ, ɑpi kɔ kɛnɑɑpi kunípɛ lukɛ-lukɛ-mɛ̃ u múísɛ. Api pɛɛ ukɛyu-i wúlɑ ɑpi sɛ́nnyi ɑpi rɛ: Tɔ pɔ́ yɑ́ɑsi, Pisuifi uyɔɔpi! 
Api pɛɛ mɛtɑ u n túkɛntɛ. Api yɛ kɔ kɛnɑɑpi kɛ̃ kpísi ɑpí nɛ ríyu kɛcɑ́ɑ́ u pepu. 
Kɑpi nnyei u m mɑsí, ɑpi kukpɛ́lɛnku u mɑhɑni ɑpi uilũ pitɑnɛ u pɛsɛ ɑpí nɛ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ pikɑrii u tɔ́mpɔ. 
Kɑpi kuyu-i n léeri pin nɛ Yeesu pikɑrii tɔ́su, ɑpí nɛ Sirɛɛni kɛtẽ utisi unyinɛ sɑ́nɛ. Pi yɛ u sée rɛ Simɔɔ. Api ńnɑŋɛ nɛ Yeesu kunɑpɛ́ɛ́kɔɔ́ u cɔnsɛ. 
Api pɛɛ́ hɑ nɛ kɛlõ kɛnyinɛ tulu. Pi yɛ kei sée rɛ Kɔlikɔtɑ. Liɑsei rɛ: Kúyu Koŋoo kɛlõ. 
Api pítɑ nɛ pitɛ tulɛ-tulɛ pinyinɛ cɔ́hɔnɛ ɑpi u pɑ rɛ ukɛ́ n ntí. Uu lélé uu yúlu rɛ úu nírunɛ. 
Api u kpísi ɑpí hɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii, ɑpi pɑŋɑi ɑpí nɛ pimɛcɔpɛcirɛ uilũ hɔɔnɛntɛ. ... 
Api pɛɛ kei n tũ pin u mɛ̃́. 
Api kɛkpénkpélépi kɛnyinɛ kɛcɑ́ɑ́ wɔ́lu lɛ̃ nnyɑ kɑpi u ń kpu ɑpi kunɑpɛ́ɛ́kɔɔ́-i uriyu kɛcɑ́ɑ́ kɑrii. Pi wɔi rɛ: Nkó yɛɛ Yeesu, Pisuifi uyɔɔpi. 
Api pɛɛ kɔ limɛmɑ́ɑ́ pisoi kpɑ́ɑ́ree pitɛ́ pinyinɛ kɛsẽ́ kɑrii, ucɔ ukulukɛ-lukɛ-mɛ̃ ucɔ ukumii-mɛ̃ ɑpí nɛ u kɔ. 
Picéetɔŋɛ́ ɑpi u lɑ́mɑɑnkɛɛ ɑpi ríyu m pɔuli. 
Api yɛ u mɑɑ rɛ: Pɔɔ́ yɛ Uléécɑɑ kɛyɔ n forii ɑɑ siyɑ́ɑ sitɑɑni nɛ pimɔ́mɛ kɛ pɛsɛ, pɔn mɛsei Uléécɑɑ Kɛpipi, ɑ nkpéni ripɔ́yu lɔ ɑɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ súiri! 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ nɛ Pisuifi piwɛ́ɛ́sɛ pɔ́ɔ kɔ limɛcɔ kɛcɑ́ɑ́ u nyɑ́nɛi ɑpi yɛ nɛ mɑɑ rɛ: 
U picɔ yoriyɛntɛlɛ ɑmɑ́ úu pɛɛ fe ukɛ́ uricuruu yoriyɛ! Nɛ́ntɛ Isirɑyɛɛli Uyɔɔpi lo! Ukɛ́ kɔ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ súimɛ tɔkɛ́ pɛɛ́ nɛ kɛfɑ u tɛnɛ. 
Uléécɑɑ kuú nɛ ritɑ́ uu yɛ nɛ mɑɑ rɛ: Nɛ Uléécɑɑ Kɛpipi lɛ. Lɛ̃ nnyɑ, in tɛ Uléécɑɑ yɛ ukɛcɑ́ɑ́ kpɑ́lɛ̃, ukɛ́ nkpéni u yoriyɛ tɔkɛ́ yɛ̃́ nkɛ̃́! 
Hɑ́i ɑí nɛ pisoi kpɑ́ɑ́ree pɛ̃ kɛtɛ́ kɑpi kɛsẽ́ n kɑrii pɛ́nɛ ɑpi kɔ lɛ̃ u lɑ́misi. 
Ituŋɛ kɛcírɛ kumúŋɛ́ hɑ́i nɛ kɛtúŋɛ́ mɛtɑɑni kumúŋɛ́ kɛ kuŋmɑhɑ ɑku kɛtẽ nnɛ́í kɛcɑ́ɑ́ cɔ̃. 
Kɑi kɛtúŋɛ́ mɛtɑɑni kumúŋɛ́ ń tu, Yeesu uu kɛcɑ́ɑ́ilɑ píimɑ kɛnyinɛ súúni rɛ Elii, Elii lɑmɑ sɑpɑtɑni? Lɛlɛɛ rɛ: Unɛ́léécɑɑ, Unɛ́léécɑɑ, yo nnyɑ kɑɑ nɛ́ ŋmɛriyɛ? 
Pisoi pɛɛ kei n cɑ́pinɛlɛ̃ kɛcɔpɛ, pinyinɛ ɑpi lɛ̃ kom ɑpi rɛ: Elii kuu nní sélei. 
Mɛsɛ nɛ mɛsɛ, pikɛcɔpɛ unyinɛ uu wuru uú hɑ sinɛ́ɛ̃ kpísi uu pítɑ nyɛpɛ-nyɛpɛ pinyinɛ fɛ uú nɛ kɛnɑɑpi kɛnyinɛ kɛcɑ́ɑ́ tuhu uu u pɑ rɛ ukɛ́ n ntí. 
Picɔ ɑpi pɛɛ rɛ: Ani riyɑ́ tɔkɛ́ yɛ̃́ tɛ Elii yɛ́ kɑm uu u yoriyɛ nɛ́ɛ. 
Yeesu uu mɛyɑ̃́ pipupɛi kpɑ́ uu pɛɛ kpi. 
Likumúŋɛ́ kpɛ-i kɛ kusɑ́ŋɑ́ɑ́ kɑpí nɛ Uléécɑɑ kɛyɔ kɛ́mɛɛ n kénɛlɛ̃ ɑku ricɛɛ́rinɛ mɛtɛ́ hɑ́i kɛcɑ́ɑ́ nɛ mɛtene-pɔ. Kɛtẽ ɑkɛ yéŋesi, ɑpɑrɛcɑŋɑɑ ɑɑ forɛsi, 
ɑkpíí mɛyɑ̃ ɑɑ wúkulɛntɛ, Uléécɑɑ pisoi kulúi pɛɛ n kpí ɑpi yisikɛɛ. 
Api ɑkpíí kɛ́mɛɛ léenti. Kɛ Yeesu uu ń kɑ uu nkpɔ kɛ́mɛɛ yisi, ɑpi Uléécɑɑ kuyu lompɔ, pisoi mɛyɑ̃ ɑpi pi yɛnu. 
Kɛ Rɔm usɔ́ɔ́cɑ kpéẽ nɛ pisɔ́ɔ́cɑ picɔ pɛɛ kei ń we pin kɛsẽ́ Yeesu mɛ̃́ ɑpi n yɛ̃́ lɛ̃ kɛ kɛtẽ ɑkɛ n yéŋesi nɛ licɔ licɔ lɛɛ n wɑpisi, iwɑmɛ ii mɛyíkíyiki ŋmɑɑ pi loni ɑpi rɛ: Usoi nkó yɛ pɛɛ mɛsei Uléécɑɑ Kɛpipi lɛ. 
Pinɔ́si yɛ pɛɛ kulúi welɛ, pin kɛtɑɑ-pɔ nyɛnu pin nyɑ́nɛi. Pimɛyɑ̃ pɛɛ hɑ́i Kɑlilee-pɔ u ritikimɛ pin u lɛ́ni. 
Pikɛcɔpɛ pinyinɛ pɛɛ nní Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri nɛ Yɑkupu nɛ Yosɛfi uni Mɑɑri nɛ Sepetee piŋmɑ́nɛ uni. 
Kɑi nnyɔ́ɔ́ n wɑ, utisi mɑ́mɑ́ uyɛ̃ kɑpi yɛ n sée rɛ Yosɛfi un Arimɑtee kuyu ukɔ́, uú hɑpɔ. Uyɛ̃ nɛ kɔ pɛɛ Yeesu upirɛtiki lɛ. 
Uu Pilɑti kɛ́mɛɛ sĩ́ uú hɑ u pisɛ rɛ ukɛ́ Yeesu u pɑ. Kei kɛ Pilɑti uu ŋmurɛi uu ncée hɛ rɛ pikɛ́ hɑ Yeesu u pɑ. 
Yosɛfi uu Yeesu kpísi uu kɛsɑ́ŋɑ́ɑ́pi tomɛ́ kɛcirɛ-i wɑi 
uú hɑ nɛ nhórɛ fɑlɛ kuu kupɑrɛpoo-i n wɑ kɛ́mɛɛ u finsɛ, uu pɛɛ mɛpɑrɛ píimɑ mɛnyinɛ pímiisɛ uu rinɔnɔɔ fi, uu pɛɛ yɑ́ uu tɔ́mpɔ. 
Likumúŋɛ́ kpɛ-i, Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri nɛ Mɑɑri ucɔ uyɛ̃ pɔ́ɔn kei rikpíí kɛyúrí tṹ. 
Kɛ kɛyɑ́ɑ kɛɛ yɛ nɛ kɛwéntɛyɑɑ n tisɛ ɑkɛ n tu, pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi ɑpi yisi ɑpi Pilɑti kɛyɔ sĩ, 
ɑpi u mɑɑ rɛ: Urɔ́sɑ́ɑ, tɔ léisɛ rɛ kɛ unɔ́ɔmɛ nkó uú pɛɛ nfɑ́ɑ m mɑ́, u pɛɛ́ mɑɑ rɛ u yɛ́ unkpɔ mɛmɑ́ɑ́ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ yisi. 
Lɛ̃ nnyɑ, ɑ ncée hɛ rɛ pisoi pikɛ́ hɑ siyɑ́ɑ sitɑɑni urikpíí ńsɔnɛ m mɛ̃́, upipirɛtiki ɑ́pi kɑpɛ nɛ u n yɑ́ɑi pikɛ́ lesɛ, pikɛ́ pɛɛ́ nɛ pisoi n kirɑɑsɛ rɛ: u nkpɔ kɛ́mɛɛ yisilɛ nnyɑ. Lin lɛ̃ n wɑ, nnɔ́ɔmɛ mmɛ̃ nɛ́ pɛɛ cɑɑi ɑi nfoí feriyɛ. 
Pilɑti uu pi pɛsɛ rɛ: Pimɛ́rɛ́ yɛ mpí! Ani nɛ pi hɑ pikɛ́ hɑ rikpíí m mɛ̃́ yɑrɛ kɑni ń lɑ. 
Kei kɑpi yisi ɑpí hɑ músu, ɑpi yɛ́nu lɛ̃ kɑpi yɛ́ pimɛ́rɛ́kɛi pɛ̃ n wɑ. Api mɛpɑrɛ mɛɛ rinɔnɔɔ n fílɛ̃ ńsɔnɛ kɑrii ɑpi pɛɛ pimɛ́rɛ́ tónsɛntɛ rɛ pikɛ́ m mɛ̃́.
