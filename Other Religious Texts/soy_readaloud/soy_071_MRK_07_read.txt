Mɑriki.
7.
*Pifɑrisi nɛ *isé picélɑɑ pinyinɛ pɛɛ *Yerusɑlɛm n léeri ɑpi Yeesu kɛwúípi wɑi. 
Api upipirɛtiki pinyinɛ yɛnu pin ilukɛ le nɛ ɑnípɛ kpɛrinkpɛkɔɔ́, ɑ́pi piɑnipɛ nɑlɛ yɑrɛ kɑi piinyɛkii kɛ́mɛɛ m pisɛ. 
Kɛ Pifɑrisi nɛ *Pisuifi picɔ nnɛ́í ɑpi mɛyɑ́nsei pikpurɛ inyɛkii m múílɛ̃ nnyɑ, pinsɑ́ ńsɔnɛ n nɑlɛ, ɑ́pi yɛ li. 
Pin kɔ kuyɑ́ɑ n leemɛ, pi yɛ piinyɛkii mɛnɑlɛ mɛ̃ nɑlɛlɛ ɑpi kɛlenɛ le. Pi nɛ pimɛkpurɛwɑi mɛyɑ̃ yɑrɛ nní piheehɛɛrɛ nɛ sicɑ́ripi nɛ ɑpɔ́ri pihɛɛrɛ múílɛnlɛ. 
Pifɑrisi nɛ isé picélɑɑ ɑpi pɛɛ u pisɛ rɛ: Yo nnyɑ kɛ pipɔ́pirɛtiki ɑ́pi pirɔ́kpurɛ inyɛkii tikilɛ̃ pin nɛ ɑnípɛ kpɛrinkpɛkɔɔ́ ilukɛ le? 
Yeesu uu rinɔ́ɔ pi pɛsɛ rɛ: Asei kɛ *ɑntepu *Esɑyi uu mɛkɛɛ-mɛ nɔ́nnɛ́símɛ́ yɛ́mmɛ. Nɔ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ lɛ yɑrɛ kɛ Esɑyi uu n wɔ́i rɛ Uléécɑɑ yɛ mɑɑ rɛ: Nnɔ́ɔ-i ŋmɑnɛ kɛ pisoi mpí ɑpi ríyu nɛ́ wɑisɛlɛ̃, ɑmɑ́ ɑ́pi pɛɛ pisifɑ-i nɛ nɛ́ kɔ́lɛ̃. 
Piiyɑ́hɑɑ yɛ likpɑ́rɑ́kɔɔ́ lɛ. Pi rinyɑlɛ pin sisoipipi inyɛkii kpísi pin céési yɑrɛ inɛ́kɔ́ yɛ nyi. 
Uléécɑɑ isé kɑni kɛpirɛ ritɔ ɑni sisoipipi inyɛkii m múílɛ̃. 
Uú nɔŋ́ tɛ: Ani riŋmɔ́ɔ́pú ɑni Uléécɑɑ isé kɛpirɛ tɔ ɑní nɛ nɔ́inɛ́nyɛ́kii picélɑɑ kɛ́yu mɑhɑni. 
Ticuruu *Moisi yɛ mɛkɛɛ-mɛ mɑɑ rɛ: A upɔ́sɑ́ɑ nɛ upɔ́ni ríyu wɑisɛ. Uu kɔ rɛ úye un usɑ́ɑ nɛ́ɛ úni ɑnɔ́ɔ n wɑ, li pisɛ rɛ pikɛ́ liute kpu. 
Amɑ́ nɔ̃́ nɛ yɛ rɛ: Usoi un usɑ́ɑ nɛ́ɛ úni m mɑɑ rɛ: Lɛ̃ kuú pɛɛ́ ń lɑ ukɛ́ nɛ u lɛ̃́ Uléécɑɑ hɛlɛ, 
ɑ́ni yɛ liute riyɑ́ ukɛ́ linyinɛ licɔ piwɑi usɑ́ɑ nɛ́ɛ úni rikpɑ́. 
Mɛ́wee mɛ̃ yɛɛ tíyɛsɛlɛ ɑni Uléécɑɑ Nsímɛ́ cɔ́kɔi nɛ nɔ́inɛ́nyɛ́kii isé kɑni n yekeilɛ̃. Mɛwɑi mɛ̃ mɛcɔ kɑni n wɑpisi yɛ kɔ piyɛlɛ. 
Yeesu uu limɛmɑ́ɑ́ pisoi picɑ́pinɛ kpɑ́ uu rɛ: Ani nɔ́nnɛ́nɛ́í kutu nɛ́ ricɔ ɑni kóm. 
Litɑhɑikɔ́ líkɑ ɑ́i we lɛɛ yɛ́ lin usoi kɛ́mɛɛ n lõ, ɑi Uléécɑɑ inipɛɛ-i mɛ́kpɛrinkpɛ u wɑisɛ, ɑmɑ́ lɛlɛɛ usoi kɛ́mɛɛ n léenti lɛɛ yɛ Uléécɑɑ inipɛɛ-i mɛ́kpɛrinkpɛ u wɑisɛ.  
Úye un ukómɛ ukɛ́ kṍ. 
Kɛ Yeesu uu pisoi pɛ̃ kɛcɔpɛ n yisi, u nɛ upirɛtiki ɑpi kɛ́yɔ loni ɑpi kɛnyɑ́rũ kɛ̃ ɑsei u pisɛ. 
Uu rinɔ́ɔ pi pɛsɛ rɛ: Nɔ̃́ ticuruu ɑ́ni kɔ mɛsɔhɔ mɑ́ɑ? Lɛ̃ nnyɑ, ɑ́ni nyu rɛ lɛ̃ nnɛ́í lɛɛ usoi kɛ́mɛɛ n léeripɔ ɑ́i yɛ́ fe likɛ́ Uléécɑɑ inipɛɛ-i mɛ́kpɛrinkpɛ u wɑisɛɛ? 
Likumúŋɛ́ rɛ ɑ́i urikiŋ kɛ́mɛɛ kɑi léeripɔ, ɑmɑ́ kuloi-i le, uu kɔ pɛɛ́ cukú. Yeesu mɛmɑɑ mɛɛ nyísɛlɛ̃ tɛ ilukɛ nnɛ́í yɛ kulukɛ welɛ. 
Uu pimɑ́ɑ pi kpɑ́ rɛ lɛlɛɛ usoi kɛ́mɛɛ n léenti lɛɛ yɛ mɛ́kpɛrinkpɛ u wɑisɛ. 
Liriyíkí rɛ usoi kɛfɑ-i kɛ simúŋɛ́ kópɛ nfɑsimɛ́ nɛ iwɑ́sɑ́ nɛ piyɑɑ nɛ pisoikõ 
nɛ ucɔ unɔsi nɛ́ɛ ucɔ ulɑ kuwɑ́ncɔ piwɑi nɛ mɛmɑ́ nnɑ́ɑ́pí nɛ rikpɑ́kɑ́rɑ́ nɛ piyɑɑlukɛ nɛ nturi nɛ inípɛɛ cɑɑ-cɑɑ nɛ picɔ ɑlɛ̃ nɛ rikɔ́kɔri nɛ píyuwɛ́lɑɑ ɑi léenti. 
Nní nnɛ́í, usoi kɛfɑ-i kɛ nní nnɛ́í ɑi léenti, lɛlɛɛ yɛ kɔ mɛ́kpɛrinkpɛ u wɑisɛ. 
Kɛ Yeesu uu kei n yisi, uu Tiiri kuyu kuwɛ́lɛ́-mɛ̃ sĩ. Uu kɛ́yɔ kɛnyinɛ-i tonɛ. Úu lɑ unyinɛ ukɛ́ u yɛ̃. Amɑ́ ḿpɑ́ nɛ lɛ̃, úu fe ukɛ́ umɛcirɛ pesí. 
Unɔ́si unyinɛ kɛwɑ̃ kɛɛ kuníri n héesi uu kóm pin Yeesu nsímɛ́ wɑi. Uu pɛɛ́ hɑ Yeesu kɛyu-i wúlɑ. 
Unɔ́si uyɛɛ usɑ́nɛ lɛ, úu *Usuifi. Sirii kɛteni-i kɑpi u mɑri. U Fenisii ukɔ́ lɛ. Uú nɛ Yeesu téni rɛ ukɛ́ uukpére kuníri lɑkɑsɛ. 
Yeesu uu pɛɛ u mɑɑ rɛ: Ái nyɑḿ tɛ pikɛ́ siwɑ̃́ ilukɛ kpísi pikɛ́ sipɔpipi risɑ́. 
Unɔ́si uu rinɔ́ɔ u pɛsɛ rɛ: Mɛsei yɛ mɛ Sɑ́ɑ, ɑmɑ́ siwɑ̃́ sin pilukɛ m mɑsí, sipɔpipi yɛɛ lɛ̃ kɛ siwɑ̃́ ɑsi kɛteni-i m póroisɛ hínɛilɛ ɑsi le. 
Yeesu uu pɛɛ u mɑɑ rɛ: Tipɔ́nɔ́ɔ ntí nnyɑ, ɑ n ŋme, kuníri yɛ upɔ́kpére kɛ́mɛɛ léepɔlɛ. 
Kuu kɛ́yɔ n kúi, uu kɛwɑ̃́ lɛɛpɔ kɛn nɛ kɛripɔɔ finu, kuníri kun tɔ́mpɔ. 
Yeesu uu limɛmɑ́ɑ́ Tiiri kɛtẽ yisi uu Sitɔɔ kuyu cɔpu, uu kuwɛ́lɛ́ kpɛ̃ kɑpi yɛ n sée rɛ Ayu Kɛfi kɛtẽ-mɛ̃ wɑlɑ́ uu Kɑlilee kupiyɛ-mɛ̃ pɛlɛ. 
Api pɛɛ́ nɛ utusuí yɛɛ úu nsímɛ́ n fe ńsɔnɛ Yeesu weri. Apí nɛ u téni rɛ ukɛ́ ɑnípɛ ukɛcɑ́ɑ́ rilɑɑ. 
Yeesu uú nɛ iyɑɑ u lelu, uú nɛ pisoi kɛtɑɑ wɑi uu usinipɛpi ɑtu-i u tɑmpɔ uu rilémpi kɛcɑ́ɑ́ mɛtɑ u tɛŋ́. 
Uu lɛ̃ mɛmɑ́ɑ́ nfɑ́ɑ tɔ́ɔ́ri uu rɛ: Efɑɑtɑ, liɑsei rɛ: A tɑ́pilɛ. 
Mɛsɛ nɛ mɛsɛ uɑtu ɑɑ tɑ́pilɛ, tilémpi ɑri fénnɛ, uu nsímɛ́ loni ńsɔnɛ. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ rinɔ́ɔ pi hɛ rɛ ɑ́pi kɑpɛ tíyɛsɛ unyinɛ ukɛ́ kṍ. Amɑ́ ḿpɑ́ Yeesu mɛmɑlɛ mɛ́ye kɛ pɛ̃ ɑpi nsímɛ́ piyóó ŋmɔ́ɔ́púlɛ̃. 
Pírí píimɑ kɑi mɛyɑ́nsei pi wɑ ɑpí m mɑ́ikɛɛ rɛ: Lɛ̃ nnɛ́í kuu n wɑ, ɑsɑɑ lo, ḿpɑ́ pitusuí, u yɛ pi komɛisɛlɛ, uu pɛ̃ ɑlempi nyɛɛ m pɑhɑɑlɛ̃ símɑɑsɛ.
