Mɑriki.
13.
Kɛ Yeesu uu Uléécɑɑ kɛyɔ-i n léeri, upirɛtiki usɛ uu u mɑɑ rɛ: Sɑ́ɑ, ɑ ripɑí, kɛ́yɔ kɛcirɛ nɛ ɑpɑrɛ sɔnɛ-sɔnɛ, nɛ́ɛ yo? 
Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Pɔ kɛ́yɔ píimɑ nkɛ́ nyɑ́nii? Li sɔ́nti likɛ́ wɑ ripɑrɛ ríkɑ ɑ́ri yɛ́ ricɔ kɛcɑ́ɑ́ n tɔsilɛ̃. Linnɛ́í mmú yɛ sɔ́ntilɛ likɛ́ fori tɛ́kɛ́-tɛ́kɛ́. 
Uu Olifiyee rikúú kɛcɑ́ɑ́ tonɛ Uléécɑɑ kɛyɔ kɛyu-mɛ̃. Piyɛɛ nɛ Yɑkupu nɛ Yohɑni nɛ Antiree mɛcirɛ nɛ uyɛ̃ pin we, ɑpi u pisɛ rɛ: 
A rɔ́ símisi kumúŋɛ́ kpɛ-i kɛ lɛ̃ ɑi ń werinɛ. Yo yɛɛ rɔ́ nyísɛnɛ rɛ lɛ̃ nnɛ́í yɛ́ tɛnɛ? 
Yeesu uu pɛɛ pi mɑɑ rɛ: Ani nɔ́mɛnɛ́círɛ tĩ ńsɔnɛ, ɑ́ni kɑpɛ tíyɛ unyinɛ ukɛ́ kuyúi nɔ́ tɑ̃. 
Pisoi mɛyɑ̃́ yɛ́ sɔ́nɑɑmɛ usɛ usɛ nɛ rinɛ́nyíri, ḿpɑ́ úye uu yɛ mɑɑ rɛ nɛ̃́ kɛ Uléécɑɑ uu wɛ́ɛ. Lɛ̃ kɛ́mɛɛ kɑpi yɛ́ pisoi kulúi kuyúi tɑ̃. 
Nɔn píyei n kõ tɛ ɑtɔpi yɛ nnyɑ́ɑ̃ nɛ kɛtɑɑ-pɔ wemɛ, ɑ́ni kɑpɛ nfɑsimɛ́ tónɛ, likumúŋɛ́ rɛ lɛlɛɛ sɔ́nti. Amɑ́ kɛ́tɔ ɑ́kɛ kɑhɑnɛ n tu. 
Nsoipuri nnyinɛ yɛ́ yisi nn nsoipuri ncɔ nɛ tɔpu, kɛyɔ́ɔpitẽ ɑkɛ kɛyɔ́ɔpitẽ kɛcɔ nɛ tɔpu, kɛtẽ yɛ́ɛ́ silõ silõ yeŋesi, nkṹ yɛ́ kɔ wɑ. Limpɔlɛ yɛ́ n we yɑrɛ kɛ mɛmɑ́rɛnlõ ɑmɛ yɛ unɔ́si pikɛ́hɛ̃ n kórɑɑnɛ. 
Ani nɔ́mɛnɛ́círɛ tĩ. Pi yɛ́ nɛ sitúhɑɑnɛ nɔ́ hɑ, ɑpí nɛ sipɔ́pí nɔ́ kɑ́ii ɑyómɛyɑ́hɑɑlee-i. Pi yɛ́ nɛ̃́ nnyɑ kuyu piwɛ́ɛ́sɛ kɛ́mɛɛ nɛ pitúhɑɑnɛ kɛ́mɛɛ nɔ́ sɔ́nɛ. Nɔ́ kɔ pikɛyu-i inɛ́tɑnsei li. 
Li pisɛ rɛ pikɛ́ Nsímɛ́ Kɛcirɛ isoipuri nnɛ́í riyóó kɛlenɛ. 
Pin nɔ́ n tĩ ɑpí nɛ kɛtúhɑɑnɛ nɔ́ sĩ́, ɑ́ni kɑpɛ pírí tónɛ kɛlenɛ nɛ mmɛ̃ kɑni yɛ́ m mɑɑ, ɑmɑ́ ɑni mmɛ mɛɛ kumúŋɛ́ kpɛ-i nɔ́ n kɑ mɑɑ, likumúŋɛ́ rɛ ɑ́i nɔ́ɔ símisi, ɑmɑ́ Nfɑ́ɑsɔnɛ yɛ mu. 
Usoi yɛ́ umɑ́ɑ nɛ́ɛ uwɑ̃́ tĩ uu tíyɛsɛ ɑpi u kopu. Pisɑ́ɑ yɛ́ kɔ limɛcɔ pisipipi wɑ. Siwɑ̃́ yɛ́ pisɑ́ɑ nɛ píni kɛcɑ́ɑ́ nyɛrɛ ɑsi tíyɛsɛ ɑpi pi kóni. 
Tinɛ́nyíri nnyɑ, ḿpɑ́ úye yɛ nɔ́inɛ́puri yulunɛ, ɑmɑ́ úye un tinkɑ́hɑ́-kɑ́hɑ́ hɑ́i nɛ kɛ́tɔ-pɔ, Uléécɑɑ yɛ́ u yoriyɛ. 
Kumúŋɛ́ kunyinɛ yɛ́ kɑm ɑni ncɑɑi píimɑ mɛɛ yɛ n cɔ́kɔi yɛnu, kei kɑ́i nɛ n sɑ́ rɛ nkɛ́ n we. Úye un nsímɛ́ mmɛ̃ n kɛɛ̃, ukɛ́ musí ukɛ́ liɑsei kõ. Lin lɛ̃ n wɑ, pɛpɛɛ Yutee kɛtẽ kɛ́mɛɛ ń we pikɛ́ wuri ɑpi ɑyɔ́pɛ tɑɑ́. 
Úye un kɛ́yɔ ipirɑɑ kɛcɑ́ɑ́ n we, úu kɑpɛ kɛ́yɔ-i ricɛpimɛ rɛ ukɛ́ linyinɛ kpísi. 
Úye un kɔ kɛcɑrɛ-i n we, úu kɑpɛ kɛpirɛ pɛlɛ rɛ ukɛ́ hɑ ukɛtukɑnkɑ kpísi. 
Íwɛ kɛ pifɑni nɛ pimɑ́rɛni ɑpí hɑ siyɑ́ɑ sɛ-i topori. 
Ani kɛyómɛ yɑ́ɑ́si, ɑ́i kɑpɛ nnyiyɛ kumúŋɛ́ kɑ. 
Likumúŋɛ́ rɛ siyɑ́ɑ sɛɛ íwɛ píimɑ sikɔ́ lɛ. Kɛ Uléécɑɑ uú nɛ kɛtẽ n wɑ, ɑ́pi íwɛ iyɛ̃ icɔ yɛnɑɑlɛ̃, iicɔ íi kɔ píkɑi léeri. 
Uléécɑɑ unsɑ́ pɛɛ siyɑ́ɑ sɛ̃ n hɑsi, úkɑ úu yɛ́ pɛɛ nfɑ́ɑ yɛ̃. Pɛ̃ kuu n tɛ́si uú n te nnyɑ kuu si hɑsi. 
Úye un pɛɛ nɔ́ m mɑɑ rɛ: Ukirisi kɛ Uléécɑɑ uu n wɛ́ɛ yɛ nté nɛ́ɛ nté-pɔ welɛ, ɑ́ni kɑpɛ wɑisɛ. 
Likumúŋɛ́ rɛ pikirisi nɔɔmɛ nɛ ɑntepuyɛ nɔɔmɛ yɛ leerinɛ. Pi yɛ́ɛ́ mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ wɑpisi pikɛ́ nɛ ḿpɑ́ pɛ̃ kɛ Uléécɑɑ uú n te pɔisɛ. 
Áni kɑpɛ lɔpilɛ, ḿpɑ́ yo kɑm nní mɛncirɛ nɔ́ símisi. 
Amɑ́ siyɑ́ɑ sɛ̃ kɛ́mɛɛ, íwɛ píimɑ iyɛ̃ mɛmɑ́ɑ́ kɛ ituŋɛ ií sɔmɑɑnɛ, iwɑ́rɛ ii mɛtɛ́í pɑɑ. 
Awɑ́rɛpi ɑɑ kɛyómɛcɑɑ-pɔ poroiri, linɑ́ŋɛ-nɑŋɛ lɛɛ kɛyómɛcɑɑ-pɔ ń we ɑi mɛyɑ̃́ yeŋesi. 
Pi yɛ́ pɛɛ Usoi Kɛpipi yɛ̃ kɛn ɑhopɛ kɛcɑ́ɑ́ wemɛ nɛ kɛnnɑŋɛ nɛ kɛmɛyɔɔpi. 
U yɛ́ pɛɛ upitumɛ pɛɛ kɛtẽ nɛ kɛyómɛ-pɔ ń we ɑwɛ́lɛ́ nnɛ́í túmɛi ɑpi pɛ̃ kuu n wɛ́ɛkɛɛ kɛtẽ nkɛ́ nɛ kɛkɛtɔ-pɔ cɑ́pinɛ. 
Ani kúfikiyee kɛnyɑ́rũ icélɑɑ kutu ricɔ. Ku yɛ kun ilésɛ ŋmṹ n wɑiri, mɛpɛ́i mɛn n leeri, nɔ yɛ mɛsɛ nɛ mɛsɛ ceri rɛ kupii yɛ nyɑhɑimɛlɛ. 
Limɛcɔ kɑí we, nɔn píyei likɛi nní mɛwɑi n yɛ̃, ɑni ceri rɛ Usoi Kɛpipi yɛ nyɑhɑimɛlɛ un tinɔnɔɔ nɛ kɔlɛ̃. 
Asei kɑm nɔ́ símisi: Pisoi kunɑi nkú ɑ́ku tɔ́sunɛ kɛlenɛ nní nnɛ́í likɛ́ nɛ ń kɑm. 
Kɛtẽ nɛ kɛyómɛ yɛ́ tɔsí, ɑmɑ́ ɑnɛ́nɔ́ɔ ɑ́ɑ mɛtɔ́ŋɛ́ mɑ́. 
Amɑ́ in kɛtẽ mɛtɛnɛ nkɔ́, úkɑ úu likɛyɑɑ kɛ̃ nɛ́ɛ liituŋɛ iyɛ̃ nyu, ḿpɑ́ piléécɑɑtumɛ pɛɛ kɛyómɛcɑɑ-pɔ ń we nɛ́ɛ Uléécɑɑ Kɛpipi ricuruu. Amɑ́ Usɑ́ɑ mɛcirɛ yɛɛ nyu. 
Ani nɔ́mɛnɛ́círɛ tĩ, ɑ́ni kɑpɛ ripɔ́ɔ fóm, (ɑni yɛ kɛyómɛ yɑ́ɑ́si), likumúŋɛ́ rɛ ɑ́ni nyu píyei kɛ liituŋɛ ii yɛ́ kɑm. 
Li yɛ́ n we yɑrɛ usoi yɛɛ ncée ń hɑ uu ukɛyɔ tíyɛ, uu upikɛikɔ́ ńnɑŋɛ hɛ, uu ḿpɑ́ úye pikɛi u múísɛ, uu rinɔnɔɔ umɛ́rɛ́ rinɔ́ɔ hɛ rɛ úu kɑpɛ ripɔ́ɔ fóm. 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ ripɔ́ɔ fóm, likumúŋɛ́ rɛ ɑ́ni nyu píyei kɛ úyɔsɑɑ uu pɛɛri, nnyɔ́ɔ́ nɛ́ɛ kɛsinɛ pele nɛ́ɛ mɛcɑ́nkooi nɛ́ɛ mmɑrɛ. 
Ái kɑpɛ wɑ rɛ u mɛkɑ́ikɑi kɑ uu nɔ́ lɛɛri nnɔ̃ kɛ́mɛɛ. 
Lɛ̃ kɑm nní nɔ́ n símisi, ḿpɑ́ úye kɑḿ nɛ mɑ́nɛ, ɑ́ni kɑpɛ ripɔ́ɔ fóm.
