Uléécɑɑ Pitumɛ Mɛwɑi.
7.
Uléécɑɑ usinɑ uu pɛɛ Etiyɛɛni pisɛ rɛ: Lɛɛ ni mɛseii? 
Etiyɛɛni uu rɛ: Nɔ̃́ pinɛ́mɑ́rɛcɔ nɛ pinɛ́sɑ́ɑ, ɑni kutu ricɔ ńsɔnɛ ɑni kóm! Uléécɑɑ, mɛyɔ́ɔpi ute yɛ urɔ́sɑ́ɑ Apirɑhɑm tikpɑ́fúmɛlɛ pɛɛ́ uyɛ-i kuú pɛɛ Mesopotɑmii-pɔ ń we, uu kɛlenɛ Hɑrɑɑni-pɔ finɛ. 
Uu pɛɛ u mɑɑ rɛ: A kɛpɔ́tẽ nɛ kupɔ́mɑ́rɛ riyɑ́ ɑɑ kɛtẽ kɛ̃ kɑm pɔ́ n nyísɛnɛ kɛ́mɛɛ sĩ́. 
Apirɑhɑm uu pɛɛ Kɑlitee kɛtẽ yisi uú hɑ Hɑrɑɑni-pɔ finɛ. Usɑ́ɑ mɛkpɔ-mɑɑ́, Uléécɑɑ uu kei u kpísi uú nɛ kɛtẽ nkɛ́-i kɑni nɛni nní ń we weri. 
Uléécɑɑ úu kei kɛtẽ u hɛ. Ḿpɑ́ kɛ́nɑcɔpɛ ritísí, úu u hɛ rɛ ukɛ́ kei ripɛ́ɛ́lú. Amɑ́ uu pɛɛ rinɔ́ɔ yekei rɛ u yɛ́ kɛnnɛ́í u nɛ mɑsí upipirɛ̃ pɑ ɑpí n te. Úu kɔ pɛɛ kumúŋɛ́ kpɛ-i kɛwɑ̃́ mɑ́. 
Lɛ̃ kɛ Uléécɑɑ uu u ḿ mɑ yɛ nní: Kucɔ̃yu-pɔ kɛ pipɔ́pírɛ̃ ɑpí mɑsí finɛnɛ, ɑpi ilɑ́si pi tini, ɑpi íwɛ pi wɑi ɑŋmɛ̃ pílɛ pílɛ mɛnɑ. 
Amɑ́ nɛ́ nɛ mpuri mmɛɛ ilɑ́si pi n tininɛ túhɑɑnɛ ɑpi pɛɛ limɛmɑ́ɑ́ léeri ɑpí weri ɑpi nté kɛlõ nkɛ́-i nɛ́ yɑ́ɑ́si. 
Kei kɛ Uléécɑɑ uu pɛɛ u nɛ Apirɑhɑm kɛcɔpɛ rinɔ́ɔ yekei rɛ ukɛ́ riké. Lɛ̃ nnyɑ kɑpi Isɑɑki mɑri Apirɑhɑm uu pɛɛ kɛyɑ́ɑ pɑ́hɑ-pɑhɑ tuŋɛ́ u ké. Isɑɑki pɔ́ɔ kɔ uuŋmɑ́nɛ Yɑkupu ké. Yɑkupu pɔ́ɔ kɔ limɛcɔ upiŋmɑ́nɛ kɛfi nɛ pitɛ́ pɛ̃ kérii. Pɛpɛɛ nní pirɔ́kpurɛ kɛfi nɛ pitɛ́ pɛ̃. 
Pikpurɛ pɛ̃ ɑpi pikɛcɔpɛ usɛ kɑpi yɛ n sée rɛ Yosɛfi nɛ kɛfɑ tóósi ɑpi u yɑ́i, uú hɑ Esipiti-pɔ ilɑ́si le. Amɑ́ Uléécɑɑ yɛ pɛɛ Yosɛfi kɛkúrí welɛ, uu ncɔ́ŋ nnɛ́í kɛ́mɛɛ u lesɛ 
uú nɛ kɛfɑ u fénnɛ uu mɛwɛ́ɛ́sɛsɔhɔ u hɛ. Esipiti uyɔɔpi uu Esipiti kɛtẽ kɛcɑ́ɑ́ iyɔ́ɔpi u tonsɛ uu kɔ ukɛyɔ likɔ́ nnɛ́í kpísi uu uɑnipɛ-i wɑi. 
Nkṹ nnyinɛ nn Esipiti kɛtẽ nnɛ́í nɛ Kɑnɑɑ kɛtẽ loni. Pirɔ́sɑ́ɑyɑhɑ ɑpi íwɛ píimɑ le, ɑ́pi yɛ yɛ̃ pikɛ́ li. 
Yɑkupu uu kóm tɛ mɛlukɛpipi yɛ Esipiti kɛtẽ-pɔ we, uu pirɔ́sɑ́ɑyɑhɑ tum mɛfoí. 
Kɑpi mɛlírũ n hɑ́pɔ, Yosɛfi uu tíyɛsɛ upimɑrɛcɔ ɑpi u ceru. Esipiti uyɔɔpi pɔ́ɔ kɔ Yosɛfi riyíkí céru. 
Yosɛfi uu pɛɛ tum ɑpi usɑ́ɑ Yɑkupu séipɔ nɛ ukumɑrɛ nnɛ́í pisoi kuwóó nɛ ɑfɛɛrɛ́ nɛ pinupũ. 
Yɑkupu uu yisi u nɛ pirɔ́sɑ́ɑyɑhɑ picɔ ɑpí hɑ Esipiti kɛteni-i finɛ ɑpí hɑ nɛ kei kpíni. 
Api pi cɔni ɑpí nɛ Sikɛm kuyu sĩ ɑpí hɑ kei rikpíí kɛ Apirɑhɑm uu Hɑmɔɔ piŋmɑ́nɛ-i nɛ usiwóó ń lɔ kɛ́mɛɛ pi kulɛsi. 
Kɛ rinɔ́ɔ kɛ Uléécɑɑ uú pɛɛ Apirɑhɑm n yekei ituŋɛ ii n nyɑhɑiri, ntɔ́puri nn pɛɛ mɛyɑ̃́ kulúi n wɑi Esipiti kɛtẽ kɛcɑ́ɑ́. 
Uyɔ́ɔpi fɑlɛ unyinɛ yɛɛ úu píkɑi Yosɛfi nkɔ́ ńkɑ n kómɑɑlɛ̃ uu Esipiti kɛteni-i iyɔ́ɔpi tonɛ. 
Uyɔ́ɔpi uyɛ̃ uu ntɔ́puri nɛ isɔhɔɔcee tóói uu pirɔ́sɑ́ɑyɑhɑ íwɛ lukɛisɛ, hɑ́i un nɑ́ɑ́si yɑrɛ pikɛ́ pisipipi riyɑ́ sikɛ́ kpí. 
Likumúŋɛ́ kpɛ-i kɑpi Moisi mɑri un Uléécɑɑ inipɛɛ-i nyɑḿ. Iwɑ́rɛ itɑɑni kɑpi usɑ́ɑ kɛyɔ-i u kpínii. 
Api limɛmɑ́ɑ́ u fómni, Esipiti uyɔɔpi ukpére uu u kpísi uu nyɔ́lu yɑrɛ uricuruu yɛɛ u mɑri. 
Esipiti pikɔ́ mɛnyuwɛ nnɛ́í kɑpi u céesi, unsímɛ́ nɛ umɛwɑi ɑmɛ tíyɛsɛ uu ńnɑŋɛ m mɑ́. 
Kuu iŋmɛ̃ ɑfɛɛnɑ n yɛ̃́, uu rɛ u yɛ́ hɑ upimɑrɛcɔ Isirɑyɛɛli pikɔ́ rilóólú. 
Kuu n hɑ́pɔ, uu Esipiti ukɔ́ unyinɛ lɛɛpɔ un Isirɑyɛɛli ukɔ́ unyinɛ kɑii. Uu unɑ́rɛi kɛcɑ́ɑ́ kpɑ́ uu Esipiti ukɔ́ uyɛ̃ kɑii uyɛ̃ uú hɑ nɛ kpi. 
Moisi yɛ pɛɛ músu rɛ upimɑrɛcɔ Isirɑyɛɛli pikɔ́ yɛ́ kṍ tɛ uyɛ̃ kɛ Uléécɑɑ uú lɑ ukɛ́ nɛ ritiki ukɛ́ nɛ ilɑ́si kɛ́mɛɛ pi lesɛ. Nɛ́ḿpɑ́kɑnɛ upimɑrɛcɔ ɑ́pi lɛ̃ kõ. 
Moisi uu kɔ kɛyɑ́ɑ lírũ tuŋɛ́ Isirɑyɛɛli pikɔ́ pitɛ́ lɛɛpɔ pin cɛ́ŋɛsi, uu pɛɛ́ hɑpɔ rɛ ukɛ́ pi yoriyɛ. Uu pi pisɛ rɛ: Nɔ pisoi pisɛ lɛ, pinɛ́sɑ́nɛ! Yo nnyɑ kɑni pɛɛ cɛ́ŋɛsi? 
Uyɛɛ pɛɛ ucɔ n íwɛ n wɑi uu pɑnsɛpɔ uu Moisi lɑ́upɔ uu u pisɛ rɛ: Wóo iyɔ́ɔpi pɔ́ tónsɛ rɛ ɑ irɔ́símɛ́ n túhɑɑnɛ? 
Pɔ lɑlɛ pɔkɛ́ nɛ́ kpu yɑrɛ kɑɑ sé Esipiti ukɔ́ ń kpu, nɛ́ɛ íye? 
Kɛ Moisi uu ɑnɔ́ɔ nyɛ̃ n kṍ, uu wuru uú hɑ Mɑtiyɑɑ kɛtẽ-pɔ tonɛ. Kei kuu piŋmɑ́nɛ pitɛ́ mɑ́ri. 
Iŋmɛ̃ ɑfɛɛnɑ mɛmɑ́ɑ́, Moisi uu pɛɛ kɛyɑ́ɑ kɛnyinɛ kucɛsi kóimɑ-i Sinɑyi riyɔpɛ kɛkúrí n we. Kei kɛ Uléécɑɑtumɛ unyinɛ uu kuhííhíí kunyinɛ kɛcɔpɛ inɑlempi kɛ́mɛɛ u rikpɑ́fúmɛ. 
Kɛ Moisi uu lɛ̃ n yɛ̃́, ɑi mɛnyinɛ u wɑi. Uu rɛ ukɛ́ rinyɔsɔ́pɔ ukɛ́ hɑ mɛwɑi mɛ̃ nyɑ́nɛi. Uu Uléécɑɑ rinɔɔ kom tɛ: 
Nɛ́ɛ pipɔ́sɑ́ɑyɑhɑ Uleecɑɑ, nɛ Apirɑhɑm nɛ Isɑɑki nɛ Yɑkupu Uleecɑɑ. Moisi uu pɛɛ iwɑmɛ nɛ n tɛrii úu kɑhɑ ukɛ́ riweríi. 
Upíimɑ uu u mɑɑ rɛ: A ɑpɔ́nɛ́ɛ́ri mɑhɑ̃, li we rɛ kɛlõ kɛ-i kɑɑ nní n nyɛnu yɛ Uléécɑɑ ripɔhɔ lɛ. 
Nɛ Esipiti kɛteni-i nnɛ́puri iwɛ yɛnlɛ, ɑm kɔ piiŋmɔpi kom. Lɛ̃ nnyɑ kɑḿ kɑ kɛ́ pi yoriyɛ. Lɛ̃ nnyɑ, ɑ nkpéni kɑm kɛ́ Esipiti kɛtẽ-pɔ pɔ́ tũ. 
Moisi uyɛ̃ kɛ Isirɑyɛɛli pikɔ́ ɑpí pɛɛ n kɛ́si, ɑpi u mɑɑ rɛ: Wóo iyɔ́ɔpi nɛ itúhɑɑnɛ pɔ́ tónsɛ? Uyɛ̃ cirɛ kɛ Uléécɑɑ uu tummɛ rɛ ukɛ́ piuyɔɔpi, ukɛ́ kɔ ilɑ́si kɛ́mɛɛ pi lesɛ. Uléécɑɑtumɛ uyɛɛ kuhííhíí kɛcɔpɛ inɑlempi kɛ́mɛɛ u rinkpɑ́fúmɛ yɛɛ u lɛ́ninɛ. 
Moisi yɛɛ Isirɑyɛɛli pikɔ́ Esipiti kɛteni-i lésɛmɛ. Iŋmɛ̃ ɑfɛɛnɑ kɛ́mɛɛ kuú nɛ Esipiti kɛteni-i nɛ Míni-Wɛɛ́-Kukoi kɛ́mɛɛ nɛ kucɛsi kóimɑ-i mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ mɛ̃ wɑpisi. 
Uyɛ̃ cirɛ yɛɛ pɛɛ Isirɑyɛɛli pikɔ́ símisi rɛ: Uléécɑɑ yɛ́ ɑntepu unyinɛ nɔ́ pɑ́mɛ yɑrɛ nɛ̃́ un nɔ́nnɛ́puri ukɔ́. 
Kɛ Isirɑyɛɛli pikɔ́ ɑpí pɛɛ kucɛsi kóimɑ-i n cɑ́pinɛ, uyɛɛ pɛɛ pirɔ́sɑ́ɑyɑhɑ nɛ Uléécɑɑtumɛ yɛɛ Sinɑyi riyɔpɛ kɛcɑ́ɑ́–pɔ u n símɑɑri kɛcɔpɛ we. Uléécɑɑ nsímɛ́ mɛɛ yɛ nfɑ́ɑ n hɛ kuú pɛɛ yɔ́su un tɔ́ pɑ́lɛ̃. 
Amɑ́ pirɔ́sɑ́ɑyɑhɑ ɑ́pi pɛɛ́ lɑ pikɛ́ u pɑkɑrɛ, ɑpi uipuri yulu ɑpí n kɑ́ípi pikɛ́ Esipiti pɛlɛ. 
Api pɛɛ Arɔɔ mɑɑ rɛ: A piléécɑɑ rɔ́ mɔ̃́ pɛɛ yɛ́ kɛrɔ́yu-i n sɔ́nɛ, li we rɛ ɑ́ri nyu lɛlɛɛ Moisi uyɛɛ nní Esipiti kɛtẽ-pɔ rɔ́ n lésɛmɛ n wɑ. 
Api kɛyɑ́ɑ kɛ̃ tinɑ́ɑpipi mɔm, ɑpi ri nyɔ́ɔnsɛ, ɑpi piɑnipɛ limɔmɛ-mɔmɛ lɛ̃ ɑnyɑ̃́ wɑi, piɑkiŋ ɑɑ niŋɛsi. 
Amɑ́ Uléécɑɑ uu kɛpirɛ pi tɔ uu pi yɑ́ ɑpi ituŋɛ nɛ iwɑ́rɛ nɛ ɑwɑ́rɛpi n yɑ́ɑ́si yɑrɛ kɑi ɑntepuyɛ ɑtɛlɛ́-i n wɔ́lɑɑlɛ̃. Li kei wɔlɑɑlɛ̃ tɛ: Nɔ̃́ Isirɑyɛɛli pikɔ́, nɛ̃́ kɑni iŋmɛ̃ ɑfɛɛnɑ kɛ́mɛɛ kucɛsi kóimɑ-i isɛɛ nɛ nyɔ́ɔnsɛntɛ ɑni kɔ licɔ licɔ hɛkɛsii? 
Amɑ́ piléé kɑpi yɛ n sée rɛ Molɔki kucɑ́ŋíí nɛ nɔ́pinɛ́léé Refɑɑ riwɑrɛpi kɑní pɛɛ topori. Lɛ̃ kɑní sɛrí nɔn nɛ pi yɑ́ɑ́si. Lɛ̃ nnyɑ kɑm nɔ́ lésɛnɛ kɛ́ nɛ Pɑpilɔɔni kɛtẽ kɛpirɛ-mɛmpɔ hɑ. 
Pirɔ́sɑ́ɑyɑhɑ yɛ pɛɛ kucɛsi kóimɑ-i kucɑ́ŋíí mɑ́lɛ kpɛɛ n nyísɛlɛ̃ tɛ Uléécɑɑ yɛ tɔ́ nɛ uyɛ̃ kɛcɔpɛ rinɔ́ɔ yekeilɛ. Ku pɛɛ́ welɛ yɑrɛ kpɛ̃ kɛ Uléécɑɑ uú pɛɛ Moisi n símisi rɛ ukɛ́ wɑ. U pɛɛ Moisi mɑɑ rɛ ukɛ́ kpɛ̃ kuú pɛɛ n yɛ̃́ kucɔ wɑ. 
Pirɔ́sɑ́ɑyɑhɑ yɛ kucɑ́ŋíí kpɛ̃ nɛ pipipirɛ̃ tíyɛlɛ. Pɛɛ́ uyɛ-i kɛ Yosuwee uú pɛɛ pi n kpɑ́úlɛ̃ kɑpí pɛɛ́ nɛ ku ripɛ́nɛ ɑpí nɛ kɛtẽ kɛɛ pikɛyu-mɛmpɔ ń we tɛ́ŋ́. Kɛtẽ kɛ̃ pikɔ́ kɛ Uléécɑɑ uú pɛɛ lɑ́kɑsɛ. Kucɑ́ŋíí kpɛ̃ ɑku lɛ̃ Kɑnɑɑ-i n we ɑí nɛ Tɑfiti kumúŋɛ́ tulu. 
Uléécɑɑ yɛ pɛɛ́ nɛ Tɑfiti kɛfɑ fénnɛlɛ. Tɑfiti uu pɛɛ u pisɛ rɛ ukɛ́ ncée u hɛ ukɛ́ kɛ́yɔ mɔ̃ kɛkɛ́ wɑ kei kɛ Yɑkupu Uleecɑɑ uu ń we. 
Amɑ́ Sɑlomɔɔ yɛɛ kɑ uu kɛ́yɔ u mɔm. 
Ḿpɑ́ nɛ lɛ̃, Uléécɑɑ yɛɛ ḿpɑ́ yo kɛcɑ́ɑ́ ń tṹ, úu síyɔ kɛ pisoi ɑpi m mɔ̃́ kɛ́mɛɛ sói. Yɑrɛ kɛ ɑntepu Esɑyi uu ḿ mɑɑ rɛ: 
Upíimɑ yɛ mɑɑ rɛ Kɛyómɛcɑɑ yɛ rinɛ́yɔ́ɔpitũ lɛ, kɛtẽ pɔ́ɔ kɛnɛ́nɑcɔpɛ kɛ́yɔ kɛ́ye kɛ nɔ́ɔ yɛ́ nɛ́ mɔ̃́? Yei kɑm yɛ́ tonɛ kɛ́ nɛ n wéntɛ? 
Ái rinɛ́cúruu nɛ́ɛ lɛ̃ nnɛ́í wɑɑ? 
Etiyɛɛni uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɔ́ pisoi mpí ɑtii yɛ pɑ́ɑ́púlɛ hɑ́i. Nɔ́ ɑkíŋ tɑkɑ́lɛ, nɔ́ɑnɛ́tu ɑ́ɑ yɛ kɔ nsímɛ́ kṍ. Nɔ́pinɛ́sɑ́ɑyɑhɑ mɛcɔ kɑni nní we nɔn Nfɑ́ɑsɔnɛ rinɔɔ kɛsu. 
Antepu úye mpuri kɛ nɔ́pinɛ́sɑ́ɑyɑhɑ ɑ́pi wéékusɛlɛ̃? Pɛ̃ nnɛ́í pɛɛ n símisi rɛ Asei ute yɛ wemɛ kɑpi kṍ. Asei ute uyɛ̃ kɛ nɔ́ɔ kɔ nkpéni nní pilɑ́ɑrɔ ɑnipɛ-i wɑ ɑni u kopu. 
Uléécɑɑ yɛ nɛ piléécɑɑtumɛ ritikilɛ uú nɛ uisé nɔ́ pɑ, ɑmɑ́ ɑ́ni i ritiki. 
Kɛ pɛpɛɛ pɛɛ kɛtúhɑɑnɛ-i ń we ɑpi lɛ̃ n kṍ, ɑi kuwɔ́i pi fukɔɔmɛ ɑpi pɛɛ Etiyɛɛni nsímɛ́ mmɛ̃ nɛ ɑnĩ́ n tɑkɑi. 
Amɑ́ Etiyɛɛni pɔ́ɔ pɛɛ kɛléécɑɑ inípɛɛ n tɑpɑɑlɛ̃, Nfɑ́ɑsɔnɛ nn u yipɑɑlɛ̃ un nɛ Uléécɑɑ mɛyɔɔpi nɛ Yeesu yɛɛ Uléécɑɑ kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i n nyɛnu nyɑ́nɛi. 
Kei kuu pɛɛ rɛ: Nɛ kɛyómɛcɑɑ nyɑ́nilɛ kɛn wúkulɛlɛ̃, Usoi Kɛpipi kɛn Uléécɑɑ kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i nyɛnu. 
Kuu lɛ̃ ḿ mɑ, ɑpi kɛpupɛilɑ píimɑ kɛnyinɛ té ɑpi ɑtu tɑ́pisi ɑpi pinnɛ́í kɛsẽ́ pehépɔ ɑpi u fílí, 
ɑpi u lɑkɑsɛ ɑpí nɛ kuyu iyɑɑ leepɔ ɑpi ɑpɑrɛ u tɑpisi ɑpi kópu. Pɛpɛɛ kei ń we ɑpi pisitukɑnkɑ cícɑɑ́ mɑ́hɑni ɑpi uŋmɑ́nɛ unyinɛ kɑpi yɛ n sée rɛ Sɔɔli pɑ uu si m mɛ̃́. 
Pi Etiyɛɛni ɑpɑrɛ tɑpisilɛ un we un kɔ nɛ kɛyómɛ yɑ́ɑ́si. Uu rɛ: Upíimɑ Yeesu, ɑ nnɛ́fɑ́ɑ yɔsí, 
uu pɛɛ ɑnui kɛcɑ́ɑ́ wúlɑ uu mɛsɔ́ri pupɛi rɛ: Upíimɑ, kɑpɛ ɑkópɛ nnyɛ́ piriyu kɛcɑ́ɑ́ wɑ. Kuu lɛ̃ pimɑ́ɑ ḿ mɑsí, uu pɛɛ kpi.
