Yohɑni.
18.
Kɛ Yeesu uu lɛ̃ pisímɛ́ ḿ mɑsí, u nɛ upipirɛtiki ɑpi yisi ɑpi Seturɔɔ kɛfɑŋɑɑ mɛtɛ́ŋ́-mɛ̃ sĩ ɑpí hɑ kɛléécɑrɛ kɛnyinɛ kɛɛ kei ń we kɛ́mɛɛ lompɔ. 
Yutɑsi yɛɛ Yeesu n yɑ́ɑ́ yɛ kei nyulɛ. Yeesu nɛ upipirɛtiki yɛ̀ɛ̀ pɛɛ kei mɛsɛ́rɛ cɑ́pinɛlɛ. 
Kei kɛ Yutɑsi uu yisi uu pisɔ́ɔ́cɑ kpísi uu kɔ Uléécɑɑ kɛyɔ pimɛ́rɛ́ kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi ɑpi n tumpɔ nɛ pɛ́nɛ, pin iyúipukɛ nɛ sífirɑ́ɑ nɛ itɔpilũ mulɛilɛ̃. 
Yeesu yɛ pɛɛ lɛlɛɛ u n wɑinɛ pinyuwɛ mɑ́ɑ́lɛnlɛ, uu pɛɛ́ nɛ nyɔsɔ́pɔ uu pi pisɛ rɛ: Úye kɑni wɛ́ɛ́si? 
Api u pɛsɛ rɛ: Nɑsɑrɛti ukɔ́ Yeesu. Uu pɛɛ pi pɛsɛ rɛ: Nɛ́ɛ lo! Yutɑsi yɛɛ u n yɑ́ɑ́ yɛ pɛɛ kei welɛ lin nɛ lɛ̃ wɑi. 
Kumúŋɛ́ kpɛ-i kɛ Yeesu uú pɛɛ pi ḿ mɑɑ rɛ: Nɛ́ɛ lo, pi ipírɛ̃ pírɛ̃ nɛ sɔ́nɛlɛ ɑpí hɑ nɛ lólu. 
Yeesu uu kɔ pipisɛ pi kpɑ́ rɛ: Úye kɑni wɛ́ɛ́si? Api rɛ: Nɑsɑrɛti ukɔ́ Yeesu. 
Yeesu uu pɛɛ pi pɛsɛ rɛ: Ńté nɛ nɔ́ mɑɑ rɛ nɛ́ɛ lo. In tɛ nɛ̃́ kɑni mɛsei wɛ́ɛ́si, ɑni picɔ riyɑ́ pikɛ́ tɔ́mpɔ. 
Lɛ̃ kuú pɛɛ mɛkɛɛ ḿ mɑ likɛ́ nɛ n wɑ nnyɑ kuu lɛ̃ mɑ. U pɛɛ́ mɑɑ rɛ: Pɛ̃ kɑɑ nɛ́ rimmúísɛ, úkɑ úu nɛ nɛ́ pɔ. 
Simɔɔ Piyɛɛ yɛ pɛɛ ritɛ́hɛ cɑɑcɑɑ múílɛnlɛ. Uu ri mɑhɑni uu Uléécɑɑ usinɑ ukɛikɔ́ usɛ kutu lukɛ-lukɛ seḿ. Pi yɛ ukɛikɔ́ uyɛ̃ sée rɛ Mɑlikuusi. 
Kei kɛ Yeesu uu pɛɛ Piyɛɛ mɑɑ rɛ: A ripɔ́tɛ́hɛ kukɑi kɛ́mɛɛ pɛsɛpɔ. Íwɛ iyɛ̃ kɛ Unɛ́sɑ́ɑ uu wɑ rɛ kɛ́ li. Ám kɑpɛ pɛɛ i li, nɛ́ɛ íye? 
Pisɔ́ɔ́cɑ pɛ̃ nɛ piukpeenwɑi nɛ Pisuifi pimɛ́rɛ́ ɑpi pɛɛ Yeesu tini ɑpi pɑɑsi 
ɑpí nɛ Anɑ kɛyɔ u sĩ kɛlenɛ. Anɑ yɛ pɛɛ Kɑyifu yɛɛ kumúŋɛ́ kpɛ-i isínɑ n tṹ ulɔ́hɔ́ lɛ. 
Kɑyifu yɛɛ pɛɛ Pisuifi piwɛ́ɛ́sɛ mɛsɔhɔ rikpɑ́ rɛ: Li kutɔsi we rɛ usoi usɛ ukɛ́ kuyu nnɛ́í nkpɔ kpu. 
Pi pɛɛ lɛ̃ nɛ Yeesu tɔ́sulɛ, Simɔɔ Piyɛɛ nɛ uupirɛtikicɔ unyinɛ pin Yeesu tikilɛ̃. Usínɑ yɛ pɛɛ upirɛtiki ucɔ uyɛ̃ nyulɛ. Lɛ̃ nnyɑ kuu fe u nɛ Yeesu ɑpi kucɔ usínɑ kɛyɔ-i lompɔ. 
Amɑ́ kɛtɑhɑi kɛ Piyɛɛ pɔ́ɔ nyɛrɛ uú nɛ ripoo kɔ. Kei kɛ uupirɛtikicɔ uyɛ̃ kɛ usínɑ uu nní ń nyu uu pɛɛ léemɛ uú nɛ unɔ́sikɛikɔ́ uyɛɛ ripoo-i m mɛ̃́ símisi uu pɛɛ Piyɛɛ tɑmpɔ. 
Unɔ́si uyɛ̃ uu pɛɛ Piyɛɛ pisɛ rɛ: Ńté pɔ̃́ nɛ kɔ utisi uyɛ̃ upirɛtiki lɛ? Piyɛɛ uu rɛ: Aɑí, ɑ́m utisi uyɛ̃ upirɛtiki. 
Nnyiyɛ yɛ pɛɛ likumúŋɛ́ kpɛ-i welɛ. Pikɛikɔ́ nɛ pimɛ́rɛ́ ɑpi pɛɛ nkɑlɛnɑ wɑi ɑpi kɛkɑ́lɛ wɑi pin pɛɛ wéni. Piyɛɛ pɔ́ɔ kɔ pikɛmɛɛ́ hɑpɔ uú n tũ un nnɑ wéni. 
Usínɑ uu Yeesu upipirɛtiki nɛ uicélɑɑ kɛcɑ́ɑ́ písɛisɛ. 
Yeesu uu u pɛsɛ rɛ: Ḿpɑ́ úye kɛyu-i kɑm símisi ɑm kɔ mɛsɛ́rɛ Pisuifi ɑyomɛyɑ́hɑɑlee-i nɛ Uléécɑɑ kɛyɔ-i kɛ Pisuifi nnɛ́í ɑpi yɛ n cɑ́pinɛ céesi. Ám kɛtumɛ ńkɑ mɛyɑɑ mɑ. 
Yo nnyɑ kɑɑ pɛɛ nɛ́ písɛisɛ? Pɛpɛɛ n kṍ nɛn símisi kɑɑ yɛ́ pisɛ rɛ yo kɛcɑ́ɑ́ kɑm pi símisi. Pɛ̃ nɛ nyulɛ ńsɔnɛ lɛ̃ kɛ nɛ́ɛ ḿ mɑ. 
Kɛ pimɛ́rɛ́ pɛ̃ usɛ uu lɛ̃ n kṍ, uu Yeesu kɛpɛsɛ tɔlu uu u pisɛ rɛ: Lɛ̃ kɑɑ yɛ́ pɛɛ usínɑ rinɔ́ɔ pɛsɛɛ? 
Yeesu uu pɛɛ u mɑɑ rɛ: In tɛ ɑ́m ńsɔnɛ símisi, ɑ nɛ́ símisi kei kɑm m púnnɛ. Amɑ́ in tɛ nɛ ńsɔnɛ símisilɛ, yo nnyɑ kɑɑ pɛɛ nɛ́ pi? 
Anɑ uu pɛɛ u kpísi un lɛ̃ pɑhɑɑlɛ̃ uú nɛ usínɑ Kɑyifu u pɑpɔ. 
Simɔɔ Piyɛɛ yɛ pɛɛ likumúŋɛ́ kpɛ-i welɛ un nnɑ wéni. Api u pisɛ rɛ: Ńté pɔ̃́ nɛ kɔ utisi uyɛ̃ upirɛtiki lɛ? Uu kɛ́su uu rɛ: Aɑí, ɑ́m utisi uyɛ̃ upirɛtiki. 
Usínɑ ukɛikɔ́ usɛ yɛ pɛɛ utisi uyɛ̃ kɛ Piyɛɛ uu kutu rińseḿ umɑrɛcɔ lɛ. Uu pɛɛ Piyɛɛ pisɛ rɛ: Ái nɔ́ nɛ uyɛ̃ kɑm ɑléé kɛcɑrɛ-pɔ yɛ̃́ɛ? 
Piyɛɛ uu kɔ Yeesu pikɛ́ŋɛ kpɑ́. Mɛsɛ nɛ mɛsɛ, icɑ̃́ ii kooi. 
Api mmɑrɛ mɑrɛ Kɑyifu kɛyɔ-i Yeesu kpísi ɑpí nɛ riyɔ́ɔpipoo sĩ. Amɑ́ Pisuifi piwɛ́ɛ́sɛ ɑ́pi pɛɛ lompɔ rɛ ɑ́i kɑpɛ mɛ́kpɛrinkpɛ pi wɑisɛ, ɑ́pi yɛ́ pɛɛ fe pikɛ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ li nnyɑ. 
Lɛ̃ nnyɑ kɛ Pilɑti uu léepɔ uu pi lɛɛpɔ, uu pi pisɛ rɛ: Ń-ye kɑni nɛ usoi nkó túhɑɑnɛ? 
Api u pɛsɛ rɛ: Usoi nkó unsɑ́ pɛɛ mɛwɑi kópɛ n wɑi, ɑ́ri yɛ́ pɛɛ ɑpɔ́nípɛ-i u wɑ. 
Kei kɛ Pilɑti uu pɛɛ pi mɑɑ rɛ: In lɛ̃, ɑni u kpísi ɑní nɛ nɔ́rinɛ́cúruu u túhɑɑnɛ yɑrɛ kɑi nɔ́inɛ́sé kɛ́mɛɛ m pisɛ. Pisuifi ɑpi u pɛsɛ rɛ: Ári ncée mɑ́ tɔkɛ́ unyinɛ kpu. 
Tinɔ́ɔ tɛ̃ kɛ Yeesu uú pɛɛ ḿ mɑ uú nɛ nkpɔ mmɛ̃ kuu n kpinɛ kɛcɑ́ɑ́ símisi kɑi lɛ̃ titiki. 
Pilɑti uu riyɔ́ɔpipoo-i lompɔ uu Yeesu pisɛ rɛ: Pɔ́ɔ Pisuifi Uyɔɔpii? 
Yeesu uu rɛ: Pɔ́ɔ ripɔ́cúruu lɛ̃ pisɛ nɛ́ɛ unyinɛ yɛɛ nnɛ́símɛ́ pɔ́ símisi? 
Pilɑti uu rɛ: Pɔ nyu rɛ nɛ̃́ nɛ Usuifi lɛɛ? Mpɔ́puri pikɔ́ nɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ pɛɛ ɑnɛ́nípɛ-i pɔ́ wɑ. Yo kɑɑ mpíí wɑ? 
Yeesu uu rɛ: Kɛnɛ́yɔ́ɔpitẽ ɑ́kɛ kɛtẽ nté kɛkɔ́. Nɛn pɛɛ kɛtẽ nté iyɔɔpi n le, pinɛ́kɛikɔ́ yɛ́ pɛɛ kɛnɛ́cɑ́ɑ́ ɑtɔpi wɑ rɛ ɑ́pi kɑpɛ Pisuifi ɑnipɛ-i nɛ́ wɑ nnyɑ. Amɑ́ inɛ́yɔ́ɔpi íi nkpɑ́ni kɛtẽ nté ikɔ́. 
Kei kɛ Pilɑti uu pɛɛ u pisɛ rɛ: Lɛ̃ nnyɑ, pɔ uyɔ́ɔpi lɛɛ? Yeesu uu rɛ: Lɛ̃ kɑɑ nní mɑ. Nɛ uyɔ́ɔpi lɛ. Pi nɛ́ mɑ́rilɛ ɑm kɛtẽ kɛcɑ́ɑ́ weri rɛ kɛ́ nsímɛ́sei kɛcɑ́ɑ́ símisi. Nkó yɛɛ nsímɛ́sei m mɑ́ yɛ̀ɛ̀ kutu nɛ́ ricɔlɛ. 
Pilɑti uu u pisɛ rɛ: Yo kɑpi yɛ sée rɛ nsímɛ́sei? Kɛ Pilɑti uu lɛ̃ Yeesu pipisɛ ḿ mɑsí, uu yisi uu piléepɔ kpɑ́ uu Pisuifi lɛɛpɔ, uu pi mɑɑ rɛ: Nɛ̃́ ɑ́m yɛ̃ yo kuu cɑɑi nnyɑ kɑpi yɛ́ nɛ u ń kpu. 
Amɑ́ li nɔ́kɛnɛ́mɛɛ inyɛ́kii lɛ rɛ kɛ́ yɛ ḿpɑ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ɑ́ye kɛ́mɛɛ ukpɑniikɔ́ usɛ unyinɛ ɑkpɑnii tɔ. Nɔ lɑ rɛ kɛ́ Pisuifi Uyɔɔpi ɑkpɑnii nɔ́ tɔɔ? 
Api pɛɛ picɑ́ɑ́i kpɑ́ rɛ: Aɑí, ɑ́i uyɛ̃, ɑ Pɑrɑpɑsi ɑkpɑnii rɔ́ tɔ. Pɑrɑpɑsi yɛ pɛɛ kɔ usoi kpɑ́ɑ́ree lɛ.
