Yohɑni.
15.
Yeesu uu pimɑ́ɑ pi kpɑ́ rɛ: Nɛ̃́ nɛ fiinyi kuléé riyíkí lɛ, Unɛ́sɑ́ɑ yɛɛ kuute. 
U yɛ ḿpɑ́ nlésɛ ń-ye mɛɛ kɛnɛ́cɑ́ɑ́ n tórɑɑlɛ̃ ńn yɛ pɛɛ mɑ́ri rikélɛ uu kɔ iyɛɛ m mɑ́ru hɛ́ru, ii pɛɛ mɛyɑ̃́ pimɑ́rɛ kpɑ́. 
Icélɑɑ nnyí kɑm nní nɔ́ ḿ pɑ yɛ pihɛ́rɛ́ nɔ́ mɑsilɛ ɑni pɛɛ nkpɑ́ni cɑ́kɑ́-cɑ́kɑ́. 
Tɔkɛ́ m pɛ́nɛlɛ̃ nɔn kɛnɛ́mɛɛ we yɑrɛ kɛ nɛ́ɛ nɔ́kɛnɛ́mɛɛ ń we mɛcɔ. Nlésɛ nnsɑ́ nɛ kuléé n torɑɑlɛ̃, ńn yɛ fe nkɛ́ mmɛcirɛ n we nń nɛ ɑpipi mɑru. Limɛcɔ kɛ nɔ̃́ ɑ́ni yɛ́ fe nɔkɛ́ linyinɛ yoriyɛ nɔnsɑ́ nɛ nɛ́ m pɛ́nɛlɛ̃. 
Nɛ́ɛ kuléé riyíkí, nɔ́ɔ ilésɛ. Nkó yɛɛ nɛ nɛ́ m pɛ́nɛlɛ̃ un nɛ we yɑrɛ kɛ nɛ́ɛ ukɛmɛɛ ń we mɛcɔ yɛɛ yɛ linyinɛ yoriyɛ. Insɑ́ nɛ̃́, ɑ́ni yɛ́ fe nɔkɛ́ ɑpipi torɛsi. 
Úye unsɑ́ nɛ nɛ́ m pɛ́nɛlɛ̃ ukɛ́ nɛ n we, pi yɛ kɛtɑhɑi-mɛmpɔ u fómpɔlɛ yɑrɛ nlésɛ, uu kóósi ɑpi pɛɛ ilésɛ kóimɑ iyɛ̃ cɑ́pinɛ ɑpi nnɑ sɛ́ ií torɛ. 
Nɔn nɛ nɛ́ m pɛ́nɛlɛ̃ nɔn nɛ kɛnɛ́mɛɛ n we, nnɛ́símɛ́ nn kɔ nɔ́kɛnɛ́mɛɛ n we, ɑni yɛ lɛ̃ kɑni ń lɑ pisɛ, pi yɛ́ nɔ́ li pɑ. 
Nɔn ɑpipi mɛyɑ̃́ n torɛsi, nɔ lɛ̃ nyísɛlɛ rɛ nɔ pinɛ́pirɛtiki lɛ. Unɛ́sɑ́ɑ uu pɛɛ ríyu yɛnu. 
Nɛ nɔ́ lɑlɛ yɑrɛ kɛ Sɑ́ɑ uu nɛ́ ń lɑ mɛcɔ. Ani ńlɑ mmɛ̃ kɛ́mɛɛ n we. 
Nɔn inɛ́sé m múílɛ̃, nnɛ́lɑ kɛ́mɛɛ kɑni lɛ̃ we yɑrɛ kɑm Unɛ́sɑ́ɑ isé rimmúlú ɑm pɛɛ unlɑ kɛ́mɛɛ n we mɛcɔ. 
Nɛ lɛ̃ nɔ́ símisilɛ nɔkɛ́ nɛ mɛnɛ́cɔ mpɔ́ɔnɑrɛ m mɑ́, nɔ́nnɛ́pɔ́ɔnɑrɛ nkɛ́ kɔ n we nɛ nkɛtɔ-pɔ nnyɑ. 
Isé iyɛ̃ kɑm nɔ́ ḿ pɑ́lɛ̃ yɛɛ rɛ: Ani n lɑkɑɑnɛ yɑrɛ kɑm nɔ́ ń lɑ mɛcɔ. 
Lɛlɛɛ n fe kɛ usoi uu yɛ nɛ n nyísɛ rɛ u ńlɑ píimɑ mɑ́lɛ yɛɛ rɛ: Usoi ukɛ́ unfɑɑ cirɛ kpísi ukɛ́ upisɑnɛ riyu kɛcɑ́ɑ́ hɛ. 
Nɔn yɛ lɛ̃ kɑm nɔ́ m pisɛ n wɑ, nɔ pinɛ́sɑ́nɛ lɛ. 
Ám nkpéni piséi nɔ́ kpɑ́nɛ rɛ pikɛikɔ́. Ukɛikɔ́ úu yɛ n nyu lɛ̃ kɛ ukɛisɑɑ uu n wɑi. Lɛ̃ nnɛ́í kɛ Unɛ́sɑ́ɑ uu nɛ́ n nyísɛ kɑm nɔ́ céesi. Lɛ̃ nnyɑ kɑm nɔ́ sée rɛ pinɛ́sɑ́nɛ. 
Ái nɔ́ɔ nɛ́ wɛ́ɛ, nɛ́ɛ nɔ́ wɛ́ɛ ɑm nɔ́ tum tɛ ɑni n ŋme ɑni hɑ pikɛi wɑ likɛ́ yóriyɛ ḿpɑ́ píyei-pɔ. Lɛ̃ kɛ́mɛɛ, Unɛ́sɑ́ɑ yɛ́ pɛɛ lɛ̃ nnɛ́í kɑni yɛ́ nɛ rinɛ́nyíri u m pisɛ nɔ́ pɑ. 
Lɛ̃ nnyɑ, lɛ̃ kɑm nɔ́ m pisɛ yɛɛ rɛ ɑni n lɑkɑɑnɛ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Kɛtẽ nkɛ́ pikɔ́ pin nɔ́inɛ́puri n yulu, ɑni ceri rɛ inɛ́puri kɑpi mɛfoí yɛ̀ kɛlenɛ. 
Nɔn pɛɛ kɛtẽ nkɛ́ pikɔ́, pi yɛ́ pɛɛ nɔ́inɛ́puri n lɑ rɛ nɔ pipikɔ́ nnyɑ. Amɑ́ kɑ́ni nní kɛtẽ nkɛ́ pikɔ́, ɑm kɔ nɔ́ wɛ́ɛ, nɔ́ nɛ pɛ̃ ɑ́ni pɛɛ líkɑ pɛ́nɛlɛ̃ nnyɑ, ɑ́pi pɛɛ nɔ́inɛ́puri lɑ. 
Ani rinɔ́ɔ tɛ̃ kɑm nɔ́ ḿ mɑ léisɛ rɛ: “Ukɛikɔ́ úu yɛ ukɛisɑɑ n fe. Pin nɛ́ n wéékusɛlɛ̃, ɑni n nyu rɛ pi kɔ nɔ̃́ wéékusɛnɛ, pin yɛ nnɛ́símɛ́ rimmúlú, pi yɛ́ kɔ nɔ̃́ nkɔ́ rimúlú.” 
Amɑ́ nɛ̃́ nnyɑ kɑpi lɛ̃ nnɛ́í nɔ́ wɑinɛ, kɑ́pi nní uyɛɛ nɛ́ n tummɛ n nyu nnyɑ. 
Nɛnsɑ́ pɛɛ́ n kɑ, nɛnsɑ́ pɛɛ kɔ pi n símisi, ɑ́pi yɛ́ pɛɛ ɑkópɛ m mɑ́. Ápi nkpéni ńkɑ mɑ́ pikɛ́ nɛ piɑkópɛ yɑrii. 
Nkó yɛɛ inɛ́puri n yúlu yɛ kɔ Unɛ́sɑ́ɑ ipuri yululɛ. 
Nɛnsɑ́ pɛɛ pikɛcɔpɛ mɛwɑi n wɑpisi kɛ úkɑ mpuri úu n wɑilɛ̃, ɑ́pi yɛ́ pɛɛ ɑkópɛ m mɑ́. Amɑ́ pi nkpéni mɛ yɛnlɛ ɑpi kɔ pɛɛ tɔ́ nɛ Unɛ́sɑ́ɑ ipuri yulu. 
Lɛlɛɛ piisé kɛ́mɛɛ n wɔ́lɑɑlɛ̃ likɛ́ nɛ n wɑ nnyɑ kɑi lɛ̃ wɑi. Lɛlɛɛ rɛ: Mɛ́woo kɑpi inɛ́puri yulu. 
Ucomɛ yɛɛ n sɔ́nti yɛɛ nní nsímɛ́sei Nfɑɑsɔnɛ kɑḿ nɛ Sɑ́ɑ rinyiri nɔ́kɛnɛ́mɛɛ n tumtinɛ. Sɑ́ɑ kɛ́mɛɛ kuu léeri ukɛ́ kɛnɛ́cɑ́ɑ́ ɑseérɑ lesɛ. 
Nɔ̃́ nɛ́ kɛnɛ́cɑ́ɑ́ ɑseérɑ lesɛ rɛ tɔ́ nɛ nnɔ̃́ tɔ́ɔ hɑ́i kɛ́korɑɑnɛ-mɛ we nnyɑ.
