Luki.
23.
Pisoi nnɛ́í pɛɛ n cɑ́pinɛlɛ̃ ɑpi yisi ɑpí nɛ Yeesu Pilɑti kɛ́mɛɛ sĩ́. 
Api kei nɛ nsímɛ́ u wɑ́ɑ́nɑ́ pin tee rɛ: Tɔ usoi nkó lɛɛpɔlɛ un kurɔ́yu pikɔ́ tɑ́lisi rɛ pikɛ́ ncée kópɛ ritiki. U pi símɑɑnkɛɛ rɛ ɑ́pi kɑpɛ lɑmpoo Rɔm uyɔɔpi Sesɑɑ hɛ́ɛlɛ, un kɔ tee rɛ uyɛ̃ kɛ Uléécɑɑ uu wɛ́ɛ, uyɔ́ɔpi. 
Pilɑti uu u pisɛ rɛ: Pɔ́ɔ Pisuifi uyɔɔpii? Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Mmɛ̃ kɑɑ nní mɑ. 
Pilɑti uu pɛɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ risoiwuí mɑɑ rɛ: Ám ncɑɑi ńkɑ nyɑ́ni mɛɛ yɛ́ usoi nkó ɑkpɑnii n tɑ̃. 
Amɑ́ ɑpi pimɑ́ikɛɛ n kpɑ́lɛ̃ nɛ ńnɑŋɛ rɛ: Usoi nkó yɛɛ kuyu pikɔ́ pihɑ́ikusɛ tɑ́lɑɑnkɛɛ, Kɑlilee-pɔ kuu kóri uu Yutee kɛtẽ nnɛ́í cɔpu uú nɛ nté tuiri. 
Kɛ Pilɑti uu ɑnɔ́ɔ nyɛ̃ n kṍ, uu pisɛ rɛ: Usoi nkó yɛ Kɑlilee ukɔ́ lɛɛ? 
Kuu n céri rɛ Erooti yɛɛ Kɑlilee kɛtẽ-pɔ Yeesu uyɔɔpi, uu Yeesu u pɑ́ɑ́ipɔ, liriyíkí rɛ Erooti ricuruu yɛ siyɑ́ɑ sɛ̃ kumúŋɛ́-i Yerusɑlɛm-i welɛ. 
Kɛ Erooti uu Yeesu n yɛ̃́, ɑi u lɑrisi, liriyíkí rɛ u pɛɛ mɛkɛɛ-mɛ Yeesu nnɛ̃́ kómɛilɛ lin tipɔ́ɔ u we rɛ ukɛ́ u yɛ̃. U pɛɛ tɑ́lɛ̃ tɛ Yeesu yɛ́ uinipɛɛ-i mɛwɑisɑŋɑ mɛnyinɛ wɑ. 
Uu u písɛisɛ ɑi nɑ́ŋɑi, ɑmɑ́ Yeesu úu rinɔ́ɔ ríkɑ u yɔsí. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ pin kei we pin nɛ Yeesu isímɛ́ piwɑɑnɑi kɛ́yu mɑhɑni. 
Erooti nɛ upisɔ́ɔ́cɑ ɑpi kɛcɑ́ɑ́ u nyɑ́nɛi ɑpi u n sɛ́nnyi yɑrɛ kɛsoi kpɑ́rɑ́pi. Api kɛtúkɑnkɑ kɛcirɛ u tɑni ɑpi Pilɑti-mɛ̃ u pɛsɛpɔ. 
Erooti nɛ Pilɑti yɛ pɛɛ pilɑ́ɑrɔ lɛ, ɑmɑ́ ɑpi kɛyɑ́ɑ kɛ̃ kusɑ́nɛ tonɛ. 
Pilɑti uu pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ kuyu piyukɔɔ́ nɛ kuyu pikɔ́ cɑ́pinɛ, 
uu pi mɑɑ rɛ: Nɔ nɛ usoi nkó nɛ́ kɑlɛ nɔn tee rɛ u pisoi tɑ́lisi rɛ pikɛ́ ncée kópɛ ritiki. Too, nɔ́kɛnɛ́yu-i kɑm u písɛi ɑ́m kɔ ncɑɑi kɑni rɛ u wɑi ńkɑ yɛ̃. 
Erooti úu kɔ uncɑɑi ńkɑ yɛ̃, liriyíkí rɛ u rɔ́ u pɛsɛmɛlɛ. Kɑi nní lɛ̃, usoi nkó úu ncɑɑi ńkɑ mɑ́ mɛɛ yɛ́ ń tíyɛsɛ pikɛ́ u ń kpu. 
Lɛ̃ nnyɑ nɛ́ tíyɛsɛ ɑpi kuyíyɑɑ u fɑpii ɑm lɛ̃ mɛmɑ́ɑ́ u tíyɛ. 
Li pɛɛ tílɑsi lɛ rɛ Pilɑti ukɛ́ ukpɑniikɔ́ usɛ pi riyɑ́ ḿpɑ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ɑ́ye. 
Api pinnɛ́í pipupukɛɛ kpɑ́ rɛ: A usoi nkó rɔ́ kpu ɑɑ Pɑrɑpɑsi yɑ́! 
Pihɑ́ikusɛ mɛwɑi mɛnyinɛ kɛ Pɑrɑpɑsi pɔ́ɔ pɛɛ kuyu-i tɑmmɛ uu kɔ usoi kopu. Lɛ̃ nnyɑ kɑpí pɛɛ uyɛ̃ kukpɑniilee tɑ̃. 
Kɛ Pilɑti uu nní ń lɑ ukɛ́ Yeesu ŋmɛ́hɛrɛɛsɛ nnyɑ, uu risoiwuí nɛ pisímɛ́ kpɑ́. 
Amɑ́ ɑpi pupɛi rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii, ɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii. 
Pilɑti uu mɛtɑ́ɑ́nũ pimɑ́ɑ pi kpɑ́ rɛ: Ncɑɑi ń-ye kuu mpíí wɑ? Ám ncɑɑi ńkɑ ukɛcɑ́ɑ́ yɛ̃́ mɛɛ yɛ́ ń tíyɛsɛ pikɛ́ nɛ u ń kpu. Lɛ̃ nnyɑ nɛ́ tíyɛsɛ pikɛ́ nɛ kuyíyɑɑ u kɑii ɑm pɛɛ u yɑ́pɔ. 
Amɑ́ ɑpi pipupukɛɛ nɔŋ́ tɛ pikɛ́ Yeesu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii. Ai Pilɑti feriyɛ. 
Pilɑti uu hɔ́ rɛ ukɛ́ lɛ̃ kɑpi u m pisɛ pi wɑ. 
Uu usoi uyɛ̃ kɑpi n tee rɛ ukɛ́ riyɑ́ yɑ́, uu Yeesu pi múísɛ rɛ pikɛ́ nɛ u li lɛ̃ kɑpi ń lɑ. Uyɛ̃ kuu nní rin-yɑ́ yɛ pɛɛ pihɑ́ikusɛ mɛwɑi mɛnyinɛ kuyu-i tɑmmɛlɛ uu kɔ usoi kopu. Lɛ̃ nnyɑ kɑpí pɛɛ uyɛ̃ kukpɑniilee tɑ̃. 
Kɑpí nɛ Yeesu n tɔ́su, ɑpí nɛ Sirɛɛni kuyu ukɔ́ unyinɛ kɑpi yɛ n sée rɛ Simɔɔ sɑ́nɛ un sicɑrɛ-mɛ̃ leeri. Api u tini ɑpi Yeesu kunɑpɛ́ɛ́kɔɔ́ u cɔnsɛ rɛ ukɛ́ nɛ u n tikilɛ̃. 
Pisoi kulúi nɛ pinɔ́si pin u tikilɛ̃. Pinɔ́si pin siwuu pirɑlɛ̃ pin nɛ ukɛcɑ́ɑ́ téni pin kɔ nɛ u tikilɛ̃. 
Yeesu uu pinɔ́si kuwɛ́lɛ́-mɛ̃ pɑnsɛpɔ uu rɛ: Yerusɑlɛm pinɔsi, ɑ́ni kɑpɛ nɛ̃́ nnyɑ n ténisi. Amɑ́ ɑni nɔ́rinɛ́cúruu nɛ nɔ́sinɛ́wɑ̃́ nnyɑ ténisi. 
Likumúŋɛ́ rɛ siyɑ́ɑ yɛ sɔ́ntilɛ, sɛ-i kɑpi yɛ́ ḿ mɑɑ rɛ: Pinɔ́si nyɑpi pɛɛ ɑ́pi píkɑi m mɑ́rɑɑlɛ̃ ɑ́pi kɔ píkɑi n nyɛnsɛlɛ̃ yɛ pínɑrɛkomɛ lɛ. 
Pisoi yɛ́ pɛɛ ɑkúú pimɑ́ikɛɛ lõ tɛ: Ani kɛrɔ́cɑ́ɑ́ loó, ɑpi sikúúpi mɑ́ikɛɛ rɛ: Ani tɔ́ yɑ́rii. 
Likumúŋɛ́ rɛ pin nɛ́ɛ ń we yɑrɛ kunɑ ŋmṹ nní n wɑ, íye kɑpi yɛ́ pɛɛ nɔ́ɔ nní ń we yɑrɛ kunɑ kóimɑ pɔ̃́ wɑ? 
Api kɔ nɛ piwɑikópɛkɔɔ́ pitɛ́ pinyinɛ n tɔ́su pikɛ́ hɑ pi nɛ Yeesu kɛsẽ́ kpu. 
Kɑpi kɛlõ kɛ̃ kɑpi yɛ n sée rɛ Kúyu Koŋoo n tu, ɑpi kei Yeesu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii, ɑpi kɔ piwɑikópɛkɔɔ́ pitɛ́ pɛ̃ kɑrikɛɛ ɑnɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́, usɛ ukulukɛ-lukɛ, ucɔ ukumii-mɛ̃. 
Yeesu uu pɛɛ rɛ: Sɑ́ɑ, ɑ piɑkópɛ pi sɑ́rɛi, likumúŋɛ́ rɛ ɑ́pi nyu lɛ̃ kɑpi n wɑi. Api pɑŋɑi ɑpí nɛ pimɛcɔpɛcirɛ uilũ hɔɔnɛntɛ. 
Piyukɔ́ pin kei nyɛnu pin nyɑ́nɛi. Pisuifi piwɛ́ɛ́sɛ ɑpi kɛcɑ́ɑ́ u nyɑ́nɛi pin tee rɛ: U picɔ yoriyɛntɛlɛ, ukɛ́ uricuruu yoriyɛ, in tɛ u Kirisi lɛ, uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ. 
Pisɔ́ɔ́cɑ ricuruu ɑpi kɛcɑ́ɑ́ u nyɑ́nɛi ɑpí nɛ u kɔ ɑpi pítɑ nyɛpɛ-nyɛpɛ pinyinɛ u tuŋɑ́ 
ɑpi rɛ: Pɔn Pisuifi uyɔɔpi, ɑ ripɔ́cúruu yoriyɛ. 
Api pɛɛ uriyu kɛcɑ́ɑ́ wɔ́lu rɛ: Nkó yɛɛ Pisuifi uyɔɔpi. 
Uwɑikópɛkɔɔ́ usɛ kɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ ń kɑriilɛ̃ uu u lɑ́misi un tee rɛ: Nɛ́ntɛ pɔ́ɔ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ? A ripɔ́cúruu yoriyɛ ɑɑ kɔ tɔ̃́ yoriyɛ! 
Amɑ́ ucɔ nkó uú nɛ u cɛ́si rɛ: Pɔn nɛ Uléécɑɑ wuruu? Pɔ́ɔ nní uɑkpɑnii ɑcɔ kɛ́mɛɛ ń we. 
Tɔ̃́ ɑkpɑnii piwɑi yɛ nɛ risɑ́lɛ, likumúŋɛ́ rɛ mɛrɔ́wɑi kópɛ kulɑɑ kɑri nní le, ɑmɑ́ uyɛ̃ úu ncɑɑi ńkɑ wɑ. 
Uu limɛmɑ́ɑ́ kpɑ́ rɛ: Yeesu ɑ kɛnɛ́cɑ́ɑ́ léisɛ pɔn píyei ipɔ́yɔ́ɔpi pilukɛ n wemɛ. 
Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Asei kɑm pɔ́ símisi, nɛni kɛyɑɑ nkɛ́, tɔ́ nɛ mpɔ̃́ nɛ pi Uléécɑɑ iyɔɔpi kɛ́mɛɛ. 
Ituŋɛ kɛcírɛ kumúŋɛ́, kuŋmɑhɑ ɑku kɛtẽ nnɛ́í kɛcɑ́ɑ́ cɔni hɑ́i nɛ ituŋɛ mɛtɑɑni kumúŋɛ́-pɔ. Kusɑ́ŋɑ́ɑ́ kɑpí nɛ Uléécɑɑ kɛyɔ n kénɛlɛ̃ ɑku kɛcɔpɛ cɛɛ́rinɛ. 
Yeesu uu mɛyɑ̃́ pupɛi rɛ: Sɑ́ɑ, nɛ nnɛ́fɑ́ɑ ɑpɔ́nípɛ-i wɑ. Kuu ɑnɔ́ɔ nyɛ̃ pimɑ́ɑ ḿ mɑsí, uu kpi. 
Rɔm usɔ́ɔ́cɑ wɛ́ɛ́sɛ uu lɛ̃ nnɛ́í lɛɛ n wɑpisi yɛnu, uu Uléécɑɑ ríyu wɑisɛ uu rɛ: Asei kɛcɑ́ɑ́, usoi nkó yɛ pɛɛ ɑsei ute lɛ. 
Pɛ̃ nnɛ́í pɛɛ pɛɛ ń kɑ ɑwúí ɑwúí pikɛ́ nyɑ́nɛi, ɑpi lɛlɛɛ n wɑ yɛnu, ḿpɑ́ úye uu pɛɛ pɛlɛ un mpɔ́ɔcɑɑi nɛ kɛwuu pirɑlɛ̃. 
Yeesu pisɑnɛ nnɛ́í nɛ pinɔ́si pɛɛ pɛɛ u n sɑ́rummɛ hɑ́i Kɑlilee-pɔ, ɑpi iyɑɑ-pɔ nyɛrɛ ɑpi lɛlɛɛ n wɑi n nyɑ́nɛi. 
Pisuifi kuyu Arimɑtee kɛ́mɛɛ kɛ utisi unyinɛ uú pɛɛ́ we. Pi yɛ u sée rɛ Yosɛfi. U pɛɛ kɛtúhɑɑnɛ píimɑ uwɛ́ɛ́sɛ usɛ lɛ, ɑmɑ́ lɛ̃ kɛ upiwɛ́ɛ́sɛcɔ ɑpi rinhɔ́ ɑpi wɑi ɑ́i nɛ u risɑ́. Utisi uyɛ̃ nɛ pɛɛ ɑsei ute lɛ un kɔ Uléécɑɑ iyɔɔpi mɛkɑmɛ mɛ̃́. 
Uu Pilɑti kɛ́mɛɛ sĩ́ uú hɑ u pisɛ rɛ ukɛ́ Yeesu u pɑ. 
Uu limɛmɑ́ɑ́ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ Yeesu súisɛri, uu kusɑ́ŋɑ́ɑ́ u pílɛsi uu nhórɛ kɑpi kupɑrɛcɑŋɑɑ-i n wɑ kɛ́mɛɛ u finsɛ. Ápi nhórɛ mmɛ-i úkɑ kulɑɑlɛ̃. 
Kɛyɑ́ɑ kɛɛ nɛ risɑ́lɛ kɛwéntɛyɑɑ usé, kɛwéntɛyɑɑ kɛn nɛ korɑɑnɛnɛ. 
Pinɔ́si pɛɛ pɛɛ Yeesu n sɑ́rummɛ hɑ́i Kɑlilee-mɛ nɛ Yosɛfi ɑpi rikpíí-i símpɔ pikɛ́ yɛ̃́ lɛ̃ kɑpi Yeesu n finsɛ. 
Api pɛɛ kɛ́yɔ pɛlɛ ɑpí hɑ mɛ́kpɔ nɛ tulɑɑli ncɔpuri wɑi pikɛ́ Yeesu níŋɛsi. Amɑ́ ɑpi kɛwéntɛyɑɑ kɛtúŋɛ́ wéntɛ yɑrɛ kɛ isé ii m pisɛ ɑpi kɛlenɛ simpɔ.
