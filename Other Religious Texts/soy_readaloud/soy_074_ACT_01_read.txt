Uléécɑɑ Pitumɛ Mɛwɑi.
1.
Unɛ́sɑ́nɛ Teofiili, nɛ rítɛlɛ́ foí kɛ́mɛɛ pɔ́ símisilɛ lɛ̃ nnɛ́í kɛ Yeesu uu piwɑi nɛ picélɑɑ n kórɑɑnɛ 
hɑ́i nɛ kɛyɑ́ɑ kɛ̃ kɑi kɛyómɛcɑɑ-mɛ̃ u n kpíipɔ. Ukɛ́ kɛlenɛ rińtɑɑ́, u upitumɛ kuu n wɛ́ɛ uicélɑɑ pɑlɛ nɛ Nfɑ́ɑsɔnɛ nnɑŋɛ. 
U umɛkpɔ-mɑɑ́ mɛ́wee mpehẽ mpehẽ kɛ́mɛɛ umɛcirɛ pi nyísɛlɛ pikɛ́ céri rɛ u nfɑ́ɑ mɑ́lɛ. Siyɑ́ɑ ɑfɛɛnɑ kuu pi rikpɑ́fúmɛ uu Uléécɑɑ iyɔɔpi nsímɛ́ pi símisi. 
Kɛyɑ́ɑ kɛnyinɛ kɑpi kɛsẽ́ we nɛ Yeesu, uu pɛɛ isé nnyí pi pɑ rɛ: Áni kɑpɛ yisi nɔkɛ́ Yerusɑlɛm nɛ kɛtɑɑ wɑ, ɑmɑ́ ɑni lisɔnɛ lɛ̃ kɑni n kṍ nɛn símisi rɛ Sɑ́ɑ Uléécɑɑ yɛ nɛ nɔ́ mɛ̃́ kutu ricɔ. 
Míni ŋmɑnɛ kɛ Yohɑni uu nɔ́ wólɛ. Amɑ́ li tisɛ nkɑ́ripi, Uléécɑɑ ukɛ́ nɛ Nfɑ́ɑsɔnɛ nɔ́ wolɛ. 
Pɛpɛɛ pɛɛ Yeesu kɛyúrí n cɑ́pinɛlɛ̃ ɑpi pɛɛ u pisɛ rɛ: Kumúŋɛ́ kpɛ-i kɑɑ Isirɑyɛɛli kɛyɔɔpitẽ pɛsɛrinɛɛ, Upíimɑ? 
Uu pi pɛsɛ rɛ: Ái nɔ́nnɛ́kɔ́ ńkɑ rɛ nɔkɛ́ mɛwɑi mɛɛ n sɔ́nti kumúŋɛ́ nɛ mɛituŋɛ n nyu. Sɑ́ɑ Uléécɑɑ yɛɛ lɛ̃ nɛ uricuruu nnɑŋɛ yekei. 
Amɑ́ Nfɑ́ɑsɔnɛ nn nɔ́kɛnɛ́cɑ́ɑ́ n súimɛ, nɔ́ ńnɑŋɛ nnyinɛ yɛ̃, ɑni pɛɛ pɑnsɛ pinɛ́tɑnsei Yerusɑlɛm kuyu-i nɛ Yutee kɛtẽ kɛcɑ́ɑ́ nnɛ́í nɛ Sɑmɑrii kɛtẽ kɛcɑ́ɑ́, hɑ́i nɛ kɛtẽ nnɛ́í kɛtɔ-pɔ. 
Kɛ Yeesu uu rinɔ́ɔ tɛ̃ pimɑ́ɑ ḿ mɑsí, ɑi kɛyómɛcɑɑ-mɛ̃ u kpíípɔ, ḿpɑ́ úye un u nyɑ́nɛipɔ ɑhopɛ ɑnyinɛ ɑɑ́ weri ɑɑ́ nɛ u yɑrii ɑpi pɛɛ u pɑɑpɔ. 
Pin kei nní lɛ̃ we pin kɛléécɑɑ inípɛɛ wiilɛ̃ un tɔ́su kɛ pitisi pitɛ́ pinyinɛ pɛɛ ilũ tomɛ́ n tɑnɑɑlɛ̃ ɑpi pikɛyúrí rikpɑ́fúmɛ. 
Pitisi pɛ̃ ɑpi pi mɑɑ rɛ: Yo nnyɑ kɛ nɔ̃́ Kɑlilee pikɔ́ mpí ɑni nní nyɛnu nɔn kɛléécɑɑ-mɛ̃ nyɑ́nɛi. Lɛ̃ kɑi nní nɔ́kɛnɛ́cɔpɛ nkɛ́-i Yeesu uyɛ̃ kɛléécɑɑ-mɛ̃ n kpíipɔ nɔn nyɑ́ni mɛcɔ kuú hɑ pɛɛri. 
Pitumɛ ɑpi pɛɛ kɛlõ kɛ̃ kɑpi yɛ n sée rɛ Olifiyee rikúú yisi ɑpi Yerusɑlɛm pɛlɛ. Tikúú tɛɛ nɛ Yerusɑlɛm kɔlɛnlɛ. Lirinɑ̃ nɛ sɑ́ yɑrɛ kilomɛtiri usɛ. 
Kɑpi n tuipɔ, ɑpi kulee kunyinɛ-i tɑɑ́ kɛ́yɔ kɛnyinɛ kɛcɑ́ɑ́-pɔ. Kei kɑpi yɛ cɑ́pinɛ. Pɛpɛɛ kei n cɑ́pinɛlɛ̃ pɛɛ Piyɛɛ nɛ Yohɑni nɛ Yɑkupu nɛ Antiree nɛ Filipu nɛ Tomɑɑ nɛ Pɑtelemii nɛ Mɑtiyee nɛ Alifee uŋmɑ́nɛ Yɑkupu nɛ Simɔɔ yɛɛ ukuyu ń lɑ nɛ Yɑkupu uŋmɑ́nɛ Yuuti 
nɛ Yeesu uni Mɑɑri nɛ pinɔ́si picɔ pinyinɛ nɛ Yeesu piwɑ̃. Pinnɛ́í mmɛ̃ pɛɛ rinɔ́ɔ risɛ wɑ ɑpi yɛ cɑ́pinɛ ɑpí nɛ piyómɛyɑ́hɑɑ n ŋmɔ́ɔ́púlɛ̃. 
Likumúŋɛ́ kpɛ-i, Piyɛɛ uu kɛyɑ́ɑ kɛnyinɛ yisi, piyómɛyɑ́hɑɑ pílɛ nɛ ɑfɛɛrɛ́ pinyinɛ pin cɑ́pinɛlɛ̃, uu pikɛcɔpɛ mɑɑ rɛ: 
Pimɑ́rɛcɔ, li pisɛ rɛ lɛ̃ kɛ Nfɑ́ɑsɔnɛ nn Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n símisi likɛ́ wɑ. Nfɑ́ɑsɔnɛ yɛ Yutɑsi kɛcɑ́ɑ́ Tɑfiti rinɔ́ɔ rinyinɛ mɑɑsɛlɛ. Yutɑsi uyɛɛ pɑnsɛ pɛpɛɛ Yeesu n tĩ́ ukpeenwɑi. 
U pɛɛ urɔ́cɔ usɛ lɛ, tɔn kɛsẽ́ pikɛi pisɛ mpí wɑi. 
Siwóó sɛ̃ kuu usoi ń kpu ɑpi u hɛ́ɛ́lɛ kuú nɛ kɛcɑrɛ lɔ, uu pɛɛ kei lóiri uu ríyu mɛfoí kɛtẽ fɑpi, uu kɛcɔpɛ kpɑ́fú, ɑnyɔ́hɔ nnɛ́í ɑɑ cúmɛ. 
Yerusɑlɛm pikɔ́ nnɛ́í pɛɛ mɛwɑi mɛ̃ kõ, hɑ́i ɑpí nɛ kɛcɑrɛ kɛ̃ sée rɛ Hɑkɛlitɑmɑ, lɛlɛɛ rɛ unyinɛ mɛnyɛ yɛ kei kɔɔnu. 
Ticuruu li Ipɑkɑrɛ Siyomɛ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Ukɛyɔ kɛkɛ́ mɛŋmɑnɛ tonɛ! Úkɑ úu kɑpɛ kɛkɛmɛɛ tónɛ! Li kɔ wɔlɑɑlɛ̃ tɛ: Usoi féé ukɛ́ upikɛi yɔsí uú nɔŋ́! 
Kumúŋɛ́ kpɛ̃ nnɛ́í kɛ Upíimɑ Yeesu uú pɛɛ kɛrɔ́mɛɛ ń we, pisoi pinyinɛ yɛ pɛɛ ḿpɑ́ yei rɔ́ tíkilɛnlɛ. 
Hɑ́i kɛ Yohɑni uú nɛ míni u n wólɛ nɛ pɛɛ́ uyɛi-mɛ kɑi kɛrɔ́cɔpɛ nkɛ́-i u n kpíipɔ ɑí nɛ kɛyómɛcɑɑ sĩ́, pisoi pinyinɛ yɛ pɛɛ rɔ́ tíkilɛnlɛ. Lɛ̃ nnyɑ, li pisɛ rɛ pisoi pɛ̃ kɛcɔpɛ usɛ ukɛ́ kɛrɔ́mɛɛ n we ukɛ́ mɛrɔ́cɔ itɑnsei li rɛ Yeesu yɛ nkpɔ kɛ́mɛɛ yisi. 
Kei kɑpi pɛɛ pisoi pitɛ́ sée ɑpi nyɛrɛsɛ. Pɛpɛɛ Yosɛfi Pɑrisɑpɑsi kɑpi yɛ kɔ n sée rɛ Yusituusi. Ulírũ yɛɛ Mɑtiyɑsi. 
Api pɛɛ kɛyómɛ yɑ́ɑ́si rɛ: Upíimɑ, pɔ́ɔ pisoi nnɛ́í sifɑ kɛ́mɛɛ nyu. A pisoi pitɛ́ mpí kɛcɔpɛ uyɛ̃ kɑɑ n wɛ́ɛ rɔ́ nyísɛ 
ukɛ́ pɑnsɛ utumɛ uu Yutɑsi pikɛi yɔ́su uú nɔŋ́. Yutɑsi yɛ ukɛtonɛ riyɑ́lɛ uu sĩ́ kei kɑí nɛ u rinsɑ́ rɛ ukɛ́ tonɛ. 
Kei kɑpi pɛɛ pɑŋɑi ɑi Mɑtiyɑsi tini, uu pɛɛ Yeesu pitumɛ kɛfi nɛ usɛ pɛ̃ kɛcɑ́ɑ́ kpɑ́pɔ.
