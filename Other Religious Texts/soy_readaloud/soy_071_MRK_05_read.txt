Mɑriki.
5.
Lɛ̃ mɛmɑ́ɑ́, ɑpi Kɑlilee kupiyɛ tɛ́ŋ́ ɑpi Kerɑsɑɑ kɛtẽ loni. 
Kɛ Yeesu uu kúninɔi-i rincúlɑmɛ, uú nɛ usoi unyinɛ yɛɛ ɑkpíí-mɛ̃ n léeri sɑ́nɛ. Kunírilɑ̃ kunyinɛ kpɛɛ yɛ utisi uyɛ̃ m pɛkɛsi. 
Akpíí-i kuu yɛ n sói. Usoi uyɛ̃ úu kupɑhɑɑ we. Ḿpɑ́ ɑkpɑnii, ɑ́pi yɛ fe pikɛ́ u pɑɑsi. 
Mɛpehẽ kulúi kɑpi yɛ uɑnɑ nɛ nweni hɔpi ɑpi uɑnipɛ nɛ ɑkpɑnii pɑɑsi, ɑmɑ́ u yɛ ipehe iyɛ̃ nnɛ́í túúnlɛ. Úkɑ nnɑŋɛ ńn pɛɛ tulɑɑlɛ̃ nkɛ́ nɛ utisi uyɛ̃ tĩ. 
Kɛtúŋɛ́ nɛ kɛsinɛ kuu yɛ ɑkpíí-i nɛ ɑkúú kɛcɑ́ɑ́ n kɔɔnii un pupukɛɛ, un umɛcirɛ itói wɑi nɛ ɑpɑrɛ. 
Uu Yeesu kɛtɑɑ-pɔ yɛnu uu pɛɛ wurupɔ uú hɑ ukɛyu-i wúlɑ 
uu pupɛi rɛ: Uléécɑɑ yɛɛ ḿpɑ́ yo kɛcɑ́ɑ́ n tṹ Kɛpipi Yeesu, tɔ́ nɛ mpɔ̃́ nɛ ń-ye? Nɛ nɛ pɔ́ ténilɛ, nɛ Uléécɑɑ rinyiri, kɑpɛ íwɛ nɛ́ wɑ. 
Yeesu yɛ kunírilɑ̃ kpɛ̃ tinɔ́ɔ hɛ rɛ kukɛ́ utisi uyɛ̃ kɛ́mɛɛ le. Lɛ̃ nnyɑ kɑku lɛ̃ mɑ́ikɛɛ. 
Yeesu uu kɔ ku pisɛ rɛ: Íye kɑpi yɛ pɔ́ sée? Aku rinɔ́ɔ u yɔ́su rɛ: Pi yɛ nɛ́ sée rɛ kulúi, likumúŋɛ́ rɛ tɔ kulúi welɛ. 
Akú nɛ Yeesu téni rɛ úu kɑpɛ ɑ lɑkɑsɛ ɑkɛ́ kuyu kɛtɔ rilɔ́ɔ́. 
Kúfɔŋlee píimɑ kunyinɛ yɛ pɛɛ kei rikúuésɑɑ-i túkurilɛ. 
Aníri ɑɑ pɛɛ́ nɛ Yeesu téni rɛ: A ncée rɔ́ hɛ tɔkɛ́ ɑ́fɔŋ nnyɛ́ lõ. 
Yeesu uu ncée hɛ. Aníri ɑɑ utisi uyɛ-i léeri ɑɑ ɑ́fɔŋ loni. Kúfɔŋlee kpɛɛ tu yɑrɛ ɑ́fɔŋ ɑ́kotokú ɑtɛ́ (2.000). Áfɔŋ nyɛ̃ nnɛ́í ɑɑ wuru ɑɑ́ nɛ rikúú cɛ́pi ɑɑ́ hɑ nɛ mínimɑɑ-i kpúupɔ. 
Pisoi pɛɛ pɛɛ ɑ n séni ɑpi wuru ɑpí hɑ nɛ nsímɛ́ kuyu nɛ siyupi kɔ́ɔ́nú. Pisoi ɑpi sĩ́ pikɛ́ lɛlɛɛ n wɑ yɛ̃. 
Api Yeesu-i sĩ́ ɑpi utisi uyɛɛ pɛɛ ɑníri kulúi n héesi yɛnu un tũ nɛ ukɛyu cɑ́kɑ́-cɑ́kɑ́ un kɔ lɑpɑɑlɛ̃ ńsɔnɛ, ɑi iwɑmɛ pi wɑi. 
Pɛ̃ nnɛ́í inipɛɛ-i kɑi lɛ̃ n wɑpisi ɑpi lɛ̃ kɛ ɑníri ɑɑ́ nɛ utisi uyɛ-i ń le kɛɛnkɛɛ nɛ lɛ̃ kɑi ɑ́fɔŋ n wɑ. 
Api pɛɛ Yeesu suúlusɛ rɛ ukɛ́ pikɛteni-i le. 
Kɛ Yeesu uu kúninɔi-i n loni, utisi uyɛ̃ uu u suúlusɛ rɛ u yɛ́ u ritiki. 
Yeesu úu ncée u hɛ, ɑmɑ́ u rinɔ́ɔ u hɛ rɛ: A kui ɑɑ pipɔ́kɔ́ lɛɛpɔ. Aɑ lɛ̃ kɛ Upíimɑ uu íwɛ pɔ́ n tẽ́ uu pɔ́ wɑi pi kɛɛnkɛɛ. 
Utisi uu mɛsei pɛlɛ uú hɑ nɛ nsímɛ́ mmɛ̃ kɔ́ɔ́nú kuwɛ́lɛ́ kpɛ-mɛ̃ kɑpi yɛ n sée rɛ Ayu Kɛfi kɛtẽ. Pɛ̃ nnɛ́í pɛɛ yɛ n kṍ, nnɔ́ɔ kɑi yɛ pi yi. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu uú nɛ kúninɔi míni ritimɛ ricɔ tɛ́ŋ́. Pisoi kulúi ɑpi kɔ kei lɛ̃ u n kɑ́lɑɑlɛ̃. 
Kei kɛ Pisuifi kuyomɛyɑ́hɑɑlee uwɛ́ɛ́sɛ unyinɛ kɑpi yɛ n sée rɛ Yɑirusi uu Yeesu yɛ̃ uu uɑnɑ mɛtene wulɑ. 
Uú nɛ Yeesu mɛyɑ́nsei téni, uu rɛ: Kɛnɛ́kpérepi yɛ kɛkpɔ tulɛ, ɑ kɑm, ɑɑ ɑnípɛ kɛkɛcɑ́ɑ́ lɑ́ɑ likɛ́ kɛ yoriyɛ ɑ́kɛ kɑpɛ kpu. 
Yeesu uú nɛ nsẽ́ u pɛ́nɛ. Pisoi kulúi pin u nyikii pin nɛ u tikilɛ̃. 
Kumúŋɛ́ kpɛ-i, unɔ́si unyinɛ kɛ mɛ́nyɛ ɑmɛ n foho hɑ́i iŋmɛ̃ kɛfi nɛ itɛ́-mɛ un kei we. 
Pipɔisɛ mɛyɑ̃ kuu rikɑ ɑi u pelu, usiwóó nnɛ́í ɑsi kɔ itói iyɛ-i tɛnɛ, úu kɔ pɛɛ peí. Amɑ́ itói in nɑ́kíísɛlɛ̃. 
Kuu nní Yeesu nsímɛ́ n kómɑɑlɛ̃, uu pɛɛ riwúí tɛ-i lompɔ uu kɛpirɛ-mɛ̃ Yeesu ilũ cɑ. 
Likumúŋɛ́ rɛ u ukɛfɑ-i pisímɛ́ mɑsí rɛ: Ḿpɑ́ Yeesu ilũ ŋmɑnɛ kɑm fe ɑḿ cɑ, nɛ́ peí. 
Kuu lɛ̃ tińcɑ, mɛ́nyɛ mɛɛ pɛɛ n foho ɑmɛ mɛsɛ nɛ mɛsɛ nyɛrɛ, uu pélu. 
Yeesu uu cérupɔ rɛ unnɑŋɛ yɛ pikɛi yisilɛ. Uu riwúí tɛ̃ kɛcɔpɛ pɑnsɛ uu pisɛ rɛ: Wóo inɛ́lũ ticɑ? 
Upipirɛtiki ɑpi rinɔ́ɔ u yɔ́su rɛ: Pɔ nyɑ́nilɛ rɛ pisoi yɛ pɔ́ nyíkiilɛ̃, ɑɑ kɔ pɛɛ́ m písɛi rɛ wóo pɔ́ ricɑɑ? 
Yeesu uu uilúkɛ́ pɑí ukɛ́ uyɛɛ u rińcɑ yɛ̃. 
Unɔ́si uyɛ̃ uu pitɛ́rii kɑ́pɑ́ɑ́ nɛ iwɑmɛ mɛpehẽ. U lɛlɛɛ n wɑ nyulɛ. Uu lɛ̃ nnyɑ Yeesu ɑnɑ mɛtene wulɑ uu ɑsei nnɛ́í u símisi. 
Amɑ́ Yeesu pɔ́ɔ u mɑɑ rɛ: Unɛ́kpére, mpɔ́fɑtɛnɛ yɛ pɔ́ yóriyɛlɛ. A n ŋme nɛ nkíŋniŋɛ ipɔ́tói ikɛ́ tɛnɛ. 
Úu kɑhɑnɛ nkpɑ́ni nsímɛ́ n tɛnɛ kɛ pisoi pɛɛ Pisuifi kuyomɛyɑ́hɑɑlee uwɛ́ɛ́sɛ uyɛ̃ kɛyɔ-i n léeri ɑpí mɑɑ rɛ: Kɛpɔ́wɑ̃́ yɛ nkpɔ mɑsilɛ. Yo nnyɑ kɑɑ nkpɑ́ni Ucélɑɑ mɛ́woo cɔ́ŋlɛ̃? 
Amɑ́ Yeesu úu nkpɑ́ni piɑnɔɔ nyɛ̃ kutu ricɔ, uu kuyómɛyɑ́hɑɑlee uwɛ́ɛ́sɛ mɑɑ rɛ: Kɛpɔ́wuu ɑ́kɛ kɑpɛ tɔ, mpɔ́kɔ́ yɛɛ rɛ ɑ nɛ kɛfɑ tɛnɛ. 
Lɛ̃ mɛmɑ́ɑ́, úu úkɑ ncée hɛ rɛ ukɛ́ u sɑrũ, insɑ́ Piyɛɛ nɛ Yɑkupu nɛ Yɑkupu uwɑ̃ Yohɑni. 
Kɑpi kuyómɛyɑ́hɑɑlee uwɛ́ɛ́sɛ uyɛ̃ kɛyɔ n tuipɔ, iwɔ́ɔ kɛ Yeesu uu kṍ. Pisoi yɛ pɛɛ ikpɔtu wɔ́riilɛ pin pupukɛɛ. 
Uu lompɔ uu pi mɑɑ rɛ: Yo nnyɑ kɑni iwɔ́ɔ likirɛlɛ̃ nɔn kɔ téni? Kɛkpérepi kɛ̃ ɑ́kɛ kpu ɑmɑ́ kɛ lɔnilɛ. 
Api pɛɛ Yeesu n sɛ́nnyi. Yeesu uu pɛɛ lɛ̃ mɛmɑ́ɑ́ pi lesɛri. U nɛ kɛwɑ̃́ kɛ̃ usɑɑ nɛ kɛuni nɛ upipirɛtiki pitɑɑni pɛ̃ kuú pɛɛ n tɛ́si ɑpi kei kɛ kɛwɑ̃́ kɛ̃ ɑkɛ́ ń we lompɔ. 
Uu kɛwɑ̃́ kunipɛ tini uu kɛ mɑɑ rɛ: Tɑlitɑ kum, li ɑsei rɛ kɛkpérepi, ɑ yisi, nɛ́ɛ rinɔ́ɔ pɔ́ hɛ. 
Kɛkpérepi ɑkɛ mɛsɛ nɛ mɛsɛ yisi ɑkɛ nsẽ́ kɑ́pɑ́ɑ́. Aŋmɛ̃ kɛfi nɛ ɑtɛ́ kɑkɛ yɛ m mɑ́. Kɑpi mɛwɑi mɛ̃ lɛ̃ n yɛ̃́, ɑi ḿpɑ́ úye nnɔ́ɔ yipu. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ pi kúrúrúsɛ rɛ ɑ́pi kɑpɛ tíyɛsɛ unyinɛ ukɛ́ céri. Uu kɔ pimɑ́ɑ pi kpɑ́ rɛ pikɛ́ kɛkpérepi kɛ̃ ilukɛ hɛ.
