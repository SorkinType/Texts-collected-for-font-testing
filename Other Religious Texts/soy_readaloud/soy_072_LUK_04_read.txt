4.
Yeesu uu Yuritɛɛ nkoi-mɛ̃ n leeri Nfɑ́ɑsɔnɛ nn u yipɑɑlɛ̃. Nfɑ́ɑsɔnɛ nn u kpísi nń nɛ kucɛsi kóimɑ-mɛ̃ sĩ. 
Setɑni uú hɑ kei u peikɛɛ siyɑ́ɑ ɑfɛɛnɑ. Úu siyɑ́ɑ sɛ̃ kɛ́mɛɛ líkɑ li. Lɛ̃ mɛmɑ́ɑ́ nkṹ nn u wɑi. 
Setɑni uu pɛɛ u mɑɑ rɛ: Pɔn Uléécɑɑ Kɛpipi, ɑ rinɔ́ɔ ripɑrɛ ntí hɛ rikɛ́ ripɔ́lɔ́ kukpɔ́nɔ́. 
Yeesu uu rinɔ́ɔ ku yɔ́su rɛ: Li rítɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Ái ilukɛ ŋmɑnɛ yɛɛ yɛ usoi nfɑ́ɑ hɛ. 
Setɑni uú nɛ kɛlõ cɑ́cɑ́ u sĩ, uu mɛyéne mɛsɛ kɛtẽ nnɛ́í ɑyɔɔpiyu u nyísɛntɛ 
uu u mɑɑ rɛ: Nɛ́ ɑyɔ́ɔpiyu nnyɛ́ nnɑŋɛ nɛ ɑmmulɛ pɔ́ hɛ. Anɛ́nípɛ-i kɑpi lɛ̃ nnɛ́í wɑ, nɛ́ fe ɑḿ uyɛ̃ kɑm ń lɑ hɛ. 
Pɔn nɛ́ rinwulɑ ɑɑ nɛ́ yɑ́ɑ́si, lɛ̃ nnɛ́í yɛ́ lipɔ́kɔ́ wɑ. 
Yeesu uu rinɔ́ɔ ku yɔ́su rɛ: Li rítɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Upíimɑ mɛcirɛ kóló kɑí nɛ sɑ́ rɛ usoi ukɛ́ riwulɑ ukɛ́ yɑ́ɑ́si. 
Kuníri ɑku lɛ̃ mɛmɑ́ɑ́ u kpɑ́ú nɛ Yerusɑlɛm-pɔ, ɑkú hɑ Uléécɑɑ kɛyɔ kɛcɑ́ɑ́ u tɔsi, ɑku u mɑɑ rɛ: Pɔn Uléécɑɑ Kɛpipi, ɑ kɛcɑ́ɑ́ nkɛ́-i mɛpɔ́círɛ risɑ́pɔ. 
Likumúŋɛ́ rɛ li rítɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Uléécɑɑ yɛ́ upitumɛ rinɔ́ɔ hɛ 
Api pɔ́ yéu kupɔ́nɑ ɑ́ku kɑpɛ nɛ ripɑrɛ rinyinɛ rinkpɔ́u nnyɑ. 
Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Li rítɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Kɑpɛ Upíimɑ Upɔ́léécɑɑ peikɛɛ pɔkɛ́ ukɛtɔ yɛ̃. 
Kɛ Setɑni uu mɛ́wee mpehẽ mpehẽ nɛ Yeesu m peikɛɛ, uu lɛ̃ mɛmɑ́ɑ́ nɛ kɛtɑɑ u wɑi un nɛ kɛpolɛ kɛcɔ mɛ̃́. 
Yeesu uu Kɑlilee kɛtẽ pɛlɛ. Nfɑ́ɑsɔnɛ nnɑŋɛ nn ukɛcɑ́ɑ́ we. Urinyiri ɑri kɛtẽ kɛ̃ nnɛ́í kɔ́ɔ́nú. 
Uu ɑyómɛyɑ́hɑɑlee-i n céési, ḿpɑ́ úye un ríyu u wɑisɛ. 
Yeesu uu Nɑsɑrɛti-i kɑpí pɛɛ u rimmúlú uú hɑ nɛ pɑpisi pɛlɛ. Uu kɛwéntɛyɑɑ tuŋɛ́ kuyómɛyɑ́hɑɑlee-i loni yɑrɛ kuu yɛ n lõ mɛcɔ. Uu yisi rɛ ukɛ́ Nléécɑɑsimɛ́ kɛ́ɛ̃. 
Api ɑntepu Esɑyi ritɛlɛ́ u múísɛ. Uu ripínnɛ uu yɛ́nu kei kɑi n wɔ́lɑɑlɛ̃ tɛ: 
Upíimɑ Nfɑɑsɔnɛ yɛ kɛnɛ́mɛɛ welɛ. U nɛ́ wɛ́ɛlɛ rɛ kɛ́ pɛpɛɛ nɛ mpɔ́ɔcɑɑi n tṹ ncɔ́ŋ pi lesɛ kɛ́ kɔ píwɛkɔɔ́ Nsímɛ́ Kɛcirɛ riyóó, kɛ́ pikpɑniikɔ́ símisi rɛ pi yɛ́ ɑkpɑnii kɛ́mɛɛ pi lesɛ, ɑm kɔ pinyíyɛ símisi rɛ pi yɛ́ kɑm ɑpi yɛ yɛnu. Am pɛ̃ kɑpi íwɛ n wɑi ŋmɛ́hɛrɛɛsɛ 
tɛ kɛ́ kuŋmɛ̃ kpɛ̃ kɛ Upíimɑ uu uipɛɛlɛ́ɛ n nyísɛnɛ nkɔ́ símisi. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ rítɛlɛ́ pílɛsi uu ukɛikɔ́ ri múísɛ uú tonɛ. Pisoi pɛ̃ nnɛ́í pɛɛ kuyómɛyɑ́hɑɑlee-i ń we ɑpi inípɛɛ u wii. 
Uu pɛɛ pi mɑɑ rɛ: Nléécɑɑsimɛ́ mmú kɑni nní n kṍ, nɛni kɛ rinɔ́ɔ tɛ̃ ɑri li. 
Anɔ́ɔ nyɛ̃ kɛ Yeesu uu m mɑ́ikɛɛ ɑɑ ḿpɑ́ úye loni, ɑi pírí pi wɑi kuú nɛ nsímɛ́ sɔnɛ mmú n símisi, ɑpi yɛ pɛɛ mɑɑ rɛ: In nɛ Yosɛfi Kɛpipii? 
Yeesu uu pi mɑɑ rɛ: Nɛ nyu rɛ nɔ́ kɛnyɑ́rũ kɛɛ n tee rɛ “Upɔisɛ, ɑ mɛpɔ́círɛ pɔisɛ” nɛ́ mɑlɛ. Nɔ́ kɔ nɛ́ mɑɑ rɛ: Tɔ konlɛ lɛ̃ nɛ lɛ̃ kɑɑ Kɑpɛrinɑwum-pɔ n wɑpisi. A kɔ limɛcɔ kupɔ́yu-i wɑpisi. 
Uu lɛ̃ mɛmɑ́ɑ́ kpɑ́ rɛ: Asei kɑm nɔ́ símisinɛ: Ápi yɛ ɑntepu úkɑ kusɑ́nɛ yɔsí ukuyu cirɛ-i. 
Asei kɑm nɔ́ símisi, pikúmɑnnɔsi yɛ pɛɛ Elii kumúŋɛ́ piyɛlɛ Isirɑyɛɛli-i, kɛ nkpei nn n wɑ ɑŋmɛ̃ ɑtɑɑni nɛ kuwɛ́lɛ́ kumúŋɛ́, nkṹ píimɑ nn kuyu nnɛ́í loni. 
Uléécɑɑ úu pikúmɑnnɔsi pɛ̃ unyinɛ kɛ́mɛɛ Elii tũ, insɑ́ uyɛɛ Sitɔɔ kɛtẽ kɛ́mɛɛ Sɑrɛpitɑ-pɔ ń we. 
Piyɔ̃́ yɛ kɔ pɛɛ kulúi welɛ Isirɑyɛɛli-i ɑntepu Elisee kumúŋɛ́, ḿpɑ́ nɛ lɛ̃, piukɑ úu wolɛ ukɛ́ fũ, insɑ́ Sirii ukɔ́ Nɑɑmɑ mɛcirɛ. 
Kɛ pɛ̃ nnɛ́í pɛɛ kuyómɛyɑ́hɑɑlee-i ń we ɑpi lɛ̃ n kṍ, kuwɔ́i ɑku pi wɑi. 
Api yisi ɑpi u tɔrɔi ɑpí nɛ kuyu lelu, ɑpí nɛ rikúú tɛ̃ kɛcɑ́ɑ́ kɛ pikuyu ɑku m mɔ́mɑɑlɛ̃ tɑɑ́ rɛ pikɛ́ hɑ kei-pɔ u rilɑ́umɛ. 
Amɑ́ uu nnɑ́nnɑ́ri wɑi uu pikɛpɑ́ɑ́pɑ́ɑ́ tiki uu tɔ́mpɔ. 
Yeesu uu pɛɛ Kɑlilee kɛtẽ kɛ́mɛɛ Kɑpɛrinɑwum-pɔ sĩ. Uu kɛwéntɛyɑɑ kɛtúŋɛ́ pisoi Nléécɑɑsimɛ́ céési. 
Umɛcélɑɑ yɛ pɛɛ pírí pi welɛ, liriyíkí rɛ u céésilɛ nɛ ńnɑŋɛ. 
Kuyómɛyɑ́hɑɑlee kɛ́mɛɛ, unírihélɑɑ unyinɛ un we. Uu pipupukɛɛ kɑ́pɑ́ɑ́. 
Ah! Ń-ye kɑɑ́ nɛ rɔ́ wɛ́ɛ́si, Nɑsɑrɛti ukɔ́ Yeesu? Pɔ kɑlɛ rɛ pɔkɛ́ rɔ́ nɑ̃́ɑ? Nɛ pɔ́ nyulɛ rɛ Uléécɑɑ yɛ pɔ́ wɛ́ɛlɛ uu iyɑɑ yekei. 
Yeesu uú nɛ ku cɛ́si uu ku mɑɑ rɛ: A risɛ́ɛ́ ɑɑ usoi uyɛ̃ kɛ́mɛɛ́ lelu. Kuníri ɑku pisoi kɛcɔpɛ kɛtẽ u tɑpu ɑku pɛɛ ukɛmɛɛ́ lelu ɑ́ku líkɑ u wɑ. 
Ai pinnɛ́í nnɔ́ɔ yipu ɑpi pimɛcɔpɛcirɛ m mɑ́ikɛɛnɛ rɛ: kpí! Nsímɛ́ ń-ye mpuri yɛ mmú? Usoi nkó yɛ ɑníri rinɔ́ɔ hɛlɛ nɛ ńnɑŋɛ ɑɑ́ lelu. 
Api Yeesu rinyiri nɛ kɛtẽ kɛ̃ nnɛ́í kɔ́ɔ́nú. 
Yeesu uu Pisuifi kuyomɛyɑ́hɑɑlee-i léeri uu Simɔɔ kɛyɔ sĩ, Simɔɔ ulɔ́hɔ́ni ipiŋɛ in mɛyɑ̃́ tolú lin u pɛkɛsi. Apí nɛ Yeesu téni rɛ ukɛ́ u pɔisɛ. 
Uu ukɛcɑ́ɑ́ pɛ, uu kupiŋɛtoi rinɔ́ɔ hɛ nɛ kuwɔ́i, kupiŋɛtoi ɑku u kpɑ̃́ii, uu mɛsɛ nɛ mɛsɛ yisi uu pikɛcɑ́ɑ́ pɑí. 
Ituŋɛ mɛlólɛ́-mɑɑ́, pɛ̃ nnɛ́í pɛɛ itói ncɔpuri pitoikɔ́ m mɑ́ ɑpí nɛ Yeesu pi lɛɛpɔ. Uu ḿpɑ́ úye kɛcɑ́ɑ́ ɑnípɛ lɑɑ uu pi péisɛ. 
Aníri ɑɑ kɔ pisoi mɛyɑ̃ kɛ́mɛɛ́ lelu ɑn pupukɛɛ rɛ: Pɔ́ɔ Uléécɑɑ Kɛpipi. Amɑ́ Yeesu uú nɛ ɑ cɛ́si uu nsímɛ́ nɛ ɑ yulu, ɑníri ricuruu yɛ nyu rɛ Yeesu kɛ Uléécɑɑ uu wɛ́ɛ. 
Kɑi n weesi, Yeesu uu kuyu lelu uu kei kɛ úkɑ úu ń we sĩ. Tisoiwuí ɑri piwɛ́lɑɑ u loni. Kɑpi u n lɛɛpɔ, ɑpí n lɑ yɑrɛ ukɛ́ pikɛkúrí n we, úu kɑpɛ pi riyɑ́. 
Amɑ́ Yeesu uu pi mɑɑ rɛ: Li pisɛ rɛ kɛ́ Uléécɑɑ iyɔɔpi Nsímɛ́ Kɛcirɛ ɑyu ɑcɔ símisi. Lɛ̃ nnyɑ kɛ Uléécɑɑ uu nɛ́ tummɛ. 
Uu Kɑlilee Pisuifi ɑyomɛyɑ́hɑɑlee-i Nléécɑɑsimɛ́ n símisi.
