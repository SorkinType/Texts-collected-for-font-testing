Yohɑni.
9.
Kɛ Yeesu uu ncée-i ń we un tɔ́su, uu utisi unyinɛ kɑpi m mɑ́ri unyíyɛ yɛnu. 
Upipirɛtiki ɑpi u pisɛ rɛ: Ucélɑɑ, wóo kɛcirɛ yɛɛ ɑkópɛ wɑ nnyɑ kɑpí nɛ utisi nkó mɑ́ri unyíyɛ? Uricuruu ɑkópɛ nnyɑ nɛ́ɛ pɛpɛɛ u m mɑ́ri ɑkópɛ nnyɑ? 
Yeesu uu pi pɛsɛ rɛ: Ái uɑkópɛ nnyɑ, ɑ́i kɔ pɛpɛɛ u m mɑ́ri ɑkópɛ nnyɑ. Lɛ̃ kɑi yɛ́ n wɑ Uléécɑɑ nnɑŋɛ nkɛ́ nɛ ukɛcɑ́ɑ́ ń we lin nyɑ́ni nnyɑ kuu unyíyɛ. 
Kɛ ituŋɛ ii nní ń we, li pisɛ rɛ tɔkɛ́ uyɛɛ nɛ́ n tummɛ pikɛi wɑ. Kɛsinɛ yɛ wɑinɛ, úkɑ úu yɛ́ pɛɛ fe ukɛ́ kɛisi. 
Kumúŋɛ́ nkú-i kɑm nní kɛtẽ kɛcɑ́ɑ́ ń we, nɛ́ɛ kɛtẽ kɛcɑ́ɑ́ mɛtɛ́í. 
Kuu ɑnɔ́ɔ nyɛ̃ pimɑ́ɑ ḿ mɑsí, uu kɛteni-i mɛtɑ tukɛ uú nɛ sipɛ́rɛ́pi wɑi, uu unyíyɛ uyɛ̃ inipɛɛ pɔ́rii, 
uu pɛɛ u mɑɑ rɛ: A n ŋme ɑ hɑ Silowee kupɔrɛ-pɔ lihí. Silowee ɑsei rɛ Utumɛ. Uu mɛsei yisi uu sĩ́ uú hɑ líhu. Kuú hɑ m pɛɛmɛ, uu pɛɛ́ n nyɑ́ni. 
Upipɛrɛcɔ nɛ pɛpɛɛ n nyu rɛ u pɛɛ wéleilɛ, ɑpi pɛɛ́ m písɛinɛ rɛ: Ái nkó yɛɛ pɛɛ tṹ un wéleii? 
Picɔ ɑpi yɛ rɛ: Uyɛɛ lo. Picɔ ɑpi rɛ ɑɑí, mɛnyínɛ kɑpi wenɛ. Uricuruu uu pɛɛ rɛ: Nɛ̃́ ŋmɑɑ lo. 
Api pɛɛ u pisɛ rɛ: Íye kɑi wɑ kɛ ipɔ́nípɛɛ ií nɛ nkpéni nyɑ́ni? 
Uu pi pɛsɛ rɛ: Utisi uyɛ̃ kɑpi yɛ n sée rɛ Yeesu yɛɛ sipɛ́rɛ́pi wɑ, uu inípɛɛ nɛ́ pɔ́rii uu pɛɛ nɛ́ mɑɑ rɛ: “A n ŋme ɑ hɑ Silowee kupɔrɛ-pɔ lihí.” Am sĩ́ ɑḿ hɑ líhu, ɑm pɛɛ́ n nyɑ́ni. 
Api u pisɛ rɛ: Yei kɛ utisi uyɛ̃ uú we? Uu pi pɛsɛ rɛ: Ám nyu. 
Api pɛɛ utisi uyɛɛ pɛɛ mɛkɛɛ n nyíinlɛ̃ nɛ Pifɑrisi kɛ́mɛɛ sĩ́. 
Kɛyɑ́ɑ kɛ̃ kɛ Yeesu uu sipɛ́rɛ́pi n wɑ uú nɛ inípɛɛ u wúkulɛ yɛ pɛɛ kɛwéntɛyɑɑ lɛ. 
Pifɑrisi pɔ́ɔ kɔ pipísɛi u kpɑ́ rɛ íye kɑi wɑ kuú nɛ nkpéni nyɑ́ni? Uu pi mɑɑ rɛ: Sipɛ́rɛ́pi kuu inípɛɛ nɛ́ pɔ́rii, ɑm sĩ́ ɑḿ hɑ líhu, ɑm pɛɛ nkpéni n nyɑ́ni. 
Lɛ̃ nnyɑ kɛ pikɛcɔpɛ pinyinɛ ɑpí mɑɑ rɛ: Utisi uyɛɛ mɛwɑi mmɛ́ n wɑ úu yɛ́ fe ukɛ́ Uléécɑɑ-i n leeri. Kúu nní kɛwéntɛyɑɑ isé rintíki nnyɑ. Picɔ ɑpi rɛ: Íye kɛ ukópɛkɔɔ́ uu yɛ́ fe uu mɛwɑisɑŋɑ mmɛ́ mɛcɔ wɑi? Kei kɑpi pɛɛ ɑnɔ́ɔ ɑkɔ́ ɑkɔ́ nɛ rikénɛ. 
Pifɑrisi ɑpi kɔ unyíyɛ uyɛ̃ pipisɛ kpɑ́ rɛ: Íye kɛ pɔ́ɔ utisi uyɛɛ inípɛɛ pɔ́ m pɔisɛ kɛcɑ́ɑ́ músu? Uu rɛ: Antepu lo. 
Ḿpɑ́ nɛ lɛ̃, Pisuifi piwɛ́ɛ́sɛ ɑ́pi pɛɛ lɑ pikɛ́ kṍ tɛ u pɛɛ unyíyɛ lɛ kuu kɔ nkpéni nyɑ́ni. Api pɛɛ upikɔ́ séi 
ɑpi pi pisɛ rɛ: Nɔ́unɛ́ŋmɑ́nɛ uyɛ̃ kɑni n tee rɛ nɔ u mɑrilɛ unyíyɛ yɛ mɛsei nkóo? Íye kɑi wɑ kuu kɔ nɛ nkpéni nyɑ́ni? 
Upikɔ́ ɑpi pi pɛsɛ rɛ: Tɔ u nyulɛ rɛ u kɛrɔ́pipi. Tɔ kɔ u mɑrilɛ unyíyɛ. 
Amɑ́ ɑ́ri nyu lɛ̃ kɑi n wɑ kuú nɛ nkpéni n nyɑ́ni, ɑ́ri kɔ nyu wóo u pɔisɛ. Ani u pisɛ, úu kɛwɑ̃́, u yɛ́ fe uu uriyu kɛcɑ́ɑ́ nkɔ́ símisi. 
Pisuifi piwɛ́ɛ́sɛ kɑpi wuru nnyɑ kɑpi lɛ̃ mɑ. Li we rɛ Pisuifi piwɛ́ɛ́sɛ yɛ pɛɛ kómɛinɛlɛ rɛ úye un m mɑɑ rɛ Yeesu kɛ Uléécɑɑ uu wɛ́ɛ, pikɛ́ liute kuyómɛyɑ́hɑɑlee-i lesɛ. 
Lɛ̃ nnyɑ kɛ upikɔ́ ɑpí mɑɑ rɛ: Úu kɛwɑ̃́, ɑni uricuruu pisɛ! 
Pifɑrisi ɑpi kɔ uyɛ̃ kɑpi m mɑ́ri unyíyɛ pipísɛi kpɑ́ ɑpi u mɑɑ rɛ: A ɑsei símisi likɛ́ Uléécɑɑ rinyiri ritɑɑ́sɛ. In tɔ́ɔ, tɔ nyulɛ ŋmɑɑ rɛ utisi uyɛɛ ukópɛkɔɔ́ lɛ. 
Uu pɛɛ pi pɛsɛ rɛ: Un ukópɛkɔɔ́, nɛ̃́ ɑ́m nyu. Amɑ́ likɛi lisɛ kɑm nyu rɛ nɛ pɛɛ unyíyɛ lɛ nɛn kɔ pɛɛ nkpéni nyɑ́ni. 
Api pɛɛ u pisɛ rɛ: Yo kuu pɔ́ wɑ? Íye kuú nɛ inípɛɛ pɔ́ wúkulɛ? 
Uu pi pɛsɛ rɛ: Nɛ pisímɛ́ nɔ́ mɑsilɛ ɑ́ni kutu nɛ́ ricɔ. Yo nnyɑ kɑní lɑ kɛ́ kɔ pisímɛ́ rikpɑ́ nɔkɛ́ kṍ? Nɛ́ɛ nɔ̃́ nɛ kɔ lɑlɛ nɔkɛ́ pɑnsɛ upipirɛtiki? 
Kei kɑpi ɑlɛ̃́ u rikɑ́pɑ́ɑ́ ɑpi rɛ: Pɔ́ɔ uyɛ̃ upirɛtiki. Amɑ́ tɔ̃́ nɛ Moisi pipirɛtiki lɛ. 
Tɔ nyulɛ rɛ Uléécɑɑ yɛ nɛ Moisi símisilɛ. Amɑ́ ɑ́ri nyu ricuruu mɛ́ye kɛ nkó pɔ́ɔ léeri. 
Unyíyɛ yɛɛ ḿ peí uu pi mɑɑ rɛ: Lɛlɛɛ pírí nɛ́ ń we yɛɛ rɛ ɑ́ni nyu mɛ́ye kuu léeri, uu kɔ pɛɛ tíyɛsɛ inɛ́nípɛɛ ií n nyɑ́ni. 
Tɔ nyulɛ rɛ Uléécɑɑ úu yɛ ukópɛkɔɔ́ nɛ kutu ricɔ, ɑmɑ́ nkó yɛɛ ríyu u n wɑisɛlɛ̃ un umɛlɑ wɑi kuu yɛ kutu ricɔ. 
Kɑí nɛ n wemɛ, ɑ́ri píkɑi kómɑɑlɛ̃ tɛ unyinɛ yɛ tíyɛsɛ usoi kɑpi m mɑ́ri unyíyɛ uu yɛ kɔ yɛnu. 
Utisi nkó unsɑ́ pɛɛ Uléécɑɑ kɛ́mɛɛ n leeri, úu yɛ́ pɛɛ fe ukɛ́ linyinɛ wɑ. 
Api u pɛsɛ rɛ: Hɑ́i kɑpí nɛ pɔ́ ḿ mɑ́ri kɑɑ ɑkópɛ kɛ́mɛɛ pɔ́lɔlɛ̃. Pɔ́ɔ yɛ́ pɛɛ nkpéni tɔ̃́ céési, nɛ́ɛ yo? Api pɛɛ kuyómɛyɑ́hɑɑlee-i u lesɛ. 
Yeesu uu kóm tɛ pi utisi uyɛ̃ lesɛ. Uú nɛ utisi uyɛ̃ sɑ́nɛ uu u pisɛ rɛ: Pɔ Usoi Kɛpipi nɛ kɛfɑ tɛnɛlɛ̃ɛ? 
Uu Yeesu pɛsɛ rɛ: Uyɛɛ úye, Upíimɑ? A nɛ́ símisi kɛ́ nɛ kɛfɑ u tɛnɛ. 
Yeesu uu u pɛsɛ rɛ: Uyɛ̃ kɑɑ nní nyɑ́ni, uyɛɛ nní nɛ pɔ́ símisi. 
Kei kuu pɛɛ rɛ: Nɛ nɛ kɛfɑ pɔ́ tɛnɛlɛ, Upíimɑ. Uu pɛɛ Yeesu kɛnyɑlɛ-i wúlɑ uu ríyu u wɑisɛ. 
Yeesu uu pɛɛ rɛ: Nɛ kɛtẽ kɛcɑ́ɑ́ kɑlɛ kɛ́ nɛ pisoi túhɑɑnɛ, pɛpɛɛ n nyíinlɛ̃ pikɛ́ n nyɑ́ni, pɛpɛɛ n nyɑ́ni pɔ́ɔkɛ́ nyíinu. 
Pifɑrisi pinyinɛ yɛ pɛɛ ukɛkúrí welɛ. Kɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, ɑpi u pisɛ rɛ: Yɑrɛ tɔ̃́ nɛ kɔ pinyíyɛ lɛ, nɛ́ɛ íye? 
Yeesu uu pi pɛsɛ rɛ: Nɔn pɛɛ pinyíyɛ, ɑ́ni yɛ́ pɛɛ ɑkópɛ m mɑ́. Amɑ́ nɔ yɛ mɑɑ rɛ nɔ nyɑ́nilɛ nnyɑ kɛ nɔ́ɑnɛ́kópɛ ɑɑ nɔ́kɛnɛ́cɑ́ɑ́ we.
