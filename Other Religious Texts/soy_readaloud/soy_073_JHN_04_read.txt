Yohɑni.
4.
Pifɑrisi ɑpi kóm tɛ Yeesu yɛ pipirɛtiki mɛyɑ̃́ yɛnlɛ un kɔ pisoi mɛyɑ̃́ míni wolɛ ɑi tɔ́su Yohɑni pikɔ́. 
Asei kɛcɑ́ɑ́, Yeesu ricuruu úu pɛɛ úkɑ míni wolɛ. Upipirɛtiki pɛɛ yɛ pɛɛ pisoi wolɛ. 
Kɛ Yeesu uu n kṍ tɛ lɛ̃ kɛ pisoi ɑpi símɑɑnkɛɛ, uu Yutee kɛteni-i yisi uu Kɑlilee kɛtẽ-mɛ̃ pɛlɛ. 
Uncee yɛ pɛɛ kɔ Sɑmɑrii kɛtẽ-mɛ̃ cɔlɛ nń nɛ kei lompɔ. 
Uu pɛɛ Sɑmɑrii kɛyupi kɛnyinɛ kɑpi yɛ n sée rɛ Sisɑɑ tulu. Ái kei nɛ kɛcɑrɛ kɛ̃ kɛ Yɑkupu uú pɛɛ uuŋmɑ́nɛ Yosɛfi n hɛ kɛtɑɑ we. 
Kei kɛ Yɑkupu kɛlukɑnkɑ ɑkɛ́ pɛɛ́ we. Kɛtúŋɛ́ kɛfi nɛ mɛtɛ́ kumúŋɛ́ kɛ Yeesu uu kei tuipɔ, nsẽ́ nn u pelɑɑlɛ̃, uu pɛɛ kɛlúkɑnkɑ kɛ̃ kɛkúrí tonɛ. 
Sɑmɑrii kɛtẽ unɔsi unyinɛ uu pínilṹ hɑpɔ. Yeesu uu pɛɛ u mɑɑ rɛ: A míni nɛ́ hɛ kɛ́ n ntí. 
Upipirɛtiki pɔ̃́ nɛ pɛɛ likumúŋɛ́ kpɛ-i lilukɛ-lukɛ piwɛ́lɑɑ hɑlɛ kuyu-pɔ. 
Sɑmɑrii kɛtẽ unɔsi uyɛ̃ uu pɛɛ Yeesu mɑɑ rɛ: Íye kɛ pɔ̃́ Usuifi, ɑɑ yɛ́ fe ɑɑ nɛ̃́ Sɑmɑrii ukɔ́ mɑɑ rɛ kɛ́ míni pɔ́ hɛ pɔkɛ́ n ntí? Pisuifi nɛ Sɑmɑrii pikɔ́ ɑ́pi yɛ pɛɛ rikɔnɛ. 
Yeesu uu u pɛsɛ rɛ: Pɔn pɛɛ lɛ̃ kɛ Uléécɑɑ uu yɛ n hɛ n nyu, pɔn kɔ pɛɛ uyɛɛ nní pɔ́ m pisɛ rɛ ɑ míni u hɛ n nyu, pɔ́ɔ yɛ pɛɛ u pisɛ rɛ ukɛ́ míni pɔ́ hɛ, uu pɛɛ míni mɛɛ yɛ nfɑ́ɑ n hɛ pɔ́ hɛ. 
Unɔ́si uyɛ̃ uu u mɑɑ rɛ: Upíimɑ, kɛlúkɑnkɑ nkɛ́ ɑ́kɛ kusɑ címú, ɑ́ɑ kɔ líkɑ mɑ́ pɔkɛ́ nɛ lṹ. Íye kɑɑ yɛ́ pɛɛ fe ɑɑ́ nɛ míni mɛɛ yɛ nfɑ́ɑ n hɛ yɛnu? 
Urɔ́sɑ́ɑyɑhɑ Yɑkupu yɛɛ kɛlúkɑnkɑ nkɛ́ nɛ rɔ́ tíyɛ. U nɛ usipipi nɛ uɑsɛɛlee yɛɛ pɛɛ míni mɛ̃ n ntílɛ. Yɑrɛ pɔ músu rɛ pɔ Yɑkupu felɛ nɛ́ɛ? 
Yeesu uu u pɛsɛ rɛ: Nnírɛ́ yɛ́ usoi nkó yɛɛ míni mmɛ́ ń ntí piwɑi rikpɑ́. 
Amɑ́ nnírɛ́ ńn píkɑi nkó kɛ nɛ́ɛ míni n hɛ uu mɛ níru piwɑi kpɑ́nɛ. Míni mɛ̃ kɑm u n hɛnɛ yɛ ukɛmɛɛ pɑnsɛnɛ ńnihɔi mɛɛ yɛ nfɑ́ɑ tɛnɛcirɛ́ n hɛ. 
Kei kɛ unɔ́si uyɛ̃ uu pɛɛ u mɑɑ rɛ: Upíimɑ, ɑ mɛpɔ́ni mɛ̃ nɛ́ hɛ nnírɛ ńn kɑpɛ yɛ kɔ piwɑi nɛ́ rikpɑ́. Pɛɛ́ uyɛ-i, ɑ́i pɛɛ pisɛ rɛ kɛ́ yɛ kɔ nté pínilṹ pɛɛmɛ. 
Yeesu uu pɛɛ u mɑɑ rɛ: A n ŋme ɑ hɑ upɔ́lɑ séemɛ ɑɑ́ weri. 
Unɔ́si uyɛ̃ uu u pɛsɛ rɛ: Ám ulɑ mɑ́. Yeesu uu pɛɛ u mɑɑ rɛ: Pɔ ɑlɑri mɑ́lɛ pɔkɛ́ rɛ ɑ́ɑ ulɑ mɑ́. 
Pilɑ pinupũ kɑɑ pisɔ́ɔ̃ mɑsí. Nɔ́ nɛ utisi uyɛɛ kɔ nkpéni ń we úu kɔ upɔ́lɑ. Pɔ ɑsei mɑ́lɛ. 
Unɔ́si uu pɛɛ u pɛsɛ rɛ: Upíimɑ, nɛ yɛ̃ tɛ pɔ ɑntepu lɛ. 
Tiyɔ́pɛ ntí kɛcɑ́ɑ́ kɛ tɔ̃́ pisɑɑyɑhɑ ɑpí pɛɛ Uléécɑɑ yɑ́ɑsi. Amɑ́ nɔ̃́ Pisuifi ɑni rɛ Yerusɑlɛm-i kɑi pisɛ rɛ pikɛ́ yɛ Uléécɑɑ yɑ́ɑ́si. 
Yeesu uu pɛɛ u mɑɑ rɛ: A kõ tɛ nú unɔ́si, li ituŋɛ inyinɛ-i sɔ́ntilɛ, ɑ́i pɛɛ pisɛ rɛ riyɔ́pɛ ntí-i nɛ́ɛ Yerusɑlɛm-pɔ kɑni yɛ́ɛ́ Sɑ́ɑ Uléécɑɑ yɑ́ɑ́si. 
Nɔ̃́ Sɑmɑrii pikɔ́, ɑ́ni Uléécɑɑ uyɛ̃ kɑni n yɑ́ɑ́si nyu. Tɔ̃́ Pisuifi yɛ Uléécɑɑ uyɛ̃ kɑri n yɑ́ɑ́si nyulɛ. Tɔ̃́ Pisuifi kɛ́mɛɛ kɛ píyulɑlɛ ɑpi léeri. 
Amɑ́ li ituŋɛ inyinɛ-i sɔ́ntilɛ, ticuruu li pikɑ́mɛ mɑsilɛ. Pɛpɛɛ Sɑ́ɑ Uléécɑɑ n yɑ́ɑ́si yɛ́ɛ́ nɛ piisoi nnɛ́í u yɑ́ɑ́si nɛ ɑsei. Pɛpɛɛ lɛ̃ u n yɑ́ɑ́si kuu wɛ́ɛ́si. 
Uléécɑɑ yɛ Nfɑ́ɑsɔnɛ lɛ. Lɛ̃ nnyɑ, li pisɛ rɛ pɛpɛɛ u n yɑ́ɑ́si pikɛ́ yɛ nɛ piisoi nnɛ́í ritiki ɑpí nɛ u yɑ́ɑ́si nɛ ɑsei. 
Unɔ́si uyɛ̃ uu pɛɛ Yeesu mɑɑ rɛ: Nɛ nyulɛ rɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ yɛ wemɛlɛ. Uyɛ̃ kɑpi yɛ sée rɛ Kirisi. Un píyei n kɑ, u yɛ́ linnɛ́í tɔ́ riyóó. 
Yeesu uu pɛɛ u mɑɑ rɛ: Nɛ́ɛ nɛ nní pɔ́ n símisi, nɛ́ɛ liute uyɛ̃. 
Yeesu pipirɛtiki ɑpi likumúŋɛ́ kpɛ-i tuipɔ, ɑi nnɔ́ɔ pi yipu rɛ pi u yɛ̃ un nɛ unɔ́si unyinɛ símisi nnyɑ. Amɑ́ piukɑ úu kɑhɑ ukɛ́ u pisɛ rɛ: Yo kɑɑ ukɛmɛɛ wɛ́ɛ́si nɛ́ɛ yo nnyɑ kɑɑ́ nɛ u símisi? 
Unɔ́si uyɛ̃ uu pɛɛ ukɛnintúhu kei yɑ́ uu kuyu pɛlɛ uú hɑ pisoi símisi rɛ: 
Ani kɑm ɑni usoi unyinɛ yɛ̃ yɛɛ mɛnɛ́wɑi nnɛ́í kɑm n wɑpisi nɛ́ n kɛ́ɛ́nkɛɛ. In n lɑ, ɑ́i nkpéni uyɛ̃ kɛ Uléécɑɑ uu wɛ́ɛ? 
Api pɛɛ kuyu-i léenti ɑpi símpɔ ɑpí hɑ Yeesu lɛɛpɔ. 
Upipirɛtiki ɑpi pɛɛ likumúŋɛ́ kpɛ-i u kpukɛisɛ rɛ: Ucélɑɑ, ɑ kɑm ɑ li. 
Kei kuu pɛɛ pi pɛsɛ rɛ: Ilukɛ inyinɛ kɑ́ni n nyu kɑm mɛlukɛ mɑ́. 
Kei kɛ upipirɛtiki ɑpi pɛɛ pimɛcɔpɛcirɛ pipísɛinɛ lõ tɛ: Nɛ́ɛ unyinɛ yɛ nɛ ilukɛ u kɑlɛ uu le? 
Yeesu uu pɛɛ pi mɑɑ rɛ: Inɛ́lukɛ yɛɛ rɛ kɛ́ uyɛɛ nɛ́ n tummɛ mɛlɑ n wɑi, kɛ́ kɔ pikɛi pɛ̃ kuu nɛ́ n forii n wɑi nɛ kɛ́tɔ-pɔ. 
Nɔ́rinɛ́cúruu nɔ́ɔ yɛ mɑɑ rɛ: Li tisɛ iwɑ́rɛ inɑ likɛ́ nɛ kúkpɑɑ tu. Kɛ́ nɔ́ símisi rɛ ɑni ńsɔnɛ ripɑí ɑni yɛ́nu rɛ ilukɛ yɛ sicɑrɛ-i pipele mɑsilɛ rɛ pikɛ́ i kpɑsi. 
Ticuruu ukpɑ́ɑ yɛ welɛ un uihɛ́ɛ́ yɔ́su, un kɔ ilukɛ kuu n kpɑ́si cɑ́pinɛntɛ rɛ ikɛ́ nfɑ́ɑ tɛnɛcirɛ́ yɛ̃́. Kei kɛ ulukɑɑ nɛ ukpɑ́ɑ ɑpɔɔ ɑɑ yɛ́ pɛɛ kɛsẽ́ n lɑ́ɑ́rú. 
Lɛ̃ kɛ́mɛɛ, mɛyu mmɛ́ yɛ ɑsei lɛ rɛ: Ucɔ yɛɛ yɛ lukɛsi, ucɔ uú kpɑsi! 
Nɛ nɔ́ tunlɛ rɛ ɑni hɑ kɛcɑrɛ kɛ-i kɑ́ni pikɛi píkɑ n wɑ kɛ́mɛɛ kpɑsi. Picɔ pɛɛ kei kɛisi, nɔ́ɔ nkpéni weri ɑni pipikɛi kulɑɑ n le. 
Sɑmɑrii pikɔ́ mɛyɑ̃ yɛ nɛ Yeesu kɛfɑ tɛnɛlɛ rɛ unɔ́si uyɛɛ mɑɑ rɛ: Lɛ̃ nnɛ́í kɑm n wɑ kuu nɛ́ símisi. 
Lɛ̃ nnyɑ kɑpi Yeesu-i hɑ́pɔ ɑpi u pisɛ rɛ ukɛ́ pikɛkúrí kɛtónɛ rikpɑ́. Uu ŋmurɛi uu pikɛkúrí tonɛ siyɑ́ɑ sitɛ́. 
Kɛ Yeesu uu uricuruu nɛ unnɔɔ nsímɛ́ pi n símisi nnyɑ, pisoi kulúi píimɑ picɔ ɑpi kɔ nɛ kɛfɑ u tɛnɛ. 
Api yɛ pɛɛ unɔ́si uyɛ̃ mɑɑ rɛ: Ái nsímɛ́ mmɛ̃ kɑɑ rɔ́ n símisi ŋmɑnɛ nnyɑ kɑrí nɛ kɛfɑ Yeesu tɛnɛ. Amɑ́ tɔ uricuruu konlɛ un símisi, ɑri céru rɛ u mɛsei pisoi nnɛ́í Uyulɑlɛ lɛ. Lɛ̃ nnyɑ kɑrí nɛ kɛfɑ u tɛnɛ. 
Kɛ Yeesu uu siyɑ́ɑ sitɛ́ sɛ̃ kei n tónɛ, uu yisi uu Kɑlilee kɛtẽ-mɛ̃ tɔ́mpɔ. 
Yeesu ricuruu yɛ pɛɛ́ mɑɑ rɛ: Ápi yɛ ɑntepu úkɑ ukɛtẽ cirɛ-i n wɑisɛlɛ̃. 
Amɑ́ kuu Kɑlilee kɛteni-i n tuipɔ, kei pikɔ́ yɛ kusɑ́nɛ u yɔsilɛ ńsɔnɛ, kɛ pɛ̃ ŋmɑɑ ɑpí pɛɛ Yerusɑlɛm-pɔ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ń hɑ ɑpi lɛ̃ nnɛ́í kuu n wɑ yɛnu nnyɑ. 
Yeesu uu pɛɛ Kɑlilee kɛtẽ-pɔ Kɑnɑɑ kuyu-i pɛlɛ. Kei kuú pɛɛ wɑ míni ɑmɛ pɑnsɛ pítɑ. Uyɔ́ɔpi usɔ́ɔ́cɑ wɛ́ɛ́sɛ unyinɛ yɛɛ pɛɛ Kɑpɛrinɑwum-pɔ we, uuŋmɑ́nɛ un pɔ́ɔ́lú. 
Kuu n kṍ tɛ Yeesu yɛ Yutee kɛtẽ-pɔ yisi uu Kɑlilee kɛteni-i hɑpɔ, uu ukɛmɛɛ sĩ́ uú hɑ u welu rɛ ukɛ́ Kɑpɛrinɑwum hɑpɔ ukɛ́ uuŋmɑ́nɛ yɛɛ nɛ nkpɔ n nɑ́ɑ́si pɔisɛ. 
Yeesu uu pɛɛ u pisɛ rɛ: Nɔnsɑ́ mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ n yɛ̃, ɑ́ni yɛ́ fe nɔkɛ́ nɛ kɛfɑ nɛ́ tɛnɛ, nɛ́ɛ yo. 
Uyɔ́ɔpi usɔ́ɔ́cɑ uyɛ̃ uu Yeesu pɛsɛ rɛ: Upíimɑ, ɑ kɛnɛ́yɔ hɑ unɛ́ŋmɑ́nɛ ukɛ́ kɛlenɛ kpu. 
Kei kɛ Yeesu uu pɛɛ u mɑɑ rɛ: A n ŋme ɑ kui, upɔ́ŋmɑ́nɛ yɛ nfɑ́ɑ mɑ́lɛ. Uu Yeesu nsímɛ́ mmɛ̃ ŋmurɛi, uu yisi uú kulu. 
Kuu nkúlɛcee n kpísi, upikɛikɔ́ ɑpí nɛ u sɑ́nɛ ɑpi u mɑɑ rɛ: Upɔ́ŋmɑ́nɛ yɛ nfɑ́ɑ mɑ́lɛ. 
Uu pi pisɛ rɛ: Ituŋɛ íye kumúŋɛ́ kɛ uipiŋɛ ii kutɔsi piwɑi korɑɑnɛ? Api rɛ: Sé kɛtúŋɛ́, mɛsɛ kumúŋɛ́ kɛ kupiŋɛtoi ɑku ukɛcɑ́ɑ́ le. 
Usɑ́ɑ uyɛ̃ uu pɛɛ léisɛ rɛ ituŋɛ iyɛ-i ŋmɑɑ kɛ Yeesu uu sé u mɑɑ rɛ: Upɔ́ŋmɑ́nɛ yɛ nfɑ́ɑ mɑ́lɛ. Lɛ̃ nnyɑ kɛ u nɛ ukɛyɔ pikɔ́ nnɛ́í ɑpi Yeesu nɛ kɛfɑ tɛnɛ. 
Kɛ Yeesu uu Yutee kɛtẽ-pɔ n léemɛ uu Kɑlilee kɛteni-i pɛɛri, mɛwɑisɑŋɑ lírũ mɛcɔ kuu n wɑ yɛ mɛ̃.
