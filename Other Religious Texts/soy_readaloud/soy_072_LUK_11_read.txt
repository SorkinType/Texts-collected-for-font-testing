Luki.
11.
Kɛyɑ́ɑ kɛnyinɛ kɛ Yeesu uu kɛlõ kɛnyinɛ kɛyómɛ yɑ́ɑ́si. Kuu ḿ mɑsí, uupirɛtiki unyinɛ uu u pisɛ rɛ: Upíimɑ, ɑ piyómɛyɑ́hɑɑ rɔ́ céési yɑrɛ kɛ Yohɑni uu upipirɛtiki n céesi mɛcɔ. 
Yeesu uu pi mɑɑ rɛ: Nɔn yɛ kɛyómɛ n yɑ́ɑ́sinɛ, ɑni yɛ mɑɑ rɛ: Sɑ́ɑ, tipɔ́nyíri tikɛ́ n kóólɛlɛ̃. Ipɔ́yɔ́ɔpi ikɛ́ lommɛ. 
A ḿpɑ́ kɛyɑ́ɑ kɛ́ye ilukɛ rɔ́ hɛ. 
A ɑrɔ́kópɛ rɔ́ sɑ́rɛi, Yɑrɛ kɛ tɔ́ɔ yɛ kɔ picɔ n sɑ́rɛi pin ɑkópɛ rɔ́ n wɑ mɛcɔ. Kɑpɛ tíyɛsɛ tɔkɛ́ pipeikɛɛ kɛ́mɛɛ loó. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɔ́ unyinɛ un nkpéni usɑ́nɛ m mɑ́ uu kɛsinɛ u lɛɛpɔ uu u mɑɑ rɛ: Unɛ́sɑ́nɛ, ɑ ɑkpɔ́nɔ́ ɑtɑɑni nɛ́ lonsɛ. 
Unɛ́sɑ́nɛ unyinɛ yɛɛ ncée leeri uu kɛnɛ́yɔ-i weri, ɑ́m kɔ pɛɛ líkɑ mɑ́ kɛ́ u pɑ. 
In n lɑ uusɑnɛ uyɛ̃ yɛ́ hɑ́i kɛ́yɔ kɛmɛɛ-pɔ rinɔ́ɔ u yɔ́ɔmɛ rɛ: Kɑpɛ nɛ́ ricɔ́ŋ! Nɛ rinɔnɔɔ hɑ́nɛsilɛ ɑm kɑ́, tɔ́ nɛ sinɛ́wɑ̃́ yɛ lonlɛ, ɑ́m fenɛ kɛ́ yisi kɛ́ ɑkpɔ́nɔ́ pɔ́ pɑ. 
Asei kɑm nɔ́ símisinɛ rɛ in n lɑ úu yɛ́ yisi ukɛ́ ɑ u pɑ rɛ u uusɑnɛ nnyɑ, ɑmɑ́ u yɛ́ yisi uu lɛ̃ nnɛ́í kɑi u m pisɛ u pɑ rɛ uusɑnɛ yɛ picɔ́ŋ u nɔŋ́lɛ̃ nnyɑ. 
Nɛ̃́ nɛ nɔ́ tee rɛ: Ani yɛ we, pi yɛ́ nɔ́ hɛ, ɑni yɛ wɛ́ɛ́si, nɔ́ yɛ̃́, ɑni yɛ rikóḿkóḿ, pi yɛ́ rinɔnɔɔ nɔ́ hɑ́nnɛ. 
Likumúŋɛ́ rɛ usoi uyɛɛ n we kɑpi yɛ hɛ, uyɛɛ n wɛ́ɛ́si yɛɛ yɛ yɛ̃, uyɛɛ rinkóḿkóḿ kɑpi yɛ hɑ́nnɛ. 
Nɛ́ɛ usɑ́ɑ úye kɛwɑ̃́ kɛɛ yɛ́ nkpíntomɛ́ u we, uu pɛɛ iwɑ́ɑ kɛ pɑ? 
Nɛ́ɛ úye yɛɛ yɛ́ ukɛwɑ̃ inɑ̃ hɛ, kɛn tipele u n we? 
Ḿpɑ́ nɔ́ pikpɑ́ɑ́ree mpí, nɔ yɛ felɛ ɑni nɔ́sinɛ́pipi lisɔnɛ hɛ. Nɔ́ pɛɛ rɛ urɔ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we yɛɛ yɛ́ pɛɛ mɛníŋɛ pɛpɛɛ Nfɑ́ɑsɔnɛ u m pisɛ pihɛɛ riyɑ́ɑ́? 
Kɛyɑ́ɑ kɛnyinɛ kɛ Yeesu uú we un kuníri kunyinɛ kpɛɛ utisi unyinɛ nsímɛ́ ḿ pɔɔnɛsɛlɛ̃ lɑkɑsɛ. Kɑku ń le, utisi uyɛ̃ uu nsímɛ́ loni ɑi pisoi pɛɛ n cɑ́pinɛlɛ̃ mɛyíkíyiki nnɔ́ɔ yipu. 
Amɑ́ pikɛcɔpɛ pinyinɛ ɑpí mɑɑ rɛ: Aníri uyɔɔpi Pɛlisepulɛ nnɑŋɛ kuu yɛ nɛ ɑníri lɑkɑsɛ! 
Amɑ́ picɔ pɔ́ɔn lɑ pikɛ́ ripinɛ u wɑ, ɑpi pɛɛ u pisɛ rɛ ukɛ́ mɛwɑisɑŋɑ mɛnyinɛ wɑ ukɛ́ nɛ nyísɛ rɛ Uléécɑɑ kɛ́mɛɛ kuu léeri. 
Amɑ́ Yeesu pɔ̃́ nɛ pɛɛ pikɛmúŋɛ́ nyulɛ. Uu pɛɛ pi mɑɑ rɛ: Kuyɔ́ɔpiyu kunyinɛ pisoi pin pimɛcɔpɛcirɛ n tɔpu, ku yɛ kpulɛ, kusiyɔ ɑsi kɛsɛ kɛsɛ lólɛsi. 
Lɛ̃ nnyɑ in tɛ Setɑni yɛ nɛ umɛcirɛ tɔpu, íye kɛ ukuyɔɔpiyu ɑku yɛ́ pɛɛ fe ɑkú nɛ n we? Nɔ mɑɑ rɛ Pɛlisepulɛ nnɑŋɛ kɑḿ nɛ ɑníri lɑkɑsɛ nnyɑ kɑm lɛ̃ mɑ. 
In tɛ Pɛlisepulɛ yɛɛ ńnɑŋɛ nɛ́ hɛ kɑḿ nɛ ɑníri lɑkɑsɛntɛ, wóo pɛɛ nɔ́pinɛ́kɔ́ pɔ̃́ ńnɑŋɛ hɛ kɑpí nɛ ɑ lɑkɑsɛntɛ? Ani n nyu rɛ nɔ́pinɛ́kɔ́ cirɛ pɛɛ nɔ́ nyísɛnɛ rɛ ɑ́ni ɑlɑri mɑ́. 
Asei kɛcɑ́ɑ́, Uléécɑɑ nnɑŋɛ kɑḿ nɛ ɑníri lɑkɑsɛntɛ. Lɛlɛɛ nyísɛlɛ̃ tɛ Uléécɑɑ iyɔɔpi yɛ nɔ́kɛnɛ́mɛɛ pituimɛ mɑsilɛ. 
Usoi nɑŋɛ-nɑŋɛ un itɔpilũ m mɑ́ un nɛ ukɛyɔ m mɛ̃́, úkɑ úu yɛ́ fe ukɛ́ ulikɔ́ linyinɛ yɑ́ɑ́i. 
Amɑ́ unyinɛ un ńnɑŋɛ u n feriyɛ, liute yɛ̀ɛ̀ uitɔpilũ iyɛ̃ kuú pɛɛ́ nɛ n tɑ́lɛ̃ yɔsilɛ uu umɛmɑ́ mɛ̃ kpísi uu hɔ́ɔ́nɛntɛ. 
Nkó yɛɛ úu kɛnɛ́cɑ́ɑ́ n kpɑ́lɛ̃ yɛ ilɑ́ɑrɔ nɛ́ wɑilɛ, nkó yɛɛ úu picɑ́pinɛ nɛ́ n lɛ́ni yɛ hɑrɛsilɛ. 
Kuníri kun usoi kɛ́mɛɛ́ n le, kɛlõ kóimɑ kɛnyinɛ-i kɑku yɛ hɑ n kerɑɑi kun kɛwéntɛ wɛ́ɛ́si. Kunsɑ́ kɛwéntɛ n yɛ̃, ku yɛ kukɛfɑ-i mɑɑ rɛ: Nɛ kɛnɛ́yɔ kpurɛ kɑm n tíyɛmɛ-i pɛlɛnɛ. 
Kun m pɛɛpɔ, ɑku kɛ́yɔ lɛɛpɔ kɛn pééilɛ̃ kɛn kɔ nyɔ́ɔnsɛlɛ̃ cɑ́kɑ́-cɑ́kɑ́. 
Kei kɑku yɛ pɛɛ́ hɑ kuɑcɔ ɑsɛɛi nyɛɛ kuricuruu rikpɑ́kɑ́rɑ́ ń fe kpíipɔ ɑɑ́ nɛ kɛ́yɔ kɛ-i tonɛ. Lɛlɛɛ yɛ tíyɛsɛ kɛ liute uyɛ̃ mɛwee fɑlɛ ɑmɛ yɛ n cɑɑilɛ̃ ɑmɛ mɛkpurɛ mɛ-i kuú pɛɛ ń we tɔ́su. 
Kɛ Yeesu uu lɛ̃ pisímɛ́ ḿ mɑsí, unɔ́si unyinɛ uu riwúí kɛcɔpɛ mɑɑri rɛ: Unɔ́si uyɛ̃ kɛfɑ-i kɑɑ n tónɛ uu kɔ pɔ́ nyɛnsɛ yɛ únɑrɛkomɛ lɛ. 
Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Pɛpɛɛ yɛ Nléécɑɑsimɛ́ kutu rińcɔ ɑpi lɛ̃ knn m pisɛ tiki pɛɛ mɛníŋɛ pínɑrɛkomɛ. 
Kumúŋɛ́ kpɛ-i kɛ ɑsoiwuí ɑɑ Yeesu n kɑ́lɑɑlɛ̃, Yeesu uú mɑɑ rɛ: Nfɑ́ɑni kɛtẽ nkɛ́ pisoi yɛ cɑɑilɛnlɛ hɑ́i. Pi mɛwɑisɑŋɑ kórulɛ. Mɛwɑisɑŋɑ mɛ̃ kɛ ɑntepu Yonɑɑsi uu n yɛ̃́ mɛnyinɛcɔ kɑpi yɛ́ kɔ yɛ̃. 
Yɑrɛ kɛ Yonɑɑsi uu Niniifi pikɔ́ mɛwɑisɑŋɑ n nyísɛ, limɛcɔ kɛ Usoi Kɛpipi ɑkɛ yɛ́ nfɑ́ɑni kɛtẽ nkɛ́ pisoi mɛ nyísɛ. 
Kɛyɑ́ɑ kɛ̃ kɛ Uléécɑɑ uú nɛ pisoi n túhɑɑnɛnɛ, unɔ́siyɔɔpi yɛɛ Sɑɑpɑ kɛtẽ kɛcɑ́ɑ́ n tṹ yɛ́ hɑ yisi, uu nfɑ́ɑni kɛtẽ nkɛ́ pisoi ɑkópɛ pɑ. Li we rɛ uyɛɛ pɛɛ hɑ́i kɛtẽ kɛtɔ-pɔ leemɛlɛ uú weri rɛ ukɛ́ Sɑlomɔɔ mɛwɛ́ɛ́sɛsɔhɔ kõ. Uyɛɛ kɔ nté nní ń we yɛ kɔ pɛɛ Sɑlomɔɔ felɛ! 
Niniifi pikɔ́ yɛ́ hɑ kɛyɑ́ɑ kɛ̃ yisi, ɑpi nfɑ́ɑni kɛtẽ nkɛ́ pisoi ɑkópɛ pɑ. Li we rɛ kumúŋɛ́ kpɛ-i kɛ Yonɑɑsi uú pɛɛ Nléécɑɑsimɛ́ rin-yóó ɑpi kóm, pi pɛɛ mɛfinɛ consɛlɛ. Uyɛɛ nté nní ń we yɛ kɔ pɛɛ Yonɑɑsi felɛ. 
Yeesu uu kɔ rɛ: Úkɑ úu yɛ kɛ́firɑ́ɑ risɛ́ uu kɛ pésu nɛ́ɛ uu kucɑ́ri kɛ kipi, ɑmɑ́ kɛlõ cɑ́cɑ́ kuu yɛ kɛ ritɔsi rɛ pɛpɛɛ n lompɔ pikɛ́ mɛtɛ́í yɛ̃́ nnyɑ. 
Ipɔ́nípɛɛ yɛ ipɔ́piŋɛ kɛfirɑ́ɑ lɛ. Ipɔ́nípɛɛ in n nyɑ́ni ńsɔnɛ, ipɔ́piŋɛ nnɛ́í kɑi yɛ n kpɑ̃́iilɛ̃, ɑmɑ́ ipɔ́nípɛɛ insɑ́ ńsɔnɛ n nyɑ́ni, kuŋmɑhɑ-i kɛ ipɔ́piŋɛ ii yɛ́ n we. 
Lɛ̃ nnyɑ ɑ mɛpɔ́círɛ tĩ, mɛtɛ́í mɛɛ kɛpɔ́mɛɛ ń we ɑ́mɛ kɑpɛ nɛ riŋmɑhɑ rimpɔ́lɔ́ nnyɑ. 
Ipɔ́piŋɛ nnɛ́í in mɛtɛ́í m mɑ́ kɛ ikɛlõ kɛ́kɑ ɑ́kɛ kuŋmɑhɑ-i we, innɛ́í yɛ́ mɛtɛ́í-i m pɔlɔlɛnlɛ yɑrɛ lɛ̃ kɛ kɛ́firɑ́ɑ ɑkɛ yɛ nɛ kɛmɛtɛ́í nnɛ́í pɔ n kpɑ̃́iilɛ̃. 
Kɛ Yeesu uu ń we un símisi, Ufɑrisi unyinɛ uu ukɛyɔ-i u séi rɛ pikɛ́ kɛsẽ́ ilukɛ li. Yeesu uu lompɔ uu pilukɛ kɑ́pɑ́ɑ́. 
Ai Ufɑrisi uyɛ̃ pírí wɑi kɛ Yeesu uú nɛ n le úu kɔ pɛɛ ɑnípɛ nɑlɛ kɛlenɛ yɑrɛ kɛ piinyɛkii ii m pisɛ. 
Upíimɑ uu pɛɛ u mɑɑ rɛ: Lɛ̃ kɛ nɔ̃́ Pifɑrisi ɑni ń we yɛ lɛ̃. Kɛpóripi nɛ ricɑ́ripi kɛpirɛ kɑni yɛ hɛɛrɛ, ɑmɑ́ nɔ́sinɛ́fɑ kɛ́mɛɛ, piyɑɑ nɛ mɛwɑi kópɛ nfɑsimɛ́ mɛɛ yipɑɑlɛ̃. 
Nɔ piníri lɛ. In nɛ Uléécɑɑ yɛɛ kɛpirɛ n wɑ cirɛ yɛɛ kɔ kɛmɛ́ɛ wɑɑ? 
Lɛ̃ nnyɑ, ɑni lɛlɛɛ nɔ́sinɛ́póripi nɛ nɔ́ɑnɛ́cɑ́ripi-i ń we píwɛkɔɔ́ hɛkɛsi, li yɛ́ pɛɛ ḿpɑ́ yo nɔ́ sɔnukusɛ. 
Nɔ̃́ Pifɑrisi, íwɛ kɑni topori! Nɔ yɛ iyɛ́hɛ yɑrɛ mɑnti nɛ ruu nɛ ɑyɛ́hɛpɛi ncɔpuri ripɛ́nɛlɛ ɑni hɔ́ɔ́nɛ ɑwɛ́lɛ́ kɛfi, ɑni kuwɛ́lɛ́ kusɛ Uléécɑɑ pɑ. Amɑ́ ɑ́ni pɛɛ Uléécɑɑ ɑsei nɛ unlɑ wɑisɛlɛ̃. Lɛ̃ kɑi kɔ pɛɛ m pisɛ rɛ ɑni n wɑi nɔn kɔ pɛɛ́ nɛ litɔ́rɔɔ kɛcɑ́ɑ́ léisɛlɛ̃. 
Nɔ̃́ Pifɑrisi, íwɛ kɑni topori. Síyukɔɔ́tonɛ kɑni yɛ ɑyómɛyɑ́hɑɑlee-i n lɑ. Nɔ kɔ lɑ pikɛ́ yɛ kɛ́yɑɑ́lɑ-i nɔ́ n yɑ́hɑɑnkɛɛ. 
Íwɛ kɑni topori! Nɔ we yɑrɛ ihórɛ iyɛ̃ kɛcɑ́ɑ́ kɑpi n sɔ́nɛ ɑ́pi kɔ pɛɛ ikɛcɑ́ɑ́ nyu. 
Isé ucélɑɑ usɛ uu u mɑɑ rɛ: Sɑ́ɑ, pɔn lɛ̃ n símisi, tɔ̃́ kɑɑ kɔ lɛ̃ lɑ́misi. 
Yeesu uu rinɔ́ɔ yɔ́su rɛ: Nɔ̃́ isé picélɑɑ, íwɛ kɛ nɔ́ɔ kɔ topori! Ncɔni mulɛ-mulɛ kɑni yɛ picɔ cɔnsɛ, ɑ́ni yɛ kɔ pɛɛ ḿpɑ́ fíí nɛ nɔ́kɛnɛ́nípɛpi ricɑ. 
Íwɛ kɑni topori! Akpíí sɔnɛ kɑni yɛ ɑntepuyɛ kɛ nɔ́pinɛ́sɑ́ɑyɑhɑ ɑpi n kṍ mɔ́mti. 
Nɔ́mɛnɛ́wɑi mɛɛ nyísɛlɛnlɛ rɛ lɛ̃ kɛ nɔ́pinɛ́sɑ́ɑyɑhɑ ɑpi n wɑ yɛ nɛ nɔ́ risɑ́lɛ, likumúŋɛ́ rɛ pi ɑntepuyɛ kónlɛ, nɔ́ɔ piɑkpíí mɔ́mti. 
Lɛ̃ nnyɑ kɛ Uléécɑɑ uú pɛɛ́ nɛ umɛwɛ́ɛ́sɛsɔhɔ mɑɑ rɛ: Nɛ̃́ Uléécɑɑ, nɛ́ ɑntepuyɛ nɛ pitumɛ pi pɑ, ɑpi pikɛcɔpɛ picɔ kóni ɑpi picɔ íwɛ wɑi. 
Kɑi nní lɛ̃, nfɑ́ɑni kɛtẽ nkɛ́ pisoi pɛɛ yɛ́ ɑntepuyɛ pɛɛ hɑ́i kɛtẽ kɛkorɑɑnɛ-mɛ n kpí nkpɔ nnyɑ ɑkpɑnii yɛ̃. 
Kɑi n kpíimɛ Apɛɛli nkpɔ-mɛ hɑ́i nɛ Sɑkɑrii kɑpi kɛnyɔ́ɔnsɛhúúpi nɛ Uléécɑɑ kɛyɔ kɛpɑ́ɑ́pɑ́ɑ́ ń kpu nkpɔ-pɔ. Asei kɑm nɔ́ símisi, nfɑ́ɑni kɛtẽ nkɛ́ pisoi mpí kɑpi ɑntepuyɛ pɛ̃ nkpɔ riwomɛ kórunɛ. 
Íwɛ kɑni topori, nɔ̃́ isé picélɑɑ! Nɔ rinɔnɔɔ tɛ̃ kɑpi yɛ nɛ rintíki ɑpí nɛ Uléécɑɑ céru kɛhɑ́nnɛ lesɛlɛ, nɔ̃́ ɑ́ni kɔ pɛɛ fenɛ nɔkɛ́ lompɔ, ɑni kɔ pɛɛ pɛpɛɛ pilompɔ ń lɑ ncée tɑ́pisi. 
Kɛ Yeesu uu kɛlõ kɛ-i n yisi, isé picélɑɑ nɛ Pifɑrisi ɑpí nɛ u wɔ́ɔsi, ɑpi isímɛ́ mpehẽ mpehẽ kɛcɑ́ɑ́ u m písɛisɛ. 
Api ɑpinɛ u n wɑpisi rɛ ukɛ́ nkópɛ nnyinɛ mɑlɛ.
