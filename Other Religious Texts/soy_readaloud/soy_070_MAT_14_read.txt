Mɑtiyee.
14.
Uyɔ́ɔpi Erooti yɛɛ pɛɛ likumúŋɛ́ kpɛ-i Kɑlilee kɛtẽ kɛcɑ́ɑ́ pɑílɛ̃. Uu pɛɛ kóm pin Yeesu nsímɛ́ símisi. 
Uu upikɛikɔ́ mɑɑ rɛ: Yohɑni Úniwolɛ yɛɛ n kpu nsímɛ́ kɑpi símisi. U pikpɔkpɔ kɛ́mɛɛ yisilɛ. Lɛ̃ nnyɑ kuu nní ńnɑŋɛ yɛ̃ un nɛ mɛwɑisɑŋɑ wɑpisi. 
Asei kɛcɑ́ɑ́, Erooti yɛ pɛɛ tíyɛsɛlɛ ɑpi Yohɑni Úniwolɛ tini ɑpi pɑɑsi ɑpi kukpɑniilee tɑni. Umɑ́ɑ Filipu unɔsi kɑpi yɛ n sée rɛ Erootiyɑti nnyɑ kuú pɛɛ́ lɛ̃ wɑ. 
Yohɑni yɛ pɛɛ Erooti mɑɑ rɛ: Áɑ ncée mɑ́ pɔkɛ́ upɔ́mɑ́ɑ unɔsi kpísi. 
Lɛ̃ nnyɑ kɛ Erooti uú pɛɛ́ lɑ ukɛ́ Yohɑni kpu, ɑmɑ́ un pɛɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ wuru. Pɛ̃ nɛ pɛɛ nyɑ́nilɛ rɛ Yohɑni yɛ ɑntepu lɛ. 
Api pɛɛ kɛyɑ́ɑ kɛnyinɛ Erooti kɛmɑrɛyɑɑ ɑnyɑ̃́ n wɑi, Erootiyɑti ukpére uu léépɔ uu pisɑ́nɛ pɛɛ n cɑ́pinɛlɛ̃ kɛcɔpɛ yesu ɑi Erooti yúkúlú 
uu pisoi inipɛɛ iyɛ-i símisi uu wééri rɛ ukpére uyɛ̃ un yo mpuri n lɑ, un li u m pisɛ, u yɛ́ li u pɑ. 
Kei kɛ Erootiyɑti uu ukpére tɑ́lɑɑnkɛɛ, uyɛ̃ uu pɛɛ Erooti mɑɑ rɛ: A nté nní Yohɑni Úniwolɛ riyu kɛcɑ́ripi tɑ́í-tɑ́í kɛ́mɛɛ wɑ ɑɑ nɛ́ pɑ. 
Kɛ Erooti uu lɛ̃ n kṍ, uripɔɔ ɑri cɑɑi. Amɑ́ uɑwééri nɛ pisɑnɛ isɛi nnyɑ, uu rinɔ́ɔ hɛ rɛ pikɛ́ lɛ̃ kuu m pisɛ u pɑ. 
Uu unyinɛ tum tɛ pikɛ́ hɑ kukpɑniilee-i Yohɑni riyu tɔ. 
Apí hɑ ríyu tɔlu ɑpí nɛ weri ɑpi ukpére uyɛ̃ pɑ, uyɛ̃ uu ri kpísi uu úni pɑ. 
Kei kɛ Yohɑni pipirɛtiki ɑpi pɛɛ́ hɑ ɑpi u kpísi ɑpi kúlɛsi, ɑpi limɛmɑ́ɑ́ pɛɛ sĩ́ ɑpí hɑ lɛlɛɛ n wɑ nnɛ́í Yeesu kɛɛni. 
Kɛ Yeesu uu nsímɛ́ mmɛ̃ n kṍ, uu yisi uu kúninɔi loni rɛ ukɛ́ tɔ́mpɔ ukɛ́ hɑ umɛcirɛ kei kɛ úkɑ úu ń we n tũ. Pisoi ɑpi kóm ɑpi ɑyu nyɛɛ nɛ kei n kɔ́lɛ̃ kɛ́mɛɛ nɛ ɑ́nɑ yisi ɑpi lɛ̃ kupiyɛ n kɑnɛi ɑpí hɑ nɛ u lɛɛpɔ. 
Kuu kúninɔi-i ń suí, uu risoiwuí yɛ́nu, tiicɔ ii íwɛ u wɑi uu pitóikɔ́ pɛɛ n hɑ́pɔ pipɔisɛntɛ loni. 
Kɑi nnyɔ́ɔ́ n wɑ, upipirɛtiki ɑpi ukɛkúrí hɑpɔ ɑpi rɛ: Ituŋɛ yɛ nkpéni pitɛnɛ mɑsilɛ, ɑ́ri kɔ nté nɛ kuyu kúkɑ kɔlɛ̃. A pisoi kulúi mpí riyɑ́ pikɛ́ siyupi-i pilukɛlɑlɛ hɑ pikɛ́ li. 
Yeesu uu pi pɛsɛ rɛ: Ái tílɑsi rɛ pikɛ́ tɔ́mpɔ. Ani nɔ́rinɛ́cúruu ilukɛ pi hɛ pikɛ́ li. 
Api pɛɛ u mɑɑ rɛ: Akpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́ ŋmɑnɛ kɑri nté mɑ́. 
Yeesu uu pi pɛsɛ rɛ: Ani nɛ nté nɛ́ kɑm. 
Kɑpí nɛ n hɑ́pɔ, uu pɛɛ pisoi riwúí tɛ̃ mɑɑ rɛ pikɛ́ mɛyúi kɛcɑ́ɑ́ tonɛ. Uu ɑkpɔ́nɔ́ ɑnupũ nyɛ̃ nɛ ikpíntomɛ́ itɛ́ iyɛ̃ kpísi uu kɛléécɑɑ weríí uu Uléécɑɑ pɔɔnɛsɛ, uu ɑkpɔ́nɔ́ kpɔ́kɔrinɛ uu upipirɛtiki pɑ ɑpi pisoi riwúí hɔ́ɔ́nɛ. 
Pinnɛ́í ɑpi le ɑpi lɛpu. Upipirɛtiki ɑpi ɑkéri nyɛɛ m pɔɔnɛ cɑ́pinɛ ɑi ɑnɛ́rɛ kɛfi nɛ ɑtɛ́ yipu. 
Pɛpɛɛ ilukɛ iyɛ̃ n li yɛ tu pitisi ŋmɑnɛ ɑ́kotokú ɑnupũ (5.000). Ápi pinɔ́si nɛ siwɑ̃́ kɛɛ̃. 
Kɑpi pilukɛ ḿ mɑsí, Yeesu uu upipirɛtiki rinɔ́ɔ hɛ rɛ pikɛ́ kúninɔi lõ pikɛ́ hɑ kupiyɛ ritimɛ ricɔ-pɔ u fɔsí, uyɛ̃ ukɛ́ pɛɛ kei pisoi riwúí tɛ̃ n kpiisɛlɛ̃ kɛlenɛ. 
Kɛ pisoi ɑpi pikúũ ḿ mɑsí, uu yisi uu riyɔ́pɛ tɑɑ́ rɛ ukɛ́ kei umɛcirɛ n we un kɛyómɛ yɑ́ɑ́si. Uu pɛɛ kei lɛ̃ umɛcirɛ n we ɑí nɛ nnyɔ́ɔ́ wɑi. 
Kúninɔi yɛ pɛɛ likumúŋɛ́ kpɛ-i míni ritimɛ nɛ kɛtɑɑ piwɑi mɑsilɛ. Kɛ kuyɔ ɑku nní kukɛyu-mɛ̃ m pépuri nnyɑ, ɑ́niwɑlɛ́ ɑɑ pɛɛ ku n tɑ́ŋɑisɛ. 
Yeesu uu riweeki kumúŋɛ́ míni kɛcɑ́ɑ́ n sɔ́nɛ un nɛ upipirɛtiki kɛ́mɛɛ sĩ́. 
Kɑpi lɛ̃ u n yɛ̃́ un míni kɛcɑ́ɑ́ sɔ́nɛ, piipiŋɛ ii kpi nɛ iwɑmɛ mɛpehẽ, ɑpi pɛɛ pimɛcɔpɛcirɛ n tee rɛ: Kunírilɑ̃ yɛ ku. Api pɛɛ iwɑmɛ nɛ m pupukɛɛ. 
Yeesu uu pɛɛ mɛsɛ nɛ mɛsɛ pi mɑɑ rɛ: Áni kɑpɛ pírí tónɛ. Nɛ́ɛ lo, iwɑmɛ íi kɑpɛ nɔ́ wɑ! 
Piyɛɛ uu u mɑɑ rɛ: In tɛ mɛsei pɔ́ɔ lo, ɑ tíyɛsɛ kɛ́ míni kɛcɑ́ɑ́ sɔ́nɛ kɛ́ nɛ pɔ́ lɛɛpɔ, Upíimɑ! 
Uu rɛ: A kɑm! Piyɛɛ uu kúninɔi-i léepɔ uu míni kɛcɑ́ɑ́ sɔ́nɛ rɛ ukɛ́ Yeesu lɛɛpɔ. 
Amɑ́ kuu n yɛ̃́ tɛ kuyɔ yɛ pɑ́ɑ́púlɛ, iwɑmɛ ii u loni, míni ɑmɛ pɛɛ u m muuni. Uu cɑ́ɑ́i rɛ: Upíimɑ, ɑ nɛ́ yóriyɛ! 
Yeesu uu mɛsɛ nɛ mɛsɛ kunípɛ sɑɑpɔ uu u tini uu u pisɛ rɛ: Mpɔ́fɑtɛnɛ ńn piyɛ. Yo nnyɑ kɑɑ́ tirí? 
Kei kɛ pikɛtɛ́ ɑpi pɛɛ kúninɔi kɛ́mɛɛ lompɔ, kuyɔ ɑku sɛ́ɛ́. 
Yeesu pipirɛtiki picɔ pɛɛ kúninɔi kɛ́mɛɛ ń we ɑpi pɛɛ́ hɑ Yeesu mɛtene wulɑ ɑpi rɛ: Pɔ mɛsei Uléécɑɑ Kɛpipi lɛ. 
Kɑpi kupiyɛ pitɛ́ŋ́ ḿ mɑsí ɑpi Kenesɑrɛti kɛtẽ tulu, 
kei pikɔ́ ɑpi Yeesu ceru, ɑpi pitumɛ ḿpɑ́ yei yɛ́kɑ́ɑ́sɛ rɛ pikɛ́ pisoi símisi rɛ u kɑ. Pisoi ɑpi pɛɛ pitóikɔ́ nnɛ́í nɛ u sɔ́nɑɑpɔ. 
Api u welu rɛ ukɛ́ ŋmurɛi pitóikɔ́ pɛ̃ pikɛ́ ḿpɑ́ ukutukɑnkɑ ritimɛ ŋmɑnɛ ricɑ. Pɛ̃ nnɛ́í pɛɛ rińcɑ ɑpi pélu cɑ́kɑ́-cɑ́kɑ́.
