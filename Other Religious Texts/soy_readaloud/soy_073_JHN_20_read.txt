Yohɑni.
20.
Kɛwéntɛyɑɑ ukóso, Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri uu kɛmɑrɛsinɛ yisi, lin hiihiilɛ̃ uu nhórɛ-i sĩ́, uu yɛnu rɛ pi ripɑrɛ kɑpi nhórɛ rińfi lesɛlɛ. 
Uu itóó wɑi uú nɛ Simɔɔ Piyɛɛ nɛ uupirɛtikicɔ uyɛ̃ kɛ Yeesu uú pɛɛ mɛyɑ̃́ ń lɑ lɛɛpɔ, uu pi mɑɑ rɛ: Pi Upíimɑ nhórɛ kɛ́mɛɛ lésɛlɛ, ɑ́ri nyu yei kɑpi u wɑ. 
Kei kɛ Piyɛɛ nɛ uupirɛtikicɔ uyɛ̃ ɑpi yisi ɑpi nhórɛ-i sĩ́. 
Api pikɛtɛ́ itóó wɑi, ɑmɑ́ ucɔ uyɛ̃ uu Piyɛɛ fɑɑu uú hɑ nhórɛ-i mɛfoí tulu. 
Uú fo uu lóólúpɔ, uu sisɑ́ŋɑ́ɑ́pɑmpi yɛnu sin kɛteni-i lɑɑnu, ɑmɑ́ úu pɛɛ lompɔ. 
Simɔɔ Piyɛɛ yɛɛ u n tíkilɛ̃ pɔ́ɔ kɔ tuipɔ. Uyɛ̃ uu nhórɛ-i lompɔ uu sisɑ́ŋɑ́ɑ́pɑmpi sɛ̃ kɛteni-i yɛ́nu. 
Uu kɔ kusɑ́ŋɑ́ɑ́ kpɛ̃ kɑpí pɛɛ Yeesu ríyu m pɑɑsi yɛnu kun pílɑɑlɛ̃ kun kumɛcirɛ kɛlõ iyɑɑ yisunu, sisɑ́ŋɑ́ɑ́pɑmpi ɑ́si kei we. 
Kei kɛ uupirɛtiki ucɔ uyɛɛ pɛɛ mɛfoí nhórɛ-i n tuipɔ pɔ́ɔ kɔ lompɔ, uu yɛ́nu uu pɛɛ́ nɛ kɛfɑ tɛnɛ. 
Asei kɛcɑ́ɑ́, hɑ́i nɛ kumúŋɛ́ kpɛ-i, ɑ́pi pɛɛ kɑhɑnɛ lɛlɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ ɑsei n kõ tɛ li pisɛ rɛ Yeesu ukɛ́ pikpɔkpɔ kɛ́mɛɛ yisi. 
Lɛ̃ mɛmɑ́ɑ́, pipirɛtiki pɛ̃ kɛtɛ́ ɑpi yisi ɑpí kulu. 
Mɑɑri pɔ́ɔ pɛɛ kɛtɑhɑi-pɔ nhórɛ kɛkúrí tonɛ uú n téni. Kuu lɛ̃ ń we un téni, uú fo uu nhórɛ kɛ́mɛɛ lóólúpɔ. 
Uu piléécɑɑtumɛ pitɛ́ pinyinɛ yɛnu pin ilũ tomɛ́ tɑnɑɑlɛ̃ pin nɛ kei kɑpí pɛɛ Yeesu n finsɛ tũ, ucɔ ríyu-mɛ̃, ucɔ ɑ́nɑ-mɛ̃. 
Api u pisɛ rɛ: Yo nnyɑ kɑɑ téni, unɔ́si? Uu pi pɛsɛ rɛ: Pi Unɛ́píimɑ kpísilɛ ɑpí nɛ tɔ́mpɔ, ɑ́m pɛɛ nyu yei kɑpi u wɑ. 
Kuu lɛ̃ ḿ mɑ, uu pɛɛ kɛpirɛ-mɛ̃ weríí uu Yeesu yɛnu un nyɛnu, úu kɔ ceri rɛ uyɛɛ lo. 
Kei kɛ Yeesu uu pɛɛ u pisɛ rɛ: Yo nnyɑ kɑɑ téni, unɔ́si? Úye kɑɑ wɛ́ɛ́si? Uu pɛɛ músu rɛ uyɛɛ kɛléécɑrɛ kɛcɑ́ɑ́ ḿ pɑílɛ̃ lo. Lɛ̃ nnyɑ kuu u pɛsɛ rɛ: Urɔ́sɑ́ɑ, in tɛ pɔ́ɔ u kpísi ɑɑ́ nɛ tɔ́mpɔ, ɑ nɛ́ símisi kei kɑɑ u n wɑ kɛ́ hɑ u kpísi. 
Kei kɛ Yeesu uu pɛɛ u sée rɛ: Mɑɑri! Uu pɛɛ ukuwɛ́ɛ́-mɛ̃ pɑnsɛpɔ uú nɛ Mɛ́-epiree u pɛsɛ rɛ: Rɑpuni. Liɑsei rɛ Ucélɑɑ. 
Yeesu uu u mɑɑ rɛ: Kɑpɛ nɛ́ ricɑ, ɑ́m kɑhɑnɛ Unɛ́sɑ́ɑ kɛ́mɛɛ rintɑɑ́. Amɑ́ ɑ hɑ pinɛ́mɑ́rɛcɔ símisi rɛ Unɛ́sɑ́ɑ yɛɛ kɔ nní Nɔ́unɛ́sɑ́ɑ nɛ Unɛ́léécɑɑ un kɔ Nɔ́unɛ́léécɑɑ kɛ́mɛɛ kɑḿ tɑɑ́lɛ̃. 
Kei kɛ Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri uu pɛɛ́ hɑ pipirɛtiki símisi rɛ u Upíimɑ yɛnlɛ. Uu pɛɛ lɛ̃ kɛ Upíimɑ uu u n símisi pi símisi. 
Kɛwéntɛyɑɑ kɛ̃ ukóso, Yeesu pipirɛtiki pin nnyɔ́ɔ́ mmɛ̃ kɛ́yɔ kɛnyinɛ-i cɑ́pinɛlɛ̃. Pisuifi iwɑmɛ yɛ pɛɛ pi welɛ. Api lɛ̃ nnyɑ hɑ́nɑɑnkɛɛ ɑpi pɛɛ kɛ́mɛɛ n we. Kei kɛ Yeesu uu pɛɛ pikɛcɔpɛ leemɛ uu pi mɑɑ rɛ: Nɔ́ɑnɛ́kíŋ ɑkɛ́ n niŋú! 
Kuu lɛ̃ pi ḿ mɑ, uu pɛɛ uɑnipɛ nɛ uilúkɛ́ pi nyísɛ. Upipirɛtiki rikiŋ ɑri níŋɛsi rɛ pi Upíimɑ yɛ̃ nnyɑ. 
Kei kɛ Yeesu uu kɔ pimɑ́ɑ pi rikpɑ́ rɛ: Nɔ́ɑnɛ́kíŋ ɑkɛ́ n niŋú! Yɑrɛ kɛ Unɛ́sɑ́ɑ uu nɛ́ n tummɛ mɛcɔ kɛ nɛ́ɛ kɔ nɔ̃́ tum. 
Kuu lɛ̃ pimɑ́ɑ ḿ mɑsí, uu pikɛcɑ́ɑ́ fúupɔ uu rɛ: Ani Nfɑ́ɑsɔnɛ yɔsí. 
Nɔn yɛ píye ɑkópɛ n sɑ́rɛi, Uléécɑɑ yɛ́ kɔ piɑkópɛ pi sɑ́rɛi. Nɔn yɛ rɛ ɑ́ni pinyinɛ ɑkópɛ sɑ́rɛinɛ, Uléécɑɑ pɑ́ɑ kɔ piɑkópɛ pi sɑ́rɛinɛ. 
Tomɑɑ kɑpi yɛ n sée rɛ Úsikɑ yɛ Yeesu pipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ kɛcɔpɛ usɛ lɛ. Uyɛ̃ úu pɛɛ kumúŋɛ́ kpɛ-i kɛ Yeesu uu pikɛmɛɛ́ n hɑ́pɔ we. 
Upipirɛtikicɔ ɑpi pɛɛ u mɑɑ rɛ: Tɔ Upíimɑ yɛnlɛ. Uu pi pɛsɛ rɛ: Nɛnsɑ́ ihímɛ ɑpolɛ uɑnipɛ-i n yɛ̃ kɛ kɛnípɛpi tɑhɑmpɔ, kɛ́ kɔ kunɛ́nípɛ nɛ uilúkɛ́ ricɑ, ɑ́m yɛ́ ŋmurɛi. 
Siyɑ́ɑ sipɑhɑ mɛmɑ́ɑ́, pipirɛtiki pɛ̃ ɑpi kɔ kɛ́yɔ-i n cɑ́pinɛlɛ̃ kɛsẽ́ nɛ Tomɑɑ. Pin kɔ nɛ hɑ́nɑɑnkɛɛ, Yeesu uu kɔ pikɛcɔpɛ piléemɛ kpɑ́ uu nyɛrɛ, uu pi mɑɑ rɛ: Nɔ́ɑnɛ́kíŋ ɑkɛ́ n niŋú! 
Uu pɛɛ Tomɑɑ mɑɑ rɛ: A kɛpɔ́nípɛpi nté tɑhɑmpɔ ɑɑ ɑnɛ́nípɛ pɑí, ɑɑ kɔ kupɔ́nípɛ nɛ inɛ́lúkɛ́ cɑ. A lɛ̃ pitírɛ́ tíyɛ ɑɑ́ nɛ nkpéni kɛfɑ tɛnɛ. 
Kei kɛ Tomɑɑ uu pɛɛ u pɛsɛ rɛ: Unɛ́píimɑ nɛ Unɛ́léécɑɑ! 
Yeesu uu u mɑɑ rɛ: Kɑɑ nɛ́ ń yɛ̃́ nnyɑ kɑɑ́ nɛ kɛfɑ nɛ́ tɛnɛ. Pɛpɛɛ ɑ́pi nɛ́ ń yɛ̃́, pin kɔ pɛɛ́ nɛ kɛfɑ nɛ́ tɛnɛlɛ̃ pɛɛ pínɑrɛkomɛ. 
Yeesu yɛ kɔ upipirɛtiki inipɛɛ-i mɛwɑisɑŋɑ mɛyɑ̃ ncɔpuri wɑpisilɛ mɛɛ ɑ́mɛ rítɛlɛ́ ntí-i n wɔ́lɑɑlɛ̃. 
Pi lɛlɛɛ nní rikɛmɛɛ ń we wɔilɛ rɛ nɔkɛ́ nɛ n ŋmurɛi rɛ Yeesu kɛ Uléécɑɑ uu n wɛ́ɛ nnyɑ. Uyɛɛ kɔ Uléécɑɑ Kɛpipi. Nɔn nɛ kɛfɑ u n tɛnɛ, nɔ́ urinyiri nnyɑ nfɑ́ɑ yɛ̃.
