Mɑriki.
3.
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu kuyómɛyɑ́hɑɑlee-i pɛɛri. Utisi unyinɛ yɛ pɛɛ kei welɛ un kunípɛ kópɛ mɑ́. 
Pɛpɛɛ kɛyɑ́ɑ kɛ̃ kei ń we, ɑpi kutu n cɔlɛ̃ tɛ Yeesu yɛ́ kɛwéntɛyɑɑ kɛtúŋɛ́ utisi uyɛ̃ pɔisɛɛ? Likumúŋɛ́ rɛ kɛwɑ́ɑ́nɑ́ kɑpi yɛ n wɛ́ɛ́si pikɛ́ nɛ Yeesu n tĩ́. 
Yeesu uu unípɛkópɛkɔɔ́ uyɛ̃ mɑɑ rɛ: A yisi, ɑ nyɛrɛ ḿpɑ́ úye ukɛ́ pɔ́ yɛ̃́. 
Yeesu uu pɛɛ pɛpɛɛ nní n nyɑ́nɛi pisɛ rɛ: Íye kɑi irɔ́sé kɛ́mɛɛ pisɛ? Li pisɛ rɛ tɔkɛ́ lisɔnɛ wɑ kɛwéntɛyɑɑ nɛ́ɛ likópɛ kɑi pisɛ rɛ tɔkɛ́ wɑ mɑ́? Li pisɛ rɛ tɔkɛ́ usoi nfɑɑ lɔ nɛ́ɛ li pisɛ rɛ tɔkɛ́ u kpu? Amɑ́ ɑ́pi ŋmurɛi pikɛ́ Yeesu rinɔ́ɔ yɔsí. 
Yeesu uu pinnɛ́í nyɑ́nɛi, piicɔ ii kuwɔ́i u wɑi hɑ́i. Uripɔɔ ɑri cɑɑi, ɑtu kɑpi m mɑ́ nnyɑ. Yeesu uu lɛ̃ mɛmɑ́ɑ́ u utisi uyɛ̃ mɑɑ rɛ: A kupɔ́nípɛ nɛ́ rituŋɑ́mɛ. Kuu yɛ́ kunípɛ rińtuŋɑ́pɔ, kunípɛ ɑku pélu. 
Pifɑrisi ɑpi kuyómɛyɑ́hɑɑlee-i léeri, pi nɛ pɛpɛɛ Erooti kɛmúŋɛ́ ń tíkilɛ̃ ɑpi mɛsɛ nɛ mɛsɛ cɑ́pinɛ pikɛ́ yɛ̃́ tɛ íye kɑpi yɛ́ wɑ pikɛ́ nɛ Yeesu kpu. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu kei pi yɑ́, u nɛ upipirɛtiki ɑpi Kɑlilee kupiyɛ ritimɛ sĩ. Kɑlilee pikɔ́ mɛyɑ̃ pɛɛ pɛɛ u tikilɛ̃. 
Yutee nɛ Yerusɑlɛm nɛ Itumee nɛ Yuritɛɛ mɛtɛ́ŋ́ nɛ Tiiri nɛ Sitɔɔ sitẽ pikɔ́ mɛyɑ̃ pɛɛ pɛɛ kɔ ukɛmɛɛ sɔ́nɑɑpɔ. Uliwɑiwɑi kɑpi n kṍ yɛɛ nɛ pi sɔ́nɑɑpɔ. 
Yeesu uu pɛɛ upipirɛtiki mɑɑ rɛ pikɛ́ kúninɔi kunyinɛ u nyɔ́ɔnsɛ, pisoi ɑ́pi kɑpɛ u nyikii nnyɑ. 
Ái líkɑ, kuu pisoi m pɔisɛntɛ mɛyɑ̃́ nnyɑ kɛ pitóikɔ́ tɔrɔɔ ɑpi yɔ́ɔnɛpɔ, ḿpɑ́ úye un nɑ́ɑ́si ukɛ́ u ricɑ. 
Ḿpɑ́ kumúŋɛ́ kúye-i kɛ ɑníri ɑɑ u yɛ̃, ɑ yɛ u riwulɑlɛ, ɑn pupukɛɛ rɛ: Pɔ Uléécɑɑ Kɛpipi lɛ. 
Yeesu uu rinɔ́ɔ ɑ hɛ nɛ ńnɑŋɛ rɛ: Áni kɑpɛ tíyɛsɛ unyinɛ ukɛ́ céri rɛ nɛ́ɛ úye. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ riyɔ́pɛ tɑɑ́ uu pisoi pɛ̃ kuu ń lɑ séi ɑpi u tiki. 
Uu pisoi kɛfi nɛ pitɛ́ wɛ́ɛ. U pi wɛ́ɛlɛ rɛ u nɛ pɛ̃ pikɛ́ n we ukɛ́ yɛ kɔ pi tũ pikɛ́ nɛ Nléécɑɑsimɛ́ rikɔ́ɔ́nú. 
Uu kɔ ńnɑŋɛ pi hɛ pikɛ́ yɛ nɛ ɑníri lɑkɑsɛ. 
Pisoi kɛfi nɛ pitɛ́ pɛ̃ kuu n lésɛ pɛɛ Simɔɔ kɛ Yeesu uu rinyíri n hɛ rɛ Piyɛɛ, 
nɛ Sepetee sipipi Yɑkupu nɛ uuwɑ̃ Yohɑni kɛ Yeesu uu rinyíri n hɛ rɛ Pɔwɑnɛrɛkɛsɛ. Tinyíri tɛ̃ ɑsei rɛ ikónɛ pisɛnkɛɛ. 
Nɛ Antiree nɛ Filipu nɛ Pɑtelemii nɛ Mɑtiyee nɛ Tomɑɑ, nɛ Alifee kɛpipi Yɑkupu, nɛ Tɑtee nɛ Simɔɔ yɛɛ pɛɛ ukuyu pimúlɛ́ kɛcɑ́ɑ́ n tɔpu, 
nɛ Yutɑsi Isikɑriyɔɔti yɛɛ Yeesu n tĩ́ uu pilɑ́ɑrɔ ɑnipɛ-i wɑi. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu kɛ́yɔ-i pɛɛri. Pisoi ɑpi kɔ pikɑ́lɑɑ u kpɑ́, u nɛ upipirɛtiki ɑ́pi pɛɛ kɛ́yu yɛ̃ pikɛ́ ilukɛ li. 
Kɛ Yeesu pikɔ́ ɑpi lɛ̃ n kṍ, ɑpi piwɛ́lɑɑ u sĩ pikɛ́ nɛ u kui pin músu rɛ li yɛ́ u m pɑnsɛlɛ. 
Uléécɑɑ isé picélɑɑ pɛɛ Yerusɑlɛm n léeri ɑpí m mɑ́ikɛɛ rɛ Pɛlisepulɛ kuu héesi. Api kɔ rɛ ɑníri uyɔɔpi nnɑŋɛ kuu yɛ nɛ ɑníri lɑkɑsɛ. 
Yeesu uu pɛɛ pi séi uu kɛnyɑ́rũ pi mɑɑ rɛ: Íye kɛ Setɑni uu yɛ Setɑni lɑkɑsɛ mɑ́? 
Kuyɔ́ɔpiyu kusɛ pisoi pin n tɔpu, kuyɔ́ɔpiyu kpɛɛ kpinɛ. 
Lɛ̃ cirɛ kɛ́yɔ kɛsɛ pisoi pin n tɔpu, kɛ́yɔ kɛɛ kpinɛ. 
In lɛɛ ni, Setɑni un nɛ uricuruu n tɔpu, pihɔ́ɔnɛnɛ pin ukuyɔɔpiyu-i n we, unnɑŋɛ ńn nɔŋ́nɛ. Amɑ́ mɛ́woo knn wɑinɛ. 
Úye un n lɑ ukɛ́ unɑ́ŋɛ-nɑŋɛ kɛyɔ-i lõ ukɛ́ ulikɔ́ yɔsí, u yɛ unɑ́ŋɛ-nɑŋɛ uyɛ̃ íŋmɛ pɑɑsilɛ. Umɛpɑhɑɑ mɑɑ́ kuu yɛ́ fe uu lɛlɛɛ ukɛyɔ-i ń we nnɛ́í yɔ́su. 
Asei kɑm nɔ́ símisi, ɑkópɛ nnɛ́í nɛ ɑlɛ̃́ yɛ kusɑ́rɛi welɛ. 
Amɑ́ úye un Nfɑ́ɑsɔnɛ n lɑ́mɑɑnkɛɛ, ɑkópɛ nyɛ̃ ɑ́ɑ kusɑ́rɛi we, ɑtɛnɛcirɛ́ yɛ nyɛ. Mɛsɛ́rɛ kɑɑ yɛ́ u m pɔ́riilɛ̃. 
Kɑpi Yeesu ḿ mɑɑ rɛ u ɑníri héesilɛ nnyɑ kuu mmú pi rihɑɑ. 
Yeesu uni nɛ upiwɑ̃ ɑpi pɛɛ́ hɑpɔ, ɑpi kɛtɑhɑi nyɛrɛ, ɑpi usoi tumpɔ rɛ pikɛ́ u séemɛ. 
Aí nɛ sɑ́ pisoi kulúi pin Yeesu kɑ́lɑɑlɛ̃. Api pɛɛ u mɑɑ rɛ: A nyɛrɛ nkɛ̃́, upɔ́ni nɛ pipɔ́wɑ̃́ yɛ pɔ́ wɛ́ɛ́si. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Wóo unɛ́ni? Píye pɛɛ pinɛ́wɑ̃́? 
Uu pɛɛ lɛ̃ mɛmɑ́ɑ́ pisoi pɛɛ nní u n kɑ́lɑɑlɛ̃ wérii uu rɛ: Unɛ́ni nɛ pinɛ́wɑ̃́ yɛ mpí. 
Úye un Uléécɑɑ mɛlɑ n wɑi, uyɛɛ unɛ́wɑ̃́ nɛ́ɛ unɛ́yɛ́i nɛ́ɛ unɛ́ni.
