Yohɑni.
21.
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu pɛɛ kɛtumɛ Tiperiyɑɑti kupiyɛ ritimɛ umɛcirɛ upipirɛtiki nyísɛ. Lɛ̃ kuu umɛcirɛ pi n nyísɛ yɛ nní: 
Simɔɔ Piyɛɛ nɛ Tomɑɑ kɑpi yɛ n sée rɛ Úsikɑ nɛ Nɑtɑnɑyɛɛli yɛɛ Kɑlilee kɛteni-i Kɑnɑɑ-pɔ ń we nɛ Sepetee piŋmɑ́nɛ nɛ upipirɛtiki pitɛ́ picɔ yɛ pɛɛ cɑ́pinɛlɛnlɛ. 
Simɔɔ Piyɛɛ uu pi mɑɑ rɛ: Ikpíntomɛ́ kɑm picɔ́pii tɔ́su. Api u pɛsɛ rɛ: Tɔ̃́ nɛ́ kɔ pɔ́ ritiki ɑri kɛsẽ́ sĩ. Kei kɑpi yisi ɑpí hɑ kúninɔi loni ɑpi tɔ́mpɔ. Amɑ́ ɑ́pi kɛsinɛ kɛ̃ líkɑ tĩ. 
Kɑi piweesi n kórɑɑnɛ, Yeesu un kei kupiyɛ ritimɛ we, ɑmɑ́ upipirɛtiki ɑ́pi pɛɛ céri rɛ Yeesu lo. 
Kei kuu pɛɛ pi pisɛ rɛ: Nɔ ikpíntomɛ́ yɛ̃́ ɑni cɔ́pii, sinɛ́wɑ̃́? Api rɛ: Eehei! 
Uu pi mɑɑ rɛ: Ani kúninɔi kulukɛ-lukɛ-mɛmpɔ kunyɛŋ́ fṍ, nɔ́ linyinɛ tĩ. Api mɛsei kunyɛŋ́ fom ɑpi ikpíntomɛ́ tini ɑi ku yipu péḿ, ɑ́pi fe ricuruu pikɛ́ míni-i ku pɔnnɛ kɛ ikpíntomɛ́ ii kulúi ń we nnyɑ. 
Kei kɛ upirɛtiki uyɛ̃ kɛ Yeesu uú pɛɛ ń lɑ uu pɛɛ Piyɛɛ mɑɑ rɛ: Asé Upíimɑ lo! Kɛ Simɔɔ Piyɛɛ uu lɛ̃ n kṍ tɛ Upíimɑ lo, uu uilũ kuú pɛɛ m mɑhɑ̃ un pɛɛ mɛŋmɑnɛ ikpíntomɛ́ cɔ́pii pitɑnɛ pɛsɛ uú nɛ míni-i kpúupɔ. 
Pi pɛɛ kɛtẽ kóimɑ nɛ kɛtɑɑ welɛ yɑrɛ ɑpɑ́hɑ pílɛ kumúŋɛ́. Upipirɛtikicɔ ɑpi léeri, ɑpi pɛɛ kunyɛŋ́ nɛ ikpíntomɛ́ mɛyipɛ pɑsɑi ɑpi lésɛpɔ. 
Kɑpi kúninɔi kɛ́mɛɛ ń suí, ɑpi kupiyɛ ritimɛ nkɑlɛnɑ lɛɛpɔ, ikpíntomɛ́ nɛ ɑkpɔ́nɔ́ ɑn nkɛcɑ́ɑ́ lɑ́ɑlɛ̃. 
Yeesu uu pi mɑɑ rɛ: Ani ikpíntomɛ́ iyɛ̃ kɑni n cɔ́pii kɛ́mɛɛ nɛ inyinɛ kɑm. 
Simɔɔ Piyɛɛ uu kúninɔi kɛ́mɛɛ lompɔ uú hɑ kunyɛŋ́ nɛ ikpíntomɛ́ mɑɑmɑɑ mɛyipɛ súisɛri, ií lelu pílɛ nɛ kuwóó nɛ itɑɑni. Amɑ́ ḿpɑ́ nɛ imɛsɑ́ mɛ̃, kunyɛŋ́ ɑ́ku ricɛɛ́ri. 
Yeesu uu pɛɛ pi séi uu rɛ: Ani kɑm ɑni li. Uupirɛtiki úkɑ úu kɔ pɛɛ kɑhɑ ukɛ́ u pisɛ rɛ: Pɔ́ɔ wóo? Pi pɛɛ nyulɛ rɛ Upíimɑ lo. 
Yeesu uú nɛ pi kɔ, uu kukpɔ́nɔ́ kpísi uu pi hɔɔnɛ, uu kɔ ikpíntomɛ́ pi hɔɔnɛ. 
Kɛ Yeesu uú nɛ pikpɔkpɔ kɛ́mɛɛ n yisi, mɛtɑ́ɑ́nũ yɛ mɛ̃ kuu lɛ̃ umɛcirɛ upipirɛtiki nyísɛ. 
Kɑpi pilukɛ ḿ mɑsí, Yeesu uu pɛɛ Simɔɔ Piyɛɛ pisɛ rɛ: Yonɑɑsi uŋmɑ́nɛ Simɔɔ, pɔ nɛ́ lɑ ɑi tɔ́su lɛ̃ kɛ picɔ mpí ɑpi nɛ́ ń lɑɑ? Uu rɛ: Ɛɛɛ, Upíimɑ, tipɔ́cúruu yɛ nyulɛ rɛ nɛ pɔ́ lɑlɛ. Yeesu uu pɛɛ u mɑɑ rɛ: In lɛ̃, ɑ sinɛ́sɑ́ŋpi kɛcɑ́ɑ́ ńsɔnɛ m pɑílɛ̃. 
Uu kɔ mɛlírũ pipisɛ u kpɑ́ rɛ: Yonɑɑsi uŋmɑ́nɛ Simɔɔ, pɔ nɛ́ lɑɑ? Piyɛɛ uu u pɛsɛ rɛ: Ɛɛɛ, Upíimɑ, tipɔ́cúruu yɛ nyulɛ rɛ nɛ pɔ́ lɑlɛ. Yeesu uu u mɑɑ rɛ: In lɛ̃, ɑ inɛ́sɑ́ŋ ńsɔnɛ n séni. 
Uu kɔ mɛtɑ́ɑ́nũ pipisɛ u kpɑ́ rɛ: Yonɑɑsi uŋmɑ́nɛ Simɔɔ, pɔ nɛ́ lɑɑ? Simɔɔ Piyɛɛ ripɔɔ ɑri cɑɑi rɛ mɛtɑ́ɑ́nũ yɛ mmɛ́ kuu lɛ̃ u pisɛ rɛ “pɔ́ nɛ́ lɑɑ” nnyɑ. Uu pɛɛ u pɛsɛ rɛ: Pɔ ḿpɑ́ yo nyulɛ, Upíimɑ, pɔ kɔ nyulɛ rɛ nɛ pɔ́ lɑlɛ. Yeesu uu pɛɛ u mɑɑ rɛ: In lɛ̃, ɑ inɛ́sɑ́ŋ kɛcɑ́ɑ́ ńsɔnɛ m pɑílɛ̃. 
Asei kɛcɑ́ɑ́, kɛ́ pɔ́ símisi rɛ kɑɑ́ pɛɛ iŋmɑ́nɛ ń we, pɔ́ɔ yɛ pɛɛ ripɔ́cúruu kupɔ́tɑmpɑ́lɑ rité ɑɑ kei kɑɑ ń lɑ sĩ. Amɑ́ pɔn n kpurunu, pɔ́ pɛɛ ɑnípɛ risíkɑ, ucɔ uu pɔ́ ku pɑɑsi uú nɛ kei kɑ́ɑ ń lɑnɛ pɔ́ sĩ́. 
U lɛ̃ mɑlɛ ukɛ́ nɛ nyísɛ lɛ̃ kɛ Piyɛɛ uu n kpinɛ likɛ́ nɛ Uléécɑɑ ríyu n wɑisɛ. Kuu lɛ̃ pisímɛ́ u ḿ mɑsí, uu pɛɛ u mɑɑ rɛ: A nɛ́ ritiki. 
Piyɛɛ uu kɛpirɛ ŋmɛɛlú uu upirɛtiki uyɛ̃ kɛ Yeesu uú pɛɛ ń lɑ yɛnu un pi tikipɔlɛ̃. Uyɛɛ pɛɛ ilukɛ kɛlukɛ-i Yeesu kɛcɑ́ɑ́ n finɛpɔ uu u pisɛ rɛ: Upíimɑ, wóo pilɑ́ɑrɔ ɑnipɛ kɛ́mɛɛ pɔ́ wɑinɛ? 
Kɛ Piyɛɛ uu lɛ̃ u n yɛ̃́, uu Yeesu pisɛ rɛ: Nkó ní, Upíimɑ, yo kɛ uyɛ̃ uú nɛ kɛ́tɔ kopunɛ? 
Yeesu uu u pɛsɛ rɛ: Nɛn n lɑ rɛ ukɛ́ n we kɛ́ hɑ nɛ pɛɛmɛ, ɑi yei nɛ pɔ́ yɛ́nu? Pɔ́ɔkɛ́ nɛ́ ritiki. 
Lɛ̃ nnyɑ, nsímɛ́ nn piyómɛyɑ́hɑɑ kɛ́mɛɛ kɔ́ɔ́nú rɛ upirɛtiki uyɛ̃ úu kpinɛ. Ái kɔ rɛ Yeesu yɛ u mɑɑ rɛ úu kpinɛ, ɑmɑ́ u pɛɛ rɛ: Nɛn n lɑ rɛ ukɛ́ n we kɛ́ hɑ nɛ pɛɛmɛ, ɑi yei nɛ pɔ́ yɛnu? 
Upirɛtiki uyɛ̃ ticuruu yɛɛ ɑseérɑ lesɛ uu nsímɛ́ mmɛ̃ wɔlu. Tɔ kɔ nyulɛ rɛ lɛ̃ kuu n yɛ̃́ uú mɑlɛ yɛ nsímɛ́sei lɛ. 
Yeesu yɛ kɔ mɛwɑi mɛyɑ̃ mɛcɔ wɑpisilɛ. Pin pɛɛ mɛnnɛ́í n wɔlunɛ mɛsɛ mɛsɛ, ɑ́m musí rɛ ɑ́tɛlɛ́ nyɛ̃ nnɛ́í kɑpi yɛ́ n wɔ́i yɛ́ kɛtẽ nkɛ́ nnɛ́í kɛcɑ́ɑ́ kɛwɑi yɛ̃.
