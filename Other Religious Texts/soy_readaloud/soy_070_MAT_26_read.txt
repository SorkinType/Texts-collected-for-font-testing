Mɑtiyee.
26.
Kɛ Yeesu uu lɛ̃ picélɑɑ ḿ mɑsí, uu pɛɛ upipirɛtiki mɑɑ rɛ: 
Nɔ yɛ̃ tɛ li tisɛ siyɑ́ɑ sitɛ́ pikɛ́ nɛ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ li. Kei kɑpi Usoi Kɛpipi pilɑ́ɑrɔ ɑnipɛ-i wɑinɛ, pɛ̃ pikɛ́ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɛ kɑrii. 
Kei kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ ɑpi Uléécɑɑ usinɑ kɑpi yɛ n sée rɛ Kɑyifu kɛyɔcɔpɛ cɑ́pinɛ, 
ɑpi rinɔ́ɔ risɛ wɑi rɛ pikɛ́ mɛyɑɑ Yeesu tĩ pikɛ́ kpu. 
Api pɛɛ rɛ: Amɑ́ ɑ́i kɑpɛ pɛɛ ɑnyɑ̃́ kɛ́mɛɛ. Insɑ́ lɛ̃, pisoi pɛɛ n cɑ́pinɛlɛ̃ yɛ́ yisi ɑpi nyɛrɛ. 
Yeesu uu Petɑnii kɛtẽ-mɛ̃ sĩ uú hɑ uyɔ̃́ Simɔɔ kɛyɔ-i topilɛ. 
Kɑpi n tṹ pin ilukɛ le, unɔ́si unyinɛ uu kɛnúḿpi sɔnɛ kɛnyinɛ kɛ́mɛɛ tulɑɑli nɛ́íkɔɔ́ píimɑ unyinɛ múípɔ uú nɛ sɔ́nɛpɔ uú nɛ Yeesu kɔ. Uu Yeesu riyu kɛcɑ́ɑ́ tulɑɑli uyɛ̃ kɔɔni. 
Kɛ upipirɛtiki ɑpi lɛ̃ n yɛ̃́, ɑ́i nɛ pi risɑ́, ɑpi pɛɛ unɔ́si uyɛ̃ mɑɑ rɛ: Yo picɑɑinkɛɛ yɛ nkpéni mpí? 
Pi yɛ́ pɛɛ fe ɑpi tulɑɑli nkó yɑ́i nnɛ́í píimɑ ɑpi lisiwóó sɛ̃ píwɛkɔɔ́ hɔ́ɔ́nɛ. 
Yeesu uu céru rɛ lɛ̃ kɑpi símisi, uu pɛɛ pi mɑɑ rɛ: Yo nnyɑ kɑni lɛ̃ unɔ́si uyɛ̃ cɔ́ŋlɛ̃? Mɛwɑi sɔnɛ kuu nɛ́ wɑ. 
Likumúŋɛ́ rɛ ḿpɑ́ píyei kɛ nɔ́ nɛ píwɛkɔɔ́ ɑní we, ɑ́i ḿpɑ́ píyei kɛ tɔ́ nɛ nnɔ̃́ ɑrí we. 
Inɛ́piŋɛ kuu mɛncirɛ tulɑɑli fṍ pikɛ́ kɛlenɛ nhórɛ nɛ́ tɑ̃. 
Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ ḿpɑ́ yei kɑpi kɛtẽ nnɛ́í kɛcɑ́ɑ́ Nsímɛ́ Kɛcirɛ yóólɛ̃, pi yɛ́ɛ́ unɔ́si nkó mɛwɑi mmɛ́ símisi ɑpí nɛ u n léisɛlɛ̃. 
Kei kɛ Yeesu pipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ usɛ kɑpi yɛ n sée rɛ Yutɑsi Isikɑriyɔɔti uu pɛɛ yisi uu pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ lɛɛpɔ 
uu pi pisɛ rɛ: Nɛn Yeesu nɔ́ɑnɛ́nípɛ-i n wɑ, yo kɑni yɛ́ nɛ́ pɑ? Pɛ̃ ɑpi mɛsɛ nɛ mɛsɛ kei nní nwóóweni mɛpipi ɑfɛɛtɑɑni kɛɛni ɑpi u múísɛ. 
Kɑí nɛ kei n kpísi, Yutɑsi uu pɛɛ icée n wɛ́ɛ́si ukɛ́ nɛ Yeesu piɑnipɛ-i n wɑ. 
Anyɑ̃́ nyɛ-i kɑpi yɛ ɑkpɔ́nɔ́ nyɛɛ ɑ́ɑ pitɛ m mɑ́ n li kɛyɑɑ foí kɛ Yeesu pipirɛtiki ɑpi ukɛkúrí hɑ́pɔ, ɑpi u pisɛ rɛ: Yei kɑɑ́ lɑ tɔkɛ́ ɑnyɑ̃́ nyɛ̃ ilukɛ pɔ́ wɑ? 
Uu pi pɛsɛ rɛ: Ani kuyu-i kɔɔ́ kɛyɔ-i hɑpɔ ɑni u mɑɑ rɛ: Ucélɑɑ yɛ rɛ unkpɔ ituŋɛ yɛ tulɛ. Kɛpɔ́yɔ-i kɛ u nɛ upipirɛtiki ɑpi sɔ́nti pikɛ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ li! 
Upipirɛtiki ɑpi mɛsei lɛ̃ kuu pi ḿ mɑ wɑi, ɑpi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ilukɛ wɑi. 
Kɑi nnyɔ́ɔ́ ń wɑ, u nɛ upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ ɑpí tonɛ ɑpi ilukɛ n le. 
Kɛlukɛ kɛ-i kuu pi mɑɑ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ nɔ́kɛnɛ́cɔpɛ unyinɛ yɛ́ nɛ́ tĩ́ uu pinɛ́lɑ́ɑrɔ ɑnipɛ-i wɑi. 
Kuu lɛ̃ ḿ mɑ, upipirɛtiki ɑpɔɔ ɑɑ mɛyíkíyiki cɑɑi ɑpi usɛ usɛ u m písɛi rɛ: Upíimɑ, yɑrɛ nɛ̃́ nɛ́ɛ? Ucɔ uu kɔ rɛ: Upíimɑ, yɑrɛ nɛ̃́ nɛ́ɛ? 
Kei kuu pɛɛ pi pɛsɛ rɛ: Tɔ́ nɛ uyɛ̃ tɔ́ɔ nní ricɑ́ripi risɛ-i n le yɛɛ sɔ́nti ukɛ́ pinɛ́lɑ́ɑrɔ ɑnipɛ-i nɛ́ wɑ. 
Usoi Kɛpipi yɛ kɛtẽ nkɛ́ kɛcɑ́ɑ́ tɔ́sulɛ yɑrɛ lɛ̃ kɑi Nléécɑɑsimɛ́ kɛ́mɛɛ kɛkɛcɑ́ɑ́ ń wɔ́lɑɑlɛ̃ mɛcɔ. Amɑ́ uyɛɛ pilɑ́ɑrɔ ɑnipɛ-i kɛ n wɑinɛ yɛ íwɛ toporilɛ. Li kutɔsi we ɑ́pi kɑpɛ pɛɛ liute m mɑri. 
Yutɑsi yɛɛ pilɑ́ɑrɔ ɑnipɛ-i u n wɑinɛ pɔ́ɔ pɛɛ u pisɛ rɛ: Yɑrɛ nɛ́ɛ pɛɛ lo nɛ́ɛ, Upíimɑ? Yeesu uu rɛ: Yɑrɛ kɑɑ nní ḿ mɑ. 
Kɛ Yeesu nɛ upipirɛtiki ɑpi lɛ̃ n tṹ pin ilukɛ le, uu kukpɔ́nɔ́ kpísi uu kukɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu ku kpɔ́kɔrinɛ uu pi pɑ uu rɛ: Inɛ́piŋɛ yɛ nnyí, ɑni yɔsí ɑni li. 
Uu lɛ̃ mɛmɑ́ɑ́ kɛpóripi kpísi uu kɛkɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu pi pɑ uu rɛ: Ani yɔsí ɑni nɔ́nnɛ́nɛ́í n ntí. 
Mɛnɛ́nyɛ yɛ mmɛ́ mɛɛ n kɔɔnu pisoi mɛyɑ̃ nnyɑ rɛ Uléécɑɑ ukɛ́ piɑkópɛ pi sɑ́rɛi. Mɛ́nyɛ mɛ̃ kɑí nɛ ritiki Uléécɑɑ uú nɛ pisoi séi rɛ u nɛ pɛ̃ pikɛ́ kómɛinɛ. 
Kɛ́ nɔ́ símisi rɛ kɑí nɛ nɛni rimpɑ́, ɑ́m ɑléépipi pitɑ mpí pinírɛ́ kpɑ́nɛ likɛ́ hɑ nɛ tu kɛyɑ́ɑ kɛ̃ kɛ tɔ́ nɛ nnɔ̃́ ɑri yɛ́ hɑ kɔ pítɑ fɑlɛ Unɛ́sɑ́ɑ iyɔɔpi kɛ́mɛɛ ń ntí. 
Kuu lɛ̃ pisímɛ́ pi ḿ mɑsí ɑpi pɛɛ ipɑkɑrɛ siyomɛ n yom pin pɑkɑrɛntɛ, ɑpi wɑi ɑpi yisi ɑpi Olifiyee rikúú tɑɑ́. 
Kei kɛ Yeesu uu pɛɛ upipirɛtiki mɑɑ rɛ: Nɔ́ nɔ́nnɛ́nɛ́í kɛsinɛ nkɛ́ nɛ́ ŋmɛriyɛ ɑni wúrunkɛɛ. Li Uléécɑɑ ritɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Nɛ́ usɛɛsẽ́ kpu, kusɛɛlee isɛɛ nnɛ́í ii kpíi. 
Yeesu uu kɔ kpɑ́ rɛ: Amɑ́ nɛn píyei nkpɔ kɛ́mɛɛ piyisɛ m mɑsí, nɛ́ hɑ Kɑlilee kɛtẽ-pɔ nɔ́ m mɛ̃́. 
Piyɛɛ uu rinɔ́ɔ yɔ́su uu rɛ: Ḿpɑ́ pisoi nnɛ́í pin pɔ̃́ nnyɑ kɛlólɛ́ n yɛ̃, nɛ̃́ ɑ́m yɛ́ píkɑi loó. 
Yeesu uu u pɛsɛ rɛ: Asei kɛcɑ́ɑ́, kɛ́ pɔ́ símisi rɛ pɔ́ kɛsinɛ nkɛ́ cirɛ kɛsi mɛtɑɑni rɛ ɑ́ɑ nɛ́ nyu, icɑ̃́ ii kɛlenɛ kooi. 
Kei kɛ Piyɛɛ uu pɛɛ u mɑɑ rɛ: Ḿpɑ́ lin nkpɔ yɛ mu, tɔ́ nɛ mpɔ̃́ tɔkɛ́ kɛsẽ́ kpu, ɑ́m kuŋmɛriyɛ pɔ́ we. Pipirɛtiki nnɛ́í ɑpi limɛcɔ mɑlɛ. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu nɛ upipirɛtiki ɑpi kɛlõ kɛnyinɛ kɑpi yɛ n sée rɛ Kɛtisemɑnee kɛ́mɛɛ tuipɔ. Uu upipirɛtiki mɑɑ rɛ: Ani nté n tũ nɛ́ɛkɛ́ rinyɔsɔ́ kɛ́ hɑ kɛyómɛ yɑ́ɑ́si. 
Uu Piyɛɛ nɛ Sepetee piŋmɑ́nɛ kɛtɛ́ kpísi uú nɛ pɛ́nɛ. Nfɑsimɛ́ píimɑ nɛ mpɔ́ɔcɑɑi nnyinɛ nn pɛɛ u tini. 
Kei kuu pɛɛ pi mɑɑ rɛ: Tinɛ́pɔ́ɔ yɛ cɑɑilɛnlɛ hɑ́i lin nɛ nkpɔ mɛnyínɛ we. Ani nté n tũ, ɑ́ni kɑpɛ lɔpilɛ, tɔkɛ́ kɛsẽ́ n nyɑ́ni. 
Kei kuu nkɑ́ripi nɛ kɛtɑɑ pi wɑ, uu wúlɑ uu kɛ́yu kɛtẽ kipi uu Uléécɑɑ yɑ́ɑ́si rɛ: Unɛ́sɑ́ɑ, pɔ́ fe ɑɑ ḿpɑ́ yo wɑi, ɑ nɛ ntóósi kɛporipi nkɛ́ kɛtɑɑ nɛ́ wɑ. Ḿpɑ́ nɛ lɛ̃, kɑpɛ mɛnɛ́pɔ́ɔlɑ wɑ, ɑmɑ́ mɛpɔ́kɔ́ wɑ. 
Kuu piyómɛyɑ́hɑɑ ḿ mɑsí, uu upipirɛtiki pɛ̃ kɛ́mɛɛ pɛɛpɔ, uu pi lɛɛpɔ pin lɔni. Uu pɛɛ Piyɛɛ pisɛ rɛ: Lɛ̃ nnyɑ, ɑ́ni nkpɑ́ni fe tɔ́ nɛ nnɔ̃́ tɔkɛ́ kɛnɑmpi kɛsɛ kɛsẽ́ n nyɑ́nii? 
Ani n tũ nɔn nyɑ́ni nɔn kɛyómɛ yɑ́ɑ́si ɑ́ni kɑpɛ nɛ pipeikɛɛ kɛ́mɛɛ ń loó nnyɑ. Li kɛsoipipi ripɔ́ɔ we rɛ kɛkɛ́ yɛ lisɔnɛ wɑ, ɑmɑ́ kɛ pɛɛ rinɑ́rɛi mɑ́lɛ. 
Uu kɔ mɛlírũ nkɑ́ripi nɛ kɛtɑɑ pi wɑi, uu kɔ Uléécɑɑ yɑ́ɑ́si rɛ: Unɛ́sɑ́ɑ, in tɛ ɑ́ɑ yɛ́ ŋmurɛi ntóósi kɛporipi nkɛ́ kɛkɛ́ nɛ kɛtɑɑ nɛ́ wɑ rɛ nɛ mɛnírɛ́ kɛ mɑ́lɛ, mɛpɔ́lɑ mɛkɛ́ wɑ. 
Uu pɛɛri uu kɔ pi lɛɛri pin lɔni, likumúŋɛ́ rɛ piinipɛɛ yɛ mulɛsilɛ. 
Uu kɔ mɛtɑ́ɑ́nũ piyisɛ kpɑ́ uú nɛ kɛtɑɑ pi wɑi, uú hɑ kɔ mɛsɛ mɛ̃ nɛ Uléécɑɑ téni 
uu pɛɛ kɔ upipirɛtiki kɛ́mɛɛ pɛɛpɔ uu pi mɑɑ rɛ: Nɛ́ntɛ nɔ nkpéni lɔnilɛ nɔn wéntɛlɛ̃? Ituŋɛ yɛ nkpéni tulɛ rɛ pikɛ́ Usoi Kɛpipi tĩ pikɛ́ pikópɛkɔɔ́ ɑnipɛ-i wɑ. 
Ani yisi tɔkɛ́ u tɛ́pɛi. Uyɛɛ nɛ́ n kpísinɛ ukɛ́ pikópɛkɔɔ́ ɑnipɛ-i wɑ yɛ nyɑhɑimɛlɛ. 
Kɛ Yeesu uu lɛ̃ ń we un símisi, upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ kɛcɔpɛ usɛ kɑpi yɛ n sée rɛ Yutɑsi un tuipɔ nɛ risoiwuí, ɑtɛ́hɛ nɛ ɑpúnípi ɑn mulɛilɛ̃. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pisuifi piwɛ́ɛ́sɛ pɛɛ lɛ̃ pi tumpɔ. 
Yutɑsi uyɛɛ pɛɛ piɑnipɛ-i Yeesu n wɑinɛ yɛ pɛɛ risoiwuí céesilɛ lɛ̃ kɑpi yɛ́ nɛ Yeesu n céri pikɛ́ tĩ́. Uu pɛɛ pi mɑɑ rɛ: Nɛn hɑ úye rimpirɑ ɑḿ u yɑ́ɑ́si, uyɛɛ lo. Ani u tĩ. 
Kuu n tuipɔ, uú nɛ Yeesu kɛ́mɛɛ n sɑirilɛ̃ uú hɑ u mɑɑ rɛ: Nɛ pɔ́ yɑ́ɑsi, Sɑ́ɑ! Uu pɛɛ u pirɑ uu u yɑ́ɑ́si. 
Yeesu uu pɛɛ u mɑɑ rɛ: Unɛ́sɑ́nɛ, ɑ pikɛi pɛ̃ kɑɑ piwɑi ń kɑ mɛkɛɛ wɑ! Kei kɛ pɛ̃ ɑpi pɛɛ cɑ́pinɛpɔ, ɑpi Yeesu tini ɑpi múlú. 
Kei kɛ Yeesu upirɛtiki usɛ uu risɑɑpɔ uu ritɛ́hɛ kpeéu uu Uléécɑɑ usinɑ ukɛikɔ́ kutu seḿ. 
Yeesu uu u mɑɑ rɛ: A ripɔ́tɛ́hɛ tɛ̃ tikukɑi-i pɛsɛpɔ. Li we rɛ pɛ̃ nnɛ́í pɛɛ nɛ ritɛ́hɛ n kóni, titɛ́hɛ kɑpi yɛ kɔ nɛ pɛ̃ kpu. 
Áɑ nyu rɛ nɛn pɛɛ nté Unɛ́sɑ́ɑ m pisɛ rɛ ukɛ́ nɛ́ lɛ́mmɛ, u yɛ́ pɛɛ nté nní piléécɑɑtumɛ nɛ́ pɑmɛ ɑi inɔŋ kɛfi nɛ itɛ́ fɑ́ɑuu? 
Nɛn nkpéni lɛ̃ n wɑ, li yɛ́ pɛɛ mmɛ̃ kɑpi nní Uléécɑɑ ritɛlɛ́-i n wɔ́i rɛ nní kɑi nní n wɑi kɑi pisɛ rɛ likɛ́ wɑ kɛcɑ́ɑ́ ritikii? 
Yeesu uu pɛɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ mɑɑ rɛ: Nɔ ɑtɛ́hɛ nɛ ɑpúnípi nɛ pitínɛ nɛ́ kɑ yɑrɛ usoi kpɑ́ɑ́ree unyinɛ kɑni pitínɛ kɑ. Ḿpɑ́ kɛyɑ́ɑ kɛ́ye kɛ tɔ́ nɛ nnɔ̃́ ɑrí pɛɛ Uléécɑɑ kɛyɔ-i we nɛn céési, ɑ́ni nɛ́ tĩ́. 
Amɑ́ lɛ̃ kɛ ɑntepuyɛ ɑpi Nléécɑɑsimɛ́ kɛ́mɛɛ n wɔ́i likɛ́ nɛ n wɑ nnyɑ kɑi lɛ̃ wɑ. Kei kɛ upipirɛtiki nnɛ́í ɑpi pɛɛ u ŋmɛriyɛ ɑpi wuru. 
Pɛpɛɛ Yeesu n tĩ́ yɛ pɛɛ u kpísilɛ ɑpí nɛ Uléécɑɑ usinɑ Kɑyifu kɛyɔ sĩ. Kei kɛ isé picélɑɑ nɛ Pisuifi piwɛ́ɛ́sɛ ɑpí pɛɛ cɑ́pinɛlɛ̃. 
Piyɛɛ uu kɛtɑɑ-pɔ Yeesu n tikilɛ̃ uú nɛ Uléécɑɑ usinɑ kɛyɔcɔpɛ tulu, uu lompɔ uu pimɛ́rɛ́ kɛkúrí tonɛ un nɛ mɛ̃́ ukɛ́ yɛ̃́ lɛ̃ kɑi yɛ́ nɛ kɛ́tɔ ń kpu. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ pitúhɑɑnɛ píimɑ nnɛ́í ɑpi pisoi wɛ́ɛ́si pɛɛ yɛ́ Yeesu nɛ nsímɛ́ rinkpɑ́rɑ́ pikɛ́ nɛ u ń kpu nnyɑ. 
Píseérɑkɔɔ́ nɔɔmɛ ɑpi cɑ́pinɛpɔ. Ḿpɑ́ nɛ lɛ̃, ɑ́pi ńkɑ yɛ̃ pikɛ́ nɛ u rikpɑ́rɑ́. Kɑi yɛ́ rinkpɑ́sɛ, pitisi pitɛ́ pinyinɛ ɑpí hɑpɔ 
ɑpi rɛ: Usoi nkó yɛ mɑɑ rɛ u yɛ́ fe uu Uléécɑɑ kɛyɔ foru uu siyɑ́ɑ sitɑɑni nɛ pimɔ́mɛ kɛ pɛsɛ. 
Kei kɛ Uléécɑɑ usinɑ uu pɛɛ yisi uu u mɑɑ rɛ: Nsímɛ́ mmú nnɛ́í kɛ pisoi mpí ɑpi nní kɛpɔ́cɑ́ɑ́ n símisi kɛ́mɛɛ, ń-ye kɑɑ yɛ̃́ pɔkɛ́ pɛsɛ? 
Yeesu uú n sɛ́ɛ́lɛ̃ lɛ́kɛ́-lɛ́kɛ́. Uléécɑɑ usinɑ uu pɛɛ u mɑɑ rɛ: Nɛ lɑ rɛ ɑ Uléécɑɑ yɛɛ n nyɑ́ni rinyiri sée ɑɑ́ nɛ wééri, in tɛ pɔ̃́ kɛ Uléécɑɑ uu wɛ́ɛ. Nɛ́ɛ pɔ́ɔ Uléécɑɑ Kɛpipii? 
Yeesu uu u pɛsɛ rɛ: Lɛ̃ kɑɑ nní mɑ. Limɛmɑ́ɑ́, nɛ́ kɔ nɔ́ rikpɑ́sɛ rɛ kɑí nɛ nɛni n kpísi, nɔ́ Usoi Kɛpipi yɛ̃ kɛn Uléécɑɑ, ńnɑŋɛ nnɛ́í ute kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tṹ. Nɔ́ kɔ u yɛ̃ un kɛyómɛcɑɑ-pɔ ɑhopɛ kɛcɑ́ɑ́ tṹ un nɛ wemɛ. 
Kei kɛ Uléécɑɑ usinɑ uu pɛɛ uricuruu ilũ kuu n tɑnɑɑlɛ̃ cɛ́ɛ́rii ɑi nyísɛ rɛ li u tóósilɛ hɑ́i. Uu pɛɛ rɛ: Uléécɑɑ ŋmɑɑ kuu nní lɑ́mɑɑnkɛɛ. Ái nkpɑ́ni nkpéni pisɛ rɛ tɔkɛ́ píseérɑkɔɔ́ pinyinɛ wɛ́ɛ́si. Nɔ́rinɛ́cúruu yɛ ɑlɛ̃́ kuu Uléécɑɑ n lɑ́mɑɑnkɛɛ konlɛ. 
Lɛ̃ nnyɑ, íye kɑni músu? Api u pɛsɛ rɛ: Li nɛ sɑ́ rɛ pikɛ́ u kpu. 
Api pɛɛ mɛtɑ kɛ́yu-i pitúkɛntɛ u loni. Api ɑkpini u tini, picɔ ɑpi sipɛsɛ u kɑ́pɑ́ɑ́ 
ɑpi yɛ pɛɛ rɛ: Pɔ̃́ kɛ Uléécɑɑ uu n wɛ́ɛ, ɑ ŋmɔinu ɑɑ rɔ́ símisi nkɛ̃́ tɛ wóo pɔ́ pi? 
Kɛ Piyɛɛ uu kei lɛ̃ tipoo-i n tṹ, unɔ́si kɛikɔ́ unyinɛ uu léepɔ uu kei u lɛɛpɔ uu u mɑɑ rɛ: Nɛ pɔ́ɔ lo! Pɔ̃́ nɛ kɔ pɛɛ Kɑlilee ukɔ́ Yeesu uyɛ̃ tikilɛnlɛ. 
Uu pinnɛ́í inipɛɛ iyɛ-i kɛ́su uu rɛ: Nɛ̃́ ɑ́m nyu ń-ye kɑɑ símisi. 
Uu limɛmɑ́ɑ́ yisi uu rinɔnɔɔ-mɛ̃ sĩ. Unɔ́si kɛikɔ́ ucɔ pɔ́ɔ kɔ kei piyɛ́nɛ u kpɑ́ uu pɛpɛɛ kei n cɑ́pinɛlɛ̃ mɑɑ rɛ: Nkó pɔ̃́ nɛ kɔ pɛɛ Nɑsɑrɛti ukɔ́ Yeesu uyɛ̃ tikilɛnlɛ. 
Piyɛɛ uu kɔ pikɛ́ŋɛ kpɑ́ uu wééri rɛ: Ám usoi nkó nyu. 
Likɛ́ nɛ rinsɑ́sɛ, pɛpɛɛ kei ń we ɑpí nyɔsɔ́pɔ ɑpí nɛ u kɔ ɑpi u mɑɑ rɛ: Asei kɛcɑ́ɑ́, pɔ pisoi pɛ̃ usɛ lɛ. Mɛpɔ́símɛ́ kɛ́mɛɛ kɑi lɛ̃ nyísɛlɛ̃. 
Kei kɛ Piyɛɛ uu kɛ́si ŋmɑɑ uú nɛ kɛtẽ kpɛɛni uu wééri rɛ: Nɛ Uléécɑɑ, ɑ́m usoi nkó nyu. Mɛsɛ nɛ mɛsɛ, icɑ̃́ ii kooi. 
Piyɛɛ uu rinɔ́ɔ tɛ̃ kɛ Yeesu uú pɛɛ u ḿ mɑ léisɛ rɛ: Icɑ̃́ ikɛ́ kɛlenɛ n kooi, pɔ́ kɛsi mɛtɑɑni rɛ ɑ́ɑ nɛ́ nyu. Kei kɛ Piyɛɛ uu léepɔ uú nɛ mpɔ́ɔcɑɑi téni.
