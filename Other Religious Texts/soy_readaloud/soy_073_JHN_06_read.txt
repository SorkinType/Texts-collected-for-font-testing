Yohɑni.
6.
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu Kɑlilee kupiyɛ kɑpi yɛ kɔ n sée rɛ Tiperiyɑɑti mɛnimɑɑ tɛ́ŋ́ uu ritímɛ ricɔ-mɛ̃ sĩ. 
Pisoi kulúi píimɑ pinyinɛ yɛ pɛɛ u tikilɛnlɛ, kɑpi n yɛ̃́ tɛ u mɛwɑisɑŋɑ wɑpisi un pitóikɔ́ pɔisɛntɛ nnyɑ. 
Kei kɛ Yeesu uu pɛɛ riyɔ́pɛ ritɑɑ́ u nɛ upipirɛtiki ɑpí hɑ tonɛ. 
Pisuifi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ yɛ pɛɛ́ uyɛ-i nyɑhɑimɛlɛ. 
Yeesu uú weríí uu yɛ́nu rɛ pisoi kulúi píimɑ yɛ ukɛmɛɛ sɔ́nɑɑpɔ, uu pɛɛ Filipu mɑɑ rɛ: Yei kɑri yɛ́ ilukɛ yɛɛ yɛ́ pisoi mpí nnɛ́í mɛlukɛ n tu lɔ? 
Sikɔ́rí kuú nɛ lɛ̃ mɑ ukɛ́ peẽ ukɛ́ yɛ̃́ tɛ íye kɛ Filipu uu yɛ́ u pɛsɛ. Uricuruu yɛ pɛɛ nyulɛ lɛ̃ kuu n wɑinɛ. 
Filipu uu u pɛsɛ rɛ: Ḿpɑ́ tɔkɛ́ nwóóweni mɛpipi pílɛ pílɛ mɛtɛ́ nɛ ilukɛ lɔ ɑ́i nkpɑ́ni ḿpɑ́ úye nkɑ́ripi nkɑ́ripi mɛlukɛ tulunɛ. 
Uupirɛtiki ucɔ, Simɔɔ Piyɛɛ uwɑ̃ kɑpi yɛ n sée rɛ Antiree pɔ́ɔ pɛɛ u mɑɑ rɛ: 
Kɛwɑ̃́ ŋmɑ́nɛpi kɛnyinɛ yɛ nkɛ́ nté kɛn ɑkpɔ́nɔ́ ɑnupũ nɛ ikpíntomɛ́ itɛ́ múílɛ̃. Yɑrɛ yo kɛ lɛ̃ ɑi yɛ́ nkpɑ́ni pisoi kulúi píimɑ nkú wɑ? 
Kei kɛ Yeesu uu pɛɛ rɛ: Ani pisoi pɛ̃ mɑɑ rɛ pikɛ́ tonɛ. Mɛyúi ɑ́mɛ pɛɛ kusɑ kɛlõ kɛ-i piyɛ. Api pɛɛ túnti, ɑí lelu pisoi ɑ́kotokú ɑnupũ kumúŋɛ́ (5.000). 
Kei kuu ɑkpɔ́nɔ́ kpísi uu ɑkɛcɑ́ɑ́ Uléécɑɑ pɔɔnɛsɛ uu pɛɛ pɛpɛɛ kei ń we ɑ hɔɔnɛ. Uu kɔ limɛcɔ ikpíntomɛ́ wɑi, ɑi pinnɛ́í mɛlukɛ tulu. 
Kɑpi pinnɛ́í ń li ɑpi lɛpu, Yeesu uu pɛɛ upipirɛtiki mɑɑ rɛ: Ani ɑkéri tɔrɔɔ kɔ́ɔ́i, líkɑ ɑ́i kɑpɛ pɔ. 
Api ɑkpɔ́nɔ́ ɑnupũ nyɛ̃ kɑpi n li ɑtɔrɔɔ kɔ́ɔ́i ɑi ɑnɛ́rɛ kɛfi nɛ ɑtɛ́ yipu. 
Kɛ pisoi pɛ̃ ɑpi mɛwɑisɑŋɑ kɛ Yeesu uu n wɑpisi n yɛ̃́, ɑpi pɛɛ́ n símɑɑnkɛɛ rɛ: Mɛsei, usoi nkó yɛɛ ɑntepu kɑpi n tee rɛ u kɛtẽ kɛcɑ́ɑ́ sɔ́nti. 
Yeesu pɔ́ɔ céru rɛ pi sɔ́ntilɛ pikɛ́ u tĩ pikɛ́ ńnɑŋɛ iyɔ́ɔpi u tonsɛ. Uu lɛ̃ nnyɑ riyɔ́pɛ pítɑɑ́ pɛlɛ uú hɑ umɛcirɛ n tũ. 
Kɑi nnyɔ́ɔ́ n wɑ, Yeesu pipirɛtiki ɑpi cɛ́pi ɑpí hɑ kupiyɛ ritimɛ n we. 
Api kúninɔi loni ɑpi kupiyɛ n tɛ́ŋ́lɛ̃ pin Kɑpɛrinɑwum kuwɛ́ɛ́-mɛ̃ sĩ. Kɛsinɛ yɛ pɛɛ likumúŋɛ́ kpɛ-i piwɑi mɑsilɛ, Yeesu úu kɔ pɛɛ kɑhɑnɛ pikɛkúrí n hɑpɔ. 
Kei kɛ mɛyɔ píimɑ mɛnyinɛ ɑmɛ pipépɛ lõ, míni ɑmɛ pɛɛ́ n tɔɔnkɛɛ. 
Kɑpi kúninɔi rintíkíí yɑrɛ kilomɛtiri pinupũ kumúŋɛ́, ɑpi Yeesu yɛnu un míni kɛcɑ́ɑ́ sɔ́nɛ un nɛ pikɛkúrí wépɔ, iwɑmɛ ii pi wɑi. 
Kei kɛ Yeesu uu pɛɛ pi mɑɑ rɛ: Nɛ́ɛ lo, iwɑmɛ íi kɑpɛ nɔ́ wɑ! 
Api pɛɛ́ n lɑ pikɛ́ kúninɔi kɛ́mɛɛ u kpísi. Amɑ́ kúninɔi ɑku kei nní mɛsɛ nɛ mɛsɛ kei kɑpi n sĩ́ tulu. 
Kɑi kóso n weesi, pisoi riwúí tɛɛ kupiyɛ ritimɛ tɛ-i m mɛ́sũ ɑri yɛ́nu rɛ kúninɔi kusɛ kpɛɛ yɛ kei n we, Yeesu úu kɔ ku lõ u nɛ upipirɛtiki pikɛ́ nɛ kɛsẽ́ tɔ́mpɔ. Upipirɛtiki ŋmɑnɛ pɛɛ pɛɛ tɔ́mpɔ. 
Amɑ́ ɑ́ninɔi ɑcɔ yɛ pɛɛ Tiperiyɑɑti kɛtẽ-mɛ̃ leemɛlɛ ɑɑ kei kɛ Upíimɑ uú pɛɛ ɑkpɔ́nɔ́ kɛcɑ́ɑ́ Uléécɑɑ m pɔɔnɛsɛ ɑpi pɛɛ ɑ le tuipɔ. 
Kɛ pisoi riwúí tɛ̃ ɑri n yɛ̃́ tɛ Yeesu nɛ upipirɛtiki ɑ́pi kei we, ɑpi pɛɛ piricuruu ɑ́ninɔi nyɛ̃ loni ɑpí nɛ Kɑpɛrinɑwum-pɔ Yeesu piwɛ́lɑɑ sĩ. 
Api kupiyɛ ritimɛ ricɔ-mɛmpɔ Yeesu lɛɛpɔ ɑpi u pisɛ rɛ: Píyei kɑɑ nté tuimɛ, Ucélɑɑ? 
Yeesu uu pi pɛsɛ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ ɑ́i rɛ kɑni mɛwɑisɑŋɑ kɑm n wɑpisi n yɛ̃́ nnyɑ kɑni nɛ́ wɛ́ɛ́si, ɑmɑ́ kɑni ɑkpɔ́nɔ́ ń li ɑni lɛpu nnyɑ kɑni nɛ́ wɛ́ɛ́si. 
Áni kɑpɛ ilukɛ yɛɛ yɛ n fɔ́ni nnyɑ n nɑ́ɑ́si, ɑmɑ́ ɑni ilukɛ yɛɛ íi yɛ píkɑi n fɔ́ni ii yɛ kɔ pɛɛ nfɑ́ɑ tɛnɛcirɛ́ hɛ nnyɑ n nɑ́ɑ́si. Usoi Kɛpipi kɛɛ yɛ́ ilukɛ iyɛ̃ nɔ́ hɛ, li we rɛ Sɑ́ɑ Uléécɑɑ yɛ unwɔi ukɛcɑ́ɑ́ wɑlɛ. 
Api u pisɛ rɛ: Íye kɑri yɛ́ wɑ tɔkɛ́ nɛ pikɛi pɛ̃ kɛ Uléécɑɑ uu ń lɑ piwɑi n fe? 
Yeesu uu pi pɛsɛ rɛ: Uléécɑɑ pikɛi kuu nɔ́ m pisɛ yɛ mpí rɛ ɑni nɛ uyɛ̃ kuu n tummɛ kɛfɑ tɛnɛ. 
Api u mɑɑ rɛ: Mɛwɑisɑŋɑ mɛ́ye kɑɑ yɛ́ wɑ tɔkɛ́ yɛ̃́ tɔkɛ́ pɛɛ́ nɛ kɛfɑ pɔ́ tɛnɛ? Mɛwɑi mɛ́ye kɑɑ wɑinɛ? 
Pirɔ́sɑ́ɑyɑhɑ yɛ iléécɑɑlukɛ kucɛsi kóimɑ-i lilɛ yɑrɛ kɑi Nléécɑɑsimɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ tɛ: Ilukɛ yɛɛ kɛyómɛcɑɑ-pɔ n léeri kuu pi hɛ rɛ pikɛ́ li. 
Yeesu uu pi pɛsɛ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi ɑni kóm tɛ ɑ́i Moisi yɛɛ iléécɑɑlukɛ nɔ́ hɛ, ɑmɑ́ Unɛ́sɑ́ɑ yɛɛ ilukɛ yɛɛ mɛyíkíyiki kɛyómɛcɑɑ-pɔ n léeri nɔ́ hɛ. 
Ilukɛ kɛ Uléécɑɑ uu yɛ n hɛ yɛɛ nní uyɛɛ kɛyómɛcɑɑ-pɔ n léeri uu yɛ pisoi nfɑ́ɑ hɛ. 
Api u mɑɑ rɛ: Upíimɑ, ɑ mɛsɛ́rɛ ilukɛ iyɛ̃ tɔ́ n hɛ. 
Kei kɛ Yeesu uu pɛɛ pi símisi rɛ: Nɛ́ɛ nní lɛ̃ ilukɛ iyɛɛ yɛ nfɑ́ɑ n hɛ. Nkṹ ńn píkɑi nkó yɛɛ kɛnɛ́mɛɛ ń kɑ wɑinɛ. Nnírɛ́ ńn kɔ píkɑi nkó yɛɛ nɛ kɛfɑ nɛ́ n tɛnɛ wɑinɛ. 
Amɑ́ nɛ nɔ́ símisilɛ: Nɔ nɛ́ yɛ́nlɛ, ḿpɑ́ nɛ lɛ̃, ɑ́ni nɛ kɛfɑ tɛnɛ. 
Pɛ̃ nnɛ́í kɛ Unɛ́sɑ́ɑ uu nɛ́ m pɑ yɛ kɛnɛ́mɛɛ sɔ́ntilɛ, ɑ́m pikɛcɔpɛ úkɑ mpuri mɛ́yulɛ mɑ́. 
Li we rɛ ɑ́m kɛyómɛcɑɑ-pɔ súimɛ rɛ kɛ́ mɛnɛ́lɑ wɑ, ɑmɑ́ uyɛɛ nɛ́ n tummɛ mɛlɑ kɑm piwɑi kɑ. 
Uyɛɛ nɛ́ n tummɛ mɛlɑ yɛ kɔ pɛɛ mmɛ́ rɛ ɑ́m kɑpɛ pɛ̃ kuu nɛ́ m pɑ kɛcɔpɛ úkɑ fóm, ɑmɑ́ u lɑ rɛ kɛ́ hɑ kɛyɑ́ɑ tɔrɔɔ kɛtúŋɛ́ nkpɔ kɛ́mɛɛ pi yukusɛ. 
Unɛ́sɑ́ɑ yɛ lɑ rɛ nkó yɛɛ Ukɛpipi n yɛ̃́ uú nɛ kɛfɑ kɛ tɛnɛ ukɛ́ nfɑ́ɑ tɛnɛcirɛ́ m mɑ́. Nɛ kɔ pɛɛ liute kɛyɑ́ɑ tɔrɔɔ kɛtúŋɛ́ nkpɔ kɛ́mɛɛ yukusɛnɛ. 
Pisuifi ɑpi Yeesu nɛ ŋmúlɑɑnkɛɛ rɛ u mɑɑ rɛ: Nɛ́ɛ nní lɛ̃ ilukɛ iyɛɛ kɛyómɛcɑɑ-pɔ n léeri nnyɑ. 
Api pɛɛ́ m písɛinɛ rɛ: Ái Yosɛfi uŋmɑ́nɛ Yeesuu? Tɔ usɑ́ɑ nɛ úni nyulɛ. Íye kuu yɛ́ pɛɛ́ mɑɑ rɛ: Kɛyómɛcɑɑ-pɔ kuu léeri? 
Kei kɛ Yeesu uu pɛɛ pi mɑɑ rɛ: Ani lɛ̃ nɔ́mɛnɛ́cɔpɛcirɛ piŋmúlɑɑnkɛɛ tíyɛ. 
Úkɑ úu yɛ́ fe ukɛ́ kɛnɛ́mɛɛ kɑm, insɑ́ Sɑ́ɑ yɛɛ nɛ́ n tummɛ yɛɛ nɛ kɛnɛ́mɛɛ u kɑ. Nɛ́ kɔ ḿpɑ́ úye yɛɛ kɛnɛ́mɛɛ ń kɑ kɛyɑ́ɑ tɔrɔɔ kɛtúŋɛ́ nkpɔ kɛ́mɛɛ yukusɛ. 
Li ɑntepuyɛ ɑtɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Uléécɑɑ yɛ́ pinnɛ́í céési. Nkó yɛɛ Sɑ́ɑ kutu rińcɔ uu uicélɑɑ kom, yɛɛ yɛ́ kɛnɛ́mɛɛ kɑm. 
Lɛ̃ ɑ́i nyísɛ rɛ unyinɛ yɛ Sɑ́ɑ yɛnɑɑlɛ̃. Amɑ́ uyɛɛ Uléécɑɑ kɛ́mɛɛ n léeri mɛcirɛ yɛɛ u yɛnɑɑlɛ̃. 
Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi ɑni kóm tɛ nkó yɛɛ nɛ kɛfɑ nɛ́ n tɛnɛ yɛ nfɑ́ɑ tɛnɛcirɛ́ mɑ́lɛ. 
Nɛ́ɛ ilukɛ iyɛɛ yɛ nfɑ́ɑ n hɛ. 
Nɔ́pinɛ́sɑ́ɑyɑhɑ yɛ pɛɛ kucɛsi kóimɑ-i mɑɑni lilɛ, ḿpɑ́ nɛ lɛ̃, ɑpi kɔ pɛɛ kpíni. 
Ilukɛ iyɛɛ kɛyómɛcɑɑ-pɔ n léeri yɛ nnyí nté. Nkó yɛɛ i n li úu píkɑi kpinɛ. 
Nɛ́ɛ ilukɛ iyɛɛ kɛyómɛcɑɑ-pɔ n léemɛ ii yɛ pisoi nfɑ́ɑ hɛ. Ḿpɑ́ úye yɛɛ ilukɛ iyɛ̃ li, nfɑ́ɑ tɛnɛcirɛ́ kuu yɛ́ yɛ̃́. Inɛ́piŋɛ yɛɛ ilukɛ iyɛ̃ kɑm u n hɛnɛ. Nɛ i hɛlɛ rɛ kɛtẽ nnɛ́í pisoi pikɛ́ nfɑ́ɑ yɛ̃ nnyɑ. 
Kuu lɛ̃ ḿ mɑ, Pisuifi ɑpi picɛ́ŋɑɑnkɛɛ loni pin pimɛcɔpɛcirɛ písɛinɛ rɛ: Íye nnyɑ kɛ nkó uu yɛ́ mɑɑ rɛ u yɛ́ uipiŋɛ rɔ́ hɛ ɑri le? 
Kei kɛ Yeesu uu pɛɛ pi mɑɑ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi ɑni kóm tɛ nɔnsɑ́ Usoi Kɛpipi ipiŋɛ n li, nɔnsɑ́ kɔ kɛmɛnyɛ n ntí, ɑ́ni nfɑ́ɑ mɑ́. 
Usoi nkó yɛɛ inɛ́piŋɛ n li uu kɔ mɛnɛ́nyɛ níru yɛ nfɑ́ɑ tɛnɛcirɛ́ yɛ́nlɛ. Nɛ hɑ kɛyɑ́ɑ tɔrɔɔ kɛtúŋɛ́ nkpɔ kɛ́mɛɛ u yukusɛnɛ. 
Li we rɛ inɛ́piŋɛ yɛ ilukɛ yíkíyiki lɛ, mɛnɛ́nyɛ pɔ́ɔn kɔ linírɛ́-nirɛ́ yíkíyiki. 
Usoi nkó yɛɛ inɛ́piŋɛ n li uu kɔ mɛnɛ́nyɛ níru yɛ kɛnɛ́mɛɛ welɛ nɛ́ɛn kɔ ukɛmɛɛ we. 
Sɑ́ɑ yɛɛ nɛ́ n tummɛ yɛɛ nfɑ́ɑ ute. Uyɛ̃ nnyɑ kɑm nfɑ́ɑ mɑ́. Limɛcɔ, úye un inɛ́piŋɛ n li, u nɛ̃́ nnyɑ nfɑ́ɑ yɛnunɛ. 
Ilukɛ iyɛɛ kɛyómɛcɑɑ-pɔ n léeri yɛ iyɛ̃. Íi iyɛ̃ kɛ nɔ́pinɛ́sɑ́ɑyɑhɑ ɑpí pɛɛ n li, ḿpɑ́ nɛ lɛ̃, ɑpi kɔ pɛɛ kpíni mɛcɔ we. Amɑ́ nkó yɛɛ nnyí n li yɛ mɛsɛ́rɛ nfɑ́ɑ mɑ́nɛ. 
Kumúŋɛ́ kpɛ-i kɛ Yeesu uú pɛɛ Kɑpɛrinɑwum-pɔ Pisuifi kuyomɛyɑ́hɑɑlee-i pisoi n céési kuu ɑnɔ́ɔ nyɛ̃ mɑ. 
Kɛ upipirɛtiki ɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, pimɛyɑ̃ ɑpi rɛ: Nsímɛ́ mmú yɛ ḿpɑpɛ tɔsilɛ. Wóo yɛ́ mmú kutu n cɔlɛ̃? 
Yeesu uu céru rɛ upipirɛtiki yɛ nsímɛ́ mmɛ̃ kɛcɑ́ɑ́ ŋmúlɑɑkɛɛlɛ. Uu pɛɛ pi mɑɑ rɛ: Li nɔ́ ricɔ́ŋlɛ mɛyɑ̃́, nɛ́ɛ yo? 
In lɛ̃, nɔn píye Usoi Kɛpipi n yɛ̃ kɛn kei kɑkɛ́ pɛɛ ń we pítɑɑ́ m pɛlɛ ní, íye kɑi yɛ́ pɛɛ nɔ́ wɑ? 
Uléécɑɑ Nfɑɑsɔnɛ mɛɛ yɛ nfɑ́ɑ hɛ. Kɛsoipipi ɑ́kɛ yɛ́ líkɑ fe. Amɑ́ ɑnɔ́ɔ nnyɛ́ kɑm nní nɔ́ ḿ mɑ yɛ Nfɑ́ɑsɔnɛ ɑkɔ́ lɛ nyɛɛ yɛ nfɑ́ɑ n hɛ. 
Amɑ́ pinyinɛ ɑ́pi nɔ́kɛnɛ́cɔpɛ nɛ kɛfɑ nɛ́ tɛnɛ. Ái líkɑ nnyɑ kuu lɛ̃ mɑlɛ. U hɑ́i mɛkɛɛ-mɛ pɛpɛɛ ɑ́pi n ŋmurɛinɛ nɛ uyɛɛ n sɔ́nti ukɛ́ pilɑ́ɑrɔ ɑnipɛ-i u n wɑ nyulɛ. 
Uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Lɛ̃ nnyɑ kɑm nɔ́ mɑɑ rɛ úkɑ úu we yɛɛ yɛ́ n fe uu kɛnɛ́mɛɛ weri insɑ́ Unɛ́sɑ́ɑ yɛɛ ncée u hɛ. 
Kɑí nɛ kei n kpísi, upipirɛtiki mɛyɑ̃ ɑpi kɛpirɛ u tɔ ɑ́pi yɛ pɛɛ pitíki u rikpɑ́. 
Uu pɛɛ upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ pisɛ rɛ: Nɔ̃́ ní, nɔ̃́ ɑ́ni lɑ nɔkɛ́ tɔ́mpɔɔ? 
Simɔɔ Piyɛɛ uu u pɛsɛ rɛ: Upíimɑ, úye ucɔ kɛ́mɛɛ kɑri yɛ́ nkpɑ́ni hɑ? Kɛpɔ́mɛɛ́ kɛ nsímɛ́ mɛɛ yɛ nfɑ́ɑ tɛnɛcirɛ́ n hɛ nń we. 
Tɔ nɛ kɛfɑ pɔ́ tɛnɛlɛ ɑri yɛ́nu rɛ pɔ̃́ kɛ Uléécɑɑ uu wɛ́ɛ uu tumti. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Ái nɛ́ɛ nɔ́ kɛfi nɛ pitɛ́ mpí wɛ́ɛɛ? Ḿpɑ́ nɛ lɛ̃, usɛ yɛ nɔ́kɛnɛ́cɔpɛ kuníri lɛ! 
Simɔɔ Isikɑriyɔɔti uŋmɑ́nɛ Yutɑsi nsímɛ́ kuu lɛ̃ símisi. U kɛfi nɛ pitɛ́ pɛ̃ usɛ lɛ. Ḿpɑ́ nɛ lɛ̃, uyɛɛ sɔ́nti ukɛ́ pilɑ́ɑrɔ ɑnipɛ-i u wɑ.
