Nsímɛ́ Kɛcirɛ ritɛlɛ́ kɛ Mɑriki uu n wɔ́i.
1.
Nté kɑi wɔ́lɑɑlɛ̃ lɛ̃ kɛ Yeesu Kirisi, Uléécɑɑ Kɛpipi Nsímɛ́ Kɛcirɛ nn n kóri. 
Li ɑntepu Esɑyi ritɛlɛ́-i wɔ́lɑɑlɛ̃ tɛ: Nɛ́ unɛ́tumɛ kɛpɔ́kpéẽ wɑisɛ ukɛ́ ncée pɔ́ nyɔ́ɔnsɛ. 
Usoi rinɔɔ yɛ ri tɛɛ kucɛsi kóimɑ-i n yóólɛ̃ tɛ: Ani Upíimɑ ncee nyɔ́ɔnsɛ, ɑni usiceepi tɛ̃iisɛ. 
Lɛ̃ kɑi wɑ Yohɑni uu kucɛsi kóimɑ-i pisoi míni wolɛ, un pi símisi rɛ: Ani mɛfinɛ consɛ, ɑni tíyɛsɛ kɛ́ míni nɔ́ wolɛ, Uléécɑɑ uu nɔ́ɑnɛ́kópɛ nɔ́ sɑ́rɛi. 
Yutee kɛtẽ pikɔ́ nnɛ́í nɛ Yerusɑlɛm pikɔ́ nnɛ́í ɑpi ukɛmɛɛ́ n sɔ́nɑɑpɔ pin piɑkópɛ yɑ́mnɛntɛ, Yohɑni uu Yuritɛɛ nkoi kɛ́mɛɛ míni pi wolɛ. 
Kuyooyoo nkpíí kɛtukɑnkɑ kɛ Yohɑni uú pɛɛ tɑnɑɑlɛ̃ un kɛkónɛpɑmpi kɛhɑlɛ-i tɛ́lɛ̃. Apɑ́ɑ́ni nɛ mɛtɛ̃́ kuú pɛɛ le 
un yóólɛ̃ tɛ: Uyɛɛ nɛ̃́ mɛmɑ́ɑ́ n sɔ́nti yɛ ńnɑŋɛ nɛ̃́ felɛ. Inɛ́soi íi tu rɛ kɛ́ ripɛ kɛ́ ukunɛ́ɛ́ri nŋmɛ fénnɛ. 
Míni kɛ nɛ́ɛ nɛ nɔ́ wólɛ, ɑmɑ́ Nfɑ́ɑsɔnɛ kɛ uyɛ̃ uu yɛ́ nɛ nɔ́ wolɛ. 
Yeesu uu pɛɛ Kɑlilee kuyu kunyinɛ kɑpi yɛ n sée rɛ Nɑsɑrɛti leeri, Yohɑni uu Yuritɛɛ nkoi kɛ́mɛɛ míni u wolɛ. 
Kɛ Yeesu uu mɛsɛ nɛ mɛsɛ míni-i n léemɛ, uu kɛyómɛ yɛnu kɛn wúkulɛlɛ̃, Nfɑ́ɑsɔnɛ nn ukɛcɑ́ɑ́ súiri yɑrɛ tilɔ́ɔlɑ̃. 
Uu rinɔ́ɔ kɛyómɛcɑɑ-pɔ komti rɛ: Pɔ́ɔ Kɛnɛ́pipi lɑlɑ kɛcirɛ. Nɛ pɔ́ wɛ́ɛlɛ. 
Mɛsɛ nɛ mɛsɛ Nfɑ́ɑsɔnɛ nń nɛ kucɛsi kóimɑ-pɔ Yeesu sĩ. 
Uu kei n we hɑ́i nɛ siyɑ́ɑ ɑfɛɛnɑ. Kei kɛ Setɑni uu u peikɛɛ. Icɑrɛsɛɛ kɛpɑ́ɑ́pɑ́ɑ́ kuu yɛ n we, piléécɑɑtumɛ pin ukɛcɑ́ɑ́ pɑílɛ̃. 
Kɑpi Yohɑni kukpɑniilee n tɑ̃ mɛmɑ́ɑ́, Yeesu uu Kɑlilee sĩ uu Uléécɑɑ Nsímɛ́ Kɛcirɛ n yóólɛ̃ 
un símisi rɛ ituŋɛ yɛ tulɛ, Uléécɑɑ iyɔɔpi yɛ nyɑhɑimɛlɛ, ɑni mɛfinɛ consɛ ɑni Nsímɛ́ Kɛcirɛ ŋmurɛi. 
Kɛ Yeesu uu Kɑlilee kupiyɛ mɛcɑ́ɑ n tíkilɛ̃, uu Simɔɔ nɛ uuwɑ̃ Antiree yɛnu pin ɑnyɛŋ́ fóm pin nɛ ikpíntomɛ́ cɔ́pii. Pi pɛɛ pikpíntomɛ́cɔ́pii lɛ. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Ani nɛ́ ritiki, nɛ́ tíyɛsɛ ɑni pisoi pɛɛ yɛ́ nɛ́ rintíki wɛ́ɛ́si. 
Api mɛsɛ nɛ mɛsɛ piɑnyɛŋ́ sɑ́ ɑpi u tiki. 
Kɛ Yeesu uu nsẽ́ ń nɔŋ́lɛ̃, uú hɑ Sepetee sipipi Yɑkupu nɛ uuwɑ̃ Yohɑni yɛnu pin piɑnyɛŋ́ nyɔ́ɔnsɛ. 
Yeesu uu mɛsɛ nɛ mɛsɛ pi séi, ɑpi piusɑɑ Sepetee nɛ pikɛikɔ́ kúninɔi-i tíyɛ ɑpi u tiki. 
Yeesu nɛ upipirɛtiki ɑpi Kɑpɛrinɑwum sĩ. Yeesu uu mɛsɛ nɛ mɛsɛ kɛwéntɛyɑɑ kɛtúŋɛ́ Pisuifi kuyomɛyɑ́hɑɑlee loni uu pisoi picélɑɑ kɑ́pɑ́ɑ́. 
Pírí kɛ umɛcélɑɑ ɑmɛ kɛyɑ́ɑ kɛ̃ pisoi pɛ̃ nnɛ́í pɛɛ kutu u n cɔ́lɛ̃ we, likumúŋɛ́ rɛ umɛcélɑɑ nɛ isé picélɑɑ mɛcélɑɑ ɑ́mɛ sɑ́. Amɑ́ uyɛ̃ nɛ pɛɛ pisoi céésilɛ nɛ n-yɔ́ɔpinɑŋɛ. 
Kumúŋɛ́ kpɛ-i, usoi unyinɛ kɛ ɑníri ɑɑ m pɛkɛsi uu Pisuifi kuyomɛyɑ́hɑɑlee-i lompɔ, uu pupɛi rɛ: 
Ń-ye kɑɑ́ nɛ rɔ́ wɛ́ɛ́si, Nɑsɑrɛti ukɔ́ Yeesu? Pɔ kɑlɛ rɛ pɔkɛ́ rɔ́ nɑ̃́ɑ? Nɛ pɔ́ nyulɛ rɛ Uléécɑɑ yɛ pɔ́ wɛ́ɛlɛ uu iyɑɑ yekei. 
Yeesu uu kuníri kpɛ̃ sɛmni uu rinɔ́ɔ ku hɛ rɛ: A kei risɛ́ɛ́, ɑɑ usoi uyɛ-i lelu. 
Kuníri kpɛ̃ ɑku usoi uyɛ̃ lémesi ɑku ukɛmɛɛ́ lelu kun pupukɛɛ. 
Ai pinnɛ́í nnɔ́ɔ yipu ɑpí m písɛinɛ rɛ: Yoo nní? Icélɑɑ fɑlɛ inyinɛ yɛ nyi nɛ innɑŋɛ! Ḿpɑ́ ɑníri kuu rinɔ́ɔ hɛ, ɑ́ɑ yɛ ti rilɔ́ɔ́. 
Mɛsɛ nɛ mɛsɛ Yeesu nsímɛ́ nn Kɑlilee kɛtẽ nnɛ́í kɔ́ɔ́nú. 
Api Pisuifi kuyomɛyɑ́hɑɑlee-i léeri ɑpí nɛ Simɔɔ nɛ Antiree kɛyɔ n tɔ́su, Yɑkupu nɛ Yohɑni pin tikilɛ̃. 
Api Simɔɔ ulɔ́hɔ́ni lɛɛpɔ un finu nɛ kupiŋɛtoi. Kɛ Yeesu uu n tuipɔ, ɑpi unɔ́si uyɛ̃ nsímɛ́ u wɑi. 
Yeesu uú nyɔsɔ́pɔ, uu ukunipɛ tini uu u yukusɛ. Kupiŋɛtoi ɑku mɛsɛ nɛ mɛsɛ tɛnɛ, uu yisi uu pikɛcɑ́ɑ́ pɑí. 
Mɛtuŋɛlolɛ́-mɑɑ́, pisoi ɑpí nɛ pitóikɔ́ ncɔpuri nɛ pinírihélɑɑ Yeesu lɛɛpɔ. 
Kuyu pikɔ́ nnɛ́í ɑpi kɛ́yɔ kɛ̃ tipoo yipu. 
Yeesu uu pisoi mɛyɑ̃ itói ncɔpuri pɔisɛntɛ, uu kɔ ɑníri mɛyɑ̃ lɑkɑsɛ. Úu tíyɛsɛ ɑníri ɑkɛ́ rinɔ́ɔ mɑlɛ, kɛ ɑricuruu ɑɑ u nyu nnyɑ. 
Kɑi n weesi, Yeesu uú nɛ kɛsinɛ tɔrɔɔ kɛ́yɔ lelu. Uu iyɑɑ sĩ uú hɑ kei umɛcirɛ kɛyómɛ n yɑ́ɑ́si. 
Simɔɔ nɛ upicɔ ɑpi pɛɛ piwɛ́lɑɑ u loni. 
Kɑpi u n yɛ̃́ ɑpi u mɑɑ rɛ: Pisoi nnɛ́í pɛɛ pɔ́ wɛ́ɛ́si. 
Amɑ́ Yeesu uu pi mɑɑ rɛ: Tɔkɛ́ kɛlõ féé hɑ, ɑyu ɑcɔ-i, li pisɛ rɛ kɛ́ kɔ kei Nléécɑɑsimɛ́ símisi, likumúŋɛ́ rɛ lɛlɛɛ nɛ nɛ́ kɑ. 
Lɛ̃ kɑi wɑ Yeesu uu Kɑlilee kɛtẽ nnɛ́í kɔ́ɔ́nú un Pisuifi ɑyomɛyɑ́hɑɑlee-i Nléécɑɑsimɛ́ símisi un ɑníri lɑkɑsɛ. 
Uyɔ̃́ unyinɛ uu Yeesu lɛɛpɔ uu ukɛyu-i wúlɑ uú nɛ u téni rɛ: Pɔn n lɑ, pɔ́ nɛ́ wolɛ ɑm funi. 
Uicɔ ii Yeesu íwɛ wɑi hɑ́i, Yeesu uu pɛɛ kunípɛ sɑɑpɔ uu u cɑ, uu lɛ̃ mɛmɑ́ɑ́ u mɑɑ rɛ: Nɛ lɑ rɛ ɑ peí. 
Mɛsɛ nɛ mɛsɛ, utisi uyɛ̃ kɛyɔ̃́ ɑkɛ tɛnɛ uu funi. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ u cɛ́si uu u lɑkɑsɛ 
un u kúrúrúsɛlɛ̃ tɛ: A ńsɔnɛ kutu ricɔ, kɑpɛ wɑ unyinɛ ukɛ́ mɛwɑi mɛ̃ kõ. Amɑ́ ɑ hɑ uyɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ mɛpɔ́círɛ nyísɛ ukɛ́ yɛ̃́ lɛ̃ kɑɑ ń we. Lɛ̃ mɛmɑ́ɑ́, ɑɑ inyɔ́ɔnsɛ kɛ Moisi isé ii m pisɛ wɑi pɔkɛ́ nɛ ḿpɑ́ úye nyísɛ rɛ pɔ peí. 
Amɑ́ utisi uyɛ̃ uu sĩ́ uu lɛ̃ nnɛ́í kɑi n wɑ ḿpɑ́ yei kɛ́ɛ́nkɛɛ. Lɛ̃ nnyɑ, Yeesu úu yɛ fe ukɛ́ umɛcirɛ riwúí-i pinyísɛ rikpɑ́, ɑmɑ́ iyɑɑ kuú le, kei kɛ úkɑ úu ń we. Pisoi pin ḿpɑ́ mɛ́ye léenti pin ukɛmɛɛ sɔ́nɑɑpɔ.
