Yohɑni.
2.
Siyɑ́ɑ sitɑɑni mɛmɑ́ɑ́, ɑpi pinɔ́sikpikɛ́ ɑnyɑ̃́ Kɑlilee kɛtẽ-pɔ Kɑnɑɑ kuyu-i n wɑi. Yeesu uni un ɑnyɑ̃́ nyɛ-i we. 
Pi Yeesu nɛ upipirɛtiki pɔ̃́ kɔ kei kusɑ́nɛ séelɛ. 
Pítɑ ɑpí weri ɑpi kumúŋɛ́ kunyinɛ-i tɛnɛ. Úni uu pɛɛ u mɑɑ rɛ: Pítɑ yɛ nɛ pi tɛnɛlɛ. 
Yeesu uu u pɛsɛ rɛ: Unɔ́si! Nɛ nyulɛ, ɑ́i pɔ́ɔ yɛ́ nɛ́ símisi lɛ̃ kɑm yɛ́ ń wɑ. Inɛ́tuŋɛ íi kɑhɑnɛ n tu. 
Kei kɛ úni uu pɛɛ upikɛikɔ́ pɛɛ kei ń we mɑɑ rɛ: Ani lɛ̃ nnɛ́í kuu nɔ́ m pisɛnɛ wɑ. 
Mɛpɛ́rɛ́ ɑpɔri ɑkpuulũ ɑnyinɛ yɛ pɛɛ welɛ. Kei kɛ Pisuifi ɑpi yɛ míni wɑ ɑpí nɛ pimɛcirɛ nyɔ́ɔnsɛ. Ḿpɑ́ kupɔ́ri kúye yɛ́ fe ɑku ɑ́nicɑri ɑtɛ́ nɛ́ɛ ɑtɑɑni yɔ́su. 
Yeesu uu upikɛikɔ́ mɑɑ rɛ: Ani míni lṹ, ɑni ɑpɔ́ri nnyɛ́ yípũ. Pɛ̃ ɑpi mɛsei lúni ɑi ɑ yípuni ɑí wurɛ. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Ani nkpéni kpɛhɛ̃ ɑní hɑ uyɛɛ ɑnyɑ̃́ ilukɛ kɛcɑ́ɑ́ m pɑílɛ̃ pɑ. Pɛ̃ ɑpi kpɛhɛni ɑpí hɑ u pɑ. 
Kɑpi u ḿ pɑ, uu lélé míni mɛn pɑnsɛ pítɑ. Úu nyu kei kɛ pítɑ pɛ̃ ɑpi n léeri. Amɑ́ pikɛikɔ́ pɛɛ n kpɛhɛ̃ ɑpí nɛ hɑpɔ pɔ́ɔn n nyu kei kɑpi n léeri. Kei kɛ uyɛɛ ɑnyɑ̃́ ilukɛ kɛcɑ́ɑ́ m pɑílɛ̃ uu pɛɛ unɔ́sikɔ́ uyɛ̃ sée, 
uu u mɑɑ rɛ: Pítɑ lɑri-lɑri kɛ pisoi ɑpi yɛ nɛ pinírɛ́ kórɑɑnɛ. Pin n lɛ, ɑpi pɛɛ pikópɛ lesɛri. Amɑ́ pítɑ lɑri-lɑri kɛ pɔ́ɔ yekei ɑí nɛ nkpéni tulu! 
Lɛ̃ kɛ Yeesu uu Kɑlilee kɛteni-i Kɑnɑɑ-pɔ mɛwɑisɑŋɑ piwɑi kori. Lɛ̃ kuú nɛ ummulɛ nyísɛ upipirɛtiki ɑpi pɛɛ́ nɛ kɛfɑ u tɛnɛ. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu nɛ úni nɛ upimɑrɛcɔ nɛ upipirɛtiki ɑpi yisi ɑpi Kɑpɛrinɑwum tɔ́mpɔ ɑpí hɑ kei siyɑ́ɑ sinyinɛ wɑi, ɑ́pi nɑ́ŋɑi. 
Kɛ Pisuifi nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ ɑɑ nnyɑ́ɑ̃ n wɑ, Yeesu uu pɛɛ yisi uu Yerusɑlɛm tɔ́mpɔ. 
Uu kei Uléécɑɑ kɛyɔ kɛ́mɛɛ pikpéensɛ lɛɛpɔ pin inɑ́ɑ nɛ isɑ́ŋ nɛ ɑlɔ́ɔlɑ̃ yɑ́i, picɔ pɔ́ɔn túntilɛ̃ pin nɛ siwóó foru. 
Kei kuu íŋmɛ kpísi uú nɛ nsɛ́í wɑi uú nɛ pi nɛ piisɑŋ nɛ piinɑɑ Uléécɑɑ kɛyɔ kɛ́mɛɛ lɑ́kɑsɛ ɑpí lelu. Uu piwóóforɛ ɑtɑɑpili lɑ́uli, siwóó ɑsi yɛ́kɑ́ɑ́. 
Uu pɛɛ pilɔ́ɔlɑ̃yɑ́i pɔ̃́ mɑɑ rɛ: Kɛ́ nté nɔ́ pɑɑ nfɑ́ɑni, ɑ́ni kɑpɛ Unɛ́sɑ́ɑ kɛyɔ kuyɑ́ɑ pɑnsɛsɛ. 
Kei kɛ upipirɛtiki ɑpi pɛɛ nsímɛ́ mmú mɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ léisɛ rɛ: Hɑi Uléécɑɑ! Lɛ̃ kɑm nní kɛpɔ́yɔ ń lɑ yɛ nɛ́ torɛlɛ yɑrɛ nnɑ. 
Kei kɛ Pisuifi piwɛ́ɛ́sɛ ɑpi pɛɛ u pisɛ rɛ: Mɛwɑisɑŋɑ mɛ́ye kɑɑ yɛ́ fe ɑɑ wɑi ɑɑ́ nɛ rɔ́ nyísɛ rɛ pɔ ncée mɑ́lɛ pɔkɛ́ lɛ̃ wɑ? 
Yeesu uu pi pɛsɛ rɛ: Ani Uléécɑɑ kɛyɔ nkɛ́ fori kɛ́ siyɑ́ɑ sitɑɑni nɛ pimɔ́mɛ kɛ pɛsɛ. 
Api u pisɛ rɛ: Aŋmɛ̃ ɑfɛɛnɑ nɛ ɑkpuulũ kɑpí nɛ kɛ́yɔ nkɛ́ mɔ̃́. Kɛ̃ kɛ pɔ́ɔ músu rɛ pɔ́ fe ɑɑ siyɑ́ɑ sitɑɑni nɛ mɔmm? 
Nɛ́ḿpɑ́kɑnɛ umɛcirɛ kuú pɛɛ́ nɛ lɛ̃ mɑ́nɛ kuú nɛ Uléécɑɑ kɛyɔ nkɔ́ símisi. 
Kɛ Uléécɑɑ uu Yeesu nkpɔ kɛ́mɛɛ n yukusɛ, upipirɛtiki ɑpi pɛɛ léisɛ rɛ u pɛɛ ɑnɔ́ɔ nyɛ̃ mɑlɛ. Api pɛɛ Nléécɑɑsimɛ́ nɛ ɑnɔ́ɔ nyɛ̃ kɛ uricuruu uu ḿ mɑ nɛ kɛfɑ tɛnɛ. 
Nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ kumúŋɛ́, Yeesu un Yerusɑlɛm-i we. Pisoi ɑpi mɛwɑisɑŋɑ kuu n wɑpisi yɛ́nti ɑpi pɛɛ́ nɛ kɛfɑ u tɛnɛ. 
Amɑ́ kɛ Yeesu pɔ́ɔ pikɛmúŋɛ́ nnɛ́í n nyu nnyɑ, úu nɛ ripɔ́ɔ pi lɑrisi. 
Ái we rɛ pikɛ́ pisoi kɛcɑ́ɑ́ nsímɛ́ u símisi. Uricuruu yɛ lɛlɛɛ sisoipipi sifɑ-i ń we nyulɛ.
