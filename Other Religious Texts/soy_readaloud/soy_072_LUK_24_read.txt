Luki.
24.
Kɛwéntɛyɑɑ ukóso kɛmɑrɛsinɛ, pinɔ́si ɑpi mɛ́kpɔ nɛ tulɑɑli kɑpi n wɑ múlú ɑpí nɛ rikpíí símpɔ. 
Api lɛɛpɔ lin tipɑrɛ tɛɛ pɛɛ rikpíí n fílɛ̃ pímiisɛ ɑi iyɑɑ yekei. 
Api lompɔ ɑmɑ́ ɑ́pi Upíimɑ Yeesu lɛɛpɔ. 
Ápi ceri íye kɑpi yɛ́ wɑ. Kumúŋɛ́ kpɛ-i kɛ pisoi pitɛ́ pinyinɛ ɑpi rikpɑ́fúmɛ pin ilũ yɛɛ n tɛ́lu tɑnɑɑlɛ̃. 
Kɛ iwɑmɛ ii pi n wɑ ɑpi kpɑ́rɑi, pisoi pɛ̃ ɑpi pi mɑɑ rɛ: Yo nnyɑ kɑni pikpɔkpɔ kɛcɔpɛ uyɛɛ n nyɑ́ni wɛ́ɛ́si? 
Úu nté we, u nkpɔ kɛ́mɛɛ yisilɛ un nfɑ́ɑ mɑ́. Ani léisɛ lɛ̃ kuu nɔ́ ḿ mɑ pɛɛ́ uyɛ-i kuú pɛɛ Kɑlilee-pɔ ń we. 
Li pisɛ rɛ pikɛ́ pikópɛkɔɔ́ ɑnipɛ-i Usoi Kɛpipi wɑ, ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɛ kɑrii Uléécɑɑ ukɛ́ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ nkpɔ kɛ́mɛɛ kɛ yukusɛ. 
Api pɛɛ Yeesu ɑnɔɔ léisɛ. 
Api rikpíí-i yisi ɑpi lɛ̃ nnɛ́í pipirɛtiki kɛfi nɛ usɛ nɛ picɔ nnɛ́í kɛ́ɛ́nkɛɛ. 
Mɑɑri Mɑkitɑlɑ ukɔ́ nɛ Yohɑnɑ nɛ Yɑkupu uni Mɑɑri yɛ pɛɛ pi. Pi nɛ pinɔ́si picɔ pɛɛ pɛɛ kɛsẽ́ ń we ɑpi nsímɛ́ pitumɛ kɛɛni. 
Amɑ́ pitumɛ ɑpi pinɔ́si nsímɛ́ mmɛ̃ yɛnu rɛ nnírisimɛ́ yɛ mu, ɑ́pi pɛɛ pɑkɑrɛ. 
Kumúŋɛ́ kpɛ-i, Piyɛɛ uu yisi uu rikpíí-mɛ̃ wurupɔ, uú kpɑrɑ́pɔ uu ɑsɑ́ŋɑ́ɑ́ ŋmɑnɛ yɛnu ɑn cúlɛ̃. Uu pɛɛ ukɛyɔ pɛlɛ. Lɛ̃ nnɛ́í lɛɛ nní n wɑpisi lin nnɔ́ɔ u yipɑɑlɛ̃. 
Kɛyɑ́ɑ kɛ̃ cirɛ, pipirɛtiki pitɛ́ ɑpi kɛyupi kɛnyinɛ kɑpi yɛ n sée rɛ Emɑyusi n simpɔ. Ituŋɛ nwɔi mɛkɔ́ɔ́nú mɛtɛ́ kɑpi yɛ kei nɛ Yerusɑlɛm sɔ́nɛ. 
Lɛ̃ nnɛ́í lɛɛ pɛɛ n wɑpisi nsímɛ́ kɑpí pɛɛ símisi. 
Kumúŋɛ́ kpɛ-i kɑpi lɛ̃ n símisi pin kɛŋɛnɛ, Yeesu ricuruu uú nɛ pi kɔ ɑpi kɛsẽ́ n sɔ́nɛ, 
pin u nyɑ́ni ɑmɑ́ linyinɛ lin pi hilɑlɛ̃ ɑ́pi pɛɛ u ceri. 
Yeesu uu pi pisɛ rɛ: Yo yɛɛ nɔ́ kɛ́ŋɛnɛsɛ nɔn nɛ sɔ́nɛ? Api nyɛrɛ nɛ mpɔ́ɔcɑɑi. 
Pikɛcɔpɛ unyinɛ kɑpi yɛ n sée rɛ Keleopɑsi uu u mɑɑ rɛ: Yɑrɛ pɔ̃́ mɛcirɛ yɛɛ Yerusɑlɛm kuyu-i we ɑ́ɑ nyu lɛlɛɛ siyɑ́ɑ tɔrɔɔ nsí kɛ́mɛɛ kuyu kpɛ-i n wɑii? 
Yeesu uu pi pisɛ rɛ: Yɑrɛ mɛwɑi mɛ́ye mpuri yɛ mɛ? Api rinɔ́ɔ u yɔ́su rɛ: Lɛlɛɛ Nɑsɑrɛti ukɔ́ Yeesu n lɛɛmɛ mɑ́. Antepu lo yɛɛ ńnɑŋɛ m mɑ́. U Uléécɑɑ nɛ pisoi kɛyu-i unnɑŋɛ nyísɛlɛ uliwɑiwɑi nɛ uɑnɔɔlempi kɛ́mɛɛ. 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ kurɔ́yu piyukɔɔ́ ɑpi u tini ɑpi nsímɛ́ u cɔnsɛ ɑi wɑi rɛ pikɛ́ u kpu. Api kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii. 
Tɔ pɛɛ tɑ́lɛ̃ tɛ uyɛɛ yɛ́ Isirɑyɛɛli riyu lɔ. Amɑ́ kɛyɑ́ɑ tɑ́ɑ́nũ yɛ nkɛ́ kɛ mɛwɑi mɛ̃ ɑmɛ́ nɛ n wɑpisi. 
Asei kɛcɑ́ɑ́, kɛrɔ́cɔpɛ, pinɔ́si pinyinɛ yɛ tíyɛlɛ ɑi pírí rɔ́ wɑi. Kɛmɑrɛsinɛ kɑpi rikpíí-i hɑ. 
Amɑ́ ɑ́pi u lɛɛpɔ. Api pɛɛri ɑpi rɔ́ símisi rɛ piléécɑɑtumɛ pinyinɛ yɛ pikɛyu-i rikpɑ́fúmɛlɛ ɑpi pi mɑɑ rɛ u nyɑ́nilɛ. 
Pirɔ́séncɔ pinyinɛ ɑpi rikpíí-i símpɔ ɑpi lɛɛpɔ yɑrɛ lɛ̃ kɛ pinɔ́si ɑpi rɔ́ ń kɛ́ɛ̃. Amɑ́ ɑ́pi pɛɛ uricuruu yɛ̃. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Nɔ́ piníri mpí, ɑ tíyɛ nɔ́ ɑtu mɑ́ ɑ́ni lɑ nɔkɛ́ mmɛ̃ kɛ ɑntepuyɛ ɑpi nɔ́ n símisi nɛ kɛfɑ tɛnɛ. 
Ái pɛɛ pisɛ rɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ ukɛ́ lɛ̃ íwɛ li uu kɛlenɛ umɛyɔɔpi-i lonii? 
Uu pɛɛ limɛmɑ́ɑ́ mmɛ̃ nnɛ́í kɑi uriyu kɛcɑ́ɑ́ ɑ́tɛlɛ́-i n wɔ́lɑɑlɛ̃ pi kpɑ̃́iisɛ, Moisi ritɛlɛ́ kuú nɛ kori uú nɛ ɑntepuyɛ ɑkɔ́ tɛnɛsɛ. 
Kɑpi kuyu kpɛ-i kɑpi n símpɔ n nyɑhɑi, Yeesu uu wɑi yɑrɛ u lɑlɛ ukɛ́ kɛtɑɑ-tɑɑ hɑ. 
Amɑ́ ɑpi u nyɛrɛsɛ pin tee rɛ: A kɛrɔ́kúrí tonɛ, li nnyɔ́ɔ́ wɑlɛ kɛsinɛ ɑkɛ nyɑhɑi. Uu kɛ́yɔ pi tiki ɑpí hɑ kɛsẽ́ tonɛ. 
Apí tonɛ rɛ pikɛ́ ilukɛ li, uu kukpɔ́nɔ́ kpísi uu Uléécɑɑ pɔɔnɛsɛ uu pɛɛ ku kpɔkɔ́rinɛ uu pi ku pɑ. 
Piinipɛɛ ii pɛɛ wúkulɛ ɑpi u ceru, ɑmɑ́ uu pikɛyu kɛ-i mílu. 
Api pimɛcɔpɛcirɛ m mɑ́ikɛɛnɛ rɛ: Pɔ̃́ tɛ piyo kɛ nsímɛ́ kuú pɛɛ ncée-i n símisi nɛ Nléécɑɑsimɛ́ ɑsei kuú pɛɛ rɔ́ ń símisi nn tɔ lõ ŋmɑɑ. 
Api mɛsɛ nɛ mɛsɛ yisi ɑpi Yerusɑlɛm simpɔ. Api pipirɛtiki kɛfi nɛ usɛ nɛ pipisencɔ lɛɛpɔ pin cɑ́pinɛlɛ̃, 
pin tee rɛ: Mɛsei yɛ mɛ, Upíimɑ yɛ nkpɔ kɛ́mɛɛ yisilɛ uu tíyɛsɛ Simɔɔ uu u yɛnu! 
Pɛ̃ ticuruu ɑpi lɛlɛɛ ncée-i n wɑ pi kɛɛnkɛɛ nɛ lɛ̃ kɑi n wɑ ɑpí nɛ Yeesu ceru kumúŋɛ́ kpɛ-i kuú pɛɛ kukpɔ́nɔ́ rińkpɔkɔ́rinɛ. 
Ápi kɑhɑnɛ nkpɑ́ni nsímɛ́ n tɛnɛ, Yeesu ricuruu uu pikɛcɔpɛ kpɑ́fúmɛ uu pi mɑɑ rɛ: Nɔ́ɑnɛ́kíŋ ɑkɛ́ n niŋú. 
Pikɛwuu ɑkɛ tɔ́lu, iwɑmɛ píimɑ ii pi tini, likumúŋɛ́ rɛ li pi wɑ yɑrɛ kuníri kunyinɛ kɑpi nyɑ́ni. 
Amɑ́ Yeesu uu pi mɑɑ rɛ: Yo nnyɑ kɑi nɔ́ ricɔ́ŋ? Yo nnyɑ kɑni nɔ́sinɛ́fɑ-i tíru? 
Ani ɑnɛ́nípɛ nɛ ɑnɛ́nɑ ripɑí. Mɛsei nɛ́ɛ lo. Ani nɛ́ ricɑ ɑni yɛ́nu. Kuníri ɑ́ku ipiŋɛsɑrɛ nɛ́ɛ ɑkɔhɔ mɑ́, yɑrɛ kɑni nní lɛ̃ nnɛ́í kɛnɛ́cɑ́ɑ́ n nyɑ́ni. 
Uu ɑnɔ́ɔ nyɛ̃ mɑlɛ uu uɑnipɛ nɛ uɑnɑ pi nyísɛ. 
Mpɔ́ɔnɑrɛ nɛ pírí píimɑ yɛɛ pi ń we nnyɑ, ɑ́pi nkpɑ́ni fe pikɛ́ nsímɛ́ mmɛ̃ pɑkɑrɛ. Uu pi pisɛ rɛ: Nɔ nté lilukɛ-lukɛ linyinɛ mɑ́lɛɛ? 
Api nkpíntomɛ́ ŋmɑ́ɑ́-ŋmɑ́ɑ́ keri u pɑ. 
Uu n kpísi uu tɑkɑi piinipɛɛ-i. 
Uu pɛɛ pi mɑɑ rɛ: Mmɛ̃ kɑḿ pɛɛ nɔ́ ń símisi kumúŋɛ́ kpɛ-i kɑrí pɛɛ kɛsẽ́ ń we yɛ mmú. Li pisɛ rɛ lɛ̃ nnɛ́í kɑi kɛnɛ́cɑ́ɑ́ n wɔ́lɑɑlɛ̃ Moisi isé kɛ́mɛɛ nɛ ɑntepuyɛ ɑtɛlɛ́ kɛ́mɛɛ nɛ siyómɛ ɑtɛlɛ́ kɛ́mɛɛ likɛ́ wɑ. 
Uu pɛɛ mɛsɔhɔ pi tɑ́pilɛ pikɛ́ nɛ Nléécɑɑsimɛ́ ɑsei n kṍ nnyɑ. 
Uu pi mɑɑ rɛ: Lɛ̃ kɑi n wɔ́lɑɑlɛ̃ yɛ nní: Li pisɛ rɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ ukɛ́ íwɛ li, uu pɛɛ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ pikpɔkpɔ kɛ́mɛɛ yisi. 
Li kɔ pisɛ rɛ pikɛ́ urinyiri nɛ sitẽ nnɛ́í kɛ́mɛɛ pisoi riyóó rɛ pikɛ́ mɛfinɛ consɛ Uléécɑɑ ukɛ́ piɑkópɛ pi sɑ́rɛi. Amɑ́ Yerusɑlɛm kuyu kɑpi yɛ́ nɛ korɑɑnɛ. 
Nɔ́ɔ lɛ̃ piseérɑkɔɔ́. 
Iyɛɛ nnyí, ihɛɛ iyɛ̃ kɛ Unɛ́sɑ́ɑ uu rinhɔ́ rɛ u yɛ́ nɔ́ hɛ. Nɛ́ɛ rinɛ́cúruu nɔ́ i pɑmɛnɛ. Amɑ́ li pisɛ rɛ nɔkɛ́ kuyu-i n tũ hɑ́i nɛ kumúŋɛ́ kpɛ-i kɛ kɛyómɛcɑɑ nnɑŋɛ nn yɛ́ nɔ́kɛnɛ́cɑ́ɑ́ n súimɛ. 
Yeesu uu limɛmɑ́ɑ́ nɛ kuyu iyɑɑ pi simpɔ, Petɑnii kɛkúrí, uu ɑnípɛ síkɑ uu rinɔ́ɔ sɔnɛ pi tikisɛ. 
Kumúŋɛ́ kpɛ-i kuu rinɔ́ɔ sɔ́nɛ pi n tíkisɛlɛ̃, uú nɛ pi tɔlu, ɑi kɛyómɛ-mɛ̃ u sɑkɑɑpɔ. 
Pɛ̃ ɑpi Uléécɑɑ yɑ́ɑ́si ɑpi pɛɛ Yerusɑlɛm pɛlɛ nɛ mpɔ́ɔnɑrɛ píimɑ. 
Api yɛ pɛɛ mɛsɛ́rɛ Uléécɑɑ kɛyɔ-i n cɑ́pinɛlɛ̃ pin Uléécɑɑ pɑkɑrɛntɛ.
