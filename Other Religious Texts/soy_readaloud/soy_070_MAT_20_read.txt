Mɑtiyee.
20.
Yeesu uu kɔ kpɑ́ rɛ: Lɛ̃ nnyɑ, Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ nsímɛ́ mmú: Ukɛisɑɑ unyinɛ yɛɛ kɛmɑrɛsinɛ leepɔ rɛ ukɛ́ pikɛikɔ́ wɛ́ɛ́si pikɛ́ uɑléé kɑpi yɛ n sée rɛ fiinyi kɛcɑrɛ kɛ́mɛɛ u kɛisi. 
U nɛ pɛ̃ ɑpi kómɛinɛ rɛ ukɛ́ pi hɛ́ɛ́lɛ nnɛ́í mmɛ̃ kɑpi yɛ kɛyɑ́ɑ kɛsɛ ukɛikɔ́ n hɛ́ɛlɛ. Kei kuu pɛɛ kɛcɑrɛ-i pi tɔ́ŋsɛpɔ. 
Kɑi ḿ pi yɑrɛ ituŋɛ mɛsɛ ɑ́mɛ we kɛfi kumúŋɛ́, uu kɔ leepɔ uu pikɛikɔ́ picɔ yɛnu, pɛ̃ pin kɛ́yɑɑ́lɑ-i mɛ́woo nyɛnu. 
Uu pi mɑɑ rɛ: Nɔ́ɔkɛ́ hɑ kɔ kɛnɛ́cɑrɛ-i kɛisi kɛ́ nnɛ́í mmɛ̃ kɑi m pisɛ nɔ́ hɛ́ɛ́lɛ. 
Api yisi ɑpi sĩ́. Uu kɔ ituŋɛ kɛcírɛ kumúŋɛ́ nɛ nnyɔ́ɔ́ mɛtɑɑni kumúŋɛ́ piléepɔ kpɑ́, uu kɔ pikɛikɔ́ picɔ kpísi. 
Kɑi nnyɔ́ɔ́ mɛnupũ ḿ pi, uu kɔ piléepɔ kpɑ́ uu pinyinɛ yɛnu pin kei nyɛnu, uu pi mɑɑ rɛ: Yo nnyɑ kɑni nté mɛ́woo nyɛnu ituŋɛ nɛ imɛlólɛ́? 
Api u pɛsɛ rɛ: Lɛlɛɛ rɛ úkɑ úu pikɛi rɔ́ kpísi. Uu pi mɑɑ rɛ: Nɔ́ɔkɛ́ kɔ n ŋme ɑni hɑ kɛnɛ́cɑrɛ-i kɛisi. 
Kɑi nnyɔ́ɔ́ n wɑ, ucɑrɛte uyɛ̃ uu uyɛɛ ulikɔ́ kɛcɑ́ɑ́ m pɑílɛ̃ mɑɑ rɛ: A pikɛikɔ́ pɛ̃ sée ɑɑ pɛpɛɛ mɛtɔ́rɔɔ ń kɑ nɛ pihɛ́ɛ́lɛ korɑɑnɛ ɑɑ́ nɛ pifoí kɛ́tɔ pɑ́ɑ́lɑ́. 
Pɛpɛɛ nnyɔ́ɔ́ mɛnupũ kumúŋɛ́ pikɛi n kórɑɑnɛ ɑpí weri, ḿpɑ́ úye uu nnɛ́í mmɛ̃ kɑpi yɛ kɛyɑ́ɑ kɛsɛ ukɛikɔ́ n hɛ́ɛlɛ yɔ́su. 
Kɛ pikɛikɔ́ foí pɛ̃ ɑpi ń kɑ, ɑpí m músu rɛ piihɛ́ɛ́ yɛ́ picɔ ikɔ́ rifɑɑu. Amɑ́ ḿpɑ́ úye uu nnɛ́í nsɛ mmɛ̃ kɑpi yɛ kɛyɑ́ɑ kɛsɛ ukɛikɔ́ n hɛ́ɛlɛ yɔ́su. 
Api kɛwóóyɔɔ́ kɛ-i iŋmúlɑɑ nɛ ukɛisɑɑ uyɛ̃ iwɔ́hɔ́ loni. 
Api rɛ: Mpí pɛɛ nní pikɛi mɛtɔ́rɔɔ mmɛ́ ń kɑ yɛ wɑlɛ ituŋɛ mɛkɔ́ɔ́nú mɛsɛ. Aɑ pɛɛ tɔ́ nɛ pɛ̃ nnɛ́í nsɛ hɛ́ɛlɛ, in kɔ nɛ tɔ́ɔ mɛyɑ̃́ ntuŋɛ nɛ mpɔɔnɛ kõ. 
Ukɛisɑɑ uyɛ̃ uu piusɛ rinɔ́ɔ yɔ́su rɛ: Ám inípɛɛ pɔ́ li, unɛ́sɑ́nɛ! Ái rɔ kómɛinɛlɛ rɛ nnɛ́í mmɛ̃ kɑpi yɛ kɛyɑ́ɑ kɛsɛ ukɛikɔ́ n hɛ́ɛlɛ kɑɑ yɛ́ yɔsíi? 
A lipɔ́kɔ́ kpísi ɑɑ tɔ́mpɔ! Nɛ lɑlɛ kɛ́ nɔ́ nɛ uyɛɛ pikɛi mɛtɔ́rɔɔ mmɛ́ ń kɑ nnɛ́í nsɛ hɛ́ɛ́lɛ. 
Yɑrɛ ɑ́m ncée mɑ́ kɛ́ linɛ́kɔ́ nɛ mɛnɛ́lɑ wɑ, nɛ́ɛ íye? Nɛ́ɛ pɔ inípɛɛ wɔŋnɛ rɛ nɛ kɛfɑ lɑ́ɑ́rú nnyɑ? 
Kei kɛ Yeesu uu rikpɑ́ rɛ limɛcɔ kɛ pitɔ́rɔɔ ɑpi yɛ́ pɑnsɛ pifoí, pifoí ɑpi pitɔ́rɔɔ. 
Yeesu nɛ upipirɛtiki ɑpi kɛyɑ́ɑ kɛnyinɛ Yerusɑlɛm n tɔ́su, uu upipirɛtiki kɛfi nɛ pitɛ́ pɛ̃ iyɑɑ séi uu pi mɑɑ rɛ: 
Nɔ nyɑ́nii, Yerusɑlɛm kɑri nní símpɔ pikɛ́ hɑ kei Usoi Kɛpipi tĩ pikɛ́ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ isé picélɑɑ ɑnipɛ-i wɑ, pɛ̃ pikɛ́ nɛ u túhɑɑnɛ ɑpi yekei rɛ pikɛ́ u kpu. 
Pi yɛ́ pɛpɛɛ ɑ́pi Uléécɑɑ m pɑkɑrɛlɛ̃ ɑnipɛ-i u wɑ, ɑpi mɛ́woo u nyɑ́nɛi ɑpi isɛ́í u fopii ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii, uu pɛɛ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ nkpɔ kɛ́mɛɛ yisi. 
Kei kɛ unɔsi uu upiŋmɑ́nɛ kɛtɛ́, Yɑkupu nɛ Yohɑni kpísi uú nɛ Yeesu lɛɛpɔ, uu ukɛyu-i wúlɑ uú nɛ u téni. 
Yeesu uu pɛɛ u pisɛ rɛ: Yo kɑɑ́ lɑ kɛ́ pɔ́ wɑ? Uu rinɔ́ɔ u yɔ́su rɛ: Nɛ lɑ rɛ ɑ ŋmurɛi rɛ pɔ́ tíyɛsɛ pinɛ́ŋmɑ́nɛ mpí kɛtɛ́ ɑpí mɑsí ipɔ́yɔ́ɔpi kɛ́mɛɛ tonɛ, ucɔ kupɔ́lukɛ-lukɛ, ucɔ kupɔ́míi-mɛ̃. 
Yeesu uu pɛɛ pi pɛsɛ rɛ: Áni nyu lɛ̃ kɑni kei nní n welu. Nɔ́ fe ɑni ntóósi kɛporipi kɑm n nírunɛ níruu? Api rɛ: Ɛɛɛ, tɔ́ fe. 
Yeesu uu pi mɑɑ rɛ: Nɔ́ mɛsei ntóósi kɛporipi mɛnɛ́cɔ n ntí. Amɑ́ in kunɛ́lukɛ-lukɛ nɛ́ɛ kunɛ́míi mɛtonɛ nkɔ́ mmú, ɑ́i nɛ́ɛ yɛ́ tíyɛsɛ likɛ́ nɛ lɛ̃. Pɛ̃ nnyɑ kɛ Unɛ́sɑ́ɑ uu sitónɛ sɛ̃ n nyɔ́ɔnsɛlɛ̃ kuu si pɑnɛ. 
Kɛ upipirɛtiki kɛfi tɔrɔɔ mpí pɔ́ɔ lɛ̃ n kṍ, piripɔɔ ɑrí nɛ pimɑ́rɛcɔ pitɛ́ mpí cɑɑi. 
Yeesu uu pinnɛ́í séi uu pi mɑɑ rɛ: Nɔ nyulɛ rɛ pɛ̃ kɑpi n kpíilɛ̃ tɛ ɑyu piwɛ́ɛ́sɛ yɛ pinnɑŋɛ nɛ ɑ wéékusɛlɛnlɛ, pisoi kɛcirɛ pin pinnɑŋɛ ɑ nyísɛlɛ̃. 
Amɑ́ ɑ́i limɛcɔ kɑi nɔ̃́ kɛ́mɛɛ we. Úye un n lɑ ukɛ́ nɔ́kɛnɛ́cɔpɛ uwɛ́ɛ́sɛ, ukɛ́ pɑnsɛ nɔ́unɛ́kɛikɔ́. 
Úye un kɔ nɔ́kɛnɛ́cɔpɛ ikpéẽ n lɑ, liute ukɛ́ nɔ́unɛ́lɑ́si. 
Limɛcɔ kɛ Usoi Kɛpipi ɑ́kɛ kɑ rɛ pikɛ́ kɛ n kɛisi. Amɑ́ kɛ kɑlɛ rɛ kɛkɛ́ picɔ kɛisi kɛkɛ́ kɔ kɛnfɑɑ kpísi kɛkɛ́ piɑkópɛ nnyɑ hɛ likɛ́ nɛ pisoi mɛyɑ̃ ɑyu lɔ. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu nɛ upipirɛtiki ɑpi Yeriko kuyu tulu. Kɑpi kei n yisi pin tɔ́su, tisoiwuí píimɑ rinyinɛ ɑri un n tikilɛ̃. 
Kei kɛ pinyíyɛ pitɛ́ pinyinɛ pɛɛ ricéetimɛ n tṹ ɑpi kṍ tɛ Yeesu yɛɛ tɔ́su. Api pɛɛ sicɑ́ɑ́ilɑ kɑ́pɑ́ɑ́ rɛ: Upíimɑ, Tɑfiti Kɛpipi, ɑ irɔ́wɛ ripɑí! 
Pisoi pɛɛ n cɑ́pinɛlɛ̃ ɑpi pi cɛ́si rɛ pikɛ́ risɛ́ɛ́, ɑmɑ́ ɑpi pɛɛ mɛyɑ̃́ n cɑ́ɑ́i ŋmɑɑ nɛ ńnɑŋɛ rɛ: Upíimɑ, Tɑfiti Kɛpipi, ɑ irɔ́wɛ ripɑí! 
Yeesu uu pɛɛ nyɛrɛ, uu pi séi uu pi pisɛ rɛ: Yo kɑní lɑ kɛ́ nɔ́ wɑ? 
Api u pɛsɛ rɛ: Upíimɑ, tɔ lɑ rɛ ɑ wɑ irɔ́nípɛɛ ikɛ́ wúkulɛ. 
Yeesu uu piiwɛ pɑí uu piinipɛɛ cɑ, mɛsɛ nɛ mɛsɛ, inípɛɛ ii wúkulɛ, pikɛtɛ́ ɑpi pɛɛ u tiki.
