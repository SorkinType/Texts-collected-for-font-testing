Mɑriki.
9.
Uu pimɑ́ɑ pi kpɑ́ rɛ: Asei kɑm nɔ́ símisi, pinyinɛ yɛ nté welɛ pɛɛ yɛ́ Uléécɑɑ iyɔɔpi n yɛ̃́ in nɛ innɑŋɛ wemɛ ɑpi kɛlenɛ kpíni. 
Siyɑ́ɑ sikpuulũ mɛmɑ́ɑ́, Yeesu uu Piyɛɛ nɛ Yɑkupu nɛ Yohɑni kpísi ɑpí nɛ riyɔ́pɛ cɑ́cɑ́ rinyinɛ tɑɑ́ ɑpí hɑ kei pimɛcirɛ n we. Uipiŋɛ ii consɛ piinipɛɛ-i. 
Uilũ ií n tɛ́lu, ii wɑ́rɑrɑ póí-póí, ntómɛ́ mmɛ̃ ńn ncɔ mɑ́ kɛtẽ nkɛ́ nɛ kɛnkinii. 
Kei kɛ Elii nɛ Moisi ɑpi pɛɛ rikpɑ́fúmɛ, pi nɛ Yeesu pin yɔ́i. 
Piyɛɛ uu pɛɛ Yeesu mɑɑ rɛ: Sɑ́ɑ, li nyɑmlɛ rɛ tɔkɛ́ pɛɛ nté n we. Tɔ lɑ rɛ tɔkɛ́ sicɑ́ŋíípi sitɑɑni kɑrii, pɔ̃́ kɛsɛ, Moisi kɛsɛ, Elii kɛsɛ. 
Piyɛɛ úu nkpɑ́ni nyu mmɛ̃ kuu yɛ́ pɛɛ ḿ mɑlɛ, likumúŋɛ́ rɛ iwɑmɛ yɛ u nɛ Yeesu pipirɛtiki tɔrɔɔ wɑlɛ hɑ́i. 
Kuhopɛ kunyinɛ ɑkú weri ɑku pi hilɑ. Tinɔ́ɔ rinyinɛ ɑri kuhopɛ kpɛ-i léeri rɛ: Nkó yɛɛ Kɛnɛ́pipi lɑlɑ, ɑni kutu kɛ ricɔ. 
Mɛsɛ nɛ mɛsɛ ɑpi piilúkɛ́ nnɛ́í pɑ́ii, ɑmɑ́ ɑ́pi úkɑ yɛ̃, Yeesu yɛɛ pikɛkúrí ń we mɛmɑ́ɑ́. 
Kɑpi riyɔ́pɛ n cɛ́pilɛ̃, Yeesu uu rinɔ́ɔ pi hɛ rɛ ɑ́pi kɑpɛ tíyɛsɛ unyinɛ ukɛ́ lɛ̃ kɑpi n yɛ̃́ kõ hɑ́i nɛ kɛyɑ́ɑ kɛ̃ kɛ Usoi Kɛpipi ɑkɛ yɛ́ pikpɔkpɔ-i n yisi. 
Api nsímɛ́ mmɛ̃ nyi, ɑmɑ́ ɑpi pimɛcɔpɛcirɛ m písɛinɛ rɛ yoo pikpɔkpɔ kɛ́mɛɛ mɛyisɛ? 
Lɛ̃ mɛmɑ́ɑ́, upipirɛtiki ɑpi u pisɛ rɛ: Yo nnyɑ kɛ isé picélɑɑ ɑpi mɑ́ikɛɛ rɛ Elii yɛɛ pɛɛrinɛ likɛ́ kɛlenɛ kɛtẽ kɛtɔ tu? 
Uu pi pɛsɛ rɛ: Mɛsei yɛ mɛ rɛ Elii yɛ pɛɛrinɛ kɛlenɛ ukɛ́ ḿpɑ́ yo nyɔ́ɔnsɛ. Yo nnyɑ kɑi mpíí wɔ́lɑɑlɛ̃ tɛ Usoi Kɛpipi yɛ íwɛ lenɛ mɛyɑ̃́ ɑpi kɛipuri yulu? 
Amɑ́ nɛ lɑ kɛ́ nɔ́ símisi rɛ Elii yɛ pipɛɛmɛ mɑsilɛ pisoi ɑpí nɛ u wɑi lɛ̃ kɑpi ń lɑ yɑrɛ kɑi ukɛcɑ́ɑ́ n wɔ́lɑɑmɛlɛ̃. 
Kɑpi pipirɛtiki tɔrɔɔ n nyɑhɑipɔ, ɑpi pisoi kulúi yɛnu pin kɛwúípi pi wɑilɛ̃, isé picélɑɑ pin nɛ pi kɛŋɛnɛ. 
Kɛ pisoi ɑpi mɛsɛ nɛ mɛsɛ Yeesu n yɛ̃́, ɑi pi kɑikɑi, ɑpi pɛɛ n wúrunkɛɛpɔ pikɛ́ u yɑ́ɑ́si. 
Yeesu uu upipirɛtiki pisɛ rɛ: Yoo mpíí nɔ́ nɛ pɛ̃ kɛŋɛnɛsɛ? 
Utisi unyinɛ uu pɛɛ riwúí tɛ-i rinɔ́ɔ yɔ́ɔri rɛ: Sɑ́ɑ, kɛpɔ́mɛɛ́ kɑrí nɛ kɛnɛ́pipi kɑ. Kuníri kpɛɛ kɛ n lõ yɛ nsímɛ́ kɛ pɔɔnɛsɛlɛnlɛ. 
Ḿpɑ́ yei kɑi kɛ yisi, ku yɛ kɛtẽ kɛ tɑlɛ ɑkɛ ɑtɑlɑlɑɑ́ léeri, kɛn ɑnĩ́ tɑkɑi, ipiŋɛ ii nɑ́nu. Aḿ nɛ pipɔ́pirɛtiki téni rɛ pikɛ́ kuníri kpɛ̃ lɑkɑsɛ, ɑmɑ́ ɑ́pi fe. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Nɔ́ nfɑtɛnɛcirɛ́ pikɔ́ mpuri mmú, ituŋɛ ilɛ́ kɑm nɔ́kɛnɛ́mɛɛ pitónɛ kpɑ́nɛ? Ituŋɛ ilɛ́ kɑm yɛ́ lɛ̃ nɛ nɔ́ n kɑhɑri? Ani nɛ kɛwɑ̃́ nɛ́ kɑm. 
Apí nɛ kɛ u hɑpɔ. Kɛ kɛwɑ̃́ ɑkɛ Yeesu n yɛ̃́, kuníri ɑku kei nní kɛ lémesi, ɑkɛ kɛteni-i lólu ɑkɛ́ m pímiisɛ kɛn ɑtɑlɑlɑɑ́ léeri. 
Yeesu uu kɛusɑɑ pisɛ rɛ: Píyei-mɛ kɑi yɛ nní kɛ wɑ? Uu rinɔ́ɔ yɔ́su rɛ: Hɑ́i uiwɑ̃-mɛ le. 
Mɛpehẽ kulúi kɛ kuníri kpɛ̃ ɑku yɛ nnɑ nɛ míni-i kɛ risɑ́pɔ kɛkɛ́ nɛ n kpu nnyɑ. Amɑ́ in tɛ pɔ́ fe ɑɑ linyinɛ wɑi, ɑ kɑm ɑɑ rɔ́ comɛ. A irɔ́wɛ ripɑí. 
Yeesu uu rinɔ́ɔ u pɛsɛ rɛ: In tɛ pɔ́ fe nɛ́ɛ? Úye un nɛ ukɛfɑ Uléécɑɑ n tɛnɛ, úu líkɑ pɔɔnɛnɛ. 
Mɛsɛ nɛ mɛsɛ, kɛwɑ̃́ kɛ̃ usɑɑ u pupɛi rɛ: Nɛ̃́ nɛ kɛfɑ tɛnɛlɛ! Amɑ́ ɑ kɑm ɑ nɛ́ comɛ, nnɛ́fɑtɛnɛ ńn piyɛ. 
Kɛ Yeesu uu risoiwuí n yɛ̃́ tin u kɑ́lɑɑri, uu kuníri semni rɛ: Pɔ̃́ kuníri kpɛ̃, pɔ́ɔ yɛ ɑtusuí usoi n wɑisɛ ɑɑ nsímɛ́ u pɔɔnɛsɛ, ɑ kɛwɑ̃́ kɛ-i le, kɑpɛ píkɑi kɛkɛmɛɛ pɛɛpɔ, nɛ́ɛ rinɔ́ɔ pɔ́ hɛ. 
Akú lelu kun pupukɛɛ, kun kɔ kɛwɑ̃́ nɑnɛisɛlɛ̃. Kɛwɑ̃́ ɑkɛ́ n we yɑrɛ kɛ kpulɛ, pisoi mɛyɑ̃ ɑpi wɑisɛ rɛ kɛ kpulɛ. 
Amɑ́ Yeesu uu kɛkunipɛ múlú uu kɛ yukusɛ, ɑkɛ yisi ɑkɛ nyɛrɛ. 
Kɛ Yeesu uu lɛ̃ mɛmɑ́ɑ́ kɛ́yɔ n lõ, upipirɛtiki ɑpi pimpehencirɛ u pisɛ rɛ: Yo nnyɑ kɑ́ri fe tɔkɛ́ kuníri nkú lɑ́kɑsɛ? 
Uu pi mɑɑ rɛ: Kɛyómɛ mɛcirɛ kɛ usoi uu yɛ́ yɑ́ɑ́si uú nɛ kuníri nkú mpuri lɑkɑsɛ. 
Api kei yisi ɑpi Kɑlilee kɛtẽ tɛ́ŋ́. Yeesu úu pɛɛ́ lɑ pikɛ́ kei kuu ń we ceri. 
Likumúŋɛ́ rɛ u yɛ pɛɛ upipirɛtiki céésilɛ uu yɛ pi mɑɑ rɛ: Pi Usoi Kɛpipi tininɛ ɑpi pisoi ɑnipɛ-i kɛ wɑi pikɛ́ kɛ kpu. Kɛmɛkpɔ-mɑɑ́, kɛ yɛ́ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ nkpɔ kɛ́mɛɛ yisi. 
Amɑ́ Yeesu pipirɛtiki ɑ́pi mɛyu mɛ̃ kõ. Iwɑmɛ íi kɔ ŋmurɛi pikɛ́ pisɛ. 
Api Kɑpɛrinɑwum tulu. Kɑpi kɛ́yɔ n tu, Yeesu uu pi pisɛ rɛ: Yoo pɛɛ ncée kɛ́mɛɛ nɔ́ kɛ́ŋɛnɛsɛ? 
Amɑ́ ɑpi sɛ́ɛ́ keu. Ái líkɑ nnyɑ, pi pɛɛ ncée kɛ́mɛɛ kɛ́ŋɛnɛ rɛ úye yɛɛ pikɛcɔpɛ uwɛ́ɛ́sɛ? 
Yeesu uu pɛɛ́ tonɛ, uu upipirɛtiki kɛfi nɛ pitɛ́ séi uu pi mɑɑ rɛ: Úye un n lɑ ukɛ́ ufoí, li pisɛ rɛ ukɛ́ nɔ́nnɛ́nɛ́í utɔrɔɔ uu nɔ́nnɛ́nɛ́í ukɛikɔ́. 
Uu kɛwɑ̃́ kɛnyinɛ kpísi uu pikɛcɔpɛ nyɛrɛsɛ, uu kɛ kpísi uu pɛ́kɛ́rí uu pɛɛ pi mɑɑ rɛ: 
Úye un siwɑ̃́ nsí kɛcɔpɛ kɛ́ye kusɑ́nɛ n yɔsí nɛ̃́ nnyɑ, tinɛ́cúruu kuu kusɑ́nɛ yɔsí, úye un kɔ kusɑ́nɛ nɛ́ n yɔsí, ɑ́i rinɛ́cúruu kuú yɔsí, ɑmɑ́ uyɛɛ nɛ́ n tummɛ kuú yɔsí. 
Yohɑni uu u mɑɑ rɛ: Sɑ́ɑ, tɔ unyinɛ yɛnlɛ un ɑníri lɑkɑsɛ nɛ ripɔ́nyíri nnɑŋɛ úu kɔ pɛɛ rɔ́ tíkilɛ̃. Kúu rɔ́ n tíkilɛ̃ nnyɑ, ɑrí nɛ u yulu. 
Amɑ́ Yeesu uu rinɔ́ɔ yɔ́su rɛ: Áni kɑpɛ nɛ u yɛ̀, likumúŋɛ́ rɛ úkɑ úu we yɛɛ yɛ rinɛ́nyíri nɛ mɛwɑisɑŋɑ n wɑ uu kei nní cirɛ nnɛ́kópɛ símisi. 
Úye un kɛrɔ́pirɛ n we uyɛ̃ úu urɔ́lɑ́ɑrɔ. 
Kɑni nɛ̃́ Kirisi pikɔ́ nnyɑ, úye un kɔ rinɛ́nyíri nnyɑ míni nírɛ́-nirɛ́ nɔ́ n hɛ, ɑsei kɑm nɔ́ símisi, liute úu uihɛ́ɛ́ pɑɑnɛ. 
Amɑ́ úye un siwɑ̃́ sɛɛ nɛ nní kɛfɑ nɛ́ n tɛnɛ kɛnyinɛ ɑkópɛ n wɑisɛ, li kutɔsi we pikɛ́ rinɑ́ipɑrɛ mulɛ-mulɛ ńŋmɛ wɑ ɑpi ripɔ́ɔ u lɛsi ɑpi mínimɑɑ-i u sɑ́pɔ. 
In tɛ kupɔ́nípɛ kpɛɛ ɑkópɛ pɔ́ wɑisɛ, ɑ ku riké. Li kutɔsi we ɑ nfɑ́ɑ yɛ̃ nɛ kunípɛ keri nɛ kɑɑ yɛ́ ɑnípɛ kɛtɛ́ m mɑ́ ɑɑ́ nɛ nnɑ kpɔcirɛ́ kɛlõ loni, kei kɛ nnɑ ńn yɛ píkɑi n kpu. 
Kei kɛ ɑpɑpi nyɛɛ yɛ ipiŋɛ n li ɑ́ɑ yɛ n kpí, kei kɛ nnɑ ńn yɛ kɔ n kpu. 
In tɛ kupɔ́nɑ kpɛɛ ɑkópɛ pɔ́ wɑisɛ, ɑ ku riké, li kutɔsi we ɑ kúnɑ kusɛ m mɑ́ ɑɑ́ nɛ nfɑ́ɑ yɛnu nɛ kɑɑ yɛ́ ɑ́nɑ kɛtɛ́ m mɑ́ ɑpí nɛ nnɑ kpɔcirɛ́ kɛlõ kɛ́mɛɛ pɔ́ sɑ́pɔ, kei kɛ nnɑ ńn yɛ píkɑi n kpu. 
Kei kɛ ɑpɑpi nyɛɛ yɛ ipiŋɛ n li ɑ́ɑ yɛ n kpí, kei kɛ nnɑ ńn yɛ kɔ n kpu. 
In tɛ mɛpɔ́nípɛɛ mɛɛ ɑkópɛ pɔ́ wɑisɛ, ɑ mɛ lesɛ ɑɑ kɛtɑɑ-pɔ fómni. Li kutɔsi we pɔkɛ́ Uléécɑɑ iyɔɔpi kɛ́mɛɛ lõ nɛ mɛnípɛɛ mɛsɛ nɛ kɑɑ yɛ́ inípɛɛ itɛ́ m mɑ́ ɑpi pɛɛ nnɑ kpɔcirɛ́ kɛlõ kɛ́mɛɛ pɔ́ sɑ́pɔ, kei kɛ nnɑ ńn yɛ píkɑi n kpu. 
Kei kɛ ɑpɑpi nyɛɛ yɛ ipiŋɛ n li ɑ́ɑ yɛ n kpí, kei kɛ nnɑ ńn yɛ kɔ n kpu. 
Likumúŋɛ́ rɛ: Pisoi nnɛ́í kɛcɑ́ɑ́ kɑpi nnɑ kɔɔnunɛ yɑrɛ kɑpi yɛ nní ilukɛ kɛcɑ́ɑ́ nnyɑ́si n fulɛi. 
Nnyɑ́si yɛ linyinɛ kɛcirɛ lɛ, ɑmɑ́ mmɛlõ mɛn n kpu, yo kɑpi yɛ nɛ n lɑrukusɛ? Ani nnyɑ́si mɛcɔ n lɑ́ɑ́rú, nkíŋniŋɛ nkɛ́ kɔ nɔ́kɛnɛ́cɔpɛ n we.
