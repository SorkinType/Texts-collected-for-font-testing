Mɑtiyee.
6.
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Ani nɔ́mɛnɛ́círɛ tĩ, ɑ́ni kɑpɛ yɛ lɛ̃ kɑí nɛ n sɑ́ rɛ nɔkɛ́ Uléécɑɑ nnyɑ wɑ pisoi inipɛɛ-i wɑ rɛ pikɛ́ nɔ́ yɛ̃́ pikɛ́ pɑkɑrɛ nnyɑ. Insɑ́ lɛ̃, Nɔ́unɛ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we úu ihɛ́ɛ́ íkɑ nɔ́ pɑnɛ. 
Lɛ̃ nnyɑ pɔn úwɛkɔɔ́ linyinɛ n hɛ, kɑpɛ kuloo pi pɔkɛ́ riyóó yɑrɛ kɛ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ ɑpi yɛ ɑyómɛyɑ́hɑɑlee-i nɛ icée kɛcɑ́ɑ́ n wɑi rɛ pisoi pikɛ́ pi pɑkɑrɛ mɛcɔ. Asei kɛcɑ́ɑ́, pi piihɛ́ɛ́ piyɔ́ɔ́ mɑsilɛ lɛ̃! 
Amɑ́ pɔ́ɔn yɛ linyinɛ úwɛkɔɔ́ n hɛ, kupɔ́nípɛ mii ɑ́ku kɑpɛ lɛ̃ kɛ kulukɛ-lukɛ ɑku n wɑ ceri. 
A yɛ risɛ́ɛ́ nɛ hɛ. Upɔ́sɑ́ɑ Uléécɑɑ yɛɛ mɛpɔ́wɑi kɑɑ́ nɛ risɛ́ɛ́ n wɑi n nyɑ́ni yɛ́ pɛɛ pɔ́ hɛ́ɛ́lɛ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɔn yɛ kɛyómɛ n yɑ́ɑ́si, ɑ́ni kɑpɛ yɛ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ mɛcɔ n wɑi. Pi yɛ n lɑlɛ pikɛ́ ɑyómɛyɑ́hɑɑlee-i nɛ sicéepɛnɛ-i n nyɛnu pin nɛ kɛyómɛ yɑ́ɑ́si rɛ pisoi pikɛ́ pi yɛ̃ nnyɑ. Asei yɛ nyɛ kɑm nní nɔ́ símisi rɛ pi piihɛ́ɛ́ piyɔ́ɔ́ mɑsilɛ. 
Amɑ́ pɔ́ɔn yɛ n lɑ pɔkɛ́ kɛyómɛ yɑ́ɑ́si, ɑ yɛ kupɔ́lee lõ ɑɑ hɑ́nɛsi ɑɑ pɛɛ Upɔ́sɑ́ɑ Uléécɑɑ kɛ úkɑ úu n nyɑ́ni yɑ́ɑ́si. Upɔ́sɑ́ɑ Uléécɑɑ yɛɛ n nyɑ́ni lɛ̃ kɑɑ́ nɛ risɛ́ɛ́ n wɑi yɛ́ ipɔ́hɛ́ɛ́ pɔ́ pɑ. 
Nɔn yɛ kɛyómɛ n yɑ́ɑ́si, ɑ́ni kɑpɛ yɛ mɛ́woo kusímɛ́nɑɑpí n wɑi yɑrɛ pɛpɛɛ ɑ́pi Uléécɑɑ n nyu mɛcɔ. Pi yɛ m músu rɛ pin lɛ̃ mɛyɑ̃́ n símɑɑnkɛɛ, Uléécɑɑ yɛ́ nɛ pi kõ. 
Áni kɑpɛ pimɛcɔ n wɑi. Nɔ́unɛ́sɑ́ɑ Uléécɑɑ yɛ lɛ̃ kɑi nɔ́ m pisɛ nyulɛ ɑni yɛ kɛlenɛ li u welu. 
Lɛ̃ kɑi m pisɛ rɛ nɔkɛ́ yɛ kɛyómɛ n yɑ́ɑ́si yɛ nní: Urɔ́sɑ́ɑ, pɔ́ɔ kɛyómɛcɑɑ-pɔ ń we, 
Pisoi pikɛ́ céri rɛ tipɔ́nyíri yɛ kóólɛlɛnlɛ, Ipɔ́yɔ́ɔpi ikɛ́ kɑm, Mɛpɔ́lɑ mɛkɛ́ kɛtẽ kɛcɑ́ɑ́ wɑ yɑrɛ kɛyómɛcɑɑ-pɔ. 
A yɛ ḿpɑ́ kɛweesi kɛ́ye ilukɛ rɔ́ hɛ, 
A ɑrɔ́kópɛ rɔ́ sɑ́rɛi yɑrɛ kɛ tɔ́ɔ yɛ kɔ picɔ n sɑ́rɛi pin ɑkópɛ rɔ́ n wɑ mɛcɔ. 
Kɑpɛ tíyɛsɛ tɔkɛ́ pipeikɛɛ kɛ́mɛɛ loó. Amɑ́ ɑ kuníri ɑnipɛ-i rɔ́ lesɛ. Li we rɛ pɔ́ɔ iyɔ́ɔpi nɛ ńnɑŋɛ nɛ ḿmulɛ te ḿpɑ́ píyei-pɔ. Amí. 
Nɔn yɛ picɔ ɑkópɛ kɑpi nɔ́ n wɑ pi n sɑ́rɛi, Nɔ́unɛ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we yɛ́ kɔ nɔ́ɑnɛ́kɔ́ nɔ́ sɑ́rɛi. 
Amɑ́ nɔnsɑ́ yɛ picɔ ɑkópɛ pi n sɑ́rɛi, Nɔ́unɛ́sɑ́ɑ pɑ́ɑ yɛ́ kɔ nɔ́ɑnɛ́kɔ́ nɔ́ sɑ́rɛi. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɔn yɛ ɑnɔ́ɔ m pɑhɑɑlɛ̃, ɑ́ni kɑpɛ yɛ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ mɛcɔ kɛ́yu n cɑɑilɛ̃. Pi yɛ kɛ́yu n cɔ́hɔnɛlɛnlɛ rɛ pisoi pikɛ́ céri rɛ ɑnɔ́ɔ kɑpi pɑhɑɑlɛ̃. Asei kɑm nní nɔ́ símisi rɛ pi piihɛ́ɛ́ lɛ̃ piyɔ́ɔ́ mɑsilɛ. 
Amɑ́ pɔ́ɔn yɛ ɑnɔ́ɔ m pɑhɑɑlɛ̃, ɑ yɛ lihí ɑɑ ríyu tulɑɑli fóm. 
Pisoi ɑ́pi kɑpɛ nɛ n céri rɛ ɑnɔ́ɔ kɑɑ pɑhɑɑlɛ̃ nnyɑ. Upɔ́sɑ́ɑ Uléécɑɑ kɛ úkɑ úu n nyɑ́ni ŋmɑnɛ yɛɛ nyɑ́ni lɛ̃ kɑɑ́ nɛ risɛ́ɛ́ n wɑi. Uyɛɛ yɛ́ pɛɛ ipɔ́hɛ́ɛ́ pɔ́ pɑ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Áni kɑpɛ n wɛ́ɛ́si nɔkɛ́ kɛtẽ nté mɛmɑ́ yɛ̃́ nɔkɛ́ curii. Apɑpi nɛ ncɑ́mɛ́ yɛ̀ɛ̀ nté mɛ cɑɑilɛ, piyɑɑ pɔ́ɔ ihɑ́nɑɑ foru ɑpi mɛ yɑ́ɑ́i. 
Amɑ́ kɛyómɛcɑɑ-pɔ kɑí nɛ sɑ́ rɛ nɔkɛ́ mɛmɑ́ yɛ̃́ nɔkɛ́ curii. Apɑpi nɛ ncɑ́mɛ́ nn yɛ kei fe nkɛ́ mɛ cɑɑi, piyɑɑ ɑ́pi yɛ́ kɔ fe pikɛ́ ihɑ́nɑɑ fori pikɛ́ mɛ yɑ́ɑ́i. 
Li we rɛ kei kɛ mɛpɔ́mɑ́ ɑmɛ ń we kɛ kɛpɔ́múŋɛ́ ɑkɛ́ wepɔ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Inípɛɛ yɛɛ ipiŋɛ kɛfirɑ́ɑ. Ipɔ́nípɛɛ in n nyɑḿ, i yɛ ipiŋɛ nnɛ́í n kpɑ̃́iilɛnlɛ. 
Amɑ́ ipɔ́nípɛɛ in m pɔ́ɔ́lú, ipɔ́piŋɛ nnɛ́í yɛɛ yɛ kuŋmɑhɑ-i n we. Lɛ̃ nnyɑ, in tɛ mɛtɛ́í mɛ̃ kɑɑ m mɑ́ yɛ pɑnsɛlɛ kuŋmɑhɑ, ɑ yɛ̃ tɛ kuŋmɑhɑ píimɑ-i kɑɑ́ we. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Úkɑ úu yɛ́ fe ukɛ́ kumúŋɛ́ kusɛ kɛ́mɛɛ píyɔsɑɑ pitɛ́ n tikilɛ̃. Ḿpɑ́ likɛ́ íye wɑ, u yɛ́ usɛ ipuri yɛ̀ uu ucɔ ikɔ́ n lɑ, nɛ́ɛ u yɛ́ usɛ nɛ kutu n cɔlɛ̃ úu yɛ pɛɛ ucɔ n wɑisɛlɛ̃. Áni yɛ́ fe nɔkɛ́ kumúŋɛ́ kusɛ kɛ́mɛɛ Uléécɑɑ nɛ siwóó n tikilɛ̃. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Lɛ̃ nnyɑ kɑm nɔ́ mɑɑ rɛ ɑ́ni kɑpɛ lɛ̃ kɑni yɛ́ n li nɛ lɛ̃ kɑni yɛ́ ń ntí nɔkɛ́ nɛ nfɑ́ɑ m mɑ́ nɛ ilũ kɑni yɛ́ nɔ́inɛ́piŋɛ n yɑ́rii nfɑsimɛ́ wɑ. Nfɑ́ɑ nn ilukɛ fee, nɛ́ɛ ipiŋɛ yɛɛ ilũ fe? 
Ani sinúipi ripɑí nkɛ̃́. Ási yɛ lukɛsi, ɑ́si yɛ kpɑsi, ɑ́si yɛ líkɑ risímɛ súũ. Amɑ́ Nɔ́unɛ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we yɛ si kpíniilɛ! Yɑrɛ nɔ̃́ ŋmɑɑ ɑ́ni pɛɛ ríyu mɑ́ lin sinúipi sɛ̃ fee? 
Nɔ́ úye yɛɛ yɛ́ fe uu nfɑsimɛ́ wɑi uú hɑ nɛ unfɑɑ ɑŋmɛ̃ ḿpɑ́ nkɑ́ripi kpɑ́sɛ? 
Yo nnyɑ kɑni pɛɛ nkpɑ́ni ilũ kɑni yɛ́ nɔ́inɛ́piŋɛ n yɑ́rii nfɑsimɛ́ wɑi? Ani mɛyúi mɛɛ yɛ sicɑrɛ-i n fɛlɛsi ripɑí nkɛ̃́! Ámɛ pikɛi píkɑ wɑi, ɑ́mɛ kɔ ilũ íkɑ nyɛlɑɑnkɛɛ. 
Amɑ́ kɛ́ nɔ́ símisi rɛ ḿpɑ́ nɛ lɛ̃, Uyɔ́ɔpi píimɑ Sɑlomɔɔ ricuruu nɛ ummulɛ mmú nnɛ́í, úu pɛɛ ilũ inyinɛ mɑ́ yɛɛ ń nyɑḿ ɑi mɛfɛlɛ mɛnyinɛ tulu. 
Nɔ́ pisoi mpí nfɑtɛnɛ ńn piyɛ. In tɛ Uléécɑɑ yɛ mɛyúi mɛɛ nɛni kɛcɑrɛ-i ń we pin kɔ kóso nnɑ mɛ ŋmɑɑsinɛ lɛ̃ sɔnukusɛ, nɔ̃́ kɛcɑ́ɑ́ kúu yɛ́ pɛɛ mɛníŋɛ ripɑíí? 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ yɛ nfɑsimɛ́ wɑ nɔkɛ́ mɑɑ rɛ: Yo kɑri yɛ́ li nɛ́ɛ yo kɑri yɛ́ ń ntí nɛ́ɛ yo kɑri yɛ́ tɑ̃. 
Pɛpɛɛ ɑ́pi Uléécɑɑ n nyu pɛɛ lɛ̃ likɔ́ mɛsɛ́rɛ wɛ́ɛ́si. Amɑ́ Nɔ́unɛ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we yɛ nyulɛ lɛ̃ kɑi nɔ́ m pisɛ. 
Uléécɑɑ iyɔɔpi kɑni yɛ́ mɛfoí wɛ́ɛ́si, ɑni kɔ lɛ̃ kuu m pisɛ n wɑi, u yɛ́ kɔ litɔ́rɔɔ nní nnɛ́í mpuri pihɛɛ nɔ́ rikpɑ́. 
Lɛ̃ nnyɑ, ɑ́ni kɑpɛ yɛ nfɑsimɛ́ wɑ rɛ íye kɑi yɛ́ kóso n we. Li we rɛ ḿpɑ́ kɛyɑ́ɑ kɛ́ye nɛ kɛnkɔ́ yɛ mu.
