Mɑtiyee.
15.
Kei kɛ Pifɑrisi nɛ isé picélɑɑ pinyinɛ ɑpi Yerusɑlɛm-pɔ leemɛ ɑpi Yeesu lɛɛri ɑpi u pisɛ rɛ: 
Yo nnyɑ kɛ pipɔ́pirɛtiki ɑpi yɛ inyɛ́kii kɛ pirɔ́kpurɛ ɑpi n yekei rilɔ́ɔ́? Lɛlɛɛ rɛ ɑ́pi yɛ nɑlɛ pikɛ́ kɛlenɛ li yɑrɛ kɑi mɛkɛɛ-mɛ n wemɛ? 
Yeesu uu pi pɛsɛ rɛ: Nɔ̃́ ni, yo nnyɑ kɛ nɔ́ɔ Uléécɑɑ isé rilɔ́ɔ́ ɑni nɔ́pinɛ́kpurɛ inyɛkii n tikilɛ̃? 
Uléécɑɑ yɛ mɑɑ rɛ: A upɔ́sɑ́ɑ nɛ upɔ́ni ríyu wɑisɛ. Uu kɔ rɛ úye un usɑ́ɑ nɛ́ɛ úni ɑnɔ́ɔ n wɑ, li pisɛ rɛ pikɛ́ liute kpu. 
Amɑ́ nɔ̃́ nɛ yɛ rɛ: Úye un usɑ́ɑ nɛ́ɛ úni m mɑɑ rɛ u lɛ̃ kuú pɛɛ́ ń lɑ ukɛ́ nɛ u n lɛ̃́ Uléécɑɑ hɛlɛ, 
u yɛ́ fe uu lɛ̃ nnyɑ usɑ́ɑ nɛ́ɛ úni nkpéni pilɛ̃́ yɑ́. Uléécɑɑ nsímɛ́ kɑni lɛ̃ fómpɔ ɑni pɛɛ nɔ́pinɛ́kpurɛ inyɛkii n tikilɛ̃. 
Nɔ́ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ mpí! Antepu Esɑyi yɛ ɑlɑri mɑ́lɛ ukɛ́ nɔ́kɛnɛ́cɑ́ɑ́ mɑɑ rɛ: 
Nnɔ́ɔ-i ŋmɑnɛ kɛ pisoi mpí ɑpi ríyu nɛ́ wɑisɛlɛ̃, ɑmɑ́ ɑ́pi pɛɛ pisifɑ-i nɛ nɛ́ kɔ́lɛ̃. 
Piiyɑ́hɑɑ yɛ likpɑ́rɑ́kɔɔ́ lɛ. Pi rinyɑlɛ pin sisoipipi inyɛkii kpísi pin céési yɑrɛ inɛ́kɔ́ yɛ nyi. 
Yeesu uu pɛɛ risoiwuí cɑ́pinɛ uu pi mɑɑ rɛ: Ani kutu nɛ́ ricɔ ɑni kóm ńsɔnɛ: 
Ái lɛlɛɛ yɛ usoi nnɔɔ-i n lompɔ lɛɛ yɛ mɛ́kpɛrinkpɛ u wɑisɛ, ɑmɑ́ lɛlɛɛ yɛ unnɔɔ-i n léemɛ lɛɛ yɛ mɛ́kpɛrinkpɛ u wɑisɛ. 
Kei kɛ upipirɛtiki ɑpi ukɛkúrí rinyɔsɔ́pɔ ɑpi u mɑɑ rɛ: Pɔ nyu rɛ lɛ̃ kɑɑ nní n símisi yɛ Pifɑrisi mɛyɑ̃́ ticɔ́ŋ? 
Uu pi pɛsɛ rɛ: Kuléé kúye kun n we, kɑ́i Unɛ́sɑ́ɑ yɛɛ kɛyómɛcɑɑ-pɔ ń we yɛɛ ku tɑmɛsi, pi ku wɛhɛnɛ. 
Ani pi riyɑ́ kei. Pi nkpɑ́ni yɑrɛ pinyíyɛ pɛɛ pinyíyɛcɔ n tórikɛɛ? Unyíyɛ un unyíyɛcɔ n tɔrɔi, pi sɔ́ntilɛ pikɛ́ hɑ nɛ pikɛtɛ́ kúcoŋ́horɛ kunyinɛ-i lóipɔ. 
Piyɛɛ uu pɛɛ rinɔ́ɔ yɔ́su uu Yeesu mɑɑ rɛ: A kɛnyɑ́rũ nkɛ́ ɑsei rɔ́ símisi. 
Yeesu uu u pɛsɛ rɛ: Aí nɛ nɔ̃́ pɛ́nɛ ɑ́ni kɑhɑnɛ nkpɑ́ni líkɑ n kõo? 
Áni kõ tɛ lɛ̃ nnɛ́í lɛɛ yɛ usoi nnɔɔ-i n lompɔ yɛ̀ɛ̀ kɛfɑ-i hɑlɛ uu wɑi uu iyɑɑ leepɔ uu li lesɛrii? 
Kɛfɑ kɛ́mɛɛ kɛ nsímɛ́ nn yɛ n we, nn pɛɛ nnɔ́ɔ-i tíkimɛ nń nɛ leeri. Mmɛ mɛɛ yɛ usoi mɛ́kpɛrinkpɛ wɑisɛ. 
Lɛ̃ nnyɑ, kɛfɑ-i kɛ simúŋɛ́ kópɛ ɑsi léeri. Sɛsɛɛ yɛ tíyɛsɛ kɛ usoi uu yɛ usoi kpu, uu ucɔ unɔsi nɛ́ɛ ucɔ ulɑ kuwɑ́ncɔ wɑi, uu iwɑ́sɑ́ wɑi, uu yɑ́ɑ́i, uu kɔ ucɔ nɛ nsímɛ́ kpɑ́rɑ́, nɛ́ɛ uu ucɔ lɑ́mɑɑnkɛɛ. 
Lɛlɛɛ yɛ usoi mɛ́kpɛrinkpɛ n wɑisɛ yɛ lɛ̃. Amɑ́ usoi unsɑ́ n nɑlɛ yɑrɛ kɛ irɔ́nyɛ́kii ii ḿ mɑ ukɛ́ kɛlenɛ li, ɑ́i yɛ́ mɛ́kpɛrinkpɛ u wɑisɛ. 
Yeesu uu limɛmɑ́ɑ́ kei yisi uu Tiiri nɛ Sitɔɔ sitẽ-mɛ̃ tɔ́mpɔ. 
Kɑnɑɑ kɛtẽ unɔsi unyinɛ yɛɛ kei n hɑ́pɔ uu yisi uu Yeesu lɛɛpɔ, uu pupɛi rɛ: Upíimɑ, Tɑfiti Kɛpipi, ɑ inɛ́wɛ yɛ̃. Kuníri yɛ unɛ́kpére lémɑɑpɔlɛnlɛ ɑ́i nyɑḿ! 
Yeesu uu kɔ lɛ̃ n sɛ́ɛ́lɛ̃ lɛ́kɛ́-lɛ́kɛ́. Kei kɛ upipirɛtiki ɑpi ukɛkúrí rinyɔsɔ́pɔ ɑpi rɛ: A unɔ́si nkó nɛ kutu ricɔ ukɛ́ tɔ́mpɔ. U welɛ un pupukɛɛ un nɛ rɔ́ tíkilɛ̃! 
Yeesu uu pi pɛsɛ rɛ Isirɑyɛɛli pikɔ́ pɛɛ ń we yɑrɛ isɑ́ŋ yɛɛ m pɔ íi nɛ usẽ́ ŋmɑnɛ kɛ́mɛɛ kɛ Uléécɑɑ uu nɛ́ tummɛ. 
Kei kɛ unɔ́si uyɛ̃ uu pɛɛ yisipɔ uú hɑ ukɛnyɑlɛ-i wúlɑ uu u mɑɑ rɛ: A risuúlu ɑ nɛ́ lɛ̃́, Upíimɑ. 
Yeesu uu u pɛsɛ rɛ: Ái nyɑḿ tɛ pikɛ́ siwɑ̃́ ilukɛ kpísi pikɛ́ sipɔpipi risɑ́. 
Unɔ́si uyɛ̃ uu kɔ kpɑ́ rɛ: Mɛsei yɛ mɛ, Upíimɑ. Amɑ́ ḿpɑ́ nɛ lɛ̃, sipɔpipi pite pin n le, si yɛ lɛlɛɛ kɛteni-i m póroi n hínilɛ sin le. 
Yeesu uu pɛɛ u pɛsɛ rɛ: A tíyɛ mpɔ́fɑtɛnɛ yɛ piyɛ kpɑ! Uléécɑɑ yɛ́ tíyɛsɛ lɛ̃ kɑɑ ń lɑ ɑi wɑi. Kei nní uukpére uyɛ̃ uu mɛsɛ nɛ mɛsɛ pélu. 
Yeesu uu kei yisi uu Kɑlilee kupiyɛ ritimɛ sĩ. Uu pɛɛ rikúú rinyinɛ tɑɑ́ uú tonɛ. 
Kei kɛ pisoi kulúi ɑpí nɛ pipɑ́íkɔɔ́ nɛ pinyíyɛ nɛ síkɑnkɑ́lɑ́kɑ́ nɛ pisímɛ́cirɛ́ nɛ pitóikɔ́ picɔ picɔ u kɑ́lɑɑpɔ ɑpi uɑnɑ-i pi lɑikɛɛ, uu pi pɔisɛntɛ. 
Kɛ pisoi ɑpi pisímɛ́cirɛ́ n yɛ̃́ pin símisi, síkɑnkɑ́lɑ́kɑ́ sin pélɑɑlɛ̃, pipɑ́íkɔɔ́ pin sɔ́nɛ, pinyíyɛ pin nyɑ́ni, ɑi nnɔ́ɔ pi yipu ɑpi pɛɛ Isirɑyɛɛli pikɔ́ Uleecɑɑ ríyu n wɑisɛ. 
Yeesu uu pɛɛ upipirɛtiki séi uu pi mɑɑ rɛ: Pisoi mpí icɔ yɛ íwɛ nɛ́ welɛ. Siyɑ́ɑ sitɑɑni yɛ nsí kɑpi kɛnɛ́kúrí we ɑ́pi kɔ nkpéni líkɑ mɑ́ pikɛ́ li. Ám kɔ lɑ kɛ́ nɛ nkṹ pi kuisɛ. Insɑ́ lɛ̃, li yɛ́ hɑ ncée pi feriyɛ. 
Upipirɛtiki ɑpi pɛɛ u pisɛ rɛ: Yei kɑri yɛ́ kuyúi nkú-i ilukɛ mɛyɑ̃́ tu pisoi píimɑ mpí pikɛ́ nɛ n lɛ? 
Yeesu uu pɛɛ pi pisɛ rɛ: Akpɔ́nɔ́ ɑlɛ́ kɑni mɑ́? Api rɛ: Asɛɛi nɛ sikpíntomɛ́pi nkɑ́ripi. 
Kei kuu pɛɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ tinɔ́ɔ hɛ rɛ pikɛ́ kɛteni-i tonɛ. 
Kɑpi n tónɛ, uu ɑkpɔ́nɔ́ ɑsɛɛi nyɛ̃ nɛ ikpíntomɛ́ iyɛ̃ kpísi uu Uléécɑɑ pɔɔnɛsɛ uu pɛɛ ɑ kpɔ́kɔrinɛ uu upipirɛtiki pɑ, ɑpi pisoi nnɛ́í hɔ́ɔ́nɛ. 
Ḿpɑ́ úye uu le uu lɛpu. Pipirɛtiki ɑpi ɑkpɔ́nɔ́ keri nyɛɛ m pɔɔnɛ kɔ́ɔ́i ɑi ɑnɛ́rɛ ɑsɛɛi yipu. 
Pɛpɛɛ pɛɛ n li yɛ lelɛ pitisi ŋmɑnɛ ɑ́kotokú ɑnɑ (4.000), ɑ́pi pinɔ́si nɛ siwɑ̃́ kɛmɛ́ɛ kɛɛ̃. 
Yeesu uu pɛɛ lɛ̃ mɛmɑ́ɑ́ pi kuisɛ, uyɛ̃ uu pɛɛ kúninɔi loni uú nɛ Mɑkɑtɑni kɛtẽ-mɛ̃ tɔ́mpɔ.
