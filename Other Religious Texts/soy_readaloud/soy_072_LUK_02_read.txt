Luki.
2.
Kɛtẽ nnɛ́í uyɔɔpi Okusiti uu kumúŋɛ́ kpɛ-i rinɔ́ɔ hɛ rɛ pikɛ́ kɛtẽ nnɛ́í pisoi kɛɛ̃. 
Pikɛ́ɛ̃ pɛpɛɛ pifoí. Pi wɑlɛ kumúŋɛ́ kpɛ-i kɛ Kiriniisi uú pɛɛ Sirii kɛtẽ kɛcɑ́ɑ́ iyɔ́ɔpi n le. 
Ḿpɑ́ úye uu ukuyu pikɛ́ɛ̃ sĩ. 
Yosɛfi uu Kɑlilee kɛtẽ kɛ́mɛɛ Nɑsɑrɛti kuyu-i yisi uu Yutee kɛteni-i Tɑfiti kuyu kɑpi yɛ n sée rɛ Pɛtɛlɛhɛm sĩ. Kuyu kpɛ-i kɑpi uyɔ́ɔpi Tɑfiti mɑri. Yosɛfi ricuruu yɛ Tɑfiti kɛyɔ nɛ ukɛpirɛ ukɔ́ lɛ. 
Uu kei pikɛ́ɛ̃ sĩ, u nɛ uunɔsi felei-felei Mɑɑri yɛɛ kɛfɑ m mɑ́. 
Pin Pɛtɛlɛhɛm-i we, ukɛmɑrɛyɑɑ ɑkɛ tulu 
uu ukɛwɑ̃ foí mɑ́ru kɛtisi, uu ɑsɑ́ŋɑ́ɑ́ kɛ yɑrii uu kuhɔpi kɛcɑ́ɑ́ kɛ finsɛ, kɑpi picéetɔŋɛ́ kulee-i kɛtónɛ m pɑɑ nnyɑ. 
Kɛlõ kɛsɛ kɛ-i kɛ pisɛɛsẽ́ pinyinɛ ɑpí pɛɛ kɛsinɛ we pin sicɑrɛ-i piisɛɛ kɛcɑ́ɑ́ pɑílɛ̃. 
Upíimɑ uleecɑɑtumɛ uu pikɛyu-i kpɑ́fúmɛ, Upíimɑ mɛyɔɔpi ɑmɛ́ n tɛ́lu mɛn kɛkɑ́lɛ pi wɑilɛ̃. Iwɑmɛ ii pi wɑi mɛyɑ́nsei. 
Uléécɑɑtumɛ uu pɛɛ pi mɑɑ rɛ: Iwɑmɛ íi kɑpɛ nɔ́ wɑ. Nsímɛ́ sɔnɛ mɛɛ yɛ́ Isirɑyɛɛli pikɔ́ nnɛ́í ripɔɔ mɛyɑ̃́ n níŋukusɛ kɑḿ nɛ nɔ́ kɑ. 
Nɛni kɛyɑɑ nkɛ́ kɑpi Tɑfiti kuyu-i Nɔ́unɛ́yulɑlɛ mɑri, uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ, Upíimɑ. 
Lɛ̃ kɑni yɛ́ nɛ u n céri yɛ nní: Nɔ́ kɛwɑ̃́ fɛnfɛ̃í lɛɛpɔ kɑpi kusɑ́ŋɑ́ɑ́ ḿ pílɛsi ɑpi kuhɔpi kɛcɑ́ɑ́ finsɛ. 
Mɛsɛ nɛ mɛsɛ, piléécɑɑtumɛ picɔ ɑpi ufoí nkó kɛcɑ́ɑ́ kpɑ́pɔ ɑpi kulúi wɑi, ɑpi pɛɛ Uléécɑɑ mɛyɔɔpi n yom tɛ 
Uléécɑɑ yɛɛ kɛyómɛcɑɑ-pɔ ipɑkɑrɛ te. Nkíŋniŋɛ nkɛ́ pɛ̃ kuu kɛtẽ kɛcɑ́ɑ́ ń lɑ kɛcɔpɛ n we. 
Kɛ piléécɑɑtumɛ ɑpi pi rin-yɑ́ ɑpi kɛyómɛcɑɑ-mɛ̃ pɛlɛ, pisɛɛsẽ́ ɑpi pɛɛ símɑɑnɛ rɛ: Tɔkɛ́ Pɛtɛlɛhɛm hɑ. Li pisɛ rɛ tɔkɛ́ lɛlɛɛ n wɑ kɛ Upíimɑ uu rɔ́ n céreisɛ yɛ̃. 
Api kei mɛkɛɛkɛɛ sĩ ɑpí hɑ Mɑɑri nɛ Yosɛfi lɛɛpɔ, kɛwɑ̃́ kɛn kuhɔpi kɛ́mɛɛ finu. 
Kɑpi kɛ n yɛ̃́, ɑpi lɛ̃ kɛ uléécɑɑtumɛ uu kɛwɑ̃́ kɛ̃ nsímɛ́ n símisi kɛɛnkɛɛ. 
Lɛ̃ kɛ pisɛɛsẽ́ ɑpi n símisi ɑi pɛ̃ nnɛ́í pɛɛ kutu pi n cɔ́lɛ̃ pírí wɑi. 
Mɑɑri uu mmɛ̃ nnɛ́í ukɛfɑ-i múlú un nkɛcɑ́ɑ́ músu ńsɔnɛ. 
Pisɛɛsẽ́ ɑpi lɛ̃ mɛmɑ́ɑ́ nkúlɛcee pɔlɔ. Api Uléécɑɑ rinyiri tɑɑ́sɛ ɑpi umɛyɔɔpi n yom tɛ u tíyɛsɛ ɑpi kóm ɑpi kɔ yɛnu nnyɑ. Linnɛ́í yɛ wɑ yɑrɛ lɛ̃ kɛ uléécɑɑtumɛ uu pi n símisi. 
Siyɑ́ɑ sisɛɛi mɛmɑ́ɑ́, kɛwɑ̃́ kɛkéyɑɑ ɑkɛ tulu, ɑpi rinyíri kɛ hɛ rɛ Yeesu. Tinyíri tɛ̃ kɛ uléécɑɑtumɛ uu pi céesi Mɑɑri uu kɛlenɛ fɑpisi. 
Lɛ̃ mɛmɑ́ɑ́, ituŋɛ ii tulu rɛ pikɛ́ inyɛ́kii wɑ pikɛ́ nɛ pimɛcirɛ hɛɛrɛ yɑrɛ kɑi Moisi isé kɛ́mɛɛ m pisɛ, Mɑɑri nɛ Yosɛfi ɑpi pɛɛ́ nɛ kɛwɑ̃́ Yerusɑlɛm kuyu sĩ tɛ pikɛ́ Upíimɑ kɛ nyísɛ. 
Likumúŋɛ́ rɛ li Upíimɑ isé kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Upíimɑ yɛɛ kɛwɑ́ntisi foí te. 
Api kɔ inyɔ́ɔnsɛ kɛ isé isɛ iyɛ̃ ii m pisɛ wɑi: Sikpéékpeepi sitɛ́ nɛ́ɛ ɑlɔ́ɔlɑ̃ síńsɑ́ ɑtɛ́ kɑpi yɛ́ nɛ wɑ. 
Utisi unyinɛ kɑpi yɛ n sée rɛ Simeyɔɔ yɛ pɛɛ Yerusɑlɛm-i welɛ. Utisi uyɛ̃ yɛ pɛɛ usoi sɔnɛ lɛ un Uléécɑɑ riwurɛ mɑ́ un uyɛ̃ kɑí pɛɛ m pisɛ rɛ ukɛ́ Isirɑyɛɛli yoriyɛ mɛ̃́. Nfɑ́ɑsɔnɛ yɛ pɛɛ ukɛmɛɛ́ welɛ. 
Uléécɑɑ uú nɛ Nfɑ́ɑsɔnɛ tiki uú nɛ u símisi rɛ u yɛ́ uyɛ̃ kɛ Upíimɑ uu n wɛ́ɛ uu tumti nɛ uinipɛɛ yɛ̃ uu kɛlenɛ kpi. 
Nfɑ́ɑsɔnɛ nn tíyɛsɛ Simeyɔɔ uu Uléécɑɑ kɛyɔ sĩ. Kɛ Yeesu pikɔ́ ɑpí nɛ u n hɑ́pɔ pikɛ́ inyɛ́kii iyɛ̃ kɛ isé ii m pisɛ wɑ, 
Simeyɔɔ uu u kpísi, uu Uléécɑɑ pɑkɑrɛ uu rɛ: 
Upíimɑ, tipɔ́nɔ́ɔ kɑɑ n yekei yɛ nkpéni lilɛ. Pɔ́ fe ɑɑ nkpéni nɛ̃́ upɔ́kɛikɔ́ tíyɛ ɑm nkíŋniŋɛ-i kpi. 
Likumúŋɛ́ rɛ nɛ nɛ inɛ́nípɛɛ pipɔ́yulɑlɛ yɛnlɛ. 
Pɔ Uyóriyɛ kpíimɛlɛ ɑɑ isoipuri nnɛ́í kɛyu-i wɑi. 
Uyɛɛ mɛtɛ́í mɛɛ yɛ́ n tíyɛsɛ kɛtẽ nnɛ́í ɑyu ɑɑ pɔ́ céru. Uyɛɛ yɛ́ tíyɛsɛ mpɔ́puri Isirɑyɛɛli rinyiri ɑrí tɑɑ́. 
Yeesu usɑɑ nɛ uuni ɑpi pírí wɑi nɛ lɛ̃ kɛ Simeyɔɔ uu Yeesu kɛcɑ́ɑ́ n símisi. 
Simeyɔɔ uu ɑnɔ́ɔ sɔnɛ pi tikisɛ, uu Yeesu uni Mɑɑri mɑɑ rɛ: Uléécɑɑ yɛ kɛwɑ̃́ nkɛ́ wɛ́ɛ rɛ kɛkɛ́ tíyɛsɛ pisoi mɛyɑ̃ pikɛ́ loó, pimɛyɑ̃ pikɛ́ kɔ Isirɑyɛɛli-i yisi. Kɛkɛɛ Uléécɑɑ linyísɛ-nyísɛ lɛɛ yɛ́ pisoi nɛ Uléécɑɑ kɛcɔpɛ m pɑ́sɑisɛnɛ. 
U yɛ́ lɛ̃ kɛ pisoi mɛyɑ̃ ɑpi risɛ́ɛ́ nɛ pisifɑ-i m músu yɑ́mnɛ. Amɑ́ pɔ̃́ kɛuni, pɔ́ ntóósi kõ yɑrɛ kɛtɛ́hɛ kɑpi pɔ́ tuhu. 
Unɔ́si unyinɛ kɑpi yɛ n sée rɛ Aɑni yɛ kɔ pɛɛ́ welɛ. U ɑntepu lɛ. Fɑnuwɛɛli ukpére lo Asɛɛ kumɑrɛ-i. U pɛɛ kpurunulɛ ńsɔnɛ. Aŋmɛ̃ ɑsɛɛi kɛ u nɛ uulɑ pɛɛ n sɔ́ɔnnɛ unnɑŋɛ kɛ́mɛɛ ɑpi wɑ. 
Lɛ̃ mɛmɑ́ɑ́, uu ikúmɑnnɔsi tonɛ uú hɑ nɛ ɑŋmɛ̃ kuwóó nɛ ɑfɛɛtɑɑni nɛ ɑnɑ tulu. Úu yɛ píkɑi Uléécɑɑ kɛyɔ riyɑ́, ɑmɑ́ kɛtúŋɛ́ nɛ kɛsinɛ kuu yɛ Uléécɑɑ pikɛi n wɑi un ɑnɔ́ɔ pɑɑsi un welu. 
Uu kumúŋɛ́ kpɛ-i Uléécɑɑ kɛyɔ-i hɑpɔ uu Uléécɑɑ pipɑkɑrɛ loni. Uu pɛ̃ nnɛ́í pɛɛ pɛɛ m mɛ̃́ tɛ Uléécɑɑ ukɛ́ Yerusɑlɛm riyu lɔ kɛwɑ̃́ kɛ̃ nsímɛ́ símisi. 
Kɛ Yeesu pikɔ́ ɑpi lɛ̃ kɛ Upíimɑ isé ii m pisɛ rɛ pikɛ́ wɑ piwɑi ḿ mɑsí, ɑpí nɛ Kɑlilee u pɛlɛ, pikuyu Nɑsɑrɛti kɛ́mɛɛ. 
Kɛwɑ̃́ ɑkɛ́ m pɑpisi kɛn kɔ ńnɑŋɛ kpɑ́lɛ̃. Mɛwɛ́ɛ́sɛsɔhɔ píimɑ kuú pɛɛ mɑ́, Uléécɑɑ ipɛɛlɛ́ɛ in kɔ ukɛcɑ́ɑ́ we. 
Ḿpɑ́ kuŋmɛ̃ kúye-i, Yeesu usɑɑ nɛ uuni yɛ̀ɛ̀ pɛɛ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ hɑlɛ Yerusɑlɛm-pɔ. 
Kumúŋɛ́ kpɛ-i, Yeesu yɛ pɛɛ ɑŋmɛ̃ kɛfi nɛ ɑtɛ́ mɑ́lɛ. Usɑ́ɑ nɛ úni ɑpí nɛ u pɛ́nɛ ɑpi ɑnyɑ̃́ sĩ́ yɑrɛ kɑpi yɛ ń hɑ. 
Kɛ ɑnyɑ̃́ ɑɑ n tɛnɛ, usɑ́ɑ nɛ úni ɑpi pɛlɛ, ɑmɑ́ Yeesu uu Yerusɑlɛm-i mɛ́suni, upikɔ́ ɑ́pi kɔ nkpɑ́ni céri. 
Api músu rɛ Yeesu yɛ́ pipisencɔ pinyinɛ rintikilɛ, ɑpi sɔ́nɛ yɑrɛ kɛyɑ́ɑ kɛsɛ kumúŋɛ́, ɑpi pɛɛ lɛ̃ mɛmɑ́ɑ́ pipimɑrɛcɔ nɛ pipisɑnɛ kɛ́mɛɛ piwɛ́lɑɑ u sĩ. 
Amɑ́ ɑ́pi u yɛ̃, ɑpi Yerusɑlɛm-mɛ̃ pɛlɛ pin piwɛ́lɑɑ u nɔŋ́lɛ̃. 
Api kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ Uléécɑɑ kɛyɔ-i u yɛnu un isé picélɑɑ kɛcɔpɛ tũ un kutu pi cɔlɛ̃ un kɔ nsímɛ́ pi písɛi. 
Pírí kɛ Yeesu mɛsɔhɔ nɛ uɑnɔɔ mɛyɔ́ɔ́i ɑi pɛ̃ nnɛ́í pɛɛ kutu u n cɔ́lɛ̃ wɑ. 
Kɛ usɑ́ɑ nɛ úni ɑpi u n yɛ̃́, ɑi pírí pi wɑi, úni uu u mɑɑ rɛ: Kɛnɛ́wɑ̃́, yo nnyɑ kɑɑ lɛ̃ wɑ? Tɔ́ nɛ upɔ́sɑ́ɑ yɛ pɛɛ pɔ́ wɛ́ɛ́silɛ kɛrɔ́wuu kɛn túúni. 
Uu rinɔ́ɔ pi yɔ́su rɛ: Yo nnyɑ kɑni nɛ́ wɛ́ɛ́si? Áni nyu rɛ li pisɛ rɛ kɛ́ unɛ́sɑ́ɑ pikɛi wɑɑ? 
Amɑ́ ɑ́pi lɛ̃ kuu pi ḿ mɑ ɑsei kõ. 
Yeesu uu pi tiki ɑpi Nɑsɑrɛti pɛlɛ. U pɛɛ usɑ́ɑ nɛ úni pɑkɑrɛlɛnlɛ. Úni uu kɔ mɛwɑi mɛ̃ nnɛ́í ukɛfɑ-i n wɑilɛ̃. 
Yeesu uú m pelu, umɛsɔhɔ mɛn kpɑ́lɛ̃ un kɔ isoi sɔnɛ le, Uléécɑɑ nɛ sisoipipi inipɛɛ-i, Uléécɑɑ ipɛɛlɛ́ɛ in kɔ ukɛcɑ́ɑ́ we.
