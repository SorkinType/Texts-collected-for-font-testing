6.
Kɛwéntɛyɑɑ tuŋɛ́ kɛnyinɛ kɛ Yeesu uu kɛcɑrɛ kɛ́mɛɛ sɔ́nɛ un tɔ́su. Upipirɛtiki ɑpi mɛpipi n kpééni pin piɑnipɛ-i tɛkɛisɛ pin tɑkɑi. 
Pifɑrisi pinyinɛ ɑpi pi mɑɑ rɛ: Yo nnyɑ kɑni lɛ̃ kɛ isé ii ḿ mɑɑ rɛ pisoi ɑ́pi kɑpɛ yɛ kɛwéntɛyɑɑ kɛtúŋɛ́ wɑ wɑi? 
Kei kɛ Yeesu uu pɛɛ pi pɛsɛ rɛ: Lɛ̃ nnyɑ ɑ́ni lɛ̃ kɛ Tɑfiti uu kumúŋɛ́ kpɛ-i kɛ nkṹ nn u nɛ upikɔ́ n wɑ kɛɛnlɛ̃, nɛ́ɛ yo? 
U Uléécɑɑ kɛyɔ-i lonlɛ uu ɑkpɔ́nɔ́ kɑpi Uléécɑɑ n yekei kpísi uu le, uu kɔ upikɔ́ pɑ, lin kɔ nɛ irɔ́sé-i wɔ́lɑɑlɛ̃ tɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ŋmɑnɛ pɛɛ yɛ́ fe ɑpi ɑ le. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɛ̃́, Usoi Kɛpipi, nɛ́ɛ kɛwéntɛyɑɑ ute. 
Kɛwéntɛyɑɑ kɛtúŋɛ́ kɛcɔ kɛnyinɛ, Yeesu uu kuyómɛyɑ́hɑɑlee kunyinɛ-i loni un pisoi céési. Uu kei utisi unyinɛ lɛɛpɔ nɛ kunípɛ lukɛ-lukɛ kópɛ. 
Isé picélɑɑ nɛ Pifɑrisi ɑpi Yeesu kutu n cɔlɛ̃ pikɛ́ yɛ̃́ in tɛ u yɛ́ kɛwéntɛyɑɑ kɛtúŋɛ́ usoi pɔisɛ, likumúŋɛ́ rɛ pi pɛɛ kɛwɑ́ɑ́nɑ́ wɛ́ɛ́silɛ pikɛ́ nɛ ɑkópɛ u n lɑpɑɑsɛ. 
Amɑ́ Yeesu yɛ pɛɛ pinfɑsimɛ́ cérilɛ. Uu unípɛkópɛkɔɔ́ mɑɑ rɛ: A yisi ɑ nyɛrɛ ḿpɑ́ úye ukɛ́ pɔ́ yɛ̃́. Utisi uu yisi uu nyɛrɛ. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Nɛ nɔ́ pisɛ rɛ íye kɑi irɔ́sé kɛ́mɛɛ pisɛ? Li pisɛ rɛ tɔkɛ́ kɛwéntɛyɑɑ kɛtúŋɛ́ lisɔnɛ nɛ́ɛ likópɛ wɑ? Tɛ tɔkɛ́ usoi nfɑɑ lɔ nɛ́ɛ tɔkɛ́ u kpu? 
Uu pinnɛ́í nyɑ́nɛi uu pɛɛ limɛmɑ́ɑ́ usoi uyɛ̃ mɑɑ rɛ: A kupɔ́nípɛ rituŋɑ́mɛ. Uu kunípɛ tuŋɑ́pɔ, kunípɛ ɑku pélu. 
Amɑ́ ɑi mpí pɔ̃́ tóósi ɑpi pimɛcɔpɛcirɛ n kɛŋɛnɛ pin lɛ̃ kɑpi yɛ́ nɛ Yeesu n wɑ tɑikɛɛ. 
Yeesu uu likumúŋɛ́ kpɛ-i kɛkúúpi tɑɑ́ rɛ ukɛ́ kɛyómɛ yɑ́ɑ́si. Kɛsinɛ nnɛ́í kuu kei tonɛ un Uléécɑɑ yɑ́ɑ́si. 
Kɑi n weesi, uu upipirɛtiki séi, uu pikɛcɔpɛ kɛfi nɛ pitɛ́ wɛ́ɛ uu pi séi rɛ pitumɛ. 
Pɛpɛɛ Simɔɔ kuu rinyíri n hɛ rɛ Piyɛɛ nɛ uyɛ̃ uwɑ̃ Antiree nɛ Yɑkupu nɛ Yohɑni nɛ Filipu nɛ Pɑtelemii 
nɛ Mɑtiyee nɛ Tomɑɑ, nɛ Alifee kɛpipi Yɑkupu nɛ Simɔɔ yɛɛ ukuyu ń lɑ 
nɛ Yɑkupu kɛpipi Yutɑsi Isikɑriyɔɔti yɛɛ ń kɑ uu kɛpirɛ Yeesu yɑ́i. 
Yeesu nɛ upipirɛtiki ɑpi kɛkúúpi cɛpimɛ ɑpi kutẽ mmɛ́ weri, kei kɛ upipirɛtiki mɛyɑ̃ ɑpi ń we. Yutee kɛtẽ nnɛ́í pisoi mɛyɑ̃ pin kɔ kei we, nɛ Yerusɑlɛm pikɔ́ nɛ Tiiri nɛ Sitɔɔ mɛnimɑɑ ritimɛ ɑyu pikɔ́. 
Pi cɑ́pinɛmɛlɛ pikɛ́ Yeesu icélɑɑ kutu ricɔ, ɑpi kɔ piitoi pi pɔisɛ. Pɛ̃ kɛ ɑníri ɑɑ m pɛkɛsi ɑpi pélu. 
Ḿpɑ́ úye uu nɑ́ɑ́si ukɛ́ u ricɑ, liriyíkí rɛ u pɛɛ ńnɑŋɛ nnyinɛ mɑ́lɛ un nɛ pinnɛ́í pɔisɛntɛ. 
Yeesu uu upipirɛtiki inípɛɛ wii uu rɛ: Nɔ́ mpí nɔ́ɔ nní íwɛ m mɑ́, nɔ pínɑrɛkomɛ lɛ. Nɔ́ɔ Uléécɑɑ iyɔɔpi te. 
Nɔ́ mpí nɔ́ɔ nní nkṹ n tṹ, nɔ pínɑrɛkomɛ lɛ, nɔ kɛyɑ́ɑ lɛpunɛ. Nɔ́ mpí nɔ́ɔ nní n téni, nɔ kɛyɑ́ɑ sɛ́nnyinɛ. 
In tɛ pisoi yɛ nɔ́inɛ́puri yulu, ɑpi nɔ́ lɑ́kɑsɛ, pin lɑ́misi pin kɔ nɔ́nnɛ́kópɛ símisi rɛ nɔ Usoi Kɛpipi nɛ kɛfɑ tɛnɛ nnyɑ, nɔ pínɑrɛkomɛ lɛ. 
Tikpɑ́kɑ́rɑ́ tɛ̃ ticɔ kɛ pipisɑɑ ɑpí pɛɛ ɑntepuyɛ wɑ. In tɛ lɛlɛɛ nɔ́ lɛɛmɛ, nɔ́rinɛ́pɔ́ɔ rikɛ́ n lɑ́ɑ́rú, ɑni nɛ mpɔ́ɔnɑrɛ cɔ, nɔ́inɛ́hɛɛ yɛ kɛyómɛcɑɑ-pɔ piyɛlɛ ɑ́i kumúŋɛ́ mɑ́. 
Amɑ́ nɔ́ mpí nɔ́ɔ nní ḿ mɑ́, íwɛ kɑni topori, nɔ nɔ́nnɛ́nɑrɛ pilukɛ mɑsilɛ. 
Nɔ́ mpí nɔ́ɔ nɛni nní n lɛpɑɑlɛ̃, íwɛ kɑni topori, nkṹ yɛ sɔ́ntilɛ nkɛ́ nɔ́ wɑ. Nɔ́ mpí nɔ́ɔ nɛni nní n sɛ́nnyi, íwɛ kɑni topori, nɔ mpɔ́ɔcɑɑi-i loninɛ ɑni ténisi. 
In tɛ ḿpɑ́ úye yɛ nɔ́nnɛ́sɔnɛ símisi, íwɛ kɑni topori, likumúŋɛ́ rɛ lɛ̃ kɛ pipisɑɑ ɑpí pɛɛ ɑntepuyɛ nɔɔmɛ wɑi. 
Nɔ́ɔ nní kutu nɛ́ n cɔ́lɛ̃, nɛ nɔ́ símisilɛ rɛ ɑni nɔ́pinɛ́lɑ́ɑrɔ n lɑ, ɑni pɛpɛɛ nɔ́ n yúlu lisɔnɛ n wɑi, 
ɑni pɛpɛɛ ɑwééri nɔ́ n wɑi ɑnɔ́ɔ sɔnɛ ritikisɛ, ɑni pɛpɛɛ íwɛ nɔ́ n wɑi kɛcɑ́ɑ́ n welu. 
In tɛ usoi yɛ kumɔ́rí kusɛ-i kɛpɛsɛ pɔ́ tɔ, ɑ kutɔ́rɔɔ u rituŋɑ́, usoi un kupɔ́kpɛ́lɛnku pɔ́ n yɔsíé, ɑ tíyɛsɛ ukɛ́ kɔ kɛpɔ́túkɑnkɑ tene kpísi. 
Ḿpɑ́ úye yɛɛ linyinɛ pɔ́ we ɑ li u hɛ, úye un kɔ lipɔ́kɔ́ linyinɛ pɔ́ n yɔsí, kɑpɛ li u korí. 
Lɛ̃ kɑni ń lɑ picɔ pikɛ́ nɔ́ wɑ, ɑni yɛ limɛcɔ pi wɑ. 
In tɛ pɛpɛɛ nɔ́ ń lɑ ŋmɑnɛ kɑní lɑ, ihɛ́ɛ́ íye icɔ kɑni pɛɛ mɛ̃́. Pikópɛkɔɔ́ ricuruu yɛ pɛpɛɛ pi ń lɑ lɑlɛ. 
In tɛ pɛpɛɛ lisɔnɛ nɔ́ n wɑi ŋmɑnɛ kɑni yɛ lisɔnɛ wɑ, ihɛ́ɛ́ íye icɔ kɑni pɛɛ mɛ̃́? Pikópɛkɔɔ́ yɛ̀ɛ̀ kɔ lɛ̃ wɑlɛ. 
In tɛ nɔ tɑ́lɛ̃ tɛ pɛ̃ kɑni n lonsɛ yɛ́ nɔ́ hɛ́ɛlɛ, ihɛ́ɛ́ íye icɔ kɑni pɛɛ mɛ̃́? Pikópɛkɔɔ́ ricuruu yɛ̀ɛ̀ pikópɛkɔɔ́ lonsɛlɛ pin tɑ́lɛ̃ tɛ pi yɛ́ lɛ̃ kɑpi n lonsɛ ritísí linyinɛ pi hɛ́ɛ́lɛ. 
Amɑ́ nɔ́ɔkɛ́ nɔ́pinɛ́lɑ́ɑrɔ n lɑ, ɑni lisɔnɛ pi n wɑi. Áni kɑpɛ yɛ n tɑ́lɛ̃ tɛ pi yɛ́ nɔ́ hɛ́ɛ́lɛ ɑni kɛlenɛ pi lonsɛ. Nɔ́ ihɛ́ɛ́ kɛcirɛ yɛ̃ ɑni pɔ́lɔ́ Uléécɑɑ yɛɛ ḿpɑ́ yo kɛcɑ́ɑ́ n tṹ sipipi, likumúŋɛ́ rɛ u pɛpɛɛ ɑ́pi rícere m mɑ́ nɛ pisoi kpɑ́kɑ́rɑ́ lɑlɛ. 
Ani yɛ nɔ́mɛnɛ́cɔpɛcirɛ íwɛ ténɑɑnɛ yɑrɛ kɛ Nɔ́unɛ́sɑ́ɑ uu yɛ íwɛ nɔ́ n tẽ́ mɛcɔ. 
Áni kɑpɛ yɛ picɔ ɑkópɛ m pɑlɛ̃, Uléécɑɑ úu kɑpɛ kɔ nɛ nɔ̃́ ɑkópɛ m pɑ́lɛ̃ nnyɑ. Áni kɑpɛ yɛ picɔ nɛ túhɑɑnɛ, Uléécɑɑ úu kɑpɛ nɛ nɔ̃́ n túhɑɑnɛ nnyɑ. Ani yɛ picɔ ɑkópɛ pi sɑ́rɛi, Uléécɑɑ uu nɔ́ɑnɛ́kɔ́ nɔ́ sɑ́rɛi. 
Ani yɛ picɔ hɛ, Uléécɑɑ uu nɔ́ hɛ. Pi nɔ́ músunɛ ɑi yipu ɑí wurɛ ɑpi tíŋɑ-tiŋɑ ɑpi nɔ́ kópɔ. Kɛmúŋɛ́ kɛ̃ kɑni yɛ nɛ picɔ ḿ musí kɑpi kɔ nɛ nɔ̃́ músunɛ. Ái líkɑ nnyɑ, Uléécɑɑ yɛ músunɛ ɑi yipu uu tíŋɑ-tiŋɑ uu tírii uu nɔ́ kópɔ. Kulũ kpɛ̃ kɑni yɛ nɛ picɔ ḿ musí kucɔ kɛ Uléécɑɑ uu kɔ nɛ nɔ̃́ músunɛ. 
Yeesu uu kɔ kɛnyɑ́rũ kpɑ́ rɛ: Unyíyɛ úu yɛ́ fe ukɛ́ unyíyɛcɔ tɔrɔi. Lin lɛ̃ n wɑ, pinnɛ́í yɛ hɑ kuhórɛ-i lóipɔnɛ. 
Uyɛɛ pikɛi m peikɛɛ úu yɛ uyɛɛ pi u m peikɛɛsɛ n fe. Amɑ́ uyɛɛ pi m peikɛɛ un pikɛi n ceri ńsɔnɛ, u nɛ uyɛɛ pi u m peikɛɛsɛ yɛ́ n sɑ́. 
Yo nnyɑ kɑɑ kɛyúipi kɛɛ upɔ́mɑ́rɛcɔ mɛnipɛɛ-i ń we nyɑ́ni ɑ́ɑ pɛɛ kɛnɑwɛlɛ́pi kɛɛ mɛpɔ́kɔ́-i ń we nyɑ́ni? 
Íye kɑɑ yɛ́ fe ɑɑ upɔ́mɑ́rɛcɔ mɑɑ rɛ: Unɛ́mɑ́rɛcɔ, ɑ riyɑ́ kɛ́ kɛyúipi kɛɛ mɛpɔ́nípɛɛ-i ń we lesɛ, pɑ́ɑ pɛɛ kɛnɑwɛlɛ́pi kɛɛ mɛpɔ́nípɛɛ-i ń we lesɛ kɛlenɛ. A tíyɛ pɔ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi mɑ́ kpɑ! Kɛnɑwɛlɛ́pi kɛɛ mɛpɔ́nípɛɛ-i ń we kɑɑ yɛ́ mɛfoí lesɛ ɑɑ pɛɛ ńsɔnɛ n nyɑ́ni ɑɑ́ nɛ kɛyúipi kɛɛ ucɔ mɛnipɛɛ-i ń we lesɛ. 
Kuléé kpɛɛ isɑrɛ n lɑ́ɑ́rú ɑ́ku yɛ ɑpipi kópɛ mɑri, kuléé toikɔ́ ɑ́ku yɛ kɔ ɑpipi sɔnɛ mɑri. 
Pi yɛ ḿpɑ́ kuléé kúye nɛ kuɑpipi cerilɛ. Ápi yɛ sinípɛleépi kɛcɑ́ɑ́ ɑfiiki tɔ, ɑ́pi yɛ kɔ ɑhííhíí kɛcɑ́ɑ́ ɑrɛɛsɛɛ tɔ. 
Lisɔnɛ kɛ usoi sɔnɛ uu yɛ ukɛfɑ-i lésɛmɛ, usoi kópɛ pɔ́ɔ likópɛ ukɛfɑ-i lésɛri, likumúŋɛ́ rɛ lɛlɛɛ usoi kɛfɑ-i n yipɑɑlɛ̃ kɛ nnɔ́ɔ nn yɛ símisi. 
Yo nnyɑ kɑni nɛ́ sélei rɛ Upíimɑ, Upíimɑ ɑ́ni pɛɛ lɛ̃ kɑm nɔ́ n céesi wɑi? 
Ani tíyɛ kɛ́ nɔ́ nyísɛ lɛ̃ kɛ uyɛɛ nnɛ́símɛ́ kutu n cɔ́lɛ̃ un nɛ n kɛisi uú nɛ mɛnyínɛ ń we. 
Liute yɛ we yɑrɛ usoi yɛɛ kɛ́yɔ m mɔ́m uu túni ɑi cimisi uu kɛpɑrɛ tulu, uu kɛkɛcɑ́ɑ́ rikúi tɔsi. Kɛ ikói ii n yi ii wɑɑ, míni ɑmɛ kɛ fɛ, ɑmɑ́ ɑ́mɛ kɔ fe mɛkɛ́ kɛ ritiŋɑ-tiŋɑ, likumúŋɛ́ rɛ kɛ mɔmɑɑlɛnlɛ ńsɔnɛ. 
Amɑ́ uyɛɛ nnɛ́símɛ́ kutu n cɔ́lɛ̃ úu pɛɛ́ nɛ n kɛisi yɛ we yɑrɛ usoi yɛɛ kɛ́yɔ m mɔ́m uu ntɑɑi purɛ-purɛ kɛcɑ́ɑ́ kɛ tɔsi, rikúi ɑ́ri we. Kɛ ikói mɛni ɑmɛ kɛ rińfɛ, mɛsɛ nɛ mɛsɛ kɑkɛ tɔhɔ̃ ɑkɛ foru tɛ́kɛ́-tɛ́kɛ́.
