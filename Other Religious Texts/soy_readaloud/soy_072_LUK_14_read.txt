Luki.
14.
Kɛwéntɛyɑɑ kɛtúŋɛ́ kɛnyinɛ, Yeesu uu Pifɑrisi uwɛ́ɛ́sɛ unyinɛ kɛyɔ sĩ tɛ ukɛ́ ilukɛ li. Pɛpɛɛ kɛyɑ́ɑ kɛ̃ kei ń we ɑpi ńsɔnɛ u m pɑílɛ̃. 
Utisi unyinɛ yɛɛ mɛpílɛ́ itoi m pɔ́ɔ́lú un ukɛyu-i nyɛnu. 
Yeesu uu nsímɛ́ kpísi uu isé pinyuwɛ nɛ Pifɑrisi pisɛ rɛ: Irɔ́sé yɛ ncée hɛ nɛ́ɛ íi hɛ rɛ pikɛ́ usoi kɛwéntɛyɑɑ kɛtúŋɛ́ pɔisɛ? 
Amɑ́ ɑ́pi ŋmurɛi pikɛ́ rinɔ́ɔ yɔsí. Yeesu uu pɛɛ utóikɔ́ kpísi uu u pɔisɛ uu u yɑ́pɔ. 
Uu pɛɛ pi mɑɑ rɛ: In tɛ nɔ́kɛnɛ́cɔpɛ unyinɛ kɛwɑ̃ nɛ́ɛ uinɑɑ yɛɛ kɛlúkɑnkɑ-i lóipɔ, liute úu yɛ́ mɛsɛ nɛ mɛsɛ kɛ pɔɔnɛmɛ ḿpɑ́ lin kɛwéntɛyɑɑ kɛtúŋɛ́ɛ? 
Api rinɔ́ɔ piyɔ́ɔ́ u pɔɔnɛ. 
Yeesu uu yɛ́nu lɛ̃ kɛ pisɑ́nɛ kɑpi n sée ɑpi sitónɛ n wɛ́ɛkɛɛ. Uu pɛɛ pinnɛ́í kɛnyɑ́rũ nkɛ́ mɑɑ rɛ: 
Usoi un pisɔ́ɔnnɛ pilukɛ pɔ́ n sée, kɑpɛ hɑ kɛtónɛ kɛcirɛ kɛ́mɛɛ tónɛ. In n lɑ, pi yɛ́ úyukɔɔ́ unyinɛ yɛɛ pɔ́ n fe n sée. 
Ai wɑi rɛ uyɛɛ nní nɔ́kɛnɛ́tɛ́ n sée uú weri uu rɛ: A kɛtónɛ kɛ̃ nɛ uyɛ̃ tiyɑ́. Isɛi kɑɑ yɛ́ hɑ pɛɛ́ nɛ kɛtónɛ tɔrɔɔ-i tonɛ. 
Amɑ́ li nyɑmlɛ rɛ usoi un kusɑ́nɛ pɔ́ n sée, ɑ hɑ kɛtónɛ tɔrɔɔ-i tonɛ, likɛ́ wɑ rɛ kumúŋɛ́ kpɛ-i kɛ uyɛɛ nní pɔ́ n sée uu ń kɑ, ukɛ́ pɔ́ mɑɑ rɛ: Unɛ́sɑ́nɛ, ɑ kɛ́yu-mɛ̃ kɑm kɛtónɛ kɛcirɛ kɛ́mɛɛ. Lin lɛ̃ n wɑ, pɔ́ pɛɛ pipɔ́sɑ́nɛcɔ kɛyu-i ríyu yɛ̃. 
Lɛ̃ nnyɑ úye un umɛcirɛ rintɑɑ́sɛ, li yɛ u ricɛpisɛlɛ. Úye un umɛcirɛ rincɛpisɛ, li yɛ u ritɑɑ́sɛlɛ. 
Yeesu uu pɛɛ uyɛɛ kusɑ́nɛ u n sée mɑɑ rɛ: Pɔn pisoi pilukɛ n séinɛ, kɑpɛ yɛ pipɔ́sɑ́nɛ nɛ kupɔ́mɑ́rɛ pikɔ́ nɛ pipɔ́pɛ́rɛcɔ pɛɛ m mɑ́ sée. Pɛ̃ nɛ́ fe ɑpi kɔ pɔ̃́ kusɑ́nɛ séi. Lin lɛ̃ n wɑ, pi lɛ̃ kɑɑ pi ḿ pɑ pɔ́ pɛsɛlɛ. 
Amɑ́ pɔn ɑnyɑ̃́ ilukɛ n wɑi, ɑ yɛ píwɛkɔɔ́ nɛ pinɑ́rɛi nɛ síkɑnkɑ́lɑ́kɑ́ nɛ pinyíyɛ sélei. 
Pɔ́ pɛɛ mpɔ́ɔnɑrɛ yɛ̃, likumúŋɛ́ rɛ pɛ̃ ɑ́pi fenɛ pikɛ́ pɔ́ li pɛsɛ. Uléécɑɑ yɛɛ yɛ́ pɔ́ li pɛsɛ kumúŋɛ́ kpɛ-i kɛ pɛ̃ nnɛ́í pɛɛ umɛlɑ n wɑpisi ɑpi nkpɔ kɛ́mɛɛ n yisinɛ. 
Anɔ́ɔ nyɛ̃ mɛkomɛ-mɑɑ́, pɛ̃ nɛ pɛ̃ pɛɛ pɛɛ kɛsẽ́ ilukɛ n le usɛ uu Yeesu mɑɑ rɛ: Uyɛɛ yɛ́ hɑ Uléécɑɑ iyɔɔpi-i ilukɛ n li yɛ únɑrɛkomɛ lɛ. 
Yeesu uu pɛɛ kɛnyɑ́rũ nkɛ́ u wɑi rɛ: Utisi unyinɛ yɛɛ kɛyɑ́ɑ ilukɛ kɛcirɛ sɔ́ɔnkɛɛ uu pisoi kulúi séi. 
Kɑi pilukɛ kumúŋɛ́ n tu, uu uukɛikɔ́ tum tɛ ukɛ́ hɑ pɛ̃ kuu n sélei mɑɑ rɛ: Ani kɑm, ḿpɑ́ yo yɛ nkpéni pipele mɑsilɛ. 
Amɑ́ pinnɛ́í mmɛ̃, ḿpɑ́ úye uu uncɔ́ŋ pinyísɛ koru. Ufoí uu ukɛikɔ́ mɑɑ rɛ: Nɛ kɛcɑrɛ lɔlɛ, li kɔ pɛɛ pisɛ rɛ kɛ́ hɑ kɛ yɛ̃. Lɛ̃ nnyɑ, ɑ u mɑɑ rɛ úu kɑpɛ wɔ́ɔsi. 
Ucɔ uu u mɑɑ rɛ: Nɛ inɑ́ɑ lɑmɛ-lɑmɛ kɛfi lɔlɛ, nɛ lɑ kɛ́ nɛ pikɛi i peenu. Lɛ̃ nnyɑ, ɑ u mɑɑ rɛ úu kɑpɛ wɔ́ɔsi. 
Ucɔ uu kɔ u mɑɑ rɛ: Nfɑ́ɑni kɑm unɔ́si kpísi, lɛ̃ nnyɑ kɑ́m fenɛ kɛ́ hɑ. 
Ukɛikɔ́ uu uuyɔsɑɑ kɛ́mɛɛ pɛlɛ uú hɑ ɑnɔ́ɔ nyɛ̃ u símisi. Ai úyɔsɑɑ kuwɔ́i wɑi uu uukɛikɔ́ mɑɑ rɛ: A mɛkɛɛ ɑyɑ́ɑ-i nɛ icée kɛ́mɛɛ hɑ, ɑɑ píwɛkɔɔ́ nɛ pinɑ́rɛi nɛ pinyíyɛ nɛ síkɑnkɑ́lɑ́kɑ́ séleiri. 
Kɑi rinkpɑ́sɛ, ukɛikɔ́ uu pɛɛri uu úyɔsɑɑ mɑɑ rɛ: Nɛ lɛ̃ kɑɑ m pisɛ piwɑi mɑsilɛ, ɑmɑ́ sitónɛ yɛ kɔ pɔɔnɛlɛ. 
Úyɔsɑɑ uu pɛɛ ukɛikɔ́ mɑɑ rɛ: A sicɑrɛyupi siceepi ritiki ɑɑ pɛ̃ kɑɑ ncée-i n yɛ́ntinɛ nɛ kɛ́yu mɑhɑni pikɛ́ kɑm, kɛnɛ́yɔ kɛkɛ́ nɛ n yi nnyɑ. 
Nɛ nɔ́ símisilɛ rɛ pɛ̃ nnɛ́í kɑḿ pɛɛ nní n sélei, úkɑ úu inɛ́lukɛ lenɛ. 
Asoiwuí píimɑ ɑn Yeesu tikilɛ̃ pin nsẽ́ pɛ́nɛlɛ̃, Yeesu uu pɑnsɛpɔ uu pi mɑɑ rɛ: 
Úye un kɛnɛ́mɛɛ n kɑ, li pisɛ rɛ ukɛ́ nɛ́ n lɑ ɑi tɔ́su uusɑɑ nɛ uuni nɛ uunɔsi nɛ usiwɑ̃ nɛ upiwɑ̃ nɛ upimɑɑ nɛ upiyɛi nɛ uricuruu isoi. Insɑ́ lɛ̃ úu fenɛ ukɛ́ unɛ́pirɛtiki. 
Nkó yɛɛ úu n ŋmurɛi ukɛ́ mɛnɛ́cɔ ukunɑpɛ́ɛ́kɔɔ́ cɔ̃ ukɛ́ nɛ nɛ́ ritiki, úu nɛ sɑ́ rɛ ukɛ́ unɛ́pirɛtiki. 
In tɛ nɔ́kɛnɛ́cɔpɛ unyinɛ yɛ lɑ ukɛ́ kulee cɑ́cɑ́ mɔ̃́, liute yɛ̀ɛ̀ tonɛlɛ uu músu, uu yɛ́nu rɛ ɑlɛ́ kɛ pikɛi pɛ̃ ɑpi wɑ, ɑi tíyɛsɛ uu céru, in tɛ usiwóó yɛ piyɛ un yɛ́ fe uu pikɛi pɛ̃ kɛtɔ tulu. 
Insɑ́ lɛ̃, un tikúi n yekei kúu pɛɛ kulee cɑ́cɑ́ pimɔ́mɛ tɛnɛsɛ, pɛ̃ nnɛ́í pɛɛ yɛ́ lɛ̃ n yɛ̃́ yɛ́ u sɛ́nnyi. 
Pi yɛ́ mɑɑ rɛ: Usoi nkó yɛ pimɔ́mɛ korɑɑnɛ ɑmɑ́ úu fe ukɛ́ tɛnɛsɛ. 
Lɛ̃ cirɛ yɛ ni, uyɔ́ɔpi úye un n lɑ ukɛ́ hɑ nɛ ucɔ tɔ, u yɛ tonɛlɛ uu músu kɛlenɛ, in tɛ u yɛ́ nɛ upisoi ɑ́kotokú kɛfi (10.000) fe uú hɑ nɛ uulɑɑrɔ yɛɛ pisoi ɑ́kotokú ɑfɛɛrɛ́ (20.000) m mɑ́ tɔpu. 
Unsɑ́ n fenɛ, u yɛ uuyɔɔpicɔ-i pitumɛ tunlɛ úu kɑhɑnɛ u n nyɑhɑimɛ, uu u pisɛ rɛ pikɛ́ kómɛinɛ pikɛ́ ɑtɔpi riyɑ́. 
Yeesu uu pimɑ́ɑ kpɑ́ rɛ: Kɑi nní lɛ̃, nɔ́ úkɑ úu fenɛ ukɛ́ unɛ́pirɛtiki, liute unsɑ́ lɛ̃ kuu m mɑ́ nnɛ́í n ŋmɛriyɛ. 
Nnyɑ́si yɛ likɛi kɛcirɛ lɛ. Amɑ́ mmɛlõ mɛn n kpu, yo kɑpi yɛ nɛ n lɑrukusɛ? 
Ńn pɛɛ kɛtẽ nɛ́ɛ mɛfɛ́nɛ kɛ́mɛɛ linyinɛ yoriyɛnɛ. Pi yɛ kɛtɑhɑi n fómpɔlɛ. Úye un ukómɛ, uu kóm.
