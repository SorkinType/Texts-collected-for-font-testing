Luki.
13.
Pisoi pinyinɛ ɑpi kumúŋɛ́ kpɛ-i Yeesu lɛɛpɔ, ɑpi u símisi lɛ̃ kɛ uyɔ́ɔpi Pilɑti uu rinɔ́ɔ n hɛ rɛ pikɛ́ Kɑlilee pikɔ́ pɛɛ ń we pin Uléécɑɑ inyɔ́ɔnsɛ wɑi kṍ. 
Yeesu uu pi mɑɑ rɛ: Nɔ músu rɛ kɛ pisoi pɛ̃ ɑpi pikópɛkɔɔ́ píimɑ ɑpi Kɑlilee pikɔ́ tɔrɔɔ tɔ́su nnyɑ kɑpi lɛ̃ pi kṍo? 
Nɛ nɔ́ mɑɑ rɛ “ɑɑí”, ɑmɑ́ nɔnsɑ́ mɛfinɛ n consɛ, nɔ nɔ́nnɛ́nɛ́í kpíninɛ pimɛcɔ. 
Pisoi pitɛ́ ɑ́pi we ɑfɛɛrɛ́ pɛ̃ kɛcɑ́ɑ́ kɛ Silowee kulee cɑ́cɑ́ ɑku ń loó ɑku kóni ní? Nɔ músu rɛ piɑkópɛ nyɛɛ Yerusɑlɛm kuyu pikɔ́ tɔrɔɔ ɑkɔ́ tɔsii? 
Nɛ nɔ́ mɑɑ rɛ “ɑɑí”, ɑmɑ́ nɔnsɑ́ mɛfinɛ n consɛ, nɔ nɔ́nnɛ́nɛ́í kpíninɛ pimɛcɔ. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ kɛnyɑ́rũ nkɛ́ pi mɑɑ rɛ: Utisi unyinɛ yɛɛ pɛɛ kúfikiyee kunyinɛ tɑmɑɑlɛ̃ uɑfikiyee kɛcɑrɛ kɛ́mɛɛ. Uu kɛyɑ́ɑ sĩ ukɛ́ kukɛcɑ́ɑ́ ɑpipi túúnu, ɑmɑ́ úu líkɑ yɛ̃. 
Uu pɛɛ uléétɑmɑɑ mɑɑ rɛ: A ripɑí, ɑŋmɛ̃ ɑtɑɑni yɛ nnyɛ́ kɑm sɔ́nɑɑri rɛ kɛ́ yɛ kúfikiyee nkú kɛcɑ́ɑ́ ɑpipi túúnu ɑ́m yɛ kɔ líkɑ yɛ̃. A ku riké. Yo nnyɑ kɑku yɛ́ mɛ́woo kɛtẽ nɛ́ n cɑɑi? 
Uléétɑmɑɑ uu u mɑɑ rɛ: Sɑ́ɑ, ɑ nɔŋɔni kuŋmɛ̃ nkú pikɑŋnɛ ku rikpɑ́, nɛ́ tṹ ɑm ku kɔ́ɔ́núsɛ, ɑm pɛɛ ntɑɑi mɛɛ mɛfɛ́nɛ m mɑ́ súúni. 
Lin lɛ̃ n wɑ, in n lɑ ku yɛ́ kɛlɑɑnɛ mɑri, insɑ́ lɛ̃ n wɑ, ɑɑ ku ké. 
Kɛwéntɛyɑɑ kɛtúŋɛ́ kɛnyinɛ kɛ Yeesu uu Pisuifi kuyomɛyɑ́hɑɑlee kunyinɛ-i céési. 
Unɔ́si toikɔ́ unyinɛ un kei we. Li wɑlɛ ɑŋmɛ̃ ɑtɛ́ ɑ́ɑ we ɑfɛɛrɛ́ kɛ kuníri kunyinɛ ɑkú nɛ u n lõ kun u kpɑrɑ́sɛlɛ̃ un kɛcírɛ pinyɛrɛ pɔɔnɛlɛ̃. 
Kɛ Yeesu uu unɔ́si uyɛ̃ n yɛ̃́, uu u séi uu u mɑɑ rɛ: Uléécɑɑ yɛ pɔ́ pɔisɛlɛ. 
Uu ɑnípɛ ukɛcɑ́ɑ́ lɑ́ɑ, unɔ́si uu mɛsɛ nɛ mɛsɛ kɛcírɛ nyɛrɛ uu Uléécɑɑ pítɑɑ́sɛ kɑ́pɑ́ɑ́. 
Amɑ́ ɑ́i nɛ kuyómɛyɑ́hɑɑlee uwɛ́ɛ́sɛ risɑ́, kɑi n wɑ rɛ Yeesu yɛ kɛwéntɛyɑɑ kɛtúŋɛ́ usoi pɔisɛ nnyɑ. Uu pɛɛ rinɔ́ɔ yɔ́su, uu risoiwuí mɑɑ rɛ: Siyɑ́ɑ sikpuulũ kɑri mɑ́, sɛ-i kɑi pisɛ rɛ tɔkɛ́ kɛisi. Lɛ̃ nnyɑ, ɑni yɛ siyɑ́ɑ sɛ-i kɑm pikɛ́ nɔ́ pɔisɛ, ɑ́i kɑpɛ pɛɛ kɛwéntɛyɑɑ kɛtúŋɛ́. 
Upíimɑ uu pɛɛ rinɔ́ɔ u yɔ́su rɛ: Nɔ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ lɛ. Ḿpɑ́ nɔ́ úye yɛ̀ɛ̀ kɛwéntɛyɑɑ kɛtúŋɛ́ kusɛɛlee-i uinɑɑ nɛ́ɛ ukɛminɑɑ toriyɛlɛ uú nɛ kɛ́ninirɛ́ sĩ́, nɛ́ɛ ɑ́i lɛ̃? 
Unɔ́si nkó yɛɛ nní Apirɑhɑm kɛpirɛ ukɔ́ kɛ Setɑni uu n tórɑɑlɛ̃ hɑ́i ɑŋmɛ̃ ɑtɛ́ ɑ́ɑ we ɑfɛɛrɛ́-mɛ, ɑ́i pisɛ rɛ pikɛ́ kɛwéntɛyɑɑ kɛtúŋɛ́ u toriyɛɛ? 
Yeesu rinɔɔ mɛyɔ́ɔ́ mɛ̃ ɑmɛ upilɑɑrɔ isɛi wɑi hɑ́i, ɑmɑ́ risoiwuí nnɛ́í ripɔɔ ɑrí nɛ umɛwɑi píimɑ kuu n wɑpisi nnɛ́í lɑrisi. 
Yeesu uú mɑɑ rɛ: Yo kɛ Uléécɑɑ iyɔɔpi ií nɛ mɛnyínɛ we? Yo kɑm yɛ́ fe kɛ́ nɛ i múŋɛisɛnɛ? 
I we yɑrɛ mpuri yukɛ-yukɛ nnyinɛ kɛ usoi uu ukɛcɑrɛ-i n lukɛsi nn lɛlu, nn kuléé wɑi, sinúipi ɑsi kuilesɛ kɛcɑ́ɑ́ ɑyɑ́mɛ́ wɑpisi. 
Yeesu uu pimɑ́ɑ kpɑ́ rɛ: Yo kɑm yɛ́ fe kɛ́ nɛ Uléécɑɑ iyɔɔpi múŋɛisɛnɛ? 
Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ nkpɔ́nɔ́tɔ pitɛ kɛ unɔ́si uu ntɔ píimɑ kɛ́mɛɛ n cɔ́hɔnɛpɔ uu pulu pikpɛ́ɛ ɑpi yisi. 
Yeesu uu ɑyu mɑɑ nɛ ɑsíńsɑ́ n tɛŋni un céési un nɛ Yerusɑlɛm-mɛ̃ sĩ. 
Unyinɛ uu u pisɛ rɛ: Sɑ́ɑ, mɛsei yɛ mɛ rɛ pisoi nkɑ́ripi kɛ Uléécɑɑ uu yɛ́ yóriyɛɛ? Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: 
Ani riŋmɔ́ɔ́pú ɑní nɛ rinɔnɔɔ tɛɛ m mɔ́kú-mɔ́kúlɛ̃ loni, ɑsei kɑm nɔ́ símisi. Mɛyɑ̃́ yɛ́ nɑ́ɑ́si rɛ mɛkɛ́ lompɔ, ɑmɑ́ ɑ́mɛ kɔ fenɛ. 
Úyɔte un píyei n yisi uu rinɔnɔɔ kɑ́, nɔ́ kɛtɑhɑi nyɛrɛ, ɑni rinɔnɔɔ n komnɛntɛ nɔn mɑ́ikɛɛ rɛ: Sɑ́ɑ, ɑ rɔ́ hɑ́nnɛ! Uu rinɔ́ɔ nɔ́ yɔ́su rɛ: Ám nyu rɛ nɔ yei pikɔ́ lɛ. 
Ani pɛɛ u mɑɑ rɛ: Kɛsẽ́ kɑri li ɑri níru, ɑrɔ́yu icee kɛ́mɛɛ kɑɑ tóói pɔn céési. 
Uu pimɑ́ɑ nɔ́ kpɑ́ rɛ: Ám nyu rɛ nɔ yei pikɔ́ lɛ. Ani nɛ kɛtɑɑ nɛ́ wɑ, nɔ́ pɛ̃ nnɛ́í nɔ́ɔ nní likópɛ n wɑi. 
Kei kɑni yɛ́ ténesi, ɑni ɑnĩ́ tɑkɑi kumúŋɛ́ kpɛ-i kɑni yɛ́ Apirɑhɑm nɛ Isɑɑki nɛ Yɑkupu nɛ ɑntepuyɛ nnɛ́í n yɛ̃́ Uléécɑɑ iyɔɔpi-i, ɑi pɛɛ n we rɛ kɛtɑhɑi kɑpi nɔ́ fómpɔnɛ. 
Pisoi yɛ́ kɛtuŋɛleemɛ-mɛ̃ nɛ kɛtuŋɛlolɛ́-mɛ̃ nɛ ituŋɛ kumii-mɛ̃ nɛ ituŋɛ kulukɛ-lukɛ-mɛ̃ leemɛ ɑpi weri ɑpi Uléécɑɑ iyɔɔpi kɛ́mɛɛ tonɛ ɑpi ilukɛ le. 
Ai pɛɛ wɑi rɛ pɛpɛɛ nɛni nní pitɔ́rɔɔ pinyinɛ yɛ́ hɑ pifoí, pɛpɛɛ nɛni nní pifoí ɑpí hɑ pitɔ́rɔɔ. 
Kumúŋɛ́ kpɛ-i, Pifɑrisi pinyinɛ ɑpí nɛ Yeesu kɔ ɑpi u mɑɑ rɛ: A nté yisi ɑɑ kɛlõ féé sĩ. Erooti yɛ lɑlɛ ukɛ́ tíyɛsɛ pikɛ́ pɔ́ kpu. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Ani n ŋme ɑni hɑ kɛsɑ́ŋɑ́ mpuri mmɛ̃ mɑɑ rɛ nɛ nɛni nɛ kóso ɑníri lɑkɑsɛntɛlɛ nɛn pisoi pɔisɛntɛ kɛ́ pɛɛ kɛyɑ́ɑ tɑ́ɑ́nũ tuŋɛ́ pikɛi mpí tɛnɛsɛ. 
Li pisɛ rɛ kɛ́ nɛni kɛyɑɑ nɛ kóso nɛ likɛweeki nnɛ́cée rinɔŋ́, likumúŋɛ́ rɛ ɑ́i nɛ sɑ́ rɛ ɑntepu ukɛ́ kɛlõ féé kpu kɑ́i Yerusɑlɛm-i. 
Nɔ̃́ Yerusɑlɛm pikɔ́, nɔ ɑntepuyɛ kónlɛ ɑni kɔ pɛ̃ kɛ Uléécɑɑ uu nɔ́kɛnɛ́mɛɛ n tummɛ ɑpɑrɛ tɑpisi ɑni kóni. Mɛlɛ́ kɛsẽ kɑḿ pɛɛ́ lɑ kɛ́ nɔ́ cɑ́pinɛ kɛ́ rimúlú líkɑ ɑ́i kɑpɛ nɔ́ wɑ, nɔkɛ́ n we yɑrɛ kɛ kucɑ̃́yɑhɑ ɑku yɛ kuɑmɛ́ŋ́ mɛtene kusipipi n tɑ̃ líkɑ ɑ́i yɛ́ si wɑ mɛcɔ, ɑni yúlu. 
Lɛ̃ nnyɑ kɑi yɛ́ nɔ́kɛnɛ́yɔ riyɑ́ kɛkɛ́ mɛŋmɑnɛ n nyɛnu. Ani kõ tɛ ɑ́ni nɛni piyɛ́nɛ nɛ́ kpɑ́nɛ likɛ́ hɑ nɛ tu kumúŋɛ́ kpɛ-i kɑni yɛ́ hɑ ḿ mɑɑ rɛ: Uléécɑɑ ukɛ́ uyɛɛ Upíimɑ rinyiri nɛ n wemɛ rikpɑ́!
