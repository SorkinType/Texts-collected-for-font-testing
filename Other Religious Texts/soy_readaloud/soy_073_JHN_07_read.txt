Yohɑni.
7.
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu Kɑlilee kɛtẽ kɔ́ɔ́nú. Úu pɛɛ́ hɑ ukɛ́ Yutee kɛkɔ́ rikɔ́ɔ́nú rɛ Pisuifi piwɛ́ɛ́sɛ yɛ wɛ́ɛ́si pikɛ́ u kpu nnyɑ. 
Pisuifi ɑnyɑ̃́ nyɛ̃ kɛ́mɛɛ kɑpi yɛ sicɑ́ŋíípi kɛ́mɛɛ n sói yɛ kɔ pɛɛ nyɑhɑimɛlɛ. 
Upimɑrɛcɔ ɑpi pɛɛ u mɑɑ rɛ: A nté yisi ɑɑ Yutee kɛtẽ-pɔ sĩ, pipɔ́pirɛtiki pɔ́ɔkɛ́ kɔ pikɛi kɑɑ n wɑi yɛ̃! 
Úkɑ úu yɛ umɛwɑi m pékesi un kɔ pɛɛ́ lɑ kuyu kukɛ́ u ceri. Lɛ̃ nnyɑ, ɑ tíyɛ kɛtẽ nnɛ́í pisoi pikɛ́ mɛpɔ́wɑi mmɛ́ yɛ̃́. 
Asei kɛcɑ́ɑ́, upimɑrɛcɔ ricuruu pɑ́ɑ pɛɛ kɔ nɛ kɛfɑ u tɛnɛlɛ̃. 
Kei kɛ Yeesu uu pɛɛ pi mɑɑ rɛ: Inɛ́tuŋɛ íi kɑhɑnɛ n tu. Amɑ́ in nɔ́ɔ, ḿpɑ́ ituŋɛ íye yɛ nɛ nɔ́ sɑ́lɛ. 
Kɛtẽ nté pisoi ɑ́pi nɔ́inɛ́puri yulu, ɑmɑ́ pi nɛ̃́ ipuri yululɛ rɛ nɛ we nɛn pi símisi rɛ pimɛwɑi ɑ́mɛ nyɑḿ nnyɑ. 
Nɔ́ɔkɛ́ n ŋme ɑni hɑ ɑnyɑ̃́ li. Nɛ̃́ ɑ́m kei simpɔ kɛlenɛ. Inɛ́tuŋɛ íi kɑhɑnɛ n tu. 
Kuu lɛ̃ pisímɛ́ pi ḿ mɑsí, uu Kɑlilee kɛteni-i mɛ́suni. 
Kɛ Yeesu pimɑrɛcɔ ɑpi ɑnyɑ̃́ nyɛ̃ ń hɑ, uricuruu pɔ́ɔ kɔ yisi uu sĩ́, úu tíyɛ pikɛ́ u n nyɑ́ni, uu pɛɛ n we yɑrɛ u pékesilɛ. 
Pisuifi piwɛ́ɛ́sɛ yɛ pɛɛ ɑnyɑ̃́ nyɛ-i u wɛ́lɑɑnkɛɛlɛ ɑpi yɛ pɛɛ rɛ: Yei kuú we? 
Pisoi pɛɛ n cɑ́pinɛlɛ̃ yɛ pɛɛ unsímɛ́ ŋmúlɑɑnkɛɛnɛlɛ. Pinyinɛ ɑpi yɛ rɛ: Usoi sɔnɛ lo. Picɔ ɑpi rɛ: Eehei, pisoi kuu kírɑɑsɛntɛ. 
Amɑ́ úkɑ úu pɛɛ unsímɛ́ símisi likɛ́ n cɑ́lɛ̃. Pisuifi piwɛ́ɛ́sɛ kɑpí pɛɛ lɛ̃ wuru. 
Anyɑ̃́ siyɑɑ sin kɛcɔpɛ pitulɛ mɑsí kɛ Yeesu uu Uléécɑɑ kɛyɔ hɑ uú hɑ n céési. 
Ai Pisuifi pírí wɑi ɑpi yɛ pɛɛ rɛ: Usoi nkó úu icélɑɑ íkɑ yɔsí, íye kɑi pɛɛ wɑ kuú nɛ nní ńsɔnɛ Nléécɑɑsimɛ́ nyu? 
Yeesu uu pi pɛsɛ rɛ: Icélɑɑ nnyí kɑm nní pisoi m pɑ́lɛ̃ íi kɛnɛ́mɛɛ léeri, ɑmɑ́ Uléécɑɑ yɛɛ nɛ́ n tummɛ kɛ́mɛɛ kii léeri. 
Úye un n lɑ ukɛ́ Uléécɑɑ mɛlɑ wɑ, u yɛ́ céri rɛ Uléécɑɑ kɛ́mɛɛ kɛ icélɑɑ nnyí ii léeri nɛ́ɛ nnɛ́nɔ́ɔsimɛ́ cirɛ yɛ mu. 
Nkó yɛɛ unnɔɔsimɛ́ cirɛ n símisi yɛ lɑlɛ pikɛ́ ríyu u wɑisɛ. Amɑ́ nkó yɛɛ ń lɑ pikɛ́ uyɛɛ u n tummɛ ríyu wɑisɛ yɛ ɑsei mɑ́lɛ. Nnɔ́ɔmɛ ńkɑ ńn ukɛmɛɛ́ we. 
Ńté Moisi yɛ isé nɔ́ pɑ mɑ́? Nɔ́ úkɑ úu kɔ pɛɛ isé iyɛ̃ tikilɛ̃. Yo nnyɑ kɑni pɛɛ́ lɑ nɔkɛ́ nɛ́ kpu? 
Pisoi pɛɛ n cɑ́pinɛlɛ̃ ɑpi u pɛsɛ rɛ: Pɔ kuníri héesilɛ. Wóo lɑ ukɛ́ pɔ́ kpu? 
Yeesu uu pi pɛsɛ rɛ: Mɛwɑi mɛsɛ kɑm wɑ ɑi nɔ́nnɛ́nɛ́í pírí wɑi. 
Moisi yɛɛ isé nɔ́ pɑ rɛ ɑni yɛ siwɑ́ntisi kérii, ɑ́i kɔ pɛɛ uyɛɛ nɛ li kori, hɑ́i nɔ́pinɛ́kpurɛ foí-mɛ kɑi léeri. Ani kɔ ŋmurɛi, hɑ́i ricuruu ɑni yɛ kɛwéntɛyɑɑ tuŋɛ́ usoi ké. 
In tɛ nɔ yɛ ŋmurɛi ɑni kɛwéntɛyɑɑ tuŋɛ́ kɛwɑ́ntisi ké, ɑ́i kɔ pɛɛ Moisi isé kɑni rilɔ́ɔ́, yo nnyɑ kɑni pɛɛ́ nɛ nɛ́ wɔ́lɑɑlɛ̃ tɛ nɛ kɛsoipipi nɛ kɛriyu kɛwéntɛyɑɑ tuŋɛ́ pɔisɛ? 
Áni kɑpɛ yɛ usoi kɛyu ripɑí ɑní nɛ u túhɑɑnɛ, ɑmɑ́ ɑni yɛ ɑsei kɛmúŋɛ́ ritiki ɑní nɛ túhɑɑnɛ. 
Yerusɑlɛm pikɔ́ pinyinɛ ɑpi rɛ: Ái utisi nkó kɑpi wɛ́ɛ́si pikɛ́ kpuu? 
Uyɛɛ nkó un nɛ uripɔɔ símisi lin cɑ́lɛ̃, úkɑ úu kɔ ríkɑ u mɑ! Yɑrɛ pirɔ́yɔ́ɔpi yɛ cerilɛ rɛ mɛsei uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ nɛ́ɛ? 
Tɔ nkó pɔ̃́ kɛleemɛ nyulɛ. Amɑ́ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ un píyei n kɑ, úkɑ úu yɛ́ céri mɛ́ye kuu léeri. 
Yeesu yɛ pɛɛ Uléécɑɑ kɛyɔ-i welɛ un likumúŋɛ́ kpɛ-i céési. Uu pupɛi rɛ: Mɛsei kɑni nɛ́ nyu nɔn kɔ kɛnɛ́leemɛ nyu, nɛ́ɛ yo? Ḿpɑ́ nɛ lɛ̃, ɑ́i mɛnɛ́lɑ kɛcɑ́ɑ́ kɑḿ nɛ kɑ. Amɑ́ uyɛɛ nɛ́ n tummɛ yɛ ɑsei mɑ́lɛ. Áni uyɛ̃ nyu. 
Nɛ̃́ nɛ u nyulɛ. Ukɛmɛɛ́ kɑm léeri. Uyɛɛ nɛ́ tummɛ. 
Kɛ Yeesu uu lɛ̃ ḿ mɑ nnyɑ, ɑpi pɛɛ́ n lɑ pikɛ́ u tĩ. Amɑ́ úkɑ úu kɑhɑ ukɛ́ u ricɑ, kɛ uituŋɛ íi pɛɛ kɑhɑnɛ nní n tu nnyɑ. 
Pisoi mɛyɑ̃ ɑpi riwúí tɛ̃ kɛcɔpɛ nɛ kɛfɑ u tɛnɛ ɑpí mɑɑ rɛ: Uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ un píyei n kɑ, yɑrɛ u yɛ́ mɛwɑisɑŋɑ wɑpisi ɑi tɔ́su lɛ̃ kɛ utisi nkó uu nní n wɑpisii? 
Pifɑrisi ɑpi lɛ̃ kɛ pisoi ɑpi riwúí kɛ́mɛɛ Yeesu kɛcɑ́ɑ́ ń wɛirɛsɛntɛnɛ kom. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi ɑpi pɛɛ Uléécɑɑ kɛyɔ pimɛ́rɛ́ pinyinɛ tum tɛ pikɛ́ hɑ Yeesu tĩ. 
Yeesu uu pɛɛ rɛ: Nkɑ́ripi kɛ tɔ́ nɛ nnɔ̃́ ɑri kɛtónɛ kpɑ́nɛ ɑm pɛɛ limɛmɑ́ɑ́ uyɛɛ nɛ́ n tummɛ kɛ́mɛɛ pɛlɛ. 
Nɔ́ kɑm ɑni nɛ́ wɛ́ɛ́si, ɑmɑ́ ɑ́ni yɛ́ pɛɛ nɛ́ yɛ̃́. Li we rɛ ɑ́ni yɛ́ fe nɔkɛ́ kei kɑm yɛ́ hɑ ń we hɑ. 
Pisuifi ɑpi pɛɛ pimɛcɔpɛcirɛ m písɛinɛ rɛ: Yei kuu sĩ́ kɑ́ri yɛ́ fe tɔkɛ́ u yɛ̃? Nɛ́ɛ Pisuifi pɛɛ Pikirɛki kɛtẽ kɛcɑ́ɑ́ n yɛkɑɑlɛ̃ kɛ́mɛɛ kuu sĩ́ ukɛ́ hɑ Pikirɛki n céési? 
Tɛ íye kuu nní mɑɑ rɛ: Nɔ́ kɑm ɑni nɛ́ wɛ́ɛ́si, ɑmɑ́ ɑ́ni yɛ́ pɛɛ nɛ́ yɛ̃́. Li we rɛ ɑ́ni yɛ́ fe nɔkɛ́ kei kɑm yɛ́ hɑ ń we hɑpɔ? 
Anyɑ̃́ kɛyɑɑ tɔrɔɔ kɛɛ sinnɛ́í kɛcɔpɛ kɛyɑ́ɑ píimɑ. Kɛ̃ kɛ Yeesu uu pisoi kɛyu-i nyɛrɛ uu pupɛi rɛ: Nnírɛ́ nn úye n we, ukɛ́ kɛnɛ́mɛɛ kɑm ukɛ́ míni n ntí. 
Míni mɛɛ yɛ nfɑ́ɑ n hɛ yɛ́ nkó yɛɛ nɛ kɛfɑ nɛ́ n tɛnɛ kɛ́mɛɛ n leerilɛ mɛn wɔ́lu yɑrɛ ńnihɔi, yɑrɛ kɑi Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ mɛcɔ. 
Uléécɑɑ Nfɑɑsɔnɛ kɛ pɛpɛɛ nɛ kɛfɑ u n tɛnɛ ɑpi yɛ́ ń yɔsí nkɔ́ kɛ Yeesu uu lɛ̃ símisi. Ápi pɛɛ kɑhɑnɛ likumúŋɛ́ kpɛ-i Nfɑ́ɑsɔnɛ n yɔsí, kɛ Uléécɑɑ úu pɛɛ kɑhɑnɛ Yeesu ríyu n wɑisɛ nnyɑ. 
Kɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ kɛcɔpɛ pinyinɛ ɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, ɑpi rɛ: Mɛsei usoi nkó yɛɛ ɑntepu uyɛɛ pɛɛ n sɔ́nti. 
Picɔ pɔ́ɔ rɛ: Uyɛ̃ kɛ Uléécɑɑ uu wɛ́ɛ. Kei kɛ picɔ pɔ́ɔ kɔ rɛ: Uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ yɛ́ fe uu pɛɛ Kɑlilee-pɔ leerii? 
Ńté li Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ yɛ Tɑfiti kɛpirɛ̃ lɛ? Pɛtɛlɛhɛm kuyu-i kɛ Tɑfiti uú pɛɛ ń we kɑpi u mɑrunɛ. 
Lɛ̃ nnyɑ kɛ pisoi ɑpi Yeesu kɛcɑ́ɑ́ ɑnɔ́ɔ ɑkɔ́ ɑkɔ́ símɑɑnkɛɛ. 
Pinyinɛ yɛ pɛɛ pikɛcɔpɛ lɑlɛ pikɛ́ u tĩ, ɑmɑ́ úkɑ úu pɛɛ kɑhɑ ukɛ́ u ricɑ. 
Uléécɑɑ kɛyɔ pimɛ́rɛ́ ɑpi yisi ɑpi pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi kɛ́mɛɛ pɛlɛ. Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi ɑpi pi pisɛ rɛ: Yo nnyɑ kɑ́ni u tĩ nɔkɛ́ nɛ kɑm? 
Api pi pɛsɛ rɛ: Nɛ kɛweesi, usoi úkɑ úu símɑɑlɛ̃ yɑrɛ usoi nkó mɛcɔ. 
Kei kɛ Pifɑrisi ɑpi pɛɛ pi pisɛ rɛ: Yɑrɛ nɔ̃́ nɛ kɔ tíyɛlɛ uu kɔ nɔ̃́ feriyɛɛ? 
Yɑrɛ pirɔ́yɔ́ɔpi nɛ tɔ̃́ Pifɑrisi kɛcɔpɛ usɛ unyinɛ yɛ nɛ kɛfɑ u tɛnɛlɛ nɛ́ɛ? 
Amɑ́ risoiwuí ntí tɛɛ nní u n tíkilɛ̃ ɑ́ri isé nyu. Uléécɑɑ yɛ lɛ̃ nnyɑ ɑnɔ́ɔ ri wɑlɛ. 
Pifɑrisi kɛcɔpɛ usɛ kɑpi yɛ n sée rɛ Nikotɛm yɛ pɛɛ kei welɛ. U pinyinɛ-i Yeesu kɛ́mɛɛ hɑ́pɑɑlɛnlɛ. Uyɛɛ pi mɑɑ rɛ: 
Irɔ́sé íi ncée hɛ rɛ pikɛ́ usoi ɑkópɛ pɑ ɑ́pi kɑhɑnɛ unkɔ́ n kõ, nɛ́ɛ ɑ́pi kɑhɑnɛ n ceri lɛ̃ kuu n wɑ. 
Kei kɑpi pɛɛ u pisɛ rɛ: Nɛ́ntɛ pɔ̃́ nɛ kɔ nkpéni Kɑlilee ukɔ́ lɛ, nɛ́ɛ yo. A ńsɔnɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́ɛ̃ ɑɑ kóm tɛ ɑntepu úkɑ úu píkɑi Kɑlilee-pɔ leemɛlɛ̃. 
Lɛ̃ mɛmɑ́ɑ́, ḿpɑ́ úye uu yisi uú kulu.
