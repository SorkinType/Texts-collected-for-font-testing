Yohɑni.
11.
Utisi unyinɛ yɛɛ pɛɛ́ we un pɔ́ɔ́lú, ɑpi yɛ u séi rɛ Lɑsɑɑ. Mɑɑri nɛ umɑ́ɑ Mɑritɑ kuyu Petɑnii-pɔ kuú pɛɛ́ we. 
Mɑɑri uyɛɛ Upíimɑ ɑnɑ tulɑɑli kɔɔnu uú nɛ unnyúpi tinnɛ. Uuyɛi Lɑsɑɑ yɛɛ pɛɛ pɔ́ɔ́lú. 
Lɑsɑɑ piyɛi pɛ̃ kɛtɛ́ ɑpi Yeesu kɛ́mɛɛ tum ɑpí hɑ u mɑɑ rɛ: Upíimɑ, usoi uyɛ̃ kɑɑ nní ń lɑ yɛ pɔ́ɔ́lúlɛ. 
Kɛ Yeesu uu nsímɛ́ mmɛ̃ n kṍ, uu rɛ: Ái nkpɔ itoi kuu pɔ́ɔ́lú, ɑmɑ́ Uléécɑɑ ukɛ́ nɛ ríyu n yɛ̃́ nnyɑ kuu pɔ́ɔ́lú. Kei kɑi yɛ́ kɔ nɛ Uléécɑɑ Kɛpipi ríyu wɑisɛ. 
Yeesu yɛ pɛɛ Mɑritɑ nɛ uwɑ̃́ Mɑɑri nɛ Lɑsɑɑ lɑlɛ. 
Kuu pikómɛ ḿ mɑsí rɛ Lɑsɑɑ yɛ pɔ́ɔ́lú mɛmɑ́ɑ́, uu kei kuú pɛɛ ń we siyɑ́ɑ sitɛ́ kɛtónɛ kpɑ́. 
Uu pɛɛ upipirɛtiki mɑɑ rɛ: Tɔkɛ́ Yutee kɛtẽ-mɛ̃ pɛlɛ. 
Upipirɛtiki ɑpi u pɛsɛ rɛ: Ucélɑɑ, ɑ́i kɑhɑnɛ n nɑ́ŋɑi kɛ Pisuifi ɑpí pɛɛ kei-pɔ lɑ pikɛ́ ɑpɑrɛ pɔ́ tɑpisi pikɛ́ kpu. Kei kɑɑ kɔ lɑ pɔkɛ́ pɛlɛ nɛ́ɛ? 
Yeesu uu pi pɛsɛ rɛ: Kɑi yɛ nní n weesi, ituŋɛ yɛɛ leemɛlɛ nɛ́ɛ íi yɛ leemɛ? Úye un nɛ mɛtuŋɛtɛí n sɔ́nɛ, úu yɛ rikpɔu. Li we rɛ u yɛ kɛtẽ nkɛ́ kɛcɑ́ɑ́ mɛtɛ́í n nyɑ́nilɛ. 
Amɑ́ úye un kɛsinɛ n sɔ́nɛ, u yɛ rikpɔulɛ rɛ mɛtɛ́í ɑ́mɛ ukɛmɛɛ́ we nnyɑ. 
Kɛ Yeesu uu lɛ̃ pimɑ́ɑ ḿ mɑsí, uu kɔ kpɑ́ rɛ: Urɔ́sɑ́nɛ Lɑsɑɑ yɛ lɔnilɛ. Nɛ sinlɛ kɛ́ hɑ u yóŋsɛ. 
Upipirɛtiki ɑpi u mɑɑ rɛ: Upíimɑ, in tɛ u lɔnilɛ, u yɛ́ peí. 
Lɑsɑɑ nkpɔ nsímɛ́ kɛ Yeesu uu lɛ̃ tee. Upipirɛtiki pɔ́ɔn músu rɛ nnɔni yíkíyiki nsímɛ́ kuu símisi. 
Yeesu uu pɛɛ pi luru rɛ: Lɑsɑɑ yɛ kpulɛ. 
Nɔ̃́ nnyɑ kɑi nɛ́ lɑ́ɑ́rú rɛ ɑ́m pɛɛ likumúŋɛ́ kpɛ-i kei we. Li lɛ̃ wɑlɛ rɛ nɔkɛ́ nɛ kɛfɑ nɛ́ n tɛnɛ nnyɑ. Tɔkɛ́ nkpéni ukɛmɛɛ tɔ́mpɔ. 
Kei kɛ Tomɑɑ kɑpi rinyíri n hɛ rɛ Úsikɑ uu pɛɛ pipirɛtiki tɔrɔɔ mɑɑ rɛ: Ani mpɔ tɔ́ɔkɛ́ kɔ hɑ, tɔ́ nɛ uyɛ̃ tɔkɛ́ hɑ kɛsẽ́ kpu. 
Kɛ Yeesu uu n tuipɔ, uu kóm tɛ Lɑsɑɑ yɛ nhórɛ kɛ́mɛɛ siyɑ́ɑ sinɑ piwɑi mɑsilɛ. 
Petɑnii yɛ nɛ Yerusɑlɛm kɔlɛnlɛ yɑrɛ kilomɛtiri pitɑɑni kumúŋɛ́. 
Pisuifi mɛyɑ̃ yɛ pɛɛ Mɑritɑ nɛ Mɑɑri piyɑ́hɑɑ kɑlɛ pikɛ́ piuyɛi nkpɔ pi riyɑ́kɑsɛ. 
Kɛ Mɑritɑ uu n kṍ tɛ Yeesu yɛ nnyɑ́ɑ̃ wɑimɛ, uu yisi uu u tɛ́pɛi, Mɑɑri pɔ́ɔ kɛ́yɔ-i tonɛ. 
Mɑritɑ uu Yeesu mɑɑ rɛ: Upíimɑ, pɔn pɛɛ nté n we, unɛ́yɛ́i úu yɛ́ pɛɛ kpu. 
Amɑ́ nɛ nyulɛ rɛ ḿpɑ́ nkpéni, Uléécɑɑ yɛ́ lɛ̃ nnɛ́í kɑɑ u m pisɛnɛ pɔ́ pɑ. 
Yeesu uu u mɑɑ rɛ: Upɔ́yɛ́i yɛ nkpɔ kɛ́mɛɛ yisinɛ. 
Mɑritɑ uu rɛ: Nɛ nyulɛ rɛ u yɛ́ kɑm uu kɛyɑ́ɑ tɔrɔɔ kɛtúŋɛ́ kɛ pikpɔkpɔ ɑpi n yisinɛ yisi. 
Yeesu uu u mɑɑ rɛ: Nɛ́ɛ yɛ pikpɔkpɔ yukusɛ, nɛ́ɛ kɔ nfɑ́ɑ. Usoi nkó yɛɛ nɛ kɛfɑ nɛ́ n tɛnɛ yɛ nfɑ́ɑ wɑinɛ, ḿpɑ́ un píyei n kpu. 
Nkó yɛɛ nfɑ́ɑ m mɑ́ un pɛɛ́ nɛ kɛfɑ nɛ́ tɛnɛlɛ̃ úu píkɑi kpinɛ. Pɔ lɛ̃ ŋmurɛii? 
Mɑritɑ uu u pɛsɛ rɛ: Ɛɛɛ Upíimɑ, nɛ nɛ kɛfɑ tɛnɛ rɛ pɔ̃́ kɛ Uléécɑɑ uu wɛ́ɛ. Pɔ́ɔ Uléécɑɑ Kɛpipi kɛɛ kɛtẽ kɛcɑ́ɑ́ mɛ́kɑmɛ m mɑ́. 
Kɛ Mɑritɑ uu lɛ̃ pimɑ́ɑ ḿ mɑsí, uu yisi uu tɔ́mpɔ, uú hɑ uwɑ̃́ Mɑɑri séi uu u wɛ́í-wɛ́í rɛ: Ucélɑɑ yɛ kɑlɛ, u pɔ́ séilɛ. 
Kɛ Mɑɑri uu lɛ̃ n kṍ, uu mɛsɛ nɛ mɛsɛ yisi uu Yeesu tɛ́pɛi. 
Yeesu úu pɛɛ kɑhɑnɛ kuyu kpɛ-i n lompɔ. Kei kɛ Mɑritɑ uú pɛɛ́ nɛ u rinsɑ́nɛ cirɛ kuú pɛɛ́ kɔ nɛ lɛ̃ we. 
Pisuifi pɛɛ pɛɛ Mɑɑri kɛyɔ-i ń we pin nkpɔ u yɑ́ɑ́si, ɑpi yɛ́nu rɛ u riwɑkɑsilɛ uu léepɔ. Api yisi ɑpi u tiki. Pi pɛɛ lɛ̃ músulɛ rɛ rikpíí-i kuu sĩ́ ukɛ́ hɑ tẽ́. 
Kɛ Mɑɑri uu kei kɛ Yeesu uu ń we n tuipɔ, uu uɑnɑ mɛtene lólu uu u mɑɑ rɛ: Upíimɑ, pɔn pɛɛ nté n we, unɛ́yɛ́i úu yɛ́ pɛɛ kpu. 
Kɛ Yeesu uu n yɛ̃́ tɛ Mɑɑri yɛ ténilɛ, Pisuifi pɛɛ u rintíkipɔ pɔ́ɔn kɔ téni, ɑi u lémesi uripɔɔ ɑri cɑɑi hɑ́i, 
uu pɛɛ pi pisɛ rɛ: Yei kɑni u wɑ? Api u pɛsɛ rɛ: A kɑm ɑ yɛ̃, Upíimɑ. 
Yeesu uu téni. 
Pisuifi ɑpi pɛɛ rɛ: A tíyɛ u pɛɛ u lɑ kpɑ! 
Pikɛcɔpɛ pinyinɛ pɔ́ɔ rɛ: Uyɛɛ unyíyɛ inipɛɛ n wúkulɛ úu yɛ́ pɛɛ fe uu wɑi usoi nkó úu yɛ kpuu? 
Ai Yeesu pilémɑɑ kpɑ́, uripɔɔ ɑri cɑɑi hɑ́i uu yisi uu rikpíí-i tɔ́mpɔ. Kupɑrɛpoo kunyinɛ yɛ ku kɑpi nnɔ́ɔ-i mɛpɑrɛ rińfi. 
Yeesu uu pi mɑɑ rɛ: Ani kɛpɑrɛ lesɛ. Ukpɔkpɔ uyɛi, Mɑritɑ uu Yeesu mɑɑ rɛ: U nkpéni pikpɑ́hɑi mɑsí, Upíimɑ. Kuú nɛ n kpu, li siyɑ́ɑ sinɑ piwɑi mɑsilɛ. 
Yeesu uu u pɛsɛ rɛ: Ám pɔ́ mɑɑ rɛ pɔn nɛ kɛfɑ n tɛnɛ, pɔ́ yɛ̃́ tɛ Uléécɑɑ yɛ ríyu mɑ́lɛɛ? 
Kei kɑpi pɛɛ kɛpɑrɛ wúkulɛ, Yeesu uu kɛléécɑɑ weríí uu rɛ: Sɑ́ɑ, nɛ pɔ́ pɔɔnɛsɛ rɛ pɔ nɛ kutu nɛ́ ricɔ nnyɑ. 
Nɛ̃́ nɛ nyulɛ rɛ mɛsɛ́rɛ kɑɑ́ nɛ kutu nɛ́ cɔ́lɛ̃. Amɑ́ pisoi riwúí píimɑ ntí tɛɛ nní nɛ́ n kɑ́lɑɑlɛ̃ nnyɑ kɑm lɛ̃ mɑ rɛ pikɛ́ fe pikɛ́ nɛ kɛfɑ tɛnɛ rɛ pɔ́ɔ nɛ́ tummɛ nnyɑ. 
Kuu lɛ̃ pimɑ́ɑ ḿ mɑsí, uu cɑ́ɑ́i ɑi cɑ́ rɛ: Lɑsɑɑ, ɑ leemɛ! 
Usoi uu kei nní yisi uu léeri, uɑnɑ nɛ uɑnipɛ ɑn sisɑ́ŋɑ́ɑ́pɑmpi pílɑɑnkɛɛlɛ̃, ukɛyu kɛn kusɑ́ŋɑ́ɑ́ yɑ́riilɛ̃. Yeesu uu pɛɛ pi mɑɑ rɛ: Ani u fénnɛntɛ ɑni u yɑ́ ukɛ́ tɔ́mpɔ. 
Pisuifi mɛyɑ̃ pɛɛ Mɑɑri kɛyɔ-i n hɑ́pɔ ɑpi lɛ̃ kɛ Yeesu uu n wɑ yɛnu, ɑpí nɛ kɛfɑ u tɛnɛ. 
Amɑ́ picɔ pɔ́ɔ sĩ́, ɑpí hɑ lɛ̃ kɛ Yeesu uu n wɑ Pifɑrisi kɛɛnkɛɛ. 
Kei kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi ɑpi pitúhɑɑnɛ píimɑ cɑ́pinɛ ɑpi rɛ: Íye kɑri wɑinɛ? Usoi nkó yɛ mɛwɑisɑŋɑ wɑpisilɛ hɑ́i! 
Tɔn n tíyɛ un lɛ̃ n wɑi, pisoi nnɛ́í yɛ́ nɛ kɛfɑ u tɛnɛ, Rɔm pikɔ́ ɑpi pɛɛ́ weri ɑpi Uléécɑɑ kɛyɔ foru ɑpí nɛ kurɔ́yu pɛ́nɛ. 
Pisuifi piwɛ́ɛ́sɛ pɛ̃ usɛ kɑpi yɛ sée rɛ Kɑyifu. Uyɛɛ pɛɛ kuŋmɛ̃ kpɛ̃ Uléécɑɑ isinɑ tũ. Uyɛ̃ uu pɛɛ pi mɑɑ rɛ: Áni ńkɑ kómɛi! 
Áni yɛ̃ tɛ li nɛ kutɔsi nɔ́ we rɛ usoi usɛ ukɛ́ nɔ́nnɛnɛ́í nkpɔ kpu, kuyu nnɛ́í ɑ́ku kɑpɛ nɛ mɛ́woo rinsɛ́ɛ́ nnyɑɑ? 
Ái rɛ ukɛmúŋɛ́ cirɛ kuú nɛ lɛ̃ mɑ. Kɑi nní uyɛɛ pɛɛ likumúŋɛ́ kpɛ-i Uléécɑɑ usinɑ nnyɑ kuú nɛ Uléécɑɑ rinyiri símisi rɛ Yeesu yɛɛ Pisuifi kɛcɑ́ɑ́ kpinɛ. 
Ái kɔ Pisuifi kɛcɑ́ɑ́ ŋmɑnɛ nnyɑ kuu kpinɛ, ɑmɑ́ li pisɛ rɛ ukɛ́ kpu ukɛ́ nɛ Uléécɑɑ sipipi sɛɛ kɛtẽ nnɛ́í kɛcɑ́ɑ́ n yɛkɑɑlɛ̃ cɑ́pinɛ sikɛ́ pɑnsɛ pisoi pisɛ. 
Kɑí nɛ kɛyɑ́ɑ kɛ̃ n kpísi, Pisuifi piwɛ́ɛ́sɛ ɑpi rinɔ́ɔ risɛ wɑi rɛ pikɛ́ Yeesu kpu. 
Lɛ̃ nnyɑ, Yeesu úu yɛ pɛɛ Pisuifi kɛcɔpɛ pinyɛ́rɛi rikpɑ́ pikɛ́ u n nyɑ́ni. Uu pɛɛ yisi uu kɛtẽ kɛnyinɛ kɛɛ nɛ kucɛsi kóimɑ n kɔ́lɛ̃-mɛ̃ tɛ́ŋ́, u nɛ upipirɛtiki ɑpí hɑ kei-pɔ n we. Pi yɛ kei sée rɛ Efirɑyim. 
Isirɑyɛɛli pikɔ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ yɛ pɛɛ nyɑhɑimɛlɛ. Lɛ̃ nnyɑ kɛ kei pikɔ́ mɛyɑ̃ ɑpi yisi ɑpi Yerusɑlɛm sĩ tɛ pikɛ́ hɑ inyɔ́ɔnsɛ inyɛkii wɑ ɑnyɑ̃́ ɑkɛ́ kɛlenɛ tu. 
Api Yeesu wɛ́lɑɑnkɛɛ, ɑpi pɛɛ Uléécɑɑ kɛyɔ-i n we pin pɛɛ pimɛcɔpɛcirɛ písɛinɛ rɛ: Ń-ye kɑni músu? Yɑrɛ u yɛ́ kɑm nɛ́ɛ úu yɛ́ kɑm? 
Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ Pifɑrisi yɛ pɛɛ rinɔ́ɔ pihɛɛ mɑsilɛ rɛ: Úye un ukɛlõ n nyu, ukɛ́ símisi pikɛ́ fe pikɛ́ u tĩ.
