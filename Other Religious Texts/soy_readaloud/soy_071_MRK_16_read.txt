Mɑriki.
16.
Kɛ kɛwéntɛyɑɑ kɛtúŋɛ́ ɑkɛ ń tɔsí, Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri nɛ Mɑɑri Yɑkupu uni nɛ Sɑlomee ɑpi tulɑɑli lɔlu pikɛ́ hɑ Yeesu yɑkɑsi. 
Kɛwéntɛyɑɑ ukóso, ɑpi mɛkɛɛkɛɛ mɛtuŋɛleemɛ rikpíí sĩ́. 
Api pimɛcɔpɛcirɛ písɛinɛ rɛ: Wóo yɛ́ hɑ ripɑrɛ tɛɛ rikpíí rinɔnɔɔ n fílɛ̃ tɔ́ pímiisɛ? 
Kɑpi yɛ́ inípɛɛ rinsíkɑ, ɑpi yɛ́nu rɛ li mɛpɑrɛcɑŋɑɑ mɛ̃ pímiisɛlɛ. 
Api rikpíí-i lompɔ ɑpi kɛŋmɑ́nɛpi kɛnyinɛ yɛnu kɛn kulukɛ-lukɛ-mɛ̃ tũ kɛn kɛtúkɑnkɑ tomɛ́ tɑnɑɑlɛ̃, iwɑmɛ ii pi wɑi. 
Akɛ pi mɑɑ rɛ: Iwɑmɛ íi kɑpɛ nɔ́ wɑ, nté Yeesu Nɑsɑrɛti kuyu ukɔ́ uyɛ̃ kɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ n kɑrii kɑni wɛ́ɛ́si? U nkpɔ kɛ́mɛɛ yisilɛ, úu nté we. Ani nyɑ́nii, kɛlõ nkɛ́-i kɑpi u finsɛ. 
Amɑ́ ɑni hɑ Piyɛɛ nɛ upipirɛtiki tɔrɔɔ mɑɑ rɛ u yɛ́ hɑ Kɑlilee-pɔ pi m mɛ̃́, kei kɑpi yɛ́ hɑ u yɛ̃ yɑrɛ kuu pi n símɑɑlɛ̃. 
Api rikpíí-i léeri ɑpi ntóó sɛ́ pin tɛrii nɛ iwɑmɛ, ɑmɑ́ ɑ́pi úkɑ ríkɑ mɑ nɛ iwɑmɛ mɛpehẽ. 
Kɛ Yeesu uu kɛwéntɛyɑɑ ukóso mmɑrɛ nkpɔ kɛ́mɛɛ n yisi, u tíyɛsɛlɛ Mɑɑri Mɑkitɑlɑ uu mɛfoí u yɛnu. Mɑɑri Mɑkitɑlɑ uyɛ̃ kɛ́mɛɛ kuú pɛɛ ɑníri ɑsɛɛi lɑkɑsɛ. 
Mɑɑri Mɑkitɑlɑ uú hɑ nsímɛ́ mmɛ̃ pɛpɛɛ pɛɛ Yeesu rintíki símisi. Pɛ̃ pin kumúŋɛ́ kpɛ-i nkpɔ tũ pin kɔ ténesi. 
Kɑpi n kṍ tɛ Yeesu yɛ nfɑ́ɑ mɑ́lɛ, tɛ Mɑɑri Mɑkitɑlɑ yɛ u yɛ̃, ɑ́pi nɛ kɛfɑ tɛnɛ. 
Lɛ̃ mɛmɑ́ɑ́, Yeesu uu mɛ́wee féé mɛcɔ kpísi uu tíyɛsɛ upipirɛtiki pitɛ́ pinyinɛ pɛɛ kɛcɑrɛ-mɛ̃ n sĩ́ ɑpi u yɛnu. 
Pɛ̃ ticuruu ɑpí weri ɑpi picɔ nsímɛ́ mmɛ̃ símisi, picɔ pɛ̃ ɑpi kɔ nɛ kɛfɑ tɛnɛ. 
Yeesu uu mɛtɔ́rɔɔ umɛcirɛ upipirɛtiki kɛfi nɛ usɛ nyísɛ kumúŋɛ́ kpɛ-i kɑpi ilukɛ n le. Uú nɛ pinfɑtɛnɛ kucírɛ́ nɛ pinkiŋtɑkɛ pi cɛ́si, kɑ́pi pɛpɛɛ pɛɛ nkpɔ kɛ́mɛɛ mɛyisɛ-mɑɑ́ u n yɛ̃́ nɛ kɛfɑ u tɛnɛ nnyɑ. 
Uu pɛɛ lɛ̃ mɛmɑ́ɑ́ pi mɑɑ rɛ: Ani kɛtẽ nnɛ́í rikɔ́ɔ́nú ɑni pisoi nnɛ́í Nsímɛ́ Kɛcirɛ símisi. 
Uléécɑɑ yɛ́ nkó yɛɛ nɛ kɛfɑ u n tɛnɛ uu kɔ míni wolɛ yoriyɛ. Amɑ́ u yɛ́ nɛ nkó yɛɛ úu nɛ kɛfɑ u n tɛnɛ túhɑɑnɛ ɑí nɛ u kpi. 
Mɛwɑisɑŋɑ kɛ pɛpɛɛ nɛ kɛfɑ Uléécɑɑ n tɛnɛ ɑpi yɛ́ ń wɑpisi yɛ mmɛ́: Pi yɛ́ rinɛ́nyíri nɛ ɑníri lɑkɑsɛ, ɑpi mɛyu féé símisi. 
Pin iwɑ́ɑ n cɔ́pii, íi yɛ́ líkɑ pi wɑ. Ḿpɑ́ pin kɔ linírɛ́-nirɛ́ lɛɛ yɛ́ n kpu n ntí, ɑ́i yɛ́ nkpɑ́ni líkɑ pi wɑi. Pi yɛ́ pitóikɔ́ kɛcɑ́ɑ́ ɑnípɛ rilɑɑ, pitóikɔ́ pɛ̃ ɑpi pélu. 
Kɛ Upíimɑ uu pisímɛ́ pi ḿ mɑsí, ɑi kɛyómɛcɑɑ-mɛ̃ u kpíípɔ uú hɑ Uléécɑɑ kulukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tonɛ. 
Upipirɛtiki ɑpi sĩ́ ɑpí hɑ ḿpɑ́ yei Nléécɑɑsimɛ́ símisi. Upíimɑ uu pikɛi pi n lɛ́ni, uu tíyɛ mɛwɑisɑŋɑ kɑpi n wɑpisi ɑmɛ pinsímɛ́ ɑsei n nyísɛ.
