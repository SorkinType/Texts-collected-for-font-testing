Luki.
10.
Lɛ̃ mɛmɑ́ɑ́, Upíimɑ uu pisoi kuwóó nɛ ɑfɛɛrɛ́ wɛ́ɛ uu pitɛ́ pitɛ́ pi tum ɑyu nɛ silõ nnɛ́í kɛ́mɛɛ kɑi m pisɛ rɛ uricuruu ukɛ́ kɔ sɔ́nɛ. 
Uu pi mɑɑ rɛ: Ilukɛ kɑpi ń kpɑsinɛ yɛ piyɛlɛ, ɑmɑ́ pikɛikɔ́ pɛɛ yɛ́ i n kpɑsi ɑ́pi piyɛ. Ani nɛ ucɑrɛte tẽ́ ukɛ́ pɛpɛɛ yɛ́ i n kpɑsi rikpɑ́. 
Ani ncée kpísi! Nɛ nɔ́ tumlɛ rɛ ɑni isɑ́ŋ yɛɛ ɑ́kpɛ kɛcɔpɛ ń we mɛcɔ n sɔ́nɛ. 
Áni kɑpɛ siwóó nɛ́ɛ kulɔ́ɔ nɛ́ɛ ɑnɛ́ɛ́ri rimúlú, ɑ́ni kɑpɛ ncée-i nyɛrɛ rɛ nɔkɛ́ unyinɛ yɑ́ɑ́si. 
Nɔkɛ́ kɛlenɛ kɛ́yɔ kɛnyinɛ-i n lompɔ, ɑni yɛ mɑɑ rɛ: Nkíŋniŋɛ nkɛ́ kɛ́yɔ nkɛ́-i n we. 
In tɛ nkíŋniŋɛ ukɔ́ unyinɛ yɛ kɛ́yɔ kɛ-i we, nɔ́nnɛ́kíŋniŋɛ yɛ́ u ritiki. Insɑ́ lɛ̃, nɔ́kɛnɛ́mɛɛ kɛ nɔ́nnɛ́kíŋniŋɛ nn pɛlɛnɛ. 
Ani kɛ́yɔ kɛ-i tonɛ, ɑni lɛ̃ kɑpi kei nɔ́ n hɛ le ɑni níru, likumúŋɛ́ rɛ ukɛikɔ́ yɛ uihɛ́ɛ́ mɑ́lɛ. Áni kɑpɛ síyɔ n tɔrii. 
Nɔn kuyu kúye-i n lompɔ pin kusɑ́nɛ nɔ́ n yɔ́su, pin yo nɔ́ m pɑ, ɑni li. 
Ani pitóikɔ́ m pɔisɛntɛ, ɑni kuyu pikɔ́ mɑɑ rɛ: Uléécɑɑ iyɔɔpi yɛ nɔ́ nyɑhɑimɛlɛ. 
Amɑ́ nɔn kuyu kúye-i n lompɔ pin piyɔ́ɔ́ nɔ́ n yɛ̀, ɑni icée-i léepɔ ɑní mɑɑ rɛ: 
Tɔ nɔ́kunɛ́yu nkoŋo mɛɛ ɑrɔ́nɑ riḿmɛɛlú nɔ́kɛnɛ́cɑ́ɑ́ ripéi. Ḿpɑ́ nɛ lɛ̃, ɑni ceri rɛ Uléécɑɑ iyɔɔpi yɛ nɛ nɔ́ rikɔlɛ. 
Asei kɑm nní nɔ́ símisi rɛ kɛyɑ́ɑ kɛ̃ kɛ Uléécɑɑ uu n sɔ́nti ukɛ́ nɛ pisoi n túhɑɑnɛ, Sotɔm pikɔ́ kɛ Uléécɑɑ uu nnɑ n ŋmɑɑsi iwɛ yɛ́ nɛ kuyu kpɛ̃ ikɔ́ kutɔsi wɑ. 
Nɔ̃́ Korɑsin pikɔ́, íwɛ kɑni topori. Nɔ̃́ Petisɑyitɑ pikɔ́, íwɛ kɑni topori! Likumúŋɛ́ rɛ mɛwɑisɑŋɑ kɑpi nɔ́kɛnɛ́mɛɛ n wɑpisi mɛn pɛɛ Tiiri nɛ Sitɔɔ kɛ́mɛɛ n wɑ, ɑi nní ń we pin piɑkópɛ kɛpirɛ ritɔ ɑpi ɑ́fɔrɔnti tɑni ɑpi pimɛcirɛ nfúrɔ̃ kɔikɛɛ likɛ́ nyísɛ rɛ pi consɛ. 
Lɛ̃ nnyɑ kɛ Tiiri nɛ Sitɔɔ pikɔ́ iwɛ ii yɛ́ hɑ kɛtúhɑɑnɛ kɛ́mɛɛ́ nɛ nɔ́inɛ́kɔ́ kutɔsi wɑ. 
Ńté nɔ̃́ Kɑpɛrinɑwum pikɔ́, nɔ músu rɛ kɛléécɑɑ-mɛ̃ kɑní tɑɑ́lɛ̃? Ái nkpɑ́ni lɛ̃. Amɑ́ kukúlɑ̃yu-i kɛ Uléécɑɑ uu kɛtẽ-mɛ̃ nɔ́ lɑ́upɔlɛ. 
Uu upipirɛtiki pimɑ́ɑ kpɑ́ rɛ: Úye un kutu nɔ́ n cɔlɛ̃, nɛ̃́ kuu kutu cɔlɛ̃, úye un nɔ́ n yulu nɛ̃́ kuu yúlu. Úye un kɔ nɛ̃́ n yulu, uyɛɛ nɛ́ n tummɛ kuu yúlu. 
Pisoi kuwóó nɛ ɑfɛɛrɛ́ pɛ̃ kɑpi n tũ ɑpi pɛɛri nɛ mpɔ́ɔnɑrɛ, ɑpi rɛ: Sɑ́ɑ, ɑníri ricuruu yɛɛ rɔ́ pɑkɑrɛlɛ, tɔn tinɔ́ɔ ɑ n hɛ nɛ ripɔ́nyíri. 
Yeesu uu rinɔ́ɔ pi yɔ́su rɛ: Nɛ pɛɛ Setɑni yɛnlɛ un kɛyómɛcɑɑ-pɔ lóiri yɑrɛ kɛkónɛsɛípi. 
Ani kutu ricɔ! Nɛ́ɛ ńnɑŋɛ nɔ́ hɛ rɛ nɔkɛ́ iwɑ́ɑ nɛ inɑ̃ kɛcɑ́ɑ́ sɔ́nɛ ɑni ulɑ́ɑrɔ nnɑŋɛ nnɛ́í tɛ́kɛ́ɛ́ líkɑ ɑ́i yɛ́ kɔ nɔ́ wɑ. 
Amɑ́ ɑ́i kɑpɛ nɔ́ n lɑ́ɑ́rú rɛ ɑníri yɛ nɔ́ wuru nnyɑ. Lɛlɛɛ yɛ́ nɔ́ n lɑ́ɑ́rú lɛɛ rɛ Uléécɑɑ yɛ nɔ́ɑnɛ́nyíri uritɛlɛ́ kɛ́mɛɛ wɔ́i. 
Kumúŋɛ́ kpɛ-i, Nfɑ́ɑsɔnɛ nn Yeesu rikiŋ niŋukusɛ uú mɑɑ rɛ: Sɑ́ɑ, kɛtẽ nɛ kɛyómɛ Upiimɑ, nɛ pɔ́ pɑkɑrɛ rɛ pɔ tíyɛsɛ siwɑ̃́ síńsɑ́pi ɑsi nsímɛ́ mmɛ̃ kɑɑ pisɔhɔɔ nɛ pinyuwɛ ḿ pesí ɑsei kom. Pɔ́ɔ mpɔ́fɑnɑrɛ kɛ́mɛɛ lɑ rɛ likɛ́ lɛ̃ wɑ nnyɑ kɑi kɔ mɛsei lɛ̃ wɑ, Sɑ́ɑ. 
Unɛ́sɑ́ɑ yɛ ḿpɑ́ yo nnɛ́í nɛ́ pɑlɛ. Úkɑ ucɔ úu nɛ́ nyu Unɛ́sɑ́ɑ mɛmɑ́ɑ́. Úkɑ ucɔ úu kɔ Unɛ́sɑ́ɑ nyu insɑ́ nɛ̃́ Ukɛpipi nɛ pɛ̃ kɑm n tíyɛsɛ ɑpi u ceru. 
Yeesu uu lɛ̃ mɛmɑ́ɑ́ upipirɛtiki-mɛ̃ pɑnsɛpɔ uu pi mɑɑ rɛ: Pɛpɛɛ lɛ̃ kɑni nní n nyɑ́ni n nyɑ́ni yɛ pínɑrɛkomɛ lɛ. 
Asei kɛcɑ́ɑ́, ɑntepuyɛ nɛ piyɔ́ɔpi mɛyɑ̃ yɛ kɑ́ipilɛ pikɛ́ lɛ̃ kɑni n yɛ̃́ yɛ̃, ɑmɑ́ ɑ́pi li yɛ̃, ɑpi kɑ́ipi rɛ pikɛ́ lɛ̃ kɑni nní n kṍ kõ, ɑmɑ́ ɑ́pi li kõ. 
Isé ucélɑɑ unyinɛ uu rinɔ́ɔ mɑɑ un lɑ ukɛ́ Yeesu ripinɛ wɑ, uu u pisɛ rɛ: Sɑ́ɑ, íye kɑi pisɛ rɛ kɛ́ wɑ kɛ́ nɛ nfɑ́ɑ tɛnɛcirɛ́ n yɛ̃́? 
Yeesu uu rinɔ́ɔ u yɔ́su rɛ: Íye kɑi irɔ́sé kɛ́mɛɛ wɔ́lɑɑlɛ̃? Kɑɑ yɛ isé ritɛlɛ́ n kɛ́ɛ̃, íye kɑɑ yɛ mpíí i kõ. 
Usoi uyɛ̃ uu rinɔ́ɔ yɔ́su rɛ: A Upíimɑ Upɔ́léécɑɑ n lɑ nɛ kɛpɔ́fɑ nnɛ́í, nɛ ipɔ́soi nnɛ́í, nɛ mpɔ́nɑŋɛ nnɛ́í nɛ kɛpɔ́múŋɛ́ nnɛ́í, ɑɑ kɔ upɔ́cɔ n lɑ yɑrɛ kɑɑ ripɔ́cúruu ń lɑ. 
Yeesu uu pɛɛ u mɑɑ rɛ: Pɔ ńsɔnɛ rinɔ́ɔ nɛ́ pɛsɛlɛ. A lɛ̃ titiki, pɔ́ nfɑ́ɑ yɛ̃. 
Amɑ́ isé ucélɑɑ uu ɑlɑri n wɛ́ɛ́si uu lɛ̃ nnyɑ Yeesu pisɛ rɛ: Wóo unɛ́soicɔ? 
Yeesu uu rinɔ́ɔ u pɛsɛ rɛ: Utisi unyinɛ yɛɛ kɛyɑ́ɑ Yerusɑlɛm leeri un Yeriko-mɛ̃ cɛpilɛ̃, piyɑɑ ɑpi u tini ɑpi uilũ yɔ́su ɑpi u kpɑfui ɑpi itói u wɑi, ɑpi kɛteni-i u lɑlɛi hɛihɛi ɑpi tɔ́mpɔ. 
Aí nɛ sɑ́ uyɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ unyinɛ un ncée mmɛ̃ tikilɛ̃. Kuu utisi uyɛ̃ n yɛ̃́, uu kɛ́ŋɛ́ɛ́ uu fɑ́ɑu. 
Lefi ukɔ́ unyinɛ uu kɔ limɛcɔ kɛlõ kɛ-i tuipɔ, uu utisi uyɛ̃ yɛnu, uyɛ̃ uu kɔ kɛ́ŋɛ́ɛ́ uu fɑ́ɑu. 
Amɑ́ Sɑmɑrii ukɔ́ unyinɛ yɛɛ ncée mmɛ̃ n tíkimɛlɛ̃ uu utóikɔ́ uyɛ̃ kɛlõ tuipɔ. Kuu u n yɛ̃́, ɑi mɛyíkíyiki íwɛ u wɑi. 
Uú nyɔsɔ́pɔ uu mɛ́kpɔ nɛ fɛɛ̃́ uitoi kɛcɑ́ɑ́ kɔikɛɛ, uu i pɑhɑɑnkɛɛ. Uu lɛ̃ mɛmɑ́ɑ́ ukɛminɑɑ kɛcɑ́ɑ́ u tonsɛ uú nɛ picéetɔŋɛ́ kɛyɔ kɛnyinɛ-i u sĩ uú hɑ kei ukɛcɑ́ɑ́ pɑí. 
Kɑi n weesi, uu mɛwóópipi mɛtɛ́ lesɛ uu uyɛɛ picéetɔŋɛ́ kɛyɔ kɛcɑ́ɑ́ ḿ pɑílɛ̃ pɑ uu u mɑɑ rɛ: A usoi nkó kɛcɑ́ɑ́ ripɑí, nɛn hɑ píyei nté m pɛɛmɛ, nɛ́ riwómɛ tɛɛ n tisɛ pɔ́ hɛ́ɛ́lɛ. 
Yeesu uu pimɑ́ɑ kpɑ́ rɛ: Pisoi pitɑɑni mpí kɛcɔpɛ úye yɛɛ uyɛ̃ kɛ piyɑɑ ɑpi nní itói n wɑ usoicɔ? 
Isé ucélɑɑ uu rinɔ́ɔ yɔ́su rɛ: Uyɛɛ kusoi u n wɑ yɛɛ uusoicɔ. Yeesu uu pɛɛ u mɑɑ rɛ: A n ŋme, ɑɑ yɛ limɛcɔ wɑi. 
Kɛ Yeesu nɛ upipirɛtiki ɑpi pinsẽ ń nɔŋ́lɛ̃, ɑpi kuyu kunyinɛ tulu, unɔ́si unyinɛ kɑpi yɛ n sée rɛ Mɑritɑ un kei we, uu ukɛyɔ-i kusɑ́nɛ pi yɔ́su. Mɑritɑ yɛ pɛɛ uwɑ̃́ unyinɛ mɑ́lɛ ɑpi yɛ u séi rɛ Mɑɑri. U pɛɛ kei welɛ. Uu Upíimɑ ɑnɑ mɛtene tonɛ un uicélɑɑ kutu cɔlɛ̃. 
Mɑritɑ pɔ́ɔn nɛ pisɔ́ɔ̃ kɛ́yu mɑhɑni. Uu yisi uu rɛ: Upíimɑ, ɑ́i pɔ́ cɔ́ŋlɛ̃ kɛ unɛ́wɑ̃́ uú nɛ mɛnɛ́círɛ pihúúkɛi n nyɑ́nɛii? A u mɑɑ rɛ ukɛ́ nɛ́ lɛ̃́. 
Upíimɑ uu u mɑɑ rɛ: Mɑritɑ, Mɑritɑ, pɔ mɛpɔ́círɛ ḿpɑ́ yo kɛ́mɛɛ picɔ́ŋ tɔsilɛ nɛ pinɑ́kɑɑnkɛɛ. 
Amɑ́ likɛi lisɛ kɑi usoi pisɛ. Likɛi kɛcirɛ kɛ Mɑɑri uu wɛ́ɛ, úkɑ úu yɛ́ fe ukɛ́ li u yɔsí.
