Yohɑni.
8.
Kɛ pisoi ɑpi n kúũ, Yeesu pɔ́ɔ yisi uu Olifiyee rikúú tɑɑ́. 
Uu kɛmɑrɛsinɛ yisi uu Uléécɑɑ kɛyɔ-i pɛlɛ, pisoi ɑpi u kɑ́lɑɑpɔ. Uu pɛɛ́ tonɛ uu pi n céési. 
Kei kɛ isé picélɑɑ nɛ Pifɑrisi ɑpí nɛ unɔ́si unyinɛ hɑpɔ rɛ pi u torilɛ un iwɑ́sɑ́ wɑi. Api riwúí kɛcɔpɛ u nyɛrɛsɛ, 
ɑpi Yeesu mɑɑ rɛ: Ucélɑɑ, pi unɔ́si nkó tórilɛ un iwɑ́sɑ́ wɑi. 
Moisi yɛ isé rɔ́ pɑ rɛ tɔkɛ́ yɛ lipinɔsi pɛ̃ ɑpɑrɛ tɑpisi tɔkɛ́ kpu. Ń-ye kɛ pɔ́ɔ yɛ̃́? 
Tinɔ́ɔ kɑpi lɛ̃ wɛ́ɛ́si nnyɑ kɑpi lɛ̃ u pisɛ pikɛ́ nɛ nnyinɛ n yɛ̃́ pikɛ́ nɛ pitele u n wɑ nnyɑ. Amɑ́ Yeesu uú pɛ, uu kɛnípɛpi nɛ ntɑɑi kɛ́mɛɛ piwɔ́lɛ kɑ́pɑ́ɑ́. 
Kɑpi ń we pin pipísɛisɛ nɛ kɛ́yu u mɑhɑni nnyɑ, Yeesu uu yisi uu kɛcírɛ nyɛrɛ uu pi mɑɑ rɛ: Uyɛɛ úu nɔ́kɛnɛ́cɔpɛ nkɛ́-i ɑkópɛ n wɑilɛ̃ ukɛ́ mɛfoí ripɑrɛ u tɔ́ɔ́ri. 
Uu kɔ pípɛ kpɑ́ uu ntɑɑi kɛ́mɛɛ piwɔ́lɛ nɔŋ́. 
Kɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, piwɛ́ɛ́sɛ ɑpi pinnɛ́í kpíi usɛ usɛ. Piwɛ́ɛ́sɛ kɑí nɛ kori ɑí nɛ siwɑ̃́ pɑ́ɑ́lɑ́, ɑi pɛɛ nkpéni unɔ́si uyɛ̃ mɛcirɛ kɛcɔpɛ yɑ́, u nɛ Yeesu pin we. 
Yeesu uu pɛɛ yisi uu nyɛrɛ uu u pisɛ rɛ: Unɔ́si, yei kɑpí we? Úkɑ úu ɑkópɛ pɔ́ pɑɑ? 
Uu u pɛsɛ rɛ: Úkɑ úu we, Upíimɑ. Yeesu uu u mɑɑ rɛ: Nɛ̃́ ɑ́m kɔ ɑkópɛ pɔ́ pɑnɛ. A n ŋme ɑɑ tɔ́mpɔ. Amɑ́ kɑpɛ kɔ ɑkópɛ piwɑi rikpɑ́. 
Yeesu uu kɔ pisoi pɛɛ n cɑ́pinɛlɛ̃ pimɑ́ɑ kpɑ́ rɛ: Nɛ̃́ nɛ pisoi nnɛ́í mɛtɛ́í lɛ. Nkó yɛɛ nɛ́ n tíkilɛ̃ úu píkɑi kuŋmɑhɑ kɛ́mɛɛ sɔ́nɛnɛ, ɑmɑ́ mɛtɛ́í mɛɛ yɛ nfɑ́ɑ n hɛ kuu yɛ́ m mɑ́. 
Kei kɛ Pifɑrisi ɑpi pɛɛ u mɑɑ rɛ: Pɔ́ɔ ripɔ́cúruu kɛpɔ́cɑ́ɑ́ ɑseérɑ lesɛ. In lɛ̃, ɑ́i nsímɛ́sei kɑɑ símisi. 
Yeesu uu pi pɛsɛ rɛ: Ḿpɑ́ kɑm tinɛ́cúruu kɛnɛ́cɑ́ɑ́ nsímɛ́ n símisi, nsímɛ́sei yɛ mu. Li we rɛ nɛ nyulɛ kei kɑm n léeri nɛ kei kɑm n sĩ́. Amɑ́ nɔ̃́ ɑ́ni nyu kei kɑm n léeri nɛ kei kɑm n sĩ́. 
Sisoipipi mɛtúhɑɑnɛ mɛcɔ kɛ nɔ́ɔ túhɑɑnɛ. Nɛ̃́ ɑ́m úkɑ nɛ túhɑɑnɛ. 
Amɑ́ in tɛ nɛ nɛ unyinɛ túhɑɑnɛ, ɑsei kɑm tíkilɛ̃ nɛn nɛ u túhɑɑnɛ. Li we rɛ ɑ́i mɛnɛ́círɛ nɛ́ɛ nɛ u túhɑɑnɛ, tɔ́ nɛ Unɛ́sɑ́ɑ yɛɛ nɛ́ n tummɛ yɛ pi. 
Li nɔ́inɛ́sé kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ pisoi pitɛ́ pin linyinɛ kɛcɑ́ɑ́ nsɛ n símisi, nsímɛ́sei yɛ mu. 
Nɛ̃́ nɛ rinɛ́cúruu kɛcɑ́ɑ́ ɑseérɑ lesɛlɛ, Unɛ́sɑ́ɑ yɛɛ nɛ́ n tummɛ un kɔ kɛnɛ́cɑ́ɑ́ ɑseérɑ lesɛ. 
Api pɛɛ u pisɛ rɛ: Yei kɛ Upɔ́sɑ́ɑ uú we? Yeesu uu pi pɛsɛ rɛ: Áni nɛ́ nyu ɑ́ni kɔ Unɛ́sɑ́ɑ nyu. Nɔn pɛɛ nɛ́ n nyu, nɔ́ kɔ pɛɛ Unɛ́sɑ́ɑ pɔ̃́ kɔ n nyu. 
Yeesu yɛ ɑnɔ́ɔ nnyɛ́ mɑlɛ un Uléécɑɑ kɛyɔ kɛ́mɛɛ ɑwóótɑkɑi kɛyúrí we un Nléécɑɑsimɛ́ céési. Kɛ uituŋɛ íi pɛɛ kɑhɑnɛ n tu nnyɑ, úkɑ úu u tĩ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɛ tɔ́sulɛ. Nɔ́ kɑm ɑni nɛ́ wɛ́ɛ́si ɑ́ni yɛ́ nɛ́ yɛ̃́, ɑní nɛ nɔ́ɑnɛ́kópɛ kpíni. Áni yɛ́ fe nɔkɛ́ kei kɑm n símpɔ hɑ. 
Pisuifi ɑpi pɛɛ pimɛcirɛ m písɛinɛ rɛ: Kuu nní m mɑɑ rɛ ɑ́ri yɛ́ fe tɔkɛ́ kei kuu n símpɔ hɑ, yɑrɛ u lɑlɛ ukɛ́ umɛcirɛ kpu nɛ́ɛ? 
Yeesu uu pi mɑɑ rɛ: Nɔ̃́ nɛ mɛtene nté pikɔ́ lɛ, nɛ́ɛn kɛcɑ́ɑ́-pɔ ukɔ́. Nɔ̃́ nɛ kɛtẽ nté pikɔ́ lɛ, nɛ̃́ ɑ́m kɛtẽ nté ukɔ́. 
Lɛ̃ nnyɑ kɑm nɔ́ mɑɑ rɛ nɔnsɑ́ n ŋmurɛi rɛ nɛ́ɛ ḿpɑ́ píyei uwee, nɔ nɛ nɔ́ɑnɛ́kópɛ kpíninɛ. 
Kei kɑpi pɛɛ u pisɛ rɛ: Pɔ́ɔ mpíí úye kɛcirɛ? Yeesu uu pi pɛsɛ rɛ: Tɛ íye kɑm mpíí mɛkɛɛ-mɛ nɔ́ símɑɑri? 
Nɛ nɔ́kɛnɛ́cɑ́ɑ́ nsímɛ́ mɑ́lɛ mɛyɑ̃́ kɛ́ símisi kɛ́ kɔ nɛ nɔ́ túhɑɑnɛ. Amɑ́ mmɛ̃ kɑm uyɛɛ nɛ́ n tummɛ kɛ́mɛɛ n kṍ ŋmɑnɛ kɑm pisoi símisi. Asei kɛ uyɛ̃ uu símisi. 
Amɑ́ ɑ́pi nkpɑ́ni céri rɛ Usɑ́ɑ nsímɛ́ kuu símisi. 
Yeesu uu pɛɛ pi mɑɑ rɛ: Nɔn píyei Usoi Kɛpipi rintɑɑ́sɛ, nɔ́ pɛɛ céri rɛ nɛ́ɛ ḿpɑ́ píyei uwee. Nɔ́ kɔ ceri rɛ ɑ́m linɛ́kɔ́ cirɛ líkɑ wɑi, ɑmɑ́ lɛ̃ kɛ Sɑ́ɑ uu nɛ̃́ n céesi kɑm símisi. 
Tɔ́ nɛ uyɛɛ nɛ́ n tummɛ tɔ́ɔ kɛsẽ́ we. Lɛ̃ kuu ń lɑ ŋmɑnɛ kɑm yɛ wɑ. Lɛ̃ nnyɑ, úu nɛ́ riyɑ́ kɛ́ mɛnɛ́círɛ n we. 
Kɛ Yeesu uu lɛ̃ n símisi, pisoi mɛyɑ̃ ɑpí nɛ kɛfɑ u tɛnɛ. 
Kei kɛ Yeesu uu pɛɛ Pisuifi pɛɛ nɛ kɛfɑ u n tɛnɛ mɑɑ rɛ: Nɔn nnɛ́símɛ́ n tikilɛ̃, nɔ pinɛ́pirɛtiki yíkíyiki lɛ. 
Nɔ́ pɛɛ nsímɛ́sei ceri, nsímɛ́sei mmɛ̃ nn tíyɛsɛ ɑni nɔ́mɛnɛ́círɛ n te. 
Api u pɛsɛ rɛ: Tɔ Apirɑhɑm siyɑsɛ lɛ, ɑ́ri kɔ píkɑi unyinɛ ilɑsi tonɛlɛ̃. Yo nnyɑ kɑɑ pɛɛ rɛ: Nɔ́ kɑm ɑni nɔ́mɛnɛ́círɛ n te? 
Yeesu uu pi pɛsɛ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ usoi nkó yɛɛ ɑkópɛ n wɑi yɛ ɑkópɛ ulɑsi lɛ. 
Ulɑ́si úu yɛ mɛsɛ́rɛ kɛ́yɔ kɛ-i kɑpi u m múílɛ̃ n we. Úyɔpipi yɛɛ yɛ mɛsɛ́rɛ kɛ́yɔ-i n we. 
Lɛ̃ nnyɑ, Kɛpipi kɛn ilɑ́si kɛ́mɛɛ nɔ́ n lesɛ, nɔ mɛyíkíyiki nɔ́mɛnɛ́círɛ telɛ. 
Nɛ nyulɛ rɛ nɔ Apirɑhɑm siyɑsɛ lɛ. Amɑ́ nɔ kɔ pɛɛ́ lɑ nɔkɛ́ nɛ́ kpu rɛ ɑ́ni nnɛ́símɛ́ ŋmurɛi nnyɑ. 
Lɛ̃ kɛ nɛ́ɛ Unɛ́sɑ́ɑ kɛ́mɛɛ n yɛ̃́ nkɔ́ kɛ nɛ́ɛ símisi. Lɛ̃ kɛ nɔ́ɔ kɔ nɔ́unɛ́sɑ́ɑ kɛ́mɛɛ n kṍ kɛ nɔ́ɔ kɔ wɑ. 
Api u pɛsɛ rɛ: Apirɑhɑm yɛ urɔ́sɑ́ɑ lɛ. Yeesu uu pi mɑɑ rɛ: In tɛ nɔ Apirɑhɑm sipipi lɛ, ɑni Apirɑhɑm mɛwɑi n wɑi. 
Mɛnɛ́wɑi nnɛ́í yɛɛ rɛ kɛ́ nsímɛ́sei kɑm Uléécɑɑ kɛ́mɛɛ n kṍ nɔ́ ritɛ́ŋ́sɛ. Amɑ́ ḿpɑ́ nɛ lɛ̃, nɔ kɔ pɛɛ wɛ́ɛ́silɛ nɔkɛ́ nɛ́ kpu. Apirɑhɑm úu pɛɛ lɛ̃ wɑi! 
Amɑ́ nɔ́unɛ́sɑ́ɑ sei mɛwɑi mɛcɔ kɛ́ nɔ́ɔ wɑi. Kei kɑpi pɛɛ Yeesu pɛsɛ rɛ: Ári siwɑ́sɑ́nkɑipipi. Usɑ́ɑ usɛ kóló kɑri mɑ́. Uyɛɛ nní Uléécɑɑ. 
Yeesu uu pi mɑɑ rɛ: Uléécɑɑ un pɛɛ mɛsei Nɔ́unɛ́sɑ́ɑ, nɔ yɛɛ pɛɛ nɛ́ n lɑ rɛ ukɛmɛɛ kɑm léeri nɛn kɔ uyɛ̃ nnyɑ nté we. Ái mɛnɛ́lɑ kɛ́mɛɛ kɑḿ kɑ. Uyɛɛ nɛ́ tummɛ. 
Yo nnyɑ kɑ́ni lɛ̃ kɑm nɔ́ n símisi ɑsei kómɛi? Kɑ́ni yɛ́ n fe nɔkɛ́ nnɛ́nɔ́ɔsimɛ́ kutu rińcɔ nnyɑ yɛ ni. 
Kuníri kpɛɛ nɔ́unɛ́sɑ́ɑ. Nɔ́unɛ́sɑ́ɑ mɛlɑ kɑni kɔ lɑ nɔkɛ́ n wɑi. Hɑ́i mɛkɛɛ-mɛ kuu usoikṍ. Úu yɛ ɑsei ritiki. Li we rɛ ɑsei ɑ́ɑ ukɛmɛɛ́ we. U yɛ un nnɔ́ɔmɛ n wɑi kɑí nɛ u sɑ́. U unɔ́ɔmɛ lɛ. Uyɛɛ kɔ nnɔ́ɔmɛ nnɛ́í riyíkí. 
Kɑi nní lɛ̃ ń we nnyɑ kɑ́ni yɛ nsímɛ́sei kɛ nɛ́ɛ n símisi ŋmurɛi. 
Nɔ́ úye yɛɛ yɛ́ fe uu nɛ́ nyísɛ rɛ nɛ ɑkópɛ wɑ? In tɛ nsímɛ́sei kɑm símisi, yo nnyɑ kɑ́ni pɛɛ́ nɛ kɛfɑ nɛ́ tɛnɛ? 
Usoi nkó yɛɛ Uléécɑɑ ukɔ́, yɛ̀ɛ̀ Uléécɑɑ nsímɛ́ kutu ricɔlɛ. Kɛ nɔ̃́ ɑ́ni nní upikɔ́ nnyɑ kɑ́ni Uléécɑɑ Nsímɛ́ kutu cɔlɛ̃. 
Pisuifi ɑpi pɛɛ Yeesu pɛsɛ rɛ: Ái tɔ mɛsei ɑlɑri mɑ́lɛ tɔkɛ́ mɑɑ rɛ pɔ Sɑmɑrii ukɔ́ lɛ ɑɑ kɔ ɑníri héésii? 
Uu pi pɛsɛ rɛ: Ám ɑníri héesi, ɑmɑ́ Unɛ́sɑ́ɑ kɑm tíyu wɑisɛ, nɔ́ɔn pɛɛ tinɛ́kɔ́ kópu! 
Ám wɛ́ɛ́si kɛ́ rinɛ́cúruu ríyu wɑisɛ. Unyinɛ yɛɛ wɛ́ɛ́si ukɛ́ ríyu nɛ́ wɑisɛ. Uyɛɛ kɔ utúhɑɑnɛ. 
Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ nkó yɛɛ nnɛ́símɛ́ rimmúlú úu píkɑi kpinɛ. 
Kei kɛ Pisuifi ɑpi pɛɛ u mɑɑ rɛ: Tɔ nkpéni mɛyíkíyiki yɛ́nlɛ rɛ pɔ ɑníri héesilɛ. Apirɑhɑm yɛ kpulɛ, ɑntepuyɛ pɔ́ɔ kɔ kpíni. Pɔ́ɔ pɛɛ nkpéni rɛ nkó yɛɛ mpɔ́símɛ́ rintíki úu píkɑi kpinɛ! 
Yɑrɛ pɔ̃́ nɛ urɔ́sɑ́ɑ Apirɑhɑm yɛɛ n kpu felɛ nɛ́ɛ? Antepuyɛ pɔ̃́ nɛ kɔ kpínlɛ. Yɑrɛ pɔ́ɔ pɛɛ úye? 
Yeesu uu pɛɛ pi pɛsɛ rɛ: In tɛ nɛ́ɛ rinɛ́cúruu ríyu wɑisɛ, ɑ́m yɛ́ ríyu yɛ̃. Amɑ́ Unɛ́sɑ́ɑ uyɛ̃ kɑni ḿ mɑɑ rɛ uyɛɛ Nɔ́unɛ́léécɑɑ yɛɛ ríyu nɛ́ wɑisɛ. 
Áni kɔ pɛɛ u nyu. Amɑ́ nɛ̃́ nɛ u nyulɛ. Nɛn m mɑɑ rɛ ɑ́m u nyu, nɛ lɛ̃ wɑlɛ nɔ́mɛnɛ́cɔ unɔ́ɔmɛ. Amɑ́ nɛ̃́ nɛ u nyulɛ nɛn kɔ unsímɛ́ tíkilɛ̃. 
Nɔ́unɛ́sɑ́ɑ Apirɑhɑm yɛ pɛɛ́ nɛ mpɔ́ɔnɑrɛ cuunkɛɛlɛ rɛ u yɛ́ kɑm uu kɛnɛ́kɑ́mɛyɑɑ yɛnu. Uu kɔ mɛsei kɛ yɛnu, urikiŋ ɑri níŋɛsi hɑ́i. 
Kei kɛ Pisuifi ɑpi pɛɛ u pisɛ rɛ: Pɑ́ɑ kɑhɑnɛ nkpɑ́ni iŋmɛ̃ kuwóó n tu, pɔn pɛɛ tee rɛ pɔ Apirɑhɑm yɛnɑɑlɛ̃, nɛ́ɛ yo? 
Yeesu uu pi mɑɑ rɛ: Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ Apirɑhɑm ukɛ́ kɛlenɛ ń we, nɛ̃́ nɛ pɛɛ́ welɛ. 
Kei kɑpi pɛɛ ɑpɑrɛ kpísi rɛ pikɛ́ u tɑpisi. Amɑ́ uú nɛ pi pɔlu uu Uléécɑɑ kɛyɔ-i léepɔ uu tɔ́mpɔ.
