Yohɑni.
19.
Kei kɛ Pilɑti uu pɛɛ ncée hɛ rɛ pikɛ́ Yeesu tĩ pikɛ́ isɛ́í u súúkɛɛ. 
Pisɔ́ɔ́cɑ ɑpi inípɛ nɛ riyɔ́ɔpikɔŋ yóóni ɑpi u kipisɛ, ɑpi kɔ kukpɛ́lɛnku wɛɛ́ nyɑii kpísi ɑpi u tɑni. 
Api pɛɛ ukɛkúrí nyɔsɔ́pɔ ɑpi yɛ pɛɛ rɛ: Tɔ pɔ́ yɑ́ɑsi, Pisuifi uyɔɔpi! Apí nɛ sipɛsɛ u tɔlu. 
Pilɑti uu kɔ piléepɔ kpɑ́, uu pisoi pɛɛ n cɑ́pinɛlɛ̃ mɑɑ rɛ: Uyɛɛ nkó kɑḿ nɛ kɛtɑhɑi nɔ́ léemɛ nɔkɛ́ céri rɛ ɑ́m yɛ̃ yo kuu cɑɑi nnyɑ kɑpi yɛ́ nɛ u ń kpu. 
Yeesu uu pɛɛ léepɔ un tiyɔ́ɔpikɔŋ kɑpí nɛ inípɛ n yóõ kipilɛ̃ un kɔ kukpɛ́lɛnku wɛɛ́ nyɑii tɑnɑɑlɛ̃. Pilɑti uu pɛɛ pi mɑɑ rɛ: Usoi uyɛɛ nkó! 
Kɛ pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ nɛ pimɛ́rɛ́ ɑpi u n yɛ̃́ ɑpi cɑ́ɑ́i rɛ: A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii! A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii! Kei kɛ Pilɑti uu pɛɛ pi mɑɑ rɛ: Ani nɔ́rinɛ́cúruu u kpísi ɑní hɑ kɑrii. Nɛ̃́ ɑ́m yɛ̃ yo kuu cɑɑi nnyɑ kɑpi yɛ́ nɛ u ń kpu. 
Pisuifi ɑpi u pɛsɛ rɛ: Tɔ isé inyinɛ mɑ́lɛ yɛɛ m pisɛ rɛ ukɛ́ kpu rɛ u umɛcirɛ wɑisɛ Uléécɑɑ Kɛpipi nnyɑ. 
Kɛ Pilɑti uu ɑnɔ́ɔ nyɛ̃ n kṍ, iwɑmɛ ii mɛyɑ̃́ pilonɛ u kpɑ́. 
Uu riyɔ́ɔpipoo-i lompɔ uu Yeesu pisɛ rɛ: Pɔ yei ukɔ́ lɛ? Amɑ́ Yeesu úu rinɔ́ɔ u yɔsí. 
Pilɑti uu pɛɛ u mɑɑ rɛ: Áɑ lɑ pɔkɛ́ rinɔ́ɔ nɛ́ yɔsí, nɛ́ɛ yo? Áɑ nyu rɛ nɛ́ fe ɑm tíyɛsɛ ɑpi pɔ́ yɑ́ nɛ́ɛ ɑm tíyɛsɛ ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ pɔ́ kɑrii? 
Yeesu uu pɛɛ u pɛsɛ rɛ: Áɑ ńnɑŋɛ ńkɑ kɛnɛ́cɑ́ɑ́ mɑ́, insɑ́ mmɛ̃ kɛ Uléécɑɑ uu pɔ́ ḿ pɑ rɛ ɑ kɛnɛ́cɑ́ɑ́ m mɑ́. Lɛ̃ nnyɑ kɛ uyɛɛ ɑpɔ́nípɛ-i nɛ́ ń wɑ ɑkópɛ ɑɑ ɑpɔ́kɔ́ fe. 
Kɑí nɛ kumúŋɛ́ kpɛ-i n kpísi, Pilɑti uu pɛɛ́ n wɛ́ɛ́si ukɛ́ Yeesu riyɑ́pɔ. Amɑ́ Pisuifi ɑpi sicɑ́ɑ́ilɑ loni rɛ: Pɔn u rin-yɑ́pɔ, ɑ́ɑ pɛɛ Rɔm uyɔɔpi usɑnɛ! Usoi nkó yɛɛ umɛcirɛ n nyɑ́ni rɛ uyɔ́ɔpi yɛ Rɔm uyɔɔpi ulɑɑrɔ lɛ. 
Kɛ Pilɑti uu ɑnɔ́ɔ nyɛ̃ n kṍ, uú nɛ Yeesu kɛtɑhɑi leepɔ, uu pɛɛ́ hɑ kei kɑpi yɛ n sée rɛ ɑsɛ́hɛ́wɛɛ́pi kɛlõ riyɔ́ɔpitũ kɛcɑ́ɑ́ tonɛ. Pi yɛ kɛlõ kɛ̃ Mɛ́-epiree kɛ́mɛɛ sée rɛ Kɑpɑtɑ. 
Likɛ́ nɛ n weesi pikɛ́ nkpɔ mɛlɔ́ɔ́ ɑnyɑ̃́ n li, ituŋɛ kɛcírɛ kumúŋɛ́ kɑí pɛɛ lɛ̃ we. Pilɑti uu Pisuifi mɑɑ rɛ: Nɔ́unɛ́yɔ́ɔpi yɛ nkó! 
Amɑ́ ɑpi pɛɛ sicɑ́ɑ́ilɑ loni rɛ: A kpu! A kpu! A kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ u kɑrii! Pilɑti uu pi pisɛ rɛ: Kɛ́ nɔ́unɛ́yɔ́ɔpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii? Pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpi u pɛsɛ rɛ: Rɔm uyɔɔpi mɛmɑ́ɑ́, ɑ́ri uyɔ́ɔpi úkɑ ucɔ mɑ́. 
Uu pɛɛ Yeesu kpísi uu piɑnipɛ-i wɑi rɛ pikɛ́ hɑ u kɑrii. Api pɛɛ Yeesu kpísi ɑpí nɛ tɔ́mpɔ. 
Yeesu uu uricuruu kunɑpɛ́ɛ́kɔɔ́ kpɛ̃ kɛcɑ́ɑ́ kɑpi u n kɑriinɛ cɔni uú nɛ kuyu-i léepɔ, uú nɛ kɛlõ kɛ̃ kɑpi yɛ n sée rɛ Kúyu Koŋoo sĩ. Pi yɛ kei nɛ Mɛ́-epiree sée rɛ Kɔlikɔtɑ. 
Kei kɑpi u nɛ pisoi pitɛ́ pinyinɛ ɑnɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii, ucɔ uilúkɛ́ isɛ, ucɔ icɔ, Yeesu kunɑpɛ́ɛ́kɔɔ́ ɑku pɛɛ kɛcɔpɛ n we. 
Pilɑti uu kɔ tíyɛsɛ ɑpi kɛkpénkpélépi kɛcɑ́ɑ́ wɔ́lu ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii. Pi kɛkɛcɑ́ɑ́ wɔ́i rɛ: Nɑsɑrɛti ukɔ́ Yeesu, Pisuifi uyɔɔpi. 
Pisuifi mɛyɑ̃ ɑpi iwɔ́i iyɛ̃ kɛɛni, kɑí nɛ kei kuyu n kɔ́lɛ̃ nnyɑ. Mɛ́-epiree nɛ Mɛ́lɑtɛɛ nɛ Mɛ́kirɛki kɑpí nɛ wɔi. 
Pisuifi pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ piwɛ́ɛ́sɛ ɑpi Pilɑti mɑɑ rɛ: Pi wɔi rɛ Pisuifi Uyɔɔpi, ɑmɑ́ ɑ wɔi rɛ u mɑɑ rɛ: Nɛ Pisuifi Uyɔɔpi lɛ. 
Pilɑti uu pi pɛsɛ rɛ: Lɛ̃ kɑm n wɔ́i kɑm wɔ́i. 
Kɛ pisɔ́ɔ́cɑ ɑpi Yeesu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ pikɑrii ḿ mɑsí, ɑpi uilũ kpísi ɑpi hɔ́ɔ́nɛ ɑpi ɑfiri ɑnɑ wɑi, ḿpɑ́ usɔ́ɔ́cɑ úye uu rifiri risɛ kpísi. Api kɔ ukukpɛlɛnku wɛɛ́ nyɑii kpɛɛ ɑ́ku nnyɛlɑɑ m mɑ́ kpísi. Mɛsɑmɛ kɑpi ku yóõ kɛcɑ́ɑ́ nɛ mɛtene-pɔ. 
Api pɛɛ pimɛcɔpɛcirɛ n tee rɛ: Ári kɑpɛ ku ricɛɛ́ri. Amɑ́ tɔkɛ́ pɑŋɑi ɑrí nɛ yɛnu uyɛɛ yɛ́ ku n kpísi. Li lɛ̃ wɑlɛ likɛ́ nɛ lɛlɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ kɛcɑ́ɑ́ rintíki nnyɑ. Li wɔlɑɑlɛ̃ tɛ: Pi inɛ́lũ hɔɔnɛ ɑpi pɑŋɑi ɑpí nɛ uyɛɛ yɛ́ kunɛ́kpɛ́lɛnku wɛɛ́ nyɑii n kpísi yɛnu. Lɛ̃ kɛ pisɔ́ɔ́cɑ ɑpi n wɑ yɛ lɛ̃. 
Kunɑpɛ́ɛ́kɔɔ́ kpɛ̃ kɛcɑ́ɑ́ kɛ Yeesu uú pɛɛ n kɑriilɛ̃ kɛkúrí kɛ úni nɛ úni uwɑ̃ nɛ Kulopɑɑsi unɔsi Mɑɑri nɛ Mɑkitɑlɑ kuyu ukɔ́ Mɑɑri ɑpí pɛɛ nyɛnu. 
Yeesu uu úni yɛnu, uupirɛtiki uyɛ̃ kuú pɛɛ ń lɑ pɔ́ɔn úni kɛkúrí nyɛnu. Yeesu uu úni mɑɑ rɛ: Unɔ́si, upɔ́ŋmɑ́nɛ yɛ nɛni nkó. 
Uu kɔ uupirɛtiki uyɛ̃ mɑɑ rɛ: Upɔ́ni yɛ nɛni nkó. Kɑí nɛ kei n kpísi, uyɛ̃ uu pɛɛ Yeesu uni kpísi uu ukɛyɔ-i yekei. 
Kɛ Yeesu uu n yɛ̃́ tɛ u lɛ̃ nnɛ́í kuu piwɑi ń kɑ piwɑi mɑsilɛ, uu rɛ: Nnírɛ́ yɛ nɛ́ welɛ. Likɛ́ nɛ lɛlɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ kɛcɑ́ɑ́ rintíki nnyɑ kuu lɛ̃ mɑ. 
Kulũ kunyinɛ yɛ pɛɛ kei kɛkúrí nɛ pítɑ nyɛpɛ-nyɛpɛ mɛyipɛ yisunulɛ. Pisɔ́ɔ́cɑ ɑpi kuyúi kpɛ̃ kɑpi yɛ n sée rɛ isɔɔpi kɛkpɑ́pi kpísi, ɑpi sinɛ́ɛ̃ sinyinɛ pítɑ nyɛpɛ-nyɛpɛ pɛ̃ fɛ ɑpi kɛkɛcɑ́ɑ́ tuhu ɑpí nɛ Yeesu nnɔɔ kuwɛ́ɛ́-mɛ̃ sĩ. 
Yeesu uu pítɑ nyɛpɛ-nyɛpɛ pɛ̃ sɔɔmú uu pɛɛ rɛ: Ḿpɑ́ yo yɛ piwɑi mɑsilɛ. Kuu lɛ̃ ḿ mɑ, uu pɛɛ kpɑrɑ́, nfɑ́ɑ nn tɛnɛ. 
Likɛweesi kɛɛ kɛwéntɛyɑɑ kɛtúŋɛ́. Kɛwéntɛyɑɑ kɛ̃ nɛ kɔ pɛɛ kɛyɑ́ɑ píimɑ lɛ. Sisoikópɛ ɑ́si kɑpɛ nɛ kɛwéntɛyɑɑ kɛ̃ kɛtúŋɛ́ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ ń kɔ́lɔilɛ̃ nnyɑ, ɑpi Pilɑti pisɛ rɛ ukɛ́ tíyɛsɛ pikɛ́ ɑpɑ́í si selesi ɑpi pɛɛ si súisɛntɛ. 
Pisɔ́ɔ́cɑ ɑpi pɛɛ́ hɑpɔ ɑpi pisoi pitɛ́ pɛ̃ nɛ Yeesu kɑpi kɛsẽ́ u kɑrii ufoí ɑpɑ́í selu, ɑpi kɔ ulírũ ɑkɔ́ selu. 
Api Yeesu-i tuipɔ. Kɑpi n yɛ̃́ tɛ u pikpɔ mɑsilɛ nnyɑ, ɑ́pi pɛɛ ɑpɑ́í u sei. 
Amɑ́ pisɔ́ɔ́cɑ pɛ̃ usɛ uu ilúkɛ́-i kupɑ́ŋ́ u tuhu uu luru, mɛsɛ nɛ mɛsɛ, mɛ́nyɛ nɛ míni ɑmɛ léeri. 
Uyɛɛ mɛwɑi mɛ̃ n yɛ̃́ yɛ mɑɑ rɛ ɑsei yɛ nyɛ. Lɛ̃ kuu ḿ mɑ yɛ kɔ ɑsei lɛ. Uyɛɛ nyulɛ rɛ ɑsei kuu símisi. U símisilɛ rɛ nɔ́ɔkɛ́ kɔ nɛ kɛfɑ tɛnɛ. 
Li lɛ̃ wɑlɛ likɛ́ nɛ lɛlɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ n wɔ́lɑɑlɛ̃ tintíki nnyɑ. Lɛlɛɛ rɛ: Ápi ukukɔhɔ kúkɑ selunɛ. 
Li kɔ rikɛlõ kɛcɔ-i wɔ́lɑɑlɛ̃ tɛ: Pi yɛ́ kɑm ɑpi yɛ uyɛ̃ kɑpi ilúkɛ́ n luri kuwɛ́ɛ́-mɛ̃ n nyɑ́nɛi. 
Lɛ̃ mɛmɑ́ɑ́, Arimɑtee kuyu ukɔ́ kɑpi yɛ n sée rɛ Yosɛfi uu Pilɑti ncée welu rɛ ukɛ́ Yeesu kpísi ukɛ́ hɑ kulɛsi. Pilɑti uu ŋmurɛi. Pisuifi iwɑmɛ yɛ pɛɛ Yosɛfi uyɛ̃ welɛ, úu pɛɛ lɛ̃ nnyɑ nyísɛlɛ̃ tɛ u Yeesu upirɛtiki lɛ. Uyɛɛ hɑ uu Yeesu kpísi uú nɛ tɔ́mpɔ. 
Nikotɛm yɛɛ kɛsinɛ kɛnyinɛ Yeesu kɛ́mɛɛ ń hɑ, pɔ́ɔ kɔ tulɑɑliyɛ pɛ̃ kɑpi yɛ n sée rɛ miiri nɛ ɑlowɛɛsi mɛcɔ́hɔnɛ sikiloo ɑfɛɛtɑɑni kumúŋɛ́ nɛ hɑpɔ. 
Pikɛtɛ́ ɑpi Yeesu kpísi ɑpi sisɑ́ŋɑ́ɑ́pɑmpi pílɛsi, ɑpi tulɑɑliyɛ pɛ̃ u fom yɑrɛ kɑpi yɛ pɛɛ Pisuifi inyɛkii kɛ́mɛɛ usoi n kúlɛsi mɛcɔ. 
Kei kɑpi Yeesu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ n kɑrii kɛkúrí kɛ kɛléécɑrɛ kɛnyinɛ ɑkɛ́ pɛɛ́ we. Nhórɛ fɑlɛ nnyinɛ kɛ́mɛɛ kɑ́pi úkɑ n wɑilɛ̃ yɛ pɛɛ kɛléécɑrɛ kɛ-i welɛ. 
Likɛweesi kɛɛ Pisuifi kɛwéntɛyɑɑ. Nhórɛ nn kɔ nɛ kei kɔlɛ̃. Api lɛ̃ nnyɑ kei Yeesu wɑipɔ.
