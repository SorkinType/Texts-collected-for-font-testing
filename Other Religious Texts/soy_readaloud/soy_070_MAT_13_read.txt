Mɑtiyee.
13.
Yeesu uu kɛyɑ́ɑ kɛ̃ kɛ́yɔ-i léeri uu sĩ́ uú hɑ kupiyɛ ritimɛ tonɛ. 
Pisoi kulúi ɑpi ukɛkúrí cɑ́pinɛpɔ uu pɛɛ lɛ̃ nnyɑ kúninɔi loni uú tonɛ. Pisoi pɛ̃ pɔ́ɔ kupiyɛ ritimɛ tonɛ. 
Uu sinyɑ́rũ kpísi uú nɛ isímɛ́ isímɛ́ pi símɑɑnkɛɛ. Uu rɛ: Utisi unyinɛ yɛɛ we, uu kɛyɑ́ɑ ukɛcɑrɛ-i ipuri pifómɛ leepɔ. 
Kuu lɛ̃ n fóm, ipuri inyinɛ ii ricéetimɛ lólɛsi, sinúipi ɑsí hɑpɔ ɑsi i kpóuli. 
Icɔ ii kɛpɑrɛtẽ kɛcɑ́ɑ́ kɛ ntɑɑi ńn m piyɛ lólɛsi. Kɛ kɛtẽ ɑ́kɛ n címú nnyɑ, ipuri iyɛ̃ ii mɛsɛ nɛ mɛsɛ lɛlu. 
Amɑ́ kɛ ntuŋɛ nn n tɑ, mɛlɛi fɑlɛ mɛ̃ ɑmɛ iníŋí kuweecirɛ́ nnyɑ torɛ. 
Ipuri icɔ ii sinípɛleépi kɛpɑ́ɑ́pɑ́ɑ́ lólɛsi, sinípɛleépi ɑsi pelu ɑsi i hilɑ. 
Amɑ́ ipuri icɔ ií hɑ kɛtẽ sɔnɛ kɛcɑ́ɑ́ lólɛsi ii lɛlu ii pelu ii fólu. Mpuri nsɛ nn fólu ipipi pílɛ, ncɔ ipipi kuwóó nɛ kɛfi, ncɔ ipipi ɑfɛɛtɑɑni. 
Kɛ Yeesu uu lɛ̃ pimɑ́ɑ pi m mɑsí, uu kɔ pi kpɑ́ rɛ: Úye un ukómɛ uu kóm. 
Kei kɛ upipirɛtiki ɑpi pɛɛ́ nɛ u rikɔ ɑpi u pisɛ rɛ: Yo nnyɑ kɑɑ sinyɑ́rũ kpísi pɔn nɛ pisoi símisi? 
Yeesu uu pi pɛsɛ rɛ: Nɔ̃́ kɛ Uléécɑɑ uu uiyɔɔpi nsímɛ́ mɛɛ m pékɑɑlɛ̃ cereisɛ, úu pɛ̃ cereisɛ. Lɛ̃ nnyɑ kɑḿ nɛ lɛ̃ pi símisi. 
Li we yɑrɛ kɑpi yɛ ḿ mɑɑ rɛ: Nkó yɛɛ linyinɛ m mɑ́ kɑpi yɛ licɔ rikpɑ́ ɑi mɛyɑ̃́ wɑi. Amɑ́ nkó yɛɛ úu líkɑ m mɑ́ kɑpi yɛ likɑ́ripikɔɔ́ lɛ̃ kuu m mɑ́ yɔsí. 
Lɛ̃ nnyɑ kɑm sinyɑ́rũ kpísi nɛn nɛ pi símisi. Li we rɛ pi yɛ n wiilɛnlɛ, ɑmɑ́ ɑ́pi pɛɛ nyɑ́ni, ɑpi yɛ kutu n cɔlɛ̃, ɑmɑ́ ɑ́pi pɛɛ kɔ kómɛi. Ápi kɔ ɑsei ceri. 
Lɛ̃ kɛ ɑntepu Esɑyi uú pɛɛ mɛkɛɛ-mɛ ḿ mɑ yɛɛ lɛ̃ wɑi rɛ: Nɔ́ kɑm ɑni yɛ n kómɛi, ɑ́ni yɛ́ɛ́ pɛɛ liɑsei ceri. Nɔ́ kɑm ɑni yɛ n nyɑ́nɛi, ɑ́ni yɛ́ɛ́ pɛɛ yɛ̃́. 
Li we rɛ mpuri mmú pisoi ɑkiŋ yɛ tɑkɑsilɛ. Pi pimɛcirɛ ɑtu tɑ́pisilɛ. Api inípɛɛ wúkɛsi, ɑpi yúlu rɛ piɑtu ɑ́ɑ kɑpɛ yɛ kõ, piinipɛɛ íi kɑpɛ yɛ yɛ̃ pimɛsɔhɔ ɑ́mɛ kɑpɛ yɛ kɔ líkɑ ceri. Lɛ̃ kɛ́mɛɛ, ɑ́pi yɛ́ pɛɛ fe pikɛ́ pimɛfinɛ consɛ. Nɛ̃́ Uléécɑɑ, kɛ́ nɛ pi m pɔisɛ. 
Amɑ́ in nɔ́ɔ, nɔ pínɑrɛkomɛ lɛ rɛ nɔ́inɛ́nípɛɛ yɛ nyɑ́ni, nɔ́ɑnɛ́tu ɑn kómɛi. 
Asei kɑm nní nɔ́ símisi rɛ ɑntepuyɛ nɛ ɑsei pite Uléécɑɑ inipɛɛ-i yɛ kulúi welɛ pɛɛ pɛɛ́ ń lɑ pikɛ́ lɛ̃ kɑni nní ń nyɑ́ni yɛ̃, ɑmɑ́ ɑ́pi li yɛ̃. Pi pɛɛ kɔ lɑlɛ pikɛ́ lɛ̃ kɑni nní nɛ nɔ́ɑnɛ́tu n kómɛi kõ, ɑmɑ́ ɑ́pi li kõ. 
Kei kɛ Yeesu uu pɛɛ pi mɑɑ rɛ: Ani nkpéni ulukɑɑ kɛnyɑ́rũ kɛ̃ ɑsei kõ: 
Pisoi pɛɛ yɛ Uléécɑɑ iyɔɔpi nsímɛ́ kutu rińcɔ ɑ́pi yɛ́ nkɛmɛɛ́ kṍ yɛ we yɑrɛ ricéetimɛ tɛ-i kɛ ipuri ii n lólɛsi. Kuníri yɛ̀ɛ̀ kɑlɛ ɑku nsímɛ́ mmɛ̃ pikɛmɛɛ́ lesɛ. 
Picɔ yɛ kɔ we yɑrɛ kɛpɑrɛtẽ kɛ-i kɛ ipuri ii n lólɛsi. Lipite yɛ̀ɛ̀ nsímɛ́ kutu ricɔlɛ ɑpi mɛsɛ nɛ mɛsɛ piɑkiŋ kɛ́mɛɛ n yɔ́su nɛ mpɔ́ɔnɑrɛ. 
Amɑ́ ɑ́pi yɛ tíyɛ nkɛ́ pikɛmɛɛ iníŋí wɑ, ńn yɛ pikɛmɛɛ tónɛ likɛ́ nɑ́ŋɑi. Ncɔ́ŋ nn nsímɛ́ mmɛ̃ nnyɑ pi n lɛɛmɛ nɛ́ɛ mɛ́wee kópɛ mɛnyinɛ mɛn pi m pɛkɛsi, ɑpi kei nɛ kɛwɑ́ɑ́nɑ́ yɛ́nu ɑpí nɛ Uléécɑɑ kɛpirɛ tɔ. 
Picɔ yɛ kɔ we yɑrɛ ipuri yɛɛ sinípɛleépi kɛpɑ́ɑ́pɑ́ɑ́ ń loó. Lipite yɛ̀ɛ̀ nsímɛ́ kutu ricɔlɛ, ɑmɑ́ kɛtẽ kɛcɑ́ɑ́ pikɛi kɛ pisoi ɑpí nɛ ń kpɑrɑ́lɛ̃ nɛ mɛmɑ́ nnɑ́ɑ́pí mɛɛ yɛ n kírɑɑsɛ yɛ̀ɛ̀ nsímɛ́ mmɛ̃ tihilɑlɛ ńn yɛ pɛɛ líkɑ pi yoriyɛ. 
Picɔ yɛ kɔ welɛ yɑrɛ ipuri yɛɛ kɛfɛ́nɛtẽ kɛcɑ́ɑ́ ń loó. Lipite yɛ̀ɛ̀ nsímɛ́ mmɛ̃ kutu ricɔlɛ ɑpi nkɛmɛɛ́ kóm nn pi lɛ́ni ɑpi pɛɛ́ n we yɑrɛ ɑléé nyɛɛ yɛ ɑpipi m mɑ́ri, kunyinɛ pílɛ, kucɔ kuwóó nɛ kɛfi, kucɔ ɑfɛɛtɑɑni. 
Yeesu uu kɔ kɛnyɑ́rũ kɛcɔ pi kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ we yɑrɛ usoi nkó nsímɛ́ mmú: Utisi unyinɛ yɛɛ ukɛcɑrɛ-i mɛlukɛpipi lukɛsi. 
Ulɑ́ɑrɔ uu kɛsinɛ kɛnyinɛ hɑpɔ, pisoi pin lɔni uu ilukɛ kɛpɑ́ɑ́pɑ́ɑ́ mɛyúipuri kópɛ uu yɑɑ, uu tɔ́mpɔ. 
Kɛ ilukɛ ii n lɛ ii mɛpɛ́i leeri ii fólu, mɛyúipuri pɔ́ɔ kɔ leeri. 
Pikɛikɔ́ ɑpí weri ɑpi ucɑrɛte uyɛ̃ mɑɑ rɛ: Ái ilukɛpuri kɑɑ kɛpɔ́cɑrɛ-i lukɛsii, Upíimɑ? Kɑi pɛɛ íye wɑ kɛ mɛyúipuri kópɛ ɑmɛ́ nɛ n lɛ? 
Uu pi pɛsɛ rɛ: Unɛ́lɑ́ɑrɔ unyinɛ yɛɛ lɛ̃ wɑ. Pikɛikɔ́ ɑpi rɛ: Pɔ lɑ tɔkɛ́ hɑ mɛ wɛhɛɛ? 
Uu rɛ: Eehei. Nɛ wuru rɛ nɔ́ mɛyúipuri kópɛ pilésɛ n lɑ ɑní nɛ ilukɛ pɛ́nɛ ɑní wɛhɛ. 
Lɛ̃ nnyɑ, ɑni likɛtɛ́ riyɑ́ likɛ́ kɛsẽ́ m pelu ɑí nɛ kúkpɑɑ tulu. Lin kúkpɑɑ n tu, nɛ́ pɛpɛɛ ń kpɑsi mɑɑ rɛ: Ani mɛyúipuri kópɛ mɛfoí lesɛ ɑni ɑpehe mɛ pɑhɑɑnkɛɛ tɔkɛ́ mɛ risɛ́, ɑni pɛɛ kɔ ilukɛ cɑ́pinɛ ɑni rinɛ́símɛ-i súúni. 
Yeesu uu kɔ kɛnyɑ́rũ pi kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ mpuri yukɛ-yukɛ nnyinɛ kɛ usoi uu n kpísi uu ukɛcɑrɛ-i lukɛsi. 
Ipuri kɛcɔpɛ mmɛ mɛɛ ń-yukɛ icɔ fe. Amɑ́ nn n lɛ, nkuléé kpɛɛ yɛ kɛcɑrɛ-i ɑyɛ́hɛléé kɛcɔpɛ kumɑ́ɑ. Ai pɛɛ wɑi kuléé mɑɑ kpɛ̃ ilesɛ kɛcɑ́ɑ́ kɛ sinúipi ɑsi yɛ ɑyɑ́mɛ́ wɑ. 
Yeesu uu kɔ kɛnyɑ́rũ pi wɑi rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ nkpɔ́nɔ́tɔ pitɛ kɛ unɔ́si uu ntɔ píimɑ kɛ́mɛɛ n cɔ́hɔnɛpɔ uu pulu pikpɛ́ɛ ɑpi yisi. 
Sinyɑ́rũ kɛ Yeesu uu yɛ pɛɛ kpísi uú nɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ n céési. Sinyɑ́rũ ŋmɑnɛ kuu yɛ pɛɛ kpísi uú nɛ pi céési. 
Lɛ̃ kɛ ɑntepu uú pɛɛ n wɔ́i likɛ́ nɛ n wɑ nnyɑ. U pɛɛ rɛ: Sinyɑ́rũ kɑm kpísinɛ kɛ́ nɛ n símisi. Nɛ́ sinyɑ́rũ kpísi ɑḿ nɛ lɛ̃ nnɛ́í lɛɛ ḿpɑ́ yo kɛkorɑɑnɛ-mɛ m pékɑɑlɛ̃ yóó. 
Kei kɛ Yeesu uu pɛɛ yisi uu pisoi pɛɛ n cɑ́pinɛlɛ̃ yɑ́ uu kɛ́yɔ sĩ. Upipirɛtiki ɑpi u lɛɛpɔ ɑpi u pisɛ rɛ: A mɛyúipuri kópɛ mɛɛ kɛcɑrɛ-i n lɛ kɛnyɑ́rũ kɛ̃ ɑsei rɔ́ símisi. 
Uu pi pɛsɛ rɛ: Uyɛɛ ilukɛpuri n lukɛsi yɛɛ nní Usoi Kɛpipi. 
Kɛcɑrɛ kɛɛ nní kɛtẽ, nlukɛpuri sɔnɛ mɛɛ nní Uléécɑɑ iyɔɔpi pikɔ́, pɛpɛɛ Setɑni n tíkilɛ̃ pɛɛ nní mɛyúipuri kópɛ. 
Ulɑ́ɑrɔ uyɛɛ mɛyúipuri kópɛ n lukɛsi yɛɛ nní Setɑni. Kɛtẽ nkɛ́ mɛtɛnɛ yɛɛ nní kúkpɑɑ. Pɛpɛɛ ń kpɑsinɛ pɛɛ nní piléécɑɑtumɛ. 
Yɑrɛ kɑpi yɛ nní mɛyúipuri kópɛ n wɛ́hɛ ɑpi nnɑ sɛ́ mɛcɔ kɑi sɔ́nti likɛ́ kɛtẽ mɛtɛnɛ kɛtúŋɛ́ wɑ. 
Nɛ̃́, Usoi Kɛpipi yɛ́ pinɛ́léécɑɑtumɛ tũ ɑpi pɛ̃ nnɛ́í pɛɛ inɛ́yɔ́ɔpi kɛ́mɛɛ picɔ ɑkópɛ n wɑisɛ nɛ pɛ̃ nnɛ́í pɛɛ mɛwɑi kópɛ n wɑi lesɛ. 
Api nnɑ píimɑ kɛ́mɛɛ pi sɑ́pɔ. Kei kɛ pisoi ɑpi yɛ́ mɛyɑ̃́ ténesi pin inĩ́ tɑkɑi. 
Pɛɛ́ uyɛ-i kɛ pɛpɛɛ ɑsei n tíkilɛ̃ pɔ́ɔ yɛ́ pɛɛ piusɑɑ Uléécɑɑ iyɔɔpi kɛ́mɛɛ n we pin tɛ́lu yɑrɛ ituŋɛ. Úye un ukómɛ uu kóm! 
Yeesu uu kɔ upipirɛtiki kɛnyɑ́rũ kɛcɔ kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ nsímɛ́ mmú: Mɛmɑ́ píimɑ mɛnyinɛ mɛɛ kɛcɑrɛ kɛnyinɛ-i kúlɑɑlɛ̃. Utisi unyinɛ uu kɛyɑ́ɑ mɛmɑ́ píimɑ mɛ̃ yɛnu, uu pikúlɑɑ mɛ pɛsɛ, uu kɔ mpɔ́ɔnɑrɛ nɛ ulikɔ́ nnɛ́í kpísi uu yɑ́i uú nɛ kɛcɑrɛ kɛ̃ lɔlu. 
Yeesu uu kɔ kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ ukpéénsɛ unyinɛ yɛɛ kɛyɑ́ɑ ikólɛ́ kɛcirɛ wɛ́ɛ́si. 
Kuú hɑ nkólɛ́ mɛɛ mɛyíkíyiki ń nyɑḿ n yɛ̃́, uu sĩ́ uú hɑ ulikɔ́ nnɛ́í yɑ́i uú nɛ n lɔlu. 
Uu kɔ kpɑ́ rɛ: Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ kunyɛŋ́ kunyinɛ kɛ usoi uu mínimɑɑ-i n fṍ ɑku ikpíntomɛ́ nnyí ncɔpuri pɑɑ́lɑ. 
Kɑku n yi, pikpíntomɛ́cɔ́pii ɑpi ku pɑsɑi ɑpi ritimɛ lesɛri ɑpi pɛɛ́ n tũ pin ikpíntomɛ́ tɛ́si, pin isɔnɛ ɑnɛ́rɛ-i wɑi pin ikópɛ fóŋɛɛ. 
Lɛ̃ kɑi sɔ́nti likɛ́ kɛtẽ mɛtɛnɛ kumúŋɛ́-i wɑ. Piléécɑɑtumɛ yɛ hɑ sɔ́ntilɛ pikɛ́ pɛpɛɛ Uléécɑɑ mɛlɑ n wɑ nɛ pɛpɛɛ ɑ́pi Umɛlɑ n wɑ kóólɛnɛ, 
ɑpi pɛpɛɛ ɑ́pi Umɛlɑ n wɑ nnɑ píimɑ kɛ́mɛɛ sɑ́pɔ. Kei kɛ pisoi ɑpi yɛ́ mɛyɑ̃́ ténesi pin inĩ́ tɑkɑi. 
Yeesu uu upipirɛtiki pisɛ rɛ: Nɔ pɛɛ lɛ̃ nnɛ́í kɑm nní n símisi ɑsei konlɛ nníí? Api rɛ: Ɛɛɛ, tɔ konlɛ. 
Kei kuu pɛɛ pi mɑɑ rɛ: Lɛ̃ nnyɑ, isé ucélɑɑ úye un Uléécɑɑ iyɔɔpi nkɔ́ n ŋmurɛi, u we yɑrɛ úyɔsɑɑ unyinɛ yɛɛ ulikɔ́ kɛ́mɛɛ lifɑ́lɛ nɛ likpurɛ ń lesɛ. 
Kɛ Yeesu uu lɛ̃ nɛ sinyɑ́rũ sɛ̃ picélɑɑ pi m mɑsí, uu kei yisi uu tɔ́mpɔ, 
uu kuyu kpɛ-i kuu iwɑ̃́ n li kɛ́mɛɛ sĩ́ uú hɑ kei Pisuifi kuyomɛyɑ́hɑɑlee-i n céési. Ai pɛpɛɛ kei ń we nnɛ́í nnɔ́ɔ yipu, ɑpi yɛ pɛɛ rɛ: Yei kɛcirɛ kɛ nkó uu mpíí mɛwɛ́ɛ́sɛsɔhɔ mmɛ́ tu? Íye kuu yɛ nɛ mɛwɑisɑŋɑ mmɛ́ wɑpisi? 
Nɛ́ntɛ uyɛɛ usɛ́rɛ́ uyɛ̃ uŋmɑ́nɛ? Ái úni yɛɛ Mɑɑri? Ái upiwɑ̃ pɛɛ nní Yɑkupu nɛ Yosɛfi nɛ Simɔɔ nɛ Yuutii? 
Ái tɔ́ nɛ upiyɛi nnɛ́í tɔ́ɔ nté kɛsẽ́ wee? Yei kɛcirɛ kuu pɛɛ mpíí ńnɑŋɛ mmú yɛ̃́? 
Nní nnɛ́í ɑi pi tɑpisi ɑ́pi nɛ Yeesu kɛfɑ tɛnɛ. Yeesu uu pɛɛ pi mɑɑ rɛ: Ápi yɛ ɑntepu kuyu-i nɛ ukɛyɔ-i u n wɑisɛlɛ̃. 
Pinfɑtɛnɛ kucírɛ́ nnyɑ, Yeesu úu kei mɛwɑisɑŋɑ mɛyɑ̃́ wɑpisi.
