Yohɑni.
14.
Yeesu uu kɔ upipirɛtiki pimɑ́ɑ kpɑ́ rɛ: Nɔ́rinɛ́pɔ́ɔ ɑ́ri kɑpɛ n cɑɑilɛ̃. Ani Uléécɑɑ nɛ kɛfɑ tɛnɛ ɑni kɔ nɛ̃́ nɛ kɛfɑ tɛnɛ. 
Sitónɛ yɛ Unɛ́sɑ́ɑ kɛyɔ-i kulúi welɛ. Nɛ sinlɛ kɛ́ hɑ kɛtónɛ nɔ́ nyɔ́ɔnsɛ. Insɑ́ pɛɛ lɛ̃, ɑ́m yɛ́ pɛɛ nɔ́ símisi rɛ nɛ sinlɛ kɛ́ hɑ kɛtónɛ nɔ́ nyɔ́ɔnsɛ. 
Nɛn n hɑ kɛtónɛ pinyɔ́ɔnsɛ nɔ́ m mɑsí, nɛ pɛɛrilɛ kɛ́ nɔ́ kpísi kɛ́ nɛ ripɛ́nɛ, nɔ́ɔkɛ́ kɔ nɛ kei kɑm ń we n we nnyɑ. 
Nɔ kɔ kei kɑm n sĩ́ ncee nyulɛ. 
Tomɑɑ uu pɛɛ u mɑɑ rɛ: Upíimɑ, ɑ́ri nyu kei kɑɑ n sĩ́, íye kɑri yɛ́ wɑ tɔkɛ́ nɛ lincee n nyu? 
Yeesu uu u pɛsɛ rɛ: Nɛ́ɛ ncée, nɛ́ɛ nsímɛ́sei, nɛ́ɛ nfɑ́ɑ. Úkɑ úu we yɛɛ yɛ́ Sɑ́ɑ Uléécɑɑ-i ń hɑ kɑ́i nɛ̃́ kuú nɛ ritiki. 
Nɔn pɛɛ nɛ́ n nyu, nɔ́ pɛɛ kɔ Unɛ́sɑ́ɑ n nyu. Kɑí nɛ nɛni n kpísi, nɔ u nyulɛ ɑni kɔ u yɛnu. 
Filipu pɔ́ɔ pɛɛ rɛ: Upíimɑ, pɔ kpére Sɑ́ɑ rɔ́ nyísɛ, lɛɛ nɛ rɔ́ piyɛ. 
Yeesu uu u mɑɑ rɛ: Filipu, hɑ́i ɑ́i nɛni-mɛ kɛ tɔ́ nɛ nnɔ̃́ ɑrí we, ɑ́ɑ kɔ pɛɛ nɛ́ nyuu? Nkó yɛɛ nɛ́ n yɛ̃́ yɛ Unɛ́sɑ́ɑ yɛnlɛ. Íye kɑi wɑ kɑɑ pɛɛ́ nɛ mɑɑ rɛ kɛ́ Sɑ́ɑ nɔ́ nyísɛ? 
Áɑ nɛ kɛfɑ tɛnɛ rɛ tɔ́ nɛ Unɛ́sɑ́ɑ yɛ pɛ́nɛlɛnlɛ, nɛn ukɛmɛɛ we uyɛ̃ un kɔ kɛnɛ́mɛɛ we nɛ́ɛ yo? Nsímɛ́ mmú kɑm nní kɔ n símisi ńn kɛnɛ́mɛɛ léeri. Amɑ́ Sɑ́ɑ yɛɛ kɛnɛ́mɛɛ ń we yɛɛ umɛwɑi wɑi. 
Ani nɛ kɛfɑ tɛnɛ rɛ nɛ Unɛ́sɑ́ɑ kɛ́mɛɛ welɛ, Unɛ́sɑ́ɑ pɔ́ɔn kɔ kɛnɛ́mɛɛ we. Ḿpɑ́ nɔnsɑ́ nɛ lɛ̃ kɛfɑ n tɛnɛnɛ, ɑni nɛ lɛ̃ kɑm n wɑi nnyɑ kɛfɑ nɛ́ tɛnɛ. 
Asei kɛcɑ́ɑ́, kɛ́ nɔ́ símisi rɛ nkó yɛɛ nɛ kɛfɑ nɛ́ n tɛnɛ pɔ̃́ nɛ́ kɔ mɛnɛ́wɑi mɛcɔ wɑ. Ticuruu u yɛ́ mɛnɛ́kɔ́ mɛtɔ́ŋɛ́-tɔŋɛ́ wɑ rɛ Unɛ́sɑ́ɑ kɛ́mɛɛ kɑm tɔ́su nnyɑ. 
Lɛ̃ nnɛ́í kɑni yɛ́ nɛ rinɛ́nyíri m pisɛ, nɛ li wɑinɛ Sɑ́ɑ ukɛ́ nɛ Kɛpipi nnyɑ ríyu yɛ̃. 
Nɔn yɛ yo mpuri nɛ rinɛ́nyíri m pisɛ, nɛ́ nɔ́ li pɑ. 
Yeesu uu kɔ pimɑ́ɑ pi kpɑ́ rɛ: Nɔn nɛ́ n lɑ, nɔ́ yɛ inɛ́sé ritiki. 
Nɛ̃́ nɛ́ pɛɛ Sɑ́ɑ we rɛ ukɛ́ Ucomɛ nɔ́ pɑ ukɛ́ mɛsɛ́rɛ nɔ́kɛnɛ́mɛɛ n we. 
Uyɛɛ nní nsímɛ́sei Nfɑɑsɔnɛ. Kɛtẽ nkɛ́ pisoi ɑ́pi yɛ́ fe pikɛ́ n yɔsí. Ápi n nyɑ́ni, ɑ́pi kɔ n nyu. Amɑ́ nɔ̃́ nɛ n nyulɛ rɛ n nɔ́kɛnɛ́kúrí we. Nɔ́ nɛ mmɛ̃ nɛ́ ripɛ́nɛ nń nɛ nɔ́kɛnɛ́mɛɛ n we nnyɑ. 
Ám nɔ́mɛnɛ́círɛ yɑ́nɛ yɑrɛ sikúmɑmpipi. Nɛ nɔ́kɛnɛ́mɛɛ pɛɛrilɛ. 
Li tisɛ nkɑ́ripi, kɛtẽ nkɛ́ pisoi ɑ́pi yɛ́ pɛɛ piyɛ́nɛ nɛ́ rikpɑ́. Amɑ́ nɔ̃́ nɛ́ nɛ́ yɛ̃́ tɛ nɛ nfɑ́ɑ mɑ́lɛ nnyɑ. Nɔ̃́ nɛ kɔ sɔ́ntilɛ nɔkɛ́ nfɑ́ɑ m mɑ́ nnyɑ. 
Likɛyɑɑ kɛ̃ kɑni yɛ́ céri rɛ tɔ́ nɛ Unɛ́sɑ́ɑ yɛ pɛ́nɛlɛnlɛ, nɛn ukɛmɛɛ we, nɔ́ɔn kɛnɛ́mɛɛ we, nɛ́ɛn kɔ nɔ́kɛnɛ́mɛɛ we. 
Nkó yɛɛ inɛ́sé ḿ múílɛ̃ un i tikilɛ̃ yɛɛ nɛ́ lɑ. Unɛ́sɑ́ɑ yɛ́ kɔ uyɛɛ nɛ́ n lɑ n lɑ. Nɛ̃́ nɛ́ kɔ u n lɑ ɑm kɔ mɛnɛ́círɛ u nyísɛ. 
Yuuti, ɑ́i Yutɑsi Isikɑriyɔɔti, uu u mɑɑ rɛ: Upíimɑ, íye kɑɑ yɛ́ mɛpɔ́círɛ rɔ́ nyísɛ ɑ́ɑ yɛ́ pɛɛ kɛtẽ nkɛ́ pisoi mɛpɔ́círɛ nyísɛ? 
Yeesu uu u pɛsɛ rɛ: Úye un nɛ́ n lɑ, u yɛ́ nnɛ́símɛ́ rimúlú. Unɛ́sɑ́ɑ yɛ́ u n lɑ. Tɔ́ nɛ Unɛ́sɑ́ɑ ɑri ukɛkúrí sĩ́, ɑrí hɑ ukɛmɛɛ́ n we. 
Nkó yɛɛ úu nɛ́ ń lɑ úu yɛ nnɛ́símɛ́ rimúlú. Nsímɛ́ mmú kɑm nní nɔ́ n símisi ńn kɔ pɛɛ kɛnɛ́mɛɛ léeri, ɑmɑ́ Sɑ́ɑ yɛɛ nɛ́ n tummɛ kɛ́mɛɛ le. 
Kumúŋɛ́ nkú-i kɛ tɔ́ nɛ nnɔ̃́ ɑri nkpéni nní ń we kɛ́mɛɛ kɑm lɛ̃ nɔ́ símisi. 
Nɔ́unɛ́comɛ yɛɛ nní Nfɑ́ɑsɔnɛ kɛ Sɑ́ɑ Uléécɑɑ uú nɛ rinɛ́nyíri m pɑ́mɛnɛ. Mmɛ mɛɛ ḿpɑ́ yo nɔ́ céésinɛ nn kɔ lɛ̃ nnɛ́í kɑm nɔ́ n símisi nɔ́ léisɛ. 
Nɛ nkpéni nkíŋniŋɛ nɛ nɔ́ riyɑ́. Nnɛ́kíŋniŋɛ kɑm nɔ́ n hɛ. Ái sisoipipi mɛhɛɛ mɛcɔ kɑm nɔ́ hɛ. Nɔ́rinɛ́pɔ́ɔ ɑ́ri kɑpɛ cɑɑi, iwɑmɛ íi kɑpɛ kɔ nɔ́ n we. 
Yɑrɛ kɑm nɔ́ ḿ mɑɑ rɛ: Nɛ́ nɔ́ riyɑ́ ɑḿ hɑ kɔ nɔ́kɛnɛ́mɛɛ pɛɛri. Nɔn nɛ́ n lɑ, kɑm nní Unɛ́sɑ́ɑ kɛ́mɛɛ n sĩ́, li yɛ́ pɛɛ nɔ́ n lɑ́ɑ́rúlɛ rɛ nɛ ukɛmɛɛ sĩ́ nnyɑ. Unɛ́sɑ́ɑ yɛ nɛ́ felɛ. 
Nɛ nsímɛ́ mmú nnɛ́í nɔ́ símisilɛ nkɛ́ kɛlenɛ kɑm tɛ nn píyei n kɑ, nɔkɛ́ nɛ kɛfɑ n tɛnɛ nnyɑ. 
Ám nkpéni nɛ mɛyɑ̃́ pisímɛ́ nɔ́ kpɑ́nɛ. Kɛtẽ nkɛ́ uyɔɔpi yɛ wemɛlɛ. Úu ńnɑŋɛ ńkɑ kɛnɛ́cɑ́ɑ́ mɑ́. 
Amɑ́ kɛtẽ nkɛ́ pisoi pikɛ́ nɛ n céri rɛ nɛ Sɑ́ɑ lɑlɛ nnyɑ kɑm yɛ lɛ̃ kuu yɛ nɛ́ ḿ mɑ wɑ. Ani nté yisi tɔkɛ́ ncée kpísi.
