Mɑtiyee.
22.
Yeesu uu kɔ kɛnyɑ́rũ kɛcɔ pi kpɑ́ rɛ: 
Uléécɑɑ iyɔɔpi yɛ kɔ we yɑrɛ nsímɛ́ mmú: Uyɔ́ɔpi unyinɛ yɛɛ kɛyɑ́ɑ uuŋmɑ́nɛ yɛɛ unɔ́si n kpísi ɑnyɑ̃́ píimɑ wɑi. 
Uu upikɛikɔ́ tum tɛ pikɛ́ hɑ pisoi séleimɛ pikɛ́ ɑnyɑ̃́ ilukɛ li. Amɑ́ pisoi ɑpi isɑ́nɛséi iyɛ̃ yulu ɑpi ɑnyɑ̃́ wéntɛ. 
Uu kɔ pikɛikɔ́ picɔ pitumɛ kpɑ́ uu pi mɑɑ rɛ: Ani pisɑ́nɛ sée ɑni pi símisi rɛ: Anyɑ̃́ ilukɛ yɛ pipele mɑsilɛ. Anɛ́nɑ́ɑlɑ nɛ inɛ́sɛɛ kpɔkɔɔ́ kɑm kṍ. Ḿpɑ́ yo yɛ kɛ́yu wɑlɛ. Ani kɑm ɑni pinɔ́sikpikɛ́ ɑnyɑ̃́ ilukɛ li. 
Ápi nkpɑ́ni isɑ́nɛséi iyɛ̃ kutu ricɔ. Ḿpɑ́ úye uu ulikɔ́-mɛ̃ tɔ́mpɔ. Picɔ ɑpi sicɑrɛ wɑikɛɛ, picɔ ɑpi ɑyɑ́ɑ tɔ́ŋɑɑpɔ 
pitɔ́rɔɔ pɔ́ɔ pikɛikɔ́ pɛ̃ cɔ́pii ɑpi isɛi pi wɑisɛ ɑpi pɛɛ pi kóni. 
Ai uyɔ́ɔpi kuwɔ́i wɑi, uu upisɔ́ɔ́cɑ tum ɑpí hɑ pisoi pɛ̃ kóni ɑpi pikuyu nnɑ ŋmɑɑsi. 
Uu pɛɛ upikɛikɔ́ picɔ mɑɑ rɛ: Anyɑ̃́ ilukɛ yɛ pipele mɑsilɛ, ɑmɑ́ pisɑ́nɛ ɑ́pi pɛɛ ikulukɛ tulɑɑlɛ̃. 
Lɛ̃ nnyɑ, ɑni sicéepɛnɛ-i léepɔ, nɔn píye n yɛ̃, ɑni kusɑ́nɛ pi séiri. 
Upikɛikɔ́ pɛ̃ ɑpi yisi ɑpi icée tóói ɑpi pisoi nnɛ́í kɑpi n yɛ̃́ sélei, pisɔnɛ nɛ pikópɛ, pisɑ́nɛ ɑpi pɛɛ kɛnyɑ̃́yɔ yipu. 
Kei kɛ uyɔ́ɔpi uu pɛɛ lompɔ rɛ ukɛ́ pisɑ́nɛ pɛ̃ yɑ́ɑ́si, uu usoi unyinɛ yɛnu yɛɛ úu inyɑ́nlũ n tɑnɑɑlɛ̃. 
Uu u pisɛ rɛ: Unɛ́sɑ́nɛ, íye kɑɑ wɑ ɑɑ́ nɛ nté lonti ɑ́ɑ kɔ inyɑ́nlũ tɑnɑɑlɛ̃? Ai utisi muuni uú n sɛ́ɛ́lɛ̃. 
Uyɔ́ɔpi uu pɛɛ pikɛikɔ́ mɑɑ rɛ: Ani ɑ́nɑ nɛ ɑnípɛ u pɑɑsi ɑni kɛtɑhɑi kuŋmɑhɑ-i u sɑ́pɔ. Kei kuu yɛ́ n téni un inĩ́ tɑkɑi. 
Yeesu uu pɛɛ rɛ: Li we rɛ pisoi kulúi kɑpi sée, ɑmɑ́ pɛ̃ kɑpi pikɛcɔpɛ n wɛ́ɛ ɑ́pi pɛɛ kulúi we. 
Kei kɛ Pifɑrisi ɑpi pɛɛ símɑɑnɛ rɛ íye kɑpi yɛ́ nɛ Yeesu ripinɛ wɑ pikɛ́ nɛ unnɔɔsimɛ́ cirɛ u n tĩ́. 
Api pipirɛtiki pinyinɛ nɛ Erooti pipirɛtiki pinyinɛ lesɛ ɑpi tum tɛ pikɛ́ hɑ Yeesu mɑɑ rɛ: Upíimɑ, tɔ nyulɛ rɛ ɑsei ŋmɑnɛ kɑɑ yɛ símisi. Pɔ yɛ kɔ Uléécɑɑ ncee pisoi céesilɛ nɛ ɑsei. Ḿpɑ́ usoi ukɛ́ íye nsɑ́ ńnɑŋɛ m mɑ́, ɑ́ɑ yɛ uinipɛɛ n wuru pɔn nɛ símisi. 
Tɔ lɑ rɛ ɑ kɛpɔ́múŋɛ́ rɔ́ símisi tɔkɛ́ yɛ̃́ tɛ ncée yɛ we rɛ tɔkɛ́ yɛ Rɔm uyɔɔpi lɑmpoo hɛ́ɛ́lɛ nɛ́ɛ ɑ́ri kɑpɛ yɛ hɛ́ɛlɛ? 
Amɑ́ Yeesu pɔ̃́ nɛ pimɛsɔhɔ kópɛ kɑpi n wɑi pinyuwɛ mɑ́ɑ́lɛnlɛ. Uu pɛɛ pi mɑɑ rɛ: Nɔ́ kɛcɑ́ɑ́ kɛcɑ́ɑ́ mɛwɑi pikɔ́ mpí, yo nnyɑ kɑni nɛ́ peikɛɛ? 
Ani nɛ siwóó sɛ̃ kɑni yɛ nɛ lɑmpoo n hɛ́ɛlɛ kɑm nkɛ̃́! Api mɛwóópipi mɛsɛ nɛ u hɑpɔ. 
Uu pɛɛ pi pisɛ rɛ: Úye riyu nɛ úye rinyiri tɛɛ nní mɛkɛcɑ́ɑ́ we? 
Api u pɛsɛ rɛ: Sesɑɑ lo. Kei kɛ Yeesu uu kɔ pi mɑɑ rɛ: In lɛ̃, ɑni Sesɑɑ likɔ́ Sesɑɑ pɑ, ɑni Uléécɑɑ likɔ́ Uléécɑɑ pɑ. 
Kuu lɛ̃ tinɔ́ɔ tɛ̃ pi m pɛsɛ, ɑi nnɔ́ɔ pi yipu ɑpi pɛɛ u yɑ́ ɑpi tɔ́mpɔ. 
Kɛyɑ́ɑ kɛ̃ cirɛ kɛ Pisɑtusee ɑpi Yeesu kɛ́mɛɛ hɑ. Pisɑtusee pɛɛ tee rɛ pikpɔkpɔ ɑ́pi yɛ nkpɔ-i yisi. Api sĩ́ ɑpí hɑ Yeesu lɛɛpɔ, ɑpi u pisɛ rɛ: Sɑ́ɑ, Moisi yɛ mɑɑ rɛ: 
Unɔ́sikɔ́ úye un n kpu unsɑ́ kɛwɑ̃́ m mɑ́, li pisɛ rɛ uuwɑ̃ yɛɛ yɛ́ unɔ́si uyɛ̃ kukúmɑnnɔsi kpísi uu umɑ́ɑ mpuri wɑi. 
Too! Amɑ́ piŋmɑ́nɛ pinyinɛ pɛɛ pɛɛ kɛrɔ́cɔpɛ nté we. Pi pimɑ́rɛcɔ pisɛɛi lɛ. Umɑ́ɑ uu unɔ́si kpísi, uu kpi úu kɛwɑ̃́ yɛ̃. Uwɑ̃́ uu pɛɛ umɑ́ɑ ukúmɑnnɔsi kpísi. 
Uyɛ̃ uu kɔ limɛcɔ kpi, uu ukúmɑnnɔsi uyɛ̃ nɛ utɑ́ɑ́nũ yɑ́. Ai lɛ̃ n wɑi hɑ́i nɛ usɛ́ɛ́i-sɛ́ɛ́i kɛ́mɛɛ-pɔ. 
Pɛ̃ mɛmɑ́ɑ́, unɔ́si uyɛ̃ pɔ́ɔ kɔ kpi. 
Tɔ lɑ tɔkɛ́ nkpéni pɔ́ pisɛ rɛ kɛ pimɑ́rɛcɔ pisɛɛi pɛ̃ ɑpi nní pinnɛ́í unɔ́si uyɛ̃ n kpísi, pi uye unɔsi yɛ hɑ lo, kɛyɑ́ɑ kɛ̃ kɛ pikpɔkpɔ ɑpí hɑ nkpɔ kɛ́mɛɛ n yisinɛ? 
Kei kɛ Yeesu uu pɛɛ pi pɛsɛ rɛ: Lɛ̃ nnyɑ kɑni nɔ́mɛnɛ́círɛ kirɑɑsɛntɛ. Áni Nléécɑɑsimɛ́ ɑsei kómɛi, ɑ́ni kɔ Uléécɑɑ nnɑŋɛ nyu. 
Kumúŋɛ́ kpɛ-i kɛ pikpɔkpɔ ɑpí hɑ n yisinɛ, pitisi ɑ́pi pinɔ́si kpíkɛsinɛ, pinɔ́si pɑ́ɑ kɔ pilɑ sɔ́ɔnkɛɛnɛ. Amɑ́ pimɛfinɛ yɛ́ hɑ n we yɑrɛ piléécɑɑtumɛ kɛyómɛcɑɑ-pɔ. 
Áni lɛ̃ kɛ Uléécɑɑ uu pikpɔkpɔ mɛyisɛ kɛcɑ́ɑ́ nɔ́ ḿ mɑ ɑpi Unsímɛ́ Ritɛlɛ́ kɛ́mɛɛ wɔ́lu kɛ́ɛ̃, nɛ́ɛ yo? U mɑɑ rɛ: 
Nɛ́ɛ Apirɑhɑm Uleecɑɑ nɛ Isɑɑki Uleecɑɑ nɛ Yɑkupu Uleecɑɑ. Uléécɑɑ úu pikpɔkpɔ Uleecɑɑ ɑmɑ́ u pinyɑ́nnyɑ̃́ Uleecɑɑ lɛ. 
Kɛ pisoi pɛ̃ nnɛ́í pɛɛ n cɑ́pinɛlɛ̃ pin kutu u cɔlɛ̃ ɑpi lɛ̃ n kṍ, uicélɑɑ iyɛ̃ ii pi loni ńsɔnɛ. 
Kɛ Pifɑrisi ɑpi n kṍ tɛ Yeesu yɛ Pisɑtusee feriyɛ uu sɛ́ɛ́sɛ, ɑpi cɑ́pinɛ. 
Pikɛcɔpɛ isé ucélɑɑ unyinɛ uu pɛɛ ripinɛ Yeesu wɑi uu u pisɛ rɛ: 
Ucélɑɑ, Uléécɑɑ iseé nnɛ́í kɛ́mɛɛ íye yɛɛ imɑ́ɑ? 
Yeesu uu rɛ: A Upíimɑ, Upɔ́léécɑɑ n lɑ nɛ kɛpɔ́fɑ nnɛ́í, nɛ ipɔ́soi nnɛ́í, nɛ kɛpɔ́múŋɛ́ nnɛ́í. 
Iyɛɛ isé foí, iyɛɛ kɔ innɛ́í kɛcɔpɛ imɑ́ɑ. 
Ilírũ yɛɛ ifoí iyɛ̃ mɛcɔ n sɑ́ yɛ nnyí: A upɔ́cɔ n lɑ yɑrɛ kɑɑ ripɔ́cúruu ń lɑ. 
Isé nnyí kɛtɛ́ yɛɛ isé nɛ ɑntepuyɛ icélɑɑ riyíkí. 
Kɛ Pifɑrisi ɑpi lɛ̃ n cɑ́pinɛ, Yeesu uu pɛɛ pi pisɛ rɛ: 
Íye kɑni uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ kɛcɑ́ɑ́ músu? Úye kɛpipi yɛ kɛ? Api u pɛsɛ rɛ Tɑfiti Kɛpipi yɛ kɛ. 
Yeesu uu pɛɛ pi mɑɑ rɛ: In lɛ̃, íye kɑi wɑ kɛ Nfɑ́ɑsɔnɛ nn Tɑfiti mɛtɛ́í hɛ uú nɛ u séi rɛ Upíimɑ? Tɑfiti yɛ pɛɛ́ mɑɑ rɛ: 
Upíimɑ yɛ Unɛ́píimɑ mɑɑ rɛ: A kunɛ́lukɛ-lukɛ-mɛ̃ kɛ́yukɔɔ́tonɛ-i tonɛ hɑ́i nɛ kumúŋɛ́ kpɛ-i kɑm yɛ́ pipɔ́lɑ́ɑrɔ kɛpɔ́nyɑ́lɛ-i n cɔ. 
In tɛ Tɑfiti yɛ mɛsei u sée rɛ Upíimɑ, íye kɛ uyɛ̃ cirɛ uu yɛ́ kɔ pɛɛ Tɑfiti Kɛpipi? 
Pifɑrisi pɛ̃ ɑ́pi fe pikɛ́ ńkɑ Yeesu pɛsɛ. Kɑí nɛ kei n kpísi, úkɑ úu yɛ pɛɛ kɑhɑ ukɛ́ nnyinɛ u pisɛ.
