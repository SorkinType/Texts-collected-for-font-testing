Zɑ̃ɑ.
17.
Iyi Jesu í fɔ bɛɛbɛ í tɑ̃, ŋɔi í wu iju lele í ni, Bɑɑbɑ, ɑwɑɑti í to. Jɔ̀ irii ɑmu Amɑɛ ku nɛ ɑmboe ku bɑ ɑmu mɔ n jɔ̀ titɛɛ ku nɛ ɑmboe, 
nɔ inɛ ŋɑ iyi ì nɑm fei n mu nŋɑ kuwɛɛ hɑi tɑ̃ si bɛi ì muum yiiko si inɛ fei. 
Kuwɛɛ hɑi tɑ̃ui í jɛ inɛ ŋɑ ɑ mɑ̀ɛ, ɑwɔ Ilɑɑɔ̃ ɑkɑ̃ ilu ntɔu, nɔ ɑ mɑ̀m, ɑmu Jesu Kirisi iyi ì bɛ wɑu. 
Ǹ jɔ̀ iriɛ í nɛ ɑmboe si ilɛu ihɛ̃, nɔ icɛ ŋɑ iyi í dɑsim ku ceu fei ǹ coo. 
Bɑɑbɑ, nsɛi ǹ wɑ n tɔɔɛi, jɔ̀ n nɛ ɑmboe bi tɛɛ bɛi ǹ tɑko ǹ nɛ wo hee ɑ mɑɑ tɑkɑ ɑndunyɑ. 
Inɛ ŋɑ iyi ì cicɑ hɑi si ɑndunyɑ ì nɑm, ǹ jɔ̀ ɑ̀ mɑ̀ɛ. Tɛɛi ɑ̀ jɛ wo, nɔ ì so ŋɑ ì nɑm. À nɔ ɑ̀ wɑɑ jirimɑ ideɛ. 
Nsɛi ɑ̀ mɑ̀ mii iyi ì nɑm fei, hɑi bi tɛɛi í nɑɑ, 
domi ide ŋɑ iyi ì sɔ̃m nŋui ɑmu mɔ ǹ sɔ̃ ŋɑ, nɔ ɑ̀ ɡbɑ ideu. À mɑ̀ iyi ntɔ ntɔ hɑi bi tɛɛi ǹ nɑɑ nɔ ɑ̀ dɑsi nɑɑnɛ iyi ɑwɔi ì bɛm wɑ. 
Ǹ wɑ n tɔɔɛ nɑ iri nŋɑ. Kù jɛ nɑ irii inɛi ɑndunyɑ ŋɑi ǹ wɑ n tɔɔɛ, ɑmmɑ nɑ inɛ ŋɑ iyi ì nɑm ŋɑi, domi ɑŋɑi ɑ̀ jɛ tɛɛ. 
Mii iyi ǹ nɛ fei tɛɛi, nɔ mii iyi ɑwɔ mɔ ì nɛ fei tom nii, nɔ irim í nɛ ɑmboe nɑ iri nŋɑ. 
Nsɛi ǹ wɑ n nyi wɑ bi tɛɛ. Bii í kɔ̃ɔsi keeke n kɑɑ n mɑɑ n wɑ si ɑndunyɑ mɑ́, ɑŋɑi ɑɑ mɑɑ wɑ si ɑndunyɑu. Bɑɑbɑ kumɑ́, deɡbe ŋɑ do ɡbuɡbɑ̃i iriɛ iyi ì nɑm, ku bɑ ɑ jɛ ɑkɑ̃ si bɛi ɑwɑɛ ɑ̀ jɛ ɑkɑ̃. 
Wɑɑti iyi ǹ wɛɛ do ɑŋɑ ǹ deɡbe ŋɑ do ɡbuɡbɑ̃i iriɛ, iri iyi ì nɑm nɔu. Ǹ deɡbe ŋɑ nɔ si inɔ nŋɑ bɑɑ inɛ ɑkɑ̃ kù ce nfe, bii kù jɛ ɑmɑi nfeu bɑɑsi. Bɛɛbɛi kukɔi ideɛ í kɔ̃. 
Nsɛi ǹ wɑ n nyi wɑ bi tɛɛ. Ide ŋɑu ihɛ̃ si ɑndunyɑu ihɛ̃i ǹ wɑ n fɔɔ, ku bɑ ɑŋɑ mɔ ɑ kɔ̃ò inɔ didɔ̃ nɑ irim. 
Ǹ nɑ ŋɑ ideɛ, nɔ inɛi ɑndunyɑ ŋɑ ɑ̀ cé ŋɑ si nɑ iyi í jɔ̀ ɑ kù jɛ ti ɑndunyɑ, si bɛi ɑmu mɔ n kù jɛ ti ɑndunyɑ. 
N kù wɑ n tɔɔɛ i nyɑ ŋɑ hɑi si ɑndunyɑ, ɑmmɑ ǹ wɑ n tɔɔɛi i deɡbe ŋɑ ku bɑ Seetɑm inɛ lɑɑlɔu ku mɑɑ̀ nɔ ŋɑ. 
A kù jɛ ti ɑndunyɑ si bɛi ɑmu mɔ n kù jɛ ti ɑndunyɑ. 
Ideɛi í jɛ ntɔ. Jɔ̀ ideɛu ku nyɑ ŋɑ ikɑ̃ ɑ jɛ titɛɛ. 
Ǹ bɛ ŋɑi si ɑndunyɑ si bɛi ɑwɔ mɔ ì bɛm si ɑndunyɑu. 
Nɑ iri nŋɑi ǹ so ɑrɑm ǹ nɑɛ ku bɑ ɑŋɑ mɔ ɑ jɛ titɛɛ mɑm mɑm. 
Kù jɛ ɑŋɑ ɑkɑ̃i ǹ wɑ n ceɑ kutɔɔ, ɑmmɑ do inɛ ŋɑ mɔ iyi ɑɑ nɑ ɑ dɑsim nɑɑnɛ nɑ sɑɑbui ide nŋɑ. 
Ǹ wɑ n tɔɔɛi ɑŋɑ fei ɑ jɛ ɑkɑ̃. Si bɛi ɑwɔ Bɑɑbɑ ì wɑ sim, nɔ ɑmu mɔ ǹ wɑ siɛ, jɔ̀ ɑ jɛ ɑkɑ̃ si wɑ ku bɑ inɛi ɑndunyɑ ŋɑ ɑ dɑsi nɑɑnɛ iyi ɑwɔi ì bɛm wɑ. 
Amboeɛ iyi ì jɔ̀ inɛ ŋɑ ɑ̀ yɛ sim, nŋui ǹ jɔ̀ ɑ̀ wɑɑ yɛ si ɑŋɑ mɔ ku bɑ ɑŋɑ mɔ ɑ jɛ ɑkɑ̃ bɛi ɑwɑɛ ɑ̀ jɛ ɑkɑ̃. 
Jɔ̀ n wɑ si ŋɑ nɔ ɑwɔ mɔ i wɑ sim ku bɑ ɑ jɛ ɑkɑ̃ mɑm mɑm, nɔ inɛi ɑndunyɑ ŋɑ ɑ mɑ̀ iyi ɑwɔi ì bɛm wɑ nɔ ì bi ŋɑi bɛi ì bim. 
Bɑɑbɑ, ǹ bi inɛ ŋɑ iyi ì nɑm ɑ mɑɑ wɑ bi tom bii ɑn wɑ, ku bɑ ɑ yɛ ɑmboem, ɑmboe iyi ì muum. Ì muum ɑmboeui si nɑ iyi í jɔ̀ ì bim hɑi wɑɑti iyi ɑ kù tɑkɑ ɑndunyɑ. 
Bɑɑbɑ ɑwɔ iyi ì jɛ dee dee, bɑɑ do iyi inɛi ɑndunyɑ ŋɑ ɑ kù mɑ̀ɛ, ɑmu ǹ mɑ̀ɛ, nɔ inɛ ihɛ̃ ŋɑ mɔ ɑ̀ mɑ̀ iyi ɑwɔi ì bɛm wɑ. 
Ǹ jɔ̀ ɑ̀ mɑ̀ɛ, nɔ ɑn kɔ̃ɔ nŋɑ si kumɑ̀ɛ mɑ́ ku bɑ kubi iyi ì bim ku wɑ si ŋɑ, nɔ ɑmu mɔ n wɑ si ŋɑ.
