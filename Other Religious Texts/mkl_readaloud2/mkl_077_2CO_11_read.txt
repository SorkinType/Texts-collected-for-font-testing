Kɔrɛnti II.
11.
Oo, ǹ bi i temuɑòm ŋɑ bii ǹ fɔ ide keeke iyi kù nɛ yɑɑse. Ntɔ ntɔ, i temuɑòm ŋɑ. 
Ǹ wɑ n ce iɡu iyi wɑ nɑɑ hɑi bi Ilɑɑɔ̃ nɑ iri nŋɛ, domi ì yɛ ŋɑi bɛi mudɛ̃ɛ iyi ǹ muɑ mɔkɔ ɑkɑ̃. Mɔkɔui í jɛ Kirisi nɔ ǹ bi n kooò ŋɛ bi tɛɛi bɛi mudɛ̃ɛ iyi kù mɑ̀ mɔkɔ. 
Ammɑ si bɛi Seetɑm njou, í dĩ ijui Ɛfɑ wo do bisi lɑɑlɔɛ, ǹ wɑ n ce njo lɑsɑbu nŋɛ mɔ ku mɑɑ nɑ ku sindɑ hɑi si kpɑ̃ɑi Kirisi iyi ì wɑɑ too nsɛi ŋɑ do nɑɑnɛ hɑi nɛ muɑfiti. 
Njo í nyi wɑ mum mɑ́ si nɑ iyi í jɔ̀ bii inɛ ɡɔ í nɑɑ wɑ sisi nŋɛ idei Jesu mmu ɡɔ i ci yɑ kɔ ŋɑ. Ì bɑ Hundei Ilɑɑɔ̃ tɑ̃ ŋɑ, ɑmmɑ ì nyi ì tɑ̃ sɔɔlu i ɡbɑ hunde mmu ɡɔ mɑ́ ŋɑ. Ì ɡbɑ lɑɑbɑɑu jiidɑu tɑ̃ ŋɑ, ɑmmɑ ì nyi ì tɑ̃ sɔɔlu i ɡbɑ lɑɑbɑɑu ikɑ̃ mɑ́ ŋɑ. 
Ǹ wɑ n tɑmɑɑ inɛ ŋɑu ihɛ̃ iyi ɑ̀ wɑɑ cɔ ɑrɑ nŋɑ woo bɛ nlɑ ŋɑu, kɑ̃mɑ nŋɑ ɡɔ kù rem do nɡɔɡɔ. 
Bɑɑ bii n kù dɔ̃ ɡɛlɛ bɛi inɛ ŋɑu ihɛ̃, kù jɛ ǹ kuɑ kumɑ̀, ǹ nɔ ǹ nyisi ŋɛ si fei ndɛɛ fei mɑm mɑm. 
Wɑɑti iyi ǹ sisi nŋɛ lɑɑbɑɑu jiidɑi Ilɑɑɔ̃, ǹ cooi nɡbɛ nɡbɛ. Ǹ kɑye ɑrɑm ku bɑ n so ŋɛ lele. Iyi ǹ ce bɛɛbɛ ǹ rɑrɑi? 
Ǹ ɡbɑ fiɑi iɡbɛi inɛi Jesu ɡɔ ŋɑ wɑi fei sɑ̃ɑ sɑ̃ɑ ǹ nɑ ǹ sobi ŋɛò. 
Wɑɑti iyi ǹ wɑ bi tu ŋɛ n kù jɛ ɑsoi inɛ ɡɔ nŋɛ. Bii ǹ nɛ bukɑɑtɑi nɡɔɡɔ, inɛ nwɑ ŋɑ iyi ɑ̀ yɑ nɑɑ hɑi ilɛi Mɑsedɔni ɑŋɑi ɑ̀ yɑ nɑɑm mii iyi ǹ nɛ bukɑɑtɑɛ fei wɑ. Ǹ mu ɑrɑm si fei ndɛɛ ku bɑ n mɑɑ̀ n jɛ ɑso nŋɛ. Bɛɛbɛ mɔi ɑn nɔ n mɑɑ n mu ɑrɑm. 
Do ide ntɔi Kirisi iyi í wɑ sim, n wɑ n sɔ̃ ŋɛ iyi inɛ ɡɔ kù wɛɛ si ilɛi Gɛrɛsi fei iyi ɑ́ ɡɑnjiim n tɔɔ fufu si ideu ihɛ̃. 
Nɑ mii í jɔ̀ ǹ ce bɛɛbɛ. Ì wɑɑ tɑmɑɑ n kù bi ŋɛi? Ilɑɑɔ̃ í mɑ̀ sɑ̃ɑ sɑ̃ɑ iyi ǹ bi ŋɛ. 
Ammɑ bɛi ǹ wɑ n ce nsɛi, bɛɛbɛi ɑn mɑɑ n ce mɑ́ ku bɑ n dĩ nŋɑ kpɑ̃ɑ, ɑŋɑ iyi ɑ̀ wɑɑ dɛ ɑ tɔɔ fufu ɑ ni ɑwɑò ŋɑ ŋɑ ɑkɑ̃i. 
Inɛ ŋɑu, woo bɛ ilu ibo ŋɑi iyi ɑ̀ yɑ dĩ ijui inɛ ɡɔ ŋɑ, nɔ ɑ̀ wɑɑ kpe ɑrɑ nŋɑ woo bɛi Kirisi ŋɑ. 
Nkɑ̃mɑ kù wɑ bɛ iyi ɑ́ biti inɛ, domi Seetɑm tɑkɑɛ ɑ́ yɔkɔ ku kpɑɑsi ɑrɑɛ bɛi ɑmɑlekɑi Ilɑɑɔ̃. 
Nɑ nŋu, biti ɡɔ kù wɛɛ bii woo ce icɛɛ ŋɑ ɑ̀ kpɑɑsi ɑrɑ nŋɑ bɛi woo ce icɛi Ilɑɑɔ̃ ŋɑ. Ammɑ ɑɑ bɑ ribɑ nŋɑ si bɛi icɛ nŋɑ í yɛ. 
Ǹ wɑ n fɔ mɑ́, inɛ ɡɔ ku mɑɑ̀ cɔm bɛi sũmu. Ammɑ bɑɑ bii ì wɑɑ cɔm sũmu ŋɑ, i ɡbɑm do bɛɛbɛ, ku bɑ ɑmu mɔ n tɔɔ fufu keeke. 
Iyi ǹ wɑ n fɔ nsɛi kù jɛ bɛi Lɑfɛ̃ɛ í bi inɛ ŋɑ ɑ yɑ fɔ. Ammɑ si ideu ihɛ̃ ǹ wɑ n tɔɔ fufu nɔ ǹ fɔ ide bɛi sũmu. Nɔ bii ǹ wɑ n tɔɔ fufu ǹ nɛ mii iyi ǹ wɑ n tɔɔò fufui. 
Amu tɑkɑm mɔ ɑn tɔɔ fufu si nɑ iyi í jɔ̀ inɛ nkpɔ wɑ tɔɔ fufu nɑ ɑrɑ. 
Ì wɑɑ temuɑò inɛ hɑi nɛ yɑɑse ŋɑ, iŋɛ iyi ì wɑɑ tɑmɑɑ ì nɛ yɑɑse jiidɑ ŋɑ. 
Ì wɑɑ coko ɑ̀ wɑɑ ce ŋɛ ɑru ŋɑ, ɑ̀ wɑɑ ce ŋɛ ile, ɑ̀ wɑɑ tɑki ŋɛ, nɔ ɑ̀ wɑɑ dondɑ ŋɛ, nɔ ɑ̀ wɑɑ sɑmbɑlɑ ŋɛ. 
Anyɔ wɑ mum n sɔ̃ ŋɛ iyi n kù nɛ ɡbuɡbɑ̃ n ce nŋɛ bɛɛbɛ. Do nŋu fei, bii inɛ ɡɔ í nɛ wɔɔkɔ wɑ tɔɔò fufu ɑmu mɔ ǹ nɛ. Ǹ wɑ n fɔ ide bɛi sũmu. 
À jɛ inɛi Ebɛɛ ŋɑi? Bɛɛbɛ mɔi ɑmu tɑkɑm. À jɛ inɛi Izirɛli ŋɑi? Bɛɛbɛ mɔi ɑmu tɑkɑm. À jɛ tɔkui Aburɑhɑmui? Bɛɛbɛ mɔi ɑmu tɑkɑm. 
À jɛ woo ce icɛi Kirisi ŋɑi? Ǹ re ŋɑ si bɛ. Idem nɔu, idei sũmui. Ǹ ce icɛ ǹ re ŋɑ, ɑ̀ dɑsim piisɔ̃ɔ í re ŋɑ, ɑ̀ tɛɛm kpɔtɔɔ n kù mɑ̀ bɑɑ jiɑɛ. Isɛ nkpɔ nkpɔi ǹ mɑɑi iku. 
Isɛ ɡbɛɛwui Zuifu ŋɑ ɑ̀ tɛɛm kpɔtɔɔ ɑnu ciiji fɑkɑ̃ kù wɑ si. 
Inɛi Romu ŋɑ ɑ̀ cɑ̃m do ɡolo í to isɛ ɡbɛɛtɑ. À tɑm kutɑ ɑ̀ bi ɑ kpɑm ɡbɑkɑ̃ ɑkɑ̃. Isɛ ɡbɛɛtɑi ɑkɔi inyi iyi ǹ lɔ í jɛ inyi, nɔ ǹ dɑsi inyi ɡbɑkɑ̃ ɑkɑ̃ ǹ wɑ bɛ hɑi dɑɑkɔ hee iju í koo í mɑ́. 
Si isɛɛnɛm ŋɑ ɑjɔ nkpɔ nkpɔ si mbirisii ǹ yɑ n mɑɑ n wɑ nɑ kukɔ̃i ido do woo bɑtɑ ku ɡbɑ ŋɑ iyi ɑ̀ wɑ si kpɑ̃ɑ. Ajɔ ɡɔ ŋɑ ǹ yɑ n wɑ si mbirisi bi dimii inɛm ŋɑ, ɑjɔ ɡɔ ŋɑ mɔ bi dimi mmu ŋɑ, ɑjɔ ɡɔ ŋɑ mɔ si inɔ ilu, ɑjɔ ɡɔ ŋɑ mɔ si ɡbɑbuɑ, ɑjɔ ɡɔ ŋɑ mɔ si tenku, ɑjɔ ɡɔ ŋɑ mɔ bi inɛ ŋɑ iyi ɑ̀ sɔ ibo ɑ̀ wɑɑ ni ɑŋɑ ɑ̀ jɛ kpɑɑsi woo dɑsi nɑɑnɛ ŋɑ. 
Ǹ mɑ̀ icɛ ku ɡɑɑbu ŋɑ do wɑhɑlɑ nlɑ nlɑ ŋɑ, ɑ̀ ɡɑnjiim njoo ku sũ. Ari í yɑ kpɑm nɔ ɑɡbɛ í yɑ kpɑm, ǹ ce ɑnu ku dĩ do tilɑsi, ǹ wɑhɑlɑ do tũtũ nɔ n kù nɛ jinɛ ku dɑsi ŋɑ. 
Si ɑnyii mii ŋɑu bɛ fei, lɑsɑbum í yɑ mɑɑ wɑ si icɛ ŋɑ iyi ǹ ceɑ iɡbɛi inɛi Jesu ŋɑ fei. 
Bii inɛ ɡɔ kù nɛ ɡbuɡbɑ̃, ɑmu mɔ n ci yɑ n nɛ ɡbuɡbɑ̃. Bii inɛ ɡɔ í dɑsi dulum ǹ yɑ n nɛ ɑrɑ kuroi jiidɑ. 
Bii í jɛ ǹ wɑ n tɔɔ fufui, ɑn tɔɔi si mii iyi wɑ nyisi ɡbuɡbɑ̃ hɑi nɛm fei. 
Kɑ sɑɑbu Ilɑɑɔ̃ bɑɑi Lɑfɛ̃ɛ nwɑ Jesu do ɑjɔ fei. Nŋui í mɑ̀ iyi n kù wɑ n sɔ ibo. 
Wɑɑti iyi ɑ̀ wɑ Dɑmɑsi, inɛ nɡbo iyi ɑmɑnlu Aretɑsi í jilɛ si iluu í jilɛ inɛ ŋɑi ɑ mɑɑ deɡbe ɑnu koofɑi ilu nlɑu ɑ bɑ ɑ mum. 
Ammɑ ɑ̀ jɔ̀ ǹ fitɑi hɑi do fɛrɛntii bindii iluu nɔ ɑ̀ kitɑɑm nii si kɔlɔ, nɔ do yɑɑse bɛɛbɛi ǹ ce ǹ ɡbɑ̃ò.
