Kɔrɛnti I.
9.
N kù nɛ ɑrɑm nii? N kù jɛ woo bɛi? N kù yɛ Jesu Lɑfɛ̃ɛ nwɑi? I kù jɛ isoi icɛm iyi ǹ ceɑ Lɑfɛ̃ɛ ŋɑi? 
Bɑɑ bii inɛ ɡɔ ŋɑ ɑ̀ kɔ ku mɑ̀m bɛi woo bɛ, ǹ jɛ woo bɛ bi tu ŋɛ, domi iŋɛi ì jɛ sɛɛdɑ iyi wɑ nyisi iyi ɑmuu woo bɛi Lɑfɛ̃ɛi ntɔ ntɔ. 
Inɛ ŋɑ iyi ɑ̀ wɑɑ wusɛm, ide iyi ɑn jɛ nŋɑ wee. 
A kù nɛ kpɑ̃ɑ kɑ jɛ nɔ kɑ mɔi? 
A kù nɛ kpɑ̃ɑ kɑ nɛ ɑbo woo dɑsi Jesu nɑɑnɛ iyi ɑ́ si wɑ bɛi woo bɛ ɡɔ ŋɑ ɑ̀ wɑɑ cei? Bɛi ifɔi Lɑfɛ̃ɛ ŋɑu do Sefɑɑ Piɛɛ? 
Mɑ̀ ɑmu do Bɑɑnɑbɑsi ɑwɑ ɑkɑ̃i ɑ kù nɛ kpɑ̃ɑi ɑ yɑ nɑ wɑ ijɛ, í ɡbe kɑ ce icɛ? 
Inɛ kɑ̃mɑ kɑɑ bɔ ikoi iɡũ do fiɑi nŋu tɑkɑɛ. Inɛ kɑ̃mɑ kɑɑ lɔ jĩi ndii rezɛ̃ɛ iyi kù jɛ isoɛ. Nɔ inɛ kɑ̃mɑ kɑɑ mɑɑ deɡbe mɛɛmu ɡɑɑ iyi kù wɑɑ mɔ ɑmɔɛ. 
Kù jɛ do bisii ɑmɑnɛ nŋu ɑkɑ̃i ǹ wɑ n fɔò ideu ihɛ̃. Ide ɑkɑ̃ui do iyi woodɑi Moizi wɑ fɔ. 
Moizi í kɔ si tiɑi woodɑɛu í ni, i mɑɑ̀ ti i dĩ ɡɛlɛi kɛtɛ iyi wɑ jojo si bilee wɑ nyɑ yɔyɔɛ. Nɑ kɛtɛ ŋɑi Ilɑɑɔ̃ í fɔ iyi bɛ bɑ. 
Mɑ̀ nɑ iri nwɑi wɑ fɔ ideu. Oo, nɑ iri nwɑi ɑ̀ kɔɔ, domi í sĩɑ woo loɡoo do inɛ iyi wɑ kpokpo ɑŋɑ fei ɑ mɑɑ cɔ kpɑ̃ɑi ikpɛ̃ nŋɑ. 
Ide iyi ɑ̀ sɔ̃ ŋɛ í yɛi bɛi idei Hundei ɑ̀ ɡbɛ̃ nŋɛ. Debɛi, bii ì mu nwɑ sɔndɑi ɑrɑ ŋɑ, ɑɑ jɔ̀ ku lɑ nŋɛi? 
Bii inɛ ɡɔ ŋɑ ɑ̀ nɛ kpɑ̃ɑɛ bi tu ŋɛ, ɑwɑ mɔ ni ɑ kù re ŋɑ? Ammɑ ɑ kù ceò kpɑ̃ɑu icɛ si ŋɛ. À temuɑ si fei ndɛɛ ku bɑ kɑ mɑɑ̀ ti kɑ dĩ kpɑ̃ɑi lɑɑbɑɑu jiidɑi Kirisi. 
I kù mɑ̀ ŋɑ iyi inɛ ŋɑ iyi ɑ̀ wɑɑ ce icɛ kpɑsɛ̃i Ilɑɑɔ̃, bɛi ɑ̀ yɑ bɑ ijɛ nŋɑ? Inɛ ŋɑ mɔ iyi ɑ̀ jɛ woo wee ŋɑ, si kuweeu bɛi ɑ̀ yɑ bɑ mii ku jɛ nŋɑ. 
Bɛɛbɛ mɔi Lɑfɛ̃ɛ í jilɔɔ si inɛ ŋɑ iyi ɑ̀ wɑɑ sisi lɑɑbɑɑu jiidɑu ɑŋɑ mɔ ɑ bɑ ijɛ nŋɑ si bɛ. 
Ammɑ n kù ceò kɑ̃mɑɛ icɛ. N kù nɔ n kù wɑ n kɔ iyi ihɛ̃ ku bɑ ɑ ceem bɛɛbɛ. Ǹ ɡbɑsi n kú do iyi ide iyi ǹ wɑ n tɔɔò fufuu ihɛ̃ ɑ́ ce nfe. 
N kù wɑ n tɔɔ fufu si nɑ iyi í jɔ̀ ǹ wɑ n sisi lɑɑbɑɑu jiidɑ. Ammɑ tilɑsii bi tom n coo. Nɔ ijuukpɑ̃i ɑ́ jɛ titom bii n kù wɑɑzo ideu. 
Bii do idɔɔbim nii ǹ wɑ n coo, ǹ nɛ kpɑ̃ɑi n bɑ ribɑ. Ammɑ tilɑsi ǹ nɛ ku ceɛ mɔm nii domi Ilɑɑɔ̃i í dɑsim. 
Debɛi, yoomɑi í jɛ ribɑm. Ribɑm nii í jɛ n wɑɑzo lɑɑbɑɑu jiidɑi Kirisi, ku mɑɑ̀ ti ku jɛ do idei fiɑ nɔ n mɑɑ̀ n ti n ce icɛ do kpɑ̃ɑ iyi ǹ nɛ. 
Ǹ nɛ ɑrɑm, n kù jɛ ɑrui inɛ kɑ̃mɑ. Ammɑ do nŋu fei ǹ ce ɑrɑm ɑrui inɛ fei ku bɑ n fɑ inɛ nkpɔ n nɑɑò wɑ bi Kirisi. 
Bi Zuifu ŋɑ ǹ bɑɑ ǹ jɛ bɛi Zuifu, ku bɑ n fɑ ŋɑ wɑ. Inɛ ŋɑ mɔ iyi ɑ̀ wɑɑ jirimɑ woodɑ ǹ jɛ inɛ ɑkɑ̃ nŋɑ ku bɑ n fɑ ŋɑ wɑ bɑɑ bii ɑmu tɑkɑm n kù wɑ si ɑwɔi woodɑ. 
Bi inɛ ŋɑ iyi ɑ kù nɛ woodɑ ǹ jɛ inɛ ɑkɑ̃ nŋɑ ku bɑ n fɑ ŋɑ wɑ. Kù jɛ n kù wɑ n jirimɑ woodɑi Ilɑɑɔ̃, domi ǹ wɑ n too woodɑi Kirisi. 
Bi inɛ ŋɑ iyi ɑ kù nɛ ɡbuɡbɑ̃ ɑmu mɔ n ce bɛi inɛ iyi kù nɛ ɡbuɡbɑ̃ ku bɑ n fɑ ŋɑ wɑ. Ǹ ce bɛi inɛ fei í yɛ ku bɑ ɡɔɡɔ nŋɑ ŋɑ ɑ bɑ fɑɑbɑ, bɑɑ bii bɛirei. 
Ǹ wɑ n ce fei ndɛɛi ku bɑ kuwee iyi lɑɑbɑɑu jiidɑu í nɛ n bɑsi ikpɛ̃. 
Iŋɛ fei ì mɑ̀ ŋɑ si ɡɑɑsɑi kusɛi inɛ feii í yɑ sɛi ɑmmɑ inɛ ɑkɑ̃i í yɑ bɑ ribɑɛ. Nɑ nŋu, i hɑnyɑ ku bɑ i bɑ ribɑɛ ŋɑ. 
Woo ce ɡɑɑsɑi kusɛi ŋɑ ɑ̀ yɑ mɑɑ mu ɑrɑ nŋɑi si fei ndɛɛ. À wɑɑ ce bɛɛbɛ ku bɑ ɑ bɑ fulɑi wuɑ ŋɑ iyi ɑ́ ɡbɛ nɔ ku wɔ. Ammɑ ɑwɑ ɑ̀ wɑɑ cei ku bɑ kɑ bɑ iyi ci yɑ tɑ̃. 
Nɑ ŋɔi í ce ǹ wɑ n sɛi bɛi inɛ iyi í nɛ mii iyi wɑ lele. Ǹ wɑ n jɑbu ɑmmɑ kù jɛ si nɡbɛ. 
Ǹ wɑ n wɑhɑlɑ ɑrɑm ntɔ ntɔ ku bɑ n kɑ̃miɑɛ nɔ n mɑɑ̀ n ti n jɛ inɛ iyi ɑɑ nɑ ɑ lele ɑwɔ nɡbɛ wɑɑti iyi ǹ ceɑ inɛ ŋɑ wɑɑzo ǹ tɑ̃.
