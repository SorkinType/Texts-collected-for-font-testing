Tenketɛ́.
31.
Kɛ̀ Sɑkɔbuu keè Dɑmɑɑ bí nɑ́ɑ́mmɛ̀ kɛ tú: Sɑkɔbu fìètɛ ti cicɛ kpɑ̀tì nti, dɛɛ̀ te kòò kpenkɛ miɛ. 
Kòo yɑ̀ Dɑmɑɑ yɛ̀mmɛ̀ tɛ̃́nkɛ í ò nɑɑtinɛ̀mɛ̀ mɛ̀ do ɔ̃ ndòmmɛ̀ nɛ̀ dimɔ̀nnì. 
Kɛ̀ ti Yiɛ̀ nKuyie nnɑ́kɛ́ Sɑkɔbu kɛ dɔ̀: Kò nhɑ cicɛ ciɛ bɛ̀ dɑ pɛitɛ́ dɛ̀. N yó ndɑ bonɛ̀mu. 
Kɛ̀ Sɑkɔbuu yú Dɑsɛɛdi nɛ̀ Deɑ dikpɑ́ɑ̀ ò cɛ̃mmú dɛ̀ o wũɔ̃, 
kɛ́ bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: N yɑ̀ di cicɛ yɛ̀mmɛ̀ tɛ̃́nkɛ í n nɑɑtinɛ̀mɛ̀ mɛ̀ do ɔ̃ ndòmmɛ̀, Kuyie n cicɛ tũ nkù mɛ nni m bonɛ̀mu. 
Di yɛ̃́mu m pĩ̀ḿmɛ̀ di cicɛ mutɔ̃mmú nɛ̀ n wɛ̃rímú mumɔu. 
Kòò n ciitɛ́, kɛ cèètɛ n yeti kucɛ tɛpíítɛ̀, Kuyie mmɛ nyí yóu kòò n dɔ̀ɔ̀ mɛyɛi mmɑmɛ̀. 
Ò yĩ dìì mɔ̀nnì ìì ɑ̃ sisuɔ ì bo ntú n kpɛyi kɛ̀ ìi piɛ́ ìì ɑ̃ sisuɔ, kòò dɔ̀ ìì dò nyɑ̀ùyɑ̀ù ì bo ntú n kpɛyi, kɛ̀ ìi piɛ́ ìì dò nyɑ̀ùyɑ̀ù. 
Kuyie nkuù cɔutɛ́ di cicɛ wũɔ̃ kɛ n duɔ́. 
Ipe nɛ̀ sibɔɔ́ dɛ̀ dèkù dìì mɔ̀nnì, kɛ̀ n yɑ̀ nɛ̀ tidɔuǹtì dɛ̀ɛ̀ ɑ̃ sisuɔ nɛ̀ dɛ̀ɛ̀ dò nyɑ̀ùyɑ̀ù dɛɛ̀ dèkùmɛ̀. 
Kɛ̀ Kuyie ntɔ̃nnì n yú kɛ dɔ̀: Sɑkɔbu! Kɛ̀ n dɔ̀: Ntɛ mí. 
Kɛ̀ dì dɔ̀: Wéntɛ́ kɛ́yɑ̀ yɛbɔdɑkɛ̀ nɛ̀ ipe dɛ̀ɛ̀ dò nyɑ̀ùyɑ̀ù dɛɛ̀ dèkùmɛ̀. N yɑ̀mu Dɑmɑɑ dɑ dɔɔ̀rimɛ̀. 
Míì tu di Yiɛ̀ nKuyie nhɑ do yɑ̀ kù Betɛɛdi, kɛ́fíí nditɑ̃́rì, kɛ́cóú mmɛkùɔ̀ nkɛ́ n nɑ́kɛ́ ɑ dɔ́ tì. Di mmɔ̀nnì, ítɛ́ kɛ́kò nhɑ ɛì bɛ̀ dɑ pɛitɛ́ dì. 
Kɛ̀ Dɑsɛɛdi nɛ̀ Deɑ kɛ̀ bɛ̀ dɔ̀: Ti cicɛ í yóó kú kɛ̀ ti túótɛ́ dɛ̀mɑrɛ̀ o cĩ́ɛ̃̀tɛ̀. 
Ò ti wèńtɛ́ bɛpɔ̀ɔ̀bɛ̀ mbɛ, kɛ tì fìtɛ́ kɛ di ti díítí, bɑ́ wè í ti duɔ́ nwe do dò nkɛ́ ti duɔ́ nyì. 
Kuyie ncɔutɛ́ tìì kpɑ̀tì ti cicɛ borɛ̀ kɛ dɑ duɔ́ ntì tu ti kpɛti nti, tínti nɛ̀ ti bí, dɔɔ̀ Kuyie ndɑ nɑ̀kɛ́ tì. 
Kɛ̀ Sɑkɔbuu bɑ́ɑ́tí, kɛ́dee nho bí nɛ̀ o pobɛ̀ yòyóbɛ̀, 
kɛ́túótɛ́ o wũɔ̃ nɛ̀ o kpɑ̀tì timɔu ò pɛ̀tɛ́ tì Mɛsopotɑmii, kɛ́kétɛ́ kɛ́nkũnti o cicɛ Isɑki borɛ̀ Kɑnnɑhɑ̃ɑ̃. 
Kɛ sɔ̃́ nDɑmɑɑ kɔ̀tɛ kɛ kɛ̃ĩ o pe cìtì, kɛ̀ Dɑsɛɛdi yúúkú o bɔkɛ̀. 
Kɛ̀ Sɑkɔbuu soutɛ́ Dɑmɑɑ Arɑmu botí kou, kɛ́ítɛ́ bɑ́ ò í ò cɑu, 
kɛ́túótɛ́ o kpɛrɛ dɛmɔu kɛ́cokɛ́, kɛ́séntɛ́ kukó ndiɛkù Efɑdɑti, kɛ́kétɛ́, kɛ́nwetí Kɑdɑɑdi tɑ̃rì. 
Diyiè tɑ̃ɑ̃́nnì yiè kɛ̀ bɛ̀ɛ nɑ́kɛ́ Dɑmɑɑ Sɑkɔbu cokɛ́mɛ̀. 
Kòo túótɛ́ o kɔbɛ, kɛ́tũnnɛ Sɑkɔbu kɛ́mmɔkɛ yɛwe yɛ̀yiekɛ̀, kóò nìntɛ Kɑdɑɑdi tɑ̃rì. 
Kɛ̀ Kuyie nnɑ́kɛ́ Dɑmɑɑ, Arɑmu botí kou nɛ̀ tidɔuǹtì kɛ dɔ̀: A bɑ́ɑ́ dɔɔ̀ Sɑkɔbu tìmɑtì. 
Wè ò nìntɛ mɛmmɛ kɛ sɔ̃́ nSɑkɔbu còńnɛ́ o touku Kɑdɑɑdi tɑ̃rì ĩ́nkɛ̀, kɛ̀ Dɑmɑɑ múnkɛɛ cónnɛ́ o kɔku, 
kɛ́bekɛ Sɑkɔbu kɛ dɔ̀: A n dɔ̀ɔ̀ bɑ, kɛ n soutɛ́? Kɛ tùótɛ́ n kó bɛsɑpɑ̀mbɛ̀ kɛ dò nhɑ kpɑ̀ nkɛ bɛ̀ ɛí. 
Dɛ̀ dòmmɛ kɑ̀ɑ n soutɛ́, kɛ sɔ̀ri kɛ cokɛ́, bɑ́ ɑ í n nɑ̀kɛ́? Ti nɑ n dɑ cíemmu kɛ́ndiè, kɛ bie nyɛbɑ̀rɛ̀ nɛ̀ tikùtìdùùtì kɛ̀ dɛ̀ nɑɑti. 
A í yóu kɛ̀ nh ɔrí n kó bɛsɑpɑ̀mbɛ̀ nɛ̀ n yɑɑ̀bí. A dɔ̀ɔ̀ dɛyɛinkpɛrɛ ndɛ. 
M mɔkɛmu muwɛ̃rímú kɛ nɑ ndɑ dɔ̀ɔ̀ mɛyɛi, kɛ̀ Kuyie nhɑ cicɛ tũ nkù nɛ́ n nɑ̀kɛ́ kɛyènkɛ̀ kɛ̀ɛ̀ pɛ̃ɛ̃tɛ́ kɛ tú m bɑ́ɑ́ dɑ dɔɔ̀ tìmɑtì. 
A dɔ́ mɛcɑ̃ɑ̃ mmɛ kɛ bo kò nhɑ cicɛ cĩ́ɛ̃̀tɛ̀, dɛ̀ nɛ́ yĩ́mɛ kɑ̀ɑ yùúkú m bɔkɛ̀? 
Kɛ̀ Sɑkɔbu dɔ̀: N yĩɛ̃̀kù ɑ bo fietɛ mɛ̀mmɛ ɑ kó bɛsɑpɑ̀mbɛ̀. 
Wéntɛ́ n kó bɛnìtìbɛ̀, kɑ̀ɑ yɑ̀ wèè borɛ̀ ɑ bɔkɛ̀ bɛ̀ɛ ò kùɔ. Sɑkɔbu do í yɛ̃́ kɛ dò nDɑsɛɛdi weè yɛ̀ tùótɛ́. 
Kɛ̀ Dɑmɑɑ tɑ Sɑkɔbu kó kutouku, kɛ́tɑ Deɑ kɔku, kɛ́tɑ o kó bɛnitipotɔ̃mbɛ̀ bɛdɛ́ kpɛti, ò í nyɑ̀ mùmɑmù, kɛ́yɛ̀ kɛ́tɑ Dɑsɛɛdi kɔku. 
Kɛ yɛ̃́ bɑ! Kɛ̀ Dɑsɛɛdi weè yɛ̀ tùótɛ́ kɛ sɔ̀nnɛ yɛsɑ̃nkɔ̃̀nkɛ̀ miɛkɛ kɛ kɑri yɛ ĩ́nkɛ̀. Kɛ̀ Dɑmɑɑ wɑmmú titouti timɔu miɛkɛ kɛ́mɔ́ntɛ́. 
Kɛ̀ Dɑsɛɛdi dɔ̀: N cicɛ ɑ miɛkɛ bɑ́ɑ́ pɛikɛ n yí mɛ̀ nyìtɛ́mɛ̀ n kɑ̀rì, m bo n tɑ̃̀nkù yɛirɛ ndɛ. Kɛ̀ Dɑmɑɑ wɑmmú kɛ́mɔ́ntɛ́. 
Kɛ̀ dɛ̀ɛ yonkɛ Sɑkɔbu kòo nkpɑnnɛ̀ Dɑmɑɑ, kɛ tú: N cɑ̀kɛ bɑ? N dɔ̀ɔ̀ ɔ̃̀mmɛ yɛi nkɑ̀ɑ n tũ̀ nnɛ̀ kɛmiɛkɛ? 
A fiɛ́mu n nɛntì, ɑ yɑ̀ bɑ dɛ̀ɛ̀ tu ɑ kpɛrɛ? Bɛnkɛ dɛ n kɔbɛ nɛ̀ ɑ kɔbɛ, kɛ̀ bɛ̀ɛ yɑ̀ kɛ́ ti pútɛ́. 
M bo ɑ cĩ́ɛ̃̀tɛ̀ kɛ mɔkɛ yɛbie nsipísìdɛ́ ndi, m mɛ nyí cɑ̀ɑ́kɛ́ ɑ pedɑɑ̀ mɑrì. 
Musĩ̀mmù í pĩ nhɑ ɔ̃̀fɛ̀ mɑfɛ̀, kɛ̀ n fɛ̀ tɔní kɛ dɑ bɛnkɛ, nh ɔ̃ɔ̃ fɛ̀ yietímu. Kɑ̀ɑ m békú bɛ̀ yúúkú ì kuyie nnɛ̀ kɛyènkɛ̀, kɛ̀ ǹ dɑ yietí. 
Kɛ̀ fɛwentɑɑfɛ̀ m potɛ́, kɛ̀ muséé nni m potɛ́ kɛyènkɛ̀, kɛ̀ n níí nyié nkɛ wúó. 
Yɛbie nsipísìdɛ́ ndi m bomɛ̀ ɑ cĩ́ɛ̃̀tɛ̀, kɛ pĩ mmutɔ̃mmú yɛbie ntɛpíítɛ̀ nɛ̀ yɛ̀nɑ̀ɑ̀ ɑ kó bɛsɑpɑ̀mbɛ̀ kpɛ́í, nkɛ cɛ̃mmú ɑ wũɔ̃ yɛbie nyɛ̀kuɔ̀. Kɑ̀ɑ cèètɛ n yeti kucɛ tɛpíítɛ̀. 
Kɛ̀ Kuyie, n yɑ̀ɑ̀rì Abɑrɑhɑmmu do tũ nkù, n cicɛ Isɑki dé kù do í n teennɛ̀, ɑ nɑ nni m bɛ̀ti n nɔu sinùmmù ndi. Kù mɛ nyɑ̀mu n fɛ̃̀ṹtɛ́mɛ̀ ɑ cĩ́ɛ̃̀tɛ̀, kɛ n teennɛ̀ wenkɛ kɛyènkɛ̀. 
Dɛ mɔ̀nnì kɛ̀ Dɑmɑɑ ò tɛ̃̀ńnɛ́ kɛ dɔ̀: N kó bɛsɑpɑ̀mbɛ̀ tu biɛ, bɛ bí tú n kpɛyi nyi, kɛ̀ iwũɔ̃ ii ntú n kpɛyi, dɛmɔu ɑ mɛ̀ nwùó ndɛ̀ tu n kpɛrɛ ndɛ. M mɛ nyí yóó nɑ kɛ́dɔɔ̀nɛ̀ mùmɑmù n kó bɛsɑpɑ̀mbɛ̀ nɛ̀ bɛ bí. 
Yóu kɛ̀ ti tɑu mmí nnɛ̀ fɔ̃́ nkɛ́dɔɔ̀ mɛbɛnkùmɛ̀ mɑmɛ̀. 
Kɛ̀ Sɑkɔbuu túótɛ́ ditɑ̃́rì kɛ́fíí, 
kɛ́nɑ́kɛ́ o kɔbɛ kɛ dɔ̀: Wɑɑ́nnɛ̀nní yɛtɑ̃́rɛ̀! Kɛ̀ bɛ̀ɛ yɛ̀ wɑɑnní kɛ́cóú nditɑ̃́cóù, kɛ̀ bɛ̀ɛ kɑri bɛmɔu di ĩ́nkɛ̀ kɛ́di. 
Kòo dì yu kɛ dɔ̀: Mɛtɑummɛ̀ kó mɛbɛnkùmɛ̀ kó ditɑ̃́cóù. Nɛ̀ Dɑmɑɑ kó tinɑ́ɑǹtì Yekɑɑ-Sɑɑdutɑɑ nɛ̀ Sɑkɔbu kpɛti Kɑdɛdi. 
Kɛ̀ Dɑmɑɑ dɔ̀: Di ntɑ̃́cóù diì yó mbɛnkú mí nnɛ̀ fɔ̃́ nti tɑummɛ̀. Nɛ̀ dɛ kpɛ́í nkɛ bɛ̀ɛ dì yu kɛ dɔ̀ Kɑdɛdi, 
kɛ́yíɛ́ kɛ́ dì yu kɛ dɔ̀ Misipɑɑ. Kɛ̀ Dɑmɑɑ dɔ̀: Ti Yiɛ̀ nKuyie nní nti wúó nti dɛ̀tɛ́nɛ̀ dìì mɔ̀nnì titɔbɛ̀. 
Kɑ̀ɑ fɛ̃́ũnko n kó bɛsɑpɑ̀mbɛ̀, kɛ tùókɛ́ bɛtɔbɛ̀, ɑ nyɛ̃́ kɛ dò nhonìtì weè í ti wúó, Kuyie nkuù ti wúó. 
Kɛ̀ Dɑmɑɑ dɔ̀: Ntɛ ditɑ̃́cóù n còú ndì nɛ̀ ditɑ̃́rì n fìí ndì mí nnɛ̀ fɔ̃́ nti cuokɛ̀. 
Di ntɑ̃́cóù nɛ̀ di ntɑ̃́rì dɛ̀ ntú ti tɑummɛ̀ kó dibɛnkɛ̀rì. M bɑ́ɑ́ dɛ̀ pɛ̃ɛ̃tɛ́ kɛ́kɔtɛ ɑ bíɛ́kɛ̀ kɛ́dɔɔ̀ mɛyɛi, ɑ mɛ mbɑ́ɑ́ dɛ̀ pɛ̃ɛ̃tɛ́ kɛ́kɔtɛní m bíɛ́kɛ̀ kɛ́dɔɔ̀ mɛyɛi. 
Kuyie nhAbɑrɑhɑmmu nɛ̀ Nɑɔɔ bɛ̀ tũ nkù ni nti wúó! Kɛ̀ Sɑkɔbuu pɑrìkɛ̀ kɛ́yú Kuyie nho cicɛ Isɑki dé kù yètìrì, 
kɛ́féútɛ́ fɛɔ̃̀fɛ̀ Kuyie, kɛ́yú o kɔbɛ kɛ̀ bɛ̀ɛ cɑ́ɑ́kɛ́ kɛ́nyié nditɑ̃rì ĩ́nkɛ̀.
