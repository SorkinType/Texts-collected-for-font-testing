Tenketɛ́.
27.
Kɛ̀ Isɑkii kótɛ́ kɛ tɛ̃́nkɛ í nwúó, kɛ́yú o kóo Po Esɑyuu, kòò dɔ̀: Ntɛ mí. 
Kɛ̀ Isɑki dɔ̀: N kòtɛ́mu, n yí yɛ̃́ n kṹṹ nyiè. 
Túótɛ́ ɑ tɑ̃mmù nɛ̀ ɑ pie nkɛ́tɑ dikpɑ́ɑ̀ kɛ́ n kuɔní tɛhontɛ̀, 
kɛ́bénnɛ́ n dɔ́ mùù diì kɛ́kɔtɛnɛ̀ní kɛ̀ ǹ di, kɛ́ dɑ pɑ̃ mɛsɑ̀ɑ̀ nkɛ nɛ́ nɑ kɛ́kú. 
Kɛ̀ Debekɑɑ keè Isɑki nɑ̀kɛ́ tì Esɑyuu kɛ tú wèe tɑ dikpɑ́ɑ̀ kóò kùɔní tɛhontɛ̀, 
kɛ́nɑ́kɛ́ Sɑkɔbu kɛ dɔ̀: N kèè ɑ cicɛ nɑ̀kɛ́mɛ̀ ɑ kóo kótì Esɑyuu kɛ tú 
wèe tɑ dikpɑ́ɑ̀ kóò kùɔní tɛhontɛ̀ kɛ́bénnɛ́ mudiì ò dɔ́ mù, kòo di, kóò pɑ̃ ti Yiɛ̀ nKuyie nkó mɛsɑ̀ɑ̀ nkɛ́nɑ kɛ́kú. 
M birɛ kéntɛ́ kɛ́keè n yóó dɑ nɑ́kɛ́ tì kɛ́ tì dɔ̀ɔ̀. 
Kɔtɛ kɛ́pĩ́nní yɛbɔpɑ̀ɑ̀ nyɛ̀dɛ́ɛ̀ kɛ̀ m bénnɛ́ ɑ cicɛ dɔ́ mùù diì. 
Kɑ̀ɑ ò duɔ́ nkòo di, kɛ́ dɑ pɑ̃ mɛsɑ̀ɑ̀ nkɛ́nɑ kɛ́kú. 
Kɛ̀ Sɑkɔbuu nɑ́kɛ́ o yɔ̃ kɛ dɔ̀: N kóo kótì Esɑyuu mɔ̀kɛ ticìtì nti, mí m mɛ nyí tì mɔ̀kɛ. 
Kɛ̀ dɛ̀ yɛ̀mmɛ n cicɛ n kɑ́ɑ́kɛ́ kɛ́yɑ̀ n yí mɔkɛmɛ̀ ticìtì kɛ̀ n nɑɑ́ siyɑ́ɑ̀bìsí yiɛ̀, kòò n dɔɔ̀ mɛyɛi nkɛ̀ m mɔ́ntɛ́ mɛsɑ̀ɑ̀. 
Kòo yɔ̃ dɔ̀: Kòò dɑ dɔ̀ɔ̀ mɛyɛi mmɛ̀ɛ do n yĩ́nkɛ̀. Cɛ̃́nkɛɛ yie n dɑ nɑ̀kɛ́ tì kɛ́pĩ́nní yɛbɔpɑ̀ɑ̀. 
Kɛ̀ Sɑkɔbuu kɔtɛ kɛ́pĩ́nní yɛbɔpɑ̀ɑ̀ nkɛ́duɔ́ nho yɔ̃, kòo bénnɛ́ o cicɛ dɔ́ mùù diì. 
Kɛ̀ bɛ yɔ̃ Debekɑɑ túótɛ́ o kóò Po Esɑyuu yɑɑ̀sɑ̀ɑ̀tì tìì bo tɛcĩ́ɛ̃̀tɛ̀ kɛ́dɑ́tínnɛ́ Sɑkɔbu o bíyɑ̃ɑ̃tɛ̀, 
kɛ́túótɛ́ kubɔkɔ̃̀nkù kɛ́dɑ́tínnɛ́ o nɔu nɛ̀ o fɔ̃̀níí, 
kóò duɔ́ mmudiì ò bèńnɛ́ mù nɛ̀ pɛ̃́ɛ̃̀. 
Kɛ̀ Sɑkɔbuu kɔtɛ o cicɛ borɛ̀ kɛ dɔ̀: N cicɛ! Kòò dɔ̀: Ntɛ mí! Fɔ̃́ nwennìnwe? 
Kɛ̀ Sɑkɔbu dɔ̀: N cicɛ mínwe ɑ kóo Po Esɑyuu. N dɔ̀ɔ̀ ɑ yɛ̃ n dɔɔ̀ dɛ̀ ndɛ. N dɑ bɑ́ɑ́mmu, ítɛ́ kɛ́kɑri kɛ́cɑ́ɑ́kɛ́ iorimɑɑ n kùɔní ì, kɛ́ m pɑ̃ mɛsɑ̀ɑ̀. 
Kɛ̀ Isɑki dɔ̀: A pɑ̃̀nkɛ pɛ̀tɛ́mu dɛndɛ bɑ̀mbɑ̀ ntɛhontɑ̀ɑ̀? Kòò dɔ̀: A Yiɛ̀ nKuyie nkuù tɛ̀ kɔ̀tɛnɛ̀ní n yììkɛ̀. 
Kɛ̀ Isɑki dɔ̀: Tɔ́ɔ́nní kɛ̀ n dɑ kɑ́ɑ́kɛ́ kɛ́yɑ̀ kɛ̀ nsɑ̀ kɛ̀ fɔ̃́ nwe m birɛ Esɑyuu. 
Kɛ̀ Sɑkɔbuu tɔ́ɔ́nko, kòo cicɛ Isɑkii ò kɑ̀ɑ́kɛ́ kɛ dɔ̀: Mɛtɑmmɛ̀ dònnɛ̀ Sɑkɔbu kɔ̃mɛ kɛ̀ dɛbɑɑ̀ ndònnɛ̀ Esɑyuu kpɛrɛ. 
Bɑ́ ò í nhò bɑntɛ́, kɛ yɛ̃́ o bɑɑ̀ ndo ɑ̃mɛ̀ ticìtì kɛ dònnɛ̀ o kóo kótì Esɑyuu kpɛrɛ, kòo ò dɔ̀ɔ̀ mɛsɑ̀ɑ̀. 
Kóò bekɛ kɛ dɔ̀: Fɔ̃́ nwe m birɛ Esɑyuɑ̀ɑ̀? Kòò dɔ̀: Ɛ̃ɛ̃ mínwe. 
Kɛ̀ Isɑki dɔ̀: Nh ɑ̃nnɛ́, kɛ̀ n cɑ́ɑ́kɛ́ iorimɑɑ nkɛ́ dɑ pɑ̃ mɛsɑ̀ɑ̀. Kɛ̀ Sɑkɔbuu ɑ̃nnɛ́ o cicɛ, kòo cɑ́ɑ́kɛ́, kòo ò duɔ́ mmɛnɑɑ̀ nkòo yɑ̃̀. 
Kɛ dɔ̀: Tɔ́ɔ́nní m birɛ kɛ́ nh ɔrí. 
Kòo tɔ́ɔ́nko kóò ɔrí kòo keè o yɑ̀ɑ̀tì nɔ́ú, kóò pɑ̃ mɛsɑ̀ɑ̀ nkɛ dɔ̀: M birɛ kó kunɔ́ú dònnɛ̀ dikpɑ́ɑ̀ ti Yiɛ̀ nKuyie mpɑ̃ dì mɛsɑ̀ɑ̀ nkó kunɔ́ú nku. 
Kuyie ndɑ pɑ̃ kɛĩ́nkɛ̀ kó kubooku, kɑ̀ɑ pɑɑ nní mpɛí mɛsɑ̀ɑ̀, kɑ̀ɑ mmɔkɛ tidiitì nɛ̀ mɛnɑɑ̀. 
Kɑ̀ɑ mbɑkɛ́ ibotí imɔu, kɛ̀ titentì timɔu kɔbɛ nninku ɑ ììkɛ̀, kɑ̀ɑ mbɑkɛ́ ɑ tebìí kɛ sì ninku ɑ ììkɛ̀. Kɛ̀ wèè dɑ dɔ̀ɔ̀ mɛyɛi nwèe pɛ́tɛ́ mɛyɛi, kɛ̀ wèè dɑ dɔ̀ɔ̀ mɛsɑ̀ɑ̀ nwèe pɛ́tɛ́ mɛsɑ̀ɑ̀. 
Isɑki pɑ̃ dìì mɔ̀nnì Sɑkɔbu mɛsɑ̀ɑ̀ nkɛ dèè kòo nyiti, kòo kóo kótì Esɑyuu konní kupɑɑwɑɑ́, 
kɛ́bénnɛ́ mudiì kɛ́tɔo kɛ́duɔ́ nho cicɛ kɛ dɔ̀: N cicɛ ítɛ́ kɛ́cɑ́ɑ́kɛ́ iorimɑɑ n kùɔní ì kɛ́ m pɑ̃ mɛsɑ̀ɑ̀. 
Kɛ̀ Isɑkii bekɛ kɛ dɔ̀: Fɔ̃́ nwennìnwe? Kòò dɔ̀: Mínwe Esɑyuu ɑ kóo Po. 
Kɛ̀ dɛ̀ɛ di Isɑki mɛdiɛ̀ nkòo kɔ̃̀ntìi ɑɑ́, kòò dɔ̀: Wenninwe wèè kùɔní tɛhontɛ̀ kɛ n duɔ́ nkɛ̀ n cɑ̀ɑ́kɛ́ kɛ dèè kɑ̀ɑ kpɑɑní, kɛ̀ nh ò pɑ̃ mɛsɑ̀ɑ̀ nkòo yóó mɛ̀ pɛ̀tɛ́? 
Kɛ̀ Esɑyuu keè o cicɛ bɛ́i ntì kòo yɛ̀mmɛ̀ɛ cɑɑ̀rɛ̀, kòo kuɔ́nko mɛdiɛ̀ nkɛ dɔ̀: N cicɛ múnkɛ m pɑ̃ mɛsɑ̀ɑ̀! 
Kɛ̀ Isɑki dɔ̀: A nɑntɛ̀ tɛɛ̀ kɔ̀tɛní kɛ soutɛ́ kɛ cɔutɛ́ ɑ kó mɛsɑ̀ɑ̀. 
Kɛ̀ Esɑyuu dɔ̀: Bɛ̀ ò yu kɛ tú Sɑkɔbu osoùntì, dɛɛ̀ te kòò n soutɛ́ kucɛ mɛ̀dɛ́mɛ̀ kɛ cɔutɛ́ Mpo ɔ̃ nte dɛ̀ kɛ wɛ̃̀tɛ kɛ cɔutɛ́ n kó mɛsɑ̀ɑ̀. Kɛ dɔ̀: Mɛsɑ̀ɑ̀ mmɑmɛ̀ í kpɑɑ́ n kpɛ́ínɑ́ɑ̀? 
Kɛ̀ Isɑki dɔ̀: M bɛ́immu kɛ tú ò bo n dɑ bɑkɛ́, kɑ̀ɑ tebìí simɔu ntú o kó bɛtɔ̃mbɛ̀. M bɛ́immu kɛ tú ò bo mmɔkɛ tidiitì nɛ̀ mɛnɑɑ̀, n tɛ̃́nkɛ bo nɑ kɛ́ dɑ dɔɔ̀ bɑ m birɛ! 
Kɛ̀ Esɑyuu dɔ̀: N cicɛ ɑ mɔkɛ mɛɛ̀ sɑ̀ɑ̀ mmɑ́ɑ̀? Múnkɛ n dɔɔ̀ mɛsɑ̀ɑ̀. Dɛ mɔ̀nnì kɛ́kuɔ́. 
Kɛ̀ Isɑki dɔ̀: A bo mbo kɛ̀ɛ̀ tenkɛ̀ bɑ́ nnɑɑti, kɛĩ́nkɛ̀ kó kubooku bɑ́ɑ́ cútɛ́ní ɑ tenkɛ̀. 
A níí yóó dokɛ́mu kɛ́pɛ́tɛ́ kɛ́di, kɛ́ntú ɑ nɑntɛ̀ kóo tɔ̃ntì, kɛ́nhɑ̃́ɑ̃ntɛ tipíìtì timɔu, diyiè mɑrì kɛ́ nɛ́ fietɛ ɑmɑ́ɑ̀ ɑ nɑntɛ̀ nɔu miɛkɛ. 
Kɛ̀ Esɑyuu miɛkɛɛ pɛikɛ mɛdiɛ̀ nSɑkɔbu o cicɛ o pɑ̃ mɛ̀ɛ̀ kpɛ́í mmɛsɑ̀ɑ̀, kòò nnɑ́ɑ́ nkɛ tú: Dɛ̀ í kpɑɑ́ dɛ́tirì n cicɛ bo kúmɛ̀, kɛ̀ ǹ kuɔ n nɑntɛ̀ Sɑkɔbu. 
Kɛ̀ bɛ̀ɛ nɑ́kɛ́ Debekɑɑ Esɑyuu nɑ̀ɑ́ ntì, kòo nɑ́kɛ́ Sɑkɔbu kɛ dɔ̀: A kóo kótì Esɑyuu miɛkɛ dɑ pɛikɛmu kòò dɔ́ kɛ́ dɑ kuɔ. 
M birɛ kéntɛ́ n yóó dɑ nɑ́kɛ́ tì, kɛ́cokɛ́ kɛ́kɔtɛ n tɑ̃ũ̀ Dɑmɑɑ ciɛ Anɑɑ ɛì, 
kɛ́ nkɛ mbo kòo miɛkɛ yɑ̀ɑ donɛ̀. 
Kòo yɛ̃̀ nhɑ ò dɔ̀ɔ̀ dɛ̀, kɛ̀ n dɑ nɑ́kɛ́ kɑ̀ɑ wɛ̃tɛní. N yí dɔ́ ndi pɑɑ̀ didɛ́ diyiè dimɑ́ɑ̀. 
Kɛ̀ Debekɑɑ nɑ́kɛ́ Isɑki kɛ dɔ̀: N dɛmmu mufòmmu nɛ̀ n cɔ̀ɔ̀bɛ̀ Itiibɛ kó mɛdɔɔ̀rìmɛ̀ kpɛ́í, kɛ̀ Sɑkɔbu puokɛ di nhɛì kóo nitipòkù kɛ̀ n ku dɛ̀ ntɔ̃ũ.
