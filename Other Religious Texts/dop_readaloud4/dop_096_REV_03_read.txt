Kukulutu.
3.
Nmɑɑ tɑkǝlɑɣɑ nɑ ń heeli Sɑɑtɩ Yesu sɛɛlɑɑ kpekǝle ɩsɔtɑɑ tillu sɩ: Mɑ wei mɑ tɩ Ɩsɔ feesiŋ nɑɑtosompɔɣɔlɑɣɑ nɑ ɩsɔtʋlʋŋɑsɩ nɑɑtosompɔɣɔlɑɣɑ tɔ, mɑ yɔɣɔtɑɑ sɩ: Mɑ nyǝmɑ pǝ tǝnɑ mpi ń lɑpɑ tɔ. Mɑ nyǝmɑ́ teu sɩ ɩsɛntɔɔ kɛ́ n nyǝnǝɣɩ sɩ n wɛnɑ weesuɣu pǝyele n sǝpɑɣɑ. 
Kʋlɩ nyɑ́ nɑɑsɩ tɑɑ nɑ ń lɑ mpi pǝ kɑɑsɑɑ tɔ nɑ pǝ́ nyɔɔ toŋ, pǝ́ tɑɑ kɔɔ nɑ pǝ́ tɛɛsɩ sǝpʋ tǝnɑŋnɑŋ. Pǝ tɑɣɑ pʋlʋ, mɑ nɑwɑɣɑ sɩ nyɑ́ kʋlɑpʋtʋnɑɑ tɑ te teu kɛ mɑ Ɩsɔ ɩsɛntɑɑ. 
Sɛɣɛsʋɣʋ ŋku pɑɑ sɛɣɛsɑ-ŋ tɔ tɔɔsɩ kʋ tɔɔ. Tɔɔsɩ sɩ n kɑ tɛmɑ sɛɣɛsʋɣʋ ŋkʋɣʋ nɩɩʋ, tɔkɩ-kʋ nɑ ń yele nyɑ́ ɩsɑɣɑtʋ. Ye n tɑ kʋlɩ nyɑ́ nɑɑsɩ tɑɑ, mɑ kɔŋɩ-ŋ tutuɣu kɛ́ ɩsɩɩ ŋmɩɩlʋ kɔŋʋɣʋ tɔ, n kɑɑ nyɩ wɑɑtʋ wei mɑɑ kɔɔ tɔ se. 
Pɑɑ nɑ mpʋ nyɑ́ yǝlɑɑ mpɑ pɑ wɛ Sɑɑtɩ tɑɑ tɔ pɑ tɑɑ lɛlɑɑ tɑ pilisi pɑ wontu se. Wontu kʋhʋlʋmtʋ kɛ pɑɑ suu nɑ mɑ́ nɑ-wɛ tǝ́ tɔŋ. Pǝ tɑɣɑ pʋlʋ, pǝ mʋnɑ-wɛɣɛ ḿpʋ́ɣʋ́. 
Mpɑ pɑɑ lɑ kǝllɑɑ tɔ wontu kʋhʋlʋmtʋ kɛ ḿpʋ́ɣʋ́ pɑɑ suu, mɑ kɑɑ kʋ pɑ hǝlɑ kɛ weesuɣu tɑkǝlɑɣɑ tɑɑ. Mɑɑ yɔɣɔtɩ mɑ Cɑɑ nɑ ɩ ɩsɔtɑɑ tillɑɑ pɑ ɩsɛntɑɑ sɩ pɑ kɛ́ mɑ nyǝ́mɑ. 
Ye wei ɩ wɛnɑ ŋkpɑŋŋ pʋntʋ ɩ tʋ-ɩ teu, nɑ ɩ́ nɩɩ nti Feesuɣu heeliɣi Yesu sɛɛlɑɑ kpekɑ tɔ ɩlɛ. 
Nmɑɑ tɑkǝlɑɣɑ nɑ ń heeli Filɑtɛlǝfii Yesu sɛɛlɑɑ kpekǝle ɩsɔtɑɑ tillu sɩ: Mɑ́ wei mɑ wɛ tǝnɑŋŋ nɑ mɑ kɛ́ tɑmpɑnɑ tʋ, nɑ mɑ tɔkɑ wulɑʋ sɔsɔ Tɑfiiti sɑɑfʋ nɑ mɑ tǝkǝɣɩ nɑ nɔɣɔlʋ ɩɩ pǝsǝɣɩ tʋlʋɣʋ, nɑ mɑ tʋlɑɑ nɔɣɔlʋ ɩɩ pǝsǝɣɩ tǝkʋɣʋ tɔ, mɑ yɔɣɔtɑɑ sɩ: 
Mɑ nyǝmɑ́ pǝ tǝnɑ mpi n lɑpɑ tɔ. Mɑ nyǝmɑ́ sɩ nyɑ́ tomɑ tɑ tɔɔ te, pɑɑ nɑ mpʋ tɔ n tɔkɑ mɑ tɔm nɑ n lɑpɑ yʋlʋpɑŋ. Mɑ hɑ ɩkpɑtɛ kɛ nyɑ́ ɩsɛntɑɑ nɑ nɔɣɔlʋ kɑɑ wɑkǝlɩ-tɛ. 
Sɑtɑnɩ nyǝ́mɑ pɔpɔtʋnɑɑ mpɑ pɑ yɑɑ pɑ tǝɣɩ Yutɑ nyǝ́mɑ, pǝyele kpɑɩtǝko pɑ tɑ kɛ polo tɔ, mɑɑ lɑnɑ-wɛ. Mɑɑ cɑɑlɩ-wɛ nɑ pɑ́ hǝntɩ-ŋ ɑtɛ nɑ pɑ se-ŋ. Ḿpʋ́ɣʋ́ pɑɑ nyǝnɑ sɩ mɑ sɔɔlɑ nyɑ́ tɔm. 
Ɩsǝnɑ mpi n nɩɩnɑ mɑ nɔɣɔ tɑɑ tɔm sɩ ń tɔkɩ suulu, nɑ tɑmpɑnɑ n tɔkɩ-ɩ tɔ, mɑ tɔtɔ mɑɑ kenti nyɑ́ tɔɔ kɛ́ wɑɑtʋ wei wɑhɑlɑ kɔŋ ɩ́ kɔɔ sɩ ɩ mɑɣɑsǝɣɩ ɑntulinyɑ tǝnɑ tɑɑ yǝlɑɑ sɩ ɩ nɑɑ tɔ. 
Mɑɑ kɔɔ nɔɔnɔɔ yoo. Mpi n wɛnɑ tɔ tɔkɩ-wǝɣɩ teu tǝkem, pǝcɔ nɔɣɔlʋ ɩ́ tɑɑ kɔɔ nɑ ɩ́ lɛɛkɩ-ŋ nyɑ́ kǝlʋɣʋ teeli ntenuɣu. 
Wei ɩ lɑpɑ kǝllʋ tɔɣɔ mɑɑ lɑ mɑ Ɩsɔ tǝsɛɛlɛ sɔsɔɔlɛ ɑkele, nɑ pʋntʋ kɑɑ tɑsɑ fituɣu. Mɑɑ ŋmɑɑ ɩ tɔɔ kɛ́ mɑ Ɩsɔ hǝtɛ nɑ mɑ Ɩsɔ ɩcɑtɛ nyǝntɛ tɔtɔ, Yosɑlɛm kʋfɑlʋ wei ɩ kɑ lɩɩnɑ mɑ Ɩsɔ kiŋ kɛ ɩsɔtɑɑ nɑ ɩ tii tɔ. Nɑ mɑɑ ŋmɑɑ pʋntʋ tɔɔ tɔtɔɣɔ mɑ hǝtɛ kʋfɑtɛ nte pɑ hɑ-m tɔ. 
Ye wei ɩ́ wɛnɑ ŋkpɑŋŋ pʋntʋ ɩ́ tʋ-ɩ teu, nɑ ɩ́ nɩɩ nti Feesuɣu heeliɣi Yesu sɛɛlɑɑ kpekɑ tɔ ɩlɛ. 
Nmɑɑ tɑkǝlɑɣɑ nɑ ń heeli Lɑyotisee Yesu sɛɛlɑɑ kpekǝle ɩsɔtɑɑ tillu sɩ: Mɑ́ wei mɑ kɛ́ Ami, nɑ mɑ́ hʋlǝɣǝnɑ pǝ tǝnɑ pǝ tɑmpɑnɑ kɛ teu nɑ mɑ́ kɛnɑ pǝ tǝnɑ mpi Ɩsɔ ŋmɑwɑ tɔ pǝ tʋ tɔ, mɑ yɔɣɔtɑɑ sɩ: 
Mɑ nyǝmɑ́ pǝ tǝnɑ mpi ń lɑkɩ tɔ. Mɑ nyǝmɑ́ sɩ nyɑɣɑlɛ nyɑ́, n tɑ niɣiti pǝyele n tɑ hɑŋ. Mɑ cɑɑ kɛ́ sɩ ye n kɑ́ niɣiti, niɣiti, ye n kɑ́ hɑŋ, hɑŋ. 
Anɩ n tɑ hɑŋ pǝyele n tɑ niɣiti, nɑ ń wɛ tǝtɛmlɛɛ kɛ́ mpʋ tɔ, mɑɑ tɔ-ŋ kɛ́. 
N fɑɑlʋɣʋ mpʋ sɩ n pilɑɑ, sɩ n wɛnɑ kɛ́, n tɑ lɑŋ pʋlʋ tɔ, n tɑ nyɩ sɩ nyɑ́ tɔm wɛ wɑɩyo, nɑ n kɛ́ kʋnyɔntʋ kutukutu. N hʋlʋmɑɑ kɛ́, nɑ tɑpɑkpɛtɛ kɛ n wɛɛ, nɑ n kɛ́ yʋlʋm kɛ́. 
Pǝ tɔɔ kɛ́ mɑ tɑsǝɣɩ tɔm sɩ ń yɑ mɑ cɔlɔɣɔ wʋlɑnɑɑ mpɑ kɔkɔ tɑɣɑnɑɑ tɔ nɑ ń pili. Yɑ tɔtɔɣɔ wontu kʋhʋlʋmtʋ nɑ ń suu nɑ pǝ́ wɑɑsɩ nyɑ́ tɑpɑkpɛtɛ fɛɛlɛ tɔɔ. Nɑ ń yɑ tɔtɔɣɔ kɔɔlɛ nɑ ń tʋ nyɑ́ ɩsɛ nɑ ɑ́ kuli. 
Pɑ tǝnɑ mpɑ mɑ sɔɔlɑɑ tɔɣɔ mɑ kɑlǝɣǝnɑ nɑ mɑ́ hɔŋ pɑ ŋkpɑŋŋ. Ye mpʋ sɛɛsɩ ɩsǝlɛ nɑ ń yele nyɑ́ ɩsɑɣɑtʋ. 
N nɑɑ mɑ sǝŋʋɣʋ nɔnɔɣɔ nɑ mɑ́ lɑkɩ kɑɑfɑlɑ, ye nɔɣɔlʋ nɩɩ mɑ nɔɣɔ nɑ ɩ́ tʋlɩ nɔnɔɣɔ, mɑɑ sʋʋ pʋntʋ tɛ nɑ mɑ́ nɑ-ɩ tǝ́ kɑɑ nyɑnɑɣɑ. 
Mpɑ pɑ lɑpɑ kǝllɑɑ mɑɑ hɑ-wɛɣɛ mpɑɑʋ nɑ pɑ́ cɑɣɑ mɑ cɔlɔɣɔ mɑ kɑwulɑɣɑ kumte tɔɔ ɩsɩɩ mɑ lɑpʋ kǝllʋ nɑ mɑ́ cɑɣɑ mɑ Cɑɑ kiŋ kɛ ɩ kɑwulɑɣɑ kumte tɔɔ tɔ. 
Ye wei ɩ wɛnɑ ŋkpɑŋŋ pʋntʋ ɩ́ tʋ-ɩ teu, nɑ ɩ́ nɩɩ nti Feesuɣu heeliɣi Yesu sɛɛlɑɑ kpekɑ tɔ ɩlɛ.
