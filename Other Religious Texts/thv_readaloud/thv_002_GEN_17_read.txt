Tizaret.
17.
As yekreh Abram teẓẓahet temerwén n ǎwetay d teẓẓa yemun asin Amaɣlol yenna has : « Nk Messineɣ wan Ǎneddabu. Egel dati, umas yebbillel. 
Éd eḍmeneɣ téséqin ǧéri dk éd key ekfeɣ iheyawen eknanén ǎǧut hullan. » 
Yenḍew Abram udem ennét fol ǎmǎḍal ezzar yenna has Yaḷḷah daɣ : 
« Nɣtah téséqin ǧéri dk : éd teqqeled tis n ellametén ǎǧǧotnén. 
Ǎbas key enɣar s Abram, éd teqqeled Ibrahim édét éd teqqeled tis n ellametén ǎǧǧotnén. 
Éd key ekfeɣ iheyawen eknanén ǎǧut hullan, éd dk ekneɣ ellametén d imenokalen a key-d éd eǧmeḍen. 
Ogaẓeɣ téséqin ǧéri dk, d ǧéri d iheyawen ennek ḍeffer-k, éd tumas téséq teɣlalet, baš éd umaseɣ Messineɣ ennek d Messineɣ n térwa ennek. 
Éd key ekfeɣ key d iheyawen ennek ǎkal waɣ dɣ tǎmosed ǎɣǎrib, ǎkal n Kenƹan. Éd yumas ǎkal nwn ar abedǎh. Éd umaseɣ Messineɣ n térwa ennek. 
Yenna Yaḷḷah i Ibrahim : « Keyunan, éd tagẓed téséqin, key d iheyawen ennek ḍeffer-k dɣ ezzemanen emdan. 
Nɣtah téséqin ta hé taǧẓim ǧéri dk d ǧéri d iheyawen ennek : ak ales dɣ winnek ilzam as ǎzeyyen. 
Édt tezzeyyenem, éd yumas éhwel n téséq ta ǧéri dewen. 
Dɣ ak ezzeman, éyyan emdan n ǎddunet ennek éd ezzeyyenen as ekrehen ettam iheḍan, awén aked éklan wi uwenén ɣor-k méɣ wi tezzenhed s aẓref wrn ǎmos in térwa ennek. 
Akli wa yuwen ɣor-k d wa tezzenhed s aẓref éd ezzeyenen, téséqin dɣ isan nwn téséq teɣlalet. 
Ales wa wrn yezzeyyen éd yemmukkes dɣ ellamet ennét folin yeɣtes téséqin. » 
Yenna Yaḷḷah i Ibrahim daɣ : « Saray, hennik tadih, ǎbas tt tɣarred s isem wan Saray, dimaɣ isem ennét Sara. 
Éd has eǧeɣ elbaraka, éd key ekfeɣ arraw n éy ds. Éd has eǧeɣ elbaraka éd tumas mas n ellametén ǎǧǧotnén, imenokalen n ellametén ǎǧǧotnén a tt-d éd yeǧmeḍen. » 
Yenḍew Ibrahim udem ennét fol ǎmǎḍal ezzar yeṭsa, yenna dɣ iman ennét : « Menék awa hé eǧeɣ éd ekreheɣ arraw, nk wa ilen téméḍé n ǎwetay d Sara telat teẓẓahet temerwén n ǎwetay ? » 
Yenna i Yaḷḷah : « Wr eréɣ ar éd yedder Ismaƹil dat-k. » 
Yenna Yaḷḷah : « Kela, hennik Sara éd key tekf arraw d teǧed as isem Isḥaq. Éd agzeɣ téséqin ǧéri ds d ǧéri d iheyawen ennét ar abedǎh. 
Awa tennéd fol Ismaƹil, esléɣ as : éd has eǧeɣ elbaraka, éd yekreh ǎǧut arraw, éd yekreh iheyawen. Éd yumas tis n meraw imenokalen d essin. Éd yeqqel tis n ellamet mǎqqeret. 
Beššan téséqin éd tt agzeɣ ǧéri d Isḥaq, arraw wa hak éd taru Sara ǎwetay wad iglen hund émér waɣ. » 
As yesmenda Yaḷḷah awal ǧérés d Ibrahim, yefelt. 
Yezzeyyen Ibrahim arraw ennét Ismaƹil d médden wi ewenén dɣ tǎɣǎhamt ennét d wi yezzenha s aẓref ennét. Éyyan emdan dɣ ǎddunet win tǎɣǎhamt tan Ibrahim ezzeyyenen ahel héndɣ hund awa has yenna Yaḷḷah. 
As yezzeyen Ibrahim ila teẓẓahet temerwén n ǎwetay d teẓẓa. 
As yezzeyen rurés Ismaƹil ila meraw iwetyan d keraḍ. 
Ahel iyyen ad ezzeyenen Ibrahim d arraw ennét Ismaƹil, 
aked médden win tǎɣǎhamt n Ibrahim emdan, wi ewenén déndɣ méɣ wi enhenén s aẓref ɣor iɣǎriben, emdan ezzeyyenen.
