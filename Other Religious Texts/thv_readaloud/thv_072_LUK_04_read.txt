Luqa.
4.
Ƹisa yeṭkar Unfas Heddigen, yefel eɣǎher wan Elurdun, yelwayt Unfas dɣ ténéré, 
dɣ yessokey ekkoẓet temerwén n ahel itajerrabt Iblis. Wr yekšé ula enḍerren iheḍan windɣ, as okeyen, yeŋɣé laẓ. 
Yenna has Iblis : « Kud tidet as Ag Yaḷḷah a tǎmosed key, en as i ékedé waɣ éd yeqqel tagella. » 
Yenna has Ƹisa : « Yektab : -Ǎwadem wr éd yeddar s tagella ɣas, beššan aked s ak tefért n Yaḷḷah ». 
Yeweyt Iblis édeg iknan aṭkul, yessekné-hin hik id-t illa ǎkal yetiwezzaɣen 
ezzar yenna has : « Éd key ekfeɣ terna n ikallen wiɣ emdan d elmejd nsn : awaɣ imda egreweɣt, addobéɣ édt ekfeɣ i was eréɣ. 
Kud terkeƹed dati, ak heret innek, ekféɣ akt. » 
Yenna has Ƹisa : « Yektab : « Erkeƹ dat Émeli Messineɣ ennek enta ɣas a hé teƹbeded.” » 
Yeweyt Iblis adih s Yerušelem, yeǧé afella n Éhen wan ǎmud, yenna has : « Kud tǎmosed Ag Yaḷḷah, enḍu iman ennek s ǎmǎḍal ; 
édét yektab : « Yaḷḷah yenna hasn i ǎngelosen ennét éd key agẓin, key.” 
Méɣ daɣ : « Éd key eṭṭefen s ifassen nsn, baš wr éd yeqqef aḍer ennek i ékedé.” » 
Yenna has Ƹisa : « Yahéwel : « Wr éd taremed Émeli, Messineɣ ennek.” » 
As yesmenda Iblis awa yeddobet dɣ térmitén ennét, yegla, yoyyé ar émér iyyen heḍen. 
Yeqqel Ƹisa Jalila yeṭkar terna n Unfas Heddigen. Esineqqesen fols ǎddunet dɣ ǎkal héndɣ imda. 
Yeqqim isaɣra dɣ ihenan n téɣeré nsn, ǎddunet emdan tamelent. 
Yekka Ƹisa Ennaṣira, ǎkal wa dɣ yedwel. Yosin ahel n essebet, ahel n tesonfat, yeggeh éhen n téɣeré, hund emǧom. Yenker, éd yeɣer, 
yegrew elkettab wan ennebi Išeƹya. Yoré, yosa édeg wa dɣ ikteb aẓel waɣ : 
« Unfas n Émeli iwar ahi, yesnefren ahi nk, baš éd egeɣ isǎlan n elxér i tilaqqiwen, éd zuzyeɣ ulawen iha elhem. Yezmahel ahi-d baš éd enneɣ i kél tǎkormut aǧmoḍ yoheẓ, éd enneɣ i immedorɣal ǎhenay yeǧrew, éd exxerereɣ wi atiwekrasnén, 
baš éd enneɣ ǎwetay n elxér n Émeli. » 
As yesmenda, yoḍa Ƹisa elkettab, yerré i ǎnaxdam, ezzar yeqqim. Ǎddunet wi ehenén éhen n téɣeré, ekyedent emdan. 
Dédih yenna hasn : « Ahel waɣ aẓel wa yektaben dɣ Elkettab yetbet émér héndɣ d has tsallem, kewenéḍ wiɣ has esgednén. » 
Emdan tamelent, eqqimen ǎkunen dɣ tifér n elxér ti yenna. Ennen : « Awah weggéɣ rurés n Yusef ?” 
Yenna hasn Ƹisa : « Essaneɣ as teddobém éd hi tennim anhi wah yenna : « Ǎḍebib ! Zuzey iman ennek ! » Éd tennim aked : « Nessan awa tgéd imda dɣ Kafranahum, egé déɣ dɣ ǎkal ennek. » 
Ezzar yesséta : « Éd hawen tn enneɣ, awah tidet : wrt illé ennebi yeknan etwerexxeb dɣ ǎkal ennét. 
Awah tidet, eddobéɣ éd hawen tn enneɣ, élwanet téḍéḍén s eba ales nesnet dɣ ǎkal n Israyil dɣ ezzeman wan Eliya. Émér wéndɣ dɣ wr iwét aǧenna ad éweḍen keraḍ iwetyan d aɣil n iyyen. Émér héndɣ d temel menna mǎqqeret dɣ ǎkal imda. 
Beššan wr yesséwey Messineɣ Eliya ɣor wl iyyet dsnt, yesseweyt ɣor iyyet heḍet tezzaɣet dɣ Ṣerfa dɣ ǎkal n Ṣeyda. 
Awén yola d wa iǧen dɣ ezzeman wan ennebi Elišeƹ, élwan ǎddunet wi tegrew torna tan élem mellen ; beššan wl iyyen dsn wr yezzéy, wr yezzéy ar Naƹman d ifalen ǎkal wan Surya. » 
Emdan, dɣ éhen n téɣeré, ekkusen, ewweššenen as eslen i awén. 
Enkeren, emhelen Ƹisa essegmeḍent aɣrem, eweyent s afella n térgit tan tadreq ta ɣor tebna Ennaṣira, eran édt in sebbereɣwelen. 
Beššan yokey gérésn, yegla. 
Yekka Ƹisa Kafranahum, dɣ ǎkal n Jalila, isaɣra ds ahel wan essebet, yǎmosen ahel n tesonfat. 
Akunen ǎddunet dɣ ǎseɣri ennét folin awal ennét ila elḥikma. 
Dɣ éhen wan téɣeré, illé ales iyyen iha ag ǎsuf. Yeṭkel amesli ennét yesiɣeret : 
« Hey ! Ƹisa wan Ennaṣira, ma dnɣ teréd ? Teréd éd haneɣ tehheddemed ? Essaneɣ ma tǎmosed : Wa Heddigen d yezmahel Yaḷḷah ! » 
Yezmeheḍ Ƹisa fol ag ǎsuf wa ihen ales wah, yenna has : « Susem folnɣ d tegmeḍed ales wah !” Yegmeḍ-d ag ǎsuf, yessoḍa ales fol ǎmǎḍal dat ǎddunet, beššan wr has yeɣhéd ula enḍerren. 
Ǎddunet emdan akunen, ennen gérésn : « Ma yǎmos awal wah ? Ales waɣ yeḥkam s terna d elḥikma dɣ kél ǎsuf, ǧammeḍen !” 
Dédih enferen isǎlan ennét dɣ ǎkal imda. 
Yefel Ƹisa éhen n téɣeré, yekka tǎɣǎhamt tan ales iyyen igan isem Simƹun. Tamɣart iyyet, tǎḍeggalt n Simgenƹun tegraw tt tazzaq teṣṣohet, yehéwel i Ƹisa éd has yeg heret iyyen. 
Yeneh-in fols, yezmeheḍ fol tazzaq. Tefel tt tazzaq, tenker tǎmǎṭ déndɣ a tn simgurut. 
As toḍa tǎfuk, i ilen ǎméren, ak iyyen d torna ennét, yéweyt ɣor Ƹisa. Isansa ifassen ennét fol ak iyyen dsn, tazzeyen emdan. 
Aked déndɣ, gammeḍen-d kel ǎsuf dɣ ǎddunet ǎǧǧotnén, gannén : « Key as Elmesiḥ Ag Yaḷḷah ! » Beššan izamahaḍ dsn Ƹisa, yekkas asn awal, édét entenéḍ essanen as enta Elmesiḥ. 
Déndɣ d yeffo, yefel Ƹisa aɣrem, yekka édeg iyyen yesofen. Egmeyent ǎddunet ; ast osen, eɣélen édt ermesin, wr erén éd tn yefel. 
Beššan yenna hasn Ƹisa : « Yelkam ahi éd ebbeššereɣ Isǎlan n Elxér win Elmelek n Yaḷḷah dɣ iɣermen wi heḍnén emdan. Fol awéndɣ as hid yesséwey Yaḷḷah. » 
Yeqqim ixatteb dɣ ihenan n téɣeré dɣ ǎkal n Jalila imda.
