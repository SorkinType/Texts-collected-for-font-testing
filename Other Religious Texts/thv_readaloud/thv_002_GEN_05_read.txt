Tizaret.
5.
Awah tǎnǎqqést tan Adam d iheyawen ennét : ahel wad yexlek Yaḷḷah Adam yekné yola ds. 
Yexlek tn ǎmosen éy d tunté. Yeǧa sn elbaraka ezzar yeǧa sn isem ales d tǎmǎṭ ahel wa dɣ exleken. 
As yekreh Adam téméḍé n ǎwetay d keraḍet temerwén yekreh arraw hund enta, yola ds. Igas isem Šet. 
Ḍeffer tiwit n Šet yedder Adam ettamet temaḍ n ǎwetay arweh. Yekreh arrawen wiyeḍ heḍnén d tarrawén. 
Yedder teẓẓahet temaḍ n ǎwetay d keraḍet temerwén ezzar yemmut. 
As yekreh Šet téméḍé n ǎwetay d semmos yekreh arraw isem ennét Enoš. 
Ḍeffer tiwit n Enoš, yedder Šet ettamet temaḍ n ǎwetay d essa. Yekreh arrawen d tarrawén wiyeḍ heḍnén. 
Yedder teẓẓahet temaḍ n ǎwetay d meraw d essin ezzar yemmut. 
As yekreh Enoš teẓẓahet temerwén n ǎwetay yekreh arraw isem ennét Qenan. 
Ḍeffer tiwit n Qenan yedder Enoš ettamet temaḍ n ǎwetay d meraw d semmos. Yekreh arrawen d tarrawén heḍnén. 
Yedder teẓẓahet temaḍ n ǎwetay d semmos ezzar yemmut. 
As yekreh Qenan essahet temerwén n ǎwetay yekreh arraw isem ennét Mahlalyel. 
Ḍeffer tiwit n Mahlalyel yedder Qenan ettamet temaḍ n ǎwetay d ekkoẓet temerwén. Yekreh arrawen d tarrawén heḍnén. 
Yedder teẓẓahet temaḍ n ǎwetay d meraw ezzar yemmut. 
As yekreh Mahlalyel seḍiset temerwén n ǎwetay d semmos yekreh arraw isem ennét Yared. 
Ḍeffer tiwit n Yared, yedder Mahlalyel ettamet temaḍ n ǎwetay d keraḍet temerwén. Yekreh arrawen d tarrawén heḍnén. 
Yedder ettamet temaḍ n ǎwetay d teẓẓahet temerwén d semmos ezzar yemmut. 
As yekreh Yered téméḍé n ǎwetay d seḍiset temerwén d essin yekreh arraw isem ennét Hénok. 
Ḍeffer tiwit n Hénok yedder Yered ettamet temaḍ n ǎwetay. Yekreh arrawen d tarrawén heḍnén. 
Yedder teẓẓahet temaḍ n ǎwetay d seḍiset temerwén d essin ezzar yemmut. 
As yekreh Hénok seḍiset temerwén n ǎwetay d semmos yekreh arraw isem ennét Matušaleḥ. 
Ḍeffer tiwit n Matušaleḥ, yedder Hénok keraḍet temaḍ n ǎwetay, yelkam i ǎbǎreqqa n Yaḷḷah. Yekreh arrawen d tarrawén heḍnén. 
Yedder keraḍet temaḍ n ǎwetay d seḍiset temerwén d semmos. 
Dɣ tǎmeddort ennét yelkam i ǎbǎreqqa n Yaḷḷah, ad yexrek édét Yaḷḷah yéweyt ɣors. 
As yekreh Matušaleḥ téméḍé n ǎwetay d ettamet temerwén d essa yekreh arraw isem ennét Lamek. 
Ḍeffer tiwit n Lamek yedder Matušaleḥ essahet temaḍ n ǎwetay d ettamet temerwén d essin. Yekreh arrawen d tarrawén heḍnén. 
Yedder teẓẓahet temaḍ n ǎwetay d seḍiset temerwén d teẓẓa ezzar yemmut. 
As yekreh Lamek téméḍé n ǎwetay d ettamet temerwén d essin yekreh arraw, 
yeǧas isem Noḥ. Yenna : « Édét waɣ éd haneɣ isesmeḍ fol oḍuh d elxedmet ta teṣṣohét n ǎmǎḍal wa fol iǧa Amaɣlol tilɣant. » 
Ḍeffer tiwit n Noḥ, yedder Lamek semmoset temaḍ n ǎwetay d teẓẓahet temerwén d semmos. Yekreh arrawen d tarrawén heḍnén. 
Yedder essahet temaḍ n ǎwetay d essahet temerwén d essa ezzar yemmut. 
Ila Noḥ semmoset temaḍ n ǎwetay as yekreh keraḍ arrawen ismawen nsn Sam, Ḥam d Jafet.
