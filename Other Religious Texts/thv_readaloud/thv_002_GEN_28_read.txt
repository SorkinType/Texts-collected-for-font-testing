Tizaret.
28.
Yeɣra Isḥaq Yaƹqub yeǧas elbaraka ezzar yenna has : « Wr téweyed tarrawt n Kenƹan. 
Enker ! As Fadan-Aram ɣor Betuwel, tis n mak. Awi tǎmǎṭ ɣors, iyyet dɣ eššéš n Laban, eŋŋas n mak. 
É Messineɣ wan Ǎneddabu a key yekfen elbaraka, arew, ǎqqel i ǎǧǧotnén baš éd teqqeled tis n ellametén ǎǧǧotnén. 
Yeǧét ak elbaraka tan Ibrahim key d iheyawen ennek baš éd teǧrewed ǎkal waɣ dɣ tezzeɣed tǎmosed ds ǎɣerib, ǎkal wa yekfa Yaḷḷah i Ibrahim. » 
Yessoka Isḥaq Yaƹqub Fadan-Aram ɣor Laban ag wan Arami Betuwel, eŋŋas n Rébéka mas n Esayu d Yaƹqub. 
Yelmed Esayu as Isḥaq yeǧa elbaraka i Yaƹqub, yesséweyt éd yeǧ éhen dɣ Fadan-Aram. Yelmed as yenna Isḥaq i Yaƹqub in wr éd yawi tarrawt n Kenƹan. 
Yeney Esayu as yelkem Yaƹqub awa has ennen imerawen ennét yekka Fadan-Aram. 
Yelmed Esayu as tarrawén n Kenƹan wr egréẓnet i tis. 
Yegla yosa Ismaƹil ag Ibrahim, yéwey tarrawt ennét Mahalaṭ welt-mas n Nebayoṭ, yessétét-in fol téḍéḍén ennét. 
Yefel Yaƹqub Beršéba, yekka berén Haran. 
Yensa dɣ édeg wa dɣ-t yeɣder ǎgǎḍel n tǎfuk. Yeṭkel aẓẓed yessomet déndɣ. 
Yehhorǧet, nɣwah awa yehhorǧet : tisekkukal ewarnet ǎmǎḍal, afella yewaḍ aǧenna. Ǎngelosen n Yaḷḷah tawenen errassen. 
Yebdad Amaɣlol dats iganna as : « Nk Amaɣlol, Messineɣ n abbaten ennek Ibrahim d Isḥaq. Ǎmǎḍal yemda wa fol tenséd éd hakt ekfeɣ key d iheyawen ennek. 
Iheyawen ennek éd umasin eǧǧeten hund tǎkessélt. Ǎkal nwn éd yefel aǧmoḍ n tǎfuk, yekka ǎgǎḍel n tǎfuk, afella d ǎteram. Tawsitén tin ǎmǎḍal emdanet éd eǧrewnet elbaraka-hin srk d iheyawen ennek. 
Nɣwah awa hé eǧeɣ : éd dk idaweɣ, éd key agẓeɣ is tekkéd, éd key-d ǎrreɣ ǎkal waɣ. Wr key éd eyyeɣ, éd eǧeɣ awas ak ennéɣ imda. » 
Yenker-d Yaƹqub dɣ éḍes ennét, yenna dɣ iman ennét : « Itbat as Amaɣlol illa déɣ beššan wr ss esséneɣ. » 
Yermeɣ, yenna : « Édeg waɣ yessukseḍ ! Itbat as tǎɣǎhamt n Yaḷḷah d émi n aǧenna ! » 
Yenker déndɣ-d yeffo. Yéwey aẓẓed wa yessomet, yessebdedt yǎmos as ǎmesenner n awa iǧen enḍéheḍ, yesseŋɣel fols ǎhatim. 
Iǧa isem i édeg wéndɣ Beytel (elmeƹna ennét éhen n Yaḷḷah), dat awén isem ennét Luz. 
Yesmeḍren Yaƹqub, yenna : « Kud Yaḷḷah yeddéw dɣi, yogaẓ ahi fol ǎbǎreqqa hin, ihak ahi a ekšéɣ d a elséɣ, 
kud eqqeleɣ-d eddareɣ eslameɣ ɣor abbahin, dédih Amaɣlol éd yeqqel Messineɣin. 
Aẓẓed waɣ essebdedeɣ, éd yumas éhen ennek key é Yaḷḷah, éd hak ekkateɣ elɣašer n awa hi éd tekfed. »
