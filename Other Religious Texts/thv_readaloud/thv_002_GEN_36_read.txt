Tizaret.
36.
Awah iheyawen n Esayu wa yǎmosen Edom. 
Yéwey Esayu téḍéḍén n Kenƹan : Ada welt Elon wan Ḥethit, Oholibama welt Ana, welt Sibɣon wan Hiwit, 
d Basmat welt Ismaƹil, welt mas n Nebayoṭ. 
Yekreh Esayu dɣ Ada, Elifaz d Basmat térew as Raƹuyil. 
Oholibama tekreh Yeƹuš, Yelam, Qoraḥ. Awén dg Esayu wi yekreh dɣ ǎkal n Kenƹan. 
Yéwey Esayu téḍéḍén ennét, meddans d eššéš, d ǎddunet win tǎɣǎhamt ennét, éheré ennét d awa yezzenha imda dɣ ǎkal n Kenƹan, ezzar yekka ǎkal yoǧǧeǧen ǎkal wa yezzeɣ eŋŋas Yaƹqub. 
Tela nsn ti mǎqqeret, wr eddobén éd ezzeɣen édeg iyyen, ǎkal wa dɣ ezzaɣen wr yeddobet éd yekf akesa i éheré nsn édét imaqqeren. 
Yezzeɣ Esayu tidreɣén tin Seƹir : Esayu enta as Edom. 
Awah iheyawen n Esayu, tis n kél Edom, dɣ tidreɣén tan Seƹir : 
Nɣwih ismawen n dg Esayu : Elifaz ag Ada, hennis n Esayu d Raƹuyil ag Basmat, hennis n Esayu. 
Dg Elifaz entenéḍ Téman, Omar, Sefo, Jaƹtam d Qenaz. 
Elifaz ag Esayu yekreh éy iyyen dɣ Timna tǎmǎt iyyet heḍet, yekreh ds éy, isem ennét Amalék. Awén iheyawen win Ada, hennis n Esayu. 
Nɣwih dg Raƹuyil : Nahat, Zera, Šamma d Mizza. Awén iheyawen win Basmat, hennis n Esayu. 
Nɣwih dg Oholibama, welt Ana welt Sibɣon, hennis n Esayu, térew as Yeƹuš, Yelam d Qoraḥ. 
Nɣwih imɣaren n dg Esayu : Dg Elifaz, wa yezzaren dɣ dg Esayu : amɣar Téman, amɣar Omar, amɣar Sefo, amɣar Qenaz, 
amɣar Qora, amɣar Jaƹtam, amɣar Amalék. Awén imɣaren win Elifaz dɣ ǎkal n Edom, awén iheyawen win Ada. 
Nɣwih dg Raƹuyil, ag Esayu : amɣar Nahat, amɣar Zera, amɣar Šamma, amɣar Mizza. Awén imɣaren n Raƹuyil dɣ ǎkal n Edom ; awén iheyawen win Basmat, hennis n Esayu. 
Nɣwih dg Oholibama, hennis n Esayu : amɣar Yeƹuš, amɣar Yelam, amɣar Qoraḥ. Awén imɣaren n Oholibama, welt Ana, hennis n Esayu. 
Wiɣ entenéḍ dg Esayu, awén imɣaren nsn. Esayu enta as Edom. 
Nɣwih dg Seƹir wan Horit, ǎmezzaɣ n ǎkal : Lotan, Šobal, Sibƹon, Ana, 
Dišon, Eser d Dišan. Awén imɣaren n Horiten, dg Seƹir dɣ ǎkal n Edom. 
Dg Lotan ǎmosen Hori d Hemam, welt mas n Lotan enta as Timna. 
Nɣwih dg Šobal : Alwan, Manaḥat, Eybal, Šefo d Onam. 
Nɣwih dg Sibƹon : Ayya d Ana. Ana a yosen téṭ n aman wi ekkusnén dɣ ténéré, awad yeḍan ihéḍan win tis Sibƹon. 
Nɣwih dg Ana : Dišon d Oholibama welt Ana. 
Nɣwih dg Dišan : Hemdan, Ešban, Yitran d Keran. 
Nɣwih dg Eser : Bilhan, Zaƹwan, Aqan. 
Nɣwih dg Dišan : Uṣ d Aran. 
Nɣwih imɣaren n Horiten : amɣar Lotan, amɣar Šobal, amɣar Sibƹon, amɣar Ana, 
amɣar Dišon, amɣar Eser, amɣar Dišan. Awén imɣaren n Horiten ak iyyen d tawsit ennét dɣ ǎkal n Seƹir. 
Nɣwih imenokalen wi emnakelnén dɣ ǎkal n Edom arweh wr ekréhen Israyiliten imenokalen : 
Béla ag Beƹor, yemnakel fol Edom, isem n aɣrem ennét Dinhaba. 
Yemmut Béla, yelkem as Yobab ag Zera wan Boṣra. 
Yemmut Yobab, yelkem as Hušam wan ǎkal wan Temaniten. 
Yemmut Hušam, yelkem as Hadad ag Bedad. Yewet Madian dɣ ǎkal wan Mowab, isem n aɣrem ennét Awit. 
Yemmut Hadad, yelkem as Semla wan Masreqa. 
Yemmut Semla, yelkem as Šaul wan Rehobot ɣor ǎɣǎher. 
Yemmut Šaul, yelkem as Baƹl-Hanan ag Akbor. 
Yemmut Baƹl-Hanan ag Akbor, yelkem as Hadar ; isem n aɣrem ennét Faƹu. Isem n hennis Mehetabeel welt Matred welt Me-Zahab. 
Nɣwih ismawen n imɣaren n Esayu, ak iyyen d tawsit ennét d ǎkal ennét : amɣar Timna, amɣar Alwa, amɣar Yatét, 
amɣar Oholibama, amɣar Ela, amɣar Finon, 
amɣar Qenaz, amɣar Téman, amɣar Mibṣar, 
amɣar Magdiel, amɣar Iram. Awén imɣaren n Edom ak iyyen d awad yezzeɣ dɣ ǎkal wa dɣ iǧa tela. Awén iheyawen win Esayu tis n Edom.
