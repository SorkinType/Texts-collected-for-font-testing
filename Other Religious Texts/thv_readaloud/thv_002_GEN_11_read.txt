Tizaret.
11.
Ǎddunet emdan sawalen awal iyyen d tifér tiyyeḍ. 
Ǎddunet efelen berén awas d tǧammeḍ tǎfuk osen édeg iyyen dɣ asyaf dɣ ǎkal n Šinƹar ǎqqimen ds. 
Ennen ǧérésn : « Eywah ! Neknét oṭṭében ezzar éd tn enseŋ dɣ témsé ! » Oṭṭében ǎmosen asn éẓẓeden d ǎbeleɣleɣ teqqal asn tǎlaq ta temmalet ǧér oṭṭében. 
Ennen arweh : « Eywah ! Nexdemét baš éd neken aɣrem d elburj iyyen yéwaḍen aǧenna ! Éd nekreh tisellet, wr éd nemmezzey fol ǎrori n ǎmǎḍal imda. » 
Yeres-d Amaɣlol itassen ma-d yola aɣrem d elburj wi kannen ǎddunet. 
Ezzar yenna Amaɣlol dɣ iman ennét : « Ellamet iyyet, awal iyyen ! Nɣwah awas ebden emuken ennét, kud ebden hund awaɣ éd eknin awa eran imda. 
Eywah ! Endawtet neres, éd ensemmerti awal nsn baš éd ǎbas éd tanefhamen ǧérésn. » 
Yezmezzey tn Amaɣlol fol ǎmǎḍal imda, ezzar oyyen elbeni n aɣrem.. 
Fol awéndɣ as aɣrem iǧa isem Babel, -ǎsemmerti-, édét Amaɣlol déndɣ ad yesmertey awal n ǎddunet, yesmertey-tn fol ǎrori n ǎmǎḍal imda. 
Awah iheyawen n Sam. Ila Sam téméḍé n ǎwetay as yewa rurés Arfakšad, essin iwetyan ḍeffer anǧi wa mǎqqeren. 
Ḍeffer tiwit n Arfakšad yedder Sam semmoset temaḍ n ǎwetay. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Arfakšad keraḍet temerwén n ǎwetay d semmos yekreh arraw isem ennét Šelaḥ. 
Ḍeffer tiwit n Šelaḥ yedder Arfakšad ekkoẓet temaḍ n ǎwetay d keraḍ. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Šelaḥ keraḍet temerwén n ǎwetay yekreh arraw isem ennét Eber. 
Ḍeffer tiwit n Eber yedder Šelaḥ ekkoẓet temaḍ n ǎwetay d keraḍ. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Eber keraḍet temerwén n ǎwetay d ekkoẓ yekreh arraw isem ennét Feleg. 
Ḍeffer tiwit n Feleg yedder Eber ekkoẓet temaḍ n ǎwetay d keraḍet temerwén. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Feleg keraḍet temerwén n ǎwetay yekreh arraw isem ennét Reƹu. 
Ḍeffer tiwit n Reƹu, yedder Feleg senatet temaḍ n ǎwetay d teẓẓa. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Reƹu keraḍet temerwén n ǎwetay d essin yekreh arraw isem ennét Serug. 
Ḍeffer tiwit n Serug yedder Reƹu senatet temaḍ n ǎwetay d essa. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Serug keraḍet temerwén n ǎwetay yekrah arraw isem ennét Naḥor. 
Ḍeffer tiwit n Naḥor yedder Serug senatet temaḍ n ǎwetay. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Naḥor senatet temerwén n ǎwetay d teẓẓa yekreh arraw isem ennét Teraḥ. 
Ḍeffer tiwit n Teraḥ yedder Naḥor téméḍé n ǎwetay d meraw d teẓẓa. Yekreh arrawen d tarrawén heḍnén. 
As yekreh Teraḥ essahet temerwén n ǎwetay yekreh keraḍ arrawen Abram d Naḥor d Haran. 
Awah iheyawen n Teraḥ : Teraḥ yérew Abram d Naḥor d Haran. Yekreh Haran arraw isem ennét Loṭ. 
Haran yemmut arweh yeddar tis Teraḥ dɣ Ur ǎkal wa dɣ yewa dɣ ǎkal n Kaldé. 
Yewey Abram Saray, Naḥor yéwey Milka wlt Haran. Haran tis n Milka aked n Yiska. 
Saray, tǎmeǧǧaǧrut, wr telé arraw wl iyyen. 
Yéwey Teraḥ rurés Abram d aheya ennét Loṭ ag Haran d tǎḍeggalt ennét Saray, hennis n Abram, efelen Ur dɣ ǎkal n Kaldé ekken ǎkal n Kenƹan, as éweḍen Ḥaran ǎqqimen ds. 
Yedder Teraḥ senatet temaḍ n ǎwetay d semmos ezzar yemmut dɣ Haran.
