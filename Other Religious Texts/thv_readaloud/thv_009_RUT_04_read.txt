Raɣut.
4.
Yekka Boɣaz émi n aɣrem, yeqqigm, émér iyyen yokey-in ales wanǧam s iganna, wa yoŋŋaten d Alimelek. Yenna has Boɣaz : « Aheẓ, ɣayem deh. » Yoheẓ in ales, yeqqim. 
Dédih yeɣra Boɣaz meraw imɣaren n aɣrem, yenna hasn : « Γeymet deh », ǎqqimen. 
Yenna has i ales wanǧam : « Ǎfǎraǧ wan eŋŋa- nnɣ Alimelek, zanhé hin Naɣomi, ta-d tefelet ǎkal n Mowab. » 
Yenna has : « Ǎfǎraǧ waɣ id-t izzenhen telléd asd seres key, édét key wa yezzaren, édét key a yoŋŋaten d Alimelek, awaɣ as ak tn ennéɣ dat imɣaren-nnɣ éd ǎmosen tiǧuhawén, kudét zanhéd-t, ennén dɣ eljemat, kudét wrt teréd ennén dɣ eljemat, éd nelmed awa hé yeǧin, édét key ǎmuzer, nk ḍefferk ad elléɣ, nk, as tennéd nk wr teǧǧed awaɣ, tekféd-i tefért ennek n-innin wrt eddobéɣ awaɣ éd-t aǧeɣ, eǧé key, ɣor déndɣ éd awreɣ nk. » Yenna has ales wah : « Ayoh, ǎréɣ édt zenheɣ. » 
Beššan yenna has Boɣaz : « Ahel wa-d éd tzenhed ǎfǎraǧ wéndɣ fol Naɣomi, éd tzenhed aked Raɣut telmowabit, hennis n wa eba, baš éd tekrehed arrawen hé umasnin meddans n wa eba fol tǎkasit ennét. » 
Yenna has ales wanǧam : « Γor déɣ-d iǧǧeh ǎdewenné wan tediwt, nk ahel waɣ eǧmeḍeɣ. Wr eréɣ éd zenheɣ, édét wr eréɣ éd enfeẓeɣ tela-hin. Oyyéɣ ak édeg, zenh key, nk wrt eddobéɣ. » 
Dɣ ezzeman wéndɣ dɣ Israyil, tellé elɣada tah, tamos tǎmǎtart : ɣor inezzan méɣ ɣor ǎsemmeskel n heret, as tekna taɣera iyyet, éd yekkes ǎwadem eɣǎtém ennét édt yekf i wa heḍen. Awéndɣ a yǎmosen tǎǧuhé. 
Yenna ales wanǧam i Boɣaz : « Zenh key. » Ezzar yekkes eɣǎtém ennét. 
Yenna Boɣaz i imɣaren d ǎddunet widt ellenén emdan : « Tešhadem ahel waɣ as zanhéɣ i Naɣomi, tela d ǎrrezɣen n Alimelek emdan d win Maxlon d Kilyon meddans, 
éweyeɣ aked Raɣut telmowabit, hennis n Maxlon, baš éd teqqayem tǎkasit n Maxlon dɣ ǎddunet ennét, baš éd yedder isem ennét ǧér ayet-mas d dɣ émi n aɣrem waɣ. Ahel waɣ tešhadem fol awén. » 
Ǎddunet emdan wi ellenén ɣor émi n aɣrem ennen : « Nešhad »; ennen imɣaren : « É Messineɣ a yessolen tǎmǎṭ ta hé teǧǧehet ihenan ennek d Rašél d Leya, essenetésnet, annǎtén n Kél-Israyil. É Messineɣ a key yekfen isem déɣ, dɣ Beytlaḥm d tela déɣ, dɣ Efrata. 
É Messineɣ a key yerzeɣen s inuba dɣ tǎmǎṭ taɣ, yumasét éhen ennek hund wan Fereṣ, wa térew Tamar i Yehuda. » 
Yéwey Boɣaz Raɣut. Éheḍ héndɣ ad iǧa éhen, yerzeɣt ds Messineɣ s arraw n éy. 
Ennenet téḍéḍén i Naɣomi : « Elbaraka n Messineɣ wa kem yekfen ahel waɣ wa kem éd yagẓin. Yetwezzayét isem ennét dɣ Israyil ! 
Éd kem-d yer s tǎmeddort, éd ham yilal dɣ tewǎhré ennem. Édét mas ta-d térewet tessimɣer kem, tufam essa arrawen. » 
Teṭkel Naɣomi arraw, teswert éfef ennét, enta adt yessedwelen. 
Ennenet tinaraǧén : « Tekreh Naɣomi arraw. » Eǧenet as isem Γubeyd. Γubeyd waɣ a yérewen Yessa, tis n Dawud. 
Awah abbaten n Dawud mn Fereṣ : Fareṣ tis n Hesron, 
Hesron tis n Ram, Ram tis n Amminadab, 
Amminadab tis n Našon, Našon tis n Salmun, 
Salmun tis n Buɣaz, Buɣaz tis n Γubeyd, 
Γubeyd tis n Yessa, Yassa tis n Dawud.
