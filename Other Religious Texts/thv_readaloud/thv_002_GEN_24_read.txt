Tizaret.
24.
Weššar Ibrahim, yekna tewehré hullan. Iǧas Amaɣlol elbaraka fol ak heret. 
Ahel iyyen yenna Ibrahim i wa yezzaren dɣ éklan ennét, wa yǎmosen amɣar nsn : « Eǧ ǎfus ennek dag taɣmahin. 
Éd key zehḍeɣ s Amaɣlol, Yaḷḷah n aǧenna d ǎmǎḍal, in wr éd teǧmeyed i ruré tǎmǎṭ n ǎkal waɣ n Kenƹan, ǎkal waɣ dɣ ezzaɣeɣ. 
Beššan éd tekked ǎkalin wan tidet ɣor ǎddunetin ezzar éd hi teǧmeyed tǎmǎṭ i ruré, Isḥaq. » 
Yenna has akli : « Tiǧenén tǎmǎṭ éd tugay éd hi telkem dɣ ǎkal waɣ. Éd awyeɣ dédih rurék s ǎkal wa tefeled ? » 
Yenna Ibrahim : « Kela. Hékey éd tawyed ruré séndɣ. 
Amaɣlol, Yaḷḷah n aǧenna, wa hi yesfelen tǎɣǎhamt n abbahin d ǎkal wan ǎddunetin, yesséwel ahi yeheḍ ahi in éd yekf ǎkal waɣ i iheyawenin. Enta a hé isiwyin ǎngelos ennét datk baš éd-d tawyed tǎmǎṭ i ruré. 
Beššan kud wr teré tǎmǎṭ éd hak telkem, dédih éd key yefel éhéḍ wa teǧéd beššan wr teweyed ruré sendɣ fo ! » 
Yeǧa akli ǎfus ennét dag taɣma n messis Ibrahim ezzar yeheḍ as in éd yeǧ awa has yenna. 
Yéwey meraw imnas dɣ imnas win messis. Yéwey hereten wi tn unfenén dɣ win messis. Yegla yekka ǎkal wan Aram-Ennahreyn dɣ aɣrem wan Naḥor. 
As yéweḍ ɣor anu wa yoheẓen aɣrem, yesgen imnas ɣor anu. Awén iẓelluwaẓ émér wad harreǧnet téḍéḍén aman. 
Yenna : « É Amaɣlol, Messineɣ n messi Ibrahim, ahi yellilen éd yeǧ amhiyyu ahel waɣ. Seken ahi sémɣar ennek i Ibrahim. 
Elléɣ ɣor téṭ, tarrawén ti ezzeɣnén aɣrem éd-d asinet ehraǧnet. 
Éd egmeyeɣ dɣ iyyet éd hi tekf aman baš éd esweɣ. Kud tenna : "Esu, éd ekfeɣ aman aked i imnas ennek", éd elmedeɣ dédih as enta a teréd éd tt yawi Isḥaq. Éd elmedeɣ adih as tessimɣered messi. » 
Animér wr yesmenda awa iganna as in tosa Rébéka welt Betuwel, wa érewen Milka d Naḥor, eŋŋas n Ibrahim. Teway elgeḷḷa ennét fol éẓér. 
Tǎmos tǎmawaṭ tǎhoseyet, atewa wr tezzéy médden. Teres s téṭ, teṭker elgeḷḷa ennét ezzar téwen. 
Yohelin srs akli n Ibrahim yenna has : « Inhod, seswahi dɣ aman n elgeḷḷa ennem. » 
Tenna has : « Esu, Sidi ! » Tesres-in elgeḷḷa ennét hik tekfé aman. 
As tesmenda tenna has : « Éd ehreǧeɣ aman i imnas ennek baš éd eknin tésesé aked entenéḍ. » 
Termeḍ tesseŋɣel aman dɣ édeg wa dɣ sassen imnas ezzar tohel tehreǧ animér. Tesseswa imnas emdan. 
Yekyad tt ales, yessossem, itaxemmam in Amaɣlol yellil as méɣ dɣ asikel ennét. 
As esmenden imnas tésesé, yekfa ales i tǎmawaṭ tiǧemt n oreɣ yogdan d aɣil n erriyal d essin ihebgan n oreɣ ogdanén d meraw erriyalen. 
Yessesten tt : « Inhod leɣtahi mi kem yérewen ? D leɣted ahi kud illé édeg ɣor tim baš éd enseɣ éheḍ waɣ nk d wi dɣi eddéwnén ? » 
tenna has : « Nk welt Betuwel ag Milka d Naḥor. 
Γornɣ illé édeg wa dɣ éd tensim d iškan d ǎrummu s ǎǧut. » 
Yeneh ales, yerkeƹ dat Amaɣlol. 
Yenna : « Ǎborekeɣ Amaɣlol, Messineɣ wan messi Ibrahim, wa yogaẓen ellemana d sémɣar ennét i Ibrahim. Yeznehel ahi ɣor ǎddunet n messi déndɣ yezzaren. » 
Tohel tǎmawaṭ s tǎɣǎhamt n mas éd teǧ isǎlan. 
Rébéka tela eŋŋas igan isem Laban. Yegmeḍ Laban hik yosa ales ɣor teṭ. 
Édét yenay tiǧemt d ihebgan dɣ ǎfus n welt mas, yeslâ i isǎlan wi yeǧa ales i Rébéka, yosa ales enta d imnas ennét ɣor téṭ. 
Yenna has : « As-d ɣori, key wa ilen elbaraka n Amaɣlol. Mafol teqqimed dɣ ténéré ? Ezwejjedeɣ tǎɣǎhamt, imanin, eknéɣ édeg i imnas. » 
Yekka ales ɣor Laban. Yesseɣser Laban imnas, egrewen iškan d ǎrummu. Egrewen enta d ǎddunet wi ds eddéwnén aman éd serdin éḍaren nsn. 
Yeǧrew imekšan beššan yenna : « Wr tetteɣ ar eskundebat ennéɣ awas eréɣ éd tn enneɣ. » Yenna has Laban : « Siwel adih ! » 
Yenna ales : « Nk, akli n Ibrahim. 
Amaɣlol iǧa elbaraka mǎqqeret i Ibrahim, wan messi, yeɣmer, yeqqel ales mǎqqeren édét yekfé Yaḷḷah éheré wa ǧezzulen d wa heǧrén d aẓref d oreɣ, éklan d téklatén, imnas d iheḍan. 
Hennis Sara, dɣ tewehré ennét, tekfa arraw i messi, yekfé tela ennét temda. 
Yezheḍi messi Ibrahim, yenna : "Wr tǧemmiyed tǎmǎṭ i ruré dɣ šét Kenƹaniten, dɣ ǎkal waɣ ezzaɣeɣ. 
Yuf tekkéd ǎddunet n ti, dɣ tawsitin baš éd tawyed tǎmǎṭ i ruré". 
Ezzar essestenneɣt : "Kud tuǧey tǎmǎṭ éd dɣi tidaw ?" 
Adih yenna hi : "Amaɣlol, was dat erriǧeheɣ abedǎh, éd yesiwiy ǎngelos ennét, yeddéw dk, éd hak yilal dɣ ǎsikel ennek baš éd tawyed tǎmǎṭ i arrawin ɣor ǎddunet n ti. 
Beššan kud tekkéd ɣor ǎddunet n tawsitin, kud uǧeyen ak éd key ekfin tǎmawaṭ, dédih yefal key éhéḍ wa hi teǧéd". 
As-d éweḍeɣ ahel waɣ ɣor téṭ ennéɣ : "Amaɣlol, Messineɣ n messi Ibrahim, inhod a hi telliled dɣ ǎsikelin. 
Adih nk deh elléɣ deh ebdadeɣ dat téṭ taɣ, dédih as-d teǧmeḍ tǎmawaṭ, tehraǧet, éd has enneɣ : ”Inhod ahi tekféd heret n aman n elgeḷḷa ennem'. 
Adih, kud tenna hi : “Esu key, éd sesweɣ aked imnas ennek', éd elmedeɣ as enta, tǎmǎṭ ta yesinefren Amaɣlol i rurés n messi. 
Arweh wr esmendeɣ awal dɣ imanin, Rébéka nɣtah, tosed, tǎsiwer elgeḷḷa éẓér, teras s téṭ, tehraǧ aman. Ennéɣ as : "Inhod ahi tekféd aman". 
Tesres-d elgeḷḷa ennét ta téway hik, tenna hi : "Esu key, ezzar éd sesweɣ imnas ennek." Eswéɣ ezzar tesseswa imnas. 
As tt essesteneɣ mi tt yérewen, tenna : “Nk welt Betuwel ag Naḥor d Milka. » Eǧéɣ as tiǧemt dɣ tinhar d ihebgan dɣ ifassen. 
Eneheɣ, erkeƹeɣ dat Amaɣlol, Messineɣ n messi Ibrahim, emmoyéɣ as, édét yesséwey ahi dɣ ǎbǎreqqa wan ellemana, yesséwey ahi s tǎheyawt n eŋŋas n Ibrahim. Ǎddobéɣ éd tt awyeɣ i rurés n Ibrahim. 
Dimaɣ, kud teknam sémɣar i messi, teǧam ds ellemana, ennét ahi tn, kud kela, ennet ahi tn. Nk éd semmelelleyeɣ a iǧen. » 
Nɣwah awa has ennen Betuwel d Laban : « Amaɣlol, iman ennét, a yesiweggen awén. Wr nelé a hé nen, ula a yehoseyen ula a yexlan. 
Rébéka nɣtah, datk. Awi tt. Éd tumas hennis n rurés n messik, hund awa yenna Amaɣlol. » 
As yesla akli n Ibrahim awén, yerkeƹ ar ǎmǎḍal dat Amaɣlol. 
Yekkes-d dɣ kaya ennét oɣnen n aẓref d oɣnen n oreɣ d iselsa, yekfén i Rébéka. Eǧrewen eŋŋas d mas n tǎmawaṭ hereten eknanén tihusay aked entenéḍ. 
Ḍeffer awén, ekšen, eswen, enta d ǎddunet wi ds eddéwnén ezzar ekken éḍes. As yeffo, enkeren, yenna akli n Ibrahim : « Eyyet ahi éd eqqeleɣ ɣor messi. » 
Ennen as : « Tǎqqeymét tǎmawaṭ ɣornɣ iheḍan wiyyeḍ, meraw iheḍan arweh ezzar teglét. » 
Yenna akli : « Wr hi téwéɣem, dimaɣ-d hi yellil Amaɣlol dɣ ǎsikelin, eyyet ahi éd eqqeleɣ ɣor messi. » 
Adih ennen : « Neɣrét tǎmawaṭ éd tt nesesten enta » 
Eɣren Rébéka essestenen tt : « Teréd éd tegled teddéwed d ales waɣ ? » Tenna : « Éwalla. » 
Eɣbelen éd eyyin Rébéka éd tegel teddéw d ta tt tessedwelet. Eglenet eddéwnet d akli n Ibrahim d ǎddunet wi ds eddéwnén. 
Émér wan téklé, eǧen i Rébéka elbaraka nsn ennen as : « Kem, tǎmeḍrayt nnɣ umas mas n iǧiman d iǧiman n ǎddunet. Iheyawen ennem awyin iɣermen n ihenǧa ennem ! » 
Enkernet Rébéka d téklatén ennét, oɣarnet imnas, eǧlenet elkamnet i ales. Yéwey akli Rébéka, yegla. 
Émér héndɣ, yefel-d Isḥaq anu wan Lahay-Roy, yezzaɣ ǎkal n Negeb. 
Ahel iyyen d igmaḍ s iẓelluwaẓ, yerriǧeh fol iman ennét, yeṭkel éɣef, yemheyyet ǎsewaḍ ennét d imnas wi hin eglenén. 
Teṭkel Rébéka ǎsewaḍ ennét aked enta, teney Isḥaq, teres amis 
tessesten akli : « Ma yǎmos ales wén haneɣ-d inkaden dɣ ténéré ? » Yenna : « Awén messi. » Teswer afer udem ennét. 
Yenna akli i Isḥaq awa iǧen imda. 
Yessofeḍ Isḥaq Rébéka s éhen n mas, Sara, yéwey tt, teqqel hennis, yeɣhel tt. Tera taɣ has iǧa, tellil as fol iba n mas.
