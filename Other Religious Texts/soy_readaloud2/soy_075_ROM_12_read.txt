Rɔm.
12.
Pimɑ́rɛcɔ, lɛ̃ kɛ Uléécɑɑ uu nní íwɛ rɔ́ n tẽ́ nnyɑ, nɛ nɔ́ ŋmɔ́ɔ́púsɛlɛnlɛ rɛ ɑni nɔ́mɛnɛ́círɛ kpísi ɑni u hɛ yɑrɛ inyɔ́ɔnsɛsɛɛ kɛcirɛ. Inyɔ́ɔnsɛ iyɛ̃ ikɛ́ n kpɑ̃́iilɛ̃ in kɔ nɛ u sɑ́. Lɛ̃ kɑni yɛ́ u yɑ́ɑ́si ɑi ɑsei m mɑ́. 
Áni kɑpɛ kɛtẽ nté pisoi mɛwɑi n tikilɛ̃. Amɑ́ ɑni tíyɛsɛ Uléécɑɑ ukɛ́ mɛsɔhɔ fɑlɛ nɔ́ hɛ mɛkɛ́ nɔ́ consɛsɛ. Kei kɑni yɛ́ pɛɛ́ nɛ Uléécɑɑ mɛlɑ ceri. Nɔ́ lisɔnɛ nɛ lɛlɛɛ yɛ u n lɑrisi nɛ lɛlɛɛ ń nyɑḿ nɛ kɛ́tɔ-pɔ ceri. 
Kɛfɑ kɛ Uléécɑɑ uú nɛ nɛ́ n fénnɛ nnyɑ, nɛ lɑ kɛ́ nɔ́nnɛ́nɛ́í mɑɑ rɛ úkɑ úu kɑpɛ yɛ umɛcirɛ n tɑɑ́sɛlɛ̃ likɛ́ tɔsí, ɑ́i kɔ pɛɛ ɑsei mɑ́. Amɑ́ ɑni yɛ kɛmúŋɛ́ wɑ kɛkɛ́ nɛ n sɑ́. Uléécɑɑ yɛ ḿpɑ́ úye nfɑtɛnɛ nɛ kumúŋɛ́ kunyinɛ hɛlɛ. Ḿpɑ́ úye ukɛ́ nɛ ukɛpɑ́ m pɑ́lɛ̃. 
Ipiŋɛ isɛ kɛ usoi uu mɑ́, iyɛ̃ in ilésɛ kulúi mɑ́, ilésɛ iyɛ̃ in pikɛi pikɔ́ pikɔ́ wɑi. 
Limɛcɔ ḿpɑ́ kɑri nní kulúi ń we, tɔ wɑlɛ ipiŋɛ isɛ tɔ́ nɛ Kirisi kupɛ́nɛcɔ kɛ́mɛɛ, tɔn ntɔ́nɛ́í mulɛinɛlɛ̃ yɑrɛ kɛ usoi ipiŋɛ ilesɛ ii nní m mulɛinɛlɛ̃ mɛcɔ. 
Amɑ́ ḿpɑ́ úye nɛ ulikɔ́ yɛ ni kɛ Uléécɑɑ uú nɛ nfɑnɑrɛ u n hɛ. Li pisɛ rɛ ḿpɑ́ úye ukɛ́ ihɛɛ iyɛ̃ kuu n yɔsí nɛ pikɛi ńsɔnɛ wɑ. Uléécɑɑ un úye n hɛ rɛ ukɛ́ ɑntepu pikɛi n wɑi, liute ukɛ́ yɛ Uléécɑɑ rinyiri nɛ símisi uu lipikɛi nɛ unfɑtɛnɛ nnɛ́í n wɑi. 
Uléécɑɑ un úye n hɛ rɛ ukɛ́ picɔ lɛ̃́, ukɛ́ yɛ upikɛi n wɑi lin nɛ pisoi lɛ́ni. Uléécɑɑ un úye n hɛ rɛ ukɛ́ n céési, li pisɛ rɛ ukɛ́ n céési. 
Uyɛ̃ kuu n hɛ rɛ ukɛ́ yɛ picɔ ikɑri rikpɑ́sɛ, ukɛ́ picɔ ikɑri n kpɑriisɛ. Uyɛ̃ kuu n hɛ rɛ ukɛ́ kɛfɑ n lɑ́ɑ́rú, ukɛ́ yɛ nɛ nfɑnɑrɛ hɛ. Nkó yɛɛ iwɛ́ɛ́sɛ n tṹ ukɛ́ ńsɔnɛ upikɛi n wɑi. Nkó yɛɛ píwɛkɔɔ́ n lɛ́ni pɔ́ɔkɛ́ yɛ pi lɛ̃́ nɛ mpɔ́ɔnɑrɛ. 
Ńlɑ ńn kɑpɛ kɛcɑ́ɑ́ kɛcɑ́ɑ́ nkɔ́. Ani mɛwɑi kópɛ n kɔ́hɔ. Ani lisɔnɛ piwɑi nɛ kɛ́yu m mɑhɑni. 
Ani n lɑkɑɑnɛ yɑrɛ kɛ pimɑ́rɛcɔ ɑpi yɛ n lɑ́kɑɑnɛ mɛcɔ. Ḿpɑ́ úye ukɛ́ yɛ uucɔ pɑkɑrɛ uu yɛ ríyu u wɑisɛ. 
Ani kɛkɛi-i n tolú, ɑ́ni kɑpɛ sikɑ̃́ m mɑ́. Ani n ŋmɔ́ɔ́púlɛ̃ nɔn kɛfɑ kɛsɛ nɛ Upíimɑ pikɛi wɑi. 
Nɔ́rinɛ́pɔ́ɔ rikɛ́ lɛ̃ kɑni n tɑ́lɛ̃ nɔn mɛ̃́ nnyɑ n lɑ́ɑ́rú. Ani lɛlɛɛ nɔ́ n wéékusɛlɛ̃ nɛ n niŋú. Ani piyómɛyɑ́hɑɑ nɛ kɛ́yu m mɑhɑnlɛ̃. 
Ani yɛ Uléécɑɑ pikɔ́ pɛɛ ncɔ́ŋ kɛ́mɛɛ ń we lɛ̃́. Áni kɑpɛ yɛ pisɑ́nɛyɔɔ́ nɛ sikɑ̃́ wɑ. 
Ani yɛ pɛpɛɛ nɔ́ n wéékusɛlɛ̃ tinɔ́ɔ sɔnɛ ritikisɛ. Ani yɛ pisoi rinɔ́ɔ sɔnɛ ritikisɛ, ɑ́ni kɑpɛ yɛ úkɑ ɑnɔ́ɔ wɑ. 
Píye pin mpɔ́ɔnɑrɛ kɛ́mɛɛ n we, nɔ́ nɛ pɛ̃ ɑni kɛsẽ́ mpɔ́ɔnɑrɛ-i n we, píye pin n téni, nɔ́ nɛ pɛ̃ ɑni kɛsẽ́ tẽ́. 
Nɔ́ nɛ nɔ́pinɛ́cɔ ɑni nfɑsimɛ́ m pɛ́nɛlɛ̃. Áni kɑpɛ limɑ́ɑ-mɑɑ nɛ líyukɔɔ́ nɛ nɔ́mɛnɛ́círɛ m músu. Amɑ́ ɑni yɛ lɛ̃ kɛ úkɑ úu nɛ kutu n cɔ́lɛ̃ kuwɛ́ɛ́-mɛ̃ nɔ́kɛnɛ́múŋɛ́ pɛsɛ. Áni kɑpɛ nɔ́mɛnɛ́círɛ n nyɑ́ni rɛ nɔ mɛwɛ́ɛ́sɛsɔhɔ mɑ́lɛ. 
Áni kɑpɛ yɛ unyinɛ mɛwɑi kópɛ nɛ mɛwɑi kópɛ hɛ́ɛlɛ. Ani riŋmɔ́ɔ́pú ɑni yɛ ḿpɑ́ úye kɛyu-i lisɔnɛ wɑi. 
Lin yo kuwɑi n we, in tɛ nɔ́kɛnɛ́mɛɛ́ kɑi yɛ́ léemɛ nɔ́ nɛ ḿpɑ́ úye nɔkɛ́ nɛ n lɑ́ɑ́rúnɛ, ɑni li wɑ. 
Pinɛ́sɑ́nɛ lɑlɑ kɛcirɛ, ɑ́ni kɑpɛ yɛ mɛwɑi kópɛ kɛ pisoi ɑpi nɔ́ n wɑ nɔ́rinɛ́cúruu pi hɛ́ɛlɛ, ɑmɑ́ ɑni yɛ riyɑ́ Uléécɑɑ ricuruu ukɛ́ ukuwɔi pi nyísɛ. Li Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ Upíimɑ yɛ mɑɑ rɛ: Nɛ́ɛ yɛ usoi mɛwɑi kópɛ u hɛ́ɛlɛ. Nɛ́ɛ yɛ ḿpɑ́ úye mɛwɑi u cɛɛi. 
Li kɔ wɔlɑɑlɛ̃ tɛ: Nkṹ nn upɔ́lɑ́ɑrɔ n we, ɑ u hɛ ukɛ́ li, nnírɛ́ nn u n we, ɑ u hɛ ukɛ́ n ntí. Pɔn yɛ lɛ̃ n wɑ, ɑnɑkɑlɛ wɛɛ́ kɑɑ lɛ̃ uriyu kɛcɑ́ɑ́ kɔɔnu. 
Kɑpɛ tíyɛ mɛwɑi kópɛ mɛkɛ́ pɔ́ feriyɛ, ɑmɑ́ pɔ́ɔkɛ́ lisɔnɛ n wɑi ɑɑ́ nɛ mɛwɑi kópɛ feriyɛ.
