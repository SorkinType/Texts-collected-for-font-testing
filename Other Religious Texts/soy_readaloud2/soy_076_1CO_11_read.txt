1 Korɛnti.
11.
Ani mɛnɛ́cɔ n wɑi yɑrɛ kɛ nɛ́ɛ Kirisi mɛcɔ n wɑi. 
Nɛ nɔ́ pɑkɑrɛ kɑní nɛ ḿpɑ́ píyei kɛnɛ́cɑ́ɑ́ n léisɛlɛ̃ nɔn kɔ icélɑɑ kɑm nɔ́ rintɛ́ŋ́sɛ ńsɔnɛ múílɛ̃. 
Ḿpɑ́ nɛ lɛ̃, nɛ lɑ rɛ ɑni mmú kṍ tɛ Kirisi yɛɛ ḿpɑ́ utisi úye uyɔɔpi. Utisi yɛɛ unɔ́si uyɔɔpi, Uléécɑɑ pɔ́ɔ Kirisi uyɔɔpi. 
Utisi nkó yɛɛ linyinɛ ríyu n yɑ́riilɛ̃ un nɛ kɛyómɛ yɑ́ɑ́si nɛ́ɛ un nɛ Uléécɑɑ rinyiri símisi yɛ uuyɔɔpi Kirisi isɛi wɑisɛlɛ. 
Amɑ́ unɔ́si nkó yɛɛ ríyu ŋmɑnɛ nɛ kɛyómɛ n yɑ́ɑ́si nɛ́ɛ un nɛ Uléécɑɑ rinyiri símisi yɛ mɛníŋɛ uulɑ isɛi wɑisɛlɛ. Li we rɛ unɔ́si nkó kɑpi rinkoósi rɛ isɛi ikɛ́ u wɑ kuu lɛ̃ nɛ mɛnyínɛ wɑ. 
Unɔ́si unsɑ́ n lɑ ukɛ́ uriyu linyinɛ yɑrii, ukɛ́ mɛsɛ nnyúpi nyɑi. Li kɔ pɛɛ isɛi welɛ rɛ unɔ́si ukɛ́ n nyɑlɑɑlɛ̃ nɛ́ɛ ukɛ́ n koósilɛ̃. Lɛ̃ nnyɑ, ukɛ́ yɛ ríyu linyinɛ yɑrii. 
Ái nɛ sɑ́ rɛ utisi ukɛ́ linyinɛ ríyu yɑrii. Li we rɛ uyɛɛ nɛ Uléécɑɑ mɛnyínɛ we, un kɔ ríyu u wɑisɛ. Utisi kɛ unɔ́si pɔ́ɔ ríyu wɑisɛ. 
Ái unɔ́si ipiŋɛ kɛ Uléécɑɑ uu kpísi uú nɛ utisi wɑi, ɑmɑ́ utisi ipiŋɛ kuu kpísi uú nɛ unɔ́si wɑi. 
Ái kɔ unɔ́si nnyɑ kɛ Uléécɑɑ uu utisi wɑ, ɑmɑ́ utisi nnyɑ kuu unɔ́si pɔ̃́ wɑ. 
Lɛ̃ nnyɑ kɑi pisɛ rɛ unɔ́si ukɛ́ piléécɑɑtumɛ nnyɑ linyinɛ ríyu yɑrii likɛ́ n nyísɛlɛ̃ tɛ u umɛcirɛ utisi mɛtene ricɛpisɛlɛ. 
Amɑ́ Upíimɑ kɛ́mɛɛ́, utisi unsɑ́ n we, unɔ́si úu líkɑ tu, unɔ́si unsɑ́ n we, utisi pɑ́ɑ kɔ líkɑ tu. 
Yɑrɛ lɛ̃ kɑpi utisi ipiŋɛ n kpísi ɑpí nɛ unɔ́si wɑi mɛcɔ kɛ unɔ́si uu yɛ utisi mɑri. Uléécɑɑ yɛɛ lɛ̃ nnɛ́í wɑi. 
Ani nɔ́rinɛ́cúruu musí ɑni yɛ́nu nkɛ̃́ tɛ li nɛ sɑ́ rɛ unɔ́si ukɛ́ ríyu ŋmɑnɛ Uléécɑɑ n yɑ́ɑ́sii? 
Lɛ̃ kɑi nní kɛtẽ kɛcɑ́ɑ́ ń we ricuruu yɛ rɔ́ nyísɛlɛ rɛ li isɛi we rɛ utisi ukɛ́ nnyúpi riyɑ́ nkɛ́ pi nn cɑ́. 
Amɑ́ li pɛɛ ríyu mɑ́ rɛ unɔ́si pɔ̃́ nnyúpi nkɛ́ pi nn cɑ́. Lɛ̃ nnyɑ kɑi unɔ́si nnyúpi hɛ rɛ nkɛ́ yɛ ríyu u n yɑriilɛ̃. 
Amɑ́ úye unsɑ́ n lɑ ukɛ́ lɛ̃ wɑ, liute ukɛ́ nsɛ kõ tɛ tɔ̃́ ɑ́ri mɛwɑi féé mɛ́kɑ mɛcɔ kɛyómɛyɑ́hɑɑ-i wɑi, ɑ́pi kɔ Uléécɑɑ icɑ́pinɛ-i lɛ̃ wɑi. 
Ám nkpéni mmɛ̃ kɑm nní nɔ́ n símisinɛ kɛ́mɛɛ nɔ́ pɑkɑrɛnɛ. Nɛ nɛ nɔ́ cɛ́sinɛ rɛ nɔ́inɛ́cɑ́pinɛ yɛ nɔ́ kópulɛ ɑi tɔ́su kii nɔ́ ń nyɔ́ɔnsɛ. 
Nsímɛ́ foí mɛɛ rɛ, nɛ kõ tɛ nɔ yɛ nɔn n cɑ́pinɛlɛ̃, mɛkóólɛntɛnɛ yɛ̀ɛ̀ nɔ́kɛnɛ́cɔpɛ n welɛ. Nɛ nyɑ́ni yɑrɛ in n lɑ, ɑ́i yɛ́ nnɔ́ɔmɛ. 
Li mɛyíkíyiki pisɛ rɛ mɛkóólɛntɛnɛ mɛkɛ́ nɔ́kɛnɛ́cɔpɛ n we likɛ́ nɛ pɛpɛɛ nɔ́kɛnɛ́cɔpɛ ɑsei n tíkilɛ̃ ń lesɛ likɛ́ ń nyísɛ nnyɑ. 
Nɔ yɛ nɔn n cɑ́pinɛlɛ̃, ɑ́i nkpɑ́ni Upíimɑ kupɛ́nɛcɔ ilukɛ nnyɑ kɑni yɛ n cɑ́pinɛlɛ̃. 
Likumúŋɛ́ rɛ nɔ́ úkɑ úu yɛ ucɔ mɛrí pikɛ́ kɛsẽ́ li. Ḿpɑ́ úye yɛɛ yɛ n tũ un uilukɛ cirɛ le. Pɛɛ́ uyɛ-i kɛ nkṹ nn picɔ ń we kɛ picɔ ɑpi yɛ m pomɑɑlɛ̃. 
Áni síyɔ mɑ́ rɛ nɔkɛ́ yɛ kei-pɔ li ɑni níruu? Nɛ́ɛ Uléécɑɑ icɑ́pinɛ pikɔ́ kɑ́ni wɑisɛlɛ̃ nɔn pɛɛ́ lɑ nɔkɛ́ pɛpɛɛ ɑ́pi líkɑ m mɑ́ isɛi wɑisɛ. Kɛ́ nkpéni nɔ́ mɑɑ rɛ íye? Nɔ lɑ rɛ kɛ́ nɔ́ pɑkɑrɛ nɛ́ɛ? Ám yɛ́ mmú kɛ́mɛɛ nɔ́ pɑkɑrɛ! 
Li we rɛ nsímɛ́ mmɛ̃ kɛ Upíimɑ uu nɛ́ n símisi kɑm nɔ́ tɛ́ŋ́sɛlɛ̃ tɛ: Kɛsinɛ kɛ̃ kɑpi Upíimɑ Yeesu pɛpɛɛ pikópɛ u ń lɑ ɑnipɛ-i n wɑ, u kukpɔ́nɔ́ kpísilɛ, 
kuu Uléécɑɑ pipɔɔnɛsɛ ḿ mɑsí, uu ku kpɔkɔ́rinɛ uu rɛ: Inɛ́piŋɛ iyɛ̃ kɑpí nɛ nɔ̃́ nnyɑ n nyɔ́ɔnsɛ yɛ nnyí. Ani yɛ lɛ̃ wɑ ɑní nɛ nɛ́ n léisɛlɛ̃. 
Kɑpi pilukɛ ḿ mɑsí, uu kɔ limɛcɔ kɛpóripi kpísi uu rɛ: Nkómɛinɛ fɑlɛ mɛɛ mɛnɛ́nyɛ nnyɑ nɔ́ nɛ Uléécɑɑ kɛcɔpɛ ń we yɛɛ kɛpóripi nkɛ́. Ḿpɑ́ píyei nɔn yɛ kɛ n níru, ɑni yɛ mɛsɛ́rɛ nɛ́ n léisɛlɛ̃. 
Nɔn yɛ ḿpɑ́ píyei kukpɔ́nɔ́ nkú n le, nɔn kɛpóripi nkɛ́ n níru, Upíimɑ nkpɔ kɑni lɛ̃ yóólɛ̃ likɛ́ hɑ nɛ umɛpɛɛmɛ tu. 
Lɛlɛɛ tíyɛsɛ kɛ úye uu yɛ un mɛhɑ́ikusɛ wee kɛ́mɛɛ n we un nɛ kukpɔ́nɔ́ kpɛ̃ n le un kɔ nɛ kɛpóripi kɛ̃ n níru, Upíimɑ ipiŋɛ nɛ umɛnyɛ kuu lɛ̃ nɛ púnnɛ. 
Lɛ̃ nnyɑ, ḿpɑ́ úye ukɛ́ yɛ musí uu uisoi pɑí ńsɔnɛ uu pɛɛ kɛlenɛ kukpɔ́nɔ́ le uu kɔ kɛpóripi kɛ̃ níru. 
Úye un mɛ́woo n le un n níru, unsɑ́ n ceri rɛ Upíimɑ ipiŋɛ yɛ nyi, umɛcirɛ kuu lɛ̃ mɛwɑi mɛ-i ɑkópɛ pɑlɛ̃. 
Lɛ̃ nnyɑ kɛ pisoi mɛyɑ̃ ɑpi nɔ́kɛnɛ́cɔpɛ pitóikɔ́, picɔ síkɑnkɑ́lɑ́kɑ́, picɔ mɛyɑ̃ pɔ́ɔ kɔ kpíni. 
Tɔn yɛ pɛɛ́ m musí ɑri irɔ́soi pɑí ńsɔnɛ, Uléécɑɑ úu yɛ́ɛ́ pɛɛ́ nɛ rɔ́ túhɑɑnɛ. 
Amɑ́ Upíimɑ yɛ nɛ rɔ́ túhɑɑnɛlɛ, uu kutu rɔ́ pɑ́sɑ uu ntɔ́símɛ́ nyɔ́ɔnsɛ rɛ ɑ́pi kɑpɛ nɛ pɛpɛɛ ɑ́pi Kirisi n nyu tɔ́ ripɛ́nɛ pikɛ́ túhɑɑnɛ. 
Pinɛ́mɑ́rɛcɔ, lɛ̃ nnyɑ, nɔn yɛ n cɑ́pinɛlɛ̃ tɛ nɔkɛ́ Upíimɑ kupɛ́nɛcɔ ilukɛ li, ɑni yɛ mɛ́rɑɑnɛ ɑni kɛlenɛ le. 
Nkṹ nn úye n we, ukɛ́ yɛ ukɛyɔ-pɔ limɛ. Úu kɑpɛ wɑ Uléécɑɑ ukɛ́ yɛ nɔ́inɛ́cɑ́pinɛ-i nɛ nɔ́ túhɑɑnɛ ukɛ́ ɑkópɛ nɔ́ pɑ. Nɛn píyei nɔ́kɛnɛ́mɛɛ n hɑpɔ kɑm yɛ́ isímɛ́ tɔrɔɔ iyɛ̃ kɛcɑ́ɑ́ ripɑí.
