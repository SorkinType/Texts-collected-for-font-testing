2 Korɛnti.
11.
Hói pinɛ́kɔ́! Ani n kɑhɑri ɑní nɛ nɛ́ kóm, ḿpɑ́ nɛn n wɑi yɑrɛ isɔhɔ kɑm tóri. Ani nɛ nɛ́ n kɑhɑri. 
Nɛ nɔ́kɛnɛ́cɑ́ɑ́ ilɑ́ɑrɔ yɛɛ Uléécɑɑ kɛ́mɛɛ n léeri wɑilɛ. Nɔ we yɑrɛ ukpére yɛɛ úu utisi n nyu un kɔ kpɑ̃́iilɛ̃ kɑm utisi usɛ kóló ń félei. Utisi uyɛɛ nní Kirisi ŋmɑɑ. 
Amɑ́ iwɑmɛ yɛ nɛ́ welɛ rɛ nɔ́kɛnɛ́múŋɛ́ yɛ́ cɑɑi ɑni Kirisi nɛ kɛtɑɑ wɑi, ɑni kɛfɑ kɛsɛ nɛ pitíki u yɑ́ yɑrɛ kɛ iwɑ́ɑ ii Ɛfɑ kɛmúŋɛ́ nɛ mɛsɔhɔ n cɑɑi mɛcɔ. 
Ái líkɑ nnyɑ kɑm lɛ̃ mɑ. Úye yɛ̀ɛ̀ un n kɑ, un Yeesu nsímɛ́ nɔ́ rin-yóó, ḿpɑ́ linsɑ́ mmɛ̃ kɑri nɔ́ rin-yóó, un kɔ nfɑ́ɑsɔnɛ nnyinɛ nsímɛ́ nɔ́ n símisi, ḿpɑ́ linsɑ́ Nfɑ́ɑsɔnɛ kɑni ń yɔsí, nɛ́ɛ un Nsímɛ́ Kɛcirɛ nnyinɛ nɔ́ rin-yóó, ḿpɑ́ linsɑ́ mmɛ̃ kɑni ń yɔsí, li yɛ nɛ nɔ́ risɑ́lɛ ɑni mɛsɛ nɛ mɛsɛ ŋmurɛi. 
Ám pɑkɑrɛ rɛ pisoi pɛ̃ kɑni nní n nyɑ́ni rɛ pitumɛ píimɑ yɛ linyinɛ nɛ́ fe. 
In n lɑ, ɑ́m nkpéni usímɛ́nyuwɛ píimɑ unyinɛ, ɑmɑ́ Uléécɑɑ mɛnyuwɛ ɑ́mɛ nɛ́ pɑ́rílɛ̃. Ḿpɑ́ píyei nɛ ḿpɑ́ yo kɛ́mɛɛ kɑri tíyɛsɛ ɑni lɛ̃ ceru. 
Nɛ Kirisi Nsímɛ́ Kɛcirɛ nɔ́ riyóólɛ ɑ́m ihɛ́ɛ́ íkɑ pisɛ. Nɛ mɛnɛ́círɛ ricɛpisɛlɛ kɛ́ nɛ n fe kɛ́ nɔ́ rińtɑɑ́sɛ nnyɑ. Akópɛ kɑm pɛɛ lɛ̃ wɑɑ? 
Kɑḿ pɛɛ nɔ́kɛnɛ́mɛɛ n kɛisi, Uléécɑɑ icɑ́pinɛ icɔ yɛɛ pɛɛ nɛ́ hɛ́ɛ́lɛ. Nɛ́ fe ɑm tɛ iyɛ̃ kɑm yɑ́ɑi ɑḿ nɛ nɔ́ lɛ́ni. 
Kumúŋɛ́ kpɛ-i kɑḿ pɛɛ nɔ́kɛnɛ́mɛɛ ń we, íwɛ yɛ pɛɛ nɛ́ lonlɛ, ɑmɑ́ ɑ́m nɔ́ úkɑ líkɑ pisɛ. Yeesu pikɔ́ pɛɛ Mɑsetuwɑɑni-pɔ n léemɛ yɛ pɛɛ́ nɛ lɛlɛɛ m pɑ́rílɛ̃ nɛ́ kɑlɛ. Ám pɛɛ́ lɑ kɛ́ nɔ́ n nyikilɛ̃ yɑrɛ ncɔni. Nɛ kɛweesi, ɑ́m kɔ lɑ kɛ́ nɔ́ n nyikilɛ̃. 
Kirisi nsímɛ́sei mɛɛ kɛnɛ́mɛɛ ń we kɑḿ lɑ kɛ́ nɛ nɔ́ símisi rɛ: Úkɑ úu yɛ́ fe ukɛ́ Akɑyii kɛtẽ nnɛ́í kɛcɑ́ɑ́ lirikɔkɔri nɛ́ kpu! 
Yɑrɛ ɑ́m nɔ́ lɑ nnyɑ kɑm lɛ̃ símisi nɛ́ɛ íye? Ái lɛ̃! Uléécɑɑ ricuruu yɛ nyɑ́nilɛ rɛ nɛ nɔ́ lɑlɛ! 
Lɛ̃ kɑm nkpéni nní n wɑi kɑm yɛ́ nɛ piwɑi n ŋmɔ́ɔ́púlɛ̃, pɛpɛɛ icée n wɛ́ɛ́si pikɛ́ nɛ rikɔ́kɔri n wɑi rɛ tɔ́ nɛ pɛ̃ pikɛi yɛ pisɛ lɛ ɑ́pi kɑpɛ lincee n yɛ̃́ nnyɑ. 
Lipite pɛ̃ ɑ́pi pitumɛ yíkíyiki. Pi pikɛikɔ́ lɛ pɛɛ n kírɑɑsɛ pin wɑi yɑrɛ pi Kirisi pitumɛ lɛ. 
Lɛ̃ ɑ́i nkpɑ́ni lisɑ́ŋɑ linyinɛ! Setɑni ricuruu yɛ̀ɛ̀ ripɔ́lɔ́lɛ uu kírɑɑsɛ rɛ u mɛtɛ́í uleecɑɑtumɛ yíkíyiki lɛ. 
Lɛ̃ ɑ́i nkpɑ́ni lisɑ́ŋɑ linyinɛ rɛ upikɛikɔ́ pikɛ́ ripɔ́lɔ́ pikɛ́ kírɑɑsɛ rɛ pi Uléécɑɑ ɑsei ute pikɛikɔ́ lɛ. Amɑ́ lɛ̃ kɛ piliwɑiwɑi ɑi yɛ nɛ kɛ́tɔ ń kpu lɛɛ sɔ́nti likɛ́ nɛ pi tɛnɛ. 
Kɛ́ kɔ pimɑ́ɑ nɔ́ rikpɑ́ rɛ úkɑ úu kɑpɛ nɛ́ n nyɑ́ni yɑrɛ nɛ usɔhɔtorɛ lɛ. Nɛ́ɛ nɔn lɛ̃ nɛ́ n nyɑ́ni, ɑni mɛsɛ ŋmurɛi kɛ́ lɛ̃ n we nɛ́ɛkɛ́ kɔ nɛ lɛ̃ nnyɑ nkɑ́ripi rikɔ́kɔri n wɑi. 
In n lɑ lɛ̃ kɑm nní ń lɑ kɛ́ mɑɑ ɑ́i lɛ̃ kɛ Upíimɑ uú lɑ. Li nɛ́ we yɑrɛ kɛ́ rikɔ́kɔri wɑ nnyɑ kɑm lɛ̃ símisi yɑrɛ usɔhɔtorɛ. 
Lɛ̃ kɛ pisoi mɛyɑ̃ ɑpi nní kɛsoimuŋɛ́ nɛ rikɔ́kɔri n wɑi nnyɑ kɛ nɛ́ɛ kɔ lɑ kɛ́ rikɔ́kɔri wɑ. 
Nɔ́rinɛ́cúruu nɔn kɔ nɛ mɛwɛ́ɛ́sɛsɔhɔ mɑ́, ɑmɑ́ nɔ yɛ kɔ pɛɛ piníri nɛ n kɑhɑrilɛ. 
Nɔ yɛ n kɑhɑrilɛ ɑpi ilɑ́si nɔ́ tini ɑpi nɔ́linɛ́kɔ́ le, ɑpi nɔ́linɛ́kɔ́ nɔ́ yɔ́su, ɑpi kɛcɑ́ɑ́ nɔ́ nyɑ́nɛi ɑpi sipɛsɛ nɔ́ túúni. 
Li isɛi nɛ́ welɛ kɛ́ nɛ ḿ mɑɑ rɛ tɔ rinɑ́rɛi mɑ́lɛ. Ḿpɑ́ nɛ lɛ̃, nɛ nkpéni símisilɛ yɑrɛ usɔhɔtorɛ rɛ lɛ̃ kɛ picɔ ɑpí nɛ rikɔ́kɔri n wɑi kɛ nɛ́ɛ kɔ fenɛ kɛ́ nɛ rikɔ́kɔri wɑ. 
Nɛ́ntɛ lipite pɛɛ Pi-Epiree lɛ nɛ́ɛn kɔ U-Epiree? Nɛ́ntɛ pi Isirɑyɛɛli pikɔ́ lɛ̃ nɛ́ɛn kɔ Isirɑyɛɛli ukɔ́? Nɛ́ntɛ pi Apirɑhɑm pipirɛ̃ lɛ nɛ́ɛn kɔ Apirɑhɑm kɛpirɛ̃? 
Nɛ́ntɛ pi Kirisi pikɛikɔ́ lɛ? Kɛ́ nkpéni símisi yɑrɛ li nɛ́ pɑnsɛlɛnlɛ: Nɛ̃́ nɛ Kirisi ukɛikɔ́ lɛ nɛn pi fe. Li we rɛ nɛ kɛisilɛ ɑm pi feriyɛ. Pi mɛsɛ́rɛ ɑkpɑnii nɛ́ wɑilɛ ɑi pi feriyɛ. Nɛ ipépi yɛnlɛ ńsɔnɛ ɑm pi feriyɛ, ɑm kɔ mɛyíkíyiki mɛsɛ́rɛ nkpɔ hɛirɛntɛ. 
Mɛpehẽ mɛnupũ kɛ Pisuifi ɑpi nɛ́ kɑ́ii. Mɛsɛ ɑ́mɛ we ɑfɛɛnɑ kɑpi yɛ ḿpɑ́ mɛ́kɑii mɛ́ye kɛ́mɛɛ isɛ́í nɛ́ súúkɛɛ. 
Mɛpehẽ mɛtɑɑni kɛ Rɔm pikɔ́ ɑpí nɛ nsɛ́í nɛ́ kɑ́ii. Mɛsɛ kɑpi ɑpɑrɛ nɛ́ tɑpisi rɛ kɛ́ kpu. Mɛtɑɑni kɛ míni ɑmɛ kurɔ́ninɔi rimí, ɑm míni-i tonɛ kɛtúŋɛ́ nɛ kɛsinɛ nnɛ́í. 
Nɛ inɛ́cée kului kɛ́mɛɛ ikói nɛ míni nɛ piyɑɑ nɛ pinɛ́yucɔ Pisuifi nɛ mpuri sɑnɛ pikɔ́ nnyɑ nkpɔ hɛirɛntɛlɛ. Nɛ ɑyu nɛ ɑcɛsi kɛcɔpɛ nkpɔ hɛirɛntɛlɛ, ɑm kɔ mínimɑɑ kɛ́mɛɛ nɛ pɛpɛɛ n kpɑ́rɑ́lɛ̃ tɛ Kirisi kɑpi tíkilɛ̃ ɑnipɛ-i nkpɔ hɛirɛntɛ. 
Nɛ pikɛi pɑpɛ-pɑpɛ wɑlɛ ɑm kɔ íwɛ píimɑ yɛnu. Mɛsɛ́rɛ kɛ nnɔni ńn yɛ nɛ́ tu, nkṹ nɛ nnírɛ nn nɛ́ wɑi, ɑm yɛ ɑnɔ́ɔ pɑɑsi ɑ́m lɑ, nnyiyɛ nn nɛ́ loni ɑm ilũ pɑɑ kɛ́ tɑ̃. 
Nɛ kɔ ḿpɑ́ kɛyɑ́ɑ kɛ́ye Uléécɑɑ icɑ́pinɛ nnɛ́í kɛcɑ́ɑ́ músulɛ. Ám nkpɑ́ni lɑ kɛ́ nnɛ́cɔ́ŋ mmɛ̃ pɔ̃́ tinɔɔ mɑɑ. 
Úye un tinɑ́rɛi rinyinɛ m mɑ́, nɛ̃́ nɛ kɔ pɑnsɛlɛ unɑ́rɛi. Úye un ɑkópɛ n wɑ, li yɛ rinɛ́pɔ́ɔ cɑɑisɛlɛ. 
Lin n we rɛ usoi ukɛ́ rikɔ́kɔri wɑ, tinɛ́nɑ́rɛi kɛ nɛ́ɛ nɛ rikɔ́kɔri wɑinɛ. 
Upíimɑ Yeesu usɑɑ Uléécɑɑ yɛ nyulɛ rɛ ɑ́m kirɑɑsɛ. Uyɛɛ ḿpɑ́ píyei ipɑkɑrɛ te. 
Kɑḿ pɛɛ Tɑmɑɑsi kuyu-i ń we, uyɔ́ɔpi píimɑ Aretɑsi yɛ pɛɛ unyinɛ kuyu iwɛ́ɛ́sɛ tinlɛ, uyɛ̃ uu pɛɛ pisoi kuyu sipoo-i tɑ́pɑɑsɛ rɛ pikɛ́ nɛ́ tĩ́. 
Amɑ́ pinyinɛ ɑpi nɛ́ kpísi ɑpi mɛnɛ́rɛ mɛnyinɛ-i wɑi ɑpí nɛ mmɛlɛ kupolɛ kɛpirɛ-mɛ̃ súisɛpɔ ɑḿ nɛ uɑnipɛ-i ŋmɛ́hɛrɛɛ.
