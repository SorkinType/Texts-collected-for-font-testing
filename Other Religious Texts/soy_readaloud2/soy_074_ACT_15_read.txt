Uléécɑɑ Pitumɛ Mɛwɑi.
15.
Kirisi pikɔ́ pinyinɛ ɑpi kɛyɑ́ɑ kɛnyinɛ Yutee kɛtẽ-pɔ yisi ɑpi Antiyɔɔsi sĩ, ɑpí hɑ Kirisi pikɔ́ picɔ mɑɑ rɛ: Nɔnsɑ́ n kérii yɑrɛ kɛ Moisi isé ii m pisɛ, ɑ́ni yɛ́ píyulɑlɛ yɛ̃. 
Pɔɔli nɛ Pɑrinɑpɑsi ɑpi pi kɛsu, íkɛŋɛnɛ píimɑ inyinɛ ii likɛcɑ́ɑ́ pikɛcɔpɛ leeri. Api símisi ɑí tonɛ rɛ Pɔɔli nɛ Pɑrinɑpɑsi nɛ pinyinɛ picɔ pikɛ́ Yerusɑlɛm hɑ, pi nɛ pitumɛ nɛ piwɛ́ɛ́sɛ pikɛ́ hɑ kɛsẽ́ nsímɛ́ mmɛ̃ tipɑí. 
Uléécɑɑ icɑ́pinɛ pikɔ́ ɑpi pikɛcɑ́ɑ́ pɑí ɑpí nɛ ncée tiki. Api Fenisii nɛ Sɑmɑrii sitẽ-mɛ̃ tiki, ɑpi lɛ̃ kɛ mpuri sɑnɛ pikɔ́ ɑpi Upíimɑ rintíki pisoi símisi. Nsímɛ́ mmɛ̃ nn Kirisi pikɔ́ nnɛ́í lɑrisi hɑ́i. 
Kɑpi Yerusɑlɛm n tuipɔ, Uléécɑɑ icɑ́pinɛ pikɔ́ nɛ pipiwɛ́ɛ́sɛ nɛ pitumɛ pɛɛ pi yɔsí. Api Yerusɑlɛm pikɔ́ pɛ̃ símisi lɛ̃ nnɛ́í kɛ Uléécɑɑ uú nɛ pi rintíki uú nɛ wɑi. 
Amɑ́ Pifɑrisi pinyinɛ pɛɛ Yeesu nɛ kɛfɑ n tɛnɛ ɑpi pɛɛ yisi ɑpi rɛ: Li pisɛ rɛ mpuri sɑnɛ pikɔ́ pɛɛ Yeesu nɛ kɛfɑ n tɛnɛ pikɛ́ kérii tílɑsi, pikɛ́ kɔ Moisi isé ritiki. 
Pitumɛ nɛ piwɛ́ɛ́sɛ ɑpi cɑ́pinɛ rɛ pikɛ́ nsímɛ́ mmɛ̃ tipɑí. 
Kɑpi nsímɛ́ m pɑ́sɑinɛ ɑi nɑ́ŋɑi mɛmɑ́ɑ́ kɛ Piyɛɛ uu pɛɛ rinɔ́ɔ yɔsí, uu rɛ: Pimɑ́rɛcɔ, nɔ́rinɛ́cúruu yɛ yɛnlɛ rɛ ɑ́i nɛni-mɛ kɛ Uléécɑɑ uu nɔ́kɛnɛ́cɔpɛ nkɛ́-i nɛ́ wɛ́ɛ rɛ kɛ́ mpuri sɑnɛ pikɔ́ Nsímɛ́ Kɛcirɛ riyóó pikɛ́ nnɛ́nɔ́ɔ-i n kõ, pikɛ́ kɔ nɛ kɛfɑ n tɛnɛ. 
Uléécɑɑ yɛɛ nyu lɛlɛɛ pisoi sifɑ-i ń we. Uyɛɛ kɔ pɛ̃ Unfɑɑsɔnɛ hɛ yɑrɛ kuu tɔ̃́ n hɛ mɛcɔ. U lɛ̃ nyísɛlɛ rɛ pɛ̃ nɛ kɔ pɑnsɛlɛ upisoi. 
Úu tɔ́ nɛ pɛ̃ kɛcɔpɛ kóólɛnɛ. Lɛ̃ nnyɑ kuu pinfɑtɛnɛ nnyɑ piɑkiŋ hɛɛrɛ ɑɑ funi. 
Yo nnyɑ kɑni nkpéni Uléécɑɑ peikɛɛ nɔkɛ́ yɛ̃́, nɔn ncɔni kɛ úkɑ úu hɑ́i pirɔ́sɑ́ɑ-i-mɛ nɛ tɔ́-i n fe ukɛ́ ń cɔ̃ Yeesu pipirɛtiki nyikisɛlɛ̃? 
Amɑ́ Upíimɑ Yeesu ipɛɛlɛ́ɛ nnyɑ kɑrí nɛ kɛfɑ tɛnɛ rɛ tɔ̃́ nɛ kɔ pimɛcɔ píyulɑlɛ yɛnlɛ. 
Pɛpɛɛ n cɑ́pinɛlɛ̃ nnɛ́í ɑpi sɛ́ɛ́ keu, ɑpi pɛɛ Pɑrinɑpɑsi nɛ Pɔɔli kutu n cɔlɛ̃ pin mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ kɛ Uléécɑɑ uú nɛ pi rintíki uu mpuri sɑnɛ pikɔ́ kɛcɔpɛ wɑpisi kɛɛnkɛɛ. 
Kɑpi pisímɛ́ ḿ mɑsí, Yɑkupu uu rinɔ́ɔ yɔ́su uu rɛ: Ani kutu nɛ́ ricɔ, pimɑ́rɛcɔ. 
Simɔɔ yɛ rɔ́ símisilɛ lɛ̃ kɛ Uléécɑɑ ricuruu uu hɑ́i mɛkɛɛ-mɛ mpuri sɑnɛ pikɔ́ kɛcɔpɛ umpuri kuu n te n wɛ́ɛ. 
Antepuyɛ ɑnɔɔ kɑi lɛ̃ titiki yɑrɛ kɑi n wɔ́lɑɑlɛ̃. Li wɔlɑɑlɛ̃ tɛ: 
Lɛ̃ mɛmɑ́ɑ́, nɛ Tɑfiti kɛyɔ kɛɛ n fori yukusɛnɛ, ɑm ncɑɑi nnɛ́í nyɔ́ɔnsɛ. Nɛ́ pimɔ́mɛ kɛ pɛsɛ. 
Kei kɛ pisoi tɔrɔɔ ɑpi yɛ́ pɛɛ nɛ̃́ Upíimɑ wɛ́ɛ́si. Lɛlɛɛ isoipuri nnɛ́í kɑm n sée rɛ ikɛ́ pɑnsɛ inɛ́kɔ́. Nɛ̃́ Upíimɑ rinɔɔ yɛ tɛ̃, 
nɛ́ɛ tíyɛsɛ kɛ pisoi ɑpi hɑ́i mɛkɛɛ-mɛ mɛwɑi mɛ̃ ceri. 
Lɛ̃ nnyɑ kɛ nɛ̃́ Yɑkupu, ɑm músu rɛ ɑ́ri ncée mɑ́ tɔkɛ́ mpuri sɑnɛ pikɔ́ pɛɛ pimɛcirɛ Uléécɑɑ ḿ pɑ Pisuifi isé rinyikisɛ. 
Amɑ́ tɔkɛ́ pi wɔipɔ ɑri pi símisi rɛ ɑ́pi kɑpɛ yɛ inyɔ́ɔnsɛsɑrɛ li. I tɔ̃́ kɛ́mɛɛ ikpɛ́rinkpɛkɔɔ́ lɛ. Pikɛ́ pimɛcirɛ tĩ, ɑpi yɛ sɔ́ɔnnɛ yɑrɛ kɛ isé ii m pisɛ. Ápi kɑpɛ yɛ isɛɛ kɑ́pi rimpɑ́ li, ɑ́pi kɑpɛ yɛ kɔ mɛ́nyɛ li. 
Kɑí pɛɛ mɛkɛɛ ń we, pisoi yɛɛ pɛɛ ḿpɑ́ kuyu kúye-i Moisi isé riyóólɛ, ɑpi yɛ kɔ ḿpɑ́ kɛwéntɛyɑɑ kɛ́ye kɛtúŋɛ́ kuyómɛyɑ́hɑɑlee-i i kɛɛni. 
Pitumɛ nɛ piwɛ́ɛ́sɛ nɛ Uléécɑɑ icɑ́pinɛ pikɔ́ nnɛ́í ɑpi yɛ́nu rɛ li nyɑm tɛ pikɛ́ pikɛcɔpɛ pisoi lesɛ pikɛ́ Pɔɔli nɛ Pɑrinɑpɑsi ritikisɛ pikɛ́ nɛ Antiyɔɔsi pi tũ. Kei kɑpi pɛɛ Yuuti kɑpi yɛ kɔ n sée rɛ Pɑrisɑpɑsi nɛ Silɑsi lesɛ. Pipicɔ yɛ pɛɛ pi wɑisɛlɛnlɛ ńsɔnɛ. 
Piɑnipɛ-i kɑpi rítɛlɛ́ tɛ̃ wɑ. Pi rikɛmɛɛ wɔ́i rɛ: Tɔ̃́ nɔ́pinɛ́mɑ́rɛcɔ, pitumɛ nɛ piwɛ́ɛ́sɛ, tɔ nɔ̃́ pimɑ́rɛcɔ, nɔ́ɔ yɛ ɑ́ni nní Pisuifi nɔn Antiyɔɔsi-pɔ nɛ Sirii nɛ Silisii kɛteni-i we yɑ́ɑsi ńsɔnɛ. 
Tɔ kõ tɛ pisoi pinyinɛ yɛ kɛrɔ́mɛɛ nté yisipɔ ɑpi nɔ́ lɛɛpɔ, ɑpi pinsímɛ́ kɛ́mɛɛ nɔ́ɑnɛ́pɔ́ɔ cɑɑisɛ ɑpi kɔ nɔ́sinɛ́wuu tɔisɛ. Ani kõ tɛ ɑ́i tɔ́ɔ pi tumpɔ. 
Lɛ̃ nnyɑ kɑri kómɛinɛ rɛ li nyɑm tɛ tɔkɛ́ pisoi lesɛ tɔkɛ́ pirɔ́mɑ́rɛcɔ lɑlɑ Pɑrinɑpɑsi nɛ Pɔɔli ritikisɛ tɔkɛ́ nɛ nɔ́kɛnɛ́mɛɛ pi tumpɔ. 
Pɑrinɑpɑsi nɛ Pɔɔli ɑ́pi nkpɔ iwɑmɛ wɑ Upíimɑ Yeesu Kirisi pikɛi kɑpi n wɑi nnyɑ. 
Tɔ Yuuti nɛ Silɑsi nɔ́kɛnɛ́mɛɛ tumpɔ rɛ pikɛ́ nɛ pinnɔɔ nsímɛ́ mmɛ̃ kɑri n wɔ́i nɔ́ símisi. 
Li we rɛ tɔ́ nɛ Nfɑ́ɑsɔnɛ yɛ yɛ̃ tɛ ɑ́i nyɑḿ tɛ tɔkɛ́ ncɔni féé nnyinɛ mɛ́woo nɔ́ rinyikisɛ ɑ́i kɔ lɛ̃ kɑi pisɛ. 
Lɛlɛɛ nɛ n sɑ́ rɛ tɔkɛ́ nɔ́ pisɛ yɛ nní: Áni kɑpɛ yɛ inyɔ́ɔnsɛsɑrɛ nɛ́ɛ mɛ́nyɛ nɛ́ɛ isɑrɛ kɑ́pi rimpɑ́ pikɛ́ mɛ́nyɛ lesɛ li. Ani kɔ nɔ́mɛnɛ́círɛ tĩ, ɑni yɛ sɔ́ɔnnɛ yɑrɛ kɛ isé ii m pisɛ. Tɔ mɛmɑ́rɛcɔ nɛ kɛfɑ kɛsɛ nɔ́ yɑ́ɑsi! 
Pɔɔli nɛ Pɑrinɑpɑsi nɛ Yuuti nɛ Silɑsi ɑpi yisi ɑpi Antiyɔɔsi tɔ́mpɔ. Kɑpi n hɑ́pɔ, ɑpi kei piyómɛyɑ́hɑɑ nnɛ́í cɑ́pinɛ ɑpi rítɛlɛ́ tɛ̃ pi pɑ. 
Kɑpi pinnɛ́í kɛyu-i ri n kɛ́ɛ̃, ɑi ḿpɑ́ úye ikɑri wɑisɛ ɑi kɔ piɑkiŋ niŋukusɛ. 
Yuuti nɛ Silɑsi ricuruu yɛ pɛɛ ɑntepuyɛ lɛ. Api nsímɛ́ nsímɛ́ Antiyɔɔsi pikɔ́ símɑɑnkɛɛ ɑpí nɛ ikɑri pi kpɑ́sɛ rɛ pikɛ́ nɛ nfɑtɛnɛ n ŋmɔ́ɔ́púlɛ. 
Api kei siyɑ́ɑ sinyinɛ wɑi. Pimɑ́rɛcɔ ɑpi pɛɛ limɛmɑ́ɑ́ pi yɑ́ ɑpi pɛpɛɛ pi n tumpɔ kɛ́mɛɛ pɛɛsɛ nɛ piripɔɔ. 
Amɑ́ Silɑsi pɔ́ɔ rɛ u yɛ́ kei tonɛ. 
Pɔɔli nɛ Pɑrinɑpɑsi pɔ́ɔ hɑ Antiyɔɔsi-i tonɛ, pi nɛ picɔ pinyinɛ pin pisoi céési pin kɔ Upíimɑ nsímɛ́ kɛcirɛ yóólɛ̃. 
Kɑpi Antiyɔɔsi-i n tónɛ ɑi siyɑ́ɑ wɑi, Pɔɔli uu Pɑrinɑpɑsi mɑɑ rɛ pikɛ́ silõ sɛ̃ nnɛ́í kɑpi Upíimɑ nsímɛ́ nɛ rinkɔ́ɔ́nú pɛɛsɛ pikɛ́ hɑ yɛ̃ tɛ íye kɛ Yeesu pikɔ́ ɑpi kei we. 
Pɑrinɑpɑsi uú n lɑ ukɛ́ nɛ Yohɑni kɑpi rinyíri n hɛ rɛ Mɑriki ripɛ́nɛ. 
Amɑ́ Pɔɔli pɔ́ɔ yúlu rɛ ɑ́pi kɑpɛ kɔ nkpéni pikpíkɛ́ u rikpɑ́ ukɛ́ pi ritiki, li we rɛ u pɛɛ mɛfoí hɑ́i Pɑmfilii-pɔ pi ŋmɛriyɛlɛ úu pi ritiki pikɛ́ hɑ nɛ pipikɛi kɛtɔ tu. 
Ai wɑi rɛ li Pɔɔli nɛ Pɑrinɑpɑsi símɑɑsɛ, hɑ́i ɑpí nɛ likɛcɑ́ɑ́ túúnnɛ. Pɑrinɑpɑsi uu Mɑriki kpísi ɑpi kúninɔi loni ɑpi Siipuru tɔ́mpɔ. 
Pɔɔli pɔ́ɔ Silɑsi kuséncɔ wɛ́ɛ, Yeesu pikɔ́ ɑpí nɛ pi welu rɛ Uléécɑɑ ukɛ́ pi sɑrumpɔ. Api pɛɛ ncée kpísi ɑpi tɔ́mpɔ. 
Api Sirii nɛ Silisii sitẽ kuwɛ́ɛ́-mɛ̃ tiki, pin Uléécɑɑ icɑ́pinɛ ŋmɔ́ɔ́púsɛlɛ̃ tɛ ikɛ́ Uléécɑɑ ncee kɛ́mɛɛ pɑpisi.
