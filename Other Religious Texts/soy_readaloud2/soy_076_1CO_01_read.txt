Títɛlɛ́ foí kɛ Pɔɔli uu Korɛnti pikɔ́ n wɔ́i.
1.
Nɛ̃́ Pɔɔli kɛ Uléécɑɑ uu umɛlɑ kɛ́mɛɛ m pɑnsɛsɛ Kirisi Yeesu utumɛ nɛ unɛ́cɔ Sɔsitɛɛni, 
tɔ́ɔ Uléécɑɑ icɑ́pinɛ pikɔ́ pɛɛ Korɛnti-i ń we rítɛlɛ́ ntí wɔ́lu. Tɔ pɛ̃ nnɛ́í kɛ Uléécɑɑ uu n lésɛ uu iyɑɑ yekei uu pi nɛ Kirisi Yeesu pɛ́nɛ uu pi pɑnsɛsɛ upikɔ́ yɑ́ɑsi. Tɔ kɔ pɛ̃ nnɛ́í pɛɛ ḿpɑ́ yei-pɔ ń we, pin kɛsẽ́ Upíimɑ Yeesu Kirisi rinyiri sélei yɑ́ɑsi. Yeesu Kirisi yɛ Piupiimɑ lɛ. Uyɛɛ kɔ tɔ̃́ Upiimɑ. 
Urɔ́sɑ́ɑ Uléécɑɑ nɛ Upíimɑ Yeesu Kirisi pikɛ́ ípɛɛlɛ́ɛ nɔ́ nyísɛ ɑpi kɔ nkíŋniŋɛ nɔ́ hɛ! 
Ḿpɑ́ píyei kɑm yɛ nɔ̃́ nnyɑ Uléécɑɑ pɔɔnɛsɛ rɛ u nɔ́ nɛ Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ ípɛɛlɛ́ɛ nɔ́ nyísɛ nnyɑ. 
U nɔ́ nɛ Kirisi kupɛ́nɛcɔ kɛ́mɛɛ mɛsɔhɔ nnɛ́í nɔ́ hɛlɛ ɑni pɑnsɛ linnɛ́í kɛ́mɛɛ pimɑ́mɑ́, ɑí nɛ tɔ́su nsímɛ́ piyóó nɛ mɛnyuwɛ kɛ́mɛɛ́. 
Kirisi kɛcɑ́ɑ́ nsímɛ́ kɑri nɔ́ rin-yóó yɛ nɔ́kɛnɛ́mɛɛ iníŋí wɑlɛ ńsɔnɛ. 
Lɛ̃ nnyɑ, nɔ́ mpí nɔ́ɔ nní m mɛ̃́ tɛ Urɔ́píimɑ Yeesu Kirisi yɛ́ kɛyɑ́ɑ umɛcirɛ lesɛ uu nyísɛ, nfɑ́ɑsɔnɛ lihɛɛhɛɛ líkɑ ɑ́i nɔ́ pɑ́rílɛ̃. 
Uléécɑɑ uyɛ̃ yɛ́ nɔ́ m múílɛ̃ nɔn nɛ nyɛnu, ɑí hɑ nɛ kɛyɑ́ɑ tɔrɔɔ tulu rɛ ɑ́pi kɑpɛ nɔ́kɛnɛ́cɑ́ɑ́ ncɑɑi ńkɑ yɛ̃ kɛyɑ́ɑ kɛ̃ kɛ Upíimɑ Yeesu Kirisi uu n sɔ́nti. 
Uléécɑɑ uyɛɛ nɔ́ n sée rɛ nɔ́ nɛ Ukɛpipi, Urɔ́píimɑ Yeesu Kirisi nɔkɛ́ ripɛ́nɛ yɛ ɑsei ute lɛ. 
Pimɑ́rɛcɔ, nɛ Urɔ́píimɑ Yeesu Kirisi rinyiri nɛ nɔ́ pisɛ rɛ ɑni ńsɔnɛ n kómɛinɛ, nɔn kɔ nsímɛ́ nsɛ símisi. Mɛkóólɛntɛnɛ ɑ́mɛ kɑpɛ nɔ́kɛnɛ́cɔpɛ n we, ɑmɑ́ ɑni kɛsẽ́ lisɛ nɛ n nɑ́ɑ́si nɔn kɔ kɛmúŋɛ́ kɛsɛ mɑ́. 
Pinɛ́mɑ́rɛcɔ, Kulowee kɛyɔ pikɔ́ yɛ nɛ́ símisi rɛ íkɛŋɛnɛ yɛ nɔ́kɛnɛ́cɔpɛ welɛ. 
Liɑsei rɛ pisoi pinyinɛ yɛ nɔ́kɛnɛ́cɔpɛ we, ɑpi yɛ mɑɑ rɛ Pɔɔli kɛ pɛ̃ ɑpi tíkilɛ̃, picɔ rɛ Apoloosi kɛ pɛ̃ ɑpi tíkilɛ̃, picɔ ɑpi kɔ rɛ Piyɛɛ kɛ pɛ̃ ɑpi tíkilɛ̃. Picɔ pɔ́ɔ kɔ rɛ Kirisi kɛ pɛ̃ ɑpi tíkilɛ̃! 
Kirisi yɛ kériinɛlɛnlɛɛ́? Yɑrɛ Pɔɔli kɑpi nɔ̃́ nnyɑ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kpu nɛ́ɛ Pɔɔli rinyiri kɑpí nɛ míni nɔ́ wólɛ? 
Nɛ Uléécɑɑ pɔɔnɛsɛ kɑ́m Kirisipuusi nɛ Kɑyusi mɛmɑ́ɑ́ nɔ́ úkɑ ucɔ míni n wólɛ nnyɑ. 
Lɛ̃ nnyɑ, úkɑ úu yɛ́ fe ukɛ́ rɛ rinɛ́nyíri kɑpí nɛ míni nɔ́ wólɛ. 
Nɛ léisɛ rɛ nɛ kɔ Sitefɑnɑɑsi kɛyɔ pikɔ́ míni wolɛ, ɑmɑ́ ɑ́m pɑkɑrɛ rɛ nɛ kɔ unyinɛ ucɔ piwólɛ rikpɑ́. 
Kirisi úu nɛ́ tummɛ rɛ kɛ́ pisoi míni wolɛ, u nɛ́ tummɛlɛ rɛ kɛ́ Nsímɛ́ Kɛcirɛ riyóó. Ái kɛsoipipi mɛsímɛ́sɔhɔ kɑḿ nɛ n yóólɛ̃. Insɑ́ lɛ̃, Kirisi nkpɔ kuu kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ n kpu ńn líkɑ yoriyɛ. 
Kirisi nkpɔ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ nsímɛ́ yɛ pɛpɛɛ ncée m pɔ kɛ́mɛɛ nnírisimɛ́ lɛ. Tɔ́ mpí riyu kɛ Uléécɑɑ uu n lɔ yɛ nyɑ́ni rɛ Uléécɑɑ nnɑŋɛ yɛ mu. 
Uléécɑɑ Nsímɛ́ yɛ mɑɑ rɛ: Nɛ́ mpí pɛɛ m músu rɛ pi mɛwɛ́ɛ́sɛsɔhɔ mɑ́ mɛwɛ́ɛ́sɛsɔhɔ cɑɑi, ɑm mpí pɛɛ m músu rɛ pi mɛnyuwɛ mɑ́ mɛnyuwɛ wɑisɛ mɛ́woo. 
Lɛ̃ nnyɑ, pɛpɛɛ mɛwɛ́ɛ́sɛsɔhɔ m mɑ́ nɛ isé picélɑɑ nɛ nfɑ́ɑni kɛtẽ nkɛ́ pisímɛ́nyuwɛ pin yei n we, pikɛ́ léemɛ pikɛ́ símisi kɛ́ kṍ nkɛ̃́. Uléécɑɑ yɛ nyísɛlɛ rɛ lɛ̃ kɑpi yɛ kɛtẽ kɛcɑ́ɑ́ n sée rɛ mɛwɛ́ɛ́sɛsɔhɔ yɛ nnírisimɛ́ lɛ. 
Mɛsei kɛ sisoipipi ɑ́si nɛ simɛwɛ́ɛ́sɛsɔhɔ cirɛ fe sikɛ́ Uléécɑɑ céri kei kuu umɛwɛ́ɛ́sɛsɔhɔ n lésɛ uu nyísɛ. Lɛ̃ nnyɑ kɛ Uléécɑɑ uu rɛ u yɛ́ pɛpɛɛ nɛ nsímɛ́ mmú mɛɛ nní ń we yɑrɛ nnírisimɛ́ kɛfɑ n tɛnɛ riyu lɔ. 
Pisuifi rɛ pinsɑ́ mɛwɑisɑŋɑ n yɛ̃, ɑ́pi yɛ́ ŋmurɛi. Pikirɛki pɔ́ɔn mɛwɛ́ɛ́sɛsɔhɔ wɛ́ɛ́si kɛlenɛ. 
Amɑ́ Kirisi yɛɛ kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ ń kpu nkɔ́ kɛ tɔ́ɔ símisi. Nsímɛ́ mmɛ̃ ńn Pisuifi kɛ́mɛɛ kuŋmurɛi we rɛ pikɛ́ nɛ kɛfɑ n tɛnɛ, n kɔ mpuri sɑnɛ pikɔ́ kɛ́mɛɛ nnírisimɛ́ lɛ. 
Amɑ́ Kirisi yɛɛ Pisuifi nɛ Pikirɛki pɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ kɛ́mɛɛ́ Uléécɑɑ nnɑŋɛ nɛ Uléécɑɑ mɛwɛ́ɛ́sɛsɔhɔ. 
Lɛlɛɛ ń we yɑrɛ Uléécɑɑ nnirisimɛ́ yɛ sisoipipi mɛwɛ́ɛ́sɛsɔhɔ likɔ́ felɛ. Kei kɛ Uléécɑɑ uu yɛ n wɑ yɑrɛ unɑ́rɛi yɛ sisoipipi rikɔhɔ likɔ́ felɛ. 
Ani ripɑí, pimɑ́rɛcɔ, ɑni yɛ́nu isoi iyɛ̃ kɛ́mɛɛ́ kɑní pɛɛ ń we Uléécɑɑ uú nɛ nɔ́ wɛ́ɛ. Pɛ̃ kɑni n nyɑ́ni rɛ pi mɛwɛ́ɛ́sɛsɔhɔ mɑ́ nɛ pinɑ́ŋɛ-nɑŋɛ nɛ píyukɔɔ́ ɑ́pi nɔ́kɛnɛ́cɔpɛ kulúi we. 
Lɛ̃ kɛ sisoipipi ɑsi n kpíílɛ̃ tɛ linírikɔ́ kɛ Uléécɑɑ uu yɛ kpísi uú nɛ lɛ̃ kɛ sisoipipi ɑsi n kpíílɛ̃ tɛ mɛwɛ́ɛ́sɛsɔhɔ likɔ́ sɛ́ɛ́sɛ. Lɛ̃ kɛ sisoipipi ɑsi n kpíílɛ̃ tɛ linɑ́rɛikɔ́ kɛ Uléécɑɑ uu yɛ kpísi uú nɛ pɛpɛɛ m músu rɛ pi pinɑ́ŋɛ-nɑŋɛ sɛ́ɛ́sɛ. 
Lɛlɛɛ ɑ́i sisoipipi inipɛɛ-i ríyu m mɑ́ nɛ́ɛ lɛ̃ kɑpi n lómiisɛ nɛ́ɛ lɛlɛɛ ɑ́i líkɑ n tulɑɑlɛ̃ kɛ Uléécɑɑ uu yɛ nɛ lɛlɛɛ yɛ siinipɛɛ n yi rifíísɛ. 
U yɛ lɛ̃ wɑlɛ rɛ úkɑ úu kɑpɛ nɛ n fe ukɛ́ ukɛyu-i umɛcirɛ rińtɑɑ́sɛ nnyɑ. 
Amɑ́ Uléécɑɑ yɛ nɔ̃́ wɛ́ɛlɛ uu nɔ́ kpísi uú nɛ Kirisi Yeesu nɔ́ pɛ́nɛ. Uyɛɛ mɛrɔ́wɛ́ɛ́sɛsɔhɔ mɛɛ Uléécɑɑ-i n léeri. Uyɛɛ tíyɛsɛlɛ̃ kɑri Uléécɑɑ inipɛɛ-i ɑsei pite. Uyɛ̃ nnyɑ kɛ Uléécɑɑ uu rɔ́ pɑnsɛsɛ upikɔ́, uu ɑkópɛ kɛ́mɛɛ rɔ́ lesɛ. 
Ai pɛɛ nkpéni wɑi yɑrɛ kɑi Nléécɑɑsimɛ́ ritɛlɛ́-i n wɔ́lɑɑlɛ̃ tɛ: Úye un n lɑ ukɛ́ umɛcirɛ ritɑɑ́sɛ, ukɛ́ lɛ̃ kɛ Upíimɑ uu n wɑ nnyɑ umɛcirɛ ritɑɑ́sɛ.
