Rɔm.
6.
Lɛ̃ nnyɑ, tɔkɛ́ nkpéni rɛ íye? Tɛ tɔkɛ́ nkpéni ɑkópɛ piwɑi nɛ kɛ́yu mɑhɑ̃ uipɛɛlɛ́ɛ ikɛ́ pɛɛ mɛyɑ̃́ wɑ nɛ́ɛ? 
Ái nkpɑ́ni lɛ̃! Akópɛ ɑ́ɑ kɛrɔ́cɑ́ɑ́ ńnɑŋɛ mɑ́. Íye kɛ tɔ́ɔ yɛ́ kɔ pɛɛ́ nɛ ɑkópɛ kɛ́mɛɛ n we? 
Áni nyu rɛ tɔ́ɔ nní ntɔ́nɛ́í Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ míni n wólɛ, tɔ kɔ míni piwolɛ pɛ̃ kɛ́mɛɛ́ nɛ u ripɛ́nɛlɛ ɑri kɛsẽ́ kpii? 
Kɑpi míni rɔ́ ń wólɛ, tɔ́ nɛ uyɛ̃ kɑpi lɛ̃ tipɛ́nɛ ɑpi kúlɛsi. Kɛsẽ́ kɑri lɛ̃ kpu. Yɑrɛ kuu kɔ Sɑ́ɑ Uléécɑɑ nnɑŋɛ nnyɑ pikpɔkpɔ kɛ́mɛɛ n yisi, limɛcɔ kɛ tɔ́ɔ yɛ́ kɔ nkpéni nfɑ́ɑ fɑlɛ yɛ̃. 
Asei kɛcɑ́ɑ́, in tɛ tɔ ripɛ́nɛ ɑri unkpɔ kɛ́mɛɛ kɛsẽ́ kpi, tɔ kɔ limɛcɔ pɛ́nɛnɛ ɑrí n we ɑrí nɛ umɛyisɛ kɛ́mɛɛ kɛsẽ́ yisi. 
Tɔ kɔ nyulɛ rɛ isoi kpurɛ iyɛ-i kɑrí pɛɛ ń we kɑpí nɛ ripɛ́nɛ ɑpi kunɑpɛ́ɛ́kɔɔ́ kɛcɑ́ɑ́ kɑrii, ɑi wɑi rɛ ɑkópɛ isoi iyɛ̃ íi nkpéni kɛrɔ́cɑ́ɑ́ ńnɑŋɛ ńkɑ mɑ́, ɑ́ri kɔ nkpéni ɑkópɛ ilɑsi le. 
Li we rɛ usoi nkó yɛɛ nkpɔ ḿ mɑsí yɛ ɑkópɛ ilɑsi kɛ́mɛɛ lelɛ. 
Tɔn m mɑɑ rɛ tɔ́ nɛ Kirisi yɛ ripɛ́nɛlɛ ɑri kɛsẽ́ unkpɔ kɛ́mɛɛ kpi, tɔ kɔ nɛ kɛfɑ tɛnɛlɛnlɛ rɛ tɔ́ nɛ uyɛ̃ yɛ́ kɛsẽ́ nfɑ́ɑ m mɑ́. 
Tɔ nyulɛ rɛ Kirisi yɛɛ pikpɔkpɔ kɛ́mɛɛ n yisi úu kɔ píkɑi kpinɛ, nkpɔ ńn kɔ nkpéni ukɛcɑ́ɑ́ ńnɑŋɛ ńkɑ mɑ́. 
Kuu n kpu nnyɑ, ɑkópɛ ɑ́ɑ kɔ nkpéni ukɛcɑ́ɑ́ ńnɑŋɛ ńkɑ mɑ́. Nkpɔ cirɛníŋɛ kuu kpu. U nkpéni nfɑ́ɑ mɑ́lɛ un nɛ Uléécɑɑ mɛkɔ́-i we. 
Lɛ̃ nnyɑ, nɔ́ɔkɛ́ nɔ́mɛnɛ́círɛ n nyɑ́ni rɛ ɑkópɛ ɑ́ɑ nɔ́kɛnɛ́cɑ́ɑ́ ńnɑŋɛ ńkɑ mɑ́. Ani kɔ nɔ́mɛnɛ́círɛ n nyɑ́ni rɛ nɔ Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ nfɑ́ɑ mɑ́lɛ nɔn nɛ Uléécɑɑ mɛkɔ́-i we. 
Lɛ̃ nnyɑ, ɑkópɛ ɑ́ɑ kɑpɛ kɔ nkpéni nɔ́inɛ́piŋɛ yɛɛ kɛyɑ́ɑ n kpinɛ kɛ́mɛɛ ńnɑŋɛ m mɑ́. Áni kɑpɛ ilikɑ́ipi-kɑ́ipi n tikilɛ̃. 
Áni kɑpɛ nɔ́inɛ́piŋɛ kɛlõ kɛnyinɛ kpísi nɔkɛ́ ɑkópɛ-i n wɑi. Amɑ́ ɑni nɔ́mɛnɛ́círɛ kpísi ɑni Uléécɑɑ ɑnipɛ-i wɑi yɑrɛ nɔ kpulɛ ɑni yisi, ɑni nɔ́inɛ́piŋɛ silõ kpísi ɑni Uléécɑɑ hɛ ɑní nɛ lisɔnɛ n wɑi. 
Akópɛ ɑ́ɑ yɛ́ nɔ́kɛnɛ́cɑ́ɑ́ ńnɑŋɛ yɛ̃. Li we rɛ isé íi kɔ nkpéni nɔ́kɛnɛ́cɑ́ɑ́ ńnɑŋɛ ńkɑ mɑ́. Amɑ́ Uléécɑɑ nfɑnɑrɛ mɛɛ nɔ́kɛnɛ́cɑ́ɑ́ ńnɑŋɛ mɑ́. 
Lɛ̃ nnyɑ, íye nkpéni? Tɛ tɔ́ nkpéni ɑkópɛ piwɑi nɛ kɛ́yu mɑhɑ̃ tɛ isé íi kɛrɔ́cɑ́ɑ́ ńnɑŋɛ mɑ́ rɛ Uléécɑɑ ipɛɛlɛ́ɛ kɛ́mɛɛ kɑrí we nnyɑɑ? Ái nkpɑ́ni lɛ̃! 
Áni nyu rɛ nɔn nɔ́mɛnɛ́círɛ n kpísi nɔn usoi m pɑ rɛ nɔkɛ́ uilɑsi n le nɔn u pɑkɑrɛlɛ̃, nɔ mɛyíkíyiki uyɛ̃ kɑni nní m pɑkɑrɛlɛ̃ pilɑsi lɛɛ? Nɔn nɔ́mɛnɛ́círɛ n kpísi nɔn ɑkópɛ-i n tɑmpɔ ɑni ɑilɑsi n le, nɔ́ lɛ̃ nnyɑ kpí. Nɔn Uléécɑɑ ilɑsi n le nɔn u m pɑkɑrɛlɛ̃, nɔ́ pɛɛ lɛ̃ nnyɑ Uléécɑɑ inipɛɛ-i pɑnsɛ ɑsei pite. 
Amɑ́ tɔ Uléécɑɑ pɑkɑrɛ: Nɔ pɛɛ ɑkópɛ ilɑsi lelɛ, nɔ nkpéni kɛfɑ kɛsɛ nɛ icélɑɑ tikilɛnlɛ yɑrɛ kɑpi nɔ́ ɑ n céesi. 
Yeesu Kirisi yɛ ɑkópɛ ilɑsi kɛ́mɛɛ nɔ́ lésɛlɛ ɑni nkpéni pɑnsɛ nɔn ɑsei ilɑsi le. 
Nɛ sisoipipi mɛcɔ símisilɛ rɛ ɑ́ni yɛ mɛkɛɛ nsímɛ́ kṍ nnyɑ. Nɔ pɛɛ mɛkɛɛ nɔ́mɛnɛ́círɛ kpísilɛ nɔn mɛ́kpɛrinkpɛ nɛ mɛwɑi kópɛ ilɑsi kɛ́mɛɛ wɑilɛ̃. Li tíyɛsɛlɛ ɑni pɛɛ isoi yɛɛ íi nɛ Uléécɑɑ n sɑ́ n le. Limɛcɔ kɑni yɛ́ kɔ nɔ́mɛnɛ́círɛ kpísi ɑni lɛlɛɛ ń nyɑḿ piwɑi ilɑsi n le, ɑi tíyɛsɛ ɑni pɛɛ isoi yɛɛ nɛ Uléécɑɑ n sɑ́ n le. 
Kumúŋɛ́ kpɛ-i kɑní pɛɛ ɑkópɛ ilɑsi n le, lisɔnɛ piwɑi ɑ́pi pɛɛ́ nɛ nɔ́ yɛ̃́. 
Mɛwɑi mɛ̃ kɑní pɛɛ n wɑpisi yɛ nkpéni isɛi nɔ́ welɛ. Yo kɑi nɔ́ yóriyɛ? Nkpɔ-mɛ̃ kɑmɛ́ nɛ nɔ́ sĩ. 
Amɑ́ Uléécɑɑ yɛ nkpéni ɑkópɛ kɛ́mɛɛ nɔ́ lésɛlɛ uu nɔ́ pɑnsɛsɛ upilɑsi. Nɔ́kunɛ́lɑ́ɑ kpɛɛ kei rɛ lɛlɛɛ nɛ Uléécɑɑ ń sɑ́ kɑni nkpéni wɑi nɔn nɛ kɛ́tɔ-pɔ nfɑ́ɑ tɛnɛcirɛ́ mɛ̃́. 
Akópɛ kulɑɑ kpɛɛ nkpɔ, ɑmɑ́ lɛ̃ kɛ Uléécɑɑ uu yɛ fɑɑlɑɑ n hɛ yɛɛ nfɑ́ɑ tɛnɛcirɛ́ kɑri yɛ tɔ́ nɛ Urɔ́píimɑ Kirisi Yeesu kupɛ́nɛcɔ kɛ́mɛɛ n yɛ̃́.
