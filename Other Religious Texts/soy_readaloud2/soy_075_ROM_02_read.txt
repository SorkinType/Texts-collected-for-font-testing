Rɔm.
2.
Ḿpɑ́ pɔn usoi úye mpuri, pɔ́ nkó pɔ́ɔ nní nɛ picɔ n túhɑɑnɛ pɔn ɑkópɛ pi pɑlɛ̃, ɑ́i we rɛ pikɛ́ íwɛ pɔ́ tẽ́. Pɔ́ɔ nní nɛ picɔ n túhɑɑnɛ pɔn ɑkópɛ pi pɑlɛ̃, mɛpɔ́círɛ kɑɑ lɛ̃ ɑkópɛ pɑlɛ̃. Liriyíkí rɛ pɔ́ɔ nní nɛ picɔ n túhɑɑnɛ, pɔ̃́ nɛ kɔ pimɛcɔ wɑilɛ. 
Tɔ mɛyíkíyiki nyulɛ rɛ nsímɛ́sei kɛ Uléécɑɑ uu yɛ nɛ pɛpɛɛ lɛ̃ n wɑi túhɑɑnɛ. 
Pɔ́ nkó pɔ́ɔ nní nɛ pɛpɛɛ lɛ̃ n wɑi n túhɑɑnɛ pɔn kɔ pimɛcɔ wɑi, pɔ músu rɛ pɔ̃́ nɛ́ Uléécɑɑ kɛtɑhɑi kɛ́mɛɛ́ lee? 
Nɛ́ɛ Uléécɑɑ mɛwɑi sɔnɛ píimɑ nɛ uisoi niŋɛ-niŋɛ nɛ unfɑnɑrɛ kɑɑ lómiisɛ. Áɑ nyu rɛ umɛwɑi sɔnɛ píimɑ mɛɛ lɑ rɛ ɑ mɛfinɛ consɛɛ? 
Amɑ́ ripɔ́kíŋ ntɑkɛ nɛ kɑɑ n yúlu rɛ ɑ́ɑ yɛ́ mɛfinɛ consɛ nnyɑ, mɛpɔ́círɛ kɑɑ lɛ̃ kuwɔ́i píimɑ cɑ́pinɛ kɛyɑ́ɑ kɛ̃ kɛ Uléécɑɑ uú nɛ nsímɛ́sei ukuwɔi kɛ́mɛɛ pisoi n túhɑɑnɛnɛ. 
Kɛtúhɑɑnɛ kɛ-i kɛ Uléécɑɑ uu yɛ́ ḿpɑ́ úye mɛwɑi ihɛ́ɛ́ u pɑ. 
Uu pɛpɛɛ lisɔnɛ piwɑi nɛ n ŋmɔ́ɔ́púlɛ̃ nfɑ́ɑ tɛnɛcirɛ́ hɛ. Pi lɛ̃ wɑilɛ rɛ ukɛ́ umɛyɔɔpi pi hɛ ukɛ́ kɔ ríyu pi wɑisɛ ukɛ́ kɔ tíyɛsɛ pikɛ́ pɑnsɛ picɑɑicirɛ́. 
Amɑ́ Uléécɑɑ yɛ́ pɛpɛɛ nɛ u n kɛ́ŋɛnɛ ɑ́pi lɑ pikɛ́ nsímɛ́sei kutu ricɔ, pin pɛɛ lɛlɛɛ ɑ́i nɛ n sɑ́ kutu cɔlɛ̃ kɛcɑ́ɑ́ ukuwɔi píimɑ súisɛ. 
Píwɛlukɛ nɛ nwuutúũ mɛɛ mpí nnɛ́í pɛɛ mɛwɑi kópɛ n wɑi likɔ́. Pisuifi kɑí nɛ mɛfoí mɑ́nɛ, ɑi pɛɛ́ nɛ mpuri sɑnɛ pikɔ́ kpɑ́. 
Tíyu nɛ ipɑkɑrɛ nɛ nkíŋniŋɛ mɛɛ mpí nnɛ́í pɛɛ lisɔnɛ n wɑi likɔ́. Pisuifi kɑí nɛ mɛfoí mɑ́nɛ, ɑi pɛɛ́ nɛ mpuri sɑnɛ pikɔ́ kpɑ́. 
Uléécɑɑ úu pisoi nɛ picɔ kóólɛnɛ. 
Mpí nnɛ́í pɛɛ ɑ́pi isé kɛcɑ́ɑ́ n nyu ɑpí nɛ ɑkópɛ wɑi yɛ kpíninɛ ɑ́pi ikɛcɑ́ɑ́ nɛ pi túhɑɑnɛnɛ. Amɑ́ pɛpɛɛ ikɛcɑ́ɑ́ ń nyu ɑpí nɛ ɑkópɛ wɑi yɛ kpíninɛ ɑpi ikɛcɑ́ɑ́ nɛ pi túhɑɑnɛ. 
Asei kɛcɑ́ɑ́, ɑ́i mpí pɛɛ yɛ isé kutu ŋmɑnɛ rińcɔ pɛɛ Uléécɑɑ inipɛɛ-i ɑsei pite. Amɑ́ mpí pɛɛ isé n tíkilɛ̃ kɑpi yɛ hɑ mɑɑ rɛ ɑsei pite. 
In tɛ mpuri sɑnɛ pikɔ́ pɛɛ ɑ́pi Uléécɑɑ isé n nyu yɛ pɛɛ isé iyɛ̃ tikilɛ̃, li wɑ yɑrɛ i pisifɑ kɛ́mɛɛ́ welɛ pin nɛ i sɔ́nɛ, ḿpɑ́ kɑ́pi nní i n nyu. 
Pi lɛ̃ nyísɛlɛ rɛ lɛlɛɛ isé iyɛ̃ kɛ́mɛɛ ń we yɛ pisifɑ kɛ́mɛɛ́ welɛ. Pikɛmúŋɛ́ yɛ̀ɛ̀ pi nyísɛlɛ ɑpi céru rɛ pi yɛ pinyinɛ-i ɑkópɛ wɑlɛ, ɑkɛ yɛ kɔ pinyinɛ-i ɑlɑri pi pɑ. 
Lɛlɛɛ hɑ leerinɛ kɛyɑ́ɑ kɛ̃ kɛ Uléécɑɑ uu Nsímɛ́ Kɛcirɛ kɑm n yóólɛ̃ ḿ pɑínɛ uú nɛ pisoi túhɑɑnɛ. Kirisi Yeesu kuú nɛ tikinɛ uú nɛ lɛ̃ nnɛ́í kɛ pisoi ɑpi n wɑi pin pékesi kɛcɑ́ɑ́ pi túhɑɑnɛ. 
Pɔ́ nkó pɔ́ɔ nní mɛpɔ́círɛ rinyíri n hɛ rɛ Usuifi, pɔn nɛ isé tɑ́lɛ̃, pɔn Uléécɑɑ nnyɑ rikɔ́kɔri wɑi, 
pɔn umɛlɑ nyu, pɔ̃́ kuu uisé kɛ́mɛɛ n céesi rɛ ɑ yɛ lɛlɛɛ licɔ nɛ kɛkpéẽ ń we ceri, 
pɔ́ɔ nní mɛpɔ́círɛ n nyɑ́ni rɛ pɔ́ɔ pinyíyɛ tɔrikɛɛ, pɔn kɔ pɛpɛɛ kuŋmɑhɑ kɛ́mɛɛ ń we mɛtɛ́í, 
nɛ piníri nɛ siwɑ̃́ ucélɑɑ, rɛ pɔ isé kɛ́mɛɛ mɛnyuwɛ nɛ nsímɛ́sei yɛ̃ nnyɑ, 
pɔ nkpéni nní lɛ̃ picɔ céesilɛ, ɑɑ pɛɛ mɛpɔ́círɛ picélɑɑ pɔɔnɛ. Pɔ yɛ rɛ ɑ́pi kɑpɛ yɛ yɑ́ɑi, ɑmɑ́ pɔ́ɔn pɛɛ yɑ́ɑ́i! 
Pɔ yɛ kɔ rɛ ɑ́pi kɑpɛ yɛ iwɑ́sɑ́ wɑ, ɑmɑ́ pɔ́ɔn pɛɛ iwɑ́sɑ́ wɑi! Pɔ yɛ rɛ pɔ piléé kɔ́hɔlɛ, pɔn kɔ pɛɛ pilikɔ́ yɑ́ɑ́inkɛɛ! 
Pɔ́ɔ nní isé nnyɑ rikɔ́kɔri n wɑi, pɔ yɛ kɔ isé rilɔ́ɔ́lɛ ɑɑ́ nɛ Uléécɑɑ riyu kopu. 
Li Nléécɑɑsimɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Nɔ̃́ nnyɑ, Uléécɑɑ rinyiri yɛ mpuri sɑnɛ pikɔ́ kɛ́mɛɛ cɑɑilɛ. 
Asei kɛcɑ́ɑ́, in tɛ nɔ isé tíkilɛ̃ kɛ piké ɑpi yɛ́ kulɑ́ɑ nɔ́ wɑ. In tɛ nɔ isé rilɔ́ɔ́, nɔ́pinɛ́ké yɛ we yɑrɛ kukécirɛ́. 
In tɛ ukécirɛ́ yɛ lɛ̃ kɛ isé ii m pisɛ kɛcɑ́ɑ́ sɔ́nɛ, ɑ́ni yɛ̃ tɛ ukukécirɛ́ yɛ́ Uléécɑɑ kɛ́mɛɛ́ n we yɑrɛ pi u rikélɛɛ? 
Mpuri kécirɛ́ ukɔ́ un isé n tikilɛ̃, úu yɛ́ tíyɛsɛ pɔ́ nkó pɔ́ɔ nní isé n nyu ɑ́ɑ yɛ pɛɛ i ritiki, pɔn kɔ kélɛ̃, ɑɑ yɛ́nu rɛ pɔ ɑkópɛ mɑ́lɛɛ? 
Lɛ̃ nnyɑ, ɑ́i ipiŋɛ kɛcɑ́ɑ́ piké mpí pɛɛ yɛ nyísɛ rɛ usoi yɛ Usuifi lɛ. Ái kɔ ipiŋɛ kɛcɑ́ɑ́ piké mpí pɛɛ piké yíkíyiki. 
Amɑ́ lɛ̃ kɛ usoi uu kɛfɑ kɛ́mɛɛ ń we lɛɛ yɛ nyísɛ rɛ u Usuifi lɛ. Lɛ̃ kɑpi yɛ mɑɑ rɛ usoi yɛ mɛyíkíyiki kélɛnlɛ. Uléécɑɑ Nfɑɑsɔnɛ mɛké yɛ mɛ̃, ɑ́i isé mɛkɔ́. Ái pisoi kɛ́mɛɛ́ kɛ Usuifi uyɛ̃ mpuri ipɑkɑrɛ ii yɛ leemɛ, ɑmɑ́ Uléécɑɑ kɛ́mɛɛ le.
