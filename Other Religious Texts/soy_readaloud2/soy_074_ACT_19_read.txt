Uléécɑɑ Pitumɛ Mɛwɑi.
19.
Kumúŋɛ́ kpɛ-i kɛ Apoloosi uú pɛɛ Korɛnti-i ń we, Pɔɔli yɛ pɛɛ yisilɛ uu Asii ɑyɔpɛ silõ lɔ́ɔ́ uu Efɛɛsi sĩ, uu kei Yeesu pipirɛtiki pinyinɛ yɛnu. 
Uu pi pisɛ rɛ: Kɑni Yeesu nɛ kɛfɑ n tɛnɛ, Uléécɑɑ yɛ Nfɑ́ɑsɔnɛ nɔ́ hɛɛ? Api u pɛsɛ rɛ: Ticuruu ɑ́ri Nfɑ́ɑsɔnɛ nkɔ́ ńkɑ komɑɑlɛ̃. 
Uu kɔ pipisɛ pi kpɑ́ rɛ: Míni mɛ́ye mpuri kɑpi pɛɛ nɔ́ wólɛ? Api rɛ Yohɑni mɛni kuu pisoi n wólɛ kɑpi rɔ́ wólɛ. 
Pɔɔli uu pɛɛ pi mɑɑ rɛ: Pɛpɛɛ n ŋmurɛi rɛ pi yɛ́ mɛfinɛ consɛ kɛ Yohɑni uu yɛ pɛɛ míni wolɛ. Uu yɛ kɔ Isirɑyɛɛli pikɔ́ mɑɑ rɛ pikɛ́ Yeesu yɛɛ mɑsí ukɛpirɛ n sɔ́nti nɛ kɛfɑ tɛnɛ. 
Kɛ pipirɛtiki pɛ̃ ɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, ɑpi tíyɛsɛ ɑpi Upíimɑ Yeesu rinyiri nɛ míni pi wolɛ. 
Pɔɔli uu ɑnípɛ pikɛcɑ́ɑ́ lɑ́ɑ, Nfɑ́ɑsɔnɛ nn pikɛcɑ́ɑ́ súiri. Api pɛɛ mɛyu mpehẽ mpehẽ pisímɛ́ loni pin kɔ nɛ Uléécɑɑ rinyiri pisoi símisi. 
Pinnɛ́í yɛ pɛɛ wɑlɛ pitisi kɛfi nɛ pitɛ́. 
Pɔɔli uu kuyómɛyɑ́hɑɑlee-i lompɔ, uu kɛ́yu kpɑ̃́iisɛ uu iwɑrɛ itɑɑni nɛ Uléécɑɑ iyɔɔpi kɛcɑ́ɑ́ pi símisi cɑ́kɑ́-cɑ́kɑ́ rɛ pikɛ́ kṍ. 
Amɑ́ pinyinɛ pɔ́ɔn pɛɛ yúlu rɛ ɑ́pi nɛ kɛfɑ tɛnɛnɛ, pin kɔ Upíimɑ icélɑɑ iyɛ̃ tiwúí kɛ́mɛɛ sɛ́nnyi pin lɑ́misi. Pɔɔli uu mɛsɛ upipirɛtiki kpísi uú nɛ kei nyɔsɔ́, uu yɛ pɛɛ ḿpɑ́ kɛyɑ́ɑ kɛ́ye utisi unyinɛ kɑpi yɛ n sée rɛ Tirɑnuusi ripoo-i isúkuli kɛ́mɛɛ pi n céési. 
Ai wɑi iŋmɛ̃ itɛ́ nɛ iriyu. Lɛlɛɛ tíyɛsɛ kɛ Pisuifi nɛ Pikirɛki nnɛ́í pɛɛ Asii kɛteni-i ń we ɑpi Upíimɑ nsímɛ́ kṍ. 
Uléécɑɑ yɛ pɛɛ kumúŋɛ́ kpɛ-i Pɔɔli nɛ ritikilɛ uú nɛ mɛwɑisɑŋɑ wɑpisi. 
Lɛ̃ ɑi tíyɛsɛ ricuruu pisoi ɑpi yɛ ilũ nɛ sinírípi sɛɛ Pɔɔli rińcɑ kpísi ɑpí hɑ pitóikɔ́ kɛcɑ́ɑ́ lɑ́ɑ, ɑpi pélu, ɑníri ɑɑ kɔ lelu. 
Kei kɛ Pisuifi pinyinɛ pɛɛ n kɔɔnii pin pisoi ɑníri pɛ́sɛi pɔ́ɔ kɔ rɛ pi yɛ́ Upíimɑ Yeesu rinyiri sée ɑpí nɛ pisoi ɑníri pɛ́sɛi. Api yɛ pɛɛ ɑ mɑɑ rɛ: Nɛ Yeesu uyɛ̃ nsímɛ́ kɛ Pɔɔli uu n yóólɛ̃ tinyiri nɛ rinɔ́ɔ nɔ́ hɛ rɛ ɑni pisoi mpí kɑni nní n cɔ́ŋlɛ̃ kɛ́mɛɛ le. 
Pisuifi pɛpɛɛ yɛ Uléécɑɑ inyɔ́ɔnsɛ n wɑ uwɛ́ɛ́sɛ usɛ kɑpi yɛ n sée rɛ Sefɑ piŋmɑ́nɛ pisɛɛi pɛɛ mɛwɑi mɛ̃ lɛ̃ wɑi. 
Amɑ́ kuníri ɑku kɛyɑ́ɑ kɛnyinɛ pi pɛsɛ rɛ: Nɛ Yeesu nyulɛ, nɛn kɔ Pɔɔli isoi nyu. Nɔ̃́ ní, nɔ́ɔ pɛɛ mpíí píye? 
Kei kɛ unírihélɑɑ uyɛ̃ uu yisi uu pi kɑ́pɑ́ɑ́, uu pinnɛ́í rikɔhɔ feriyɛ. Uu rikpɑ́kɑ́rɑ́ nɛ íwɛ pi wɑi ɑpí hɑ nɛ itói yɔ́ɔi, ɑi tɔ́su ɑpí nɛ ukɛyɔ-i mɛŋmɑnɛ popirimɛ ɑpi itóó súúni. 
Pisuifi nɛ Pikirɛki pɛɛ Efɛɛsi-i ń we ɑpi kóm tɛ lɛ̃ kɑi wɑ. Iwɑmɛ ii pi loni, pisoi ɑpi pɛɛ Upíimɑ Yeesu rinyiri mɛyɑ̃́ n tɑɑ́sɛlɛ̃. 
Pɛpɛɛ pikɛmɛɛ Kirisi nɛ kɛfɑ n tɛnɛ mɛyɑ̃ ɑpi yɛ pɛɛ sĩ́ ɑpí hɑ pisoi kɛcɔpɛ lɛ̃ kɑpi n wɑpisi yɑ́mnɛntɛ. 
Pɛpɛɛ pɛɛ mɛŋmɑnɛiwɑi n wɑi mɛyɑ̃ pɔ́ɔ kɔ nɛ piɑtɑrɛtɛlɛ́ sɔ́nɑɑpɔ ɑpi pisoi kɛcɔpɛ nnɑ ɑ ŋmɑkɑɑnkɛɛ. Pi ɑsilɑlɛwoó kɛ́ɛ́nlɛ ɑí lelu mɛwóópipi ɑ́kotokú kuwóó (50.000). 
Lɛ̃ kɛ Upíimɑ nnɑŋɛ nn tíyɛsɛ unsímɛ́ nń nɛ ḿpɑ́ yei kɔ́ɔ́nú ńsɔnɛ. 
Lɛ̃ mɛmɑ́ɑ́, Pɔɔli uu pɛɛ kɛmúŋɛ́ wɑi rɛ ukɛ́ Mɑsetuwɑɑni nɛ Akɑyii sitẽ-mɛ̃ cɔ ukɛ́ nɛ Yerusɑlɛm tu. Uu rɛ un kei n tu, li kɔ pisɛ rɛ ukɛ́ Rɔm kuyu rilóólú. 
Uu pɛɛ pisoi pɛɛ pikɛi u n lɛ́ni kɛcɔpɛ pitɛ́ yukusɛ uu Mɑsetuwɑɑni-pɔ tum. Pɛ̃ kuu n tũ pɛɛ Timotee nɛ Erɑsiti. Uricuruu uu nkɑ́ripi Asii kɛteni-i kɛtónɛ kpɑ́. 
Ńcɔkɔi píimɑ nnyinɛ nn likumúŋɛ́ kpɛ-i Upíimɑ ncee pitiki kɛcɑ́ɑ́ nnyɑ Efɛɛsi kuyu-i léeri. 
Utisi unyinɛ yɛɛ pɛɛ kei we, ɑpi yɛ u séi rɛ Temetiriisi. U pɛɛ úyu lɛ. Piléé pinyinɛ kɑpi yɛ pɛɛ kɔ kei n sée rɛ Aritemiisi siwosoopi kuu yɛ pɛɛ́ nɛ yírɑɑnɛ uu sisiyukɛ-yukɛ nɛ nwóóweni yuhu yɑrɛ sítumpiwɛɛ́u. Upikɛi pɛɛ pɛɛ u nɛ upinípɛkɛikɔ́cɔ kulɑ́ɑ wɑilɛ ńsɔnɛ. 
Uu kɛyɑ́ɑ kɛnyinɛ upikɛicɔ pɛ̃ nnɛ́í cɑ́pinɛ uu pi mɑɑ rɛ: Pinɛ́sɑ́nɛ, nɔ́rinɛ́cúruu yɛ nyulɛ rɛ pirɔ́kɛi mpí kɑrí nɛ le. 
Nɔ́rinɛ́cúruu yɛ kɔ nyɑ́nilɛ nɔn kɔ kómɛi rɛ ɑ́i nkpɑ́ni Efɛɛsi nté ŋmɑnɛ, ɑmɑ́ Asii kɛtẽ nnɛ́í kɛcɑ́ɑ́ kɛ Pɔɔli uyɛ̃ uu pisoi tɑ́lɑɑnkɛɛ ɑpi ŋmurɛi rɛ ɑ́pi yɛ́ piléé kɛ sisoipipi ɑsi ń sɛrí nɛ kutu n cɔlɛ̃, tɛ ɑ́i Uléécɑɑ. 
Tɔnsɑ́ piwɑi n ceri, mɛwɑi mɛ̃ nɛ́ pirɔ́kɛi riyu kpu, ɑmɛ pirɔ́lééyɑhɑ píimɑ Aritemiisi kɑpi Asii nnɛ́í nɛ sitẽ sicɔ-pɔ n yɑ́ɑ́si fíísɛ, úkɑ úu yɛ pɛɛ pi n wɑisɛlɛ̃. 
Kɛ pisoi pɛɛ n cɑ́pinɛlɛ̃ ɑpi ɑnɔ́ɔ nyɛ̃ n kṍ, ɑi kuwɔ́i pi feriyɛ ɑpi cɑ́ɑ́i rɛ: Lipíimɑ lɛɛ Efɛɛsi piléé Aritemiisi! 
Ńcɔkɔi nn pɛɛ kuyu nnɛ́í yipu. Pisoi ɑpi kómɛinɛ ɑpí nɛ kɛkpíntɑhɑi n sɑirilɛ̃, ɑpi Pɔɔli pisencɔ Mɑsetuwɑɑni pikɔ́ pitɛ́ tini ɑpí nɛ pɛ́nɛpɔ. Pɛpɛɛ Kɑyusi nɛ Arisitɑriki. 
Pɔɔli uú n lɑ ukɛ́ pisoi pɛɛ n cɑ́pinɛlɛ̃ kɛyu-i léépɔ, ɑmɑ́ Yeesu pipirɛtiki ɑpí nɛ u yulu. 
Aí nɛ upisɑnɛ Asii pikɑwe píimɑ pinyinɛ pɛ́nɛ ɑpi ukɛmɛɛ pisoi tumpɔ ɑpí nɛ u kɛsu rɛ úu kɑpɛ kɛkpíntɑhɑi kɛ-i lompɔ. 
Pisoi yɛ pɛɛ welɛ pin kɛcɑ́pinɛ kɛ-i mɛ́woo cɑ́ɑ́i, úkɑ úu ucɔ nkɔ́ kómɛi, lin pi túírɛsɛntɛ. Pimɛyɑ̃ ɑ́pi kɔ nyu lɛlɛɛ kei pi n cɑ́pinɛsɛlɛ̃. 
Kei kɛ Pisuifi pinyinɛ ɑpi Alɛkisɑntiri kɛ́yu-mɛ̃ titépɔ, pisoi ɑpi pɛɛ́ m músu rɛ uyɛɛ nɛ ńcɔkɔi mmɛ̃ kɑ. Alɛkisɑntiri pɔ́ɔ pɛɛ kunípɛ síkɑ rɛ u lɑ ukɛ́ uriyu hɛɛrɛ. 
Kɛ pisoi ɑpi n céri rɛ Alɛkisɑntiri yɛ Usuifi lɛ, ɑpi pɛɛ rinɔ́ɔ risɛ wɑi ɑpí n cɑ́ɑ́i rɛ: Efɛɛsi piléé Aritemiisi pɛɛ Lipíimɑ! Api lɛ̃ n cɑ́ɑ́i ɑí nɛ ituŋɛ mɛkɔ́ɔ́nú mɛtɛ́ tulu. 
Kuyu utɛlɛwɔlɛ uu fe uu pisoi sɛ́ɛ́sɛ, uu pɛɛ pisɛ rɛ: Nɔ̃́ Efɛɛsi pikɔ́, wóo yɛ úu kɛtẽ kɛcɑ́ɑ́ nyu rɛ Efɛɛsi kuyu pikɔ́ pɛɛ pilééyɑhɑ píimɑ Aritemiisi nɛ pikɛyu kɛɛ kɛléécɑɑ-pɔ n lóimɛ kɛcɑ́ɑ́ pɑílɛ̃? 
Úkɑ úu we yɛɛ yɛ́ nsímɛ́ mmɛ̃ n kɛ́si. Lɛ̃ nnyɑ, ɑni nɔ́ɑnɛ́kíŋ niŋukusɛ, ɑ́ni kɑpɛ ń-yɔɔnɛ-yɔɔnɛ nɛ linyinɛ wɑ. 
Pisoi mpí kɑní nɛ ń kɑ ɑ́pi kurɔ́yómɛyɑ́hɑɑlee-i líkɑ cɑɑi, ɑ́pi kɔ pirɔ́léé Aritemiisi lɑ́mɑɑnkɛɛ. 
In tɛ úyu Temetiriisi nɛ upikɛicɔ yɛ unyinɛ nɛ nsímɛ́ mɑ́, sitúhɑɑnɛyɑɑ yɛ welɛ, pitúhɑɑnɛ pin kɔ we. Pi yɛ́ fe ɑpi kei kɛtɑhɑi u séi ɑpí nɛ pi túhɑɑnɛ. 
Nɔn kɔ pɛɛ ń-ye ncɔ m mɑ́, tɔ́ hɑ riyɔ́ɔpipoo-i mmɛ̃ símisi yɑrɛ kɑi m pisɛ. 
Insɑ́ lɛ̃, lɛlɛɛ nɛni kɛyɑɑ nkɛ́ nní ń wɑ yɛ́ tíyɛsɛ ɑrí n we yɑrɛ pitencɔkɔi. Ári yɛ́ ńkɑ yɛ̃ tɔkɛ́ símisi likɛ́ nɛ n nyísɛ rɛ tɔ ɑlɑri mɑ́lɛ tɔkɛ́ nní n cɑ́pinɛlɛ̃. 
Tinɔ́ɔ tɛ̃ kuú mɑ uú nɛ pisoi kpiisɛ.
