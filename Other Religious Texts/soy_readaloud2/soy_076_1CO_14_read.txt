1 Korɛnti.
14.
Ńlɑ kɑni yɛ́ mɛfoí n wɛ́ɛ́si nɔkɛ́ m mɑ́. Ani n nɑ́ɑ́si nɔkɛ́ kɔ Nfɑ́ɑsɔnɛ lihɛɛhɛɛ yɛ̃, ɑí nɛ tɔ́su nɔkɛ́ Uléécɑɑ rinyiri nɛ n símisi. 
Mɛsei yɛ mɛ rɛ usoi nkó yɛɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi úu pisoi nɛ símisi, ɑmɑ́ Uléécɑɑ rinyiri kuú nɛ símisi. Li we rɛ úkɑ úu u kómɛi. Nfɑ́ɑsɔnɛ kuú nɛ nsímɛ́ mɛɛ m pékɑɑlɛ̃ nyɑ́ni un símisi. 
Amɑ́ úye un yɛ Uléécɑɑ rinyiri nɛ n símisi, pisoi kɛ uyɛ̃ uú nɛ símisi. U yɛ́ pi lɛ̃́ ɑpi Uléécɑɑ ncee-i pɑpisi, uu ikɑri pi kpɑ́sɛ, uu kɔ piɑkiŋ niŋukusɛ. 
Nkó yɛɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi yɛ umɛcirɛ Uléécɑɑ ncee-i pɑpukusɛlɛ. Amɑ́ nkó yɛɛ Uléécɑɑ rinyiri nɛ n símisi yɛ picɔ Uléécɑɑ ncee-i pɑpukusɛlɛ. 
Nɛ pɛɛ́ lɑ yɑrɛ nɔ́nnɛ́nɛ́í nɔkɛ́ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi. Amɑ́ lɛ̃ kɑm mɛyɑ́nsei ń lɑ yɛɛ nɔkɛ́ pɛɛ Uléécɑɑ rinyiri nɛ n símisi. Nkó yɛɛ Uléécɑɑ rinyiri nɛ n símisi yɛ nkó yɛɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi felɛ. Amɑ́ in tɛ unyinɛ yɛ liɑsei lesɛ un símisi mɛcirɛ kɑi yɛ́ picɔ kulɑ́ɑ wɑ, ɑi tíyɛsɛ ɑpi Uléécɑɑ ncee-i pɑpisi. 
Pimɑ́rɛcɔ, nɛ lɑ kɛ́ nɔ́ pisɛ rɛ nɛn nɔ́kɛnɛ́mɛɛ́ n hɑpɔ nɛn Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi, yo kɑi yɛ́ nɔ́ yóriyɛ? Ái nɛn nsímɛ́ kɛ Uléécɑɑ nɛ́ n nyísɛ nɔ́ n símisi, nɛ́ɛ nɛn nsɔhɔɔsimɛ́ nnyinɛ nɔ́ n símisi, nɛ́ɛ nɛn Uléécɑɑ rinyiri nɛ n símisi, nɛ́ɛ nɛn icélɑɑ inyinɛ nɔ́ m pɑ kɑi yɛ́ kulɑ́ɑ wɑɑ? 
Limɛcɔ kɑí we nɛ iyómɛlũ yɑrɛ nní risũ nɛ́ɛ kusolɛ kɑpi yɛ nɛ ɑnípɛ ḿ pi. Liɑnɔɔ ɑnsɑ́ n kóólɛntɛnɛlɛ̃, íye kɛ usoi uu yɛ́ nɛ ceri rɛ risũ kɑpi símisi nɛ́ɛ kusolɛ kɑpi pépu? 
Uyɛɛ rinɑ́ɑtɔ́ŋ n símisi unsɑ́ ɑnɔ́ɔ n lesɛ likɛ́ n kpɑ̃́iilɛ̃, wóo yɛ́ pɛɛ céri ukɛ́ umɛcirɛ cɑ́pinɛ ukɛ́ nɛ ɑtɔpi leepɔ? 
Limɛcɔ, nɔn Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi, nɔnsɑ́ yɛ rinɔ́ɔ m mɑ rikɛ́ ɑsei m mɑ́, íye kɛ usoi uu yɛ́ nɛ lɛ̃ kɑni n símisi ceri? Mɛ́woo kɑni pɛɛ lɛ̃ símisi. 
Mɛyu yɛ kulúi welɛ kɛtẽ kɛcɑ́ɑ́, ɑmɑ́ mɛ́kɑ ɑ́mɛ mɛkɛcɔpɛ we rɛ ɑ́mɛ ɑsei mɑ́. 
Lɛ̃ nnyɑ, usoi un nɛ nɛ́ n símisi, nɛnsɑ́ umɛyu n kómɛi, nɛ ukɛmɛɛ usɑ́nɛ lɛ, uyɛ̃ un kɔ nɛ̃́ kɛ́mɛɛ usɑ́nɛ. 
Limɛcɔ, yɑrɛ kɑni nní ń lɑ nɔkɛ́ Nfɑ́ɑsɔnɛ lihɛɛhɛɛ mɛyɑ̃́ m mɑ́, ɑni n wɛ́ɛ́si mɛfoí rɛ nɔkɛ́ mɛyɑ̃́ lihɛɛhɛɛ m mɑ́ lɛɛ yɛ Uléécɑɑ icɑ́pinɛ pikɔ́ n lɛ̃́ pikɛ́ nɛ nfɑtɛnɛ kɛ́mɛɛ́ m pɑpisi. 
Lɛ̃ nnyɑ, li pisɛ rɛ nkó yɛɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi ukɛ́ Uléécɑɑ pisɛ ukɛ́ u hɛ ukɛ́ yɛ fe ukɛ́ mɛ tuisɛ. 
Nɛn limɛyu mɛ̃ nɛ kɛyómɛ n yɑ́ɑ́si, Nfɑ́ɑsɔnɛ mɛɛ kɛnɛ́mɛɛ ń we kɑḿ nɛ yɑ́ɑ́si. Amɑ́ mɛnɛ́sɔhɔ ɑ́mɛ pɛɛ líkɑ wɑi. 
Lɛ̃ nnyɑ, íye kɑi pisɛ rɛ kɛ́ wɑ? Li pisɛ rɛ kɛ́ Nfɑ́ɑsɔnɛ mɛɛ kɛnɛ́mɛɛ ń we nɛ kɛyómɛ yɑ́ɑ́si, ɑm kɔ mɛnɛ́sɔhɔ nɛ kɛyómɛ yɑ́ɑ́si. Nɛ́ Nfɑ́ɑsɔnɛ mɛɛ kɛnɛ́mɛɛ ń we nɛ yõ, ɑm kɔ nɛ mɛnɛ́sɔhɔ yom. 
In tɛ Nfɑ́ɑsɔnɛ ŋmɑnɛ kɑɑ́ nɛ Uléécɑɑ pɔɔnɛsɛ, íye kɛ nkó yɛɛ úu mɛyu mɛ̃ n kómɛi uu yɛ́ fe uu pipɔ́pɔɔnɛsɛ iyɑ́hɑɑ kɛcɑ́ɑ́ rinɔ́ɔ yɔ́su rɛ Amí! Úu kɔ pɛɛ kómɛi lɛ̃ kɑɑ n tee. 
Ḿpɑ́ ipɔ́pɑkɑrɛ kɑɑ Uléécɑɑ m pɑkɑrɛntɛ ikɛ́ íye nsɑ́ n nyɑḿ, íi nkpɑ́ni ucɔ lɛ́ninɛ ukɛ́ nɛ nfɑtɛnɛ kɛ́mɛɛ m pɑpisi. 
Nɛ Uléécɑɑ pɔɔnɛsɛ rɛ u tíyɛsɛ nɛn Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ símisi lin nɔ́nnɛ́nɛ́í mmú fe. 
Amɑ́ ḿpɑ́ nɛ lɛ̃, piyómɛyɑ́hɑɑ pin n cɑ́pinɛlɛ̃, li yɛ kutɔsi nɛ́ n we rɛ kɛ́ ɑnɔ́ɔ fíí nyɛɛ kúkomɛ ń we símisi kɛ́ nɛ pi céési, nɛ kɑm yɛ́ mɛyu kɑ́pi n kómɛi nɛ ɑnɔ́ɔ kulúi pi n símɑɑnkɛɛ. 
Áni kɑpɛ mɛwɑ̃́ m músu. Ani ɑkópɛ piwɑi ncee kɛ́mɛɛ n nyɑḿ yɑrɛ siwɑ̃́ fɛnfɛ̃í. Amɑ́ ɑni yɛ mɛwɛ́ɛ́sɛ musí. 
Yɑrɛ kɛ Upíimɑ uu ḿ mɑ ɑpi Uléécɑɑ Nsímɛ́ ritɛlɛ́-i wɔ́lu rɛ: Pisoi mpí pɛɛ mɛyu féé n símisi nɛ pisɑ́nɛ ɑnɔɔ kɑḿ nɛ tikinɛ kɛ́ nɛ pinɛ́soi símisi. Amɑ́ ḿpɑ́ nɛ lɛ̃, ɑ́pi wɑisɛnɛ. 
Lɛ̃ nnyɑ, Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ pisímɛ́ yɛ pɛpɛɛ ɑ́pi Kirisi m pɑkɑrɛlɛ̃ kɛ́mɛɛ rícere lɛ rɛ linyinɛ yɛ wɑi. Amɑ́ ɑ́i Kirisi pikɔ́ kɛ́mɛɛ linyinɛ yoriyɛ. Limɛcɔ, nɔkɛ́ nɛ Uléécɑɑ rinyiri n símisi yɛ Kirisi pikɔ́ kɛ́mɛɛ rícere lɛ rɛ linyinɛ yɛ wɑi. Amɑ́ ɑ́i pɛɛ pɛpɛɛ ɑ́pi Kirisi m pɑkɑrɛlɛ̃ kɛ́mɛɛ líkɑ yoriyɛ. 
Uléécɑɑ pikɔ́ pin nkpéni n cɑ́pinɛlɛ̃ pin pinnɛ́í Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisi, piyómɛyɑ́hɑɑ pinyinɛ pɛɛ ɑ́pi limɛkɔ́ n nyu nɛ́ɛ pɛpɛɛ ɑ́pi Uléécɑɑ m pɑkɑrɛlɛ̃ pin kei n lompɔ, ɑ́pi yɛ́ mɑɑ rɛ nɔ pisɔhɔtorɛ lɛɛ? 
In tɛ pi pinnɛ́í Uléécɑɑ rinyiri nɛ símisi, kɛ unyinɛ yɛɛ úu Uléécɑɑ m pɑkɑrɛlɛ̃ nɛ́ɛ unyinɛ yɛɛ úu limɛkɔ́ n nyu uu kei lompɔ, u yɛ́ ukɛfɑ-i musí, nsímɛ́ mmɛ̃ nn u nyísɛ rɛ úu ɑlɑri mɑ́, 
kɛmúŋɛ́ kɛɛ ukɛfɑ-i m pékɑɑlɛ̃ ɑkɛ kpɑ̃́ii, uu pɛɛ kɛ́yu kɛtẽ kipi uu Uléécɑɑ pɑkɑrɛ, uu yóó rɛ mɛseisei kɛ Uléécɑɑ uu nɔ́kɛnɛ́cɔpɛ we. 
Yo kɑi nkpéni pisɛ rɛ nɔkɛ́ n wɑi, pimɑ́rɛcɔ? Nɔn yɛ n cɑ́pinɛlɛ̃ tɛ kɛyómɛ kɑni yɑ́ɑ́si, nɔ́kɛnɛ́cɔpɛ unyinɛ yɛ́ɛ́ fe uu yóm, ucɔ uu icélɑɑ inyinɛ pɑ, Uléécɑɑ uu ucɔ nsímɛ́ nyísɛ, ucɔ uu Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ símisi, ucɔ uu mɛɑsei lesɛ. Li pisɛ rɛ linnɛ́í likɛ́ n lɛ́ni, piyómɛyɑ́hɑɑ pin nɛ nfɑtɛnɛ kɛ́mɛɛ pɑpisi. 
Pisoi pin yɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ n símisinɛ, ɑ́pi kɑpɛ yɛ pitɛ́ nɛ́ɛ pitɑɑni rifɑɑu. Pikɛ́ yɛ kɔ usɛ usɛ símisi. Li kɔ pisɛ rɛ unyinɛ ukɛ́ yɛ n we un pinsímɛ́ mmɛ̃ tuisɛ. 
Úkɑ unsɑ́ n we ukɛ́ tuisɛ, li pisɛ rɛ pikɛ́ yɛ risɛ́ɛ́. Pikɛ́ yɛ nɛ pimɛcirɛ kɛfɑ-i n símisi, ɑpi yɛ kɔ nɛ Uléécɑɑ n símisi. 
Li pisɛ rɛ pɛpɛɛ Uléécɑɑ rinyiri nɛ n símisi pitɛ́ nɛ́ɛ pitɑɑni pikɛ́ yɛ símisi picɔ ɑpi yɛ pinsímɛ́ mmɛ̃ músu ɑpi yɛ́nu. 
Amɑ́ in tɛ Uléécɑɑ yɛ icɑ́pinɛ iyɛ-i unyinɛ nsímɛ́ nyísɛ, li pisɛ rɛ uyɛɛ pɛɛ n símɑɑri ukɛ́ risɛ́ɛ́ pikɛ́ nkó kutu ricɔ. 
Nɔ́ yɛ fe ɑni nɔ́nnɛ́nɛ́í Uléécɑɑ rinyiri nɛ símisi ɑni tisɛntɛnɛ, ḿpɑ́ úye uú nɛ icélɑɑ inyinɛ yɛnu uu múlú, ɑi kɔ nɛ ḿpɑ́ úye n ŋmɔ́ɔ́púsɛlɛ̃. 
Pɛpɛɛ Uléécɑɑ rinyiri nɛ n símisi yɛ ihɛɛ iyɛ̃ kɛ Uléécɑɑ uu pi n hɛ kɛcɑ́ɑ́ ńnɑŋɛ mɑ́lɛ. 
Li we rɛ Uléécɑɑ úu mɛ́woo mɛ́woo ukɔ́, ɑmɑ́ nkíŋniŋɛ ukɔ́ lo. Yɑrɛ kɑi Uléécɑɑ pikɔ́ icɑ́pinɛ nnɛ́í kɛ́mɛɛ ń we, 
li pisɛ rɛ pinɔ́si pikɛ́ yɛ Uléécɑɑ icɑ́pinɛ kɛ́mɛɛ n sɛ́ɛ́lɛ̃. Ápi ncée mɑ́ pikɛ́ yɛ kei n símisi. Li pisɛ rɛ pikɛ́ pimɛcirɛ ricɛpisɛ yɑrɛ kɛ isé ii ḿ mɑ. 
Pin linyinɛ n lɑ pikɛ́ céri, pikɛ́ yɛ pipilɑ pisɛ síyɔ-i. Ái nɛ sɑ́ rɛ unɔ́si ukɛ́ yɛ Uléécɑɑ icɑ́pinɛ kɛ́mɛɛ n símisi. 
Nɛ́ɛ nɔ músu rɛ nɔ́kɛnɛ́mɛɛ nkɛ́-i kɛ Nsímɛ́ Kɛcirɛ nn léemɛ? Nɛ́ɛ nɔ̃́ ŋmɑnɛ kɛ́mɛɛ knn hɑ́pɔ? 
Úye un m músu rɛ u Uléécɑɑ rinyiri nɛ símisilɛ nɛ́ɛ u Uléécɑɑ Nfɑɑsɔnɛ ihɛɛ inyinɛ mɑ́, liute ukɛ́ céri rɛ mmɛ̃ kɑm ní nɔ́ ń wɔ́lu yɛ Upíimɑ isé lɛ. 
Amɑ́ úye unsɑ́ lɛ̃ n nyu, Uléécɑɑ pɑ́ɑ kɔ liute nyu. 
Lɛ̃ nnyɑ, pimɑ́rɛcɔ, ɑni n wɛ́ɛ́si nɔkɛ́ yɛ nɛ Uléécɑɑ rinyiri símisi. Amɑ́ ɑ́ni kɑpɛ n yulu rɛ pisoi ɑ́pi kɑpɛ yɛ Nfɑ́ɑsɔnɛ mɛyu mpehẽ mpehẽ símisi. 
Amɑ́ li pisɛ rɛ nɔkɛ́ linnɛ́í n wɑi lin nyɑḿ, ɑ́i kɑpɛ m pulu.
