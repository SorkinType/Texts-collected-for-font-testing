Uléécɑɑ Pitumɛ Mɛwɑi.
8.
Etiyɛɛni nkpɔ mmɛ̃ nń nɛ Sɔɔli sɑ́. Likɛyɑɑ kɛ̃ cirɛ kɑpi Yerusɑlɛm kuyomɛyɑ́hɑɑlee pikɔ́ piwéékusɛ korɑɑnɛ ɑi wɔ́ŋɔɔ. Pinnɛ́í ɑpi Yutee sitẽ nɛ Sɑmɑrii sitẽ kɛcɑ́ɑ́ yɛ́kɑ́ɑ́, ɑi yɑ́ pitumɛ ŋmɑnɛ. 
Pisoi pɛɛ Uléécɑɑ likɔ́ ń lɑ pɛɛ Etiyɛɛni kulɛsi ɑpi nkpɔ tonɛ ɑpi ukɛcɑ́ɑ́ téni hɑ́i. 
Sɔɔli pɔ́ɔn ŋmɔ́ɔ́púlɛ̃ tɛ ukɛ́ Uléécɑɑ icɑ́pinɛ kpu. Síyɔ kuú pɛɛ kɔ́ɔ́núlɛ̃ un Kirisi pikɔ́ cɔ́pii, pitisi nɛ pinɔ́si, un kukpɑniilee súúni. 
Pɛpɛɛ rin-yɛ́kɑɑ yɛ pɛɛ kɛtẽ kɔ́ɔ́núlɛnlɛ pin Nsímɛ́ Kɛcirɛ yóólɛ̃. 
Filipu uu yisi uú hɑ Sɑmɑrii kuyu mɑɑ-i yóó rɛ Yeesu yɛɛ uyɛ̃ kɛ Uléécɑɑ uu n wɛ́ɛ. 
Pisoi nnɛ́í yɛ pɛɛ rinɔ́ɔ risɛ wɑlɛ pin Filipu kutu cɔlɛ̃. Pi pɛɛ kómɛilɛ rɛ u mɛwɑisɑŋɑ wɑpisi, ɑpi kɔ mɛ yɛnu. 
Aníri yɛ pɛɛ pisoi mɛyɑ̃ kɛ́mɛɛ lelɛ ɑn pupukɛɛ. Síkɑnkɑ́lɑ́kɑ́ nɛ pipɑ́íkɔɔ́ kulúi yɛ pɛɛ pélesilɛ. 
Ai pɛɛ lɛ̃ nnyɑ pisoi n lɑ́ɑ́rú kuyu kpɛ-i. 
Utisi unyinɛ yɛɛ hɑ́i mɛkɛɛ-mɛ kuyu kpɛ-i we, ɑpi yɛ u séi rɛ Simɔɔ. U pɛɛ mɛtɑrɛwɑi wɑpisilɛ ɑi yɛ Sɑmɑrii pikɔ́ nnɔ́ɔ yipu. U kɔ pɛɛ umɛcirɛ kpíilɛnlɛ rɛ u úyukɔɔ́ unyinɛ lɛ. 
Pisoi nnɛ́í, kɑi n kpísi siwɑ̃́ nɛ piwɛ́ɛ́sɛ yɛ pɛɛ u wɑisɛlɛnlɛ hɑ́i. Api yɛ mɑɑ rɛ: Uléécɑɑ nnɑŋɛ kɑpi yɛ n sée rɛ ńnɑŋɛ mɑɑ mɛɛ usoi nkó kɛ́mɛɛ we. 
Hɑ́i mɛkɛɛ-mɛ kɛ umɛtɑrɛwɑi ɑmɛ nnɔ́ɔ pi yipɑɑlɛ̃. Api pɛɛ lɛ̃ nnyɑ u n wɑisɛlɛ̃. 
Amɑ́ kɛ Filipu uu Uléécɑɑ iyɔɔpi nɛ Yeesu Kirisi kɛcɑ́ɑ́ Nsímɛ́ Kɛcirɛ pi n símisi ɑpi ŋmurɛi, pi tíyɛsɛlɛ ɑpi míni pi wolɛ, kɑi n kpísi pitisi nɛ pinɔ́si. 
Simɔɔ uyɛ̃ ticuruu pɔ́ɔ kɔ nɛ Nsímɛ́ Kɛcirɛ mmɛ̃ kɛfɑ tɛnɛ ɑpi míni u wolɛ. Uu pɛɛ Filipu n tikilɛ̃ un mɛwɑisɑŋɑ nɛ mɛwɑi píimɑ kɛ uyɛ̃ uu n wɑpisi nyɑ́nɛi mɛn nnɔ́ɔ u yipɑɑlɛ̃. 
Pitumɛ pɛɛ Yerusɑlɛm-i ń we ɑpi kóm tɛ Sɑmɑrii kɛtẽ pikɔ́ yɛ Nléécɑɑsimɛ́ yɔsí. Api pɛɛ Piyɛɛ nɛ Yohɑni kei tumpɔ. 
Pikɛtɛ́ kɛ̃ ɑpi yisi ɑpi sĩ́ ɑpí hɑ nɛ Uléécɑɑ welu rɛ ukɛ́ Sɑmɑrii pikɔ́ pɛ̃ Nfɑ́ɑsɔnɛ hɛ. 
Nfɑ́ɑsɔnɛ ńn pɛɛ kɑhɑnɛ likumúŋɛ́ kpɛ-i úkɑ kɛcɑ́ɑ́ n súimɛ. Míni ŋmɑnɛ kɑpí pɛɛ́ nɛ Yeesu rinyiri pi wolɛ. 
Kei kɛ Piyɛɛ nɛ Yohɑni ɑpi pɛɛ ɑnípɛ pikɛcɑ́ɑ́ rilɑɑ, Nfɑ́ɑsɔnɛ nn pɛɛ pi loni. 
Kɛ Simɔɔ uu n yɛ̃́ tɛ pitumɛ yɛ pisoi kɛcɑ́ɑ́ ɑnípɛ rilɑɑ Nfɑ́ɑsɔnɛ nn pi loni, uú nɛ siwóó hɑpɔ uu pitumɛ pɛ̃ mɑɑ rɛ: 
Ani kɔ nɛ̃́ ńnɑŋɛ mmɛ̃ hɛ kɛ́ yɛ nɛn úye kɛcɑ́ɑ́ ɑnípɛ rinlɑɑ, ukɛ́ Nfɑ́ɑsɔnɛ yɔsí. 
Amɑ́ Piyɛɛ uu u pɛsɛ rɛ: Nɔ́ nɛ sipɔ́wóó sɛ̃ ɑni ɑ́sɑ́lɑ́ú wɑ. Pɔ musí rɛ pɔ́ fe ɑɑ Uléécɑɑ lihɛɛhɛɛ nɛ siwóó lɔluu? 
Lipɔ́kɔ́ líkɑ ɑ́i nté we, ɑ́ɑ kɔ líkɑ kɛcɑ́ɑ́ nté ncée mɑ́. Li we rɛ ɑ́ɑ Uléécɑɑ kɛ́mɛɛ ɑsei mɑ́. 
A kɛpɔ́múŋɛ́ kópɛ kɛpirɛ ritɔ ɑɑ Upíimɑ nɛ téni rɛ lin kusɑ́rɛi n we, ukɛ́ kɛmúŋɛ́ kɛ̃ pɔ́ sɑ́rɛi. 
Nɛ nyɑ́ni yɑrɛ nfɑsimɛ́ kópɛ nɛ rikpɑ́kɑ́rɑ́ kɑɑ n fɑnɑɑlɛ̃ yɛ pɔ́ múílɛnlɛ yɑrɛ ulɑ́si. 
Kei kɛ Simɔɔ uu pɛɛ rinɔ́ɔ yɔsí rɛ: Ani nɔ́rinɛ́cúruu nɛ Upíimɑ nɛ́ tẽ́, lɛ̃ kɑni nní ḿ mɑ ɑ́i kɑpɛ nɛ nɛ́ n wɑ nnyɑ. 
Kɛ Piyɛɛ nɛ Yohɑni ɑpi Sɑmɑrii kuyu kpɛ-i Upíimɑ nsímɛ́ piyóó nɛ picélɑɑ ḿ mɑsí, ɑpi yisi ɑpi Yerusɑlɛm pɛlɛ, pin ncée-i siyupi mɛyɑ̃ kɛ́mɛɛ Sɑmɑrii kɛtẽ kɛcɑ́ɑ́ Yeesu Nsímɛ́ kɛcirɛ yóólɛ̃. 
Upíimɑ uleecɑɑtumɛ unyinɛ uu Filipu mɑɑ rɛ: A yisi ɑɑ ituŋɛ kumii-mɛ̃ pɑnsɛ ɑɑ Yerusɑlɛm ncee mɛɛ Kɑsɑɑ n cɛ́pilɛ̃ kpísi. Ápi yɛ ncée mmɛ̃ titiki. 
Filipu uu mɛsɛ nɛ mɛsɛ yisi uu tɔ́mpɔ, uú nɛ ncée Etiyopii kɛtẽ utisi unyinɛ sɑ́nɛ. Ukɑwe píimɑ unyinɛ lo . Uyɛɛ Etiyopii unɔsiyɔɔpi Kɑntɑɑsi mɛmɑ́ nnɛ́í kɛcɑ́ɑ́ pɑílɛ̃. U Yerusɑlɛm-i kɑlɛ rɛ ukɛ́ Uléécɑɑ yɑ́ɑ́si 
un pɛɛ nkpéni kulu, un ukɛsɑnnɔipi kɛcɑ́ɑ́ tṹ un ɑntepu Esɑyi ritɛlɛ́ kɛ́ɛ́ni. 
Nfɑ́ɑsɔnɛ nn Filipu mɑɑ rɛ: A kɛnɔ́ipi nkɛ́ kɛkúrí hɑpɔ. 
Filipu uú nɛ itóó nyɔsɔ́pɔ uu kóm Etiyopii ukɔ́ uyɛ̃ un ɑntepu Esɑyi ritɛlɛ́ kɛ́ɛ́ni. Filipu uu u pisɛ rɛ: Pɔ lɛ̃ kɑɑ n kɛ́ɛ́ni ɑsei kómɛii? 
Utisi uu rɛ: Unyinɛ unsɑ́ nɛ́ n lɛ̃́, íye kɑm yɛ́ nɛ kõ? Uu pɛɛ Filipu séi rɛ ukɛ́ hɑpɔ ukɛ́ kɛsɑnnɔipi kɛ́mɛɛ ukɛkúrí tonɛ. 
Li Uléécɑɑ Nsímɛ́ kɛlõ kɛ̃ kuu n kɛ́ɛ́ni kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Pi nɛ u tɔ́sulɛ yɑrɛ kɑpi yɛ kusɑ́ŋyɑhɑ n tɔ́rɔi pin nɛ pikópɛ sĩ, un sɛ́ɛ́lɛ̃ yɑrɛ risɑ́ŋpipi kɑpi nkpíí ń nyɑ́lu, úu rinɔ́ɔ wúkulɛ. 
Apí nɛ u túhɑɑnɛ ɑpi u cɛpisɛ, úu pɛɛ nkpɑ́ni líkɑ tu. Wóo yɛ́ pɛɛ nkpɑ́ni upipirɛ̃ tinɔɔ mɑɑ? Pi kɛtẽ kɛcɑ́ɑ́ unfɑɑ lesɛlɛ. 
Kuu lɛ̃ pikɛ́ɛ̃ m mɑsí, uu pɛɛ Filipu pisɛ rɛ: A risuúlu ɑ nɛ́ símisi nkɛ̃́ tɛ úye nsímɛ́ kɛ ɑntepu nkó uu nní símisi. Uricuruu kuú nɛ mɑ́nɛ nɛ́ɛ unyinɛ nsímɛ́ kuu símisi? 
Kei kɛ Filipu uu pɛɛ́ nɛ ɑntepu Esɑyi nsímɛ́ mmɛ̃ riwɑ́ɑ́nɑ́ uú nɛ Yeesu nsímɛ́ kɛcirɛ u símisi. 
Api nsẽ́ nɔŋ́ ɑpí hɑ nɛ míni kɛlõ kɛnyinɛ tulu. Ukɑwe píimɑ uyɛ̃ uu pɛɛ rɛ: Míni yɛ mmɛ́ nní. Yo icɔ yɛɛ pɑ́rílɛ̃ kɛ́ nɛ míni ń wolɛ? 
Filipu uu u pɛsɛ rɛ: In tɛ pɔ nɛ kɛpɔ́fɑ nnɛ́í nsímɛ́ mmú tɛnɛ, pi yɛ́ fe ɑpi míni pɔ́ wolɛ. Utisi uu rɛ: Nɛ nɛ kɛfɑ tɛnɛ rɛ Yeesu Kirisi yɛ Uléécɑɑ Kɛpipi lɛ. 
Uu kɛsɑnnɔipi nyɛrɛsɛ, u nɛ Filipu ɑpi míni-i lompɔ, Filipu uu míni u wolɛ. 
Kɑpi míni-i n léepɔ, Upíimɑ Nfɑɑsɔnɛ nn Filipu kpíípɔ, ukɑwe uyɛ̃ úu pɛɛ piyɛ́nɛ u rikpɑ́. Uyɛ̃ uu kɔ mpɔ́ɔnɑrɛ nɛ uncee nɔŋ́. 
Filipu pɔ́ɔ hɑ Asɔɔti kɛtẽ-pɔ leeri. Uu pɛɛ kei ɑyu nnɛ́í nyɛɛ ncée-i ń we kɛ́mɛɛ Nsímɛ́ Kɛcirɛ n yóólɛ̃ uú hɑ nɛ Sesɑree kɛtẽ tulu.
