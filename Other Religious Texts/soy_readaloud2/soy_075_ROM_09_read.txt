Rɔm.
9.
Lɛ̃ kɑm nní ḿ mɑɑnɛ yɛ nsímɛ́sei lɛ. Nɛ Kirisi ukɔ́ lɛ, ɑ́m kirɑɑsɛ. Uléécɑɑ Nfɑɑsɔnɛ yɛ tíyɛsɛlɛ ɑm céru rɛ ɑsei yɛ nyɛ. 
Asei yɛ nyɛ rɛ tinɛ́pɔ́ɔ yɛ mɛyɑ́nsei cɑɑilɛnlɛ nɛn kɔ mɛsɛ́rɛ kɛnɛ́fɑ-i nɛ íwɛ pɑnsɛntɛ. 
Nɛ́ pɛɛ Uléécɑɑ pisɛ rɛ ukɛ́ ɑnɔ́ɔ nɛ́ wɑ tɔ́ nɛ Kirisi tɔkɛ́ túúnnɛ likɛ́ pinɛ́mɑ́rɛcɔ, tɔ́ nɛ pɛ̃ kɑpi mpuri nsɛ-i m mɑ́ri yoriyɛ. 
Pi Isirɑyɛɛli pikɔ́ lɛ kɛ Uléécɑɑ uu m pɑnsɛsɛ upikɔ́, pin umɛyɔɔpi kɛ́mɛɛ́ we. Pɛ̃ kuú nɛ nkómɛinɛ sée uu isé pi pɑ, uu kɔ pi céési lɛ̃ kɑpi yɛ́ uinyɔ́ɔnsɛ n wɑ, uu rinɔ́ɔ pi yekei. 
Uléécɑɑ yɛ hɑ́i pɛɛ mɛkɛɛ-mɛ pipikpurɛ wɛ́ɛlɛ. Kirisi ricuruu yɛ ukɛpirɛ ukɔ́ lɛ, tɔn kei kɑpi kɛtẽ nkɛ́ kɛcɑ́ɑ́ u m mɑ́ri m pɑinɛ. Kirisi uyɛɛ pɛɛ kɔ ḿpɑ́ yo fe. Uyɛɛ Uléécɑɑ yɛɛ ḿpɑ́ píyei ipɑkɑrɛ n te. Amí! 
Ái we rɛ usoi ukɛ́ musí rɛ Uléécɑɑ rinɔɔ ɑ́ri nkpéni ńnɑŋɛ ńkɑ mɑ́. Ái rɛ Isirɑyɛɛli pikɔ́ nnɛ́í pɛɛ Isirɑyɛɛli pikɔ́ yíkíyiki. 
Limɛcɔ, ɑ́i rɛ Apirɑhɑm kumɑrɛ pikɔ́ nnɛ́í pɛɛ Apirɑhɑm sipipi yíkíyiki. Uléécɑɑ yɛ Apirɑhɑm mɑɑ rɛ: Isɑɑki kɛ́mɛɛ́ kɑɑ sipírɛ̃ yɛnunɛ ɑpi yɛ ripɔ́nyíri si séi. 
Lɛ̃ nnyɑ, ɑ́i Apirɑhɑm sipipi nnɛ́í sɛɛ Uléécɑɑ sipipi. Uléécɑɑ yɛ mɑɑ rɛ usipipi yíkíyiki sɛɛ nní sɛ̃ kɑpi Urinɔɔ kuu n yekei kɛcɑ́ɑ́ ḿ mɑ́ri. 
Asei kɛcɑ́ɑ́, tinɔ́ɔ tɛɛ n yisunu yɛ ntí: Kɛ́ nɛ kɛlɑɑnɛ nní icɔ m pɛɛmɛ, Sɑrɑ un kɛwɑ́ntisi mɑ́. 
Ái nɛ kei ŋmɑnɛ ripɑ́. Limɛcɔ kɑi kɔ Repekɑ piŋmɑ́nɛ kɛtɛ́ kɛ́mɛɛ́ we. Urɔ́sɑ́ɑyɑhɑ Isɑɑki yɛɛ pɛɛ pikɛtɛ́ usɑɑ. 
Úu pɛɛ kɑhɑnɛ pi m mɑri, ɑ́pi pɛɛ kɔ kɑhɑnɛ lisɔnɛ nɛ́ɛ likópɛ n wɑ. Ḿpɑ́ nɛ lɛ̃, Uléécɑɑ yɛ pɛɛ Repekɑ mɑɑ rɛ: Uwɛ́ɛ́sɛ yɛ́ pɑnsɛ uwɑ̃́ ulɑsi yɑrɛ kɑi n wɔ́lɑɑlɛ̃ tɛ: "Yɑkupu ipuri kɑḿ lɑ, ɑm Esɑu ikɔ́ yúlu." Uléécɑɑ kɛmúŋɛ́ kɛkɛ́ nɛ n wɑ nnyɑ kɑi lɛ̃ wɑ. Ái kɛsoipipi mɛwɑi kuú pɑílɛ̃ un nɛ wɛ́ɛ́kɛɛ, ɑmɑ́ uricuruu kɛmúŋɛ́ kɛ́mɛɛ́ kɑi léeri kuu yɛ nɛ usoi wɛ́ɛ. 
Lɛ̃ nnyɑ, tɔ́ nkpéni mɑɑ rɛ íye? Tɛ Uléécɑɑ úu ɑsei ute nɛ́ɛ? Ái nkpɑ́ni lɛ̃. 
U pɛɛ Moisi mɑɑ rɛ: Nɛn úye n lɑ kɛ́ íwɛ tẽ́, nɛ yɛ liute íwɛ ténlɛ, nɛn kɔ úye n lɑ kɛ́ nɛ kɛfɑ fénnɛ, nɛ yɛ liute nɛ kɛfɑ fénnɛlɛ. 
Lɛ̃ nnyɑ, ɑ́i usoi mɛlɑ kɛ́mɛɛ léeri, ɑ́i kɔ usoi nŋmɔ́ɔ́pú kɛ́mɛɛ léeri, ɑmɑ́ Uléécɑɑ yɛɛ yɛ usoi íwɛ n tẽ́ ŋmɑnɛ kóló kɛ́mɛɛ kɑi léeri. 
Lɛ̃ kɛ Uléécɑɑ uu Esipiti Uyɔɔpi mɑɑ rɛ: Nɛ pɔ́ lésɛmɛlɛ rɛ kɛ́ nɛ pɔ́ ritiki kɛ́ nɛ nnɛ́nɑŋɛ nyísɛ rinɛ́nyíri tikɛ́ nɛ kɛtẽ nnɛ́í rinkɔ́ɔ́nú nnyɑ. 
Lɛ̃ nnyɑ, Uléécɑɑ un úye n lɑ ukɛ́ íwɛ tẽ́, u yɛ liute íwɛ ténlɛ, un kɔ úye rikiŋ n lɑ ukɛ́ tɑ́kɑɑsɛ, u yɛ liute rikiŋ tɑkɑɑsɛlɛ. 
In n lɑ unyinɛ yɛ́ fe uu nɛ́ pisɛ rɛ: In lɛɛ ni, yo nnyɑ kɛ Uléécɑɑ uu yɛ pɛɛ́ nɛ pisoi cɛ́si? Wóo yɛ́ mpíí fe ukɛ́ umɛlɑ kɛsi? 
Pɔ́ kɛsoipipi, pɔ́ɔ úye pɔ́ɔkɛ́ nɛ Uléécɑɑ n kɛ́ŋɛnɛ? Kɛtúhu yɛ́ fe ɑkɛ umɔ́mɛ pisɛ rɛ yo nnyɑ kɑɑ nní nɛ́ mɔ̃́ɔ? 
Utúhumɔmɛ yɛ ncée mɑ́lɛ ukɛ́ nɛ lɛ̃ kuu ń lɑ mɛtipipɛrɛ́ wɑ. U yɛ́ fe uu mɛpɛ́rɛ́ mɛsɛ mɛ̃ kpísi uú nɛ kɛtúhu mɔm, ɑpi yɛ nɛ líyukɔɔ́ kɛ wɑi, uu kɔ kɛcɔ mɔm, ɑpi yɛ kɛ̃ nɛ likpɑ́rɑ́kɔɔ́ wɑi. 
Uléécɑɑ yɛ pɛɛ lɑlɛ ukɛ́ ukuwɔi nɛ unnɑŋɛ pinyinɛ kɛ́mɛɛ nyísɛ. Pɛpɛɛ sitúhu sɛ̃ kɑpi yɛ nɛ likpɑ́rɑ́kɔɔ́ n wɑ. Amɑ́ ḿpɑ́ nɛ lɛ̃, uu umɛcirɛ tini uu pɛpɛɛ pɛɛ́ nɛ nkpɔ n sɑ́ m múílɛ̃ un tikíŋ tɑ́pɑɑlɛ̃. 
U kɔ pɛɛ́ lɑlɛ ukɛ́ picɔ kɛ́mɛɛ nyísɛ rɛ umɛyɔɔpi ɑ́mɛ kumúŋɛ́ mɑ́. Pɛ̃ kuu íwɛ tẽ́. Pɛpɛɛ sitúhu sɛ̃ kɑpi yɛ nɛ líyukɔɔ́ n wɑ. Pɛ̃ kuu kɔ mɛkɛɛ-mɛ wɛ́ɛ rɛ pikɛ́ umɛyɔɔpi kɛ́mɛɛ n we. 
Lɛlɛɛ nní tɔ́ mpí kuu nní n sée. Ái kɔ Pisuifi ŋmɑnɛ, ɑmɑ́ nɛ mpuri sɑnɛ pikɔ́ kuu sée. 
Lɛ̃ kɛ Uléécɑɑ uu Osee ritɛlɛ́ kɛ́mɛɛ mɑɑ rɛ: Mpuri mmɛ mɛɛ ńn pɛɛ nnɛ́kɔ́ kɑm sɔ́nti kɛ́ sée rɛ nnɛ́puri. Mpuri mmɛ̃ isoi kɑ́m yɛ pɛɛ ń lɑ kɑm sɔ́nti kɛ́ sée rɛ nnɛ́puri lɑlɑ píimɑ. 
Pɛ̃ kɑm yɛ pɛɛ ḿ mɑɑ rɛ: Áni pinɛ́kɔ́, kɑm sɔ́nti kɛ́ sée rɛ Uléécɑɑ yɛɛ nfɑ́ɑ n te sipipi. 
Antepu Esɑyi pɔ̃́ nɛ kɔ Isirɑyɛɛli pikɔ́ kɛcɑ́ɑ́ nsímɛ́ mɑɑ rɛ: Ḿpɑ́ Isirɑyɛɛli sipipi sikɛ́ kɑm sikɛ́ kulúi n we yɑrɛ mínimɑɑ ritimɛ nnyɛnɛ, pisoi nkɑ́ripi nnyinɛ ŋmɑnɛ riyu kɛ Uléécɑɑ uu lɔlunɛ. 
Upíimɑ yɛ rinɔ́ɔ tɛ̃ kuu n yekei wɑinɛ nɛ kɛ́tɔ-pɔ, kɛtẽ kɛcɑ́ɑ́ ɑ́i nkpɑ́ni sésénɛ. 
Lɛ̃ kɛ Esɑyi uu mɛkɛɛ-mɛ wɔi rɛ: Uléécɑɑ, kɛtẽ nɛ kɛyómɛ ute unsɑ́ pɛɛ ntɔ́puri kɛ́mɛɛ pinyinɛ n tíyɛ, tɔ́ pɛɛ pɑnsɛlɛ yɑrɛ Sotɔm pikɔ́, Komɔɔri pikɔ́ kɑri yɛ́ pɛɛ́ nɛ mɛnyinɛ n we. 
Lɛ̃ nnyɑ, tɔ́ nkpéni mɑɑ rɛ íye? Tɛ mpuri sɑnɛ pikɔ́ pɛɛ ɑ́pi n wɛ́ɛ́si pikɛ́ pɑnsɛ ɑsei pite yɛ kɛfɑ kɑpí nɛ u n tɛnɛ nnyɑ pɑnsɛ ɑsei pite. 
Isirɑyɛɛli pikɔ́ pɛɛ n ŋmɔ́ɔ́púlɛ̃ pikɛ́ isé ritiki pikɛ́ nɛ pɑnsɛ ɑsei pite pɔ́ɔ pɛɛ mɛníŋɛ́ lólu, ɑ́pi lɛ̃ kɑpí pɛɛ ń lɑ yɛ̃. 
Íye kɑi pɛɛ lɛ̃ wɑ? Ápi wɛ́ɛsi rɛ pikɛ́ nɛ Uléécɑɑ kɛfɑ tɛnɛ. Amɑ́ pi pɛɛ́ lɑlɛ pikɛ́ piricuruu liwɑiwɑi nnyɑ pɑnsɛ ɑsei pite Uléécɑɑ inipɛɛ-i. Kei kɑpi pɛɛ́ nɛ “ripɑrɛ tɛɛ yɛ pisoi n lóisɛ rikpɔu.” 
Tipɑrɛ tɛ̃ nsímɛ́ mɛɛ Nléécɑɑsimɛ́ ritɛlɛ́ kɛ́mɛɛ wɔ́lɑɑlɛ̃ tɛ: Nɔ yɛ̃ɛ, nɛ Siyɔɔ kɛtẽ-pɔ ripɑrɛ yekeilɛ tɛɛ yɛ pisoi rinkpɔ́usɛ nɛ kupɑrɛcɑŋɑɑ kpɛɛ yɛ pisoi n lóisɛ. Amɑ́ nkó yɛɛ ripɑrɛ tɛ̃ nɛ kɛfɑ n tɛnɛ úu isɛi mɛlukɛ mɑ́.
