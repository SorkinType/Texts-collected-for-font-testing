Lom.
2.
Ɩlɛ nyɑ́ wei n footiɣi nyɑ́ lɛlʋ tɔ, pɑɑ n kɛ́ we te, n fɛɩnɑ tɑmpɑnɑ. Pǝ tɑɣɑ pʋlʋ, nyɑ́ footiɣi lɛlʋ pǝyele ḿpʋ́ɣʋ́ nyɑ́ mɑɣɑmɑɣɑ nyɑ́ lɑkɩ tɔ, n kʋɣɩ nyɑ́ tɩ tɔm kɛ́. 
Tǝ nyǝmɑ́ sɩ tɑmpɑnɑ kɛ Ɩsɔ tǝŋǝɣɩ nɑ ɩ́ hʋʋkǝnɑ pɑ tǝnɑ mpɑ pɑ lɑkɩ mpʋ ɩnɩ pǝ tɑkɑ tɔ te. 
Ɩlɛ ɑpɑŋnyɑ, ɩsɩɩ nyɑ́ footuɣuu mpɑ pɑ lɑkɩ mpʋ, pǝcɔ nyɑ́ mɑɣɑmɑɣɑ nyɑ́ lɑkɩ teitei ɩsɩɩ pɛlɛ tɔ, n hʋʋ sɩ Ɩsɔ kɑɑ pɔɔsɩ-ŋ? 
Ɩsɔ kʋcɔɔʋ ntɛ́ n mǝlɑ ń kpɑlǝɣɩ nɑ ń kisiɣi ɩ suulu sɩ ɩ́ tɑɑ wii nyɑ́ pǝtɔɔtǝlɛ? N tɑ nyɩ sɩ Ɩsɔ lɑkɩ-ŋ kʋpɑntʋ kɛ ḿpʋ́ɣʋ́ sɩ ɩsɔwɛɛ n kɑ́ lɑɣɑsɩ tɔntɛ? 
Nyɑ́ lɑŋkpɩɩsǝŋ ɩnɛ. Amɑ nn cɑɑ lɑɣɑsʋɣʋ se. N lɑkʋɣʋ mpʋ tɔ, sɑlǝkɑ kʋkǝlʋ kɛ n cɑɑkɩ nɑ ń kɔŋnɑ nyɑ́ nyʋɣʋ tɑɑ kɛ́ kʋyɑŋku Ɩsɔ kɑ́ cɑɣɑ nɑ ɩ́ hʋ́lɩ́ ɩ pɑ́ɑ́nɑ́ nɑ ɩsǝnɑ ɩ tǝŋǝɣɩ siɣisuɣu nɑ ɩ́ hʋʋkǝnɑ tɔ. 
Kʋyɑkʋ ŋkʋɣʋ Ɩsɔ kɑ́ fɛlɩ pɑɑ wei kɛ́ ɩ kʋlɑpʋtʋ. 
Mpɑ pɑ sɔɔlɑ kʋpɑntʋ lɑpʋ kɛ tɑm, nɑ pɑ́ pɛɛkǝɣɩ Ɩsɔ kiŋ teeli, nɑ sɑm nɑ tɑm wɛɛtʋ tɔ, mpɛɣɛ Ɩsɔ kɑ́ hɑ weesuɣu ŋku kɩɩ tɛŋ tɔ. 
Amɑ mpɑ pɑ sɔpɑ nyʋɣʋ nɑ pɑ nyǝntʋ tɛlǝsʋɣʋ kɛ lɔlɔ, nɑ pɑɑ tǝŋǝɣɩ tɑmpɑnɑ, nɑ pɑ́ lɑkɩ pǝ tǝnɑ mpi pǝ fɛɩ teu tɔ, Ɩsɔ kɑ́ hʋ́lɩ́ pɛlɛɣɛ ɩsǝnɑ ɩ tɑɑ kpɑɑnɑ-wɛ nɑ pɑ tɔm tɑ mɑɣɑ-ɩ tɔ. 
Wɑhɑlɑ nɑ lɑŋwɑkǝllɛ sɔsɔɔlɛ kɑ́ kpɑ pɑ tǝnɑ mpɑ pɑ lɑkɩ ɩsɑɣɑtʋ tɔ. Yutɑ nyǝ́mɑ kɛ kɑncɑɑlɑɣɑ, pǝ wɑɑlɩ kɛ́ mpɑ pɑ tɑ kɛ Yutɑ nyǝ́mɑ tɔ. 
Amɑ mpɑ pɑ lɑkɩ kʋpɑntʋ tɔ, Ɩsɔ kɑ́ kʋsɩ pɛlɛ pɑ nyʋɣʋ nɑ ɩ́ sɑ-wɛ nɑ ɩ́ hʋlǝsɩ pɑ lɑŋɑ. Yutɑ nyǝ́mɑ kɛ kɑncɑɑlɑɣɑ, pɑ wɑɑlɩ kɛ́ mpɑ pɑ tɑ kɛ Yutɑ nyǝ́mɑ tɔ. 
Mpi tɔ Ɩsɔ ɩɩ fɑɣɑsǝɣɩ yǝlɑɑ tɔ se. 
Mpɑ pɑ tɑ nyɩ Moisi kʋsǝsɩɩtʋ nɑ pɑ́ pǝntǝɣɩ tɔ, pɑɑ le nɑ kʋsǝsɩɩtʋ ntɩ tǝ tɔm tɑ nɑ-wɛ. Amɑ mpɑ pɛlɛ pɑ nyǝmɑ́ Moisi kʋsǝsɩɩtʋ nɑ pɑ́ tɔŋnɑ pǝntʋɣʋ tɔ, kʋsǝsɩɩtʋ ntǝɣɩ pɑɑ kɛɛsǝnɑ nɑ pɑ́ hʋʋnɑ-wɛɣɛ tɔm. 
Mpi tɔ pǝ tɑɣɑ yǝlɑɑ mpɑ pɑ nɩɩkɩ kʋsǝsɩɩtʋ tɔm kɛ kʋnɩɩ tɔ mpɛɣɛ Ɩsɔ nyǝnǝɣɩ kʋpɑmɑ se. Amɑ mpɑ pɑ tɔkɑ-tɩ tɔɣɔ. 
Ye piitim lɛmpɩ nyǝ́mɑ nɔkɑɑ nɑ pɑ́ tǝŋǝɣɩ mpi Yutɑ nyǝ́mɑ kʋsǝsɩɩtʋ kɛɛsǝɣɩ tɔ, pǝyele nɔɣɔlʋ tɑ hʋlɩ-wɛ, ɩlɛ pɑ kʋsǝsɩɩtʋ wɛ pɑ mɑɣɑmɑɣɑ pɑ tɑɑ kɛlɛ, pɑɑ nɑ pɑ tɑɑ fɛɩnɑ-tɩ tɔ. 
Pɑ lɑkʋɣʋ mpʋ tɔ, pɑ hʋ́lǝ́ɣɩ́ kɛlɛ sɩ, pɑɑ pɑ tɑɑ fɛɩnɑ kʋsǝsɩɩtʋ, pǝ tʋ Moisi nyǝntʋ ntɛ́ pɑ lotunɑɑ tɑɑ. Nɑ pǝ nɑɑkɩ ḿpʋ́ɣʋ́ pɑ lɑɑkɑlǝnɑɑ tɑɑ. Pǝ tɑɣɑ pʋlʋ tɔɔ, wutee wutee pɑ pǝtǝɣɩ pɑ kʋlɑpʋtʋ tɑɑ kɛ́, nɑ wutee wutee pɑ wɛnɑ nɑɑni. 
Ḿpʋ́ɣʋ́ pɩɩ lɩɩ teitei kɛ́ kʋyɑŋku Ɩsɔ kɑ́ yele sɩ Yesu Kilisiti ɩ́ hʋʋnɑ yǝlɑɑ kɛ pɑ mukɑɣɑ tɛɛ kʋlɑpʋtʋ tɔɔ tɔ. Nɑ pǝ mɑɣɑmɑɣɑ kɛ Lɑɑpɑɑlɩ Kʋpɑŋ wei mɑ kpɑɑlǝɣɩ tɔ ɩ hʋ́lǝ́ɣɩ́. 
Amɑ nyɑ́, n yɑɑ nyɑ́ tǝɣɩ Yutɑ tʋ Yutɑ tʋ, nɑ Moisi kʋsǝsɩɩtʋ ntǝɣǝlɛ Moisi kʋsǝsɩɩtʋ, nɑ ń mʋɣʋlǝɣǝnɑ nyɑ́ Ɩsɔ. 
Pǝ́cɔ́ n nyǝmɑ́ mpi Ɩsɔ cɑɑ sɩ ń lɑ tɔ, nɑ kʋsǝsɩɩtʋ ntɩ tǝ kulɑ nyɑ́ ɩsɛ kɛ teu sɩ ń cɛkǝnɑ kʋpɑntʋ. 
N nyǝnǝɣɩ nyɑ́ tǝɣɩ yʋlʋmɑɑ hɔntʋ nɑ sǝkpɛtʋɣʋ tɛɛ nyǝ́mɑ ɩsɛntɔɔ cɛtʋ, 
nɑ ɩsǝkpɛɛtɛɛ nyǝ́mɑ ɩsǝkullu, nɑ piyɑ sɛɣɛsǝlʋ. Mpi pǝ tɔɔ n nyǝnǝɣɩ nyɑ́ tǝɣɩ mpʋ tɔ, n tɛɛlǝɣɩ kɛ́ sɩ nyɑ́ nɑnɑ pǝ tǝnɑɣɑ kʋsǝsɩɩtʋ ntɩ tǝ tɑɑ, pɑɑ nyǝm, pɑɑ tɑmpɑnɑ. 
Aŋhɑ, nyɑ́ wei n sɛɣɛsǝɣɩ lɛlɑɑ tɔ, nn sɛɣɛsǝɣɩ nyɑ́ tɩ suwe? Nyɑ́ wei n lɑkɩ wɑɑsʋ sɩ pɑ́ tɑɑ ŋmɩɩlɩ tɔ, pepe tɔɔ kɛ́ nyɑ́ ŋmɩɩlǝɣɩ? 
Nyɑ́ wei n tɔŋ sɩ pɑ́ tɑɑ lɑ wɑsɑŋkɑlǝtʋ tɔ, pepe tɔɔ kɛ́ nyɑ́ lɑkɩ? Nyɑ́ wei nn cɑɑ tɩɩŋ tɔm kɛ pɑɑ pǝcɔ tɔ, pepe tɔɔ kɛ́ n nɑɣɑlǝɣɩ ɩ tɛɛ kʋpɑmpɑm? 
Pǝ tɑɣɑ nyɑ́ mʋɣʋlǝɣǝnɑ sɩ n wɛnɑ Ɩsɔ kʋsǝsɩɩtʋ? Ɩlɛ pepe tɔɔ kɛ́ n pǝntǝɣɩ-tɩ nɑ pǝ́ kʋɣɩ Ɩsɔ nyʋɣʋ? 
Ɩsɔ Tɔm tɑɑ pɑ ŋmɑɑwɑ tǝkelekele sɩ: Yutɑ nyǝ́mɑ mɛ mǝ yelinɑ nɑ piitim lɛmpɩ nyǝ́mɑ tʋʋkɩ Ɩsɔ. 
Ye n tɔkǝɣɩ Moisi kʋsǝsɩɩtʋ kɛ pɛlʋɣʋ wɑɑsǝɣɩ-ŋ. Amɑ ye nn tɔkǝɣɩ-tɩ, pǝ wɛ ɩsɩɩ pɑ tɑ pɛlɩ-ŋ kɛlɛ, pɑɑ n pɛlɑɑ. 
Ye yʋlʋ ɩ́ tɑ pɛlɩ, pǝyele ɩ tǝŋǝɣɩ teu kɛ Moisi kʋsǝsɩɩtʋ kɛɛsɑ nti tɔ, Ɩsɔ kɑɑ nyǝnɩ pʋntʋ ɩsɩɩ pɛllʋ? 
Wei ɩ tɑ pɛlɩ pǝyele ɩ tɔkɑ kʋsǝsɩɩtʋ ntɩ tɔ, ɩ kɑ́ kʋ Yutɑ tʋ nyɑ́ nyɑ́ tɔm. Pǝ tɑɣɑ pʋlʋ tɔɔ, nyɑ́ wɛnɑ kʋsǝsɩɩtʋ ntɩ tǝ tɑkǝlɑsɩ, nɑ n pɛlɑɑ pǝyele nn tǝŋǝɣɩ-tɩ. 
Tɔʋ, ɩsǝnɑɣɑ Yutɑ tʋ siɣisiɣi nɑ́ wɛɛ? Pǝ tɑɣɑ ɩsɛntɔɔ nɑ ɩ pɛlʋɣʋ kiŋ kɛ pɑ nyǝŋnɑ-ɩ sɩ Yutɑ tʋ. 
Amɑ Yutɑ tʋ siɣisiɣi ɩlɛ, pɑ nyǝŋnɑ-ɩ nɑ ɩ tɑɑ kɛ́, nɑ hɑtuu ɩ lotu tɑɑ kɛ́ pɑ pɛlɑ-ɩ. Ɩsɔ Feesuɣu pɛlǝnɑ-ɩ, pǝ tɑɣɑ Ɩsɔ kʋsǝsɩɩtʋ nti pɑ ŋmɑɑ kʋŋmɑɑ tɔ tǝ tɔm tɔkʋɣʋ tɔɔ. Yutɑ tʋ siɣisiɣi kɛ mpʋ, Ɩsɔ sɑŋnɑ-ɩ, pǝ tɑɣɑ yǝlɑɑ.
