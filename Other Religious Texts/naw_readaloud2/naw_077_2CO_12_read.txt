Ɩ dagaa n maa kyaga sʋ kaapʋ mɛ n‐yɩɩ. Halɩɩ ɩ maa man sʋ tɔnɔ gbaa mʋ, nbɩɩnbɩɩ bɩrɛ n biti n tɔwɛ atɔ mʋ nɩɩ Ɩbwaarɛ yɛgɛ nɩɩ n wu fɛɛ idee, yɛ asɩnlɛɛ kaapʋsa mʋ nɩɩ ɔ taa sa mɛ. 
Ɔnyɩn kʋ kyɩna ɔ bʋ tɔ, nɩɩ ɔ gyɛ Krisito obuupu. Nsu kudu ana ndee nɩɩ Ɩbwaarɛ taa mʋ kpee sʋsʋ sasapʋ mʋ tɔ. N man nyi yɛ ɩnɩmʋ ɩ waa kanɩn nɩn kesintin, abɛɛ ansi sʋ atɔ nɩɩ o wu fɛɛ idee, amɔɔ Ɩbwaarɛ nkʋn i nyi. 
N bwii n tɔwɛ yɛ, ɔnyɩn nɩmʋ mʋ kayɩɩ‐wʋlɛ kpini nɩɩ ɩ taa kpee sʋsʋ nɩn o, mʋ kala nkʋn nɩɩ ɩ taa kpee nɩn o, n man nyi, amɔɔ Ɩbwaarɛ nkʋn i nyi. 
Amaa Ɩbwaarɛ maa taa mʋ kpee paradise mʋ, o nu asɩn mʋ nɩɩ ba mɛɛ taalɛ tɔwɛ, yɛ ɩmʋ nɩɩ nyɩmɩsa man sʋ ɔkpa yɛ ɔ tɔwɛ. 
Ɩmʋ sʋ nɩɩ n biti n kaapʋ mɛ n‐yɩɩ mʋ, kʋtɔ mʋ nɩɩ kɩ ba ɔnyɩn nɩmʋ sʋ nɩɩ n biti n kaapʋ n‐yɩɩ. N mɛɛ yɛ n kaapʋ mɛ n‐yɩɩ lɩɩ mɛ n‐yɩɩ sʋ, amɔɔ mɛ kapɔɔsɛ sʋ nkʋn. 
Halɩɩ nɩɩ n kan n kaapʋ mɛ n‐yɩɩ gbaa mʋ, n mɛɛ yɛ n waa omelensipu. Lɩɩ fɛɛ ɩmʋ nɩɩ n tɔwɛ mʋ ɩ gyɛ kesintin. Amaa n lɛɛ mɛ n‐yɩɩ lɩɩ n‐yɩɩ kɩkaapʋ tɔ, kɛ ɔkʋ ɔ man nyɛ kaala mɛ kyʋn kʋtɔ mʋ nɩɩ o wu mɛ maa waa sʋ, abɛɛ ɩmʋ nɩɩ n tɔwɛ sʋ. 
Kanan mʋ nɩɩ ɩ waa kɛ n man taa Ɩbwaarɛ kɩlɛɛkaapʋ damantɛ mʋ nɩɩ ɔ taa sa mɛ mʋ waa n‐yɩɩ alala‐sɩn sʋ mʋ, ɔ yɛgɛ nɩɩ Sɩtaanɛ sun kʋlɔ‐libi kʋ waa mɛ n‐yɩɩ tɔ fɛɛ owii, nɩɩ kii yii mɛ. Nɩɩ kʋlɔ kɩɩ nyanla mɛ, nɩɩ kii tii mɛ ɔkpa, kɛ n man nyɛ n waa mɛ n‐yɩɩ alala‐sɩn tɔ. 
N kʋlɛ Ɔnyɩrɩpɛ Yesu ɩlɩɩ ɩsa fɛɛ ɔ lɛɛ ɩmʋ lɩɩ mɛ sʋ. 
Amaa ɔ lɛɛ mɛ kɔnɔ yɛ, “Mɛ kabwaala mʋ nɩɩ n sʋ sa fʋ mʋ, ɩ kʋkwɛ fʋ. Lɩɩ fɛɛ fʋ kɩpɔɔsɛ mʋ tɔ nɩɩ mɛ ɔlʋn mʋ ɩɩ lɩɩ ifuli.” Ɩmʋ sʋ n biti n kaapʋ mɛ n‐yɩɩ kensi‐gyii tɔ gaa lɩɩ mɛ kɩpɔɔsɛ tɔ, kɛ Krisito ɔlʋn mʋ kɩ ba bun mɛ sʋ. 
Ɩmʋ sʋ lɩɩ Krisito sʋ mʋ, mi ansi i gyii lɩɩ mɛ kɩpɔɔsɛ tɔ, yɛ asebi tɔ, yɛ kayɩɩ kɩtaa tɔ, yɛ kanyanla tɔ, yɛ awʋrʋfɔ tɔ. Lɩɩ fɛɛ saŋa mʋ nɩɩ n pɔɔ mʋ, saŋa maa nɩɩ n sʋ ɔlʋn. 
N waa fɛɛ omelensipu, amaa fanɛ i nin mɛ nɩɩ n waa kanɩn. Naafɔɔ fanɛ ɩ dagaa fan maa lɛɛ kanan mʋ nɩɩ n du mʋ kaapʋ. Lɩɩ fɛɛ halɩɩ n maa man gyɛ sɛhn gbaa mʋ, bamʋ nɩɩ bɛɛ waa bamʋ n‐yɩɩ yɛ ba gyɛ “Krisito asunpu kparɛ” ba man kyɔ mɛ pii falɛ gbaa. 
Nɩɩ atɔ mʋ nɩɩ ɩɩ lɛɛ ɩɩ kaapʋ fɛɛ n gyɛ Krisito osunpu mʋ ɩ gyɛ yɛ, n nyɛ kɔkɔlɔ yɩlɛ tɔ, nɩɩ ɩ yɛgɛ nɩɩ Ɩbwaarɛ n waa ɩsɩnkaala yɛ adangana yɛ asɩn kyinkyinsa fanɛ asɛ. 
Mɩnɛ atɔ nɩɩ Krisito kʋbʋ‐kʋbʋ asansa mʋ ba nyɛ nɩɩ fanɛ bɩrɛ fan man nyɛ? Amɔɔ afulee asɩn nɩɩ n man taa ba sʋla fanɛ. Nɩɩ ɩ gyɛ kanɩn bɩrɛ fan taa mɛ alibi kɛɛ mɛ. 
Nbɩɩnbɩɩ n kʋʋla lʋwɛ n ba bɔɔ fanɛ sʋ ɩlɩɩ sasapʋ. Nɩɩ mee biti fɛɛ n maa biliŋi kasʋla sa fanɛ, lɩɩ fɛɛ man gyɛ fanɛ atɔ nɩɩ n biti, fanɛ gbaa‐gbaa nɩɩ n biti. Nɩɩ man gyɛ nbii ii biti atɔ bɛɛ yɩla bamʋ akʋʋgɛpʋ, amɔɔ akʋʋgɛpʋ ii biti bɛɛ yɩla bamʋ‐bii ana. 
Ɩmʋ sʋ n biti n kpɩnɛ kɛ n nyita mɛ kʋtɔ kʋmaa fanɛ sʋ, kɛ n taa mɛ n‐yɩɩ gbaa tii sʋ. Fɛɛ n maa sʋ kebiti lala falɛ sa fanɛ mʋ, fen biti fan nyɛ kebiti pii sa mɛ nɩn? 
N nyi fan maa kɔɔlɛ kyula fɛɛ n man biliŋi kasʋla fanɛ sʋ. Amaa akʋ bɛɛ tɔwɛ yɛ n taa mɛ kanyaasɩn ɔkpa sʋ n nyɛ fanɛ. 
Nɛnɛ sʋ? Asa mʋ nɩɩ n sun bamʋ fanɛ asɛ mʋ, n bɔla bamʋ sʋ gyii fanɛ nɩn abɛɛ? 
Mɛ ɩ kʋlɛ Titusi yɛ o kpee fanɛ asɛ, nɩɩ n sun anɛ bɩrɩsa kʋ yɛ o buu mʋ kpee. Abɛɛ fɛn fɛ nfɛɛrɛ yɛ Titusi ba pɩna fanɛ gyii nɩn? Man gyɛ nfɛɛrɛ kʋlʋn yɛ ɔkpa kʋlʋn sʋ nɩɩ anaa mʋ an naa? 
Akʋtɔ fen fɛ nfɛɛrɛ nɩn fɛɛ an tɔwɛ an lɛɛ anɛ kɔnɔ nɩn. Ayee! Man gyɛ kanɩn nɩn. Amaa an tɔwɛ nɩn fɛɛ kanan mʋ nɩɩ Krisito ii biti fɛɛ a tɔwɛ Ɩbwaarɛ asɛ mʋ. Mɛ kyɛmɩnɛ ana danbɩrasa, kʋtɔ kʋmaa mʋ nɩɩ an waa mʋ, an waa nɩn kanan mʋ nɩɩ ɩ kyɛ fanɛ tɔ. 
Ɩ sʋ mɛ kufuu fɛɛ n kan ba fanɛ asɛ mʋ, n mɛɛ yɛ n ba wu fanɛ fɛɛ kanan mʋ nɩɩ n biti fɛɛ n ba wu fanɛ mʋ, nɩɩ fanɛ gbaa fɛn mɛɛ yɛ fen wu mɛ kanan mʋ nɩɩ fen biti fan wu mɛ mʋ. Ɩ sʋ mɛ kufuu fɛɛ n biti n ba wu nluu yɛ kayɩɩ‐basa yɛ agbʋ lala yɛ kigyi‐lɛɛ yɛ asa kɩsaalɛ yɛ asa asɩn kɩtɔwɛ yɛ alala‐sɩn yɛ nbarɩgɛ tɔ mʋ, ɩ maa bʋ fanɛ tɔ. 
Ɩ sʋ mɛ kufuu fɛɛ nɩɩ n kan bwii ba fanɛ asɛ mʋ, mɛ Ɩbwaarɛ ii biti ɔ bɩya mɛ kasɛ fanɛ ansi tɔ, kɛ mɛ kɔkɔlɔ ki nyita mɛ lɩɩ asa gaalaagaa mʋ nɩɩ ba wʋla waa alibi, nɩɩ ba man kyɛɛgɛ bamʋ nfɛɛrɛ lɩɩ kayɩɩ‐wʋlɛ ineesi, yɛ kakyɩɩ yɛ kanyɩn biti kɩgyagaa, yɛ agyaga‐sɩn kʋmaa mʋ nɩɩ i tii sʋ mʋ.
