11.
Nɩɩ fan kan nyɛ kenyiita gbɛrɛɛ sa mɛ lɩɩ mɛ amelensi‐sɩn nɩmʋ sʋ mʋ, naafɔɔ ɩ bʋrɔn. Nɩɩ n nyi fɛɛ fen biti fɛn waa kanɩn. 
Krisito ayaa tɔ nɩɩ n yɩlɛ n kɔ fanɛ sʋ kɔkʋrʋkɔ falɛ. Fen du nɩn fɛɛ mʋ nɩɩ ɔ man nyi ɔnyɩn nɩɩ ɔ man sʋ ineesi ɩkʋ, nɩɩ n sʋ n sa ɔnyɩn kekili, nɩɩ ɔnyɩn maŋa ɩ gyɛ Krisito. 
Amaa n sʋ kufuu fɛɛ ɔkʋ ii biti o be nyita fanɛ nfɛɛrɛ lɩɩ Krisito mʋ nɩɩ fan taa fanɛ nkɔlɔ kpini buu mʋ mʋ, fɛɛ kanan mʋ nɩɩ kɩnansɛ taa mʋ kɔnɔ ɔkɔn pɩna Hawa mʋ. 
Lɩɩ fɛɛ ɔkʋmaa mʋ nɩɩ ɔ ba tɔwɛ asɩn lɩɩ Yesu kʋ sʋ, nɩɩ ɔ kwɩɩ yɛ Yesu mʋ nɩɩ an tɔwɛ mʋ asɩn mʋ, abɛɛ nɩɩ fan kan nyɛ kayaayu kʋ, nɩɩ kɩ kwɩɩ yɛ kayaayu mʋ nɩɩ fan daa kɩnyɛ mʋ, abɛɛ nɩɩ fan nu asɩn danbɩrasa kʋ, nɩɩ ɩ kwɩɩ lɩɩ asɩn danbɩrasa mʋ nɩɩ fan kɔɔlɛ gyii mʋ, ɩɩ kaapʋ nɩn fɛɛ fɛn mɛɛ kɛrɛ kanɩn atɔ kɩkɔɔlɛ kyula. 
Amaa n mɛɛ fɛ nfɛɛrɛ yɛ n bʋ bamʋ nɩɩ bɛɛ waa bamʋ n‐yɩɩ asunpu kparɛ mʋ kasɛ pii falɛ gbaa. 
Nɩɩ n kan n gyɛ ɔmʋ nɩɩ ɔ man nyi katɔwɛ gbaa mʋ, n sʋ kinyi lɩɩ asɩn danbɩrasa mʋ tɔ. Nɩɩ an kaapʋ fanɛ ɔkpa kʋmaa sʋ yɛ saŋa kʋmaa, nɩɩ kanɩn nɩɩ ɩ gyɛ. 
N maa tɔwɛ Ɩbwaarɛ asɩn danbɩrasa sa fanɛ mʋ, n man kɔɔlɛ fanɛ sɛhn, nɩɩ bɩya mɛ n‐yɩɩ kasɛ, kɛ ɩ nyɛ waa fanɛ asa kparɛ mʋ, n waa alibi nɩn abɛɛ? 
N taa kɩkpɩnɛ tʋʋ Krisito kʋbʋ‐kʋbʋ mʋ ɩkʋ sʋ, lɩɩ fɛɛ n kɔɔlɛ nkyɛtɔ lɩɩ bamʋ asɛ, kɛ n nyɛ ba kyɛ fanɛ tɔ. 
Saŋa mʋ nɩɩ n kyɩna n bʋ fanɛ asɛ nɩɩ i tiri mɛ mʋ, n man yɛgɛ nɩɩ ɩ waa kasʋla fanɛ sʋ. Lɩɩ fɛɛ mɛ bɩrɩsa ana mʋ nɩɩ ba lɩɩ Masedoniya mʋ ba bɩya mɛ atɔ mʋ nɩɩ ii tiri mɛ. N kɩtaa mɛ n‐yɩɩ ɔkpa kʋmaa nɩmʋ sʋ kanan mʋ nɩɩ n man ba waa kasʋla fanɛ sʋ, nɩɩ n biti n kyaga sʋ waa kanɩn. 
Fɛɛ Krisito kesintin mʋ maa bʋ mɛ tɔ falɛ mʋ, n tɔwɛ yɛ ɔkʋ man bʋ Akiya kɩbaafʋn nɩmʋ sʋ, nɩɩ ɔkʋ mɛɛ taalɛ tii mɛ n‐yɩɩ kɩkaapʋ nɩmʋ sʋ. 
N maa tɔwɛ falɛ mʋ, ɩ kaapʋ nɩn fɛɛ n mee biti fanɛ asɩn abɛɛ? Ayee! Ɩbwaarɛ nyi fɛɛ n biti fanɛ asɩn. 
Nɩɩ ɩmʋ nɩɩ n waa mʋ n biti n kyaga sʋ n waa, kɛ ɩ nyɛ tii bamʋ nɩɩ bee biti ɩkpa ba kʋsaa bamʋ n‐yɩɩ mʋ ɔkpa. Lɩɩ fɛɛ bɛɛ kaapʋ yɛ kanan mʋ nɩɩ bɛɛ waa kusun mʋ, anaa bamʋ lɛɛ gyɛ kʋlʋn nɩn. 
Kanɩn asa maŋa ba gyɛ antɩŋɛsa asunpu mʋ nɩɩ bɛɛ pɩna asa lɩɩ kanan mʋ nɩɩ bee sun kusun mʋ sʋ, nɩɩ ba baala bamʋ daa bɛɛ waa fɛɛ ba gyɛ Krisito asunpu. 
Ɩnɩmʋ ɩ man gyɛ asɩn kyinkyinsa, lɩɩ fɛɛ Sɩtaanɛ gbaa‐gbaa ɩɩ taalɛ baala mʋ daa ɛɛ ŋalɩgɛ fɛɛ Ɩbwaarɛ kabɔɔ. 
Kanɩn gbaa nɩɩ ɩ man gyɛ asɩn kyinkyinsa fɛɛ Sɩtaanɛ ayaafɔlɛ gbaa ba baala bamʋ daa, nɩɩ ba waa bamʋ n‐yɩɩ fɛɛ nbɔɔ mʋ nɩɩ bamʋ ɩkpa ii kyiigi mʋ. Laalaalʋwɛ mʋ bee biti ba nyɛ bamʋ asun libi kʋkɔka. 
N pɩla tɔ n tɔwɛ yɛ, ɔkʋ ɔ man fɛ nfɛɛrɛ fɛɛ n gyɛ omelensipu. Amaa nɩɩ fɛn kan fɛ nfɛɛrɛ kanɩn mʋ, ɩmʋ bɩrɛ fɛn taa mɛ kanan mʋ nɩɩ fen biti fan taa omelensipu mʋ, kɛ n nyɛ kaapʋ mɛ n‐yɩɩ gbɛrɛɛ. 
Ɩmʋ nɩɩ n tɔwɛ falɛ, man gyɛ kʋtɔ mʋ nɩɩ Ɔnyɩrɩpɛ mʋ ii biti fɛɛ n tɔwɛ, amaa n‐yɩɩ kɩkaapʋ asɩn nɩmʋ tɔ mʋ, n tɔwɛ nɩn fɛɛ omelensipu. 
Fɛɛ asa gaalaagaa ba maa kaapʋ bamʋ n‐yɩɩ fɛɛ kanan mʋ nɩɩ dulinyaa abi bɛɛ waa mʋ, ngbaa mɛ n biti n kaapʋ mɛ n‐yɩɩ. 
Fan maa yɛ fan sʋ kanyaasɩn gaa sʋ mʋ, fɛn nyɛ kenyiita fen nu amelensipu asɛ. 
Kesintin tɔ bamʋ nɩɩ bɛɛ waa fanɛ fɛɛ anyɛ, abɛɛ ba sʋ fanɛ bee biliŋi sʋ, abɛɛ bee gyii fanɛ, abɛɛ bɛɛ kɩɩ fanɛ kɩyan, abɛɛ bɛɛ kpaalɛ fanɛ ɩtan gbaa mʋ, fɛn taalɛ fɛn nyɛ kenyiita fen sa bamʋ. 
Ɩ sʋ mɛ ipeeli nɩɩ n kɔɔlɛ kyula fɛɛ, atɔ mʋ nɩɩ bɛɛ waa falɛ bɩrɛ, an gyɛ apɔɔsɩpʋ ɩmʋ tɔ gaa. Amaa nɩɩ ɔkʋ kan ɔ man sʋ kufuu fɛɛ ee biti ɔ kaapʋ mʋ n‐yɩɩ kʋtɔ kʋ sʋ mʋ, n maa tɔwɛ fɛɛ omelensipu falɛ mʋ, ngbaa mɛ n mɛɛ yɛ n waa kufuu fɛɛ n kaapʋ mɛ n‐yɩɩ ɩmʋ sʋ. 
Asa maŋa ba gyɛ Hibriwu abi nɩn abɛɛ? Kanɩn nɩɩ ngbaa mɛ n gyɛ. Ba gyɛ Ɩsɩrayɩ abi nɩn abɛɛ? Kanɩn nɩɩ ngbaa mɛ n gyɛ. Ba gyɛ Abraham anaana‐bii nɩn abɛɛ? Kanɩn nɩɩ ngbaa mɛ n gyɛ. 
Ba gyɛ Krisito ayaafɔlɛ nɩn abɛɛ? N nyi mɛ katɔwɛ nɩmʋ maa duli ɩbʋn bɩrɛ, amaa n gyɛ kɩyaafɔlɛ kparɛ n kyɔ bamʋ. N sun kusun ɔlʋn sʋ n kyɔ bamʋ, nɩɩ ba tii mɛ ibu saŋa kʋmaa nɩmʋ ɩ kyɔ bamʋ, ba taa awulibi tɩŋɛ mɛ libi‐libi ɩ kyɔ bamʋ, nɩɩ n gyaŋa lewu saŋa kʋkyɔ. 
Gyiwu abi ba taa awulibi tɩŋɛ mɛ adɩsa yɛ akpanɔ kanɩn ɩlɩɩ inuu. 
Nɩɩ Roma abi gbaa ba taa abɩlɩtɔ kpekpesen da mɛ ɩlɩɩ ɩsa, nɩɩ saŋa kʋ asa ba twɩɩ mɛ abuu, nɩɩ ɔkʋlɩ lala taa mɛ naa watɛ ɩlɩɩ ɩsa, nɩɩ kakaakʋ n dɛ ɔpʋ sʋ kalayɩ tɩyɛ kyɔwɛ. 
Mɛ kɩnatɛ tɔ mʋ, n gyaŋa kinsi‐payɩ gaalaagaa. N nyɛ kinsi‐payɩ lɩɩ ɩbʋn kɩbɔla sʋ, yɛ ayu mʋ nɩɩ bɛɛ nyan bɛɛ kɔɔlɛ atɔ abaa tɔ, yɛ mɛ asa Gyiwu abi, yɛ bamʋ nɩɩ ba man gyɛ Gyiwu abi mʋ abaa tɔ. Nɩɩ n nyɛ kinsi‐payɩ ndɛ lala tɔ, yɛ ɩsʋwʋlɛ sʋ, yɛ ɔpʋ sʋ. Nɩɩ n nyɛ kinsi‐payɩ lɩɩ bamʋ nɩɩ bɛɛ pɩna yɛ ba gyɛ anɛ Krisito abuupu mʋ asɛ. 
N sun kusun kenken nɩɩ n kpɩnɛ, halɩɩ nkɛ damantɛ n mɛɛ nyɛ nsɩɩrɛ n dɛ. Akʋn yɛ bʋlɩfɔ nɩɩ n naa, nɩɩ saŋa gaalaagaa sɛhn man loo mɛ kɔnɔ tɔ. Nɩɩ n pan atɔ bunsa nɩɩ n dɛ awɔɔ. 
Ɩnɩmʋ kpini ɩ naa dɛ, ɩmʋ nɩɩ ɩ nyan mɛ kakɛ kʋmaa mʋ ɩ gyɛ yɛ n fɛ Krisito kʋbʋ‐kʋbʋ mʋ sʋ. 
Anɩmʋ kɩkɔɔlɛ‐gyii ɩɩ kɔɔgɛ sʋ, nɩɩ n mee su n sa mʋ? Anɩmʋ nɩɩ ba pɩna mʋ ba waa alibi tɔ, nɩɩ mɛ ɔtɔ tɔ mɛɛ fɔyɩ mɛ? 
Nɩɩ n kan i tiri fɛɛ n kaapʋ mɛ n‐yɩɩ mʋ, n biti n kaapʋ mɛ n‐yɩɩ lɩɩ mɛ kɩpɔɔsɛ sʋ nɩn. 
Anɛ Ɩbwaarɛ mʋ nɩɩ ɔ gyɛ anɛ‐Nyɩrɩpɛ Yesu Mʋ‐Sɛ, nɩɩ ayɩn‐bi ɩ gyɛ mʋ lɛɛ kakpaa mʋ, i nyi yɛ ɩmʋ nɩɩ n tɔwɛ falɛ man gyɛ antɩŋɛsa. 
Saŋa mʋ nɩɩ n bʋ Damasiko kadɛ tɔ mʋ, Owura Aretasi mʋ ɔsʋwʋlɛ ɔkɩɩsʋpʋ mʋ lɛɛ akunpu yɛ ba kun nbunɔ mʋ nɩɩ ii loo Damasiko kadɛ mʋ nɩɩ kɩbandɩyɛ kɩ kyaabɔɔ kɩmʋ mʋ, kɛ ba nyɛ kɩtaa mɛ. 
Amaa akʋ ba taa mɛ waa kɩlantan tɔ, bʋlɛ kɩbandɩyɛ mʋ ɔbɔ tɔ, kpɩlɩgaa mɛ yii, nɩɩ n nyɛ sɩlɛ lɩɩ mʋ abaa tɔ.
