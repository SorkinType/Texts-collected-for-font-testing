13.
Nɩɩ n kan n taalɛ n tɔwɛ asa abɛɛ Ɩbwaarɛ nbɔɔ ndɛ tɔwɛsa, nɩɩ n man sʋ kebiti sa mɛ bɩrɩsa mʋ, i du nɩn fɛɛ agongon mʋ nɩɩ kɩ sʋ ɔlanba mʋ, abɛɛ kidulonbi mʋ nɩɩ kɩ sʋ ɩlawʋ mʋ. 
Nɩɩ n kan n sʋ kakɛɛ mʋ nɩɩ Ɩbwaarɛ ɩɩ bɔla mɛ sʋ ɛɛ tɔwɛ, nɩɩ n nyi Ɩbwaarɛ asɩn baalasa kpini, nɩɩ n sʋ kinyi, nɩɩ n sʋ kɩkɔɔlɛ‐gyii mʋ nɩɩ n taalɛ kʋsaa abɩɩ gbaa, nɩɩ n man sʋ kebiti bɩrɛ, n man gyɛ sɛhn. 
Nɩɩ n kan taa mɛ kʋtɔ kʋmaa mʋ nɩɩ n sʋ kɛɛ atiripu, nɩɩ n kan taa mɛ n‐yɩɩ sa yɛ ba tuwa gbaa, nɩɩ n man sʋ kebiti mʋ, n mɛɛ yɛ n nyɛ tɔnɔ kʋ. 
Ɔkʋmaa mʋ nɩɩ ɔ sʋ kebiti mʋ, ɔ sʋ kenyiita, nɩɩ ɛɛ waa kɩdanbɩrasa, nɩɩ ɔ man sʋ kayɩɩ‐basa, nɩɩ ɔ mɛɛ waa alala‐sɩn, nɩɩ ɔ mɛɛ kaapʋ mʋ n‐yɩɩ. 
Ɔ mɛɛ waa kɩmʋ nɩɩ kɩ man dagaa, nɩɩ ɔ mɛɛ kɩɩ ee biti mʋ nkʋn lɛɛ, nɩɩ ɔ mɛɛ nyɛ agbʋ bilen‐bilen, nɩɩ ɔ mɛɛ taa ɔkʋ asɩn ɛɛ waa mʋ ɔtɔ tɔ. 
Ɔkʋmaa mʋ nɩɩ ɔ sʋ kebiti mʋ, alibi kɩwaa ɩ mɛɛ waa mʋ ɔkɔn, amaa mu ansi i gyii kesintin sʋ. 
Saŋa kʋmaa ɛɛ yɩlɛ tɔ naa fʋʋ ɔkaa, nɩɩ ɔ sʋ kɩkɔɔlɛ‐gyii, yɛ kenyiita, yɛ tamaa kʋtɔ kʋmaa nɩmʋ tɔ. 
Kebiti kɩ man sʋ ɔkaa. Amaa Ɩbwaarɛ kɩbɔla sʋ ɛɛ tɔwɛ, yɛ katɔwɛ mʋ nɩɩ be mee nu kasɛ, yɛ kinyi, ɩmʋ kpini ii biti ɩ ba fʋʋ ɔkaa. 
Lɩɩ fɛɛ kʋtɔ mʋ nɩɩ Ɩbwaarɛ yɛgɛ nɩɩ an nyi, yɛ ɩmʋ nɩɩ ɛɛ bɔla anɛ sʋ ɛɛ tɔwɛ mʋ ɩ gyɛ gbɛrɛɛ nɩn. 
Amaa nɩɩ kʋtɔ mʋ nɩɩ ki du kyɛɛkyɛɛ lʋwɛ kan ba mʋ, kɩmʋ nɩɩ kɩ man du kyɛɛkyɛɛ mʋ ii biti ɩ kyʋn. 
Saŋa mʋ nɩɩ n gyɛ kebii mʋ, n tɔwɛ fɛɛ kebii, nɩɩ n fɛ nfɛɛrɛ fɛɛ kebii, nɩɩ n gyii ɩmɔɔrɛ fɛɛ kebii. Amaa n maa ba dan mʋ n yɛgɛ abii‐sɩn mʋ. 
Nbɩɩnbɩɩ bɩrɛ, an wu atɔ nɩn fuu anɛ ansi sʋ, fɛɛ a maa kɩɩ digi mʋ nɩɩ kɩ man kɩɩ tɔ mʋ tɔ, amaa saŋa kʋ an biti a wu kʋtɔ kʋmaa anɛ ansi sʋ danbɩrasa. Nbɩɩnbɩɩ asɩn mʋ nɩɩ n nyi mʋ ɩ gyɛ gbɛrɛɛ, amaa saŋa kʋ n biti n bɩɩ kʋtɔ kʋmaa tɔ lʋwɛ, fɛɛ kanan mʋ nɩɩ Ɩbwaarɛ nyi mɛ mʋ. 
Fɛɛ kanan mʋ nɩɩ i du mʋ, atɔ asá anɩmʋ ɩ bʋ tɔ. Ɩmʋ ɩ gyɛ kɩkɔɔlɛ‐gyii, yɛ tamaa yɛ kebiti. Amaa kebiti ɩ kyɔ ɩmʋ kpini.
