1 Nsyɔ̃.
2.
Ń wɛ̃́npì, mé sɛ́bɛ́y mɛ̀nɛ n tɛkɩ n mɔ yé tyɩ́, nɛ̂-á kɩ yé kɔ̃ yè káh tyípékèlɛ n pɩ. Ǹtɛ kàh se pɔ pɩ ǹnɩ̂ yɔ́ tyípékè pɩ, wrɔ́ tɔ̃́-ò mɛ á tyɩ́ Sú Liyel yah sɔkɔ, nɛ̂-á Yesu Kristalɛ, ǹmɔ nɛ̂-á mɛ tómm. 
Ǹmɔ ye ǹ gblɔ̌y kɔ̃ sárkálɛ nɛ̂-á kɩ Liyel kɔ̃ à kɩ ápi tyípêl yɔ̃ mɔ. Tɛ́ à náh ǹ gblɔ̌y kɔ̃ ápi ó tyípêl yɔ̃̀nmɔnɔ yĩnnɛ dɛ́, kèkõyṍ névye pól tɔ tyípêl yɔ̃̀nmɔnɔ yĩn ye à ǹ gblɔ̌y kɔ̃. 
Áyâh Liyel tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ, képah sɔkɔ ye á kɩ pɩ̃ nɛ áyáh ǹnɛ n pnɛ. 
Névi nɛ̂-à nɛ páh ǹnɛ n pnɛ, tɛ́ tɛ̃̀ náh mɛ n kɔ ǹ tyi kõ̀nsàhnɔ́ wã̀l yõ, mɩ̀nyo ye tɛ̃̀nɛ. Gbɩ náh mɛ tɛ̃̀ sɔkɔ. 
Ǹtɛ nɛ̂ wɔ-à n kɔ ǹ yóré yõ, ǹ tyɩ́ mɛ nɔ tɛ̃̀ wɔ tyɩ́ kègbɩ tɛ̃̀ yõ. Képah sɔkɔ ye ǎ kélɛ n pnɛ nɛ áyáh mɛ ǹ tyɩ́ yõ. 
Névi nɛ̂-à nɛ páh mɛ ǹ tyɩ́ yõ, Yesu-á kɔ dùkù nɛ̂nɛ, tɛ̃̀ tɔ yɩ̃nɛ à nɛ n kɔ sɛ̃́ntɛ̃̀nɛ. 
Ń nɛ́nɔ́nɔ́, tir kõ̀nsàhnɔ́ késã́n náh kélɛ mé n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ dɛ́, tir kõ̀nsàhnɔ́ kédyâh ye kélɛ, nɛ̂-á yo sɔkɔ yé tyɩ́ tyah gbĩ́ dùkùlɛ. Yáh wɛy nɛ̂ noh sɔkɔ tɛ, képah ye tir kõ̀nsàhnɔ́ kédyâh mɛ̀nɛ. 
Ǹtɛ fɩ̃́ yɔ́ tĩ̀nnɛ, tir kõ̀nsàhnɔ́ késã́n ye mó tɔ̃ kélɛ mé n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. Yesu ànɛ̂ yépi pɩ dùkù mɛ képah gbɩlɛ n wɛ̃kɩ. Yé náh wɛ di, fɛ̃́ntũ̂ mɛ n sɔkɔ, kyòo yèsyɩ̀kɩ̀ mó mɛ pɔ tɛ. 
Névi nɛ̂-à nɛ páh mɛ kyòo sɔkɔ, tɛ́ mó tɔ̃ mɛ ǹ yṹnpi yɔ́ dyɔlɛ n yɔ, tɛ̃̀ mɛ sɛ̃́ fɛ̃́ntũ̂ sɔkɔ. 
Névi nɛ̂ yṹnpi tyɩ́-à nɔ ǹ tyɩ́, tɛ̃̀ wɔ mɛ kyòo sɔkɔ. Ǹnɩ̂ yɔ́ ńtɛ̃ náh n de n pi tyípékè sɔkɔ ǹmɔ gbõ̀ yõ. 
Ǹtɛ nɛ̂-à ǹ yṹnpi dyɔlɛ n yɔ, tɛ̃̀ mɛ fɛ̃́ntũ̂ sɔkɔ. Tɛ́ wǎh mɛ n kɔ fɛ̃́ntũ̂ sɔkɔ sɛ̃́, fɛ̃́ntũ̂ tɛ̃̀ tyɩ́, ǹ yɩ̃́ kɩ wɔ, à náh ǹ yúkú-ńsah pɩ̃ n pi. 
Ń wɛ̃́npì, Yesu yĩn gbõ̀ yõ sɔkɔ ye yé tyípêl yɔ̃ mɔ yé kɔ̃. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. 
Súbí, nɛ̂-á sõ̀ mɛ tyi pól tyah gbĩ́ dùkùlɛ, yé ǹnɛ n pnɛ. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. Kètɛ̃̀npì, yé tɛ̃ wɛ pèkè pɩ́-ò tyɩ́, nɛ̂-á sétã̀n yõ̀tɛ̃̀nɛ. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. 
Yépi nónó-á wɛ̃́npìlɛ, nɛ̂-á Sú Liyellɛ, yé ǹnɛ n pnɛ. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. Yépi nónó-á súbílɛ, nɛ̂-á sõ̀ mɛ tyi pól tyah gbĩ́ dùkùlɛ, yé ǹnɛ n pnɛ. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. Kètɛ̃̀npì, tǎhkɩ̀ mɛ yé sɔkɔ, ànɛ̂ Liyel wɛy mɛ n tɛ̃ yé nnɔ sɔkɔ, tɛ́ yé tɛ̃ wɛ pèkè pɩ́-ò tyɩ́, nɛ̂-á sétã̀n yõ̀tɛ̃̀nɛ. Képah ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. 
Yé káh yé yĩ́ wil kèkõyṍ tyɩ́, képah náh pɩ ké sɔkɔ kwâh yɔ́ ńtɛ̃ tyɩ́. Névi nɛ̂-à ǹ yĩ́ wil kèkõyṍ tyɩ́, Sú Liyel tyɩ́ náh fɛ̃ n nɔ n pi tɛ̃̀ tyɩ́. 
Yé yah, tyi nónó-á mɛ kèkõyṍ sɔkɔ, sɔ̃́-á gblɔ̌y dyɔ yĩ́ĩ́ tyi pɩ́nɔ́ dékèlɛ, ànɛ̂ yĩ́wílkìlɛ, ànɛ̂ gblɔ̌y wɛ̃́kɩ́nɔ́lɛ à yuku tahrɩ tyɩ́ tĩ̀nnɛ, sépi sɔkɔ yɔ́ ńtɛ̃ náh n wil Sú Liyel tyɩ́. Kèkõyṍ sɔkɔ ye sè n wil. 
Tɛ́ kèkõyṍ ànɛ̂ ké yĩ́wílkì kɩ pɔ n tyɛ n pi. Ǹtɛ névi nɛ̂-à Liyel dyɔ yĩ́ĩ́ tyilɛ n pɩ, tɛ̃̀ wɔ kɩ tɛ̃ tyɛ́-ńkɛ̃̂nɛ. 
Ń wɛ̃́npì, kèkõyṍ tyɛ gbĩ́ ye mɛ nɔ n tɛ. Yé kè noh sɔkɔ tɛ nɛ Krista sépɛ-á kɩ n pɔ n pi. Núkúnúkú, Krista sépɛ́bɩ́ yísyɔ́ mɛ wil mɔ pí. Képah n wɛ̃kɩ nɛ kèkõyṍ tyɛ gbĩ́ sɔkɔ-á álɛ. 
Ápi tĩ̀nkì sɔkɔ ye sépɛ́bɩ́ nɔ́pi wil. Ǹtɛ kègbɩ yõ, pè náh sõ̀ mɛ núkúlɛ ápilɛ. Pàh sõ̀ mɛ núkúlɛ álɛ, pè náh náhkɩ n gbɛ n wil n pi ápi sɔkɔ. Ǹtɛ kè ye sõ̀ yɩ̃nɛ kè pɩ sɛ̃́, nɛ̂-á kɩ kè wɛ̃kɩ nɛ pé sɔkɔ nɛy yɔ́ ńtɛ̃ náh sõ̀ mɛ núkúlɛ ápilɛ kègbɩ yõ. 
Ǹtɛ yépi wɔlɛ, Liyel Mirki mɛ yé sɔkɔ Krista-á nɛ̂nɛ yé kɔ̃. Képah ye yé pól kɔ̃ yé tyilɛ n pnɛ. 
Kè náh mɛ á kɩ nɛ yé kósyì gbɩlɛ n pnɛ ń kɔ̃ mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ dɛ́, yáh kélɛ n pnɛ, tɛ́ pɩ̃ nɛ mɩ̀nɩ̀ yɔ́ ńtɛ̃ náh n wil ké sɔkɔ, képah ye ń kɔ̃ mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. 
Névi nɛ̂-à nɛ Yesu náh Kristalɛ, nɛ̂-á Liyel nɛy yáhntɔkɔnɔlɛ, mɩ̀nyo yɔ́ ńtɛ̃ náh kah mɔ tɛ̃̀ yõ. Krista sépɛ ye tɛ̃̀nɛ. Sú Liyel ànɛ̂ ǹ Pídĩ́ Krista ye tɛ̃̀ yahle. 
Yé yah, névi nɛ̂-à Liyel Pídĩ́ yahle, Sú Liyel náh mɛ tɛ̃̀nɛ. Tɛ́ nɛ̂-à Liyel Pídĩ́ fɛ̃ syi, Sú Liyel mɛ tɛ̃̀ wɔlɛ. 
Yépi tɛ̃̀ wɔlɛ, yáh wɛy nɛ̂ noh tyah gbĩ́ dùkùlɛ, yé yɩ̃nɛ yè kè koh n sah yé nnɔ sɔkɔ. Yáh wɛy nɛ̂ noh tyah gbĩ́ dùkùlɛ, yâh fɛ̃ kè koh sah yé nnɔ sɔkɔ, yépi tɔ wrɔ́ kɩ mɛ kwrɔ́nmɔnɔlɛ Pídĩ́nɛ, ànɛ̂ Súlɛ. 
Tɛ́ Pídĩ́-á ǹ vyãh kɔ̃ nɛ̂ tyɩ́ sɔkɔ, tyɛ́-ńkɛ̃̂ min ye. 
Nónó-á n yah n kɔ̃ yé yõ yɩ́kɩ́-ńsahlɛ, pépi tyɩ́ sɔkɔ ye mé sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́. 
Yépi tɛ̃̀ wɔlɛ, Yesu-á Liyel Mirki nɛ̂nɛ yé kɔ̃, kè mɛ tɛ̃̀nɔ́lɛ yé sɔkɔ. Képah ye, kwɔ́-ò yétõ̂ náh tɔ̃ mɛ yé tyɩ́. Liyel Mirki mɛ yélɛ n kwɔ tyi póllɛ. Tɛ́ ǹ kwɔ́nɔ́nɔ póllɛ mó tɔ̃, gblɔ ye sélɛ, mɩ̀nɩ̀ náh mɛ sé sɔkɔ. Képah ye, yé nɛ mɛ kwrɔ́nmɔnɔlɛ Kristalɛ, sɔ̃́-á Liyel Mirki yé kwɔ kélɛ. 
Núkúnúkú ńtɛ̃nɛ ń wɛ̃́npì, yé yɛ yé wrɔ́ nɛ mɛ kwrɔ́nmɔntɛ̃̀nɔ́lɛ Kristalɛ. Képah sɔkɔ, wàh sõ nɛ̂nɛ pɔ tíkí náh á tɛ̃ n pi, ànɛ̂ sen náh á kɔkɔ n sah n pi ǹ yah sɔkɔ lésõlɛ. 
Yé se pɩ̃ tɛ nɛ wǎh mɛ tómm tɛ́. Tɛ́ yé yɩ̃nɛ yè kè pɩ̃ tɔlɛ nɛ, névi nɛ̂-à n kɔ tómm, Liyel pi-á tɛ̃̀nɛ.
