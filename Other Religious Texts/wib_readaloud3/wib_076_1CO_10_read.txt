Korɛ̃nti tãn.
10.
Ń yṹnpyé, nɛ̂-á á sútóbé yĩni tɛ̃ Moyisi gbĩ sɔkɔ, mé mɛ ké tyɩ́ nɛ yé yé dyɔ sah nɛ n kwnɛ kélɛ. Pé pól sõ̀ mɛ n kɔ ńyĩ́rkì fob sõ̀, tɛ́ de kah pnɛ yɔ́nsyã̂h sɔkɔ pé tɔ̂ɔ́. 
Pé pól de sɔkɔ ńyĩ́rkì fob sɔkɔ ànɛ̂ pnɛ yɔ́nsyã̂h sɔkɔ wèlnwìlkìnɔ́ tyɩ́ kɔ̃lɛ, à pɩ Moyisi sõ̀ tɛ̃́púlɛ. 
Pé pól yõke núkú yõ wɛ̃ tyɩ́, nɛ̂-á wil Liyel tyɩ́, 
tɛ́ mɛ wɔ wɔ ni núkú wɛ wɔ, nɛ̂-á wil Liyel tyɩ́. Tɛ́dáhkɩ̀ yɔ́ sɔkɔ ni ye sõ̀ kélɛ. Kénɛ tɛ́dáhkɩ̀lɛ sõ̀, Krista ye nɛ̂-á mɛ n yuku pélɛ. 
Ǹtɛ pé kan náh sõ̀ pɩ nɛ́gblɔ́lɛ Liyel yah. Kǎh pɩ sɛ̃́, pè ku kwɛ gbɛ̃́nnɩ sɔkɔ. 
Kénɛ tyi ye pɩ, à ápi yɩ̃́ yɩkɩ, nɛ̂-á kɩ á kɔ̃ á yĩ́ náh tɔ̃ n wil n pi tyítúkúnì pɩ́nɔ́ tyɩ́ pépi tyɩ́ kɔ̃lɛ. 
Yé káh pɩ sãh fálpúlɛ pépi túkù tyɩ́ kɔ̃lɛ, pépi nónó tyɩ́-á yónɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Tɔ́wû ye kɔ̃ tɛ̃ à n yõ, tɛ́ n wɔ, képah náh tɛ́ yuku à syáhlɛ n syah.» 
Á ye káh á gblɔ̌y pɩ prɔ̀npɩbɩlɛ, sɔ̃́-á lésõ pé túkù kélɛ n pɩ, tɛ́ nɛ́pĩ̂ kèyõ gbɔ̃núkú kyɛgbãm (23 000) mɛ ku pé sɔkɔ gbĩ̀yĩ̀kì núkúlɛ. 
Á ye káh Tɛ̃̀ ǹgbɛ̃nɛ n mɔ n yah, sɔ̃́-á lésõ pé túkù pɩ kélɛ, tɛ́ vyeh mɛ pè kwãkɩ ko. 
Yé káh Liyel sèyòkèlɛ n yo sɔ̃́-á lésõ pé túkù kélɛ n pɩ, tɛ́ yĩ̂nyõ tõ̀npɩ nɛ̂-á nɛ́kókèlɛ n mɔ, ǹmɔ mɛ pɔ pé duku to. 
Tyi nɔ́pi ye pè yĩni tɛ̃, à névye yĩ́ yɩkɩ. Ápi nónó-á mɛ kèkõyṍ tyɛ́ntɛnɔ gbĩ́ mɛ sɔkɔ yɔ̀, ápi yĩn ye pè tyi nɔ́pi wãrkɩ, à gbáhdyâh di sah á tyɩ́. 
Képah ye, nɛ̂-à mɛ n sõ nɛ páh mɛ wɛ su yĩn pallɛ, tɛ̃̀ kyɔ̀mɩ̀nɔ́lɛ n pɩ, à káh pɔ yɔ̃ syɩkɩ. 
Tɔ̃̀nmɔnɔ nónó-á yé yĩni tɛ̃ yah, sɛ̃́nknɔ ye n nɔ névi yõ̀tàhsyilɛ. Ǹtɛ Liyel n kɔ ǹ tyíyónɔ́ sõ̀. À náh n yɛ tɔ̃̀nmɔnɔ yɔ́ ńtɛ̃ kɔ̃ kè kǎn pɩ n pi yélɛ. Tɛ́ tɔ̃̀nmɔnɔ-à yé wɛ, à kɩ tɛ̃́nwɛnɔlɛ yé kɔ̃ yé kɩ kè fwo, tɛ́ kɩ yé wɛ̃kɩ syɔ̃̀-ǹsahlɛ, yáh nɛ wɛ n wil n pi ké sɔkɔ sɔ̃́. 
Ń yṹnpyé pèplɔ, yé yé náh vi tã̀n tyi kɔ̃ pnɛ́pnɛ́. 
Ńmɔ mɛ yélɛ n mɔ n kõ yɩ̃ pɩ̃́pú sɔkɔ. Képah ye, tyi nónó-á mé n yo yé tyɩ́, yé sõ n yah sé tyɩ́ yõ yé gblɔ̌y. 
Liyel yõke yõ gbĩ́nɛ, áyáh ńkɛ̃́nɛ wɔ wɔ kwâh nɛ̂ wɔ á wɛ́kɩ́ntɛnɔ náh Liyel tyɩ́, képah kélɛ n wɛ̃kɩ nɛ áyáh pɩ núkúlɛ Yesu Kristalɛ ǹ tṍ kwɛ́nɔ́ gbõ̀ yõ. Búrn nɛ̂ tɔ-á ńkɛ̃́nɛ à n kwɔkɩ n dibi n tah wɛ̃ tyɩ́, ké tɔ álɛ n pɩ núkúlɛ Yesu Krista wil gbɛ̃́nsêlɛ. 
Búrn núkú ó ye mɛ. Tɛ́ ápi pípí nɔ́pi pól mɛ kénɛ búrn núkú mɛ̀ ólɛ n tah. Képah ye, kè mɛ á kɩ nɛ á pól sɔ̃́ pɩ wil gbɛ̃́nsê núkúlɛ. 
Tir nɛ̂-á mɛ n pɩ Yisrayel no nɩyṍ, yé yé fɔkɔ mɔ n yah képah tyɩ́ yõ kɛ̀. Pé sárká súkɛ́nkɔ̃-ńsah mɛ Liyel tyɩ́ yõ. Képah ye, yípɛ̂y nónó-á pè n ko ké yõ sɔkɔ, pé nónó-á sé kyɔlɛ n kɔ, pè mɛ núkúlɛ Liyellɛ. 
Képah se n wɛ̃kɩ nɛ máh nɛ yõre-á mɛ tã̀n ànɛ̂ sãh fálnɔ́ kyɔ tyɩ́? 
Ɔ̃́ɔ̃ dɛ́, mé nɛ sãh fálpú-á sãhlɔ nónónɛ n fal, sétã̀n tyɩ́-á pè n fal sélɛ, Liyel tyɩ́ náh. Mé náh mɛ ké tyɩ́ nɛ yé pɩ núkúlɛ sétã̀nnɛ gbógbó. 
Yé náh fɛ̃ Tɛ̃̀ ǹgbɛ̃ wóh sɔkɔ wɔ wɔ kwâh wɔ n pi, tɛ́ tɔ̃ yé yah vi n wɔ n pi sétã̀n tyɩ́ sɔkɔ. Yé náh fɛ̃ nɛ n yõ n pi Tɛ̃̀ ǹgbɛ̃ tyɩ́, tɛ́ tɔ̃ yé yah vi kyɩ n yõ n pi sétã̀n tyɩ́. 
Tĩ̂, Tɛ̃̀ ǹgbɛ̃ nɛ̂-á nɛ á nɛ n mɛ péwɔ ó tyɩ́ yõ, ǹmɔ vyãh sõ̀ se á n yah n kɔ̃ nóh-ńsahlɛ? Ápi tǎhkɩ̀ se kal ǹ tyɩ́lɛ? 
Kè mɛ yónɔ́lɛ nɛ ńkɛ̃́: «Tyi pól pɩ́nɔ́ gbàh mɛ dɛ́nmɔnɔlɛ.» Ǹtɛ yõre náh n wɛ sé túkù sɔkɔ. «Tyi pól pɩ́nɔ́ gbàh mɛ dɛ́nmɔnɔlɛ.» Ǹtɛ sé túkù pɩ́nɔ́ náh yah yúkúnɔ́lɛ névilɛ n kɔ̃. 
Ǹnɩ̂ yɔ́ ńtɛ̃ káh ǹ gblɔ̌y pɛ́kɩ́-ńsah ólɛ n yah n kɔ̃ dɛ́, yé no tɛ́lɔ́ pɛ́kɩ́-ńtã̂nnɛ n yah n kɔ̃ n kal. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Tɛ̃̀ ǹgbɛ̃ tyɩ́ ye sétáh ànɛ̂ ké sɔkɔ yíyìn póllɛ.» Képah ye, yî nónó pól-á fyɔ̀ mɛ yãbrɩ sɔkɔ yòh sɔkɔ, yé kɩ fɛ̃ sé póllɛ n yõ. Kè náh yɩ̃nɛ kè nɛ n mɛ yṍtɔ́kɔ́bɩ́lɛ yé tyɩ́, yé kɔ̃ yè píkínyahnɔ yɔ́ ńtɛ̃ pɩ sé wìl-ǹtã̀n tyɩ́ sɔkɔ. 
Yesu yõ sɛ̃́-ńkɛ̃̂ névi yɔ́-à ǹ ye yõke yṍnɔ́lɛ, tɛ́ á fɛ̃ syi kyɩ, wàh yé nónó tɔkɔ pɔ, sélɛ n yõ fwòó. Kè náh yɩ̃nɛ kè pɩ yṍtɔ́kɔ́bɩ́lɛ á tyɩ́ ǹ kɔ̃ ǹ pɔ píkínyahnɔ yɔ́lɛ n pɩ sé wìl-ǹtã̀n tyɩ́ sɔkɔ. 
Ǹtɛ ǹnɩ̂ yɔ́-à yo á tyɩ́ nɛ ńkɛ̃́, kɔ̀ mɛ̀nɛ, sãh fálnɔ́ kɔ̀-á kélɛ, tɛ̃̀ wɔ yĩnnɛ ànɛ̂ ké tyɩ́ sõnɔ yṍtɔ́kɔ́bɩ́ yĩnnɛ, káh kè kɔ. 
Sõnɔ yṍtɔ́kɔ́bɩ́ nɛ̂ tyɩ́-á mé yo ásõ̂ sɔkɔ yɔ̀, áwɔ tyɩ́ sɔkɔ náh ńnɛ dɛ́, nɛ̂-á yo ǹ syah kélɛ, tɛ̃̀ wɔ tyɩ́ sɔkɔ ye ńnɛ. Ǹtɛ ǹnɩ̂ kɩ fɛ̃ nɛ kwâh nɛ̂ se pɩ tɛ́ ǹnɩ̂ fɔkɔ sõnɔ yṍtɔ́kɔ́bɩ́ yĩnnɛ, péwɔ náh fɛ̃ pé dyɔ dyɔ tyi pɩ n pi nɩ? 
Pâh wɛkɩ Liyel tyɩ́ yõke nɛ̂ yĩnnɛ, tɛ́ fyɔ̀ kélɛ n yõ, nɛy se yɩ̃nɛ à pé tyɩ́lɛ n yo kénɛ yõke yṍnɔ́ yĩnnɛ, képah nɛ̂ yĩn-á pé wɛkɩ Liyel tyɩ́ nɩ? 
Ǹtɛ sɛ̃́ náh dɛ́, wâh kwâh nɛ̂nɛ n yõ ò, wâh kwâh nɛ̂nɛ n wɔ ò, wâh se tir dùkù nɛ̂nɛ n pɩ ò, yɛ Liyel yĩn nɛ n gbilki ké sɔkɔ. 
Kè se Nsyifunɔ́ ò, kè se nónó náh pɩ Nsyifunɔlɛ ò, kè se Yesu yõ sɛ̃́pú tĩ̀nkìlɛ ò, yépi tyípɩ́nɔ́ káh pé sɔkɔ yɔ́ ńtɛ̃ yõ yɩkɩ n mɔ tyípékè pɩ́nɔ́ sɔkɔ. 
Yé ńmɔlɛ n yah n pɩ. Ńmɔ ńkɛ̃́nɛ n gbah n tɛ̃ ǹ no pól nɩ̀vɩ̀nɩ̀nɛ n pɩ tyi pól sɔkɔ. Ńmɔ náh mɛ ń gblɔ̌y pɛ́kɩ́-ńsahlɛ n yah n kɔ̃ dɛ́, ǹtɛ mé mɛ nɛ́kwĩ́nkílɛ n yohnɩ n mɔ pé wɛ n pwah n mɔ yĩnnɛ.
