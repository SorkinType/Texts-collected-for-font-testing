1 Tesaloniki tãn.
3.
Képah sɔkɔ, áyáh pópó pɔ tɛ̃ kwlɔ yé wɛ́-ńkɛ̃̂nɛ, á kè yah tɔkɔ nɛ ápi-á kɩ wũ á gblɔ̌y Atɛnɩ kwil sɔkɔ, 
tɛ́ mɛ á yṹnpi Timote tɛkɩ mɔ yé kɔ̃, ǹmɔ nɛ̂-á mɛ Liyel tõ̀lɛ n pɩ ápilɛ wɛ̃ tyɩ́ Nɩ̀vɩ̀nɩ̀ tir yónkɔ̃nɔ sɔkɔ Krista tyɩ́ sɔkɔ. Á nɛ à pɔ yé yohnɩ mɔ yé tǎhkɩ̀ wɛ Liyel yõ sɛ̃́nɔ́ sɔkɔ, à mó sṍré mɔ yé sɔkɔ, 
sɔ̃̀nko kwɛy gbĩ yèvyãhrɩ káh pɔ yé kɔ̃ kwɔ. Yé pɩ̃́nɔ́ se mɛ ké sɔkɔ nɛ yèvyãhrɩ mɛ̀nɛ n sɔkɔ tɛ́ á sah n pi tɛ́. 
Yé náh wɛ di, áyáh lésõ mɛ yé tĩ̀nnɛ gbĩ́ nɛ̂nɛ, á lésõ kè yo sah yé tyɩ́ nɛ áyáh kɩ pɔ yèvyãhrɩ pɩ n pi. Képah ye pɩ yɔ̀, ké sèndì náh yélɛ dò. 
Képah ye, máh pɔ tɛ̃ sṍmnɔ́ pɩ kwlɔ, mé Timote tɛkɩ mɔ nɛ à kyɩ yé Liyel yõ sɛ̃́nɔ́ dùkù yah n pɔ n yo ń syah. Tíkí sõ̀ mɛ ń tɛ̃ nɛ sétã̀n yõ̀tɛ̃̀ tɔ̃̀nmɔnɔ káh nɛ mɛ tɛ̃ n wɛ yé tyɩ́, ápi kwlɔ́nɔ́ kɔ̃ kè pɩ yõ kɛ̃́nɛ. 
Ǹtɛ núkúnúkú, Timote ye wil yé tĩ̀nnɛ à pɔ. À mɛ pɔ yé yĩ́ngbɩ́lɛ n yo nɛ Liyel yõ sɛ̃́nɔ́-á mɛ yé sɔkɔ, ànɛ̂ tyɩ́nɔnɔlɛ. Ńkɛ̃́ yáh mɛ n sõ ápi tyɩ́ sɔkɔ gbĩ́mɛgbĩ́. Á wɛ dékè-á mɛ yé tɛ̃, sɔ̃́-á yépi tɔ wɛ dékè mɛ ápi tɛ̃. 
Kǎh pɩ sɛ̃́ á yṹnpyé, áyáh sõ̀ mɛ yèkwɔrɩ ànɛ̂ yèvyãhrɩ nónó pól sɔkɔ, yé Liyel yõ sɛ̃́nɔ́ tyɩ́ nóhnɔ́ pɩ nɩ dénɔ́ ǹgbɛ̃nɛ á tyɩ́. 
Ǹtɛ yáh fyɔ̀ mɛ sɛ̃́nyĩnɔlɛ Tɛ̃̀ ǹgbɛ̃ yõ, á mɩ̀nɩ̀ ye pyě tiki de á fɔkɔ sɔkɔ núkúnúkú yɔ̀. 
Wɛy dùkù nɛ̂ tɛ̃̀ mɛ ápi kɩ fɛ̃ Liyel kɔ̃ n pi yé tyɩ́ sɔkɔ yɔ̀, wǎh nɩ̀vɩ̀nɩ̀ ǹgbɛ̃ nɛ̂nɛ á kɔ̃ yépi yĩnnɛ? 
Ké lékã̂h ké gbĩ̀ntɔ̃̀, á mɛ kélɛ n gbah n ni Liyel tyɩ́ nɛ à wɛ̃̀kɩ̀ yɩkɩ mɔ á kɔ̃ á fɛ̃ pɔ yah yé tyɩ́. Tɛ́ fɩ̃́ nɛ̂-à kɩ mɛ yé tɛkɩ Liyel yõ sɛ̃́nɔ́ sɔkɔ, á mó yé yohnɩ n mɔ ké tyɩ́ sɔkɔ. 
Á kélɛ n yah n kɔ̃ á Sú Liyel ńtɛ̃ tyɩ́ ànɛ̂ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́ nɛ, pè wɛ̃̀kɩ̀ yɩkɩ mɔ á kɔ̃ á pɔ yé tĩ̀nnɛ. 
Wɛ̃ tyɩ́nɔnɔ ànɛ̂ no pól tyɩ́nɔnɔ nɛ̂-á mɛ yé sɔkɔ, á kélɛ n yah n kɔ̃ Tɛ̃̀ ǹgbɛ̃ tyɩ́ nɛ, à wãnnɛ n mɔ ké yõ á kɩ nɛ yé tyɩ́-á nɔ sɔ̃́ ápi tyɩ́. 
À mó nɩ̀sãhlɛ yé kɔ̃ yĩ́ĩ́ Liyel yõ sɛ̃́nɔ́ sɔkɔ, yé kɔ̃ yè pɩ wɛ́nyo-ńsah kɛ̃́nɛ yé tyɩ́. Yè mó nɛ n mɛ yályál á Sú Liyel yah sɔkɔ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista-à sõ nɛ̂nɛ ǹ syɩ pɔ ǹ yõ ǹ yĩ̂nyõ tõ̀npɩbɩ póllɛ.
