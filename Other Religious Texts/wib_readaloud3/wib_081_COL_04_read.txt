Kolɔsɩ.
4.
Yõ̀tãm, yé yé yɔ̃bɩlɛ n tɛ̃ n plɛ gbɩ wɛ̃̀kɩ̀ yõ sɔkɔ, yé mó yé dyɔ sah nɛ n kwnɛ nɛ yõ̀tɛ̃̀ yɔ́-á mɛ yépi tɔ tyɩ́ yĩ̂nyõ sɔkɔ. 
Yé gbah n tɛ̃ nírí pɩ́nɔ́ sɔkɔ. Yé nírílɛ n pɩ yè pépɔrɩ pɩ̃́nɔ́ wɛynɛ Liyellɛ n kɔ̃. 
Yé Liyellɛ n ni ápi tɔ tyɩ́ sɔ̃̀nko à wɛ̃̀kɩ̀ yɩkɩ mɔ á kɔ̃ à fɛ̃ ǹ wɛynɛ n yo, ànɛ̂ mè fɛ̃ sètĩ̀n sõ̀ tir tyɩ́lɛ n yo nɛ̂-á Liyel wɔ̃bɩ sah Krista sɔkɔ. Képah nɛ̂ yĩn-á ńnɛ wɔ̀kɔ̀npèkè sɔkɔ. 
Yé Liyellɛ n ni ń tyɩ́ sɔ̃̀nko à ń yah tɛ̃ ń kɔ̃ mè fɛ̃ sètĩ̀n sõ̀ tir mɛ̀nɛ n yo n wah, máh yɩ̃nɛ mè kè yo sɔ̃́. 
Yé yé minnɛ n pɩ lékã́mnɛ Yesu yõ sɛ̃́-ńkɛ̃̂pú wrɔ́. Syɔ̃̀-ǹsah-à fyɔ̀ wɛ yé tyɩ́ pèpɔrɩ pɩ pɩ tɛ̃̀nɛ pé tyɩ́, yé kè pɩ pé tyɩ́. 
Yé yɛ yé wɛlɔ nóhnɔ́ nɔ névye tyɩ́, ànɛ̂ yõre mó nɛ n mɛ sé tyɩ́. Névi nɛ̂-á píkínyahnɔ pɩ, yé ké yah tɔ̃ n sah n yo tɛ̃̀ tyɩ́. 
Á yṹnpi Tisyiki nɛ̂-á ń nɛ́nɔ́nɩ́nɛ ànɛ̂ ń kɔ́-òlɛ tɛ́ mɛ tõ̀lɛ n pɩ pallɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́, ǹmɔ kɩ pɔ ń tyɩ́ kèdùkù pól yo yé tyɩ́. 
Ńmɔ ye wɛ tɛ n tɛkɩ n mɔ ǹnɛ yé tyɩ́ nɛ à kyɩ á tyɩ́ kèdùkù yo yé syah à mó yé nɩ sũ. 
Wonesimi tɔ mɛ ǹnɛ, ǹmɔ nɛ̂-á á yṹnpilɛ nɛ̂-á mɛ tómm, tɛ́ pɩ á nɛ́nɔ́nɩ́nɛ. Yépi sɔkɔ yɔ́ ye ǹnɛ. Tyi nónó pól-á mɛ n pɩ ásõ̂, pè kɩ pɔ sè yãh yé syah. 
Arisitarkɩ nɛ̂-á mɛ wɔ̀kɔ̀npèkè sɔkɔ ànɛ̂ ńmɔlɛ, à n wɛkɩ yé tyɩ́. Markɩ tɔ n wɛkɩ yé tyɩ́ ǹmɔ nɛ̂-á Barnabasɩ tyɩ́ névilɛ. Markɩ tyɩ́ kèdùkù yo sɔkɔ yé tyɩ́ nɛ wàh kɩ pɔ yé tĩ̀nnɛ, yé à tɛ̃ plɛ. 
Yesu nɛ̂-á pè n ye Nsyusitusilɛ, à n wɛkɩ yé tyɩ́ tɔlɛ. Nsyufunɔ Yesu yõ sɛ̃́pú sɔkɔ, pépi ó ye mɛ tõ̀lɛ n pɩ ànɛ̂ ńmɔlɛ à Liyel tɔ̃́rɩ́ tyɩ́lɛ n yo. Pè ń nɩ de yĩ́ĩ́ kègbɩlɛ. 
Yepafrasɩ nɛ̂-á yépi sɔkɔ yɔ́lɛ, à n wɛkɩ yé tyɩ́. Yesu tõ̀npɩ ye ǹnɛ, nɛ̂-á mɛ nírílɛ n pɩ yé tyɩ́ gbĩ́mɛgbĩ́ sɔ̃̀nko yè nɛ n mɛ sṹnyĩnɔlɛ Liyel yõ sɛ̃́nɔ́ sɔkɔ, Liyel dyɔ yĩ́ĩ́ tyi pól pɩ pɩ tɛ̃̀nɛ. 
À yèvyãhrɩ kah pɩ yépi ànɛ̂ Lawodise tãn, ànɛ̂ Yerapolisi tãn yĩnnɛ. Ńmɔ ńtɛ̃ kɩ fɛ̃ pɩ képah tàrfyɔ̀lɛ. 
Dɔ̀ktɔ̂r dĩ́ Likyi nɛ̂-á ápi kódĩ́nɛ, à n wɛkɩ yé tyɩ́ ànɛ̂ Damasɩlɛ. 
Á yṹnpyé nónó-á mɛ Lawodise, yé wɛkɩ pé tyɩ́ à tahlɩ Nɛfa tyɩ́ ànɛ̂ Yesu yõ sɛ̃́pú tĩ̀nkì nɛ̂-á n tuke n mɔ n pɔ ǹ gbô. 
Yáh kɩ sɛ́bɛ́y mɛ̀ wɛ kar, yé gbah n tɛ̃ Lawodise Yesu yõ sɛ̃́pú tĩ̀nkì tɔ kɔ̃ pè kè wɛ n kar. Nɛ̂ tɔ-à kɩ se mó mɛ tɛkɩ mɔ pépi tyɩ́, pàh kɩ kélɛ yé kɔ̃, yé kè kar. 
Yè mó n yo Arsyipi tyɩ́ nɛ, Tɛ̃̀ ǹgbɛ̃-á tõ̀ nɛ̂ dahbɩ ǹ gbõ̀, à gbah n tɛ̃ kè pɩ pallɛ. 
Ńmɔ Pole ye n wɛkɩ yé tyɩ́, tɛ́ kélɛ n wãrkɩ ń gbõ̀lɛ. Yé káh n yɛ yé dyɔ fwɔ nɛ máh mɛ wɔ̀kɔ̀npèkè sɔkɔ. Mé kélɛ n yah n kɔ̃ Liyel tyɩ́ nɛ, à yõ̀bènɔrɩ pɩ yé pól tyɩ́.
