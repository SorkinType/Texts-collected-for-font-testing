Nsyakɩ.
2.
Ń yṹnpyé, yáh sɛ̃ tɔ̃́rɩ́ Tɛ̃̀ ǹgbɛ̃ Yesu Krista yõ, yé náh yɩ̃nɛ yè nɛynɛ n wah. 
Sɔ̃́-á dĩ́ yɔ́ kɩ fɛ̃ pɔ de yé tɔ́wû nɩyṍ sɔkɔ, tɛ́ kɩ mɛ sã́n kèngbɔ tã́hnɩ́ dyɩ, tɛ́ kɩ mɛ fwɔ̀tǎhnɩ̀ mɔ; tɛ́ yãm nɛy yɔ́ tɔ kɩ fɛ̃ pɔ de yé nɩyṍ fwɔ̀yàllɛ ǹ wil: 
Yâh fwɔ̀pèplɔ mɔ́-ò dĩ́ wɔ tɛ̃́nɔ́ plɛ, tɛ́ yo ǹ tyɩ́ nɛ, à pɔ kɔ̃ tɛ̃ tɛ̃̀-ǹsah pèpɛy mɛ̀ sɔkɔ, tɛ́ yo yãm nɛy mɛ̀ wɔ tyɩ́ nɛ, à yuku yĩn tɛ̃, képah náh pɩ, à kɔ̃ tɛ̃ ásõ̂ sétáh sɔkɔ; 
yâh pɩ sɛ̃́, wǎhkɩ̀ náh yé n pɩ wɛ̃ wrɔ́ di? Yé náh ǹnɩ̂ yɔ́ tɔkɔ fṍnfṍ pɩ ǹ ní-òlɛ yé fɔ́kɔ́sõnɔ túkúnì gbõ̀ yõ di? 
Ń yṹnpyé, yé tyɩ́ nɔ ń tyɩ́. Yé yé níkí kwɔ noh. Névye nónó-á yãm tãnnɛ kèkõyṍ névye tɛ̃̀ tyɩ́, pépi ye Liyel yah wilki, à pɩ tahbɩlɛ Yesu yõ sɛ̃́nɔ́ sɔkɔ. Ànɛ̂ pépi ye tɔ̃ à kɩ kɔ̃ pè kɩ n de n pi ǹ tɔ̃́rɩ́ sɔkɔ, képah nɛ̂ tyɩ́-á à ǹ vyãhlɛ névye kɔ̃, ǹ tyɩ́-á nɔ nónó tyɩ́. 
Ǹtɛ, yépi-à yãm tãnnɛ n yilɔ, tɛ́ tahbɩ wɔlɛ n gbilki-à; kénɛ tahbɩ tɛ̃̀ wɔ náh n fõh yélɛ, tɛ́ yélɛ n tɛ̃ n tuke n pɔ tukey tùkè-ǹtã̀n sɔkɔ di? 
Tɛ́ Krista yĩn pèpɛy-á yé yõ sɔkɔ, pépi ye tɔ̃ képah tyɩ́ yónkòpulɛ. 
Ǹtɛ, Liyel tyi kõ̀nsàhnɔ̀ yɔ́ mɛ wã́rkɩ́nsàhnɔ́lɛ nɛ̂-á gblò kal sé no tɛ́lɔ́lɛ. Képah n yo nɛ ńkɛ̃́: «Yɛ á ní-ò tyɩ́ nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ». Képah wɛ̃̀kɩ̀-à kɩ nɛ̂ yé n kɔ, yé mɛ n pɩ plɛ. 
Ǹtɛ, wǎhkɩ̀ tɛ̃̀-à kɩ nɛ̂ yé n pɩ, tyípékè ye kélɛ. Yé mɛ pílkínɔ́lɛ Liyel tyi kõ̀nsàhnɔ́ sɔkɔ, yáh sɔ̃́npɩ́ yahle kɔ́lɔ́lɛ ké wɛ̃̀kɩ̀ yõ. 
Yah, Liyel nɛ̂-á yo nɛ: «Káh fɔ̀fɩ́kɩ̀ pɩ», ǹmɔ ó ye tɔ̃ yo nɛ: «Káh nɛ́kókè mɔ.» Wâh sɔ̃́ fɔ̀fɩ́kɩ̀ pɩ́-ńkɛ̃̂nɛ, tɛ́ nɛ́kókè mɔ, á ye Liyel tyi kõ̀nsàhnɔ́ yahle. Képah ye, névi-à sɔ̃́ Liyel tyi kṍnsàhnɔ́ pól pɩ, tɛ́ sé sɔkɔ tyíyóné núkú fyé yahle, tɛ̃̀ mɛ pílkínɔ́lɛ sé pól sɔkɔ. 
Liyel tyi kõ̀nsàhnɔ́ ye sah yélɛ yé gblɔ̌y yõ, tɛ́ Liyel kɩ yé tuke n yɩ̃nɛ n pi sépilɛ. Képah ye, yé yɛ yé wɛy, ànɛ̂ yé pɩ́dúkù nɛ mɛ yɩ̃́nɛ́nɔ́lɛ Liyel tyi kõ̀nsàhnɔ́ nɔ́pilɛ. 
Ńkɛ̃́, névi nɛ̂-à névye yãmnɛ n yáh-ńkɛ̃̂nɛ, Liyel tɔ náh tɛ̃̀ yãm yah n pi tukey tuke sṍlɛ. Ǹtɛ, névi nɛ̂-à névye yãmnɛ n yah, tɛ̃̀ yõ kɩ yɔ̃ tukey tuke sṍlɛ. 
Ń yṹnpyé, ǹnɩ̂ yɔ́-à ńtyɩ̃́nɔ nɛ, páh sɛ̃ Liyel yõ, tɛ́ ǹ tyípɩ́nɩ̀ náh mɛ kélɛ n wɛ̃kɩ, yõ se kɩ wɛ ké tyɩ́? Kénɛ Liyel yõ sɛ̃́nɔ́ dùkù mɛ̀ se kɩ fɛ̃ à pwah mɔ? 
Sɔ̃́-á, á yṹnpi yɔ́ tyɩ́ flɔ kɩ fɛ̃ n knɛ, ànɛ̂ à náh mɛ n wɛ n yõ swɔ̃swɔ̃ pól sɔkɔ: 
Yõ se mɛ ké tyɩ́, wâh tɛ̃̀ syah nɛ, à de lènɔlɛ, à mó flɔlɛ n mɔ vɩrɩlɛ n tũ, ànɛ̂ à nɛ n yõ n tɔ̃; tɛ́ á náh se mó tɔ̃ à kɔ̃ ǹ yétõ̂ yî nɔ́pilɛ? 
Sɛ̃́ ó ye, névi nɛ̂-à nɛ, páh sɛ̃ Liyel yõ, tir yɔ́ ńtɛ̃ pɩ́-ńkɛ̃̂nɛ nɛ̂-á ǹ Liyel yõ sɛ̃́nɔ́lɛ n wɛ̃kɩ, tɛ̃̀ Liyel yõ sɛ̃́nɔ́ mɛ kúntɛ̃̀nɔ́lɛ. 
Gbɔ̀kénpɩ ǹnɩ̂ yɔ́ kɩ fɛ̃ nɛ, túkù-á mɛ sɛ̃ Liyel yõ, túkù mó mɛ tyípéplɔlɛ n pɩ. Kàh pɩ sɛ̃́, tɛ̃̀ ń wɛ̃kɩ ǹ Liyel yõ sɛ̃́nɔ́lɛ, tyípɩ́nɩ̀ kɛ̃́nɛ wɛ́. Ńmɔ wɔ kɩ à wɛ̃kɩ ń Liyel yõ sɛ̃́nɔ́lɛ ń tyípɩ́nɩ̀ sɔkɔ. 
Wâh mɛ sɛ̃ ké yõ nɛ Liyellɛ, núkú-á ǹnɛ, kè plɛ. Tɛ́ pɩ̃ nɛ, sétã̀n tɔ sɛ̃́nɔ́-á mɛ képah tɛ̃̀ wɔ yõ dɛ́. Ànɛ̂, ǹ yah tíkí pɩ́nɔ́ tɛ̃̀ tyɩ́, sé wil n nɛ n syah. 
Lékã́m se mɛ ǹ tɛkɩ? Wâh mɛ ké tyɩ́, mé kɩ fɛ̃ ǹ wɛ̃kɩ nɛ, tyípéplɔ pɩ́-ńkɛ̃̂ Liyel yõ sɛ̃́nɔ́lɛ, yõ náh mɛ ké tyɩ́: 
Ápi náh nɛy Abrahmɩ-á tyípɩ́kɩ̀ nɛ̂ pɩ, képah nɛ̂-á pɩ ǹ pídĩ́ Yisakɩ dɔ́kɔ́nmɔnɔlɛ sárká súkɛ́nkɔ̃-ńsah yõ, nɛ pé à ko Liyel kɔ̃, sɔ̃́-á Liyel yo kélɛ ǹ tyɩ́. Képah sɔkɔ náh Liyel à pɩ nɛ́gbɩ́lɛ di? 
Á náh wɛ di, Abrahmɩ Liyel yõ sɛ̃́nɔ́ ànɛ̂ ǹ tyípɩ́nɩ̀ sõ̀ mɛ n kɔ wɛ̃ tyɩ́. Tɛ́, ǹ tyípɩ́nɩ̀ gbõ̀ yõ, ǹ Liyel yõ sɛ̃́nɔ́ pɩ pallɛ. 
Képah sɔkɔ ye Liyel wɛy mɛ̀ pɩ tũ nɛ̂-á yo nɛ ńkɛ̃́: «Abrahmɩ sɛ̃ yĩ́ĩ́ ǹ Tɛ̃̀ ǹgbɛ̃ Liyel yõ, Liyel mó à pɩ nɛ́gbɩ́lɛ», tɛ́ ǹnɛ n ye ǹ kódĩ́nɛ. 
Yé kè wɛ kɛ̀, névi Liyel yõ sɛ̃́nɔ́ ó náh dɛ́! Ǹ tyípɩ́nɩ̀ tɔ gbõ̀ yõ ye Liyel à pɩ nɛ́gbɩ́lɛ. 
Sɛ̃́ ó ye tɔ̃, fɔ̀fɩ́npɩ pi nɛ̂-á lésõ pè n ye Wrahbɩlɛ: À lésõ fɛ̃ syi Yisrayel tɛ̃ntõ̀ névye kɔ̃ pè wɔ̃bɩ ǹ tyɩ́. Tɛ́ pè yohnɩ mɔ, à pè mɔ wɛ̃̀kɩ̀ yɔ́ yõ, pè kɔ̃ pè to pwah. Kénɛ tyípépɛ̌y gbõ̀ yõ ye Liyel ǹ tɔ pɩ nɛ́gbɩ́lɛ. 
Sɔ̃́-á névi min-à wil ǹ sɔkɔ, à n nɛ mɛ kúnɔ́lɛ, sɛ̃́ ó ye tyípɩ́nɩ̀ tɔ-à n knɛ, névi Liyel yõ sɛ̃́nɔ́ n nɛ mɛ kúnɔ́lɛ.
