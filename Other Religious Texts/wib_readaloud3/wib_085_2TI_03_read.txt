2 Timote.
3.
Timote, á yɩ̃nɛ ǹ kè pɩ̃ nɛ kèkõyṍ tyɛ gbĩ́nɛ, tyah táhnɔ́nɔ-á kɩ n wɛ. 
Névye kɩ pɩ gblɔ̌y yõ yɩ̃́kɩ́púlɛ, pɔ́ kódíbílɛ, wɛ̀ngbìl wɛ́kɩ́púlɛ, gblɔ̌y gbílkípúlɛ, Liyel tyɩ́ yóndepulɛ, dúbípú tyɩ́ yáhlepulɛ, pèpɔrɩ pɩ̃́-ńkɛ̃̂ névyelɛ, Liyel tyi mɔ́nkõ̀-ǹkɛ̃̂púlɛ, 
kùkùtǎhkɩ̀ névyelɛ, nɛ́pĩ́kɔbɩlɛ, névye yĩn yɩ́kɩ́púlɛ, gblɔ̌y tɛ̃́-ńkɛ̃̂ névyelɛ, nɛ́kókè mɔ́púlɛ, pèpɛy fwópúlɛ, 
wɛ̃ náh vìpulɛ, fɔ̀kɔ̀nvyã̀hnɩ̀ névyelɛ, gblɔ̌y yɔ̃̀nsàhpulɛ. Pé dyɔ yĩ́ĩ́ tyi tyɩ́ kɩ nɔ kal pé tyɩ́ Liyellɛ. 
Pè kɩ pɩ yĩ́ yah Liyel yah tíkí pɩ́púlɛ, ǹtɛ kègbɩ yõ, pè náh Liyel yah tíkí pɩ́nɔ́ tǎhkɩ̀ fɛ̃ n syi n pi. Káh nɛ n pɩ n soh sɛ̃́nknɔ névyelɛ. 
Pé túkù n nɛ n de n kɔ̃ gbên sɔkɔ ǹ syɔ́ túkùlɛ n vi n tɛ̃, nónó lékã́m-á tuke. Kénɛ syɔ́ mɛ tyípêl ó pé tɔ̂ɔ́. Tyi duke pól dékè n nɛ n mɛ pé sɔkɔ. 
Pè mɛ sã́hnɔ́ sɔkɔ kṹmm, ǹtɛ pè náh mɛ fɛ̃ gbɩlɛ n pnɛ póllɛ. 
Nsyãnɛsɩ ànɛ̂ Nsyãbrɛsɩ-á Moyisi kɔ̃lɩ sɔ̃́, sɛ̃́ ye pépi tɔ mɛ n kɔ̃lɩ gbɩlɛ. Pé sõnɔ náh mɛ pallɛ. Liyel yõ sɛ̃́nɔ́ náh yuku yahlɛ. 
Ǹtɛ pè náh n yuku n pi yahlɛ. Yé náh wɛ di, no pól kɩ pé nɛ́wṍré wɛ kwèyy sɔ̃́-á Nsyãnɛsɩ ànɛ̂ Nsyãbrɛsɩ tyɩ́ pɩ. 
Ǹtɛ áwɔ Timote tɛ̃̀ wɔ ye ńmɔ sèfiki mɔ tyi tyi pól sɔkɔ: Máh kwɔ́nɔ́ nɛ̂ pɩ á kè fɛ̃ syi, tɛ́ mɛ ń pɩ dùkùlɛ n yah n pɩ. Á ń sõnɔlɛ n pnɛ, ànɛ̂ ń Liyel yõ sɛ̃́nɔ́ dùkùlɛ, ànɛ̂ ń tyi srónɔ́ dùkùlɛ, ànɛ̂ tyɩ́nɔnɔ nɛ̂-á mɛ ń sɔkɔ, à tahlɩ máh mɛ gbah tɛ̃ dùkù nɛ̂nɛ. 
Pǎh ń fõh dùkù nɛ̂nɛ á kélɛ n pnɛ, à tahlɩ máh yèvyãhrɩ nónó pɩ Antyɔsyɩ, ànɛ̂ Yikonyumi, ànɛ̂ Lisitir sɔkɔ. Fṍhnɔ́ nɛ̂ dùkù tɛ̃̀ mɛ mé náh kè wɛ yah? Ǹtɛ Tɛ̃̀ ǹgbɛ̃ ń pwah mɔ sépi pól sɔkɔ. 
Gbɩ yõ, nónó pól-á sɛ̃ Yesu yõ, tɛ́ mɛ ké tyɩ́ nɛ pé pé minnɛ n pɩ Liyel yah tíkí pɩ́nɔ́ sɔkɔ, pé kɩ fṍhnɔ́ wɛ. 
Ǹtɛ nɛ́pêl ànɛ̂ névye pílkípú tɛ̃̀ wɔ kɩ yɔ̃ sɔkɔ pèkè pɩ́nɔ́lɛ kɩ névye túkù pilki, tɛ́ kɩ pé gblɔ̌y ńtɛ̃ tɔ pilki. 
Ǹtɛ áwɔ tɛ̃̀ wɔ-á kwɔ́nɔ́ nɛ̂ wɛ tɛ́ kè fɛ̃ syi nɛ gbɩ-á kélɛ, kè tɛ̃ yĩn pallɛ. Wáh kwɔ́nɔ́ mɛ̀ wɛ nɛ̂ tyɩ́, á ǹnɛ n pnɛ ó. 
Á wɛ̃́npir gbĩ́ dùkùlɛ á Liyel wɛynɛ n pnɛ. Kè kɩ fɛ̃ á lékã́m yɩkɩ mɔ ǹ kɔ̃ á kɩ pwáhnmɔnɔ wɛ Yesu Krista yõ sɛ̃́nɔ́ gbõ̀ yõ. 
Liyel wɛy pól wãrkɩ Liyel tǎhkɩ̀ gbõ̀ yõ. Kè wãrkɩ névye kwɔ yĩnnɛ, ànɛ̂ mɩ̀nɩ̀ gbel kye yĩnnɛ, ànɛ̂ pílkí-ńtã̂n tɔ̃ yĩnnɛ, ànɛ̂ névi mɔ yĩnnɛ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ, 
sɔ̃̀nko, Liyel tyɩ́ névi pɩ nɛ́gbɩ́lɛ tã̀n pèplɔ pól fɛ̃ n pɩ yĩnnɛ.
