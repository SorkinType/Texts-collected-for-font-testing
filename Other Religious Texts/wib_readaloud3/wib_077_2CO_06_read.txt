Korɛ̃nti tãn.
6.
Ápi-á pɩ Liyel tõ̀ pɩ mùnɛ sɛ̃́, képah ye á kélɛ n yah n kɔ̃ yé tyɩ́ nɛ wǎh yõ̀bènɔrɩ nɛ̂ pɩ yé tyɩ́, yé káh yɛ kè pɩ yõ kɛ̃́nɛ. 
Yé náh wɛ di, Liyel nɛ ńkɛ̃́: «Tyah-á sõ̀ mɛ pallɛ gbĩ́ nɛ̂nɛ, mé á nírí tɛ̃. Pwáhnmɔnɔ sõlɛ, mé ǹ yohnɩ mɔ. Tyah pèpɛy gbĩ́ mɛ̀ ye núkúnúkú yɔ̀. Yah, pwáhnmɔnɔ yòh ye kwɛynɛ.» 
Á náh mɛ ké tyɩ́ nɛ á ǹnɩ̂ yɔ́ ńtɛ̃ yõ yɩkɩ, ǹnɩ̂ kɔ̃ à pɔ wɛ́nyo-ńsah wɛ á tõ̀ tyɩ́ sɔkɔ. 
Ǹtɛ tyi tyi pól sɔkɔ, ápi mɛ á gblɔ̌ynɛ n wɛ̃kɩ Liyel tõ̀npɩbɩ kègblɔlɛ. Á mɛ gbah tɛ̃ yèvyãhrɩ, ànɛ̂ yèkwɔrɩ, ànɛ̂ tíkí képékèlɛ n sro náhnáh. 
Á ko syi, tɛ́ wɔ̀kɔ̀npèkè pɩ. Pè névye mɔ á yõ, tɛ́ á mɔ tã̀n pèl pɩ́nɔ́ sɔkɔ. Á náh mɛ n wɛ n dɔ, yõke tɔ náh mɛ n wɛ n yõ. 
Á mɛ tõ̀lɛ n pɩ Liyel tyɩ́ fɔkɔlɛ píkínɔ́lɛ ànɛ̂ á lékã́m póllɛ. Á mɛ tyilɛ n sro mɛ pèpɔrɩlɛ n pɩ Liyel Mirki gbõ̀ yõ. Tyɩ́nɔnɔ yèsyɩ̀kɩ̀ mɛ á sɔkɔ, 
ànɛ̂ gbɩ wɛlɔ ye á mɛ n yo Liyel tǎhkɩ̀ gbõ̀ yõ. Gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ ye ápi pɩ á mɛ́fwó yîlɛ à á sépɛ́bɩ́lɛ n fwo, tɛ́ á gblɔ̌y yõlɛ n syɩkɩ. 
Névye á dɔkɔ n mɔ, képah náh pɩ pě á kõ fɛ̃. Pě á yĩ́gbɩ́lɛ n yo, képah náh pɩ pě á yĩ́pollɛ n yo. Pě álɛ n pɩ mɩ̀nyobelɛ, tɛ́ gblɔ yópú ye mó tɔ̃ álɛ. 
Pě álɛ n pɩ pé kósyì á pɩ̃, tɛ́ pè mó tɔ̃ álɛ n pnɛ kyɔ́. Pě álɛ n pɩ lékyɩ̂ sɔ̃́ álɛ tɛ́ vye ye mó tɔ̃ álɛ. Pě fṍhnɔ́lɛ n sah á yõ, tɛ́ pè náh álɛ n ko n mɔ. 
Pě yèkwɔrɩ tyilɛ n pɩ á tyɩ́, tɛ́ gbĩ́mɛgbĩ́ à n nɛ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ. Pě álɛ n tɔkɔ n pɩ yãm tãnnɛ, tɛ́ á mó tɔ̃ mɛ névye náhnáhlɛ n pɩ tahbɩlɛ. Pě álɛ n pɩ kwâh kɛ̃́púlɛ, tɛ́ yíyìn pól mó tɔ̃ mɛ á tyɩ́. 
Korɛ̃ntɩ tãn, gbɩ ye ápi n yo yé tyɩ́. Á á fɔkɔ dal mɔ yélɛ. 
Ápi náh á fɔkɔ kwĩnki yélɛ. Yépi ye yé fɔkɔ kwĩnki ápilɛ. 
Sú-á ńkɛ̃́nɛ tyilɛ n yo ǹ wɛ̃́npì tyɩ́ sɔ̃́, sɛ̃́ ye mé mɛ n yo tyilɛ yé tyɩ́. Yé yɩ̃nɛ yè kè pɩ̃ á tyɩ́, yépi tɔ mó yé fɔkɔ dal n mɔ yĩ́ĩ́ álɛ. 
Yé káh nɛ n pɩ n soh Yesu yõ sɛ̃́-ńkɛ̃́púlɛ tir yɔ́ ńtɛ̃ sɔkɔ. Gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ ànɛ̂ tyítúkúnì pɩ́nɔ́ náh mɛ wɛ̃̀kɩ̀ núkú yõ. Kyòó ànɛ̂ fɛ̃́ntũ̂ náh fɛ̃ n kwrɔ n mɔ. 
Krista ànɛ̂ sétã̀n yõ̀tɛ̃̀ wrɔ́ náh fɛ̃ n kwrɔ n mɔ. Yesu yõ sɛ̃́-ò ànɛ̂ Yesu yõ sɛ̃́-ńkɛ̃́ névi wrɔ́ náh fɛ̃ n kwrɔ n mɔ. 
Fal fal tã̀n náh fɛ̃ n kwrɔ n mɔ Liyel ní-ńsah gbô sɔkɔ. Yé náh wɛ di, ápilɛ, Liyel ní-ńsah gbô ye álɛ à yɩ̃nɛ Liyel-á kè yo sɔ̃́ nɛ ńkɛ̃́: «Mé kɩ tɛ̃ pélɛ tɛ́ kɩ mɛ pélɛ pé kɔ́-ńtã̂n sɔkɔ. Mé kɩ pɩ pé Liyellɛ, tɛ́ pè kɩ mó pɩ ń nolɛ.» 
Képah ye Tɛ̃̀ ǹgbɛ̃ Liyel nɛ: «Yé wil kénɛ névye wrɔ́, yè gbɛ n tɛ̃ yé tyɩ́lɛ. Yé káh n tũ kwâh yɔ́ ńtɛ̃ tyɩ́ nɛ̂ náh mɛ yályál ńmɔ yah, képah sɔkɔ ńmɔ kɩ mó yé tɛ̃ plɛ. 
Mé kɩ pɩ yé Súlɛ, tɛ́ yé kɩ mó pɩ ń dĩ́npì ànɛ̂ ń syɔ́npìlɛ.» Sɛ̃́ ye Tɛ̃̀ ǹgbɛ̃ yo, ǹmɔ nɛ̂-á tǎhkɩ̀ nɛ́tɛ̃̂nɛ.»
