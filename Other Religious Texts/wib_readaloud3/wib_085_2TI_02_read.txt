2 Timote.
2.
Áwɔ tɛ̃̀ wɔlɛ ń pídĩ́ Timote, yõ̀bènɔrɩ nɛ̂-á ápi wɛ á wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Yesu Kristalɛ, képah yĩnnɛ, á tah n sah yĩ́ĩ́. 
Wáh ń kwɔ́nɔ́ nɛ̂ noh névye náhnáh yĩ́ yah, kyɩ nɛ́gblɔ́ kwɔ kélɛ, nónó tɔ-á kɩ tɔ̃ fɛ̃ túkù kwɔ kélɛ. 
Kwlɔ́nɔ́ nɛ̂-á yɩ̃nɛ álɛ, kè fɛ̃ n syi Yesu Krista sràsyi yèsyɩ̀kɩ̀ tyɩ́ kɔ̃lɛ. 
Névi-à fyɔ̀ de sràsyìyá sɔkɔ tɛ́ mɛ ké tyɩ́ nɛ pé pé yõ̀tɛ̃̀ nɩ̀vɩ̀nɩ̀nɛ n pɩ, ǎ ǹ vyãh dɛ kèkõyṍ tyi tɛ́lɔ́ sɔkɔ. 
Wɛ̃ kàlnɔ́ tólɔ́ tó-ò tɔ náh tɔ̃́rɩ́ yṍkúrílɛ n wɛ n pnɛ nɛ́ à náh to ké wɛ̃̀kɩ̀ yõ. 
Sètõ nɛ̂-á tõ swãhlɛ à kwlɔ, ǹmɔ ǹgbò ye yɩ̃nɛ ké sèmukɔ wɛ́nɔ́lɛ 
Sõ n yah ń wɛlɔ yõ, Tɛ̃̀ ǹgbɛ̃ kɩ sé pól syɩ́kɩ́-ńsahlɛ ǹ wɛ̃kɩ. 
Káh á dyɔ fwɔ Yesu Krista tyɩ́lɛ, ǹmɔ nɛ̂ lékó-á yiki mɔ. Davidi náh nɛy ye ǹnɛ. Sɛ̃́ ye kélɛ yónɔ́lɛ Nɩ̀vɩ̀nɩ̀ tir sɔkɔ, képah nɛ̂-á mé mɛ n yo n kɔ̃. 
Kénɛ Nɩ̀vɩ̀nɩ̀ tir yĩn ye kwlɔ́nɔ́lɛ ń yõ sɔkɔ, ànɛ̂ kârnnɛ mɔ́nɔ́lɛ ń tyɩ́ á kɩ nɛ pèkè pɩ́-ò sɔ̃́ ńnɛ. Ǹtɛ kârn náh se yɔ́ mɛ mɔ Liyel wɛy tɛ̃̀ wɔ tyɩ́ tɛ́. 
Képah ye, mé mɛ tyi póllɛ n sro névye yĩnnɛ Liyel-á nónó yah tɔkɔ, sɔ̃̀nko pé tɔ pwáhnmɔnɔ wɛ Yesu Krista gbõ̀ yõ, pé mó gbílkínɔ́ wɛ nɛ̂-á kè náh n tyɛ. 
Gbɩ wɛy yɔ́ wɛ yɔ̀: Áyáh kɩ mɛ ku Yesu Kristalɛ wɛ̃ tyɩ́, á kɩ mɛ min tɔ sɔkɔ ǹnɛ wɛ̃ tyɩ́. 
Áyáh kɩ á tah sah wɛ̃ tyɩ́ ǹnɛ, á kɩ tɔ̃́rɩ́ pɩ ǹnɛ wɛ̃ tyɩ́. Áyáh kɩ à yahle, ǹ tɔ kɩ á yahle. 
Áyáh kɩ sɔ̃́ pɩ́-ńkɛ̃̂nɛ nɛ́gblɔ́lɛ, ǹmɔ tɛ̃̀ wɔ kɩ tɛ̃ nɛ́gbɩ́lɛ, à náh fɛ̃ ǹ syɩ n kur n pi. 
Névye dyɔ to n kõ n mɔ kénɛ tyilɛ, ǹ tɛ́ ké gbáhdyâh di pé tyɩ́ Liyel yah sɔkɔ nɛ, pè káh yékã̂b wã̀nnɛ n pɩ. Sè náh tir yɔ́lɛ n pɛkɩ. Nónó-á mɛ sélɛ n nohnɩ, sè n pyě pépi mɔ n pu. 
Á tah n sah Liyel yĩ́ kɔ̃ kè yuku álɛ. Pɩ tõ̀npɩlɛ, nɛ̂-á gbɩ wɛlɔlɛ n yo ké yah yõ, tɛ́ sen náh n pɩ ǹ tyɩ́ à yuku ǹ tõ̀ tyɩ́ tĩ̀nnɛ. 
káh yã́hwã̂nnɛ n yãh nónó-á sè náh mɛ yuku Liyel yõ sɛ̃́nɔ́ tyɩ́ tĩ̀nnɛ. Nónó-à kélɛ n pɩ, pé wrɔ́ n nɛ n gbah n gbɛ Liyellɛ. 
Pé yã́hwã̂n yɩ́kɩ́nɔ́lɛ n pɩ á kɩ nɛ nówũ̂-á ńkɛ̃́nɛ willɛ n yõ sɔ̃́. Yimene ànɛ̂ Filɛtɩ tɔ mɛ kénɛ sɛ̃́nknɔ névye sɔkɔ. 
Pépi gbɩ wɛ̃̀kɩ̀ fĩn mɔ tɛ́ mɛ n yo n nɛ lékyɩ̂-á yilki mɔ sɔkɔ tɛ. Sɛ̃́ ye pè pɩ tɛ́ névye náhnáh Liyel yõ sɛ̃́nɔ́ yɩkɩ. 
Ǹtɛ sépinɔ pól sɔkɔ, Liyel-á sètyah nɛ̂ wilki, képah tɛ̃̀ wɔ mɛ. Wɛlɔ nónó-á mɛ wã́rkɩ́nɔ́lɛ ké tyɩ́ yõ, sè wɛ yɔ̀: «Nónó-á mɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́ yõ, à pélɛ n pnɛ.» Ànɛ̂ tɔ̃: «Nónó pól-á Tɛ̃̀ ǹgbɛ̃nɛ n ye n gbo, pè yɩ̃nɛ pè tyítúkúnì wɛ n yɛ.» 
Wɔ̀kɔ̀ ǹgbɛ̃ sɔkɔ, sã́n vyɔ́ ànɛ̂ wárn vyɔ́ ó kɔ́kɔ́ náh mɛ dɛ́; sésár tyɩ́nɔ mɛ, puri tyɩ́nɔ tɔ mɛ. Túkù mɛ yî tǎhnɩ̀ mɔ yĩnnɛ, túkù mó mɛ ńyã̌n yî mɔ yĩnnɛ. 
Tyi nónó tyɩ́-á mé yo, Névi nɛ̂-à sè wɛ yɛ tɛ́ pɩ yályál Liyel yah, tɛ̃̀ kɩ mɛ wìlkìnsàhnɔ́lɛ tyítǎhnɩ̀ pɩ yĩnnɛ. Ǹ yétõ̂ kɩ mɛ yõ̀tɛ̃̀ dĩ́ tyɩ́ tyípéplɔ pɩ yĩnnɛ. 
Kètɛ̃̀npìkìnì dékè tyi wɛ n yɛ. Tyi nónó-á á yɩ̃nɛ n yah n kɔ̃, sè wɛ yɔ̀: Gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́lɛ, Liyel yõ sɛ̃́nɔ́, tyɩ́nɔnɔlɛ, ànɛ̂ yèvɩnɩnɛ. Yé sépi póllɛ n pɩ ànɛ̂ névyelɛ nónó-á mɛ Tɛ̃̀ ǹgbɛ̃nɛ n ye n gbo fɔkɔ pérpérlɛ. 
Yékã̂b wã̀n pɩ́nɔ́ wɛ n yɛ. Á pɩ̃ nɛ sépi-á n pɔ n pɔ mâllɛ dò. 
Ǹgbǒ Tɛ̃̀ ǹgbɛ̃ tõ̀npɩ náh yɩ̃nɛ mɛ́kɩ̀ pɩ́nɔ́lɛ dɛ́. À pyě yɩ̃nɛ à nɛ n mɛ nɛ́pír névilɛ no pól tyɩ́, à mó nɛ n mɛ névye kwɔ́nɔ́ yah wɛ, tɛ́ pàh ǹ tyɩ́lɛ n yo, à tɛ̃ wɛ kè sṍmnɔ́. 
À yɩ̃nɛ à ǹ wɛy kɔ̃̀lɩ̀pulɛ n tɛ̃ mármár à vìnmɔnɔlɛ pélɛ n kɔ̃. Nɛ̂ pɩ̃, tɛ́nkàh Liyel kɩ yɛ pè kɔ̃ pè kɩ pé sõnɔ wɛ vi mɔ kɩ gbɩ wɛ pɩ̃ tɛ́. 
Képah-à pɩ, pé yah kɩ pɩ kyòo pè kɩ wɛ dɛ sétã̀n yõ̀tɛ̃̀ vyɔ́y sɔkɔ, nɛ̂ sɔkɔ-á sõ̀ à pè mɔ tɛ̃ sah pè kɔ̃ pè ǹ dyɔ yĩ́ĩ́nɛ n pɩ.
