Wrome tãn.
10.
Ń yṹnpyé, máh mɛ nɛ̂ tyɩ́ ń nɩ póllɛ, képah ye nɛ Nsyifunɔ pwáhnmɔnɔ wɛ. Mé mɛ Liyellɛ n ni pé tyɩ́ képah tyɩ́ sɔkɔ. 
Yé náh wɛ di, mé kɩ fɛ̃ kè yo nɛ pǎh mɛ gbah tɛ̃ yĩ́ĩ́ Liyel wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ, ǹtɛ pǎh mɛ kélɛ n pɩ yɩ̃́tũri sɔkɔ. 
Liyel-á névilɛ n pɩ nɛ́gbɩ́lɛ ǹ yah sɔkɔ dùkù nɛ̂nɛ, pè náh kè pɩ̃. Tɛ́ pé yɛ̃́ngbɛ̃́ tyɩ́lɛ n pɩ, tɛ́ Liyel tyɩ́ wɔ yahle. 
Krista ye pɔ Moyisi tyi kõ̀nsàhnɔ́ tyi pɩ syɩ sé yĩn tyɩ́. Névi nɛ̂-à sɛ̃ Krista yõ, tɛ̃̀ kɩ pɩ nɛ́gbɩ́lɛ Liyel yah sɔkɔ. 
Névi-á fɛ̃ n pɩ n pi nɛ́gbɩ́lɛ tyi kõ̀nsàhnɔ́ gbõ̀ yõ dùkù nɛ̂nɛ, Moyisi-á nɛ̂ wãrkɩ ké tyɩ́ sɔkɔ kè wɛ yɔ̀: «Névi nɛ̂-à tyi kõ̀nsàhnɔ́ sɔkɔ tyíyónɔ́lɛ n pɩ, tɛ̃̀ kɩ ǹ minnɛ n pɩ sépi gbõ̀ yõ.» 
Ǹtɛ névi-á n pɩ nɛ́gbɩ́lɛ Krista yõ sɛ̃́nɔ́ gbõ̀ yõ dùkù nɛ̂nɛ, kè mɛ yónɔ́lɛ képah tyɩ́ sɔkɔ nɛ: «Káh nɛ n sõ á fɔkɔ sɔkɔ nɛ: ‹Nɛ̂ kɩ dɔkɔ kyɩ yĩ̂nyõ?›» Nɛ̂ kɔ́r-á, kɩ kyɩ Krista kɔ̃ à kɩ tiki pɔ. 
Képah náh pɩ: «Nɛ̂ kɩ tiki yuku sétáh sõwãh?» Nɛ̂ kɔ́r-á, kɩ kyɩ Krista kɔ̃ à kɩ wil lékókwíl kɩ dɔkɔ pɔ. 
Ǹtɛ tir nɛ̂ mɛ tɔ̃ yónɔ́lɛ wɛ́? Kè mɛ yónɔ́lɛ nɛ: «Liyel wɛy mɛ á tnɔ̂. Kè mɛ á vyãh sɔkɔ ànɛ̂ á nɩ sɔkɔ.» Kénɛ wɛynɛ, Yesu Krista yõ sɛ̃́nɔ́ wɛy ye kélɛ, áyáh mɛ nɛ̂nɛ n yo n kɔ̃. 
Wâh kè yo á vyãhlɛ nɛ Tɛ̃̀ ǹgbɛ-á Yesu Kristalɛ, tɛ́ sɛ̃ ké yõ á nɩ sɔkɔ nɛ Liyel-á ǹ lékó yilki mɔ, á kɩ pwáhnmɔnɔ wɛ. 
Wâh sɛ̃ Liyel yõ á nɩ sɔkɔ, képah álɛ n pɩ nɛ́gbɩ́lɛ; ànɛ̂ wâh kè yo á vyãhlɛ, képah álɛ n pwah n mɔ. 
Á náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ névi nɛ̂-à sɛ̃ Krista yõ, sen náh tɛ̃̀ tɛ̃ n pi. 
Kǎh pɩ sɛ̃́, wǎhkɩ̀ náh mɛ Nsyifunɔ wrɔ́ ànɛ̂ nónó náh pɩ Nsyifunɔlɛ. Tɛ̃̀ ǹgbɛ̃ núkú ó ye mɛ pé pól tyɩ́, nɛ̂-á pèpɔrɩlɛ n pɩ no pól tyɩ́ nónó-á ǹnɛ n ye n gbo. 
Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Névi nɛ̂-à Tɛ̃̀ ǹgbɛ̃ ye gbo, tɛ̃̀ kɩ pwáhnmɔnɔ wɛ.» 
Ǹtɛ sɔ̃́ mɛ pè kɩ ye gbo ǹnɛ, nɛ́ pè náh sɛ̃ ǹ yõ? Ǹ tyɩ́ wɛ́nnoh-ńkɛ̃̂nɛ, pè kɩ sɛ̃ sɔ̃́ ǹ yõ? Pè kɩ wɛ noh sɔ̃́ ǹ tyɩ́lɛ nɛ́ nɛy náh ǹ tyɩ́ wɛ yo pé tyɩ́? 
Tɛ́ sɔ̃́ mɛ Liyel wɛy kɩ yo, nɛ́ névi náh tɛkɩ mɔ ké yo yĩnnɛ? Képah ye kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Kè mɛ nɩ̀vɩ̀nɩ̀nɛ wâh Nɩ̀vɩ̀nɩ̀ tir yópú wɛ pè pé gbáhkɩ̀ tɔkɔ n pi.» 
Ǹtɛ no pól se yɔ́ Nɩ̀vɩ̀nɩ̀ tir fɛ̃ syi? Yé yah, Liyel tɛ̃ntõ̀ syínyo-ò Yesayi lésõ mɛ píkínyahnɔlɛ n pɩ nɛ ńkɛ̃́: «Tɛ̃̀ ǹgbɛ̃, ápi-á tir nɛ̂nɛ n yo, nɛ̂ sɛ̃ ké yõ?» 
Sɛ̃́ ye Krista yõ sɛ̃́nɔ́ n wɛ Nɩ̀vɩ̀nɩ̀ tir nóhnɔ́ sɔkɔ. Tɛ́ Nɩ̀vɩ̀nɩ̀ tir mɛ̀nɛ mó, Yesu Krista tyɩ́ yónɔ́ ye. 
Ǹtɛ à; pè náh ǹ tyɩ́ yónɔ́ noh yah di? Pè kè noh kyɔ́! Kè mɛ yónɔ́ ńtɛ̃nɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Pé yĩ́npɔ̃́ mɛ n dɔkɔ sétáh yõ pól sɔkɔ, pé wɛy mɛ n yĩni n tɛ̃ kèkõyṍ tã̀ntã̀n pól tyɩ́.» 
Ǹtɛ à; Yisrayel no náh lésõ Liyel wɛy mɛ̀ syɩ́kɩ́-ńsah pɩ̃ di? Moyisi mó tɔ̃ kwãh tɛ̃ kè yo nɛ Liyel-á nɛ ńkɛ̃́: «Mé kɩ kè pɩ yé kɔ̃ yé kɩ yìkì fṍrɔɔ yɔ́ yèwannɛ n yah. Yìkì nɛ̂ névye-á lékã́m kɛ̃́ névyelɛ, mé kɩ kè pɩ yé kɔ̃ yé kɩ pé dyɔlɛ n yɔ.» 
Yesayi ǹ nɩ syãh kè yo ńtɛ̃nɛ nɛ Liyel-á nɛ ńkɛ̃́: «Névye nónó náh sõ̀ mɛ ńmɔlɛ n yah n kɔ̃, pépi ye wɛ ńnɛ. Névye nónó náh sõ̀ mɛ kwâh yɔ́ ńtɛ̃nɛ n yah n kɔ̃ ń tyɩ́, mé ń gblɔ̌ynɛ pépi wɛ̃kɩ.» 
Ǹtɛ Yisrayel no tɛ̃̀ wɔ tyɩ́ sɔkɔ, Yesayi nɛ Liyel-á nɛ ńkɛ̃́: «Gbĩ́mɛgbĩ́ mè n nɛ n mɛ ń gbã̀n yɩkɩ mɔ pè syɩkɩ n tɛ̃, pépi nónó-á nòhnɩ̀-ǹkɛ̃̂ ànɛ̂ kèlékɛ̃ névyelɛ.»
