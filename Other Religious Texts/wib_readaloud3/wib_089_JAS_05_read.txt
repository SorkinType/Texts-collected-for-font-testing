Nsyakɩ.
5.
Núkúnúkú, yépi nónó-á mɛ tahbɩlɛ, yé noh ń tyɩ́ yɔ̀: Yèvyã̀hnpèl nónó-á kɩ pɔ yé yĩni n tɛ̃ n pi, yé nɛ n kah n gbo sépi yĩnnɛ. 
Yé tahrɩ mɛ n pyãh, tɛ́ sóhrɩ́ mɛ yé flɔlɛ n yõ. 
Yé sã́n ànɛ̂ yé pɔ́nɔ mɛ n pɩ syɩ̃́mɩ́nɛ. Kénɛ syɩ̃́mɩ́ ye kɩ pɔ wilki mɔ yé gbõ̀nkɔ̀kɔ̀lɛ, tɛ́ kɩ yé wil gbɛ̃́nsê yõ, nyàh tyɩ́ kɔ̃lɛ. Kèkõyṍ tyɛ gbĩ́ mɛ̀ fyé nɔ́ntɛnɔ sɔkɔ, yépi mɛ tahrɩlɛ n yah n tuke n mɔ n sah yé gblɔ̌y tyɩ́ kɛ̀! 
Tɛ́, tõ̀npɩbɩ nónó ńtɛ̃-á tõ̀lɛ n pɩ yé swɛ̃y sɔkɔ, yè n yahle pé pɔ́lɛ pé kɔ̃́nɔ́lɛ. Pépi Liyel yéngbonɔ yĩ́npɔ̃́ ye n de Liyel níkí sɔkɔ, ǹmɔ nɛ̂-á kèkõyṍ Tɛ̃̀ ǹgbɛ̃nɛ. 
Yépi mɛ yõ̀tɔ̃rɩ́ yélɛ n yõ kèkõyṍ mɛ̀ sɔkɔ, à yé dyɔ yĩ́ĩ́ tyilɛ n pɩ, mɛ gbilki tɛ̃, á kɩ nɛ ko ko nyeh sɔ̃́. 
Yé lékàhlɛ névye kɔ̃, nónó-á pè náh tyítúkù pɩ. Pè mó pé gblɔ̌y fĩn mɔ yé kɔ̃, yé pè ko mɔ. 
Ń yṹnpyé, yé syɩ́kɩ́ntɛ̃̀nɔ́lɛ n pɩ, fɔ́ɔ́ kè pɔ nɔ Tɛ̃̀ ǹgbɛ̃ syɩ́npɔnɔ tyɩ́. Yé yah sètõlɛ kɛ̀, à ńkɛ̃́nɛ sèmukɔ pèpɛy wɛ́ntɔkɔnɔ syɩkɩ n tɛ̃, à tɔkɔ pṹgbro pɩ́nɔ́ tyɩ́, pópó ǹ pɔ n syɩ pṹtɛ́lɔ́ pɩ́nɔ́ tyɩ́. 
Tɛ̃̀ ǹgbɛ̃ syɩ́npɔnɔ mɛ yúkɛy; képah ye, yépi tɔ sṍré mɔ yĩ́ĩ́ yé gblɔ̌y sɔkɔ, yè syɩ́kɩ́ntɛ̃̀nɔ́ pɩ. 
Ń yṹnpyé, yé káh nɛ n tɔkɔ wɛ̃nɛ n kɔ̃; képah sɔkɔ, Liyel náh yé tuke n pi. Yé yah, Tɛ̃̀ ǹgbɛ̃ nɛ̂-á kɩ pɔ á tuke n pi, à mɛ yúkɛy. 
Ń yṹnpyé, Liyel tɛ̃ntõ̀ syínyopu-á lésõ ǹ wɛynɛ n yo n kɔ̃ yèvyãhrɩ ànɛ̂ syɩ́kɩ́ntɛ̃̀nɔ́ sɔkɔ sɔ̃́, yé yɩ̃nɛ yè nɛ n pɩ pépi tyɩ́ kɔ̃lɛ. 
Yé yah, nónó-á gbah tɛ̃ à kyɩ syɩ ké yĩ̀n tyɩ́, ǎ pépilɛ n sõ. Yé Nsyɔbɩ gbáhntɛ̃nɔ tyɩ́ noh ǹ yèvyãhrɩ gbĩ́ sɔkɔ kɛ̀. Tɛ́ tɔ̃ kè wɛ, Tɛ̃̀ ǹgbɛ̃-á tyí-ǹgbnɔ nónó pɩ ǹ tyɩ́ ké tyɛ́-ńsah sɔkɔ. Yé náh wɛ, Tɛ̃̀ ǹgbɛ̃ tɛ̃̀ wɔ névi yãm yáhnɔ́ ànɛ̂ ǹ pèpɔrɩ mɛ gblò kègbɩ tɛ̃̀nɛ. 
Ǹtɛ, ń yṹnpyé, sépi pól sɔkɔ, yé káh yélɛ n wo Liyellɛ, képah náh pɩ sétáhlɛ. Ànɛ̂ yé káh yélɛ n wo kwâh yɔ́ ńtɛ̃nɛ. Yé yɛ yé àwe nɛ mɛ ké gbɩ yõ, yé ɔ̃́ɔ̃ mó nɛ mɛ ké gbɩ yõ. Képah sɔkɔ, Liyel náh lékàhlɛ yé kɔ̃ n pi. 
Tɛ́ yèvyãhrɩ-à mɛ yépi sɔkɔ nɛy nɛ̂ yõ, tɛ̃̀ Liyel ni. Nɩ̀vɩ̀nɩ̀-à se mó mɛ nɛ̂ yõ, tɛ̃̀ Liyel yĩn gbilki yɩ̀gbɩ̀nɩ̀ sɔkɔ. 
Yé sɔkɔ nɛy nɛ̂-à mɛ n yɔ, tɛ̃̀ Liyel nípú yah tɛ̃́pú ye, pépi kɩ pɔ nɩrɩ tuke ǹ tyɩ́ Tɛ̃̀ ǹgbɛ̃ Yesu yĩn yõ, tɛ́ kɩ Liyel ni ǹ tyɩ́. 
Tɛ́ nírí mɛ̀-à pɩ Liyel yõ sɛ̃́nɔ́ sɔkɔ, yɔ́nwó mɛ̀ kɩ pwáhnmɔnɔ wɛ, ànɛ̂ Tɛ̃̀ ǹgbɛ̃ Liyel kɩ à dɛ mɔ. Tɛ́ wàh se mɛ tyípêl pɩ, à kɩ sè yɔ̃ mɔ à kɔ̃. 
Képah ye, yâh mɛ dɛ́nmɔnɔ tyɩ́, yé yé tyípêl wil n syi, yè mó Liyellɛ n ni wɛ̃nɛ n kɔ̃. Tǎhkɩ̀ mɛ yĩ́ĩ́ nɛ́gbɩ́ nírí sɔkɔ dɛ́. 
Yé náh Yeli wɛ-à: Névi ye lésõ ǹnɛ ápi tyɩ́ kɔ̃lɛ. Ǹtɛ, à mɛ nírí gbah pɩ nɛ, pũ̀ káh tɔ̃ pɩ. Pũ̀ náh mó ńtyɩ̃́nɔ pɩ sétáh yõ, fɔ́ɔ́ à pɔ nɔ ye tɔ̃́nɔ́ ànɛ̂ tɔ́nɩ́ tyɩ́. 
Képah náh, à tɔ̃ ǹ syɩ Liyel ni nɛ, pũ̀ pɩ. Pũ̀ mɛ pɩ, sèmukɔ pɛkɩ. 
Ń yṹnpyé, yé sɔkɔ nɛy yɔ́-à mɔ pu gbɩ wɛ̃̀kɩ̀lɛ, tɛ́ nɛ̂ mɛ à tɛ̃ pɔ ké yõ, tɛ̃̀ pɩ plɛ. 
Yé pɩ̃ nɛ, névi nɛ̂-à tyípêl pɩ́-ò vi mɔ pèpɛy pɩ́nɔ́ sɔkɔ, tɛ̃̀ ǹ min pwah mɔ ǹ kúntɛnɔ náh yèvyãhrɩ yah. Ànɛ̂ ǹ tyípêl náhnáh nɔ́pi kɩ se tɔ̃ yɔ̃ mɔ à kɔ̃.
