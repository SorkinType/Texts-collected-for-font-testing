Kolɔsɩ.
3.
Yépi tɔ lékyɩ̂-á fyɔ̀ yilki mɔ ànɛ̂ Kristalɛ wɛ̃ tyɩ́, yé Liyel fṍ yîlɛ n yah n kɔ̃, ńyãh sɔkɔ-á Kristalɛ tɛ̃̀nɔ́lɛ Liyel gbõ̀yõ̀ sõ̀lɛ. 
Yé nɛ n sõ Liyel fṍ yî tyɩ́ sɔkɔ, yé káh nɛ n sõ sétáh yõ yî tyɩ́ sɔkɔ. 
Yé náh wɛ di, yé ku tɛ́ yé min mɛ wɔ̃̀bɩ̀nsàhnɔ́lɛ Krista sɔkɔ Liyel tyɩ́. 
Ǹtɛ Krista nɛ̂-á yépi minnɛ, ǹmɔ-à sõ nɛ̂nɛ wil mɔ, yépi tɔ kɩ wil mɔ ǹnɛ wɛ̃ tyɩ́ ǹ tɔ̃́rɩ́ sɔkɔ. 
Tyi nónó-á pɩ kèkõyṍ mɛ̀ tyilɛ, yé sépi yal n mɔ, sépi nónó-á pɩ: Prɔ̀prɔ̀ pɩ́nɔ́lɛ, põ̌põ̌ pɩ́nɔ́lɛ, dékè tyi pɩ́nɔ́lɛ, dékè túkúnìnɛ, à tahlɩ yĩ́sesúkú pɩ́nɔ́ tyɩ́, képah nɛ̂-á sãh fálnɔ́ dùkù yɔ́lɛ. 
Kénɛ tyi dùkù ye n yɔrɩ n mɔ Liyel fɔ̀kɔ̀vyã̀hnɩ̀nɛ ǹ yóré yáhnlepu yõ. 
Ǹgbòlɛ sõ̀, yé min-á lésõ mɛ tyípékè yɔ̃rɩ sɔkɔ, sɛ̃́ntɛ̃̀ ye yé mɛ n pɩ. 
Ǹtɛ núkúnúkú, tyi nónó-á yé yɩ̃nɛ yè milki, sè wɛ yɔ̀: Fɔ̀kɔ̀vyã̀hnɩ̀nɛ, ànɛ̂ mâl pɩ́nɔ́lɛ, ànɛ̂ nɛ́pékè pɩ́nɔ́lɛ. Gbyɔ ànɛ̂ wɛ̀npèkè yɔ́ ńtɛ̃ káh nɛ n wil yé vyɛ̃y sɔkɔ. 
Yé káh mɩ̀nɩ̀nɛ n yo wɛ̃ tyɩ́. Yé náh wɛ di, yé yé náh vi yé min pɩ dùkù kédyâh kɔ̃ tɛ, à ké tyípɩ́nɩ̀ pól milki, 
tɛ́ mɛ pɩ dùkù késã́nnɛ n pɩ, képah nɛ̂-á Liyel mɛ n vi n mɔ gbĩ́mɛgbĩ́, sɔ̃̀nko yè pɩ ǹmɔ tyɩ́ kɔ̃lɛ, yè mó ǹnɛ n pɩ̃ yuku yahlɛ ǹmɔ nɛ̂-á mɔ yélɛ. 
Képah sɔkɔ, wǎhkɩ̀ náh tɔ̃ n wɛ n pi yìkìnì tɛ́lɔ́ ànɛ̂ Nsyifunɔ wrɔ́ sɔkɔ, kènkõbe ànɛ̂ kõ̀-ǹkɛ̃̂pú wrɔ́. Kè náh tɔ̃ mɛ á kɩ nɛ yã̀hnkõ wɔ-á yɩ̃ pɩ̃, tɛ́ ńyɔ mɛ náh yɩ̃ pɩ̃. Kè náh tɔ̃ mɛ á kɩ nɛ ńyɔlɛ yɔ̃̀-á tɛ́ ńyɔ mɛ gblɔ̌y yõ névilɛ. Nɛ̂-á mɛ n mɔ n kõ, Krista ye. Ǹmɔ ye yíyìnì póllɛ tɛ́ mɛ no pól min sɔkɔ. 
Liyel-á fyɔ̀ yépi yah wilki pɩ ǹ nolɛ, à yé pɩ yályál ǹ yah sɔkɔ, tɛ́ yé tyɩ́ mɛ nɔ ǹ tyɩ́. Képah ye, yé pɩ névye yãm yáhpú, ànɛ̂ pèpɔrɩ pɩ́púlɛ, ànɛ̂ gblɔ̌y tíkínmɔpulɛ, ànɛ̂ nɛ́pír névyelɛ, ànɛ̂ nónó-á tyilɛ n sro. 
Yé wɛ̃ sṍmnɔ́lɛ n pɩ. Ǹnɩ̂ yɔ́-à sɔ̃́ pɩ de ǹ ní-ò tyɩ́, yé yɔ̃ mɔ wɛ̃ kɔ̃. Tɛ̃̀ ǹgbɛ̃-á yɔ̃ mɔ yé kɔ̃ sɔ̃́, yépi tɔ nɛ n yɔ̃ n mɔ wɛ̃nɛ n kɔ̃ sɛ̃́. 
Ǹtɛ nɛ̂-á kal sépi póllɛ, yé tyɩ́ nɔ wɛ̃ tyɩ́. Tyɩ́nɔnɔ gbõ̀ yõ ye yé wrɔ́ kɩ mɛ kwrɔ́nmɔnɔlɛ pallɛ. 
Krista yèvɩnɩ nɛ̂ mɛ yé sɔkɔ. Liyel ye yé ye kénɛ yèvɩnɩ wɛ yĩnnɛ, sɔ̃̀nko yè pɩ wil gbɛ̃́nsê núkúlɛ. Yé pɩ pèpɔrɩ pɩ̃́púlɛ. 
Yé yɛ Krista wɛy tõ̀ pɩ yĩ́ĩ́ yé sɔkɔ. Yé wɛ̃nɛ n kwɔ, yè mó wɛ̃nɛ n vi n mɔ yɩ̃́pɩ̃̂ sɔkɔ. Yé pɩ pèpɔrɩ pɩ̃́púlɛ Liyel tyɩ́ yé nnɔ póllɛ, yè mó ǹnɛ n gbal, yè ǹ yĩn gbilki gbilki yɩ̀lɛ n gbɩ, à tahlɩ Liyel Mirki-á yɛ̀ nónó dahbɩ yé sɔkɔ. 
Yâh tyi nónónɛ n yo tɛ́ tyi nónónɛ n pɩ, yé sélɛ n pɩ Tɛ̃̀ ǹgbɛ̃ Yesu yĩn yõ, yè wɛynɛ Sú Liyellɛ n kɔ̃ ǹmɔ gbõ̀ yõ. 
Syɔ́, yé nɛ n sah yé pɛbɩ tyɩ́ yõ kǎh yɩ̃nɛ sɔ̃́ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ. 
Pɛbɩ, yé yɛ yé syɔ́ tyɩ́ nɔ yé tyɩ́, yé káh peke tɛ̃ pé yõ. 
Wɛ̃́npì, yé nɛ n sah yé dúbípú tyɩ́ yõ tyi tyi pól sɔkɔ. Képah ye Tɛ̃̀ ǹgbɛ̃ n yah n kɔ̃ yé tyɩ́. 
Dúbípú, yé káh fɔ̀kɔ̀nvyã̀hnɩ̀nɛ n mɔ yé wɛ̃́npì sɔkɔ kè kɔ̃ kè pɔ pɩ wílkúnìnɛ pé tyɩ́. 
Yɔ̃bɩ, tyi tyi pól sɔkɔ, yé nɛ n sah yé yõ̀tãm tyɩ́ yõ nónó-á mɛ sétáh mɛ̀ yõ. Yé káh tõ̀lɛ n pɩ pé tyɩ́ pàh yélɛ n kyɔmɩ gbĩ́ nɛ̂ ólɛ, á kɩ nɛ névye dyɔ yĩ́ĩ́ tɛ̃̀ sɔ̃́ yé n yah n kɔ̃ pɩ́-ńsahlɛ dɛ́. Ǹtɛ yé yé gblɔ̌y tiki mɔ kélɛ n pɩ fɔkɔ pérpérlɛ yáh sɔ̃́npɩlɛ mɛ Tɛ̃̀ ǹgbɛ̃ yah tíkílɛ n pɩ. 
Yâh sɔ̃́ tõ̀ nɛ̂ dùkùlɛ n pɩ, yé kélɛ n pɩ yé nnɔ póllɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́ sɔ̃́ yé n pɩ kélɛ, kè káh nɛ n mɛ nɛ névi tyɩ́ sɔ̃́ yé n pɩ. 
Yé pɩ̃ nɛ ńkɛ̃́ Tɛ̃̀ ǹgbɛ̃-á kɩ pɔ yépi tɔ dyah n pi yípéplɔlɛ, wǎh sépi nónó yah sah ǹ no yĩnnɛ. Yé náh wɛ di, ǹmɔ Krista ye yõ̀tɛ̃̀ kègbɩlɛ nɛ̂ tyɩ́-á yé n pɩ tõ̀lɛ. 
Ǹtɛ névi nɛ̂-à ǹ gblɔ̌y pɩ nɛ́túkùlɛ, ǹ nɛ́túkúnì tyi lédyáhnɩ́ ye à kɩ wɛ. Yé náh wɛ di, Liyel náh wǎhkɩ̀lɛ n pɩ.
