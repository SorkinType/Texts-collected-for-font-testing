Wrome tãn.
9.
Krista tyɩ́ yõ ye ńnɛ. Képah ye, máh kɩ nɛ̂ yo n pi, gbɩ ye kélɛ, mɩ̀nɩ̀ náh mɛ ké sɔkɔ. Liyel Mirki ye ké sõnɔ dahbɩ tɛ ń sɔkɔ nɛ nɛ̂-á mé kɩ n yo n pi, gbɩ-á kélɛ. 
Yèkwɔrɩ ǹgbɛ̃ mɛ ń sɔkɔ. Ń nɩ mɛ n sukɛ gbĩ́mɛgbĩ́. 
Yé náh wɛ di, máh mɛ nɛ̂ tyɩ́, ńmɔ tyɩ́ sɔ̃́ náhkɩ yɩkɩ ànɛ̂ ń wrɔ́ mó gbɛ Kristalɛ ń yṹnpyé swɔ sɔkɔ, pé pwah n mɔ yĩnnɛ, pépi nónó-á ń yìkì névyelɛ. 
Pépi ye Yisrayel nolɛ, Liyel-á nónó pɩ ǹ wɛ̃́npìlɛ. À yõ yɔ̃̀nɔ́lɛ pè kɔ̃, tɛ́ yégbékè kwrɔ mɔ pélɛ gbáyɩ̃́ náhnáhlɛ, tɛ́ tyi kõ sah pè kɔ̃, tɛ́ ǹ ni dùkùlɛ pè wɛ̃kɩ, tɛ́ ǹ vyãhlɛ pè kɔ̃. 
Abrahmɩ, ànɛ̂ Yisakɩ, ànɛ̂ Nsyakɔbɩpi ye pé náh tãnnɛ. Yìkì núkú ye pélɛ Yesulɛ wǎh pɔ sétáh yõ. Ǹmɔ ye Liyellɛ, nɛ̂-á mɛ yíyìn pól yõ. Ǹ yĩn nɛ n gbilki gbĩ́mɛgbĩ́. Amiina! 
Máh nɛ̂ yo sɔkɔ yõ̀tah, képah náh n wɛ̃kɩ nɛ Liyel náh n kɔ ǹ tyíyónɔ́ sõ̀. Ǹtɛ Yisrayel no pól náh pɩ Yisrayel no yèsyɩ̀kɩ̀nɩ̀nɛ Liyel yah sɔkɔ. 
Sɛ̃́ ó ye, Abrahmɩ wɛ̃́npì pól tɔ náh pɩ ǹ wɛ̃́npì yèsyɩ̀kɩ̀nɩ̀nɛ. Ǹ pídĩ́ Yisakɩ wɔ wɛ̃́npì ye kɩ n mɔ n kõ ǹ pétóbélɛ sɔ̃́-á Liyel yo kélɛ ǹ tyɩ́. 
Mâh nɛ mé kè yo pallɛ, wɛ̃́npì nónó-á dubi gbɔynɛ, pépi náh Liyel wɛ̃́npìlɛ. Nónó-á dubi à yɩ̃nɛ Liyel vyãh kɔ̃́nɔ́lɛ, pépi ye ǹ wɛ̃́npìlɛ. 
Yé yah, Liyel lésõ ǹ vyãhlɛ Abrahmɩ kɔ̃ nɛ ńkɛ̃́: «Gbĩ́ yɔ́ yɩ̃̌nkɔ̃ gbĩ́nɛ, mé kɩ ń syɩ pɔ á tĩ̀nnɛ, tɛ́ Sara kɩ mɛ dĩ́npir wɛ dubi.» 
Tɔ̂ɔ náh dɛ́, yé yah Wrebeka tɔ tyɩ́lɛ kɛ̀: Ápi sútó Yisakɩ mɛ̀ ye dubi à kɔ̃ dĩ́npì nɛ́pĩ̂ nímínɛ nónó-á kènìnìnɛ. 
Gbĩ́ nɛ̂-á wɛ̃́npì nɔ́pi náh tɔ̃̌nɩ̀ dubi, pè náh tyítúkúnì pɩ, pè náh tyípéplɔ tɔ pɩ. Ǹtɛ Liyel mɛ yo Wrebeka tyɩ́ nɛ: «Wɛ̃́npì nɔ́pi sɔkɔ, nɛ́nkyɩ́ tɛ̃̀ kɩ pɔ pɩ gbɛ̃̀ tɛ̃̀ yɔ̃̀lɛ.» Kè tɔ̃ mɛ wã́rkɩ́nɔ́lɛ Liyel wɛy sɛ́bɛ́y ńtɛ̃ sɔkɔ nɛ ńkɛ̃́: «Nsyakɔbɩ wɔ tyɩ́ ye nɔ ń tyɩ́, Yesayu náh» Yé náh wɛ di, képah n wɛ̃kɩ nɛ névi tyípɩ́nɔ́ náh Liyel n yah ǹ tɛ́ tɛ̃̀ yah n wilki. Névi nɛ̂-à ǹ dyɔ, tɛ̃̀ wɔ ye à n yah n wilki. 
Sɔ̃́ mɛ ápi kɩ yo yɔ̀ wɛ́? Liyel náh mɛ n kɔ gbɩ wɛ̃̀kɩ̀ yõ di? À mɛ n kɔ gbɩ wɛ̃̀kɩ̀ yõ kɛ̀. 
Yé náh wɛ di, à yo Moyisi tyɩ́ nɛ: «Mâh mɛ ké tyɩ́ nɛ mé pèpɔrɩ pɩ névi nɛ̂ tyɩ́, mě kè pɩ tɛ̃̀ tyɩ́. Tɛ́ mâh mɛ ké tyɩ́ nɛ mé névi nɛ̂ yãm yah, mě tɛ̃̀ yãm yah.» 
Kǎh pɩ sɛ̃́, képah náh n wil névi dyɔ yĩ́ĩ́ tɛ̃̀ sɔkɔ, képah náh pɩ ǹ gbáhntɛ̃nɔ sɔkɔ dɛ́. Ǹtɛ kè n wil Liyel ó tyɩ́, nɛ̂-á névi yãmnɛ n yah. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ, Liyel-á yo Yesyipti yõ̀tɛ̃̀ dĩ́ tyɩ́ nɛ: «Ń tɛ̃́nwɛnɔ wɛ̃kɩ yĩn ànɛ̂ ń yĩn wil yĩn ye kèkõyṍ pól sɔkɔ, ń kɔ̃ mé ǹ pɩ yõ̀tɛ̃̀nɛ.» 
Kǎh pɩ sɛ̃́, kàh Liyel dyɔ, ǎ ǹnɩ̂ yãm yah, tɛ́ kàh ǹ dyɔ ǎ ǹnɩ̂ pɩ kùkùtǎhkɩ̀ névilɛ. 
Gbɔ̀kénpɩ túkù kɩ fɛ̃ nɛ, kwâh nɛ̂ se pɩ tɛ́ Liyel kɩ tɔ̃ n tɔkɔ névyelɛ n kɔ̃ nɩ? Nɛ̂ se kɩ fɛ̃ ǹ dyɔ dyɔ tyi yah kõ nɩ? 
Ǹtɛ nɛ̂ tɛ̃̀ ó mɛ áwɔlɛ, tɛ́ á kɩ pɔ Liyellɛ n kɔ̃lɩ? Pĩ́n se kɩ fɛ̃ ké mɔ́-ò piki yah nɛ, kwâh nɛ̂ se pɩ tɛ́ à pé mɔ sɛ̃́ nɩ? 
Pní mɔ́-ò tyɩ́ náh ǹ fãhrɩlɛ di? À kɩ fɛ̃ tyígbɩ́ pɩ pɩ pĩ́n mɔ kélɛ, tɛ́ kɩ tɔ̃ fɛ̃ ńyã̌n tyi pɩ pɩ pĩ́n mɔ kénɛ fãhrɩ ólɛ. 
Liyel mɛ ké tyɩ́ nɛ pé pé fɔ̀kɔ̀vyã̀hnɩ̀ wilki mɔ, pé mó pé tǎhkɩ̀ wɛ̃kɩ. Tɛ́ wǎh mó tɔ̃ sõ̀ yɩ̃nɛ à ǹ fɔkɔ vyãh névye nónónɛ, à sṍmnɔ́ pɩ tɛ́ ǹ gblɔ̌y yɛ náhnáh pé tyɩ́. 
Ǹtɛ ǹ tɔ̃́rɩ́-á mɛ gblò dùkù nɛ̂nɛ, à sõ̀ mɛ ké tyɩ́ nɛ pé kélɛ no tɛ́lɔ́ wɛ̃kɩ, wǎh pépi nónó yãm yah tɛ́ pè yah wilki sah ǹ tɔ̃́rɩ́ sɔkɔ de yĩnnɛ. 
Tɛ́ kénɛ névyelɛ, ápi ye. À náh á yah wilki Nsyifunɔ ó sɔkɔ dɛ́, ǹtɛ à túkù yah wilki yìkìnì tɛ́lɔ́ sɔkɔ, 
sɔ̃́-á Liyel kè yo sah Wose sɛ́bɛ́y sɔkɔ nɛ: «Névye nónó náh sõ̀ mɛ ńmɔ nolɛ, mé kɩ pélɛ n ye ń nolɛ. Yìkì nɛ tyɩ́ náh sõ̀ nɔ ń tyɩ́, mé kɩ kélɛ n ye ń dyɔ yĩ́ĩ́ yìkìlɛ. 
Tɛ́ ńyãh sɔkɔ-á lésõ kè yo pé tyɩ́ nɛ ńmɔ no náh pélɛ, nɛ́npɔ́ mɛ̀ ó sɔkɔ ye tɔ̃ pè kɩ ye pélɛ Min nɛ́tɛ̃̂ Liyel wɛ̃́npìlɛ.» 
Yesayi tɔ mó yo Yisrayel no tyɩ́ sɔkɔ nɛ ńkɛ̃́: «Yisrayel névye-à sɔ̃́ mɛ dò pnɛ yɔ gbɔ̀pɔ sèmìyã́hrɩ́ sɔ̃́, kwéy ó ye kɩ n pwah n pi pé sɔkɔ. 
Yé náh wɛ di, TƐ̃̀ ǸGBƐ̃ Liyel-á tyi nónó yo, à kɩ sè pɩ dal mɔ súú sétáh yõ.» 
Tɛ́ Yesayi tɔ̃ lésõ kè yo sah nɛ ńkɛ̃́: «Tyi pól tyɩ́ tɛ̃́nwɛ-ò TƐ̃̀ ǸGBƐ̃ Liyel-à sõ̀ mɛ á túkù sàh-ǹkɛ̃̂nɛ min sɔkɔ, á duku náhkɩ to Sodɔmɩ ànɛ̂ Gbomɔrɩ névye tyɩ́ kɔ̃lɛ.» 
Sɔ̃́ mɛ ápi kɩ yo yɔ̀ wɛ́? Névye nónó náh pɩ Nsyifunɔlɛ, pè pɩ nɛ́gblɔ́lɛ Liyel yah ǹ yõ sɛ̃́nɔ́ gbõ̀ yõ, pépi nónó-á pè náh sõ̀ mɛ kélɛ n yah n kɔ̃. 
Tɛ́ Yisrayel no nónó wɔ-á pyě sõ̀ mɛ n yah n kɔ̃ nɛ pé pɩ nɛ́gblɔ́lɛ Liyel yah Moyisi tyi kõ̀nsàhnɔ́ wã̀l kɔ́lɔ́ gbõ̀ yõ, pè náh wɛ pɩ nɛ́gblɔ́lɛ képah gbõ̀ yõ. 
Sɔ̃́ pɩ tɛ́ pé náh wɛ pɩ nɛ́gblɔ́lɛ wɛ́? Pè náh sõ̀ mɛ kélɛ n yah n kɔ̃ Liyel yõ sɛ̃́nɔ́ gbõ̀ yõ, pè sõ̀ wɛ nɛ ńkɛ̃́ pé tyípɩ́nɩ̀-á kɩ fɛ̃ pé pɩ nɛ́gblɔ́lɛ Liyel yah képah ye. Pé gbáhkɩ̀ ye pyě pè mɩ pé wilki n mɔ mɔ dáhkɩ̀ tyɩ́ yɔ̀ sɔ̃́. 
Kénɛ dáhkɩ̀lɛ, Yesu Krista ye. Képah tyɩ́ ye wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Yé yah, mé kɩ gbáhkɩ̀ mɩ mɩ dáhkɩ̀ yɔ́ sah Siyõ kwil sɔkɔ. Névye wilki n mɔ mɔ tɛ́dáhkɩ̀ yɔ́ ye kélɛ. Ǹtɛ névi nɛ̂-à ǹ gbõ̀ sah ké yõ, sen náh tɛ̃̀ tɛ̃ n pi.»
