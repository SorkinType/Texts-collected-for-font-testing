1 Tesaloniki tãn.
2.
Á yṹnpyé, yépi ńtɛ̃ kè pɩ̃ yĩ́ĩ́ nɛ á pɔ́nɔ́ náh pɩ nɔ̀nɔ́lɛ yé tĩ̀nnɛ. 
Yé kè pɩ̃ tɔlɛ nɛ áyáh yèvyãhrɩ wɛ sɔkɔ Filipi kwil sɔkɔ, ńyãh sɔkɔ-á pè fṍhnɔ́ sah á yõ tɛ́ Liyel mɛ nɩ̀sãh mɔ á sɔkɔ á kɔ̃ á mɛ fɛ̃ pɔ ǹ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo yépi tɔ tyɩ́, gbĩ́ nɛ̂-á névye sõ̀ mɛ álɛ n fwo. 
Tyi nónó-á á n yo névye tyɩ́, á náh mɛ pílkínɔ́ sɔkɔ, yĩ́nvǐ wɛlɔ náh sélɛ, vìntɛ̃nɔ tɔ náh à n vi n tɛ̃ pélɛ. 
Liyel ye kè wɛ nɛ ápi-á kɩ fɛ̃ Nɩ̀vɩ̀nɩ̀ tir yo, tɛ́ mɛ ké tyɩ́ dahbɩ ápi gbõ̀ á kɔ̃ á kélɛ n yo ké yah yõ. Névye dyɔ yĩ́ĩ́ tɛ̃̀ náh ápi n yah n kɔ̃ pɩ́-ńsahlɛ dɛ́, Liyel dyɔ yĩ́ĩ́ tɛ̃̀ ye ǹmɔ nɛ̂-á ápi sõnɔlɛ n pnɛ. 
Yépi ńtɛ̃ pɩ̃́nɔ́ mɛ ké sɔkɔ nɛ wɛlɔ nónó-á á yo yé tyɩ́, yé vi n tɛ̃ tɛ̃ wɛlɔ náh sélɛ. Liyel ye á tàrfyɔ̀lɛ nɛ á náh mɛ n yah n kɔ̃ yé yṍ-ńsahlɛ. 
Ápi náh mɛ n yah n kɔ̃ nɛ ǹnɩ̂ yɔ́ ńtɛ̃ álɛ n gbilki. Kàh sɔ̃́ yépilɛ ò, kàh sɔ̃́ no tɛ́lɔlɛ ò, á náh mɛ n yah n kɔ̃ nɛ ǹnɩ̂ yɔ́ ńtɛ̃ álɛ n gbilki. 
Tɛ́ ápi kɩ mó tɔ̃ sõ̀ fɛ̃ á tɔ̃́rɩ́ sah yé yõ áyáh sɔ̃́npɩ́lɛ pɩ Yesu Krista tɛ̃ntõ̀ névyelɛ. Ǹtɛ á yé tɛ̃ mármár pídúbísè-á ǹ pyelɛ n tɔ̃ n sah n tɛ̃ sɔ̃́. 
Yé tyɩ́ sõ̀ nɔ yĩ́ĩ́ ápi tyɩ́, képah ye á mɛ Nɩ̀vɩ̀nɩ̀ tir yo yé tyɩ́ nɛ̂-á n wil Liyel tyɩ́. Tɛ́ kè náh yĩn képah ó tyɩ́ dɛ́, á á min ńtɛ̃ mɔ kũ kɔ̃ yépi yĩnnɛ yé tyɩ́ táhnɔ́ tɛ̃̀ yĩnnɛ á tyɩ́. 
Á yṹnpyé, áyáh yèvyãhrɩ ànɛ̂ kwlɔ́nɔ́ nónó pɩ, yé dyɔ mɛ n kwnɛ sélɛ kyɔ́. Áyáh lésõ mɛ yé tĩ̀nnɛ à Nɩ̀vɩ̀nɩ̀ tirlɛ n yo n kɔ̃ yé tyɩ́, á lésõ mɛ tõ̀ tɔlɛ n pɩ lékã̂h ké gbĩ̀ntɔ̃̀ à yĩn á gblɔ̌y yĩ́nnɔ́lɛ, sɔ̃̀nko à káh n pɩ n tɛ̃ yṍtɔ́kɔ́bɩ́lɛ yé sɔkɔ yɔ́ ńtɛ̃ tyɩ́. 
Yépi ńtɛ̃ ye ápi tàrfyɔ̀lɛ nɛ, áyáh tyi nónó pól pɩ yépi Yesu yõ sɛ̃́pú tyɩ́, sé pól-á mɛ plɛ, tɛ́ mɛ gbɩ wɛ̃̀kɩ̀ yõ wɛ́nyo-ńsah kɛ̃́nɛ; Liyel tɔ ye á tàrfyɔ̀lɛ. 
Yé pɩ̃ tɔlɛ nɛ áyáh yé tɛ̃ plɛ, sɔ̃́-á súbí pé wɛ̃́npìlɛ n tɛ̃ plɛ. 
Á vìnmɔnɔlɛ yé kɔ̃, tɛ́ sṍré mɔ yé sɔkɔ, tɛ́ kè gbah yo yé tyɩ́ nɛ yé yɛ yé kɔ dùkù nɛ n mɛ Liyel dyɔ yĩ́ĩ́, ǹmɔ nɛ̂-á ye yélɛ ǹ tɔ̃́rɩ́ sɔkɔ de yĩnnɛ kɩ ké nɔlɩ wɛ yõ. 
Yáh Liyel wɛy nɛ̂ noh ápi tyɩ́, yé náh kè pɩ névi vyãh wɛynɛ. Yé kè pɩ Liyel wɛynɛ. Tɛ́ sɛ̃́ ńtɛ̃ ye kélɛ kyɔ́. Kǎh pɩ sɛ̃́, kè mɛ tõ̀lɛ n pɩ yé sɔkɔ, yépi nónó-á Yesu yõ sɛ̃́púlɛ. Képah ye, á n wɛkɩ Liyel tyɩ́ gbĩ́mɛgbĩ́ ńtɔ̃. 
Ń yṹnpyé, Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ nónó-á mɛ Liyel tyɩ́ yõ Nsyude mara sɔkɔ, tir nɛ̂-á nɔ pélɛ, sɛ̃́nkɔ̃ ye nɔ yépi tɔlɛ. Yé náh wɛ di, yé sú fṍ névye-á yé kwlɔ sɔ̃́, sɛ̃́ ye Nsyifunɔ kwlɔ pépi tɔlɛ. 
Nsyifunɔ ye ko mɔ Tɛ̃̀ ǹgbɛ̃ Yesu ànɛ̂ Liyel tɛ̃ntõ̀ syínyopulɛ. Pè fṍhnɔ́ sah ápi tɔ yõ. Pé tyɩ́ náh nɔ Liyel tyɩ́. No pól sépɛ́bɩ́ ye pélɛ. 
Pè mɛ ké tyɩ́ nɛ pé ápi yah kõ á káh Nɩ̀vɩ̀nɩ̀ tir yo yìkìnì tɛ́lɔ́ névye tyɩ́ pé tɔ kɔ̃ pè wɛ n pwah. Kǎh pɩ sɛ̃́, dòhnɩ̀nɔ́ ó ye pè mɛ n pɩ n yuku pé tyípêl yõ. Ǹtɛ ké syì-ǹsah, Liyel fṍhnɔ́ sah pé yõ. 
Ń yṹnpyé, á yal yé tnɔ̂ wrɔ́ kwéy mɛ̀ sɔkɔ, á yĩ́ tɛ̃̀ wɔ náh mɛ tɛ̃̀nɔ́lɛ yé yõ ó, ǹtɛ á mɛ yé tyɩ́lɛ n sõ gbĩ́mɛgbĩ́. Yé wɛ dékè mɛ á kah tɛ̃, képah ye á mɛ gbah tɛ̃ nɛ á yé wɛ. 
Á lésõ mɛ ké tyɩ́ nɛ á á syɩ n pɔ yé tĩ̀nnɛ, tɛ́ gbáhyɩ̃́ náhnáh ńtɛ̃ ye ńmɔ Pole yah kɔ̃ kélɛ tɛ́ sétã̀n yõ̀tɛ̃̀ á yah kõ. 
Ápi Tɛ̃̀ ǹgbɛ̃ Yesu syɩ n pɔ sõlɛ, nónó gbõ̀ yõ mɛ á yáh-ńsah náh n tyɛ n pi, ànɛ̂ á nɩ kɩ vɩ, ànɛ̂ á yõ kɩ yɔ̃? Yépi ye. 
Kègbɩ yõ, yépi yĩn ye gbílkínɔ́ kɩ sah ápi yõ, tɛ́ nɩ̀vɩ̀nɩ̀ kɩ mɛ á sɔkɔ.
