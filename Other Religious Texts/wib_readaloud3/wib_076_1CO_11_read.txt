Korɛ̃nti tãn.
11.
Yé ńmɔ pɩ dùkùlɛ n yah n pɩ, sɔ̃́-á mé Krista pɩ dùkùlɛ n yah n pɩ. 
Mé yélɛ n sõ náhnáh, yáh sɔ̃́npɩ́lɛ yé dyɔ sah n kwnɛ ńmɔ tyɩ́lɛ gbĩ́mɛgbĩ́, tɛ́ se tɔ̃ mɛ n kɔ tyi yónɔ́ wã̀l yõ, máh kwɔ́nɔ́ wɛ sépi nónó yõ à pɔ yé kwɔ. 
Ǹtɛ mé mɛ ké tyɩ́ nɛ yé kè pɩ̃ tɔlɛ nɛ ńkɛ̃́, sê yõ̀tɛ̃̀-á dĩ́nɛ, díbí pól yõ̀tɛ̃̀ mó Yesu Kristalɛ, Yesu Krista wɔ yõ̀tɛ̃̀ mó Liyellɛ. 
Képah ye, dĩ́ nɛ̂-à yṍkúrí kwĩnki sah tɛ́ nírílɛ n pɩ, képah náh pɩ tɛ́ mɛ Liyel tɛ̃ntõ̀lɛ n syi n yo, ǹ yõ̀tɛ̃̀ ye à n yilɔ. 
Ǹtɛ sê nɛ̂-à ǹ yõ dɛ mɔ sah tɛ́ nírílɛ n pɩ, képah náh pɩ à Liyel tɛ̃ntõ̀lɛ n syi n yo, ǹ yõ̀tɛ̃̀ ye à n yilɔ. Kè núkú mɛ á kɩ nɛ ǹ yõ sɔ̃́ à kye mɔ. 
Sê nɛ̂-à nɛ mɔ̀nsɔr tɛ̃ n pi ńkɛ̃̂nɛ, à ǹ yõ kye. Ǹtɛ sen-à nɛ sê tyɩ́ ǹ yõ kyénɔ́lɛ, képah náh pɩ ké kṍnɔ́lɛ, à káh kè dɛ n mɔ n sah kɛ̀. 
Dĩ́nɛ, Liyel kètɔ̀ ye ǹnɛ, ànɛ̂ à Liyel yõ yɔ̃̀nɔ́lɛ n wɛ̃kɩ. Képah ye, à náh yɩ̃nɛ yṍkúrí kwĩ́nkínsàhnɔ́lɛ. Ǹtɛ sê wɔ dĩ́ yõ yɔ̃̀nɔ́lɛ n wɛ̃kɩ. 
Kèkõyṍ mɔ gbĩ́nɛ, dĩ́ náh wil sê sɔkɔ, sê wɔ ye wil dĩ́ sɔkɔ. 
Dĩ́ náh mɔ sê yĩnnɛ dɛ́, ǹtɛ sê mɔ dĩ́ yĩnnɛ. 
Képah ye, yĩ̂nyõ tõ̀npɩbɩ yĩnnɛ, kwâh yɔ́ yɩ̃nɛ kè nɛ n mɛ sê yõ sɔkɔ, nɛ̂-á kɩ ǹ yĩ́n-ńsahlɛ n wɛ̃kɩ. 
Ǹtɛ sépinɔ pól sɔkɔ, ápi nónó tɛ̃̀-á mɛ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀lɛ n kɔ, á ye pɩ̃ nɛ ńkɛ̃́, dĩ́ yétõ̂-á mɛ sê tyɩ́, sê tɔ yétõ̂ mó mɛ dĩ́ tyɩ́. 
Yé náh wɛ di, sê wil dĩ́ sɔkɔ, tɛ́ sê ye mó n dubi dĩ́nɛ. Ǹtɛ Liyel ye pɩ sépi póllɛ sɛ̃́. 
Yépi ńtɛ̃ kè sõ n yah kɛ̀; ké yah se sê Liyellɛ nɛ n ni ǹ yõlɛ dɛ́nmɔnsàhnɔ́lɛ? 
À yɩ̃nɛ kégbɔ́ dùkùlɛ, yé náh pɩ̃ nɛ ńkɛ̃́ dĩ́-à yõ̀yùkù mɔ, sen-á kélɛ ǹ tyɩ́ di? 
Ǹtɛ yõ̀yùkù mɔ́nɔ́lɛ, yõ yɔ̃̀nɔ́ ye kélɛ sê tɛ̃̀ wɔ tyɩ́. Liyel kélɛ à kɔ̃, ǹ yõ yah vike yĩnnɛ. 
Névi nɛ̂ tɛ̃̀ wɔ-à kɩ tɔ̃ mɛ n yah n kɔ̃ yékã̂b pɩ́-ńsahlɛ yĩ mɛ̀ tyɩ́ yõ, tɛ̃̀ pɩ̃ nɛ ńkɛ̃́, sɛ̃́-á ápi n pɩ kélɛ à tahlɩ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ pól tɔ tyɩ́. 
Tɛ́ yah, máh tyi nónó tɔ yo n pi yé tyɩ́ núkúnúkú, sè wɛ yɔ̀: Yé wɛ̃ tùkènmɔnɔnɔ mɛ yényêhlɛ n pɩ n kal yé tyɩ́ pèpɛynɛ. Képah ye, mé náh fɛ̃ yé sõ n pi képah sɔkɔ lɔ́. 
Ké gbennɛ, pè nɛ yâh wɛ̃ tuke mɔ, gbɛ́kɩ́nɔ́ n nɛ n mɛ yé wrɔ́. Tɛ́ ńmɔ sɛ̃́nɔ́ mɛ ké yõ nɛ gbɩ-á mɛ pé wɛy sɔkɔ. 
Ǹtɛ gbɛ́kɩ́nɔ́ dùkù yɔ́ tɛ̃̀ wɔ yɩ̃nɛ kè wɛ yé wrɔ́ sɔkɔ, á kɔ̃ à Yesu yõ sɛ̃́pú kègblɔ pɩ̃ n wah yé sɔkɔ. 
Ǹtɛ gbɛ́kɩ́nɔ́ nɛ̂ tyɩ́-á mé n yo yé wrɔ́ sɔkɔ, yâh wɛ̃ tuke mɔ Tɛ̃̀ ǹgbɛ̃ yõke yõ gbĩ́nɛ, yé nɛ́núkù núkú pól n núkú n yɩ̃kɩ n kyɩ yé yɛ̃́ngbɛ̃́ yõkenɔ tɔkɔ n yõ. Képah sɔkɔ, túkù n wũ kwɛkɩlɛ, tɛ́ túkù n nɛ n mɛ sètɛ̃̀nɩ̀ sɔkɔ. Kàh pɩ sɛ̃́, Tɛ̃̀ ǹgbɛ̃ yõke tɛ̃̀ wɔ náh tɔ̃ kélɛ yé n yõ légbĩ́nɛ dɛ́. 
Gbên náh mɛ yé tyɩ́ nónó sɔkɔ-á yé kɩ fɛ̃ yõkelɛ n yõ, tɛ́ kɩ n wɔ di? Tĩ̂, Yesu yõ sɛ̃́pú tĩ̀nkì se yé n yilɔ, tɛ́ mɛ ké tyɩ́ nɛ yé sennɛ kwâh kɛ̃́pú tɛ̃? Sɔ̃́ mɛ mé kɩ yo yé tyɩ́ yɔ̀ wɛ́? Mé se yɩ̃nɛ mè yé sõ? Póllɛ, mé náh yé sõ n pi képah sɔkɔ lɔ́. 
Yé yah, Tɛ̃̀ ǹgbɛ̃-á nɛ̂nɛ ńmɔ kwɔ tɛ́ mé mɛ kélɛ yé kwɔ, kè wɛ yɔ̀: Tɛ̃̀ ǹgbɛ̃ Yesu-á kɩ sõ̀ n dahbɩ n pi gbõ̀ sɔkɔ sõ nɛ̂ lékã̂hnɛ, lésõlɛ à búrn tɔkɔ, 
tɛ́ wɛkɩ Liyel tyɩ́, tɛ́ mɛ kè kwɔkɩ dibi pé tyɩ́, tɛ́ mɛ n yo pé tyɩ́ nɛ: «Ńyɔlɛ, ńmɔ wil ye, nɛ̂-á kɩ kɔ̃ yépi yĩnnɛ. Yé kélɛ n pɩ, yè yé lékã́m sah ńmɔ tyɩ́lɛ.» 
Yõke yṍntɛnɔ náh, à tɔ̃ wóh tɔkɔ sɛ̃́ntɛ̃̀ ólɛ dìvɛ̃nɛ ké sɔkɔ, tɛ́ yo pé tyɩ́ nɛ: «Ńyɔlɛ, ńmɔ tṍ ye, nɛ̂-á kɩ kwɛ yépi yĩnnɛ, kɩ pɩ yégbékè kwrɔ́nmɔnɔ késã́nnɛ Liyel ànɛ̂ névye wrɔ́. Yé kélɛ n pɩ, yè yé lékã́m sah ńmɔ tyɩ́lɛ, gbĩ́mɛgbĩ́ yâh kélɛ n wɔ.» 
Yé yah, gbĩ́mɛgbĩ́ yâh búrn mɛ̀nɛ n tah tɛ́ dìvɛ̃ mɛ̀nɛ n wɔ gbĩ́ nɛ̂nɛ, Tɛ̃̀ ǹgbɛ̃ kúnɔ́ tyɩ́ ye yé mɛ n yo, fɔ́ɔ́ kɩ pɔ syɩ ǹ syɩ́npɔnɔ tyɩ́. 
Képah ye, névi nɛ̂-à búrn mɛ̀nɛ n tah tɛ́ dìvɛ̃ mɛ̀nɛ n wɔ, tɛ́ tɛ̃̀ pɩ dùkù náh mɛ plɛ, tɛ̃̀ mɛ pèkèlɛ n pɩ Tɛ̃̀ ǹgbɛ̃ wil ànɛ̂ ǹ tṍ tyɩ́ pɩ́ndenɔ sɔkɔ. 
Nɛ́núkù núkú pól yɩ̃nɛ pé pé gblɔ̌y dùkù kyɔmɩ yah pallɛ, pè mó pyě búrn mɛ̀ tah, pè mó dìvɛ̃ mɛ̀ wɔ. 
Yé náh wɛ à, névi nɛ̂-à búrn mɛ̀ tah tɛ́ dìvɛ̃ mɛ̀ wɔ, tɛ́ tɛ̃̀ náh sè mɔ kõ Tɛ̃̀ ǹgbɛ̃ willɛ, tɛ̃̀ mɛ pèkèlɛ n yɔrɩ n mɔ ǹ gblɔ̌y yõ. 
Képah yĩn ye yámpú ànɛ̂ nɛ́fukulɔ́lɛ pípí yé wrɔ́, tɛ́ yé náhnáh ńtɛ̃ mɛ ku. 
Áyâh mɛ n sõ n yah pallɛ á gblɔ̌y tyɩ́ sɔkɔ, Liyel tukey náh n syɩkɩ n pi á yõ. 
Ǹtɛ Tɛ̃̀ ǹgbɛ̃ náh mɛ ké tyɩ́ nɛ ápi tɔ tyɩ́ kyɩ yɩkɩ wɛ̃ tyɩ́ ànɛ̂ kèkõyṍ névyelɛ, képah ye à kɔ̃ à ápilɛ n fõh á kwɔ yĩnnɛ. 
Képah ye ń yṹnpyé, yâh tuke mɔ Tɛ̃̀ ǹgbɛ̃ yõke yõ yĩnnɛ, yé pól wɛ̃ syɩkɩ. 
Kwɛkɩ-à mɛ nɛ̂ tɛ̃, tɛ̃̀ yõ ǹ tĩ̀nnɛ. Kàh pɩ sɛ̃́, yé náh Tɛ̃̀ ǹgbɛ̃ pèkè yɔrɩ n mɔ n pi yé gblɔ̌y yõ yé wɛ̃ tùkènmɔnɔnɔ sɔkɔ. Tyi tɛ́lɔ́ nónó-á wũ, mâh pɔ pɔ yé tĩ̀nnɛ gbĩ́ nɛ̂nɛ, mé kɩ sépi tyɩ́ yah wah.
