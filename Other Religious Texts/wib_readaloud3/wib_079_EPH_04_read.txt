Yefɛsɩ tãn.
4.
Ńmɔ nɛ̂-á mɛ wɔ̀kɔ̀npèkè sɔkɔ Tɛ̃̀ ǹgbɛ̃ tõ̀ yĩnnɛ, mé yélɛ n ni yépi nónó-á Liyel ye, nɛ yé yé minnɛ n pɩ à yɩ̃nɛ ǹ yénɔ́ mɛ̀nɛ. 
Yé pɩ gblɔ̌y tíkínmɔpulɛ, ànɛ̂ nɛ́pír névyelɛ, ànɛ̂ sṍmnɔ́ pɩ́púlɛ. Yé wɛ̃nɛ n n tɛ̃ tyɩ́nɔnɔ sɔkɔ. 
Yáh mɛ yèvɩ̀nɩ̀ nɛ̂ sɔkɔ wɛ̃ tyɩ́, képah gbõ̀ yõ, yé gbah n tɛ̃ yé wrɔ́ kwrɔ́nmɔnɔ tɛ̃ n sah Liyel Mirki-á nɛ̂nɛ yé kɔ̃. 
Yé pól pɩ wil gbɛ̃́nsê núkú Krista gbõ̀ yõ, ànɛ̂ Liyel Mirki núkú ye mɛ. Liyel yé ye tɔlɛ kwâh núkú yĩnnɛ nɛ yé yé gbõ̀ sah ké yõ. 
Tɛ̃̀ ǹgbɛ̃ núkú ye mɛ, ànɛ̂ Liyel yõ sɛ̃́nɔ́ núkúlɛ, ànɛ̂ ni sɔkɔ wèlnwìkìnɔ́ núkúlɛ. 
Liyel núkú ye mɛ. No pól Sú ye ǹnɛ. Ǹ tɔ̃́rɩ́ mɛ no pól yõ. À mɛ tõ̀lɛ n pɩ no pól sɔkɔ. À mɛ tɛ̃̀nɔ́lɛ no pól sɔkɔ. 
Ǹtɛ á nɛ́núkù núkú pól ye Liyel yõ̀bènɔrɩ wɛ à yɩ̃nɛ Krista-á mɛ ké tyɩ́ nɛ pé kélɛ á kɔ̃ sɔ̃́. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «Wǎh dɔkɔ yuku yĩ̂nyõ sɛ̃́, à sɔkɔ wɔ̀kɔ̀npèkè sɔkɔ névyelɛ. À tɛ̃́npnɛ̀nɔ́ yɩ̃ névye kɔ̃.» 
Ǹtɛ kwâh nɛ̂ mɛ pè n ye yĩ̂nyõ dɔ́kɔ́nyukunɔlɛ wɛ́? Képah ye n wɛ̃kɩ nɛ wǎh sõ̀ tiki yuku sétáh sõwãh tɔ sɔkɔ. 
Nɛ̂-á tiki pɔ, ǹmɔ ó ye tɔ̃ dɔkɔ yuku yĩ̂nyõ léyy tɛ̃̀-ǹsah sɔkɔ, kɩ kèkõyṍ pól yuku. 
Ǹmɔ Krista ye kɔ̃ névyelɛ tɛ̃́npnɛ̀nɔ́ duke dukelɛ. À túkù pɩ ǹ tɛ̃ntõ̀ névyelɛ, tɛ́ túkù pɩ Liyel tɛ̃ntõ̀ syínyopulɛ, tɛ́ túkù pɩ Nɩ̀vɩ̀nɩ̀ tir yónkɔ̃pulɛ, tɛ́ túkù pɩ kwɔ́púlɛ. 
Sɛ̃́ ye à vi mɔ Liyel nolɛ pè kɔ̃ pè kɩ ǹ tõ̀ pɩ dal mɔ, wãn kɔ̃ kè kɩ n mɔ Krista yõ sɛ̃́pú yõ. 
Kǎh pɩ sɛ̃́, Liyel yõ sɛ̃́nɔ́ kɩ á pól wrɔ́ kwrɔ mɔ pɩ núkúlɛ, tɛ́ á kɩ Liyel Pídĩ́ pɩ̃ tɛ́ kɩ yuku yahlɛ Liyel wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ fɔ́ɔ́ kɩ pɔ pɩ Krista tyɩ́ kɔ̃lɛ. 
Képah sɔkɔ, á náh tɔ̃ n pɩ n tɛ̃ n pi wɛ̃́npìlɛ nónó-á kwɔ́nɔ́ kétúkúnì pɩ́pú ànɛ̂ pé névye vìntɛ̃nɔ kɩ álɛ n tɔkɔ n fĩ n kɔ̃ yã́hkɩ́ tyɩ́ kɔ̃lɛ, kɩ á mɔ pílkínɔ́ wɛ̃̀kɩ̀ yõ. 
Ǹtɛ áyáh kɩ gbɩlɛ n yo tyɩ́nɔnɔ sɔkɔ, á kɩ yah yúkúnɔ́ wɛ tyi pól sɔkɔ kɩ n nɔ n de Kristalɛ ǹmɔ nɛ̂-á yõlɛ. 
Ǹmɔ gbõ̀ yõ ye wil gbɛ̃́nsê tã̀ntã̀n póllɛ núkúlɛ. Gbèl tél-ńtã̂n gbõ̀ yõ ye wil gbɛ̃́nsê tã̀ntã̀n pól wɛ̃ tɛ̃ yĩn sah. Kǎh pɩ sɛ̃́, wil tã̀ntã̀n pól-à tõ̀lɛ n pɩ kǎh yɩ̃nɛ sɔ̃́, wãn kɩ n mɔ wil gbɛ̃́nsê yõ, kè kɩ n yuku yahlɛ tyɩ́nɔnɔ sɔkɔ. 
Nɛ̂-á mé n gbah n yo yé tyɩ́ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ sɔkɔ, kè wɛ yɔ̀: Yé káh tɔ̃ yé minnɛ n pɩ Liyel pɩ̃́-ńkɛ̃̂ névye tyɩ́ kɔ̃lɛ, pépi nónó-á pé lékã́m pé yahlɛ n tɛ̃ n mɔ yõre kɛ̃́ tyi sɔkɔ. 
Pé lékã́m mɛ fɔ́kɔ́ntũnɔlɛ. Liyel-á min nɛ̂nɛ n kɔ̃, kè náh mɛ pé tyɩ́. Képah ye pè pɩ yah wɛ́-ńkɛ̃̂pú ànɛ̂ kùkùtǎhkɩ̀ névyelɛ. 
Pè pɩ sen pɩ̃́-ńkɛ̃̂púlɛ, pè náh n fyãh. Tɛ́ pè pɩ prɔ̀npɩbɩlɛ à tyítúkúnì duke póllɛ n pɩ yĩ̀n kɛ̃́nɛ. 
Ǹtɛ yépi tɛ̃̀ wɔlɛ, sɛ̃́ntɛ̃̀ náh yé pɩ̃ Kristalɛ dɛ́. 
Mé sɛ̃ ké yõ nɛ ǹ tyɩ́ dùkù pól-á yo yé tyɩ́. Tɛ́ gbɩ nɛ̂-á mɛ ǹ tyɩ́ sɔkɔ, yé kwɔ́nɔ́ wɛ ké tɔ yõ. 
Yáh sõ̀ mɛ dùkù nɛ̂nɛ ǹgbòlɛ, yé yɩ̃nɛ yè képah milki. Yé yé náh vi yé pɩ dùkù kédyâh kɔ̃, képah nɛ̂-á sõnɔ túkúnìnɛ n mɔ yé sɔkɔ, nónó-á yé tyɩ́lɛ n yɩkɩ. 
Liyel Mirki-á lékã́m nɛ̂nɛ yé kɔ̃, yé yɛ kè yé vi mɔ. 
Yé vi mɔ nɛ n pɩ pɩ dùkù késã́nnɛ à yɩ̃nɛ Liyel-á mɛ ké tyɩ́ sɔ̃́, yè mó nɛ n mɛ tómm ànɛ̂ yályál liyel yah gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ gbõ̀ yõ. 
Képah yĩnnɛ, yé mɩ̀nɩ̀ yónɔ́ yal n mɔ, yè mó gbɩlɛ n yo wɛ̃ tyɩ́. Yé náh wɛ di, wil gbɛ̃́nsê núkú tã̀ntã̀n ye á póllɛ. 
Yé fɔkɔ-à vyãh yé káh tyípékè pɩ. Gbĩ́yĩ́ náh yɩ̃nɛ kè pɔ de yè mó nɛ n mɛ fɔ̀kɔ̀vyã̀hnɩ̀ sɔkɔ sɛ̃́. 
Yé káh yé mɔ n tɛ̃ tɛ̃ wɛ̃̀kɩ̀ yɔ́ ńtɛ̃ yɩkɩ mɔ sétã̀n yõ̀tɛ̃̀ kɔ̃. 
Névi nɛ̂-à sõ̀ mɛ yɩ̃belɛ n yɩ̃, tɛ̃̀ kè yal mɔ, à mó ǹ gblɔ̌y kɔ̃ tõ̀lɛ n pɩ ǹ gbã̀nnɛ yĩ́nvǐ kɛ̃́nɛ, à fɛ̃ kwâh wɛ à névi yɔ́ tɔ yohnɩ n mɔ nɛ̂-á mɛ yãm nɛynɛ. 
Wɛ̀ntùkù yɔ́ ńtɛ̃ náh yɩ̃nɛ kè nɛ n wil yé vyãh sɔkɔ. Yé wɛ̀npèplɔ ólɛ n wɛkɩ nónó-á kɩ fɛ̃ névye yohnɩ mɔ, ànɛ̂ nónó-á kɩ fɛ̃ pèpɔrɩ pɩ sé nóhpú tyɩ́. 
Liyel Mirki mɛ yé sɔkɔ kètõ̀npɔ́kɔ́ sɔ̃́ yé wil tyɩ́, yé káh yèkwɔrɩlɛ n mɔ ké sɔkɔ. Yé náh wɛ di, képah gbõ̀ yõ sɔkɔ ye yé sɛ̃ yĩ́ĩ́ ké yõ nɛ sõ yɔ́-á mɛ n pi, Liyel-á kɩ yé pwah mɔ n pi pnɛ́pnɛ́. 
Yé káh yrɔlɛ n tɛ̃ n sah yé fɔkɔnɩ sɔkɔ. Yé fɔ̀kɔ̀vyã̀hnɩ̀, ànɛ̂ mâl, ànɛ̂ wɛy sǒhsǒh, ànɛ̂ gbyɔ yal n mɔ, à tahlɩ nɛ́pékè pɩ́nɔ́ duke pól tyɩ́. 
Yé pɩ nɛ́gblɔ́lɛ, yé tyɩ́ mó nɛ n mɛ nɔ́nɔ́lɛ wɛ̃ tyɩ́. Yé nɛ n yɔ̃ n mɔ wɛ̃nɛ n kɔ̃, sɔ̃́-á Liyel yɔ̃ mɔ yé kɔ̃ Krista gbõ̀ yõ.
