Korɛ̃nti tãn.
9.
Yòhnɩ̀mɔnɔ nɛ̂-á kɩ n tɛkɩ n mɔ n pi Yude mara Yesu yõ sɛ̃́pú tyɩ́, yõ náh mɛ ké tyɩ́ mè sɛ́bɛ́y tõ yé tyɩ́ képah tyɩ́ sɔkɔ. 
Mé pɩ̃ nɛ yáh mɛ ké tyɩ́ nɛ yé kè pɩ. Mé mɛ yélɛ n sõ ńtɛ̃nɛ Masedɔnɩ mara Yesu yõ sɛ̃́pṹ tyɩ́, mɛ n yo pé tyɩ́ nɛ, Akayi mara Yesu yõ sɛ̃́pú-á mɛ pé gblɔ̌y vi tɛ yòhnɩ̀nmɔnɔ pɩ yĩnnɛ fɔ́ɔ́ yán dùkùlɛ. Kǎh pɩ sɛ̃́, yépi nónó-á mɛ kénɛ mara sɔkɔ, yépi gbáhntɛ̃nɔ mɛ̀ sṍré mɔ névye kan sɔkɔ. 
Mé á yṹnpyélɛ n tɛkɩ n mɔ yé tyɩ́, sɔ̃̀nko máh mɛ yélɛ n sõ sõ̀nɔ́ nɛ̂nɛ yĩ mɛ̀ tyɩ́ sɔkɔ, kè káh pɔ pɩ yõ kɛ̃́nɛ. Ǹtɛ yè nɛ n mɛ yé gblɔ̌y vi sah kègbɩ yõ sɔ̃́-á mé yo kélɛ. 
Kàh kɩ pɩ dùkù yɔ́lɛ, tɛ́ Masedɔnɩ mara tãn-à kɩ pɔ́nɔ́ yĩ́ ńmɔlɛ, tɛ́ pɔ nɛ yah yé náh mɛ yé gblɔ̌y vi sah, kè kɩ pɩ sen ǹgbɛ̃nɛ ápi tyɩ́ áyáh sɔ̃́npɩ́lɛ kàh sɛ̃ yé yõ. Tɛ́ yépi ńtɛ̃ wɔ tyɩ́ ye kè kɩ pɩ sennɛ. Ǹtɛ mé náh mɛ ké tyɩ́ nɛ mé képah yo. 
Képah ye, mé kè sõ nɛ máh yɩ̃nɛ mé á yṹnpyé nɔ́pi ni pè mɔ ńmɔ yahlɛ, pè pɔ pɔ́ mɔ́rɩ́ mɛ̀ tyɩ́ yahlɛ n wah yáh yé vyãh kɔ̃ nɛ̂ tyɩ́ sɔkɔ; sɔ̃̀nko máh kɩ pópó n pi, ké tyɩ́ yah nɛ n mɛ wàhntɛnɔlɛ, nɛ̂-á kɩ kè wɛ̃kɩ nɛ yáh kè pɩ yé dyɔ dyɔ tɛ̃̀nɛ, kyɩ̃́nmɔnɔ náh pè kyɩ̃ mɔ yélɛ. 
Yé se kè pɩ̃ nɛ, névi nɛ̂-à duku mɔ kwéy, tɛ̃̀ n pɔ sèmukɔ kwéy tɔkɔ; tɛ́ nɛ̂-à duku mɔ dò, tɛ̃̀ n pɔ sèmukɔ dò tɔkɔ tɛ́. 
Yé nɛ́núkù núkú pól yɩ̃nɛ yè kɔ̃ à yɩ̃nɛ yáh kè yah tɔkɔ sɔ̃́ yé nnɔ sɔkɔ. Nɛy káh kɔ̃ à mó tɔ̃ ké nàhnpɩ̃nɛ n gbo, képah náh pɩ, pè sɔ̃́ tɛ̃̀ kyɩ̃ mɔ. Yé náh wɛ di, nɛ̂-á n yɩ̃ n kɔ̃ nɩ̀vɩ̀nɩ̀ sɔkɔ, ǹmɔ tyɩ́ nɔ Liyel tyɩ́. 
Tɛ̃́nwɛnɔ mɛ Liyel tyɩ́ kɩ fɛ̃ pèpɔrɩ duke pól pɩ yĩ́ĩ́ yé tyɩ́, sɔ̃̀nko yî nónó pól yétõ̂-á mɛ yé tyɩ́, sè nɛ n mɛ yé tyɩ́ gbĩ́mɛgbĩ́ ànɛ̂ dòhnɔ̀ tɔlɛ, yé kɔ̃ yè fɛ̃ pèpɔrɩ duke póllɛ n pɩ, 
sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «À ǹ nɩlɛ n dal n mɔ n yɩ̃ n kɔ̃, à n yɩ̃ yãm tãnnɛ n kɔ̃ n yĩ́ĩ́. ǹ fɔ̀kɔ̀pènɔrɩ tyɩ́ yĩ̀n náh mɛ.» 
Liyel nɛ̂-á kwáhdúkúlɛ sètõlɛ n kɔ̃, tɛ́ ǹ yõ yõ yõkelɛ ǹnɛ n kɔ̃, ǹmɔ ye kɩ kwáhdúkúlɛ yépi kɔ̃ n pi, tɛ́ kɩ kè kɔ̃ kè kɩ kwel, kɩ wãn mɔ yé fɔ̀kɔ̀pènɔrɩ pɩ́nɔ́ yõ. 
À kɩ yé pɩ kwâh nátãmnɛ gbĩ́mɛgbĩ́, sɔ̃̀nko yè fɛ̃ fɔ̀kɔ̀pènɔrɩlɛ n pɩ tyɛ́-ńkɛ̃̂nɛ. Képah sɔkɔ, yáh yî nónónɛ álɛ n kɔ̃ á névyelɛ n kɔ̃ n pɔ, pé náhnáh kɩ wɛkɩ Liyel tyɩ́ sépi yĩnnɛ. 
Yé náh wɛ di, yé pɔ́ mɔ́rɩ́ mɛ̀ kɩ Yesu yõ sɛ̃́pú yohnɩ mɔ. Tɛ́ kè náh n yĩn n pi képah ó tyɩ́ dɛ́, kè kɩ névye náhnáh kɔ̃ pè kɩ pèpɔrɩ pɩ̃́nɔ́ wɛynɛ Liyellɛ n kɔ̃. 
Yõre tõ̀ nɛ̂-á yé pɩ, pàh kɩ kè wɛ pè kɩ Liyel yĩn gbilki yáh sɔ̃́npɩ́lɛ pɩ nɛ́gblɔ́lɛ à fɛ̃ syi tõ̀ pɩ Krista Nɩ̀vɩ̀nɩ̀ tir yónɔ́ sɔkɔ. Pè kɩ tɔ̃ Liyel yĩn gbilki yáh sɔ̃́npɩ́lɛ yé gbã̀n yîlɛ pépi ànɛ̂ no tɛ́lɔ́ pól tɔ yohnɩ mɔ. 
Pè kɩ nírí pɩ yé tyɩ́, kɩ pé tyɩ́nɔnɔlɛ yé wɛ̃kɩ yõ̀bènɔrɩ ǹgbɛ̃ mɛ̀ yĩnnɛ, Liyel-á nɛ̂ pɩ yé tyɩ́. 
Gbílkínɔ́ nɛ n sah Liyel yõ ǹ pèpɔrɩ ǹgbɛ̃ pɩ́nɔ́ yĩnnɛ, nɛ̂-á ké kénkɔ̃ náh mɛ.
