Nsyakɩ.
3.
Ń yṹnpyé, yé pɩ̃́nɔ́ mɛ ké sɔkɔ nɛ, ápi nónó-á pɩ névye kwɔ́púlɛ, áyáh kɩ n tuke n pi képékèyɔ̂, kɩ kal no tɛ́lɔ́lɛ kɛ̀. Képah ye, yé náhnáh káh nɛ n yah n kɔ̃ nɛ, yé pɩ kwɔ́púlɛ. 
Yé pɩ̃ nɛ, á nɛ́núkù núkú pól-á n pilki náhnáh. Ǹtɛ, névi nɛ̂-à n pílkí-ńkɛ̃̂nɛ ǹ vyãh wɛy sɔkɔ, ǹmɔ nɛ́pépɛ̌y syɩ ké yĩ̀n tyɩ́. À kɩ fɛ̃ tɛ̃ ǹ wil gbɛ̃́nsê pól tɔ tyɩ́. 
Sɔ̃́-á áyâh kàrfyɛ̂ dyɩ sèsèn vyɛ̃y tyɩ́ nɛ, sè nɛ n nohnɩ á tyɩ́, sé wil pól tɔ yah ye ńkɛ̃́nɛ à n nɛ n tɛ̃ kélɛ. 
Képah náh pɩ, yé krótókè ǹgbɛ̃ tɔlɛ n yah yɔ sɔkɔ kɛ̀: Kàh sɔ̃́ mɛ gblò sɔ̃́, tɛ́ yã́hkɩ́ sɔ̃́ mɛ kélɛ n tu n mɔ sɔ̃́, sésár pír yɔ́ ó ye ké tó-ò n nɛ n tɛ̃ ké yahlɛ. Wàh mɛ ké tyɩ́ nɛ, kè syɔ̃ ńyãh sɔkɔ, kè n syɔ̃ nɛ́npɔ́lɛ. 
Sɛ̃́ ó ye nɛ́npɔ́rɩ́ tɔlɛ, wil tyah yɔ́ ye tyɩ̃́ɩ̃ ǹsɔ̃̀, tɛ́ kè ké gblɔ̌ynɛ n gbilki n kah. Yé náh nyàh tɔ wɛ di? Kàh sɔ̃́ mɛ tyɩ̃́ɩ̃ sɔ̃́, kàh soh dùkù ǹgbɛ̃ sɔkɔ, kè n fɛ̃ ké pól yõ. 
Ǹtɛ, nɛ́npɔ́rɩ́ tɔ mɛ kénɛ nyàh tyɩ́ kɔ̃lɛ. Tyípêl náhnáh n wil képah sɔkɔ. Kè mɛ déntɛ̃̀nɔ́lɛ wil sɔkɔ, tɛ́ wil tɛ́y póllɛ n yõ. Tyípêl nónó-á n wil ké sɔkɔ, ní-ńkɛ̃̂ nyàhnpèkè sɔkɔ ye sè n wil. Képah ye, kě yɩ́kɩ́nɔ́lɛ n pɩ á min pól sɔkɔ, fɔ́ɔ́ kè pɔ nɔ á kúnɔ́ tyɩ́. 
Yah, gbɛ̃́nkyɔ duke duke pól n kwɔ n pnɛ. Kèpye, ànɛ̂ vyeh, à tahlɩ yɔ sɔkɔ yî tyɩ́, névi tɛ̃ wɛ sépi pól kwɔ. 
Ǹtɛ, nɛ́npɔ́rɩ́ tɛ̃̀ wɔlɛ, ǹnɩ̂ yɔ́ ńtɛ̃ náh tɛ̃ wɛ képah kwɔ n pi. Kwáhpékè nɛ̂-á kè náh n tɛ̃ syɩ́syɩ́, képah ye nɛ́npɔ́rɩ́lɛ. Kè mɛ névi ko ko sohrɩ ólɛ ké tɔ̂ɔ́. 
Kénɛ nɛ́npɔ́rɩ́ núkú mɛ ye ápi Tɛ̃̀ ǹgbɛ̃, á sú Liyellɛ n gbilki, ǹ tɛ́ n tɔ̃ képah ólɛ pèkèlɛ n ye n mɔ névye yõ, ǹtɛ Liyel ye mó tɔ̃ mɔ pélɛ ǹ kètɔ̀lɛ. 
Gbílkínɔ́ wɛlɔ ànɛ̂ pèkè yénmɔnɔ wɛlɔ kɩ n wil vyãh núkú sɔkɔ nɩ? Ń yṹnpyé, yé káh nɛ n pɩ sɛ̃́. 
Nípénɛ̀kɩ̀ ànɛ̂ ni páhpáh se kɩ fɛ̃ n wil kwlɛ̀y núkú sɔkɔ? 
Ń yṹnpyé, dah se kɩ fɛ̃ wɛ gbɛ sènsóké séswɔ̂ sɔkɔ? Tĩ̂ sènsóké séswɔ̂ pye se kɩ fɛ̃ wɛ gbɛ kèpɔ̃npɔ̃ séswɔ̂ sɔkɔ? Sɛ̃́ ó ye pnɛ ni, ànɛ̂ nípénɛ̀kɩ̀ náh fɛ̃ n wil n pi kwlɛ̀y núkú sɔkɔ. 
Tɛ́ yépi sɔkɔ, nɛ̂ tɛ̃̀ mɛ yɩ̃́pɩ̃̂ névilɛ tɛ́ lékã́m mɛ ǹ sɔkɔ? Tɛ̃̀ yɩ̃nɛ à nɛ n kɔ pallɛ gblɔ̌y tíkínmɔnɔlɛ ǹ tyípɩ́nɩ̀ sɔkɔ, képah nɛ̂-á n wɛ̃kɩ nɛ, wǎh yɩ̃ pɩ̃. 
Ǹtɛ, dyɔ́yɔ́kɩ̀ képékè, ànɛ̂ wɛ̃ fwónɔ́ sõnɔ-à kénɛ yé nnɔ sɔkɔ, yé káh yé gblɔ̌ynɛ gbilki nɛ, yáh mɛ yɩ̃ pɩ̃. Yé káh gbɩlɛ n kɔ̃lɩ. 
Kénɛ yɩ̃́pɩ̃̂ mɛ̀ dùkù náh n wil Liyel tyɩ́ dɛ́! Kèkõyṍ mɛ̀ tyɩ́ ye kélɛ. Névi yɩ̃́pɩ̃̂ ye kélɛ, kè n wil sétõ tyɩ́. 
Yah, dyɔ́yɔ́kɩ̀ ànɛ̂ wɛ̃ fwónɔ́ sõnɔ-à mɛ ńyãh sɔkɔ, tyífɛ̃̌ ànɛ̂ tyítúkúnì duke duke pól n nɛ n pɩ nɛ́npɔ́ sɔkɔ. 
Ǹtɛ, yɩ̃́pɩ̃̂ nɛ̂-á n wil Liyel tyɩ́, képah-à mɛ névi nɛ̂ sɔkɔ, ké gbennɛ, tɛ̃̀ nɩ n nɛ n mɛ yályál, ànɛ̂ yèvɩnɩ tyɩ́ n nɔ ǹ tyɩ́, tɛ́ à n pɩ nɛ́pépɛ̌ynɛ. Ǹ tõ̀ náh n tah, ànɛ̂ à névi yãmnɛ n yah, tɛ́ tyípéplɔ ó ye à n nɛ n pɩ. À náh yèwǎhkɩ̀ ànɛ̂ fɔ̀kɔ̀pǐllɛ n pɩ. 
Yèvɩnɩ-á mɛ nónó tyɩ́, tɛ́ pè mɛ ké tõ̀lɛ n pɩ, pě ké yõre wɛ, nɛ̂-á pélɛ n pɩ nɛ́gblɔ́lɛ Liyel yah.
