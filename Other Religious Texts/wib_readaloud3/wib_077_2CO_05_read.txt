Korɛ̃nti tãn.
5.
Wil gbɛ̃́nsê nɛ̂-á ápi min tɛ̃̀-ǹsahlɛ sétáh mɛ̀ yõ sɔkɔ yɔ̀, ké tyɩ́-à pɔ yɩkɩ, Liyel kɩ min tɛ̃̀-ǹsah yɔ́ yah á kɔ̃ yĩ̂nyõ, nɛ̂-á kɩ mɛ kṹmm yõ sɔkɔ. Kénɛ tɛ̃̀-ǹsah mɛ̀nɛ, névi gbɛ̃̀npɩnɩ náh kélɛ. 
Ǹtɛ núkúnúkú, ké sɔkɔ de dékè mɛ á kah tɛ̃, képah ye á mɛ n gbĩ ké yĩnnɛ. 
Áyáh kɩ wɛ de ké sɔkɔ, á wil tìtí náh tɔ̃ n wɛ n pi. 
Wɔ̂ nɛ̂ pól-á á min kɩ n pɩ n pi sétáh mɛ̀ yõ sɔkɔ, á kɩ mɛ gbĩ̀nɔ́ sɔkɔ tɔ́kɔ́pékè sɔ̃́ á yõ. Min tɛ̃̀-ǹsah nɛ̂-á á tyɩ́ sétáh mɛ̀ yõ, kè náh mɛ á kɩ nɛ á sɔ̃́ mɛ ké tyɩ́ yɩ́kɩ́-ńsahlɛ n yah n kɔ̃ dɛ́. Yɩ́kɩ́-ńkɛ̃̂ min tɛ̃̀-ǹsah nɛ̂-á Liyel kɩ á kɔ̃ n pi yĩ̂nyõ, á mɛ ké tyɩ́ nɛ képah pɔ yɩkɩ yɩkɩ tyɩ́ mɛ̀ swɔ tɔkɔ. 
Liyel ye sépi pól yah sah ápi yĩnnɛ. Pèpɔrɩ nónó-á à kɩ n pɩ n pi á tyɩ́, ǹ Mirki ǹgbò ye à tɛkɩ mɔ á kɔ̃ sé sɔkɔ. 
Képah ye, á sɛ̃́nyĩnɔ mɛ ké yõ gbĩ́mɛgbĩ́ nɛ á min tɛ̃̀-à kɩ fyɔ̀ tɛ̃̀ á wil gbɛ̃́nsê mɛ̀ tɛ̃̀ sɔkɔ, á wrɔ́-á mɛ dɔ́nɔ́lɛ Tɛ̃̀ ǹgbɛ̃nɛ. 
Ǹ yõ sɛ̃́nɔ́ gbõ̀ yõ ye á mɛ n yuku yahlɛ, ǹ wɛ́nɔ́ gbõ̀ yõ náh. 
Á sɛ̃́nyĩnɔ mɛ ké yõ, képah ye á mɛ ké tyɩ́ nɛ á min yal wil gbɛ̃́nsê mɛ̀ sɔkɔ, kè kyɩ n tɛ̃ Tɛ̃̀ ǹgbɛ̃ tnɔ̂. 
Ǹtɛ kàh kɩ se wɛ wil ò; kàh kɩ se wɛ wìl-ǹkɛ̃̂nɛ ò; áyáh pɩ sɔ̃́ tɛ́ á tyɩ́ n nɔ n pi Tɛ̃̀ ǹgbɛ̃ tyɩ́, képah ye á mɛ n yah n kɔ̃. 
Yé náh wɛ di, á pól yɩ̃nɛ Krista á tuke. Képah sɔkɔ, nɛ̂-à kɩ mɛ pèpɛy pɩ kèkõyṍ mɛ̀ sɔkɔ, à kɩ à dyah kélɛ. Nɛ̂-à kɩ mó mɛ pèkè pɩ, à kɩ à dyah kélɛ. 
Kwâh nɛ̂-á Tɛ̃̀ ǹgbɛ̃ yah tíkí pɩ́nɔ́lɛ, ápi kélɛ n pnɛ. Képah ye, á mɛ n yah n kɔ̃ nɛ á névye sõnɔ vi mɔ pè ǹ yah tíkílɛ n pɩ. Ápi fɩ̃́ yɔ́ ńtɛ̃ náh mɛ Liyel pu. Tɛ́ ńmɔ sõnɔ sɔkɔ, yépi tɔ api pɩ̃ yĩ́ĩ́. 
Á gblɔ̌y dùkù náh tɔ̃ á n yo yélɛ n syah, ǹtɛ syɔ̃̀-ǹsah ye á yélɛ n kɔ̃ yè fɛ̃ yé gblɔ̌ynɛ n gbilki ápi yĩnnɛ. Képah sɔkɔ, wɛ́nyo-ńsah kɩ wɛ yé tyɩ́ yĩ́ yah pɩ́pú tyɩ́. 
Ápi yõ̀gbèl-à kɩ sɔ̃́ mɛ yuku, Liyel yĩn ye. Ǹtɛ á yah-à kɩ se mó mɛ gbrɛ́y, yépi yĩn ye. 
Krista tyɩ́nɔnɔ ye yah syi ápi tyɩ́, ápi nónó-á pɩ̃ nɛ nɛ́núkú-á ku no pól swɔ sɔkɔ. Kǎh pɩ sɛ̃́, kè mɛ á kɩ nɛ no pól tɔ sɔ̃́ ku. 
À ku no pól yĩnnɛ sɔ̃̀nko nónó-á mɛ min sɔkɔ, pè káh pé minnɛ n pɩ pé gblɔ̌y yĩnnɛ, nɛ̂-á ku tɛ́ ǹ lékó yilki mɔ, pè kélɛ n pɩ ǹmɔ yĩnnɛ. 
Kǎh pɩ sɛ̃́, núkúnúkú tɛ̃̀nɛ á náh tɔ̃ ǹnɩ̂ yɔ́ ńtɛ̃ yah n pi à yɩ̃nɛ névi yõ̀tàhsyi yáhnɔ́lɛ. Áyâh sɔ̃́ sõ̀ mɛ Kristalɛ n yah sɛ̃́, núkúnúkú á náh tɔ̃ mɛ ǹnɛ n yah sɛ̃́. 
Névi nɛ̂ wrɔ́-à mɛ kwrɔ́nmɔnɔlɛ Kristalɛ, tɛ̃̀ nɩ n vi n mɔ. Ǎ ǹ pɩ dùkù kédyâh fĩn n mɔ, ǹ tɛ́ pɩ dùkù késã́n tɔkɔ. 
Liyel tõ̀ vyãh ye sépi póllɛ, ǹmɔ nɛ̂-á ápi wrɔ́ kwrɔ mɔ ǹnɛ Krista gbõ̀ yõ, tɛ́ mɛ wrɔ́ kwrɔ́nmɔnɔ tõ̀ dahbɩ ápi gbõ̀. 
Yé náh wɛ di, Liyel náh kèkõyṍ névye tyípêl mɔ kõ, à syɔ̃ Krista tyɩ́ à pé wrɔ́ kwrɔ mɔ ǹnɛ, tɛ́ mɛ wrɔ́ kwrɔ́nmɔnɔ tyɩ́ yónɔ́ dahbɩ ápi gbõ̀ nɛ, á névye wrɔ́lɛ n kwrɔ n mɔ péwɔlɛ. 
Krista yĩn yõ ye ápi mɛ n wɛkɩ. Kè mɛ á kɩ nɛ Liyel sɔ̃́ n yo n mɔ névye tyɩ́ ápi gbõ̀ yõ. Képah ye á kélɛ n yah n kɔ̃ yé tyɩ́ Krista yĩn yõ nɛ, yé yé wrɔ́ kwrɔ n mɔ Liyellɛ. 
Krista nɛ̂-á à náh tyípékè pɩ yah, Liyel ǹmɔ twah mɔ ápi tyípêllɛ, sɔ̃̀nko ǹmɔ gbõ̀ yõ, ápi tɔ pɩ yályál Liyel yah sɔkɔ.
