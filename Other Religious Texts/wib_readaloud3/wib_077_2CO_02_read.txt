Korɛ̃nti tãn.
2.
Ńmɔ kè yah ń gblɔ̌y nɛ yõ náh mɛ ké tyɩ́ mé n syɩ n pɔ yé tĩ̀nnɛ mé pɔ yèkwɔrɩ yɔrɩ mɔ yé yõ ńtɔ̃. 
Ńmɔ-à kɩ yèkwɔrɩ mɔ yépi sɔkɔ, nɛ̂ kɩ pɩ ńmɔ nɩ̀vɩ̀nɩ̀ yõlɛ, nɛ́ yépi náh máh yèkwɔrɩ mɔ yépi nónó sɔkɔ? 
Mé sɛ́bɛ́y mɛ̀ wãrkɩ tɛkɩ mɔ yé tyɩ́, sɔ̃̀nko mé káh pɔ yé tĩ̀nnɛ yé túkù kɔ̃ pè yèkwɔrɩ mɔ ń sɔkɔ, pépi nónó-á sõ̀ yɩ̃nɛ nɩ̀vɩ̀nɩ̀ mɔ́nɔ́lɛ ń sɔkɔ. Mé kè pɩ̃ yĩ́ĩ́ nɛ, mâh mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ, kè n pɩ nɩ̀vɩ̀nɩ̀nɛ yépi tɔ tyɩ́. 
Ńmɔ-á sɛ́bɛ́y mɛ̀nɛ n wãrkɩ gbĩ́ nɛ̂nɛ, ń nɩ sõ̀ mɛ yɩ́kɩ́nɔ́lɛ, ń lékã́m sõ̀ mɛ vye fɛ̃kɩ tɛ́ yétĩ mɛ n kah n kwɛ. Yèkwɔrɩ náh sõ̀ mé n yah n kɔ̃ mɔ́-ńsahlɛ yé sɔkɔ dɛ́, mé sõ̀ mɛ ké tyɩ́ nɛ yé pɩ̃, yé tyɩ́-á mɛ nɔ ń tyɩ́ à kyɩ syɩ ńyãh tyɩ́. 
Yèkwɔrɩ-à wɛ ǹnɩ̂ yɔ́ gbõ̀ yõ, ńmɔ tyɩ́ náh tɛ̃̀ pɩ dɛ́, yé pól yõ ye tɛ̃̀ yɔrɩ mɔ kélɛ. Ǹtɛ áyâh nɛ á káh kè yo n kah mɔ, á kɩ nɛ yé pól yõ-á tɛ̃̀ yɔrɩ mɔ kélɛ. 
Kénɛ tɛ̃̀ tɛ̃̀nɛ, tɔ́wû-á kɩ lékàh nɛ̂nɛ à kɔ̃, képah kɩ yɩ̃nɛ ǹnɛ. 
Képah ye, yé yɩ̃nɛ yè yɔ̃ mɔ tɛ̃̀ kɔ̃ yè mó ǹ nɩ de, sɔ̃̀nko yèkwɔrɩ káh pɔ ǹ yáh-ńsah tyɛ n kah mɔ. 
Mé kélɛ n yah n kɔ̃ yé tyɩ́ tɔlɛ nɛ, yé kélɛ tɛ̃̀ wɛ̃kɩ nɛ ǹ tyɩ́-á mɛ nɔ yé tyɩ́. 
Máh sɛ́bɛ́y mɛ̀nɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ yõ nɛ̂ yõ tɔlɛ, mé sõ̀ mɛ ké tyɩ́ nɛ mé yé gbɩ yah, yáh kɩ mɛ n sah ń yóré yõ tyi tyi pól sɔkɔ. 
Yâh yɔ̃ mɔ névi nɛ̂ kɔ̃, ńmɔ tɔ kɩ yɔ̃ mɔ tɛ̃̀ kɔ̃. Tɛ́ yâh wɛ mé yɔ̃ mɔ ǹnɩ̂ kɔ̃, yépi yĩn ye mè n pɩ kélɛ Krista yah sɔkɔ. 
Képah sɔkɔ, sétã̀n yõ̀tɛ̃̀ náh syɔ̃̀-ǹsah wɛ à tɛ̃ wɛ n pi á tyɩ́. Á pól ǹ yah mɔ́-ńsahlɛ n pnɛ dò. 
Máh yĩni tɛ̃ Turasɩ mara sɔkɔ Yesu Krista Nɩ̀vɩ̀nɩ̀ tir yó-ńsah, mé kè wɛ Liyel ké wɛ̃̀kɩ̀ yɩkɩ mɔ ń kɔ̃ pallɛ. 
Ǹtɛ mé náh ń yṹnpi Tite wɛ nɛ́npɔ́ sɔkɔ. Kǎh pɩ sɛ̃́, ń sõnɔ náh sõ̀ mɛ tɛ̃̀nɔ́lɛ fɩ̃́ núkú sɔkɔ. Képah ye, mé mɛ wɛ̃̀kɩ̀ ni nɛ́npɔ́ tãn tyɩ́, tɛ́ sɔkɔ Masedonɩ mara sɔkɔ. 
Gbílkínɔ́ nɛ n sah Liyel yõ, ǹmɔ nɛ̂-á á pɩ tɛ̃́nwɛpulɛ á wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Kristalɛ. À á kɔ̃ á Krista dùkùlɛ n yo n tɔ̃ n mɔ n kɔ̃ tã̀ntã̀n pól sɔkɔ, sɔ̃́-á làsèkwěl pãn n tɔ̃ n mɔ tyah pól sɔkɔ. 
Ápi pɩ làsèkwěl pãn pènɛ̀kɩ̀ tyɩ́ kɔ̃lɛ Liyel tyɩ́, á wrɔ́ kwrɔ́nmɔnɔ gbõ̀ Kristalɛ. Névye nónó-á pwáhnmɔnɔ wɛ̃̀kɩ̀ yõ, ànɛ̂ nónó-á mɛ tyɩ́ yɩ́kɩ́nɔ́ wɛ̃̀kɩ̀ yõ, pãn mɛ̀ mɛ n yĩni n tɛ̃ pé pól tyɩ́. 
Túkù tyɩ́, á mɛ n põ kétukùyɔ̂ nɛ̂-á n pɔ n pɔ kũnɛ. Túkù tyɩ́ mó, á mɛ n põ pallɛ nɛ̂-á n pɔ n pɔ minnɛ. Ǹtɛ nɛ̂ tɛ̃̀ wɔ kɩ fɛ̃ képah pɩ wɛ́? 
Névye náhnáh mɛ Liyel wɛy tɔkɔ gbɔ̀lɛ n yɩ̃. Ápi tɛ̃̀ wɔ náh mɛ pépi tyɩ́ kɔ̃lɛ. Liyel ye tõ ápilɛ, képah ye á mɛ á fɔkɔ dal mɔ ǹ wɛynɛ n yo ǹ yah sɔkɔ, á wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Kristalɛ.
