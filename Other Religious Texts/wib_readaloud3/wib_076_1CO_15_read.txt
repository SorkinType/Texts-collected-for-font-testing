Korɛ̃nti tãn.
15.
Ń yṹnpyé, mé mɛ ké tyɩ́ nɛ mé yé dyɔ to kõ Nɩ̀vɩ̀nɩ̀ tir mɛ̀ tyɩ́lɛ, nɛ̂-á mé yo sɔkɔ yé tyɩ́, képah nɛ̂-á yé kè fɛ̃ syi tɛ́ mɛ ké wɛ̃̀kɩ̀lɛ n kɔ n yĩ́ĩ́. 
Képah gbõ̀ yõ sɔkɔ, yé kɩ pwáhnmɔnɔ wɛ, yáh kɩ n kɔ ké wɛ̃̀kɩ̀ yõ máh kè yo sɔ̃́ yé tyɩ́. Kàh kɩ pɩ́-ńkɛ̃̂nɛ sɛ̃́, ké yõ sɛ̃́nɔ́ kɩ pɩ yõ kɛ̃́nɛ yé tyɩ́. 
Ńmɔ-á kwɔ́nɔ́ wɛ nɛ̂ yõ, képah ǹgbò ye mé yé kwɔ nɛ Liyel nɛy yáhntɔkɔnɔ-á ku ápi tyípêl yĩnnɛ. Sɛ̃́ ye kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ. 
Mé tɔ̃ kwɔ́nɔ́lɛ yé kɔ̃ nɛ wǎh yɩ̃, tɛ́ ké sèpĩ̀n tɔ̃́-ò sõlɛ ǹ lékó mɛ yilki mɔ. Sɛ̃́ ye kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ. 
À ǹ gblɔ̌ynɛ Pyɛrɩ wɛ̃kɩ, képah náh tɛ́ gblɔ̌ynɛ pé nɛ́pĩ̂ gbɔ̃nimí wɛ̃kɩ, pépi nónó-á ǹ tɛ̃ntõ̀ névyelɛ. 
Képah náh, à pɔ ǹ gblɔ̌ynɛ ǹ tõ̀ sã́hpú nɛ́pĩ̂ kyɛkwɛ́ (500) kahnɩ wɛ̃kɩ fɩ̃́ núkú sɔkɔ. Pépi kan mɛ min sɔkɔ kwɛy nɔ yɔ̀, tɛ́ pé túkù wɔ ku. 
Képah náh, tɛ́ ǹ gblɔ̌ynɛ Nsyakɩ wɛ̃kɩ, tɛ́ mɛ ǹ gblɔ̌ynɛ ǹ tɛ̃ntõ̀ névye pól wɛ̃kɩ. 
Pépi pól tyɩ́ sɔ́kɔ́ntɛnɔ náh, à mɛ ǹ gblɔ̌ynɛ ńmɔ tɔ wɛ̃kɩ, ńmɔ nɛ̂-á piri kòndal-ńkɛ̃̂ tyɩ́ kɔ̃lɛ. 
Ńmɔ ye Yesu tɛ̃ntõ̀ névye sɔkɔ tyɩ̃́ɩ̃ tɛ̃̀nɛ. Mé náh yɩ̃nɛ yénɔ́lɛ Yesu tɛ̃ntõ̀ névilɛ, máh sɔ̃́npɩ́lɛ fṍhnɔ́ sah ǹ yõ sɛ̃́pú tĩ̀nkì yõ. 
Ǹtɛ máh swɔ nɛ̂ sɔkɔ kwɛynɛ yɔ̀, Liyel pèpɔrɩ gbõ̀ yõ sɔkɔ ye ńnɛ ké sɔkɔ. Pèpɔrɩ nɛ̂-á à pɩ ńmɔ tyɩ́, kè náh sɔkɔ de ni sɔkɔ. Yé náh wɛ di, ńmɔ ye tõ̀ pɩ kal á no tɛ́lɔ́ póllɛ. Ǹtɛ ńmɔ tɛ̃̀ wɔ tǎhkɩ̀ náh n pɩ kélɛ dɛ́, Liyel-á pèpɔrɩ nɛ̂ pɩ ń tyɩ́, képah gbõ̀ yõ sɔkɔ ye mé n pɩ kélɛ. 
Képah ye, kè sɔ̃́ ńmɔ ò, kè sɔ̃́ pépi ò, kénɛ Nɩ̀vɩ̀nɩ̀ tir mɛ̀ ó ye á pól mɛ n yo. Képah yõ ye mó tɔ̃ yépi tɔ mɛ sɛ̃. 
Ápi-á kélɛ n yo n wah nɛ ńkɛ̃́ Krista lékó-á yilki mɔ, sɔ̃́ mɛ tɛ́ yé túkù nɛ lékyɩ̂ náh n yilki n mɔ n pi? 
Lékyɩ̂-à kɩ mɛ pɔ n yilki n mɔ n pi ńkɛ̃̂nɛ, Krista lékó tɔ náh mó yilki mɔ dò. 
Tɛ́ wàh mɛ yílkínmɔ-ńkɛ̃̂nɛ, ápi Nɩ̀vɩ̀nɩ̀ tir yónɔ́lɛ, nɔ̀nɔ́ ye kélɛ, yépi tɔ Krista yõ sɛ̃́nɔ́lɛ mó, nɔ̀nɔ́ ye ké tɔlɛ. 
À wil képah sɔkɔ, kàh sõ̀ mɛ gbɩlɛ nɛ lékyɩ̂ náh n yilki n mɔ n pi, mɩ̀nɩ̀ yónsàhpu ye ápi ńtɛ̃nɛ Liyel yõ sɔkɔ, áyáh sɔ̃́npɩ́lɛ nɛ wǎh Krista lékó yilki mɔ, tɛ́ à náh se mó tɔ̃ kè pɩ. 
Lékyɩ̂-à kɩ mɛ n yilki n mɔ n pi ńkɛ̃̂nɛ, Krista lékó tɔ náh mó yilki mɔ dò. 
Tɛ́ Krista lékó-à kɩ mɛ yílkínmɔ-ńkɛ̃̂nɛ, yé sɛ̃́nɔ́lɛ ǹ yõ, nɔ̀nɔ́ ye kélɛ. Yé mɛ sɛ̃́ yé tyípêl sɔkɔ. 
Tɛ́ névye nónó-á sɛ̃ Krista yõ tɛ́ ku, pépi tɔ tyɩ́ ye mó yɩkɩ. 
Kàh kɩ mɛ ápi á gbõ̀ sah Krista yõ, á min pɩ yĩnnɛ kèkõyṍ mɛ̀ tɛ̃̀ ó sɔkɔ, yãm pɩ́pú ye álɛ à kal no póllɛ. 
Ǹtɛ gbɩ yõ sɔkɔ, Krista lékó ye yilki mɔ gbennɛ lékyɩ̂ sɔkɔ, nɛ̂-á kélɛ álɛ n wɛ̃kɩ n dal n mɔ nɛ névye nónó tɔ́-á ku sɔkɔ, pé tɔ lékyɩ̂-á kɩ pɔ yilki mɔ. 
Yé náh wɛ di, nɛ́núkù yɔ́ yĩn ye névye póllɛ kũ mnɔ sɔkɔ yɔ̀. Tɛ́ nɛ́núkù yɔ́ tɔ gbõ̀ yõ ye névye pól lékyɩ̂ kɩ pɔ yilki mɔ. 
Sɔ̃́-á Adama náh tãn pól n ku, sɛ̃́ ó ye Krista yõ sɛ̃́pú pól tɔ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Ǹtɛ no pól lékyɩ̂ kɩ yilki mɔ syɩ yuku wɛ̃ náh, kǎh yɩ̃nɛ sɔ̃́. Krista ǹgbò lékó ye yilki mɔ. Képah náh, ǹ syɩ n pɔ gbĩ́nɛ, nónó-á mɛ ǹmɔ tyɩ́ yõ, pépi tɔ lékyɩ̂ kɩ yilki mɔ. 
Képah náh, tyi tyi pól tyɛ gbĩ́ kɩ nɔ. Képah tɛ̃̀nɛ, Krista kɩ mɛ névye fnɛ̀pu ànɛ̂ tɔ̃́rɩ́nɔ, ànɛ̂ tǎhkɩ̀nɔ́ duke duke pól tyɩ́ yɩkɩ, tɛ́ kɩ yõ̀tɔ̃rɩ́ dahbɩ Sú Liyel gbõ̀. 
Yé náh wɛ di, Krista yɩ̃nɛ à nɛ n mɛ tɔ̃́rɩ́ yõ, fɔ́ɔ́ kè pɔ n nɔ gbĩ́ nɛ̂-á Liyel kɩ sépɛ́bɩ́ pól pɩ Krista gbáhkɩ̀ súnsàh-ǹsahlɛ. 
Kũ ye kɩ pɩ sépɛ tɛ́ynɛ, nɛ̂ tyɩ́-á kɩ yɩkɩ. 
Liyel yíyìn pól pɩ Krista gbáhkɩ̀ súnsàh-ǹsahlɛ. Ǹtɛ kǎh yo sɛ̃́, pè náh nɛ Liyel tɔ-á dɛ́, ǹmɔ Liyel nɛ̂-á yíyìn pól pɩ Krista gbáhkɩ̀ súnsàh-ǹsahlɛ. 
Yíyìn pól-à kɩ pɩ Krista gbáhkɩ̀ súnsàh-ǹsahlɛ gbĩ́ nɛ̂nɛ, ǹmɔ Krista nɛ̂-á Pídĩ́nɛ, à kɩ ǹ gblɔ̌y tiki mɔ Liyel yah, ǹmɔ nɛ̂-á yíyìn pól pɩ Krista gbáhkɩ̀ súnsàh-ǹsahlɛ. Képah sɔkɔ, Liyel kɩ tɔ̃́rɩ́ tɔkɔ dal mɔ yíyìn pól yõ. 
Ǹtɛ à, lékyɩ̂-à pɔ n yilki n mɔ n pi ńkɛ̃̂nɛ, névye nónó-á pé gblɔ̌ynɛ n wel n wilki ni sɔkɔ lékyɩ̂ yĩnnɛ, pè se náhkɩ kélɛ n pɩ? Lékyɩ̂ tɛ̃̀-à mɛ pɔ n yilki n mɔ n pi ńkɛ̃̂nɛ, sɔ̃́ pɩ tɛ́ pè pé gblɔ̌ynɛ n wel n wilki ni sɔkɔ pépi yĩnnɛ wɛ́? 
Tɛ́ lékyɩ̂-à kɩ mɛ n yilki n mɔ n pi ńkɛ̃̂nɛ, sɔ̃́ pɩ tɛ́ ápilɛ dlo sɔkɔ gbĩ́mɛgbĩ́ wɛ́? 
Ń yṹnpyé, sɔ̃́-á kélɛ gbɩlɛ nɛ ń nɩ-á mɛ vɩ́nɔ́lɛ yélɛ Yesu Krista yõ sɛ̃́nɔ́ yĩnnɛ, sɛ̃́ tɔ ye kélɛ gbɩlɛ nɛ swɔ̃swɔ̃ póllɛ máh mɛ kũnɛ n dɔ n yah. 
Gbɛ̃́nképèl nónó-á sõ̀ mɛ Yefɛsɩ kwil sɔkɔ, máh mɛ́kɩ̀ fwo sélɛ dùkù nɛ̂nɛ nɛ́npɔ́, kàh sõ̀ mɛ nɛ kèkõyṍ mɛ̀ sɔkɔ min pɩ yĩn tɛ̃̀ ó sɔ̃́, yõre nɛ̂ mɛ mé náhkɩ wɛ ké sɔkɔ? Ǹtɛ kàh mɛ pɩ nɛ lékyɩ̂ tɛ̃̀ náh pɔ n yilki n mɔ n pi, á ye nɛ n yõ à mó nɛ n wɔ kɛ̀, ńsõ̂nɛ á kɩ se pɔ n ku n pi tɛ́. 
Yé káh yé gblɔ̌ynɛ n tɔ̃ n mɔ dɛ́, ǹgbǒ wâh nɛ́túkù pɩ á kɔ́-òlɛ, ǎ á yõ yɩkɩ dɛ́! 
Mé kélɛ n yo yé tyɩ́, túkù mɛ yé sɔkɔ, pè náh Liyellɛ n pnɛ. Ǹtɛ yé pól sen ye kélɛ dò. Yé tɔ̃ n tɛ̃ kègbɩ yõ, yè yèsrey tyilɛ n pɩ kǎh yɩ̃nɛ sɔ̃́, yè mó pèl pɩ́nɔ́ yal n mɔ. 
Ǹnɩ̂ kɩ fɛ̃ n piki n yah nɛ, lékyɩ̂ se kɩ yilki mɔ sɔ̃́ nɩ? Wil gbɛ̃́nsê dùkù nɛ̂ se kɩ mɛ pé tyɩ́ nɩ? 
Lékã́m kɛ̃́ névi! Wâh sèmukɔ pi duku mɔ, kàh fi, dúkúnmɔnɔ mɛ̀ ye ńkɛ̃́nɛ n ku dò. 
Wáh ńkɛ̃́nɛ nɛ̂ duku n mɔ, sèmukɔ pi yɔ́ ó ye. Kè kɩ fɛ̃ pɩ kwlɔlɛ, képah náh pɩ sèmukɔ káhkɩ́lɛ, ńkɛ̃́ yòhkɩ̀ tɛ̃̀ wɔ náh ǹ n duku mɔ kè n fi dɛ́. 
Képah náh, Liyel kè kɔ̃ kè n fi n wil yòhkɩ̀lɛ à yɩ̃nɛ ǹ dyɔ dyɔ tɛ̃̀nɛ. Sèmukɔ pye duke pól yèh mɛ sé tyɩ́ tyɩ́lɛ. 
Wil gbɛ̃́nsê dùkù núkú náh min sɔkɔ yî pól tyɩ́. Névye wil gbɛ̃́nsê mɛ ké tyɩ́lɛ, gbɛ̃́nkyɔ tyɩ́ mɛ ké tyɩ́lɛ, kèpye tyɩ́ mɛ ké tyɩ́lɛ, yɛ̃́nkyɔ tyɩ́ mɛ ké tyɩ́lɛ. 
Yĩ̂nyõ yî tɔ mɛ, sétáh yõ yî mɛ. Ǹtɛ yĩ̂nyõ yî plɛ́nɔ́ mɛ sé tyɩ́lɛ ànɛ̂ sétáh yõ yî tyɩ́lɛ. 
Gbĩ plɛ́nɔ́ mɛ ké tyɩ́lɛ, ńwɔ̃́ plɛ́nɔ́ mɛ ké tyɩ́lɛ, ńwɔ̃́ kèpye plɛ́nɔ́ mɛ sé tyɩ́lɛ. Tɛ́ ńwɔ̃́ kèpye túkù ńtɛ̃ plɛ́nɔ́ tɔ̃ sé gblɔ̌y sé túkù tyɩ́lɛ. 
Sɛ̃́ ye lékyɩ̂ yílkínmɔnɔ tɔ kɩ pɔ n pɩ n pi. Wil gbɛ̃́nsê nɛ̂-á kɩ n pyãh n pi, képah kɩ yɩ̃, tɛ́ kɩ pɔ yilki mɔ pɩ wil gbɛ̃́nsêlɛ nɛ̂ náh tɔ̃ n pyãh n yah n pi. 
Kàh n yɩ̃ gbĩ́ nɛ̂nɛ, kè n nɛ n mɛ kétúkùyɔ̂ sohkɩ kɛ̃́nɛ. Ǹtɛ kàh pɔ yilki mɔ, kè n nɛ n mɛ kètǎr ó, tǎhkɩ̀lɛ ké sɔkɔ. 
Kè n yɩ̃ wil gbɛ̃́nsê gbɔynɛ, ǹtɛ n pɔ n yilki n mɔ wil gbɛ̃́nsê késã́nnɛ Liyel Mirki gbõ̀ yõ. Wil gbɛ̃́nsê gbɔy mɛ, wil gbɛ̃́nsê késã́n tɔ mɛ Liyel Mirki gbõ̀ yõ. 
Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Nɛ́gben nɛ̂-á Adamalɛ, nɛ́gbɔy ye ǹmɔlɛ, nɛ̂-á min dah mɔ ǹ sɔkɔ.» Ǹtɛ Adama tɛ́y wɔlɛ, Liyel Mirki mɛ ǹmɔ sɔkɔ, nɛ̂-á minnɛ n kɔ̃. 
Wil gbɛ̃́nsê késã́n nɛ̂-á wɛ Liyel Mirki gbõ̀ yõ, képah náh pɔ gbennɛ dɛ́, wil gbɛ̃́nsê gbɔy ye pɔ gbennɛ. Képah náh, nɛ̂-á wɛ Liyel Mirki gbõ̀ yõ, képah pyě pɔ. 
Puri ye mɔ Adama gbennɛ, sétáh névi ye ǹmɔlɛ. Adama ní-ò wɔlɛ, ǹmɔ wil yĩ̂nyõ. 
Nɛ̂-á wil sétáh sɔkɔ, sétáh yõ névye mɛ ǹmɔ tyɩ́ kɔ̃lɛ. Nɛ̂-á wil yĩ̂nyõ, yĩ̂nyõ tyɩ́nɔ mó mɛ ǹmɔ tyɩ́ kɔ̃lɛ. 
Áyáh pɩ sétáh yõ névi mɛ̀ kètɔ̀lɛ dùkù nɛ̂nɛ, sɛ̃́ ó ye á kɩ tɔ̃ pɔ n pɩ n pi yĩ̂nyõ névi mɛ̀ tɔ kètɔ̀lɛ. 
Yé yah, nɛ̂-á mé n yo yé tyɩ́ ń yṹnpyé, kè wɛ yɔ̀: Wil gbɛ̃́nsê gbɔy náh fɛ̃ n de n pi Liyel tɔ̃́rɩ́ sɔkɔ. Pyãh pyãh kwâh tɔ náh fɛ̃ tyɛ́-ńkɛ̃̂ min wɛ. 
Yé yah, mé kɩ tir wɔ̃̀bɩ̀nsàhnɔ́ yɔ́ yo yé syah: Á pól tɛ̃̀ wɔ náh n ku n pi dɛ́. Ǹtɛ kèpɔ̃ tɛ́y-à pɔ fɩ̃bɩ, á pól kɩ viri nɛ́npɔ́ swɔ, yɩ̃́ víkényɩkɩnmɔnɔ wrɔ́. Kàh fɩ̃bɩ gbĩ́ nɛ̂nɛ, lékyɩ̂ kɩ yilki mɔ, tɛ́ pè náh tɔ̃ n ku n yah n pi. Ǹtɛ ápi nónó-à mɛ min sɔkɔ, ápi kɩ viri. 
Pyãh pyãh kwâh yɩ̃nɛ kè vi mɔ pɩ pyã́h-ńkɛ̃̂ kwâhlɛ. Tɛ́ ku ku kwâh mó vi mɔ pɩ kú-ńkɛ̃̂ kwâhlɛ. 
Ǹtɛ pyãh pyãh kwâh mɛ̀-à pɔ vi mɔ pɩ pyã́h-ńkɛ̃̂ kwâhlɛ, tɛ́ ku ku kwâh mɛ̀ mɛ vi mɔ pɩ kú-ńkɛ̃̂ kwâhlɛ, légbĩ́nɛ Liyel wɛy sɛ́bɛ́y sɔkɔ tir wã́rkɩ́nsàhnɔ́ mɛ̀ kɩ wil, nɛ̂-á yo nɛ ńkɛ̃́: «Kũ tyɩ́ ye yɩkɩ. Gbõ̀ ye syi ké tyɩ́ pnɛ́pnɛ́.» 
«Kũ, ńyãh mɛ á tɛ̃́nwɛnɔlɛ? Kũ, á kókè mɔ mɔ sõhkɩ ńyãh?» 
Névye tyípékè pɩ́nɔ́ ye kũ sõhkɩlɛ, tyi kõ̀nsàhnɔ́ mó tǎhkɩ̀lɛ tyípékèlɛ n kɔ̃. 
Ǹtɛ á ye wɛkɩ Liyel tyɩ́, ǹmɔ nɛ̂-á tɛ̃́nwɛnɔlɛ á kɔ̃ tyi nɔ́pi yõ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista gbõ̀ yõ. 
Képah ye ń yṹnpyé pèplɔ, yé yé tah n sah yé nɛ mɛ yĩ́nnɔ́lɛ gbáhkɩ̀ núkú yõ. Yé wãnnɛ n mɔ yé gbáhntɛ̃nɔ yõ Tɛ̃̀ ǹgbɛ̃ tõ̀ pɩ́nɔ́ sɔkɔ. Yè mó pɩ̃ nɛ yáh kwlɔ́nɔ́ nɛ̂nɛ n pɩ Tɛ̃̀ ǹgbɛ̃ tõ̀ sɔkɔ, kè náh n pɩ n pi nɔ̀nɔ́lɛ.
