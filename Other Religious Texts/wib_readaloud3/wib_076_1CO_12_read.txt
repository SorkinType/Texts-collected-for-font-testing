Korɛ̃nti tãn.
12.
Ń yṹnpyé, Liyel Mirki-á tyi nónó pɩ́nɔ́ yõ̀bènɔrɩ tɛ̃̀ wɔlɛ névilɛ n kɔ̃, mé náh mɛ ké tyɩ́ nɛ yé nɛ n mɛ sé yɩ̃́tũri sɔkɔ. 
Yépi ńtɛ̃ pɩ̃ nɛ gbĩ́ nɛ̂-á lésõ yé náh tɔ̃̌nɩ̀ Liyel pɩ̃, vì-ǹsah yɔ́ ńtɛ̃ náh sõ̀ mɛ yé tyɩ́ nɛ́ tã̀n yah ó tónkɔ̃nɔ náh tɔ̌ɔ̀, sépi nónó-á sè náh fɛ̃ n wɛkɩ. 
Képah ye, mé kélɛ n yo n yĩ́ĩ́ yé tyɩ́ nɛ Liyel Mirki-à mɛ névi nɛ̂ yahlɛ n tɛ̃, tɛ̃̀ náh fɛ̃ n yo n pi nɛ ńkɛ̃́ Yesu tyɩ́ yɩkɩ; ànɛ̂ nɛy tɔ náh fɛ̃ n yo n pi nɛ ńkɛ̃́ Yesu-á TƐ̃̀ ǸGBƐ̃NƐ nɛ́ Liyel Mirki náh tɛ̃ tɛ̃̀ yahlɛ. 
Tyi duke náhnáh pɩ́nɔ́ yõ̀bènɔrɩ kɩ fɛ̃ wɛ, ǹtɛ kénɛ Liyel Mirki núkúní ó ye sélɛ névyelɛ n kɔ̃. 
Liyel tã̀n duke mɛ dò, ǹtɛ Tɛ̃̀ ǹgbɛ̃ núkúní tyɩ́ ye sé póllɛ á n pɩ. 
Tã̀n pɩ dùkùnɔ́ mɛ dò, ǹtɛ Liyel núkúní ó ye sé pól pɩ́nɔ́ tǎhkɩ̀lɛ nɛ́núkù núkú póllɛ n kɔ̃. 
Liyel Mirki ye tir fɛ̃̀npɩnɔ yõ̀bènɔrɩlɛ nɛ́núkù núkú póllɛ n kɔ̃, no pól tyɩ́ pɛkɩ yĩnnɛ. 
Liyel Mirki ńkɛ̃́nɛ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, à n nɛ n wɛkɩ yɩ̃́pɩ̃̂ sɔkɔ, tɛ́ kénɛ Mirki ó ńtɔ̃ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ tyilɛ n pnɛ n yo. 
Kénɛ Liyel Mirki ó ye tɔ̃ ńkɛ̃́nɛ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, à n sɛ̃ yĩ́ĩ́ Tɛ̃̀ ǹgbɛ̃ yõ, kénɛ Liyel Mirki núkúní n tɔ̃ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ yámpúlɛ n dɛ n mɔ. 
Kě yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ gbǐl tyilɛ n pɩ, ǹ tɛ́ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ Liyel tɛ̃ntõ̀lɛ n syi n yo. Kě yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ Liyel Mirki tyípɩ́nɩ̀nɛ n wah n pnɛ sétã̀n tyɩ́ sɔkɔ. Kě yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, ǎ wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á à náh kè sãh, ǹ tɛ́ yõ̀bènɔrɩlɛ ǹnɩ̂ kɔ̃, à n fɛ̃ sé yahlɛ n yo. 
Kénɛ Liyel Mirki núkúní ó tõ̀ vyãh ye sépi póllɛ. Kě tyi fɛ̃̀npɩnɔ yõ̀bènɔrɩ dibi nɛ́núkù núkù pól kɔ̃, à yɩ̃nɛ nɛ̂-á ké dyɔ. 
Névi wil gbɛ̃́nsêlɛ, núkú ye kélɛ. Ǹtɛ tã̀ntã̀n mɛ dò ké tyɩ́. Ké tã̀ntã̀n dò fyé yõ, wil gbɛ̃́nsê núkú ye kélɛ. Névye nónó tɔ-á mɛ kwrɔ́nmɔnɔlɛ Yesu Krista sɔkɔ, sɛ̃́ ó ye pépi tɔlɛ. 
Yé náh wɛ di, kè se Nsyifunɔ ò, kè se nónó náh pɩ Nsyifunɔlɛ ò, kè se yɔ̃bɩ ò, kè se gblɔ̌y yõ névye ò, Liyel Mirki núkú mɛ̀ ó gbõ̀ yõ sɔkɔ ye á pól wèlnwìlkìnɔ́ wɛ, à pɩ wil gbɛ̃́nsê núkúlɛ. Kènɛ Liyel Mirki mɛ̀ ye mɛ á pól sɔkɔ. 
Wil gbɛ̃́nsêlɛ, tã̀ntã̀n mɛ dò ké tyɩ́, fɩ̃́ núkú ó náh ké tyɩ́. 
Gbáhkɩ̀-à sõ̀ mɛ nɛ péwɔ náh fyɔ̀ pɩ gbõ̀lɛ, wil gbɛ̃́nsê tyah yɔ́ náh mó pélɛ, képah se náhkɩ kè kɔ̃ kè náhkɩ gbɛ wilki wil gbɛ̃́nsê sɔkɔ? 
Níkí-à sõ̀ mɛ nɛ péwɔ náh fyɔ̀ pɩ yĩ́nɛ, wil gbɛ̃́nsê tyah yɔ́ náh mó pélɛ, képah se náhkɩ kè kɔ̃ kè náhkɩ gbɛ wilki wil gbɛ̃́nsê sɔkɔ? 
Wil gbɛ̃́nsê pól-à sõ̀ mɛ yĩ́nɛ, kè náhkɩ n nohnɩ sɔ̃́? Ké pól-à sõ̀ mɛ níkílɛ, kè náhkɩ pãnnɛ n tɔkɔ sɔ̃́? 
Ǹtɛ Liyel ye wil gbɛ̃́nsê tã̀ntã̀n pól mɔ ké sɔkɔ, à yɩ̃nɛ ǹ dyɔ dyɔ tɛ̃̀nɛ. 
Wil gbɛ̃́nsê pól-à sõ̀ mɛ fɩ̃́ núkú ólɛ, wil gbɛ̃́nsê náh náhkɩ n mɛ n pi. 
Ǹtɛ sɛ̃́ náh dɛ́, wil gbɛ̃́nsêlɛ, núkú ye kélɛ, tɛ́ tã̀ntã̀nnɛ dò ké tyɩ́. 
Kǎh pɩ sɛ̃́, yĩ́ náh fɛ̃ n yo n pi gbõ̀ tyɩ́ nɛ, ké yétõ̂ náh mɛ pé tyɩ́. Yõ tɔ náh fɛ̃ n yo n pi gbyẽ̂h tyɩ́ nɛ, sé yétõ̂ náh mɛ pé tyɩ́. 
Ǹtɛ wil gbɛ̃́nsê tã̀ntã̀n nónó ńtɛ̃ wɔ-á pyě á kɩ nɛ sohkɩ kósyì mɛ sé sɔkɔ, yétõ̂ mɛ sé tyɩ́. 
Wil gbɛ̃́nsê tã̀ntã̀n nónó-á sè náh mɛ n mɔ n kõ n yĩ́ĩ́, sépi ye pyě à n nɛ n tɔ̃ n sah n tɛ̃. Sé nónó-á sé tyɩ́ yónɔ́ mɛ sennɛ, à n nɛ n yah n dah n mɔ yĩ́ĩ́ sépi tyɩ́ sɔkɔ, 
tɛ́ nónó wɔ-á mɛ pallɛ, à n pyě sépi sah. Wil tã̀ntã̀n nónó-á á náh mɛ sélɛ n mɔ n kõ kwâh yɔ́lɛ, Liyel-á wil gbɛ̃́nsê mɔ, à gbílkínɔ́ sah kal sépi yõ. 
Kǎh pɩ sɛ̃́, wil gbɛ̃́nsê tã̀ntã̀n náh wɛ̃nɛ n wɛ n mɔ. Sé pól mɛ wɛ̃ tɛ̃́nɔ́ plɛ dùkù núkúlɛ. 
Ké tyah yɔ́-à mɛ kwlɔ́nɔ́ sɔkɔ, sé tɛ́lɔ́ tɔ n nɛ n mɛ n kwlɔ kélɛ wɛ̃ tyɩ́. Tɛ́ gbílkínɔ́-à mó n sah ké tyah yɔ́ yõ, sé tɛ́lɔ́ tɔ n mó ké nɩ̀vɩ̀nɩ̀ pɩ kélɛ wɛ̃ tyɩ́. 
Ǹtɛ yé yah kɛ̀, yépi ye Krista wil gbɛ̃́nsêlɛ dò. Yé nɛ́núkù núkú póllɛ, Krista wil gbɛ̃́nsê fɩ̃́ yɔ́ ye yélɛ. 
Yé náh wɛ di, Liyel mɛ nósyɔ́ wilki sah Yesu yõ sɛ̃́pú tĩ̀nkì sɔkɔ pé tyɩ́ tyɩ́lɛ: Gbennɛ, à Yesu tɛ̃ntõ̀ névye yah wilki. Ní-òlɛ, à Liyel tɛ̃ntõ̀ syínyopu yah wilki. Tɔ̃́-òlɛ, tɛ́ kwɔ́pú yah wilki. Képah náh, gbǐlnpɩbɩ mɛ syɩ. Tɛ́ mɛ yámpú fɛ̃̀ndɛnmɔnɔ yõ̀bènɔrɩlɛ túkù kɔ̃, tɛ́ névye yòhnɩ̀nmɔnɔ yõ̀bènɔrɩlɛ túkù kɔ̃, tɛ́ névye yah tɛ̃́nɔ́ yõ̀bènɔrɩlɛ túkù kɔ̃, tɛ́ yõ̀bènɔrɩlɛ túkù kɔ̃ pè fɛ̃ wɛ́nkáhkɩ́nɔlɛ n wɛkɩ nónó-á pè náh sè sãh. 
No pól se yɔ́ pɩ Yesu tɛ̃ntõ̀ névyelɛ, képah náh pɩ Liyel tɛ̃ntõ̀ syínyopulɛ, képah náh pɩ kwɔ́púlɛ, képah náh pɩ gbǐlnpɩbɩlɛ? 
Tĩ̂, no pól se yɔ́ pɩ yámpú dɛ́nmɔpulɛ, képah náh pɩ wɛ́nkáhkɩ́nɔ wɛ́kɩ́púlɛ nónó-á pè náh sè sãh, képah náh pɩ wɛ́nkáhkɩ́ nɔ́pi yah yópúlɛ? 
Kǎh pɩ sɛ̃́, tir nɛ̂ fɛ̃̀npɩnɔ-á kɩ névi yohnɩ mɔ kal, yé képah yõ̀bènɔrɩ yah n kɔ̃. Ǹtɛ mé kɩ tɔ̃ yé wɛ̃kɩ wɛ̃̀kɩ̀ yɔ́lɛ nɛ̂-á plɛ kal sé tɛ́lɔ́lɛ.
