Wrome tãn.
14.
Névi nɛ̂ Liyel yõ sɛ̃́nɔ́-à mɛ yúkú-ńkɛ̃̂nɛ yahlɛ, yé tɛ̃̀ tɛ̃́nɔ́ plɛ. Yé káh tɛ̃̀ sõnɔlɛ n kɔ̃lɩ. 
Sɔ̃́ kénkɔ̃-á ǹnɩ̂ Liyel yõ sɛ̃́nɔ́-à yuku yahlɛ tɛ̃̀ kɔ̃ à mɛ fɛ̃ yíyìn póllɛ n yõ, tɛ́ nɛ̂ Liyel yõ sɛ̃́nɔ́ náh mó yuku yahlɛ, kè mɛ tɛ̃̀ kɔ̃ à mɛ kɔ̀ yahle; 
yíyìn pól yṍ-ò káh kɔ̀ kɔ̀-ǹkɛ̃̂ névilɛ n yilɔ, kɔ̀ kɔ̀-ǹkɛ̃̂ névi tɔ káh mó yíyìn pól yṍ-òlɛ n tuke. Yé náh wɛ di, Liyel mɛ tɛ̃̀ tɛ̃ plɛ dò. 
Ǹtɛ nɛ̂ mɛ áwɔlɛ, tɛ́ á kɩ pɔ ǹnɩ̂ yɔ́ tõ̀npɩlɛ n tuke? Wàh kɩ sɔ̃́ mɛ n pɩ plɛ ò, à náh mɛ n pɩ plɛ ò, képah mɛ ǹ yõ̀tɛ̃̀ dĩ́nɛ n yah kɛ̀. Ǹtɛ à kɩ pɩ plɛ n pi. Yé náh wɛ di, tǎhkɩ̀ mɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́ kɩ fɛ̃ à kɔ̃ à kɩ n pɩ plɛ. 
Túkù tyɩ́ sèpĩ̀n mɛ sé tyɩ́ tyɩ́lɛ, túkù tyɩ́ sè mɛ núkúlɛ. Ǹtɛ nɛ̂-à sɔ̃́ mɛ nɛ̂nɛ n sõ, tɛ̃̀ sɛ̃́nyĩnɔ nɛ n mɛ ké yõ. 
Névi nɛ̂-á mɛ sèpĩ̀nnɛ n mɔ n kõ, Tɛ̃̀ ǹgbɛ̃ tyɩ́ ye tɛ̃̀ n pɩ kélɛ. Nɛ̂-à mɛ yíyìn póllɛ n yõ, Tɛ̃̀ ǹgbɛ̃ tyɩ́ ye tɛ̃̀ n pɩ kélɛ. Yé náh wɛ di, Liyel yĩn ye tɛ̃̀ n gbilki. Nɛ̂-à se yíyìn póllɛ n yṍ-ńkɛ̃̂nɛ, Tɛ̃̀ ǹgbɛ̃ tyɩ́ ye tɛ̃̀ n pɩ kélɛ. Liyel yĩn ye tɛ̃̀ tɔ n gbilki. 
Yé náh wɛ di, ápi sɔkɔ yɔ́ ńtɛ̃ náh mɛ min sɔkɔ ǹ gblɔ̌y yĩnnɛ, ànɛ̂ ǹnɩ̂ yɔ́ ńtɛ̃ náh n ku n pi ǹ gblɔ̌y yĩnnɛ. 
Áyâh mɛ min sɔkɔ, Tɛ̃̀ ǹgbɛ̃ tyɩ́ yõ ye álɛ min sɔkɔ. Áyâh se ku, ǹmɔ tyɩ́ yõ ye á ku. Képah ye, á mɛ min sɔkɔ ò, á ku ò, Tɛ̃̀ ǹgbɛ̃ tyɩ́ yõ ye álɛ. 
Krista ye ku tɛ́ ǹ lékó mɛ yilki mɔ, nɛ̂-á à kɔ̃ à pɩ lékyɩ̂ ànɛ̂ vye Tɛ̃̀ ǹgbɛ̃nɛ. 
Ǹtɛ áwɔ-á dò, kwâh nɛ̂ pɩ tɛ́ á á yṹnpilɛ n tuke? Tɛ́ áwɔ tɔ-á dò, kwâh nɛ̂ pɩ tɛ́ á á yṹnpilɛ n yilɔ? Liyel kɩ pɔ á pól tuke n pi dò. 
Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Ńmɔ nɛ̂-á TƐ̃̀ ǸGBƐ̃NƐ mé mɛ min sɔkɔ. No pól kɩ gbètukunɔ di ńmɔ yah sɔkɔ, ànɛ̂ no pól kɩ ńmɔlɛ n gbilki.» 
Kǎh pɩ sɛ̃́, á nɛ́núkù núkú pól kɩ á tyípɩ́nɔ́ yah yo n pi Liyel tyɩ́ á gblɔ̌y. 
Képah ye, á ye yal mɔ wɛ̃ tùkènɔ́lɛ, yè mó kè yah tɔkɔ nɛ ǹnɩ̂ yɔ́ ńtɛ̃ káh tir yɔ́ pɩ nɛ̂-á kɩ yé yṹnpi yɔ́ yõ yɩkɩ, képah náh pɩ tɛ̃̀ syɩkɩ mɔ Liyel yõ sɛ̃́nɔ́ sɔkɔ. 
Á wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Tɛ̃̀ ǹgbɛ̃ Yesulɛ, ńmɔ kè pɩ̃, ànɛ̂ ń sɛ̃́nyĩnɔ mɛ ké yõ sɔkɔ nɛ ńkɛ̃́, kwâh yɔ́ ńtɛ̃ náh mɛ sìlkìnɔ́lɛ Liyel yah sɔkɔ ké gblɔ̌y sɛ̃́. Ǹtɛ névi nɛ̂-à n sõ nɛ kwâh nɛ̂-á mɛ sìlkìnɔ́lɛ, kè n pɩ sìlkìnɔ́lɛ tɛ̃̀ tyɩ́ Liyel yah sɔkɔ. 
Káh n pɔ á yõke yĩnnɛ á yṹnpi mɔ yèkwɔrɩ sɔkɔ, á náh tɔ̃ mɛ n kɔ tyɩ́nɔnɔ wɛ̃̀kɩ̀ yõ dò. Káh n pɔ á yõke yĩnnɛ névi min kɔ̃ kè mɔ pu, Krista-á ku ǹmɔ nɛ̂ yĩnnɛ. 
Kwâh nɛ̂-á kwâh pèpɛynɛ áwɔ tyɩ́, képah káh pɔ pɩ yĩ́nyɩ́kɩ̀lɛ dɛ́. 
Yé náh wɛ di, Liyel tɔ̃́rɩ́lɛ, yõke ànɛ̂ wɔ wɔ yî náh. Gbɩ wɛ̃̀kɩ̀, ànɛ̂ yèvɩnɩ, ànɛ̂ nɩ̀vɩ̀nɩ̀ ye kélɛ Liyel Mirki gbõ̀ yõ. 
Névi nɛ̂-à tõ̀lɛ n pɩ Krista tyɩ́ sɛ̃́ntɛ̃̀nɛ, Liyel yĩ́ n yuku tɛ̃̀nɛ, ànɛ̂ névye ǹ yĩ́ngbɩ́lɛ n yo. 
Képah ye, áyáh nɛ yèvɩnɩ ànɛ̂ wɛ̃ yah yúkúnɔ́ wɛ n pi sɔ̃́, á ye képahlɛ n yɩ̃kɩ. 
Á náh yɩ̃nɛ ǹ pɔ Liyel tõ̀lɛ nɛ n yɩkɩ yõke yĩnnɛ. Gbɩ ye, yíyìn pól tɛ̃̀ wɔ mɛ yályál Liyel yah ó; ǹtɛ wâh kwâh yɔ́lɛ n yõ nɛ̂ yṍnɔ́-á kɩ fɛ̃ pɔ á yṹnpi yɔ́ syɩkɩ mɔ Liyel yõ sɛ̃́nɔ́ sɔkɔ, képah náh plɛ. 
Nɛ̂-á ké pèpɛynɛ, tir nɛ̂-á kɩ fɛ̃ á yṹnpi syɩkɩ mɔ Liyel yõ sɛ̃́nɔ́ sɔkɔ, kè se kɔ̀ kɔ̀nɔ́lɛ ò, kè se tɔ̃ wɔ́nɔ́lɛ ò, kè se tyi dùkù nɛ̂nɛ, káh kélɛ n pɩ. 
Áwɔ sɛ̃́nɔ́-á mɛ nɛ̂ yõ à yuku nɛ́npɔ́ tĩ̀nnɛ, yɛ képah tɛ̃ áwɔ ànɛ̂ Liyel wrɔ́. Nɛ̂-à tyilɛ n pɩ wǎh mɛ nónónɛ n yah nɛ sǎh mɛ plɛ, tɛ́ à náh mɛ lékàhlɛ ǹ gblɔ̌ynɛ n kɔ̃, tɛ̃̀ tyɩ́ nɔ. 
Ǹtɛ névi nɛ̂-à kwâh yɔ́lɛ n yõ tɛ́ mɛ ké tyɩ́lɛ n sõ tɔ̀tɔ̀, lékàh mɛ tɛ̃̀ tyɩ́ Liyel yah sɔkɔ, wǎh sɔ̃́npɩ́lɛ kélɛ n pɩ tɛ́ kè náh mɛ pɔ yɩ̃nɛ Liyel yõ sɛ̃́nɔ́ wɛ̃̀kɩ̀lɛ. Tyi nónó pól-á n pɩ nónó náh mɛ Liyel yõ sɛ̃́nɔ́ wɛ̃̀kɩ̀ yõ, tyípêl ye sélɛ.
