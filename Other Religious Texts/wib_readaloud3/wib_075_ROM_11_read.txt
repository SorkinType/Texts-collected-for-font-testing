Wrome tãn.
11.
Ǹtɛ-à, Liyel se ǹ náh vi ǹ no Yisrayel tãn kɔ̃? Ɔ̃́ɔ̃ dɛ́! Yé náh wɛ di, ńmɔ tɔlɛ Yisrayel nɛy ye dò. Abrahmɩ náh nɛy yɔ́ ye ńnɛ, nɛ̂-á wil Bɛ̃nsyamɛ yìkì sɔkɔ. 
Liyel náh ǹ náh vi ǹ no kɔ̃ wǎh kwãh tɛ̃ nónó yah wilki sah dɛ́. Yé se kɩ nɛ yé pɩ̃́nɔ́ náh mɛ ké sɔkɔ nɛ, kǎh mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ Yeli-á sõ̀ mɛ n tɔkɔ Yisrayel nolɛ n kɔ̃ Liyel tyɩ́ 
nɛ ńkɛ̃́: «Tɛ̃̀ ǹgbɛ̃, pè ye á tɛ̃ntõ̀ syínyopu ko mɔ, tɛ́ sárká súkɛ́nkɔ̃-ńtã̂n yɩkɩ nónó-á mɛ áwɔ tyɩ́ yõ. Ńmɔ ó wũ ń gblɔ̌y, tɛ́ pè tɔ̃ mɛ n yah n kɔ̃ ń kònmɔ-ńsahlɛ.» 
Ǹtɛ sɔ̃́ mɛ Liyel yo ǹ tyɩ́? À nɛ: «Mé ye díbí nɛ̂pĩ̂ kèyõ tɔ̃́nɔ́ kyɛgbãm (7 000) wilki sah ń gblɔ̌y tyɩ́, nónó-á pè náh gbètukunɔ di fal fal tõ̀ Baal yah sɔkɔ à kè gbilki.» 
Képah núkú ó ye kwɛy mɛ̀ tɔ sɔkɔ, névye kwéy yísyɔ́ mɛ wũ tɛ̃ Liyel-á yõ̀bènɔrɩ pɩ nónó tyɩ́ à pè yah wilki. 
Ǹtɛ yõ̀bènɔrɩ-à kɩ nɛ̂ Liyel pɩ pé tyɩ́ à pè yah wilki, pé tyɩ́pɩ́nɔ́ gbõ̀ yõ náh mó tɔ̃ kɛ̀. Kàh mɛ sépi gbõ̀ yõlɛ, yõ̀bènɔrɩ náh mó tɔ̃ dò. 
Sɔ̃́ mɛ ápi kɩ yo yɔ̀ wɛ́? Yisrayel no-á nɛ̂nɛ n yah n kɔ̃, pé pól náh kè wɛ. Ǹtɛ Liyel-á nónó yah wilki pé sɔkɔ, pépi ó wɛ kélɛ tɛ́ tɛ́lɔ́ wɔ ní mɛ tah tũ 
sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «Liyel ye pè pɩ tõ̌rɔ̀ɔ̀. À pé yɩ̃́ pɩ sè náh mɛ n yah, tɛ́ pé ní pɩ sè náh mɛ n nohnɩ, fɔ́ɔ́ à pɔ syɩ kwɛy mɛ̀ tyɩ́.» 
Yõ̀tɛ̃̀ dĩ́ Davidi tɔ tɔ̃ kè yo nɛ ńkɛ̃́: «Pé yõke yɩ̃nɛ kè pɩ pé tyɩ́ pé tɛ̃ tɛ̃ vyɔ́nɛ, ànɛ̂ pé tɛ̃ tɛ̃ kɛ̃́ynɛ, ànɛ̂ pé wilki n mɔ mɔ kwâhlɛ, kɩ pé tyítúkúnìnɛ pè dyah. 
Pé yɩ̃́ mó nɛ mɛ syìì, pè káh fɛ̃ nɛ n yah, pè mó nɛ mɛ kwlɔ́nɔ́ sɔkɔ gbĩ́mɛgbĩ́ tɔ́kɔ́bɩ́ sɔ̃́ mɛ pé yõ. 
Ǹtɛ-à, Yisrayel no-á pé gbáhkɩ̀ mɩ syɩkɩ sɛ̃́-à, pè se kɩ wũ tɛ̃ sɛ̃́ kṹmm? Sɛ̃́ tɛ̃̀ wɔ náh dɛ́. Ǹtɛ pépi-á pilki, yìkìnì tɛ́lɔ́ mó pwáhnmɔnɔ wɛ, pépi Nsyifunɔ kɔ̃ pè pé yèwannɛ n yah. 
Ǹtɛ pépi-à pílkínɔ́-à kɩ mɛ pɩ pèpɔrɩ wɛ́nɔ́lɛ kèkõyṍ névye tyɩ́, tɛ́ pé tɛ̃́nkwlɔnɔ mɛ pɩ pèpɔrɩ wɛ́nɔ́lɛ yìkìnì tɛ́lɔ́ tyɩ́ à, pépi-à pwáhnmɔnɔ wɛ tɛ, képah náh n pɩ n pi pèpɔrɩ ǹgbɛ̃nɛ kal di? 
Yépi nónó náh pɩ Nsyifunɔlɛ, yépi tyɩ́ ye mé n yo wɛy mɛ̀nɛ núkúnúkú. Yesu tɛ̃ntõ̀ névi ye ńmɔlɛ nɛ̂-á mɛ tõ̀lɛ n pɩ yépi yĩnnɛ. Tõ̀ mɛ̀ mɛ yõ yɔ̃̀nɔ́lɛ ńmɔ tyɩ́, 
nɛ̂-á kɩ á no Nsyifunɔ kɔ̃ kè kɩ yìkìnì tɛ́lɔ́ yèwannɛ n yah, ń kɔ̃ mé kɩ pé túkù wɛ pwah mɔ. 
Yé náh wɛ di, Liyel-á pè gbɛ milki képah-à mɛ kèkõyṍ névye wrɔ́ tɔ̃ sah Liyellɛ à, wàh tɔ̃ pɔ pé ye kwrɔ́ ǹnɛ, kè kɩ pɩ sɔ̃́ wɛ́? Kè náh n pɩ n pi á kɩ nɛ lékyɩ̂ sɔ̃́ yilki mɔ di? 
Sèmukɔ pye nónó-á ko gbrolɛ, sépi-à kɩ mɛ wilki sah Liyel tyɩ́ yõ, sé tɛ́lɔ́ pól tɔ mɛ ǹ tyɩ́ yõ. Séswɔ̂ sémîn-à mɛ sàhnɔ́lɛ ǹmɔ tyɩ́ yõ, ké yɩ̃̀ngbã̀n tɔ kɩ mɛ ǹ tyɩ́ yõ. 
Nsyifunɔ mɛ wólívî séswɔ̂ sṍnɔ́ sɔ̃́, tɛ́ Liyel mɛ ké yɩ̃̀ngbã̀n kɔ̃lɩ. Tɛ́ nónó náh pɩ Nsyifunɔlɛ, pépi mó mɛ á kɩ nɛ kyẽh sɔkɔ wólívî séswɔ̂ sɔ̃́, nɛ̂-á Liyel ké yɩ̃̀ngbã̀n kɔ̃lɩ kyɩ pukubi wólívî yòhkɩ̀ sṍnɔ́ tyɩ́. Yáh Liyel yõ̀bènɔrɩ nɛ̂ wɛ, képah n wil Nsyifunɔ tyɩ́ sɔ̃́-á wólívî séswɔ̂ sṍnɔ́ nìkì n yohnɩ n mɔ yɩ̃̀ngbã̀n púkúbínɔ́lɛ. 
Képah ye, yɩ̃̀ngbã̀n nónó-á Liyel kɔ̃lɩ kwɛ, yé káh sélɛ n yilɔ. Yé káh yé gblɔ̌ynɛ n yɔ̃ n sah, yépi tyɩ́ náh sémînnɛ, pépi tyɩ́ ye sélɛ. 
Yé kɩ fɛ̃ ń syah nɛ, Liyel-á yɩ̃̀ngbã̀n nɔ́pi kɔ̃lɩ to tɛ́ yépi wɛ pukubi. 
Gbɩ tɛ̃̀ wɔ ye, pépi Liyel yõ sɛ̃́-ńkɛ̃̂ ye Liyel kɔ̃ à pè kɔ̃lɩ to tɛ́ yépi wɛ pukubi pé swɔ sɔkɔ yé Liyel yõ sɛ̃́nɔ́ gbõ̀ yõ. Yé náh yɩ̃nɛ yè yé gblɔ̌ynɛ n gbilki. Kè pyě yɩ̃nɛ kè nɛ n mɛ tíkílɛ yé tyɩ́ ńtɛ̃nɛ ńkɛ̃́. 
Yé náh wɛ di, Liyel-à mɛ yɛ̀-ǹkɛ̃̂nɛ séswɔ̂ sṍnɔ́ yɩ̃̀ngbã̀n kɔ̃̀lɩ̀nɔ́lɛ, yépi wɔ se pyě à kɩ wɛ yɛ yâh sɛ̃́-ńkɛ̃̂nɛ ǹ yõ? 
Yé yah, Liyel plɛ, tɛ́ mó tɔ̃ peke. À peke pílkípú yõ, tɛ́ n pɩ plɛ yépi tyɩ́. Ǹtɛ yé yɩ̃nɛ yè nɛ n mɛ ǹ pèpɔrɩ mɛ̀nɛ n pnɛ gbĩ́mɛgbĩ́. Képah-à pɩ́-ńkɛ̃̂nɛ, yé tɔ kɩ kɔ̃lɩ milki. 
Ǹtɛ Nsyifunɔ tɔ-à kɩ pé kèlékɛ̃ pɩ́nɔ́ yal mɔ tɛ́ sɛ̃ Liyel yõ, Liyel kɩ pè tɔ̃ pukubi séswɔ̂ sṍnɔ́ tyɩ́. Tɛ̃́nwɛnɔ mɛ ǹmɔ tyɩ́ kɩ ǹ syɩ pè tɔ̃ pukubi. 
Yé náh wɛ di, yépi nónó náh pɩ Nsyifunɔlɛ, yé mɛ á kɩ nɛ kyẽh sɔkɔ wólívî séswɔ̂ sɔ̃́, nɛ̂-á Liyel ké yɩ̃̀ngbã̀n kɔ̃lɩ kyɩ pukubi wólívî yòhkɩ̀ sṍnɔ́ tyɩ́. Nsyifunɔ mó mɛ wólívî séswɔ̂ sṍnɔ́ yɩ̃̀ngbã̀n tyɩ́ kɔ̃lɛ nónó-á Liyel kɩ pè tɔkɔ pukubi sé séswɔ̂ tyɩ́. 
Ń yṹnpyé, mé kɩ Liyel tir wɔ̃̀bɩ̀nsàhnɔ́ yɔ́ tyɩ́ yah yo wah yé syah, sɔ̃̀nko yè káh n pɔ yé gblɔ̌ynɛ n tɔkɔ n pɩ yɩ̃ pɩ̃́púlɛ. Yé yah, Yisrayel no túkù kɩ wũ tɛ̃ níkísekɔ́kɔ́ pɩ́nɔ́ sɔkɔ, pópó kɩ pɔ nɔ gbĩ́ nɛ̂-á nónó náh pɩ Nsyifunɔlɛ, pè kɩ pwáhnmɔnɔ wɛ, tɛ́ kɩ de Liyel no sɔkɔ. 
Képah sɔkɔ, Yisrayel no pól kɩ pyě pwah sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Pwáhnmɔ-ò kɩ wil Siyõ kwil sɔkɔ. À kɩ pɔ Nsyakɔbɩ no tyípêl tɔ́kɔ́bɩ́ tɛ̃ yõ pé yõ. 
Ńmɔ Liyel-á kɩ pɔ pé tyípêl yɔ̃ mɔ pè kɔ̃, tɛ́ kɩ yégbékè nɛ̂ kwrɔ n mɔ n pi pélɛ, képah ye yɔ̀.» 
Yah yɔ́lɛ, Liyel sépɛbɩ́ ye Yisrayel nolɛ pǎh sɔ̃́npɩ́lɛ Nɩ̀vɩ̀nɩ̀ tir yahle, tɛ́ képah wɔ mó nɛkɩ yépilɛ. Ǹtɛ yah yɔ́lɛ, pépi ye sõ̀ à yah tɔkɔ pɩ ǹ nolɛ. Pé tyɩ́ nɔ ǹ tyɩ́ pé náh tãn yĩnnɛ. 
Yé náh wɛ di, Liyel-à kwâhlɛ névi kɔ̃, à náh ǹ yahlɛ n vi kélɛ n syi póllɛ. Wàh névi nɛ̂ ye, à náh ǹ nahlɛ n vi tɛ̃̀nɛ n kɔ̃ póllɛ. 
Ǹgbòlɛ, yé lésõ mɛ Liyel yóré yahle. Ǹtɛ núkúnúkú, Nsyifunɔ tɔ-á ǹ yóré yahle, à mɛ yépi yãm yah. 
Sɛ̃́ ó ye, Liyel yóré nɛ̂-á pè yahle núkúnúkú yɔ̀, tɛ́ Liyel mɛ yépi yãm yah, pépi tɔ yãm yah yĩn ye à pɩ képahlɛ. 
Yé náh wɛ di, Liyel yɛ no pól kɔ̃ pè ǹ yóré yahle pé pól yãm wɛ n yah yĩnnɛ. 
Liyel pèpɔrɩ mɛ gblò dɛ́! Lékã́m mɛ yĩ́ĩ́ ǹ sɔkɔ, ǹ fɔkɔ se mó tɔ̃ mɛ dényĩnɔlɛ. Ǹ tyi tɛ̃ dùkù náh fɛ̃ n pɩ̃, ǹ syɔ̃̀-ǹtã̀n tɔ náh fɛ̃ n pɩ̃ lɔ́. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Nɛ̂ tɛ̃̀ wɔ mɛ Tɛ̃̀ ǹgbɛ̃ sõnɔlɛ n pnɛ wɛ́? Nɛ̂ tɛ̃̀ wɔ kɩ fɛ̃ vìnmɔnɔlɛ à kɔ̃? 
Nɛ̂ kwâh yɔ́lɛ à kɔ̃, tɛ́ à pɔ ké lédyáhnɩ́nɛ tɛ̃̀ kɔ̃ náhlɛ?» 
Yé náh wɛ di, yíyìn pól wil ǹmɔ tyɩ́. Ǹmɔ mɔ́nɔ́ ye sé póllɛ. Sé pól mɛ ǹmɔ tyɩ́ yõ. Gbílkínɔ́ nɛ n sah ǹ yõ gbĩ́mɛgbĩ́. Amiina!
