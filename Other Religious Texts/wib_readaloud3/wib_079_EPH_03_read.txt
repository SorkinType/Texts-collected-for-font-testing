Yefɛsɩ tãn.
3.
Képah ye, ńmɔ Pole nɛ̂-á mɛ wɔ̀kɔ̀npèkè sɔkɔ Yesu Krista tõ̀ yĩnnɛ, ànɛ̂ yépi yĩnnɛ, yépi nónó náh pɩ Nsyifunɔlɛ... 
Liyel-á ǹ pèpɔrɩ pɩ́nɔ́ sɔkɔ tõ̀ nɛ̂ dahbɩ ń gbõ̀ yépi tyɩ́ pɛkɩ yĩnnɛ, mé sɛ̃ ké yõ nɛ yáh képah tɛ̃̀ wɔ noh ó. 
Liyel ye syɔ̃ wɛ̃́kɩ́nɔ́ yɔ́ tyɩ́ à ǹ sètĩ̀n sõ̀ tirlɛ ń wɛ̃kɩ, sɔ̃́-á mé yo sɔkɔ kélɛ yé tyɩ́ kwéy sɛ́bɛ́y mɛ̀ yõ̀tah sɔkɔ yɔ̀. 
Yáh kɩ kélɛ n kar, yé kɩ pɩ̃ nɛ sètĩ̀n sõ̀ tir nɛ̂-á mɛ Krista tyɩ́ sɔkɔ, ké sèndyàh náh ńnɛ. 
Liyel náh sõ̀ kénɛ sètĩ̀n sõ̀ tirlɛ ǹgbò tãn wɛ̃kɩ, sɔ̃́-á à kélɛ Yesu tɛ̃ntõ̀ névye ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu nɛ́gblɔ́ wɛ̃kɩ núkúnúkú ǹ Mirki gbõ̀ yõ. 
Sètĩ̀n sõ̀ tir mɛ̀ wɛ yɔ̀, nɛ Nɩ̀vɩ̀nɩ̀ tir gbõ̀ yõ, nónó náh pɩ Nsyifunɔlɛ, pé pól-á kwrɔ mɔ Nsyifunɔ sɔkɔ kɩ gbɛ̃kɩ núkú yõ n pi pélɛ, tɛ́ kɩ n pɩ n pi gbɛ̃́nsê núkúlɛ pélɛ. Tɛ́ Liyel-á ǹ vyãh kɔ̃ tir nɛ̂ tyɩ́ sɔkɔ Yesu Krista gbõ̀ yõ, pé tɔ vyãh-á kɩ n wɛ n pi ké tyɩ́ sɔkɔ. 
Liyel-á ǹ tɛ̃́npnɛ̀nɔ́ gbõ̀ yõ yõ̀bènɔrɩ nɛ̂ pɩ ńmɔ tyɩ́, képah gbõ̀ yõ sɔkɔ ye ńnɛ Nɩ̀vɩ̀nɩ̀ tir yónɔ́ tõ̀ sɔkɔ. 
Ńmɔ nɛ̂-á Yesu yõ sɛ̃́pú pól nɛ́tɛ́ynɛ, ńmɔ ye wɛ yõ̀bènɔrɩ mɛ̀nɛ kɩ Yesu pèpɔrɩ tyɩ́lɛ n yo névye tyɩ́ nónó náh pɩ Nsyifunɔlɛ. Kénɛ pèpɔrɩ mɛ̀nɛ, yĩ̀n náh mɛ ké tyɩ́. 
Liyel-á kɩ sètĩ̀n sõ̀ tir mɛ̀ pɩ n pi dùkù nɛ̂nɛ, mé yɩ̃nɛ mè kélɛ névye wɛ̃kɩ n yĩ́ĩ́ pallɛ. Ǹmɔ nɛ̂-á yíyìn pól mɔ́-òlɛ, à ye képah wɔ̃bɩ sah ǹgbò dùkùlɛ tyɛ́-ńkɛ̃̂nɛ, 
sɔ̃̀nko à tɔkɔ núkúnúkú tyɩ́, yɩ̃́pɩ̃̂ nɛ̂-á n wil Liyel tyɩ́, yõ̀tãm pól ànɛ̂ kèkõyṍ tɔ̃́rɩ́ nónó-á mɛ tõ̀lɛ n pɩ yĩ̂nyõ sɔkɔ, pé pól tɛ̃ wɛ kè pɩ̃ Yesu yõ sɛ̃́pú tĩ̀nkì gbõ̀ yõ sɔkɔ. 
Sépi pól pɩ à yɩ̃nɛ Liyel-á sõ̀ nɛ̂ yah tɔkɔ sah tyɛ́-ńkɛ̃̂, tɛ́ pɔ kè pɩ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista gbõ̀ yõ. 
Áyáh á wrɔ́ kwrɔ mɔ Kristalɛ tɛ́ sɛ̃ ǹ yõ, képah gbõ̀ yõ, wɛ̃̀kɩ̀ mɛ yɩ́kɩ́nmɔnɔlɛ á tyɩ́ kɩ fɛ̃ nɔ de Liyel tnɔ̂ nɩ̀sãh sɔkɔ. 
Kǎh pɩ sɛ̃́, ńmɔ-á mɛ yèvyãhrɩ nɛ̂nɛ n pɩ yé yĩnnɛ, yé káh yɛ képah yé wil ku. Képah ye kɩ yõ yɔ̃̀nɔ́lɛ yé kɔ̃. 
Képah ye, mé kɩ gbètukunɔ di Sú Liyel yah sɔkɔ, 
ǹmɔ nɛ̂ gbõ̀ yõ-á kétlo pól mɛ sétáh yõ ànɛ̂ yĩ̂nyõ. 
Ǹ tɔ̃́rɩ́ mɛ gblò. Képah ye, mé kélɛ n yah kɔ̃ ǹ tyɩ́ nɛ à tǎhkɩ̀lɛ yé kɔ̃ ǹ Mirki gbõ̀ yõ, sɔ̃̀nko tǎhkɩ̀ nɛ n mɛ yĩ́ĩ́ yé sɔkɔ. 
Krista nɛ n mɛ tɛ̃̀nɔ́lɛ yé nnɔ sɔkɔ ǹ yõ sɛ̃́nɔ́ gbõ̀ yõ, tyɩ́nɔnɔ mó nɛ mɛ yã̀hnɩ̀ mɔ yĩ́ĩ́ yé sɔkɔ. 
Képah sɔkɔ, tyɩ́nɔnɔ nɛ̂-á mɛ Krista sɔkɔ, yépi ànɛ̂ ǹ yõ sɛ̃́pú tɛ́lɔ́ pól kɩ ké yõ̀tukuri, ànɛ̂ ké yõ̀sɛ̃̀kɩ̀, ànɛ̂ ké kyãhlɔ, ànɛ̂ ké súkúlínɔ́ dùkù pɩ̃. 
Yé kɩ tyɩ́nɔnɔ mɛ̀ dùkù pɩ̃, képah nɛ̂-á kal pnɛ̀nɔ́ duke póllɛ. Képah sɔkɔ, Liyel kɩ mɛ ǹ tɔ̂ɔ́ yé sɔkɔ. 
Áyáh yî nónónɛ n yah n kɔ̃ ànɛ̂ áyáh mɛ nónó tyɩ́lɛ n sõ, Liyel ye kɩ fɛ̃ sélɛ álɛ n kɔ̃ gbĩ́mɛgbĩ́, fɔ́ɔ́ kɩ kah mɔ ńtɛ̃nɛ, ǹ tɛ̃́npnɛ̀nɔ́ gbõ̀ yõ nɛ̂-á mɛ tõ̀lɛ n pɩ ápi sɔkɔ. 
Yesu Krista gbõ̀ yõ, gbílkínɔ́ nɛ n sah Liyel yõ à wil Yesu yõ sɛ̃́pú tĩ̀nkì tyɩ́, knɔ knɔ pól sɔkɔ tyɛ tyɛ́nɩ́ kɛ̃́nɛ. Amiina.
