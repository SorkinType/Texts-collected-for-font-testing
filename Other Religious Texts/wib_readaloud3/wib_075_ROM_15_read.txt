Wrome tãn.
15.
Ápi nónó Liyel yõ sɛ̃́nɔ́-á yuku yahlɛ, á ye á no tɛ́lɔ́ yṍtyílɛ n twah pé tyɩ́. Á ye káh á dyɔ yĩ́ĩ́ tyi ó yah wũ. 
Nɛ̂ mɛ̀ nɛ̂ mɛ̀, à ǹ ní-ò tyɩ́ pɛkɩ pɛkɩ tyilɛ nɛ n pɩ à kɔ̃ à nɛ n yuku yahlɛ Yesu yõ sɛ̃́nɔ́ sɔkɔ. 
Krista ńtɛ̃ fyé náh ǹ dyɔ yĩ́ĩ́ tyi pɩ́-ńsah yah kɔ̃. Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ ǹ tyɩ́ sɔkɔ nɛ ńkɛ̃́: «Liyel, á gbyɔ̀pu-á ǹ gbyɔ gbyɔ nónónɛ, sè vi kwĩnki ńmɔ yõ.» 
Tyi nónó pól-á wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ ǹgbò dùkùlɛ, sè wãrkɩ nɛ sè pɩ kwɔ́nɔ́lɛ ápi tyɩ́. Sépi gbõ̀ yõ, á kɩ gbah tɛ̃ ànɛ̂ sṍré kɩ wɛ á sɔkɔ, á yáh-ńsah kɔ̃ kè náh n tyɛ n pi. 
Liyel nɛ̂-á névilɛ n pɩ gbáhntɛ̃-òlɛ ànɛ̂ sṍré névilɛ, mé kélɛ n yah n kɔ̃ ǹ tyɩ́ nɛ à yé sõnɔ pɩ núkúlɛ, yé mó Krista pɩ dùkùlɛ n yah n pɩ. 
Képah sɔkɔ, yé pól kɩ yé vyãh pɩ núkúlɛ kɩ Liyellɛ n gbilki, ǹmɔ nɛ̂-á á Tɛ̃̀ ǹgbɛ̃ Yesu Krista Súlɛ. 
Képah ye, yé wɛ̃ tɛ̃ plɛ sɔ̃́-á Krista yé tɛ̃ plɛ à gbílkínɔ́lɛ n sah Liyel yõ. 
Máh nɛ̂nɛ n yo yé tyɩ́, kè wɛ yɔ̀: Liyel n kɔ ǹ tyíyónɔ́ sõ̀. Wǎh ǹ vyãhlɛ Nsyifunɔ náh tãn kɔ̃ tir nɛ̂ tyɩ́ sɔkɔ, Krista pɔ pɩ pé tõ̀npɩlɛ à képah pɩ pé tyɩ́. 
À wil képah tyɩ́ sɔkɔ, yìkìnì tɛ́lɔ́ tɔ yĩn ye à pɔ, pè kɔ̃ pè kɩ Liyellɛ n gbilki ǹ fɔ̀kɔ̀pènɔ̀rɩ yĩnnɛ, sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɔkɔ nɛ ńkɛ̃́: «Képah ye mé kɩ álɛ n gbilki yìkìnì tɛ́lɔ́ névye nɩyṍ, tɛ́ kɩ álɛ n gbɩ.» 
Kè tɔ̃ mɛ yónɔ́lɛ ké fɩ̃́ yɔ́ sɔkɔ nɛ ńkɛ̃́: «Yìkìnì tɛ́lɔ́ névye, yé nɩ̀vɩ̀nɩ̀nɛ n pɩ wɛ̃ tyɩ́ Liyel nolɛ.» 
Tɛ́ tɔ̃ mɛ yónɔ́lɛ nɛ ńkɛ̃́: «Yìkìnì tɛ́lɔ́ névye, yé pól Tɛ̃̀ ǹgbɛ̃nɛ n gbilki. No pól mó ǹ gbilki gbilki yɩ̀lɛ n gbɩ.» 
Yesayi tɔ kè yo nɛ ńkɛ̃́: «Nsyese pétó náhkɩ̀ kɩ pɔ yuku kɩ yìkìnì pól névye yah tɛ̃, tɛ́ pè kɩ pé gbõ̀ sah ǹ yõ.» 
Liyel nɛ̂-á névi gbõ̀ sàh-ǹsahlɛ, mé kélɛ n yah n kɔ̃ ǹ tyɩ́ nɛ à nɩ̀vɩ̀nɩ̀ ǹgbɛ̃ mɔ yé sɔkɔ, à mó yèvɩnɩnɛ yé kɔ̃ yáh sɔ̃́npɩ́lɛ sɛ̃ ǹ yõ. Képah sɔkɔ, yé kɩ yé gbõ̀ sah yĩ́ĩ́ ǹ yõ ǹ Mirki tǎhkɩ̀ gbõ̀ yõ. 
Ń yṹnpyé, ńmɔ sɛ̃ yĩ́ĩ́ ké yõ nɛ pèpɔrɩ pɩ́pú-á yélɛ, yáh tyi póllɛ n pnɛ, ànɛ̂ yáh kɩ fɛ̃ vìnmɔnɔlɛ wɛ̃nɛ n kɔ̃. 
Ǹtɛ yé dyɔ to n kõ yĩn ye ń kɔ̃ mé tyi túkù wãrkɩ nɩ̀sãhlɛ tã̀ntã̀n yísyɔ́ sɔkɔ. Liyel gbõ̀ yõ ye mé pɩ képahlɛ, ǹmɔ nɛ̂-á pèpɔrɩ pɩ ńmɔ tyɩ́, 
à ń pɩ Yesu Krista tõ̀npɩlɛ névye wrɔ́, nónó náh pɩ Nsyifunɔlɛ. Liyel tõ̀ kètǎr ye mé n pɩ ǹ Nɩ̀vɩ̀nɩ̀ tir yónɔ́lɛ. Képah sɔkɔ, nónó náh pɩ Nsyifunɔlɛ, pè kɩ pé gblɔ̌ynɛ Liyel kɔ̃ kɩ pɩ yályál ǹ yah ǹ Mirki gbõ̀ yõ, pè kɔ̃ pè kɩ pɩ Liyel dyɔ yĩ́ĩ́nɛ. 
Ń wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Kristalɛ, máh tõ̀ nɛ̂nɛ n pɩ Liyel tyɩ́, kè mɛ yõ yɔ̃̀nɔ́lɛ ń tyɩ́. 
Kénɛ tõ̀ ye Krista ńmɔ tɔkɔ à pɩ névye wrɔ́ nónó náh pɩ Nsyifunɔlɛ. Mé se kɩ tɔ̃ kèyǎh yo kah mɔ képah yõ? Ǹmɔ ye pè kɔ̃ pè n sah Liyel tyɩ́ yõ, ńmɔ vyãh wɛy ànɛ̂ ń tyípɩ́nɩ̀ gbõ̀ yõ, 
ànɛ̂ gbǐl tyi ǹgbnɔ náhnáh sɔkɔ Liyel Mirki tǎhkɩ̀ gbõ̀ yõ. À tɔkɔ Yerusalɛmɩ kwil tyɩ́ ànɛ̂ ké gbɔ̀plɔ kwlolɛ, fɔ́ɔ́ à kyɩ sɔkɔ Yiliri mara pól sɔkɔ, mé Nɩ̀vɩ̀nɩ̀ tir yo yĩ́ĩ́ Krista tyɩ́ sɔkɔ. 
Tɛ́ mé kè yah tɔkɔ à Nɩ̀vɩ̀nɩ̀ tirlɛ n yo tã̀ntã̀n nónó sɔkɔ-á Krista yĩn náh yo yah. Nɩ̀vɩ̀nɩ̀ tir-á yo sɔkɔ tɛ tã̀ntã̀n nónó sɔkɔ, mé náh sõ̀ mɛ ké tyɩ́ nɛ mé kè yo nɛ́npɔ́ ńtɔ̃, á kɩ nɛ ǹnɩ̂ wɔ̀kɔ̀ sètyah yõ sɔ̃́ mé n mɔ n dohnɩ. 
Tɛ́ máh tõ̀ nɛ̂ pɩ yɔ̀, kè mɛ yɩ̃́nɛ́nɔ́lɛ Liyel wɛynɛ nɛ̂-á wã́rkɩ́nsàhnɔ́lɛ nɛ ńkɛ̃́: «Ǹ tyɩ́ náh yo sah nónó tyɩ́, pépi ye kɩ wɛ ǹnɛ. Nónó-á pè náh ǹ tyɩ́ noh yah, pépi ye kɩ ké dùkù pɩ̃.» 
Kénɛ tõ̀ ye kõ ń yahlɛ gbáhyɩ̃́ náhnáhlɛ ń kɔ̃ mé náh mɛ tɛ̃ wɛ kyɩ yé tĩ̀nnɛ. 
Ǹtɛ núkúnúkú, ń tõ̀ ye tyɛ mara mɛ̀ sɔkɔ. Tɛ́ ké ye náhnáh ye yɔ̀, yé tĩ̀n kyɩ dékè mɛ ń kah tɛ̃. 
Máh kɩ pɔ n yuku Yɛsɩpayĩ yamana sɔkɔ, mé kɩ yĩn yah yé tyɩ́ kɩ yé dékè ku yah kwéy, yè mó yòhnɩ̀nmɔnɔ pɩ ń tyɩ́ ń kɔ̃ mè fɛ̃ n yɔ̃ n sɔkɔ ń kɔ́lɔ́lɛ kénɛ yamana sɔkɔ. 
Ǹtɛ núkúnúkú, sɔ́kɔ́nɔ́ ye ńnɛ Yerusalɛmɩ kwil sɔkɔ kɩ kyɩ Yesu yõ sɛ̃́pú yohnɩ mɔ nɛ́npɔ́. 
Yesu yõ sɛ̃́pú nónó-á mɛ Masedoni ànɛ̂ Akayi sɔkɔ, pépi ye kè yah tɔkɔ à pé gbã̀n yî yísyɔ́ tuke mɔ wɛ̃ yõ, Yesu yõ sɛ̃́pú yãm tãn yohnɩ n mɔ yĩnnɛ Yerusalɛmɩ sɔkɔ. 
Pè kè yah tɔkɔ pé gblɔ̌y. Ǹtɛ gbɩ yõ, ké kɔ̃lɩ ńtɛ̃ sõ̀ mɛ pé yõ. Yah kɛ̀; Nsyifunɔ gbõ̀ yõ ye névye nónó náh pɩ Nsyifunɔlɛ pé tɔ vyãh wɛ Liyel tyi sɔkɔ. Pépi tɔ mó yɩ̃nɛ pè Nsyifunɔlɛ n yohnɩ n mɔ pé gbã̀n yîlɛ kɛ̀. 
Mâh pɔ kénɛ tyi nɔ́pi yah wah tɛ à gbã̀n yî tùkènmɔnɔ nɔ́pilɛ pè kɔ̃ tɛ, tɛ́ n sɔkɔ Yɛsɩpayĩ yamana sɔkɔ, mé kɩ syɔ̃ yé tyɩ́. 
Ń sɛ̃́nyĩnɔ mɛ ké yõ nɛ mâh pɔ yé tĩ̀nnɛ, Krista-á kɩ sãm mɔ yĩ́ĩ́ á sɔkɔ. 
Ń yṹnpyé, ápi Tɛ̃̀ ǹgbɛ̃ Yesu Krista yĩn yõ, ànɛ̂ Liyel Mirki-á tyɩ́nɔnɔ nɛ̂ dahbɩ yé sɔkɔ, sépi yĩnnɛ, mé kélɛ n yah n kɔ̃ yé tyɩ́ nɛ, yé so mɔ ń tyɩ́ náhnáh Liyel nínɔ́ sɔkɔ. 
Yé Liyellɛ n ni ńmɔ tyɩ́ ń kɔ̃ mè wɛ n pwah Nsyude mara Yesu yõ sɛ̃́-ńkɛ̃̂pú tyɩ́, ànɛ̂ yòhnɩ̀nmɔnɔ nɛ̂-á mé syi n yuku kɔ̃́-ńsah Yerusalɩmɩ kwil sɔkɔ, Yesu yõ sɛ̃́pú kè fɛ̃ n syi nɩ̀vɩ̀nɩ̀ sɔkɔ. 
Képah náh, Liyel-à kɩ fɛ̃ syi, mé kɩ yĩni tɛ̃ yé tĩ̀nnɛ nɩ̀vɩ̀nɩ̀ sɔkɔ kɩ pɔ mɩ yah kwéy. 
Liyel nɛ̂-á yèvɩnɩ nɛ́tɛ̃̂nɛ, mé kélɛ n yah n kɔ̃ ǹ tyɩ́ nɛ à nɛ n mɛ yé póllɛ. Amiina.
