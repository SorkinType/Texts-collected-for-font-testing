1 Timote.
5.
Wâh yèwàhlɛ n vi mɔ, káh nɛ n wɛkɩ ǹ tyɩ́ dírí sɔkɔ, nɛ n wɛkɩ ǹ tyɩ́ á sú sɔ̃́ ǹnɛ. Kètɛ̃̀npìlɛ n vi mɔ á yṹnpyé sɔ̃́ pélɛ. 
Syɔ́dîlɛ n vi mɔ á yṹbí sɔ̃́ pélɛ, ǹ tɛ́ syɔ́npìlɛ n vi mɔ ké yah pèpɛy yõ á dósyɔ́ sɔ̃́ pélɛ. 
Pɛ́kúsyɔ nónó-á yòhnɩ̀nmɔpu náh mɛ pé tyɩ́, pé tɛ̃́nɔ́ plɛ. 
Ǹtɛ pɛ́kúsyɔ nónó tyɩ́ wɛ̃́npì-à mɛ, képah náh pɩ pétóbélɛ, kénɛ wɛ̃́npì ye yɩ̃nɛ pè pé Liyel yah tíkí pɩ́nɔ́ wɛ̃kɩ à yuku pé gbô tãn tĩ̀nnɛ. Pǎh yɩ̃nɛ pè nɛ̂ pɩ pé dúbípú tyɩ́, pè képahlɛ n pɩ pé tyɩ́. Yé náh wɛ di, képah tyɩ́ ye nɔ Liyel tyɩ́. 
Ǹtɛ sê nɛ̂-à mɛ pɛ́kúsèlɛ kègbɩ yõ sɔkɔ, nɛ̂-á névi náh mɛ ǹ tyɩ́ kɩ à yohnɩ mɔ, ǹmɔ ye ńkɛ̃́nɛ ǹ gbõ̀ sah n yĩ́ĩ́ Liyel yõ sɔkɔ. Ǎ Liyellɛ n ye n gbo nírí sɔkɔ lékã̂h ké gbĩ̀ntɔ̃̀. 
Ǹtɛ pɛ́kúsè nɛ̂ tɛ̃̀ wɔ-à mɛ wil gbɛ̃́nsê dékè tyi ólɛ n sõ, ǹmɔ mɛ kúnɔ́lɛ tɛ́ à sah vilɛ. 
Wáh yɩ̃nɛ ǹ nɛ̂ tɔ yo pé tyɩ́ sɔ̃̀nko lékàh káh pɔ wɛ pé tyɩ́, kè wɛ yɔ̀. 
Névi nɛ̂-à n yáhndahnmɔ-ńkɛ̃̂nɛ ǹ tyɩ́ névye tyɩ́ sɔkɔ, ànɛ̂ ǹ kéte névye tɛ̃̀ tyɩ́ sɔkɔ, tɛ̃̀ Krista yõ sɛ̃́nɔ́ wɛ̃̀kɩ̀ milki. Krista yõ sɛ̃́-ńkɛ̃̂ névi ńtɛ̃ fṍnfṍ ǹmɔ yõ. 
Pɛ́kúsè nɛ̂ yĩn-à pópó n wãrkɩ, à ǹnɛ n mɔ n kõ pɛ́kúsyɔ wrɔ́ à yɩ̃nɛ à nɛ n mɛ ye tikitɔ̃nɩ́ tɛ̃̀ wɔ wɛ, à mó nɛ mɛ de yah pɛ núkú ó tyɩ́. 
Ǹ tyɩ́ yɩ̃nɛ kè nɛ mɛ pɩ̃́nɔ́lɛ ǹ tyípéplɔ pɩ́nɔ́ tyɩ́ sɔkɔ. À nɛ n mɛ ǹ wɛ̃́npì tɛ̃́nɔ́ plɛ, à nɛ n mɛ yɛ nɛ́pân nɛ n twah n tyɩ́, à nɛ n mɛ Yesu yõ sɛ̃́pú gbyẽ̂hnɛ n wu, à nɛ n mɛ névye nnɔlɛ n de nónó-á mɛ yèkwɔrɩ sɔkɔ, à mó nɛ n mɛ tyípéplɔ duke póllɛ n pɩ. 
Ǹtɛ syɔ́pâm nónó tɛ̃̀ wɔ-á pɛ́kúsyɔlɛ, káh pépi yni wãrkɩ pɛ́kúsyɔ sɔkɔ. Á náh wɛ di, lédɔ́rɩ́-à pɔ pé wrɔ́ gbɛ mɔ Kristalɛ, pè n nɛ n mɛ ké tyɩ́ nɛ pé pé syɩ de pɛbɩ tyɩ́. 
Kàh pɩ sɛ̃́, lékàh n wɛ pé tyɩ́, pǎh sɔ̃́npɩ́lɛ pé vyãhlɛ Krista kɔ̃ tɛ́ pɔ pé syɩ ǹ náh. 
Pǎh se tɔ̃ pɩ tõ̀ kɛ̃́ névyelɛ sɛ̃́, pè n wũ n tɛ̃ gbéngbèn kɔ̃́nkɔ̃nɔ ó sɔkɔ. Pépi ye tõ̀ náh mɛ pé tyɩ́ yɔ̀, tɛ́ pè n tɔ̃ n pɩ wɛ̀nkwĩnki wɛ́kɩ́púlɛ vyãhlɛ dálnɔ́lɛ, ǹ nɛ n wɛkɩ n min n dah n mɔ sɛ̃́. 
Képah ye, mé mɛ ké tyɩ́ nɛ pɛ́kúsyɔ nónó-á mɛ syɔ́pâmnɛ, pè de pɛbɩ tyɩ́. Pè fɛ̃ wɛ̃́npì wɛ, pè mó nɛ n yah n dah mɔ pé gbên tyɩ́ sɔkɔ. Sɔ̃̀nko pè káh pɔ wɛ̃̀kɩ̀ yɩkɩ mɔ á sépɛ́bɩ́ kɔ̃ pè á yĩ́npolɛ n yo. 
Yé náh wɛ di, pɛ́kúsyɔ túkù ye mɛ wɛ̃̀kɩ̀ milki tɛ, tɛ́ mɛ sétã̀n yõ̀tɛ̃̀ sõ̀ tɛ̃. 
Ǹtɛ pɛ́kúsyɔ-à mɛ Yesu yõ sɛ̃́-ò sê nɛ̂ wɔ̀kɔ̀ tãn sɔkɔ, tɛ̃̀ yɩ̃nɛ à nɛ n yah n dah mɔ pé tyɩ́ sɔkɔ, sɔ̃̀nko képah káh pɩ Yesu yõ sɛ̃́pú tĩ̀nkì yṍtɔ́kɔ́bɩ́lɛ. Képah sɔkɔ, Yesu yõ sɛ̃́pú tĩ̀nkì tɔ kɩ mó n yah n dah mɔ pɛ́kúsyɔ tyɩ́ sɔkɔ nónó-á nɛ́kã́n tãnnɛ. 
Yah tɛ̃́pú nónó-à mɛ Yesu yõ sɛ̃́pú tĩ̀nkì yahlɛ n tɛ̃ pallɛ, yõ̀yén yɩ̃nɛ kè nɛ n sah n yĩ́ĩ́ pé yõ. Ké dú dɔkɔ pé nónó-á Liyel wɛy yónɔ́ ànɛ̂ névye kwɔ́nɔ́ tõ̀ mɛ dahbɩ pé gbõ̀. 
Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ, nyeh-à mɛ swãh tṍnɔ́ sɔkɔ á tyɩ́ gbĩ́ nɛ̂nɛ, á káh sé vyɛ̃y pukubi nɛ sè káh á sèmukɔlɛ n yõ. Tɛ́ tɔ̃ mɛ wã́rkɩ́nɔ́lɛ nɛ, tõ̀npɩ-á yɩ̃nɛ ǹ tõ̀ pɩ́nɔ́ sǎr wɛ́nɔ́lɛ. 
Káh n fɛ̃ n syi lékàhlɛ n syi yah tɛ̃́-òlɛ n kɔ̃ nɛ kénɛ tir náh wil tàrfyɔ̀ nɛ́pĩ̂ nímí képah náh pɩ tɔ̃́mɩ́ vyɛ̃y sɔkɔ. 
Névi nɛ̂-à tyítúkù pɩ, kè yo gbǎb tɔ́wû pól yĩ́ yah sɔkɔ, sɔ̃̀nko kè tíkí mɔ tɛ́lɔ́ sɔkɔ. 
Mé kélɛ n gbah n yo á tyɩ́ Liyel yah, ànɛ̂ Yesu Krista yah, ànɛ̂ yĩ̂nyõ tõ̀npɩbɩ wìlkìnsàhnɔ́ yah sɔkɔ nɛ, á tyi nɔ́pi póllɛ n pɩ tómm, á káh wǎhkɩ̀lɛ n pɩ. 
Káh nɛ n sal á gbõ̀lɛ n sah névi yõ ǹ nírílɛ n pɩ tɛ̃̀ tyɩ́, ànɛ̂ káh á vyãhlɛ n dah n mɔ á ní-òlɛ ǹ tyípêl pɩ́nɔ́ sɔkɔ. Nɛ n mɛ yályál Liyel yah sɔkɔ. 
Wáh mɛ n yɔ gbáhyɩ̃́ náhnáhlɛ sɛ̃́, káh ni ó kɔ́kɔ́lɛ n wɔ, dìvɛ̃ tɔlɛ n wɔ kwéykwey á fɔkɔ markɩ yĩnnɛ. 
Névye túkù tyípêl mɛ sè n wilki n mɔ kwèyy pé tùkè-ǹkɛ̃̂ sɔkɔ. Ǹtɛ túkù tyɩ́ mɛ, náh ó sè n pɔ n pɩ̃. 
Sɛ̃́ ye tyípéplɔ tɔlɛ, sépi tɔ n wɛ kwèyy yĩ́nɛ, ànɛ̂ nónó ńtɛ̃ fyé-á sè náh mɛ n yah, sè náh n wũ n pi wìlkìnmɔ-ńkɛ̃̂nɛ.
