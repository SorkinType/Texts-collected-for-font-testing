1 Nsyɔ̃.
5.
Névi nɛ̂-à sɛ̃ ké yõ nɛ Yesu-á Kristalɛ, nɛ̂-á Liyel nɛy yáhntɔkɔnɔlɛ, Liyel pi ye tɛ̃̀nɛ. Tɛ́ Sú Liyel tyɩ́-à nɔ névi nɛ̂ tyɩ́, Liyel wɛ̃́npì tɔ tyɩ́ n nɔ tɛ̃̀ tyɩ́. 
Liyel tyɩ́-à nɔ ápi tyɩ́, tɛ́ á mɛ ǹ tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ, képah sɔkɔ ye á kɩ pɩ̃ nɛ Liyel wɛ̃́npì tɔ tyɩ́-á mɛ nɔ á tyɩ́. 
Á náh wɛ di, Liyel tyi kõ̀nsàhnɔ́ wã̀l kɔ́lɔ́ ye n wɛ̃kɩ kélɛ nɛ ǹ tyɩ́-á nɔ á tyɩ́. Tɛ́ ǹ tyi kõ̀nsàhnɔ́ náh mɛ táhnɔ́lɛ, 
á pól-á sɔ̃́npɩ́lɛ pɩ Liyel wɛ̃́npìlɛ tɛ́ n tɛ̃ n pnɛ kèkõyṍ tyɩ́. Tɛ́ tir nɛ̂-á á kɔ̃ á n tɛ̃ n pnɛ kèkõyṍ tyɩ́, á Yesu yõ sɛ̃́nɔ́ ye. 
Ǹtɛ nɛ̂ n tɛ̃ n pnɛ kèkõyṍ tyɩ́ wɛ́? Névi nɛ̂-á sɛ̃ ké yõ nɛ Liyel Pídĩ́-á Yesulɛ, ǹmɔ ye. 
Yesu Krista ye pɔ wel wilki ni sɔkɔ. Képah náh, à ko mɔ ǹ tṍ kwɛ. Tɛ́ à náh pɔ wel wilki ni ó sɔkɔ dɛ́, à ye pɔ wel wilki ni sɔkɔ, tɛ́ ko mɔ ǹ tṍ kwɛ. Kénɛ tyinɔ nínɔ́ ànɛ̂ Liyel Mirki ye n yo n wah kélɛ. Yé náh wɛ di, Liyel Mirki ye n wɛ̃kɩ gbɩlɛ gbĩ́mɛgbĩ́. 
Liyel Mirki-á nɛ̂ yo, ànɛ̂ ni-á nɛ̂ wɛ̃kɩ, ànɛ̂ tṍ-á nɛ̂ wɛ̃kɩ, sépi tɔ̃́nɔ́ ye n yo n wah kélɛ. Tɛ́ sé tɔ̃́nɔ́ pól mɛ sah núkú ó yõ. 
Névye-á tyi nónó wɛ tɛ́ sélɛ n yo n wah, áyâh mɛ sépilɛ n fɛ̃ n syi à, ǹtɛ Liyel tyi yónwàhnɔ́ náh fṍnfṍ névye tyɩ́ yõ di? Tɛ́ Liyel ye mó tɔ̃ yo wah ǹ Pídĩ́ tyɩ́lɛ. 
Névi nɛ̂-à sɛ̃ Liyel Pídĩ́ yõ, ǹ tyɩ́ yónɔ́ mɛ̀ ye tɛ̃̀ tɛ̃ sah ǹ nɩ sɔkɔ. Ǹtɛ nɛ̂-à sɛ̃́-ńkɛ̃̂nɛ Liyel yõ, tɛ̃̀ mɛ Liyellɛ n pɩ mɩ̀nyolɛ, wǎh sɔ̃́npɩ́lɛ yahle sɛ̃́nɔ́lɛ Liyel tyíyóné mɛ̀ yõ, wǎh nɛ̂ yo ǹ Pídĩ́ tyɩ́ sɔkɔ. 
Tyíyóné mɛ̀ wɛ yɔ̀ nɛ, Liyel-á tyɛ́-ńkɛ̃̂ minnɛ á kɔ̃, nɛ̂-á n wɛ ǹ Pídĩ́ gbõ̀ yõ sɔkɔ. 
Névi nɛ̂-à mɛ Pídĩ́ tyɩ́ yõ, kénɛ min mɛ tɛ̃̀ wɔ tyɩ́. Ǹtɛ nɛ̂-à mɛ́-ńkɛ̃̂nɛ ǹ tyɩ́ yõ, min mɛ̀ náh mɛ tɛ̃̀ tyɩ́. 
Yépi nónó-á sɛ̃ Liyel Pídĩ́ yõ, mé ye tyi nɔ́pilɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́, nɛ̂-á kɩ yé kɔ̃ yé kɩ pɩ̃ nɛ yáh tyɛ́-ńkɛ̃̂ min wɛ. 
Ápi sɛ̃́nyĩnɔ-á mɛ Liyel yõ nɛ̂ tyɩ́ sɔkɔ képah ye yɔ̀ nɛ ńkɛ̃́, áyâh kwâh nɛ̂nɛ n ni ǹ tyɩ́ tɛ́ kè mɛ ǹ dyɔ yĩ́ĩ́nɛ, ǎ nóhnɔ́ pɩ á tyɩ́. 
Tɛ́ áyáh fyɔ̀ kè pɩ̃ nɛ wǎh mɛ nóhnɔ́lɛ n pɩ á tyɩ́, á pɩ̃ yĩ́ĩ́ tɔlɛ nɛ, áyâh kwâh nɛ̂ ni ǹ tyɩ́, ǎ kè wɛ. 
Ǹnɩ̂ yɔ́-à ǹ yṹnpi yɔ́ wɛ à mɛ tyípékèlɛ n pɩ, nɛ̂-á kè náh névi min tyɩ́lɛ n yɩkɩ Liyel yah sɔkɔ, tɛ̃̀ nírí pɩ ǹ tyɩ́, Liyel kɩ tyɛ́-ńkɛ̃̂ minnɛ tyípékè pɩ́-ò mɛ̀ kɔ̃, kàh mɛ nɛ tyípékè nɛ̂-á à pɩ, kè náh névi min tyɩ́lɛ n yɩkɩ Liyel yah sɔkɔ. Tɛ́ tyípékè yɔ́ mɛ nɛ̂-á névi min tyɩ́lɛ n yɩkɩ Liyel yah sɔkɔ, képah dùkù tyɩ́ náh mé n yo nɛ yé nírílɛ n pɩ ké tyɩ́ sɔkɔ dɛ́. 
Tyi nónó pól tɛ̃̀-à fyɔ̀ mɛ tyíknɛ̌nɩ̀nɛ, tyípêl ye sélɛ. Ǹtɛ tyípêl pól náh névi min tyɩ́lɛ n yɩkɩ Liyel yah sɔkɔ. 
Ápi pɩ̃ nɛ névi nɛ̂-à pɩ Liyel pilɛ, tɛ̃̀ náh tɔ̃ tyípékèlɛ n pɩ. Liyel Pídĩ́ ńkɛ̃́nɛ nɛ mɛ tɛ̃̀ yõlɛ n syɩkɩ, képah ye pèkè pɩ́-ò nɛ̂-á sétã̀n yõ̀tɛ̃̀nɛ, à náh fɛ̃ tir yɔ́ pɩ n pi tɛ̃̀ sɔkɔ. 
Ápi pɩ̃ nɛ ápi-á mɛ Liyel tyɩ́ yõ, tɛ́ kèkõyṍ pól tɛ̃̀ wɔlɛ, pèkè pɩ́-ò mɛ̀ mɛ képahlɛ n fnɛ. 
Ápi tɔ̃ pɩ̃ nɛ Liyel Pídĩ́-á pɔ, ànɛ̂ wǎh lékã́mnɛ á kɔ̃ á tɛ̃ wɛ Liyel yèsyɩ̀kɩ̀lɛ n pnɛ. Ǹmɔ Liyel nɛ̂-á yèsyɩ̀kɩ̀lɛ, ápi wrɔ́ mɛ kwrɔ́nmɔnɔlɛ ǹmɔlɛ ǹ Pídĩ́ Yesu Krista gbõ̀ yõ. Ǹmɔ ye Liyel yèsyɩ̀kɩ̀lɛ, ànɛ̂ tyɛ́-ńkɛ̃̂ minnɛ. 
Ń wɛ̃́npì, yé yé gblɔ̌ynɛ n tɛ̃ sãh fálnɔ́ tyilɛ.
