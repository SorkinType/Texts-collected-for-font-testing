Filipi.
3.
Ń yṹnpyé, yé nɛ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ yé wrɔ́ kwrɔ́nmɔnɔ yĩnnɛ Tɛ̃̀ ǹgbɛ̃nɛ. Máh tyi nónó yo sɔkɔ tɛ yé tyɩ́, kè náh mɛ kwlɔ́nɔ́lɛ ń tyɩ́ kɩ tɔ̃ ń syɩ sè wãrkɩ tɛkɩ mɔ yé tyɩ́. Képah ńtɛ̃ ye kɩ yépi yohnɩ mɔ yékɩ tɛ̃ wɛ̃̀kɩ̀ pèpɛy yõ. 
Yé yé gblɔ̌ynɛ n kyɔmɩ tõ̀ntùkù pɩ́púlɛ, kénɛ pãhlɔlɛ, pépi nónó kõ̀kè kõ̀nɔ́-á wil fɩ̃́ yɔ́ ó kã̀mnɔ́lɛ. 
Ké gbɩ tɛ̃̀ yõ, ápi wɔ ye kõ kõ̀kè pèpɛynɛ, képah nɛ̂-á Liyel yĩn gbílkínɔ́lɛ ǹ Mirki gbõ̀ yõ. Kè mɛ yõ yɔ̃̀nɔ́lɛ á tyɩ́ áyáh wɛ kwrɔ mɔ Yesu Kristalɛ tɛ́ á náh mɛ á gbõ̀ sah á gblɔ̌y yõ. 
Ǹnɩ̂ yɔ́ tɛ̃̀-à fyɔ̀ fɛ̃ ǹ gbõ̀ sah n pi ǹ gblɔ̌y yõ, gbɩ kɩ sõ̀ fɛ̃ wɛ ńmɔ tyɩ́ kal tɛ̃̀nɛ kɩ ń gbõ̀ sah ń gblɔ̌y yõ, 
ńmɔ nɛ̂-á kõ ń dúbínɔ́ sèpĩ̀n kwǎrtɔ̃́-ò sõlɛ, tɛ́ mó tɔ̃ pɩ Yisrayel nɛynɛ à wil Bɛ̃nsyamɛ yìkì sɔkɔ. Yebreke ye ńnɛ, ń dúbípúlɛ Yebreni ye pélɛ. Nsyifunɔ tyi kõ̀nsàhnɔ́ wã̀l kɔ́pú nónó-á pè n ye Farisi wɛ̃̀kɩ̀ kɔ́púlɛ, pépi sɔkɔ nɛy yɔ́ ye sõ̀ ńnɛ. 
Tɛ́ à yuku gbáhntɛ̃nɔ tyɩ́ tĩ̀nnɛ kénɛ wã̀l kɔ́lɔ́ sɔkɔ, mé sõ̀ mɛ Yesu yõ sɛ̃́pú tĩ̀nkìlɛ n fõh. À tɔ̃ yuku kénɛ wã̀l kɔ́lɔ́ tyɩ́ tĩ̀nnɛ, nɛy náh sõ̀ fɛ̃ lékàhlɛ n kɔ̃ n pi. 
Ǹtɛ kénɛ tyi nónó-á sõ̀ mɛ yõre tyilɛ ńmɔ tyɩ́, núkúnúkú mé mɛ pɔ sè wɛ syɩ́kɩ́nɔ́ tyilɛ Krista yĩnnɛ. 
Gbɩ yõ, Tɛ̃̀ ǹgbɛ̃ Yesu Krista wɛ́npɩ̃nɔ-á pɩ pèpɔrɩ ǹgbɛ̃ nɛ̂nɛ ńmɔ tyɩ́, képah yĩnnɛ mé mɛ tyi nɔ́pi póllɛ n yah syɩ́kɩ́nɔ́ tyilɛ. Mé mɛ ké tyɩ́ nɛ mé pɩ Krista tyɩ́lɛ, ànɛ̂ ń wrɔ́ mó kwrɔ mɔ ǹnɛ. Képah ye ń kɔ̃ mé fɛ̃ syi ń náh vi tyi pól kɔ̃, tɛ́ sé póllɛ n yah tũ̀ yõ yî tyɩ́ kɔ̃lɛ. Mé náh tɔ̃ mɛ n sõ nɛ tyi kõ̀nsàhnɔ́ wã̀l kɔ́lɔ́-á pɩ ńnɛ tómm dɛ́. Krista yõ sɛ̃́nɔ́ gbõ̀ yõ ye mé pɩ tómm. Liyel ye n pɩ névilɛ tómm Krista yõ sɛ̃́nɔ́ gbõ̀ yõ. 
Ńmɔ-á mɛ nɛ̂ tyɩ́ képah ye yɔ̀ nɛ, mé Krista pɩ̃ ànɛ̂ tǎhkɩ̀ nɛ̂-á yilki mɔ ǹ lékólɛ, ànɛ̂ ń vyãh mó wɛ ǹ yèvyãhrɩ pɩ́nɔ́ sɔkɔ, mé mó n pɩ ǹ tyɩ́ kɔ̃lɛ ǹ ku dùkù sɔkɔ. 
Képah sɔkɔ, ń sõnɔ kɩ mɛ ké yõ nɛ ń tɔ lékó-á kɩ pɔ n yilki n mɔ n pi. 
Kè náh mɛ á kɩ nɛ nɛ̂-á mé n yah n kɔ̃ mé sɔ̃́ kè wɛ tɛ, ànɛ̂ ń tyɩ́ sɔ̃́ pɛkɩ syɩ ké yĩ̀n tyɩ́ dɛ́. Ǹtɛ mé mɛ yɔ̃ sɔkɔ ń tólɔ́lɛ nɛ mé gbah n tɛ̃ mé kè wɛ, képah nɛ̂ yĩn-á Krista ń ye pɩ ǹmɔ ó tyɩ́lɛ. 
Ɔ̃́ɔ̃ dɛ́ ń yṹnpyé, kè náh mɛ á kɩ nɛ ńmɔ sɔ̃́ kè wɛ tɛ. Ǹtɛ nɛ̂-á mé mɛ n pɩ, kè wɛ yɔ̀: Tyi nónó-á sɔkɔ tɛ, mě ń dyɔ mɔ n fwɔ sépi tyɩ́lɛ, ǹ tɛ́ yah tĩ̀n tyɩ́lɛ n yɩ̃kɩ. 
Mé mɛ n to ké yĩ̀nnɛ n yɩ̃kɩ yõre wɛ yĩnnɛ Liyel-á álɛ n ye nɛ á pɔ nɛ̂ syi yĩ̂nyõ, Yesu Krista gbõ̀ yõ. 
Ápi nónó pól Yesu yõ sɛ̃́nɔ́-á de yĩn, sɛ̃́ ye á yɩ̃nɛ à nɛ n sõ. Ǹtɛ sõnɔ káhkɩ́-à kɩ se mɛ yé sɔkɔ tir yɔ́ tyɩ́ sɔkɔ, Liyel kɩ tɔ̃ képah dùkùlɛ yé wɛ̃kɩ. 
Kàh kɩ sɔ̃́ n pɩ sɔ̃́, áyáh mɛ n kɔ wɛ̃̀kɩ̀ nɛ̂ yõ à pɔ syɩ núkúnúkú tyɩ́, á ye yɔ̃ sɔkɔ nɛ n yuku yahlɛ képah ó yõ. 
Ń yṹnpyé, yé ńmɔlɛ n yah n pɩ. Áyáh pɩ dùkù nɛ̂ wɛ̃kɩ, nónó-á mɛ képah wɛ̃̀kɩ̀lɛ n kɔ, yé pépilɛ n yah n pɩ. 
Mé ye kè yo sɔkɔ tɛ yé tyɩ́ gbáhyɩ̃́ náhnáhlɛ, tɛ́ tɔ̃ mɛ ń syɩ kélɛ n yo yé tyɩ́ gbní sɔkɔ núkúnúkú nɛ, Yesu Krista dahnɩ yõ kúnɔ́-á nɛ̂nɛ n pɛkɩ, névye náhnáh pɩ dùkù-á mɛ kélɛ n wɛ̃kɩ nɛ pǎh mɛ kélɛ n fwo. 
Pépi tyɩ́-á kɩ pɔ n yɩkɩ n pi ké tyɛ́-ńsah sɔkɔ. Pé kèkùrn-á pɩ pé líyellɛ. Tir nɛ̂-á sennɛ pè tɛ̃ n pi, pè n nɛ mɛ pé gblɔ̌ynɛ n gbilki képah sɔkɔ. Kèkõyṍ yî ó tyɩ́-á pépi n nɛ n sõ. 
Ápi tɛ̃̀ wɔ sú fṍ mɛ yĩ̂nyõ sɔkɔ, ńyãh sɔkɔ-á á pwáhnmɔ-ò nɛ̂-á Tɛ̃̀ ǹgbɛ̃ Yesu Kristalɛ, à kɩ n wil n pɔ n pi, áyáh mɛ ǹmɔ nɛ̂nɛ n dɛkɩ. 
Ápi wil gbɛ̃́nsê nɛ̂-á sohkɩ kɛ̃́nɛ, à kɩ pɔ kè vi mɔ pɩ ǹmɔ wil gbɛ̃́nsê tyɩ́ kɔ̃lɛ, nɛ̂-á gbílkínɔ́ mɛ n sah ké yõ. Ǹ tɛ̃́npnɛ̀nɔ́ gbõ̀ yõ ye à kɩ pɩ képahlɛ, nɛ̂-á yíyìn pól kɔ̃ sè n sah ǹ tyɩ́ yõ.
