Máh mɛ nɛ̂ ǹgbòlɛ n yah n kɔ̃, képah ye nɛ á yáhnkɔ̃nɔlɛ n pɩ Liyel tyɩ́, á ǹnɛ n ni, á ǹnɛ n fal, á mó wɛynɛ ǹnɛ n kɔ̃ no pól yĩnnɛ. 
Á ye nírílɛ n pɩ yõ̀tãm tyɩ́, ànɛ̂ névye pól tyɩ́ tɔ̃́rɩ́-á mɛ nónó gbõ̀, sɔ̃̀nko à á minnɛ n wɛ n pɩ syɩ́syɩ́ yèvɩnɩ sɔkɔ, á kɔ dùkù mó nɛ n mɛ pallɛ, à mó Liyel yah tíkílɛ n pɩ. 
Képah ye plɛ, ànɛ̂ ké tyɩ́ nɔ á pwáhnmɔ-ò Liyel tyɩ́, 
ǹmɔ nɛ̂-á mɛ ké tyɩ́ nɛ no pól pwáhnmɔnɔ wɛ, pè mó tɛ̃ wɛ gbɩlɛ n pnɛ. 
Yé náh wɛ di, Liyel núkú ó ye mɛ. Liyel ànɛ̂ névye wrɔ́ tɔ̃́-ò núkú tɔ ó mɛ. Névi ye ǹnɛ. Ǹmɔ ye Yesu Kristalɛ, 
ǹmɔ nɛ̂-á ǹ gblɔ̌y kɔ̃ à névye pól yõ dwe. Gbĩ́ nɛ̂-á sõ̀ yáhntɔkɔnsàhnɔ́lɛ, kè pɩ képah sɔkɔ. 
Yesu Krista gblɔ̌y kɔ̃́nɔ́ mɛ̀ yĩn ye Liyel ńmɔ yah tɔkɔ pɩ ǹ wɛy yó-òlɛ, ànɛ̂ Krista tɛ̃ntõ̀ névilɛ, nɛ névye nónó náh pɩ Nsyifunɔlɛ mé pélɛ n kwɔ Liyel yõ sɛ̃́nɔ́ ànɛ̂ gbɩ wɛ̃̀kɩ̀lɛ. Gbɩ ye mé n yo, mɩ̀nyo náh ńnɛ. 
Mé mɛ ké tyɩ́ nɛ tã̀ntã̀n pól sɔkɔ, díbí pé gbã̀n yɔ̃ dɔkɔ yĩ̂nyõ pé Liyellɛ n ni nnɔlɛ yályál, fɔ̀kɔ̀vyã̀hnɩ̀ ànɛ̂ mɛ́kɩ̀ kɛ̃́nɛ pé wrɔ́. 
Mé mɛ ké tyɩ́ nɛ syɔ́ tɔ flɔlɛ n tɔ̃ n sah n mɔ kǎh yɩ̃nɛ sɔ̃́. Pè flɔlɛ n mɔ nónó-á kɩ fɛ̃ pé sen yɩ̃ ólɛ yã́ryã́r dáhbɩ́-ńkɛ̃̂nɛ ké sɔkɔ. Pè káh yõ kàhntɔ̃nsàhnɔ́ képékè, ànɛ̂ sã́n yɛ̀sõ̀ yî, ànɛ̂ yî kètǎrnɔ, ànɛ̂ fwɔ̀tǎhnɩ̀nɛ n yah n kɔ̃. 
Ǹtɛ pé gblɔ̌y tɔ̃́nsàhnɔ́ yɩ̃nɛ kè pɩ tyípéplɔ pɩ́nɔ́lɛ, sɔ̃́-á syɔ́ kélɛ n pɩ nónó-á mɛ Liyel yah tíkílɛ n pɩ. 
Sê yɩ̃nɛ à ǹ gblɔ̌y tiki mɔ tɛ̃ syɩ́syɩ́ pàh ǹnɛ n kwɔ gbĩ́ nɛ̂nɛ Yesu yõ sɛ̃́pú tĩ̀nkì wrɔ́. 
Mé náh ké wɛ̃̀kɩ̀lɛ sê kɔ̃ nɛ à névyelɛ n kwɔ, képah náh pɩ à pɩ dĩ́ yõ̀tɛ̃̀nɛ. À yɩ̃nɛ à tɛ̃ syɩ́syɩ́. 
Yé náh wɛ di, Adama ǹgbò ye Liyel mɔ, tɛ́ pyě Awɔ mɔ dò. 
Adama náh sétõ pilki; Awɔ ye à pilki à kɔ̃ à Liyel yóré yahle. 
Ǹtɛ képah fyé yõ sê kɩ pwáhnmɔnɔ wɛ ǹ pɩ́nɔ́ yĩnnɛ yṹnɛ, wàh sɛ̃ Liyel yõ, tɛ́ tyɩ́nɔnɔ mɛ ǹ sɔkɔ, tɛ́ mɛ n kɔ yályál Liyel yah yã́ryã́r dáhbɩ́-ńkɛ̃̂nɛ sépi pól sɔkɔ.
