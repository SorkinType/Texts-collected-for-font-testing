Korɛ̃nti tãn.
14.
Yé gbah n tɛ̃, yé tyɩ́ nɔ wɛ̃ tyɩ́. Liyel Mirki-á tyi nónó pɩ́nɔ́ yõ̀bènɔrɩlɛ n kɔ̃, yé sé tɔ yah n kɔ̃, tɛ́ ké nɛ̂ wɔ-á Liyel tɛ̃ntõ̀ syínyonɔlɛ, yé képah wɔ gbah n yah n kɔ̃ n kal. 
Yé náh wɛ à, névi nɛ̂-à wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á à náh kè sãh, Liyel tyi wɔ̃̀bɩ̀nsàhnɔ́ ye ńkɛ̃́nɛ à n nɛ n yo Liyel Mirki gbõ̀ yõ, nónó-á névye náh sélɛ n nohnɩ. Kàh pɩ sɛ̃́, névye tyɩ́ náh tɛ̃̀ n nɛ n yãh dɛ́, Liyel tyɩ́ ye. 
Ǹtɛ névi nɛ̂ wɔ-à Liyel tɛ̃ntõ̀lɛ n syi n yo, névye tyɩ́ ye tɛ̃̀ n nɛ n yo n mɔ, ǹ yah yúkúnɔ́lɛ pélɛ n kɔ̃, ǹ tɛ́ sṍrélɛ n mɔ pé sɔkɔ, ǹ tɛ́ pé nnɔlɛ n sũ. 
Tɛ́ nɛ̂-à wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á à náh kè sãh, tɛ̃̀ mɛ ǹ gblɔ̌y ó yah yúkúnɔ́lɛ n yah n kɔ̃. Ǹtɛ nɛ̂-à Liyel tɛ̃ntõ̀lɛ n syi n yo, tɛ̃̀ wɔ mɛ Yesu yõ sɛ̃́pú tĩ̀nkì pól yah yúkúnɔ́lɛ n yah n kɔ̃. 
Yé pól-à sõ̀ mɛ wɛ́nkáhkɩ́nɔlɛ n wɛkɩ nónó-á yé náh sè sãh, kè náhkɩ plɛ ó; ǹtɛ yáh kɩ Liyel tɛ̃ntõ̀lɛ n syi n yo, képah kɩ nɔ kal ń tyɩ́. Névi nɛ̂-à wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á à náh kè sãh, tɛ́ ké yah náh tɛ̃ wɛ yo Yesu yõ sɛ̃́pú tĩ̀nkì wɛ̃kɩ kè kɔ̃ kè yah yúkúnɔ́lɛ pè kɔ̃, yõre náh mɛ tɛ̃̀ sɔkɔ à nɔ Liyel tɛ̃ntõ̀ syínyó-òlɛ. 
Ń yṹnpyé, sɔ̃́-á mâh ńtyɩ̃́nɔ pɔ yah yé tyɩ́, tɛ́ wɛ́nkáhkɩ́lɛ n wɛkɩ yé tyɩ́, tir nɛ̂ mɛ kè kɩ pɛkɩ yé tyɩ́, nɛ́ kè náh pɩ tir yɔ́lɛ Liyel ń wɛ̃kɩ mé yo yé syah, képah náh pɩ pnɛ̀nɔ́ yɔ́lɛ, képah náh pɩ Liyel tɛ̃ntõ̀lɛ mé syi yo yé tyɩ́, képah náh pɩ kwɔ́nɔ́ yɔ́lɛ mé yé kɔ̃. 
Yé yah yímɔ́nɩ̀nɛ kɛ̀, sɔ̃́-á yɩkɩlɛ, képah náh pɩ gbrèynɛ; sàh n mɔ́nwàhnmɔ-ńkɛ̃̂nɛ, nɛy se kɩ fɛ̃ sé yɩ̀ dùkù pɩ̃? 
Tɛ́ kèpɔ̃ nɛ̂-á ńkɛ̃́nɛ pè n fɩ̃bɩ ǹ yo névye tyɩ́ nɛ pè yuku n sɔkɔ mɛ́kɩ̀ fwó-ńsah, képah-à ńtyɩ̃́nɔ fɩ̃́bɩ́nwàhnmɔ-ńkɛ̃̂nɛ, nɛ̂ tɛ̃̀ wɔ kɩ ǹ vi mɛ́kɩ̀ fwo yĩnnɛ? 
Sɛ̃́ ó ye, yé nɛ́npɔ́rɩ́-à wɛ́nkáhkɩ́nɔ yísyɔ́lɛ n wɛkɩ nónó-á névye náh mɛ sélɛ n nohnɩ, tyi nónó-á yé n yo, pè kɩ pɩ̃ sɔ̃́ sélɛ? Wɛ́kɩ́nmíndahnmɔnɔ ye yé kɩ n wɛkɩ n min n dah n mɔ ni sɔkɔ. 
Wɛlɔ duke mɛ dò kèkõyṍ, mé náh sé dan ńtɛ̃nɛ n pnɛ. Ǹtɛ kɔ́r mɛ sé núkú núkú pól tyɩ́. 
Ǹtɛ mâh wɛy nɛ̂nɛ n nòhnɩ̀-ǹkɛ̃̂nɛ, ké wɛ́kɩ́-ò n pɩ nɛ́kã̂hnɛ ń tyɩ́, ńmɔ tɔ n mó n pɩ nɛ́kã̂hnɛ tɛ̃̀ tyɩ́. 
Képah ye, Liyel Mirki-á tyi nónó pɩ́nɔ́ yõ̀bènɔrɩlɛ n kɔ̃, yáh mɛ sépi yáhnkɔ̃nɔ sɔkɔ yɔ̀, sé nónó-á Yesu yõ sɛ̃́pú tĩ̀nkìlɛ n yuku yahlɛ, yé sépi náhnáh yah n kɔ̃. 
Kǎh pɩ sɛ̃́, nɛ̂-à wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á à náh kè sãh, tɛ̃̀ kè yah n kɔ̃ Liyel tyɩ́ à ké yah yónɔ́lɛ à wɛ̃kɩ. 
Yé náh wɛ di, nírí pɩ́nɔ́ sɔkɔ wâh wɛ́nkáhkɩ́lɛ n wɛkɩ nɛ̂-á á náh kè sãh, ǹ n pɩ̃ á nɩ sɔkɔ nɛ wáh mɛ nírílɛ n pɩ, ǹtɛ wɛy nɛ̂-á á n wɛkɩ, áwɔ ńtɛ̃ náh kélɛ n nohnɩ. 
Ǹtɛ sɔ̃́ mɛ mé yɩ̃nɛ mè n pɩ wɛ́? Mé kɩ nírílɛ n pɩ wɛ́nkáhkɩ́ sɔkɔ Liyel Mirki gbõ̀ yõ, tɛ́ kɩ kélɛ n pɩ ń noh noh wɛy tɔ sɔkɔ. Mé kɩ yɩ̀lɛ n gbɩ wɛ́nkáhkɩ́ sɔkɔ Liyel Mirki gbõ̀ yõ, tɛ́ kɩ n gbɩ ń noh noh wɛy tɔ sɔkɔ. 
Kàh pɩ́-ńkɛ̃̂nɛ sɛ̃́, tɛ́ á wɛynɛ Liyellɛ n kɔ̃ wɛ́nkáhkɩ́ ó sɔkɔ, nónó-á mɛ ké sèndìlɛ, pépi kɩ pɩ sɔ̃́ tɛ́ kɩ yohnɩ syi yo nɛ Liyel á wɛy mɛ̀ tɛ̃, wáh kɩ sɔ̃́ mɛ n tɔ̃ n sah n wɛkɩ Liyel tyɩ́ sɔ̃́ ǹ pèpɔrɩ pɩ́nɔ́ yĩnnɛ? 
Á Liyel tyɩ́ tɔ̃́nsàhnwɛkɩnɔ mɛ̀-à kɩ sɔ̃́ mɛ plɛ sɔ̃́, kè náh yah yúkúnɔ́lɛ tɛ́lɔ́ kɔ̃ n pi. 
Ńmɔ n wɛkɩ Liyel tyɩ́, máh sɔ̃́npɩ́lɛ n wɛkɩ wɛ́nkáhkɩ́ sɔkɔ à kal yépi póllɛ. 
Ǹtɛ mâh mɛ Yesu yõ sɛ̃́pú tĩ̀nkì wrɔ́, mâh wɛlɔ kwlɔ́ wɛkɩ nónó-á névyelɛ n kwɔ, képah tyɩ́ nɔ ń tyɩ́ kal wɛlɔ kèyõ kwlɔ́ (10 000) wɛ́kɩ́nɔ́ yõ wɛ́nkáhkɩ́ sɔkɔ. 
Ń yṹnpyé, yé káh yɛ yé tyi yah dùkù nɛ mɛ wɛ̃́npì tyɩ́ kɔ̃lɛ. Kàh mɛ pèkè pɩ́nɔ́lɛ, yé nɛ mɛ wɛ̃́npì yrɔ̃́ tyɩ́ kɔ̃lɛ. Ǹtɛ tyi yah dùkù tɛ̃̀ wɔ sɔkɔ, yé nɛ mɛ yèwàhnɩ̀ tyɩ́ kɔ̃lɛ. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ Tɛ̃̀ ǹgbɛ̃ Liyel-á nɛ ńkɛ̃́: «Yìkì yɔ́ névye ye mé kɩ kɔ̃ pè kɩ yo mɔ no nɔ́pi tyɩ́ wɛy yɔ́ sɔkɔ, nɛ̂-á pè náh kélɛ n nohnɩ. Ǹtɛ képah fyé yõ, pè náh fɛ̃ syi n noh n pi ń tyɩ́.» 
Képah ye, wɛ́nkáhkɩ́ wɛ́kɩ́nɔ́lɛ nɛ̂-á á náh kè sãh, kè Liyel tǎhkɩ̀lɛ Yesu yõ sɛ̃́-ńkɛ̃̂púlɛ n wɛ̃kɩ. Ǹtɛ Yesu yõ sɛ̃́pú yohnɩ n mɔ yĩn náh kélɛ. Liyel tɛ̃ntõnè wɛlɔ syínyonɔ wɔlɛ, képah Liyel tǎhkɩ̀lɛ Yesu yõ sɛ̃́púlɛ n wɛ̃kɩ. Ǹtɛ Yesu yõ sɛ̃́-ńkɛ̃̂pú yohnɩ n mɔ yĩn náh kélɛ. 
Sɔ̃́-á Yesu yõ sɛ̃́pú tĩ̀nkì pól-à ńtyɩ̃́nɔ mɛ tuke mɔ, tɛ́ yé pól núkú tyah wɛ́nkáhkɩ́nɔlɛ n wɛkɩ à, nónó-á mɛ ké sèndìlɛ, képah náh pɩ Yesu yõ sɛ̃́-ńkɛ̃̂pú yísyɔ́-à pɔ de nɛ́npɔ́, pè náh yé pɩ n pi vínpɩbɩlɛ di? 
Ǹtɛ kàh mɛ Liyel tɛ̃ntõnè wɛlɔlɛ yé pól n syi n yo, névi nɛ̂-á mɛ ké sèndyàhlɛ, képah náh pɩ Yesu yõ sɛ̃́-ńkɛ̃̂ névi nɛ̂-à de pɔ nɛ́npɔ́, yé wɛlɔ nɔ́pi pól nóhnɔ́ kɩ à kɔ̃ à kɩ pɩ̃ nɛ páh pèkè pɩ, pé tyɩ́-á kah. 
Ǹ fɔ́kɔ́sõnɔ wɔ̃̀bɩ̀ntɛ̃̀nɔ́ pól kɩ wilki mɔ kènkwã̀h yõ kwèyy. Képah sɔkɔ, à kɩ kɔ kwĩnki ǹ yah di sétáh kɩ Liyel gbilki, tɛ́ kɩ n yo nɛ, kègbɩ tɛ̃̀ yõ Liyel-á mɛ yé wrɔ́ sɔkɔ. 
Sɔ̃́ mɛ á yɩ̃nɛ à nɛ n pɩ wɛ́, ń yṹnpyé? Yâh tuke mɔ nɛ yé Liyel ni gbĩ́ nɛ̂nɛ, ǹnɩ̂ kɩ fɛ̃ yɩ̀kɩ̀ gbɩ, ǹnɩ̂ mó kwɔ́nɔ́ kɔ̃, ǹnɩ̂ mó Liyel tyi wɛ̃́kɩ́nɔ́lɛ n wɛ n yo, ǹnɩ̂ mó wɛ́nkáhkɩ́lɛ n wɛkɩ, ǹnɩ̂ mó wɛ́nkáhkɩ́ mɛ̀ yahlɛ n yo. Sépi pól yɩ̃nɛ sè yah yúkúnɔ́lɛ Yesu yõ sɛ̃́pú tĩ̀nkìlɛ n kɔ̃. 
Nósyɔ́-à mɛ ké tyɩ́ nɛ pé wɛ́nkáhkɩ́nɔlɛ n wɛkɩ, kè káh kah nɛ́pĩ̂ nímí, képah náh pɩ tɔ̃́mɩ́ yõ. Pè mó wɛ̃nɛ n syɩkɩ n wɛkɩ. Tɛ́ ǹnɩ̂ mó nɛ mɛ, nɛ̂-á kɩ fɛ̃ sé yahlɛ n yo. 
Ǹtɛ sé yah yó-ò-à mɛ́-ńkɛ̃̂nɛ, wɛ́nkáhkɩ́ wɛ́kɩ́-ò mɛ̀ yɩ̃nɛ à ǹ vyãh vike Yesu yõ sɛ̃́pú tĩ̀nkì wrɔ́. Tɛ̃̀ Liyellɛ n ni ǹ fɔkɔ táh, ǹ gblɔ̌y tyɩ́. 
Nónó-á Liyel tɛ̃ntõ̀ syínyopulɛ, pépi nɛ́pĩ̂ nímí, képah náh pɩ tɔ̃́mɩ́ wɔ yɩ̃nɛ pè wɛkɩ, tɛ́lɔ́ mó nɛ n mɛ pé wɛlɔlɛ n sohlɩ n yah. 
Ǹtɛ Liyel-à tir yɔ́lɛ ǹnɩ̂ yɔ́ wɛ̃kɩ, nɛ̂-á sõ̀ mɛ n wɛkɩ, ǹmɔ yɩ̃nɛ à ǹ vyãh vike, tɛ̃̀ mó ǹ tyɩ́ yo. 
Yé pól kɩ fɛ̃ Liyel tɛ̃ntõnè wɛlɔlɛ n syi n yo nɛ́núkù núkúlɛ, nɛ̂-á kɩ no pól kɔ̃ pè kɩ kwɔ́nɔ́ wɛ, ànɛ̂ sṍré kɩ de pé sɔkɔ. 
Liyellɛ, tyífɛ̃̌ tyɩ́ náh nɔ ǹ tyɩ́. Tir tɔ̃́nsàhnpɩnɔ tyɩ́ nɔ ǹ tyɩ́, yèvɩnɩ Liyel ye ǹnɛ. Képah ye Liyel tɛ̃ntõnè syínyopu yɩ̃nɛ pè nɛ mɛ n fyãh n pnɛ pé gblɔ̌y yõ. Sɔ̃́-á kélɛ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ pól sɔkɔ, 
wɛ̃ tùkènmɔnɔnɔ pól sɔkɔ, syɔ́ yɩ̃nɛ pè tɛ̃ syɩ́syɩ́. Pè náh yɩ̃nɛ yóndahnmɔnɔlɛ wɛy sɔkɔ. Pè yɩ̃nɛ pè tɛ̃ gblɔ̌y tíkínmɔnɔ sɔkɔ, sɔ̃́-á tyi kõ̀nsàhnɔ́ yo kélɛ. 
Pàh mɛ ké tyɩ́ nɛ pé tir yɔ́ dùkù pɩ̃, pè pé pɛbɩ piki yah gbên sɔkɔ. Ké yah náh sê nɛ n yo n dah n mɔ wɛy sɔkɔ Yesu yõ sɛ̃́pú tĩ̀nkì sɔkɔ. 
Liyel wɛy sõ se tyah yépi tyɩ́? Yépi ó se noh kélɛ? 
Névi nɛ̂-à mɛ n sõ nɛ Liyel tɛ̃ntõ̀ syínyo-ò-á pélɛ, képah náh pɩ, Liyel Mirki-á mɛ pé yahlɛ n tɛ̃, wɛy nɛ̂-á mé n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ yɔ̀, tɛ̃̀ yɩ̃nɛ à kè fɛ̃ syi nɛ Tɛ̃̀ ǹgbɛ̃ tir kõ̀nsàhnɔ́-á kélɛ. 
Ǹtɛ névi nɛ̂-à képah mɔ́nkõ̀-ǹkɛ̃̂nɛ, Liyel náh mó mɛ tɛ̃̀nɛ n mɔ n kõ lɔ́. 
Kàh pɩ sɛ̃́ ń yṹnpyé, yé Liyel tɛ̃ntõnè wɛlɔ syínyonɔlɛ n yah n kɔ̃, tɛ́ yé káh wɛ́nkáhkɩ́ wɛ́kɩ́nɔ́ tɔ yah kõ. 
Ǹtɛ tyi pól yɩ̃nɛ sè nɛ n pɩ sé yah yõ, ànɛ̂ sè mó nɛ n tɔ̃ n sah n pɩ.
