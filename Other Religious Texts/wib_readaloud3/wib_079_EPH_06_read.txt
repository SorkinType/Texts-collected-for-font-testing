Yefɛsɩ tãn.
6.
Wɛ̃́npì, yé nɛ n sah yé dúbípú tyɩ́ yõ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ. Képah ye plɛ. 
Yõ̀yénnɛ n sah á sú ànɛ̂ á yṹ yõ. Képah ye tir kõ̀nsàhnɔ́ gbennɛ, vyãh kɔ̃́nɔ́lɛ tàhlɩ̀nɔ́lɛ ké tyɩ́ nɛ: 
Wâh képah pɩ, wáh kɩ mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ tɛ́ kɩ min sɛ̃́kɩ̀ wɛ sétáh yõ sɔkɔ. 
Dúbípú, yépi tɔ káh yèkwɔrɩlɛ n mɔ yé wɛ̃́npì sɔkɔ. À yɩ̃nɛ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀lɛ, yé pélɛ n pu, yè mó kwɔ́nɔ́ pèpɛynɛ pélɛ n kɔ̃, yè mó pélɛ n vi n mɔ pallɛ. 
Yɔ̃bɩ, yé nátãm nónó-á sétáh mɛ̀ yõ sɔkɔ, yé nɛ n sah pé tyɩ́ yõ tíkí ànɛ̂ wílsyahrɩlɛ ké sɔkɔ. Yé kélɛ n pɩ yã́ryã́r dáhbɩ́-ńkɛ̃̂nɛ ké sɔkɔ Krista tyɩ́ sɔ̃́ yé n pɩ kélɛ. 
Kè káh pɩ á kɩ nɛ pǎh yélɛ n kyɔmɩ képah yĩn sɔ̃́ yé n pɩ kélɛ, á kɩ nɛ névi dyɔ yĩ́ĩ́ tɛ̃̀ sɔ̃́ yé n yah kɔ̃ pɩ́-ńsahlɛ dɛ́. Yé kélɛ n pɩ Krista yɔ̃bɩ sɔ̃́ yélɛ, yè Liyel dyɔ yĩ́ĩ́ tɛ̃̀nɛ n pɩ yé nnɔ póllɛ. 
Yé tõ̀lɛ n pɩ pé tyɩ́ yé nnɔ póllɛ, á kɩ nɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́ sɔ̃́ yé n pɩ kélɛ, kè káh nɛ n mɛ névi tyɩ́ sɔ̃́ yé n pɩ kélɛ. 
Yé yé dyɔ sah nɛ n kwnɛ nɛ yé nɛ́núkù núkú póllɛ, nɛ̂-à kɩ sɔ̃́ mɛ yɔ̃̀lɛ ò, nɛ̂ náh mɛ yɔ̃̀lɛ ò, tɛ̃̀-á kɩ ǹ tõ̀ pɩ́nɔ́ lédyáhnɩ́ wɛ n pi Tɛ̃̀ ǹgbɛ̃ tyɩ́. 
Yépi nónó-á yɔ̃bɩ nátãmnɛ, yépi tɔ sɛ̃́ntɛ̃̀nɛ n pɩ pé tyɩ́. Yé káh dírílɛ n kõ pé yõ sɔkɔ. Yé pɩ̃ nɛ yépi ò, pépi ò, yé pól tyɩ́ yõ̀tɛ̃̀ yɔ́-á mɛ yĩ̂nyõ sɔkɔ, nɛ̂-á à náh ǹnɩ̂ yɔ́ ńtɛ̃nɛ n tɔkɔ n pɩ fṍnfṍ ǹ ní-òlɛ. 
Ké tyɛ́-ńsah, yé yɛ yé tǎhkɩ̀ nɛ n wil Tɛ̃̀ ǹgbɛ̃ tɛ̃́npnɛ̀nɔ́ ǹgbɛ̃ tyɩ́. 
Yé Liyel mɛ́fwó yî tɔkɔ, sɔ̃̀nko yè tɛ̃ wɛ sétõ yĩ́nvǐ tyi yah kõ. 
Yé náh wɛ di, ápi mɛ́kɩ̀ náh mɛ ké yah mɔ névye tyɩ́ dɛ́. Ǹtɛ kǎh mɛ ké yah mɔ nónó tyɩ́ pépi ye pɩ, yõ̀tãmnɛ, ànɛ̂ tǎhkɩ̀nɔ́lɛ, ànɛ̂ névye fnɛ̀pu nónó-á mɛ tõ̀lɛ n pɩ kèkõyṍ fɛ̃́ntũ̂ mɛ̀ sɔkɔ, ànɛ̂ pèkè pɩ pɩ mrɛ́y nónó-á mɛ tõ̀lɛ n pɩ yĩ̂nyõ. 
Képah ye, yé Liyel mɛ́fwóyî tɔkɔ, sɔ̃̀nko yòhnpèkè sõ-à pɔ, yè tɛ̃ n wɛ sépɛ fwo. Képah-à pɩ yé kɩ yĩn gbáhkɩ̀ núkú yõ. 
Yé nɛ n mɛ yé gblɔ̌y vi n sah. Yé yɛ gbɩ yónɔ́ pɩ syɩ̃ kɔ̃lɛ yé tyɩ́, gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ mó pɩ vyɔrɩ fwɔ̀mɔnɩnɛ yé tyɩ́. 
Nɩ̀vɩ̀nɩ̀ tir nɛ̂-á yèvɩnɩnɛ n kɔ̃, ké yónɔ́ dékè nɛ mɛ yé sɔkɔ yé gbyẽ̂h sɔkɔ lékyêh tyɩ́ kɔ̃lɛ. 
Yé Liyel yõ sɛ̃́nɔ́ tɛ̃ n yĩn mɛ̀mɛ̀kɩ̀ swàhkɩ̀ sɔ̃́ yé gbã̀n sɔkɔ gbĩ́mɛgbĩ́, yè fɛ̃ pèkè pɩ́-ò pnɛ́pêl pól swah kélɛ. 
Yé pwáhnmɔnɔ fɛ̃ n syi á kɩ nɛ vyɔrɩ gbétú sɔ̃́ yé kwĩnki, yè mó Liyel wɛy fɛ̃ n syi kèkírkì sɔ̃́ Liyel Mirki-á nɛ̂ kɔ̃. 
Yé yɛ yé Liyel nínɔ́ ànɛ̂ yé Liyel fàlnɔ́ duke pól nɛ n pɩ Liyel Mirki yah tɛ̃́nɔ́ sɔkɔ gbĩ́mɛgbĩ́. Yé gbah tɛ̃ nɛ n yuku yahlɛ képah sɔkɔ. Yé Liyellɛ n ni Yesu yõ sɛ̃́pú tĩ̀nkì pól tyɩ́. 
Yé nírílɛ n pɩ ńmɔ tɔ tyɩ́, sɔ̃̀nko Liyel wɛlɔ pèplɔ dahbɩ ń vyãh sɔkɔ mâh n wɛkɩ gbĩ́ nɛ̂nɛ, mè n tɛ̃ n wɛ nɩ̀sãh wɛ mè tyípǐllɛ n yo n wah nónó-á mɛ Nɩ̀vɩ̀nɩ̀ tir sɔkɔ. 
Kénɛ Nɩ̀vɩ̀nɩ̀ tir yah yuku yĩn ye kârnnɛ mɔ́nɔ́lɛ ń gbã̀n tyɩ́ wɔ̀kɔ̀npèkè sɔkɔ yɔ̀. Yé nírílɛ n pɩ ń tyɩ́ ń kɔ̃ mè tɛ̃ n wɛ kélɛ n yo nɩ̀sãhlɛ kǎh yɩ̃nɛ sɔ̃́. 
Mé mɛ ké tyɩ́ nɛ yépi tɔ pɩ̃ ń tyɩ́-á mɛ dùkù nɛ̂nɛ, ànɛ̂ tir nɛ̂-á mé mɛ n pɩ. Á yṹnpi Tisyiki nɛ̂-á ń nɛ́nɔ́nɩ́nɛ, tɛ́ mɛ tõ̀lɛ n pɩ pallɛ Tɛ̃̀ ǹgbɛ̃ tyɩ́, ǹmɔ kɩ pɔ ń tyɩ́ kèdùkù pól yo yé tyɩ́. 
Ńmɔ ye wɛ tɛ n tɛkɩ n mɔ ǹnɛ yé tyɩ́ nɛ à kyɩ á tyɩ́ kèdùkù yo yé syah à mó yé nɩ sũ. 
Mé kélɛ n yah n kɔ̃ á Sú Liyel ànɛ̂ Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́ nɛ pè yèvɩnɩ, ànɛ̂ tyɩ́nɔnɔ, ànɛ̂ Liyel yõ sɛ̃́nɔ́lɛ á yṹnpyé pól kɔ̃. 
Á Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́-á mɛ nɔ́nɔ́lɛ nónó pól tyɩ́ tyɛ́tyɛ́nɩ́ kɛ̃́nɛ, mé kélɛ n yah n kɔ̃ Liyel tyɩ́ nɛ à yõ̀bènɔrɩ pɩ pé tyɩ́.
