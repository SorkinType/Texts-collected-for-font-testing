Korɛ̃nti tãn.
4.
Yé yɩ̃nɛ yè pɩ̃ nɛ Krista tõ̀npɩbɩ-á ápilɛ, ànɛ̂ tyi nónó-á lésõ wɔ̃̀bɩ̀nsàhnɔ́lɛ, Liyel-á sélɛ ápi gbõ̀ dahbɩ, á kɔ̃ á syi n yo. 
Ǹtɛ tyi-à dahbɩ nɛ̂ gbõ̀, kè n nɛ n mɛ yáhnkɔ̃nɔlɛ tɛ̃̀ tyɩ́ nɛ à nɛ n mɛ tómm sé pɩ́nɔ́ sɔkɔ. 
Ńmɔ tɛ̃̀ wɔ tyɩ́, yépi képah náh pɩ tukey tùkèpu-à kɩ sɔ̃́ lékàh, képah náh pɩ lénɔ̀lɛ ńnɛ n kɔ̃ sɔ̃́, képah náh mɛ tir yɔ́lɛ ńmɔ yah. Ńmɔ tɛ̃̀ wɔ náh mɛ lékàh, képah náh pɩ lénɔ̀lɛ ń gblɔ̌ynɛ n kɔ̃. 
Tɔ̀tɔ̀ náh mɛ ńmɔ nɩ sɔkɔ póllɛ nɛ máh tyípékè pɩ. Ǹtɛ képah náh n wɛ̃kɩ nɛ lékàh kɛ̃́ névi-á ńnɛ tɛ̃̀ wɔlɛ. Tɛ̃̀ ǹgbɛ̃ ó ye kɩ fɛ̃ lékàh, képah náh pɩ lénɔ̀lɛ ńmɔ kɔ̃. 
Képah ye, yé káh ǹmɔ dyàh tɛ̃ lékàhlɛ névyelɛ n kɔ̃ ké gbĩ́ nɔ́-ńkɛ̃̂ sɔkɔ. Yé ǹ pɔ́nɔ́ syɩkɩ. Ǹmɔ kɩ pɔ sètĩ̀n sõ̀ tyi wilki mɔ kwèyy, tɛ́ kɩ névye fɔkɔ sõnɔ wilki mɔ kégbɔ́. Képah sɔkɔ, yõ yɔ̃̀nɔ́ nɛ̂-à yɩ̃nɛ nɛ̂nɛ, tɛ̃̀ kɩ kè wɛ Liyel tyɩ́. 
Ń yṹnpyé, yépi yĩn ye mé Apɔlɔsɩ ànɛ̂ ń gblɔ̌y pɩ dùkù tɔkɔ à yé wɛ̃kɩ, nɛ̂-á kɩ yé kɔ̃ yé kɩ pɩ̃ nɛ tyi nónó-á wãrkɩ, ǹnɩ̂ yɔ́ ńtɛ̃ náh yɩ̃nɛ à nɛ n pɩ n kah n mɔ sépi yõ. Képah sɔkɔ, ǹnɩ̂ yɔ́ ńtɛ̃ náh ǹ gblɔ̌y gbílkínɔ́ sɔkɔ, ǹnɩ̂ yɔ́ tɔkɔ fṍnfṍ pɩ n kal n pi ǹ ní-òlɛ. 
Kwâh nɛ̂ kɩ ǹ kɔ̃ á kɩ kal no tɛ́lɔ́lɛ? Kwâh nɛ̂ tɛ̃̀ mɛ á tyɩ́, nɛ́ Liyel náh ǹ kɔ̃ kélɛ? Ǹtɛ kɔ̃́nɔ́ à kélɛ à, sɔ̃́ pɩ tɛ́ á á gblɔ̌ynɛ n gbilki á kɩ nɛ Liyel kósyì ǹ kɔ̃ kélɛ? 
Yî nónó pól-á yé n yah n kɔ̃, yé ye núkú sè wɛ tɛ yé yo kɛ̀? Yé pɩ tɛ tahbɩlɛ yé yo kɛ̀? Yõ̀tɔ̃rɩ́ ye núkú yé mɛ n pɩ ápi kɛ̃́nɛ yé yo kɛ̀? Yáh náhkɩ sõ̀ mɛ wɛ pɩ yõ̀tãmnɛ ápi tɔ náh kɩ mɛ yõ̀tɔ̃rɩ́ sɔkɔ yélɛ. 
Ńkɛ̃́ kè mɛ ńmɔ tɛ̃̀ tyɩ́, á kɩ nɛ Liyel sɔ̃́ ápi Yesu tɛ̃ntõ̀ névye pɩ no pól náh névyelɛ. Á pɩ névye sɔ̃́ nónó-á kɩ pɔ n ko n mɔ n pi. Kǎh pɩ sɛ̃́, á pɩ syáh syah yîlɛ kèkõyṍ pól tyɩ́, yĩ̂nyõ tõ̀npɩbɩ tɔkɔ dohnɩ névye yõ. 
Ápi pɩ nɛ́wã̂nnɛ Krista yĩnnɛ, tɛ́ yépi mɛ pɩ yɩ̃ pɩ̃́púlɛ Krista sɔkɔ yé yo kɛ̀? Ápi pɩ tɛ̃́nkwlɔpulɛ, tɛ́ yépi mɛ pɩ tɛ̃́nwɛpulɛ yé yo kɛ̀? Ápi mɛ yílɔ́nɔ́lɛ n wɛ, tɛ́ yõ̀yén mɛ n sah yépi yõ yé yo kɛ̀? 
Yé náh wɛ di, fɔ́ɔ́ à pɔ syɩ núkúnúkú mɛ̀ ńtɛ̃ tyɩ́, kwɛkɩ ànɛ̂ níkì mɛ álɛ n kwlɔ. Mɔ mɔ flɔ náh mɛ á tyɩ́. Á mɛ kolɛ n syi, tɛ́ tɛ̃̀-ǹsah náh mɛ á tyɩ́. 
Á mɛ tõ̀lɛ n pɩ kétǎhkɩ̀yɔ̂ á gbã̀nnɛ, à á yõ yõ tɛ̃̀nɛ n wɛ. Pàh álɛ n gbyɔ, ǎ sãmnɛ n mɔ pé sɔkɔ. Pàh álɛ n fõh, ǎ kè fɛ̃ n syi. 
Pàh á tyɩ́lɛ n yo n ko, ǎ wɛ̀npèplɔlɛ n wɛkɩ pé tyɩ́. Á mɛ pɩ tɛ̃ kèkõyṍ yésyɩ̂ ànɛ̂ ké yítúkúnìnɛ névye tyɩ́, pópó à pɔ syɩ núkúnúkú mɛ̀ tyɩ́ yɔ̀. 
Ǹtɛ mé náh mɛ tyi nɔ́pilɛ n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ nɛ mé sennɛ yé tɛ̃ dɛ́, gbáhdyâh ye mé n di n sah kélɛ yé tyɩ́, yépi nónó tyɩ́-á nɔ yĩ́ĩ́ ń tyɩ́ á kɩ nɛ ń wɛ̃́npì sɔ̃́. 
Yé yah, Nɩ̀vɩ̀nɩ̀ tir yónɔ́ gbõ̀ yõ sɔkɔ ye yé pɩ ńmɔ wɛ̃́npìlɛ à sɛ̃ Yesu Krista yõ. Képah ye, ńmɔ mɛ n mɔ kõ á kɩ nɛ yé sú sɔ̃́. Nónó-á mɛ yélɛ n kwɔ Yesu Krista tyilɛ, pàh sɔ̃́ mɛ nɔ nɛ́pĩ̂ kèyõ kwlɔ́lɛ ò (10 000), yé pɩ̃ nɛ sú núkú tɛ̃̀ ó-á yé tyɩ́. 
Képah ye, mé kélɛ n yah n kɔ̃ yé tyɩ́ nɛ, yé ńmɔ pɩ dùkùlɛ n yah n pɩ. 
Képah yĩn ye tɔ̃ mé Timotelɛ n tɛkɩ mɔ yé tyɩ́, nɛ̂-á mɛ ń pídĩ́ sɔ̃́. Ǹ tyɩ́ nɔ yĩ́ĩ́ ń tyɩ́. Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ kɔ́-ò pèpɛy ye ǹnɛ. Ǹmɔ ye kɩ pɔ yé dyɔlɛ n to n kõ máh mɛ Krista wɛ̃̀kɩ̀lɛ n kɔ dùkù nɛ̂nɛ, sɔ̃́-á mé kélɛ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ póllɛ n yah n kwɔ n kɔ̃ tã̀ntã̀n pól sɔkɔ. 
Ǹtɛ yé túkù mɛ n sõ nɛ mé náh ń syɩ n pɔ n pi yé tĩ̀nnɛ, tɛ́ mɛ yuku pé gblɔ̌ynɛ n yɔ̃ n sah. 
Ǹtɛ kàh kɩ nɔ Tɛ̃̀ ǹgbɛ̃ tyɩ́, mé kɩ n syɩ pɔ yé tĩ̀nnɛ núkúnúkú. Wɛ̀ngbìl wɛ́kɩ́nɔ́ náh n wɛ̃kɩ nɛ Liyel tɔ̃́rɩ́ névi-á álɛ, Liyel tǎhkɩ̀ nɛ̂-á á sɔkɔ, képah ye n wɛ̃kɩ kélɛ. Képah ye, mâh pɔ pɔ, kè náh n yĩn n pi gblɔ̌y gbílkípú wɛ̀ngbìl nóhnɔ́-ó tyɩ́ dɛ́, tɛ̃́nwɛnɔ-à kɩ mɛ pé tyɩ́, mé kɩ kè pɩ̃. 
Máh kɩ kènpyãhnɩ́ tɔkɔ pɔ yé tyɩ́, képah náh pɩ máh kɩ pɔ yárr yé tyɩ́ tyɩ́nɔnɔ sɔkɔ, nɛ̂ tɛ̃̀ mɛ yé n yah n kɔ̃?
