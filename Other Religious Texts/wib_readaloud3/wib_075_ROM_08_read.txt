Wrome tãn.
8.
Núkúnúkú tɛ̃̀nɛ, nónó wrɔ́-á mɛ kwrɔ́nmɔnɔlɛ Yesu Kristalɛ, Liyel tukey náh tɔ̃ n syɩkɩ n pi pé yõ. 
Yé náh wɛ di, Liyel Mirki nɛ̂-á minnɛ névilɛ n kɔ̃ tɛ̃̀ wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Yesu Kristalɛ, képah ye wɛ wilki ńmɔlɛ tyípékè ànɛ̂ kũ mnɔ sɔkɔ. 
Moyisi tyi kõ̀nsàhnɔ́ náh sõ̀ fɛ̃ képah pɩ n pi dò. Névye gblɔ̌y dyɔ dyɔ tyi pɩ́nɔ́ sõ̀ mɛ kǎn pɩ kélɛ képah ye. Kǎh pɩ sɛ̃́, Liyel mɛ ǹ Pídĩ́ tɛkɩ mɔ à pɔ dubi névi wil dùkùlɛ ǹ tyɩ́ ápi tyípêl pɩ́pú tyɩ́ kɔ̃lɔ, tɛ́ ku ápi tyípêl yĩnnɛ, sé tyɩ́ wɛ n yɩkɩ yĩnnnɛ névye sɔkɔ. 
Ápi nónó-á á dyɔ yĩ́ĩ́ tyi wɛ yɛ, tɛ́ Liyel Mirki sõ̀ tɛ̃. Liyel kè pɩ nɛ á fɛ̃ tyi kõ̀nsàhnɔ́ tyíyónɔ́lɛ n pɩ. Képah ye à kɔ̃ à képah pɩ. 
Yé náh wɛ di, névi nɛ̂-à yɛ ǹ dyɔ yĩ́ĩ́ tyi yah syi ǹ tyɩ́, tɛ̃̀ sõnɔ n nɛ n mɛ ǹ dyɔ yĩ́ĩ́ tyi ó yõ. Nɛ̂-à mó yɛ Liyel Mirki yah syi ǹ tyɩ́, tɛ̃̀ sõnɔ n nɛ n mɛ Liyel Mirki dyɔ yĩ́ĩ́ tyi ó yõ. 
Wâh á dyɔ yĩ́ĩ́ tyi sõ̀ tɛ̃, ké syì-ǹsah n pɩ kũnɛ. Tɛ́ wâh Liyel Mirki dyɔ yĩ́ĩ́ sõ̀ tɛ̃, ké syì-ǹsah n pɩ yèvɩnɩ ànɛ̂ tyɛ́-ńkɛ̃̂ min wɛ́nɔ́lɛ. 
Kègbɩ yõ, névye nónó dyɔ yĩ́ĩ́ tyi-á yah syi pé tyɩ́, Liyel sépɛ́bɩ́ ye pélɛ pǎh sɔ̃́npɩ́lɛ yahle sàhnɔ́lɛ Liyel tyi kõ̀nsàhnɔ́ yóré yõ. Pè náh fɛ̃ sé wã̀l kɔ ńtɛ̃nɛ dò. 
Tɛ́ névye-à pé gblɔ̌y dyɔ yĩ́ĩ́ tyilɛ n pɩ, Liyel yĩ́ náh n yuku pélɛ. 
Yépi tɛ̃̀ wɔlɛ, yé náh mɛ yé gblɔ̌y dyɔ yĩ́ĩ́ tyilɛ n pɩ. Liyel Mirki dyɔ yĩ́ĩ́ tyi ye yé mɛ n pɩ, képah nɛ̂-á mɛ yé sɔkɔ. Krista Mirki-à n knɛ névi nɛ̂ sɔkɔ, tɛ̃̀ náh mɛ ǹ tyɩ́ yõ. 
Krista-à mɛ yé yahlɛ n tɛ̃, yé kɩ sénsɔ̃ pɔ kũ n pi ó, ǹtɛ Liyel Mirki gbõ̀ yõ, yé pɩ nɛ́gblɔ́lɛ Liyel yah sɔkɔ à tyɛ́-ńkɛ̃̂ min wɛ. 
Liyel nɛ̂-á yilki mɔ Yesu lékólɛ, ǹ Mirki-à kɩ mɛ yépi tɔ sɔkɔ, yáh sɔ̃́ pɔ ku, yé lékyɩ̂ kɩ yilki mɔ kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Képah ye ń yṹnpyé, á yɩ̃nɛ à Liyel Mirki dyɔ yĩ́ĩ́ tyilɛ n pɩ, á náh tɔ̃ yɩ̃nɛ à á gblɔ̌y dyɔ yĩ́ĩ́ tyílɛ n pɩ. 
Yâh yé dyɔ yĩ́ĩ́ tyilɛ n pɩ, yé kɩ ku. Ǹtɛ Liyel Mirki gbõ̀ yõ yâh yé dyɔ yĩ́ĩ́ tyi wɛ yɛ, yé kɩ tyɛ́-ńkɛ̃̂ min wɛ. 
Yé náh wɛ di, Liyel Mirki-à nónó yahlɛ n tɛ̃, Liyel wɛ̃́npì ye pélɛ. 
Liyel-á ǹ Mirki nɛ̂nɛ yé kɔ̃, kè náh yélɛ n mɔ yɔ̃rɩ pɩ́nɔ́ sɔkɔ ànɛ̂ tíkí pɩ́nɔ́ sɔkɔ dɛ́. Képah ye pɩ yélɛ Liyel wɛ̃́npìlɛ, tɛ́ yé fɛ̃ n yo yĩ́npɔ̃́ ǹgbɛ̃nɛ nɛ: «Bɔ̌, ápi Sú!» 
Liyel Mirki mɛ̀ ńtɛ̃ ye á kɔ̃ á sɛ̃ yĩ́ĩ́ ké yõ nɛ Liyel wɛ̃́npì-á álɛ. 
Tɛ́ áyáh fyɔ̀ pɩ ǹ wɛ̃́npìlɛ, á vyãh kɩ wɛ pèpɔrɩ sɔkɔ, wǎh ǹ vyãhlɛ ǹ wɛ̃́npì kɔ̃ nɛ̂ tyɩ́ sɔkɔ. Liyel-á pèpɔrɩ nɛ̂ pɩ Krista tyɩ́, ápi tɔ vyãh kɩ wɛ ké sɔkɔ. Áyáh fyɔ̀ mɛ yèvyãhrɩlɛ n pɩ wɛ̃ tyɩ́ Kristalɛ, gbílkínɔ́ kɩ n sah n pi á yõ ǹnɛ wɛ̃ tyɩ́. 
Mé pɩ̃ nɛ áyáh mɛ yèvyãhrɩ nɛ̂ sɔkɔ kwɛy mɛ̀nɛ, képah náh mɛ sãh n yah yah tɛ̃̀nɛ gbílkínɔ́ ǹgbɛ̃ nɛ̂-á kɩ pɔ n sah n pi á yõ ńsõ̂nɛ. 
Liyel wɛ̃́npì-á kɩ gbílkínɔ́ wɛ n pi sɔ̃́, képah ye ǹ yî mɔ́nɔ́nɔ pól mɛ n dɛkɩ. 
Yé náh wɛ di, Liyel yî mɔ́nɔ́nɔ pól mɛ yõre kɛ̃́ tǎhkɩ̀ yɔ́ gbõ̀ sõ̀ sɔkɔ. Pépi dyɔ dyɔ tɛ̃̀ náh nɛ kè pɩ sɛ̃́ dɛ́. Nɛ̂-á sah pélɛ ké gbõ̀ sõ̀, ǹmɔ ye pɩ kélɛ. Ǹtɛ gbõ̀ sàh-ǹsah tɔ̃ mɛ pé tyɩ́. 
Sõ yɔ́ mɛ n pi, Liyel yî mɔ́nɔ́nɔ pól kɩ pɔ n wilki n pi yɔ̃rɩ sɔkɔ yɩ́kɩ́nɔ́ pɩ́-ò gbõ̀. Tɛ́ kɩ gbílkínɔ́ wɛ wɛ̃ tyɩ́ Liyel wɛ̃́npìlɛ. 
Ǹtɛ à pɔ syɩ kwɛy mɛ̀ tyɩ́, á pɩ̃ nɛ Liyel yî mɔ́nɔ́nɔ pól-á mɛ n gbĩ kúrntɛ̃́sè tyɩ́ kɔ̃lɛ yèvyãhrɩ tɛ̃̀ tyɩ́. 
Tɛ́ kè náh yĩn pépi ó tyɩ́ dɛ́, Liyel-á ǹ Mirkilɛ ǹgbòlɛ ápi nónó kɔ̃, ápi tɔ mɛ n gbĩ á fɔkɔ táhlɔ́ sɔkɔ, mɛ gbĩ̀yĩ̀kìlɛ n syɩkɩ sõ nɛ̂-á Liyel kɩ kè wɛ̃kɩ n pi nɛ áyáh pɩ pé wɛ̃́npìlɛ, tɛ́ kɩ pwáhnmɔnɔ wɛ dal mɔ. 
Yé náh wɛ di, á ye pwáhnmɔnɔ wɛ. Tɛ́ kè mɛ á kɩ nɛ sɔ̃́-á ńkɛ̃́nɛ névi ǹ gbõ̀ sah kwâh yɔ́ pɔ́nwɛnɔ yõ. Ǹtɛ wàh kè wɛ syi tɛ, à náh tɔ̃ ǹ gbõ̀ sah n pi ké pɔ́nwɛnɔ yõ dò. Wàh kè wɛ syi tɛ, à kɩ tɔ̃ ǹ gbõ̀ sah sɔ̃́ ké pɔ́nwɛnɔ yõ? 
Ǹtɛ áyâh tɔ̃̌nɩ̀ mɛ kwâh nɛ̂ wɛ́nsyi-ńkɛ̃̂nɛ, ǎ sṍmnɔ́ pɩ ǹ tɛ́ á gbõ̀ sah ké pɔ́nwɛnɔ yõ. 
Sɛ̃́ ó ye, Liyel Mirki tɔ mɛ ápilɛ n yohnɩ n mɔ á tɛ̃́nkwlɔ́nɔ yĩnnɛ. Yé náh wɛ di, ápi náh pɩ̃ nɛ̂-á yɩ̃nɛ nínɔ́lɛ. Ǹtɛ Liyel Mirki mɛ nírílɛ n pɩ ápi tyɩ́ nɛ̂-á ápi náh fɛ̃ ké dùkù yah yo. 
Tɛ́ Liyel nɛ̂-á névye fɔkɔ táhlɔ́lɛ n pnɛ, ǹ Mirki-á mɛ nɛ̂nɛ n yah n kɔ̃, ǹmɔ kélɛ n pnɛ. Yé náh wɛ di, ǹ Mirki nírílɛ n pɩ Yesu yõ sɛ̃́pú tyɩ́ à yɩ̃nɛ ǹmɔ dyɔ yĩ́ĩ́ tɛ̃̀nɛ. 
Á mó tɔ̃ pɩ̃ tɔlɛ nɛ Liyel tyɩ́-á nɔ nónó tyɩ́, ǎ tyi póllɛ n pɩ pépi tyɩ́ pɛkɩ yĩnnɛ, pépi nónó-á à ye à yɩ̃nɛ wǎh kè yah tɔkɔ sah sɔ̃́. 
Liyel-á nónó yah wilki ǹgbò dùkùlɛ, à ye kwãh tɛ̃ kè yah tɔkɔ tɔlɛ nɛ pǎh yɩ̃nɛ pè pɩ pé Pídĩ́ tyɩ́ kɔ̃lɛ, ǹmɔ kɔ̃ à pɩ pé pól nɛ́nkyɩ́lɛ. 
Wǎh kwãh tɛ̃ nónó yah tɔkɔ, à ye pè ye tɔlɛ. Wǎh nónó ye, à ye pè pɩ nɛ́gblɔ́ tɔlɛ ǹ yah sɔkɔ. Wǎh nónó pɩ nɛ́gblɔ́lɛ ǹ yah sɔkɔ, à ye yõ yɔ̃̀nɔ́lɛ pè kɔ̃. 
Sɔ̃́ mɛ ápi kɩ yo yɔ̀ wɛ́? Liyel-à mɛ ápilɛ, nɛ̂ kɩ fɛ̃ á fwo? 
Ǹmɔ nɛ̂-á à náh yɛ ǹ yɛ̃́ngbɛ̃́ Pídĩ́ kɔ̃́nɔ́lɛ, à à mɔ kũ kɔ̃ ápi pól yĩnnɛ. À kɩ pɩ sɔ̃́ tɛ́ à náh yî tɛ́lɔ́ póllɛ á kɔ̃ à n tahlɩ n pi ǹmɔ gbõ̀ yõ. 
Liyel-á névye nónó yah wilki, nɛ̂ tɛ̃̀ kɩ pɔ n tɔkɔ pélɛ n kɔ̃? Liyel ye pɩ névyelɛ nɛ́gblɔ́lɛ ǹ yah sɔkɔ. 
Nɛ̂ kɩ tɔ̃ fɛ̃ Liyel tukey kɔ̃ kè kɩ syɩkɩ pé yõ? Yesu Krista ye ku, tɛ́ kè náh yĩn képah ó tyɩ́ dɛ́, ǹ lékó yilki mɔ. Yé náh wɛ di, à mɛ tɛ̃̀nɔ́lɛ Liyel gbõ̀yõ̀ sõ̀lɛ, tɛ́ mɛ nírílɛ n pɩ ápi tyɩ́. 
Ápi tyɩ́nɔnɔ nɛ̂-á mɛ Krista sɔkɔ, kwâh yɔ́ ńtɛ̃ náh fɛ̃ ké yah kõ n pi. Yèvyãhrɩ ò, tíkí képékè ò, fṍhnɔ́ ò, kwɛkɩ ò, kwâh kɛ̃́ ò, dlo ò, kũ ò, sé sɔkɔ yɔ́ ńtɛ̃ náh fɛ̃ ké yah kõ n pi. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Áwɔ yĩnnɛ, ápi mɛ kũ dlo sɔkɔ sõmɛsṍ. Pè mɛ álɛ n pɩ á kɩ nɛ yípɛ̂y sɔ̃́ pè tɛ̃ n yuku kò-ǹsah sɔkɔ.» 
Ǹtɛ sépi pól sɔkɔ, á tyɩ́-á nɔ Liyel nɛ̂ tyɩ́, ǹmɔ gbõ̀ yõ á pɩ tɛ̃́npnɛ̀pu ǹgbnɔlɛ. 
Gbɩ yõ, ápi tyɩ́nɔnɔ nɛ̂-á mɛ Liyel sɔkɔ, tɛ́ à syɔ̃ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́ à kélɛ á wɛ̃kɩ, ń sɛ̃́nyĩnɔ mɛ ké yõ nɛ, kwáhmɔ́nɩ́ yɔ́ ńtɛ̃ náh fɛ̃ ké yah kõ n pi. Kàh kɩ sɔ̃́ kũnɛ ò, kàh kɩ sɔ̃́ minnɛ ò, kàh kɩ sɔ̃́ yĩ̂nyõ tõ̀npɩbɩlɛ ò, kàh kɩ sɔ̃́ yõ̀tɔ̃rɩ́nɔlɛ ò, kàh kɩ sɔ̃́ kwɛy gbĩ tyilɛ ò, kàh kɩ sɔ̃́ ńsõ̂ tyilɛ ò, kàh kɩ sɔ̃́ tɛ̃́npnɛ̀nɔ́nɔlɛ ò, kàh kɩ sɔ̃́ yĩ̂nyõ tǎhkɩ̀nɔ́lɛ ò, kàh kɩ sɔ̃́ sétáh sõwãh tǎhkɩ̀nɔ́lɛ ò, sé sɔkɔ yɔ́ ńtɛ̃ náh fɛ̃ ápi tyɩ́nɔnɔ yah kõ n pi ǹ tyɩ́.
