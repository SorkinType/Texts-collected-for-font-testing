Tyípɩ́nɩ̀.
28.
Képah sɔkɔ, áyáh pwah sɛ̃́, á kè noh nɛ pnɛ yɔ nɩyṍ gbɛ̃̀nyah mɛ̀ yĩn-á Maltɩlɛ. 
Kénɛ kwil tãn mɛ á tɔ̃ sah tɛ̃ yĩ́ĩ́ pallɛ. Tɛ́ pũ̀-á sõ̀ mɛ n pɩ, tɛ́ vɩrɩ mɛ n pɩ sɛ̃́, pè mɛ nyàh mɔ tìí á pól kɔ̃ á mɛ tuke mɔ tɛ̃ n wnɛ. 
Képah sɔkɔ, Pole-á kyɩ lésɔ́pî kwel nɛ pé pɔ mɔ nyàh sɔkɔ, nyàh swɔ̀ɔ̀-á kyɩ sé tyɩ́, wáhnpo yɔ́ wil sé sɔkɔ à Pole kõ, fɔ́ɔ́ à toh mɔ ǹ gbõ̀ tyɩ́. 
Kwil mɛ̀ tãn-á wahy wɛ tóhnmɔnɔlɛ ǹ gbõ̀ tyɩ́ sɛ̃́, pè tyah n yo wɛ̃ tyɩ́ nɛ, tɔ̀tɔ̀ náh mɛ ké sɔkɔ póllɛ, dĩ́ mɛ̀-á nɛ́kókè mɔ yah. Wǎh sénsɔ̃ pwah pnɛ yɔ gbõ̀ sɔkɔ ó, ǹtɛ pépi sétõ nɛ̂-á gbɩlɛ n wah, képah tɛ̃̀ wɔ náh à sah n pi min sɔkɔ dɛ́! 
Képah sɔkɔ, Pole wahy mɛ̀ mir dah mɔ nyàh sɔkɔ. Tɛ́ syẽ́hẽ́h kwéy ńtɛ̃ náh kyɩ Pole wil tyɩ́. 
Tɛ́ kwil tãn sõ̀ mɛ syɩ́kɩ́ntɛ̃̀nɔ́lɛ n pɩ nɛ, dǒ páh kɩ à wɛ à kɩ mɛ tyah n sɛ̃, képah náh pɩ, kɩ syɩkɩ n ku n pi nɛ́npɔ́ swɔ. Ǹtɛ pǎh képah syɩkɩ tɛ̃ wrɛ́tɛ́tɛ́, pè pɔ pɩ̃ nɛ, pèkè yɔ́ ńtɛ̃ náh à nɔ n pi. Kǎh pɩ sɛ̃́, pè pé vyɛ̃y vi à tyah n yo nɛ, tõ̀ yɔ́-á ǹnɛ. 
Tɛ́ ńyãh sɔkɔ-á sõ̀ álɛ, fɩ̃́ yɔ́ sõ̀ mɛ nɛ́npɔ́ tnɔ̂, nɛ̂-á pnɛ yɔ nɩyṍ gbɛ̃̀nyah mɛ̀ yõ̀tɛ̃̀ dĩ́ tyɩ́lɛ. Kénɛ dĩ́ yĩn ye sõ̀ Publiyusilɛ. Ǹmɔ mɛ á tɔ̃ sah tɛ̃ pallɛ ǹ kódíbí sɔ̃́, tɛ́ á kɔ̃ á twah ǹ tyɩ́ à sèpĩ̀n tɔ̃́nɩ̀ pɩ. 
Tɛ́ kénɛ dĩ́ sú sõ̀ mɛ yam pɛ̀kɩ̀ yõ. Ǹ wil sõ̀ mɛ vyãh, tɛ́ ǹ fɔkɔ tɔ mɛ kye. Képah tɛ̃̀nɛ, Pole mɛ kyɩ yah ǹ tyɩ́, tɛ́ ǹ gbã̀n sah ǹ yõ à Liyel ni ǹ tyɩ́, à à dɛ mɔ. 
Képah náh, kénɛ kwil yámpú tɛ́lɔ́ tɔ mɛ pɔ Pole tyɩ́, tɛ́ pɔ dɛ́nmɔnɔ wɛ. 
Kǎh pɩ sɛ̃́, á yõ yɔ̃ nɛ́npɔ́ sɔkɔ kègbɩlɛ. Tɛ́ áyáh pɔ n tyah sɔ́kɔ́nɔ́lɛ, yî nónó yétõ̂-á sõ̀ mɛ á tɛ̃, pè sé póllɛ á kɔ̃. 
Képah náh, ńwɔ̃́lɔ́ tɔ̃́nɔ́ sɔ́kɔ́ntɛnɔ sɔkɔ, á mɛ de sɔkɔ Alɛkisãdri kwil krótókè ǹgbɛ̃ yɔ́ sɔkɔ, nɛ̂ yĩn-á kènìnìnɛ. Képah sõ̀ mɛ kwɔ̀vɩrɩ gbĩ́ tyɛ́nɔ́ syɩkɩ tɛ̃ Maltɩ gbɛ̃̀nyah sɔkɔ. 
Áyáh kyɩ yĩni tɛ̃ Sirakusi kwil, á tɛ̃ nɛ́npɔ́ à sèpĩ̀n tɔ̃́nɩ̀ pɩ. 
Áyáh yal nɛ́npɔ́, à tyah n tɔrɩ n yuku krótókèlɛ yɔ vyãh tyɩ́, pópó à kyɩ de Wregbyo kwil. Ké dĩndallɛ, yã́hkɩ́ kõ wil á náh tĩ̀nnɛ à pɔ. Kǎh pɩ sɛ̃́, sèpĩ̀n nínì wrɔ́, á yĩni tɛ̃ Pusɔlɩ kwil, tɛ́ wil krótókè sɔkɔ. 
Áyáh kyɩ nɛ́npɔ́, á kyɩ Yesu yõ sɛ̃́pú yísyɔ́ wɛ. Pépi mɛ á ni, á kɔ̃ á wũ à sèpĩ̀n kwǎrninɔ pɩ pé tyɩ́. Képah náh, á sɔkɔ Wrome kwil. 
Tɛ́ Wrome kwil Yesu yõ sɛ̃́pú-á fyɔ̀ á pìnɔ́ noh, pé túkù pɔ á yohnɩ, fɔ́ɔ́ Apyusi yòhntàhkɩ̀ sɔkɔ; pé túkù mó pɔ á yohnɩ fɩ̃́ yɔ́ sɔkɔ pè n ye Nɛ́pân twáh-ńtã̂n tɔ̃́nɩ̀nɛ. Pole-á fyɔ̀ pè wɛ sɛ̃́, à wɛkɩ Liyel tyɩ́, ànɛ̂ ǹ nɩ sũ. 
Képah tɛ̃̀nɛ, áyáh yĩni tɛ̃ Wrome kwil, pè ké wɛ̃̀kɩ̀lɛ Pole kɔ̃ à kyɩ tɛ̃ ǹ tyɩ́ wɔ̀kɔ̀ sɔkɔ, tɛ́ sràsyí yɔ́ kɔ̃ à ǹnɛ n kyɔmɩ. 
Képah sɔkɔ, sèpĩ̀n tɔ̃́nɩ̀-á sɔkɔ, Pole Nsyifunɔ yah tãm ye ǹ tĩ̀nnɛ. Pǎh pɔ tuke mɔ, à yo pé tyɩ́ nɛ, pé no, pé náh sénsɔ̃ tir yɔ́ ńtɛ̃ pɩ de pé no Nsyifunɔ tyɩ́, pé náh pé náh tãn wã̀l dêl pɩ, ǹtɛ pè pé tɛ̃ Yerusalɛmɩ sɔkɔ, tɛ́ pé dahbɩ Wrome tãn gbõ̀ sɔkɔ. 
Ǹtɛ Wrome tãn-á pé piki yah tɛ gbĩ́ nɛ̂nɛ, pè náh pé tyípɩ́nɩ́ yɔ́ ńtɛ̃ wɛ, nɛ̂ yĩn-á pé yɩ̃nɛ pé ko mɔ. Kǎh pɩ sɛ̃́, pè mɛ nɛ páh pélɛ n fĩn n mɔ. 
Tɛ́ Nsyifunɔ mɛ képah yahle. Képah-á, pé gbõ̀ yĩ́ĩ́ náh, pé mɛ pé gblɔ̌y tyɩ́ dahbɩ Wrome kwíltãm yõ̀tɛ̃̀ Sesar gbõ̀. Ńkɛ̃́, tɔ́kɔ́nkɔ̃nɔ tɛ̃̀ wɔ náh pé nɛ pé tɔkɔ pé no Nsyifunɔ kɔ̃ dɛ́! 
Képah yĩn-á pé kɔ̃ pé kè yah kɔ̃ nɛ, pé pè wɛ, pé mó yã́hkɩ̀ yãh pélɛ. Yisrayel tãn gbõ̀ sàhnɔ́-á mɛ pwáhnmɔ-ò nɛ̂ yõ, ǹmɔ yĩn-á kârn nɔ́pilɛ mɔ́nɔ́lɛ péwɔ gbã̀n sɔkɔ yɔ̀. 
Képah sɔkɔ, pè mɛ yo Pole tyɩ́ nɛ, sɛ́bɛ́y yɔ́ ńtɛ̃ náh wil Nsyude gbɛ̃̀nyah, à pɔ pépi tyɩ́ ǹmɔ Pole tyɩ́ sɔkɔ. Pé no Nsyifunɔ sɔkɔ yɔ́ ńtɛ̃ náh mó tɔ̃ wil nɛ́npɔ́, à pɔ tir yɔ́ yo pé tyɩ́ ǹ tyɩ́ sɔkɔ, képah náh pɩ, à ǹ yĩ́po yo pépi tyɩ́. 
Ǹtɛ pépi-á pɩ̃ nɛ, wɛ̃̀kɩ̀ nɛ̂ sɔkɔ-á ǹnɛ, pǎh mɛ kélɛ n fwo ywɔ̃ ywɔ̃ pól sɔkɔ. Képah-á, ǹmɔ sõnɔ-á nɛ̂nɛ, páh mɛ ké tyɩ́ nɛ pé kè noh ǹ vyãh sɔkɔ. 
Képah tɛ̃̀nɛ, pè mɛ wɛ̃ wɛ́nɔ́ yòh mɔ wɛ̃ tyɩ́ Polelɛ. Lésõ-á pɔ nɔ, wãn tɔ̃ mɔ pé wìlnɔ́ yõ, à pɔ Pole tyɩ́ ǹ twáhntɛ̃̀-ǹsah sɔkɔ. À mɛ tyah à Liyel tɔ̃́rɩ́ tyɩ́lɛ n yo n wah pé tyɩ́, à tɔkɔ séngbɔ̃́ tyɩ́, pópó à pɔ nɔ ńvɩ̃́ tyɩ́. À sõ̀ mɛ de Moyisi tyi kõ̀nsàhnɔ́ ànɛ̂ Liyel tɛ̃ntõ̀ syínyopu wɛlɔ sɔkɔ, à Yesu tyɩ́lɛ n yo pé tyɩ́ nɛ, pè kè fɛ̃ syi. 
Képah sɔkɔ, túkù ǹ tyi yónɔ́ fɛ̃ syi, ǹtɛ túkù mó sè yahle. 
Ǹtɛ pǎh tɛ̃ wɛkɩ sah kwlɔ núkú yõ, tɛ́ n karkɩ gbĩ́ nɛ̂nɛ, Pole tɔ̃ tyínúkú yɔ́ yo pé tyɩ́ nɛ, gbɩ-á Liyel Mirki wɛy mɛ̀nɛ, nɛ̂-á lésõ Liyel tɛ̃ntõ̀ syínyo-ò Yesayi yo pé náh tãn tyɩ́ nɛ ńkɛ̃́: 
«Kùkùtǎhkɩ̀ mɛ de no nɔ́pi nnɔ sɔkɔ. Pè mɛ pé níkílɔ́ tah tũ, tɛ́ pé yɩ̃́ vike sah; nɛ ńkɛ̃́ pé yɩ̃́ káh kwâh yɔ́ ńtɛ̃ wɛ, pé níkílɔ́ káh tir yɔ́ ńtɛ̃ noh, pé nnɔ kɔ̃ sè pɔ n yɛ kùkùtǎhkɩ̀ pɩ́nɔ́lɛ, pé mó vi mɔ ńmɔ Liyel wɛ̃̀kɩ̀ tɛ̃, ń kɔ̃ mé pè wɛ pwah mɔ. Képah ye, kyɩ n yo pé tyɩ́ nɛ, pǎh kɩ n nohnɩ tɛ̃̀ wɔlɛ, tɛ́ pè náh ké tyɩ́ yah wɛ n pi. Pǎh kɩ n yah dyěh sɛ̃́ yɔ̀, tɛ́ pè náh kwâh yɔ́ ńtɛ̃ wɛ n pi.» 
Képah yóntɛnɔ náh, Pole tɔ̃ yɔ̃ sɔkɔ yo pé tyɩ́ nɛ: «Yé pɩ̃ nɛ, Liyel-á pwáhnmɔnɔ wɛy tɛkɩ mɔ névye tɔ tyɩ́, nónó náh pɩ Nsyifunɔlɛ. Pépi tɛ̃̀ wɔ kɩ kè noh n pi.»  
Wǎh képah yo sɛ̃́, Nsyifunɔ yuku sɔkɔ, tɛ́ mɛ yékã̂blɛ n pɩ n yuku képékèyɔ̂ pé wɛ̃ wrɔ́. 
Képah náh, Pole ye nínɔ́ tũ ǹ wɔ̀kɔ̀ sɔkɔ, wǎh sõ̀ nɛ̂ tɔkɔ de tɛ̃. Nónó pól-á sõ̀ n wil n pɔ ǹ tyɩ́, à sõ̀ mɛ pé póllɛ n tɛ̃ pallɛ. 
À sõ̀ mɛ Liyel tɔ̃́rɩ́ tyɩ́lɛ n yo névye tyɩ́, tɛ́ mɛ Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyilɛ nolɛ n kwɔ nɩ̀sãh sɔkɔ. Ǹnɩ̂ yɔ́ ńtɛ̃ náh sõ̀ mɛ ǹ yahlɛ n kõ ké pɩ́nɔ́lɛ.
