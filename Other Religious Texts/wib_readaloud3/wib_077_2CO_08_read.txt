Korɛ̃nti tãn.
8.
Ń yṹnpyé, Liyel-á yõ̀bènɔrɩ nɛ̂ pɩ Masedɔnɩ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ tyɩ́, á mɛ ké tyɩ́ nɛ á kè yo yé syah. 
Yèkwɔrɩ duke náhnáh sõ̀ pè yĩni tɛ̃, ǹtɛ képah fyé yõ, nɩ̀vɩ̀nɩ̀ sõ̀ mɛ kah mɔ pé sɔkɔ. Yãm sõ̀ mɛ kah peke pé yõ, ǹtɛ képah fyé yõ, pè pɩ yɩ̃́nkɔ̃pu kègblɔlɛ. 
Ńmɔ ye pé tàrfyɔ̀lɛ nɛ pǎh kè pɩ à syɩ pé gbõ̀ yĩ̀n tyɩ́, fɔ́ɔ́ à kah mɔ pé gbõ̀ yĩ̀n ńtɛ̃ yõ à yɩ̃nɛ pé dyɔ dyɔ tɛ̃̀nɛ. 
Tɛ́ mɛ kè gbah yah kɔ̃ ápi tyɩ́ nɛ á yɛ pé gbõ̀ dahbɩ á tyɩ́ yòhnɩ̀nmɔnɔ mɛ̀ tɛ́kɩ́nmɔnɔ sɔkɔ Nsyude mara Yesu yõ sɛ̃́pú tyɩ́. 
Ápi-á sõ̀ mɛ nɛ̂nɛ n yah n kɔ̃, pè kè pɩ kah mɔ képah yõ ńtɛ̃nɛ. Pè pé gblɔ̌ynɛ Liyel ǹgbò kɔ̃, képah náh, tɛ́ pé gblɔ̌y dahbɩ ápi tɔ gbõ̀ sɔkɔ à yɩ̃nɛ Liye dyɔ yĩ́ĩ́ tɛ̃̀nɛ. 
Képah ye, á mɛ Titi ni nɛ wǎh pɔ́ mɔ́rɩ́ syínɔ́ tõ̀ nɛ̂ tyah yé tĩ̀nnɛ, à kè pɩ n dal n mɔ. 
Sɔ̃́-á yélɛ yahlɛ tyi pól sɔkɔ yɔ̀: Liyel yõ sɛ̃́nɔ́lɛ yé sɔkɔ, yé n wɛkɩ n pnɛ, yɩ̃́pɩ̃̂lɛ yé sɔkɔ, sṍrélɛ yé sɔkɔ yĩ̀n kɛ̃́nɛ, à tahlɩ névye tyɩ́nɔnɔ nɛ̂-á á dahmɔ yé sɔkɔ, képah yĩnnɛ á mɛ ké tyɩ́ nɛ yé nɛ n mɛ yahlɛ pɔ́ mɔ́rɩ́ mɔ́nɔ́ mɛ̀ tɔ sɔkɔ. 
Kyɩ̃́nmɔnɔ náh mé n kyɩ̃ n mɔ yélɛ máh képahlɛ n yo dɛ́. Ǹtɛ máh no tɛ́lɔ́ gbáhntɛ̃nɔ tyɩ́lɛ n yo yé tyɩ́, mé mɛ ké tyɩ́ nɛ yépi tɔ kè wɛ̃kɩ nɛ tyɩ́nɔnɔ kègbɩ-á mɛ yé sɔkɔ. 
Tɛ̃̀ ǹgbɛ̃ Yesu Krista-á pèpɔrɩ nɛ̂ pɩ, yé kélɛ n pnɛ. Yé náh wɛ di, ǹmɔ nɛ̂-á kwâh nɛ́tɛ̃̂nɛ, à fɛ̃ syi ǹ gblɔ̌y pɩ yãm nɛynɛ sɔ̃̀nko yépi pɩ kwâh nátãmnɛ. 
Ńmɔ sõnɔ-á nɛ̂nɛ, yĩ mɛ̀ tyɩ́ tĩ̀nnɛ, képah ye mé n yo yé tyɩ́. Képah mɛ yɩ̃́nɛ́nɔ́lɛ yépi tyɩ́ yépi nónó-á pɩ gbrolɛ yán dùkùlɛ à pɔ́ mɔ́rɩ́ mɛ̀ mɔ, tɛ́ kè náh yĩn képah ó tyɩ́ dɛ́, ké pɩ́nɔ́ yáhntɔkɔnɔ wil yépi tyɩ́. 
Kǎh pɩ sɛ̃́, núkúnúkú yé tõ̀ mɛ̀ pɩ n dal n mɔ. Yáh kè yah tɔkɔ sõnɔ pèpɛynɛ sɔ̃́, yé kè pɩ n dal n mɔ sɛ̃́ à yɩ̃nɛ yé gbõ̀ yĩ̀nnɛ. 
Yé náh wɛ di, wâh yɩ̃ kɔ̃ á gbõ̀ yĩ̀nnɛ à yɩ̃nɛ wáh kè yah tɔkɔ sɔ̃́, képah tyɩ́ nɔ Liyel tyɩ́. Kwâh nɛ̂ náh mɛ á tyɩ́, à náh képah nɩ́ynɛ n tɛ̃. 
Mé náh yo nɛ yé yé gblɔ̌y mɔ yãm sɔkɔ no tɛ́lɔ́ tyɩ́ pɛkɩ yĩnnɛ dɛ́. Ǹtɛ mé mɛ ké tyɩ́ nɛ yé kè pɩ yé pól nɛ n mɛ núkúlɛ. 
Kwɛy mɛ̀ sɔkɔ, kwâh mɛ yépi gbõ̀. Képah ye, kwâh náh mɛ nónó tyɩ́, yé pélɛ n yohnɩ n mɔ. Kwâh-à mó sõ nɛ̂nɛ mɛ pépi tyɩ́ tɛ́ kɩ n knɛ yépi tyɩ́, pè kɩ yé yohnɩ mɔ. Kàh pɩ sɛ̃́, yé pól kɩ mɛ núkúlɛ, 
sɔ̃́-á kélɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «Nɛ̂-á sõ̀ kwel dò, wãn náh wũ ǹ tyɩ́. Nɛ̂-á sõ̀ kwel kwéy, ǹ tyɩ́ náh tɛkɩ.» 
Liyel yĩn nɛ n gbilki, ǹmɔ nɛ̂-á yé yòhnɩ̀nmɔnɔ dékè dahbɩ Titi nɩ sɔkɔ, sɔ̃́-á sõ̀ à kè dahbɩ ápi nnɔ sɔkɔ. 
Áyáh nɛ̂ yah kɔ̃ ǹ tyɩ́, à kè fɛ̃ syi. Tɛ́ yé yòhnɩ̀nmɔnɔ dékè-á sõ̀ mɛ ǹ nɩ sɔkɔ sɛ̃́, à núkú kè yah tɔkɔ ǹ gblɔ̌y ńtɛ̃nɛ nɛ, páh kɩ yuku yé tĩ̀nnɛ. 
Á mɛ á yṹnpi yɔ́ tɔlɛ n tɛkɩ n mɔ ǹnɛ wɛ̃ tyɩ́, nɛ̂-á Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ pól mɛ ǹ yĩ́ngbɩ́lɛ n yo Nɩ̀vɩ̀nɩ̀ tir yónɔ́ tyɩ́ sɔkɔ. 
Tɛ́ à wil képah sɔkɔ, ǹmɔ ye tɔ̃ Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ yah wilki nɛ à ápi tɛkɩ mɔ á kyɩ pèpɔrɩ tõ̀ mɛ̀ pɩ. Tɛ̃̀ ǹgbɛ̃ yĩn gbilki yĩn ye ápi n pɩ tõ̀ mɛ̀nɛ à kélɛ n wɛ̃kɩ tɔlɛ nɛ tõ̀ mɛ̀ pɩ́nɔ́ tyɩ́-á mɛ nɔ á tyɩ́. 
Á mɛ tyi póllɛ n pɩ tɔlɛ nɛ, pɔ́ ǹgbɛ̃ nɛ̂ tyɩ́-á dahbɩ á gbõ̀, á ké tyi tɔ̃ sah pɩ pɔ tyɛ á káh yĩ́nyɩ́kɩ̀ wɛ ké tyɩ́ sɔkɔ. 
Á mɛ ké tyɩ́ nɛ á nɛ n tɔ̃ n sah n pɩ Tɛ̃̀ ǹgbɛ̃ yah sɔkɔ, tɛ́ kè náh yĩn képah ó tyɩ́ dɛ́, á mɛ ké tyɩ́ nɛ á nɛ n tɔ̃ n sah n pɩ névye tɔ yah sɔkɔ. 
Á ye á yṹnpi yɔ́ tɔlɛ n tɛkɩ mɔ pélɛ wɛ̃ tyɩ́. Á à wɛ mɔ yah gbáhyɩ̃́ náhnáhlɛ, tɛ́ sé pól sɔkɔ á kè wɛ nɛ gbáhntɛ̃-ò-á ǹnɛ. Ǹtɛ à tɔ̃ mɛ gbah tɛ̃ kal núkúnúkú, wǎh sɔ̃́npɩ́lɛ mɛ sɛ̃ yĩ́ĩ́ yépi yõ sɔkɔ. 
Titi tɛ̃̀ wɔlɛ, ńmɔ kɔ́-ò ye ǹnɛ, à mɛ tõ̀lɛ n pɩ ànɛ̂ ńmɔlɛ wɛ̃ tyɩ́ yépi yĩnnɛ. Ǹtɛ á yṹnpyé tɛ́lɔ́ wɔlɛ, Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ ye tɛkɩ mɔ pélɛ, Krista yĩn gbilki yĩnnɛ. 
Képah ye, yé kélɛ pè wɛ̃kɩ nɛ pé tyɩ́-á mɛ nɔ yĩ́ĩ́ yé tyɩ́, sɔ̃̀nko Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ kè wɛ, pè mó pɩ̃ nɛ gbɩ-á mɛ á tyɩ́ á yélɛ n sõ.
