Korɛ̃nti tãn.
11.
Yé náh wɛ di, mé mɛ ké tyɩ́ nɛ yé ń vî tyi sṍmnɔ́ pɩ yah sɛ̃́ kwéy. Ũ̀ṹ, yé ń sṍmnɔ́ pɩ. 
Mé mɛ yé gbɩ̃ mɔ sah pɛ núkú ó tyɩ́, nɛ̂-á pɩ Kristalɛ, kɩ pɔ yélɛ à kɔ̃ n pi dĩ́ tyɩ́ pɩ̃́-ńkɛ̃̂ sénpir sɔ̃́ nɛ̂-á mɛ yályál Liyel yah. Képah ye, yé tyɩ́ mɛ ń nɩlɛ n yɔrɩ sɔ̃́-á kè Liyel nɩlɛ n yɔrɩ. 
Ǹtɛ tíkí mɛ ń tɛ̃ nɛ yé sõnɔ káh pɔ vi mɔ, kè pɔ yé wrɔ́ kwrɔ́nmɔnɔ pèpɛy mɛ̀ gbɛkɩ ànɛ̂ Kristalɛ, sɔ̃́-á wahy Awa vi tɛ̃ lékã́mkãm sɔkɔ. 
Yé náh wɛ di, névi yɔ́-à pɔ Yesu káhkɩ́ yɔ́ tyɩ́lɛ n yo yé tyɩ́, nɛ̂-á mɛ ǹ gblɔ̌y áyáh nɛ̂ tyɩ́ yo yé tyɩ́, yě kè fɛ̃ n syi. Névi yɔ́-à pɔ mírkí káhkɩ́ yɔ́ tyɩ́lɛ n yo yé tyɩ́, nɛ̂-á ké gblɔ̌y Liyel Mirkilɛ yáh nɛ̂ tyɩ́ noh ápi tyɩ́, yě kè fɛ̃ n syi. Névi yɔ́-à pɔ nɩ̀vɩ̀nɩ̀ tir káhkɩ́ yɔ́ tyɩ́lɛ n yo yé tyɩ́, nɛ̂-á ké gblɔ̌y Nɩ̀vɩ̀nɩ̀ tirlɛ yáh nɛ̂ tyɩ́ noh ápi tyɩ́, yě kè fɛ̃ n syi. 
Ǹtɛ nónó-á Yesu tɛ̃ntõ̀ névye ǹgbnɔlɛ yé tyɩ́ yé yo, ńmɔ náh mó tɔ̃ mɛ fɩ̃́ yɔ́ wɛ, ńyãh-á pè fṍnfṍ ńmɔ yõ. 
Máh kɩ sɔ̃́ mɛ wɛy yah wɛ́-ńkɛ̃̂nɛ, pnɛ̀nɔ́ tɛ̃̀ wɔ mɛ ń tyɩ́ à yuku Liyel tyɩ́ tĩ̀nnɛ. Á ye képah dùkùlɛ yé wɛ̃kɩ dal mɔ tyi tyi pól sɔkɔ. 
Máh Liyel Nɩ̀vɩ̀nɩ̀ tir yo yé tyɩ́ nɔ̀nɔ́lɛ, tɛ́ ń gblɔ̌y tiki mɔ yé yõ yɔ̃ yĩnnɛ, tyípékè se pyě mé pɩ képahlɛ? 
Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ yísyɔ́ ye yĩn ńmɔ yĩ́nnɔ́lɛ ń kɔ̃ mé tɛ̃ wɛ pɔ tõ̀ pɩ yépi wrɔ́ à yé yohnɩ mɔ. 
Máh lésõ mɛ yé tĩ̀nnɛ gbĩ́ nɛ̂nɛ, yî túkù yétõ̂ lésõ mɛ ń tɛ̃, ǹtɛ mé náh sè syi ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́. Á yṹnpyé nónó-á wil Masedɔnɩ mara sɔkɔ, pépi sõ̀ mɛ ń yétõ̂ yî póllɛ ńnɛ n kɔ̃ képah ye. Tyi pól sɔkɔ, mé náh sõ̀ mɛ ké tyɩ́ nɛ mé pɩ yṍtɔ́kɔ́bɩ́lɛ ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́, tɛ́ mé kɩ tɔ̃ gbah n tɛ̃ n pi sɛ̃́. 
Gbɩ yó-ò ye ńnɛ Krista gbõ̀ yõ: ǹnɩ̂ yɔ́ ńtɛ̃ náh fɛ̃ yõ yɔ̃̀nɔ́ mɛ̀ wɛ́nɔ́ yah kõ n pi ń tyɩ́ Akayi mara pól sɔkɔ. 
Kwâh nɛ̂ pɩ tɛ́ mé képahlɛ n yo? Yé tyɩ́ náh mɛ nɔ ń tyɩ́ képah se? Liyel ńtɛ̃ pɩ̃ nɛ yé tyɩ́-á nɔ ń tyɩ́ dò. 
Máh mɛ nɛ̂nɛ n pɩ yɔ̀, mé kɩ yɔ̃ n sɔkɔ n pi ké pɩ́nɔ́lɛ, kɩ névye kùrn kah nónó-á mɛ syɔ̃̀-ǹsahlɛ n yah n kɔ̃ kɩ pé gblɔ̌y gbilki, nɛ páh pɩ núkúlɛ ápilɛ. 
Kénɛ névye nɔ́pilɛ, nósyɔ́ ye nónó-á mɛ pé gblɔ̌ynɛ n pɩ Yesu tɛ̃ntõ̀ névyelɛ. Pè n nɛ n vi n syɩkɩ névye yõ, ǹ tɛ́ pé gblɔ̌ynɛ n pɩ Yesu Krista tɛ̃ntõ̀ névyelɛ. 
Ǹtɛ képah náh mɛ gbɛ̃̀ndùnìnɛ dò. Yé náh wɛ di, sétã̀n yõ̀tɛ̃̀ ńtɛ̃ tɔ ǹ gblɔ̌ynɛ n viri n pɩ yĩ̂nyõ tõ̀npɩlɛ, nɛ̂-á mɛ kyòo dò. 
Képah ye ǹ tõ̀npɩbɩ tɔ-à pé gblɔ̌ynɛ n pɩ gbɩ Liyel tõ̀npɩbɩlɛ, kè náh mɛ gbɛ̃̀ndùnìnɛ. Pè kɩ pé tõ̀ pɩ́nɔ́ lédyáhnɩ́ wɛ n pi ké tyɛ́-ńsah sɔkɔ à yɩ̃nɛ pé tyípɩ́nɩ̀nɛ. 
Mé kɩ n syɩ kè tɔ̃ yo yé tyɩ́ nɛ, ǹnɩ̂ yɔ́ ńtɛ̃ káh ńmɔlɛ n yah vínpɩlɛ. Ǹtɛ yáh kɩ se mɛ n sõ sɛ̃́, yé ń tɛ̃ vínpɩ tyɩ́ kɔ̃lɛ, ń tɔ mó ń gblɔ̌y wɛ n gbilki kwéy kɛ̀. 
Ǹtɛ máh kɩ nɛ̂ yo n pi, mé náh mɛ kélɛ n yo Tɛ̃̀ ǹgbɛ̃-á mɛ ké tyɩ́ sɔ̃́ dɛ́, mé mɛ kélɛ n yo vínpɩ tyɩ́ kɔ̃lɛ sɔ̃̀nko mè ń gblɔ̌y wɛ n gbilki. 
Névye náhnáh-á fyɔ̀ mɛ pé gblɔ̌ynɛ n gbilki kèkõyṍ dékè tyi yĩnnɛ, ńmɔ tɔ kɩ ń gblɔ̌y gbilki. 
Yépi nónó-á lékã́m névyelɛ, kè yé dyɔ yé fɛ̃ syi à vínpɩbɩ sṍmnɔ́lɛ n pɩ. 
Yè n fɛ̃ n syi pè kɔ̃ pě yélɛ n pɩ yɔ̃bɩ tyɩ́ kɔ̃lɛ, ǹ tɛ́ nɛ n syɩkɩ yé yõ, ǹ tɛ́ yélɛ n yɩ̃, ǹ tɛ́ yélɛ n yilɔ, ǹ tɛ́ yé létáhlɔ́lɛ n kɔ̃lɩ. 
Máh nɛ̂nɛ n yo yɔ̀, kè mɛ sennɛ yé tyɩ́. Kè ye á tɛ̃́nkwlɔnɔ wɛ̃kɩ. Ǹtɛ képah fyé yõ, ǹnɩ̂-à kɩ fɛ̃ syi ǹ gblɔ̌y gbilki n pi tir yɔ́ tyɩ́ sɔkɔ, ńmɔ tɔ kɩ fɛ̃ sɛ̃́ntɛ̃̀ pɩ, máh kɩ sɔ̃́ mɛ n wɛkɩ vínpɩ tyɩ́ kɔ̃lɛ. 
Pépi-à kɩ se mɛ Yebreninɛ, Yebreke ye ńmɔ tɔlɛ. Pàh kɩ se mɛ Yisrayel nolɛ, Yisrayel nɛy ye ńmɔ tɔlɛ. Pàh kɩ se mɛ Abrahmɩ náh tãnnɛ, ǹ náh nɛy ye ńmɔ tɔlɛ. 
Pàh kɩ se mɛ Krista tõ̀npɩbɩlɛ ò, mé kɩ tɔ̃ kè yo á kɩ nɛ vínpɩ sɔ̃́ ńnɛ. Ńmɔ tõ̀ pɩ kal pé póllɛ, tɛ́ yèvyãhrɩ pɩ kal pé póllɛ, tɛ́ de wɔ̀kɔ̀npèkè sɔkɔ à kal pé póllɛ, tɛ́ ko syi kal pé póllɛ, tɛ́ kũ dlo gbal gbáhyɩ̃́ náhnáhlɛ à kal pé póllɛ. 
Nsyifunɔ ń ko yèfõh yɩ̃́ yõgbãm pɔ́ kwǎryãhlɛ, fɔ́ɔ́ gbáhyɩ̃́ kwlɔ. 
Wrome tãn n ko yèfõhlɛ gbáhyɩ̃́ tɔ̃́nɔ́. Névye ń minki dyêhlɛ kɛ núkúlɛ. Krótókè tiki de yah ńnɛ yɔ sɔkɔ kɛ tɔ̃́nɔ́, ànɛ̂ yɔ ń yõ farkɩ yah gbĩ̀yĩ̀kì núkú ànɛ̂ lékã̂h núkú wrɔ́ sɔkɔ. 
Ń kɔ́lɔ́ náhnáh sɔkɔ, mé dlo gbal yrɛy yúkúnɔ́ kàh-ǹtã̀n sɔkɔ, tɛ́ de wil névye yõ vìnsyɩkɩpu dlonɔ sɔkɔ. Mé sõ̀ mɛ dlo sɔkɔ ń yìkì névye Nsyifunɔ wrɔ́ sɔkɔ, tɛ́ mɛ dlo sɔkɔ nɛ́káhbɩ́ wrɔ́. Mé sõ̀ mɛ dlo sɔkɔ kwil nɩyṍ, tɛ́ mɛ dlo sɔkɔ gbɛ̃́nnɩ sɔkɔ. Mé sõ̀ mɛ dlo sɔkɔ pnɛ yɔ sɔkɔ, tɛ́ mɛ dlo sɔkɔ ń yṹnpyé mɩ̀nyobe wrɔ́. 
Mé tã̀nkyèh náhnáh pɩ à yèvyãhrɩ pɩ sɔkɔ. Mé yah sõ lékãhlɔ náhnáhlɛ, tɛ́ kwɛkɩ ànɛ̂ níkì sṍmnɔ́ pɩ. Yõke yah kwlɔ yah ń tyɩ́ gbáhyɩ̃́ náhnáhlɛ, mé tɛ̃ kwɛkɩ sɔkɔ. Mé tɛ̃ yah vɩrɩ sɔkɔ, tɛ́ tɛ̃ yah flɔ kɛ̃́nɛ ń tyɩ́. 
Ǹtɛ nɛ̂ ńtɛ̃ tyɩ́-á mɛ tah yĩ́ĩ́ ń tyɩ́, ń vyãh náh syɩ képah tyɩ́ dɛ́: Képah nɛ̂-á pɩ fɔ́kɔ́sõnɔ nɛ̂-á mɛ ń sɔkɔ à yuku Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ tyɩ́ tĩ̀nnɛ. 
Ǹnɩ̂-à mɛ tɛ̃́nkwlɔnɔ sɔkɔ, ńmɔ tɔ n pɩ tɛ̃̀ tyɩ́ kɔ̃lɛ tɛ̃́nkwlɔ-òlɛ. Ǹnɩ̂-à tyípékè pɩ, kě ńmɔlɛ n yɔ. 
Máh kɩ sɔ̃́ mɛ yɩ̃nɛ mè ń gblɔ̌y gbilki, mé kɩ ń gblɔ̌ynɛ n gbilki ń tɛ̃́nkwlɔnɔ tyɩ́ sɔkɔ. 
Liyel nɛ̂-á Tɛ̃̀ ǹgbɛ̃ Yesu Súlɛ, ǹmɔ pɩ̃ nɛ mɩ̀nɩ̀ náh mé n yo. Sãm nɛ mɛ mɔ́nɔ́lɛ ǹ sɔkɔ gbĩ́mɛgbĩ́. 
Máh sõ̀ mɛ Damasɩ kwil sɔkɔ gbĩ́ nɛ̂nɛ, yõ̀tɛ̃̀ dĩ́ Aretasɩ-á kwíltɛ̃̂ nɛ̂ wilki sah nɛ́npɔ́, ǹmɔ sõ̀ mɛ pè kɔ̃ pè kwil dé-ńtã̂nnɛ n syɩkɩ ńmɔ tɛ̃ yĩnnɛ. 
Ǹtɛ pè mɛ ń dah mɔ gbétɔ́ yɔ́ sɔkɔ à wilki mɔ sétír pwã̀hlɛ à tiki mɔ kwil kɛ̃́nkahnɩnɛ ń kɔ̃ mé wɛ pwah ǹ gbõ̀.
