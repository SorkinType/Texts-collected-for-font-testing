1 Nsyɔ̃.
4.
Ń nɛ́nɔ́nɔ́, névye nónó-á nɛ Liyel Mirki gbõ̀ yõ-á pé n yo tyilɛ, yé káh n sɛ̃ pé pól yõ dɛ́. Yé pè sohlɩ n yah, yè ké gbɩ pɩ̃ Liyel Mirki gbõ̀ yõ-à kɩ nɛ̂ ńtyɩ̃́nɔ pè n yo tyilɛ. Yé náh wɛ di, Liyel yĩn yõ mɩ̀nyobe mɛ tɔ̃ mɔ pí kèkõyṍ. 
Liyel Mirki-á n tɛ̃ nónó yahlɛ, yáh nɛ pè pɩ̃ n pi dùkù nɛ̂nɛ, kè wɛ yɔ̀: Yâh névi nɛ̂ wɛ à mɛ n yo nɛ Yesu Krista-á pɩ névilɛ, Liyel Mirki ye n tɛ̃ tɛ̃̀ yahlɛ. 
Ǹtɛ nɛ̂-à nɛ sɛ̃́ náh, Liyel Mirki náh n tɛ̃ tɛ̃̀ wɔ yahlɛ. Krista sépɛ mírkí ye n tɛ̃ tɛ̃̀ yahlɛ. Yé kè noh sah tɛ nɛ, Krista sépɛ-á kɩ n pɔ n pi. Tɛ́ núkúnúkú, à mɛ tɛ kèkõyṍ ńtɛ̃nɛ. 
Yépi wɔlɛ ń wɛ̃́npì, Liyel tyɩ́ névye ye yélɛ. Ǹ Mirki nɛ̂-á n tɛ̃ yé yahlɛ, képah tǎhkɩ̀ kal sétã̀n yõ̀tɛ̃̀ mírkí tyɩ́lɛ, képah nɛ̂-á kèkõyṍ névye yahlɛ n tɛ̃. Képah ye, yé tɛ̃́nwɛnɔ wɛ Liyel yĩn yõ mɩ̀nyobe yõ. 
Pépilɛ, kèkõyṍ tyɩ́ ye pélɛ. Képah ye, pě pé tyilɛ n yo à yɩ̃nɛ kèkõyṍ névye tyi yo dùkùlɛ, tɛ́ kèkõyṍ névye n mó pé nínɛ n kwɔ pé tyɩ́. 
Ǹtɛ ápi mɛ Liyel tyɩ́ yõ. Névi nɛ̂-à Liyellɛ n pnɛ, tɛ̃̌ ǹ níkílɛ n kwɔ ápi tyɩ́. Tɛ́ névi nɛ̂-à mɛ́-ńkɛ̃̂nɛ Liyel tyɩ́ yõ, tɛ̃̀ náh ǹ níkílɛ n kwɔ ápi tyɩ́. Sɛ̃́ntɛ̃̀ sɔkɔ ye Liyel Mirki nɛ̂-á névyelɛ n mɔ gbɩ wɛ̃̀kɩ̀ yõ, à n tɛ̃ n wɛ ǹnɛ n wah sétã̀n yõ̀tɛ̃̀ tyɩ́ sɔkɔ, ǹmɔ nɛ̂ tyɩ́-á névyelɛ n pilki. 
Tyɩ́nɔnɔ n wil Liyel tyɩ́. Tyɩ́nɔnɔ-à mɛ névi nɛ̂ sɔkɔ, Liyel pi ye tɛ̃̀nɛ, ànɛ̂ tɛ̃̀ Liyellɛ n pnɛ. Képah ye ń nɛ́nɔ́nɔ́, á ye yɛ á tyɩ́ nɔ wɛ̃ tyɩ́. 
Liyellɛ, tyɩ́nɔnɔ Liyel ye ǹnɛ. Képah ye, á náh fɛ̃ n yo n pi nɛ wáh ǹnɛ n pnɛ tyɩ́nɔnɔ kɛ̃́nɛ á sɔkɔ. 
Yé yah, Liyel-á kè wɛ̃kɩ dùkù nɛ̂nɛ nɛ á tyɩ́-á nɔ pé tyɩ́, kè wɛ yɔ̀: Pi núkúní nɛ̂-á mɛ ǹ tyɩ́, à à tɛkɩ mɔ kèkõyṍ, ápi kɔ̃ á kɩ min wɛ ǹmɔ gbõ̀ yõ. 
Tyɩ́nɔnɔ-á nɛ̂nɛ, kè wɛ yɔ̀: Liyel tyɩ́-á nɔ ápi tyɩ́ dùkù nɛ̂nɛ, képah náh tyɩ́nɔnɔ kègbɩlɛ dɛ́. Ápi tyɩ́-á nɔ Liyel tyɩ́ nɔ dùkù nɛ̂nɛ, képah ye tyɩ́nɔnɔ kègbɩlɛ. Yé náh wɛ à ǹ Pídĩ́ tɛkɩ mɔ à pɔ pɩ sárkálɛ ápi tyípêl yɔ̃ n mɔ yĩnnɛ di? 
Ń nɛ́nɔ́nɔ́, ápi tyɩ́-á nɔ Liyel tyɩ́ sɛ̃́ à, ápi tɔ yɩ̃nɛ á tyɩ́ nɔ wɛ̃ tyɩ́ mó kɛ̀. 
Ǹnɩ̂ yɔ́ ńtɛ̃ yĩ́ náh tɛ̃ yah Liyel yõ. Ǹtɛ ápi tyɩ́-à mɛ nɔ wɛ̃ tyɩ́, á mɛ ǹ tyɩ́ yõ, ànɛ̂ ǹ tyɩ́nɔnɔ mɛ ápi sɔkɔ yèsyɩ̀kɩ̀ tɛ̃̀nɛ. 
Liyel-á ǹ Mirkilɛ á kɔ̃, képah ye á kɔ̃ á pɩ̃ nɛ áyáh mɛ ǹ tyɩ́ yõ, ànɛ̂ wǎh mɛ álɛ. 
Sú Liyel ye ǹ Pídĩ́ tɛkɩ mɔ kèkõyṍ névye pwah n mɔ yĩnnɛ. Ápi yĩ́ ye kè wɛ, képah ye á kɔ̃ á kélɛ n yo n wah. 
Névi nɛ̂ tɛ̃̀-à fyɔ̀ yo nɛ ńkɛ̃́ Yesu-á Liyel Pídĩ́nɛ, Liyel mɛ tɛ̃̀nɛ, ànɛ̂ tɛ̃̀ mɛ Liyel tyɩ́ yõ. 
Ápi tyɩ́-á nɔ Liyel tyɩ́ dùkù nɛ̂nɛ, ápi tɛ̃̀ wɔ ye kélɛ n pnɛ, ànɛ̂ á sɛ̃ ké yõ. Liyellɛ, tyɩ́nɔnɔ Liyel ye ǹnɛ. Tyɩ́nɔnɔ-à mɛ névi nɛ̂ sɔkɔ, tɛ̃̀ wɔ mɛ Liyel tyɩ́ yõ, ànɛ̂ Liyel mɛ tɛ̃̀nɛ. 
Tyɩ́nɔnɔ-à mɛ yĩ́ĩ́ ápi sɔkɔ, á mɛ á minnɛ n pɩ kèkõyṍ Yesu Krista tyɩ́ kɔ̃lɛ. Képah sɔkɔ, tíkí náh á tɛ̃ n pi Liyel tukey tuke sõlɛ. 
Yé náh wɛ di, Liyel tyɩ́nɔnɔ-à mɛ névi nɛ̂ sɔkɔ, tíkí náh tɛ̃̀nɛ n tɛ̃. Tɛ́ wâh mɛ n sõ nɛ á tyípêl wɔ-á kɩ pɔ n dɛ n pi á sɔkɔ, képah ye n pɔ n pɔ tíkílɛ. Ǹtɛ tyɩ́nɔnɔ kègbɩ wɔ tíkí yahlɛ n kõ. Tíkí tɛ̃̀-à fyɔ̀ mɛ névi nɛ̂nɛ n tɛ̃, tyɩ́nɔnɔ náh mɛ yĩ́ĩ́ tɛ̃̀ sɔkɔ. 
Liyel ǹgbò ye ápi tyɩ́ nɔ ǹ gblɔ̌y tyɩ́, képah ye tyɩ́nɔnɔlɛ ápi tɔ sɔkɔ. 
Névi nɛ̂-à nɛ Liyel tyɩ́-á nɔ pé tyɩ́, tɛ́ mó tɔ̃ mɛ ǹ yṹnpi yɔ́ dyɔlɛ n yɔ, mɩ̀nyo ye tɛ̃̀nɛ. Tɛ̃̀ yĩ́-á mɛ ǹ yṹnpi nɛ̂ yõ, ǹmɔ tyɩ́-à nɔ́-ńkɛ̃̂nɛ ǹ tyɩ́, sɔ̃́ mɛ Liyel tyɩ́ kɩ pɔ n nɔ ǹ tyɩ́, ǹmɔ nɛ̂-á à náh à wɛ yah? 
Tɛ́ Krista-á nɛ̂ kõ sah á kɔ̃, kè wɛ yɔ̀ nɛ ńkɛ̃́, Liyel tyɩ́-à mɛ nɔ névi nɛ̂ tyɩ́, tɛ̃̀ yṹnpi tɔ tyɩ́-á yɩ̃nɛ kè nɔ ǹ tyɩ́.
