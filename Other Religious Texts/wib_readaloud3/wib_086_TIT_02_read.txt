Titi.
2.
Ǹtɛ áwɔ tɛ̃̀ wɔlɛ, ápi sɛ̃́nɔ́-á mɛ nɛ̂ yõ, képahlɛ névyelɛ n kwɔ. 
Yo yèwàhnɩ̀ tyɩ́ nɛ pè pé gblɔ̌ynɛ n tɛ̃ n pnɛ, pè yɛ yõ̀yén nɛ n sah pé yõ, pè pɩ yèsrey névyelɛ, pè nɛ mɛ sũ yĩ́ĩ́ Liyel yõ sɛ̃́nɔ́ sɔkɔ, tyɩ́nɔnɔ mó nɛ n mɛ pé sɔkɔ, pè mó nɛ mɛ sṍmnɔ́ névyelɛ. 
Yo syɔ́dyî tɔ tyɩ́ nɛ pè yɛ pé kɔ dùkù nɛ mɛ pallɛ Liyel yah, pè káh névye yĩnnɛ n yɩkɩ, pè káh pɩ sèwɔbɩlɛ, pè névyelɛ n vi n mɔ pallɛ. 
Syɔ́ nónó náh wah mɔ, pǎh yɩ̃nɛ pè pépilɛ n yohnɩ n mɔ pé pɛbɩ ànɛ̂ pé wɛ̃́npì tyɩ́ kɔ̃ kè nɔ pé tyɩ́. 
Pǎh yɩ̃nɛ pè pɩ yèsrey névyelɛ nónó-á mɛ tómm, tɛ́ mɛ n yah n dah mɔ pé kétlo tyɩ́ sɔkɔ. Pǎh yɩ̃nɛ pè nɛ n mɛ syɔ́gblɔ́lɛ nónó-á mɛ n sah pé pɛbɩ tyɩ́ yõ, sɔ̃̀nko Liyel wɛy tyɩ́ káh yo ko. 
Sṍré mɔ kètɛ̃̀npì tɔ sɔkɔ pè pɩ yèsrey névyelɛ 
tyi pól sɔkɔ. Áwɔ tɛ̃̀ wɔ mó pɩ dùkù pèpɛynɛ pélɛ n wɛ̃kɩ á tyípéplɔ pɩ́nɔ́ sɔkɔ, ǹ tɛ́ á fɔkɔ dal mɔ pélɛ n kwɔ yèsyɩ̀kɩ̀lɛ. 
Yɛ á wɛlɔ nɛ n mɛ gblɔ ó sé tɔ̂ɔ́, wɛ́nyo-ńsah kɛ̃́nɛ. Képah sɔkɔ, á sépɛ́bɩ́ náh wɛ́nyo-ńsah wɛ n pi ápi tyɩ́. Kè kɩ pɩ sennɛ pé pól tyɩ́. 
Yo yɔ̃bɩ tyɩ́ nɛ pè nɛ n sah pé yõ̀tãm tyɩ́ yõ tyi tyi pól sɔkɔ, pè pɩ nɛ́gblɔ́lɛ pé tyɩ́, pè káh pé yõ̀tãmnɛ n kɔ̃lɩ, 
pè káh kwâh yɔ́ ńtɛ̃ yɩ̃ n tɔkɔ pé tyɩ́. Pè pé Liyel yõ sɛ̃́nɔ́ pèpɛynɛ n wɛ̃kɩ gbĩ́mɛgbĩ́, sɔ̃̀nko á pwáhnmɔ-ò Liyel-á kwɔ́nɔ́ nɛ̂nɛ n kɔ̃, gbílkínɔ́ nɛ n sah ké yõ. 
Yé náh wɛ di, Liyel ye ǹ yõ̀bènɔrɩ wɛ̃kɩ képah nɛ̂ gbõ̀ yõ-á névye pól pwáhnmɔnɔlɛ n wɛ. 
Kè álɛ n kwɔ nɛ á Liyel yah tíkílɛ n pɩ, ànɛ̂ á káh á yĩ́nɛ n wil kèkõyṍ yî tyɩ́. Kàh pɩ sɛ̃́, kwɛy gbĩ mɛ̀ sɔkɔ á kɩ tɛ̃ wɛ á minnɛ n pɩ gblɔ̌y tɛ̃́nɔ́ sɔkɔ, ànɛ̂ gbɩ wɛ̃̀kɩ̀ yõ, ànɛ̂ Liyel yah tíkí pɩ́nɔ́ sɔkɔ. 
Sɛ̃́ ye á yɩ̃nɛ à nɩ̀vɩ̀nɩ̀ wɛ́nɔ́ mɛ̀ syɩkɩ tɛ̃, á gbõ̀ sàhnɔ́-á mɛ nɛ̂ yõ, à tahlɩ gbĩ̀yĩ̀kì tyɩ́, sõ ye ápi Tɛ̃̀ ǹgbɛ̃ Liyel nɛ̂-á pwáhnmɔ-ò Yesu Kristalɛ, ǹ tɔ̃́rɩ́ kɩ n wilki n mɔ n pi. 
À ǹ gblɔ̌y kɔ̃ ápi yĩnnɛ à á yõ wɛ dwe wilki tyípêl pól sɔkɔ. À mɛ ké tyɩ́ nɛ pé á pɩ yályál pé yah, á mó nɛ mɛ pé tyɩ́ yõ, á mó gbah tɛ̃ tyípéplɔ pɩ́nɔ́ sɔkɔ. 
Sɛ̃́ ye á yɩ̃nɛ ǹ nɛ n wɛkɩ yõ̀tɔ̃rɩ́ sɔkɔ, ǹ sṍrélɛ n mɔ pé sɔkɔ, ǹ tɛ́ nɛ n wɛkɩ pé yõ. Káh n yɛ ǹnɩ̂ yɔ́ ńtɛ̃ álɛ n yilɔ.
