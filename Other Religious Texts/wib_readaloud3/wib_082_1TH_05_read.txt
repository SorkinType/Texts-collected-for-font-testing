1 Tesaloniki tãn.
5.
Á yṹnpyé, tyi nɔ́pi-á kɩ pɔ n pɩ n pi gbĩ́ nɛ̂nɛ, yõ náh mɛ ké tyɩ́ mè n tɔ̃ tir yɔ́ wãrkɩ n tɛkɩ n mɔ yé tyɩ́ sépi tyɩ́ sɔkɔ. 
Yépi ńtɛ̃ pɩ̃́nɔ́ mɛ ké sɔkɔ nɛ Tɛ̃̀ ǹgbɛ̃ syɩ n pɔ sõ-á kɩ névye kwɔkɩ n tɛ̃ n pi yɛ̃́nwó pɔ́nɔ́ tyɩ́ kɔ̃lɛ lékã̂hnɛ. 
Gbĩ́ nɛ̂-á névye kɩ mɛ n yo nɛ, yèvɩnɩ-á mɛ, dlo náh tɔ̃ mɛ, légbĩ́ ye tyɩ́ yɩ́kɩ́nɔ́ kɩ núkú pè kwɔkɩ tɛ̃ sê kúrkì tɛ̃́nɔ́ tyɩ́ kɔ̃lɛ, tɛ́ pè náh tɛ̃ wɛ n wil n pi ké sɔkɔ póllɛ. 
Ǹtɛ yépi wɔlɛ á yṹnpyé, yépi tɔ se yɔ́ mɛ fɛ̃́ntũ̂ sɔkɔ kénɛ gbĩ̀yĩ̀kì kɔ̃ kè kɩ pɔ yé kwɔkɩ n tɛ̃ n pi yɛ̃́nwó tyɩ́ kɔ̃lɛ? 
Yé náh wɛ di, yépi póllɛ, yé fɔkɔ mɛ dálnɔ́lɛ. Yé mɛ tyilɛ n pɩ kyòo sɔkɔ. Ápi tyípɩ́nɩ̀ náh mɛ fɛ̃́ntũ̂ sɔkɔ. 
Képah ye, á yé káh nɛ n dɔ no tɛ́lɔ́ tyɩ́ kɔ̃lɛ. Á ye nɛ mɛ n yah, à mó á gblɔ̌ynɛ n kyɔmɩ sètɛ̃̀nɩ̀ pɩ́nɔ́lɛ. 
Yé náh wɛ di, lékã̂h yé dɔy dɔ́pú n nɛ n dɔ, ànɛ̂ lékã̂h ye sèwɔbɩ tɔ sètɛ̃̀nɩ̀nɛ n pɩ. 
Ǹtɛ ápi nónó-á pɩ tyah kyòo sɔkɔ névyelɛ, á ye káh sètɛ̃̀nɩ̀nɛ n pɩ. Á ye yɛ Liyel yõ sɛ̃́nɔ́ ànɛ̂ tyɩ́nɔnɔ nɛ n mɛ á sɔkɔ á kɩ nɛ vyɔrɩ fwɔ̀mɔnɩ sɔ̃́ á mɔ. Á gbõ̀ sàhnɔ́ mó nɛ mɛ pwáhnmɔnɔ yõ á kɩ nɛ vyɔrɩ gbétú sɔ̃́ á kwĩnki. 
Yé náh wɛ di, Liyel náh ápi mɔ nɛ pé pɔ á fõh dɛ́. Ǹtɛ à mɛ ké tyɩ́ nɛ á pwáhnmɔnɔ wɛ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista gbõ̀ yõ. 
Ǹmɔ ku ápi yĩnnɛ, sɔ̃̀nko áyâh sɔ̃́ mɛ min sɔkɔ ò, áyâh sɔ̃́ ku ò, à nɛ n mɛ ǹmɔlɛ wɛ̃ tyɩ́. 
Képah ye, yé sṍrélɛ n mɔ wɛ̃ sɔkɔ, yè wãnnɛ n mɔ wɛ̃ tǎhkɩ̀ yõ Liyel yõ sɛ̃́nɔ́ sɔkɔ, sɔ̃́-á yé mɛ kélɛ n pɩ yɔ̀. 
Á yṹnpyé, á kélɛ n yah n kɔ̃ yé tyɩ́ nɛ nónó-á mɛ gbah tɛ̃ yé yahlɛ n tɛ̃, tɛ́ mɛ yélɛ n vi mɔ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ, yé yõ̀yénnɛ n sah pé yõ. 
Yé yõ̀yénnɛ n sah n yĩ́ĩ́ pé yõ tyɩ́nɔnɔ sɔkɔ pé tõ̀ yĩnnɛ. Yé nɛ n mɛ tnɔ sɔkɔ wɛ̃ tyɩ́ yèvɩnɩ sɔkɔ. 
Á yṹnpyé, á kélɛ n yah n kɔ̃ yé tyɩ́ nɛ, yé nɛ n wɛkɩ gbãndal pɩ́pú yõ, yé nɩ̀sãhlɛ n mɔ tíkí pɩ́pú sɔkɔ, yé mó tɛ̃́nkwlɔpulɛ n yohnɩ n mɔ. Yé no pól sṍmnɔ́lɛ n pɩ. 
Yé yé gblɔ̌ynɛ n kyɔmɩ, ǹnɩ̂ yɔ́ ńtɛ̃ káh pèkè wɔ́lɛ n dɛ pèkèlɛ. Ǹtɛ yé pèpɛy ó pɩ́-ńsahlɛ n yah n kɔ̃ wɛ̃ tyɩ́, ànɛ̂ no pól tyɩ́ gbĩ́mɛgbĩ́. 
Yé nɛ n mɛ nɩ̀vɩ̀nɩ̀ sɔkɔ gbĩ́mɛgbĩ́. 
Yé Liyellɛ n ni kṹmm. 
Tyi tyi pól sɔkɔ yé nɛ n wɛkɩ Liyel tyɩ́. Képah ye Liyel n yah n kɔ̃ yé tyɩ́ nɛ yé nɛ n pɩ Yesu Krista yõ sɛ̃́nɔ́ sɔkɔ. 
Yé káh Liyel Mirki yah kõ tõ̀ pɩ́nɔ́lɛ, 
yé káh Liyel tɛ̃ntõ̀ syínyopu wɛlɔ tyɩ́lɛ n yo n ko. 
Yé tyi póllɛ n sohlɩ n yah, tɛ́ nónó-à mɛ pallɛ, yè mó sépi fɛ̃ n syi. 
Yé yé gblɔ̌ynɛ n tɛ̃ tyípêl duke pól pɩ́nɔ́lɛ. 
Liyel nɛ̂-á yèvɩnɩ nɛ́tɛ̃̂nɛ, mé kélɛ n yah n kɔ̃ ǹ tyɩ́ nɛ à sãm mɔ yé min pɩ dùkù pól sɔkɔ. À mó yé sõnɔ, ànɛ̂ yé min, ànɛ̂ yé wil gbɛ̃́nsê pɩ wɛ́nyo-ńsah kɛ̃́nɛ sé tyɩ́ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista syɩ n pɔ sõlɛ. 
Ǹmɔ nɛ̂-á n ye yélɛ, à n kɔ ǹ tyíyónɔ́ sõ̀. Ǹmɔ ye kɩ pɩ sélɛ. 
Á yṹnpyé, yé nírí tɔlɛ n pɩ ápi tyɩ́. 
Yé wɛkɩ á yṹnpyé pól tyɩ́ tyɩ́nɔnɔ wɛy pèpɛynɛ. 
Mé yélɛ n gbah n ni Tɛ̃̀ ǹgbɛ̃ yĩn yõ, nɛ yé sɛ́bɛ́y mɛ̀ kar á yṹnpyé pól noh. 
Mé kélɛ n yah n kɔ̃ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́ nɛ à yõ̀bènɔrɩ pɩ yé tyɩ́.
