Wrome tãn.
16.
Mé á dósê Febelɛ n dahbɩ yé gbõ̀, ǹmɔ nɛ̂-á yòhnɩ̀nmɔ-òlɛ Sãnkrɩ kwil Yesu yõ sɛ̃́pú tĩ̀nkì sɔkɔ. 
Yé ǹ tɛ̃́nɔ́ plɛ Tɛ̃̀ ǹgbɛ̃ yĩn yõ, ǹ yõ sɛ̃́pú-á wɛ̃nɛ n tɛ̃ sɔ̃́. Yè mó ǹnɛ n yohnɩ n mɔ tyi tyi pól sɔkɔ yé yétõ̂-à syɩ ǹ tyɩ́. Yé náh wɛ di, à ye névye náhnáh yohnɩ mɔ sɔkɔ, tɛ́ ńmɔ tɔ yohnɩ mɔ. 
Yé wɛkɩ Prisili ànɛ̂ Akilasɩ tyɩ́, pépi nónó-á ńmɔ tõ̀ pɩ mûnɛ Yesu Krista tyɩ́. 
Pè ye ku farkɩ ńmɔ min pwáhnmɔ-ńsah. Ńmɔ ó náh n kɔ̃ pélɛ pèpɔrɩ pɩ̃́nɔ́ wɛynɛ dɛ́, Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ pól nónó-á pè náh pɩ Nsyifunɔlɛ, pépi tɔ mɛ pé pèpɔrɩlɛ n pnɛ. 
Yé wɛkɩ Yesu yõ sɛ̃́pú tĩ̀nkì tɔ tyɩ́, nónó-á wɛ̃nɛ n tuke n mɔ pé gbô sɔkɔ. Yé wɛkɩ ń nɛ́nɔ́nɩ́ Yepayinɛtɩ tyɩ́, ǹmɔ nɛ̂-á pɩ nɛ́gbennɛ à sɛ̃ Krista yõ Asi mara sɔkɔ. 
Yé wɛkɩ Mari tyɩ́, ǹmɔ nɛ̂-á tõ̀ kah pɩ yé tyɩ́. 
Yé wɛkɩ Andronikusi ànɛ̂ Nsyunyasɩ tyɩ́, pépi nónó-á ńmɔ tyɩ́ névyelɛ tɛ́ á wɔ̀kɔ̀npèkè pɩ wɛ̃ tyɩ́ pélɛ. Yõ̀yén mɛ n sah n yĩ́ĩ́ pé yõ pé no Yesu tɛ̃ntõ̀ névye sɔkɔ. Pépi ńtɛ̃ pɩ Yesu yõ sɛ̃́púlɛ ńmɔ yah sɔkɔ. 
Yé wɛkɩ Anpiliyatusi tyɩ́ ǹmɔ nɛ̂ tyɩ́-á tah yĩ́ĩ́ ń tyɩ́ Tɛ̃̀ ǹgbɛ̃ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ. 
Yé wɛkɩ Yuribɛ̃ tyɩ́, ǹmɔ nɛ̂-á á tõ̀ pɩ mûnɛ Krista tyɩ́. Yè mó wɛkɩ ń nɛ́nɔ́nɩ́ Syitakisi tɔ tyɩ́. 
Yé wɛkɩ Apɛlɛsɩ tyɩ́, ǹmɔ nɛ̂-á ǹ yĩ́n-ńsah fílkì viki Krista yõ sɛ̃́nɔ́ sɔkɔ. Yé wɛkɩ Arisitobuli gbô tãn tyɩ́. 
Yé wɛkɩ Yorodyõ tyɩ́, ǹmɔ nɛ̂-á ń tyɩ́ névilɛ. Yé wɛkɩ Narsisi gbô tãn tyɩ́, nónó-á sɛ̃ Tɛ̃̀ ǹgbɛ̃ yõ. 
Yé wɛkɩ Tirifɛnɩ ànɛ̂ Tirifɔsɩ tyɩ́, pépi nónó-á tõ̀ pɩ Tɛ̃̀ ǹgbɛ̃ tyɩ́. Yè mó wɛkɩ ń kósê Pɛrsidi tyɩ́, ǹmɔ nɛ̂ yi-á tõ̀ kah pɩ Tɛ̃̀ ǹgbɛ̃ tyɩ́. 
Yé wɛkɩ Wrufusi tyɩ́, ǹmɔ nɛ̂ yĩn-á mɛ wìlnɔ́lɛ Tɛ̃̀ ǹgbɛ̃ tõ̀ pɩ́nɔ́ sɔkɔ. Yè mó wɛkɩ ǹ yṹ tyɩ́, ǹmɔ nɛ̂ tɔ-á pɩ ńmɔ yṹnɛ. 
Yé wɛkɩ Asɛ̃kriti tyɩ́, ànɛ̂ Flekõlɛ, ànɛ̂ Yɛrɛmɛsɩlɛ, ànɛ̂ Patrobaselɛ, ànɛ̂ Yɛrɛmasɩlɛ, à tahlɩ á yṹnpyé nónó-á mɛ pélɛ wɛ̃ tyɩ́. 
Yé wɛkɩ Filolɔkɩ ànɛ̂ Nsyuli tyɩ́, ànɛ̂ Nere, ànɛ̂ ǹ dósêlɛ, à tahlɩ Wolɛ̃pasɩ ànɛ̂ Yesu yõ sɛ̃́pú pól tyɩ́, nónó-á mɛ pélɛ wɛ̃ tyɩ́. 
Yé wɛkɩ wɛ̃ tyɩ́ tyɩ́nɔnɔ sɔkɔ. Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ pól n wɛkɩ yé tyɩ́. 
Ń yṹnpyé, mé kélɛ n yah n kɔ̃ yé tyɩ́ nɛ kwɔ́nɔ́ nónó-á yé wɛ sɔkɔ, névye nónó-á mɛ yuku sépilɛ n fwo, tɛ́ gbɛ́kɩ́nɔ́lɛ n dah n mɔ yé wrɔ́, yé yé gblɔ̌ynɛ n tɛ̃ pélɛ. Yé káh pélɛ n tĩnki. 
Kénɛ névye nɔ́pi dùkùlɛ kɛ̀; yé káh wɛ nɛ ápi Tɛ̃̀ ǹgbɛ̃ Krista tyɩ́-á pè n pɩ tõ̀lɛ dɛ́, pé kèkùrn ye pè n yɩ̃kɩ. Vyã́hsesyɩ̃́rɩ́ sɔkɔ, pě ǹpɩ̃̀-ǹkɛ̃̂ névye yõ yɩkɩ pé yĩ́nvǐ wɛlɔlɛ. 
Yáh mɛ n sah Tɛ̃̀ ǹgbɛ̃ tyɩ́ yõ dùkù nɛ̂nɛ, no pól mɛ kélɛ n pnɛ. Képah ye ń nɩ mɛ vɩ́nɔ́lɛ yélɛ náhnáh. Ǹtɛ mé mɛ ké tyɩ́ nɛ yé lékã́m wɛ yé tyípéplɔlɛ n pɩ. Yé nɛ n mɛ yályál, yé káh tyítúkúnìnɛ n pɩ. 
Liyel nɛ̂-á yèvɩnɩ nɛ́tɛ̃̂nɛ, dǒ kwéy, à kɩ sétã̀n yõ̀tɛ̃̀ tyɩ́ yɩkɩ pɩ yé gbáhkɩ̀ súnsàh-ǹsahlɛ. Mé kélɛ n yah n kɔ̃ ápi Tɛ̃̀ ǹgbɛ̃ Yesu tyɩ́ nɛ à yõ̀bènɔrɩ pɩ yé tyɩ́. 
Timote nɛ̂-á ń tõ̀ pɩ mûnɛ, à n wɛkɩ yé tyɩ́, à tahlɩ Lusiyusi, ànɛ̂ Nsyasõ, ànɛ̂ Sosipatɛrɩ tyɩ́, pépi nónó-á ń tyɩ́ névyelɛ. 
Ńmɔ Tɛrityusi nɛ̂-á pè ń kɔ̃ mé sɛ́bɛ́y mɛ̀ wãrkɩ, ń tɔ n wɛkɩ yé tyɩ́ Tɛ̃̀ ǹgbɛ̃ yĩn yõ. 
Kayusi nɛ̂ tyɩ́-á ńmɔlɛ twáhntɛ̃̀nɔ́lɛ, tɛ́ Yesu yõ sɛ̃́pú tĩ̀nkì wɛ̃nɛ n tuke n mɔ n pɔ ǹ tĩ̀nnɛ, à n wɛkɩ yé tyɩ́. Kwil pɔ́ màr-ò dĩ́ Yerasɩtɩ ànɛ̂ á yṹnpi Kwaritusi n wɛkɩ yé tyɩ́. 
Mé kélɛ n yah n kɔ̃ ápi Tɛ̃̀ ǹgbɛ̃ Yesu tyɩ́ nɛ à yõ̀bènɔrɩ pɩ yé pól tyɩ́. Amiina! 
Gbílkínɔ́ nɛ n sah Liyel yõ. Tɛ̃́nwɛnɔ mɛ ǹ tyɩ́ kɩ fɛ̃ tǎhkɩ̀lɛ yé kɔ̃ à yɩ̃nɛ Nɩ̀vɩ̀nɩ̀ tir nɛ̂nɛ mé n yo yé tyɩ́ Krista tyɩ́ sɔkɔ. Sètĩ̀n sõ̀ tir ye lésõ kélɛ ǹgbò dùkùlɛ, 
tɛ́ pɔ wilki mɔ núkúnúkú kwèyy kyòo yah, Liyel tɛ̃ntõ̀ syínyopu tyi wã́rkɩ́nsàhnɔ́ gbõ̀ yõ. Tyɛ́-ńkɛ̃̂ Liyel ye lésõ pè kɔ̃ pè kè wãrkɩ sah sɔ̃̀nko yìkìnì pól névye kè wɛ n pɩ̃. Képah sɔkɔ, pè kɩ sɛ̃ Liyel yõ tɛ́ kɩ n sah ǹ tyɩ́ yõ. 
Liyel núkúní nɛ̂-á lékã́m ó ǹ tɔ̂ɔ́, gbílkínɔ́ nɛ n sah ǹmɔ yõ gbĩ́mɛgbĩ́ Yesu Krista gbõ̀ yõ. Amiina!
