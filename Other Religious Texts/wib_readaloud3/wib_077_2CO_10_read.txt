Korɛ̃nti tãn.
10.
Ńmɔ Polelɛ, pè nɛ mâh mɛ yé tnɔ̂ mè n nɛ n wɛkɩ dáb dáb, tɛ́ mâh gbɛ mɔ yélɛ mè n nɛ n wɛkɩ fyɔ̌rɩ̀ fyɔ̌rɩ̀. Yesu Krista nɛ́pír pɩ́nɔ́ ànɛ̂ ǹ nɩ̀gbɛ̀kɩ̀ yĩn yõ, 
ńmɔ Pole kélɛ n yah n kɔ̃ yé tyɩ́ nɛ, yé yé gblɔ̌y yɛ, máh kɩ pɔ pɔ yé tnɔ̂, yé káh ń kɔ̃ mè pɔ nɛ n wɛkɩ fyɔ̌rɩ̀ fyɔ̌rɩ̀ yépi tɔ tyɩ́, sɔ̃́ mé mɛ kè sõ sah pɩ pɩ tɛ̃̀nɛ névye tyɩ́ nónó-á mɛ n sõ nɛ névi yõ̀tàhsyi pɩ dùkù-á ápi sɔkɔ. 
Gbɩ ye névye tɛ̃̀ wɔ ye álɛ ó, ǹtɛ áyáh mɛ́kɩ̀ nɛ̂ sɔkɔ yɔ̀, névi yõ̀tàhsyi tǎhkɩ̀ náh á n fwo kélɛ. 
Ápi-á mɛ́kɩ̀lɛ n pɩ mɛ́fwóyî nónónɛ, névi gbɛ̃̀npɩnɔ náh sélɛ, sè n wil Liyel tyɩ́. Tǎhkɩ̀ mɛ sé sɔkɔ kɩ fɛ̃ Liyel sépɛ́bɩ́ tyɩ́ yɩkɩ ànɛ̂ sõnɔ túkúnìnɛ. 
Kénɛ mɛ́fwóyî gbõ̀ yõ, ápi mɛ gblɔ̌y gbílkípú tǎhkɩ̀lɛ n yɩkɩ pépi nónó-á sõ̀ mɛ kàh-ǹsah tũ névye tyɩ́ nɛ pè káh Liyel pɩ̃. Á mɛ névye yõlɛ n vi n mɔ pè kɔ̃ pè n sah Krista tyɩ́ yõ. 
Yáh kɩ pɔ n sah pallɛ Krista tyɩ́ yõ gbĩ́ nɛ̂nɛ, ápi náh tɔ̃ nòhnɩ̀-ǹkɛ̃̂ tir sah kè n sɔkɔ n pi fṍhnɔ́ kɛ̃́nɛ ké tyɩ́ sṍwãh. 
Yé yɩ̃́-á mɛ tyi nónó yõ, yě tyilɛ n yah n tɔkɔ à yɩ̃nɛ sépi ólɛ. Ǹtɛ nɛ̂-à mɛ sɛ̃ ké yõ nɛ páh mɛ Krista tyɩ́ yõ, tɛ̃̀ yɩ̃nɛ à kè pɩ̃ yĩ́ĩ́ tɔlɛ nɛ sɔ̃́-á pélɛ Krista tyɩ́ yõ, sɛ̃́ ó-á ápi tɔlɛ ǹ tyɩ́ yõ. 
Tɛ̃̀ ǹgbɛ̃ tɛ̃́nwɛnɔlɛ á kɔ̃ nɛ á yé yohnɩ mɔ yé yuku yahlɛ. À náh kélɛ á kɔ̃ yé tyɩ́ yɩkɩ yĩnnɛ. Tɛ́ ńmɔ-à kɩ sɔ̃́ mɛ ń gblɔ̌y yɔ̃ sah kah mɔ kwéy képah tyɩ́ sṍwãh, kè náh mɛ sennɛ ńmɔ tyɩ́. 
Kè náh mɛ á kɩ nɛ sɛ́bɛ́y nɔ́pi sɔ̃́ mé n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ à tíkílɛ n mɔ yé sɔkɔ dɛ́. 
Yé náh wɛ di, túkù nɛ mâh mɛ fɩ́ɩ́ yélɛ tɛ́ sɛ́bɛ́ynɛ n tɛkɩ n mɔ yé tyɩ́ mè n nɛ n nɔ n wɛkɩ fyɔ̌rɩ̀ fyɔ̌rɩ̀ ké sɔkɔ, tɛ́ mâh pɔ yé tnɔ̂ mè n pɩ tɛ̃́nkwlɔ-òlɛ, wɛ̀nfɩ̀ ó-á mè n nɛ n wɛkɩ. 
Névi nɛ̂-á n yo képahlɛ, tɛ̃̀ yɩ̃nɛ à kè pɩ̃ nɛ sɔ̃́-á ápi wɛkɩ tɛkɩ mɔ á sɛ́bɛ́ynɔ́ sɔkɔ, áyâh pɔ yé tĩ̀nnɛ sɛ̃́-á á kɩ n wɛkɩ. 
Névye nónó-á mɛ pé gblɔ̌y yĩ́gbɩ́lɛ n yo, ápi náh á nɩlɛ n syãh à á gblɔ̌ynɛ n sãh pépilɛ. Lékã́m kɛ̃́ névye ye pélɛ, nónó-á pé gblɔ̌y tɔkɔ pɩ sãh sãh dúlɛ à wɛ̃nɛ n sãh kélɛ. 
Yĩ́n-ńsah nɛ̂-á mɛ ápi tyɩ́, ápi tɛ̃ wɔ náh á gblɔ̌y gbilki n kah n mɔ n pi képah yõ. Tõ̀ nɛ̂-á Liyel dahbɩ á gbõ̀, nɛ̂ gbõ̀ yõ-á á yĩni tɛ̃ yé tĩ̀nnɛ, ápi kɩ á gblɔ̌ynɛ n gbilki à yɩ̃nɛ képahlɛ. 
Á náh n kah n mɔ n pi képah yõ, á kɩ nɛ á kósyì pɔ yé tĩ̀nnɛ. Ápi ǹgbò ye pɔ yé tĩ̀nnɛ à pɔ Yesu Krista Nɩ̀vɩ̀nɩ̀ tir yo yé tyɩ́. 
Á náh n kah n mɔ n pi képah yõ, kɩ pɔ á gblɔ̌ynɛ gbilki no tɛ́lɔ́ tõ̀ pɩ́nɔ́ yĩnnɛ. Ǹtɛ á mɛ ké tyɩ́ nɛ yé yuku yahlɛ Yesu yõ sɛ̃́nɔ́ sɔkɔ, sɔ̃̀nko wãn mɔ á tõ̀ yõ yé tĩ̀nnɛ à yɩ̃nɛ yĩn-ńsáh nɛ̂-á á kɔ̃. 
Képah náh, á kɩ fɛ̃ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo n kɔ̃ tã̀ntã̀n tɛ́lɔ́ sɔkɔ nónó-á sé gblɔ̌y yépi tĩ̀nnɛ. Á náh ǹnɩ̂ yɔ́ ńtɛ̃ tõ̀ pɩ́nɔ́ tɔkɔ á gblɔ̌y gbilki n pi. 
Kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «Névi nɛ̂-à mɛ ké tyɩ́ nɛ pé pé gblɔ̌y gbilki, Tɛ̃̀ ǹgbɛ̃-á tyi nónó pɩ ápi tyɩ́, tɛ̃̀ ǹ gblɔ̌y gbilki sépi yĩnnɛ.» 
Névi nɛ̂-à ǹ gblɔ̌ynɛ n sõ, kè náh pɩ á kɩ nɛ ǹmɔ sɔ̃́ nɛ́gbɩ́lɛ dɛ́, Tɛ̃̀ ǹgbɛ̃-à névi nɛ̂nɛ n sõ, ǹmɔ ye nɛ́gbɩ́lɛ.
