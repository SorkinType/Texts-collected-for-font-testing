Korɛ̃nti tãn.
5.
Kè mɛ n nohnɩ ywɔ̃ ywɔ̃ pól sɔkɔ nɛ ńkɛ̃́, prɔ̀prɔ̀ pɩ́nɔ́-á mɛ yé wrɔ́ sɔkɔ. Prɔ̀prɔ̀ pɩ́nɔ́ yɔ́ ńtɛ̃-á mɛ yé wrɔ́ sɔkɔ, nɛ̂-á Yesu yõ sɛ̃́-ńkɛ̃̂pú ńtɛ̃ náh fɛ̃ syi sɛ̃́nkɔ̃ pɩ n pi. Pè nɛ yé sɔkɔ yɔ́ ńtɛ̃-á mɛ ǹ sú kyɩ tɔkɔ. 
Tɛ́ sépinɔ́ pól sɔkɔ, yé tɔ̃ mɛ yé gblɔ̌ynɛ n yɔ̃ n sah n kɔ̃ kɛ̀? Kè pyě sõ̀ yɩ̃nɛ kè pɩ yĩ́ĩ́ yèkwɔrɩlɛ yé tyɩ́, yé kɔ̃ yè kénɛ tɛ̃̀ yĩni n wilki yé wrɔ́. 
Ńmɔ tɛ̃̀ wɔ náh mɛ yélɛ fɩ̃́ núkú sɔkɔ. Ǹtɛ mé mɛ yélɛ sõnɔ sɔkɔ. Képah ye, ńmɔ tɛ̃̀ wɔ lékàhlɛ yĩ mɛ̀ pɩ́-ò kɔ̃ tɛ, á kɩ nɛ mé sɔ̃́ mɛ yé wrɔ́ sɔkɔ. 
Yâh pɔ wɛ̃ tuke mɔ, ńmɔ tɔ kɩ mɛ yélɛ sõnɔ sɔkɔ. Sɛ̃́nkɔ̃ névi tɛ̃̀nɛ, yé Yesu Krista yĩn ànɛ̂ ǹ tɛ̃́nwɛnɔlɛ à wilki yé wrɔ́, yè ǹ gbàh dɛ n mɔ sétã̀n yõ̀tɛ̃̀ kɔ̃. Kàh tɛ̃̀nɛ n kwlɔ, à kɩ pɩ́-ńkɛ̃̂ tyi pɩ́nɔ́ yal mɔ ǹ min kɔ̃ kè kɩ wɛ pwah, Tɛ̃̀ ǹgbɛ̃ Yesu syɩ n pɔ sõlɛ. 
Yé gblɔ̌y gbílkínɔ́ mɛ̀ yah náh. Yé se yɔ́ pɩ̃ nɛ, kɔ́kɔ́nɔ́ kwéy ó-á n dɔkɔ mífã́hkɩ̀ póllɛ tɛ́. 
Yé kénɛ tyípékè wilki n to yé wrɔ́, képah nɛ̂-á mɛ n tɔ̃ n mɔ kɔ́kɔ́nɔ́ tyɩ́ kɔ̃lɛ. Képah sɔkɔ, yé kɩ mɛ wásáá kɔ́kɔ́nɔ́ kã̂n búrn tyɩ́ kɔ̃lɛ. Krista lésõ fɛ̃ syi pè kɔ̃ pè à ko mɔ Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh képɔ́ tyɩ́ kɔ̃lɛ ápi tyípêl wu n to yĩnnɛ á tyɩ́. Kǎh pɩ sɛ̃́, yé mɛ wásáá kɔ́kɔ́nɔ́ kã̂n búrn tyɩ́ kɔ̃lɛ. 
Yé yɛ à tyípêl duke pól wilki n to á min sɔkɔ, à mó nɛ n mɛ yályál nɛ́gblɔ́lɛ kɔ́kɔ́nɔ́ kã̂n búrn tyɩ́ kɔ̃lɛ, nɛ̂-á pè n nɛ n tah Nsyifunɔ yɔ̃rɩ tyɛ́nɔ́ syáh sɔkɔ. 
Máh lésõ sɛ́bɛ́y nɛ̂ wãrkɩ tɛkɩ mɔ yé tyɩ́, mé lésõ yo ké sɔkɔ nɛ yé káh nɛ n pɩ n soh prɔ̀npɩbɩlɛ. 
Ǹtɛ kèkõyṍ névye tyɩ́ sɔkɔ náh lésõ ńnɛ dɛ́. Kèkõyṍ mɛ̀ prɔ̀npɩbɩ, képah náh pɩ ké yĩ́sesúkú névye, ànɛ̂ ké yɩ̃́pú, képah náh pɩ, ké sãh fálpú tyɩ́ sɔkɔ-à sõ̀ ńnɛ, yé náh náhkɩ sõ̀ fɛ̃ n wũ n pi kèkõyṍ mɛ̀ sɔkɔ dò. 
Ǹtɛ sɛ̃́ náh dɛ́. Mé nɛ névi nɛ̂-à ǹ gblɔ̌ynɛ n ye Yesu yõ sɛ̃́-òlɛ, tɛ́ mɛ prɔ̀prɔ̀lɛ n pɩ, képah náh pɩ, yĩ́sesúkúlɛ n pɩ, képah náh pɩ, sãhlɛ n fal, képah náh pɩ, névye tyɩ́lɛ n yo n ko, képah náh pɩ, sètɛ̃̀nɩ̀nɛ n pɩ, képah náh pɩ, mɛ yɩ̃belɛ n yɩ̃, yé káh nɛ n pɩ n soh sɛ̃́nkɔ̃ nɛynɛ póllɛ. Yé káh yõke ńtɛ̃nɛ n yõ wɛ̃ tyɩ́ tɛ̃̀nɛ. 
Tɛ́ Yesu yõ sɛ̃́-ńkɛ̃̂pú tùkènɔ́ tɛ̃̀ wɔlɛ, ńmɔ tõ̀ náh képahlɛ. Liyel ye kɩ tuke pélɛ. Ǹtɛ kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ ńkɛ̃́: «Yé nɛ́pékè yĩni n wilki yé wrɔ́.» Kǎh yo sɛ̃́ à, yé náh yé tĩ̀nkì névye tuke n pi di?
