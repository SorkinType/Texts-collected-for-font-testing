2 Timote.
4.
Yesu Krista nɛ̂-á kɩ pɔ vye ànɛ̂ kyɩ̂ tuke n pi, mé kélɛ n gbah n yah n kɔ̃ á tyɩ́ Liyel ànɛ̂ ǹmɔ yĩ́ yah, à tahlɩ ǹ syɩ́npɔnɔ ànɛ̂ ǹ tɔ̃́rɩ́ yĩn yõ nɛ, 
á Liyel wɛynɛ nɛ n yo. Kàh sɔ̃́ yèvyãhrɩ gbĩ́ sɔkɔ ò; kàh sɔ̃́ yèvɩnɩ gbĩ́ sɔkɔ ò; kélɛ nɛ n gbah n yo. Névi nɛ̂-à pu pɩ, wɛkɩ tɛ̃̀ yõ, vìnmɔnɔlɛ tɛ̃̀ kɔ̃, tɛ̃̀ tɔkɔ yárr ǹ kwɔ́nɔ́lɛ à kɔ̃. 
Gbĩ yɔ́ mɛ n pi, túkù náh tɔ̃ kwɔ́nɔ́ yèsyɩ̀kɩ̀ fɛ̃ n syi n pi. Tyi nónó dékè-á n ko pélɛ, pè kɩ kwɔ́pú kwĩnki yísyɔ́ yah nónó-á kɩ pɔ sépi tyɩ́lɛ n wɛ n yo pé tyɩ́ nónó nóhnɔ́-á mɛ nɔ. 
Pè kɩ pé ní tah tũ yahle gbɩ nóhnɔ́lɛ, tɛ́ kɩ sèmìrn wɔlɛ n nohnɩ. 
Ǹtɛ áwɔ tɛ̃̀ wɔ á gblɔ̌y tɛ̃ pallɛ tyi tyi pól sɔkɔ. Kwlɔ́nɔ́ sro, ǹ tɛ́ á gblɔ̌y kɔ̃ yĩ́ĩ́ Nɩ̀vɩ̀nɩ̀ tir yónɔ́ sɔkɔ. Pɩ Liyel tõ̀npɩ pèpɛynɛ. 
Ńmɔ tɛ̃̀ wɔ pɩ gbĩ́ nɔ tɛ sárkálɛ. Kèkõyṍ fĩn mɔ gbĩ́ nɔ ń tyɩ́. 
Mé mɛ́kɩ̀ pèpɛy fwo à yuku Liyel yõ sɛ̃́nɔ́ tyɩ́ tĩ̀nnɛ. Tólɔ́ nɛ̂ sɔkɔ-á sõ̀ ńnɛ, kè tyɛ. Mé náh Liyel yõ sɛ̃́nɔ́ yal mɔ. 
Ǹtɛ núkúnúkú, tɛ̃́nwɛnɔ yõre wɛ́nɔ́ ye mé mɛ syɩkɩ tɛ̃, nɛ̂-á nɛ́gblɔ́ yõ̀tɔ̃rɩ́ yṍkúrílɛ. Tɛ̃̀ ǹgbɛ̃ nɛ̂-á tukey tùkè-ò pèpɛynɛ, ǹmɔ ye kɩ ń kɔ̃ n pi kélɛ. Tɛ́ à náh n yĩn n pi ńmɔ ó tyɩ́ dɛ́, nónó tɔ-á mɛ ǹ pɔ́nɔ́ syɩkɩ tɛ̃ nɩ̀vɩ̀nɩ̀ sɔkɔ, à kɩ pépi tɔ pól kɔ̃ kélɛ. 
Gbah n tɛ̃ n pɔ ń tĩ̀nnɛ súú. 
Yé náh wɛ di, kwɛy gbĩ tyi mɛ yah syi Demasɩ tyɩ́ à kɔ̃ à mɛ ǹ náh vi ń kɔ̃. À mɛ sɔkɔ Tesaloniki kwil sɔkɔ. Krɛsã mó sɔkɔ Kalasi mara sɔkɔ, tɛ́ Tite wɔ mɛ sɔkɔ Dalmati mara sɔkɔ. 
Likyi ó ye mɛ ń tnɔ̂. Markɩ tɛ̃ yè pɔ wɛ̃ tyɩ́ ǹnɛ. Ǹ yétõ̂ mɛ gblò ń tyɩ́ tõ̀ mɛ̀ sɔkɔ. 
Mé Tisyiki tɛkɩ mɔ Yefɛsɩ kwil sɔkɔ. 
Wáh kɩ n pi, ń fwɔ̀mɔ̀-ǹgbɛ̃ tɔkɔ n pɔ ń kɔ̃ nɛ̂-á mé sah Karpusi tyɩ́ Turasɩ kwil sɔkɔ. Ń sɛ́bɛ́ynɔ́ tɔ tɔkɔ n pɔ n kɔ̃, ǹ tɛ́ n pɔ sé sêl tyɩ́nɔ tɔlɛ, sépi nónó tyɩ́-á tah kal ń tyɩ́. 
Kètɔ̃ndĩ́ Alɛkisãdre nɛ́pékè kah pɩ ńnɛ. Tɛ̃̀ ǹgbɛ̃ kɩ à dyah n pi ǹ tyípɩ́nɩ̀nɛ. 
À ápi wɛlɔ kɔ̃lɩ kah mɔ. Képah ye, á tɔ á gblɔ̌ynɛ n tɛ̃ ǹnɛ. 
Pǎh ń tuke sõ nɛ̂nɛ gbennɛ, no pól pé náh vi ń kɔ̃. Ǹnɩ̂ yɔ́ ńtɛ̃ náh yo tɛ̃ mɔ ǹnɛ. Mé kélɛ n yah n kɔ̃ Liyel tyɩ́ nɛ à káh kè pɩ tirlɛ pé tyɩ́. 
Ǹtɛ Tɛ̃̀ ǹgbɛ̃ wɔ so mɔ ń tyɩ́. À tǎhkɩ̀lɛ n kɔ̃ mé fɛ̃ ǹ wɛy yo yĩ́ĩ́, tɛ́ nónó náh pɩ Nsyifunɔlɛ, pé pól kè wɛ noh. À ń pwah mɔ kyɛ̃̀kɩ́ngbɛ̃kɩ tyɩ́. 
Tɛ̃̀ ǹgbɛ̃ kɩ tɔ̃ ń yõ wilki n pi pèl duke duke pól sɔkɔ. Tɛ́ kɩ ń pwah mɔ ń kɔ̃ mé kɩ wɛ de ǹ tɔ̃́rɩ́ sɔkɔ. Gbílkínɔ́ nɛ n sah ǹmɔ yõ gbĩ́mɛgbĩ́. Amiina! 
Wɛkɩ Prisili ànɛ̂ Akilasɩ tyɩ́ ń kɔ̃, ànɛ̂ Wonesifɔrɩ nolɛ. 
Yerasɩtɩ wũ Korɛ̃nti kwil sɔkɔ. Mé Torofimi wɔ sah Milɛti kwil sɔkɔ. À sõ̀ mɛ n yɔ képah ye. 
Gbah n tɛ̃ n pɔ kwɔ̀vɩrɩ gbĩ́ nɔ́-ńkɛ̃̂ sɔkɔ. Yebulusi, ànɛ̂ Pudɛ̃sɩ, ànɛ̂ Linusi, ànɛ̂ Klodya, ànɛ̂ á yṹnpyé tɛ́lɔ́ pól n wɛkɩ á tyɩ́. 
Mé kélɛ n yah n kɔ̃ Tɛ̃̀ ǹgbɛ̃ tyɩ́ nɛ à nɛ n mɛ áwɔlɛ, à mó yõ̀bènɔrɩlɛ nɛ n pɩ yé pól tyɩ́.
