Yefɛsɩ tãn.
5.
Yáh pɩ Liyel wɛ̃́npìlɛ nónó tyɩ́-á nɔ ǹ tyɩ́ sɛ̃́, képah ye yé ǹmɔlɛ n yah n pɩ. 
Yé yɛ yé tyɩ́ nɔ wɛ̃ tyɩ́, sɔ̃́-á á pól tyɩ́ nɔ Krista tyɩ́, tɛ́ à mɛ ǹ gblɔ̌ynɛ Liyel kɔ̃ ápi yĩnnɛ à pɩ sárkálɛ, nɛ̂-á mɛ n põ sírr tɛ́ ké tyɩ́ mɛ nɔ Liyel tyɩ́. 
Yé pɩ Liyel nolɛ, képah ye fɔ̀fɩ́kɩ̀, ànɛ̂ prɔ̀prɔ̀ pɩ́nɔ́ duke pól, à tahlɩ yĩ́sesúkú tyɩ́, sépi sɔkɔ yɔ́ ńtɛ̃ pɩ́nɔ́ tyɩ́ náh yɩ̃nɛ kè nɛ n nohnɩ yé wrɔ́ sɔkɔ. 
Sen pɩ̃́-ńkɛ̃̂ wɛlɔ, ànɛ̂ wɛ̀ntùkùnì, ànɛ̂ nɛ́wã̂n wɛlɔ tɔ náh yɩ̃nɛ sè nɛ n wil yé vyɛ̃y sɔkɔ. Ǹtɛ pèpɔrɩ pɩ̃́nɔ́ wɛy ye yé yɩ̃nɛ yè Liyellɛ n kɔ̃. 
Yé yɩ̃nɛ yè kè pɩ̃ yĩ́ĩ́ nɛ prɔ̀npɩbɩ, ànɛ̂ nónó-á mɛ sìlkìnɔ́lɛ Liyel yah, ànɛ̂ yĩ́sesúkú névyelɛ, pé sɔkɔ yɔ́ ńtɛ̃ náh n de n pi Krista tɔ̃́rɩ́ ànɛ̂ Liyel tɔ̃́rɩ́ sɔkɔ. Yĩ́sesúkú pɩ́nɔ́lɛ, sãh fálnɔ́ dùkù yɔ́ ye kélɛ. 
Yé káh yɛ ǹnɩ̂ yɔ́ ńtɛ̃ kɔ̃ à pɔ yé pilki ǹ sõnɔ prɛ̌nɩ̀nɛ. Sépi dùkù pɩ́nɔ́ ye ńkɛ̃́nɛ n pɔ Liyel kɔ̃ ǎ fṍhnɔ́ sah ǹ tyɩ́ yáhlepu yõ. 
Képah ye, yé káh nɛ pɩ n soh pélɛ póllɛ. 
Yé lésõ mɛ fɛ̃́ntũ̂ sɔkɔ ǹgbòlɛ. Ǹtɛ núkúnúkú yé mɛ kyòo sɔkɔ yé wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Tɛ̃̀ ǹgbɛ̃nɛ. Kyòo sɔkɔ névye-á n pɩ sɔ̃́, yé tɔ nɛ n pɩ sɛ̃́. 
Yé náh wɛ di, kyòo sɔkɔ kɔ́lɔ́ ye n pɔ n pɔ pèpɔrɩ pɩ́nɔ́lɛ, ànɛ̂ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́lɛ, ànɛ̂ gbɩ yónɔ́lɛ. 
Nɛ̂ tyɩ́-á nɔ Tɛ̃̀ ǹgbɛ̃ tyɩ́, yé gbah tɛ̃ kélɛ n pnɛ. 
Yé káh fɛ̃́ntũ̂ sɔkɔ tyilɛ n pɩ. Yõre náh mɛ sé tyɩ́. Ǹtɛ yé sé gbɩ̃nɛ n wɔ. 
Tyi nónó-á kénɛ névye mɛ n wɔ̃bɩ n pɩ, sé tyɩ́ yã́hnɔ́ ńtɛ̃ mɛ sennɛ. 
Ǹtɛ tyi nónó gbɩ̃-à wɔ, sépi n nɛ n mɛ kyòo. 
Yé náh wɛ di, tyi nónó gbɩ̃-à wɔ, sé dùkù n nɛ n pnɛ. Képah ye, kè mɛ yónɔ́lɛ nɛ: «Áwɔ nɛ̂-á mɛ n dɔ, dɔ n yuku. Yuku n wil lékyɩ̂ wrɔ́, Krista kɩ kyòo mɔ á tyɩ́.» 
Yé yé pɩ dùkùlɛ n kyɔmɩ pallɛ. Yé káh nɛ n pɩ yɩ̃ pɩ̃́-ńkɛ̃̂ névye tyɩ́ kɔ̃lɛ, yé nɛ n pɩ yɩ̃ pɩ̃́pú tyɩ́ kɔ̃lɛ, 
nónó-á tir nɛ̂ pɩ́nɔ́ yõ̀bènɔrɩ-à fyɔ̀ wɛ pé tyɩ́, pě kè pɩ. Yé náh wɛ di, gbĩ nɛ̂ sɔkɔ-á álɛ, kè náh mɛ pallɛ. 
Képah ye, yé káh pɩ lékã́m kɛ̃́ névyelɛ, ǹtɛ Tɛ̃̀ ǹgbɛ̃ dyɔ yĩ́ĩ́ tɛ̃̀-á nɛ̂nɛ, yé gbah n tɛ̃ képahlɛ n pnɛ. 
Yé káh tɔ̃ wɔ kè yé tɛ̃, képah névi tyɩ́lɛ n yɩkɩ. Yé yɛ Liyel Mirki yé tɔkɔ. 
Yé sṍrélɛ n mɔ wɛ̃ sɔkɔ yɩ̀gbɩ̀nɩ̀ sɔkɔ yè Tɛ̃̀ ǹgbɛ̃ n gbilki, yè mó yɩ̀lɛ n gbɩ ǹ tyɩ́ ǹ Mirki-á nónó dahbɩ yé sɔkɔ. Yé yɩ̀lɛ n gbɩ yè Tɛ̃̀ ǹgbɛ̃nɛ n gbilki yé nnɔ póllɛ. 
Gbĩ́mɛgbĩ́ ànɛ̂ tyi tyi pól sɔkɔ, yé nɛ n wɛkɩ Sú Liyel tyɩ́ á Tɛ̃̀ ǹgbɛ̃ Yesu Krista yĩn yõ. 
Yépi nónó-á pɩ Krista yah tíkí pɩ́púlɛ, yé nɛ n sah wɛ̃ tyɩ́ yõ sɔkɔ. 
Syɔ́, yé nɛ n sah yé pɛbɩ tyɩ́ yõ, sɔ̃́-á yé n sah Tɛ̃̀ ǹgbɛ̃ tyɩ́ yõ. 
Yé náh wɛ di, dĩ́ ye sê yõ̀tɛ̃̀nɛ sɔ̃́-á Kristalɛ ǹ yõ sɛ̃́pú tĩ̀nkì yõ̀tɛ̃̀nɛ, pépi nónó-á ǹ wil gbɛ̃́nsêlɛ tɛ́ à mɛ pɩ pé pwáhnmɔ-òlɛ. 
Syɔ́ yɩ̃nɛ pè nɛ n sah pé pɛbɩ tyɩ́ yõ tyi tyi pól sɔkɔ, sɔ̃́-á Krista yõ sɛ̃́pú tĩ̀nkì n sah ǹ tyɩ́ yõ. 
Pɛbɩ, yé yɛ yé syɔ́ tyɩ́ nɔ yé tyɩ́, Krista yõ sɛ̃́pú tĩ̀nkì tyɩ́-á nɔ sɔ̃́ ǹ tyɩ́, fɔ́ɔ́ à ǹ min kɔ̃ pé yĩnnɛ. 
Sɛ̃́ ye sõ̀ à mɛ ké tyɩ́ nɛ pé pé yõ sɛ̃́pú tĩ̀nkì pɩ yályál Liyel yah, ni sɔkɔ wèlnwìlkìnɔ́ sɔkɔ ànɛ̂ Liyel wɛy gbõ̀ yõ. 
À sõ̀ mɛ ké tyɩ́ nɛ pè pɔ pé yah sɔkɔ yályál syíi kɛ̃́nɛ, ànɛ̂ tìkìlì-ǹsah kɛ̃́nɛ, ànɛ̂ yɩ́kɩ́-ńsah kɛ̃́nɛ, ànɛ̂ wɛ́nyo-ńsah kɛ̃́nɛ pé tyɩ́. 
Sɛ̃́ ye pɛbɩ tɔ yɩ̃nɛ pé syɔ́ tyɩ́ nɔ pé tyɩ́ pé gblɔ̌y wil gbɛ̃́nsê tyɩ́-á nɔ sɔ̃́ pé tyɩ́. Névi nɛ̂ kyɩ tyɩ́-à nɔ ǹ tyɩ́, ǹ gblɔ̌y tyɩ́ ye nɔ ǹ tyɩ́. 
Yé náh wɛ di, ǹnɩ̂ yɔ́ ńtɛ̃ wil gbɛ̃́nsê kṍ náh ǹnɛ n tɛ̃ dò, tɛ̃̀ n pyě kélɛ n pu ńkɛ̃́; ǹ tɛ́ kè tɔ̃ n sah n tɛ̃ sɔ̃́-á Krista tɔ̃ sah tɛ̃ ǹ yõ sɛ̃́pú tĩ̀nkìlɛ, 
képah nɛ̂-á ápilɛ. 
Képah ye, dĩ́ kɩ yal ǹ sú ànɛ̂ ǹ yṹ tnɔ̂, tɛ́ kɩ kwrɔ mɔ ǹ kyɩlɛ, pé nímí kɩ pɩ wil gbɛ̃́nsê núkúlɛ. 
Sètĩ̀n sõ̀ tir ǹgbɛ̃ ye kélɛ. Tɛ́ ńmɔ kélɛ n yo nɛ kǎh mɛ ké yah mɔ Krista ànɛ̂ ǹ yõ sɛ̃́pú tĩ̀nkì tyɩ́. 
Ǹtɛ kè mɛ ké yah mɔ yépi tɔ tyɩ́. Yé nɛ́núkù núkú pól syɔ́ tyɩ́ yɩ̃nɛ kè nɔ yé tyɩ́ yé gblɔ̌y sɔ̃́, syɔ́ nɛ́núkù núkú pól mó yõ̀yénnɛ n sah pé pɛbɩ yõ.
