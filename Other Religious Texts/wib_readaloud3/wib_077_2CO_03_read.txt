Korɛ̃nti tãn.
3.
Ápi se tɔ̃ yɩ̃nɛ á gblɔ̌y dùkù yónɔ́lɛ yé tyɩ́? Képah náh pɩ, á se yɩ̃nɛ à sɛ́bɛ́y yɔ́ wãrkɩ yé kɔ̃ à á gblɔ̌y dùkù yo yé syah, sɔ̃́-á túkù kélɛ n pɩ? Tĩ̂, yépi se kɩ wãrkɩ á kɔ̃ sɛ́bɛ́ynɛ, nɛ̂ sɔkɔ-á yé kɩ mɛ ápi dùkù yo? 
Yépi ye ápi sɛ́bɛ́y mɛ̀nɛ. Kè mɛ wã́rkɩ́nɔ́lɛ á nnɔ sɔkɔ. No pól kélɛ n pnɛ, tɛ́ kélɛ n kar n pnɛ. 
Kè mɛ píkínɔ́lɛ nɛ yépi-á sɛ́bɛ́y mɛ̀nɛ, nɛ̂ sɔkɔ wɛlɔ-á Krista á kɔ̃ á wãrkɩ. Wã́rkɩ́nɔ́ mɛ̀nɛ, min nɛ́tɛ̃̂ Liyel Mirki gbõ̀ nɩ́y ye, wãrkɩ wãrkɩ kwâh nɩ́y náh. Dyêh fábkɔ́ fábkɔ́ yõ sɔkɔ náh kélɛ wã́rkɩ́nɔ́lɛ dɛ́, névye nnɔ sɔkɔ ye. 
Krista gbõ̀ yõ ye á sɛ̃́nyĩnɔ mɛ kénɛ wɛlɔ yõ Liyel yah sɔkɔ. 
Ápi-á tyi nónónɛ n pɩ yɔ̀, á yɛ̃́ngbɛ̃́ tɛ̃́nwɛnɔ náh á fɛ̃ n pɩ sélɛ dɛ́, Liyel kɔ̃́nɔ́ ye. 
Ǹmɔ ye á kɔ̃ tɛ̃́nwɛnɔlɛ á fɛ̃ yégbékè kwrɔ́nmɔnɔ késã́n tyilɛ névyelɛ n kwɔ. Kénɛ yégbékè kwrɔ́nmɔnɔ náh n wil tyi kõ̀nsàhnɔ́ wã́rkɩ́nsàhnɔ́ sɔkɔ dɛ́, Liyel Mirki tõ̀ vyãh ye kélɛ. Yé náh wɛ di, tyi kõ̀nsàhnɔ́ nɛ̂-á wã́rkɩ́nsàhnɔ́lɛ, képah n pɔ n pɔ tyɩ́ yɩ́kɩ́nɔ́lɛ. Ǹtɛ Liyel Mirki wɔ minnɛ n kɔ̃. 
Tyi kõ̀nsàhnɔ́ wɔ nɛ̂-á n pɔ n pɔ tyɩ́ yɩ́kɩ́nɔ́lɛ, ké wã́rkɩ́nmɔnɔ gbĩ́nɛ dyêh fábkɔ́ fábkɔ́nɔ́ yõ, Liyel tɔ̃́rɩ́ sõ̀ mɛ tiki pɔ Moyisi yèpɔ kɔ̃ kè n tɛ̃, fɔ́ɔ́ Yisrayel no náh sõ̀ mɛ fɛ̃ n yah n dah n mɔ ǹ yah sɔkɔ. Tɛ́ kénɛ tɔ̃́rɩ́ náh mo póllɛ. 
Ǹtɛ Liyel Mirki tɛ̃̀ wɔ-á tõ̀ nɛ̂nɛ n pɩ, képah náh Liyel tɔ̃́rɩ́ wɛ̃kɩ n kal n pi di? 
Tyi kõ̀nsàhnɔ́ wã̀l kɔ́nɔ́ nɛ̂-á n pɔ n pɔ tyɩ́ yɩ́kɩ́nɔ́lɛ, gbílkínɔ́-à wɛ képah tyɩ́ sɔkɔ, gbɩ wɛ̃̀kɩ̀ kɔ́nɔ́ wɔ se pyě. 
Á kɩ fɛ̃ yo ńtɛ̃nɛ nɛ tɔ̃́rɩ́ nɛ̂-á sõ̀ mɛ gben tɛ̃̀nɛ, núkúnúkú tyɩ́-á képah tɔkɔ yõ. 
Gbílkínɔ́-à mɛ sah mó-ńkɛ̃̂ kwâh yõ, mo mo tyɩ́ wɔ se pyě. 
Á gbõ̀ sàhnyĩnɔ mɛ képah yõ, képah ye á nɩ mɛ yĩ́nnɔ́lɛ. 
Ápi náh mɛ pɩ Moyisi tyɩ́ kɔ̃lɛ, nɛ̂-á sõ̀ fwɔ vike tũ ǹ yah sɔkɔ sɔ̃̀nko Yisrayel no káh ǹ yèpɔ tɛ̃́nɔ́ yal n mɔ gbĩ́ pɩ̃, képah nɛ̂-á sõ̀ mo mo tyɩ́ náh kélɛ. 
Ǹtɛ pè mɛ sɛ̃́ fɛ̃́ntũ̂ sɔkɔ. À pɔ nɔ kwɛy tyɩ́, pàh tyi kõ̀nsàhnɔ́ kédyâhlɛ n kar, kénɛ fwɔ mɛ̀ n nɛ n mɛ fɔ́kɔ́ntũtɛ̃̀nɔ́lɛ pé yah sɔkɔ. Kè náh tɔkɔ to pé yah. Nɛ́ nɛ̂ náh ǹ wrɔ́ kwrɔ mɔ Kristalɛ, kè náh n tɔkɔ n to tɛ̃̀ yah. 
Gbɩ yõ à pɔ syɩ kwɛy tyɩ́, pàh Moyisi tyi kõ̀nsàhnɔ́lɛ n kar, fwɔ mɛ̀ n nɛ n mɛ pé lékã́m yah fɔkɔ n tũ n sah. 
Ǹtɛ nɛ̂ wɔ-à sɛ̃ Tɛ̃̀ ǹgbɛ̃ yõ, fwɔ mɛ̀ n tɔkɔ n to tɛ̃̀ yah sɔkɔ. 
Tɛ̃̀ ǹgbɛ̃ nɛ̂ tyɩ́-á n yo ásõ̂, Liyel Mirki ye. Ǹtɛ Liyel Mirki-à mɛ ńyãh sɔkɔ, névye n nɛ mɛ pé gblɔ̌y yõ nɛ́npɔ́. 
Fwɔ náh mɛ víkéntũnɔlɛ ápi tɛ̃̀ wɔ yah. Á dùkù mɛ Tɛ̃̀ ǹgbɛ̃ tɔ̃́rɩ́lɛ n wɛ̃kɩ. Á vi mɔ pɩ Tɛ̃̀ ǹgbɛ̃ tyɩ́ kɔ̃lɛ. Wãn mɛ n mɔ n yuku á tɔ̃́rɩ́ yõ. Tɛ̃̀ ǹgbɛ̃ tõ̀ vyãh ye képahlɛ, ǹmɔ nɛ̂-á Mirkilɛ.
