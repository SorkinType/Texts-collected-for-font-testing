3.
Yé yah, kègbɩ wɛy yɔ́ ye mé n tyah yónɔ́lɛ yé tyɩ́ yɔ̀. Névi nɛ̂-à mɛ ké tyɩ́ nɛ pé pɩ Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́-òlɛ, tõ̀ pèpɛy ye tɛ̃̀ n yah n kɔ̃ pɩ́-ńsahlɛ. 
Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́-ò yɩ̃nɛ à pɩ wɛ́nyo-ńsah kɛ̃́nɛ ǹ tyɩ́. À yɩ̃nɛ à pɩ sê núkú nɛ́tɛ̃̂nɛ, gblɔ̌y tɛ̃́npnɛ̀-òlɛ, nɛ̂-á mɛ ǹ tyilɛ n pɩ n yɩ̃nɛ pallɛ, mɛ nɛ́pânnɛ n tɛ̃ plɛ, tɛ́ kɩ fɛ̃ névye kwɔ n pi. 
À náh yɩ̃nɛ à pɩ sèwɔlɛ, képah náh pɩ nɛ́pékèlɛ. Ǹtɛ à yɩ̃nɛ à pɩ névilɛ, nɛ̂-á ǹ gblɔ̌ynɛ n yɛ n pnɛ. À náh yɩ̃nɛ à pɩ mâl kɔ̃̀lɩ̀-òlɛ, képah náh pɩ pɔ́ náh púkúbínmɔ-òlɛ. 
À yɩ̃nɛ à nɛ n mɛ ǹ kéte yahlɛ n tɛ̃ n pnɛ pallɛ, à mó ǹ wɛ̃́npì kwɔ pè nɛ n sah ǹ tyɩ́ yõ kǎh yɩ̃nɛ sɔ̃́. 
Yé náh wɛ di, névi-à tɛ̃ wɛ ǹ kéte yahlɛ n tɛ̃́npnɛ̀-ǹkɛ̃̂nɛ, à kɩ pɩ sɔ̃́ tɛ́ kɩ fɛ̃ Yesu yõ sɛ̃́pú tĩ̀nkì tɛ̃́nɔ́ plɛ, pépi nónó-á Liyel nolɛ? 
Tɛ̃̀ náh yɩ̃nɛ à nɛ n mɛ nɛ́sã́nnɛ Liyel wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ, sɔ̃̀nko gblɔ̌y gbílkínɔ́ káh pɔ de tɛ̃̀ sɔkɔ, ǹ tyɩ́ kɔ̃ kè pɔ n yɩkɩ sétã̀n yõ̀tɛ̃̀ tyɩ́ kɔ̃lɛ. 
À wil képah sɔkɔ, Yesu yõ sɛ̃́-ńkɛ̃̂pú tɔ yɩ̃nɛ pè nɛ mɛ tɛ̃̀ yĩ́ngbɩ́lɛ n yo, sɔ̃̀nko yílɔ́nɔ́ káh pɔ à yĩni tɛ̃, à kɔ̃ à pɔ tɛ̃ sétã̀n yõ̀tɛ̃̀ vyɔ́y sɔkɔ. 
Sɛ̃́ ó ye, Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́pú yòhnɩ̀nmɔpu tɔ yɩ̃nɛ pè pɩ nɛ́gblɔ́lɛ ànɛ̂ wɛ́kɩ́nyĩpulɛ. Pè náh yɩ̃nɛ pè pɩ sèwɔbɩlɛ ànɛ̂ kwâh yáhnkɔ̃pulɛ yĩ́nvǐ sɔkɔ. 
Gbɩ nɛ̂-á pè wɛ pɩ̃ Liyel yõ sɛ̃́nɔ́ gbõ̀ yõ, pè yɩ̃nɛ pè kè tɛ̃ n sah pé sõnɔ sɔkɔ pallɛ. 
Ké gbennɛ, pè yɩ̃nɛ mɔ́nyahnɔlɛ. Pílkí-ńsah-à wɛ́-ńkɛ̃̂nɛ pé tyɩ́, pè kɩ pyě fɛ̃ wilki sah Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́pú yòhnɩ̀nmɔpulɛ. 
Syɔ́ nónó tɔ-á Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́pú yòhnɩ̀nmɔpulɛ, pé yɩ̃nɛ pè pɩ névyelɛ nónó-á yõ̀yén mɛ n sah pé yõ. Pè náh yɩ̃nɛ pè pɩ névye tyɩ́ yónkòpulɛ. Pè yɩ̃nɛ pè pɩ gblɔ̌y tɛ̃́npnɛ̀pulɛ, pè mó nɛ n mɛ tómm tyi tyi pól sɔkɔ. 
Yesu yõ sɛ̃́pú tĩ̀nkì yah tɛ̃́pú yòhnɩ̀nmɔ-ò yɩ̃nɛ à nɛ n mɛ sê núkú nɛ́tɛ̃̂nɛ, à mó nɛ mɛ tɛ̃ wɛ ǹ wɛ̃́npì ànɛ̂ ǹ kéte yah tɛ̃́nɔ́ tyɩ́. 
Nónó-à pé tõ̀lɛ n pɩ pallɛ, pè kɩ yĩ́n-ńsah ǹgbɛ̃ wɛ ké sɔkɔ, tɛ́ gbõ̀ sàh-ǹsah tɔ kɩ wɛ pé tyɩ́ pé wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Yesu Kristalɛ. 
Mé ye sɛ́bɛ́y mɛ̀nɛ n tɛkɩ n mɔ á tyɩ́, ǹtɛ ń sõnɔ sɔkɔ mé kɩ pɔ n yah n pi á tyɩ́ dǒ kwéy. 
Ǹtɛ máh kɩ se mo pɔ́nyah-ńsah á tyɩ́, tyi nónó-á mé wãrkɩ tɛkɩ mɔ á tyɩ́, sépi kɩ ǹ yohnɩ n mɔ á kɩ á gblɔ̌y tɛ̃ dùkù pɩ̃ min nɛ́tɛ̃̂ Liyel no wrɔ́ sɔkɔ, pépi nónó-á Yesu yõ sɛ̃́pú tĩ̀nkìlɛ, nónó-á gbɩ tɛ̃́nsàhpulɛ dàhpɛ sɔ̃́. 
Liyel-á nɛ̂ yo á tyɩ́ Yesu Krista tyɩ́ sɔkɔ, tyi ǹgbɛ̃ ye kélɛ. Liyel ǹ gblɔ̌y pɩ névilɛ à pɔ sétáh yõ, nɛ̂-á Kristalɛ. Liyel Mirki kè wɛ̃kɩ nɛ nɛ́gbɩ́-á ǹnɛ, yĩ̂nyõ tõ̀npɩbɩ à wɛ, ǹ tyɩ́ yo névye tyɩ́ nónó náh pɩ Nsyifunɔlɛ, névye náhnáh sɛ̃ ǹ yõ kèkõyṍ, tɛ́ yĩ̂nyõ sɔkɔ, Liyel mɛ gbílkínɔ́ sah ǹ yõ.
