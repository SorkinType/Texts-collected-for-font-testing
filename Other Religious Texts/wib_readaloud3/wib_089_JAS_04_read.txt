Nsyakɩ.
4.
Kwâh nɛ̂ n pɔ n pɔ yékã̂b ànɛ̂ mâllɛ yé wrɔ́ sɔkɔ? Fɔ́kɔ́sõnɔ túkúnì dékè tyi náh n fwo yélɛ di? 
Yé yĩ́ mɛ n wil yî náhnáh tyɩ́, tɛ́ yé náh mɛ sélɛ n wɛ. Kàh pɩ sɛ̃́, nɛ́kókè mɔ dékè yé tɛ̃. Yáh nɛ̂nɛ n yah n kɔ̃, yé náh kélɛ n wɛ. Kàh pɩ sɛ̃́, yékã̂b ànɛ̂ mâl n pyě nɛ n de yé wrɔ́ sɔkɔ. Ǹtɛ, yé náh mɛ n ni Liyel tyɩ́, képah ye, yé náh mɛ kwâh yɔ́lɛ n wɛ. 
Yâh se sɔ̃́ n ni Liyel tyɩ́, sõnɔ pèplɔ náh ńkɛ̃́nɛ nɛ mɛ yé sɔkɔ. Yě n nɛ n ni yé tyítúkúnì dékè tyi pɩ yĩn ólɛ. Képah ye, yé náh kwâhlɛ n wɛ. 
Kwâh nɛ̂ fɔ̀fɩ́npɩbɩ pɩ dùkù mɛ yé n pɩ sɛ̃́? Yé náh pɩ̃ nɛ, wâh á yĩ wil kèkõyṍ tyi tyɩ́, Liyel sépɛ-á álɛ di? Nɛ̂-à kèkõyṍ tyi tɛ̃ kódĩ́nɛ, tɛ̃̀ kɩ pɩ Liyel sépɛlɛ. 
Yé se wɛ nɛ, yõ kɛ̃́-á Liyel wɛy mɛ̀nɛ, nɛ̂-á yo nɛ ńkɛ̃́: «Liyel-á min nɛ̂ mɔ ápi sɔkɔ, ké tyɩ́ nɔ yĩ́ĩ́ ǹ tyɩ́. Képah ye, à mɛ ké tyɩ́ nɛ á nɛ n mɛ péwɔ ó tyɩ́ yõ.» 
Ǹtɛ sépi pól sɔkɔ, à tɔ̃ pèpɔrɩ ǹgbɛ̃nɛ n pɩ ápi tyɩ́, sɔ̃́-á ǹ wɛy kè yo nɛ ńkɛ̃́: «Liyel peke gblɔ̌y gbílkípú yõ, tɛ́ pèpɔrɩlɛ n pɩ gblɔ̌y tíkínmɔpu tyɩ́.» 
Képah ye, yé nɛ n sah Liyel yóré yõ, yè mó sétõ̀ tyɩ́ yahle, képah sɔkɔ, sétõ kɩ to gbɛ mɔ yélɛ. 
Yé nɔ n de Liyellɛ, képah sɔkɔ, ǹ tɔ kɩ nɔ de yélɛ. Tyípêl pɩ́pú, yé yé gbã̀n wilki tyípêl pɩ́nɔ́ sɔkɔ. Tɔ̀tɔ̀ névye, yé yé nnɔ tɔ̃ n sah. 
Yé yɩ̃nɛ yè pɩ̃ nɛ, yáh mɛ pèkè sɔkɔ, yé ké nàhnpɩ̃ gbo. Yé yɛ yé yɩ̀syɔ̀nɩ̀ pɩ gbnílɛ, yé nɩ̀vɩ̀ mó pɩ yèkwɔrɩlɛ. 
Yé yé gblɔ̌y tiki mɔ Tɛ̃̀ ǹgbɛ̃ yah, à kɩ yé yõ yɔ̃. 
Ń yṹnpyé, yé káh wɛ̃ tyɩ́lɛ n yo n yɩkɩ. Nɛ̂-à ǹ yṹnpi tyɩ́ yo yɩkɩ, képah náh pɩ, à lékàh syi à kɔ̃, Liyel tyi kõ̀nsàhnɔ́ tyɩ́ ye tɛ̃̀ yo yɩkɩ, tɛ́ lékàhlɛ kè kɔ̃. Wâh képah pɩ, á se mɛ n kɔ Liyel tyi kõ̀nsàhnɔ́ wɛ̃̀kɩ̀ yõ? Lékàh náh á kélɛ n kɔ̃ di? 
Ǹtɛ, Liyel ó ye sah tyi kõ̀nsàhnɔ́lɛ, ànɛ̂ ǹmɔ ó ye kɩ fɛ̃ lékàhlɛ névi kɔ̃. Ǹmɔ ó ye kɩ tɔ̃ fɛ̃ névi pwah mɔ, képah náh pɩ, kɩ ǹ tyɩ́ yɩkɩ. Nɛ̂ mɛ áwɔ tɛ̃̀nɛ, tɛ́ á lékàhlɛ n syi á yṹnpilɛ n kɔ̃? 
Núkúnúkú, yé noh ń tyɩ́ yɔ̀, yépi nónó-á mɛ n yo nɛ kwɛynɛ, képah náh pɩ, ńsõ̂nɛ yáh kɩ kyɩ kwíl-ǹgbɛ̃ mɛ̀ sɔkɔ, kɩ kyɩ yi núkú pɩ gbɔ̀ yɩ̃́nɔ́ sɔkɔ nɛ́npɔ́ kɩ pɔ wɛ. 
Ǹtɛ, yé se pɩ̃ yé min-á kɩ n mɛ n pi sɔ̃́ ńsõ̂nɛ? Yé mɛ á kɩ nɛ kénfã́hkɩ́ tɔ̃́ɔ̃tɔ̃́ɔ̃ yɔ́ sɔ̃́, nɛ̂-á ńkɛ̃́nɛ n mɔ nɛ́npɔ́ swɔ, ǹ tɛ́ n karkɩ. 
Yé pyě tɔ̃ sõ̀ yɩ̃nɛ yè yo nɛ, Tɛ̃̀ ǹgbɛ̃-à kɩ fɛ̃ syi, yáh kɩ tɛ̃ min sɔkɔ, tɛ́ kɩ tir mɛ̀ pɩ, képah náh pɩ tir mɛ̀ pɩ. 
Ǹtɛ, yè n pyě yé gblɔ̌ynɛ n gbilki yé vyãh wɛy sɔkɔ. Kénɛ gblɔ̌y gbílkínɔ́ dùkù yɔ́ ńtɛ̃ náh plɛ. 
Sépi pól sɔkɔ, névi nɛ̂-à pèpɔrɩ pɩ́nɔ́ yah wɛ, tɛ́ tɛ̃̀ náh mɛ kélɛ n pɩ, tyípékè ye tɛ̃̀ n pɩ.
