Korɛ̃nti tãn.
16.
À yuku pɔ́ mɔ́rɩ́ wɔ tyɩ́ tĩ̀nnɛ, nɛ̂-á kɩ Yerusalɛmɩ kwil Yesu yõ sɛ̃́pú yohnɩ n mɔ, máh vìnmɔnɔlɛ Kalasi mara Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ kɔ̃ nɛ pè kè pɩ sɔ̃́, yépi tɔ kè pɩ sɛ̃́ntɛ̃̀nɛ. 
Dimansyi sõnɔ́ póllɛ, nɛ̂ gbõ̀-à nɔ nɛ̂nɛ, à kè wilki n kɛ̃kɩ n sah gbɔ̀pɔlɛ. Yé káh ńmɔ pɔ́nɔ́ ólɛ n syɩkɩ pɔ́ mɔ́rɩ́ mɔ́nɔ́lɛ. 
Képah sɔkɔ, mâh pɔ pɔ, mé kɩ sɛ́bɛ́ynɔ́ wãrkɩ tɛ́ kɩ sépi ànɛ̂ pɔ́ nɔ́pilɛ yé névye yáhnwìlkìnsàhnɔ́ gbõ̀ dahbɩ, pè kɔ̃ pè kɩ kyɩ sè kɔ̃ Yerusalɛmɩ sɔkɔ. 
Kàh kɩ se mɛ nɔ ńmɔ tɔ kyɩ́nɔ́ tyɩ́, á kɩ wɛ̃̀kɩ̀ pɩ núkúlɛ. 
Mé kɩ n pɔ n pi yé tĩ̀nnɛ. Máh kɩ n pi, mé kɩ sɔkɔ Masedonɩ maralɛ. Képah ye mé kɩ kah tɛ́ kɩ de yé tĩ̀nnɛ. 
Máh kɩ pɔ, kè kɩ fɛ̃ pɩ mé kɩ sèpĩ̀n yísyɔ́ pɩ yé tyɩ́. Gbɔ̀kénpɩ mé kɩ kwɔ̀vɩrɩ gbĩ́ pól ńtɛ̃ pɩ n pi yé tyɩ́. Képah sɔkɔ, yé kɩ fɛ̃ yòhnɩ̀nmɔnɔ pɩ ń tyɩ́ ń kɔ̃ mé kɩ yɔ̃ sɔkɔ ń kɔ́lɔ́lɛ nɛ́npɔ́. 
Kèpàhpɩ́kɩ̀ sɔkɔ, mé náh mɛ ké tyɩ́ nɛ mé n pɔ n yah n sɔkɔ yé tyɩ́ yĩ̂nyõ. Tɛ̃̀ ǹgbɛ̃-à kɩ fɛ̃ syi, mé kɩ wũ kɩ wɔ̂ yɔ́ pɩ yélɛ. 
Ń fwópú sénsɔ̃ mɛ dò Yefɛsɩ kwil sɔkɔ ó, ǹtɛ wɛ̃̀kɩ̀ ǹgbɛ̃ yɔ́ mɛ wɛ yɩkɩ mɔ ń tyɩ́ ń tõ̀ wɛ n pɩ yĩnnɛ nɛ́npɔ́. Képah ye, mé kɩ tɛ̃ nɛ́npɔ́ kɩ syi pɔ nɔ Pãntekɔtɩ syáh syah sõ tyɩ́. 
Timotelɛ, Liyel tõ̀ ye à n pɩ ńmɔ tyɩ́ kɔ̃lɛ. Képah ye wàh kɩ pɔ pɔ yé tĩ̀nnɛ, yé ǹ tɛ̃́nɔ́ plɛ tíkí kɔ̃ kè káh à tɛ̃ yé wrɔ́ sɔkɔ. 
Ǹnɩ̂ yɔ́ ńtɛ̃ káh ǹnɛ n yilɔ dɛ́. Ńmɔ mɛ ǹnɛ n syɩkɩ ànɛ̂ ń yṹnpyé Yesu yõ sɛ̃́púlɛ. Képah ye, wàh kɩ pɔ n yal yé tĩ̀nnɛ, yé yòhnɩ̀nmɔnɔ pɩ ǹ tyɩ́ ǹ kɔ́lɔ́ yĩnnɛ à kɔ̃ à pɔ ń tnɔ̂ yèvɩnɩ sɔkɔ. 
Tɛ́ á yṹnpi Apɔlɔsɩ wɔlɛ, mé sõ̀ kè yah kɔ̃ ǹ tyɩ́ gbáhyɩ̃́ náhnáhlɛ nɛ à kyɩ yé tĩ̀nnɛ ànɛ̂ Yesu yõ sɛ̃́pú tɛ́lɔ́lɛ. Tɛ́ à náh sõ̀ mɛ ké tyɩ́ póllɛ nɛ pé kyɩ yé tĩ̀nnɛ núkúnúkú. Ǹtɛ wrɔ́-à pɔ wɛ ǹ tyɩ́, à kɩ pɔ yé tĩ̀nnɛ sõ yɔ́lɛ. 
Yé káh yé dyɔ mɔ n fwɔ n tɛ̃. Yé su yĩn gbáhkɩ̀ núkú yõ Liyel yõ sɛ̃́nɔ́ sɔkɔ. Yé yé nɩ syãh, yè mó yé tah n sah. 
Yé tyi póllɛ n pɩ tyɩ́nɔnɔ sɔkɔ. 
Ń yṹnpyé, Sitefanasɩ ànɛ̂ ǹ no ye pɩ nɛ́gbrolɛ à pɩ Yesu yõ sɛ̃́púlɛ Akayi mara sɔkɔ. Pè tɔ̃ mɛ pé gblɔ̌y kɔ̃ yĩ́ĩ́ mɛ Yesu yõ sɛ̃́púlɛ n yohnɩ n mɔ. Yé pɩ̃́nɔ́ se yɔ́ mɛ ké sɔkɔ tɛ́. Képah ye, mé kɩ tɔ̃ kè yah kɔ̃ yé tyɩ́ nɛ, 
yé nɛ n sah sɛ̃́nknɔ névye tyɩ́ yõ, à tahlɩ nónó tɔ-á mɛ tã̀n nɔ́pilɛ n pɩ pélɛ mɛ n kwlɔ. 
Sitefanasɩ ànɛ̂ Fɔrtunatusi, ànɛ̂ Asyayikusi mɛ́nɔ́lɛ ńnɛ ásõ̂, kè ń nɩ vɩ yĩ́ĩ́. Pè ye mɛ pɔ yé swɔ tɔkɔ. 
Pè ńmɔ nɩ de à tahlɩ yépi tɔ tyɩ́ tyɩ́. Yé yɩ̃nɛ yè fɛ̃ sɛ̃́nknɔ névye yĩ́gbɩ́lɛ n yo. 
Asi mara Yesu yõ sɛ̃́pú tĩ̀nkìnɔ́ n wɛkɩ yé tyɩ́. Akilasɩ ànɛ̂ Prisili à tahlɩ Yesu yõ sɛ̃́pú nónó-á Liyellɛ n ni n pɔ Akilasɩpi gbô, pé pól n wɛkɩ yé tyɩ́ Tɛ̃̀ ǹgbɛ̃ yĩnnɛ. 
Yé yṹnpyé Yesu yõ sɛ̃́pú nónó pól-á mɛ ńnɛ ásõ̂, pé pól n wɛkɩ yé tyɩ́. Yé pól wɛkɩ wɛ̃ tyɩ́ tyɩ́nɔnɔ wɛy pèpɛynɛ. 
Yé yah, nɛ̂-á mé n wãrkɩ n tɛkɩ n mɔ yé tyɩ́ ń gbõ̀lɛ, kè wɛ yɔ̀: Ńmɔ Pole n wɛkɩ yé tyɩ́. 
Tɛ̃̀ ǹgbɛ̃ Yesu tyɩ́-à nɔ́-ńkɛ̃̂nɛ névi nɛ̂ tyɩ́, mé mɛ ké tyɩ́ nɛ tɛ̃̀ tyɩ́ yɩkɩ. Maranata. Pàh nɛ Maranata-á, ké kɔ́r ye: Tɛ̃̀ ǹgbɛ̃, pɔ! 
Mé mɛ ké tyɩ́ nɛ Tɛ̃̀ ǹgbɛ̃ Yesu yõ̀bènɔrɩ pɩ yé tyɩ́! 
Yé pól tyɩ́ nɔ ń tyɩ́ Tɛ̃̀ ǹgbɛ̃ Yesu Krista yĩnnɛ.
