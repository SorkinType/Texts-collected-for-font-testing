Pole sɛ́bɛ́y tɛ́kɩ́nmɔnɔlɛ Filipi tãn tyɩ́.
1.
Ńmɔ Pole ànɛ̂ Timotelɛ, ápi nónó-á Yesu Krista tõ̀npɩbɩlɛ, ápi ye n tɛkɩ n mɔ sɛ́bɛ́ynɛ Yesu Krista yõ sɛ̃́pú pól tyɩ́ Filipi kwil sɔkɔ, pépi nónó sɔkɔ yísyɔ́-á yah tɛ̃́pú ànɛ̂ yòhnɩ̀nmɔpulɛ, nónó-á yáhntɔkɔnsàhnɔ́lɛ. 
Á kélɛ n yah n kɔ̃ Sú Liyel ànɛ̂ Tɛ̃̀ ǹgbɛ̃ Yesu Krista tyɩ́ nɛ, pè yõ̀bènɔrɩ ànɛ̂ yèvɩnɩnɛ yé kɔ̃. 
Gbĩ́mɛgbĩ́ mâh sõ yé tyɩ́ sɔkɔ, mè n wɛkɩ Liyel tyɩ́. 
Gbĩ́mɛgbĩ́ mě nírílɛ n pɩ yé pól tyɩ́ nɩ̀vɩ̀nɩ̀ sɔkɔ, 
yáh sɔ̃́npɩ́lɛ pɩ ńmɔ yòhnɩ̀nmɔpulɛ Nɩ̀vɩ̀nɩ̀ tir yónɔ́ sɔkɔ, à tɔkɔ ké tyah sõ duku tyɩ́, pópó à pɔ syɩ núkúnúkú tyɩ́. 
Ń sɛ̃́nyĩnɔ mɛ ké yõ sɔkɔ nɛ, Liyel nɛ̂-á tyah tõ̀ pèpɛy mɛ̀nɛ yé nnɔ sɔkɔ, wǎh kɩ yɔ̃ sɔkɔ kɩ kélɛ n tɔ̃ n sah n pɩ, pópó kɩ pɔ syɩ ké pɩ́ntɛnɔ tyɩ́ Yesu Krista syɩ n pɔ sõlɛ. 
Yé tyɩ́ nɔ yĩ́ĩ́ ń tyɩ́. Képah ye, ké yah ye yé pól tyɩ́ sõnɔlɛ ń sɔkɔ sɛ̃́ntɛ̃̀nɛ. Yépi tɔ vyãh mɛ yõ̀bènɔrɩ mɛ̀ tyɩ́ sɔkɔ, Liyel-á nɛ̂nɛ n pɩ ń tyɩ́ wɔ̀kɔ̀npèkè sɔkɔ, ànɛ̂ máh sõ̀ mɛ Nɩ̀vɩ̀nɩ̀ tirlɛ n yo n wah gbĩ́ nɛ̂nɛ, à kélɛ n wɛ̃kɩ nɛ gbɩ-á kélɛ. 
Liyel ye ńmɔ tàrfyɔ̀lɛ nɛ, Yesu Krista tyɩ́nɔnɔ nɛ̂-á mɛ ńmɔ sɔkɔ, képah-á ń nɩ kɔ̃ kè n yɔrɩ yé tyɩ́lɛ. 
Máh mɛ Liyellɛ n ni yé tyɩ́ tir nɛ̂ tyɩ́ sɔkɔ, képah ye yɔ̀ nɛ: Tyɩ́nɔnɔ nɛ̂-á mɛ yé sɔkɔ, kè nɛ n yuku yahlɛ, yé kɔ̃ yé pɩ lékã́m névyelɛ, ànɛ̂ yɩ̃ pɩ̃́púlɛ. 
Képah sɔkɔ, tyi nónó-á yɩ̃nɛ kal, yé kɩ sépilɛ n yah n tɔkɔ n pnɛ. Képah-à pɩ, yé kɩ mɛ yályál wɛ́nyo-ńsah kɛ̃́nɛ yé tyɩ́ Krista syɩ n pɔ sõlɛ. 
Képah sɔkɔ, yé kɩ tyi póllɛ n pɩ gbɩ wɛ̃̀kɩ̀ yõ Yesu Krista gbõ̀ yõ, Liyel yĩn kɔ̃ kè kɩ pɩ gblò, tɛ́ kɩ n gbilki. 
Ń yṹnpyé, mé mɛ ké tyɩ́ nɛ yé pɩ̃ nɛ, tir nɛ̂-á ńmɔ yĩni tɛ̃, kǎh yòhnɩ̀nmɔnɔ pɩ Nɩ̀vɩ̀nɩ̀ tir yónɔ́ yah yúkúnɔ́ sɔkɔ. 
Yé náh wɛ di, yõ̀tɛ̃̀ dĩ́ tɔ̃́rɩ́ kéte nɛ̂ sɔkɔ-á ńnɛ wɔ̀kɔ̀npèkè sɔkɔ, nɛ́npɔ́ tãn pól ànɛ̂ no tɛ́lɔ́ pól pɩ̃ nɛ, Krista tõ̀ yĩn-á ńnɛ wɔ̀kɔ̀npèkè sɔkɔ. 
Ń yṹnpyé-á ńmɔ wɛ wɔ̀kɔ̀npèkè sɔkɔ sɛ̃́, wãn mɔ pé kan Tɛ̃̀ ǹgbɛ̃ yõ sɛ̃́nɔ́ yõ. Kǎh pɩ sɛ̃́, pé nɩ tɔ̃ gbah syãh, pè Liyel wɛynɛ n yo tíkí kɛ̃́nɛ. 
Gbɩ tɛ̃̀ wɔ ye, túkù mɛ ńmɔ dyɔ yɔ yĩnnɛ Krista tyɩ́lɛ n yo à pé gblɔ̌ynɛ n tɔkɔ n pɩ fṍnfṍ ńmɔlɛ. Ǹtɛ túkù tɛ̃̀ wɔ mɛ kélɛ n yo sõnɔ pèplɔlɛ pé sɔkɔ. 
Tyɩ́nɔnɔ sɔkɔ ye pépi n pɩ képahlɛ. Pè pɩ̃ nɛ Nɩ̀vɩ̀nɩ̀ tir gbɩ yo n wah yĩn-á ńnɛ ásõ̂. 
Ǹtɛ nónó tɛ̃̀ wɔ-á Krista tyɩ́lɛ n yo à ńmɔlɛ n fwo, pépi sõnɔ náh mɛ plɛ. Pè mɛ n sõ nɛ páh kɩ wãn mɔ n pi ńmɔ wɔ̀kɔ̀npèkè sɔkɔ dénɔ́ yèvyãhrɩ yõ. 
Ǹtɛ képah náh mɛ tir yɔ́lɛ ńmɔ tyɩ́. Kè sɔ̃́ pɩ sõnɔ túkù sɔkɔ ò, kè sɔ̃́ pɩ sõnɔ pèpɛy sɔkɔ ò, Krista tyɩ́ tɛ̃̀ wɔ mɛ n yo. Képah ye, kè mɛ nɩ̀vɩ̀nɩ̀nɛ ńmɔ tyɩ́. Tɛ́ ńmɔ pɩ̃ nɛ yépi nírí ànɛ̂ Yesu Krista Mirki yòhnɩ̀nmɔnɔ gbõ̀ yõ, yĩ mɛ̀-á mɛ n pi ńmɔ pwáhnmɔnɔ wɛ̃̀kɩ̀ yõ. Képah ye, ń nɩ kɩ tɔ̃ n vɩ n yuku n pi yahlɛ. 
Máh mɛ nɛ̂ syɩkɩ tɛ̃ kègbɩlɛ, tɛ́ ń gbõ̀ sàhnɔ́ mɛ ké yõ, képah ye nɛ, sen káh ń tɛ̃ ń tyípɩ́nɩ́ yɔ́ ńtɛ̃ sɔkɔ póllɛ. Ńmɔ sɛ̃́nyĩnɔ mɛ ké yõ sɔkɔ núkúnúkú ànɛ̂ gbĩ́mɛgbĩ́ nɛ, mâh se wũ min sɔkɔ ò, mâh se ku ò, képah-á kɩ Kristalɛ n gbilki. 
Yé náh wɛ di, ńmɔ mɛ́nɔ́ yõlɛ min sɔkɔ, Krista gbilki yĩn ye. Tɛ́ kúnɔ́lɛ, yõre wɛ́nɔ́ ye kélɛ ńmɔ tyɩ́. 
Ǹtɛ min sɔkɔ tɛ̃̀nɔ́-à kɩ ńmɔ kɔ̃ mé fɛ̃ tõ̀ wɛ n pɩ n pi yõre-á mɛ nɛ̂ tyɩ́, mé yɩ̃nɛ mè nɛ̂ tɛ̃̀ yah n tɔkɔ wɛ́? 
Ńmɔ mɛ tɔ̀tɔ̀ sõnɔ sɔkɔ. Mé mɛ ké tyɩ́ nɛ mé yal kèkõyṍ mɛ̀ sɔkɔ mé n kyɩ n tɛ̃ Kristalɛ, képah tyɩ́ mɛ kal ń tyɩ́. 
Ǹtɛ ń tɛ̃̀nɔ́lɛ min sɔkɔ, yõ mɛ képah tyɩ́ kal yépi yĩnnɛ. 
Ń sɛ̃́nɔ́ mɛ ké yõ, ànɛ̂ mé pɩ̃ nɛ máh kɩ wũ, kɩ n tɛ̃ n pi yé póllɛ, yé yah yuku yĩn ànɛ̂ yé nɩ̀vɩ̀nɩ̀ wɛ yĩnnɛ Liyel yõ sɛ̃́nɔ́ sɔkɔ. 
Képah sɔkɔ, ńmɔ syɩ́npɔnɔlɛ yé tnɔ̂, kè kɩ yé kɔ̃ yé kɩ Yesu Kristalɛ n gbah n gbilki. 
Nɛ̂ tɛ̃̀-á tah kal, yé yé minnɛ n pɩ à yɩ̃nɛ Krista Nɩ̀vɩ̀nɩ̀ tir yónɔ́lɛ. Képah sɔkɔ, máh kɩ sɔ̃́ pɔ wil yé tyɩ́ ò, máh kɩ sɔ̃́ pɔ n wìl-ǹkɛ̃̂nɛ yé tyɩ́ ò, mé kɩ mɛ yé tyɩ́lɛ n nohnɩ nɛ yáh mɛ su yĩn gbáhkɩ̀ núkú yõ sõnɔ núkúlɛ yé sɔkɔ, tɛ́ mɛ so wɛ̃ tyɩ́ à gbah tɛ̃ Nɩ̀vɩ̀nɩ̀ tir yõ sɛ̃́nɔ́ yah yúkúnɔ́ tyɩ́ sɔkɔ. 
Yé káh yɛ yé sépɛ́bɩ́ kɔ̃ pè yé wore tir yɔ́ ńtɛ̃ sɔkɔ. Képah sɔkɔ ye pè kɩ pɩ̃ nɛ pé tyɩ́ yɩkɩ yɩkɩ wɛ̃̀kɩ̀ yõ-á pélɛ, tɛ́ yépi-á mɛ pwáhnmɔnɔ wɛ̃̀kɩ̀ yõ. Liyel ye n pɩ sépi póllɛ. 
Yé náh wɛ di, Liyel ye yõ̀bènɔrɩ pɩ yé tyɩ́ yé kɔ̃ yé pɩ Krista yõ sɛ̃́púlɛ. Tɛ́ kè náh yĩn ǹ yõ sɛ̃́nɔ́ ó tyɩ́ dɛ́, yé kɩ tɔ̃ yèvyãhrɩ tɔ wɛ n pɩ n pi ǹmɔ yĩnnɛ. 
Yépi ye mɛ mɛ́kɩ̀lɛ n fwo Nɩ̀vɩ̀nɩ̀ tir yah yúkúnɔ́ tyɩ́ sɔkɔ ńmɔ tyɩ́ kɔ̃lɛ, sɔ̃́-á lésõ yé ń wɛ mé mɛ ké mɛ́kɩ̀lɛ n fwo. Tɛ́ ńmɔ mɛ yɔ̃ sɔkɔ sɛ̃́ kélɛ. Yépi ńtɛ̃ pɩ̃́nɔ́ mɛ ké sɔkɔ ó.
