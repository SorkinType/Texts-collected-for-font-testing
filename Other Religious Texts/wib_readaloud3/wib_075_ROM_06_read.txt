Wrome tãn.
6.
Sɔ̃́ mɛ ápi kɩ yo yɔ̀ wɛ́? Á se yɩ̃nɛ à yɔ̃ n sɔkɔ tyípêl pɩ́nɔ́ sɔkɔ, Liyel kɔ̃ à yõ̀bènɔrɩlɛ n gbah n pɩ á tyɩ́? 
Ɔ̃́ɔ̃ dɛ́! Ápi á náh vi tyípêl pɩ́nɔ́ kɔ̃ á kɩ nɛ á sɔ̃́ ku. Tɛ́ á kɩ pɩ sɔ̃́ tɛ́ kɩ tɔ̃ pɔ á minnɛ n pɩ tyípêl pɩ́nɔ́ sɔkɔ? 
Yé náh pɩ̃ nɛ, ápi nónó pól-á wel wilki ni sɔkɔ à kwrɔ mɔ Yesu Kristalɛ, á wèlnwìlkìnɔ́ mɛ̀-á mɛ á kɩ nɛ kúnɔ́ sɔ̃́ ápi tɔ ku ǹnɛ wɛ̃ tyɩ́ di? 
Wèlnwìlkìnɔ́ mɛ̀ sɔkɔ, ápi tɔ ye ku ǹnɛ wɛ̃ tyɩ́, tɛ́ yɩ̃ ǹnɛ wɛ̃ tyɩ́. Tɛ́ ǹ lékó-á yilki mɔ Sú Liyel tɛ̃́nwɛnɔ ǹgbɛ̃ gbõ̀ yõ sɔ̃́, sɛ̃́ ye ápi tɔ min pɩ dùkù vi mɔ pɩ sã́nnɛ. 
Yé náh wɛ di, áyáh fyɔ̀ kwrɔ mɔ ǹnɛ à kũ dùkù núkú ku, ápi tɔ lékyɩ̂ kɩ pɔ n yilki n mɔ n pi ǹmɔ tyɩ́ kɔ̃lɛ. 
Á yɩ̃nɛ à pɩ̃ nɛ áyáh sõ̀ mɛ n pɩ dùkù nɛ̂nɛ ǹgbòlɛ, áyáh képah kwrɔ mɔ syɩ Kristalɛ dahnɩ yõ à ké tyɩ́ yɩkɩ, tɛ́ wɛ wil tyípékè yɔ̃rɩ sɔkɔ. 
Yé náh wɛ di, névi nɛ̂-à ku, tɛ̃̀ vyãh n dɛ tyípékè pɩ́nɔ́ sɔkɔ. 
Tɛ́ áyâh fyɔ̀ ku ànɛ̂ Kristalɛ wɛ̃ tyɩ́, á sɛ̃́nɔ́ mɛ ké yõ nɛ áyáh kɩ á min pɩ ǹnɛ wɛ̃ tyɩ́. 
Ápi pɩ̃ nɛ Krista lékó-á yilki mɔ, à náh tɔ̃ n ku n yah n pi, kũ náh tɔ̃ tɛ̃ wɛ n pi ǹ tyɩ́. 
Wǎh kũ nɛ̂ ku yɔ̀, tyípêl tyɩ́ yɩkɩ yĩn ye kṹmm à ku. Ǹtɛ núkúnúkú wǎh mɛ min sɔkɔ, à mɛ ǹ minnɛ n pɩ Liyel tyɩ́ yõ. 
Sɛ̃́ ó ye, yépi tɔ yɩ̃nɛ yè pɩ̃ nɛ yáh yé náh vi tyípêl pɩ́nɔ́ kɔ̃ á kɩ nɛ yé sɔ̃́ ku, tɛ́ mɛ min sɔkɔ Liyel tyɩ́ yõ yé wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Yesu Kristalɛ. 
Gbĩ́ nɛ̂-á ye mɛ min sɔkɔ kèkõyṍ mɛ̀ sɔkɔ yɔ̀, yé káh tɔ̃ yɛ tyípékè pɔ fyãh yé yõ, kè pɔ yélɛ n mɔ yé dyɔ dyɔ tyi pɩ́nɔ́ sɔkɔ dɛ́. 
Yé káh tɔ̃ yé wil gbɛ̃́nsê tã̀ntã̀nnɛ tyípêllɛ n pɩ á kɩ nɛ tyípékè sɔ̃́ n tɛ̃ yé yahlɛ dɛ́. Ǹtɛ yé yé gblɔ̌ynɛ Liyel kɔ̃. Yé mɛ á kɩ nɛ yé sɔ̃́ ku tɛ́ yé lékyɩ̂ mɛ yilki mɔ. Yé yé wil gbɛ̃́nsê tã̀ntã̀n pól fĩn mɔ Liyel kɔ̃ sè pɩ tyígblɔ́ pɩ pɩ tõ̀npɩ̀yîlɛ. 
Tyípékè náh tɔ̃ n dɔkɔ n tɛ̃ n pi yé yõ póllɛ. Yé náh tɔ̃ mɛ Moyisi tyi kõ̀nsàhnɔ́ gbõ̀ sõ̀ sɔkɔ dò. Liyel-á yõ̀bènɔrɩ nɛ̂nɛ n pɩ yé tyɩ́, képah gbõ̀ sõ̀ sɔkɔ ye yélɛ. 
Á yɩ̃nɛ à yo sɔ̃́ yɔ̀ wɛ́? Áyáh fyɔ̀ mɛ yõ̀bènɔrɩ gbõ̀ sõ̀ sɔkɔ, tɛ́ á náh tɔ̃ mɛ Moyisi tyi kõ̀nsàhnɔ́ gbõ̀ sõ̀ sɔkɔ, képah se yɔ́ n wɛ̃kɩ nɛ tyípêl pɩ́nɔ́ wɛ̃̀kɩ̀-á yɩkɩ mɔ á tyɩ́? Ɔ̃́ɔ̃ dɛ́! 
Yé náh pɩ̃ nɛ ńkɛ̃́ yâh tõ̀lɛ n pɩ ǹnɩ̂ yɔ́ tyɩ́ yɔ̃bɩ sɔ̃́ yélɛ, tɛ́ n sah ǹ tyɩ́ yõ, yè n pɩ tɛ̃̀ yɔ̃̀lɛ kègbɩ yõ di? Sɛ̃́ ó ye, yâh n sah tyípékè tyɩ́ yõ, yè n pɩ ké yɔ̃bɩlɛ, ànɛ̂ kè n pɔ kũnɛ. Tɛ́ yâh n sah Liyel tyɩ́ yõ, yè n pɩ ǹ yɔ̃bɩlɛ, ànɛ̂ kè n pɔ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́lɛ. 
Yé lésõ mɛ tyípékè yɔ̃rɩ sɔkɔ. Ǹtɛ yáh kwɔ́nɔ́ nɛ̂ wɛ, képah gbõ̀ yõ, núkúnúkú yé mɛ n sah Liyel yóré yõ yé nnɔ póllɛ. Kǎh pɩ sɛ̃́, á ye wɛkɩ Liyel tyɩ́. 
Yé wil tyípékè pɩ́nɔ́ sɔkɔ, tɛ́ de gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ sɔkɔ. 
Yépi tyi yah yáhnkwlɔnɔ ye ń kɔ̃ mé tyi nɔ́pilɛ n yo à yɩ̃nɛ névye pɩ dùkùlɛ yɔ̀. Ǹgbòlɛ lésõ, yáh yé wil gbɛ̃́nsê tã̀ntã̀n póllɛ põ̌põ̌ ànɛ̂ tyítúkúnìnɛ n pɩ, à Liyel dêllɛ n pɩ sɔ̃́; núkúnúkú, yé sélɛ gbɩ wɛ̃̀kɩ̀lɛ n kɔ, yè n nɛ n mɛ wìlkìnsàhnɔ́lɛ Liyel tyɩ́ yõ. 
Yáh lésõ mɛ tyípékè yɔ̃rɩ sɔkɔ gbĩ́ nɛ̂nɛ, nɛy náh mɛ yélɛ n kyɩ̃ n mɔ nɛ yé gbɩ wɛ̃̀kɩ̀lɛ n kɔ. 
Kénɛ tyi nónó-á lésõ yé n pɩ, kwâh nɛ̂ mɛ yé wɛ sɔkɔ sé sɔkɔ, à wil sen tyɩ́ sɔkɔ, nɛ̂-á mɛ yé yõ kwɛy mɛ̀ sɔkɔ? Sépi tyɛ́-ńsahlɛ kũ ye dò. 
Ǹtɛ núkúnúkú, Liyel-á yé wilki tyípékè yɔ̃rɩ sɔkɔ, yé mɛ tõ̀lɛ n pɩ ǹmɔ tyɩ́, nɛ̂-á kɩ yé kɔ̃ yé kɩ mɛ wìlkìnsàhnɔ́lɛ ǹ tyɩ́ yõ, tɛ́ kɩ tyɛ́-ńkɛ̃̂ min wɛ n pi ké tyɛ́-ńsah. 
Yé náh wɛ di, tyípékè lédyáhnɩ́nɛ, kũ ye. Ǹtɛ Liyel-á kwâh nɛ̂nɛ n yɩ̃ névyelɛ n kɔ̃, tyɛ́-ńkɛ̃̂ min ye. À képahlɛ névyelɛ n kɔ̃ pé wrɔ́ kwrɔ́nmɔnɔ gbõ̀ yõ Yesu Kristalɛ, ǹmɔ nɛ̂-á ápi Tɛ̃̀ ǹgbɛ̃nɛ.
