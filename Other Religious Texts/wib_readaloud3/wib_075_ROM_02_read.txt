Wrome tãn.
2.
Áwɔ nɛ̂-á lékàhlɛ no tɛ́lɔ́lɛ n kɔ̃, wáh kɩ sɔ̃́ mɛ nɛ́dúkú nɛ̂nɛ, lénɔ̀ wɛ́-ńsah náh mɛ á tyɩ́. Á náh wɛ di, wáh lékàhlɛ no tɛ́lɔ́lɛ n kɔ̃, lékàh ye á á gblɔ̌ynɛ n kɔ̃, áwɔ tɔ-á sɔ̃́npɩ́lɛ mɛ n pɩ pé tyɩ́ kɔ̃lɛ, áwɔ nɛ̂-á lékàhlɛ pélɛ n kɔ̃. 
Ápi pɩ̃ nɛ ńkɛ̃́ nónó-á kénɛ tyi dùkùlɛ n pɩ, Liyel-á kɩ pè tuke n pi à yɩ̃nɛ gbɩ wɛ̃̀kɩ̀lɛ. 
Áwɔ nɛ̂-á lékàhlɛ kénɛ tyi pɩ́púlɛ n kɔ̃, tɛ́ mó tɔ̃ mɛ sɛ̃́nknɔlɛ n pɩ, á se mɛ n sõ nɛ wáh kɩ n pwah n pi Liyel tukey sɔkɔ? 
Képah náh pɩ, Liyel pèpɔrɩ ǹgbɛ̃, ànɛ̂ ǹ fɔ̀kɔ̀pènɔrɩ, ànɛ̂ ǹ sṍmnɔ́ se á yahle mɔ́nkõ̀nɔ́lɛ? Á náh pɩ̃ nɛ Liyel pèpɔrɩ mɛ̀-á kɩ ǹ kɔ̃ á kɩ á nɩ vi mɔ kɩ tyípêl pɩ́nɔ́ yal mɔ di? 
Ǹtɛ á níkísekɔ́kɔ́ ànɛ̂ á nɩ táhntũnɔ yĩnnɛ, á mɛ Liyel plɔlɛ n gbah n yɔrɩ n mɔ á gblɔ̌y yõ, nɛ̂-á kɩ n pɔ n pi ǹ fɔ̀kɔ̀vyã̀hnɩ̀ sõlɛ, sõ nɛ̂-á à kɩ névye tuke n pi à yɩ̃nɛ gbɩ wɛ̃̀kɩ̀lɛ. 
À kɩ nɛ́núkù núkú pól dyah à yɩ̃nɛ pé tyípɩ́nɩ̀nɛ. 
Nónó-á mɛ gbah tɛ̃ tyípéplɔlɛ n pɩ, nɛ pé kyɩ yõ yɔ̃̀nɔ́ wɛ, gbílkínɔ́ mó nɛ n sah pé yõ, ànɛ̂ pé mó tyɛ́-ńkɛ̃̂ min wɛ, à kɩ tyɛ́-ńkɛ̃̂ minnɛ pépi kɔ̃. 
Tɛ́ névye nónó-á yahle sàhnɔ́lɛ ǹ tyɩ́ yõ, tɛ́ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́ yahle tɛ́ pèkè pɩ́nɔ́ sõ̀ tɛ̃, à kɩ ǹ fɔ̀kɔ̀vyã̀hnɩ̀ pépi wɛ̃kɩ, tɛ́ kɩ fṍhnɔ́ sah pé yõ. 
Yèvyãhrɩ ànɛ̂ tíkí képékè kɩ pɩ tyítúkúnì pɩ́pú pól tyɩ́lɛ. Kè kɩ tyah Nsyifunɔ ǹgbò tyɩ́, tɛ́ kɩ pɔ yĩni tɛ̃ névye tyɩ́ nónó náh pɩ Nsyifunɔlɛ. 
Ǹtɛ gbílkínɔ́, ànɛ̂ yõ yɔ̃̀nɔ́, ànɛ̂ yèvɩnɩ kɩ pɩ tyípéplɔ pɩ́pú nɛ́núkù núkú pól tyɩ́lɛ. Kè kɩ tyah Nsyifunɔ ǹgbò tyɩ́, tɛ́ kɩ pɔ yĩni tɛ̃ névye tyɩ́ nónó náh pɩ Nsyifunɔlɛ. 
Yé náh wɛ di, Liyel náh névyelɛ n wah. 
Névye nónó pól-á pè náh Moyisi tyi kõ̀nsàhnɔ́lɛ n pnɛ tɛ́ tyípêl pɩ, kénɛ tyi kõ̀nsàhnɔ́ vyãh náh n wɛ n pi pépi tyɩ́ yɩ́kɩ́nɔ́ sɔkɔ. Ǹtɛ nónó pól-á tyípêl pɩ tɛ́ mɛ Moyisi tyi kõ̀nsàhnɔ́lɛ n pnɛ, pépi kɩ n tuke n pi à yɩ̃nɛ képahlɛ. 
Moyisi tyi kõ̀nsàhnɔ́ tyíyónɔ́ nóhpú náh Liyel n yah nɛ́gblɔ́lɛ dɛ́. Ké tyíyónɔ́ pɩ́pú ye. 
Névye nónó-á pè náh pɩ Nsyifunɔlɛ, pépi náh Moyisi tyi kõ̀nsàhnɔ́lɛ n pnɛ, tɛ́ pé kɔ dùkù mó tɔ̃ pɔ yɩ̃nɛ Moyisi tyi kõ̀nsàhnɔ́ yórélɛ. Kǎh pɩ sɛ̃́, képah n wɛ̃kɩ nɛ pǎh pèpɛynɛ n pnɛ pé gblɔ̌y, Moyisi tyi kõ̀nsàhnɔ́ kɛ̃́nɛ pé tyɩ́. 
Pè mɛ kélɛ n wɛ̃kɩ képah sɔkɔ nɛ, Moyisi tyi kõ̀nsàhnɔ́-á nɛ̂ yo nɛ kè nɛ n pɩ, kǎh mɛ pé nnɔ sɔkɔ. Pé sõnɔ sɔkɔ, kèyǎhlɛ pè n nɛ n mɛ lékàhlɛ pé gblɔ̌ynɛ n kɔ̃, kèyǎhlɛ ǹ tɛ n nɛ n mɛ gbɩlɛ pé gblɔ̌ynɛ n kɔ̃. 
Nɩ̀vɩ̀nɩ̀ tir nɛ̂-á ńmɔ n yo, à yɩ̃nɛ képahlɛ, Liyel kɩ syɔ̃ Yesu Krista tyɩ́ kɩ névye tuke n pi pé sètĩ̀n sõ̀ tyi pól tyɩ́ sɔkɔ. Lésõlɛ, tyi nɔ́pi kɩ wilki mɔ kwèy. 
Yépi nónó-á nɛ Nsyifunɔ-á yélɛ, yé mɛ yé gbõ̀ sah Moyisi tyi kõ̀nsàhnɔ́ yõ, tɛ́ yé gblɔ̌ynɛ n yɔ̃ n sah nɛ Liyel-á mɛ yépilɛ. 
Tɛ́ Liyel dyɔ yĩ́ĩ́-á nɛ̂nɛ, yé kélɛ n pnɛ kyɔ́! Yáh kwɔ́nɔ́ nɛ̂ wɛ Moyisi tyi kõ̀nsàhnɔ́ sɔkɔ, képah gbõ̀ yõ, yé pèpɛynɛ n pnɛ dò. 
Kénɛ kwɔ́nɔ́ gbõ̀ yõ, yé nɛ yé pnɛ̀nɔ́-á syɩ ké yĩ̀n tyɩ́, ànɛ̂ yáh gbɩ pɩ̃ dal mɔ, tɛ́ núkú yuku yɩ̃́tũbi yahlɛ n tɛ̃, tɛ́ mɛ kyòolɛ n mɔ fɛ̃́ntũ̂ sɔkɔ névye tyɩ́ yé yo kɛ̀? Yé mɛ vìnmɔnɔlɛ lékã́m kɛ̃́ névyelɛ n kɔ̃, tɛ́ mɛ wɛ̃́npì yrɔ̃́lɛ n kwɔ yé yo kɛ̀? 
Yé mɛ no tɛ́lɔ́lɛ n kwɔ, tɛ́ yé náh yé gblɔ̌y wɔlɛ n kwɔ kɛ̀? Yé mɛ névyelɛ n kwɔ nɛ pè káh yɩ̃belɛ n yɩ̃, tɛ́ yépi wɔ n yɩ̃ kɛ̀? 
Yé nɛ fɔ̀fɩ́kɩ̀ káh nɛ n pɩ, tɛ́ yépi wɔ kélɛ n pɩ kɛ̀? Yé nɛ fal fal tã̀n tyɩ́ náh nɔ yé tyɩ́ póllɛ, tɛ́ mó tɔ̃ tã̀n vɩ̀ yîlɛ n yɩ̃ n tɔkɔlɩ kɛ̀? 
Yé mɛ yé gblɔ̌ynɛ n yɔ̃ n sah nɛ yáh Moyisi tyi kõ̀nsàhnɔ́lɛ n pnɛ, tɛ́ mó tɔ̃ mɛ sé dêllɛ n pɩ. Liyel náh yé n yilɔ yɔ̀ di? 
Yé náh wɛ di, kè mɛ wã́rkɩ́nsàhnɔ́lɛ Liyel wɛy sɛ́bɛ́y sɔkɔ nɛ: «Yépi Nsyifunɔ ye yìkìnì tɛ́lɔ́ kɔ̃ pè Liyel yĩnnɛ n yɩkɩ.» 
Gbɩ tɛ̃̀ wɔ ye, yõre mɛ kõ̀kè tyɩ́ yâh mɛ fɛ̃ n kɔ Moyisi tyi kõ̀nsàhnɔ́ wã̀l yõ. Ǹtɛ yâh mɛ sé dêllɛ n pɩ, yé kõ̀nɔ́ pɩ nɔ̀nɔ́lɛ. 
Kõ̀-ǹkɛ̃̂ névi nɛ̂ tɛ̃̀-à fyɔ̀ n kɔ Moyisi tyi kõ̀nsàhnɔ́ wã̀l yõ, Liyel kɩ tɛ̃̀ mɔ kõ kènkõbe sɔkɔ. 
Yépi Nsyifunɔlɛ, Moyisi tyi kõ̀nsàhnɔ́ wã́rkɩ́nsàhnɔ́ mɛ yé tyɩ́, ànɛ̂ yé se mó tɔ̃ kõ̀kè tɔ kõ. Tɛ́ képah fyé yõ, yé mɛ Moyisi tyi kõ̀nsàhnɔ́ dêllɛ n pɩ. Kǎh pɩ sɛ̃́, kõ̀-ǹkɛ̃̂ névye nónó-á mɛ Moyisi tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ, pépi ye kɩ pɔ lékàhlɛ yé kɔ̃ n pi. 
Yé náh wɛ di, névi kõ̀nɔ́ náh n pɩ tɛ̃̀nɛ Nsyifulɛ. Kõ̀kè kõ̀nɔ́ nótɔ́lɩ́ tɔ náh n pɩ névilɛ Nsyifulɛ. 
Liyel-à névi nɛ̂ nɩ vi mɔ, tɛ̃̀ wɔ ye n pɩ Nsyifu yèsyɩ̀kɩ̀lɛ. Kõ̀kè yèsyɩ̀kɩ̀ tɔlɛ, Liyel Mirki ye n pɩ kélɛ nɩ sɔkɔ, Moyisi tyi kõ̀nsàhnɔ́ náh. Nsyifu yèsyɩ̀kɩ̀lɛ, Liyel ye n sõ ǹnɛ, névi náh.
