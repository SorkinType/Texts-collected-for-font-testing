Kalatɩ tãn.
5.
Krista mɛ ké tyɩ́ nɛ á nɛ n mɛ á gblɔ̌y yõ sɔkɔ, képah ye à kɔ̃ à á wilki yɔ̃rɩ sɔkɔ. Tɛ́ yé yé tah n sah, yé káh n tɔ̃ yé syɩ n de yɔ̃rɩ sɔkɔ. 
Ńmɔ Polelɛ, yé noh ń tyɩ́ yɔ̀: Yáh kɩ kõ̀kè kõ, Krista náh tir yɔ́ pɛkɩ n pi yé tyɩ́. 
Mé kélɛ n yo yé tyɩ́ ńtɔ̃ nɛ, névi nɛ̂ tɛ̃̀-à kɩ fyɔ̀ kõ̀kè kõ, wǎh kɩ Moyisi tyi kõ̀nsàhnɔ́ pól wã̀llɛ n kɔ káhkáh. 
Yépi nónó-á mɛ n yah n kɔ̃ nɛ yé pɩ nɛ́gblɔ́lɛ Liyel yah Moyisi tyi kõ̀nsàhnɔ́ wã̀l kɔ́lɔ́ gbõ̀ yõ, yé gblɔ̌y ye yé gbɛ Krista náh. Yé vyãh náh tɔ̃ mɛ Liyel pèpɔrɩ tyɩ́ sɔkɔ. 
Ápi tɛ̃̀ wɔlɛ, á mɛ ké tyɩ́ nɛ á pɩ nɛ́gblɔ́lɛ Liyel yah sɔkɔ ǹ yõ sɛ̃́nɔ́ gbõ̀ yõ. Képah sõnɔ ye Liyel Mirki dahbɩ ápi sɔkɔ, tɛ́ á mɛ kélɛ n dɛkɩ náhnáh. 
Yé náh wɛ di, névi nɛ̂ wrɔ́ tɛ̃̀-à fyɔ̀ mɛ kwrɔ́nmɔnɔlɛ Kristalɛ, tɛ̃̀ kõ̀kè kõ ò, tɛ̃̀ náh kõ̀kè kõ ò, tir yɔ́ náh képahlɛ. Nɛ̂-á ké yõlɛ, Liyel yõ sɛ̃́nɔ́ yɩ̃nɛ kè nɛ n wɛ̃kɩ tyi pɩ dùkù sɔkɔ, tyɩ́nɔnɔ sɔkɔ. 
Yé lésõ mɛ n yuku yahlɛ pallɛ. Ǹtɛ nɛ̂ tɛ̃̀ ó pɔ kõ yé yahlɛ yé kɔ̃ yé gbɩ wɛ̃̀kɩ̀ fĩn mɔ wɛ́? 
Liyel nɛ̂-á yé ye pɩ ǹ tyɩ́lɛ, ǹmɔ tɛ̃̀ wɔ náh se yɩkɩ yé yõlɛ tɛ́. 
Pè náh nɛ kɔ́kɔ́nɔ́ kwéy ó-á n dɔkɔ mífã́hkɩ̀ póllɛ di? 
Tɛ́ sépinɔ pól sɔkɔ, ńmɔ gbõ̀ sàhnɔ́ mɛ yé yõ Tɛ̃̀ ǹgbɛ̃ gbõ̀ yõ nɛ, yé náh n fɛ̃ n syi káhkɩ́ vi n pi. Ǹtɛ névi nɛ̂-á n yɩkɩ yé yõlɛ, tɛ̃̀-à ké se mɛ nɛ́dúkú nɛ̂nɛ, fṍhnɔ́ kɩ n sah n pi tɛ̃̀ yõ. 
Ńmɔ tɛ̃̀ wɔlɛ ń yṹnpyé, kàh sõ̀ mɛ pɩ nɛ kõ̀kè kõ̀nɔ́ tyɩ́ sɔ̃́ tɔ̃ mé mɛ gbah tɛ̃ n yo nɛ kè nɛ n pɩ, yõ se náhkɩ mɛ ké tyɩ́ pè ńnɛ nɛ n fõh ńtɔ̃? Kègbɩ yõ, képah tyɩ́-à tɔ̃ sõ̀ nɛ mé mɛ gbah tɛ̃ sɛ̃́, Krista dahnɩ yõ syɩ́nɔ́ se náhkɩ mɛ yõ yɩ́kɩ́nɔ́lɛ ǹnɩ̂ yɔ́ ńtɛ̃ tyɩ́ ńtɔ̃? 
Nónó-á n dahbɩ fɛ̃̀kɩ̀nɔ́lɛ yé wrɔ́, pè káh yĩn kõ̀kè kõ̀nɔ́ ó tyɩ́ dɛ́, pè pé tyɩ́nɔ́ kɔ̃lɩ n to pnɛ́pnɛ́ ńtɛ̃nɛ kɛ̀. 
Ń yṹnpyé, gblɔ̌y yõ wɛ n tɛ̃ yĩn ye Liyel ye yélɛ. Ǹtɛ yé káh núkú sah képah yõ, yè mó yé dyɔ dyɔ tyilɛ n pɩ dɛ́. Yé wɛ̃nɛ nɛ n yohnɩ n mɔ tyɩ́nɔnɔ sɔkɔ. 
Yé náh wɛ di, Moyisi tyi kõ̀nsàhnɔ́ tyi pól ye tir kõ̀nsàhnɔ́ núkú mɛ̀ tuke mɔ yo kɛ núkúlɛ nɛ ńkɛ̃́: «Á ní-ò tyɩ́ yɩ̃nɛ kè nɔ á tyɩ́ á gblɔ̌y tyɩ́ kɔ̃lɛ.» 
Ǹtɛ yáh wɛ̃nɛ n nwãhkɩ, tɛ́ wɛ̃nɛ n kɔ sɛ̃́ à; yáh kɩ yé gblɔ̌y tɛ̃́-ńkɛ̃̂nɛ, yé kɩ pɔ wɛ̃ tyɩ́ yɩkɩ. 
Yé noh ń tyɩ́ yɔ̀: Yé yɛ Liyel Mirki yé yahlɛ n tɛ̃. Képah sɔkɔ, yé náh tɔ̃ yé gblɔ̌y dyɔ yĩ́ĩ́ tyi pɩ n pi. 
Yé náh wɛ di, névi gblɔ̌y dyɔ yĩ́ĩ́ tyilɛ, Liyel Mirki dêl tyi ye sépilɛ. Liyel Mirki tɔ dyɔ yĩ́ĩ́ tyi-á mó tyi nónónɛ, névi dyɔ yĩ́ĩ́ tyi náh sépilɛ. Tɛ́ sépi tɛ̃̀-á pɩ wɛ̃ sépɛ́bɩ́lɛ sɛ̃́, képah ye yé kɔ̃ yé náh yé gblɔ̌y dyɔ yĩ́ĩ́ tyilɛ n pɩ n pnɛ. 
Ǹtɛ Liyel Mirki-à kɩ fyɔ̀ yé yahlɛ n tɛ̃, tyi kõ̀nsàhnɔ́ vyãh kɩ dɛ yé tyɩ́ sɔkɔ. 
Ǹtɛ á gblɔ̌y dyɔ yĩ́ĩ́ tyi-á tyi nónónɛ, á pól sélɛ n pnɛ. Sè wɛ yɔ̀: fɔ̀fɩ́kɩ̀ pɩ́nɔ́lɛ, põ̌põ̌ pɩ́nɔ́lɛ, ànɛ̂ prɔ̀prɔ̀ pɩ́nɔ́lɛ, 
ànɛ̂ sãh fálnɔ́lɛ, ànɛ̂ klɔ́ pɩ́nɔ́lɛ, ànɛ̂ nɛ́pékè pɩ́nɔ́lɛ, ànɛ̂ mɛ́kɩ̀ pɩ́nɔ́lɛ, ànɛ̂ yṹnwɔ̃rɩ́ pɩ́nɔ́lɛ, ànɛ̂ fɔkɔ vyã́hnɔ́lɛ, ànɛ̂ wɛ̃ fwónɔ́lɛ, ànɛ̂ gbɛ́kɩ́nɔ́lɛ, ànɛ̂ gbɛ́kɩ́nɔ́lɛ tĩ̀ntĩ̀nnɛ, 
ànɛ̂ yĩ́wílkìlɛ, ànɛ̂ sètɛ̃̀nɩ̀ pɩ́nɔ́lɛ, ànɛ̂ yèwò pɩ́nɔ́lɛ, ànɛ̂ sé sɛ̃́nknɔ dukelɛ. Mé ké gbáhdyâhlɛ n di yé tyɩ́, máh sõ̀ kè di sah sɔkɔ yé tyɩ́ ǹgbò dùkùlɛ sɔ̃́ nɛ, névye nónó pól-á n pɩ sɛ̃́ntɛ̃̀nɛ, pé sɔkɔ yɔ́ ńtɛ̃ náh n de n pi Liyel tɔ̃́rɩ́ sɔkɔ. 
Ǹtɛ tyi nónó-á n pɩ Liyel Mirki yah tɛ̃́nɔ́ sɔkɔ, sè wɛ yɔ̀: Tyɩ́nɔnɔlɛ, ànɛ̂ nɩ̀vɩ̀nɩ̀nɛ, ànɛ̂ yèvɩnɩnɛ, ànɛ̂ sṍmnɔ́lɛ, ànɛ̂ pèpɔrɩ pɩ́nɔ́lɛ, ànɛ̂ yãm yáhnɔ́lɛ, ànɛ̂ gbɩ wɛ̃̀kɩ̀ kɔ́lɔ́lɛ, 
ànɛ̂ gblɔ̌y yɛlɛ, ànɛ̂ gblɔ̌y tɛ̃́nɔ́lɛ. Sépi póllɛ, tyi kõ̀nsàhnɔ́ náh mɛ sélɛ n fwo. 
Névye nónó-á mɛ Krista tyɩ́ yõ, pépi ye pé gblɔ̌y dyɔ yĩ́ĩ́ tyi, ànɛ̂ pé dékè tyi, ànɛ̂ pé yĩ́wílkì tyi pól syɩ dahnɩ yõ, à sè ko. 
Liyel Mirki nɛ̂-á minnɛ á kɔ̃, á ye yɛ képah kɔ̃ kè á yahlɛ n tɛ̃ kɛ̀. 
Á ye káh pɩ gblɔ̌y gbílkípúlɛ, à nɛ n pɩ n de wɛ̃ tyɩ́, képah náh pɩ, à wɛ̃ dyɔlɛ n yɔ.
