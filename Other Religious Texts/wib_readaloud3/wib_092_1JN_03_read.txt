1 Nsyɔ̃.
3.
Yé náh wɛ à, á tyɩ́ kah nɔ Sú Liyel tyɩ́, fɔ́ɔ́ à álɛ n ye ǹ wɛ̃́npìlɛ. Tɛ́ Liyel wɛ̃́npì tɛ̃̀ wɔ ye ńtyɩ̃́nɔ álɛ ó. Tɛ́ kèkõyṍ névye-á pɩ Liyel pɩ̃́-ńkɛ̃̂púlɛ sɛ̃́, képah ye pè náh fɛ̃ ápi tɔ pɩ̃. 
Ń nɛ́nɔ́nɔ́, Liyel wɛ̃́npì ye álɛ núkúnúkú, ǹtɛ áyáh kɩ pɔ n pɩ n pi kwâh nɛ̂nɛ ńsõ̂nɛ, képah náh tɔ̃̌nɩ̀ mɛ wilki mɔ tɛ. Ǹtɛ á pɩ̃ nɛ Krista-à pɔ wil mɔ gbĩ́ nɛ̂nɛ, áyáh kɩ à wɛ wǎh mɛ dùkù nɛ̂nɛ. Képah ye, á kɩ mɛ ǹmɔ tyɩ́ kɔ̃lɛ. 
Névi nɛ̂ gbõ̀ sàhnɔ́-à mɛ képah yõ sɔkɔ Krista tyɩ́, tɛ̃̌ ǹ gblɔ̌y tɔ̃ n sah yályál Liyel yah sɔkɔ, sɔ̃́-á Kristalɛ yályál. 
Névi nɛ̂-à tyípékèlɛ n pɩ, tɛ̃̀ mɛ Liyel tyi kõ̀nsàhnɔ́ dêllɛ n pɩ. Tyípékè pɩ́nɔ́lɛ, Liyel tyi kõ̀nsàhnɔ́ dêl pɩ́nɔ́ ye kélɛ. 
Ǹtɛ yé pɩ̃ nɛ Krista pɔ́nɔ́ yõ-á pɩ tyípêl tɛ̃́nyõ̀nɔ́lɛ névye yõ. Tyípékè náh mɛ ǹmɔ sɔkɔ póllɛ. 
Névi nɛ̂-à mɛ Krista tyɩ́ yõ, tɛ̃̀ náh tyípékèlɛ n pɩ. Ǹtɛ nɛ̂-à tyípékèlɛ n pɩ, tɛ̃̀ náh à wɛ yah, à náh ǹnɛ n pnɛ. 
Ń wɛ̃́npì, yé káh yɛ ǹnɩ̂ yɔ́ ńtɛ̃ yé pilki. Névi nɛ̂-à n kɔ tómm, nɛ́gbɩ́ ye tɛ̃̀nɛ Krista tyɩ́ kɔ̃lɛ. 
Tɛ́ névi nɛ̂-à tyípékèlɛ n pɩ, sétã̀n yõ̀tɛ̃̀ tyɩ́ névi ye tɛ̃̀nɛ. Yé náh wɛ di, sétã̀n yõ̀tɛ̃̀nɛ, tyípékè pɩ́-ò ye ǹnɛ tyi pól tyah gbĩ́ dùkùlɛ. Liyel Pídĩ́-á pɔ tir nɛ̂ yĩnnɛ, sétã̀n yõ̀tɛ̃̀ tyípɩ́nɩ̀ yɩkɩ yĩn ye. 
Névi nɛ̂-à pɩ Liyel pilɛ, Liyel pɩ dùkù n nɛ n mɛ tɛ̃̀ sɔkɔ. Képah ye tɛ̃̀ náh tyípékèlɛ n pɩ. Wǎh pɩ Liyel pilɛ sɛ̃́, à náh fɛ̃ tyípékè pɩ n pi. 
Liyel wɛ̃́npì ànɛ̂ sétã̀n yõ̀tɛ̃̀ tyɩ́-á n wah n pnɛ wɛ̃ sɔkɔ dùkù nɛ̂nɛ, kè wɛ yɔ̀: Névi nɛ̂-à n kɔ́-ńkɛ̃̂nɛ tómm, Liyel tyɩ́ névi náh tɛ̃̀nɛ. Névi nɛ̂ tɔ yṹnpi tyɩ́-à nɔ́-ńkɛ̃̂nɛ ǹ tyɩ́, Liyel tyɩ́ névi náh tɛ̃̀nɛ. 
Yé náh wɛ di, yáh nɛ̂ noh tyah gbĩ́ dùkùlɛ képah ye yɔ̀ nɛ ńkɛ̃́, áyáh yɩ̃nɛ á yɛ á tyɩ́ nɔ wɛ̃ tyɩ́. 
Á ye káh pɩ Kayin tyɩ́ kɔ̃lɛ. Sétã̀n yõ̀tɛ̃̀ tyɩ́ névi ye sõ̀ ǹnɛ. À ǹ gbɛ̃̀ndĩ́ ko mɔ. Ǹtɛ tir nɛ̂ yĩn mɛ à ko mɔ ǹnɛ? Ǹmɔ tyípɩ́nɔ́-á sõ̀ mɛ kétúkùyɔ̂ Liyel yah sɔkɔ tɛ́ ǹ gbɛ̃̀ndĩ́ tyɩ́lɛ pallɛ, képah yõ ye. 
Ń yṹnpyé, kèkõyṍ névye-à yé dyɔlɛ n yɔ, yé káh n yɛ kè pɩ gbɛ̃̀ndùnìnɛ yé tyɩ́. 
Ápi tɛ̃̀ wɔlɛ, á yṹnpyé tyɩ́-á nɔ á tyɩ́ sɛ̃́, á pɩ̃ nɛ áyáh tɛ̃ syi tɛ kũ gbõ̀ sɔkɔ, tɛ́ min wɛ. Ǹtɛ tyɩ́nɔnɔ-à n knɛ névi nɛ̂ sɔkɔ, ǹmɔ mɛ kũ gbõ̀ sɔkɔ. 
Névi nɛ̂ tɛ̃̀-à fyɔ̀ ǹ yṹnpi dyɔlɛ n yɔ, nɛ́kókè mɔ́-ò ye tɛ̃̀nɛ. Tɛ́ yé se mó tɔ̃ pɩ̃ nɛ tyɛ́-ńkɛ̃̂ min náh mɛ nɛ́kókè mɔ́-ò yɔ́ ńtɛ̃ tyɩ́ tɛ́. 
Yé yah, Krista-á ǹ min kɔ̃ ápi yĩnnɛ, képah sɔkɔ ye ápi mɛ tyɩ́nɔnɔ dùkù pɩ̃. Ápi tɔ yɩ̃nɛ à á min kɔ̃ á yṹnpyé yĩnnɛ. 
Ǹtɛ kèkõyṍ tahrɩ-à mɛ ǹnɩ̂ yɔ́ tyɩ́, tɛ́ à ǹ yṹnpi yɔ́ wɛ yãm sɔkɔ tɛ́ yahle ǹ yãm yáhnɔ́lɛ, Liyel tyɩ́nɔnɔ náh mɛ tɛ̃̀ sɔkɔ. 
Ń wɛ̃́npì, ápi tyɩ́nɔnɔ káh n pɩ wɛy ànɛ̂ vyã́hsesyɩ̃́rɩ́ ó sɔkɔ dɛ́. Á tyípɩ́nɔ́ yɩ̃nɛ kè kélɛ n wɛ̃kɩ kègbɩ tɛ̃̀ yõ. 
Képah sɔkɔ ye á kɩ pɩ̃ nɛ áyáh mɛ gbɩ wɛ̃̀kɩ̀ yõ, ànɛ̂ á nnɔ kɩ mɛ yĩ́nnɔ́lɛ Liyel yah sɔkɔ. 
Ápi nnɔ-à sɔ̃́ mɛ lékàhlɛ álɛ n kɔ̃ sɔ̃́, Liyel nɛ̂-á gblò kal, ǹmɔ tyi póllɛ n pnɛ n kal ápi nnɔlɛ. 
Ń nɛ́nɔ́nɔ́, á nnɔ-à kɩ lékàhlɛ álɛ n kɔ̃́-ńkɛ̃̂nɛ, tíkí náh á tɛ̃ n pi Liyel yah sɔkɔ. 
Tɛ́ áyáh mɛ ǹ tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ, tɛ́ ǹ dyɔ yĩ́ĩ́ tyilɛ n pɩ, képah ye áyâh kwáhdúkú nɛ̂ yah kɔ̃ ǹ tyɩ́, à kɩ kélɛ á kɔ̃. 
Wǎh nɛ̂ kõ sah á kɔ̃ képah ye yɔ̀ nɛ ńkɛ̃́, á sɛ̃ pé Pídĩ́ Yesu Krista yĩn yõ, á mó yɛ á tyɩ́ nɔ wɛ̃ tyɩ́ sɔ̃́-á à kõ sah kélɛ á kɔ̃. 
Névi nɛ̂-à Liyel tyi kõ̀nsàhnɔ́ wã̀llɛ n kɔ, tɛ̃̀ kɩ mɛ Liyel tyɩ́ yõ, ànɛ̂ Liyel kɩ mɛ tɛ̃̀nɛ. Sɔ̃́-á á kɩ pɩ tɛ́ kɩ pɩ̃ nɛ Liyel-á mɛ álɛ, ǹ Mirki gbõ̀ yõ ye, wǎh képah nɛ̂nɛ á kɔ̃.
