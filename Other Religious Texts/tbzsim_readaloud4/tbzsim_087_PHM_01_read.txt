Pɔɔri wɑ̃̀ri dìì pɑ́tíri Fidemɔɔ.
1.
Mí mPɔɔri bɛ̀ kpetí wè Yesu Kirisi kpɛ́í nnɛ̀ Timɔntee tíì dɑ wɑ̃̀ri fɔ̃́ nti nɛ́po sɑ̀ù nɛ̀ wè ti wɛ̃ nti pĩ́ mmùù tɔ̃mmú. 
Kɛ dɔunnɛ̀ Kirisi kɔbɛ bɛ̀ɛ̀ tiku ɑ cĩ́ɛ̃̀tɛ̀ nɛ̀ ti tɑ̃ũ̀ Apiyɑ nɛ̀ Adisipu, wèè ti wɛ̃ńnɛ̀ dimɑ̀ù. 
Ti cicɛ Kuyie nnɛ̀ ti Yiɛ̀ nYesu Kirisi bɛ̀ di dɔɔ̀ mɛsɑ̀ɑ̀ nkɛ di duɔ́ ndiwɛ̀ì! 
Fidemɔɔ n nɛ́po sɑ̀ù nh ɔ̃ mbɑ́ɑ́ nKuyie nkɛ kù sɑ̃ntímu ɑ kpɛ́í. 
N kèèmu ɑ tɑ̃́ mɛ̀ɛ̀ botí ti Yiɛ̀ nYesu koò dɔ́ kɛ dɔ́nɛ̀ o kɔbɛ. 
M bɑ́ɑ́mmu Kuyie nkɛ tú mɛtɑ̃́kùmɛ̀ mɛ̀ɛ̀ ti wɛ̃nnìnko mí nnɛ̀ fɔ̃́, mɛ̀ dɑ teennɛ̀ kɑ̀ɑ bɑntɛ́ ti mɔkɛ mɛ̀ɛ̀ sɑ̀ɑ̀ mmɛmɔu ti wɛ̃ńnɛ̀mɛ̀ Kirisi. 
N tebitɛ, ɑ dɔ́mɛ̀ Kirisi kɔbɛ dɛɛ̀ n duɔ̀ ndiwɛ̀ì kɛ nɑ́ríkùnko n yɛ̀mmɛ̀ kɛ yɑ́úkùnnɛ Kirisi kɔbɛ kɔ̃̀ntì. 
N tú Kirisi kóo tɔ̃̀ntì nwe kɛ̀ n do dɔ́ n dɑ nɑ́kɛ́ nɛ̀ muwɛ̃rímú ɑ dò nkɛ́dɔɔ̀ tì. 
N nɛ́ dɔ́ kɛ dɑ bɑ́ntɛ̀mu n dɑ dɔ́ mɛ̀ɛ̀ kpɛ́í nnɛ̀ Yesu Kirisi ti dɔ́ wè kó diyètìrì, mí mPɔɔri odɑkótì bɛ̀ n kpetínnɛ́ Yesu Kirisi kpɛ́í nkɛ. 
N dɔ́ ɑ cɔutɛ́ Onnɛsimmu nwe kóò dɔ̀ɔ̀ mɛsɑ̀ɑ̀. Míì ò nɑ̀kɛ́ Kuyie nnɑ́ɑǹtì dikpetíntou miɛkɛ kòò cèètɛ, kɛ tú m birɛ Kuyie nkó kucɛ miɛkɛ. 
N yɛ̃́mu kɛ dò nhOnnɛsimmu do í dɑ dɔ̀ɔ̀ mɛsɑ̀ɑ̀, di mmɔ̀nnì kɛ yóó mɛ̀ ndɑ dɔɔ̀ ò mɛ̀ nni ndɔ̀ɔ̀ mɛ̀ɛ̀ botí. 
Nɛ̀ we ti nɑɑ́ nhonìtì omɑ́ɑ̀ ndi, kɛ̀ n wè ndɑ duɔnko. 
N nɑ ndɔ́mu wè nkpɑɑ́ m borɛ̀, n kpetí dɛ̀ Tinɑ́ɑǹsɑ̀ɑ̀tì kpɛ́í nkɛ n teénnɛ̀, ɑ nɑ nni nteénnɛ̀ mɛ̀ɛ̀ botí. 
N nɛ́ í dɔ́ n dɔɔ̀ dɛ̀mɑrɛ̀ kɑ̀ɑ í dɔ́, m mɛ nyí dɔ́ ɑ́ n dɔɔ̀ mɛsɑ̀ɑ̀ nnɛ̀ muwɛ̃rímú, n dɔ́ ɑ́ wè nni nduɔnní nɛ̀ ɑmɑ́ɑ̀ ndi. 
Kɛ̀ dɛ̀ yɛ̀mmɛ̀ dɛ̀ dò nhOnnɛsimmu dɑ yɑtɛnɛ̀mu yɛmɔ̀rɛ̀ sɑ́m̀pɔ́ miɛkɛ, kɛ̀ di wɛ̃tɛ kɛ́wɛ̃nnɛ́ sɑ̃́ɑ̃̀. 
Di mmɔ̀nnì ò tɛ̃́nkɛ í tú kudɑɑkù cɛ̃́nkoo, ò nɑɑ́ nti kou nwe Kirisi kó mɛwɛ̃nnímɛ̀ miɛkɛ, kɛ̀ nh ò dɔ́ mɛsɑ̀ɑ̀ nkɛ̀ n yɛ̀mmɛ̀ dò nhɑ kpɛrɛ yó mmɛ ndò. 
Kɑ̀ɑ yɛ̃́ kɛ dò m mɛ̀nkɛ tú ɑ nɛ́po nwe, cɔutɛ́ we ɑ nɑ nni ncɔutɛ́ mɛ̀ɛ̀ botí. 
Kòò dɑ cɑ̀ɑ̀rɛ̀ dɛ̀mɑrɛ̀ yoo ò dɑ bɑnnɛ̀ m bo dɛ̀ ndɑ yietí. 
Mí mPɔɔri míì wɑ̃̀ri tinɑ́ɑǹtì tii nnɛ̀ n nɔ̀ùtɛ̀ kɛ tú m bo dɛ̀ ndɑ yietí. N nɛ́ í yɛ̃ nhɑ n yietí ɑ m bɑnnɛ̀ dɛ̀, kɛ yɛ̃́ ɑ m bɑnnɛ̀mɛ̀ ɑ fòmmu. 
N tebitɛ n dɑ bɑ́ɑ́mmu nɛ̀ ti Yiɛ̀ nkó diyètìrì kɛ tú ɑ n dɔɔ̀ onìtì ɔ̃ɔ̃ dɔ̀ɔ̀mɛ̀ o kou Kirisi kó kucɛ miɛkɛ. 
N tì n dɑ wɑ̃̀ri kɛ yɛ̃́mu kɛ dò nhɑ yóó tì dɔ̀ɔ̀ kɛ̀ dɛ̀ɛ pɛ̃ɛ̃tɛ́ n yɛ̃mmɛ̀. 
A n tũntɛ kudieku, n yɛ̀mmɛ̀ dò nKuyie mbo cɔutɛ́ di bɑ́ɑmmu kɛ́ m pɑ̃ kucɛ kɛ̀ n kɔtoo di ciɛ. 
Epɑfɑdɑsi, nɛ̀ wè ti wɛ̃ nkɛ kpetí Kirisi kpɛ́í nhò dɑ dɔunko. 
Nɛ̀ Mɑriki nɛ̀ Adisitɑki nɛ̀ Demɑɑsi nɛ̀ Duku nɛ̀ bɛ̀ ti pĩ mmutɔ̃mmú bɛ̀ dɑ dɔunko. 
Ti Yiɛ̀ nYesu Kirisi kó mɛsɑ̀ɑ̀ nní ndɑ bonɛ̀!
