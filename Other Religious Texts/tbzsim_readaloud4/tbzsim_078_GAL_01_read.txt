Pɔɔri wɑ̃̀ri dìì pɑ́tíri Kirisi kɔbɛ bɛ̀ɛ̀ bo Kɑdɑsii.
1.
Mí mPɔɔri Yesu kó ditɔ̃nnì míì di wɑ̃̀ri, bɛnìtìbɛ̀ bɛɛ̀ í n duɔ́ ndɛ kó mutɔ̃mmú yoo onìtì mɔù, Yesu Kirisi nɛ̀ ti cicɛ Kuyie nkùù te kòò yɑ̃̀ńtɛ́, bɛɛ̀ mù nni nduɔ́. 
N kɔbɛ bɛ̀ɛ̀ m bonɛ̀, bɛ̀ di dɔunko díndi Kirisi kɔbɛ bɛ̀ɛ̀ bo Kɑdɑsii, kɛ tú 
Kuyie nti cicɛ nɛ̀ ti Yiɛ̀ nYesu Kirisi bɛ̀ di dɔɔ̀ mɛsɑ̀ɑ̀ nkɛ di duɔ́ ndiwɛ̀ì. 
Kirisi dɔ̀ɔ̀ Kuyie ndɔ́ mɛmmɛ kɛ duɔ́ nhomɑ́ɑ̀ kɛ̀ bɛ̀ ò kùɔ, kòò bo ti dɛɛtɛ́nɛ̀ kutenkù yɛiku kuu. 
Kuyie nyètìrì ndɛu sɑ̃́ɑ̃̀. Kɛ̀ dɛ̀ mmɛ ndò. 
Dɛ̀ n dimu di pɑ̃̀nkɛ yóumɛ̀ Kuyie, kunku kùù di yu Kirisi ti dɔ̀ɔ̀ mɛ̀ɛ̀ sɑ̀ɑ̀ nkpɛ́í, kɛ dɔ́ kɛ́tũnnɛ kucɛ tɛkù. 
N di nɑ́ɑ́mmu kucɛ tɛkù mɑkù í bo, bɛnìtìbɛ̀ bɛɛ̀ dɔ́ kɛ́ di soutɛ́ kɛ ceeri Kirisi kó tinɑ́ɑǹtì. 
Kòò mɔù, yoo mí, yoo Kuyie ntɔ̃nnì mɑrì di nɑ̀kɛ́ tìmɑtì kɛ̀ tì cɑ̃̀ɑ̃́nnɛ̀ n di nɑ̀kɛ́ tìì nɑ́ɑǹsɑ̀ɑ̀tì, Kuyie nhò cɔ̃ɔ̃. 
N tì nɑ̀kɛ́mu kɛ wɛ̃̀tɛ kɛ di nɑ́ɑ́, nkòò mɔù di nɑ̀kɛ́ tìmɑtì kɛ̀ tì cɑ̃̀ɑ̃́nnɛ̀ Tinɑ́ɑǹsɑ̀ɑ̀tì di yie ntì, Kuyie nhò cɔ̃ɔ̃. 
N nɑ́ɑ́ mmiɛ nkɛ dɔ́ bɛnìtìbɛ̀ bɛɛ̀ n sɑ̃ntɑɑ̀? Bìtì, n dɔ́ Kuyie nkuù n sɑ̃ntɛ, kɛ̀ n do dɔ́ bɛnìtìbɛ̀ bɛɛ̀ n sɑ̃ntɛ n nɑ bɑ́ nkpɑɑ́ tú Kirisi kóo tɔ̃ntì. 
N kɔbɛ n di nɑ́ɑ́mmu Tinɑ́ɑǹsɑ̀ɑ̀tì n di nɑ́ɑ́ ntì í tú onìtì kpɛti. 
Mí mmɔ́mmuɔ onìtì weè í tì nni nnɑ̀kɛ́, Yesu Odɛɛtíwè weè tì nni mbɛnkɛ. 
Di kèèmu n do dòmmɛ̀ n tũ ndìì mɔ̀nnì Sifubɛ kó itié, di yɛ̃́mu n do fɛ̃́ũmmɛ̀ Kirisi kɔbɛ nɛ̀ tikɔ̃nyɑuti kɛ́ndɔ́ kɛ́ bɛ̀ deè, 
kɛ́mpɛ̃nkù n dɑpùntɔbɛ̀ Sifubɛ kó itié mmiɛkɛ, kɛ̀ n kɔ̃̀ntì nyɑunɛ̀ m bo ntũmmɛ̀ ti yɛmbɛ̀ kó ikuɔ́ mɛsɑ̀ɑ̀. 
Kuyie nnɛ̀ ku kó mɛsɑ̀ɑ̀ nkpɛ́í nkɛ̀ kù n tɑ̃ɑ̃tɛ́ kɛ̀ bɛ̀ mu nyí m pɛitɛ́, kɛ̀ m bo pĩ́ nku tɔ̃mmú. 
Kɛ̀ dɛ mɔ̀nnì tùɔ̀kɛ kɛ̀ kù m bɛnkɛ ku Birɛ kɛ̀ m bo nɑ́kɛ́ Tinɑ́ɑǹsɑ̀ɑ̀tì bɛ̀ɛ̀ í tú Sifubɛ, n yí mbekɛ òmɔù itié. 
M mɛ n yí kɔ̀tɛ Sedisɑdɛmmu kɛ bo yɑ̀ bɛ̀ɛ̀ n niitɛ́nɛ̀ kɛ nɑɑ́ nYesu tɔ̃rɛ̀, kɛ̀ m pɑ̃̀nkɛɛ kɔtɛ Adɑbii, kɛ́wɛ̃tɛní Dɑmɑɑsi. 
Yɛbie nyɛ̀tɑ̃ɑ̃ti kó difɔ̃nkúò ndi n kɔ̀tɛmɛ̀ Sedisɑdɛmmu kɛ bo yĩ́ɛ̃́tɛ́ Piɛri, kɛ dɔ̀ yɛwe tɛpíítɛ̀ nɛ̀ yɛ̀nùmmù o borɛ̀. 
N yí nyɑ̀ Yesu tɔ̃rɛ̀ tɛyɛ̀ kɛ́yɑ̀ Isɑku nwe ti Yiɛ̀ nnɑntɛ̀. 
N di wɑ̃ũ tì tú timɔ́mmɔnti nti kɛ̀ Kuyie nyɛ̃́, n yí soú. 
Dɛ kó difɔ̃nkúò kɛ́kɔtɛ Sidii nɛ̀ Sidisii. 
Dɛ mɔ̀nnì Kirisi kɔbɛ bɛ̀ɛ̀ bo Sudee bɛ̀ mu ndo í n yɛ̃́. 
Bɛ̀ do yo bɛ̀ nɑ̀ɑ́ mmɛ̀mmɛ kɛ tú: Wèè do ti fɛ̃́ũnko, weè nɑ̀ɑ́ ndi mmɔ̀nnì Tinɑ́ɑǹsɑ̀ɑ̀tì ò do í dɔ́ tìì ḿbo. 
Kɛ̀ bɛ̀ nsɑ̃ntí Kuyie n kpɛ́í.
