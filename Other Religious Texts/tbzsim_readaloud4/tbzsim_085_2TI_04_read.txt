2 Timɔntee.
4.
N dɑ nɑ́ɑ́mmu nɛ̀ Kuyie nkó diyètìrì nɛ̀ Yesu Kirisi kpɛri, wenwe wèè wɛ̃tiní kɛ bo bɑɑtɛ́ kutenkù kɛ́bekɛ́nɛ̀ bɛfòùbɛ̀ nɛ̀ bɛcíríbɛ̀ kɛ̀ n tú: 
A yetoo kɛ́nnɑ́ɑ́ Kuyie nnɑ́ɑǹtì sɑ̃́ɑ̃̀, kɛ̀ yɛmɔ̀rɛ̀ wenni yoo yɛ̀ sĩ̀. Kɛ̀ wèè yetɛ́ ɑ ò tié nkòo bɑntɛ́ o yetímɛ̀, kɛ̀ wèè dò nhɑ ò kpɑnnɛ̀ ɑ ò kpɑnnɛ̀, kɛ̀ dɛ̀ dò nhɑ yɑ́úkùnnɛ wèè kɔ̃̀ntì, ɑ́ tì yɑ́úkùnnɛ, ɑ́ mmi nkɛ́ mbɛ̀ tiè nnɛ̀ mɛyɛ̀mmɛ̀ mɛmɑ́ɑ̀. 
Yɛmɔ̀rɛ̀ kpɑɑnímu kɛ̀ bɛnìtìbɛ̀ yóó yetɛ itié nsɑ̀ɑ̀yì, kɛ́wɑmmú bɛ̀ɛ̀ bo mbɛ̀ ni kɛ bɛ̀ nɑ̀ɑ́ mbɛ̀ dɔ́ tì. 
Kɛ̀ bɛ̀ɛ bútínnɛ́ Kuyie nnɑ́ɑǹtì kɛ́ntũnnɛ iyɑ́ɑ̀bìsítié. 
A nnɔ nkɛ pĩĩ nhɑmɑ́ɑ̀, kɛ́mmɔkɛ mɛminnímɛ̀ mɛfɛ̃́ṹtímɛ̀ miɛkɛ kɛ́nnɑ́ɑ́ nTinɑ́ɑǹsɑ̀ɑ̀tì kɛ pĩ́ nhɑ tɔ̃mmú weti weti. 
N dɑ nɑ́ɑ́ mmiɛ nkɛ yɛ̃́ n duɔ́mmɛ̀mu n fòmmu kɛ yóó kú. N kṹṹ mmɔ̀nnì diì tùɔ̀kɛ. 
M mɑ̀mu dɛ̀ bèkúmɛ̀, kɛ tɑ̃́ Kuyie nkɛ tùɔ̀kɛ kumɑ̀nku. 
Kɛ bɑɑ ticuuti obeéntì sɑ̀ɑ̀wè yó n duɔ́ ntì tibeéntì yiè, wè í yóó tì duɔ́ mmíì mmɑ́ɑ̀. Ò yóó tì duɔ́ mbɛ̀ɛ̀ kó dimɑ̀ɑ̀ ò bɑ̀ɑ bɛ̀mbɛ nɛ̀ diwɛ̀ì ò yóó wɛ̃tɛní dìì yiè. 
Yetoo kɛ́kɔtɛní mɛcɑ̃ɑ̃ nkɛ́ n yɑ̀. 
Demɑɑsi dɔ́ kutenkù kuù nkpɛrɛ ndɛ kɛ n yóu kɛ kɔ̀tɛ Tɛsɑdoniki, kɛ̀ Kɛresɑ̃ɑ̃ kɔ̀tɛ Kɑdɑsii, kɛ̀ Titi kɔ̀tɛ Dɑdimɑsii. 
Duku weè mɑ́ɑ̀ kpɑɑ́ kɛ m bonɛ̀. Kɑ̀ɑ kɔ̀riní ɑ nnɛinní Mɑriki kòo n teennɛ̀ mutɔ̃mmú. 
Tisiki yie nh ò tɔ̃ nhEfɛɛsi nwe. 
Kɑ̀ɑ kɔ̀riní ɑ n tɔní n yɑɑ̀bòrì n yóu dì Todowɑɑsi, Kɑpuusi cĩ́ɛ̃̀tɛ̀, kɛ́ntɔnní m pɑ́tɛ, ɑ bɑ́ɑ́ yɛ̃̀ nhɑ bo n tɔnnímɛ̀ Kuyie nnɑ́ɑǹtì kpɛyɛ. 
Adɛkisɑ̃ntiri omɑ́ɑ́tì n dɔ̀ɔ̀ mɛyɛi mmɛdiɛ̀, ti Yiɛ̀ nyóó mɛ̀ nhò yietímu. 
Á nyɛ̃́ o kpɛ́í, ò cɑ̀ɑ̀rimu ti nɑ́ɑ́ ntìì kó dimɑ̀ɑ̀. 
Bɛ̀ do m bekùnɛ̀ dìì mɔ̀nnì mɛketimɛ̀ kɛ̀ bɛ̀ n yóumu bɛmɔu, Kuyie mbɑ́ɑ́ bɛ̀ yietí bɛ cɑɑ̀rìmɛ̀. 
Kɛ̀ Kuyie mmɛ nní nni mbonɛ̀, kɛ́ n dɛɛtɛ́nɛ̀ yɛcìrícìrɛ̀, kɛ́ n duɔ́ mmuwɛ̃rímú kɛ̀ n nɑ̀kɛ́ Tinɑ́ɑǹsɑ̀ɑ̀tì bɛ̀ɛ̀ í tú Sifubɛ. 
Ti Yiɛ̀ mbo nni ndɛɛrí nɛ̀ mɛyɛi mmɛmɔu, kɛ́ n tɑnnɛ́ o kpɑ̀ɑ̀tìyuu, o yètìrì ndɛu sɑ̃́ɑ̃̀. Kɛ̀ dɛ̀ mmɛ ndò. 
A dɔu Pirisiidi nɛ̀ o dɔù Akidɑɑsi nɛ̀ Onnɛsifɔɔ cĩ́ɛ̃̀tɛ̀ kɔbɛ. 
Edɑsiti kpɑɑ́ Kɔdɛnti nwe, kɛ̀ Todofimmu mɔ nkɛ̀ nh ò yóu Midɛti. 
Yetoo kɛ́kɔtɛní kɛ̀ diyɔ̃ɔ̃̀ kpɑɑní. Ebuduusi nɛ̀ Pitɑ̃ɑ̃ nɛ̀ Dinuusi Kododiyɑ nɛ̀ ti kɔbɛ tɔbɛ̀ bɛ̀ dɑ dɔunko. 
Ti Yiɛ̀ nni ndɑ bonɛ̀! Kuyie nkó mɛsɑ̀ɑ̀ nní ndɑ bonɛ̀.
