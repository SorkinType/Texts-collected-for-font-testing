Isɑ̃ɑ̃.
17.
Yesu bɛ́i mmɛmmɛ kɛ́deè kɛ́bɔ́útɛ́ kɛĩ́nkɛ̀ kɛ dɔ̀: N cicɛ dɛ mɔ̀nnì tùɔ̀kɛmu, dɛ́úkùnnɛ ɑ birɛ yètìrì kɛ̀ dɛ̀ɛ dɛ́úkùnnɛ ɑ kpɛri. 
Kɛ yɛ̃́ ɑ dɛ̀ duɔ́mmɛ̀ muwɛ̃rímú kɛ̀ dɛ̀ bo duɔ́ mmufòmmu mùù bo sɑ̃́ɑ̃̀ bɛnìtìbɛ̀ ɑ dɛ̀ duɔ́ mbɛ̀. 
Bɛ̀ bo yĩ́ɛ̃́tɛ́mɛ̀ fɔ̃́ nKuyie mmɔ́mmɔnku kùù bo kumɑ́ɑ̀, nɛ̀ Yesu Kirisi ɑ tɔ̃nní wè dɛɛ̀ tú mufòmmu mùù bo sɑ̃́ɑ̃̀. 
N dèèmu mutɔ̃mmú ɑ yɛ̃ m pĩ́ mmù kɛ dɛ́úkùnnɛ ɑ yètìrì kɛtenkɛ̀ ĩ́nkɛ̀. 
Di mmɔ̀nnì n cicɛ dɛ́úkùnnɛ n yètìrì, kɛ́ n duɔ́ ndisɑ̃nni n do mɔkɛ dì ɑ borɛ̀ kɛ̀ kutenkù mu nyí dɔ̀ɔ̀. 
Míì duɔ́ nkɛ̀ bɛ̀ dɑ yĩ̀ɛ̃́tɛ́, bɛnìtìbɛ̀ ɑ dɛ̀itɛní bɛ̀ kutenkù, kɛ bɛ̀ nni nduɔ́, fɔ̃́ɔ̃̀ do bɛ̀ te kɛ bɛ̀ nni nduɔ́ nkɛ̀ bɛ̀ yie nhɑ nɑ́ɑǹtì. 
Bɛ̀ bɑntɛ́mu di m̀mɔ̀nnì kɛ dò mmɔkɛ dɛ̀ɛ̀ kó dimɑ̀ɑ̀ bonní ɑ borɛ̀. 
Kɛ yɛ̃́ m bɛ̀ nɑ̀kɛ́mɛ̀ tinɑ́ɑǹtì ɑ n duɔ́ ntì kɛ̀ bɛ̀ tì cɔutɛ́ kɛ bɑntɛ́ kɛ dò m mɛ̀nkɛ bonní ɑ borɛ̀ kɛ yie nkɛ dò nfɔ̃́ɔ̃̀ n tɔ̃nní. 
N yí dɑ bɑ́ɑ́ nkutenkù kɔbɛ kpɛ́í, n dɑ bɑ́ɑ́ nhɑ n duɔ́ mbɛ̀ kpɛ́í nkɛ, kɛ yɛ̃́ fɔ̃́ɔ̃̀ bɛ̀ temɛ̀. 
M mɔkɛ dɛ̀ tú ɑ kpɛrɛ ndɛ, kɑ̀ɑ mɔ̀kɛ dɛ̀ tú n kpɛrɛ. Mɛm̀mɛ kɛ̀ bɛ̀ dɛ́úkùnko n yètìrì. 
N kũnti ɑ borɛ̀ ndɛ, n tɛ̃́nkɛ í yó nkpɑɑ́ kutenkù miɛkɛ, kɛ̀ bɛmbɛ yó nkpɑɑ́. N cicɛ wèè dò mpɑ́íí, ḿpĩ́ mbɛ nɛ̀ ɑ kó diyètìrì kɛ̀ bɛ̀ ńwɛ̃ nkɛ dò onìtì omɑ́ɑ̀ mí nnɛ̀ fɔ̃́ nti dòmmɛ̀. 
M bɛ̀ bonɛ̀ kɛ bɛ̀ bɑ̀ɑ̀mu kutenkù miɛkɛ nɛ̀ ɑ kó diyètìrì, kɛ pĩ́ nhɑ n duɔ́ mbɛ̀ bɑ́ bɛ kóò mɔù í feti, kɛ̀ dɛ̀ í tú wèè do dò nkɛ́feti wènwe kɛ̀ dɛ̀ɛ dɔɔ̀ tì wɑ̃̀ri kɛ yɛ̃mmɛ̀. 
Di m̀mɔ̀nnì n kũnti ɑ borɛ̀ ndɛ, kɛ kpɑɑ́ kɛ bo kutenkù miɛkɛ, kɛ bɛ̀ nɑ̀ɑ́ nkɛ̀ bɛ̀ bo mmɔkɛ n kó diwɛ̀ì mɛdiɛ̀. 
M bɛ̀ nɑ̀kɛ́mu ɑ kó tinɑ́ɑǹtì kɛ̀ kutenkù kɔbɛ bɛ̀ nìí bɛ̀ í tú mɛ̀ɛ̀ kpɛ́í nkutenkù kɔbɛ kɛ́ndònnɛ̀ m mɔ́mmuɔ n yí túmɛ̀ kutenkù kou. 
N yí dɑ bɑ́ɑ́ nkɛ yɛ̃ nhɑ bɛ̀ dènnɛ kutenkù miɛkɛ, n yɛ̃ nhɑ mbɛ̀ kɑ̃nkɛ́ oyɛiwe nwe. 
Bɛ̀ í tú kutenkù kuu nkɔbɛ, m mɔ́mmuɔ n yí túmɛ̀ kutenkù kou. 
Wénkùnnɛ bɛ nɛ́ ɑ nɑ́ɑǹtì, ɑ nɑ́ɑǹtì tu timɔ́mmɔnti nti. 
A n tɔ̃nní mɛ̀ɛ̀ botí kutenkù miɛkɛ mmɛ mbɛ̀ tũɔ̃̀ nku miɛkɛ. 
N duɔ́ m mɑ́ɑ̀ bɛɛ̀ kpɛ́í, kɛ̀ dɛ̀ bo yie nkɛ̀ bɛ̀ mɛ̀nkɛ ntú ɑ kɔbɛ. 
N yí dɑ bɑ́ɑ́ mbɛɛ̀ mɑ́ɑ̀ kpɛ́í, n dɑ bɑ́ɑmmu nɛ̀ bɛ̀ɛ̀ yóó keè ɑ nɑ́ɑǹtì kɛ́ nni ntɑ̃́ bɛ kpɛ́í. 
N dɑ bɑ́ɑ́ nkɛ tú bɛ̀ ńwɛ̃ nkɛ dò nhonìtì omɑ́ɑ̀, mí nnɛ̀ fɔ̃́ nti dòmmɛ̀ kɛ́ nti tɑunɛ̀ kɛ̀ kutenkùu bɑntɛ́ fɔ̃́ɔ̃̀ n tɔ̃nnímɛ̀. 
M bɛ̀ duɔ́mmu tikpetì ɑ n duɔ́ ntì, kɛ̀ bɛ̀ bo ńwɛ̃ nkɛ dò nhonìtì omɑ́ɑ̀ m mí nnɛ̀ fɔ̃́ nti dòmmɛ̀. 
Kɛ̀ nní mbo bɛ miɛkɛ kɑ̀ɑ ḿbo m miɛkɛ kɛ̀ bɛ̀ ńwɛ̃ nkɛ dò onìtì omɑ́ɑ̀, kɛ̀ kutenkùu bɑntɛ́ fɔ̃́ɔ̃̀ n tɔ̃nnímɛ̀ kɛ bɛ̀ dɔ́ ɑ n dɔ́ mɛ̀ɛ̀ botí. 
N cicɛ n dɔ́ kɛ̀ m borɛ̀ ɑ n duɔ́ mbɛ̀ ndɛ mbo, kɛ wúó n kpetì ɑ n duɔ́ ntì kɛ yɛ̃́ ɑ n dɔkɛmɛ̀ kɛ mu nyí dɔ̀ɔ̀ kutenkù. 
N cicɛ ɑ tú timɔ́mmɔnti yiɛ̀ nwe, kutenkù kɔbɛ í dɑ yɛ̃́, mí mmɛ ndɑ yɛ̃́mu, kɛ̀ n tɑncɔ́ùmbɛ̀ yɛ̃́ kɛ dò nfɔ̃́ɔ̃̀ n tɔ̃nní. 
M bɛ̀ nɑ̀kɛ́mu ɑ tú wè kɛ yó nsɔkɛ́ kɛ bɛ̀ bɛnkú, kɛ̀ bɛ̀ nyɛ̃́ kɛ dò nhɑ bɛ̀ dɔ́ ɑ n dɔ́ mɛ̀ɛ̀ botí, kɛ̀ m bo bɛ miɛkɛ.
