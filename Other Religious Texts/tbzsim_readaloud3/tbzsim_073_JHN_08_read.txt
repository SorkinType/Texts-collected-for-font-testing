Isɑ̃ɑ̃.
8.
Kɛ̀ Yesu dekɛ Odìfíè tɑ̃rì, 
kɛ́ pikɛ wɛ̃tɛ Kuyie ncĩ́ɛ̃̀tɛ̀, kɛ̀ bɛnìtìbɛ̀ɛ tíí nkòo nkɑri kɛ bɛ tiè. 
Kɛ̀ ikuɔ́ wɑ̃ɑ̃̀bɛ̀ nɛ̀ Fɑdisĩɛ̃bɛɛ tɔní onitipòkù mɔù kóò còńnɛ́ ditĩ̀nnì cuokɛ̀, 
kɛ́nɑ́kɛ́ Yesu kɛ dɔ̀: Okótì, ti sɔ̃́ nhonitipòkù yiè nduɔ́nɛ̀ onitidɔ̀ù mɔù nwe kɛ̀ ti ò pĩ. 
Mɔyiisi kó ikuɔ́ yɛ̃ nti níí búɔ́tí wèè dɔ̀ɔ̀ mɛ kóò botí yɛtɑ̃́rɛ̀ nyɛ kɛ́kuɔ. A yɛ̀mmɛ̀ dòmmɛ? 
Bɛ̀ do nɑ́ɑ́ mmɛmmɛ kòo díintɛmu kɛ wɑnti bɛ̀ bo ò sɛi ntìì ĩ́nkɛ̀. Kɛ̀ Yesu sĩ́nnɛ́ kɛ́nwɑ̃ũ kɛtenkɛ̀ nɛ̀ o nɔ́mbii. 
Kɛ̀ bɛ̀ bɑɑ nsɔkɛ́ koò bèkú, kòo íinní o yuu kɛ́bɛ́i nkɛ dɔ̀: Di miɛkɛ wèè í dɔ̀ɔ̀ mɛyɛi nnɛ̀ ò bomɛ̀, wèe ketɛ́ kóò bɔ̀tɛ́ ditɑ̃́rì. 
Kɛ́wɛ̃tɛ kɛ́sĩ́nnɛ́ kɛ́nwɑ̃ũ kɛtenkɛ̀. 
Bɛ̀ kèè dìì mɔ̀nnì dɛ kó tinɑ́ɑǹtì kɛ́nyiiti bɛmɑ bɛmɑ, kɛ́ketɛ́nɛ̀ bɛkótíbɛ̀ kɛ yɑ̀ɑ dentɛnɛ̀ bɛbɛ́mbɛ̀, kɛ yóu Yesu nɛ̀ onitipòkù dɛborɛ̀. 
Kɛ̀ Yesu íinní o yuu kɛ́bekɛ́ onitipòkù kɛ dɔ̀: Yé bɛ̀ɛ̀ dɑ pĩ̀ńní? Òmɔù tɛ̃́nkɛ í dɔ́ kɛ́ dɑ kuɔɑ̀? 
Kòo ò tɛ̃̀ńnɛ́ kɛ dɔ̀: Ɔ́ɔ̀ n Yiɛ̀! Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: M múnkɛ í dɔ́ kɛ dɑ kuɔ, kɔtɛ, kɛ tɛ̃́nkɛ nɛ́ bɑ́ɑ́ dɔɔ̀ mɛyɛi. 
Kɛ̀ Yesu wɛ̃tɛ kɛ́bɛ́innɛ̀ ditĩ̀nnì kɛ dɔ̀: Míì tú kutenkù kó kuwenniku, wèè n tũ nhò bɑ́ nkérí dibiìnnì, dɛ kó kuwenniku ò kɔ̀rinɛ̀ mufòmmu borɛ̀ ndɛ. 
Kɛ̀ Fɑdisĩɛ̃bɛɛ ò nɑ̀kɛ́ kɛ dɔ̀: A nɑ́ɑ́ nhɑmɑ́ɑ̀ kpɛ́í nkɛ, ɑ nɑ́ɑǹtì í tú timɔ́mmɔnti. 
Kɛ̀ Yesu bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: N nɑ́ɑ́ m mɑ́ɑ̀ kpɛ́í nkɛ, kɛ̀ tì tu timɔ́mmɔnti, kɛ yɛ̃́ n yɛ̃́mɛ̀mu m boní dɛ̀ nɛ̀ n kɔri kɛ̀, díndi mɛ nyí yɛ̃́ m boní dɛ̀ nɛ̀ n kɔri kɛ̀. 
Díndi bekù di nuɔ nwùó ndɛ̀ndɛ, mí mmɛ nyí bekùnɛ̀ òmɔù. 
Kɛ̀ dɛ̀ tùɔ̀kɛní kɛ dò m bekɛ́, m bekɛ́nɛ̀ timɔ́mmɔnti, kɛ yɛ̃́ n yí bomɛ̀ m mɑ́ɑ̀, n cicɛ wèè n tɔ̃nní ò m bonɛ̀mu. 
Di kó ikuɔ́ miɛkɛ tì wɑ̃̀rimu kɛ tú: Bɛnìtìbɛ̀ bɛ̀dɛ́bɛ̀ yɑ̀ tì kɛ bɛ́i ntiì ɔ̃ ntú timɔ́mmɔnti! 
N nɑ́ɑ́mmu m mɑ́ɑ̀ kpɛ́í, nkɛ̀ n cicɛ wèè n tɔ̃nní kòò nɑ̀ɑ́ n kpɛ́í. 
Kɛ̀ bɛ̀ɛ bekɛ Yesu kɛ dɔ̀: A cicɛ borɛ? Kòo bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: Di í n yɛ̃́, di mɛ nyí yɛ̃́ n cicɛ, kɛ̀ di do n yɛ̃́ di nɑ nyɛ̃́mu n cicɛ. 
Yesu do bo Kuyie ncĩ́ɛ̃̀tɛ̀ miɛkɛ nkɛ, bɛ̀ ɑ̃ɑ̃̀ dɛ̀ idíítí kɛ nɑ́ɑ́ ndɛ kó tinɑ́ɑǹtì, òmɔù mɛ nyí nhò pĩ nkɛ yɛ̃́ o mɔ̀nnì mu ndo í tùɔ̀kɛmɛ̀. 
Kɛ̀ Yesu yíɛ́ kɛ́ bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: N yitimu kɛ̀ di bo n wɑmmú kɛ kú di yɛi mmiɛkɛ. Di bɑ́ɑ́ nɑ kɛ́kɔtɛ n kɔri kɛ̀. 
Kɛ̀ Sifubɛ nnɑ́ɑ́ mbɛmɑ́ɑ̀ kɛ tú: Ò sɑ̀ɑ̀ yóó kuɔ omɑ́ɑ̀ ndɑɑ̀? Bɑ nnɛ́ tú ti bɑ́ɑ́ nɑ kɛ́kɔtɛ ò kɔ̀ri kɛ̀. 
Kɛ̀ Yesu bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: Di tú kɛtenkɛ̀ kɔbɛ mbɛ, kɛ̀ n tu kɛĩ́nkɛ̀ kou, di tú ku ntenkù kɔbɛ mbɛ, mí mmɛ nyí tú kutenkù kou. 
Dɛɛ̀ kpɛ́í nte kɛ̀ n di nɑ̀kɛ́ kɛ tú di yóó kú di yɛi mmiɛkɛ, kɛ nsɑ̀ kɛ̀ di í yie nkɛ dò n tú n tú wè, di yóó kú di yɛi mmiɛkɛ nkɛ. 
Kɛ̀ bɛ̀ɛ bekɛ Yesu kɛ dɔ̀: A tú we? Kòo bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: N di nɑ̀kɛ́mu n tú wè nɛ̀ dimɔ̀nnì. 
M mɔkɛmu tinɑ́ɑǹtì pɛ́u n nɑ nnɑ̀kɛ́ tì di kpɛ́í nnɛ̀ m bo di bekɛ́nɛ̀ tì, n cicɛ wèè nɛ́ n tɔ̃nní ò tu timɔ́mmɔnti yiɛ̀ nwe, n kèè tì o borɛ̀, n ti nnɑ́ɑ nkutenkù. 
Bɑ́ bɛ̀ í mbɑntɛ́ kɛ dò nhò bɛ̀ nɑ̀ɑ́ nho cicɛ kpɛ́í nkɛ. 
Dɛ mɔ̀nnì kɛ̀ Yesu bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: Di bo dee ndìì mɔ̀nnì Onìtìbirɛ kudɑpɑ̃ɑ̃tí ĩ́nkɛ̀ di bo bɑntɛ́ n tú wè, kɛ́bɑntɛ́ kɛ dò n yí nɑ́ɑ́ ntìmɑtì nɛ̀ m mɑ́ɑ̀, n nɑ́ɑ́ n cicɛ m bɛnkɛ tìnti. 
Wèè n tɔ̃nní wè m bonɛ̀mu wè í n yóu m mɑ́ɑ̀, kɛ yɛ̃́ n dɔɔrimɛ̀ ò dɔ́mɛ̀ mmɛ sɑ̃́ɑ̃̀. 
Yesu nɑ̀kɛ́ dìì mɔ̀nnì mɛmmɛ kɛ̀ bɛnìtìbɛ̀ pɛ́uu yie nkɛ́ nhò tɑ̃́. 
Kɛ̀ Yesu nɑ́kɛ́ Sifubɛ bɛ̀ɛ̀ do ò tɑ̃́ kɛ dɔ̀: Kɛ̀ di yie n nɑ́ɑǹtì, ńdɔɔrinɛ̀ tì yɛ̃mmɛ̀, di bo ntú n tɑncɔ́ùmbɛ̀ mɔ́mmɔmbɛ. 
Di bo yĩ́ɛ̃́tɛ́ timɔ́mmɔnti, kɛ̀ tì di dennɛ tidɑɑtì miɛkɛ. 
Kɛ̀ bɛ̀ɛ ò tɛ̃̀ńnɛ́ kɛ dɔ̀: Ti tú Abɑrɑhɑmmu yɑɑ̀bí nyi, ti í nɑɑ́ nhòmɔù kó tidɑɑtì diyiè mɑrì, ɑ bɑ́ɑ́ nɑ kɛ dɔ̀ ti bo yɛ̀ tidɑɑtì miɛkɛ. 
Kɛ̀ Yesu bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: N di nɑ́ɑ́ mmɔ́mmuɔ nwe, wèè dɔ̀ɔ̀ri mɛyɛi nhò tú mɛyɛi nkó kudɑɑkù nku. 
Kudɑɑkù ɔ̃ɔ̃ í nhɑ̃ kɛ bo ku yiɛ̀ ncĩ́ɛ̃̀tɛ̀ sɑ̃́ɑ̃̀, tɛcĩ́ɛ̃̀tɛ̀ birɛ dɛɛ̀ ɔ̃ ḿbo sɑ̃́ɑ̃̀. 
Kɛ̀ n di dènnɛ tidɑɑtì miɛkɛ di mɛ̀nkɛ yɛ̀mu ti miɛkɛ. 
N yɛ̃́mu di túmɛ̀ Abɑrɑhɑmmu kó iyɑɑ̀bí, kɛ nɛ́ dɔ́ kɛ́ n kuɔ, di í yie mmɛ̀ɛ̀ kpɛ́í n nɑ́ɑǹtì. 
Mí n di nɑ́ɑ́ n cicɛ m bɛnkɛ tìnti, kɛ̀ di dɔɔri di kóò cicɛ di nɑ̀kɛ́ tì. 
Kɛ̀ bɛ̀ dɔ̀: Ti cicɛ tú Abɑrɑhɑmmu nwe. Kɛ̀ Yesu dɔ̀: Kɛ̀ di mɛ̀nkɛ do ntú Abɑrɑhɑmmu yɑɑ̀bí nyí di nɑ mpĩ́mmu ò do pĩ́ mmùù tɔ̃mmú. 
N di nɑ̀kɛ́ timɔ́mmɔnti nti n kèè tì Kuyie mborɛ̀ kɛ̀ di dɔ́ kɛ́ n kuɔ, Abɑrɑhɑmmu í dɔ̀ɔ̀ mɛ kóò botí. 
Di pĩ́ ndi cicɛ kó mutɔ̃mmú mmu. Kɛ̀ bɛ̀ dɔ̀ ti í tú ipocɛ̃mbí, ti cicɛ bo omɑ́ɑ̀ ndi, weè tu Kuyie. 
Kɛ̀ Yesu dɔ̀: Kɛ̀ Kuyie nkuù do ntú di cicɛ, di nɑ nni ndɔ́mu, kɛ yɛ̃́ nɛ̀ ku tíì bomɛ̀, kɛ̀ kuù n tɔ̃nní, n yí kɔ̀tɛní nɛ̀ m mɑ́ɑ̀. 
Bɑ nte kɛ̀ di í bɑnní n nɑ́ɑǹtì? Di to kpeńnì mɛ̀ɛ̀ kpɛ́í nkɛ. 
Di cicɛ tú dibɔɔ̀ ndi, kɛ̀ di dɔ́ kɛ́dɔɔ̀ di cicɛ dibɔɔ̀ dɔ́mɛ̀. Dibɔɔ̀ tu okɔ̀ùtì nwe nɛ̀ dì bomɛ̀, dì ɔ̃ɔ̃ í nnɑ kɛ́nɑ́kɛ́ timɔ́mmɔnti, di nɑ́ɑǹtì timɔu ɔ̃ɔ̃ ntú siyɑ́ɑ̀bìsí nsi, kɛ yɛ̃́ dì tumɛ̀ siyɑ́ɑ̀bìsí yiɛ̀ nwe, siyɑ́ɑ̀bìsí cicɛ. 
Mí n nɑ́ɑ́ ntimɔ́mmɔnti nti, dɛɛ̀ te kɛ̀ di í yie n kpɛti. 
Di miɛkɛ we mbo bɛnkɛ n dɔ̀ɔ̀ mɛ̀ɛ̀ yɛi? N nɑ́ɑ́ ntimɔ́mmɔnti nti, dɛ̀ yĩ́mɛ kɛ̀ di í dɔ́ kɛ́yie n kpɛti. 
Wèè tu Kuyie nkou weè kemmu ku nɑ́ɑǹtì. Díndi mɛ nyí tú Kuyie nkɔbɛ, dɛɛ̀ te kɛ̀ di í kemmu n nɑ́ɑǹtì. 
Kɛ̀ Sifubɛɛ ò nɑ̀kɛ́ kɛ dɔ̀: Ti nɑɑtimu ti yĩ́mɛ̀ ɑ tú Sɑmmɑrii ɛì kou, kɛ̀ dibɔɔ̀ dɑ tɑ. 
Kɛ̀ Yesu bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: Dibɔɔ̀ mɑrì í n tɑ, n dɛ́úkùnko n cicɛ yètìrì ndi, díndi mɛ nyí dɔ́ kɛ́ n dɛ́úkùnnɛ. 
N yí wɑnti disɑ̃nni, Kuyie nkuù dɔ́ n sɑ̃nni kɛ̀ kuù bekù. 
N di nɑ́ɑ́ mmɔ́mmuɔ nwe, wèè yie n nɑ́ɑǹtì ò bɑ́ɑ́ kú. 
Kɛ̀ Sifubɛɛ ò nɑ̀kɛ́ kɛ dɔ̀: Di mmɔ̀nnì ti bɑntɛ́mu weti weti kɛ do ndibɔɔ̀ dɑ tɑ. Abɑrɑhɑmmu kumu kɛ̀ Kuyie mpɑ̃ɑ̃nɑ́ɑǹtì nɑ́ɑm̀bɛ̀ ku, kɛ̀ fɔ̃́ ntú wèè yie nhɑ tié nhò bɑ́ɑ́ kúɑ̀? 
Fɔ̃́ɔ̃̀ dɛu kɛ pɛ̃ɛ̃tɛ́ Abɑrɑhɑmmu wèè kú nɛ̀ Kuyie mpɑ̃ɑ̃nɑ́ɑǹtì nɑ́ɑm̀bɛ̀ bɛ̀ɛ̀ kúɑ̀? A yɛ̀mmɛ̀ do nhɑ tú we? 
Kɛ̀ Yesu bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: Kɛ̀ n sɑ̃ntí m mɑ́ɑ̀ n sɑ̃nni bo ndò ndɛtetìrɛ̀. N cicɛ weè n sɑ̃ntí, wenwe di yɛ̃ nwèè tú di kó Kuyie. 
Di í ò yɛ̃́, mí mmɛ nhò yɛ̃́mu, kɛ̀ n yĩ n yí ò yɛ̃́ n tú siyɑ́ɑ̀bìsí yiɛ̀ nwe kɛ di dònnɛ̀, m mɛ nhò yɛ̃́mu, kɛ yíé nho nɑ́ɑǹtì. 
Dɛ̀ do nɑɑtimu di cicɛ Abɑrɑhɑmmu n kɔriní mɛ̀ɛ̀ kpɛ́í, kòo n yɑ̀ kɛ̀ diwɛ̀ì ò pĩ. 
Kɛ̀ Sifubɛɛ ò bekɛ kɛ dɔ̀: A bie mmu nyí tùɔ̀kɛ sipísìnùmmù kɑ̀ɑ yɛ̃ nhɑ yɑ̀ Abɑrɑhɑmmɑɑ̀? 
Kɛ̀ Yesu bɛ̀ tɛ̃̀ńnɛ́ kɛ dɔ̀: N yó mbo kɛ̀ bɛ̀ mu nyí pɛitɛ́ Abɑrɑhɑmmu. 
Dɛ mɔ̀nnì kɛ̀ bɛ̀ɛ túótɛ́ yɛtɑ̃́rɛ̀ kɛ bo ò bùɔ́tí, kóò mɔ̀ńtóo Kuyie ncĩ́ɛ̃̀tɛ̀ miɛkɛ, kòò yɛ̀.
