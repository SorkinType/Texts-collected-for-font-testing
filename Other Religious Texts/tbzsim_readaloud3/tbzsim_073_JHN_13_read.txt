Isɑ̃ɑ̃.
13.
Kɛ̀ diyentɛ́bɑnni ntú nɑnkɛ kɛ̀ Yesu bɑntɛ́ kɛ dò nho mɔ̀nnì tùɔ̀kɛ ò bo ítɛ́mɛ̀ kɛtenkɛ̀ ĩ́nkɛ̀ kɛ́dekɛ o cicɛ borɛ̀, ò nɛ́ mɛ ndɔ́mɛ̀ o kɔbɛ bɛ̀ɛ̀ bo kɛtenkɛ̀ ĩ́nkɛ̀, kòo bɛ̀ bɛnkɛ ò bɛ̀ dɔ́mɛ̀ mɛdiɛ̀. 
Bɛ̀ yo ndìì mɔ̀nnì kɛ sɔ̃́ ndibɔɔ̀ diɛ́ nkɛ ɑ̃nnɛ́ Sudɑɑsi Isikɑdiyoti Simɔɔ kó dɛbirɛ mɛyɛ̀mmɛ̀ ò bo fítɛ́nɛ̀mɛ̀ Yesu. 
Yesu mɛ nyɛ̃́mɛ̀ kɛ dò nhò bonní Kuyie mborɛ̀, kɛ wɛ̃ti ku borɛ̀, kɛ̀ kù ò duɔ́ ndɛmɔu. 
Kòo ítɛ́ kɛ́dɑ́tɛ́ o yɑɑ̀bòrì, kɛ́boutɛ́ kuyɑ̀ɑ̀kù ò do boú kù o kɑ, 
kɛ́ɑ̃nnɛ́ mɛniɛ ndibuu kɛ́ketɛ́ kɛ́nhɔu o tɑncɔ́ùmbɛ̀ nɑɑ̀cɛ̀i, kɛ ũũtìnɛ̀ kuyɑ̀ɑ̀kù ò do boú kù o kɑ. 
Kɛ́tuɔkɛ Simɔɔ Piɛri borɛ̀, kòò dɔ̀: Fɔ̃́ n Yiɛ̀, nkɛ̀ fɔ̃́ɔ̃̀ ɔ́útɛ́ n nɑɑ̀cɛ̀iɑ̀? 
Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: Di mmɔ̀nnì n dɔɔri dɛ̀ ɑ mu mbɑ́ɑ́ dɛ̀ bɑntɛ́, ɑ bo dɛ̀ bɑntɛ́ diyiè mɑrì. 
Kɛ̀ Piɛrii yetɛ kɛ dɔ̀: A bɑ́ɑ́ ɔ́ú n nɑɑ̀cɛ̀i. Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: Kɛ̀ n yí ɔ́ú ɑ nɑɑ̀cɛ̀i ti bɑ́ nkpɑɑ́ wɛ̃. 
Kɛ̀ Simɔɔ Piɛrii ò nɑ̀kɛ́ kɛ dɔ̀: Kɛ̀ mɛm̀mɛ n Yiɛ̀, ndɛ̀ í tú n nɑɑ̀cɛ̀i mɑ́ɑ̀, ɔ́únɛ̀ n nɔu nɛ̀ n yuu. 
Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: Wèè wùɔ ò tɛ̃́nkɛ í dò nkɛ́wuɔ, kòò í ɔ́ú o nɑɑ̀cɛ̀i nsi kɛ wenkɛ. Díndi wennimu dɛ̀ nɛ́ í tú dimɔu. 
Yesu do yɛ̃́mu wèè yóó ò fìtɛ́, kɛ̀ dɛɛ̀ ńte kòò dɔ̀: Di í wenni di mɔu. 
Ò ɔ́ú dìì mɔ̀nnì bɛ nɑɑ̀cɛ̀i kɛ́deè kɛ́dɑ́ɑ́tí o yɑɑ̀bòrì kɛ́kɑroo bɛ̀ yo ndɛ̀, kɛ́ bɛ̀ bekɛ kɛ dɔ̀: Di bɑntɛ́ n dɔ̀ɔ̀rɑ̀ɑ̀? 
Di n yu kɛ tú Okótì, di Yiɛ̀, mɔ́mmuɔ nwe, n we ntú. 
Mí nwèè tú Okótì, di Yiɛ̀, kɛ̀ nh ɔ́ú di nɑɑ̀cɛ̀i, di dò nkɛ́hɔ́úmu ditɔbɛ̀ nɑɑ̀cɛ̀i. 
N di bɛnkɛ di dò nkɛ́ndɔɔri mɛ̀mmɛ ditɔbɛ̀. 
N di nɑ́ɑ́ mɔ́mmuɔ nwe, otɔ̃ntì í dɛunɛ̀ ò pĩ nwè mutɔ̃mmú, ditɔ̃nnì mɛ nyí dɛunɛ̀ wèè dì tɔ̃. 
Di bɑntɛ́mu dɛ kó mɛdɔɔ̀rìmɛ̀, mmɛ̀ dɔ̀ɔ̀rinɛ̀ dɛ̀ bo ndi nɑɑti. 
N yí nɑ́ɑ́ ndimɔu di kpɛ́í, n yɛ̃́mu n tɑ̃ɑ̃tɛ́ bɛ̀, tìì wɑ̃̀ri tì nɛ́ dò nkɛ́dɔɔ̀mu tìì tú: Nɛ̀ wè ti wɛ̃ nweè kòńtɛ́ kɛ n dokùnɛ̀. 
M pɑ̃̀nkɛ ti ndi nɑ̀kɛ́mu kɛ̀ tì mu nyí tùɔ̀kɛní, kɛ̀ dɛ̀ tùɔ̀kɛní dìì mɔ̀nnì di nyɛ̃́ kɛ dò n tú, n tú wè. 
N di nɑ́ɑ́ mmɔ́mmuɔ nwe, wèè cɔutɛ́ n tɔ̃ nwè ò cɔutɛ́ mínwe, kɛ̀ wèè n cɔutɛ́ ò cɔutɛ́mu wèè n tɔ̃nní. 
Yesu bɛ́i mmɛmmɛ kòo yɛ̀mmɛ̀ɛ cɑɑ̀rɛ̀ mɛdiɛ̀ nkòò dɔ̀: N di nɑ́ɑ́ mmɔ́mmuɔ nwe di kóò mɔù weè yó n fítɛ́. 
Bɑ́ o tɑncɔ́ùmbɛ̀ í mbɑntɛ́ ò yɛ̃ nwè kɛ́nwéí wéí bɛtɔbɛ̀. 
Yesu tɑncɔ́ùmbɛ̀ kóò mɔù ò dɔ́ wè mɛsɑ̀ɑ̀, kòò kɑ̀ri koò pɛ́kɛ́nɛ̀. 
Kɛ̀ Simɔɔ Piɛrii ò pɑ̀sìrì kɛ dɔ̀: Bekɛ we kɛ́yɑ̀ ò yɛ̃ nwè nwe. 
Kòo sĩ́nnɔ́ɔ kɛ́bekɛ Yesu kɛ dɔ̀: N Yiɛ̀ wenninwe? 
Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: N yóó pɔɔ̀tɛ̀ pɛ̃́ɛ̃̀ tikṹntì kɛ́duɔ́ nwè nwe. Mɛm̀mɛ ò pɔ̀ɔ̀tɛ̀mɛ̀ pɛ̃́ɛ̃̀ kɛ́duɔ́ nSudɑɑsi, Isikɑdiyoti birɛ. 
Sudɑɑsi cɔutɛ́ dìì mɔ̀nnì pɛ̃́ɛ̃̀ kɛ̀ dibɔɔ̀ tɑ o yɛ̀mmɛ̀ kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: A yóó dɔɔ̀ tì, dɔɔ̀ ti mɛcɑ̃ɑ̃. 
(Ò kɑ̀rinɛ̀ bɛ̀ kɛ yo, bɛ kóo mɔù í mbɑntɛ́ mùù te kòò ò nɑ̀kɛ́ mɛmmɛ. 
Sudɑɑsi weè do mɛ ntɔumɛ̀ idíítí, kɛ̀ bɛ̀mɑbɛ̀ yɛ̀mmɛ̀ ńdò nYesu yɛ̃ nwèè kɔtɛ kɛ́dontɛ́ní dibɑnni kó mudiì mmu, yoo wèe pɑ̃ dɛ̀mɑrɛ̀ bɛcĩ̀rìbɛ̀.) 
Kɛ̀ Sudɑɑsii cɔutɛ́ pɛ̃́ɛ̃̀ kɛ́yɛ̀ mɛcɑ̃ɑ̃, kɛ sɔ̃́ ndɛ̀ urikɛ. 
Sudɑɑsi yɛ̀ dìì mɔ̀nnì kɛ̀ Yesu bɛ́i nkɛ dɔ̀: Dɛ mɔ̀nnì tùɔ̀kɛmu kɛ̀ Onìtìbirɛ bo dɛ́úkùnnɛ dɛ yètìrì, kɛ́dɛ́úkùnnɛ Kuyie nkpɛri. 
Kɛ̀ Onìtìbirɛ dɛ̀úkùnnɛ Kuyie nyètìrì, Kuyie mmúnkɛ bo dɛ́úkùnnɛ dɛ kpɛri bɑ̀mbɑ̀. 
M bí, n yí yó nkpɑɑ́ kɛ di bonɛ̀ kɛ̀ dɛ̀ɛ mɔntɛ. N yitimu kɛ̀ di bo n wɑmmú kɛ́mɔ́ntɛ́, n nɑ̀kɛ́ tì Sifubɛ n ti ndi nɑ́ɑ́, di bɑ́ɑ́ nɑ kɛ́kɔtɛ n kɔri kɛ̀. 
N di nɑ̀kɛ́ ditɑnnùù pɑ̀nnì ndi, n dɔ́nɛ̀ ditɔbɛ̀. N di dɔkɛ mɛ̀ɛ̀ botí ḿmɛ ndɔ́nɛ̀ ditɔbɛ̀. 
Kɛ̀ di dɔ́ ditɔbɛ̀, mɛm̀mɛ bɛ̀ bo bɑntɛ́mɛ̀ kɛ dò ndi tú n tɑncɔ́ùmbɛ̀. 
Kɛ̀ Simɔɔ Piɛrii ò bekɛ kɛ dɔ̀: N Yiɛ̀ ɑ kɔri kɛ? Kɛ̀ Yesu ò tɛ̃̀ńnɛ́ kɛ dɔ̀: A bɑ́ɑ́ nɑ kɛ́ n nɛinɛ̀ bɑ̀mbɑ̀, dɛ̀ yóó yíɛ́mu kɑ̀ɑ n tũnnɔɔ. 
Kɛ̀ Piɛrii ò bekɛ kɛ dɔ̀: N Yiɛ̀, bɑ nte kɛ̀ m bɑ́ɑ́ nɑ kɛ dɑ nɛinɛ̀ bɑ̀mbɑ̀? N yiemmu kɛ bo kú ɑ kpɛ́í. 
Kɛ̀ Yesu ò tɛ̃̀ńnɛ́ kɛ dɔ̀: A mɛ̀nkɛ yɛ̃ nhɑ bo kú n kpɛ́ínɑ́ɑ̀? N dɑ nɑ́ɑ́ mɔ́mmuɔ nwe, tɛkotɛ bo nyóó kuɔ́ nkɑ̀ɑ pɑ̀rìkɛ̀ kucɛ mɛ̀tɑ̃ɑ̃ti kɛ tú ɑ í n yɛ̃́.
