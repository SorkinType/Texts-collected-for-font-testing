Isɑ̃ɑ̃.
14.
Kɛ̀ Yesu bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: Di yɛ̀mmɛ̀ bɑ́ɑ́ cɑɑ̀rɛ̀. N tɑ̃́nɛ̀ Kuyie nkɛ́ nni ntɑ̃́. 
Mɛfíè nsũmu n cicɛ cĩ́ɛ̃̀tɛ̀, kɛ̀ dɛ̀ do í dò mɛmmɛ nnɑ bɑ́ mbɛ́i. N kũnti kɛ bo tũntɛ di yó mborɛ̀ ndɛ. 
Kɛ̀ n kò kɛ tũ̀ntɛ di yó mborɛ̀, m bo wɛ̃tɛní kɛ di túótɛ́ kɛ̀ dí ḿbo m borɛ̀. 
Di yɛ̃́mu n kũnti kɛ̀, kɛ yɛ̃́ kucɛ. 
Kɛ̀ Tomɑɑ dɔ̀: N Yiɛ̀, ti í yɛ̃́ ɑ kũnti kɛ̀, ti bo yĩ́mɛ kɛ́nyɛ̃́ kucɛ? 
Kɛ̀ Yesu ò nɑ̀kɛ́ kɛ dɔ̀: Míì tu kucɛ, nɛ̀ timɔ́mmɔnti, nɛ̀ mufòmmu. Òmɔù í kɔri n cicɛ borɛ̀ kɛ í yɛ̀ m borɛ̀. 
Kɛ̀ di do n yɛ̃́ di nɑ nyɛ̃́mu n cicɛ, di mɛ nhò yɛ̃́mu kɛ ò yɑ̀. 
Kɛ̀ Fidipuu bɛ́i nkɛ dɔ̀: N Yiɛ̀, nti bɛnkɛ ɑ cicɛ kɛ̀ dɛ̀ nti sɑnnɛ̀. 
Kɛ̀ Yesu ò tɛ̃̀ńnɛ́ kɛ dɔ̀: M mɛ ndi bonɛ̀ dìì dɑɑ̀ ɑ mu nyí n yĩ̀ɛ̃́tɑ́ɑ̀? Wèè n yɑ̀ ò yɑ̀mu n cicɛ. Dɛ̀ dòmmɛ kɑ̀ɑ tú n dɑ bɛnkɛ n cicɛ? 
A í tɑ̃́ kɛ dò n cicɛ bo m miɛkɛ kɛ̀ m bo o miɛkɑɑ̀? N di nɑ́ɑ́ ntìì nɑ́ɑǹtì í tú n kpɛti, n cicɛ wèè bo m miɛkɛ weè pĩ nho tɔ̃mmú. 
Nni ntɑ̃́nɛ̀ m bomu n cicɛ miɛkɛ kòo bo m miɛkɛ, kɛ̀ di mɛ nyí n tɑ̃́ tũ nni ntɑ̃́nɛ̀ m pĩ́ mùù tɔ̃mmú. 
N di nɑ́ɑ́ mmɔ́mmuɔ nwe, wèè n tɑ̃́ ò bo pĩ́ mmutɔ̃mmú m pĩ́ mmù kɛ̀ mù ndɛu kɛ́pɛ̃ɛ̃tɛ́ m pĩ́ mmù kɛ yɛ̃́ n kũntimɛ̀ n cicɛ borɛ̀. 
Di bo mɔɔ dɛ̀ɛ̀ kó dimɑ̀ɑ̀ nɛ̀ n kó diyètìrì, m bo dɛ̀ dɔ̀ɔ̀ kɛ́dɛ́úkùnnɛ n cicɛ yètìrì. 
Kɛ̀ di mɔɔ dɛ̀mɑrɛ̀ nɛ̀ n kó diyètìrì m bo dɛ̀ dɔ̀ɔ̀. 
Kɛ̀ di n dɔ́ mpĩ́nnɛ̀ n tɑnnɔ̀. 
M bo bɑ́ntɛ̀ n cicɛ kòo di duɔnní obɑ́ɑ́ǹtì tɔù, kòo ndi bonɛ̀ sɑ̃́ɑ̃̀. 
Wèè tu Muyɑɑ́nsɑ̀ɑ̀mù timɔ́mmɔnti kɔ̃mu, kutenkù kuu nkɔbɛ bɑ́ɑ́ nɑ kɛ́ mù cɔutɛ́, bɛ̀ í mù wùó mbɛ̀ mɛ nyí mù yɛ̃́, díndi mɛ mmù yɛ̃́mu, kɛ̀ mù di bonɛ̀ kɛ bo di miɛkɛ. 
N yí yóó di yóu kɛ̀ dí ndò nyiciribí, n wɛ̃tinímu di borɛ̀. 
Dɛ̀ bo yíɛ́ sɑ́m̀pɔ́ kutenkù kuu nkɔbɛ tɛ̃́nkɛ bɑ́ n yɑ̀, díndi mɛ n yóó n yɑ̀mu kɛ yɛ̃́ n fòùmɛ̀ kɛ̀ di yó nfòù. 
Dɛ kó diyiè di bo bɑntɛ́ kɛ dò m bo n cicɛ miɛkɛ, kɛ̀ di bo m miɛkɛ, kɛ̀ m bo di miɛkɛ. 
Wèè yie n nɑ́ɑǹtì kɛ dɔɔri tì yɛ̃mmɛ̀ weè n dɔ́, kɛ̀ wèè n dɔ́, n cicɛ bo nhò dɔ́, kɛ̀ nní nhò dɔ́ kóò bɛnkɛ m mɑ́ɑ̀. 
Kɛ̀ Sudii ò bekɛ kɛ dɔ̀: N yiɛ̀ ndɛ̀ dòmmɛ kɑ̀ɑ bo ti bɛnkɛ ɑ mɑ́ɑ̀, kɛ bɑ́ɑ́ bɛnkɛ ɑ mɑ́ɑ̀ kutenkù kɔbɛ. 
Kɛ̀ Yesu ò tɛ̃̀ńnɛ́ kɛ dɔ̀: Wèè n dɔ́ ò yó mpĩ́mmu n nɑ́ɑǹtì kɛ̀ n cicɛ nhò dɔ́, kɛ̀ tí kɔtɛní kɛ́ nhò bonɛ̀. 
Wèè í pĩ́ n nɑ́ɑǹtì ò í n dɔ́. N di nɑ́ɑ́ ntìì nɑ́ɑǹtì í tú n kpɛti, n cicɛ wèè n tɔ̃nní o kpɛti nti. 
N ti ndi nɑ̀kɛ́mu kɛ kpɑɑ́ kɛ di bonɛ̀. 
Obɑ́ɑǹtì Muyɑɑ́nsɑ̀ɑ̀mù n cicɛ yóó duɔnní mù n kó difɔ̃̀tìrì mù bo di nɑ́kɛ́ dɛmɔu, kɛ́ di dentɛ n di nɑ̀kɛ́ tì timɔu. 
N di duɔ́ ndiwɛ̀ì ndi, n kó diwɛ̀ì. N yí dì ndi duɔ́ nkutenkù kuù nkɔ̃mɛ, di yɛ̀mmɛ̀ bɑ́ɑ́ cɑɑ̀rɛ̀, di kɔ̃m̀bùɔ̀tì bɑ́ɑ́ do. 
Di kèèmu n di nɑ̀kɛ́mɛ̀ kɛ tú n kũnti n cicɛ borɛ̀ kɛ wɛ̃tiní. Kɛ̀ di do n dɔ́ dɛ̀ nɑ ndi nɑɑtimu kɛ yɛ̃́ n kũntimɛ̀ n cicɛ borɛ̀ wèè dɛu kɛ m pɛ̃ɛ̃tɛ́. 
N tì ndi nɑ̀kɛ́mu kɛ̀ dɛ̀ mu nyí tùɔ̀kɛní, kɛ̀ dɛ̀ bo yie nkɛ̀ dɛ̀ tùɔ̀kɛní dìì mɔ̀nnì di ntɑ̃́. 
N tɛ̃́nkɛ í yóó di nɑ́kɛ́ tìmɑtì pɛ́u, kɛ yɛ̃́ kutenkù kuu nkóò kpɑ̀ɑ̀tì kèrínímɛ̀, ò í mɔkɛ muwɛ̃rímú mɑmù n yĩ́nkɛ̀. 
Ò kèríní kɛ̀ kutenkù kɔbɛ bɛɛ̀ bo bɑntɛ́ kɛ dò n dɔ́ n cicɛ kɛ dɔɔri ò dɔ́mɛ̀. Ítɛ́nɛ̀ kɛ̀ tí pɛ̃ɛ̃tɛ́.
