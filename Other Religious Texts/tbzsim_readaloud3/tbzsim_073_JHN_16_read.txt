Isɑ̃ɑ̃.
16.
N tì ndi nɑ̀kɛ́ dɛ̀ bo yiemmɛ̀ mmɛ kɛ̀ di bɑ́ɑ́ duó. 
Bɛ̀ bo ndi bɛtì titííntouti, mɔ̀nnì mɑrì kɛ́ ndi kɔù kɛ̀ bɛ yɛ̀mmɛ̀ dò mbɛ̀ pĩ́ Kuyie ntɔ̃mmú. 
Bɛ̀ yó ndɔɔri mɛmmɛ bɛ̀ í yɛ̃́ mɛ̀ɛ̀ kpɛ́í nkɛ n cicɛ, bɛ̀ mɛ nyí n yɛ̃́ m mɔ́mmuɔ. 
N ti ndi nɑ̀kɛ́mu kɛ̀ dɛ mɔ̀nnì tùɔ̀kɛní, di dentɛní n di nɑ̀kɛ́ tì, n yí tì ndi nɑ̀kɛ́ nɛ̀ dimɔ̀nnì kɛ yɛ̃́ n di bonɛ̀mɛ̀. 
Di m̀mɔ̀nnì n kũnti wèè n tɔ̃nní o borɛ̀ ndɛ, di kóò mɔù mɛ nyí m békú n kɔri kɛ̀. 
Kɛ̀ di yɛ̀mmɛ̀ nɛ́ cɑ̀ɑ̀rɛ̀ n di nɑ̀kɛ́ mɛ̀ɛ̀ kpɛ́í n kɔri kɛ̀. 
N di nɑ́ɑ́ mɔ́mmuɔ nwe, kɛ̀ n kò dɛ̀ bo ntú di kó mɛsɑ̀ɑ̀, nkɛ yɛ̃́ kɛ̀ n yí kò, Obɑ́ɑǹtì sɑ̀ɑ̀wè í kɔriní, kɛ̀ m mɛ nkò m bo wè ndi duɔnní. 
Obɑ́ɑǹtì bo kɔtɛní dìì mɔ̀nnì, ò bo nte kɛ̀ bɛnìtìbɛ̀ɛ bɑntɛ́ bɛ yɛi, kɛ́bɑntɛ́ dɛ̀ɛ̀ tú timɔ́mmɔnti, kɛ́bɑntɛ́ Kuyie mbeéntì bomɛ̀. 
Bɛ yɛi ntú bɛ̀ í n tɑ̃́mɛ̀. 
Bɛ̀ í bɑntɛ́mɛ̀ kɛ dò n dɔɔri Kuyie ndɔ́mɛ̀ kɛ kũnti ku borɛ̀, di tɛ̃́nkɛ í yó n yɑ̀, 
kɛ́bɑntɛ́ kɛ dò nKuyie mbeéntì bo kɛ yɛ̃́ Kuyie mbekɛ́nɛ̀mɛ̀ kutenkù kuu nkóo kpɑ̀ɑ̀tì kɛ dèè. 
Tinɑ́ɑǹtì kpɑɑ́mu pɛ́u n nɑ ndi nɑ̀kɛ́ tì, di mɛ mmu mbɑ́ɑ́ nɑ kɛ́ tì kèè di mmɔ̀nnì. 
Muyɑɑ́nsɑ̀ɑ̀mù mùù tu timɔ́mmɔnti mù bo kɔtɛní dìì mɔ̀nnì, mù bo di nɑ́kɛ́ timɔu, kɛ yɛ̃́ mù í yó nnɑ́ɑ́mmɛ̀ nɛ̀ mu kó muwɛ̃rímú, mù yó nnɑ́ɑ́ mmù kèè tìnti kɛ di nɑ́kɛ́ dɛ̀ɛ̀ kpɑɑní. 
Mù bo nni nsɑ̃ntí kɛ yɛ̃́ mù yó nnɑ́ɑ́mmɛ̀ mù kèè tìnti m borɛ̀. 
N cicɛ mɔkɛ dɛ̀ dɛmɔu dɛ̀ tú n kpɛrɛ ndɛ, dɛɛ̀ te kɛ̀ n tu Muyɑɑ́nsɑ̀ɑ̀mù bo ndi nɑ́ɑ́ mmù kèè tì m borɛ̀. 
Dɛ̀ bo yíɛ́ sɑ́m̀pɔ́ di tɛ̃́nkɛ bɑ́ n yɑ̀, kɛ̀ dɛ̀ɛ wɛ̃tɛ kɛ́yíɛ́ sɑ́m̀pɔ́ kɛ̀ di n yɑ̀, kɛ yɛ̃́ n kũntimɛ̀ n cicɛ borɛ̀. 
Dɛ mɔ̀nnì kòo tɑncɔ́ùmbɛ̀ mɑbɛ̀ mbékú bɛtɔbɛ̀ kɛ tú: Ɔ̃̀nti nɑ́ɑǹtì tu, dɛ̀ bo yíɛ́ sɑ́m̀pɔ́ di tɛ̃́nkɛ bɑ́ n yɑ̀, kɛ̀ dɛ̀ɛ wɛ̃tɛ kɛ́yíɛ́ sɑ́m̀pɔ́ kɛ̀ di n yɑ̀, kɛ yɛ̃́ n kũntimɛ̀ n cicɛ borɛ̀? 
Bɑ ntú dɛ̀ bo yíɛ́ sɑ́m̀pɔ́? Ti í bɑntɛ́ mùù mɛ ntú. 
Kɛ̀ Yesu bɑntɛ́ bɛ̀ dɔ́mɛ̀ kóò bekɛ kɛ́ bɛ̀ nɑ̀kɛ́ kɛ dɔ̀: Di dɔ́ kɛ́bɑntɛ́ mùù tu dɛ̀ bo yíɛ́ sɑ́m̀pɔ́ kɛ̀ di tɛ̃́nkɛ bɑ́ n yɑ̀, kɛ̀ dɛ̀ɛ yíɛ́ sɑ́m̀pɔ́ kɛ̀ di wɛ̃tɛ kɛ́ n yɑɑ̀? 
N di nɑ́ɑ́ mmɔ́mmuɔ nwe, di bo kɔmmú, kɛ̀ kutenkù kɔbɛ nyɑ̃nku. Di yɛ̀mmɛ̀ bo cɑɑ̀rɛ̀, kɛ dɛ kó mɛyɛǹcɑɑ̀rìmɛ̀ nɛ́ nɑɑ́ ndiwɛ̀ì. 
Onitipòkù pɛí dìì mɔ̀nnì dɛ̀ ɔ̃ nhò yóùmu, kɛ yɛ̃́ o mɔ̀nnì tùɔ̀kɛmɛ̀ kɛ̀ dɛbirɛ yɛ̀nní kɛ dèè wèe yɛ̃̀nko kuyonku, kɛ yɛ̃́ ò pɛitɛ́mɛ̀ onìtì. 
Di kpɛrɛ mɛ ndò, di yɛ̀mmɛ̀ cɑ̀ɑ̀rɛ̀mu di m̀mɔ̀nnì, kɛ bo nɑrikɛ di bo wɛ̃tɛ kɛ́ n yɑ̀ dìì mɔ̀nnì, bɑ́ òmɔù bɑ́ɑ́ nɑ kɛ́dɛitɛ dɛ kó kuyɛǹnɑɑtí. 
Kɛ̀ dɛ kó diyiè tùɔ̀kɛ di tɛ̃́nkɛ í yó mbekɛ tìmɑtì n di nɑ́ɑ́ mmɔ́mmuɔ nwe dɛ kó diyiè di mɔ́ú dɛ̀ɛ̀ kó dimɑ̀ɑ̀ n cicɛ nɛ̀ nkó diyètìrì ò bo dɛ̀ ndi pɑ̃. 
Nɛ̀ yíenní di mu nyí mɔɔ dɛ̀mɑrɛ̀ Kuyie nɛ̀ n kó diyètìrì, mɔɔnɛ̀ kɛ̀ kù di pɑ̃, kɛ̀ di yɛ̀mmɛ̀ ńnɑɑti. 
N di nɑ̀kɛ́ dɛ kó tinɑ́ɑǹtì nɛ̀ dinɑ́ɑǹhɑ̃ntírì ndi, dɛ mɔ̀nnì kpɑɑnímu, n tɛ̃́nkɛ í yo ndi nɑ́ɑ́nnɛ̀ kɛ di ɑ̃mmù, dɛ mɔ̀nnì n yó ndi nɑ́ɑ́ mpɑ́íí nwe n cicɛ kpɛti. 
Dɛ kó dimɔ̀nnì di bo mmɔ́ú n cicɛ di dɔ́ dɛ̀ nɛ̀ n kó diyètìrì. Dɛ̀ í tú n yɛ̃ n yó nhò bɑ́ɑ́ ndi kpɛ́í. 
N cicɛ mɔ́mmuɔ ndi dɔ́mu di n dɔ́ mɛ̀ɛ̀ kpɛ́í, kɛ tɑ̃́ kɛ do m bonní o borɛ̀. 
N yìtɛ́ní n cicɛ borɛ̀ ndɛ kɛ kɔ̀tɛní kutenkù, kɛ iti kutenkù kɛ kũnti n cicɛ borɛ̀. 
Kòo tɑncɔ́ùmbɛ̀ɛ ò nɑ̀kɛ́ kɛ dɔ̀: Ntɛ ɑ nɑ́ɑ́mmɛ̀ pɑ́íí bɑ́ ɑ í ɑ̃mmù. 
Di m̀mɔ̀nnì ti bɑntɛ́mu kɛ dò nhɑ yɛ̃́ dɛmɔu, kɛ̀ dɛ̀ í dò nhòmɔùu bennɛ́ kɛ́ ndɑ békú tìmɑtì. Dɛɛ̀ te kɛ̀ ti tɑ̃́ kɛ dò nhɑ bonní Kuyie mborɛ̀. 
Kɛ̀ Yesu bɛ̀ bekɛ kɛ dɔ̀: Di tɑ̃́ di mmɔ̀nnɑ̀ɑ̀? 
Dɛ mɔ̀nnì duunnímu kɛ tùɔ̀kɛní, kɛ̀ di bo cíɛ́tɛ́ kɛ́kũṹ ndi cɛ̃́ĩ kɛ́ n yóu. M mɛ nyí yó mbo m mɑ́ɑ̀, n cicɛ m bonɛ̀mu. 
N tì ndi nɑ̀kɛ́ kɛ̀ dɛ̀ bo ndi nɑɑtimu ti kó mɛtɑummɛ̀ miɛkɛ, bɛ̀ yóó di fɛ̃́ũmmu, di yɛ̀mmɛ̀ nɛ́ bɑ́ɑ́ cɑɑ̀rɛ̀ n nɑmu kutenkù.
