Isɑ̃ɑ̃.
15.
Kɛ̀ Yesu dɔ̀: Míì tú fínyĩ̀ tie kó dikũ̀nnì kɛ̀ n cicɛ tú wèè mù te. 
Ibɑkɛ ìì n tukúnɛ̀ kɛ í pɛí ò ì kɔ̃ũmu, kɛ̀ ibɑkɛ ìì pɛí kɛ ì kɛ́í kɛ̀ ì bo dɔ̀kɛ pɛitɛ́. 
Tinɑ́ɑǹtì n di nɑ̀kɛ́ tì diɛ́ nkɛ dì cìɛ́mu, kɛ̀ di wenkɛ pɑ́íí. 
Di nni ntɑunɛ̀ kɛ̀ n di tɑunɛ̀. Kutebɑku bɑ́ɑ́ nɑ kɛ́pɛitɛ́ kɛ í tɑunɛ̀ mutie. Mɛɛ̀ botí nku di bɑ́ɑ́ nɑ kɛ́pɛitɛ́ kɛ í n tɑunɛ̀. 
Míì tú mutie kũ̀nnì kɛ̀ di tú ibɑkɛ, wèè n tɑunɛ̀ kɛ̀ nh ò tɑ̀unɛ̀ weè pɛí yɛbɛ pɛ́u, kɛ̀ di í n tɑunɛ̀ di bɑ́ɑ́ nɑ kɛ́dɔɔ̀ dɛ̀mɑrɛ̀. 
Wèè í n tɑunɛ̀ Kuyie nhò dootóomu, kutebɑku ɔ̃ɔ̃ kpeí kɛ̀ bɛ̀ɛ kù dootóo mɛ̀ɛ̀ botí, kɛ̀ bɛ̀ɛ kù tùótɛ́ kɛ́cɔ́u nkɛ̀ kùu cɔ́útɛ́. 
Kɛ̀ di n tɑunɛ̀ kɛ̀ n nɑ́ɑǹtì bo di miɛkɛ, mɔɔnɛ̀ di dɔ́ dɛ̀, m bo dɛ̀ ndi pɑ̃. 
Kɛ̀ di pɛitɛ́ yɛbɛ pɛ́u, mɛm̀mɛ n cicɛ yètìrì bo dɛukɛmɛ̀, kɛ̀ dɛ̀ɛ bɛnkɛ di túmɛ̀ n tɑncɔ́ùmbɛ̀. 
N cicɛ n dɔ́ mɛ̀ɛ̀ botí, m mɛ ndi dɔ́, n fííkúnɛ̀. 
Kɛ̀ di pĩ́ n tɑnnɔ̀ m bo ndi dɔ́ kɛ̀ di nfííkú, m pĩ́mmɛ̀ n cicɛ tɑnnɔ̀ kòò n dɔ́ kɛ̀ n fííkú. 
N ti ndi nɑ̀kɛ́ kɛ̀ dɛ̀ bo yie nkɛ̀ di mɔɔtɛ n kó diwɛ̀ì ndi kɛ̀ dì di píɛ. 
Ntɛnɛ̀ n di nɑ̀kɛ́ dìì tɑnnùù: Ndɔ́nɛ̀ ditɔbɛ̀ n di dɔ́ mɛ̀ɛ̀ botí. 
Wèè duɔ́ nho fòmmu o kɔbɛ kpɛ́í, weè bɛ̀ dɔ́ mɛdiɛ̀. 
Kɛ̀ di dɔɔri n di nɑ̀kɛ́ dìì nùù di tú n nɛ́pobɛ̀ mbɛ. 
N tɛ̃́nkɛ í di tú bɛtɔ̃mbɛ̀, kɛ yɛ̃́ otɔ̃ntì í yɛ̃́mɛ̀ wèè ò duɔ́ mmutɔ̃mmú ò dɔ̀ɔ̀ri dɛ̀. N di tú n nɛ́pobɛ̀ mbɛ, kɛ yɛ̃́ n di bɛnkɛmɛ̀ n cicɛ m bɛnkɛ dɛ̀ dɛmɔu. 
Díì í n tɑ̃ɑ̃tɛ́ míì di tɑ̃ɑ̃tɛ́, kɛ tú di kɔtɛ kɛ́pɛitɛ́ yɛbɛ yɛ̀ɛ̀ yó mbo sɑ̃́ɑ̃̀ kɛ̀ dɛ̀ɛ yie nkɛ̀ di mɔɔ dɛ̀ɛ̀ kó dimɑ̀ɑ̀ n cicɛ nɛ̀ n kó diyètìrì, wèe dɛ̀ ndi pɑ̃. 
N di duɔ́ dìì tɑnnùù tú: Di bo ndɔ́mɛ̀ ditɔbɛ̀. 
Kɛ̀ kutenkù kɔbɛ di níí, nyɛ̃́nɛ̀ kɛ dò mbɛ̀ ketɛ́ kɛ níí mínwe. 
Kɛ̀ di do ntú kutenkù kɔbɛ bɛ̀ nɑ ndi dɔ́mu, di mɛ nyí tú ku kɔbɛ, kɛ yɛ̃́ n di dènnɛnímɛ̀ kutenkù, dɛɛ̀ te kɛ̀ bɛ̀ di níí. 
Dentɛnɛ̀ní n di nɑ̀kɛ́ tì: Otɔ̃ntì mɔù í dɛunɛ̀ wèè ò duɔ́ mmutɔ̃mmú. Kɛ̀ bɛ̀ n fɛ̃̀ṹ mbɛ̀ yóó di fɛ̃́ũmmu kɛ̀ bɛ̀ mɛ nyie n kpɛti bɛ̀ yóó yiemmu di kpɛti. 
Bɛ̀ yóó di fɛ̃́ũ ndi tú mɛ̀ɛ̀ kpɛ́í nkɛ n kɔbɛ, kɛ yɛ̃́ bɛ̀ í yɛ̃́mɛ̀ wèè n tɔ̃nní. 
Kɛ̀ n do í kɔ̀tɛní kɛ bɛ̀ nɑ̀kɛ́ dɛ̀ nɑ bɑ́ ntú bɛ kó mɛyetímɛ̀, n nɛ́ mɛ nkɔ̀tɛnímɛ̀ kɛ bɛ̀ nɑ̀kɛ́ bɛ̀ bɑ́ɑ́ nɑ kɛ́peri. 
Wèè n níí ò níímmu n cicɛ. 
Kɛ̀ n do í pĩ mmutɔ̃mmú otɔù mɔù í nɑ kɛ pĩ̀ḿmú, kɛ̀ bɛ̀ yɑ̀, dɛ̀ nɑ bɑ́ ntú bɛ kó mɛyetímɛ̀. Bɛ̀ mɛ mmù yɑ̀mu kɛ ti níí, mmí nnɛ̀ n cicɛ. 
Tìì wɑ̃̀ri bɛ kó ikuɔ́ miɛkɛ tiì dɔ̀ɔ̀ tìì tú: Bɛ̀ n níí dɛtetìrɛ̀. 
Obɑ́ɑǹtì Muyɑɑ́nsɑ̀ɑ̀mù n cicɛ yó n duɔ́ mmù kɛ̀ n di duɔ́, mù bo tuɔkɛní dìì mɔ̀nnì mù yó nnɑ́ɑ́ n kpɛ́í nkɛ. 
Kɛ̀ di múnkɛ n nɑ́ɑ́ n kpɛ́í, nkɛ yɛ̃́ di m bonɛ̀mɛ̀ nɛ̀ diketírìní.
